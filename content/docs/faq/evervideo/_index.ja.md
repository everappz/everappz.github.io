{{% details title="Evervideoとは何ですか？" closed="true" %}}
**EvervideoはiPhone、iPad、Mac向けのフル機能HD・4Kクラウド動画プレーヤー**で、任意のクラウドストレージアカウント、NAS、またはメディアサーバーをファイルの完全な制御が可能な個人動画ライブラリに変換します。<br><br>

カスタムFFmpegベースのプレーヤーエンジンとハードウェアアクセラレーション付きH.264およびHEVCデコードを基盤とし、Evervideoはほぼすべての最新のコンテナとコーデック（MP4、MKV、AVI、MOV、FLV、WMV、WebM、TS、M2TS、その他多数のFFmpegフォーマット）を再生し、プライマリおよびセカンダリ字幕をサポートし、音声および映像イコライザーを内蔵し、ピクチャー・イン・ピクチャーモードで動作するため、他のアプリを使用しながら視聴を続けられます。

{{% details title="Evervideoは安全ですか？" closed="true" %}}
Evervideoは接続されたクラウドサービスとやり取りするために公式SDKとセキュアな接続のみを使用しています。ログインとパスワードはアプリが利用できません。アプリからクラウドサービスへのすべてのリクエストは暗号化されています。<br>
ログインとパスワードを入力すると、アプリはクラウドサービスプロバイダーが提供する公式認証ページを表示し、認証プロセス全体がアプリの外部で行われます。クラウドサービスプロバイダーは認証成功後にアプリに認証トークンを送信し、そのトークンがAPIコールに使用されます。<br><br>

認証トークンはサードパーティアプリケーションがクラウドストレージとやり取りできるようにするデジタルキーです。認証トークンはキーチェーンと呼ばれる安全なシステムストレージに保存されます。接続されたクラウドサービスからデバイスにファイルをダウンロードでき、それらのファイルはアプリの「ドキュメント」ディレクトリに配置されます。これらのファイルはいつでも組み込みのファイルマネージャーを使用して削除できます。<br>
アプリは接続されたクラウドアカウントからの情報を共有しません。Webブラウザーでアカウント設定ページを開くことで、いつでもクラウドアカウントへのアクセスを取り消すことができます。<br><br>

認証トークンを拒否するには、Webブラウザーでアカウントにログインして設定ページに移動してください。そこでクラウドアカウントに接続されているすべてのサードパーティアプリを見つけ、そのアプリをもう使用したくない場合は削除できます。<br><br>

完全なチュートリアルはこちら：<br>
[GoogleアカウントからサードパーティアプリBを切断する方法](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

アプリ内で接続されたクラウドアカウントを切断することもでき、認証トークンもデバイスから削除されます。デバイスからアプリを削除すると、ダウンロードされたすべてのデータとアクセストークンも削除されます。<br><br>

{{% details title="Evervideoはどのような動画フォーマットをサポートしていますか？" closed="true" %}}
**EvervideoはiPhone、iPad、Macでほぼすべてのモダンなコンテナとコーデックを再生します**。これは内蔵FFmpegエンジンとハードウェアアクセラレーション付きH.264/HEVCデコードの組み合わせによるものです。<br><br>

**コンテナ：** MP4、M4V、MKV、MOV、AVI、FLV、WMV、ASF、WebM、TS、M2TS、MTS、MPG、MPEG、OGV、3GP、3G2、F4V、RM、RMVB、VOB、DAT、その他多数。<br>
**映像コーデック：** H.264 (AVC)、H.265 (HEVC)、VP9、VP8、AV1、MPEG-2、MPEG-4、MJPEG、ProRes、Theora、WMV、FFmpegがサポートするほぼすべての他のコーデック。<br>
**音声コーデック（動画内）：** AAC、MP3、FLAC、ALAC、OGG/Vorbis、OPUS、AC-3、EAC-3、DTS、AMR、WMA、APE。<br>
**字幕：** SRT、VTT (WebVTT)、ASS/SSA（libass経由でレンダリング）、埋め込み画像およびテキストトラック。<br>
**ストリーミングプロトコル：** HTTP/HTTPS、HLS (m3u8)、RTSP（IPカメラとIPTV）。<br>
**ダイレクトストリーミング：** SMB/WebDAV/FTP/SFTP/NFS/DLNA。<br>
{{% /details %}}

{{% details title="EvervideoはiPhoneでMKVファイルを再生しますか？" closed="true" %}}
**はい — EvervideoはiPhone、iPad、MacでMKVファイルをネイティブに再生します**。変換不要で、H.264、HEVC、VP9、またはAV1映像、複数の音声トラック、埋め込みSRT/ASS字幕を含むMKVコンテナに対応しています。<br><br>

これはAppleのデフォルトプレーヤーの代わりにEvervideoをインストールする主な理由の一つです：iOSはMKVを全く開けませんが、EvervideoはFFmpegエンジンを通じてそれらを処理します。MKVファイルをダウンロードすることなくクラウドストレージやNASから直接ストリーミングできます。
{{% /details %}}

{{% details title="Evervideoは4KおよびHDR動画をサポートしていますか？" closed="true" %}}
**はい — Evervideoはデコードするためのハードウェアを持つすべてのデバイスで4K（Ultra HD）動画とHDRエンコードコンテンツを再生します**。最新のiPhone、iPad、Apple Silicon Macはすべてハードウェアで4K H.264と4K HEVCをデコードするため、再生はスムーズでバッテリーに優しいです。<br><br>

クラウドからストリーミングされる4Kの最良の結果を得るには、**設定 → プレーヤー → ファイル読み込み**で**プリロード時間**を増やしてバッファが高ビットレートのファイルに追いつけるようにし、モバイルデータではなくWi-Fiネットワークに接続してください。
{{% /details %}}

{{% details title="EvervideoはハードウェアH.264およびHEVCデコードをサポートしていますか？" closed="true" %}}
**はい — EvervideoはそれらをサポートするすべてのiPhone、iPad、MacでデフォルトでハードウェアH.264（AVC）およびH.265（HEVC）デコーダーを使用します**。これにより、純粋なソフトウェアデコードと比較して、よりスムーズな再生、低いバッテリー使用量、低いデバイス温度が実現します。<br><br>

H.264とHEVCのハードウェアデコードは**設定 → プレーヤー → 映像 → ハードウェアデコード H.264/H.265**で個別に切り替えられます。特定のファイルに互換性の問題がある場合（破損したストリーム、エキゾチックなプロファイル）、そのファイルのハードウェアデコードを無効にしてFFmpegソフトウェアデコードにフォールバックしてください。
{{% /details %}}

{{% details title="EvervideoはPicture-in-Picture（PiP）をサポートしていますか？" closed="true" %}}
**はい — EvervideoはiPhoneとiPadでPicture-in-Pictureを完全にサポートしています。** プレーヤーのPiPアイコンをタップすると、動画は他のすべてのアプリの上に浮かぶウィンドウで再生を続けます。<br><br>

浮かぶウィンドウを任意のコーナーにドラッグし、ピンチでサイズ変更し、一度タップして基本的な再生/一時停止/スキップコントロールを表示し、小さな展開ボタンをタップしてEvervideoに戻ります。PiPは**Evervideoが再生するすべての動画フォーマット**で動作し、クラウドからストリーミングされるファイルやRTSP IPカメラストリームも含まれ、電話がロックされている間も動作し続けます。
{{% /details %}}

{{% details title="EvervideoはローカルビデオのためのYouTubeスタイルのプレーヤーUIを持っていますか？" closed="true" %}}
**はい — Evervideoのデフォルトレイアウトはあなた自身の動画のためのYouTubeスタイルの体験を中心に構築されています：画面の上部に常時表示されるコンパクトな動画プレーヤーがあり、その下でライブラリをブラウズします。**<br><br>

**コンパクトプレーヤー**はすべてのタブ（最近の項目、お気に入り、メディアライブラリ、プレイリスト、ファイル、設定）で常に表示されるため、再生を一切中断することなく、ブラウズ、検索、整理、次の動画のキューイングができます。コンパクトプレーヤーをタップして全画面表示に展開し、下にスワイプして動画を止めずにコンパクトに折り畳みます。macOSでは、コンパクトプレーヤーをデスクトップでの真のピクチャー・イン・ピクチャースタイルのマルチタスキングのために、別の**常時最前面のフローティングウィンドウ**に切り離せます。<br><br>

YouTubeとは異なり、広告、アルゴリズムによるレコメンデーション、自動再生の通知、トラッキングはありません。iCloud、Google Drive、Dropbox、NAS、Plex/Jellyfin/Emby、その他どこからでも独自の動画ライブラリを完全に管理できます。
{{% /details %}}

{{% details title="EvervideoはInsta360やその他の360カメラからの360°動画を再生しますか？" closed="true" %}}
**はい — Evervideoは360°/VR（球面）動画を直接再生します。前処理や変換は不要です**。Insta360（One X、X3、X4、ONE RS、GO 3）、GoPro Max、Ricoh Theta、Samsung Gear 360、Vuze、その他の正距円筒360°ソースからの映像が対象です。<br><br>

360°動画をiCloud Drive、Google Drive、Dropbox、NASに置くか、iOS写真ライブラリから再生するだけです。Evervideoは球面フォーマットを認識し、自動的にVRビューポートレンダリングに切り替えます。そこから：<br>

- **デバイスを回転させる** — iPhoneまたはiPadを物理的に傾けて回転させると、動画ビューポートがリアルタイムで回転してシーンを見回せます。<br>
- **指でドラッグ**してデバイスを動かさずに任意の方向にパンする。<br>
- **ピンチしてズーム**して球面ビューを拡大縮小する。<br>
- プレーヤーのその他のアクションメニューで投影/表示モードを切り替え、完全な没入型体験のためにVRヘッドセットケースを使用する。<br><br>

これはInsta360の映像がiPhoneとiPadですぐに動作することを意味します。追加のアプリ、トランスコーディング、クラウドサブスクリプションは不要です。
{{% /details %}}

{{% details title="映像イコライザーとプリセットの使い方は？" closed="true" %}}
**プレーヤーを開き、その他のアクション → 映像イコライザーをタップし、輝度、コントラスト、彩度、色相のスライダーをドラッグするか、プリセットを選択します。**<br><br>

Evervideoの**映像イコライザー**はFFmpegレンダリングパイプライン内で動作するリアルタイム画質調整ツールです：<br>

- **輝度** — 暗い場所で撮影した暗いシーンを明るくするか、露出オーバーの明るすぎる部分を抑える。<br>
- **コントラスト** — 褪せたコンテンツに力強さを加えるか、コントラストが強すぎるソースを和らげる。<br>
- **彩度** — 色あせた色を温かくするか、映画的な外観のためにデサチュレートする。<br>
- **色相** — 緑または赤紫のかぶりを修正するためにカラーバランスをシフトする。<br><br>

お気に入りの設定を**カスタムプリセット**として保存し、将来の任意の動画に一度のタップで再適用できます。プリセットは**エクスポートおよびインポート**できるため、iPhone、iPad、Mac間で共有したり、バックアップを取ったりできます。映像イコライザーと**音声イコライザー**（独自のプリセットライブラリを持つ10バンドEQ）を組み合わせて、映像と音声の両方を完全にコントロールしてください。
{{% /details %}}

{{% details title="動画の回転やスケーリングモードを変更するには？" closed="true" %}}
**プレーヤーのその他のアクションをタップし、回転（0°/90°/180°/270°）またはスケーリングモード（フィット/フィル/ストレッチ/オリジナル）を選択します。**<br><br>

**回転**は横向きまたは逆さまに録画された動画に便利です。プレーヤーを離れることなく映像を回転させられます。**スケーリングモード**は映像が画面をどのように埋めるかを制御します：<br>

- **フィット** — 必要に応じてブラックバーで元のアスペクト比を保持する。<br>
- **フィル** — 必要に応じて動画をクロップして画面全体を埋める。<br>
- **ストレッチ** — 映像を歪めて画面を埋めるように引き伸ばす。<br>
- **オリジナル** — ネイティブ解像度を1:1で保持する。
{{% /details %}}

{{% details title="動画内の別の音声トラック（吹替、コメンタリー）を選択するには？" closed="true" %}}
**プレーヤーのその他のアクション（「...」）ボタンをタップして音声トラックを選択し、リストから目的のトラックを選んでください。**<br><br>

複数の音声トラック（別言語の吹替、ディレクターズコメンタリー、オリジナル/ライブミックス）を持つ動画の場合、Evervideoはすべての埋め込みトラックをその言語とコーデックとともに表示します。これはMKV、MP4、M2TS、および複数の音声ストリームを公開するその他のコンテナで機能します。
{{% /details %}}

{{% details title="Evervideoで字幕を追加または変更するには？" closed="true" %}}
**プレーヤーのその他のアクション（「...」）ボタンをタップして字幕を選択し、埋め込み字幕トラックを選ぶか、外部字幕ファイルを読み込むか、フォントを変更してください。**<br><br>

Evervideoは動画に埋め込まれたすべての字幕トラックを自動的に一覧表示します。外部字幕ファイルを読み込むには、**外部ファイル**を選択し、デバイス、iCloud Drive、または接続されたクラウドサービスから`.srt`、`.vtt`、`.ass`、または`.ssa`ファイルを選択します。**設定 → プレーヤー → 字幕**でデフォルトの字幕動作を設定することもできます。
{{% /details %}}

{{% details title="EvervideoはSRT、VTT、ASSの外部字幕ファイルをサポートしていますか？" closed="true" %}}
**はい — EvervideoはデバイスまたはAny接続されたクラウドサービス上のどこからでもSRT、VTT（WebVTT）、ASS、SSAフォーマットの外部字幕ファイルを読み込みます。**<br><br>

高度なスタイリング（カスタムフォント、色、位置、カラオケエフェクト）を持つASS/SSAファイルは、内蔵**libass**ライブラリのおかげで正しくレンダリングされます。ファンサブのアニメ、プロのキャプションファイル、プレゼンテーションに最適です。**設定 → プレーヤー → 字幕 → フォント**で字幕に特定のフォントを割り当てることもできます。
{{% /details %}}

{{% details title="クラウドストレージから削除せずにEvervideoから動画を削除するには？" closed="true" %}}
**任意の動画で「...」→ メディアライブラリから削除をタップすると、ライブラリデータベースからエントリが削除されますが、クラウドストレージ、NAS、またはiOS写真ライブラリの元のファイルはそのまま残ります。**<br><br>

ソースからファイルも削除したい場合は、代わりに**クラウドサービスから削除**または**ローカルファイルから削除**を選択してください。これらの破壊的な操作は元に戻せないため、複数の動画を選択している場合は注意してください。<br><br>

クラウドにある何も触れずにダウンロードされたコピーを削除するには、**ファイル**タブを開き、**このアプリのファイル**または**オフラインフォルダー**の下で動画を見つけ、**「...」→ 削除する**を使用してください。クラウドのオリジナルはそのままの場所に残ります。Evervideoはローカルコピーのみを削除します。
{{% /details %}}

{{% details title="Evervideoはどのクラウドサービスをサポートしていますか？" closed="true" %}}

**NASプロトコル：** SMB · WebDAV · FTP/FTPS · SFTP · NFS · DLNA/UPnP。<br>

**ライブおよびIPカメラストリーム：** RTSP。<br>

**S3互換オブジェクトストレージ：** AWS S3、Backblaze B2、Wasabi、Cloudflare R2、MinIO、DigitalOcean Spaces。<br><br>

プレミアムユーザーは無制限の数のサービスを接続できます；無料版は3つに制限されています。

{{% details title="EvervideoはPlex、Jellyfin、Emby、Subsonic、Navidromをサポートしていますか？" closed="true" %}}
**はい — EvervideoはPlex Media Server、Jellyfin、Emby、Subsonic、Navidromにネイティブに接続します**。これにより、基盤となるファイル共有を公開することなくセルフホスト型の動画ライブラリを直接ストリーミングできます。<br><br>

- **Plex Media Server** — **ファイル → クラウドストレージに接続 → Plex**をタップし、Plexアカウントでサインインしてサーバーを選択します。同じローカルネットワーク上のPlexサーバーも**利用可能なデバイス**セクションで自動検出されます。<br>
- **Jellyfin**（オープンソース）— **ファイル → クラウドストレージに接続 → Jellyfin**をタップし、サーバーURL（`http://server-ip:8096`のようなもの）とクレデンシャルを入力します。<br>
- **Emby**（商用）— **ファイル → クラウドストレージに接続 → Emby**をタップし、サーバーURLとログインを入力します。<br>
- **Subsonicおよびサブソニック互換サーバー** — **ファイル → クラウドストレージに接続 → Subsonic**をタップし、サーバーURLとクレデンシャルを入力します。同じAPIパスが**Navidrome**、**Airsonic**、**Funkwhale**、**Gonic**、**Logitech Media Server (LMS)**、**Ampache**でも機能します。<br><br>

接続後、各サーバーはファイルタブのクラウドアカウントの隣に表示されます。映画、TVショー、ホームビデオ、音楽、プレイリスト、コレクションをブラウズし、オフライン再生のためにダウンロードし、プレーヤーでアイテムをキューに入れ、グローバルメディアライブラリに取り込むことができます。すべてEvervideoを離れることなく行えます。

{{% details title="EvervideoはSMB、WebDAV、FTP/SFTP、NFS、DLNAで接続しますか？" closed="true" %}}
**はい — EvervideoはすべてのNASおよびファイル共有プロトコルをサポートしています：SMB（SMB1、SMB2、Auto）、WebDAV（HTTP/HTTPS）、FTP/FTPS、SFTP（パスワードまたは公開鍵認証）、NFS、DLNA/UPnP。**<br><br>

これにより、ほぼすべてのNASデバイス（Synology、QNAP、WD My Cloud Home、Buffalo、Apple Time Capsule）、Linux/macOS/Windowsファイル共有、セルフホスト型Nextcloud/ownCloudサーバー、またはUPnP/DLNAメディアサーバーに、**ファイル → クラウドストレージに接続**メニューからすべて接続できます。

{{% details title="EvervideoはS3互換オブジェクトストレージをサポートしていますか？" closed="true" %}}
**はい — EvervideoにはS3互換コネクターが含まれています**。これは**AWS S3、Backblaze B2、Wasabi、Cloudflare R2、MinIO、DigitalOcean Spaces**、およびS3-APIエンドポイントを公開するその他のサービスで機能します。<br><br>

**ファイル → クラウドストレージに接続 → S3ストレージ**をタップし、エンドポイントURL、リージョン、アクセスキー、シークレットキー、バケット名を入力します。接続後、バケットは他のクラウドと同様に動作します。ブラウズ、ストリーミング、ダウンロード、キューイング、ライブラリへの追加が可能です。
{{% /details %}}

{{% details title="EvervideoでRTSPストリーム（IPカメラ、IPTV）を再生できますか？" closed="true" %}}
**はい — EvervideoにはネイティブRTSPサポートがあります**。任意の`rtsp://` URL（セキュリティカメラ、ドアベルカメラ、ベビーモニター、IPTVプロバイダー、放送フィード）に向けることができ、Evervideoはライブストリームをプルしてデコードします。<br><br>

**ファイル → オンラインリンク → リンクを追加**をタップし、完全なURL（`rtsp://camera-ip:port/stream-path`）を貼り付け、必要に応じてログインとパスワードを入力し、**完了**をタップします。RTSPストリームはPicture-in-Picture、コンパクトプレーヤーで動作し、通常の動画と同様にAirPlay 2とChromecastでキャストできます。

{{% details title="コンピューターからEvervideoに動画を転送するには？" closed="true" %}}

WebDAVプロトコルの場合、URLフィールドを除いてすべての手順は同じです。URLはhttp://server-nameの形式、またはサーバーがSSLをサポートしている場合はhttps://server-nameの形式である必要があります。<br><br>

完全なチュートリアルはこちら：<br>
[WebDAVを使用してNASストレージに接続し、iPhoneまたはMacで音楽を聴く方法](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac/)<br><br>

{{% details title="EvervideoのWi-Fi Drive機能の使い方は？" closed="true" %}}

**デスクトップブラウザーを使用したワイヤレス転送**<br>
1. アプリを起動：Evervideoを開きます。<br>
2. Wi-Fi経由で接続：ファイル →「コンピューター」→「Wi-Fi経由で接続する」に移動します。<br>

6. 組み込みのファイルマネージャーを使用してファイルをアップロード、ダウンロード、名前変更、または削除します。iPhoneに直接ファイルをドラッグ＆ドロップできます。<br>
7. 完了したら、iPhoneの「Wi-Fi Driveを停止」をタップします。<br><br>

注：最良のパフォーマンスのためにJavaScriptが有効になっていて最新のブラウザーバージョンを使用していることを確認してください。<br><br>

2. Evervideoアプリに表示されているサーバーURLを入力します。<br>
3. 「接続する」をクリックして、ネットワークドライブと同様にiPhoneのファイルを管理します。<br><br>

**Windowsファイルエクスプローラーを使用したファイル転送**<br>
1. 「PC」を右クリック →「ネットワークドライブの割り当て…」。<br>
2. 「フォルダー」フィールドにEvervideoアプリのサーバーURLを入力します。<br>
3. ドライブ文字を選択し、「完了」をクリックして、iPhoneのファイルを直接ブラウズします。<br><br>

アプリをインストールしたら、「ファイル」画面を開いて「このiPhoneのファイル」セクションまでスクロールします。そこから、複数のファイルを選択する必要がある場合は「ファイルを開く...」を、メディアフォルダーを選択したい場合は「フォルダーを開く...」を選択します。アプリはフォルダーのコンテンツをスキャンし、見つかったすべてのメディアファイルが選択されます。メディアフォルダーに移動し、「開く」をタップして選択を確認すると、ファイルがプレーヤーキューに追加されます。これらのファイルはアプリバンドルにコピーされることなく、選択した場所から直接再生されます。<br><br>

**クイックアクセスのためにフォルダーをお気に入りに追加する**<br>
デバイスにあるフォルダーをお気に入りに追加することでプロセスを簡略化します。これにより、音楽を再生したいたびに手順を繰り返す必要がありません。「ファイル」画面を開き、「クイックアクセス」セクションまでスクロールして「お気に入り」をタップし、「お気に入りのファイル」画面にアクセスします。右上隅のその他のアクションボタン（三点）をタップして「フォルダーを追加」を選択します。目的のフォルダーに移動して「開く」をタップして確認します。フォルダーが「お気に入りのファイル」に追加され、メディアファイルへのクイックアクセスが提供されます。<br><br>

**ライブラリへのローカルファイルのインポート**<br>
ライブラリ内でローカルファイルを整理したい場合は、「ライブラリ」画面を開き、右上隅の三点ボタンをタップして「+ メディアファイルを追加」を選択します。「このデバイスのファイル」メニュー項目を選択して「ファイルを開く...」をタップします。追加したいファイルを選択して「開く」をタップして確認します。アプリは選択されたファイルをスキャンし、メタデータで整理してライブラリに追加します。<br><br>

**プレイリストへのローカルファイルの追加**<br>
ローカルファイルをプレイリストに追加するには、「プレイリスト」画面を開き、右上隅のその他ボタンをタップします。「+ 新しいプレイリスト」を選択し、新しいプレイリストの名前を入力して、次の画面で「このデバイスのファイル」オプションを選択して「ファイルを開く...」をタップします。追加したいメディアファイルを選択して「開く」をタップして確認します。ファイルがプレイリストに追加され、その他ボタンを使用してトラックの順序を変更したりその他のアクションを実行したりできます。<br><br>

{{% details title="動画のダウンロード方法は？" closed="true" %}}
動画をダウンロードしてオフラインで視聴するには、まずクラウドストレージを接続する必要があります。<br>
「ファイル」画面を開いてクラウドストレージを接続するだけです。<br>
追加したら、クラウドから動画をダウンロードできます。<br><br>

**クラウドから動画をダウンロードするには**<br>
– 接続されたクラウドストレージを開きます。<br>
– ダウンロードしたいフォルダー内に移動します。<br>
– 右上隅のその他のアクション「...」ボタンをタップして「選択する」メニュー項目を選択します。<br>
– ダウンロードしたいファイルを選択して「ダウンロード」ボタンをタップします。<br><br>

**アーティスト/プレイリスト/アルバムのオフラインモードを有効にするには**<br>
– アーティスト/アルバム/プレイリスト画面を開きます。<br>
– 「オフラインモード」チェックボックスをタップします。<br>
– オフラインアーティスト/アルバム/プレイリストが「ファイル」->「オフラインフォルダー」セクションに表示されます。<br><br>

もう一つのオプションはYouTubeから動画をダウンロードしてEvervideoにインポートすることです。詳細はこちら：<br>
[YouTubeから音楽をダウンロードしてiPhoneでオフライン音楽を聴く方法](/docs/howto/how-to-download-music-from-youtube-and-listen-to-offline-music-on-iphone/)<br><br>

{{% details title="インターネット接続なしでEvervideoを使用できますか？" closed="true" %}}

オフラインコンテンツは**ファイル → このアプリのファイル**の下にあり、機内モード、フライト中、Wi-Fiやモバイルデータなしの場所でも動作し続けます。クラウドのみの動画（ダウンロードしていないもの）は接続が回復するまでグレーアウトされます。旅行には、関連するフォルダーの**オフラインモード**を有効にするか、出発前に特定の動画をダウンロードしてください。

{{% details title="Evervideoのインターフェース言語を変更するには？" closed="true" %}}

アプリは英語、フランス語、ドイツ語、スペイン語、イタリア語、ポルトガル語、ロシア語、ウクライナ語、ポーランド語、オランダ語、アラビア語、ヘブライ語、ヒンディー語、日本語、韓国語、中国語（簡体字および繁体字）、ベトナム語、トルコ語などを含む多くのローカライゼーションをサポートしています。デバイスの言語設定を自動的に追うには**デフォルト**を選択してください。

{{% details title="Evervideoをパスコードで保護するには？" closed="true" %}}

ウィジェットはタイトル、ポスター、基本コントロールとともに現在再生中の動画を表示します。ウィジェットの更新は少量のエネルギーを使用するため、**ウィジェットを有効にする**トグルはデフォルトでオフです。ウィジェットをアクティブに使用する場合のみオンにしてください。ウィジェットはiPhoneとiPadのホーム画面とロック画面、およびmacOSの通知センターで動作します。

{{% details title="Evervideo Premiumを家族と共有するには？" closed="true" %}}
**すべてのEvervideo Premiumプラン（生涯、月次、年次）はAppleファミリー共有で動作するため、ファミリーグループの誰でもEvervideoをインストールして追加費用なしでPremiumを使用できます。**<br><br>

{{% details title="Evervideoライブラリのバックアップと復元を行うには？" closed="true" %}}
**設定 → バックアップと復元を開き、含めるものを選択し（データベース、アルバムカバー、設定）、「アプリケーションデータのバックアップ」をタップしてバックアップファイルを保存します。復元するには別のデバイスで開きます。**<br><br>

バックアップにはメディアライブラリエントリ、プレイリスト、お気に入り、視聴進捗、設定、ポスターキャッシュが含まれます。オフラインダウンロードされた動画ファイルは含まれ**ません**（それらはバックアップを巨大にします）。iCloud Drive、AirDrop、または接続されたクラウドサービスを通じてバックアップファイルを新しいデバイスに移動し、Evervideoで開いて適用します。

{{% details title="Evervideoが使用するストレージを解放するには？" closed="true" %}}
**設定 → ファイルマネージャー → 一時ファイルを削除するおよび設定 → メディアライブラリ → アルバムカバー → すべてを削除するを開いてキャッシュをクリアします；ファイルタブを使用して不要になったダウンロードされた動画を削除します。**<br><br>

**設定 → ファイルマネージャー → 同期されたオフラインフォルダー → 「...」→ オフラインモードを無効にする**で個別のオフラインフォルダーを削除することもできます。これによりローカルコピーが削除されます。ストリーミングのみの動画はデバイスのストレージを全く使用しません。**ファイル → このアプリのファイル**の下の**プレーヤーキャッシュ**フォルダーをクリアすることも、高ビットレートの4Kコンテンツの重い再生後に数ギガバイトを解放できます。
{{% /details %}}

{{% details title="クラウド動画がバッファリングまたはカクつく理由は？" closed="true" %}}
**バッファリングはほとんど常に遅いネットワーク、大きなファイルサイズ、またはプレーヤーバッファ設定が低いことが原因です。プリロード時間を増やして、より速いネットワークに切り替えるか、ファイルをダウンロードしてください。**<br><br>

いくつかの実践的なヒント：<br>

- 高ビットレートの4KまたはHEVCコンテンツのために**設定 → プレーヤー → ファイル読み込み → プリロード時間**で**バッファを増やす**。<br>
- モバイル通信中の場合は**Wi-Fiに切り替える**か、ルーターに近づく。<br>
- CPUがボトルネックにならないように**設定 → プレーヤー → 映像**でH.264とHEVCの**ハードウェアデコードを有効にする**。<br>
- ソースが視聴速度より遅い場合は**オフライン再生のためにファイルをダウンロードする**。大きな4Kファイルはモバイル帯域幅を容易に超える可能性があります。<br>
- 接続が期限切れになっている場合はファイルタブで**クラウドアカウントを再認証する**。

Evervideo adalah **pemutar video cloud** HD dan 4K untuk **iPhone, iPad, dan Mac** yang melakukan streaming dan mengunduh video dari iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP, dan sumber WebDAV, SMB, NFS, FTP/SFTP, DLNA, atau S3 apa pun — plus server media yang dihost sendiri seperti **Plex, Jellyfin, Emby, Subsonic,** dan **Navidrome**, dan stream **RTSP** langsung dari kamera IP. Dibangun di atas **mesin pemutar berbasis FFmpeg** kustom dengan **dekoding H.264/HEVC yang dipercepat hardware**, trek subtitle primer dan sekunder (SRT/VTT/ASS/libass), equalizer video dengan preset, equalizer audio, **Picture-in-Picture**, **pemutaran 360°/VR** untuk rekaman Insta360 dan GoPro Max, casting **AirPlay 2** dan **Chromecast**, dan pemutar ringkas yang selalu terlihat. FAQ ini menjawab pertanyaan yang paling sering dikirimkan pengguna. Untuk panduan lebih mendalam, lihat [Panduan Pengguna Evervideo](/docs/guide/evervideo/).

{{% details title="Apa itu Evervideo?" closed="true" %}}
**Evervideo adalah pemutar video cloud HD dan 4K berfitur lengkap untuk iPhone, iPad, dan Mac** yang mengubah akun penyimpanan cloud, NAS, atau server media apa pun menjadi perpustakaan video pribadi Anda dengan kontrol penuh atas file Anda.<br><br>

Dibangun di atas mesin pemutar berbasis FFmpeg kustom dengan dekoding H.264 dan HEVC yang dipercepat hardware, Evervideo memutar hampir semua container dan codec modern (MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS, dan daftar panjang format FFmpeg), mendukung subtitle primer dan sekunder, menyertakan equalizer audio dan video, dan bekerja dalam mode Picture-in-Picture sehingga Anda dapat terus menonton sambil menggunakan aplikasi lain.
{{% /details %}}

{{% details title="Apakah Evervideo gratis? Apa perbedaan antara Gratis dan Premium?" closed="true" %}}
**Ya — Evervideo gratis untuk diunduh dan digunakan**, dengan pembelian dalam aplikasi opsional untuk menghapus batasan dan membuka kunci set fitur Premium lengkap.<br><br>

Premium tersedia sebagai **pembelian seumur hidup sekali bayar** atau sebagai **langganan bulanan atau tahunan**, sehingga Anda dapat memilih opsi yang paling sesuai untuk Anda. Harga mungkin bervariasi berdasarkan negara. **Family Sharing** diaktifkan untuk setiap paket, sehingga Anda dapat berbagi Premium dengan hingga lima anggota keluarga. Pembelian seumur hidup dan langganan dibagikan antara iOS dan macOS melalui iCloud — instal versi terbaru di setiap perangkat, masuk dengan Apple ID yang sama, dan tunggu sekitar satu menit agar informasi pembelian tersinkronisasi.<br><br>

**Evervideo Premium** menghapus setiap batasan versi gratis: pemutaran bebas iklan, daftar putar tanpa batas, layanan cloud yang terhubung tanpa batas, favorit tanpa batas, unduhan offline tanpa batas, arsip (ZIP) koleksi media, dan personalisasi penuh (ikon aplikasi kustom, tema, skema warna).<br><br>

{{% details title="Apakah Evervideo Aman?" closed="true" %}}
Evervideo hanya menggunakan SDK resmi dan koneksi aman untuk berinteraksi dengan layanan cloud yang terhubung. Login dan kata sandi Anda tidak tersedia untuk aplikasi. Semua permintaan dari aplikasi ke layanan cloud dienkripsi.<br>
Ketika Anda memasukkan login dan kata sandi, aplikasi menampilkan halaman otorisasi resmi yang disediakan oleh penyedia layanan cloud dan seluruh proses otorisasi dilakukan di luar aplikasi. Penyedia layanan cloud mengirimkan auth-token ke aplikasi setelah otorisasi berhasil dan token tersebut digunakan untuk melakukan panggilan API.<br><br>

Auth-token adalah kunci digital yang memungkinkan aplikasi pihak ketiga berinteraksi dengan penyimpanan cloud. Auth-token disimpan di perangkat Anda dalam penyimpanan sistem aman yang disebut Keychain. Anda dapat mengunduh file dari layanan cloud yang terhubung ke perangkat dan file tersebut akan ditempatkan di direktori "Dokumen" aplikasi. Anda dapat menghapus file tersebut kapan saja menggunakan manajer file bawaan.<br>
Aplikasi tidak berbagi informasi apa pun dari akun cloud yang terhubung. Anda dapat mencabut akses ke akun cloud Anda kapan saja dengan membuka halaman pengaturan akun di browser web Anda.<br><br>

Untuk menolak auth-token, masuk ke akun Anda di browser web dan navigasikan ke halaman pengaturan. Di sana Anda dapat menemukan semua aplikasi pihak ketiga yang terhubung ke akun cloud Anda dan menghapus salah satunya jika Anda tidak ingin menggunakan aplikasi tersebut lagi.<br><br>

Tutorial lengkap tersedia di sini:<br>
[Cara memutuskan koneksi aplikasi pihak ketiga dari akun Google Anda](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

Anda juga dapat memutuskan koneksi akun cloud yang terhubung dalam aplikasi dan auth-token juga akan dihapus dari perangkat Anda. Jika Anda menghapus aplikasi dari perangkat Anda, semua data yang diunduh dan token akses juga akan dihapus.<br><br>

{{% details title="Format video apa yang didukung Evervideo?" closed="true" %}}
**Evervideo memutar hampir semua container dan codec video modern di iPhone, iPad, dan Mac** berkat mesin FFmpeg bawaan yang dikombinasikan dengan dekoding H.264/HEVC yang dipercepat hardware.<br><br>

**Container:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT, dan banyak lagi.<br>
**Codec video:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV, dan hampir semua codec lain yang didukung FFmpeg.<br>
**Codec audio (di dalam video):** AAC, MP3, FLAC, ALAC, OGG/Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE.<br>
**Subtitle:** SRT, VTT (WebVTT), ASS/SSA (dirender melalui libass), trek gambar dan teks yang tertanam.<br>
**Protokol streaming:** HTTP/HTTPS, HLS (m3u8), RTSP (kamera IP dan IPTV).<br>
**Streaming langsung:** SMB/WebDAV/FTP/SFTP/NFS/DLNA.<br>

{{% details title="Apakah Evervideo memutar file MKV di iPhone?" closed="true" %}}
**Ya — Evervideo memutar file MKV secara native di iPhone, iPad, dan Mac** tanpa perlu konversi, termasuk container MKV dengan video H.264, HEVC, VP9, atau AV1, beberapa trek audio, dan subtitle SRT/ASS yang tertanam.<br><br>

Ini adalah salah satu alasan utama orang menginstal Evervideo daripada pemutar default Apple: iOS tidak membuka MKV sama sekali, tetapi Evervideo menanganinya melalui mesin FFmpeg bawaan. Anda dapat melakukan streaming file MKV langsung dari penyimpanan cloud atau NAS tanpa mengunduh.
{{% /details %}}

{{% details title="Apakah Evervideo mendukung video 4K dan HDR?" closed="true" %}}
**Ya — Evervideo memutar video 4K (Ultra HD) dan konten berkode HDR** di setiap perangkat yang memiliki hardware untuk mendekodenya. iPhone, iPad, dan Mac Apple Silicon modern semuanya mendekode 4K H.264 dan 4K HEVC di hardware, sehingga pemutaran lancar dan hemat baterai.<br><br>

Untuk hasil terbaik pada 4K yang di-streaming dari cloud, tingkatkan **Waktu Pra-pemuatan** di **Pengaturan → Pemutar → Pemuatan File** agar buffer dapat mengimbangi file dengan bitrate tinggi, dan sambungkan ke jaringan Wi-Fi daripada seluler.
{{% /details %}}

{{% details title="Apakah Evervideo mendukung dekoding hardware H.264 dan HEVC?" closed="true" %}}
**Ya — Evervideo menggunakan dekoder hardware H.264 (AVC) dan H.265 (HEVC) secara default** di setiap iPhone, iPad, dan Mac yang mendukungnya, yang berarti pemutaran lebih lancar, penggunaan baterai lebih rendah, dan suhu perangkat lebih dingin dibandingkan dengan dekoding perangkat lunak murni.<br><br>

Anda dapat mengaktifkan atau menonaktifkan dekoding hardware secara independen untuk H.264 dan HEVC di **Pengaturan → Pemutar → Video → Dekoding Hardware H.264/H.265**. Jika file tertentu memiliki masalah kompatibilitas (aliran yang rusak, profil eksotis), nonaktifkan dekoding hardware untuk file tersebut agar kembali ke dekoding perangkat lunak FFmpeg.
{{% /details %}}

{{% details title="Apakah Evervideo mendukung Picture-in-Picture (PiP)?" closed="true" %}}
**Ya — Evervideo sepenuhnya mendukung Picture-in-Picture di iPhone dan iPad.** Saat Anda mengetuk ikon PiP pada pemutar, video terus diputar di jendela mengambang di atas setiap aplikasi lain.<br><br>

Seret jendela mengambang ke sudut mana pun, cubit untuk mengubah ukuran, ketuk sekali untuk memunculkan kontrol dasar putar/jeda/lewati, dan ketuk tombol perluas kecil untuk kembali ke Evervideo. PiP bekerja dengan **setiap format video yang diputar Evervideo**, termasuk file yang di-streaming dari cloud dan stream kamera IP RTSP, dan terus berjalan saat ponsel Anda terkunci.
{{% /details %}}

{{% details title="Apakah Evervideo memiliki antarmuka pemutar gaya YouTube untuk video lokal saya?" closed="true" %}}
**Ya — tata letak default Evervideo dibangun di sekitar pengalaman gaya YouTube untuk video Anda sendiri: pemutar video ringkas yang selalu terlihat di bagian atas layar saat Anda menjelajahi perpustakaan di bawah.**<br><br>

**Pemutar ringkas** tetap terlihat di setiap tab — Terbaru, Favorit, Perpustakaan Media, Daftar Putar, File, dan Pengaturan — sehingga Anda dapat menelusuri, mencari, mengatur, dan mengantri video berikutnya tanpa pernah mengganggu pemutaran. Ketuk pemutar ringkas untuk memperluasnya ke tampilan layar penuh; geser ke bawah untuk melipatnya kembali ke ringkas tanpa menghentikan video. Di macOS, pemutar ringkas dapat dilepas ke **jendela mengambang always-on-top** terpisah untuk multitasking gaya picture-in-picture yang sesungguhnya di desktop.<br><br>

Tidak seperti YouTube, tidak ada iklan, rekomendasi algoritmik, nag autoplay, dan pelacakan — Anda tetap memiliki kontrol penuh atas perpustakaan video Anda sendiri dari iCloud, Google Drive, Dropbox, NAS, Plex/Jellyfin/Emby, atau di mana pun.
{{% /details %}}

{{% details title="Apakah Evervideo memutar video 360° dari Insta360 dan kamera 360 lainnya?" closed="true" %}}
**Ya — Evervideo memutar video 360°/VR (sferis) secara langsung, tanpa pra-pemrosesan atau konversi yang diperlukan**, termasuk rekaman dari Insta360 (One X, X3, X4, ONE RS, GO 3), GoPro Max, Ricoh Theta, Samsung Gear 360, Vuze, dan sumber equirectangular 360° lainnya.<br><br>

Cukup letakkan video 360° ke iCloud Drive, Google Drive, Dropbox, NAS Anda, atau putarnya dari perpustakaan Foto iOS — Evervideo mengenali format sferis dan secara otomatis beralih ke rendering viewport VR. Dari sana Anda dapat:<br>

- **Putar perangkat Anda** — miringkan dan putar iPhone atau iPad Anda secara fisik dan viewport video berputar secara real time untuk melihat sekeliling adegan.<br>
- **Seret dengan jari Anda** untuk menggeser ke segala arah tanpa menggerakkan perangkat.<br>
- **Cubit untuk memperbesar** masuk dan keluar dari tampilan sferis.<br>
- Beralih mode proyeksi/tampilan di menu Lebih banyak tindakan pemutar, dan gunakan kandang headset VR untuk pengalaman yang sepenuhnya imersif.<br><br>

Ini berarti rekaman Insta360 Anda berfungsi langsung di iPhone dan iPad — tidak ada aplikasi tambahan, tidak ada transcoding, tidak diperlukan langganan cloud.
{{% /details %}}

{{% details title="Bagaimana cara menggunakan equalizer video dan preset?" closed="true" %}}
**Buka pemutar, ketuk Lebih banyak tindakan → Equalizer Video, lalu seret slider untuk kecerahan, kontras, saturasi, dan hue — atau pilih preset.**<br><br>

**Equalizer video** Evervideo adalah alat penyesuaian gambar real-time yang berjalan di dalam pipeline rendering FFmpeg:<br>

- **Kecerahan** — tingkatkan adegan gelap yang diambil dalam cahaya redup, atau kurangi eksposur yang terlalu terang.<br>
- **Kontras** — tambahkan ketajaman pada konten yang pudar atau perhalus sumber yang terlalu kontras.<br>
- **Saturasi** — hangatkan warna yang redup atau desaturasi untuk tampilan sinematik.<br>
- **Hue** — geser keseimbangan warna untuk memperbaiki cast hijau atau magenta.<br><br>

Anda dapat menyimpan pengaturan favorit Anda sebagai **preset kustom** dan menerapkannya kembali dengan satu ketukan pada video apa pun di masa depan. Preset juga dapat **diekspor dan diimpor** sehingga Anda dapat berbagi di antara iPhone, iPad, dan Mac, atau mencadangkannya. Kombinasikan equalizer video dengan **equalizer audio** (EQ 10 band dengan perpustakaan preset sendiri) untuk kontrol penuh atas gambar dan suara.
{{% /details %}}

{{% details title="Bagaimana cara memutar atau mengubah mode penskalaan video?" closed="true" %}}
**Ketuk Lebih banyak tindakan pada pemutar dan pilih Rotasi (0°/90°/180°/270°) atau Mode Penskalaan (Sesuai/Isi/Regangkan/Asli).**<br><br>

**Rotasi** berguna untuk video yang direkam miring atau terbalik — putar gambar tanpa meninggalkan pemutar. **Mode Penskalaan** mengontrol bagaimana gambar mengisi layar:<br>

- **Sesuai** — pertahankan rasio aspek asli dengan bilah hitam di mana diperlukan.<br>
- **Isi** — isi seluruh layar, memotong video jika perlu.<br>
- **Regangkan** — regangkan untuk mengisi layar, mendistorsi gambar.<br>
- **Asli** — pertahankan resolusi native pada 1:1.
{{% /details %}}

{{% details title="Bagaimana cara memilih trek audio yang berbeda (dubbing, komentar) dalam video?" closed="true" %}}
**Ketuk tombol Lebih banyak tindakan ("...") pada pemutar dan pilih Trek Audio — lalu pilih trek yang Anda inginkan dari daftar.**<br><br>

Untuk video dengan beberapa trek audio (dubbing bahasa alternatif, komentar sutradara, campuran asli/langsung), Evervideo menampilkan setiap trek yang tertanam dengan bahasa dan codec-nya. Ini berfungsi untuk MKV, MP4, M2TS, dan container lain yang mengekspos beberapa aliran audio.
{{% /details %}}

{{% details title="Bagaimana cara menambahkan atau mengubah subtitle di Evervideo?" closed="true" %}}
**Ketuk tombol Lebih banyak tindakan ("...") pada pemutar dan pilih Subtitle — lalu pilih trek subtitle yang tertanam, muat file subtitle eksternal, atau ubah fontnya.**<br><br>

Evervideo secara otomatis mencantumkan setiap trek subtitle yang tertanam dalam video. Untuk memuat file subtitle eksternal, pilih **File Eksternal** dan pilih file `.srt`, `.vtt`, `.ass`, atau `.ssa` dari perangkat Anda, iCloud Drive, atau layanan cloud yang terhubung. Anda juga dapat mengonfigurasi perilaku subtitle default di **Pengaturan → Pemutar → Subtitle**.
{{% /details %}}

{{% details title="Apakah Evervideo mendukung file subtitle SRT, VTT, dan ASS eksternal?" closed="true" %}}
**Ya — Evervideo memuat file subtitle eksternal dalam format SRT, VTT (WebVTT), ASS, dan SSA** dari mana saja di perangkat Anda atau layanan cloud yang terhubung.<br><br>

File ASS/SSA dengan gaya lanjutan (font kustom, warna, posisi, efek karaoke) dirender dengan benar berkat pustaka **libass** bawaan — sempurna untuk anime bersub penggemar, file keterangan profesional, dan presentasi. Anda juga dapat menetapkan font tertentu untuk subtitle di **Pengaturan → Pemutar → Subtitle → Font**.
{{% /details %}}

{{% details title="Bagaimana cara menghapus video dari Evervideo tanpa menghapusnya dari penyimpanan cloud?" closed="true" %}}
**Pada video mana pun, ketuk "..." → Hapus dari Perpustakaan Media — ini menghapus entri dari database perpustakaan Anda tetapi membiarkan file asli tidak tersentuh di penyimpanan cloud, NAS, atau perpustakaan Foto iOS.**<br><br>

Jika Anda juga ingin menghapus file dari sumbernya, pilih **Hapus dari Layanan Cloud** atau **Hapus dari File Lokal** sebagai gantinya. Tindakan destruktif ini tidak dapat dibatalkan, jadi berhati-hatilah saat Anda memilih beberapa video.<br><br>

Untuk menghapus salinan yang diunduh tanpa menyentuh apa pun di cloud, buka tab **File**, temukan video di bawah **File di Aplikasi Ini** atau **Folder Offline**, dan gunakan **"..." → Hapus**. Asli di cloud tetap persis di tempatnya — Evervideo hanya menghapus salinan lokal.
{{% /details %}}

{{% details title="Bagaimana cara mengubah kecepatan pemutaran di Evervideo?" closed="true" %}}
**Buka pemutar, ketuk kontrol Kecepatan pada toolbar, dan seret slider — kecepatan dari 0,25× hingga 3,00× didukung.**<br><br>

Anda dapat memperlambat konten untuk analisis frame-by-frame (0,25×/0,5×) atau mempercepatnya untuk tutorial dan kuliah (1,25×/1,5×/2×). Ketuk ikon konfigurasi di sudut kanan atas layar Kecepatan untuk beralih ke mode presisi untuk penyesuaian yang lebih halus. Koreksi pitch per-trek menjaga audio terdengar alami pada kecepatan non-1×.
{{% /details %}}

{{% details title="Bagaimana cara mengatur timer tidur di Evervideo?" closed="true" %}}
**Buka Pengaturan → Pemutar → Timer Tidur, aktifkan, dan pilih berapa lama pemutaran berlanjut sebelum berhenti otomatis.**<br><br>

Anda juga dapat menambahkan tombol **Timer Tidur** langsung ke layar pemutar utama melalui **Pengaturan → Pemutar → Personalisasi → Aksi Layar Utama**. Ketuk ikon konfigurasi untuk mode presisi dengan granularitas menit-demi-menit — berguna untuk tertidur saat menonton acara.
{{% /details %}}

{{% details title="Bagaimana cara menandai posisi tertentu dalam video?" closed="true" %}}
**Buka pemutar dan ketuk Tambahkan Tandai Halaman dari menu Lebih banyak tindakan untuk menyimpan posisi pemutaran saat ini — bookmark muncul di Lebih banyak tindakan → Tandai Halaman.**<br><br>

Bookmark disimpan per-video dan bertahan antar sesi, menjadikannya sempurna untuk video panjang, kuliah, buku audio video, seri tutorial, dan rekaman konser di mana Anda ingin kembali ke momen tertentu.
{{% /details %}}

{{% details title="Bagaimana cara melanjutkan daftar putar dari tempat saya berhenti?" closed="true" %}}

Ketika Anda beralih ke daftar putar lain dan kembali, Anda akan melihat empat tindakan pada toolbar atas di bawah karya seni album: "Cari," "Lanjutkan Pemutaran," "Putar Semua," dan "Acak Semua." Ketuk "Lanjutkan Pemutaran" untuk melanjutkan daftar putar dari status tersimpan dan posisi media terakhir.
{{% /details %}}

{{% details title="Bagaimana cara melempar video dari Evervideo ke Chromecast atau AirPlay?" closed="true" %}}
**Buka pemutar video, ketuk ikon AirPlay atau Chromecast, dan pilih TV, Apple TV, HomePod, atau speaker pintar Anda dari daftar.**<br><br>

Baik **AirPlay 2** maupun **Google Chromecast** didukung di iOS. AirPlay 2 juga memungkinkan Anda melakukan streaming ke beberapa perangkat yang kompatibel pada saat yang bersamaan. Beberapa file hi-res atau HEVC mungkin perlu di-transcode untuk hardware Chromecast.

Jika Anda menginginkan aplikasi yang terhubung ke cloud yang mendukung CarPlay, aplikasi musik kami **Evermusic** dan **Flacbox** keduanya berfungsi penuh di CarPlay dengan tab khusus untuk Perpustakaan, Koneksi, File Lokal, dan Pengaturan.

**Buka Pengaturan → Manajer File → Hapus File Sementara dan Pengaturan → Perpustakaan Media → Sampul Album → Hapus Semua untuk mengosongkan cache; gunakan tab File untuk menghapus video yang diunduh yang tidak lagi Anda butuhkan.**<br><br>

Anda juga dapat menghapus folder offline individual di **Pengaturan → Manajer File → Folder Offline yang Disinkronkan → "..." → Nonaktifkan Mode Offline**, yang menghapus salinan lokal. Video hanya streaming tidak menggunakan penyimpanan perangkat sama sekali. Menghapus folder **Cache Pemutar** di bawah **File → File di Aplikasi Ini** juga dapat membebaskan beberapa gigabyte setelah pemutaran intensif konten 4K bitrate tinggi.

{{% details title="Mengapa video cloud saya buffering atau tersendat?" closed="true" %}}
**Buffering hampir selalu disebabkan oleh jaringan yang lambat, ukuran file yang besar, atau pengaturan buffer pemutar yang rendah — tingkatkan Waktu Pra-pemuatan dan beralih ke jaringan yang lebih cepat atau unduh file.**<br><br>

Beberapa tips praktis:<br>

- **Tingkatkan buffer** di **Pengaturan → Pemutar → Pemuatan File → Waktu Pra-pemuatan** untuk konten 4K atau HEVC dengan bitrate tinggi.<br>
- **Beralih ke Wi-Fi** jika Anda menggunakan seluler, atau pindah lebih dekat ke router Anda.<br>
- **Aktifkan Dekoding Hardware** untuk H.264 dan HEVC di **Pengaturan → Pemutar → Video** agar CPU bukan hambatan.<br>
- **Unduh file untuk pemutaran offline** jika sumbernya lebih lambat dari kecepatan menonton Anda — file 4K besar dapat dengan mudah melebihi bandwidth seluler.<br>
- **Otorisasi ulang akun cloud Anda** di tab File jika koneksi telah kedaluwarsa.

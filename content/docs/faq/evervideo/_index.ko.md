Evervideo는 **iPhone, iPad, Mac**용 HD 및 4K **클라우드 동영상 플레이어**로, iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP 및 WebDAV, SMB, NFS, FTP/SFTP, DLNA, S3 소스에서 동영상을 스트리밍하고 다운로드할 수 있습니다. **Plex, Jellyfin, Emby, Subsonic**, **Navidrome** 등 자체 호스팅 미디어 서버와 IP 카메라의 **RTSP** 라이브 스트림도 지원합니다. 커스텀 **FFmpeg 기반 플레이어 엔진**과 **하드웨어 가속 H.264/HEVC 디코딩**, 기본 및 보조 자막 트랙(SRT/VTT/ASS/libass), 프리셋이 있는 비디오 이퀄라이저와 오디오 이퀄라이저, **Picture-in-Picture**, **360°/VR 재생**, **AirPlay 2** 및 **Chromecast** 캐스팅, 항상 화면에 표시되는 소형 플레이어를 내장하고 있습니다. 이 FAQ는 사용자들이 가장 자주 보내는 질문에 답합니다. 심층적인 가이드는 [Evervideo 사용자 가이드](/docs/guide/evervideo/)를 참조하세요.

{{% details title="Evervideo란 무엇인가요?" closed="true" %}}
**Evervideo는 iPhone, iPad, Mac용 풀 기능 HD 및 4K 클라우드 동영상 플레이어**로, 클라우드 스토리지 계정, NAS 또는 미디어 서버를 파일을 완전히 제어할 수 있는 개인 동영상 라이브러리로 만들어 줍니다.<br><br>

커스텀 FFmpeg 기반 플레이어 엔진과 하드웨어 가속 H.264 및 HEVC 디코딩을 기반으로 Evervideo는 거의 모든 최신 컨테이너와 코덱(MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS 및 FFmpeg 지원 형식)을 재생하고, 기본 및 보조 자막을 지원하며, 오디오 및 비디오 이퀄라이저를 포함하고, 다른 앱을 사용하면서도 계속 시청할 수 있는 PiP 모드로 작동합니다.
{{% /details %}}

{{% details title="Evervideo는 무료인가요? Free와 Premium의 차이는 무엇인가요?" closed="true" %}}
**네 — Evervideo는 무료로 다운로드하여 사용할 수 있으며**, 제한을 없애고 전체 Premium 기능을 잠금 해제하는 선택적 인앱 구매가 있습니다.<br><br>

Premium은 **일회성 평생 구매** 또는 **월간 또는 연간 구독**으로 제공되므로 가장 적합한 옵션을 선택할 수 있습니다. 가격은 국가에 따라 다를 수 있습니다. **Family Sharing**은 모든 플랜에서 활성화되어 있으므로 최대 5명의 가족 구성원과 Premium을 공유할 수 있습니다. 평생 구매 및 구독은 iCloud를 통해 iOS와 macOS 간에 공유됩니다 — 각 기기에 최신 버전을 설치하고 동일한 Apple ID로 로그인한 후 구매 정보가 동기화될 때까지 약 1분 기다리세요.<br><br>

**Evervideo Premium**은 무료 버전의 모든 제한을 제거합니다: 광고 없는 재생, 무제한 재생 목록, 무제한 클라우드 서비스 연결, 무제한 즐겨찾기, 무제한 오프라인 다운로드, 미디어 컬렉션의 아카이브(ZIP), 전체 개인화(커스텀 앱 아이콘, 테마, 색상 구성표).<br><br>

[전체 Free vs Premium 비교 읽기](/docs/faq/evervideo/what-is-the-difference-between-evervideo-and-evervideo-premium/)
{{% /details %}}

{{% details title="Evervideo는 안전한가요?" closed="true" %}}
Evervideo는 공식 SDK와 보안 연결만 사용하여 연결된 클라우드 서비스와 상호작용합니다. 로그인 및 비밀번호는 애플리케이션에서 사용할 수 없습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.<br>
로그인과 비밀번호를 입력하면 애플리케이션은 클라우드 서비스 공급자가 제공하는 공식 인증 페이지를 표시하며 모든 인증 과정은 애플리케이션 외부에서 이루어집니다. 클라우드 서비스 공급자는 성공적인 인증 후 애플리케이션에 인증 토큰을 전송하며 해당 토큰은 API 호출에 사용됩니다.<br><br>

인증 토큰은 타사 애플리케이션이 클라우드 스토리지와 상호작용할 수 있게 해주는 디지털 키입니다. 인증 토큰은 Keychain이라는 보안 시스템 스토리지에 기기에 저장됩니다. 연결된 클라우드 서비스에서 기기로 파일을 다운로드할 수 있으며 해당 파일은 앱 「Documents」 디렉토리에 저장됩니다. 내장 파일 관리자를 사용하여 언제든지 해당 파일을 삭제할 수 있습니다.<br>
애플리케이션은 연결된 클라우드 계정의 정보를 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정에 대한 액세스를 취소할 수 있습니다.<br><br>

인증 토큰을 거부하려면 웹 브라우저에서 계정에 로그인하여 설정 페이지로 이동하세요. 거기에서 클라우드 계정에 연결된 모든 타사 앱을 찾아 더 이상 사용하지 않으려는 앱을 삭제할 수 있습니다.<br><br>

전체 튜토리얼은 여기에 있습니다:<br>
[Google 계정에서 타사 앱 연결 해제하는 방법](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

애플리케이션에서 연결된 클라우드 계정의 연결을 끊을 수도 있으며 인증 토큰도 기기에서 제거됩니다. 기기에서 애플리케이션을 삭제하면 다운로드된 모든 데이터와 액세스 토큰도 삭제됩니다.<br><br>

{{% details title="Evervideo가 지원하는 동영상 형식은 무엇인가요?" closed="true" %}}
**Evervideo는 번들 FFmpeg 엔진과 하드웨어 가속 H.264/HEVC 디코딩의 결합 덕분에 iPhone, iPad, Mac에서 거의 모든 최신 동영상 컨테이너와 코덱을 재생합니다.**<br><br>

**컨테이너:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT 등 다수.<br>
**비디오 코덱:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV 및 FFmpeg가 지원하는 사실상 모든 코덱.<br>
**오디오 코덱(동영상 내):** AAC, MP3, FLAC, ALAC, OGG/Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE.<br>
**자막:** SRT, VTT (WebVTT), ASS/SSA (libass를 통해 렌더링), 내장 이미지 및 텍스트 트랙.<br>
**스트리밍 프로토콜:** HTTP/HTTPS, HLS (m3u8), RTSP (IP 카메라 및 IPTV).<br>
**직접 스트리밍:** SMB/WebDAV/FTP/SFTP/NFS/DLNA.<br>
{{% /details %}}

{{% details title="Evervideo가 iPhone에서 MKV 파일을 재생하나요?" closed="true" %}}
**네 — Evervideo는 iPhone, iPad, Mac에서 변환 없이 기본적으로 MKV 파일을 재생합니다.** H.264, HEVC, VP9 또는 AV1 비디오, 다중 오디오 트랙, 내장 SRT/ASS 자막이 있는 MKV 컨테이너를 포함합니다.<br><br>

이것이 사람들이 Apple의 기본 플레이어 대신 Evervideo를 설치하는 주요 이유 중 하나입니다: iOS는 MKV를 전혀 열지 않지만 Evervideo는 번들 FFmpeg 엔진을 통해 처리합니다. 다운로드 없이 클라우드 스토리지나 NAS에서 직접 MKV 파일을 스트리밍할 수 있습니다.
{{% /details %}}

{{% details title="Evervideo가 4K 및 HDR 동영상을 지원하나요?" closed="true" %}}
**네 — Evervideo는 디코딩 하드웨어를 갖춘 모든 기기에서 4K (Ultra HD) 동영상 및 HDR 인코딩 콘텐츠를 재생합니다.** 최신 iPhone, iPad, Apple Silicon Mac은 모두 하드웨어에서 4K H.264 및 4K HEVC를 디코딩하므로 재생이 부드럽고 배터리 효율이 좋습니다.<br><br>

클라우드 스트리밍 4K에서 최상의 결과를 얻으려면 **설정 → Player → File Loading**에서 **Preloading Time**을 늘려 버퍼가 고비트레이트 파일을 따라갈 수 있도록 하고, 셀룰러 대신 Wi-Fi 네트워크에 연결하세요.
{{% /details %}}

{{% details title="Evervideo가 하드웨어 H.264 및 HEVC 디코딩을 지원하나요?" closed="true" %}}
**네 — Evervideo는 지원하는 모든 iPhone, iPad, Mac에서 기본적으로 하드웨어 H.264 (AVC) 및 H.265 (HEVC) 디코더를 사용합니다.** 이는 순수 소프트웨어 디코딩에 비해 더 부드러운 재생, 낮은 배터리 사용, 낮은 기기 온도를 의미합니다.<br><br>

**설정 → Player → Video → Hardware Decode H.264/H.265**에서 H.264 및 HEVC에 대한 하드웨어 디코딩을 독립적으로 토글할 수 있습니다. 특정 파일에 호환성 문제(손상된 스트림, 특이한 프로파일)가 있는 경우 해당 파일에 대한 하드웨어 디코딩을 비활성화하여 FFmpeg 소프트웨어 디코딩으로 전환하세요.
{{% /details %}}

{{% details title="Evervideo가 Picture-in-Picture (PiP)를 지원하나요?" closed="true" %}}
**네 — Evervideo는 iPhone 및 iPad에서 Picture-in-Picture를 완전히 지원합니다.** 플레이어에서 PiP 아이콘을 탭하면 동영상이 다른 모든 앱 위에 떠 있는 창에서 계속 재생됩니다.<br><br>

플로팅 창을 아무 모서리로 드래그하고, 핀치하여 크기를 조정하고, 한 번 탭하여 기본 재생/일시정지/건너뛰기 컨트롤을 불러오고, 작은 확장 버튼을 탭하여 Evervideo로 돌아올 수 있습니다. PiP는 클라우드 스트리밍 파일 및 RTSP IP 카메라 스트림을 포함하여 **Evervideo가 재생하는 모든 동영상 형식**에서 작동하며, 전화기가 잠겨 있는 동안에도 계속 실행됩니다.
{{% /details %}}

{{% details title="Evervideo에 로컬 동영상을 위한 YouTube 스타일 플레이어 UI가 있나요?" closed="true" %}}
**네 — Evervideo의 기본 레이아웃은 내 동영상을 위한 YouTube 스타일 경험을 중심으로 구성되어 있습니다: 아래에서 라이브러리를 탐색하는 동안 화면 상단에 소형 동영상 플레이어가 항상 표시됩니다.**<br><br>

**소형 플레이어**는 최근 항목, 즐겨찾기, 미디어 라이브러리, 재생 목록, 파일, 설정 등 모든 탭에 걸쳐 화면에 유지되므로 재생을 중단하지 않고 탐색, 검색, 정리 및 다음 동영상을 큐에 추가할 수 있습니다. 소형 플레이어를 탭하여 전체 화면 보기로 확장하고, 아래로 스와이프하여 동영상을 멈추지 않고 다시 소형으로 축소할 수 있습니다. macOS에서는 소형 플레이어를 분리하여 데스크탑에서 진정한 PiP 스타일 멀티태스킹을 위한 별도의 **항상 최상위 플로팅 창**으로 만들 수 있습니다.<br><br>

YouTube와 달리 광고, 알고리즘 추천, 자동 재생 알림, 추적이 없습니다 — iCloud, Google Drive, Dropbox, NAS, Plex/Jellyfin/Emby 또는 다른 어디서든 자신의 동영상 라이브러리를 완전히 제어합니다.
{{% /details %}}

{{% details title="Evervideo가 Insta360 및 다른 360도 카메라의 360° 동영상을 재생하나요?" closed="true" %}}
**네 — Evervideo는 전처리나 변환 없이 360°/VR(구면) 동영상을 직접 재생합니다.** Insta360 (One X, X3, X4, ONE RS, GO 3), GoPro Max, Ricoh Theta, Samsung Gear 360, Vuze 및 기타 등장방형 360° 소스의 영상이 포함됩니다.<br><br>

360° 동영상을 iCloud Drive, Google Drive, Dropbox, NAS에 넣거나 iOS 사진 라이브러리에서 재생하기만 하면 됩니다 — Evervideo가 구면 형식을 인식하고 자동으로 VR 뷰포트 렌더링으로 전환합니다. 거기서부터:<br>

- **기기를 회전**하세요 — iPhone이나 iPad를 물리적으로 기울이고 돌리면 동영상 뷰포트가 실시간으로 회전하여 장면을 둘러볼 수 있습니다.<br>
- **손가락으로 드래그**하여 기기를 이동하지 않고 어느 방향으로든 이동할 수 있습니다.<br>
- **핀치 투 줌**으로 구면 뷰에서 확대하거나 축소할 수 있습니다.<br>

 • 선택한 폴더와 모든 내용이 「파일」 > 「오프라인 폴더」로 다운로드됩니다.<br><br>

- 자동 업데이트:<br>
 • 앱이 변경 사항을 지속적으로 스캔합니다. 온라인 폴더에 추가된 새 파일은 자동으로 다운로드됩니다.<br><br>

- 스캔 타임아웃 구성:<br>
 • 「설정」 > 「파일 관리자」 > 「오프라인 폴더」 > 「시간 간격」으로 이동하여 스캔 빈도를 설정하세요.<br><br>

- 수동 동기화:<br>
 • 수동으로 동기화하려면 「설정」 > 「파일 관리자」 > 「오프라인 폴더」 > 「동기화된 오프라인 폴더」로 이동하세요.<br>
 • 「추가 작업」을 탭하고 「동기화 시작」을 선택하세요.<br><br>

{{% /details %}}

{{% details title="동영상을 어떻게 다운로드하나요?" closed="true" %}}
동영상을 다운로드하고 오프라인으로 시청하기 전에 클라우드 스토리지에 연결해야 합니다.<br>
「파일」 화면을 열고 클라우드 스토리지에 연결하세요.<br>
추가한 후 클라우드에서 동영상을 다운로드할 수 있습니다.<br><br>

**클라우드에서 동영상 다운로드하기**<br>
– 연결된 클라우드 스토리지를 여세요.<br>
– 다운로드할 폴더 내로 이동하세요.<br>
– 오른쪽 상단 모서리의 추가 작업 버튼 「...」를 탭하고 「선택하다」 메뉴 항목을 선택하세요.<br>
– 다운로드할 파일을 선택하고 「다운로드」 버튼을 탭하세요.<br><br>

**아티스트/재생 목록/앨범에 오프라인 모드 활성화하기**<br>
– 아티스트/앨범/재생 목록 화면 열기<br>
– 「오프라인 모드」 체크박스 탭하기<br>
– 오프라인 아티스트/앨범/재생 목록이 「파일」 -> 「오프라인 폴더」 섹션에 표시됩니다.<br><br>

또 다른 옵션은 YouTube에서 동영상을 다운로드하여 Evervideo로 가져오는 것입니다. 자세한 내용은 여기를 참조하세요:<br>
[YouTube에서 음악을 다운로드하고 iPhone에서 오프라인으로 듣는 방법](/docs/howto/how-to-download-music-from-youtube-and-listen-to-offline-music-on-iphone/)<br><br>

{{% /details %}}

{{% details title="인터넷 연결 없이 Evervideo를 사용할 수 있나요?" closed="true" %}}
**네 — 동영상을 다운로드하거나 폴더에 오프라인 모드를 활성화하면 Evervideo는 완전히 오프라인으로 재생합니다.**<br><br>

오프라인 콘텐츠는 **파일 → 이 애플리케이션의 파일** 아래에 있으며 비행기 모드, 비행 중, Wi-Fi나 셀룰러 데이터가 없는 곳에서도 계속 작동합니다. 클라우드 전용 동영상(다운로드하지 않은 것)은 연결을 다시 얻을 때까지 회색으로 표시됩니다. 여행을 위해 관련 폴더에 **오프라인 모드**를 활성화하거나 출발 전에 특정 동영상을 다운로드하세요.
{{% /details %}}

{{% details title="Evervideo에서 다크 모드를 어떻게 활성화하나요?" closed="true" %}}
**설정 → 개인화 → 색상 구성표를 열고 어둡게, 밝게 또는 기본값(시스템 테마를 따름)을 선택하세요.**<br><br>

**설정 → 개인화 → 앱 아이콘**(Premium)에서 대체 앱 아이콘을 선택하고 **배경 스타일** 아래에서 흐릿한 포스터를 앱 배경으로 선택할 수도 있습니다.
{{% /details %}}

{{% details title="Evervideo 인터페이스의 언어를 어떻게 변경하나요?" closed="true" %}}
**설정 → 언어를 열고 120개 이상의 지원 언어 중에서 선택한 후 앱을 다시 시작하여 변경 사항을 적용하세요.**<br><br>

앱은 영어, 프랑스어, 독일어, 스페인어, 이탈리아어, 포르투갈어, 러시아어, 우크라이나어, 폴란드어, 네덜란드어, 아랍어, 히브리어, 힌디어, 일본어, 한국어, 중국어(간체 및 번체), 베트남어, 터키어 등 다양한 언어를 지원합니다. 기기의 언어 설정을 자동으로 따르려면 **기본값**을 선택하세요.
{{% /details %}}

{{% details title="Evervideo를 패스코드로 어떻게 보호하나요?" closed="true" %}}
**설정 → 패스코드를 열고 활성화를 탭한 후 4자리 코드를 선택하세요 — 앱이 실행될 때마다 입력하라는 메시지가 표시됩니다.**<br><br>

Evervideo는 고정 4자리 숫자 패스코드를 사용합니다. 패스코드는 기기에 접근할 수 있는 누구든 Evervideo를 열어 연결된 클라우드 계정, 다운로드된 동영상, 라이브러리를 탐색하지 못하게 합니다. 추가 보호를 위해 기기의 iOS Face ID/Touch ID와 결합하세요.
{{% /details %}}

{{% details title="iPhone 홈 화면이나 잠금 화면에서 Evervideo 위젯을 어떻게 활성화하나요?" closed="true" %}}
**설정 → 위젯에서 위젯 업데이트를 활성화한 후 홈 화면이나 잠금 화면을 길게 누르고 「+」를 탭하여 「Evervideo」를 검색한 후 위젯 크기를 선택하세요.**<br><br>

위젯은 제목, 포스터, 기본 컨트롤과 함께 현재 재생 중인 동영상을 표시합니다. 위젯 새로 고침은 소량의 에너지를 사용하므로 **위젯 활성화** 토글은 기본적으로 꺼져 있습니다 — 위젯을 적극적으로 사용하는 경우에만 켜세요. 위젯은 iPhone 및 iPad 홈 화면과 잠금 화면, macOS의 알림 센터에서 작동합니다.
{{% /details %}}

{{% details title="Evervideo Premium을 가족과 어떻게 공유하나요?" closed="true" %}}
**Evervideo Premium의 모든 플랜(평생, 월간, 연간)은 Apple Family Sharing과 함께 작동하므로 가족 그룹의 누구든 Evervideo를 설치하고 추가 비용 없이 Premium을 사용할 수 있습니다.**<br><br>

iOS/macOS **설정 → 가족**에서 Family Sharing을 설정한 후 각 가족 구성원이 App Store에서 Evervideo를 설치하고 자신의 Apple ID로 로그인한 상태에서 한 번 실행하게 하세요. Premium은 1분 이내에 자동으로 인식됩니다. 동일한 플랜이 모든 가족 구성원의 iPhone, iPad, Mac 간에 공유됩니다.
{{% /details %}}

{{% details title="Evervideo Premium 구독을 어떻게 취소하나요?" closed="true" %}}
**iOS 또는 macOS 설정 → [사용자 이름] → 구독을 열고 Evervideo를 찾아 구독 취소를 탭하세요 — Premium 기능은 현재 청구 기간이 끝날 때까지 활성 상태로 유지됩니다.**<br><br>

평생 인앱 구매는 구독이 아니므로 취소할 필요가 없습니다. 환불은 Apple의 **문제 신고** 페이지(`reportaproblem.apple.com`)를 사용하세요 — 환불은 Everappz가 아닌 Apple에서 발행합니다.
{{% /details %}}

{{% details title="Evervideo 라이브러리를 어떻게 백업하고 복원하나요?" closed="true" %}}
**설정 → 백업 및 복원을 열고 포함할 항목(데이터베이스, 앨범 커버, 설정)을 선택한 후 「앱 데이터 백업」을 탭하고 백업 파일을 저장하세요 — 다른 기기에서 열어 복원하세요.**<br><br>

백업에는 미디어 라이브러리 항목, 재생 목록, 즐겨찾기, 시청 진행 상황, 설정, 포스터 캐시가 포함됩니다. 오프라인으로 다운로드된 동영상 파일은 포함되지 **않습니다**(그렇게 하면 백업이 매우 커질 것입니다). iCloud Drive, AirDrop 또는 연결된 클라우드 서비스를 통해 새 기기로 백업 파일을 이동한 후 Evervideo에서 열어 적용하세요.
{{% /details %}}

{{% details title="Evervideo가 사용하는 저장소를 어떻게 확보하나요?" closed="true" %}}
**설정 → 파일 관리자 → 임시 파일 삭제 및 설정 → 미디어 라이브러리 → 앨범 커버 → 모두 삭제를 열어 캐시를 지우고, 파일 탭을 사용하여 더 이상 필요하지 않은 다운로드된 동영상을 삭제하세요.**<br><br>

**설정 → 파일 관리자 → 동기화된 오프라인 폴더 → 「...」 → 오프라인 모드 비활성화**에서 개별 오프라인 폴더도 삭제할 수 있으며 로컬 사본이 삭제됩니다. 스트리밍 전용 동영상은 기기 스토리지를 전혀 사용하지 않습니다. **파일 → 이 애플리케이션의 파일**에 있는 **Player Cache** 폴더를 지우면 고비트레이트 4K 콘텐츠를 많이 재생한 후 몇 기가바이트를 확보할 수도 있습니다.
{{% /details %}}

{{% details title="클라우드 동영상이 버퍼링되거나 끊기는 이유는 무엇인가요?" closed="true" %}}
**버퍼링은 거의 항상 느린 네트워크, 큰 파일 크기 또는 낮은 플레이어 버퍼 설정으로 인해 발생합니다 — Preloading Time을 늘리고 더 빠른 네트워크로 전환하거나 파일을 다운로드하세요.**<br><br>

몇 가지 실용적인 팁:<br>

- 고비트레이트 4K 또는 HEVC 콘텐츠의 경우 **설정 → Player → File Loading → Preloading Time**에서 **버퍼를 늘리세요.**<br>
- 셀룰러를 사용 중이라면 **Wi-Fi로 전환**하거나 라우터에 가까이 이동하세요.<br>
- **설정 → Player → Video**에서 H.264 및 HEVC에 대한 **하드웨어 디코딩을 활성화**하여 CPU가 병목이 되지 않도록 하세요.<br>
- 소스가 시청 속도보다 느린 경우 **오프라인 재생을 위해 파일을 다운로드**하세요 — 대용량 4K 파일은 셀룰러 대역폭을 쉽게 초과할 수 있습니다.<br>
- 연결이 만료된 경우 파일 탭에서 **클라우드 계정을 다시 인증**하세요.
{{% /details %}}

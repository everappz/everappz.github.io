{{% details title="Evervideo 是什麼？" closed="true" %}}
**Evervideo 是一款功能齊全的適用於 iPhone、iPad 和 Mac 的 HD 和 4K 雲端視訊播放器**，可將任何雲端儲存帳戶、NAS 或媒體伺服器變成您的個人影片資料庫，讓您完全掌控自己的檔案。<br><br>

基於自訂 FFmpeg 引擎，配備硬體加速 H.264 和 HEVC 解碼，Evervideo 幾乎可以播放任何現代容器和編解碼器（MP4、MKV、AVI、MOV、FLV、WMV、WebM、TS、M2TS 以及 FFmpeg 支援的眾多格式），支援主要和輔助字幕，內建音訊和視訊等化器，並支援畫中畫模式，讓您在使用其他應用程式時繼續觀看。

Premium 可作為**一次性買斷**或**按月或按年訂閱**購買，您可以選擇最適合自己的方案。價格可能因國家/地區而異。所有方案均啟用了**Family Sharing**，因此您最多可與五名家庭成員共享 Premium。買斷和訂閱透過 iCloud 在 iOS 和 macOS 之間共享——在每台裝置上安裝最新版本，用同一個 Apple ID 登入，等待約一分鐘即可同步購買資訊。<br><br>

{{% details title="Evervideo 安全嗎？" closed="true" %}}
Evervideo 僅使用官方 SDK 和安全連接與已連接的雲端服務互動。您的登入名稱和密碼不會提供給應用程式。所有從應用程式到雲端服務的請求均已加密。<br>
當您輸入登入名稱和密碼時，應用程式會向您顯示由雲端服務提供者提供的官方授權頁面，整個授權過程在應用程式外部完成。授權成功後，雲端服務提供者會向應用程式發送一個認證令牌，該令牌用於進行 API 呼叫。<br><br>

認證令牌是允許第三方應用程式與雲端儲存互動的數位金鑰。認證令牌儲存在您裝置上稱為 Keychain 的安全系統儲存中。您可以從已連接的雲端服務將檔案下載到裝置，這些檔案將放置在應用程式的「文件」目錄中。您可以隨時使用內建檔案管理員刪除這些檔案。<br>
應用程式不會共享來自已連接雲端帳戶的任何資訊。您可以隨時透過在網路瀏覽器上開啟帳戶設定頁面來撤銷對您雲端帳戶的存取權限。<br><br>

要拒絕認證令牌，請在網路瀏覽器上登入您的帳戶並導覽至設定頁面。在那裡您可以找到所有連接到您雲端帳戶的第三方應用程式，並刪除任何您不想繼續使用的應用程式。<br><br>

完整教學在此：<br>
[如何從您的 Google 帳戶中斷連接第三方應用程式](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

您還可以在應用程式中中斷連接已連接的雲端帳戶，認證令牌也將從您的裝置中刪除。如果您從裝置中刪除應用程式，所有已下載的資料和存取令牌也將被刪除。<br><br>

{{% details title="Evervideo 支援哪些視訊格式？" closed="true" %}}
**借助內建的 FFmpeg 引擎以及硬體加速 H.264 / HEVC 解碼，Evervideo 幾乎可以在 iPhone、iPad 和 Mac 上播放每一種現代視訊容器和編解碼器。**<br><br>

**容器：** MP4、M4V、MKV、MOV、AVI、FLV、WMV、ASF、WebM、TS、M2TS、MTS、MPG、MPEG、OGV、3GP、3G2、F4V、RM、RMVB、VOB、DAT 等更多。<br>
**視訊編解碼器：** H.264 (AVC)、H.265 (HEVC)、VP9、VP8、AV1、MPEG-2、MPEG-4、MJPEG、ProRes、Theora、WMV 以及 FFmpeg 支援的幾乎所有其他編解碼器。<br>
**音訊編解碼器（視訊內）：** AAC、MP3、FLAC、ALAC、OGG / Vorbis、OPUS、AC-3、EAC-3、DTS、AMR、WMA、APE。<br>
**字幕：** SRT、VTT (WebVTT)、ASS / SSA（透過 libass 算圖）、嵌入式圖像和文字軌道。<br>
**串流通訊協定：** HTTP / HTTPS、HLS (m3u8)、RTSP（IP 攝影機和 IPTV）。<br>
**直接串流傳輸：** SMB / WebDAV / FTP / SFTP / NFS / DLNA。<br>
{{% /details %}}

{{% details title="Evervideo 能在 iPhone 上播放 MKV 檔案嗎？" closed="true" %}}

這是人們選擇安裝 Evervideo 而非 Apple 預設播放器的主要原因之一：iOS 根本無法開啟 MKV 檔案，但 Evervideo 透過內建的 FFmpeg 引擎處理它們。您可以直接從雲端儲存或 NAS 串流傳輸 MKV 檔案，無需下載。

為在雲端串流傳輸 4K 時獲得最佳效果，請在**設定 → 播放器 → 檔案載入**中增加**預載時間**，以便緩衝區能跟上高位元率檔案，並連接 Wi-Fi 網路而非行動數據。

{{% details title="Evervideo 支援 H.264 和 HEVC 硬體解碼嗎？" closed="true" %}}
**是的——Evervideo 在每台支援的 iPhone、iPad 和 Mac 上預設使用 H.264 (AVC) 和 H.265 (HEVC) 硬體解碼器**，這表示與純軟體解碼相比，播放更流暢，電池消耗更低，裝置溫度更低。<br><br>

您可以在**設定 → 播放器 → 視訊 → 硬體解碼 H.264 / H.265** 中分別切換 H.264 和 HEVC 的硬體解碼。如果某個特定檔案存在相容性問題（損毀的串流、特殊設定檔），請停用該檔案的硬體解碼以回退到 FFmpeg 軟體解碼。

{{% details title="Evervideo 支援畫中畫（PiP）嗎？" closed="true" %}}

將浮動視窗拖到任意角落，捏合以調整大小，單擊一次可顯示基本的播放/暫停/跳過控制項，點選小展開按鈕可返回 Evervideo。PiP 支援 **Evervideo 播放的每種視訊格式**，包括雲端串流傳輸檔案和 RTSP IP 攝影機串流，即使手機鎖定後也繼續運行。

{{% details title="Evervideo 是否有適合本機影片的 YouTube 風格播放器介面？" closed="true" %}}
**是的——Evervideo 的預設版面配置圍繞 YouTube 風格的體驗為您的個人影片打造：畫面頂部始終顯示一個小型視訊播放器，而您可以在下方瀏覽資料庫。**<br><br>

與 YouTube 不同，這裡沒有廣告、沒有演算法推薦、沒有煩人的自動播放、沒有追蹤——您完全掌控來自 iCloud、Google Drive、Dropbox、NAS、Plex / Jellyfin / Emby 或任何其他來源的個人影片資料庫。

{{% details title="如何使用視訊等化器和預設？" closed="true" %}}
**開啟播放器，點選「更多操作」→ 視訊等化器，然後拖動亮度、對比度、飽和度和色調滑桿，或選擇預設。**<br><br>

Evervideo 的**視訊等化器**是在 FFmpeg 算圖管線內運行的即時畫面調整工具：<br>

- **亮度**——提亮弱光拍攝的暗場景，或降低過曝。<br>
- **對比度**——為褪色內容增加質感，或柔化過度對比的來源。<br>
- **飽和度**——增強暗淡色彩，或降低飽和度以獲得電影感。<br>
- **色調**——調整色彩平衡以修正綠色或洋紅偏色。<br><br>

您可以將喜歡的設定儲存為**自訂預設**，以後對任何影片一鍵套用。預設還可以**匯出和匯入**，在 iPhone、iPad 和 Mac 之間共享，或進行備份。將視訊等化器與**音訊等化器**（10 頻段 EQ，附獨立預設資料庫）結合使用，全面掌控畫面和聲音。
{{% /details %}}

{{% details title="如何旋轉影片或變更縮放模式？" closed="true" %}}
**點選播放器上的「更多操作」，選擇旋轉（0° / 90° / 180° / 270°）或縮放模式（符合 / 填滿 / 延伸 / 原始）。**<br><br>

**旋轉**適用於橫向或顛倒拍攝的影片——無需離開播放器即可旋轉畫面。**縮放模式**控制畫面如何填充螢幕：<br>

- **符合**——保持原始長寬比，在需要的地方顯示黑邊。<br>
- **填滿**——填充整個螢幕，必要時裁剪影片。<br>
- **延伸**——延伸以填充螢幕，使畫面變形。<br>
- **原始**——以 1:1 保持原始解析度。
{{% /details %}}

{{% details title="如何在影片中選擇不同的音訊軌道（配音、解說）？" closed="true" %}}
**點選播放器上的「更多操作」（「...」）按鈕，選擇音訊軌道——然後從清單中選擇所需軌道。**<br><br>

對於含有多個音訊軌道的影片（不同語言配音、導演解說、原聲/現場版本），Evervideo 會顯示每個嵌入式軌道及其語言和編解碼器資訊。適用於 MKV、MP4、M2TS 以及任何其他包含多個音訊串流的容器。
{{% /details %}}

{{% details title="如何在 Evervideo 中新增或變更字幕？" closed="true" %}}
**點選播放器上的「更多操作」（「...」）按鈕，選擇字幕——然後選擇嵌入式字幕軌道、載入外部字幕檔案或變更字型。**<br><br>

Evervideo 會自動列出影片中嵌入的每個字幕軌道。要載入外部字幕檔案，請選擇**外部檔案**，然後從裝置、iCloud Drive 或任何已連接的雲端服務中選擇 `.srt`、`.vtt`、`.ass` 或 `.ssa` 檔案。您還可以在**設定 → 播放器 → 字幕**中設定預設字幕行為。
{{% /details %}}

{{% details title="Evervideo 支援外部 SRT、VTT 和 ASS 字幕檔案嗎？" closed="true" %}}
**是的——Evervideo 可以從裝置上的任何位置或任何已連接的雲端服務載入 SRT、VTT (WebVTT)、ASS 和 SSA 格式的外部字幕檔案。**<br><br>

含有進階樣式（自訂字型、色彩、位置、卡拉OK效果）的 ASS / SSA 檔案可透過內建的 **libass** 函式庫正確算圖——非常適合動漫字幕組、專業字幕檔案和簡報。您還可以在**設定 → 播放器 → 字幕 → 字型**中為字幕指定特定字型。
{{% /details %}}

{{% details title="如何從 Evervideo 刪除影片而不刪除雲端儲存中的檔案？" closed="true" %}}
**在任意影片上，點選「...」→ 從媒體資料庫刪除——這將從您的資料庫中刪除該項目，但保留雲端儲存、NAS 或 iOS Photos 資料庫中的原始檔案不變。**<br><br>

如果您還想從來源刪除檔案，請選擇**從雲端服務刪除**或**從本機檔案刪除**。這些破壞性操作無法還原，因此在選取多個影片時請謹慎操作。<br><br>

要刪除下載的副本而不影響雲端任何內容，請開啟**檔案**標籤頁，在**此應用程式中的檔案**或**離線資料夾**下找到該影片，並使用**「...」→ 刪除**。雲端原始檔案保持原樣不變——Evervideo 只刪除本機副本。
{{% /details %}}

{{% details title="如何在 Evervideo 中變更播放速度？" closed="true" %}}
**開啟播放器，點選工具列上的速度控制項，拖動滑桿——支援 0.25× 到 3.00× 的速度。**<br><br>

您可以降速進行逐格分析（0.25× / 0.5×），或加速觀看教學和講座（1.25× / 1.5× / 2×）。點選速度畫面右上角的設定圖示可切換到精確模式，進行更精細的調整。每軌音調校正讓音訊在非 1× 速度下聽起來自然。

{{% details title="為什麼我的雲端影片出現緩衝或卡頓？" closed="true" %}}
**緩衝幾乎總是由網速慢、檔案體積大或播放器緩衝設定過低引起的——增加預載時間，切換到更快的網路，或下載檔案。**<br><br>

幾個實用建議：<br>

- 在**設定 → 播放器 → 檔案載入 → 預載時間**中為高位元率 4K 或 HEVC 內容**增加緩衝區**。<br>
- 如果您正在使用行動數據，請**切換到 Wi-Fi**，或靠近路由器。<br>
- 在**設定 → 播放器 → 視訊**中為 H.264 和 HEVC **啟用硬體解碼**，以避免 CPU 成為瓶頸。<br>
- 如果來源比您的觀看速度慢，請**下載檔案以供離線播放**——大型 4K 檔案很容易超過行動數據頻寬。<br>
- 如果連接已過期，請在檔案標籤頁中**重新授權您的雲端帳戶**。

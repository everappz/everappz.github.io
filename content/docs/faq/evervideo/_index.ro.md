{{% details title="Ce este Evervideo?" closed="true" %}}
**Evervideo este un player video cloud complet HD și 4K pentru iPhone, iPad și Mac** care transformă orice cont de stocare în cloud, NAS sau server media în biblioteca dvs. video personală cu control complet al fișierelor.<br><br>

Construit pe un motor personalizat bazat pe FFmpeg cu decodare hardware H.264 și HEVC, Evervideo redă practic orice container și codec modern (MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS și numeroasele formate FFmpeg), suportă subtitrări primare și secundare, include egalizatoare audio și video și funcționează în modul Picture-in-Picture astfel încât să puteți continua să vizionați în timp ce utilizați alte aplicații.
{{% /details %}}

{{% details title="Evervideo este sigur?" closed="true" %}}
Evervideo folosește doar SDK oficial și conexiuni securizate pentru a interacționa cu serviciile cloud conectate. Login-ul și parola dvs. nu sunt disponibile pentru aplicație. Toate solicitările din aplicație către serviciul cloud sunt criptate.<br>
Când introduceți login și parola, aplicația vă arată pagina oficială de autorizare furnizată de furnizorul de servicii cloud și tot procesul de autorizare se realizează în afara aplicației. Furnizorul de servicii cloud trimite un token de autentificare aplicației după autorizarea reușită, iar acel token este utilizat pentru a efectua apeluri API.<br><br>

Token-ul de autentificare este o cheie digitală care permite aplicațiilor terțe să interacționeze cu stocarea în cloud. Token-ul de autentificare este stocat pe dispozitivul dvs. în spațiul de stocare securizat de sistem numit Keychain. Puteți descărca fișierele de la serviciul cloud conectat pe dispozitiv, iar acele fișiere vor fi plasate în directorul "Documente" al aplicației. Puteți elimina acele fișiere oricând folosind managerul de fișiere încorporat.<br>
Aplicația nu partajează nicio informație din contul cloud conectat. Puteți revoca accesul la contul dvs. cloud oricând deschizând pagina de setări a contului în browserul dvs. web.<br><br>

Pentru a respinge token-ul de autentificare, conectați-vă la cont în browserul web și navigați la pagina de setări. Acolo puteți găsi toate aplicațiile terțe care sunt conectate la contul dvs. cloud și puteți elimina oricare dintre ele dacă nu mai doriți să folosiți acea aplicație.<br><br>

Un tutorial complet disponibil aici:<br>
[Cum să deconectați o aplicație terță de la contul dvs. Google](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

Puteți de asemenea deconecta conturile cloud conectate în aplicație, iar token-ul de autentificare va fi eliminat și de pe dispozitivul dvs. Dacă eliminați aplicația de pe dispozitiv, toate datele descărcate și token-urile de acces vor fi de asemenea eliminate.<br><br>

{{% details title="Ce formate video suportă Evervideo?" closed="true" %}}
**Evervideo redă practic orice container și codec video modern pe iPhone, iPad și Mac** datorită motorului FFmpeg integrat combinat cu decodarea hardware H.264 / HEVC.<br><br>

**Containere:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT și multe altele.<br>
**Codecuri video:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV și practic orice alt codec suportat de FFmpeg.<br>
**Codecuri audio (în interiorul video):** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE.<br>
**Subtitrări:** SRT, VTT (WebVTT), ASS / SSA (redate prin libass), piste de imagini și text încorporate.<br>
**Protocoale de streaming:** HTTP / HTTPS, HLS (m3u8), RTSP (camere IP și IPTV).<br>
**Streaming direct:** SMB / WebDAV / FTP / SFTP / NFS / DLNA.<br>

{{% details title="Evervideo redă fișiere MKV pe iPhone?" closed="true" %}}
**Da — Evervideo redă fișiere MKV nativ pe iPhone, iPad și Mac** fără conversie necesară, inclusiv containere MKV cu video H.264, HEVC, VP9 sau AV1, piste audio multiple și subtitrări SRT / ASS încorporate.<br><br>

Acesta este unul dintre principalele motive pentru care oamenii instalează Evervideo în locul playerului implicit Apple: iOS nu deschide fișierele MKV deloc, dar Evervideo le gestionează prin motorul FFmpeg integrat. Puteți reda în flux fișiere MKV direct din stocarea cloud sau un NAS fără a le descărca.

{{% details title="Evervideo suportă video 4K și HDR?" closed="true" %}}
**Da — Evervideo redă video 4K (Ultra HD) și conținut codificat HDR** pe fiecare dispozitiv care are hardware-ul necesar pentru a-l decoda. iPhone-urile moderne, iPad-urile și Mac-urile Apple Silicon decodează 4K H.264 și 4K HEVC în hardware, astfel redarea este fluidă și eficientă din punct de vedere al bateriei.<br><br>

Pentru cele mai bune rezultate la streaming 4K din cloud, creșteți **Timpul de Preîncărcare** în **Setări → Player → Încărcare Fișier** astfel încât buffer-ul să poată ține pasul cu fișierele de înaltă calitate, și conectați-vă la o rețea Wi-Fi în loc de rețeaua celulară.
{{% /details %}}

{{% details title="Evervideo suportă decodarea hardware H.264 și HEVC?" closed="true" %}}
**Da — Evervideo folosește implicit decodoare hardware H.264 (AVC) și H.265 (HEVC)** pe fiecare iPhone, iPad și Mac care le suportă, ceea ce înseamnă redare mai fluidă, consum mai mic al bateriei și temperatură mai scăzută a dispozitivului comparativ cu decodarea prin software pur.<br><br>

Puteți comuta decodarea hardware independent pentru H.264 și HEVC în **Setări → Player → Video → Decodare Hardware H.264 / H.265**. Dacă un anumit fișier are probleme de compatibilitate (fluxuri corupte, profiluri exotice), dezactivați decodarea hardware pentru acel fișier pentru a reveni la decodarea software FFmpeg.

{{% details title="Evervideo suportă Picture-in-Picture (PiP)?" closed="true" %}}
**Da — Evervideo suportă complet Picture-in-Picture pe iPhone și iPad.** Când atingeți pictograma PiP pe player, videoclipul continuă să ruleze într-o fereastră flotantă deasupra tuturor celorlalte aplicații.<br><br>

Trageți fereastra flotantă în orice colț, pinch pentru a redimensiona, atingeți o dată pentru a afișa controalele de bază redare / pauză / săritură și atingeți butonul mic de expandare pentru a reveni la Evervideo. PiP funcționează cu **orice format video pe care îl redă Evervideo**, inclusiv fișierele transmise în flux din cloud și fluxurile RTSP de la camerele IP, și continuă să funcționeze cât timp telefonul este blocat.

{{% details title="Evervideo are o interfață player în stil YouTube pentru videoclipurile mele locale?" closed="true" %}}
**Da — aspectul implicit al Evervideo este construit în jurul unei experiențe în stil YouTube pentru propriile dvs. videoclipuri: un player video compact permanent vizibil în partea de sus a ecranului în timp ce navigați în biblioteca dvs. de mai jos.**<br><br>

**Player-ul compact** rămâne pe ecran pe fiecare filă — Recente, Favorite, Bibliotecă Media, Liste de Redare, Fișiere și Setări — astfel încât să puteți naviga, căuta, organiza și pune în coadă următorul videoclip fără a întrerupe niciodată redarea. Atingeți playerul compact pentru a-l extinde la vizualizarea pe ecran complet; glisați în jos pentru a-l restrânge înapoi la compact fără a opri videoclipul. Pe macOS, playerul compact se poate separa într-o **fereastră flotantă mereu deasupra** pentru adevărată multitasking în stil picture-in-picture pe desktop.<br><br>

Spre deosebire de YouTube, nu există reclame, recomandări algoritmice, redare automată forțată sau urmărire — vă păstrați controlul complet asupra propriei biblioteci video din iCloud, Google Drive, Dropbox, NAS-ul dvs., Plex / Jellyfin / Emby sau oriunde altundeva.

{{% details title="Evervideo redă videoclipuri 360° de la Insta360 și alte camere 360?" closed="true" %}}

{{% details title="Cum folosesc egalizatorul video și presetările?" closed="true" %}}
**Deschideți playerul, atingeți Mai Multe Acțiuni → Egalizator Video, apoi trageți glisoarele pentru luminozitate, contrast, saturație și nuanță — sau alegeți o presetare.**<br><br>

**Egalizatorul video** Evervideo este un instrument de ajustare a imaginii în timp real care rulează în interiorul pipeline-ului de randare FFmpeg:<br>

- **Luminozitate** — luminați o scenă întunecată filmată în lumină scăzută sau reduceți o expunere supraexpusă.<br>
- **Contrast** — adăugați vivacitate conținutului decolorat sau atenuați o sursă supracontrastată.<br>
- **Saturație** — încălziți culorile estompate sau desaturați pentru un aspect cinematografic.<br>
- **Nuanță** — deplasați echilibrul de culori pentru a corecta o dominantă verde sau magenta.<br><br>

Puteți salva setările preferate ca **presetare personalizată** și o puteți reaplicta cu o singură atingere pe orice videoclip viitor. Presetările pot fi de asemenea **exportate și importate** pentru a le partaja între iPhone, iPad și Mac, sau pentru a le face backup. Combinați egalizatorul video cu **egalizatorul audio** (EQ cu 10 benzi cu propria bibliotecă de presetări) pentru control complet atât al imaginii, cât și al sunetului.

{{% details title="Cum rotesc sau schimb modul de scalare al unui videoclip?" closed="true" %}}
**Atingeți Mai Multe Acțiuni pe player și alegeți Rotire (0° / 90° / 180° / 270°) sau Mod de Scalare (Potrivit / Umplere / Întindere / Original).**<br><br>

**Rotirea** este utilă pentru videoclipurile înregistrate lateral sau cu susul în jos — rotiți imaginea fără a părăsi playerul. **Modul de Scalare** controlează cum se umple imaginea ecranul:<br>

- **Potrivit** — păstrați raportul original de aspect cu bare negre unde este necesar.<br>
- **Umplere** — umpleți întregul ecran, tăind videoclipul dacă este necesar.<br>
- **Întindere** — întindeți pentru a umple ecranul, distorsionând imaginea.<br>
- **Original** — păstrați rezoluția nativă la 1:1.
{{% /details %}}

{{% details title="Cum aleg o pistă audio diferită (dublaj, comentariu) într-un videoclip?" closed="true" %}}
**Atingeți butonul Mai Multe Acțiuni ('...') pe player și alegeți Pistă Audio — apoi alegeți pista dorită din listă.**<br><br>

Pentru videoclipurile cu piste audio multiple (dublaje în limbi alternative, comentariul regizorului, mixuri originale / live), Evervideo afișează fiecare pistă încorporată cu limba și codecul său. Aceasta funcționează pentru MKV, MP4, M2TS și orice alt container care expune fluxuri audio multiple.
{{% /details %}}

{{% details title="Cum adaug sau schimb subtitrările în Evervideo?" closed="true" %}}

Evervideo listează automat fiecare pistă de subtitrări încorporată într-un videoclip. Pentru a încărca un fișier de subtitrări extern, alegeți **Fișier Extern** și selectați un fișier `.srt`, `.vtt`, `.ass` sau `.ssa` de pe dispozitivul dvs., iCloud Drive sau orice serviciu cloud conectat. **Da — Evervideo încarcă fișiere de subtitrări externe în formatele SRT, VTT (WebVTT), ASS și SSA** de oriunde pe dispozitivul dvs. sau orice serviciu cloud conectat.<br><br>

Fișierele ASS / SSA cu stilizare avansată (fonturi personalizate, culori, poziții, efecte karaoke) sunt randate corect datorită bibliotecii **libass** integrate — perfect pentru fansub-uri anime, fișiere de subtitrări profesionale și prezentări. {{% details title="Cum șterg un videoclip din Evervideo fără a-l șterge din stocarea cloud?" closed="true" %}}
**Pe orice videoclip, atingeți '...' → Șterge din Biblioteca Media — aceasta elimină înregistrarea din baza de date a bibliotecii dvs., dar lasă fișierul original neatins în stocarea cloud, NAS-ul dvs. sau biblioteca iOS Fotografii.**<br><br>

Dacă doriți să eliminați și fișierul din sursa sa, alegeți **Șterge din Serviciul Cloud** sau **Șterge din Fișierele Locale** în schimb. Aceste acțiuni distructive nu pot fi anulate, deci fiți atenți când aveți mai multe videoclipuri selectate.<br><br>

Pentru a elimina o copie descărcată fără a atinge nimic în cloud, deschideți fila **Fișiere**, găsiți videoclipul sub **Fișiere în Această Aplicație** sau **Foldere Offline** și folosiți **'...' → Șterge**. Originalul din cloud rămâne exact unde era — Evervideo elimină doar copia locală.

{{% details title="Evervideo suportă Apple CarPlay?" closed="true" %}}
**Nu — Evervideo nu suportă Apple CarPlay.** Apple CarPlay este restricționat la aplicații exclusiv audio (muzică, podcasturi, cărți audio, navigație), deci un player video nu poate rula pe un ecran CarPlay conform politicii Apple.<br><br>

Dacă doriți o aplicație conectată la cloud care suportă CarPlay, aplicațiile noastre de muzică **Evermusic** și **Flacbox** funcționează complet pe CarPlay cu file dedicate pentru Bibliotecă, Conexiuni, Fișiere Locale și Setări.

Mutați fișierul de backup pe noul dispozitiv prin iCloud Drive, AirDrop sau orice serviciu cloud conectat, apoi deschideți-l în Evervideo pentru a aplica.
{{% /details %}}

{{% details title="Cum eliberez spațiul de stocare folosit de Evervideo?" closed="true" %}}
**Deschideți Setări → Manager Fișiere → Ștergere Fișiere Temporare și Setări → Bibliotecă Media → Coperte Album → Ștergere Tot pentru a curăța cache-urile; folosiți fila Fișiere pentru a șterge videoclipurile descărcate de care nu mai aveți nevoie.**<br><br>

Puteți de asemenea elimina foldere offline individuale în **Setări → Manager Fișiere → Foldere Offline Sincronizate → '...' → Dezactivare Mod Offline**, care șterge copiile locale. Videoclipurile doar pentru streaming nu folosesc deloc stocarea dispozitivului. Curățarea folderului **Cache Player** din **Fișiere → Fișiere în Această Aplicație** poate de asemenea elibera câțiva gigabyți după redare intensivă a conținutului 4K cu înaltă calitate.
{{% /details %}}

{{% details title="De ce video-ul meu din cloud se bufferizează sau se blochează?" closed="true" %}}
**Bufferizarea este aproape întotdeauna cauzată de rețea lentă, dimensiuni mari ale fișierelor sau setări mici ale buffer-ului playerului — creșteți Timpul de Preîncărcare și treceți la o rețea mai rapidă sau descărcați fișierul.**<br><br>

Câteva sfaturi practice:<br>

- **Creșteți buffer-ul** în **Setări → Player → Încărcare Fișier → Timp de Preîncărcare** pentru conținut 4K sau HEVC de înaltă calitate.<br>
- **Treceți la Wi-Fi** dacă sunteți pe rețea celulară, sau apropiați-vă de router.<br>
- **Activați Decodarea Hardware** pentru H.264 și HEVC în **Setări → Player → Video** astfel încât CPU să nu fie blocajul.<br>
- **Descărcați fișierul pentru redare offline** dacă sursa este mai lentă decât viteza dvs. de vizionare — fișierele 4K mari pot depăși cu ușurință lățimea de bandă celulară.<br>
- **Re-autorizați contul cloud** din fila Fișiere dacă o conexiune a expirat.

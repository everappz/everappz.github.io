Το Evervideo είναι ένας HD και 4K **cloud βιντεοαναπαραγωγέας** για **iPhone, iPad και Mac** που κάνει ροή και λήψη βίντεο από iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP και οποιαδήποτε πηγή WebDAV, SMB, NFS, FTP / SFTP, DLNA ή S3 — καθώς και διακομιστές πολυμέσων όπως **Plex, Jellyfin, Emby, Subsonic** και **Navidrome**, και ζωντανές ροές **RTSP** από κάμερες IP. Βασισμένο σε μια προσαρμοσμένη **μηχανή αναπαραγωγής βασισμένη σε FFmpeg** με **αποκωδικοποίηση H.264 / HEVC με επιτάχυνση υλικού**, πρωτεύουσες και δευτερεύουσες κομματιές υποτίτλων (SRT / VTT / ASS / libass), ισοσταθμιστή βίντεο με προεπιλογές, ισοσταθμιστή ήχου, **Εικόνα-μέσα-σε-Εικόνα**, **αναπαραγωγή 360° / VR** για υλικό Insta360 και GoPro Max, streaming **AirPlay 2** και **Chromecast**, και έναν πάντα ορατό συμπαγή αναπαραγωγέα. Αυτές οι συχνές ερωτήσεις απαντούν στις πιο συνηθισμένες ερωτήσεις χρηστών. Για πιο αναλυτικές οδηγίες, δείτε τον [Οδηγό Χρήστη Evervideo](/docs/guide/evervideo/).

{{% details title="Είναι ασφαλές το Evervideo;" closed="true" %}}
Το Evervideo χρησιμοποιεί μόνο επίσημα SDK και ασφαλείς συνδέσεις για αλληλεπίδραση με συνδεδεμένες υπηρεσίες cloud. Το login και ο κωδικός πρόσβασής σας δεν είναι διαθέσιμα στην εφαρμογή. Όλα τα αιτήματα από την εφαρμογή προς την υπηρεσία cloud είναι κρυπτογραφημένα.<br>
Όταν εισάγετε login και κωδικό πρόσβασης, η εφαρμογή σάς εμφανίζει την επίσημη σελίδα εξουσιοδότησης που παρέχεται από τον πάροχο υπηρεσιών cloud και ολόκληρη η διαδικασία εξουσιοδότησης γίνεται εκτός της εφαρμογής. Ο πάροχος υπηρεσιών cloud στέλνει ένα auth-token στην εφαρμογή μετά την επιτυχή εξουσιοδότηση και αυτό το token χρησιμοποιείται για κλήσεις API.<br><br>

Το auth-token είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με τον αποθηκευτικό χώρο cloud. Το auth-token αποθηκεύεται στη συσκευή σας σε ασφαλή αποθήκευση συστήματος που ονομάζεται Keychain. Μπορείτε να κατεβάσετε τα αρχεία σας από τη συνδεδεμένη υπηρεσία cloud στη συσκευή και αυτά τα αρχεία θα τοποθετηθούν στον κατάλογο 'Έγγραφα' της εφαρμογής. Μπορείτε να αφαιρέσετε αυτά τα αρχεία ανά πάσα στιγμή χρησιμοποιώντας τον ενσωματωμένο διαχειριστή αρχείων.<br>
Η εφαρμογή δεν κοινοποιεί καμία πληροφορία από τον συνδεδεμένο λογαριασμό cloud. Μπορείτε να ανακαλέσετε την πρόσβαση στον λογαριασμό cloud σας ανά πάσα στιγμή ανοίγοντας τη σελίδα ρυθμίσεων λογαριασμού στο πρόγραμμα περιήγησης.<br><br>

Για να απορρίψετε το auth-token, συνδεθείτε στον λογαριασμό σας στο πρόγραμμα περιήγησης και μεταβείτε στη σελίδα ρυθμίσεων. Εκεί μπορείτε να βρείτε όλες τις εφαρμογές τρίτων που είναι συνδεδεμένες με τον λογαριασμό cloud σας και να αφαιρέσετε οποιαδήποτε από αυτές αν δεν θέλετε να χρησιμοποιείτε πλέον την εφαρμογή.<br><br>

Διαθέσιμο πλήρες σεμινάριο εδώ:<br>
[Πώς να αποσυνδέσετε μια εφαρμογή τρίτου μέρους από τον λογαριασμό σας Google](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

Μπορείτε επίσης να αποσυνδέσετε τους συνδεδεμένους λογαριασμούς cloud στην εφαρμογή και το auth-token θα αφαιρεθεί επίσης από τη συσκευή σας. Αν αφαιρέσετε την εφαρμογή από τη συσκευή σας, όλα τα ληφθέντα δεδομένα και τα access token θα αφαιρεθούν επίσης.<br><br>

**Το Evervideo αναπαράγει σχεδόν κάθε σύγχρονο container βίντεο και κωδικοποιητή σε iPhone, iPad και Mac** χάρη στη μηχανή FFmpeg σε συνδυασμό με αποκωδικοποίηση H.264 / HEVC με επιτάχυνση υλικού.<br><br>

**Containers:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT και πολλά άλλα.<br>
**Κωδικοποιητές βίντεο:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV και σχεδόν κάθε άλλος κωδικοποιητής που υποστηρίζει το FFmpeg.<br>
**Κωδικοποιητές ήχου (μέσα σε βίντεο):** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE.<br>
**Υπότιτλοι:** SRT, VTT (WebVTT), ASS / SSA (αποδοσμένοι μέσω libass), ενσωματωμένα κομμάτια εικόνας και κειμένου.<br>
**Πρωτόκολλα ροής:** HTTP / HTTPS, HLS (m3u8), RTSP (κάμερες IP και IPTV).<br>
**Άμεση ροή:** SMB / WebDAV / FTP / SFTP / NFS / DLNA.<br>

Αν ένα συγκεκριμένο αρχείο έχει προβλήματα συμβατότητας (κατεστραμμένες ροές, εξωτικά προφίλ), απενεργοποιήστε την αποκωδικοποίηση υλικού για αυτό το αρχείο για επιστροφή στην αποκωδικοποίηση λογισμικού FFmpeg.

Το PiP λειτουργεί με **κάθε μορφή βίντεο που αναπαράγει το Evervideo**, συμπεριλαμβανομένων αρχείων σε ροή cloud και ροών κάμερας IP RTSP, και συνεχίζει να λειτουργεί ενώ το τηλέφωνό σας είναι κλειδωμένο.

Ο **ισοσταθμιστής βίντεο** Evervideo είναι ένα εργαλείο ρύθμισης εικόνας σε πραγματικό χρόνο που λειτουργεί μέσα στη γραμμή απόδοσης FFmpeg:<br>

- **Αντίθεση** — προσθέστε βάθος σε ξεθωριασμένο περιεχόμενο ή μαλακώστε μια υπεραντίθετη πηγή.<br>

{{% details title="Γιατί κάνει buffer ή κοκκάλωμα το βίντεο cloud μου;" closed="true" %}}
**Το buffer προκαλείται σχεδόν πάντα από αργό δίκτυο, μεγάλα μεγέθη αρχείων ή χαμηλές ρυθμίσεις buffer αναπαραγωγέα — αυξήστε τον Χρόνο Προφόρτωσης και μεταβείτε σε ταχύτερο δίκτυο ή κατεβάστε το αρχείο.**<br><br>

Μερικές πρακτικές συμβουλές:<br>

- **Αύξηση buffer** στις **Ρυθμίσεις → Αναπαραγωγέας → Φόρτωση Αρχείου → Χρόνος Προφόρτωσης** για περιεχόμενο 4K ή HEVC υψηλού ρυθμού bit.<br>
- **Μεταβείτε σε Wi-Fi** αν είστε σε κινητό δίκτυο ή πλησιάστε στο router σας.<br>
- **Ενεργοποιήστε Αποκωδικοποίηση Υλικού** για H.264 και HEVC στις **Ρυθμίσεις → Αναπαραγωγέας → Βίντεο** ώστε η CPU να μην είναι το σημείο συμφόρησης.<br>
- **Κατεβάστε το αρχείο για εκτός σύνδεσης αναπαραγωγή** αν η πηγή είναι πιο αργή από την ταχύτητα προβολής σας — τα μεγάλα αρχεία 4K μπορούν εύκολα να υπερβούν το εύρος ζώνης κινητής τηλεφωνίας.<br>
- **Επανεξουσιοδοτήστε τον λογαριασμό cloud** στην καρτέλα Αρχεία αν μια σύνδεση έχει λήξει.

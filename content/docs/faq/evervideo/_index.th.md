Evervideo คือ**เครื่องเล่นวิดีโอคลาวด์** HD และ 4K สำหรับ **iPhone, iPad และ Mac** ที่สตรีมและดาวน์โหลดวิดีโอจาก iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP และแหล่ง WebDAV, SMB, NFS, FTP / SFTP, DLNA หรือ S3 ใดก็ได้ รวมถึงเซิร์ฟเวอร์สื่อที่โฮสต์เองอย่าง **Plex, Jellyfin, Emby, Subsonic** และ **Navidrome** และสตรีม **RTSP** สดจากกล้อง IP สร้างขึ้นบน**เครื่องยนต์เล่นที่ใช้ FFmpeg** แบบกำหนดเองพร้อม**การถอดรหัสฮาร์ดแวร์ H.264 / HEVC**, แทร็กซับไตเติ้ลหลักและรอง (SRT / VTT / ASS / libass), อีควอไลเซอร์วิดีโอพร้อมพรีเซ็ต, อีควอไลเซอร์เสียง, **Picture-in-Picture**, **การเล่น 360° / VR** สำหรับฟุตเทจจาก Insta360 และ GoPro Max, การแคสต์ **AirPlay 2** และ **Chromecast** และเครื่องเล่นขนาดกะทัดรัดที่แสดงอยู่เสมอ คำถามที่พบบ่อยนี้ตอบคำถามที่ผู้ใช้ส่งมาบ่อยที่สุด สำหรับการอธิบายเชิงลึก ดู [คู่มือผู้ใช้ Evervideo](/docs/guide/evervideo/)

{{% details title="Evervideo คืออะไร?" closed="true" %}}
**Evervideo คือเครื่องเล่นวิดีโอคลาวด์ HD และ 4K ที่มีฟีเจอร์ครบครันสำหรับ iPhone, iPad และ Mac** ที่เปลี่ยนบัญชีที่เก็บข้อมูลบนคลาวด์ NAS หรือเซิร์ฟเวอร์สื่อใดก็ได้ให้เป็นคลังวิดีโอส่วนตัวของคุณพร้อมการควบคุมไฟล์ได้อย่างเต็มที่<br><br>

สร้างขึ้นบนเครื่องยนต์เล่นที่ใช้ FFmpeg แบบกำหนดเองพร้อมการถอดรหัสฮาร์ดแวร์ H.264 และ HEVC Evervideo เล่นได้แทบทุกคอนเทนเนอร์และโคเดกสมัยใหม่ (MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS และรูปแบบ FFmpeg อื่นๆ อีกมากมาย) รองรับซับไตเติ้ลหลักและรอง มีอีควอไลเซอร์เสียงและวิดีโอ และทำงานในโหมด Picture-in-Picture เพื่อให้คุณดูต่อไปได้ขณะใช้แอปอื่น

{{% details title="Evervideo ฟรีหรือไม่? ความแตกต่างระหว่างฟรีและพรีเมียมคืออะไร?" closed="true" %}}
**ใช่ — Evervideo ดาวน์โหลดและใช้งานได้ฟรี** โดยมีการซื้อในแอปเพิ่มเติมเพื่อลบข้อจำกัดและปลดล็อกชุดฟีเจอร์พรีเมียมทั้งหมด<br><br>

พรีเมียมมีให้เลือกเป็น**การซื้อตลอดชีพแบบครั้งเดียว** หรือ**การสมัครสมาชิกรายเดือนหรือรายปี** เพื่อให้คุณเลือกตัวเลือกที่เหมาะกับคุณที่สุด ราคาอาจแตกต่างกันไปตามประเทศ **Family Sharing** เปิดใช้งานสำหรับทุกแผน ดังนั้นคุณสามารถแชร์พรีเมียมกับสมาชิกในครอบครัวได้สูงสุดห้าคน การซื้อตลอดชีพและการสมัครสมาชิกถูกแชร์ระหว่าง iOS และ macOS ผ่าน iCloud — ติดตั้งเวอร์ชันล่าสุดบนแต่ละอุปกรณ์ ลงชื่อเข้าใช้ด้วย Apple ID เดียวกัน และรอประมาณหนึ่งนาทีเพื่อซิงค์ข้อมูลการซื้อ<br><br>

**Evervideo Premium** ลบข้อจำกัดทุกอย่างของเวอร์ชันฟรี: การเล่นแบบไม่มีโฆษณา เพลย์ลิสต์ไม่จำกัด บริการคลาวด์ที่เชื่อมต่อไม่จำกัด รายการโปรดไม่จำกัด การดาวน์โหลดออฟไลน์ไม่จำกัด คลังเก็บ (ZIP) ของคอลเลกชันสื่อ และการปรับแต่งส่วนบุคคลเต็มรูปแบบ (ไอคอนแอปกำหนดเอง ธีม โทนสี)<br><br>

{{% details title="Evervideo ปลอดภัยหรือไม่?" closed="true" %}}
Evervideo ใช้เฉพาะ SDK อย่างเป็นทางการและการเชื่อมต่อที่ปลอดภัยเพื่อโต้ตอบกับบริการคลาวด์ที่เชื่อมต่อ ชื่อผู้ใช้และรหัสผ่านของคุณไม่สามารถเข้าถึงได้โดยแอปพลิเคชัน คำขอทั้งหมดจากแอปพลิเคชันไปยังบริการคลาวด์ถูกเข้ารหัส<br>
เมื่อคุณป้อนชื่อผู้ใช้และรหัสผ่าน แอปพลิเคชันจะแสดงหน้าการอนุญาตอย่างเป็นทางการที่ผู้ให้บริการคลาวด์จัดหาให้ และกระบวนการอนุญาตทั้งหมดเกิดขึ้นนอกแอปพลิเคชัน ผู้ให้บริการคลาวด์ส่ง auth-token ไปยังแอปพลิเคชันหลังจากการอนุญาตสำเร็จ และโทเค็นนั้นถูกใช้เพื่อเรียก API<br><br>

Auth-token คือกุญแจดิจิทัลที่อนุญาตให้แอปพลิเคชันของบุคคลที่สามโต้ตอบกับที่เก็บข้อมูลบนคลาวด์ Auth-token ถูกจัดเก็บบนอุปกรณ์ของคุณในที่เก็บข้อมูลระบบที่ปลอดภัยที่เรียกว่า Keychain คุณสามารถดาวน์โหลดไฟล์จากบริการคลาวด์ที่เชื่อมต่อไปยังอุปกรณ์และไฟล์เหล่านั้นจะถูกวางในไดเรกทอรี "เอกสาร" ของแอป คุณสามารถลบไฟล์เหล่านั้นได้ตลอดเวลาโดยใช้ตัวจัดการไฟล์ในตัว<br>
แอปพลิเคชันไม่แบ่งปันข้อมูลใดๆ จากบัญชีคลาวด์ที่เชื่อมต่อ คุณสามารถเพิกถอนการเข้าถึงบัญชีคลาวด์ได้ตลอดเวลาโดยการเปิดหน้าการตั้งค่าบัญชีในเว็บเบราว์เซอร์ของคุณ<br><br>

เพื่อปฏิเสธ auth-token ให้เข้าสู่ระบบบัญชีของคุณในเว็บเบราว์เซอร์และไปที่หน้าการตั้งค่า ที่นั่นคุณสามารถค้นหาแอปของบุคคลที่สามทั้งหมดที่เชื่อมต่อกับบัญชีคลาวด์ของคุณและลบแอปใดก็ได้หากคุณไม่ต้องการใช้แอปพลิเคชันนั้นอีกต่อไป<br><br>

คุณยังสามารถตัดการเชื่อมต่อบัญชีคลาวด์ที่เชื่อมต่อในแอปพลิเคชันและ auth-token จะถูกลบออกจากอุปกรณ์ของคุณด้วย หากคุณลบแอปพลิเคชันออกจากอุปกรณ์ ข้อมูลที่ดาวน์โหลดและโทเค็นการเข้าถึงทั้งหมดจะถูกลบออกด้วย<br><br>

{{% details title="Evervideo รองรับรูปแบบวิดีโออะไรบ้าง?" closed="true" %}}
**Evervideo เล่นได้แทบทุกคอนเทนเนอร์และโคเดกวิดีโอสมัยใหม่บน iPhone, iPad และ Mac** ต้องขอบคุณเครื่องยนต์ FFmpeg ที่รวมมาด้วยพร้อมการถอดรหัสฮาร์ดแวร์ H.264 / HEVC<br><br>

**คอนเทนเนอร์:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT และอีกมากมาย<br>
**โคเดกวิดีโอ:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV และโคเดกอื่นๆ แทบทุกตัวที่ FFmpeg รองรับ<br>
**โคเดกเสียง (ในวิดีโอ):** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE<br>
**ซับไตเติ้ล:** SRT, VTT (WebVTT), ASS / SSA (เรนเดอร์ผ่าน libass) แทร็กภาพและข้อความที่ฝังอยู่<br>
**โปรโตคอลสตรีม:** HTTP / HTTPS, HLS (m3u8), RTSP (กล้อง IP และ IPTV)<br>
**สตรีมโดยตรง:** SMB / WebDAV / FTP / SFTP / NFS / DLNA<br>

{{% details title="Evervideo รองรับวิดีโอ 4K และ HDR ไหม?" closed="true" %}}
**ใช่ — Evervideo เล่นวิดีโอ 4K (Ultra HD) และเนื้อหาที่เข้ารหัส HDR** บนทุกอุปกรณ์ที่มีฮาร์ดแวร์ถอดรหัส iPhone, iPad และ Mac Apple Silicon สมัยใหม่ทั้งหมดถอดรหัส 4K H.264 และ 4K HEVC ด้วยฮาร์ดแวร์ ดังนั้นการเล่นจึงราบรื่นและประหยัดแบตเตอรี่<br><br>

{{% details title="Evervideo รองรับการถอดรหัสฮาร์ดแวร์ H.264 และ HEVC ไหม?" closed="true" %}}
**ใช่ — Evervideo ใช้ตัวถอดรหัสฮาร์ดแวร์ H.264 (AVC) และ H.265 (HEVC) โดยค่าเริ่มต้น** บน iPhone, iPad และ Mac ทุกเครื่องที่รองรับ ซึ่งหมายความว่าการเล่นราบรื่นกว่า ใช้แบตเตอรี่น้อยกว่า และอุณหภูมิอุปกรณ์เย็นกว่าเมื่อเปรียบเทียบกับการถอดรหัสด้วยซอฟต์แวร์ล้วน<br><br>

คุณสามารถสลับการถอดรหัสฮาร์ดแวร์แยกกันสำหรับ H.264 และ HEVC ใน **การตั้งค่า → เครื่องเล่น → วิดีโอ → ถอดรหัสฮาร์ดแวร์ H.264 / H.265** หากไฟล์เฉพาะมีปัญหาความเข้ากันได้ (สตรีมเสีย โปรไฟล์แปลก) ให้ปิดการถอดรหัสฮาร์ดแวร์สำหรับไฟล์นั้นเพื่อกลับไปใช้การถอดรหัสซอฟต์แวร์ FFmpeg
{{% /details %}}

ผู้ใช้พรีเมียมสามารถเชื่อมต่อบริการได้ไม่จำกัด เวอร์ชันฟรีจำกัดที่สาม

{{% details title="Evervideo รองรับ Plex, Jellyfin, Emby, Subsonic และ Navidrome ไหม?" closed="true" %}}
**ใช่ — Evervideo เชื่อมต่อกับ Plex Media Server, Jellyfin, Emby, Subsonic และ Navidrome แบบ native** เพื่อให้คุณสตรีมคลังวิดีโอที่โฮสต์เองโดยตรงโดยไม่เปิดเผยการแชร์ไฟล์ที่อยู่เบื้องหลัง<br><br>

- **Plex Media Server** — แตะ **ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์ → Plex** ลงชื่อเข้าใช้ด้วยบัญชี Plex และเลือกเซิร์ฟเวอร์ เซิร์ฟเวอร์ Plex ในเครือข่ายท้องถิ่นเดียวกันจะถูกค้นหาโดยอัตโนมัติในส่วน**อุปกรณ์ที่มีให้บริการ**<br>
- **Jellyfin** (โอเพ่นซอร์ส) — แตะ **ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์ → Jellyfin** ป้อน URL เซิร์ฟเวอร์ของคุณ (เช่น `http://server-ip:8096`) และข้อมูลรับรอง<br>
- **Emby** (เชิงพาณิชย์) — แตะ **ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์ → Emby** และป้อน URL เซิร์ฟเวอร์และการเข้าสู่ระบบ<br>
- **Subsonic และเซิร์ฟเวอร์ที่เข้ากันได้กับ Subsonic** — แตะ **ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์ → Subsonic** ป้อน URL เซิร์ฟเวอร์และข้อมูลรับรอง เส้นทาง API เดียวกันใช้งานได้กับ **Navidrome**, **Airsonic**, **Funkwhale**, **Gonic**, **Logitech Media Server (LMS)** และ **Ampache**<br><br>

เมื่อเชื่อมต่อแล้ว แต่ละเซิร์ฟเวอร์จะปรากฏควบคู่กับบัญชีคลาวด์ของคุณในแท็บไฟล์ คุณสามารถเรียกดูภาพยนตร์ รายการทีวี วิดีโอบ้าน เพลง เพลย์ลิสต์ และคอลเลกชัน ดาวน์โหลดเพื่อการเล่นออฟไลน์ จัดคิวรายการในเครื่องเล่น และดึงเข้าคลังสื่อส่วนกลางของคุณ — ทั้งหมดโดยไม่ต้องออกจาก Evervideo
{{% /details %}}

{{% details title="Evervideo เชื่อมต่อผ่าน SMB, WebDAV, FTP / SFTP, NFS และ DLNA ไหม?" closed="true" %}}
**ใช่ — Evervideo รองรับทุกโปรโตคอล NAS และแชร์ไฟล์หลัก: SMB (SMB1, SMB2, Auto), WebDAV (HTTP / HTTPS), FTP / FTPS, SFTP (รหัสผ่านหรือการยืนยันตัวตนด้วยคีย์สาธารณะ), NFS และ DLNA / UPnP**<br><br>

สิ่งนี้ช่วยให้คุณเชื่อมต่อกับอุปกรณ์ NAS เกือบทุกชนิด (Synology, QNAP, WD My Cloud Home, Buffalo, Apple Time Capsule) การแชร์ไฟล์ Linux / macOS / Windows เซิร์ฟเวอร์ Nextcloud / ownCloud ที่โฮสต์เอง หรือเซิร์ฟเวอร์สื่อ UPnP / DLNA ใดก็ได้ ทั้งหมดจากเมนู**ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์**
{{% /details %}}

{{% details title="Evervideo รองรับที่เก็บข้อมูลอ็อบเจกต์ที่เข้ากันได้กับ S3 ไหม?" closed="true" %}}
**ใช่ — Evervideo มีตัวเชื่อมต่อที่เข้ากันได้กับ S3** ที่ทำงานกับ **AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces** และบริการอื่นๆ ที่เปิดเผย endpoint S3-API<br><br>

แตะ **ไฟล์ → เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์ → ที่เก็บข้อมูล S3** จากนั้นป้อน URL endpoint ภูมิภาค คีย์การเข้าถึง คีย์ลับ และชื่อ bucket เมื่อเชื่อมต่อแล้ว bucket จะทำงานเหมือนคลาวด์อื่นๆ — เรียกดู สตรีม ดาวน์โหลด จัดคิว และเพิ่มในคลังของคุณ
{{% /details %}}

{{% details title="ฉันสามารถเล่นสตรีม RTSP (กล้อง IP, IPTV) ใน Evervideo ได้ไหม?" closed="true" %}}
**ใช่ — Evervideo มีการรองรับ RTSP แบบ native** ดังนั้นคุณสามารถชี้ไปที่ URL `rtsp://` ใดก็ได้ — กล้องรักษาความปลอดภัย กล้องออดี้ประตู เครื่องเฝ้าทารก ผู้ให้บริการ IPTV ฟีดสถานีออกอากาศ — และ Evervideo จะดึงและถอดรหัสสตรีมสด<br><br>

แตะ **ไฟล์ → ลิงก์ออนไลน์ → เพิ่มลิงก์** วาง URL แบบเต็ม (`rtsp://camera-ip:port/stream-path`) ระบุชื่อผู้ใช้และรหัสผ่านหากจำเป็น และแตะ**เสร็จสิ้น** สตรีม RTSP ทำงานใน Picture-in-Picture เครื่องเล่นขนาดกะทัดรัด และแคสต์ผ่าน AirPlay 2 และ Chromecast เหมือนวิดีโอปกติ
{{% /details %}}

{{% details title="ฉันโอนวิดีโอไปยัง Evervideo จากคอมพิวเตอร์อย่างไร?" closed="true" %}}
คุณสามารถเชื่อมต่อคอมพิวเตอร์หรือ NAS ส่วนตัวโดยใช้โปรโตคอล SMB, WebDAV หรือ DLNA หรือใช้ iTunes File Sharing เพื่อโอนไฟล์สื่อ<br><br>

ในการเชื่อมต่อคอมพิวเตอร์โดยใช้โปรโตคอล SMB แตะ 'ไฟล์' 'เชื่อมต่อกับที่เก็บข้อมูลบนคลาวด์' → SMB ป้อนที่อยู่ IP ของคอมพิวเตอร์และชื่อโฟลเดอร์ที่แชร์ในช่อง URL โดยใช้รูปแบบ smb://ip-address-computer/shared-folder-name ป้อนชื่อผู้ใช้และรหัสผ่านและแตะ 'เสร็จสิ้น' หากการเชื่อมต่อสำเร็จคุณจะเห็นที่เก็บข้อมูลที่เชื่อมต่อในส่วน 'ที่เก็บข้อมูลบนคลาวด์'<br><br>

สำหรับโปรโตคอล WebDAV ทุกขั้นตอนเหมือนกันยกเว้นช่อง URL URL ควรอยู่ในรูปแบบ http://server-name หรือ https://server-name หากเซิร์ฟเวอร์รองรับ SSL<br><br>

บทแนะนำฉบับสมบูรณ์มีที่นี่:<br>

Wi-Fi Drive เป็นเทคโนโลยียอดนิยมที่ช่วยให้คุณโอนไฟล์จากคอมพิวเตอร์ไปยังอุปกรณ์ iOS แบบไร้สายโดยใช้เบราว์เซอร์เดสก์ท็อป เพื่อใช้คุณสมบัตินี้ อุปกรณ์และคอมพิวเตอร์ของคุณต้องเชื่อมต่อกับเครือข่าย Wi-Fi เดียวกัน เปิด 'ไฟล์' → 'คอมพิวเตอร์' → 'เชื่อมต่อโดยใช้ Wi-Fi' และเปิดใช้งานเซิร์ฟเวอร์ หลังจากนั้นเปิดเบราว์เซอร์เดสก์ท็อปและป้อน URL จากแอปพลิเคชัน คุณสามารถลากและวางไฟล์จากคอมพิวเตอร์ไปยังหน้าเว็บที่เปิดอยู่และไฟล์จะปรากฏบนอุปกรณ์<br>

{{% details title="ฉันใช้คุณสมบัติ Wi-Fi Drive ใน Evervideo อย่างไร?" closed="true" %}}

**การโอนแบบไร้สายโดยใช้เบราว์เซอร์เดสก์ท็อป**<br>
1. เปิดแอป: เปิด Evervideo<br>
2. เชื่อมต่อผ่าน Wi-Fi: ไปที่ ไฟล์ → 'คอมพิวเตอร์' → 'เชื่อมต่อผ่าน Wi-Fi'<br>
3. (ไม่บังคับ) เพิ่มความปลอดภัย: ป้อนชื่อผู้ใช้และรหัสผ่านหากจำเป็น<br>
4. เริ่ม Wi-Fi Drive: แตะ 'เริ่ม Wi-Fi Drive' และคัดลอก URL ที่ให้มา<br>
5. เปิดเบราว์เซอร์ (Safari, Chrome, Opera, Firefox ฯลฯ) และป้อน URL<br>
6. ใช้ตัวจัดการไฟล์ในตัวเพื่ออัปโหลด ดาวน์โหลด เปลี่ยนชื่อ หรือลบไฟล์ คุณสามารถลากและวางไฟล์โดยตรงไปยัง iPhone ของคุณ<br>
7. เมื่อเสร็จสิ้น แตะ 'หยุด Wi-Fi Drive' บน iPhone ของคุณ<br><br>

หมายเหตุ: ตรวจสอบให้แน่ใจว่า JavaScript เปิดใช้งานอยู่และคุณใช้เบราว์เซอร์เวอร์ชันล่าสุดเพื่อประสิทธิภาพสูงสุด<br><br>

**โอนไฟล์โดยใช้ Mac Finder**<br>
1. ใน Finder คลิก 'ไป' → 'เชื่อมต่อกับเซิร์ฟเวอร์…'<br>
2. ป้อน URL เซิร์ฟเวอร์ที่แสดงในแอป Evervideo<br>
3. คลิก 'เชื่อมต่อ' และจัดการไฟล์บน iPhone ของคุณเหมือนไดรฟ์เครือข่ายใดก็ได้<br><br>

**โอนไฟล์โดยใช้ Windows File Explorer**<br>
1. คลิกขวาที่ 'พีซีนี้' → 'แม็พไดรฟ์เครือข่าย…'<br>
2. ป้อน URL เซิร์ฟเวอร์จากแอป Evervideo ในช่อง 'โฟลเดอร์'<br>
3. เลือกตัวอักษรไดรฟ์ คลิก 'เสร็จสิ้น' และเรียกดูไฟล์ iPhone ของคุณโดยตรง<br><br>

แอปรองรับการแปลภาษาอังกฤษ ฝรั่งเศส เยอรมัน สเปน อิตาลี โปรตุเกส รัสเซีย ยูเครน โปแลนด์ ดัตช์ อาหรับ ฮิบรู ฮินดี ญี่ปุ่น เกาหลี จีน (ตัวย่อและตัวเต็ม) เวียดนาม ตุรกี และอื่นๆ อีกมากมาย เลือก**ค่าเริ่มต้น**เพื่อตามการตั้งค่าภาษาของอุปกรณ์โดยอัตโนมัติ
{{% /details %}}

{{% details title="ฉันป้องกัน Evervideo ด้วยรหัสผ่านอย่างไร?" closed="true" %}}
**เปิดการตั้งค่า → รหัสผ่าน แตะเปิดใช้งาน และเลือกรหัส 4 หลัก — คุณจะถูกขอให้ป้อนทุกครั้งที่เปิดแอป**<br><br>

Evervideo ใช้รหัสผ่านตัวเลข 4 หลักที่ตายตัว รหัสผ่านป้องกันไม่ให้ใครที่มีสิทธิ์เข้าถึงอุปกรณ์ของคุณเปิด Evervideo และเรียกดูบัญชีคลาวด์ที่เชื่อมต่อ วิดีโอที่ดาวน์โหลด และคลัง รวมกับ iOS Face ID / Touch ID บนอุปกรณ์เพื่อการป้องกันเพิ่มเติม
{{% /details %}}

{{% details title="ฉันเปิดใช้งานวิดเจ็ต Evervideo บนหน้าจอหลักหรือหน้าจอล็อกของ iPhone อย่างไร?" closed="true" %}}
**เปิดใช้งานการอัปเดตวิดเจ็ตใน การตั้งค่า → วิดเจ็ต จากนั้นกดหน้าจอหลักหรือหน้าจอล็อกค้างไว้ แตะ '+' ค้นหา 'Evervideo' และเลือกขนาดวิดเจ็ต**<br><br>

วิดเจ็ตแสดงวิดีโอที่กำลังเล่นพร้อมชื่อ โปสเตอร์ และการควบคุมพื้นฐาน เนื่องจากการรีเฟรชวิดเจ็ตใช้พลังงานเล็กน้อย สวิตช์**เปิดใช้งานวิดเจ็ต**จึงปิดอยู่โดยค่าเริ่มต้น — เปิดใช้งานเฉพาะเมื่อคุณใช้งานวิดเจ็ตอย่างจริงจัง วิดเจ็ตทำงานบนหน้าจอหลักและหน้าจอล็อกของ iPhone และ iPad และบน macOS ใน Notification Center
{{% /details %}}

{{% details title="ฉันแชร์ Evervideo Premium กับครอบครัวอย่างไร?" closed="true" %}}
**แผน Evervideo Premium ทั้งหมด: ตลอดชีพ รายเดือน และรายปีทำงานกับ Apple Family Sharing ดังนั้นทุกคนในกลุ่มครอบครัวของคุณสามารถติดตั้ง Evervideo และใช้พรีเมียมโดยไม่มีค่าใช้จ่ายเพิ่มเติม**<br><br>

ตั้งค่า Family Sharing ใน iOS / macOS **การตั้งค่า → ครอบครัว** จากนั้นให้สมาชิกครอบครัวแต่ละคนติดตั้ง Evervideo จาก App Store และเรียกใช้ครั้งเดียวขณะลงชื่อเข้าใช้ Apple ID ของตนเอง พรีเมียมจะถูกจดจำโดยอัตโนมัติภายในหนึ่งนาที แผนเดียวกันนี้แชร์ระหว่าง iPhone, iPad และ Mac สำหรับสมาชิกครอบครัวทุกคน
{{% /details %}}

{{% details title="ฉันยกเลิกการสมัครสมาชิก Evervideo Premium อย่างไร?" closed="true" %}}
**เปิด iOS หรือ macOS การตั้งค่า → [ชื่อของคุณ] → การสมัครสมาชิก ค้นหา Evervideo และแตะยกเลิกการสมัครสมาชิก — ฟีเจอร์พรีเมียมของคุณยังคงใช้งานได้จนถึงสิ้นสุดรอบการเรียกเก็บเงินปัจจุบัน**<br><br>

การซื้อตลอดชีพในแอปไม่ใช่การสมัครสมาชิกและไม่จำเป็นต้องยกเลิก สำหรับการคืนเงิน ใช้หน้า **รายงานปัญหา** ของ Apple (`reportaproblem.apple.com`) — การคืนเงินออกโดย Apple ไม่ใช่ Everappz
{{% /details %}}

{{% details title="ฉันสำรองข้อมูลและกู้คืนคลัง Evervideo อย่างไร?" closed="true" %}}
**เปิดการตั้งค่า → สำรองข้อมูลและกู้คืน เลือกสิ่งที่จะรวม (ฐานข้อมูล ภาพปกอัลบั้ม การตั้งค่า) แตะ 'สำรองข้อมูลแอปพลิเคชัน' และบันทึกไฟล์สำรองข้อมูล — เปิดบนอุปกรณ์อื่นเพื่อกู้คืน**<br><br>

การสำรองข้อมูลประกอบด้วยรายการคลังสื่อ เพลย์ลิสต์ รายการโปรด ความคืบหน้าการดู การตั้งค่า และแคชโปสเตอร์ **ไม่รวม**ไฟล์วิดีโอที่ดาวน์โหลดออฟไลน์ (ซึ่งจะทำให้การสำรองข้อมูลมีขนาดใหญ่มาก) ย้ายไฟล์สำรองข้อมูลไปยังอุปกรณ์ใหม่ผ่าน iCloud Drive, AirDrop หรือบริการคลาวด์ที่เชื่อมต่อใดก็ได้ จากนั้นเปิดใน Evervideo เพื่อนำไปใช้
{{% /details %}}

{{% details title="ฉันจะเพิ่มพื้นที่จัดเก็บที่ Evervideo ใช้อย่างไร?" closed="true" %}}
**เปิดการตั้งค่า → ตัวจัดการไฟล์ → ลบไฟล์ชั่วคราว และการตั้งค่า → คลังสื่อ → ภาพปกอัลบั้ม → ลบทั้งหมด เพื่อล้างแคช ใช้แท็บไฟล์เพื่อลบวิดีโอที่ดาวน์โหลดที่คุณไม่ต้องการอีกต่อไป**<br><br>

คุณยังสามารถลบโฟลเดอร์ออฟไลน์แต่ละรายการใน **การตั้งค่า → ตัวจัดการไฟล์ → โฟลเดอร์ออฟไลน์ที่ซิงโครไนซ์ → '...' → ปิดโหมดออฟไลน์** ซึ่งลบสำเนาในเครื่อง วิดีโอที่สตรีมเท่านั้นไม่ใช้พื้นที่จัดเก็บอุปกรณ์เลย การล้างโฟลเดอร์**แคชเครื่องเล่น**ภายใต้ **ไฟล์ → ไฟล์ในแอปพลิเคชันนี้** ยังสามารถเพิ่มพื้นที่ว่างหลายกิกะไบต์หลังจากการเล่นเนื้อหา 4K บิตเรตสูงเป็นจำนวนมาก
{{% /details %}}

{{% details title="ทำไมวิดีโอคลาวด์ของฉันถึงบัฟเฟอร์หรือกระตุก?" closed="true" %}}
**การบัฟเฟอร์แทบทุกครั้งเกิดจากเครือข่ายช้า ขนาดไฟล์ใหญ่ หรือการตั้งค่าบัฟเฟอร์เครื่องเล่นต่ำ — เพิ่มเวลาโหลดล่วงหน้าและเปลี่ยนเป็นเครือข่ายที่เร็วกว่าหรือดาวน์โหลดไฟล์**<br><br>

เคล็ดลับปฏิบัติบางประการ:<br>

- **เพิ่มบัฟเฟอร์**ใน **การตั้งค่า → เครื่องเล่น → การโหลดไฟล์ → เวลาโหลดล่วงหน้า** สำหรับเนื้อหา 4K หรือ HEVC บิตเรตสูง<br>
- **เปลี่ยนเป็น Wi-Fi** หากคุณใช้มือถือ หรือเคลื่อนที่ใกล้เราเตอร์มากขึ้น<br>
- **เปิดใช้งานการถอดรหัสฮาร์ดแวร์** สำหรับ H.264 และ HEVC ใน **การตั้งค่า → เครื่องเล่น → วิดีโอ** เพื่อไม่ให้ CPU เป็นคอขวด<br>
- **ดาวน์โหลดไฟล์เพื่อการเล่นออฟไลน์** หากแหล่งที่มาช้ากว่าความเร็วการดูของคุณ — ไฟล์ 4K ขนาดใหญ่สามารถเกินแบนด์วิดธ์มือถือได้ง่าย<br>
- **ยืนยันบัญชีคลาวด์อีกครั้ง**ในแท็บไฟล์หากการเชื่อมต่อหมดอายุ

{{% details title="Evervideo 是什么？" closed="true" %}}
**Evervideo 是一款功能齐全的适用于 iPhone、iPad 和 Mac 的 HD 和 4K 云端视频播放器**，可将任何云存储账户、NAS 或媒体服务器变成您的个人视频库，让您完全掌控自己的文件。<br><br>

基于自定义 FFmpeg 引擎，配备硬件加速 H.264 和 HEVC 解码，Evervideo 几乎可以播放任何现代容器和编解码器（MP4、MKV、AVI、MOV、FLV、WMV、WebM、TS、M2TS 以及 FFmpeg 支持的众多格式），支持主要和辅助字幕，内置音频和视频均衡器，并支持画中画模式，让您在使用其他应用时继续观看。

{{% details title="Evervideo 安全吗？" closed="true" %}}
Evervideo 仅使用官方 SDK 和安全连接与已连接的云服务交互。您的登录名和密码不会提供给应用程序。所有从应用程序到云服务的请求均已加密。<br>
当您输入登录名和密码时，应用程序会向您显示由云服务提供商提供的官方授权页面，整个授权过程在应用程序外部完成。授权成功后，云服务提供商会向应用程序发送一个认证令牌，该令牌用于进行 API 调用。<br><br>

认证令牌是允许第三方应用程序与云存储交互的数字密钥。认证令牌存储在您设备上称为 Keychain 的安全系统存储中。您可以从已连接的云服务将文件下载到设备，这些文件将放置在应用程序的「文档」目录中。您可以随时使用内置文件管理器删除这些文件。<br>
应用程序不会共享来自已连接云账户的任何信息。您可以随时通过在网络浏览器上打开账户设置页面来撤销对您云账户的访问权限。<br><br>

要拒绝认证令牌，请在网络浏览器上登录您的账户并导航到设置页面。在那里您可以找到所有连接到您云账户的第三方应用程序，并删除任何您不想继续使用的应用程序。<br><br>

完整教程在此：<br>
[如何从您的 Google 账户断开第三方应用程序](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/)<br><br>

您还可以在应用程序中断开已连接的云账户，认证令牌也将从您的设备中删除。如果您从设备中删除应用程序，所有已下载的数据和访问令牌也将被删除。<br><br>

{{% details title="Evervideo 支持哪些视频格式？" closed="true" %}}
**借助内置的 FFmpeg 引擎以及硬件加速 H.264 / HEVC 解码，Evervideo 几乎可以在 iPhone、iPad 和 Mac 上播放每一种现代视频容器和编解码器。**<br><br>

**容器：** MP4、M4V、MKV、MOV、AVI、FLV、WMV、ASF、WebM、TS、M2TS、MTS、MPG、MPEG、OGV、3GP、3G2、F4V、RM、RMVB、VOB、DAT 等更多。<br>
**视频编解码器：** H.264 (AVC)、H.265 (HEVC)、VP9、VP8、AV1、MPEG-2、MPEG-4、MJPEG、ProRes、Theora、WMV 以及 FFmpeg 支持的几乎所有其他编解码器。<br>
**音频编解码器（视频内）：** AAC、MP3、FLAC、ALAC、OGG / Vorbis、OPUS、AC-3、EAC-3、DTS、AMR、WMA、APE。<br>
**字幕：** SRT、VTT (WebVTT)、ASS / SSA（通过 libass 渲染）、内嵌图像和文本轨道。<br>
**流媒体协议：** HTTP / HTTPS、HLS (m3u8)、RTSP（IP 摄像头和 IPTV）。<br>
**直接流式传输：** SMB / WebDAV / FTP / SFTP / NFS / DLNA。<br>
{{% /details %}}

{{% details title="Evervideo 能在 iPhone 上播放 MKV 文件吗？" closed="true" %}}
**是的——Evervideo 可以在 iPhone、iPad 和 Mac 上原生播放 MKV 文件**，无需转换，包括含有 H.264、HEVC、VP9 或 AV1 视频、多音轨以及内嵌 SRT / ASS 字幕的 MKV 容器。<br><br>

这是人们选择安装 Evervideo 而非 Apple 默认播放器的主要原因之一：iOS 根本无法打开 MKV 文件，但 Evervideo 通过内置的 FFmpeg 引擎处理它们。您可以直接从云存储或 NAS 流式传输 MKV 文件，无需下载。
{{% /details %}}

{{% details title="Evervideo 支持 4K 和 HDR 视频吗？" closed="true" %}}
**是的——Evervideo 可以在任何具备硬件解码能力的设备上播放 4K（超高清）视频和 HDR 编码内容。** 现代 iPhone、iPad 和 Apple Silicon Mac 均可以硬件方式解码 4K H.264 和 4K HEVC，播放流畅且省电。<br><br>

为在云端流式传输 4K 时获得最佳效果，请在**设置 → 播放器 → 文件加载**中增加**预加载时间**，以便缓冲区能跟上高码率文件，并连接 Wi-Fi 网络而非移动数据。
{{% /details %}}

{{% details title="Evervideo 支持 H.264 和 HEVC 硬件解码吗？" closed="true" %}}
**是的——Evervideo 在每台支持的 iPhone、iPad 和 Mac 上默认使用 H.264 (AVC) 和 H.265 (HEVC) 硬件解码器**，这意味着与纯软件解码相比，播放更流畅，电池消耗更低，设备温度更低。<br><br>

您可以在**设置 → 播放器 → 视频 → 硬件解码 H.264 / H.265** 中分别切换 H.264 和 HEVC 的硬件解码。如果某个特定文件存在兼容性问题（损坏的流、特殊配置文件），请禁用该文件的硬件解码以回退到 FFmpeg 软件解码。
{{% /details %}}

{{% details title="Evervideo 支持画中画（PiP）吗？" closed="true" %}}

将浮动窗口拖到任意角落，捏合以调整大小，单击一次可显示基本的播放/暂停/跳过控件，点击小展开按钮可返回 Evervideo。PiP 支持 **Evervideo 播放的每种视频格式**，包括云端流式传输文件和 RTSP IP 摄像头流，即使手机锁定后也继续运行。

{{% details title="Evervideo 是否有适合本地视频的 YouTube 风格播放器界面？" closed="true" %}}
**是的——Evervideo 的默认布局围绕 YouTube 风格的体验为您的个人视频打造：屏幕顶部始终显示一个小型视频播放器，而您可以在下方浏览库。**<br><br>

**小型播放器**在每个标签页上保持显示——最近使用、收藏夹、媒体库、播放列表、文件和设置——这样您就可以浏览、搜索、整理和排队下一个视频，而不会中断播放。点击小型播放器将其展开为全屏视图；向下滑动可将其收起为小型模式，而不会停止视频。在 macOS 上，小型播放器可以分离为单独的**始终置顶浮动窗口**，实现真正的画中画式桌面多任务处理。<br><br>

与 YouTube 不同，这里没有广告、没有算法推荐、没有烦人的自动播放、没有跟踪——您完全掌控来自 iCloud、Google Drive、Dropbox、NAS、Plex / Jellyfin / Emby 或任何其他来源的个人视频库。

{{% details title="Evervideo 能播放来自 Insta360 和其他 360 摄像机的 360° 视频吗？" closed="true" %}}
**是的——Evervideo 可直接播放 360° / VR（球形）视频，无需预处理或转换**，包括来自 Insta360（One X、X3、X4、ONE RS、GO 3）、GoPro Max、Ricoh Theta、Samsung Gear 360、Vuze 以及任何其他等矩形 360° 来源的素材。<br><br>

只需将 360° 视频放入 iCloud Drive、Google Drive、Dropbox、您的 NAS，或从 iOS Photos 库播放——Evervideo 会识别球形格式并自动切换到 VR 视口渲染模式。之后您可以：<br>

- **旋转设备**——物理倾斜和旋转 iPhone 或 iPad，视频视口会实时旋转，环顾四周。<br>

- **捏合缩放**球形视图。<br>

这意味着您的 Insta360 素材可以直接在 iPhone 和 iPad 上使用——无需额外应用、无需转码、无需云订阅。

{{% details title="如何使用视频均衡器和预设？" closed="true" %}}
**打开播放器，点击「更多操作」→ 视频均衡器，然后拖动亮度、对比度、饱和度和色调滑块，或选择预设。**<br><br>

Evervideo 的**视频均衡器**是在 FFmpeg 渲染管线内运行的实时画面调整工具：<br>

- **亮度**——提亮弱光拍摄的暗场景，或降低过曝。<br>
- **对比度**——为褪色内容增加质感，或柔化过度对比的来源。<br>
- **饱和度**——增强暗淡色彩，或降低饱和度以获得电影感。<br>
- **色调**——调整色彩平衡以修正绿色或洋红偏色。<br><br>

您可以将喜欢的设置保存为**自定义预设**，以后对任何视频一键应用。预设还可以**导出和导入**，在 iPhone、iPad 和 Mac 之间共享，或进行备份。将视频均衡器与**音频均衡器**（10 频段 EQ，带独立预设库）结合使用，全面掌控画面和声音。
{{% /details %}}

{{% details title="如何旋转视频或更改缩放模式？" closed="true" %}}
**点击播放器上的「更多操作」，选择旋转（0° / 90° / 180° / 270°）或缩放模式（适合 / 填充 / 拉伸 / 原始）。**<br><br>

**旋转**适用于横向或颠倒拍摄的视频——无需离开播放器即可旋转画面。**缩放模式**控制画面如何填充屏幕：<br>

- **适合**——保持原始宽高比，在需要的地方显示黑边。<br>
- **填充**——填充整个屏幕，必要时裁剪视频。<br>
- **拉伸**——拉伸以填充屏幕，使画面变形。<br>
- **原始**——以 1:1 保持原始分辨率。
{{% /details %}}

{{% details title="如何在视频中选择不同的音轨（配音、解说）？" closed="true" %}}

对于含有多个音轨的视频（不同语言配音、导演解说、原声/现场版本），Evervideo 会显示每个内嵌音轨及其语言和编解码器信息。适用于 MKV、MP4、M2TS 以及任何其他包含多个音频流的容器。
{{% /details %}}

{{% details title="如何在 Evervideo 中添加或更改字幕？" closed="true" %}}
**点击播放器上的「更多操作」（「...」）按钮，选择字幕——然后选择内嵌字幕轨道、加载外部字幕文件或更改字体。**<br><br>

Evervideo 会自动列出视频中内嵌的每个字幕轨道。要加载外部字幕文件，请选择**外部文件**，然后从设备、iCloud Drive 或任何已连接的云服务中选择 `.srt`、`.vtt`、`.ass` 或 `.ssa` 文件。您还可以在**设置 → 播放器 → 字幕**中配置默认字幕行为。
{{% /details %}}

{{% details title="Evervideo 支持外部 SRT、VTT 和 ASS 字幕文件吗？" closed="true" %}}
**是的——Evervideo 可以从设备上的任何位置或任何已连接的云服务加载 SRT、VTT (WebVTT)、ASS 和 SSA 格式的外部字幕文件。**<br><br>

含有高级样式（自定义字体、颜色、位置、卡拉OK效果）的 ASS / SSA 文件可通过内置的 **libass** 库正确渲染——非常适合番剧字幕组、专业字幕文件和演示文稿。您还可以在**设置 → 播放器 → 字幕 → 字体**中为字幕指定特定字体。
{{% /details %}}

{{% details title="如何从 Evervideo 删除视频而不删除云存储中的文件？" closed="true" %}}
**在任意视频上，点击「...」→ 从媒体库删除——这将从您的库数据库中删除该条目，但保留云存储、NAS 或 iOS Photos 库中的原始文件不变。**<br><br>

如果您还想从源头删除文件，请选择**从云服务删除**或**从本地文件删除**。这些破坏性操作无法撤销，因此在选中多个视频时请谨慎操作。<br><br>

要删除下载的副本而不影响云端任何内容，请打开**文件**标签，在**此应用中的文件**或**离线文件夹**下找到该视频，并使用**「...」→ 删除**。云端原始文件保持原样不变——Evervideo 只删除本地副本。
{{% /details %}}

{{% details title="如何在 Evervideo 中更改播放速度？" closed="true" %}}
**打开播放器，点击工具栏上的速度控件，拖动滑块——支持 0.25× 到 3.00× 的速度。**<br><br>

您可以降速进行逐帧分析（0.25× / 0.5×），或加速观看教程和讲座（1.25× / 1.5× / 2×）。点击速度屏幕右上角的配置图标可切换到精确模式，进行更精细的调整。每轨音调校正让音频在非 1× 速度下听起来自然。
{{% /details %}}

{{% details title="如何在 Evervideo 中设置睡眠定时器？" closed="true" %}}
**打开设置 → 播放器 → 睡眠定时器，开启后选择播放持续时间，到时自动停止。**<br><br>

您还可以通过**设置 → 播放器 → 个性化 → 主屏幕操作**将**睡眠定时器**按钮直接添加到主播放器屏幕。点击配置图标可进入精确模式，以分钟为单位精确设置——非常适合边看剧边入睡。
{{% /details %}}

{{% details title="如何在视频中为特定位置添加书签？" closed="true" %}}

书签按视频保存，在会话之间持久存在，非常适合长视频、讲座、视频有声书、教程系列以及演唱会录像等您想跳回特定时刻的场景。
{{% /details %}}

Evervideo 使用固定的 4 位数字密码。密码可防止任何能访问您设备的人打开 Evervideo 并浏览已连接的云账户、已下载的视频和库。结合设备上的 iOS Face ID / Touch ID 以获得额外保护。

{{% details title="如何在 iPhone 主屏幕或锁屏上启用 Evervideo 小组件？" closed="true" %}}
**在设置 → 小组件中启用小组件更新，然后长按主屏幕或锁屏，点击「+」，搜索「Evervideo」并选择小组件大小。**<br><br>

小组件显示当前播放的视频，带有标题、海报和基本控件。由于小组件刷新会消耗少量电量，**启用小组件**开关默认关闭——仅在您积极使用小组件时才开启。小组件可在 iPhone 和 iPad 主屏幕和锁屏上使用，以及在 macOS 通知中心使用。
{{% /details %}}

{{% details title="如何与家人共享 Evervideo Premium？" closed="true" %}}
**所有 Evervideo Premium 计划（买断、月付和年付）均支持 Apple Family Sharing，因此您家庭群组中的任何人都可以安装 Evervideo 并免费使用 Premium。**<br><br>

在 iOS / macOS 的**设置 → 家庭**中设置 Family Sharing，然后让每位家庭成员从 App Store 安装 Evervideo，并在使用自己的 Apple ID 登录时运行一次。Premium 会在一分钟内自动被识别。每位家庭成员的 iPhone、iPad 和 Mac 共享同一计划。

{{% details title="如何备份和恢复 Evervideo 库？" closed="true" %}}
**打开设置 → 备份与恢复，选择要包含的内容（数据库、专辑封面、设置），点击「备份应用数据」并保存备份文件——在另一台设备上打开以恢复。**<br><br>

备份包含您的媒体库条目、播放列表、收藏夹、观看进度、设置和海报缓存。它**不**包含离线下载的视频文件（那会使备份文件非常大）。通过 iCloud Drive、AirDrop 或任何已连接的云服务将备份文件移至新设备，然后在 Evervideo 中打开以应用。
{{% /details %}}

{{% details title="如何释放 Evervideo 占用的存储空间？" closed="true" %}}
**打开设置 → 文件管理器 → 删除临时文件，以及设置 → 媒体库 → 专辑封面 → 全部删除，以清除缓存；使用文件标签页删除不再需要的已下载视频。**<br><br>

您还可以在**设置 → 文件管理器 → 同步离线文件夹 → 「...」→ 禁用离线模式**中删除单个离线文件夹，这将删除本地副本。仅流式传输的视频根本不占用设备存储空间。清除**文件 → 此应用中的文件**下的**播放器缓存**文件夹，在大量播放高码率 4K 内容后也可释放几个 GB 的空间。
{{% /details %}}

{{% details title="为什么我的云端视频出现缓冲或卡顿？" closed="true" %}}
**缓冲几乎总是由网速慢、文件体积大或播放器缓冲设置过低引起的——增加预加载时间，切换到更快的网络，或下载文件。**<br><br>

几个实用建议：<br>

- 在**设置 → 播放器 → 文件加载 → 预加载时间**中为高码率 4K 或 HEVC 内容**增加缓冲区**。<br>
- 如果您正在使用移动数据，请**切换到 Wi-Fi**，或靠近路由器。<br>
- 在**设置 → 播放器 → 视频**中为 H.264 和 HEVC **启用硬件解码**，以避免 CPU 成为瓶颈。<br>
- 如果来源比您的观看速度慢，请**下载文件以供离线播放**——大型 4K 文件很容易超过移动数据带宽。<br>
- 如果连接已过期，请在文件标签页中**重新授权您的云账户**。

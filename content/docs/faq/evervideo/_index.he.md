Evervideo הוא **נגן וידאו ענן** HD ו-4K עבור **iPhone, iPad ו-Mac** המזרים ומוריד סרטונים מ-iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP וכל מקור WebDAV, SMB, NFS, FTP / SFTP, DLNA או S3 — בנוסף לשרתי מדיה מאורחים עצמאיים כמו **Plex, Jellyfin, Emby, Subsonic** ו-**Navidrome**, וזרמי **RTSP** חיים ממצלמות IP. בנוי על **מנוע נגן מבוסס FFmpeg מותאם** עם **פענוח חומרה מואץ H.264 / HEVC**, רצועות כתוביות ראשוניות ומשניות (SRT / VTT / ASS / libass), אקולייזר וידאו עם הגדרות קבועות מראש, אקולייזר אודיו, **Picture-in-Picture**, **השמעה 360° / VR** לצילומי Insta360 ו-GoPro Max, הזרמת **AirPlay 2** ו-**Chromecast**, ונגן קומפקטי תמידי על המסך. שאלות נפוצות אלו עונות על השאלות שמשתמשים שולחים לרוב. לסיורים מעמיקים יותר, ראה את [מדריך המשתמש של Evervideo](/docs/guide/evervideo/).

{{% details title="מהו Evervideo?" closed="true" %}}
**Evervideo הוא נגן וידאו ענן HD ו-4K מלא עבור iPhone, iPad ו-Mac** שהופך כל חשבון אחסון ענן, NAS או שרת מדיה לספריית הווידאו האישית שלך עם שליטה מלאה בקבצים שלך.<br><br>

בנוי על מנוע נגן מבוסס FFmpeg מותאם עם פענוח חומרה מואץ H.264 ו-HEVC, Evervideo מנגן כמעט כל מכל ואינקודק מודרניים (MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS ופורמטי FFmpeg רבים), תומך בכתוביות ראשוניות ומשניות, כולל אקולייזר אודיו ווידאו, ועובד במצב Picture-in-Picture כדי שתוכל להמשיך לצפות תוך שימוש באפליקציות אחרות.

{{% details title="אילו פורמטי וידאו Evervideo תומך?" closed="true" %}}
**Evervideo מנגן כמעט כל מכל ואינקודק וידאו מודרניים על iPhone, iPad ו-Mac** הודות למנוע FFmpeg המובנה בשילוב עם פענוח חומרה מואץ H.264 / HEVC.<br><br>

**מכלים:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT ועוד רבים.<br>
**קודקי וידאו:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV, וכמעט כל קודק אחר ש-FFmpeg תומך בו.<br>
**קודקי אודיו (בתוך וידאו):** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE.<br>
**כתוביות:** SRT, VTT (WebVTT), ASS / SSA (מעובד באמצעות libass), רצועות תמונה וטקסט מוטמעות.<br>
**פרוטוקולי הזרמה:** HTTP / HTTPS, HLS (m3u8), RTSP (מצלמות IP ו-IPTV).<br>
**הזרמה ישירה:** SMB / WebDAV / FTP / SFTP / NFS / DLNA.<br>

זה אחד הסיבות העיקריות לכך שאנשים מתקינים את Evervideo במקום הנגן המוגדר כברירת מחדל של Apple: iOS לא פותח MKV בכלל, אבל Evervideo מטפל בהם דרך מנוע FFmpeg המובנה. תוכל להזרים קבצי MKV ישירות מאחסון ענן או NAS ללא הורדה.

לתוצאות הטובות ביותר על 4K מוזרם מהענן, הגדל את **זמן הטעינה מראש** ב**הגדרות → נגן → טעינת קבצים** כדי שהמאגר יוכל לעמוד בקצב קבצים עם קצב העברת נתונים גבוה, וחבר לרשת Wi-Fi במקום לרשת סלולרית.

{{% details title="האם Evervideo תומך בפענוח חומרה H.264 ו-HEVC?" closed="true" %}}

תוכל להחליף פענוח חומרה באופן עצמאי עבור H.264 ו-HEVC ב**הגדרות → נגן → וידאו → פענוח חומרה H.264 / H.265**. אם לקובץ ספציפי יש בעיות תאימות (זרמים פגומים, פרופילים אקזוטיים), השבת פענוח חומרה עבור קובץ זה כדי לחזור לפענוח תוכנה FFmpeg.

הווידג'ט מציג את הווידאו המנגן כרגע עם כותרת, פוסטר ופקדים בסיסיים. מכיוון שרענוני ווידג'טים משתמשים בכמות קטנה של אנרגיה, כפתור **הפעל ווידג'טים** כבוי כברירת מחדל — הפעל אותו רק אם אתה משתמש באופן פעיל בווידג'טים. ווידג'טים עובדים על מסך הבית ומסך הנעילה של iPhone ו-iPad, ועל macOS במרכז ההתראות.

{{% details title="כיצד לשתף את Evervideo Premium עם המשפחה שלי?" closed="true" %}}

{{% details title="כיצד לגבות ולשחזר את ספריית Evervideo שלי?" closed="true" %}}
**פתח הגדרות → גיבוי ושחזור, בחר מה לכלול (מסד נתונים, עטיפות אלבומים, הגדרות), הקש "גבה נתוני אפליקציה" ושמור את קובץ הגיבוי — פתח אותו במכשיר אחר לשחזור.**<br><br>

הגיבוי מכיל את רשומות ספריית המדיה, רשימות ההשמעה, המועדפים, התקדמות הצפייה, ההגדרות ומטמון הפוסטרים. הוא **אינו** כולל קבצי וידאו שהורדו לא מקוון (שהיו הופכים את הגיבוי לעצום). העבר את קובץ הגיבוי למכשיר החדש דרך iCloud Drive, AirDrop או כל שירות ענן מחובר, ואז פתח אותו ב-Evervideo להחלה.

{{% details title="כיצד לפנות שטח אחסון שבשימוש Evervideo?" closed="true" %}}
**פתח הגדרות → מנהל קבצים → מחק קבצים זמניים והגדרות → ספריית מדיה → עטיפות אלבומים → מחק הכל לניקוי מטמונים; השתמש בלשונית קבצים למחיקת סרטונים שהורדת שאינך זקוק להם עוד.**<br><br>

תוכל גם להסיר תיקיות לא מקוונות בודדות ב**הגדרות → מנהל קבצים → תיקיות לא מקוונות מסונכרנות → עוד פעולות → השבת מצב לא מקוון**, שמוחק את העותקים המקומיים. סרטוני הזרמה בלבד אינם משתמשים בכלל באחסון המכשיר. ניקוי תיקיית **מטמון הנגן** תחת **קבצים → קבצים באפליקציה זו** יכול גם לפנות מספר ג'יגה-בייטים לאחר השמעה כבדה של תוכן 4K בקצב סיביות גבוה.

{{% details title="מדוע הווידאו בענן שלי מאגר או נתקע?" closed="true" %}}

כמה טיפים מעשיים:<br>

- **הגדל את המאגר** ב**הגדרות → נגן → טעינת קבצים → זמן טעינה מראש** לתוכן 4K או HEVC בקצב סיביות גבוה.<br>

- **הורד את הקובץ להשמעה לא מקוונת** אם המקור איטי יותר ממהירות הצפייה שלך — קבצי 4K גדולים יכולים בקלות לחרוג מרוחב פס סלולרי.<br>
- **אשר מחדש את חשבון הענן שלך** בלשונית קבצים אם חיבור פג.

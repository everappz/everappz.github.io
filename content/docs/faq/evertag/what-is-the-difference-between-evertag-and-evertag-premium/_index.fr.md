Evertag et Evertag Premium sont deux versions de la même puissante application d'édition de tags. Tandis qu'Evertag Free vous donne accès aux outils essentiels d'édition de métadonnées, Evertag Premium débloque l'expérience complète — sans publicité, illimitée et personnalisable.

## Choisissez votre forfait Premium

La version gratuite de l'application propose un achat unique à vie et deux options d'abonnement (1 mois et 1 an) pour supprimer toutes les restrictions et passer à la version Premium, vous permettant de choisir le prix le plus adapté et optimal pour vous. Les prix peuvent varier selon votre pays ou territoire. Veuillez également garder à l'esprit que **Family Sharing** est **activé** pour tous les achats et forfaits, vous pouvez donc partager la version Premium avec les membres de votre famille.

## Partage des achats entre iOS et Mac

 Les achats à vie et les abonnements sont partagés entre iOS et Mac, en utilisant iCloud pour synchroniser ces informations. Si vous avez la version Premium sur votre appareil iOS, assurez-vous d'avoir la dernière version installée et qu'iCloud est activé. Démarrez l'application sur iOS et attendez une minute pour que vos informations d'achat soient téléchargées sur iCloud.

Vous pouvez également essayer d'appuyer sur le bouton "Restaurer les achats" dans les Paramètres de l'application. Ensuite, installez la dernière version de l'application depuis l'App Store sur votre Mac et démarrez l'application. Assurez-vous d'avoir une connexion Internet et d'utiliser le même compte iCloud et App Store sur votre Mac que celui utilisé sur votre appareil iOS. Attendez une minute que l'application télécharge les informations d'achat depuis iCloud. La version Premium devrait s'activer automatiquement sur votre Mac.

## Restaurer les achats

Pour restaurer votre achat sur le nouvel appareil, utilisez simplement le menu "Restaurer les achats". Vous verrez la liste de vos achats. Si vous ne voyez pas tous vos achats, vérifiez si l'appareil est connecté au même compte iTunes utilisé pour effectuer les achats, et assurez-vous qu'iCloud est activé.

Une fois que vous avez mis à niveau votre application, vous verrez l'écran de statut Premium avec les détails de vos achats actuels.

## Evertag Free

- Contient des publicités
- Modifier plus de 120 tags audio
- Modifier les pochettes d'album
- Édition par lots de plusieurs fichiers
- Corriger les problèmes d'encodage de caractères
- Connecter 1 service de stockage cloud
- Ajouter jusqu'à 10 fichiers favoris
- 20 requêtes de recherche automatique de tags/jour
- 20 recherches de pochettes d'album/jour
- Personnalisation de base uniquement

## Evertag Premium

- Sans publicités
- Édition de tags illimitée
- Modifications de pochettes d'album illimitées
- Édition par lots de plusieurs fichiers
- Corriger les problèmes d'encodage
- Connexions de stockage cloud illimitées
- Favoris illimités
- Recherches de tags illimitées
- Recherches de pochettes d'album illimitées
- Personnalisation complète de l'interface, y compris l'icône et le thème

| Expérience d'édition sans publicité                                          | Non          | Oui             |
| Modifier les tags audio standard (titre, artiste, album, année, genre)       | Oui          | Oui             |
| Ajouter, remplacer ou supprimer la pochette d'album intégrée                 | Oui          | Oui             |
| Modifier plusieurs fichiers à la fois en mode par lots                       | Oui          | Oui             |
| Corriger les jeux de caractères de tags cassés ou mal encodés                | Oui          | Oui             |
| Nombre de services de stockage cloud que vous pouvez connecter               | 1            | Illimité        |

| Recherches automatiques quotidiennes de tags audio (MusicBrainz)             | 20           | Illimité        |
| Recherches automatiques quotidiennes de pochettes d'album                    | 20           | Illimité        |
| Personnalisation complète (icône d'application personnalisée, thèmes, mise en page) | Non   | Oui             |


## Essayez Premium gratuitement

Vous pouvez passer à la version Premium gratuitement mais pour une durée limitée uniquement en utilisant le menu "Essayer Premium gratuitement". Regardez simplement une publicité ou parlez de cette application à vos amis pour obtenir la version Premium gratuitement.

## Que choisir ?

Evertag Free est idéal pour une utilisation légère ou pour essayer les fonctionnalités de l'application. Mais si vous gérez de grandes collections musicales, souhaitez une intégration cloud illimitée et préférez une expérience sans publicité, Evertag Premium est le meilleur choix.

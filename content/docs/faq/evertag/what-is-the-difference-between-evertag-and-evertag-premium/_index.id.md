Evertag dan Evertag Premium adalah dua versi dari aplikasi pengeditan tag yang sama dan canggih. Sementara Evertag Free memberi Anda akses ke alat pengeditan metadata penting, Evertag Premium membuka pengalaman penuh — bebas iklan, tanpa batas, dan dapat dikustomisasi.

## Pilih Paket Premium Anda

Versi gratis aplikasi ini menawarkan pembelian dalam aplikasi seumur hidup satu kali dan dua opsi berlangganan (1 bulan dan 1 tahun) untuk menghapus semua pembatasan dan meningkatkan ke versi Premium, memungkinkan Anda memilih harga terbaik dan paling optimal. Harga mungkin berbeda-beda tergantung negara atau wilayah Anda. Selain itu, harap diingat bahwa **Family Sharing** **diaktifkan** untuk semua pembelian dan paket, sehingga Anda dapat berbagi versi Premium dengan anggota keluarga Anda.

## Berbagi Pembelian Antara iOS dan Mac

Pembelian seumur hidup dan langganan dibagikan antara iOS dan Mac, menggunakan iCloud untuk menyinkronkan informasi ini. Jika Anda memiliki versi Premium di perangkat iOS Anda, pastikan Anda telah menginstal versi terbaru dan iCloud diaktifkan. Mulai aplikasi di iOS dan tunggu satu menit agar informasi pembelian Anda diunggah ke iCloud.

Anda juga dapat mencoba menekan tombol "Pulihkan Pembelian" di Pengaturan aplikasi. Setelah itu, instal versi aplikasi terbaru dari App Store di Mac Anda dan mulai aplikasi. Pastikan Anda memiliki koneksi internet dan menggunakan akun iCloud dan App Store yang sama di Mac Anda yang Anda gunakan di perangkat iOS Anda. Tunggu satu menit agar aplikasi mengunduh informasi pembelian dari iCloud. Versi Premium seharusnya aktif di Mac Anda secara otomatis.

## Pulihkan Pembelian

Untuk memulihkan pembelian Anda di perangkat baru, cukup gunakan menu "Pulihkan pembelian". Anda akan melihat daftar pembelian Anda. Jika Anda tidak melihat semua pembelian, periksa apakah perangkat terhubung ke akun iTunes yang sama yang digunakan untuk melakukan pembelian, dan pastikan iCloud diaktifkan.

Setelah Anda meningkatkan aplikasi, Anda akan melihat layar status Premium dengan detail pembelian Anda saat ini.

## Evertag Free

- Mengandung iklan
- Edit lebih dari 120 tag audio
- Edit sampul album
- Edit massal beberapa file
- Perbaiki masalah encoding karakter
- Hubungkan 1 layanan penyimpanan cloud
- Tambahkan hingga 10 file favorit
- 20 permintaan pencarian tag otomatis/hari
- 20 pencarian sampul album/hari
- Hanya personalisasi dasar

## Evertag Premium

- Tanpa iklan
- Pengeditan tag tanpa batas
- Edit sampul album tanpa batas
- Edit massal beberapa file
- Perbaiki masalah encoding
- Koneksi penyimpanan cloud tanpa batas
- Favorit tanpa batas
- Pencarian tag tanpa batas
- Pencarian sampul album tanpa batas
- Personalisasi UI penuh, termasuk ikon dan tema

| Deskripsi Fitur                                                      | Evertag Free | Evertag Premium |
|----------------------------------------------------------------------|--------------|-----------------|
| Pengalaman pengeditan bebas iklan                                    | Tidak        | Ya              |
| Edit tag audio standar (judul, artis, album, tahun, genre)           | Ya           | Ya              |
| Tambah, ganti, atau hapus sampul album yang disematkan               | Ya           | Ya              |
| Edit beberapa file sekaligus dalam mode massal                       | Ya           | Ya              |
| Perbaiki set karakter tag yang rusak atau salah encoded              | Ya           | Ya              |
| Jumlah layanan penyimpanan cloud yang dapat Anda hubungkan           | 1            | Tanpa Batas     |
| File yang dapat Anda tandai sebagai favorit                          | 10           | Tanpa Batas     |
| Pencarian tag audio otomatis harian (MusicBrainz)                    | 20           | Tanpa Batas     |
| Pencarian sampul album otomatis harian                               | 20           | Tanpa Batas     |
| Personalisasi penuh (ikon aplikasi kustom, tema, tata letak)         | Tidak        | Ya              |


## Coba Premium secara gratis

Anda dapat meningkatkan ke versi Premium secara gratis tetapi hanya untuk waktu terbatas menggunakan menu "Coba premium secara gratis". Cukup tonton iklan atau beritahu teman-teman Anda tentang aplikasi ini untuk mendapatkan versi Premium secara gratis.

## Apa yang Harus Dipilih?

Evertag Free sangat bagus untuk penggunaan ringan atau mencoba fitur-fitur aplikasi. Tetapi jika Anda mengelola koleksi musik besar, menginginkan integrasi cloud tanpa batas, dan lebih menyukai pengalaman bebas iklan, Evertag Premium adalah pilihan terbaik.

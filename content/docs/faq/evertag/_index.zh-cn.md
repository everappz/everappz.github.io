Evertag 是一款适用于 **iPhone、iPad 和 Mac** 的音乐元数据编辑器，可让您修复音频标签、替换专辑封面、编辑歌词并整理音乐库——本地、在 iCloud Drive、Google Drive / Dropbox / OneDrive / MEGA / pCloud 上，或通过 SMB / WebDAV 在个人 NAS 上。此常见问题解答用户最常提出的问题。寻找更深入的操作说明？请从[用户指南](/docs/guide/evertag/)或[标签字段映射](/docs/guide/evertag/evertag-tag-field-mappings/)参考开始。

{{% details title="Evertag 是什么？" closed="true" %}}
Evertag 是一款适用于 **iOS 和 macOS** 的音乐元数据编辑器和专辑封面管理器，可让您修复标签、添加封面艺术并整理存储在设备或云端的音频文件。<br><br>

该应用支持多种流行音频格式——MP3、FLAC、WAV、M4A、AIFF、OGG、OPUS、WMA、APE 等——并可编辑常见标签，如 **TITLE**、**ARTIST**、**ALBUM**、**GENRE**、**YEAR**、**TRACK NUMBER**，以及扩展字段，如 **BPM**、**DISC NUMBER**、**LYRICS**、**MUSICBRAINZ IDs**、**REPLAY-GAIN** 值和 **PARENTAL ADVISORY** 评级。您可以一次处理一个文件，或切换到**批量模式**同时编辑多个曲目——非常适合整理完整专辑或播放列表。<br><br>

Evertag 的一个突出功能是能够从互联网自动获取缺失的专辑封面（通过 MusicBrainz），或让您从照片库中添加自己的封面。内置歌词编辑器支持普通和**同步（LRC）**歌词，并包含跳转到 Lrclib、Genius、Lyricsify 和 Google 搜索的一键快捷方式。该应用支持**原位编辑**，可在不复制文件的情况下修改音频标签。<br><br>

无论您是在设备还是云端（iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Synology、pCloud 等）管理音乐，Evertag 都能提供无缝的文件访问和编辑体验——非常适合希望在 iPhone、iPad 或 Mac 上拥有整洁、有序音乐库的音乐人、DJ、播客制作者和收藏家，无需使用桌面标签编辑器。请参阅完整的[用户指南](/docs/guide/evertag/)了解截图和教程。<br><br>

{{% details title="Evertag 安全吗？" closed="true" %}}
是的。Evertag 仅使用**官方云服务商 SDK 和加密连接**来访问您的账户，您的密码永远不会与应用共享。当您连接云服务时，将显示云服务商提供的官方授权页面——整个登录流程在 Evertag 外部进行，服务商返回应用用于 API 调用的授权令牌。<br><br>

授权令牌存储在设备上的安全系统 **Keychain** 中，从不发送到我们的服务器。您从连接的云服务下载的文件放置在应用的「文档」目录中，您可以随时通过内置文件管理器删除它们。<br><br>

Evertag 不共享您连接的云账户中的任何信息。您可以随时撤销访问权限：<br>
• 在 Evertag 中——打开云账户，点击 **…** 并选择**断开连接**。授权令牌将从您的设备中删除。<br>
• 在云服务商的网站上——登录，转到第三方应用/已连接应用页面，从列表中删除 Evertag。<br><br>

如果您从设备中删除 Evertag，所有下载的数据和访问令牌都会随之删除。<br><br>
{{% /details %}}

{{% details title="Evertag 会收集或分享我的个人数据吗？" closed="true" %}}

Evertag 使用最少量的分析和崩溃报告服务（Firebase Analytics 和 Crashlytics），并在免费版本中通过 AdMob 展示广告。您可以在**设置 → 分析和数据收集**中查看和关闭这些服务。Apple 的应用追踪透明度提示还允许您在应用首次启动时拒绝追踪。请参阅我们的[隐私政策](/legal/privacy-policy/)了解完整说明。<br><br>

{{% details title="Evertag 能以 ID3v2.3 或 ID3v2.4 格式写入标签吗？" closed="true" %}}
可以。打开**设置 → 音频标签编辑器 → 标签保存选项**，打开或关闭 **ID3v2.4**。<br><br>

• **ID3v2.4 开启**（默认）——具有完整 UTF-8 支持、多值帧和更丰富元数据的现代标准。非常适合当前的桌面播放器、音乐管理器和大多数现代车载音响。<br>
• **ID3v2.4 关闭**——以更广泛支持的 **ID3v2.3** 格式保存标签。如果较旧的设备、车载主机或 DJ 软件在编辑后显示空白标签，请使用此选项。<br><br>

同一设置页面还显示**重复标签**选项，该选项将常见元数据写入文件的两个 ID3 部分，以实现与旧播放器的最大兼容性。<br><br>
{{% /details %}}

{{% details title="Evertag 能修复音频标签中乱码或损坏的字符（西里尔文、中文、日文等）吗？" closed="true" %}}
可以。Evertag 包含内置的**编码规范化器**，可将错误编码的文本（例如，显示为乱码如 `Ðàñêîëüíèêîâ` 的西里尔文、中文、日文或韩文标签）转换回可读的 UTF-8。<br><br>

使用方法：<br>
1. 在标签编辑器中打开文件。<br>
2. 点击封面下方的**规范化编码**按钮。<br>
3. 保存文件。<br><br>

如果您看不到该按钮，请在**设置 → 音频标签编辑器 → 主界面上的按钮**中启用它。编码修复对于最初在 Windows 上标记的音乐库特别有用，因为那些标签读取器默认使用 ANSI / CP-1251 / Shift-JIS 而不是 UTF-8。<br><br>

{{% details title="我可以在不下载整个音乐库的情况下编辑 Google Drive、Dropbox 或 iCloud Drive 中的音频标签吗？" closed="true" %}}
可以。连接云账户后，Evertag 仅传输您即将编辑的文件——无需在本地镜像整个音乐库。<br><br>

当您点击远程文件时：<br>
1. Evertag 将其下载到临时文件夹。<br>
2. 您像往常一样编辑标签。<br>
3. 点击**保存**后，更新后的文件重新上传到云中的同一位置。<br>
4. 临时副本被删除（或保留，取决于**设置 → 音频标签编辑器 → 编辑在线文件**）。<br><br>

您可以在**设置 → 音频标签编辑器 → 更新在线文件**中控制保存行为：在每次保存前询问、始终自动上传或完全跳过上传（仅本地编辑）。<br><br>

{{% details title="我可以将 Evertag 与 Synology、QNAP、Asustor 或 Apple Time Capsule 等 NAS 配合使用吗？" closed="true" %}}
可以。使用 **SMB** 或 **WebDAV** 连接任何 NAS——这是每个主要 NAS 厂商（Synology、QNAP、Asustor、TerraMaster、Western Digital、Buffalo、Apple Time Capsule）开箱即用支持的两种协议。<br><br>

• **SMB**——点击**连接到云存储 → SMB**，输入 `smb://nas-ip-address/shared-folder-name`，选择协议版本（Auto / SMB1 / SMB2），输入登录名 + 密码，点击**完成**。完整教程：[通过 SMB 从 Mac 或 PC 传输音乐](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)。<br>
• **WebDAV**——相同流程，但输入 `http://server-name` 或 `https://server-name` 用于 SSL。完整教程：[通过 WebDAV 连接 NAS 存储](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)。<br>
• **Synology Drive**——作为一级集成支持（与通用 WebDAV 分开）。完整教程：[连接 Synology NAS 并在 iPhone 或 Mac 上收听音乐](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/)。<br><br>

在本地网络上广播的设备也会自动出现在连接屏幕的**可用设备**部分。<br><br>

{{% details title="如何在编辑后自动更新云服务上的文件元数据？" closed="true" %}}
云文件的保存行为在**设置 → 音频标签编辑器 → 更新在线文件**中控制，提供三个选项：<br><br>

• **显示确认消息**（默认）——在将编辑后的文件上传回云之前，应用会提示您确认。<br>
• **自动更新文件元数据**——在您点击标签编辑器中的**保存**后，应用立即静默重新上传文件。<br>
• **不更新文件元数据**——应用完全跳过上传；您的更改保留在本地，云副本保持不变。<br><br>

将此与**设置 → 音频标签编辑器 → 编辑在线文件**配合使用，以决定每次编辑后是否删除本地下载的副本（节省空间）或保留（更快重新打开）。<br><br>
{{% /details %}}

{{% details title="为什么编辑云文件后我的更改没有显示？" closed="true" %}}
首先，检查您的**更新在线文件**首选项是否未设置为**不更新文件元数据**——该模式将更改保留在本地，永远不会重新上传到云端。打开**设置 → 音频标签编辑器 → 更新在线文件**，选择**显示确认消息**或**自动更新文件元数据**。<br><br>

其他需要验证的事项：<br>
• 您对云文件夹有**写入权限**。某些共享/只读 WebDAV / SMB 挂载会静默接受上传但将其丢弃。<br>
• 云服务商是否悄悄将您登出——打开账户并从**连接 → … → 设置**重新验证。<br>
• 云服务的桌面或网络客户端可能仍在同步旧版本。等待一分钟并刷新。<br>
• 如果您使用 **iCloud Drive**，文件可能仍在从另一台设备上传。在 iOS 文件应用中打开文件，确认同步指示器不再旋转。<br><br>

仍然卡住？使用**设置 → 发送反馈**，附上文件格式、云服务和简短描述。<br><br>
{{% /details %}}

{{% details title="如何从云存储下载文件？" closed="true" %}}
您首先需要至少一个连接的云账户——打开**连接**屏幕并添加服务商。<br><br>

下载文件或文件夹：<br>
1. 打开连接的云存储服务。<br>
2. 导航到您想下载的文件夹。<br>
3. 点击右上角的**更多操作**（`…`）按钮，选择**选择**。<br>
4. 勾选您需要的文件或文件夹，然后点击**下载**。<br><br>

下载的文件位于**本地文件 → 下载**文件夹中，并出现在**传输队列**中——从本地文件屏幕左上角的旋转箭头图标打开它。您可以在**设置 → 文件管理器 → 将下载的文件保存到**中更改默认目标，并从同一屏幕切换后台传输。<br><br>
{{% /details %}}

{{% details title="如何管理网络存储中的文件？" closed="true" %}}
打开云文件夹，点击右上角的 **…** 按钮，选择**选择**进入选择模式。每个文件和文件夹旁边会出现复选框，让您可以选择一个或多个项目。<br><br>

可用操作：<br>
• **编辑音频标签**——在[标签编辑器](/docs/guide/evertag/evertag-guide-tag-editor/)中打开一个或多个文件（多选时使用批量模式）。<br>
• **下载**——将文件或文件夹复制到**本地文件**部分。<br>
• **移动**——将项目移到同一云账户内的其他文件夹。<br>
• **重命名**——在远程服务器上重命名文件或文件夹。<br>
• **删除**——从云中永久删除文件或文件夹（不可逆）。<br>
• **排序**——按名称、大小或编辑日期重新排序。<br>
• **网格/列表**——在表格和缩略图视图之间切换。<br><br>

如果没有足够空间显示每个操作，工具栏会显示**更多操作**溢出按钮——点击它查看完整列表。<br><br>

{{% details title="如何编辑存储在 iPhone 或 iPad 本地的音乐文件的元数据？" closed="true" %}}
打开**本地文件**并滚动到**此 iPhone 上的文件**（或 iPad / Mac）部分。<br><br>

• 选择**打开文件…**以选择设备上任意位置的一个或多个音频文件，或**打开文件夹…**以选择整个文件夹。<br>
• 点击**打开**确认——文件立即添加到编辑器，**就地编辑**，永远不会复制到 Evertag。<br><br>

**快速访问设备文件夹。** 通过将文件夹添加到该部分来跳过每次的选取器：<br>
1. 打开**本地文件**。<br>
2. 滚动到**此设备上的文件**，点击**连接文件夹**。<br>
3. 选择音乐文件夹并点击**打开**。<br><br>

文件夹以读/写访问权限出现在**此 iPhone 上的文件**列表中——非常适合存储在系统文件应用或另一个应用的暴露文件夹下的音乐库。要在以后取消链接，点击文件夹旁边的 **…** 并选择**断开连接**。详细教程：[在 iPhone 或 Mac 上播放本地音乐](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac)。<br><br>

{{% details title="如何从电脑传输音乐到 Evertag？" closed="true" %}}
您可以使用 **SMB**、**WebDAV**、**Wi-Fi Drive** 或 **Finder 文件共享**（旧版 macOS 上的 iTunes 文件共享）连接电脑或 NAS。<br><br>

**SMB**——点击**连接到云存储 → SMB**，输入 `smb://computer-ip-address/shared-folder-name`，设置凭据，点击**完成**。完整教程：[使用 SMB 协议从电脑向 iPhone 传输文件](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)。<br><br>

**WebDAV**——使用 `http://server-name`（或 `https://` 用于 SSL）的相同流程。完整教程：[通过 WebDAV 连接 NAS 存储](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)。<br><br>

**Wi-Fi Drive**——完全无线。打开**连接 → 电脑 → 通过 Wi-Fi 连接**并启动服务器，然后在电脑浏览器中打开显示的 URL 并**拖放**文件到页面。两台设备需要共享同一 Wi-Fi 网络。完整教程：[使用 Wi-Fi Drive 无线传输文件](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)。<br><br>

**Finder 文件共享**（或旧版 macOS / Windows 上的 iTunes 文件共享）——用数据线将 iPhone/iPad 连接到电脑，打开 **Finder → 您的设备 → 文件 → Evertag**，将文件拖到共享文件夹。完整教程：[在 iPhone 上播放本地 iTunes 文件](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)。<br><br>
{{% /details %}}

{{% details title="我可以在没有互联网连接的情况下离线编辑音频标签吗？" closed="true" %}}
可以。编辑本地文件完全支持离线操作——标签编辑器、批量模式、专辑封面替换、编码修复和歌词编辑器不需要网络连接。<br><br>

需要互联网的功能均有明确标注：MusicBrainz 自动搜索、手动 MusicBrainz 标签搜索、专辑封面网络搜索和歌词搜索快捷方式（Lrclib、Genius、Lyricsify、Google）。如果您在会话中断网，这些服务只是被禁用——您的本地编辑不受影响。<br><br>
{{% /details %}}

{{% details title="如何一次编辑多个文件（批量模式）？" closed="true" %}}
使用**批量模式**一次将相同的元数据更改应用于多个曲目。有两种启动方式：<br><br>

**从文件管理器**<br>
1. 打开包含文件的文件夹。<br>
2. 点击**更多操作**（`…`）按钮 → **选择**。<br>
3. 勾选要编辑的文件。<br>
4. 点击**编辑音频标签**。<br><br>

**从标签编辑器**<br>
1. 从文件夹中打开任意文件。<br>
2. 滚动到标签编辑器底部。<br>
3. 点击**同时编辑文件**——同一文件夹中的所有音频文件一起加载。<br><br>

进行一次更改并点击**保存**——新值将写入每个选定的文件。非常适合修复整张专辑的专辑名称、专辑艺术家、流派、年份或曲目总数。<br><br>
{{% /details %}}

{{% details title="如何一次更改多首歌曲的专辑封面？" closed="true" %}}
激活**批量模式**（见上文），点击封面轮播上的**相机图标**并选择新图像——相同的封面艺术将在一次保存中写入每个选定的文件。<br><br>

您可以从以下来源获取新封面：<br>
• **照片库**——适合您已保存或扫描的封面。<br>
• **本地文件/下载**——设备上已有的任何图像。<br>
• **连接的云账户**——将 Evertag 指向您云端的 JPG/PNG。<br><br>

在**设置 → 音频标签编辑器 → 专辑封面缩放**中控制保存的图像质量——小/中/大/超大，或「已停用」（Premium）以保持原始尺寸。<br><br>
{{% /details %}}

{{% details title="如何自动查找缺失的专辑封面？" closed="true" %}}
点击标签编辑器中封面下方的**搜索专辑封面**按钮。Evertag 使用曲目当前的艺术家和专辑元数据在线查找匹配的封面艺术。<br><br>

当搜索结果出现时，长按您喜欢的图像，在系统菜单中选择**保存到照片**。返回 Evertag，点击封面上的**相机图标**，选择**照片库**并选择保存的图像——封面将应用到文件。保存专辑封面按钮（在同一菜单下）还可以让您将当前封面备份到文档文件夹以便日后重用。<br><br>

如果搜索专辑封面按钮不可见，请在**设置 → 音频标签编辑器 → 主界面上的按钮**中启用它。<br><br>
{{% /details %}}

{{% details title="Evertag 使用 MusicBrainz 吗？" closed="true" %}}
是的。Evertag 的自动和手动标签搜索由 **MusicBrainz** 数据库提供支持——这是全球最大的开放音乐元数据项目之一，拥有超过 5000 万首曲目。<br><br>

标签编辑器底部工具栏提供两种流程：<br>
• **自动搜索音频标签**——使用现有的艺术家/专辑/标题元数据进行最佳匹配查找。<br>
• **手动搜索音频标签**——选择专辑、歌曲和要应用的确切字段集。<br><br>

您还可以直接存储 MusicBrainz 标识符：Album ID、Album Artist ID、Artist ID、Disc ID、Release Group ID、Release Track ID、Track ID、Work ID 等（请参阅[标签字段映射](/docs/guide/evertag/evertag-tag-field-mappings/)参考）。这些 ID 让其他支持 MusicBrainz 的应用（Picard、MusicBrainz Browser、Roon、Plex）能够可靠地交叉引用您的曲目。<br><br>
{{% /details %}}

{{% details title="如何在 Evertag 中为歌曲添加歌词？" closed="true" %}}
歌词位于**扩展标签编辑器**中。快速路径如下：<br><br>

1. 点击文件以打开标签编辑器。<br>
2. 点击**显示扩展标签**以启用完整字段集。<br>
3. 滚动到 **Lyrics unsynced** 部分，点击该字段。<br>
4. 点击**添加新页面**——每个歌词页面都有自己的**语言**和**描述**，因此单个曲目可以保存多个翻译。<br>
5. 在**文本**字段中，粘贴普通歌词或 **LRC 格式（带时间戳）的歌词**——占位符显示示例。<br>
6. 在每个屏幕上点击**完成**，然后点击**保存**将歌词写入音频文件。<br><br>

**不想手动输入？** 歌词编辑器有**一键搜索快捷方式**，直达最流行的歌词数据库——预填了当前曲目的艺术家和标题：<br>
• **Lrclib**——**定时（LRC）歌词**的首选来源。<br>
• **Genius**——拥有准确纯文本歌词的大型目录。<br>
• **Lyricsify**——包含纯文本和同步歌词的社区数据库。<br>
• **Google**——作为备用的通用网络搜索。<br><br>

每个按钮仅在对应服务可访问时出现。完整演示：[如何在 iPhone 或 Mac 上编辑音频文件的歌词](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/)。<br><br>

**同步（LRC）歌词。** 编辑器接受纯文本**和** LRC 格式歌词（`[mm:ss.ms]` 行时间戳）——将 LRC 文本粘贴到字段中并保存，音频文件的歌词标签将原样存储，随时可供任何支持 LRC 的播放器使用。内置的 **Lrclib** 和 **Lyricsify** 快捷方式默认返回 LRC 字符串，因此您可以在不离开应用的情况下在几秒内建立完整的同步歌词库。<br><br>

{{% details title="如何卸载 Evertag 并删除所有数据？" closed="true" %}}
从 iOS、iPadOS 或 macOS 中删除该应用将删除 Evertag 创建的**所有**文件，包括已下载的歌曲、缓存的缩略图、专辑封面、设置、密码和连接的云账户的授权令牌。<br><br>

如果您希望保留应用但不卸载地释放空间：<br>
• **设置 → 文件管理器 → 文件缩略图 → 全部删除**——删除缓存的缩略图。<br>
• **设置 → 文件管理器 → 删除临时文件**——清除缓存文件夹。<br>
• 打开**本地文件 → 下载**，删除不再需要的单个文件或文件夹。<br>
• 从**连接**屏幕断开云账户连接以删除其授权令牌。<br><br>

卸载 Evertag 不会删除云端文件——它们仍保留在云服务商的服务器上，可以通过任何网络客户端访问。<br><br>

Evertag 是一款適用於 **iPhone、iPad 和 Mac** 的音樂元資料編輯器，可讓您修復音訊標籤、替換專輯封面、編輯歌詞並整理音樂庫——本機、在 iCloud Drive、Google Drive / Dropbox / OneDrive / MEGA / pCloud 上，或透過 SMB / WebDAV 在個人 NAS 上。此常見問題解答使用者最常提出的問題。尋找更深入的操作說明？請從[使用者指南](/docs/guide/evertag/)或[標籤欄位對應](/docs/guide/evertag/evertag-tag-field-mappings/)參考開始。

{{% details title="Evertag 是什麼？" closed="true" %}}
Evertag 是一款適用於 **iOS 和 macOS** 的音樂元資料編輯器和專輯封面管理器，可讓您修復標籤、新增封面藝術並整理儲存在裝置或雲端的音訊檔案。<br><br>

該應用程式支援多種熱門音訊格式——MP3、FLAC、WAV、M4A、AIFF、OGG、OPUS、WMA、APE 等——並可編輯常見標籤，如 **TITLE**、**ARTIST**、**ALBUM**、**GENRE**、**YEAR**、**TRACK NUMBER**，以及延伸欄位，如 **BPM**、**DISC NUMBER**、**LYRICS**、**MUSICBRAINZ IDs**、**REPLAY-GAIN** 值和 **PARENTAL ADVISORY** 評級。您可以一次處理一個檔案，或切換到**批次模式**同時編輯多個曲目——非常適合整理完整專輯或播放清單。<br><br>

Evertag 的一個突出功能是能夠從網際網路自動取得缺失的專輯封面（透過 MusicBrainz），或讓您從照片圖庫中新增自己的封面。內建歌詞編輯器支援普通和**同步（LRC）**歌詞，並包含跳至 Lrclib、Genius、Lyricsify 和 Google 搜尋的一鍵快速鍵。該應用程式支援**原位編輯**，可在不複製檔案的情況下修改音訊標籤。<br><br>

無論您是在裝置還是雲端（iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Synology、pCloud 等）管理音樂，Evertag 都能提供無縫的檔案存取和編輯體驗——非常適合希望在 iPhone、iPad 或 Mac 上擁有整潔、有序音樂庫的音樂人、DJ、播客製作者和收藏家，無需使用桌面標籤編輯器。請參閱完整的[使用者指南](/docs/guide/evertag/)了解截圖和教學。<br><br>

{{% details title="如何在新 iPhone、iPad 或 Mac 上恢復 Premium 購買？" closed="true" %}}
登入您用於原始購買的**相同 Apple ID**，然後在 Evertag 中點選**設定 → 購買 → 恢復購買**。<br><br>

如果恢復未恢復您的 Premium 狀態：<br>

• 檢查裝置是否有作用中的網路連線，以及 **iOS 設定 → Apple ID → 媒體與購買項目**中的 App Store 帳號是否與購買升級的帳號相符。<br><br>

訂閱和終身購買在 iOS 和 Mac（Mac Catalyst）之間共享，**Family Sharing** 允許最多五名家庭成員重複使用同一 Premium 方案。<br><br>

{{% details title="Evertag 安全嗎？" closed="true" %}}
是的。Evertag 僅使用**官方雲端服務商 SDK 和加密連線**來存取您的帳號，您的密碼永遠不會與應用程式共享。當您連線雲端服務時，將顯示雲端服務商提供的官方授權頁面——整個登入流程在 Evertag 外部進行，服務商回傳應用程式用於 API 呼叫的授權令牌。<br><br>

授權令牌儲存在裝置上的安全系統 **Keychain** 中，從不傳送至我們的伺服器。您從連線的雲端服務下載的檔案放置在應用程式的「文件」目錄中，您可以隨時透過內建檔案管理器刪除它們。<br><br>

Evertag 不共享您連線的雲端帳號中的任何資訊。您可以隨時撤銷存取權限：<br>
• 在 Evertag 中——開啟雲端帳號，點選 **…** 並選擇**斷開連接**。授權令牌將從您的裝置中刪除。<br>
• 在雲端服務商的網站上——登入，前往第三方應用程式/已連線應用程式頁面，從清單中移除 Evertag。<br><br>

如果您從裝置中刪除 Evertag，所有已下載的資料和存取令牌都會隨之刪除。<br><br>
{{% /details %}}

{{% details title="Evertag 會收集或分享我的個人資料嗎？" closed="true" %}}
該應用程式不會在我們的伺服器上儲存有關您音樂庫的任何資訊；您編輯的所有內容都保留在您的裝置上或您自己的雲端帳號中。<br><br>

Evertag 使用最少量的分析和當機回報服務（Firebase Analytics 和 Crashlytics），並在免費版本中透過 AdMob 展示廣告。您可以在**設定 → 分析和資料收集**中檢視和關閉這些服務。Apple 的應用程式追蹤透明度提示還允許您在應用程式首次啟動時拒絕追蹤。請參閱我們的[隱私權政策](/legal/privacy-policy/)了解完整說明。<br><br>
{{% /details %}}

{{% details title="我可以用密碼或 Face ID 保護 Evertag 嗎？" closed="true" %}}

密碼儲存在 iOS Keychain 本機——沒有線上恢復功能。如果不再需要保護，可從同一設定畫面停用它。<br><br>

{{% details title="Evertag 能以 ID3v2.3 或 ID3v2.4 格式寫入標籤嗎？" closed="true" %}}
可以。開啟**設定 → 音訊標籤編輯器 → 標籤儲存選項**，開啟或關閉 **ID3v2.4**。<br><br>

• **ID3v2.4 開啟**（預設）——具有完整 UTF-8 支援、多值框架和更豐富元資料的現代標準。非常適合當前的桌面播放器、音樂管理器和大多數現代車載音響。<br>
• **ID3v2.4 關閉**——以更廣泛支援的 **ID3v2.3** 格式儲存標籤。如果較舊的裝置、車載主機或 DJ 軟體在編輯後顯示空白標籤，請使用此選項。<br><br>

同一設定頁面還顯示**重複標籤**選項，該選項將常見元資料寫入檔案的兩個 ID3 部分，以實現與舊播放器的最大相容性。<br><br>
{{% /details %}}

{{% details title="Evertag 能修復音訊標籤中亂碼或損毀的字元（西里爾文、中文、日文等）嗎？" closed="true" %}}
可以。Evertag 包含內建的**編碼正規化器**，可將錯誤編碼的文字（例如，顯示為亂碼如 `Ðàñêîëüíèêîâ` 的西里爾文、中文、日文或韓文標籤）轉換回可讀的 UTF-8。<br><br>

使用方法：<br>
1. 在標籤編輯器中開啟檔案。<br>
2. 點選封面下方的**正規化編碼**按鈕。<br>
3. 儲存檔案。<br><br>

如果您看不到該按鈕，請在**設定 → 音訊標籤編輯器 → 主畫面上的按鈕**中啟用它。編碼修復對於最初在 Windows 上標記的音樂庫特別有用，因為那些標籤讀取器預設使用 ANSI / CP-1251 / Shift-JIS 而不是 UTF-8。<br><br>

開啟**連接**標籤 → 點選**連線到雲端儲存** → 從清單中選擇服務商（iCloud Drive、Google Drive、Dropbox、OneDrive、Box、MEGA、Yandex.Disk、pCloud、Synology Drive、MediaFire、WD My Cloud Home、InfiniCLOUD、HiDrive、OpenDrive、MyDrive、Put.io、Cloud Mail.ru 或百度網盤）→ 在服務商官方頁面登入 → 點選**完成**。<br><br>

{{% details title="我可以在不下載整個音樂庫的情況下編輯 Google Drive、Dropbox 或 iCloud Drive 中的音訊標籤嗎？" closed="true" %}}
可以。連線雲端帳號後，Evertag 僅傳輸您即將編輯的檔案——無需在本機鏡像整個音樂庫。<br><br>

當您點選遠端檔案時：<br>
1. Evertag 將其下載到暫存資料夾。<br>
2. 您像往常一樣編輯標籤。<br>
3. 點選**儲存**後，更新後的檔案重新上傳到雲端中的相同位置。<br>
4. 暫存副本被刪除（或保留，取決於**設定 → 音訊標籤編輯器 → 編輯線上檔案**）。<br><br>

您可以在**設定 → 音訊標籤編輯器 → 更新線上檔案**中控制儲存行為：在每次儲存前詢問、始終自動上傳或完全略過上傳（僅本機編輯）。<br><br>
{{% /details %}}

{{% details title="我可以將 Evertag 與 Synology、QNAP、Asustor 或 Apple Time Capsule 等 NAS 搭配使用嗎？" closed="true" %}}
可以。使用 **SMB** 或 **WebDAV** 連線任何 NAS——這是每個主要 NAS 廠商（Synology、QNAP、Asustor、TerraMaster、Western Digital、Buffalo、Apple Time Capsule）開箱即用支援的兩種通訊協定。<br><br>

• **SMB**——點選**連線到雲端儲存 → SMB**，輸入 `smb://nas-ip-address/shared-folder-name`，選擇通訊協定版本（Auto / SMB1 / SMB2），輸入登入名稱 + 密碼，點選**完成**。完整教學：[透過 SMB 從 Mac 或 PC 串流音樂](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)。<br>
• **WebDAV**——相同流程，但輸入 `http://server-name` 或 `https://server-name` 用於 SSL。完整教學：[透過 WebDAV 連線 NAS 儲存](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)。<br>
• **Synology Drive**——作為一級整合支援（與通用 WebDAV 分開）。完整教學：[連線 Synology NAS 並在 iPhone 或 Mac 上收聽音樂](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/)。<br><br>

在本機網路上廣播的裝置也會自動出現在連接畫面的**可用設備**部分。<br><br>
{{% /details %}}

{{% details title="如何在編輯後自動更新雲端服務上的檔案元資料？" closed="true" %}}
雲端檔案的儲存行為在**設定 → 音訊標籤編輯器 → 更新線上檔案**中控制，提供三個選項：<br><br>

• **顯示確認訊息**（預設）——在將編輯後的檔案上傳回雲端之前，應用程式會提示您確認。<br>
• **自動更新檔案元資料**——在您點選標籤編輯器中的**儲存**後，應用程式立即靜默重新上傳檔案。<br>
• **不更新檔案元資料**——應用程式完全略過上傳；您的變更保留在本機，雲端副本保持不變。<br><br>

將此與**設定 → 音訊標籤編輯器 → 編輯線上檔案**搭配使用，以決定每次編輯後是否刪除本機下載的副本（節省空間）或保留（更快重新開啟）。<br><br>
{{% /details %}}

{{% details title="為什麼編輯雲端檔案後我的變更沒有顯示？" closed="true" %}}
首先，檢查您的**更新線上檔案**偏好設定是否未設定為**不更新檔案元資料**——該模式將變更保留在本機，永遠不會重新上傳到雲端。開啟**設定 → 音訊標籤編輯器 → 更新線上檔案**，選擇**顯示確認訊息**或**自動更新檔案元資料**。<br><br>

其他需要驗證的事項：<br>
• 您對雲端資料夾有**寫入權限**。某些共用/唯讀 WebDAV / SMB 掛載會靜默接受上傳但將其捨棄。<br>
• 雲端服務商是否悄悄將您登出——開啟帳號並從**連接 → … → 設定**重新驗證。<br>
• 雲端服務的桌面或網路用戶端可能仍在同步舊版本。等待一分鐘並重新整理。<br>
• 如果您使用 **iCloud Drive**，檔案可能仍在從另一台裝置上傳。在 iOS 檔案應用程式中開啟檔案，確認同步指示器不再旋轉。<br><br>

仍然卡住？使用**設定 → 傳送意見回饋**，附上檔案格式、雲端服務和簡短描述。<br><br>
{{% /details %}}

{{% details title="如何從雲端儲存下載檔案？" closed="true" %}}

4. 勾選您需要的檔案或資料夾，然後點選**下載**。<br><br>

下載的檔案位於**本機檔案 → 下載**資料夾中，並出現在**傳輸佇列**中——從本機檔案畫面左上角的旋轉箭頭圖示開啟它。您可以在**設定 → 檔案管理器 → 將下載的檔案儲存至**中更改預設目的地，並從同一畫面切換背景傳輸。<br><br>
{{% /details %}}

{{% details title="如何管理網路儲存中的檔案？" closed="true" %}}
開啟雲端資料夾，點選右上角的 **…** 按鈕，選擇**選擇**進入選擇模式。每個檔案和資料夾旁邊會出現核取方塊，讓您可以選擇一個或多個項目。<br><br>

可用操作：<br>
• **編輯音訊標籤**——在[標籤編輯器](/docs/guide/evertag/evertag-guide-tag-editor/)中開啟一個或多個檔案（多選時使用批次模式）。<br>
• **下載**——將檔案或資料夾複製到**本機檔案**部分。<br>
• **移動**——將項目移到同一雲端帳號內的其他資料夾。<br>
• **重新命名**——在遠端伺服器上重新命名檔案或資料夾。<br>
• **刪除**——從雲端永久刪除檔案或資料夾（不可逆）。<br>
• **排序**——按名稱、大小或編輯日期重新排序。<br>
• **格狀/清單**——在表格和縮圖檢視之間切換。<br><br>

如果沒有足夠空間顯示每個操作，工具列會顯示**更多操作**溢位按鈕——點選它查看完整清單。<br><br>

{{% details title="如何編輯儲存在 iPhone 或 iPad 本機的音樂檔案的元資料？" closed="true" %}}
開啟**本機檔案**並捲動到**此 iPhone 上的檔案**（或 iPad / Mac）部分。<br><br>

• 選擇**開啟檔案…**以選擇裝置上任意位置的一個或多個音訊檔案，或**開啟資料夾…**以選擇整個資料夾。<br>

**快速存取裝置資料夾。** 透過將資料夾新增到該部分來略過每次的選取器：<br>

2. 捲動到**此裝置上的檔案**，點選**連接資料夾**。<br>

資料夾以讀/寫存取權限出現在**此 iPhone 上的檔案**清單中——非常適合儲存在系統檔案應用程式或另一個應用程式的公開資料夾下的音樂庫。要在以後取消連結，點選資料夾旁邊的 **…** 並選擇**斷開連接**。詳細教學：[在 iPhone 或 Mac 上播放本機音樂](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac)。<br><br>

{{% details title="如何從電腦傳輸音樂到 Evertag？" closed="true" %}}
您可以使用 **SMB**、**WebDAV**、**Wi-Fi Drive** 或 **Finder 檔案共享**（舊版 macOS 上的 iTunes 檔案共享）連線電腦或 NAS。<br><br>

**SMB**——點選**連線到雲端儲存 → SMB**，輸入 `smb://computer-ip-address/shared-folder-name`，設定憑證，點選**完成**。完整教學：[使用 SMB 通訊協定從電腦向 iPhone 傳輸檔案](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)。<br><br>

**WebDAV**——使用 `http://server-name`（或 `https://` 用於 SSL）的相同流程。完整教學：[透過 WebDAV 連線 NAS 儲存](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)。<br><br>

**Wi-Fi Drive**——完全無線。開啟**連接 → 電腦 → 透過 Wi-Fi 連線**並啟動伺服器，然後在電腦瀏覽器中開啟顯示的 URL 並**拖放**檔案到頁面。兩台裝置需要共用同一 Wi-Fi 網路。完整教學：[使用 Wi-Fi Drive 無線傳輸檔案](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)。<br><br>

**Finder 檔案共享**（或舊版 macOS / Windows 上的 iTunes 檔案共享）——用傳輸線將 iPhone/iPad 連線到電腦，開啟 **Finder → 您的裝置 → 檔案 → Evertag** 並將檔案拖到共用資料夾。完整教學：[在 iPhone 上播放本機 iTunes 檔案](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)。<br><br>
{{% /details %}}

{{% details title="我可以在沒有網際網路連線的情況下離線編輯音訊標籤嗎？" closed="true" %}}
可以。編輯本機檔案完全支援離線操作——標籤編輯器、批次模式、專輯封面替換、編碼修復和歌詞編輯器不需要網路連線。<br><br>

需要網際網路的功能均有明確標注：MusicBrainz 自動搜尋、手動 MusicBrainz 標籤搜尋、專輯封面網路搜尋和歌詞搜尋快速鍵（Lrclib、Genius、Lyricsify、Google）。如果您在工作階段中斷網，這些服務只是被停用——您的本機編輯不受影響。<br><br>

{{% details title="如何一次編輯多個檔案（批次模式）？" closed="true" %}}
使用**批次模式**一次將相同的元資料變更套用於多個曲目。有兩種啟動方式：<br><br>

**從檔案管理器**<br>
1. 開啟包含檔案的資料夾。<br>
2. 點選**更多操作**（`…`）按鈕 → **選擇**。<br>
3. 勾選要編輯的檔案。<br>
4. 點選**編輯音訊標籤**。<br><br>

**從標籤編輯器**<br>
1. 從資料夾中開啟任意檔案。<br>
2. 捲動到標籤編輯器底部。<br>
3. 點選**同時編輯檔案**——同一資料夾中的所有音訊檔案一起載入。<br><br>

進行一次變更並點選**儲存**——新值將寫入每個選定的檔案。非常適合修復整張專輯的專輯名稱、專輯演出者、類型、年份或曲目總數。<br><br>

{{% details title="如何一次更改多首歌曲的專輯封面？" closed="true" %}}
啟動**批次模式**（見上文），點選封面輪播上的**相機圖示**並選擇新圖像——相同的封面藝術將在一次儲存中寫入每個選定的檔案。<br><br>

您可以從以下來源取得新封面：<br>
• **照片圖庫**——適合您已儲存或掃描的封面。<br>
• **本機檔案/下載**——裝置上已有的任何圖像。<br>
• **連線的雲端帳號**——將 Evertag 指向您雲端的 JPG/PNG。<br><br>

在**設定 → 音訊標籤編輯器 → 專輯封面縮放**中控制儲存的圖像品質——小/中/大/超大，或「已停用」（Premium）以保持原始尺寸。<br><br>

{{% details title="如何自動尋找缺失的專輯封面？" closed="true" %}}
點選標籤編輯器中封面下方的**搜尋專輯封面**按鈕。Evertag 使用曲目當前的演出者和專輯元資料在線上尋找相符的封面藝術。<br><br>

當搜尋結果出現時，長按您喜歡的圖像，在系統選單中選擇**儲存到照片**。返回 Evertag，點選封面上的**相機圖示**，選擇**照片圖庫**並選擇儲存的圖像——封面將套用到檔案。儲存專輯封面按鈕（在同一選單下）還可以讓您將當前封面備份到文件資料夾以便日後重用。<br><br>

如果搜尋專輯封面按鈕不可見，請在**設定 → 音訊標籤編輯器 → 主畫面上的按鈕**中啟用它。<br><br>

標籤編輯器底部工具列提供兩種流程：<br>

{{% details title="如何在 Evertag 中為歌曲新增歌詞？" closed="true" %}}
歌詞位於**延伸標籤編輯器**中。快速路徑如下：<br><br>

1. 點選檔案以開啟標籤編輯器。<br>
2. 點選**顯示延伸標籤**以啟用完整欄位集。<br>
3. 捲動到 **Lyrics unsynced** 部分，點選該欄位。<br>
4. 點選**新增頁面**——每個歌詞頁面都有自己的**語言**和**描述**，因此單個曲目可以儲存多個翻譯。<br>
5. 在**文字**欄位中，貼上普通歌詞或 **LRC 格式（含時間戳記）的歌詞**——預留位置顯示範例。<br>
6. 在每個畫面上點選**完成**，然後點選**儲存**將歌詞寫入音訊檔案。<br><br>

**不想手動輸入？** 歌詞編輯器有**一鍵搜尋快速鍵**，直達最熱門的歌詞資料庫——預填了當前曲目的演出者和標題：<br>
• **Lrclib**——**定時（LRC）歌詞**的首選來源。<br>
• **Genius**——擁有準確純文字歌詞的大型目錄。<br>
• **Lyricsify**——包含純文字和同步歌詞的社群資料庫。<br>
• **Google**——作為備用的通用網路搜尋。<br><br>

每個按鈕僅在對應服務可存取時出現。完整示範：[如何在 iPhone 或 Mac 上編輯音訊檔案的歌詞](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/)。<br><br>

**同步（LRC）歌詞。** 編輯器接受純文字**和** LRC 格式歌詞（`[mm:ss.ms]` 行時間戳記）——將 LRC 文字貼到欄位中並儲存，音訊檔案的歌詞標籤將原樣儲存，隨時可供任何支援 LRC 的播放器使用。內建的 **Lrclib** 和 **Lyricsify** 快速鍵預設返回 LRC 字串，因此您可以在不離開應用程式的情況下在幾秒內建立完整的同步歌詞庫。<br><br>

{{% details title="如何解除安裝 Evertag 並刪除所有資料？" closed="true" %}}

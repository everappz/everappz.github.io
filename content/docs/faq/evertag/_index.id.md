Evertag adalah editor metadata musik untuk **iPhone, iPad, dan Mac** yang memungkinkan Anda memperbaiki tag audio, mengganti sampul album, mengedit lirik, dan merapikan perpustakaan Anda — secara lokal, di iCloud Drive, di Google Drive / Dropbox / OneDrive / MEGA / pCloud, atau di NAS pribadi melalui SMB / WebDAV. {{% details title="Apa itu Evertag?" closed="true" %}}
Evertag adalah editor metadata musik dan manajer sampul album untuk **iOS dan macOS** yang memungkinkan Anda memperbaiki tag, menambahkan sampul, dan merapikan file audio yang tersimpan di perangkat Anda atau di cloud.<br><br>

Aplikasi ini mendukung berbagai format audio populer — MP3, FLAC, WAV, M4A, AIFF, OGG, OPUS, WMA, APE dan banyak lainnya — dan memungkinkan Anda mengedit tag umum seperti **TITLE**, **ARTIST**, **ALBUM**, **GENRE**, **YEAR**, **TRACK NUMBER**, ditambah bidang yang diperluas seperti **BPM**, **DISC NUMBER**, **LYRICS**, **MUSICBRAINZ IDs**, nilai **REPLAY-GAIN**, dan rating **PARENTAL ADVISORY**. Anda dapat bekerja dengan satu file sekaligus atau beralih ke **mode massal** untuk mengedit beberapa trek secara bersamaan — sempurna untuk mengorganisir album atau daftar putar penuh.<br><br>

Salah satu fitur unggulan Evertag adalah kemampuannya mengambil sampul album yang hilang dari internet (melalui MusicBrainz) atau memungkinkan Anda menambahkan milik sendiri dari perpustakaan Foto. Editor lirik bawaan mendukung lirik biasa dan **tersinkron (LRC)** dan menyertakan pintasan satu ketuk ke Lrclib, Genius, Lyricsify, dan pencarian Google. Aplikasi ini mendukung pengeditan **open-in-place**, sehingga Anda dapat memodifikasi tag audio tanpa menyalin file.<br><br>

Baik Anda mengelola musik di perangkat Anda atau di cloud (iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Synology, pCloud, dan banyak lagi), Evertag menawarkan akses dan pengeditan file yang mulus — ideal untuk musisi, DJ, podcaster, dan kolektor yang menginginkan perpustakaan yang bersih dan terorganisir dengan baik di iPhone, iPad, atau Mac tanpa menggunakan editor tag desktop. Lihat [panduan pengguna](/docs/guide/evertag/) lengkap untuk tangkapan layar dan tutorial.<br><br>

{{% details title="Apakah Evertag bekerja di iPhone, iPad, dan Mac?" closed="true" %}}
Ya — Evertag adalah **aplikasi universal**. Pembelian yang sama mencakup iPhone, iPad, dan Mac (Intel dan Apple Silicon, melalui Mac Catalyst), dan pengaturan, favorit, serta status Premium Anda tersinkron antar perangkat melalui iCloud.<br><br>

Tata letak menyesuaikan dengan perangkat: tab bar bawah di iPhone, tampilan terbagi dengan bilah samping di iPad dan Mac, dan dukungan pintasan keyboard penuh di Mac. Semua fitur editor tag — termasuk mode massal, pencarian lirik, bidang yang diperluas, dan integrasi cloud — tersedia di setiap platform.<br><br>
{{% /details %}}

{{% details title="Apakah Evertag gratis?" closed="true" %}}
Ya, Evertag **gratis diunduh** dengan pembelian dalam aplikasi opsional untuk menghapus batasan. Aplikasi ini menawarkan pembelian **seumur hidup satu kali** dan dua opsi berlangganan (bulanan dan tahunan) sehingga Anda dapat memilih paket yang paling sesuai. Harga mungkin berbeda-beda berdasarkan negara atau wilayah, dan **Family Sharing diaktifkan** untuk setiap paket — bagikan Premium dengan hingga lima anggota keluarga.<br><br>

Pembelian seumur hidup dan langganan dibagikan antara iOS dan Mac melalui iCloud. Jika Anda sudah membayar di iOS, instal versi aplikasi terbaru, pastikan iCloud diaktifkan, dan tunggu sekitar satu menit agar pembelian Anda diunggah — lalu mulai aplikasi di Mac (Apple ID + iCloud yang sama) dan Premium akan aktif secara otomatis. Anda juga dapat mengetuk **Pengaturan → Pembelian → Pulihkan pembelian** kapan saja.<br><br>

{{% details title="Apa perbedaan antara Evertag Free dan Evertag Premium?" closed="true" %}}

**Evertag Free** memberi Anda akses ke fitur pengeditan tag inti dengan beberapa batasan. Ini menyertakan iklan dan memungkinkan Anda menggunakan editor tag, editor sampul album, dan mode massal dengan cara yang terbatas: 1 akun cloud terhubung, hingga 10 favorit, hingga 20 pencarian tag otomatis per hari, dan hingga 20 pencarian sampul album per hari. Anda masih dapat mengedit lebih dari 120 tag audio dan mengelola file secara lokal atau melalui manajer file bawaan.<br><br>

**Evertag Premium (Bulanan, Tahunan, atau Seumur Hidup)** menghapus iklan dan membuka segalanya: bidang tag 120+ tanpa batas, pengeditan massal tanpa batas, perbaikan encoding, favorit tanpa batas, **akun cloud tanpa batas**, pencarian tag otomatis tanpa batas, pencarian sampul album tanpa batas, dan personalisasi penuh (ikon aplikasi alternatif, tema, sampul album ukuran asli). Premium ideal untuk kolektor musik, DJ, podcaster, atau siapa pun yang mengelola perpustakaan besar di seluruh perangkat dan platform cloud.<br><br>

**Opsi Harga**<br>
• Premium Bulanan — $1,99/bulan<br>
• Premium Tahunan — $12,99/tahun<br>
• Premium Seumur Hidup — $24,99 (pembelian satu kali)<br><br>

Semua opsi Premium membuka rangkaian fitur yang sama. [Halaman perbandingan khusus](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/) menunjukkan perbedaannya secara detail.<br><br>

{{% details title="Apakah Evertag aman?" closed="true" %}}
Ya. Evertag hanya menggunakan **SDK penyedia cloud resmi dan koneksi terenkripsi** untuk mengakses akun Anda, dan kata sandi Anda tidak pernah dibagikan dengan aplikasi. Saat Anda menghubungkan layanan cloud, halaman otorisasi resmi yang disediakan oleh penyedia cloud ditampilkan — seluruh alur login terjadi di luar Evertag, dan penyedia mengembalikan token autentikasi yang digunakan aplikasi untuk panggilan API.<br><br>

Token autentikasi disimpan di perangkat Anda di **Keychain** sistem yang aman, tidak pernah dikirim ke server kami. File yang Anda unduh dari layanan cloud terhubung ditempatkan di direktori "Dokumen" aplikasi, dan Anda dapat menghapusnya kapan saja dari manajer file bawaan.<br><br>

Evertag tidak berbagi informasi apa pun dari akun cloud Anda yang terhubung. Anda dapat mencabut akses kapan saja:<br>
• Di dalam Evertag — buka akun cloud, ketuk **…** dan pilih **Putuskan Koneksi**. Token autentikasi dihapus dari perangkat Anda.<br>
• Di situs web penyedia cloud — masuk, buka halaman aplikasi pihak ketiga / aplikasi terhubung dan hapus Evertag dari daftar.<br><br>

Jika Anda menghapus Evertag dari perangkat Anda, semua data yang diunduh dan token akses dihapus bersamanya.<br><br>
{{% /details %}}

{{% details title="Bisakah Evertag menulis tag dalam format ID3v2.3 atau ID3v2.4?" closed="true" %}}
Ya. Buka **Pengaturan → Editor tag audio → Opsi penyimpanan tag** dan aktifkan atau nonaktifkan **ID3v2.4**.<br><br>

• **ID3v2.4 aktif** (default) — standar modern dengan dukungan UTF-8 penuh, bingkai multi-nilai, dan metadata yang lebih kaya. Ideal untuk pemutar desktop saat ini, manajer musik, dan sebagian besar stereo mobil modern.<br>
• **ID3v2.4 nonaktif** — menyimpan tag dalam format **ID3v2.3** yang lebih didukung secara luas. Gunakan ini jika perangkat lama, head unit mobil, atau perangkat lunak DJ menampilkan tag kosong setelah pengeditan.<br><br>

Halaman pengaturan yang sama menampilkan opsi **Tag duplikat**, yang menulis metadata umum ke dalam kedua bagian ID3 dari sebuah file untuk kompatibilitas maksimal dengan pemutar lama.<br><br>
{{% /details %}}

{{% details title="Bisakah Evertag memperbaiki karakter yang rusak atau korup dalam tag audio (Sirilik, Cina, Jepang, dll.)?" closed="true" %}}
Ya. Evertag menyertakan **normalisator encoding** bawaan yang mengonversi teks yang salah encoded (misalnya, tag Sirilik, Cina, Jepang, atau Korea yang muncul sebagai karakter tidak terbaca seperti `Ðàñêîëüíèêîâ`) kembali menjadi UTF-8 yang dapat dibaca.<br><br>

Untuk menggunakannya:<br>
1. Buka file di editor tag.<br>
2. Ketuk tombol **Normalisasi encoding** di bawah karya seni.<br>
3. Simpan file.<br><br>

Jika Anda tidak melihat tombolnya, aktifkan di **Pengaturan → Editor tag audio → Tombol di layar utama**. Perbaikan encoding sangat berguna untuk perpustakaan yang awalnya diberi tag di Windows di mana pembaca tag default ke ANSI / CP-1251 / Shift-JIS alih-alih UTF-8.<br><br>

{{% details title="Bisakah saya mengedit tag audio di Google Drive, Dropbox, atau iCloud Drive tanpa mengunduh seluruh perpustakaan?" closed="true" %}}
Ya. Setelah Anda menghubungkan akun cloud, Evertag hanya melakukan streaming file yang akan Anda edit — tidak perlu mencerminkan seluruh perpustakaan Anda secara lokal.<br><br>

Saat Anda mengetuk file jarak jauh:<br>
1. Evertag mengunduhnya ke folder sementara.<br>
2. Anda mengedit tag seperti biasa.<br>
3. Setelah **Simpan**, file yang diperbarui diunggah ulang ke lokasi yang sama di cloud.<br>
4. Salinan sementara dihapus (atau disimpan, tergantung pada **Pengaturan → Editor tag audio → Edit file online**).<br><br>

Anda dapat mengontrol perilaku penyimpanan di bawah **Pengaturan → Editor tag audio → Perbarui file online**: tanya sebelum setiap penyimpanan, selalu unggah secara otomatis, atau lewati pengunggahan sepenuhnya (pengeditan hanya lokal).<br><br>
{{% /details %}}

{{% details title="Bisakah saya menggunakan Evertag dengan NAS seperti Synology, QNAP, Asustor, atau Apple Time Capsule?" closed="true" %}}
Ya. Hubungkan NAS apa pun menggunakan **SMB** atau **WebDAV** — keduanya adalah protokol yang didukung oleh setiap vendor NAS utama (Synology, QNAP, Asustor, TerraMaster, Western Digital, Buffalo, Apple Time Capsule) secara default.<br><br>

• **SMB** — ketuk **Hubungkan ke penyimpanan cloud → SMB**, masukkan `smb://alamat-ip-nas/nama-folder-bersama`, pilih versi protokol (Auto / SMB1 / SMB2), masukkan login + kata sandi, ketuk **Selesai**. Tutorial lengkap: [Stream musik dari Mac atau PC melalui SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).<br>
• **WebDAV** — alur yang sama, tetapi masukkan `http://nama-server` atau `https://nama-server` untuk SSL. Tutorial lengkap: [Hubungkan penyimpanan NAS melalui WebDAV](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).<br>
• **Synology Drive** — didukung sebagai integrasi kelas satu (terpisah dari WebDAV generik). Tutorial lengkap: [Hubungkan NAS Synology dan dengarkan musik di iPhone atau Mac Anda](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/).<br><br>

Perangkat yang menyiarkan di jaringan lokal juga muncul secara otomatis di bagian **Perangkat yang Tersedia** pada layar Koneksi.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara memperbarui metadata file secara otomatis di layanan cloud setelah mengedit?" closed="true" %}}
Perilaku penyimpanan untuk file cloud dikontrol di **Pengaturan → Editor tag audio → Perbarui file online** dengan tiga opsi:<br><br>

• **Tampilkan pesan konfirmasi** *(default)* — aplikasi meminta Anda sebelum mengunggah file yang diedit kembali ke cloud.<br>
• **Perbarui metadata file secara otomatis** — aplikasi diam-diam mengunggah ulang file segera setelah Anda mengetuk **Simpan** di editor tag.<br>
• **Jangan perbarui metadata file** — aplikasi melewati pengunggahan sepenuhnya; perubahan Anda tetap lokal dan salinan cloud tidak tersentuh.<br><br>

Padukan ini dengan **Pengaturan → Editor tag audio → Edit file online** untuk memutuskan apakah salinan yang diunduh secara lokal dihapus (menghemat ruang) atau disimpan (pembukaan ulang lebih cepat) setelah setiap pengeditan.<br><br>
{{% /details %}}

{{% details title="Mengapa perubahan saya tidak muncul setelah mengedit file cloud?" closed="true" %}}
Pertama, periksa apakah preferensi **Perbarui file online** Anda tidak disetel ke **Jangan perbarui metadata file** — mode itu menyimpan perubahan secara lokal dan tidak pernah mengunggah ulang ke cloud. Buka **Pengaturan → Editor tag audio → Perbarui file online** dan pilih **Tampilkan pesan konfirmasi** atau **Perbarui metadata file secara otomatis** sebagai gantinya.<br><br>

Hal lain yang perlu diverifikasi:<br>
• Anda memiliki **izin tulis** untuk folder cloud. Beberapa mount WebDAV / SMB bersama / hanya-baca akan menerima pengunggahan secara diam-diam tetapi membuangnya.<br>
• Penyedia cloud belum mengeluarkan Anda secara diam-diam — buka akun dan autentikasi ulang dari **Koneksi → … → Pengaturan**.<br>
• Klien desktop atau web layanan cloud mungkin masih menyinkronkan versi lama. Tunggu sebentar dan segarkan.<br>
• Jika Anda menggunakan **iCloud Drive**, file mungkin masih diunggah dari perangkat lain. Buka file di aplikasi File iOS dan konfirmasi indikator sinkronisasi sudah tidak berputar.<br><br>

Masih macet? Gunakan **Pengaturan → Kirim Umpan Balik** dengan format file, layanan cloud, dan deskripsi singkat.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara mengunduh file dari penyimpanan cloud?" closed="true" %}}
Anda memerlukan setidaknya satu akun cloud terhubung terlebih dahulu — buka layar **Koneksi** dan tambahkan penyedia.<br><br>

Untuk mengunduh file atau folder:<br>
1. Buka layanan penyimpanan cloud yang terhubung.<br>
2. Navigasi ke folder yang ingin Anda unduh.<br>
3. Ketuk tombol **Lebih banyak tindakan** (`…`) di kanan atas dan pilih **Pilih**.<br>
4. Centang file atau folder yang Anda butuhkan, lalu ketuk **Unduh**.<br><br>

Unduhan mendarat di folder **File Lokal → Unduh** dan muncul di **antrean transfer** — buka dari ikon panah berputar di kiri atas layar File Lokal. Anda dapat mengubah tujuan default dari **Pengaturan → Manajer file → Simpan file yang diunduh ke** dan mengalihkan transfer latar belakang dari layar yang sama.<br><br>

{{% details title="Bagaimana cara mengelola file di penyimpanan jaringan?" closed="true" %}}
Buka folder cloud, ketuk tombol **…** di kanan atas dan pilih **Pilih** untuk memasuki mode seleksi. Kotak centang muncul di sebelah setiap file dan folder sehingga Anda dapat memilih satu atau banyak item.<br><br>

Tindakan yang tersedia pada seleksi:<br>
• **Mengedit tag audio** — buka satu atau beberapa file di [editor tag](/docs/guide/evertag/evertag-guide-tag-editor/) (mode massal untuk multi-pilih).<br>
• **Unduh** — salin file atau folder ke bagian **File Lokal**.<br>
• **Pindah** — pindahkan item ke folder lain di dalam akun cloud yang sama.<br>
• **Ganti nama** — ganti nama file atau folder di server jarak jauh.<br>
• **Hapus** — hapus permanen file atau folder dari cloud (tidak dapat dipulihkan).<br>
• **Urutkan** — urutkan ulang berdasarkan nama, ukuran, atau tanggal diedit.<br>
• **Grid / Daftar** — beralih antara tampilan tabel dan thumbnail.<br><br>

Jika tidak ada ruang untuk setiap tindakan, toolbar menampilkan tombol overflow **Lebih banyak tindakan** — ketuk untuk melihat daftar lengkap.<br><br>

{{% details title="Bagaimana cara mengedit metadata untuk file musik yang tersimpan secara lokal di iPhone atau iPad?" closed="true" %}}
Buka **File Lokal** dan gulir ke bagian **File di iPhone ini** (atau iPad / Mac).<br><br>

• Pilih **Buka file…** untuk memilih satu atau lebih file audio di mana saja di perangkat, atau **Buka folder…** untuk memilih seluruh folder.<br>
• Konfirmasi dengan **Buka** — file ditambahkan ke editor segera, **diedit di tempat**, dan tidak pernah disalin ke Evertag.<br><br>

**Akses cepat ke folder perangkat.** Lewati picker setiap saat dengan menambahkan folder ke bagian:<br>
1. Buka **File Lokal**.<br>
2. Gulir ke **File di perangkat ini** dan ketuk **Hubungkan folder**.<br>
3. Pilih folder musik dan ketuk **Buka**.<br><br>

Folder muncul di daftar **File di iPhone ini** dengan akses baca/tulis — ideal untuk perpustakaan musik yang tersimpan di bawah aplikasi File sistem atau di folder yang diekspos aplikasi lain. Untuk memutuskan hubungan nanti, ketuk **…** di sebelah folder dan pilih **Putuskan Koneksi**. Tutorial terperinci: [Putar musik lokal yang tersimpan di iPhone atau Mac Anda](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).<br><br>

{{% details title="Bagaimana cara mentransfer musik ke Evertag dari komputer saya?" closed="true" %}}
Anda dapat menghubungkan komputer atau NAS menggunakan **SMB**, **WebDAV**, **Wi-Fi Drive**, atau **Finder File Sharing** (iTunes File Sharing di macOS yang lebih lama).<br><br>

**SMB** — ketuk **Hubungkan ke penyimpanan cloud → SMB**, masukkan `smb://alamat-ip-komputer/nama-folder-bersama`, atur kredensial, ketuk **Selesai**. Tutorial lengkap: [Transfer file dari komputer ke iPhone menggunakan SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/).<br><br>

**WebDAV** — alur yang sama dengan `http://nama-server` (atau `https://` untuk SSL). Tutorial lengkap: [Hubungkan penyimpanan NAS melalui WebDAV](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).<br><br>

**Wi-Fi Drive** — sepenuhnya nirkabel. Buka **Koneksi → Komputer → Hubungkan melalui Wi-Fi** dan mulai server, lalu buka URL yang ditampilkan di browser komputer Anda dan **seret dan lepas** file ke halaman. Kedua perangkat perlu berbagi jaringan Wi-Fi yang sama. Tutorial lengkap: [Transfer file secara nirkabel menggunakan Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).<br><br>

**Finder File Sharing** (atau iTunes File Sharing di macOS / Windows yang lebih lama) — colokkan iPhone/iPad Anda ke komputer dengan kabel, buka **Finder → perangkat Anda → File → Evertag**, dan seret file ke folder bersama. Tutorial lengkap: [Putar file iTunes lokal di iPhone saya](/docs/howto/how-to-play-local-itunes-files-on-my-iphone).<br><br>
{{% /details %}}

{{% details title="Bisakah saya mengedit tag audio secara offline tanpa koneksi internet?" closed="true" %}}
Ya. Mengedit file lokal sepenuhnya bekerja offline — editor tag, mode massal, penggantian sampul album, perbaikan encoding, dan editor lirik tidak memerlukan koneksi internet.<br><br>

Fitur yang memang membutuhkan internet ditandai dengan jelas: pencarian otomatis MusicBrainz, pencarian tag MusicBrainz manual, pencarian web sampul album, dan pintasan pencarian lirik (Lrclib, Genius, Lyricsify, Google). Jika Anda offline di tengah sesi, layanan tersebut cukup dinonaktifkan — pengeditan lokal Anda tidak terpengaruh.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara mengedit beberapa file sekaligus (mode massal)?" closed="true" %}}
Gunakan **mode massal** untuk menerapkan perubahan metadata yang sama ke banyak trek sekaligus. Ada dua cara untuk memulainya:<br><br>

**Dari manajer file**<br>
1. Buka folder yang berisi file.<br>
2. Ketuk tombol **Lebih banyak tindakan** (`…`) → **Pilih**.<br>
3. Centang file yang ingin Anda edit.<br>
4. Ketuk **Mengedit tag audio**.<br><br>

**Dari editor tag**<br>
1. Buka file mana saja dari folder.<br>
2. Gulir ke bawah editor tag.<br>
3. Ketuk **Edit file secara bersamaan** — semua file audio di folder yang sama dimuat bersama-sama.<br><br>

Buat perubahan Anda sekali dan ketuk **Simpan** — nilai baru ditulis ke setiap file yang dipilih. Ideal untuk memperbaiki nama album, artis album, genre, tahun, atau total trek di seluruh album.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara mengubah sampul album di beberapa lagu sekaligus?" closed="true" %}}
Aktifkan **mode massal** (lihat di atas), ketuk **ikon kamera** di karusel karya seni, dan pilih gambar baru — karya seni yang sama ditulis ke setiap file yang dipilih dalam satu penyimpanan.<br><br>

Anda dapat mengambil sampul baru dari:<br>
• **Perpustakaan Foto** — berguna untuk sampul yang sudah Anda simpan atau pindai.<br>
• **File Lokal / Unduhan** — gambar apa pun yang sudah ada di perangkat Anda.<br>
• **Akun cloud terhubung** — arahkan Evertag ke JPG/PNG di cloud Anda.<br><br>

Kontrol kualitas gambar yang disimpan dari **Pengaturan → Editor tag audio → Skala sampul album** — Kecil / Sedang / Besar / Ekstra Besar, atau "Dinonaktifkan" (Premium) untuk mempertahankan ukuran asli.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara menemukan sampul album yang hilang secara otomatis?" closed="true" %}}
Ketuk tombol **Cari sampul album** di bawah karya seni di editor tag. Evertag menggunakan metadata artis dan album trek saat ini untuk mencari karya seni yang cocok secara online.<br><br>

Saat hasil pencarian muncul, tekan lama gambar yang Anda suka dan pilih **Simpan ke Foto** di menu sistem. Kembali ke Evertag, ketuk **ikon kamera** di karya seni, pilih **Perpustakaan Foto**, dan pilih gambar yang disimpan — sampul diterapkan ke file. Tombol Simpan karya seni album (di bawah menu yang sama) juga memungkinkan Anda mencadangkan karya seni saat ini ke folder Dokumen untuk digunakan kembali nanti.<br><br>

Jika tombol Cari sampul album tidak terlihat, aktifkan di **Pengaturan → Editor tag audio → Tombol di layar utama**.<br><br>
{{% /details %}}

{{% details title="Apakah Evertag menggunakan MusicBrainz?" closed="true" %}}
Ya. Pencarian tag otomatis dan manual Evertag didukung oleh database **MusicBrainz** — salah satu proyek metadata musik terbuka terbesar dengan lebih dari 50 juta trek.<br><br>

Dua alur tersedia dari toolbar bawah editor tag:<br>
• **Cari tag audio secara otomatis** — pencarian kecocokan terbaik menggunakan metadata artis / album / judul yang ada.<br>
• **Cari tag audio secara manual** — pilih album, lagu, dan kumpulan bidang yang tepat untuk diterapkan.<br><br>

Anda juga dapat menyimpan pengidentifikasi MusicBrainz secara langsung: Album ID, Album Artist ID, Artist ID, Disc ID, Release Group ID, Release Track ID, Track ID, Work ID, dan lainnya (lihat referensi [Pemetaan Bidang Tag](/docs/guide/evertag/evertag-tag-field-mappings/)). ID ini memungkinkan aplikasi lain yang mendukung MusicBrainz (Picard, MusicBrainz Browser, Roon, Plex) untuk mereferensikan silang trek Anda dengan andal.<br><br>
{{% /details %}}

{{% details title="Bagaimana cara menambahkan lirik ke lagu di Evertag?" closed="true" %}}
Lirik ada di **editor tag yang diperluas**. Berikut jalur cepatnya:<br><br>

1. Ketuk file untuk membuka editor tag.<br>
2. Ketuk **Tampilkan tag yang diperluas** untuk mengaktifkan kumpulan bidang penuh.<br>
3. Gulir ke bagian **Lirik tidak tersinkron** dan ketuk bidang tersebut.<br>
4. Ketuk **Tambah halaman baru** — setiap halaman lirik memiliki **Bahasa** dan **Deskripsi** sendiri, sehingga satu trek dapat menyimpan beberapa terjemahan.<br>
5. Di bidang **Teks**, tempel lirik biasa atau **lirik berformat LRC (bertimestamp)** — placeholder menampilkan contoh.<br>
6. Ketuk **Selesai** di setiap layar, lalu **Simpan** untuk menulis lirik ke dalam file audio.<br><br>

**Tidak ingin mengetiknya?** Editor lirik memiliki **pintasan pencarian** satu ketuk ke database lirik paling populer — sudah diisi dengan artis dan judul trek saat ini:<br>
• **Lrclib** — sumber terbaik untuk **lirik berwaktu (LRC)**.<br>
• **Genius** — katalog besar dengan lirik teks biasa yang akurat.<br>
• **Lyricsify** — database komunitas dengan lirik biasa dan tersinkron.<br>
• **Google** — pencarian web umum sebagai cadangan.<br><br>

Setiap tombol hanya muncul ketika layanan yang sesuai dapat dijangkau. Panduan lengkap: [Cara mengedit lirik untuk file audio di iPhone atau Mac](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/).<br><br>

**Lirik tersinkron (LRC).** Editor menerima teks biasa **dan** lirik berformat LRC (timestamp baris `[mm:ss.ms]`) — tempel teks LRC ke dalam bidang dan simpan, dan tag lirik file audio akan menyimpannya verbatim, siap untuk pemutar yang mendukung LRC. Pintasan bawaan **Lrclib** dan **Lyricsify** mengembalikan string LRC secara default, sehingga Anda dapat membangun perpustakaan lirik yang sepenuhnya tersinkron dalam hitungan detik tanpa meninggalkan aplikasi.<br><br>

{{% details title="Bagaimana cara menghapus Evertag dan menghapus semua data saya?" closed="true" %}}
Menghapus aplikasi dari iOS, iPadOS, atau macOS menghapus **semua** file yang dibuat Evertag, termasuk lagu yang diunduh, thumbnail yang di-cache, sampul album, pengaturan, kode sandi, dan token autentikasi untuk akun cloud yang terhubung.<br><br>

Jika Anda lebih suka menyimpan aplikasi tetapi membebaskan ruang tanpa menghapus instalan:<br>
• **Pengaturan → Manajer file → Thumbnail untuk file → Hapus semua** — menghapus thumbnail yang di-cache.<br>
• **Pengaturan → Manajer file → Hapus file sementara** — menghapus folder cache.<br>
• Buka **File Lokal → Unduhan** dan hapus file atau folder individual yang tidak Anda butuhkan lagi.<br>
• Putuskan koneksi akun cloud dari layar **Koneksi** untuk menghapus token autentikasi mereka.<br><br>

File sisi cloud tidak pernah dihapus dengan menghapus instalan Evertag — file tersebut tetap di server penyedia cloud dan dapat dikunjungi kembali dari klien web mana pun.<br><br>

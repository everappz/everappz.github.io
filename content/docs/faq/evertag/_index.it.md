Evertag è un editor di metadati musicali per **iPhone, iPad e Mac** che consente di correggere i tag audio, sostituire le copertine degli album, modificare i testi e organizzare la libreria — localmente, su iCloud Drive, su Google Drive / Dropbox / OneDrive / MEGA / pCloud, o su un NAS personale tramite SMB / WebDAV. {{% details title="Cos'è Evertag?" closed="true" %}}
Evertag è un editor di metadati musicali e gestore di copertine album per **iOS e macOS** che consente di correggere i tag, aggiungere copertine e organizzare i file audio archiviati sul dispositivo o nel cloud.<br><br>

L'app supporta un'ampia gamma di formati audio popolari — MP3, FLAC, WAV, M4A, AIFF, OGG, OPUS, WMA, APE e molti altri — e consente di modificare i tag comuni come **TITLE**, **ARTIST**, **ALBUM**, **GENRE**, **YEAR**, **TRACK NUMBER**, oltre a campi estesi come **BPM**, **DISC NUMBER**, **LYRICS**, **MUSICBRAINZ IDs**, valori **REPLAY-GAIN** e classificazioni **PARENTAL ADVISORY**. È possibile lavorare su un file alla volta o passare alla **modalità in blocco** per modificare più tracce contemporaneamente — perfetto per organizzare album o playlist completi.<br><br>

Una delle funzionalità più distintive di Evertag è la capacità di recuperare le copertine degli album mancanti da Internet (tramite MusicBrainz) o di aggiungerne di proprie dalla libreria Foto. L'editor di testi integrato supporta i testi semplici e **sincronizzati (LRC)** e include scorciatoie con un tocco verso Lrclib, Genius, Lyricsify e Google. L'app supporta la modifica **open-in-place**, quindi è possibile modificare i tag audio senza copiare i file.<br><br>

Che tu gestisca la musica sul tuo dispositivo o nel cloud (iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Synology, pCloud e molti altri), Evertag offre accesso e modifica dei file senza soluzione di continuità — ideale per musicisti, DJ, podcaster e collezionisti che desiderano una libreria pulita e ben organizzata su iPhone, iPad o Mac senza usare un editor di tag desktop. Consulta la [guida utente](/docs/guide/evertag/) completa per screenshot e tutorial.<br><br>

{{% details title="Evertag è sicuro?" closed="true" %}}
Sì. Evertag utilizza solo **SDK ufficiali dei provider cloud e connessioni crittografate** per accedere ai tuoi account, e la tua password non viene mai condivisa con l'app. Quando connetti un servizio cloud, viene mostrata la pagina di autorizzazione ufficiale fornita dal provider cloud — l'intero flusso di accesso avviene al di fuori di Evertag, e il provider restituisce un token di autenticazione che l'app utilizza per le chiamate API.<br><br>

Il token di autenticazione viene archiviato sul dispositivo nel **Portachiavi** di sistema sicuro, mai inviato ai nostri server. I file scaricati da un servizio cloud connesso vengono inseriti nella directory "Documenti" dell'app, e puoi rimuoverli in qualsiasi momento dal gestore file integrato.<br><br>

Evertag non condivide alcuna informazione dal tuo account cloud connesso. Puoi revocare l'accesso in qualsiasi momento:<br>
• All'interno di Evertag — apri l'account cloud, tocca **…** e scegli **Disconnettere**. Il token di autenticazione viene rimosso dal dispositivo.<br>
• Sul sito web del provider cloud — accedi, vai alla pagina delle app di terze parti / app connesse e rimuovi Evertag dall'elenco.<br><br>

Se elimini Evertag dal dispositivo, tutti i dati scaricati e i token di accesso vengono rimossi insieme ad esso.<br><br>
{{% /details %}}

Puoi rivedere e disattivare questi servizi in **Impostazioni → Analisi e raccolta dati**. Il prompt App Tracking Transparency di Apple consente anche di rifiutare il tracciamento la prima volta che l'app viene avviata. Consulta la nostra [Informativa sulla privacy](/legal/privacy-policy/) per la panoramica completa.<br><br>

{{% details title="Evertag può scrivere tag in formato ID3v2.3 o ID3v2.4?" closed="true" %}}
Sì. Apri **Impostazioni → Editor tag audio → Opzioni di salvataggio tag** e attiva o disattiva **ID3v2.4**.<br><br>

• **ID3v2.4 attivato** (predefinito) — standard moderno con supporto UTF-8 completo, frame multi-valore e metadati più ricchi. Ideale per i player desktop attuali, i gestori musicali e la maggior parte degli stereo per auto moderni.<br>
• **ID3v2.4 disattivato** — salva i tag nel formato **ID3v2.3** più ampiamente supportato. Usa questa opzione se un dispositivo più vecchio, un'unità principale per auto o un software DJ mostra tag vuoti dopo la modifica.<br><br>

La stessa pagina delle impostazioni espone l'opzione **Tag duplicati**, che scrive i metadati comuni in entrambe le sezioni ID3 di un file per la massima compatibilità con i player legacy.<br><br>

{{% details title="Evertag può correggere caratteri illeggibili o corrotti nei tag audio (cirillico, cinese, giapponese, ecc.)?" closed="true" %}}
Sì. Evertag include un **normalizzatore di codifica** integrato che converte il testo con codifica errata (ad esempio, tag cirillici, cinesi, giapponesi o coreani che appaiono come caratteri illeggibili come `Ðàñêîëüíèêîâ`) nuovamente in UTF-8 leggibile.<br><br>

Per utilizzarlo:<br>
1. Apri un file nell'editor di tag.<br>
2. Tocca il pulsante **Normalizza codifica** sotto l'artwork.<br>
3. Salva il file.<br><br>

Se non vedi il pulsante, abilitalo in **Impostazioni → Editor tag audio → Pulsanti nella schermata principale**. Le correzioni di codifica sono particolarmente utili per le librerie originariamente taggiate su Windows dove i lettori di tag predefiniti utilizzano ANSI / CP-1251 / Shift-JIS invece di UTF-8.<br><br>

{{% details title="Posso modificare i tag audio in Google Drive, Dropbox o iCloud Drive senza scaricare l'intera libreria?" closed="true" %}}
Sì. Dopo aver connesso un account cloud, Evertag trasmette in streaming solo il file che stai per modificare — non è necessario eseguire il mirror dell'intera libreria in locale.<br><br>

Quando tocchi un file remoto:<br>
1. Evertag lo scarica in una cartella temporanea.<br>
2. Modifichi i tag come al solito.<br>
3. Alla pressione di **Salvare**, il file aggiornato viene ricaricato nella stessa posizione nel cloud.<br>
4. La copia temporanea viene rimossa (o mantenuta, a seconda di **Impostazioni → Editor tag audio → Modifica file online**).<br><br>

Puoi controllare il comportamento del salvataggio in **Impostazioni → Editor tag audio → Aggiorna file online**: chiedi prima di ogni salvataggio, carica sempre automaticamente o salta completamente il caricamento (modifiche solo locali).<br><br>

{{% details title="Posso usare Evertag con un NAS come Synology, QNAP, Asustor o Apple Time Capsule?" closed="true" %}}
Sì. Connetti qualsiasi NAS usando **SMB** o **WebDAV** — sono i due protocolli supportati da ogni principale fornitore di NAS (Synology, QNAP, Asustor, TerraMaster, Western Digital, Buffalo, Apple Time Capsule) di default.<br><br>

• **SMB** — tocca **Connetti alla cloud storage → SMB**, inserisci `smb://indirizzo-ip-nas/nome-cartella-condivisa`, scegli la versione del protocollo (Auto / SMB1 / SMB2), inserisci login + password, tocca **Fatto**. Tutorial completo: [Trasmetti musica da Mac o PC tramite SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).<br>
• **WebDAV** — stesso flusso, ma inserisci `http://nome-server` o `https://nome-server` per SSL. Tutorial completo: [Connetti storage NAS tramite WebDAV](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).<br>
• **Synology Drive** — supportato come integrazione di prima classe (separato da WebDAV generico). Tutorial completo: [Connetti NAS Synology e ascolta musica sul tuo iPhone o Mac](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/).<br><br>

I dispositivi che trasmettono sulla rete locale appaiono automaticamente anche nella sezione **Dispositivi disponibili** della schermata Connessioni.<br><br>

{{% details title="Come aggiorno automaticamente i metadati dei file su un servizio cloud dopo la modifica?" closed="true" %}}
Il comportamento di salvataggio per i file cloud è controllato in **Impostazioni → Editor tag audio → Aggiorna file online** con tre opzioni:<br><br>

• **Mostra messaggio di conferma** *(predefinito)* — l'app ti chiede prima di caricare il file modificato nel cloud.<br>
• **Aggiorna automaticamente i metadati del file** — l'app ricarica silenziosamente il file immediatamente dopo aver toccato **Salvare** nell'editor di tag.<br>
• **Non aggiornare i metadati del file** — l'app salta completamente il caricamento; le modifiche rimangono locali e la copia cloud rimane intatta.<br><br>

Abbina questa impostazione a **Impostazioni → Editor tag audio → Modifica file online** per decidere se la copia scaricata localmente viene eliminata (risparmia spazio) o mantenuta (riapertura più veloce) dopo ogni modifica.<br><br>
{{% /details %}}

{{% details title="Perché le mie modifiche non appaiono dopo aver modificato un file cloud?" closed="true" %}}
Prima, verifica che la preferenza **Aggiorna file online** non sia impostata su **Non aggiornare i metadati del file** — quella modalità mantiene le modifiche locali e non ricarica mai nel cloud. Apri **Impostazioni → Editor tag audio → Aggiorna file online** e scegli invece **Mostra messaggio di conferma** o **Aggiorna automaticamente i metadati del file**.<br><br>

Altre cose da verificare:<br>
• Hai il **permesso di scrittura** per la cartella cloud. Alcuni mount WebDAV / SMB condivisi / in sola lettura accettano il caricamento silenziosamente ma lo scartano.<br>
• Il provider cloud non ti ha disconnesso silenziosamente — apri l'account e riautentica da **Connessioni → … → Impostazioni**.<br>
• Il client desktop o web del servizio cloud potrebbe ancora sincronizzare la versione precedente. Attendi un minuto e aggiorna.<br>
• Se stai usando **iCloud Drive**, il file potrebbe essere ancora in fase di caricamento da un altro dispositivo. Apri il file nell'app File di iOS e conferma che l'indicatore di sincronizzazione non stia più girando.<br><br>

Ancora bloccato? Usa **Impostazioni → Invia feedback** con il formato file, il servizio cloud e una breve descrizione.<br><br>

{{% details title="Come scarico i file dalla cloud storage?" closed="true" %}}
Hai bisogno di almeno un account cloud connesso — apri la schermata **Connessioni** e aggiungi un provider.<br><br>

Per scaricare file o cartelle:<br>
1. Apri il servizio di cloud storage connesso.<br>
2. Naviga nella cartella che vuoi scaricare.<br>
3. Tocca il pulsante **Altre azioni** (`…`) in alto a destra e scegli **Selezionare**.<br>
4. Seleziona i file o le cartelle di cui hai bisogno, poi tocca **Scaricare**.<br><br>

I download finiscono nella cartella **File locali → Download** e appaiono nella **coda di trasferimento** — aprila dall'icona delle frecce rotanti in alto a sinistra della schermata File locali. Puoi cambiare la destinazione predefinita da **Impostazioni → Gestore file → Salva file scaricati in** e attivare/disattivare i trasferimenti in background dalla stessa schermata.<br><br>
{{% /details %}}

{{% details title="Come gestisco i file in uno storage di rete?" closed="true" %}}
Apri una cartella cloud, tocca il pulsante **…** in alto a destra e seleziona **Selezionare** per entrare in modalità selezione. Appaiono caselle di controllo accanto a ogni file e cartella così puoi scegliere uno o più elementi.<br><br>

Azioni disponibili sulla selezione:<br>
• **Modificare i tag audio** — apri uno o più file nell'[editor di tag](/docs/guide/evertag/evertag-guide-tag-editor/) (modalità in blocco per la selezione multipla).<br>
• **Scaricare** — copia file o cartelle nella sezione **File locali**.<br>
• **Sposta** — sposta l'elemento in un'altra cartella all'interno dello stesso account cloud.<br>
• **Rinomina** — rinomina il file o la cartella sul server remoto.<br>
• **Eliminare** — rimuovi permanentemente il file o la cartella dal cloud (irreversibile).<br>
• **Ordina** — riordina per nome, dimensione o data di modifica.<br>
• **Griglia / Elenco** — passa tra la visualizzazione a tabella e quella in miniatura.<br><br>

Se non c'è spazio per ogni azione, la barra degli strumenti mostra un pulsante overflow **Altre azioni** — toccalo per vedere l'elenco completo.<br><br>

{{% details title="Come modifico i metadati per i file musicali archiviati localmente su iPhone o iPad?" closed="true" %}}
Apri **File locali** e scorri alla sezione **File su questo iPhone** (o iPad / Mac).<br><br>

• Scegli **Apri file…** per selezionare uno o più file audio in qualsiasi punto del dispositivo, o **Apri cartella…** per selezionare un'intera cartella.<br>
• Conferma con **Apri** — i file vengono aggiunti immediatamente all'editor, **modificati in loco**, e non vengono mai copiati in Evertag.<br><br>

**Accesso rapido a una cartella del dispositivo.** Salta il selettore ogni volta aggiungendo una cartella alla sezione:<br>
1. Apri **File locali**.<br>
2. Scorri a **File su questo dispositivo** e tocca **Connetti una cartella**.<br>
3. Scegli la cartella musicale e tocca **Apri**.<br><br>

La cartella appare nell'elenco **File su questo iPhone** con accesso in lettura/scrittura — ideale per le librerie musicali archiviate nell'app File di sistema o in una cartella esposta da un'altra app. Per scollegarla in seguito, tocca **…** accanto alla cartella e scegli **Disconnettere**. Tutorial dettagliato: [Riproduci musica locale archiviata su iPhone o Mac](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).<br><br>
{{% /details %}}

{{% details title="Come trasferisco musica su Evertag dal mio computer?" closed="true" %}}
Puoi connettere il tuo computer o NAS usando **SMB**, **WebDAV**, **Wi-Fi Drive** o **Finder File Sharing** (iTunes File Sharing su macOS precedente).<br><br>

**SMB** — tocca **Connetti alla cloud storage → SMB**, inserisci `smb://indirizzo-ip-computer/nome-cartella-condivisa`, imposta le credenziali, tocca **Fatto**. Tutorial completo: [Trasferisci file dal computer all'iPhone usando SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/).<br><br>

**WebDAV** — stesso flusso con `http://nome-server` (o `https://` per SSL). Tutorial completo: [Connetti storage NAS tramite WebDAV](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).<br><br>

**Wi-Fi Drive** — completamente wireless. Apri **Connessioni → Computer → Connetti tramite Wi-Fi** e avvia il server, poi apri l'URL mostrato nel browser del tuo computer e **trascina e rilascia** i file sulla pagina. Entrambi i dispositivi devono condividere la stessa rete Wi-Fi. Tutorial completo: [Trasferisci file in modalità wireless usando Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).<br><br>

**Finder File Sharing** (o iTunes File Sharing su macOS / Windows precedente) — collega il tuo iPhone/iPad al computer con un cavo, apri **Finder → il tuo dispositivo → File → Evertag**, e trascina i file nella cartella condivisa. Tutorial completo: [Riproduci file iTunes locali su iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone).<br><br>
{{% /details %}}

{{% details title="Posso modificare i tag audio offline senza connessione internet?" closed="true" %}}
Sì. La modifica dei file locali funziona completamente offline — l'editor di tag, la modalità in blocco, la sostituzione della copertina album, la correzione della codifica e l'editor di testi non richiedono una connessione internet.<br><br>

Le funzionalità che richiedono internet sono chiaramente contrassegnate: ricerca automatica MusicBrainz, ricerca manuale di tag MusicBrainz, la ricerca web di copertine album e le scorciatoie di ricerca testi (Lrclib, Genius, Lyricsify, Google). Se vai offline a metà sessione, quei servizi vengono semplicemente disabilitati — le modifiche locali non sono interessate.<br><br>
{{% /details %}}

{{% details title="Come modifico più file contemporaneamente (modalità in blocco)?" closed="true" %}}
Usa la **modalità in blocco** per applicare la stessa modifica ai metadati a molte tracce contemporaneamente. Ci sono due modi per avviarla:<br><br>

**Dal gestore file**<br>
1. Apri la cartella contenente i file.<br>
2. Tocca il pulsante **Altre azioni** (`…`) → **Selezionare**.<br>
3. Seleziona i file che vuoi modificare.<br>
4. Tocca **Modificare i tag audio**.<br><br>

**Dall'editor di tag**<br>
1. Apri qualsiasi file dalla cartella.<br>
2. Scorri fino in fondo all'editor di tag.<br>
3. Tocca **Modifica i file contemporaneamente** — tutti i file audio nella stessa cartella vengono caricati insieme.<br><br>

Apporta le modifiche una volta e tocca **Salvare** — i nuovi valori vengono scritti in ogni file selezionato. Ideale per correggere il nome dell'album, l'artista dell'album, il genere, l'anno o i totali delle tracce su un intero album.<br><br>
{{% /details %}}

{{% details title="Come cambio la copertina dell'album su più canzoni contemporaneamente?" closed="true" %}}
Attiva la **modalità in blocco** (vedi sopra), tocca l'**icona della fotocamera** sul carosello delle copertine e scegli la nuova immagine — la stessa copertina viene scritta in ogni file selezionato con un singolo salvataggio.<br><br>

Puoi ottenere la nuova copertina da:<br>
• **Libreria Foto** — utile per una copertina che hai già salvato o scansionato.<br>
• **File locali / Download** — qualsiasi immagine già sul dispositivo.<br>
• **Un account cloud connesso** — punta Evertag su un JPG/PNG nel tuo cloud.<br><br>

Controlla la qualità dell'immagine salvata da **Impostazioni → Editor tag audio → Ridimensionamento copertina album** — Piccola / Media / Grande / Extra grande, o "Disattivata" (Premium) per mantenere la dimensione originale.<br><br>
{{% /details %}}

{{% details title="Come trovo automaticamente le copertine album mancanti?" closed="true" %}}
Tocca il pulsante **Cerca copertina album** sotto l'artwork nell'editor di tag. Evertag usa i metadati attuali di artista e album della traccia per cercare online la copertina corrispondente.<br><br>

Quando appaiono i risultati della ricerca, premi a lungo l'immagine che ti piace e scegli **Salva in Foto** nel menu di sistema. Torna a Evertag, tocca l'**icona della fotocamera** sulla copertina, scegli **Libreria foto** e seleziona l'immagine salvata — la copertina viene applicata al file. Il pulsante Salva artwork album (sotto lo stesso menu) consente anche di eseguire il backup dell'artwork attuale nella cartella Documenti per riutilizzarlo in seguito.<br><br>

Se il pulsante Cerca copertina album non è visibile, abilitalo in **Impostazioni → Editor tag audio → Pulsanti nella schermata principale**.<br><br>
{{% /details %}}

{{% details title="Evertag usa MusicBrainz?" closed="true" %}}
Sì. La ricerca automatica e manuale dei tag di Evertag è alimentata dal database **MusicBrainz** — uno dei più grandi progetti aperti di metadati musicali con più di 50 milioni di tracce.<br><br>

Sono disponibili due flussi dalla barra degli strumenti inferiore dell'editor di tag:<br>
• **Cerca automaticamente i tag audio** — ricerca della migliore corrispondenza usando i metadati esistenti di artista / album / titolo.<br>
• **Cerca manualmente i tag audio** — scegli l'album, la canzone e il set esatto di campi da applicare.<br><br>

Puoi anche memorizzare gli identificatori MusicBrainz direttamente: Album ID, Album Artist ID, Artist ID, Disc ID, Release Group ID, Release Track ID, Track ID, Work ID e altro (consulta il riferimento [Mappature dei campi tag](/docs/guide/evertag/evertag-tag-field-mappings/)). Questi ID consentono ad altre app compatibili con MusicBrainz (Picard, MusicBrainz Browser, Roon, Plex) di fare riferimenti incrociati alle tue tracce in modo affidabile.<br><br>
{{% /details %}}

{{% details title="Come aggiungo i testi alle canzoni in Evertag?" closed="true" %}}
I testi si trovano nell'**editor di tag esteso**. Ecco il percorso veloce:<br><br>

1. Tocca un file per aprire l'editor di tag.<br>
2. Tocca **Mostra tag estesi** per abilitare il set di campi completo.<br>
3. Scorri alla sezione **Testi non sincronizzati** e tocca il campo.<br>
4. Tocca **Aggiungi nuova pagina** — ogni pagina di testi ha il proprio **Lingua** e **Descrizione**, quindi una singola traccia può contenere più traduzioni.<br>
5. Nel campo **Testo**, incolla i testi semplici o i **testi in formato LRC (con timestamp)** — il segnaposto mostra un esempio.<br>
6. Tocca **Fatto** su ogni schermata, poi **Salvare** per scrivere i testi nel file audio.<br><br>

**Non vuoi digitarli?** L'editor di testi ha **scorciatoie di ricerca** con un tocco verso i database di testi più popolari — precompilati con l'artista e il titolo della traccia corrente:<br>
• **Lrclib** — la fonte principale per **testi temporizzati (LRC)**.<br>
• **Genius** — grande catalogo con testi in testo normale accurati.<br>
• **Lyricsify** — database della community con testi semplici e sincronizzati.<br>
• **Google** — ricerca web generica come fallback.<br><br>

Ogni pulsante appare solo quando il servizio corrispondente è raggiungibile. Guida completa: [Come modificare i testi per i file audio su iPhone o Mac](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/).<br><br>

**Testi sincronizzati (LRC).** L'editor accetta testo semplice **e** testi in formato LRC (timestamp di riga `[mm:ss.ms]`) — incolla il testo LRC nel campo e salva, e il tag dei testi del file audio lo archivierà verbatim, pronto per qualsiasi player compatibile con LRC. Le scorciatoie integrate **Lrclib** e **Lyricsify** restituiscono stringhe LRC per impostazione predefinita, quindi puoi costruire una libreria di testi completamente sincronizzata in pochi secondi senza uscire dall'app.<br><br>

{{% details title="Come disinstallo Evertag e rimuovo tutti i miei dati?" closed="true" %}}
Eliminando l'app da iOS, iPadOS o macOS si rimuovono **tutti** i file creati da Evertag, incluse le canzoni scaricate, le miniature in cache, le copertine degli album, le impostazioni, il codice e i token di autenticazione per gli account cloud connessi.<br><br>

Se preferisci mantenere l'app ma liberare spazio senza disinstallarla:<br>
• **Impostazioni → Gestore file → Miniature per file → Elimina tutto** — rimuove le miniature in cache.<br>
• **Impostazioni → Gestore file → Elimina file temporanei** — svuota la cartella cache.<br>
• Apri **File locali → Download** ed elimina singoli file o cartelle di cui non hai più bisogno.<br>
• Disconnetti gli account cloud dalla schermata **Connessioni** per rimuovere i loro token di autenticazione.<br><br>

I file lato cloud non vengono mai rimossi disinstallando Evertag — rimangono sui server del provider cloud e possono essere visitati nuovamente da qualsiasi client web.<br><br>

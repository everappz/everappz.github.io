## ปัญหาทั่วไปและคำตอบ

<div class="hx:mt-6"></div>

<div class="hx:w-full">

{{% details title="ทำไมฉันไม่สามารถเข้าสู่ระบบ pCloud บน iOS เวอร์ชันเก่า (15.8.4) ได้?" closed="true" %}}
หน้าเข้าสู่ระบบเว็บของ pCloud อาจไม่แสดงอย่างถูกต้องบน iOS เวอร์ชันเก่าเช่น 15.8.4 ซึ่งป้องกันการป้อนอีเมลและรหัสผ่านในหน้าจอการเชื่อมต่อคลาวด์<br><br>

วิธีแก้ไขคือใช้โปรโตคอล **WebDAV** ซึ่ง pCloud รองรับและทำงานได้อย่างน่าเชื่อถือในทุกเวอร์ชัน iOS

**การตั้งค่า WebDAV:**<br>
- **เซิร์ฟเวอร์สหรัฐอเมริกา:** `https://webdav.pcloud.com`  
- **เซิร์ฟเวอร์ยุโรป:** `https://ewebdav.pcloud.com`  
- **ชื่อผู้ใช้:** ที่อยู่อีเมล pCloud ของคุณ  
- **รหัสผ่าน:** รหัสผ่านบัญชี pCloud ของคุณ  

เปิดแอป → การเชื่อมต่อ → เชื่อมต่อกับพื้นที่เก็บข้อมูลคลาวด์ → เลือก **WebDAV** → ป้อนข้อมูลประจำตัวและ URL เซิร์ฟเวอร์

วิธีนี้ช่วยให้คุณเชื่อมต่อกับพื้นที่เก็บข้อมูล pCloud และเข้าถึงไฟล์ของคุณได้โดยไม่มีปัญหาบนอุปกรณ์เก่า
{{% /details %}}

{{% details title="วิธีเล่นเพลงผ่าน AirPlay จาก Mac (macOS) คืออะไร?" closed="true" %}}
แอปเวอร์ชัน macOS ไม่มีปุ่มเชื่อมต่อ AirPlay, Chromecast หรือ Bluetooth ในตัวเหมือนใน iOS<br><br>

ในการใช้ **AirPlay** บน MacBook Pro ให้ทำตามขั้นตอนเหล่านี้:

1. ไปที่ **มุมขวาบน** ของหน้าจอและเปิด **ศูนย์ควบคุม** (ใกล้นาฬิกา)  
2. คลิกไอคอน **เสียง/ระดับเสียง**  
3. ในหน้าจอถัดไป คลิก **AirPlay** เพื่อดูรายการอุปกรณ์ AirPlay ที่ใช้งานได้ทั้งหมด  
4. เลือกอุปกรณ์ที่ต้องการเพื่อเริ่มสตรีมเพลงของคุณ  

ซึ่งจะส่งเสียงระบบทั้งหมด (รวมถึงจาก Evermusic หรือ Flacbox) ไปยังอุปกรณ์ AirPlay ที่คุณเลือก
{{% /details %}}

{{% details title="ทำไมการซื้อ Premium ของฉันไม่เปิดใช้งานบน Mac ถ้าฉันซื้อบน iPhone?" closed="true" %}}
การซื้อตลอดชีพและการสมัครสมาชิกจะซิงค์ระหว่าง iOS และ Mac ผ่าน **iCloud**<br><br>

ในการเปิดใช้งาน Premium บน Mac:<br>
- ตรวจสอบให้แน่ใจว่าคุณติดตั้งแอปเวอร์ชันล่าสุดบนทั้งสองอุปกรณ์<br>
- เปิดใช้งาน **iCloud** บนทั้งสองอุปกรณ์<br>
- เปิดแอปบนอุปกรณ์ iOS และรอ 1 นาทีเพื่อให้ข้อมูลการซื้ออัพโหลด<br>
- บน Mac ติดตั้งแอปจาก App Store โดยใช้ **Apple ID เดียวกัน**<br>
- เปิดแอปและรอสักครู่เพื่อให้ข้อมูลซิงค์<br>
- หรือแตะ **กู้คืนการซื้อ** ในการตั้งค่าแอปบนทั้งสองอุปกรณ์<br><br>

ฟีเจอร์ Premium ของคุณควรเปิดใช้งานบน Mac โดยอัตโนมัติ
{{% /details %}}

{{% details title="ฉันสามารถซิงค์เพลย์ลิสต์โดยอัตโนมัติระหว่างอุปกรณ์ได้อย่างไร?" closed="true" %}}
ปัจจุบัน **ไม่มีการซิงค์อัตโนมัติ** สำหรับเพลย์ลิสต์<br><br>

คุณสามารถใช้หนึ่งในตัวเลือกต่อไปนี้:<br>
- **สำรองข้อมูลและกู้คืน** จากการตั้งค่าแอป [วิธีถ่ายโอนคลังเพลงของคุณระหว่างอุปกรณ์ใน Evermusic: คู่มือทีละขั้นตอน](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/)<br>
- **ส่งออกเพลย์ลิสต์เป็น M3U** แล้วนำเข้าบนอุปกรณ์อื่น:<br>
  - [วิธีส่งออกเพลย์ลิสต์](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)<br>
  - [วิธีนำเข้าเพลย์ลิสต์](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox/)<br>
- **เก็บถาวรเพลย์ลิสต์หรืออัลบั้ม** และถ่ายโอนผ่าน ZIP:<br>
  - [คู่มือการเก็บถาวรเพลย์ลิสต์](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/)
{{% /details %}}

{{% details title="การใช้แอปของคุณปลอดภัยหรือไม่? ฉันสามารถปิดใช้งานการวิเคราะห์ได้หรือไม่?" closed="true" %}}
ใช่ ความเป็นส่วนตัวของคุณคือสิ่งสำคัญที่สุดของเรา<br><br>

- ข้อมูลทั้งหมด — ไฟล์เพลง การตั้งค่า การเข้าสู่ระบบคลาวด์ — อยู่บนอุปกรณ์ของคุณ<br>
- ข้อมูลการเข้าสู่ระบบถูกเก็บอย่างปลอดภัยใน **iOS Keychain**<br>
- เราเก็บรวบรวมเฉพาะรายงานการล่มและการใช้งานแบบไม่ระบุตัวตนเท่านั้น<br>
- คุณสามารถออกจากระบบได้ใน **การตั้งค่าแอป → การวิเคราะห์**<br><br>

ข้อมูลเพิ่มเติม:<br>
- [นโยบายความเป็นส่วนตัว](/legal/privacy-policy/)<br>
- [ความปลอดภัย Apple Keychain](https://support.apple.com/guide/security/keychain-data-protection-secb0694df1a/web)<br><br>

หากใช้โฆษณาส่วนบุคคล Google Mobile Ads จำเป็นต้องแสดงการตั้งค่าความยินยอม<br>
ผู้ใช้ Premium ไม่เห็นโฆษณาและ SDK โฆษณาถูกปิดใช้งานอย่างสมบูรณ์
{{% /details %}}

{{% details title="แอปของคุณรองรับการแชร์กับครอบครัวหรือไม่?" closed="true" %}}
ใช่ รองรับการแชร์กับครอบครัว<br><br>

เพื่อแชร์การซื้อในแอป:<br>
- ตรวจสอบให้แน่ใจว่าการซื้อถูกตั้งค่าให้แชร์กับกลุ่มครอบครัวของคุณ<br>
- บนอุปกรณ์ของสมาชิกครอบครัว ไปที่ **การตั้งค่า > การซื้อ > กู้คืนการซื้อ**<br>
- ซึ่งจะร้องขอข้อมูลการซื้อจากเซิร์ฟเวอร์ของ Apple และเปิดใช้งานบนอุปกรณ์ของพวกเขา
{{% /details %}}

{{% details title="วิธีเร่งความเร็วเมตาดาต้าและการซิงค์คลาวด์คืออะไร?" closed="true" %}}
เพื่อปรับปรุงความเร็วการซิงค์ ให้เปิดใช้งานงานพื้นหลัง:<br><br>

- **การตั้งค่า → คลังเพลง → การอ่านเมตาดาต้า → การอ่านเมตาดาต้าในพื้นหลัง**<br>
- **การตั้งค่า → คลังเพลง → การซิงค์เพลงออนไลน์ → การซิงค์พื้นหลัง**<br><br>

นอกจากนี้ บน macOS ให้เพิ่มความเร็วการอ่านเมตาดาต้าผ่าน **การตั้งค่า → คลังเพลง**<br>
หากเครื่องเล่นทำงานอยู่ (กำลังเล่นเสียง) iOS จะไม่ระงับแอป ทำให้ซิงค์อย่างต่อเนื่องได้
{{% /details %}}

{{% details title="ฉันจะยกเลิกการสมัครสมาชิกได้อย่างไร?" closed="true" %}}
คุณสามารถยกเลิกการสมัครสมาชิกได้จากคำแนะนำอย่างเป็นทางการของ Apple:<br>
👉 [วิธียกเลิกการสมัครสมาชิก](https://support.apple.com/en-us/118428)
{{% /details %}}

{{% details title="วิธีเชื่อมต่อและสตรีมเสียงจาก WD MyCloud EX2 Ultra คืออะไร?" closed="true" %}}

เมื่อคุณเพิ่มการเชื่อมต่อในแอปผ่าน **การเชื่อมต่อ > เชื่อมต่อกับพื้นที่เก็บข้อมูลคลาวด์ > My Cloud Home** ถูกออกแบบมาอย่างเป็นทางการเพื่อรองรับอุปกรณ์ **WD MyCloud Home**<br>
WD MyCloud EX2 Ultra ใช้การเข้าถึงที่จำกัดสำหรับแอป<br><br>

อย่างไรก็ตาม หากคุณเชื่อมต่อกับ **WD MyCloud EX2 Ultra**, **WD MyCloud Mirror** หรือรุ่น **WD MyCloud** อื่นๆ สำเร็จแล้ว คุณยังสามารถใช้งานได้ด้วยวิธีแก้ปัญหาต่อไปนี้:<br><br>

1. เปิด **การเชื่อมต่อ → เชื่อมต่อกับพื้นที่เก็บข้อมูลคลาวด์ → My Cloud Home**<br>
2. สร้างโฟลเดอร์โดยใช้ตัวจัดการไฟล์ในตัว<br>
3. อัพโหลดไฟล์เพลงเข้าในโฟลเดอร์นี้<br>
4. ไฟล์เหล่านี้จะถูกเก็บไว้ใน "sandbox" ที่เข้าถึงได้เฉพาะแอปเท่านั้น<br>
5. ตอนนี้คุณสามารถสตรีมหรือดาวน์โหลดได้โดยตรง<br><br>

⚠️ มีเพียงโฟลเดอร์ที่สร้างผ่านแอปเท่านั้นที่จะเข้าถึงได้จาก NAS
{{% /details %}}

{{% details title="วิธีเชื่อมต่อกับ Koofr.eu คืออะไร?" closed="true" %}}

- คู่มือการตั้งค่า Koofr WebDAV: [บล็อก koofr.eu](https://koofr.eu/blog/posts/3-ways-to-map-koofr-as-a-network-drive-explained)<br>
- คู่มือ WebDAV ของ Evermusic/Flacbox: [วิธีเชื่อมต่อพื้นที่เก็บข้อมูล NAS โดยใช้ WebDAV และฟังเพลงบน iPhone หรือ Mac](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac/)
{{% /details %}}

{{% details title="URL scheme ของแอปคืออะไร?" closed="true" %}}
นี่คือ scheme ที่รองรับ:<br><br>

**Evermusic**<br>
- iOS (ไอคอนสีน้ำเงิน): `lsevermusic://`<br>
- macOS: `lsevermusicmac://`<br>
- iOS (ไอคอนสีแดง): `lsevermusicpro://`<br><br>

**Flacbox**<br>
- iOS: `lsflap://`<br>
- macOS: `lsflapmac://`<br><br>

**Evertag**<br>
- iOS: `lsevertag://`<br>
- macOS: `lsevertagmac://`<br><br>

หากแอปล่มหรือหยุดชั่วคราวในพื้นหลัง:<br>

- **การตั้งค่า > คลังเพลง > การอ่านเมตาดาต้า > การอ่านเมตาดาต้าในพื้นหลัง → ปิดใช้งาน**<br>
- **การตั้งค่า > ตัวจัดการไฟล์ > การถ่ายโอนพื้นหลัง → ปิดใช้งาน**

{{% details title="การเล่นต่อเนื่องไม่มีช่องว่างไม่ทำงาน — วิธีแก้ไขคืออะไร?" closed="true" %}}
การเล่นต่อเนื่องไม่มีช่องว่างขึ้นอยู่กับเวอร์ชัน iOS และเครื่องมือเสียง<br>
ลองเปลี่ยนเครื่องมือเสียง:<br>
- ไปที่ **การตั้งค่า → เครื่องเล่นเสียง → ทั่วไป → ตัวประมวลผลเสียง**<br>
- เลือก **Core Audio** เพื่อรองรับการเล่นต่อเนื่องที่ดีขึ้น

{{% details title="ทำไมแอปแสดงเพียง 100 รายการในรายการ?" closed="true" %}}
แอปใช้การแบ่งหน้าเพื่อประสิทธิภาพ<br>
วิธีปิดใช้งาน:<br>
- ไปที่ **การตั้งค่า → การปรับแต่ง → ขีดจำกัดการโหลดเนื้อหา → ปิดใช้งาน**<br>
ตอนนี้รายการทั้งหมดจะโหลดพร้อมกัน

{{% details title="ทำไมแอปไม่สามารถอ่านชื่อโฟลเดอร์ที่มีอักขระพิเศษได้?" closed="true" %}}
นี่เป็นปัญหาที่ทราบกันกับ **โปรโตคอล SMB2**<br><br>

ลองวิธีแก้ปัญหาต่อไปนี้:<br>
- เปิดใช้งาน **SMB1** บนเซิร์ฟเวอร์และในการตั้งค่าแอป<br>
- ใช้ **ตัวเลือกไฟล์ระบบ**:<br>
  - ไปที่ **ไฟล์ในเครื่อง > ไฟล์บนอุปกรณ์นี้ > เปิดไฟล์...**<br>
  - เลือกโฟลเดอร์/ไฟล์โดยใช้เมนูดั้งเดิมของ Apple<br><br>

หรือเชื่อมต่อโดยใช้ **WebDAV** หรือ **DLNA** หาก NAS ของคุณรองรับ

{{% details title="วิธีอัพโหลดและจัดการเพลงใน iCloud คืออะไร?" closed="true" %}}
– **ฉันจะอัพโหลดเพลงไปยัง iCloud ได้อย่างไร?**  <br>
ไปที่ [https://www.icloud.com](https://www.icloud.com) ในเบราว์เซอร์ของคุณ สร้างโฟลเดอร์ และอัพโหลดไฟล์เพลงของคุณโดยตรงจาก Mac หรือ PC<br>

– **ฉันจะจัดการพื้นที่เก็บข้อมูล iCloud ได้อย่างไร?**  <br>
คุณมีสองตัวเลือก:  <br>
1. ใช้ [https://www.icloud.com](https://www.icloud.com) ในเบราว์เซอร์ของคุณเพื่อจัดระเบียบ อัพโหลด หรือลบไฟล์<br>  
2. ในแอป หลังจากเชื่อมต่อกับ iCloud ผ่าน **การเชื่อมต่อ → เชื่อมต่อกับพื้นที่เก็บข้อมูลคลาวด์ → iCloud Drive** ใช้ตัวจัดการไฟล์ในตัวเพื่ออัพโหลด ดาวน์โหลด เปลี่ยนชื่อโฟลเดอร์ หรือลบไฟล์โดยตรงในพื้นที่เก็บข้อมูล iCloud ของคุณ โดยไม่ต้องบันทึกลงในอุปกรณ์ของคุณ<br>

⚠️ ระวังเมื่อลบไฟล์ แอปจะลบไฟล์อย่างถาวร (ไฟล์จะไม่ไปที่ถังขยะและไม่สามารถกู้คืนได้)<br>

เรียนรู้เพิ่มเติมที่นี่: [วิธีสตรีมเพลงจาก iCloud Drive บน iPhone หรือ Mac ของฉัน](/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/)

{{% details title="วิธีถ่ายโอนคลังเพลง 10 GB จาก Windows 11 ไปยัง iPhone เพื่อเล่นออฟไลน์คืออะไร?" closed="true" %}}

คุณมีตัวเลือกที่เชื่อถือได้หลายอย่างในการย้ายคลังเพลงของคุณจาก PC Windows 11 ไปยัง iPhone และใช้งานออฟไลน์ในแอป เลือกวิธีที่เหมาะกับคุณที่สุด:

1. **การใช้การเชื่อมต่อสาย (แนะนำสำหรับคลังขนาดใหญ่)**  <br>
   ใช้แอป **Apple Devices** บน Windows 11 เพื่อถ่ายโอนไฟล์ไปยัง iPhone โดยตรงผ่าน USB  
   ทำตามคู่มือของ Apple ที่นี่:  
   https://support.apple.com/en-ph/120402 <br>

2. **ไร้สายผ่าน Wi-Fi Drive**  <br>
   เปิดใช้งานฟีเจอร์ **WiFi Drive** ในแอปและอัพโหลดเพลงของคุณผ่านเบราว์เซอร์บนคอมพิวเตอร์ของคุณ  
   คำแนะนำทีละขั้นตอนที่นี่:  
   [วิธีถ่ายโอนไฟล์เพลงจากคอมพิวเตอร์ไปยัง iPhone โดยไม่ใช้ iTunes ด้วย WiFi-Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/) <br>

3. **ไร้สายโดยใช้เซิร์ฟเวอร์ DLNA**  <br>
   เปิดเซิร์ฟเวอร์สื่อ DLNA บนคอมพิวเตอร์ Windows ของคุณและสตรีมหรือถ่ายโอนคลังของคุณไปยังแอปโดยตรง  
   คู่มือ:  
   [วิธีเปิดใช้งานเซิร์ฟเวอร์สื่อ DLNA บน Windows 10 และเล่นเพลงบน iPhone](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/) <br>

4. **ไร้สายโดยใช้การแชร์ไฟล์ SMB**  <br>
   เปิดใช้งาน **การแชร์ไฟล์ SMB** บน PC และเชื่อมต่อกับมันในแอปเพื่อเรียกดู ดาวน์โหลด หรือถ่ายโอนไฟล์ทีละโฟลเดอร์  
   คำแนะนำ:  
   [ถ่ายโอนไฟล์จากคอมพิวเตอร์ไปยัง iPhone โดยใช้โปรโตคอล SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/) <br>

⚠️ เมื่อถ่ายโอนคลังขนาดใหญ่ (10 GB+) การถ่ายโอน USB แบบมีสายมักจะเป็นตัวเลือกที่เร็วที่สุดและเสถียรที่สุด

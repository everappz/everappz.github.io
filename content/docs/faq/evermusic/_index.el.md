Το Evermusic είναι ένα cloud music player για iPhone, iPad και Mac που μεταδίδει και κατεβάζει τραγούδια από iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, WebDAV, SMB, DLNA, συσκευές NAS και δεκάδες άλλες. Αυτές οι Συχνές Ερωτήσεις απαντούν τις πιο συχνές ερωτήσεις χρηστών σχετικά με υποστηριζόμενες μορφές και πρωτόκολλα, λειτουργίες Premium, αναπαραγωγή εκτός σύνδεσης, CarPlay, widgets, αντίγραφα ασφαλείας και αντιμετώπιση προβλημάτων. Για οδηγούς βήμα-προς-βήμα, δείτε τον [Οδηγό Χρήστη Evermusic](/docs/guide/evermusic/) ή περιηγηθείτε στα [άρθρα Πώς να](/docs/howto/).

{{% details title="Τι είναι το Evermusic;" closed="true" %}}
Το Evermusic είναι ένα **cloud music player** για iPhone, iPad και Mac που σας επιτρέπει να μεταδίδετε, να κατεβάζετε και να οργανώνετε κομμάτια από πολλές υπηρεσίες cloud, συσκευές NAS και τοπική αποθήκευση — όλα από μια ενιαία, ενοποιημένη βιβλιοθήκη.<br><br>

Μπορείτε εύκολα να κατεβάσετε μουσική για αναπαραγωγή εκτός σύνδεσης, να δημιουργείτε και να διαχειρίζεστε λίστες αναπαραγωγής και να χρησιμοποιείτε ενσωματωμένο equalizer για να βελτιώσετε την εμπειρία ακρόασής σας. Η εφαρμογή λειτουργεί με υπηρεσίες όπως **Google Drive, Dropbox, OneDrive, iCloud Drive, MEGA, Box, pCloud, Yandex Disk** και πολλά άλλα, ώστε να μπορείτε να διατηρείτε όλη τη μουσική σας σε ένα μέρος και να έχετε πρόσβαση από οποιαδήποτε συσκευή.<br><br>

Το Evermusic υποστηρίζει ευρύ φάσμα μορφών ήχου και σας επιτρέπει να οργανώνετε τη μουσική βιβλιοθήκη σας ανά **καλλιτέχνη, άλμπουμ, είδος και συνθέτη**. Μπορείτε να συγχρονίζετε τη βιβλιοθήκη σας μεταξύ cloud αποθήκευσης και της συσκευής σας, εξασφαλίζοντας ότι τα αγαπημένα σας τραγούδια είναι πάντα διαθέσιμα — ακόμη και σε πτήση ή σε λειτουργία αεροπλάνου.<br><br>

Με λειτουργίες όπως **αδιάλειπτη αναπαραγωγή, crossfade, audio equalizer, στίχοι, widgets, Apple CarPlay, AirPlay και Google Chromecast streaming**, το Evermusic είναι μια ολοκληρωμένη μουσική λύση για κάθε χρήστη iPhone, iPad ή Mac που διατηρεί μουσική στο cloud ή σε προσωπικό NAS.

{{% details title="Πώς λειτουργεί το Evermusic;" closed="true" %}}
**Το Evermusic λειτουργεί συνδέοντας τους λογαριασμούς cloud αποθήκευσης και το προσωπικό σας NAS σε μια ενιαία, αναζητήσιμη μουσική βιβλιοθήκη στη συσκευή σας.** Αφού συνδέσετε έναν λογαριασμό, μπορείτε να περιηγείστε και να μεταδίδετε κομμάτια απευθείας από το cloud ή να κατεβάζετε αγαπημένα τραγούδια, άλμπουμ και λίστες αναπαραγωγής για αναπαραγωγή εκτός σύνδεσης.<br><br>

Η εφαρμογή υποστηρίζει πολλαπλές μορφές ήχου, καθιστώντας εύκολη την αναπαραγωγή οποιουδήποτε αρχείου μουσικής που έχετε αποθηκεύσει — είτε είναι MP3, FLAC, ALAC, AAC, WAV, AIFF ή μία από δεκάδες άλλες μορφές. Οι συνδέσεις cloud χρησιμοποιούν **επίσημα SDK και OAuth** ροές, ώστε ο κωδικός σας να μην φτάνει ποτέ στην εφαρμογή και όλες οι μεταφορές να είναι κρυπτογραφημένες.<br><br>

Εκτός από ενσωμάτωση cloud, το Evermusic προσφέρει ισχυρές λειτουργίες για βελτίωση της εμπειρίας ακρόασής σας. Μπορείτε να δημιουργείτε και να διαχειρίζεστε λίστες αναπαραγωγής, να χρησιμοποιείτε τον ενσωματωμένο equalizer για προσαρμογή της ποιότητας ήχου και να απολαμβάνετε ομαλή αναπαραγωγή με **αδιάλειπτες** και **crossfade** επιλογές. Η εφαρμογή μεταδίδει επίσης σε συσκευές όπως **Chromecast** και **AirPlay**, επιτρέποντάς σας να παίζετε μουσική σε οποιοδήποτε συμβατό ηχείο, τηλεόραση ή hi-fi σύστημα.<br><br>

Μπορείτε να οργανώνετε τη μουσική βιβλιοθήκη σας ανά **καλλιτέχνη, άλμπουμ, είδος και συνθέτη** και να τη διατηρείτε συγχρονισμένη μεταξύ cloud αποθήκευσης και συσκευών, εξασφαλίζοντας ότι έχετε πάντα πρόσβαση στα αγαπημένα σας τραγούδια. Ένας ενσωματωμένος αναγνώστης μεταδεδομένων σαρώνει κομμάτια στο παρασκήνιο, διαβάζει ετικέτες και τα ομαδοποιεί αυτόματα καθώς προστίθενται νέα αρχεία.<br><br>

Επιπλέον, το Evermusic αναπαράγει τοπικά αρχεία που είναι αποθηκευμένα απευθείας στο **iPhone, iPad ή Mac** σας. Μπορείτε να εισάγετε μουσική από την αποθήκευση της συσκευής σας ή από τη βιβλιοθήκη iTunes / Apple Music. Αυτό σημαίνει ότι μπορείτε να απολαμβάνετε ολόκληρη τη συλλογή μουσικής σας σε ένα μέρος — cloud, NAS, USB, ή τοπική αποθήκευση — όλα από μια εφαρμογή, σε μια οθόνη.<br><br>

{{% details title="Είναι δωρεάν το Evermusic;" closed="true" %}}
**Ναι — το Evermusic είναι δωρεάν εφαρμογή** με ορισμένους περιορισμούς που μπορείτε να αφαιρέσετε αναβαθμίζοντας στην Premium έκδοση.<br><br>

Η εφαρμογή προσφέρει **εφάπαξ αγορά δια βίου** και **δύο επιλογές συνδρομής (1 μήνας και 1 χρόνος)** για την αφαίρεση όλων των περιορισμών, ώστε να επιλέξετε την τιμή και τη δέσμευση που σας ταιριάζει. Οι τιμές ενδέχεται να διαφέρουν ανάλογα με τη χώρα ή την περιοχή σας. Επίσης, λάβετε υπόψη ότι το **Family Sharing** είναι ενεργοποιημένο για όλες τις αγορές και τα προγράμματα, ώστε να μπορείτε να μοιράζεστε την Premium έκδοση με έως πέντε άλλα μέλη της οικογένειάς σας χωρίς επιπλέον κόστος.<br><br>

Οι αγορές δια βίου και οι συνδρομές **μοιράζονται μεταξύ iOS και Mac**, χρησιμοποιώντας iCloud για συγχρονισμό αυτών των πληροφοριών. Εάν έχετε ήδη την Premium έκδοση στη συσκευή σας iOS, βεβαιωθείτε ότι έχετε εγκαταστήσει την πιο πρόσφατη έκδοση και ότι το iCloud είναι ενεργοποιημένο. Εκκινήστε την εφαρμογή σε iOS και περιμένετε περίπου ένα λεπτό για να ανεβούν οι πληροφορίες αγοράς στο iCloud, στη συνέχεια εκκινήστε την έκδοση Mac ενώ είστε συνδεδεμένοι με το ίδιο Apple ID — το Premium ενεργοποιείται αυτόματα.<br><br>

Εάν η ενεργοποίηση δεν γίνει αυτόματα, ανοίξτε **Ρυθμίσεις → Αγορές → Επαναφορά αγορών** για να ανακτήσετε χειροκίνητα τα στοιχεία αγοράς από το App Store.<br><br>

{{% details title="Ποια είναι η διαφορά μεταξύ Evermusic Free και Evermusic Premium;" closed="true" %}}
**Το Evermusic Premium αφαιρεί κάθε περιορισμό της δωρεάν έκδοσης**: αφαιρεί τις διαφημίσεις, ξεκλειδώνει απεριόριστες λίστες αναπαραγωγής, υπηρεσίες cloud, αγαπημένα και offline λήψεις, ενεργοποιεί πλήρη εξατομίκευση και αφαιρεί τα ημερήσια όρια στις αυτόματες αναζητήσεις ετικετών και εξωφύλλων άλμπουμ και στις εξαγωγές συλλογών μέσων.<br><br>

**Evermusic Free**<br>
• **Περιέχει Διαφημίσεις:** Η δωρεάν έκδοση εμφανίζει διαφημίσεις που ενίοτε διακόπτουν την ακρόαση μουσικής.<br>
• **Περιορισμένες Λίστες Αναπαραγωγής:** Μπορείτε να δημιουργήσετε έως **10** λίστες αναπαραγωγής.<br>
• **Περιορισμένες Υπηρεσίες Cloud:** Η δωρεάν έκδοση υποστηρίζει έως **3** συνδεδεμένες υπηρεσίες cloud.<br>
• **Περιορισμένα Αγαπημένα:** Μπορείτε να προσθέσετε μέγιστο **100** τραγούδια στα αγαπημένα σας.<br>
• **Όρια Λίστας Αναπαραγωγής και Ουράς:** Η δωρεάν έκδοση περιορίζει τις λίστες αναπαραγωγής σε **1.000** τραγούδια και την ουρά αναπαραγωγής σε **750** τραγούδια.<br>
• **Περιορισμένες Λειτουργίες Εκτός Σύνδεσης:** Περιορισμός σε **1** φάκελο εκτός σύνδεσης· η λειτουργία εκτός σύνδεσης για άλμπουμ, καλλιτέχνες, είδη, λίστες αναπαραγωγής και συνθέτες είναι επίσης περιορισμένη.<br>
• **Χωρίς Εξαγωγή Συλλογών iTunes / Apple Music:** Δεν μπορείτε να εξάγετε συλλογές από iTunes / Apple Music στα Τοπικά Αρχεία στη δωρεάν έκδοση.<br>
• **Περιορισμένη Αναζήτηση Ετικετών:** Προσφέρει **20** αυτόματα αιτήματα αναζήτησης ετικετών και **20** αιτήματα αναζήτησης εξωφύλλου άλμπουμ ανά ημέρα.<br>
• **Περιορισμένες Εξαγωγές Συλλογών Μέσων:** Επιτρέπει **5** εξαγωγές συλλογών μέσων (λίστες αναπαραγωγής, άλμπουμ, καλλιτέχνες, είδη) σε M3U / CSV / TXT ανά ημέρα.<br>
• **Περιορισμένη Εξατομίκευση:** Διαθέσιμες μόνο περιορισμένες επιλογές εξατομίκευσης και η αλλαγή εικονιδίου εφαρμογής είναι περιορισμένη.<br><br>

**Evermusic Premium**<br>
• **Εμπειρία χωρίς Διαφημίσεις:** Το Premium εξαλείφει τις διαφημίσεις παρέχοντας απρόσκοπτη εμπειρία ακρόασης μουσικής.<br>
• **Απεριόριστες Λίστες Αναπαραγωγής:** Δημιουργήστε όσες λίστες αναπαραγωγής θέλετε.<br>
• **Απεριόριστες Υπηρεσίες Cloud:** Συνδεθείτε με απεριόριστο αριθμό υπηρεσιών cloud.<br>
• **Απεριόριστα Αγαπημένα:** Προσθέστε απεριόριστο αριθμό τραγουδιών στα αγαπημένα.<br>
• **Ευελιξία Λίστας Αναπαραγωγής και Ουράς:** Χωρίς όρια στο μέγεθος λίστας αναπαραγωγής ή ουράς.<br>
• **Εκτεταμένες Λειτουργίες Εκτός Σύνδεσης:** Λήψη φακέλων, άλμπουμ, καλλιτεχνών, ειδών, λιστών αναπαραγωγής και συνθετών εκτός σύνδεσης χωρίς περιορισμούς.<br>
• **Εξαγωγή Συλλογών iTunes / Apple Music:** Εξαγωγή συλλογών από iTunes / Apple Music στα Τοπικά Αρχεία.<br>
• **Απεριόριστη Αναζήτηση Ετικετών:** Χωρίς ημερήσιο όριο στα αυτόματα αιτήματα αναζήτησης ετικετών.<br>
• **Απεριόριστη Αναζήτηση Εξωφύλλου Άλμπουμ:** Χωρίς ημερήσιο όριο στα αιτήματα αναζήτησης εξωφύλλου άλμπουμ.<br>
• **Απεριόριστες Εξαγωγές Συλλογών Μέσων:** Απεριόριστες εξαγωγές M3U / CSV / TXT.<br>
• **Πλήρης Εξατομίκευση:** Όλες οι επιλογές εξατομίκευσης ξεκλειδωμένες, συμπεριλαμβανομένων εναλλακτικών εικονιδίων εφαρμογής.<br><br>

{{% details title="Ποια είναι η διαφορά μεταξύ Evermusic Free (μπλε εικονίδιο) και Evermusic Pro (κόκκινο εικονίδιο);" closed="true" %}}
**Το Evermusic Pro (κόκκινο εικονίδιο) έχει ακριβώς την ίδια λειτουργικότητα με το Evermusic Free (μπλε εικονίδιο) με ενεργοποιημένη Premium αγορά εντός εφαρμογής, ίδιες λειτουργίες, ίδια εμφάνιση. Οι διαφορές είναι η συσκευασία τους στο App Store, η συμβατότητα Mac, η τιμή και οι προεπιλογές απορρήτου.**<br><br>

**Συσκευασία App Store:**<br>
- Το **Evermusic Free (μπλε)** δημοσιεύεται ως **δύο ξεχωριστές εφαρμογές App Store** με **ξεχωριστά bundle ID** — μία για **iOS / iPadOS** και μία αφιερωμένη για **macOS** (Universal binary που τρέχει και σε **Apple Silicon και Intel Mac**). Οι αγορές σε μία από τις μπλε εφαρμογές **μοιράζονται με την άλλη μέσω iCloud** — αγοράστε Premium στο Evermusic Free σε iPhone και ξεκλειδώνει αυτόματα το Premium στο Evermusic Free σε Mac (και αντίστροφα), εφόσον και οι δύο συσκευές είναι συνδεδεμένες με το ίδιο Apple ID με ενεργοποιημένο iCloud.<br>
- Το **Evermusic Pro (κόκκινο)** είναι **μία εφαρμογή App Store** με **ένα bundle ID** που τρέχει σε **iPhone, iPad και Apple Silicon Mac (M1 και νεότερα)**. Επειδή υπάρχει μόνο ένα bundle, δεν χρειάζεται συγχρονισμός — μία αγορά καλύπτει κάθε υποστηριζόμενη συσκευή. Το Evermusic Pro **δεν υποστηρίζει Intel Mac**.<br><br>

**Βασικές Διαφορές:**<br>
- **Συμβατότητα Mac.** Η μπλε εφαρμογή Mac είναι Universal και λειτουργεί τόσο σε **Apple Silicon όσο και σε Intel Mac**. Η κόκκινη εφαρμογή είναι **μόνο για Apple Silicon (M1, M2, M3, M4, M5 και νεότερα)** — δεν υποστηρίζει Intel Mac.<br>
- **Τιμή.** Επειδή το Evermusic Pro (κόκκινο) παραλείπει το Intel Mac build, η τιμή στο App Store είναι **ελαφρώς χαμηλότερη** από την αντίστοιχη αγορά Premium εντός εφαρμογής στη μπλε εφαρμογή.<br>
- **Απόρρητο / analytics.** Το Evermusic Pro (κόκκινο) **δεν συλλέγει καθόλου διαγνωστικά ή analytics χρηστών** — τα analytics είναι πλήρως απενεργοποιημένα στο build, χωρίς επιλογή opt-in. Το μπλε Evermusic Free χρησιμοποιεί τυπικές προεπιλογές analytics που μπορείτε να απενεργοποιήσετε ανά πάσα στιγμή στις **Ρυθμίσεις → Analytics και Συλλογή Δεδομένων**.<br>
- **Αγορά μεταξύ εφαρμογών.** Τα bundle ID διαφέρουν μεταξύ μπλε και κόκκινου, οπότε μια αγορά Premium ενεργοποιημένη στη μπλε εφαρμογή **δεν ξεκλειδώνει δωρεάν την κόκκινη εφαρμογή** και αντίστροφα. Ο συγχρονισμός αγοράς λειτουργεί **εντός του ίδιου χρώματος** — μπλε iOS ↔ μπλε Mac μέσω iCloud, ή κόκκινο αυτόματα σε όλες τις υποστηριζόμενες συσκευές.<br><br>

**Τι να επιλέξω;**<br>
- **Evermusic Free (μπλε) + Premium** — καλύτερο για τη μεγαλύτερη κάλυψη συσκευών: iPhone, iPad, **Apple Silicon *και* Intel Mac**, με μία αγορά Premium που καλύπτει και τις δύο μπλε εφαρμογές iOS και Mac μέσω συγχρονισμού αγοράς iCloud.<br>
- **Evermusic Pro (κόκκινο)** — καλύτερο εάν χρησιμοποιείτε μόνο iPhone, iPad και Apple Silicon Mac (M1 ή νεότερο), προτιμάτε χαμηλότερη τιμή και θέλετε build με **χωρίς καθόλου συλλεγόμενα analytics ή διαγνωστικά** (χωρίς επιλογή opt-in).<br><br>

Εάν χρησιμοποιείτε ήδη τη μπλε εφαρμογή με ενεργοποιημένο Premium, **δεν χρειάζεται να εγκαταστήσετε την κόκκινη εφαρμογή** — έχετε ήδη όλα όσα προσφέρει το Pro.<br><br>

**Ναι — το Evermusic χρησιμοποιεί μόνο επίσημα SDK και ασφαλείς συνδέσεις για αλληλεπίδραση με συνδεδεμένες υπηρεσίες cloud. Το login και ο κωδικός σας δεν είναι ποτέ ορατά στην εφαρμογή και όλα τα αιτήματα είναι κρυπτογραφημένα.**<br><br>

Όταν εισάγετε το login και τον κωδικό σας, η εφαρμογή σας εμφανίζει την **επίσημη σελίδα εξουσιοδότησης που παρέχεται από την ίδια την υπηρεσία cloud**, και η ολόκληρη διαδικασία εξουσιοδότησης γίνεται εκτός του Evermusic. Ο πάροχος cloud στέλνει στη συνέχεια ένα **auth-token** πίσω στην εφαρμογή μετά από επιτυχημένη εξουσιοδότηση, και αυτό το token χρησιμοποιείται για όλες τις επακόλουθες κλήσεις API — το Evermusic δεν βλέπει ποτέ τα πραγματικά σας διαπιστευτήρια.<br><br>

Ένα auth-token είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με cloud αποθήκευση εκ μέρους σας. Το token αποθηκεύεται στη συσκευή σας στην **ασφαλή αποθήκευση συστήματος της Apple (Keychain)**, η οποία είναι κρυπτογραφημένη σε ηρεμία και προστατευμένη από τον κωδικό PIN ή βιομετρικό κλείδωμα. Μπορείτε να κατεβάσετε αρχεία από οποιαδήποτε συνδεδεμένη υπηρεσία cloud στη συσκευή σας· αυτά τα αρχεία τοποθετούνται στον κατάλογο **Documents** της εφαρμογής, όπου μπορείτε να τα διαχειριστείτε ή να τα αφαιρέσετε ανά πάσα στιγμή χρησιμοποιώντας τον ενσωματωμένο file manager.<br><br>

Η εφαρμογή **δεν μοιράζεται καμία πληροφορία** από τους συνδεδεμένους λογαριασμούς cloud σας με την Everappz, διαφημιστές ή τρίτους. Μπορείτε να ανακαλέσετε την πρόσβαση στον λογαριασμό cloud σας ανά πάσα στιγμή ανοίγοντας τη σελίδα ρυθμίσεων λογαριασμού στο πρόγραμμα περιήγησης ιστού σας. Για να ανακαλέσετε ένα auth-token, συνδεθείτε στον λογαριασμό cloud σας στο διαδίκτυο, μεταβείτε στη σελίδα ασφαλείας ή συνδεδεμένων εφαρμογών και αφαιρέστε την καταχώριση για το Evermusic.<br><br>

Μπορείτε επίσης να αποσυνδέσετε τον λογαριασμό cloud εντός της εφαρμογής — όταν το κάνετε, το auth-token διαγράφεται αμέσως από τη συσκευή σας. Εάν απεγκαταστήσετε την εφαρμογή από τη συσκευή σας, όλα τα ληφθέντα δεδομένα και tokens πρόσβασης αφαιρούνται αυτόματα μαζί της.<br><br>

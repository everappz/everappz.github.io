Evermusic 是一款適用於 iPhone、iPad 和 Mac 的雲端音樂播放器，可從 iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Box、pCloud、Yandex Disk、WebDAV、SMB、DLNA、NAS 裝置及數十種其他服務串流和下載歌曲。本 FAQ 解答使用者最常問到的關於支援的格式和協定、Premium 功能、離線播放、CarPlay、小工具、備份和疑難排解的問題。如需逐步操作指南，請參閱 [Evermusic 使用者指南](/docs/guide/evermusic/) 或瀏覽我們的 [操作指南文章](/docs/howto/)。

{{% details title="Evermusic 是什麼？" closed="true" %}}
Evermusic 是一款適用於 iPhone、iPad 和 Mac 的**雲端音樂播放器**，可讓您從多個雲端儲存服務、NAS 裝置和本機儲存空間串流、下載和整理音樂曲目——所有內容都在一個統一的音樂庫中。<br><br>

您可以輕鬆下載音樂進行離線播放，建立和管理播放清單，並使用內建等化器增強聆聽體驗。該應用程式支援 **Google Drive、Dropbox、OneDrive、iCloud Drive、MEGA、Box、pCloud、Yandex Disk** 等眾多服務，讓您可以將所有音樂儲存在一處並從任何裝置存取。<br><br>

Evermusic 支援多種音訊格式，可按**演出者、專輯、類型和作曲家**整理音樂庫。您可以在雲端儲存和裝置之間同步音樂庫，確保您最喜愛的音樂隨時可用——即使在飛行或飛航模式下也不例外。<br><br>

憑藉**無縫播放、交叉淡入淡出、音訊等化器、歌詞、小工具、Apple CarPlay、AirPlay 和 Google Chromecast 串流**等功能，Evermusic 是將音樂儲存在雲端或個人 NAS 上的任何 iPhone、iPad 或 Mac 使用者的完整音樂解決方案。
{{% /details %}}

{{% details title="Evermusic 如何運作？" closed="true" %}}
**Evermusic 透過將您的雲端儲存帳戶和個人 NAS 連接到裝置上的統一可搜尋音樂庫來運作。** 連接帳戶後，您可以直接從雲端瀏覽和串流曲目，或下載您喜愛的歌曲、專輯和播放清單進行離線播放。<br><br>

該應用程式支援多種音訊格式，可輕鬆播放您儲存的任何音樂檔案——無論是 MP3、FLAC、ALAC、AAC、WAV、AIFF 還是數十種其他格式。雲端連接使用**官方 SDK 和 OAuth**，因此您的密碼永遠不會進入應用程式，所有傳輸均經過加密。<br><br>

除雲端整合外，Evermusic 還提供強大功能來增強聆聽體驗。您可以建立和管理播放清單，使用內建等化器調整音質，並享受帶有**無縫**和**交叉淡入淡出**選項的流暢播放。該應用程式還可串流到 **Chromecast** 和 **AirPlay** 裝置，讓您在任何相容的喇叭、電視或音響系統上播放音樂。<br><br>

您可以按**演出者、專輯、類型和作曲家**整理音樂庫，並在雲端儲存和裝置之間保持同步。內建的中繼資料讀取器在背景掃描曲目，讀取標籤，並在新增檔案時自動對其分組。<br><br>

此外，Evermusic 還可以播放直接儲存在您的 **iPhone、iPad 或 Mac** 上的本機檔案。您可以從裝置儲存空間或 iTunes / Apple Music 資料庫匯入音樂。這意味著您可以在一處欣賞整個音樂收藏——雲端、NAS、USB 隨身碟或本機儲存空間——全部在一個應用程式、一個螢幕上。<br><br>

{{% details title="Evermusic 是免費的嗎？" closed="true" %}}
**是的——Evermusic 是一款免費應用程式**，有一些可透過升級到 Premium 版本來解除的限制。<br><br>

該應用程式提供**一次性終身應用程式內購買**和**兩種訂閱選項（1 個月和 1 年）**來解除所有限制，讓您選擇最適合自己的價格和承諾方式。價格可能因國家或地區而異。另外請注意，所有購買和方案均已啟用 **Family Sharing**，因此您最多可以與家庭中的五名其他成員共享 Premium 版本，無需額外費用。<br><br>

終身購買和訂閱在 **iOS 和 Mac 之間共享**，使用 iCloud 同步此資訊。如果您已在 iOS 裝置上購買了 Premium 版本，請確保已安裝最新版本並啟用了 iCloud。在 iOS 上啟動應用程式並等待約一分鐘，讓購買資訊上傳到 iCloud，然後在登入相同 Apple ID 的情況下啟動 Mac 版本——Premium 將自動啟用。<br><br>

{{% details title="Evermusic Free 和 Evermusic Premium 有什麼區別？" closed="true" %}}
**Evermusic Premium 解除免費版的所有限制**：去除廣告，解鎖無限播放清單、雲端服務、最愛項目和離線下載，啟用完整個人化，並取消自動標籤和專輯封面搜尋以及媒體收藏匯出的每日限制。<br><br>

**Evermusic Free**<br>
• **包含廣告：** 免費版會顯示廣告以產生收益，有時可能會打斷音樂聆聽。<br>
• **播放清單有限：** 最多可建立 **10** 個播放清單。<br>
• **雲端服務受限：** 免費版最多支援 **3** 個已連接的雲端服務。<br>
• **最愛項目有限：** 最多可將 **100** 首歌曲加入最愛項目。<br>
• **播放清單和佇列限制：** 免費版將播放清單限制為 **1,000** 首歌曲，播放器佇列限制為 **750** 首歌曲。<br>
• **離線功能有限：** 僅限 **1** 個離線資料夾；專輯、演出者、類型、播放清單和作曲家的離線模式也受到限制。<br>
• **無法匯出 iTunes / Apple Music 收藏：** 免費版無法將 iTunes / Apple Music 中的收藏匯出到本機檔案。<br>
• **標籤搜尋有限：** 每天提供 **20** 次自動標籤搜尋請求和 **20** 次專輯封面搜尋請求。<br>
• **媒體收藏匯出有限：** 每天允許將 **5** 個媒體收藏（播放清單、專輯、演出者、類型）匯出為 M3U / CSV / TXT。<br>
• **個人化有限：** 只有有限的個人化選項，更改應用程式圖示也受到限制。<br><br>

**Evermusic Premium**<br>
• **無廣告體驗：** Premium 消除廣告，提供流暢、不間斷的音樂聆聽體驗。<br>
• **無限播放清單：** 隨心建立任意數量的播放清單。<br>
• **無限雲端服務：** 連接無限數量的雲端服務。<br>
• **無限最愛項目：** 將無限數量的歌曲加入最愛項目。<br>
• **播放清單和佇列靈活性：** 播放清單或佇列大小無限制。<br>
• **全面離線功能：** 無限量下載資料夾、專輯、演出者、類型、播放清單和作曲家。<br>
• **iTunes / Apple Music 收藏匯出：** 將 iTunes / Apple Music 中的收藏匯出到本機檔案。<br>
• **無限標籤搜尋：** 自動標籤搜尋請求無每日限制。<br>
• **無限專輯封面搜尋：** 專輯封面搜尋請求無每日限制。<br>
• **無限媒體收藏匯出：** 無限 M3U / CSV / TXT 匯出。<br>
• **完整個人化：** 所有個人化選項已解鎖，包括替代應用程式圖示。<br><br>

如果您還想瞭解**紅色圖示的 Evermusic Pro**，請參閱下一個問題——它具有與 Evermusic Free + Premium 相同的功能，但 Mac 相容性和隱私權預設設定不同。<br><br>

{{% details title="Evermusic Free（藍色圖示）和 Evermusic Pro（紅色圖示）有什麼區別？" closed="true" %}}
**Evermusic Pro（紅色圖示）的功能與啟用了 Premium 應用程式內購買的 Evermusic Free（藍色圖示）完全相同，相同的功能，相同的外觀。區別在於 App Store 上的封裝方式、Mac 相容性、價格和預設隱私權設定。**<br><br>

**App Store 封裝：**<br>
- **Evermusic Free（藍色）** 以**兩個獨立的 App Store 應用程式**發佈，具有**獨立的 bundle ID**——一個用於 **iOS / iPadOS**，另一個專用於 **macOS**（Universal 二進位檔，可在 **Apple Silicon 和 Intel Mac** 上執行）。在任一藍色應用程式中進行的購買**透過 iCloud 與另一個共享**——在 iPhone 上的 Evermusic Free 中購買 Premium 會自動解鎖 Mac 上 Evermusic Free 的 Premium（反之亦然），只要兩台裝置登入相同的 Apple ID 並啟用了 iCloud。<br>
- **Evermusic Pro（紅色）** 是一個具有**單一 bundle ID** 的**單一 App Store 應用程式**，可在 **iPhone、iPad 和 Apple Silicon Mac（M1 及更高版本）** 上執行。由於只有一個 bundle，無需同步——一次購買涵蓋所有受支援的裝置。Evermusic Pro **不支援 Intel Mac**。<br><br>

**主要區別：**<br>
- **Mac 相容性。** 藍色 Mac 應用程式是 Universal 版本，可在 **Apple Silicon 和 Intel Mac 上執行**。紅色應用程式**僅支援 Apple Silicon（M1、M2、M3、M4、M5 及更高版本）**——不支援 Intel Mac。<br>
- **價格。** 由於 Evermusic Pro（紅色）略過了 Intel Mac 版本，其 App Store 價格**略低於**藍色應用程式中等效的 Premium 應用程式內購買。<br>
- **隱私權 / 分析。** Evermusic Pro（紅色）**完全不收集任何使用者診斷或分析資料**——分析在版本中完全停用，沒有選擇加入選項。藍色 Evermusic Free 使用標準分析預設設定，您可以隨時在**設定 → 分析和資料收集**中關閉。<br>
- **跨應用程式購買。** 藍色和紅色之間的 bundle ID 不同，因此在藍色應用程式中啟用的 Premium 購買**不會免費解鎖紅色應用程式**，反之亦然。購買同步在**相同顏色內**運作——藍色 iOS ↔ 藍色 Mac 透過 iCloud，或紅色自動跨所有受支援的裝置。<br><br>

**我應該選擇哪個？**<br>
- **Evermusic Free（藍色）+ Premium** ——最適合最廣泛的裝置覆蓋：iPhone、iPad、**Apple Silicon *和* Intel Mac**，一次 Premium 購買透過 iCloud 購買同步涵蓋 iOS 和 Mac 藍色應用程式。<br>
- **Evermusic Pro（紅色）** ——最適合只使用 iPhone、iPad 和 Apple Silicon Mac（M1 或更高版本）、希望價格更低，並且想要**完全不收集分析或診斷資料**（沒有選擇加入選項）的版本。<br><br>

如果您已在使用啟用了 Premium 的藍色應用程式，則**無需安裝紅色應用程式**——您已經擁有 Pro 提供的一切。<br><br>

{{% details title="Evermusic 安全嗎？" closed="true" %}}
**是的——Evermusic 僅使用官方 SDK 和安全連接與已連接的雲端服務互動。您的登入名稱和密碼對應用程式不可見，所有請求均經過加密。**<br><br>

當您輸入登入名稱和密碼時，應用程式會向您顯示**由雲端服務本身提供的官方授權頁面**，整個授權過程在 Evermusic 之外進行。成功授權後，雲端提供者會將 **auth-token** 發送回應用程式，該令牌用於所有後續 API 呼叫——Evermusic 永遠看不到您的真實憑證。<br><br>

auth-token 是一個數位金鑰，允許第三方應用程式代表您與雲端儲存互動。令牌儲存在您裝置上的 **Apple 安全系統儲存空間（Keychain）** 中，該儲存空間在靜態時加密，並受您的裝置密碼或生物辨識鎖保護。您可以從任何已連接的雲端服務下載檔案到您的裝置；這些檔案放置在應用程式的 **Documents** 目錄中，您隨時可以使用內建檔案管理員管理或刪除它們。<br><br>

該應用程式**不會將您已連接的雲端帳戶中的任何資訊**分享給 Everappz、廣告商或任何第三方。您可以隨時透過在網路瀏覽器中開啟帳戶設定頁面來撤銷對雲端帳戶的存取權限。要撤銷 auth-token，請在網路上登入您的雲端帳戶，瀏覽到安全性或已連接應用程式頁面，並刪除 Evermusic 的項目。<br><br>

您也可以在應用程式內部中斷雲端帳戶連接——這樣做時，auth-token 會立即從您的裝置中刪除。如果您從裝置上解除安裝應用程式，所有下載的資料和存取令牌都會隨之自動刪除。<br><br>

{{% details title="如何在 Evermusic 中建立播放清單？" closed="true" %}}
**要建立播放清單，開啟播放清單標籤頁，點選「+」或「...」選單，選擇「新增播放清單」，輸入名稱，然後選擇要加入的曲目。**<br><br>

- 從主導覽開啟**播放清單**部分。<br>
- 點選右上角的 **+** 或 **「...」** 按鈕，選擇**新增播放清單**。<br>
- 輸入播放清單名稱並點選**儲存**——**加入歌曲**對話方塊會自動出現。<br>
- 從以下四個來源之一選擇曲目：<br>
  - **音樂庫** ——已匯入您音樂庫的曲目。<br>
  - **此應用程式中的檔案** ——儲存在應用程式 Documents 目錄中的音訊檔案。<br>
  - **此 iPhone / iPad / Mac 上的檔案** ——位於裝置上應用程式以外的其他資料夾中的音訊檔案。<br>
  - **連接** ——儲存在已連接雲端儲存服務中的線上檔案。<br>
- 點選**完成**儲存您的選擇。<br><br>

預設情況下，每首歌曲只能加入播放清單一次。要允許重複歌曲，請啟用**設定 → 音樂庫 → 播放清單 → 播放清單中的重複項目 → 啟用**。稍後可以更改歌曲順序、重新命名播放清單、編輯其封面圖片，並將其匯出為 M3U、CSV 或 TXT。<br><br>

[閱讀完整播放清單指南](/docs/guide/evermusic/evermusic-guide-playlists/)

{{% details title="如何將 M3U 播放清單匯入 Evermusic？" closed="true" %}}
**要匯入 M3U 播放清單，開啟播放清單部分，點選「...」選單，選擇「匯入播放清單」，然後從雲端儲存或裝置中選擇 .m3u、.m3u8 或 .cue 檔案。**<br><br>

應用程式解析播放清單檔案，在您的儲存空間中定位每個參照的曲目，並在您的音樂庫中建立真正的播放清單。支援的副檔名為 **M3U**、**M3U8** 和 **CUE**。確保播放清單檔案中的路徑與音訊檔案實際儲存位置相符。<br><br>

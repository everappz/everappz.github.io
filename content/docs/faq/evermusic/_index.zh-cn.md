{{% details title="Evermusic 是什么？" closed="true" %}}
Evermusic 是一款适用于 iPhone、iPad 和 Mac 的**云端音乐播放器**，可让您从多个云存储服务、NAS 设备和本地存储流式播放、下载和整理音乐曲目——所有内容都在一个统一的音乐库中。<br><br>

您可以轻松下载音乐进行离线播放，创建和管理播放列表，并使用内置均衡器增强聆听体验。该应用支持 **Google Drive、Dropbox、OneDrive、iCloud Drive、MEGA、Box、pCloud、Yandex Disk** 等众多服务，让您可以将所有音乐保存在一处并从任何设备访问。<br><br>

Evermusic 支持多种音频格式，可按**艺术家、专辑、流派和作曲家**整理音乐库。您可以在云存储和设备之间同步音乐库，确保您最喜爱的音乐随时可用——即使在飞行或飞行模式下也不例外。<br><br>

凭借**无缝播放、交叉淡入淡出、音频均衡器、歌词、小组件、Apple CarPlay、AirPlay 和 Google Chromecast 流媒体**等功能，Evermusic 是将音乐存储在云端或个人 NAS 上的任何 iPhone、iPad 或 Mac 用户的完整音乐解决方案。
{{% /details %}}

{{% details title="Evermusic 如何工作？" closed="true" %}}
**Evermusic 通过将您的云存储账户和个人 NAS 连接到设备上的统一可搜索音乐库来工作。** 连接账户后，您可以直接从云端浏览和流式播放曲目，或下载您喜爱的歌曲、专辑和播放列表进行离线播放。<br><br>

该应用支持多种音频格式，可轻松播放您存储的任何音乐文件——无论是 MP3、FLAC、ALAC、AAC、WAV、AIFF 还是数十种其他格式。云连接使用**官方 SDK 和 OAuth**，因此您的密码永远不会进入应用程序，所有传输均经过加密。<br><br>

除云端整合外，Evermusic 还提供强大功能来增强聆听体验。您可以创建和管理播放列表，使用内置均衡器调整音质，并享受带有**无缝**和**交叉淡入淡出**选项的流畅播放。该应用还可流媒体到 **Chromecast** 和 **AirPlay** 设备，让您在任何兼容的扬声器、电视或音响系统上播放音乐。<br><br>

您可以按**艺术家、专辑、流派和作曲家**整理音乐库，并在云存储和设备之间保持同步。内置的元数据读取器在后台扫描曲目，读取标签，并在添加新文件时自动对其分组。<br><br>

此外，Evermusic 还可以播放直接存储在您的 **iPhone、iPad 或 Mac** 上的本地文件。您可以从设备存储或 iTunes / Apple Music 库导入音乐。这意味着您可以在一处欣赏整个音乐收藏——云端、NAS、USB 驱动器或本地存储——全部在一个应用程序、一个屏幕上。<br><br>

{{% details title="Evermusic 是免费的吗？" closed="true" %}}
**是的——Evermusic 是一款免费应用程序**，有一些可通过升级到 Premium 版本来解除的限制。<br><br>

该应用提供**一次性终身应用内购买**和**两种订阅选项（1 个月和 1 年）**来解除所有限制，让您选择最适合自己的价格和承诺方式。价格可能因国家或地区而异。另外请注意，所有购买和套餐均已启用 **Family Sharing**，因此您最多可以与家庭中的五名其他成员共享 Premium 版本，无需额外费用。<br><br>

终身购买和订阅在 **iOS 和 Mac 之间共享**，使用 iCloud 同步此信息。如果您已在 iOS 设备上购买了 Premium 版本，请确保已安装最新版本并启用了 iCloud。在 iOS 上启动应用并等待约一分钟，让购买信息上传到 iCloud，然后在登录相同 Apple ID 的情况下启动 Mac 版本——Premium 将自动激活。<br><br>

{{% details title="Evermusic Free 和 Evermusic Premium 有什么区别？" closed="true" %}}
**Evermusic Premium 解除免费版的所有限制**：去除广告，解锁无限播放列表、云服务、收藏夹和离线下载，启用完整个性化，并取消自动标签和专辑封面搜索以及媒体收藏导出的每日限制。<br><br>

**Evermusic Free**<br>
• **包含广告：** 免费版会显示广告以产生收入，有时可能会打断音乐聆听。<br>
• **播放列表有限：** 最多可创建 **10** 个播放列表。<br>
• **云服务受限：** 免费版最多支持 **3** 个连接的云服务。<br>
• **收藏夹有限：** 最多可将 **100** 首歌曲添加到收藏夹。<br>
• **播放列表和队列限制：** 免费版将播放列表限制为 **1,000** 首歌曲，播放器队列限制为 **750** 首歌曲。<br>
• **离线功能有限：** 仅限 **1** 个离线文件夹；专辑、艺术家、流派、播放列表和作曲家的离线模式也受到限制。<br>
• **无法导出 iTunes / Apple Music 收藏：** 免费版无法将 iTunes / Apple Music 中的收藏导出到本地文件。<br>
• **标签搜索有限：** 每天提供 **20** 次自动标签搜索请求和 **20** 次专辑封面搜索请求。<br>
• **媒体收藏导出有限：** 每天允许将 **5** 个媒体收藏（播放列表、专辑、艺术家、流派）导出为 M3U / CSV / TXT。<br>
• **个性化有限：** 只有有限的个性化选项，更改应用图标也受到限制。<br><br>

**Evermusic Premium**<br>
• **无广告体验：** Premium 消除广告，提供流畅、不间断的音乐聆听体验。<br>
• **无限播放列表：** 随心创建任意数量的播放列表。<br>
• **无限云服务：** 连接无限数量的云服务。<br>
• **无限收藏夹：** 将无限数量的歌曲添加到收藏夹。<br>
• **播放列表和队列灵活性：** 播放列表或队列大小无限制。<br>
• **全面离线功能：** 无限量下载文件夹、专辑、艺术家、流派、播放列表和作曲家。<br>
• **iTunes / Apple Music 收藏导出：** 将 iTunes / Apple Music 中的收藏导出到本地文件。<br>
• **无限标签搜索：** 自动标签搜索请求无每日限制。<br>
• **无限专辑封面搜索：** 专辑封面搜索请求无每日限制。<br>
• **无限媒体收藏导出：** 无限 M3U / CSV / TXT 导出。<br>
• **完整个性化：** 所有个性化选项已解锁，包括替代应用图标。<br><br>

如果您还想了解**红色图标的 Evermusic Pro**，请参阅下一个问题——它具有与 Evermusic Free + Premium 相同的功能，但 Mac 兼容性和隐私默认设置不同。<br><br>

{{% details title="Evermusic Free（蓝色图标）和 Evermusic Pro（红色图标）有什么区别？" closed="true" %}}
**Evermusic Pro（红色图标）的功能与激活了 Premium 应用内购买的 Evermusic Free（蓝色图标）完全相同，相同的功能，相同的外观。区别在于 App Store 上的打包方式、Mac 兼容性、价格和默认隐私设置。**<br><br>

**App Store 打包：**<br>
- **Evermusic Free（蓝色）** 以**两个独立的 App Store 应用**发布，具有**独立的 bundle ID**——一个用于 **iOS / iPadOS**，另一个专用于 **macOS**（Universal 二进制文件，可在 **Apple Silicon 和 Intel Mac** 上运行）。在任一蓝色应用中进行的购买**通过 iCloud 与另一个共享**——在 iPhone 上的 Evermusic Free 中购买 Premium 会自动解锁 Mac 上 Evermusic Free 的 Premium（反之亦然），只要两台设备登录相同的 Apple ID 并启用了 iCloud。<br>
- **Evermusic Pro（红色）** 是一个具有**单一 bundle ID** 的**单个 App Store 应用**，可在 **iPhone、iPad 和 Apple Silicon Mac（M1 及更高版本）** 上运行。由于只有一个 bundle，无需同步——一次购买涵盖所有受支持的设备。Evermusic Pro **不支持 Intel Mac**。<br><br>

**主要区别：**<br>
- **Mac 兼容性。** 蓝色 Mac 应用是 Universal 版本，可在 **Apple Silicon 和 Intel Mac 上运行**。红色应用**仅支持 Apple Silicon（M1、M2、M3、M4、M5 及更高版本）**——不支持 Intel Mac。<br>
- **价格。** 由于 Evermusic Pro（红色）跳过了 Intel Mac 版本，其 App Store 价格**略低于**蓝色应用中等效的 Premium 应用内购买。<br>
- **隐私 / 分析。** Evermusic Pro（红色）**完全不收集任何用户诊断或分析数据**——分析在版本中完全禁用，没有选择加入选项。蓝色 Evermusic Free 使用标准分析默认设置，您可以随时在**设置 → 分析和数据收集**中关闭。<br>
- **跨应用购买。** 蓝色和红色之间的 bundle ID 不同，因此在蓝色应用中激活的 Premium 购买**不会免费解锁红色应用**，反之亦然。购买同步在**相同颜色内**工作——蓝色 iOS ↔ 蓝色 Mac 通过 iCloud，或红色自动跨所有受支持的设备。<br><br>

**我应该选择哪个？**<br>
- **Evermusic Free（蓝色）+ Premium** ——最适合最广泛的设备覆盖：iPhone、iPad、**Apple Silicon *和* Intel Mac**，一次 Premium 购买通过 iCloud 购买同步涵盖 iOS 和 Mac 蓝色应用。<br>
- **Evermusic Pro（红色）** ——最适合只使用 iPhone、iPad 和 Apple Silicon Mac（M1 或更高版本）、希望价格更低，并且想要**完全不收集分析或诊断数据**（没有选择加入选项）的版本。<br><br>

如果您已在使用激活了 Premium 的蓝色应用，则**无需安装红色应用**——您已经拥有 Pro 提供的一切。<br><br>

{{% details title="Evermusic 安全吗？" closed="true" %}}
**是的——Evermusic 仅使用官方 SDK 和安全连接与已连接的云服务交互。您的登录名和密码对应用程序不可见，所有请求均经过加密。**<br><br>

当您输入登录名和密码时，应用程序会向您显示**由云服务本身提供的官方授权页面**，整个授权过程在 Evermusic 之外进行。成功授权后，云提供商会将 **auth-token** 发送回应用程序，该令牌用于所有后续 API 调用——Evermusic 永远看不到您的真实凭据。<br><br>

auth-token 是一个数字密钥，允许第三方应用程序代表您与云存储交互。令牌存储在您设备上的 **Apple 安全系统存储（Keychain）** 中，该存储在静态时加密，并受您的设备密码或生物识别锁保护。您可以从任何已连接的云服务下载文件到您的设备；这些文件放置在应用程序的 **Documents** 目录中，您随时可以使用内置文件管理器管理或删除它们。<br><br>

该应用程序**不会将您已连接的云账户中的任何信息**分享给 Everappz、广告商或任何第三方。您可以随时通过在网络浏览器中打开账户设置页面来撤销对云账户的访问权限。要撤销 auth-token，请在网络上登录您的云账户，导航到安全或已连接应用页面，并删除 Evermusic 的条目。<br><br>

您也可以在应用程序内部断开云账户连接——这样做时，auth-token 会立即从您的设备中删除。如果您从设备上卸载应用程序，所有下载的数据和访问令牌都会随之自动删除。<br><br>

{{% details title="如何在 Evermusic 中创建播放列表？" closed="true" %}}
**要创建播放列表，打开播放列表标签页，点击「+」或「...」菜单，选择「新建播放列表」，输入名称，然后选择要添加的曲目。**<br><br>

- 从主导航打开**播放列表**部分。<br>
- 点击右上角的 **+** 或 **「...」** 按钮，选择**新建播放列表**。<br>
- 输入播放列表名称并点击**保存**——**添加歌曲**对话框会自动出现。<br>
- 从以下四个来源之一选择曲目：<br>
  - **音乐库** ——已导入您音乐库的曲目。<br>
  - **此应用程序中的文件** ——存储在应用程序 Documents 目录中的音频文件。<br>
  - **此 iPhone / iPad / Mac 上的文件** ——位于设备上应用程序以外的其他文件夹中的音频文件。<br>
  - **连接** ——存储在已连接云存储服务中的在线文件。<br>
- 点击**完成**保存您的选择。<br><br>

默认情况下，每首歌曲只能添加到播放列表一次。要允许重复歌曲，请启用**设置 → 音乐库 → 播放列表 → 播放列表中的重复项 → 启用**。稍后可以更改歌曲顺序、重命名播放列表、编辑其封面图片，并将其导出为 M3U、CSV 或 TXT。<br><br>

{{% details title="如何将 M3U 播放列表导入 Evermusic？" closed="true" %}}

应用程序解析播放列表文件，在您的存储中定位每个引用的曲目，并在您的音乐库中创建真实的播放列表。支持的文件扩展名为 **M3U**、**M3U8** 和 **CUE**。确保播放列表文件中的路径与音频文件实际存储位置匹配。<br><br>

相同的操作也适用于专辑、艺术家、流派、「最近播放」列表和「收藏夹」列表。使用 M3U 重新导入其他播放器，CSV 用于电子表格，TXT 用于简单易读的副本。<br><br>

{{% details title="Evermusic 支持哪些云服务？" closed="true" %}}
**Evermusic 可连接到大多数流行的云存储提供商和个人 NAS 设备**——包括 iCloud Drive、Google Drive、Dropbox、OneDrive（个人版和商业版）、Box、MEGA、Yandex Disk、WD My Cloud Home、MediaFire、pCloud、HiDrive、百度网盘、InfiniCLOUD、Cloud Mail.ru、Put.io、MyDrive 等众多服务。<br><br>

除专用云服务外，Evermusic 还支持标准网络协议 **WebDAV**、**SMB**、**FTP / SFTP**、**NFS** 和 **DLNA / UPnP**。这意味着您几乎可以连接任何 NAS 设备——Synology、QNAP、WD My Cloud Home、Buffalo、Apple Time Capsule——以及普通的 Windows、macOS 或 Linux 文件共享、自托管的 Nextcloud / ownCloud 服务器，或 UPnP / DLNA 媒体服务器。<br><br>

每个连接都使用该服务的**官方 SDK 或开放协议**，在支持的地方使用基于 OAuth 的授权。您可以连接同一服务的多个账户（例如两个 Google Drive 账户），并在连接屏幕中并排浏览它们。Premium 用户可以添加无限数量的服务；免费版限制为三个。<br><br>

CarPlay 界面包含专用的**音乐库**、**连接**、**本地文件**和**设置**标签页，让您在驾驶途中完全控制音乐。播放控制——播放 / 暂停、下一首 / 上一首、**随机播放**、**重复**、队列管理和**音频均衡器**——都可以直接从 CarPlay 屏幕上获得，因此驾驶时您不需要触碰手机。<br><br>

要使用 CarPlay，请确保在 iPhone 上启用了 Siri，然后通过 **USB 或无线 CarPlay** 将 iPhone 连接到您的车辆。一旦汽车显示 CarPlay 主屏幕，点击 Evermusic 图标启动应用程序。<br><br>

您可以在**设置 → CarPlay** 中微调 CarPlay 体验——为所有 CarPlay 列表设置排序选项，选择是否在 CarPlay 屏幕上使用分页，更改菜单图标渐变颜色，启用或禁用图像以在大型音乐库上加快加载速度，以及开启**连接时暂停播放**，避免 iPhone 连接到汽车时突然出现大声的音频。<br><br>

[阅读完整指南](/docs/howto/how-to-play-your-own-music-on-iphone-using-carplay/)

{{% details title="如何将 Evermusic 的音乐投射到 Google Chromecast 或 AirPlay？" closed="true" %}}
**在 iPhone 或 iPad 上，打开音频播放器，点击 AirPlay 或 Chromecast 图标，然后从列表中选择您的扬声器或电视。**<br><br>

iOS 上同时支持 AirPlay 和 Google Chromecast。这些按钮可通过**设置 → 音频播放器 → 个性化 → 主屏幕操作**添加到主播放器屏幕。向 AirPlay 或 Chromecast 流媒体时，交叉淡入淡出不可用。AirPlay 在 macOS 上也可通过系统控件原生工作。
{{% /details %}}

{{% details title="如何在 iPhone 主屏幕或锁定屏幕上启用 Evermusic 小组件？" closed="true" %}}
**在设置 → 小组件中启用小组件更新，然后长按主屏幕或锁定屏幕，点击「+」，搜索「Evermusic」，然后选择小组件大小。**<br><br>

小组件显示当前播放的曲目，包含封面和基本控制。由于小组件刷新会消耗少量电量，**启用小组件**开关默认关闭——只有在您积极使用小组件时才开启。有关主屏幕和锁定屏幕集成的更多信息，请参阅[导航指南](/docs/guide/evermusic/evermusic-guide-navigation/)。
{{% /details %}}

{{% details title="如何在 Evermusic 中编辑 MP3 标签？" closed="true" %}}
**点击任何曲目上的「...」，选择「编辑音频标签」打开内置标签编辑器——您可以更改标题、艺术家、专辑、年份、流派、歌词、专辑封面等。**<br><br>

您可以在**设置 → 音频标签编辑器**中微调编辑器——包括专辑封面缩放、是否将更改写回云文件，以及编辑器主屏幕上显示哪些按钮。要一次性批量编辑多个文件，请安装我们专门用于标签编辑和音乐库整理的配套应用 **Evertag**。[了解更多关于 Evertag 的信息](/products/evertag/)。
{{% /details %}}

{{% details title="如何修复 Evermusic 中缺失的专辑封面？" closed="true" %}}
**打开设置 → 音乐库 → 专辑封面，启用「加载在线文件的专辑封面」和「在文件夹中搜索」，然后根据需要选择更高的封面质量——应用程序将从您的云文件中提取内嵌插图，并在没有内嵌封面时使用与音频文件一起存储的 JPEG / PNG 图像。**<br><br>

您也可以选择专辑封面质量并点击**全部删除**清除封面缓存并强制刷新。要手动替换特定封面，请点击专辑、播放列表或曲目上的 **「...」→ 编辑图像**，然后从您的照片库、文件应用或其中一个已连接的云服务中选择新图像。要更深入地控制存储在音频文件本身中的插图，请使用**编辑音频标签**或我们的配套应用 **Evertag**。
{{% /details %}}

{{% details title="如何在 Evermusic 中给歌曲评分？" closed="true" %}}
**打开音频播放器或任何曲目的「...」菜单，选择「编辑音频标签」，然后设置 1 到 5 星的评分——评分保存到文件的元数据中并随文件在设备间同步。**<br><br>

您可以按评分排序专辑、播放列表和音乐库，使高评分曲目排在最前面。评分在其他支持标准评分标签的标签编辑器和音乐播放器中也可读取。
{{% /details %}}

{{% details title="如何在 Evermusic 中将歌曲添加到收藏夹？" closed="true" %}}
**点击任何曲目上的「...」，选择「添加到收藏夹」——收藏夹出现在音乐库 → 收藏夹下，以及（可选）本地文件 → 收藏夹下。**<br><br>

在**设置 → 音乐库 → 收藏夹**中启用**同步编辑**，以在音乐库和文件部分之间同步收藏夹。免费版限制为 100 个收藏夹；Premium 取消限制。您还可以将收藏夹列表导出为 M3U、CSV 或 TXT 进行备份。
{{% /details %}}

{{% details title="如何查看最近播放的歌曲？" closed="true" %}}
**打开音乐库 → 最近使用，查看您最近播放的每首曲目的列表；您可以清空列表或将其导出为 M3U、CSV 或 TXT。**<br><br>

通过**设置 → 音乐库 → 最近使用 → 更改列表大小**更改列表保留的条目数。如果您想重新开始，请使用**删除列表**清空历史记录，或使用**导出歌曲列表**保留聆听历史记录的存档。
{{% /details %}}

{{% details title="如何从 Evermusic 删除歌曲而不从云存储中删除？" closed="true" %}}
**在任何曲目上，点击「...」→ 从音乐库中删除——这会从您的音乐库数据库中删除歌曲，但保留云存储和本地文件中的文件不变。**<br><br>

要同时从云端或本地设备删除文件，请改为选择**从云服务删除**或**删除文件**。这些操作无法撤销，因此选中多个文件时请小心。
{{% /details %}}

{{% details title="如何备份和恢复 Evermusic 音乐库？" closed="true" %}}
**打开设置 → 备份与恢复，选择要包含的内容（数据库、专辑封面、设置），点击「备份应用程序数据」，然后保存备份文件——在另一台设备上打开它以恢复。**<br><br>

备份包含您的音乐库条目、播放列表、评分、收藏夹、设置和专辑封面缓存。它不包括离线下载的音频文件，以保持文件大小可管理。通过 iCloud Drive、AirDrop 或任何已连接的云服务将备份文件移动到新设备，然后在 Evermusic 中打开以应用。<br><br>

{{% details title="如何用密码保护 Evermusic？" closed="true" %}}

Evermusic 使用固定的 4 位数字密码。密码可防止任何有权访问您设备的人打开 Evermusic 并浏览您已连接的云账户、下载的文件和音乐库。结合设备上的 iOS Face ID / Touch ID 以获得额外保护。

您还可以在**设置 → 个性化 → 应用程序图标**（Premium）中选择替代应用图标，以及在**背景样式**下选择模糊的专辑封面作为应用背景。

{{% details title="如何释放 Evermusic 占用的存储空间？" closed="true" %}}

您还可以在**设置 → 文件管理器 → 同步离线文件夹 → 「...」→ 禁用离线模式**中删除单个离线文件夹，这将删除本地副本。仅流媒体内容根本不占用存储空间。
{{% /details %}}

{{% details title="为什么我的云端音乐在 Evermusic 中不同步？" closed="true" %}}
**大多数同步问题是由过期的授权令牌、应用程序后台运行或没有活动的互联网连接引起的——打开连接，重新授权服务，然后从设置 → 音乐库 → 在线同步手动运行同步。**<br><br>

在线同步仅在应用程序处于前台时运行，因此同步大型音乐库可能需要一段时间。要加快速度，请保持 Evermusic 打开，将设备插入电源，并启用**设置 → 屏幕 → 始终活跃**。对于非常大的音乐库，请在应用程序的桌面版本上运行同步，并使用**备份与恢复**将结果传输到 iOS。

{{% details title="我可以用 Evermusic 播放有声书或播客吗？" closed="true" %}}
**是的——Evermusic 可以播放您存储的任何有声书或播客文件，只要是支持的格式（MP3、M4A、M4B、AAC、WAV、AIFF 等），具有书签、可变播放速度和睡眠定时器供睡前收听。**<br><br>

使用**书签**标记有趣的位置，在**设置 → 音频播放器 → 一般**中启用**保存播放位置**，以便应用程序记住您停止的位置，并将**播放速度**添加到播放器主屏幕，以便快速 1.25× / 1.5× / 2× 控制。完全支持长有声书章节和带章节标记的 `.m4b` 文件。

{{% details title="如何在曲目或有声书中添加书签到特定位置？" closed="true" %}}
**打开音频播放器并点击「添加书签」按钮保存当前播放位置——书签出现在曲目的「...」菜单的书签下。**<br><br>

在**设置 → 音频播放器 → 个性化 → 主屏幕操作**中启用**添加书签**操作，将按钮直接放在播放器上。书签按曲目存储，并在会话间保留，使其非常适合有声书、讲座和长 DJ 混音。
{{% /details %}}

{{% details title="Evermusic 可以使用蓝牙耳机、AirPods 和外部 DAC 吗？" closed="true" %}}
**是的——Evermusic 可以通过您的 iPhone、iPad 或 Mac 能够使用的任何音频输出播放：内置扬声器、有线耳机、蓝牙设备（AirPods、Beats、Sony、Bose 等）和 USB / Lightning DAC。**<br><br>

要通过外部 DAC 进行 Hi-Res 输出，请在**设置 → 音频播放器 → 一般**中将音频处理器切换到 **CoreAudio**，并选择适当的**音频输出采样率**和声道数。AirPods 的控柄控制（播放/暂停、下一首/上一首）按预期工作，锁定屏幕控制可在**设置 → 音频播放器 → 个性化 → 锁定屏幕上的播放控件**中自定义。

{{% details title="如何在 iPhone 或 iPad 上将 USB 闪存驱动器或 SD 卡与 Evermusic 一起使用？" closed="true" %}}
**通过 Lightning 转 USB 或 USB-C 适配器将驱动器插入 iPhone 或 iPad，然后在 Evermusic 中打开本地文件 → 此 iPhone 上的文件 → 打开文件夹，导航到驱动器，然后选择音乐文件夹。**<br><br>

应用程序直接从驱动器播放文件，无需复制到内部存储，这对于非常大的无损音乐库非常有用。您也可以以同样的方式将曲目导入音乐库或添加到播放列表。请参阅[如何将 USB 闪存驱动器连接到 iPhone 并听音乐](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it)中的完整分步说明。

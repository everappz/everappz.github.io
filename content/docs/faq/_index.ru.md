{{% details title="Почему я не могу войти в pCloud на старой версии iOS (15.8.4)?" closed="true" %}}
Веб-страница входа pCloud может некорректно отображаться на старых версиях iOS, таких как 15.8.4, что не позволяет ввести email и пароль на экране подключения к облаку.<br><br>

В качестве обходного решения можно использовать протокол **WebDAV**, который поддерживается pCloud и надёжно работает на всех версиях iOS.

**Настройка WebDAV:**<br>
- **Серверы США:** `https://webdav.pcloud.com`  
- **Европейские серверы:** `https://ewebdav.pcloud.com`  
- **Имя пользователя:** Ваш адрес электронной почты pCloud  
- **Пароль:** Пароль вашего аккаунта pCloud  

Откройте приложение → Подключения → Подключиться к облачному хранилищу → Выберите **WebDAV** → Введите учётные данные и URL сервера.

Этот метод позволит подключиться к хранилищу pCloud и получить доступ к файлам без проблем на старых устройствах.
{{% /details %}}

{{% details title="Как воспроизводить музыку через AirPlay с Mac (macOS)?" closed="true" %}}
Версия приложения для macOS не имеет встроенных кнопок AirPlay, Chromecast или Bluetooth, как в iOS.<br><br>

Чтобы использовать **AirPlay** на MacBook Pro, выполните следующие шаги:

1. Перейдите в **правый верхний угол** экрана и откройте **Пункт управления** (рядом с часами).  
2. Нажмите на значок **Звук/Громкость**.  
3. На следующем экране нажмите **AirPlay**, чтобы увидеть список всех доступных устройств AirPlay.  
4. Выберите нужное устройство для начала стриминга музыки.  

Это перенаправит весь системный звук (в том числе из Evermusic или Flacbox) на выбранное устройство AirPlay.
{{% /details %}}

{{% details title="Почему моя покупка Premium не активируется на Mac, если я купил её на iPhone?" closed="true" %}}
Бессрочные покупки и подписки синхронизируются между iOS и Mac через **iCloud**.<br><br>

Чтобы активировать Premium на Mac:<br>
- Убедитесь, что на обоих устройствах установлена последняя версия приложения<br>
- Включите **iCloud** на обоих устройствах<br>
- Запустите приложение на iOS-устройстве и подождите 1 минуту, пока информация о покупке загружается<br>
- На Mac установите приложение из App Store с тем же **Apple ID**<br>
- Запустите приложение и подождите несколько секунд для синхронизации информации<br>
- Либо нажмите **Восстановить покупки** в настройках приложения на обоих устройствах<br><br>

Функции Premium должны автоматически активироваться на Mac.
{{% /details %}}

{{% details title="Как подключиться и стримить аудио с WD MyCloud EX2 Ultra?" closed="true" %}}

При добавлении подключения в приложении через **Подключения > Подключиться к облачному хранилищу > My Cloud Home** оно официально разработано для поддержки устройств **WD MyCloud Home**.<br>
WD MyCloud EX2 Ultra использует ограниченный доступ для приложений.<br><br>

Однако если вы успешно подключились к **WD MyCloud EX2 Ultra**, **WD MyCloud Mirror** или другой модели хранилища **WD MyCloud**, вы всё равно можете его использовать с помощью следующего обходного решения:<br><br>

1. Откройте **Подключения → Подключиться к облачному хранилищу → My Cloud Home**<br>
2. Создайте папку с помощью встроенного файлового менеджера<br>
3. Загрузите музыкальные файлы в эту папку<br>
4. Они будут сохранены в «песочнице», доступной только приложению<br>
5. Теперь вы можете стримить или скачивать их напрямую<br><br>

⚠️ С NAS будут доступны только папки, созданные через приложение.

{{% details title="Почему приложение не может читать имена папок со специальными символами?" closed="true" %}}
Это известная проблема с **протоколом SMB2**.<br><br>

Попробуйте следующие решения:<br>
- Включите **SMB1** на сервере и в настройках приложения<br>
- Используйте **системный выбор файлов**:<br>
  - Перейдите в **Локальные файлы > Файлы на этом устройстве > Открыть файлы...**<br>
  - Выбирайте папки/файлы через стандартное меню Apple<br><br>

Также можно подключиться через **WebDAV** или **DLNA**, если ваш NAS их поддерживает.

{{% details title="Как перенести музыкальную библиотеку 10 ГБ с Windows 11 на iPhone для офлайн-воспроизведения?" closed="true" %}}

У вас есть несколько надёжных способов перенести музыкальную библиотеку с ПК на Windows 11 на iPhone и использовать её офлайн в приложении. Выберите наиболее подходящий для вас:

1. **Через кабельное подключение (рекомендуется для больших библиотек)**  <br>
   Используйте приложение **Apple Devices** на Windows 11 для прямой передачи файлов на iPhone через USB.  
   Следуйте руководству Apple:  
   https://support.apple.com/en-ph/120402 <br>

2. **Беспроводно через Wi-Fi Drive**  <br>
   Включите функцию **WiFi Drive** в приложении и загружайте музыку через браузер на компьютере.  
   Пошаговые инструкции:  
   [Как перенести музыкальные файлы с компьютера на iPhone без iTunes через WiFi-Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/) <br>

3. **Беспроводно через DLNA-сервер**  <br>
   Включите медиасервер DLNA на компьютере с Windows и стримьте или переносите библиотеку прямо в приложение.  
   Руководство:  
   [Как включить медиасервер DLNA в Windows 10 и слушать музыку на iPhone](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/) <br>

4. **Беспроводно через общий доступ к файлам SMB**  <br>
   Включите **общий доступ к файлам SMB** на ПК и подключитесь к нему в приложении для просмотра, скачивания или передачи файлов по папкам.  
   Инструкции:  
   [Передача файлов с компьютера на iPhone через протокол SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/) <br>

⚠️ При переносе больших библиотек (10 ГБ+) проводная передача через USB обычно является самым быстрым и стабильным вариантом.

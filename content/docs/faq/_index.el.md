{{% details title="Γιατί δεν μπορώ να συνδεθώ στο pCloud σε παλαιότερη έκδοση iOS (15.8.4);" closed="true" %}}
Η ιστοσελίδα σύνδεσης του pCloud ενδέχεται να μην εμφανίζεται σωστά σε παλαιότερες εκδόσεις iOS, όπως η 15.8.4, γεγονός που εμποδίζει την εισαγωγή email και κωδικού πρόσβασης στην οθόνη σύνδεσης cloud.<br><br>

Ως εναλλακτική λύση, μπορείτε να χρησιμοποιήσετε το πρωτόκολλο **WebDAV**, που υποστηρίζεται από το pCloud και λειτουργεί αξιόπιστα σε όλες τις εκδόσεις iOS.

**Ρύθμιση WebDAV:**<br>
- **Διακομιστές ΗΠΑ:** `https://webdav.pcloud.com`  
- **Ευρωπαϊκοί διακομιστές:** `https://ewebdav.pcloud.com`  
- **Όνομα χρήστη:** Η διεύθυνση email σας στο pCloud  
- **Κωδικός πρόσβασης:** Ο κωδικός πρόσβασης του λογαριασμού σας στο pCloud  

Ανοίξτε την εφαρμογή → Συνδέσεις → Σύνδεση με αποθηκευτικό χώρο cloud → Επιλέξτε **WebDAV** → Εισαγάγετε τα διαπιστευτήριά σας και τη διεύθυνση URL του διακομιστή.

Αυτή η μέθοδος θα σας επιτρέψει να συνδεθείτε στον αποθηκευτικό σας χώρο pCloud και να έχετε πρόσβαση στα αρχεία σας χωρίς προβλήματα σε παλαιότερες συσκευές.
{{% /details %}}

{{% details title="Πώς μπορώ να αναπαράγω μουσική μέσω AirPlay από Mac (macOS);" closed="true" %}}
Η έκδοση macOS της εφαρμογής δεν περιλαμβάνει ενσωματωμένα κουμπιά σύνδεσης AirPlay, Chromecast ή Bluetooth όπως στο iOS.<br><br>

Για να χρησιμοποιήσετε **AirPlay** στο MacBook Pro σας, ακολουθήστε τα παρακάτω βήματα:

1. Μεταβείτε στην **επάνω δεξιά γωνία** της οθόνης σας και ανοίξτε το **Κέντρο ελέγχου** (κοντά στο ρολόι).  
2. Κάντε κλικ στο εικονίδιο **Ήχος/Ένταση**.  
3. Στην επόμενη οθόνη, κάντε κλικ στο **AirPlay** για να δείτε μια λίστα με όλες τις διαθέσιμες συσκευές AirPlay.  
4. Επιλέξτε την επιθυμητή συσκευή για να ξεκινήσετε τη μετάδοση της μουσικής σας.  

Αυτό θα δρομολογήσει όλον τον ήχο του συστήματος (συμπεριλαμβανομένου του Evermusic ή Flacbox) στην επιλεγμένη συσκευή AirPlay.

Έχετε αρκετές αξιόπιστες επιλογές για να μεταφέρετε τη μουσική βιβλιοθήκη σας από τον υπολογιστή Windows 11 στο iPhone και να τη χρησιμοποιείτε offline στην εφαρμογή. Επιλέξτε τη μέθοδο που σας ταιριάζει καλύτερα:

1. **Με καλωδιακή σύνδεση (συνιστάται για μεγάλες βιβλιοθήκες)**  <br>
   Χρησιμοποιήστε την εφαρμογή **Apple Devices** στα Windows 11 για να μεταφέρετε αρχεία απευθείας στο iPhone μέσω USB.  
   Ακολουθήστε τον οδηγό της Apple εδώ:  
   https://support.apple.com/en-ph/120402 <br>

2. **Ασύρματα μέσω Wi-Fi Drive**  <br>
   Ενεργοποιήστε τη λειτουργία **WiFi Drive** στην εφαρμογή και ανεβάστε τη μουσική σας μέσω browser στον υπολογιστή.  
   Οδηγίες βήμα προς βήμα εδώ:  
   [Πώς να μεταφέρετε αρχεία μουσικής από υπολογιστή στο iPhone χωρίς iTunes με WiFi-Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/) <br>

3. **Ασύρματα με διακομιστή DLNA**  <br>
   Ενεργοποιήστε τον διακομιστή πολυμέσων DLNA στον υπολογιστή Windows και κάντε streaming ή μεταφέρετε τη βιβλιοθήκη σας απευθείας στην εφαρμογή.  
   Οδηγός:  
   [Πώς να ενεργοποιήσετε τον διακομιστή πολυμέσων DLNA στα Windows 10 και να αναπαράγετε μουσική στο iPhone](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/) <br>

4. **Ασύρματα με κοινή χρήση αρχείων SMB**  <br>
   Ενεργοποιήστε **κοινή χρήση αρχείων SMB** στον υπολογιστή σας και συνδεθείτε σε αυτόν στην εφαρμογή για περιήγηση, λήψη ή μεταφορά αρχείων φάκελο-φάκελο.  
   Οδηγίες:  
   [Μεταφορά αρχείων από υπολογιστή στο iPhone με το πρωτόκολλο SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/) <br>

⚠️ Κατά τη μεταφορά μεγάλων βιβλιοθηκών (10 GB+), η ενσύρματη μεταφορά USB είναι συνήθως η πιο γρήγορη και σταθερή επιλογή.

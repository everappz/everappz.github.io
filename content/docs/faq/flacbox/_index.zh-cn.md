Flacbox 是适用于 **iPhone、iPad 和 Mac** 的 **hi-res 音乐播放器**，可从 iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Box、pCloud、Yandex Disk、Synology、QNAP、WD My Cloud Home 以及任何其他 WebDAV、SMB 或 DLNA 服务器流式传输和下载曲目。基于系统编解码器和内置 **FFmpeg** 库构建，它增加了对 Apple 默认播放器无法处理的格式的支持——WMA、OGG、OPUS、APE、DSD——并为发烧友提供对输出采样率、多声道路由、音调校正和 IO 缓冲区持续时间的控制。本 FAQ 回答了用户最常发送的问题。如需更深入的指南，请参阅 [Flacbox 用户指南](/docs/guide/flacbox/) 或浏览我们的 [操作文章](/docs/howto/)。

{{% details title="Flacbox 如何工作？" closed="true" %}}
**Flacbox 是一款 hi-res 音乐播放器，让您可以像管理普通文件一样管理音频曲目**，并从任何云服务或 NAS 进行流式传输——无需 iTunes 同步。<br><br>

您可以将整个音乐收藏上传到 **Dropbox、Google Drive、OneDrive、iCloud Drive、MEGA、Box、pCloud** 等云服务或个人 NAS，并直接从云端播放音乐，拥有完全的控制权。若要离线收听，请使用内置下载管理器将音乐保存到设备。文件以您放置的方式存储在您的存储中——Flacbox 只是简单地播放它们。<br><br>

{{% details title="Flacbox 和 Evermusic 有什么区别？" closed="true" %}}
**Flacbox 是专注于发烧友的播放器，具有 FFmpeg 驱动的格式支持、可配置的采样率、多声道输出和音调校正；Evermusic 是主流播放器，具有交叉淡入淡出、无缝播放、空间音频和更广泛的云功能集。**<br><br>

**Flacbox** 旨在支持所有默认 iOS 音频格式，以及 iPhone 上原生不支持的许多附加格式，如 **WMA、OGG、OPUS、APE、DSD、DSF、DFF、TTA、MPC、WV** 等。它使用自定义音频引擎（系统编解码器 + FFmpeg）处理几乎每种格式，并提供发烧级功能，如**可调音频输出采样率（44.1 – 96 kHz）、多声道输出（Mono → 5.1 → SDDS）、IO 缓冲区调整和音调校正（-1000 至 +1000）**。<br><br>

**Evermusic** 支持默认 Apple 音频格式，并包含消费者友好的功能，如**交叉淡入淡出播放、无缝播放、空间音频、音频均衡器、播放速度控制**和更广泛的云流媒体工具。如果您主要使用 MP3、AAC、ALAC 或 FLAC 并希望使用交叉淡入淡出、无缝或空间音频，**Evermusic** 可能是更好的选择。如果您需要与更广泛的音频格式兼容——特别是 DSD 或 WMA——并希望对音频输出链进行精细控制，**Flacbox** 是正确的选择。<br><br>

[了解更多关于差异的信息](/docs/faq/evermusic/what-is-the-difference-between-evermusic-and-flacbox/)

{{% details title="Flacbox 安全吗？" closed="true" %}}
**是的——Flacbox 仅使用官方 SDK 和安全的 OAuth 连接与连接的云服务交互，从不看到您的密码，并且仅在 iOS Keychain 中存储加密的访问令牌。**<br><br>

当您输入登录名和密码时，应用程序会显示云服务提供商提供的官方授权页面，整个授权过程在 Flacbox 之外进行。成功授权后，云提供商会向应用程序发送 **auth-token**，该令牌用于进行 API 调用。<br><br>

auth-token 是允许第三方应用程序与云存储交互的数字密钥。令牌存储在设备的 **Apple 安全系统存储（Keychain）**中，静态加密，并受设备密码或生物识别锁保护。下载的文件存储在应用程序的 **Documents** 目录中；您可以随时使用内置文件管理器删除它们。<br><br>

应用程序**不会与 Everappz、广告商或任何第三方共享**来自已连接云帐户的任何信息。您可以随时通过在网络浏览器中打开帐户设置页面来撤销对您云帐户的访问权限。要撤销 auth-token，请在网络上登录您的云帐户，导航到安全或已连接应用程序页面，然后删除 Flacbox 的条目。<br><br>

您也可以在应用程序内部断开云帐户连接——这样做时，auth-token 会立即从您的设备中删除。卸载应用程序会自动删除所有已下载的数据和令牌。<br><br>

{{% details title="Flacbox 支持哪些云服务、媒体服务器和协议？" closed="true" %}}
**Flacbox 支持几乎所有流行的云存储服务、自托管媒体服务器、文件共享协议和 S3 兼容对象存储——全部通过一个「连接到云存储」屏幕完成。**<br><br>

**个人云存储：** iCloud Drive · Google Drive · Dropbox · OneDrive（个人和商业）· Box · MEGA · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive（阿里云盘）· Baidu Pan（百度网盘）。<br><br>

**自托管媒体服务器和音乐服务器 API：** **Plex** · **Jellyfin** · **Emby** · **Subsonic**（以及每个 Subsonic 兼容服务器——Airsonic、Funkwhale、Gonic、Logitech Media Server、Ampache）· **Navidrome** · Nextcloud（以及通过共享 API 的 ownCloud）· **Synology Drive** · **QNAP**。<br><br>

**NAS 和文件共享协议：** **SMB**（SMB1、SMB2、Auto）· **WebDAV**（HTTP / HTTPS）· **FTP / FTPS** · **SFTP**（密码或公钥认证）· **NFS** · **DLNA / UPnP**（播放和下载）。适用于 Apple Time Capsule、Synology、QNAP、WD My Cloud Home、Buffalo、任何 Linux Samba / NFS / SSH 主机，或 Mac 或 Windows PC 上的共享文件夹。<br><br>

**S3 兼容对象存储：** AWS S3、Backblaze B2、Wasabi、Cloudflare R2、MinIO、DigitalOcean Spaces，以及任何其他 S3-API 端点。<br><br>

**本地网络发现：** **可用设备**部分自动列出 Wi-Fi 网络上的每个 Bonjour / mDNS 服务——Plex、Jellyfin、Emby 服务器、Synology、QNAP、带有附加驱动器的 AirPort 路由器、Time Capsule——这样您可以点击连接而无需输入 IP 地址。<br><br>

每个连接都使用服务的**官方 SDK 或开放协议**，在支持的情况下使用基于 OAuth 的授权。您可以连接同一服务的多个帐户（例如，两个 Google Drive 帐户、个人 Dropbox 和工作 Dropbox，或同时连接 Plex 和 Jellyfin 服务器）。Premium 用户可以添加无限数量的服务；免费版本限制为三个。<br><br>

{{% details title="如何在 iPhone 上播放本地下载的音乐？" closed="true" %}}
**打开本地文件，滚动到「此 iPhone 上的文件」，点击「打开文件…」或「打开文件夹…」，然后选择要播放的音频文件或文件夹。**<br><br>

应用程序扫描文件夹内容，将找到的音频文件添加到播放器队列，并直接从其原始位置播放，无需将任何内容复制到应用程序包。<br><br>

**将文件夹添加到收藏夹以便快速访问**<br>
打开**本地文件**屏幕，滚动到**快速访问**，点击**收藏夹**以访问**收藏文件**屏幕。点击右上角的更多操作按钮 **「...」** 并选择**添加文件夹**。导航到所需文件夹并点击**打开**确认。您的文件夹将被添加到**收藏文件**，让您无需重复步骤即可快速访问音乐。<br><br>

**将本地文件导入音乐库**<br>
如果您希望在音乐库中整理本地文件，打开**音乐库**屏幕，点击右上角的三点按钮，选择 **+ 添加音乐**。选择**本地文件**菜单项，滚动到**此 iPhone 上的文件**部分，点击**打开文件…**。选择要添加的文件并点击**打开**确认。应用程序扫描所选文件并将其添加到音乐库，按艺术家、专辑和流派等元数据进行整理。<br><br>

**将本地文件添加到播放列表**<br>
要将本地文件添加到播放列表，打开**播放列表**屏幕并点击右上角的更多按钮。选择 **+ 新建播放列表**，输入新播放列表的名称，在下一个屏幕上选择**本地文件**选项。滚动到**此 iPhone 上的文件**部分并点击**打开文件…**。选择要添加的音频文件并点击**打开**确认。文件将被添加到播放列表，您可以使用更多按钮更改歌曲顺序并执行其他操作。<br><br>

{{% details title="如何从计算机向 Flacbox 传输音乐？" closed="true" %}}
**您可以通过 SMB、WebDAV、FTP / SFTP、DLNA、Wi-Fi Drive 或 iTunes / Finder 文件共享向 Flacbox 传输音乐——任何将文件暴露给设备的方法都可以。**<br><br>

要使用 **SMB** 协议连接计算机，点击**连接云服务 → SMB**。在 URL 字段中以 `smb://computer-ip-address/shared-folder-name` 格式输入计算机的 IP 地址和共享文件夹名称，输入登录名和密码，然后点击**完成**。如果连接成功，共享将出现在连接屏幕的**云服务**部分。<br><br>

[完整教程：使用 SMB 协议将文件从计算机传输到 iPhone](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

{{% details title="如何从 Flacbox 将播放列表导出为 M3U、CSV 或 TXT？" closed="true" %}}
**打开播放列表，点击「...」菜单，选择「导出歌曲列表」——您可以将结果保存为 M3U、M3U8、CSV 或 TXT。**<br><br>

同样的操作也适用于专辑、艺术家、流派、最近列表和收藏列表。使用 M3U 重新导入其他播放器，使用 CSV 制作电子表格，使用 TXT 制作简单易读的副本。<br><br>

[阅读完整指南](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)

{{% details title="如何将 Flacbox 的音乐投射到 Google Chromecast 或 AirPlay？" closed="true" %}}
**打开音频播放器，点击 AirPlay 或 Chromecast 图标，然后从列表中选择扬声器、电视或 AV 接收器。**<br><br>

iOS 上同时支持 **AirPlay 2** 和 **Google Chromecast**——可通过**设置 → 音频播放器 → 个性化 → 主屏幕操作**将按钮添加到主播放器屏幕。某些 hi-res 格式可能需要转码才能进行 Chromecast 播放。AirPlay 2 还允许同时向多个 HomePod 或 Apple TV 播放。
{{% /details %}}

{{% details title="如何在 iPhone 主屏幕或锁屏上启用 Flacbox 小组件？" closed="true" %}}
**在设置 → 小组件中启用小组件更新，然后长按主屏幕或锁屏，点击「+」，搜索「Flacbox」，并选择小组件大小。**<br><br>

小组件显示当前播放的曲目，包含封面和基本信息。由于小组件刷新使用少量能量，**启用小组件**开关默认关闭——仅在主动使用小组件时打开。小组件在 iPhone 和 iPad 主屏幕和锁屏上以及 macOS 通知中心中工作。
{{% /details %}}

{{% details title="如何在 Flacbox 中编辑 MP3 标签？" closed="true" %}}
**点击任何曲目上的「...」并选择「编辑音频标签」打开内置标签编辑器——您可以更改标题、艺术家、专辑、年份、流派、歌词、专辑封面等。**<br><br>

对于在线文件，Flacbox 临时下载曲目，让您编辑，然后重新上传到云服务。您可以在**设置 → 音频标签编辑器**中配置此行为——包括专辑封面缩放、是否将更改自动写回云文件，以及哪些按钮出现在编辑器主屏幕上。对于一次对多个文件进行更深入的批量编辑，请安装我们的配套应用 **Evertag**。[了解更多关于 Evertag 的信息](/products/evertag/)。
{{% /details %}}

{{% details title="如何修复 Flacbox 中缺少的专辑封面？" closed="true" %}}
**打开设置 → 资料库 → 专辑封面，启用「加载在线文件的专辑封面」和「在文件夹中搜索」，然后根据需要提高封面质量——应用程序将从您的云文件中提取嵌入的封面，并在没有嵌入封面时使用与音频文件一起存储的 JPEG / PNG 图像。**<br><br>

您也可以点击**全部删除**以清除封面缓存并强制刷新。要手动替换特定封面，请点击专辑、播放列表或曲目上的 **「...」→ 编辑图像**，并从照片库、文件应用或已连接的云服务之一中选择新图像。要更深入控制存储在音频文件本身内的封面，请使用**编辑音频标签**或我们的配套应用 **Evertag**。
{{% /details %}}

{{% details title="如何在不从云存储中删除文件的情况下从 Flacbox 删除歌曲？" closed="true" %}}
**在任何曲目上，点击「...」→ 从音乐库删除——这会从您的库数据库中删除歌曲，但云存储和本地文件中的文件保持不变。**<br><br>

要同时从云端或本地设备删除文件，请改为选择**从云服务删除**或**从本地文件删除**。这些破坏性操作无法撤消，因此当选择了多个文件时请小心。
{{% /details %}}

{{% details title="如何备份和恢复 Flacbox 音乐库？" closed="true" %}}
**打开设置 → 备份和恢复，选择要包含的内容（数据库、专辑封面、设置），点击「备份应用数据」，并保存备份文件——在另一台设备上打开以恢复。**<br><br>

备份包含您的音乐库条目、播放列表、评分、收藏、设置和专辑封面缓存。它不包括离线下载的音频文件，以保持文件大小可管理。通过 iCloud Drive、AirDrop 或任何已连接的云服务将备份文件移动到新设备，然后在 Flacbox 中打开以应用。<br><br>

{{% details title="如何与家人共享 Flacbox Premium？" closed="true" %}}
**所有 Flacbox Premium 计划——终身、每月和每年——都支持 Apple Family Sharing，因此家庭组中的任何人都可以安装 Flacbox 并免费使用 Premium。**<br><br>

在 iOS / macOS **设置 → 家庭**中设置 Family Sharing，然后让每位家庭成员从 App Store 安装 Flacbox，并在登录自己的 Apple ID 后运行一次。Premium 会在一分钟内自动识别。每位家庭成员的 iPhone、iPad 和 Mac 之间共享相同的计划。
{{% /details %}}

{{% details title="如何取消 Flacbox Premium 订阅？" closed="true" %}}
**打开 iOS 或 macOS 设置 → [您的名字] → 订阅，找到 Flacbox，点击「取消订阅」——您的 Premium 功能在当前计费期结束前保持活跃。**<br><br>

终身应用内购买不是订阅，不需要取消。退款请使用 Apple 的 **报告问题**页面（`reportaproblem.apple.com`）——退款由 Apple 发放，而非 Everappz。
{{% /details %}}

{{% details title="如何用密码保护 Flacbox？" closed="true" %}}
**打开设置 → 密码，点击「启用」，选择 4 位数字代码——每次应用启动时都会提示您输入。**<br><br>

Flacbox 使用固定的 4 位数字密码。密码可防止任何有权访问您设备的人打开 Flacbox 并浏览您已连接的云帐户、下载的文件和库。结合设备上的 iOS Face ID / Touch ID 以获得额外保护。
{{% /details %}}

{{% details title="如何在 Flacbox 中启用深色模式？" closed="true" %}}
**打开设置 → 个性化 → 颜色方案，然后选择深色、浅色或默认（跟随系统外观）。**<br><br>

您还可以在**设置 → 个性化 → 应用图标**（Premium）中选择备用应用图标，并在**背景样式**下选择模糊的专辑封面作为应用背景。
{{% /details %}}

{{% details title="如何在 Flacbox 中更改播放速度？" closed="true" %}}
**打开音频播放器，点击速度控制，拖动滑块——速度可从 0.02× 调整到 3.00×。**<br><br>

您也可以在**设置 → 音频播放器 → 播放速度**中更改默认速度。点击右上角的配置图标切换到**精确模式**以进行非常精细的调整——对有声书、讲座和语言学习内容很方便。
{{% /details %}}

{{% details title="如何在 Flacbox 中设置睡眠定时器？" closed="true" %}}
**打开设置 → 音频播放器 → 睡眠定时器，打开它，并选择您希望音乐播放多长时间后停止。**<br><br>

您也可以通过**设置 → 音频播放器 → 个性化 → 主屏幕操作**添加**睡眠定时器**按钮，直接在音频播放器屏幕上使用。点击睡眠定时器屏幕右上角的配置图标，启用精确的逐分钟调整。
{{% /details %}}

{{% details title="没有网络连接可以使用 Flacbox 吗？" closed="true" %}}
**可以——一旦您下载了音乐或为文件夹启用了离线模式，Flacbox 就可以完全离线播放所有内容。**<br><br>

离线内容保存在**本地文件**下，在飞行模式、飞行途中以及任何没有 Wi-Fi 或蜂窝数据的地方都能继续工作。仅在云端的曲目（您未下载的）在重新获得连接之前将显示为灰色。对于旅行，请在出发前为相关文件夹启用**离线模式**或下载特定专辑和播放列表。
{{% /details %}}

{{% details title="Flacbox 可以 scrobble 到 Last.fm 吗？" closed="true" %}}
**可以——Flacbox 支持 Last.fm scrobbling，因此您播放的每首曲目都会自动记录到您的 Last.fm 个人资料中。**<br><br>

从**连接 → 其他服务**部分连接您的 Last.fm 帐户。连接后，所有播放统计信息都会在后台发送到 Last.fm。然后您可以访问您的 Last.fm 个人资料，查看收听趋势、每周最热艺术家和个性化推荐。<br><br>

{{% details title="如何更改 Flacbox 界面语言？" closed="true" %}}
**打开设置 → 语言，从 120 多种支持的语言中选择，然后重启应用以使更改生效。**<br><br>

该应用支持包括英语、法语、德语、西班牙语、意大利语、葡萄牙语、俄语、乌克兰语、波兰语、荷兰语、阿拉伯语、希伯来语、印地语、日语、韩语、中文（简体和繁体）、越南语、土耳其语等多种语言。选择**默认**以自动跟随设备的语言设置。选择新语言后，完全退出 Flacbox 并重新打开，以便每个屏幕使用新翻译重新渲染。

{{% details title="如何在 Flacbox 中搜索歌曲、专辑或艺术家？" closed="true" %}}
**点击任何列表中的放大镜图标——音乐库、播放列表、专辑、艺术家、流派或文件夹内——输入名称即可立即过滤结果。**<br><br>

搜索是本地的，完全离线地对库数据库进行，因此即使在慢速网络上，结果也会在您输入时出现。您还可以在特定播放列表、专辑或文件夹中搜索，以在数百首中找到单首曲目。曲目、专辑、艺术家、流派、作曲家和文件夹都可以搜索。

{{% details title="Flacbox 支持蓝牙耳机、AirPods 和外部 DAC 吗？" closed="true" %}}
**是的——Flacbox 通过 iPhone、iPad 或 Mac 可以使用的任何音频输出播放：内置扬声器、有线耳机、蓝牙设备（AirPods、Beats、Sony、Bose 等）以及 USB / Lightning DAC。**<br><br>

对于外部 DAC 的 Hi-Res 输出，请在**设置 → 音频播放器 → 一般**中将音频编解码器切换到 **FFmpeg**，并选择适当的**音频输出采样率**和声道数。AirPods 的柄上控制（播放 / 暂停、下一首 / 上一首）按预期工作，锁屏控制可在**设置 → 音频播放器 → 个性化 → 锁屏上的播放控制**中自定义。
{{% /details %}}

{{% details title="如何在 iPhone 或 iPad 上将 USB 闪存驱动器或 SD 卡与 Flacbox 一起使用？" closed="true" %}}
**通过 Lightning 转 USB 或 USB-C 适配器将驱动器插入 iPhone 或 iPad，然后在 Flacbox 中打开本地文件 → 此 iPhone 上的文件 → 打开文件夹，导航到驱动器并选择音乐文件夹。**<br><br>

Flacbox 直接从驱动器播放文件，无需将其复制到内部存储，这对非常大的无损库很有用。您也可以以同样的方式将曲目导入音乐库或添加到播放列表。该应用支持 **Apple 认证的读卡器**和 **iXpand 闪存驱动器**——对于 iXpand，将其插入 Lightning 接口，驱动器会自动出现在连接屏幕中。请参阅[如何将 USB 闪存驱动器连接到 iPhone 并收听音乐](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it)中的完整分步说明。

{{% details title="为什么我的云音乐在 Flacbox 中无法同步？" closed="true" %}}
**大多数同步问题是由过期的身份验证令牌、应用在后台运行或没有活跃的网络连接引起的——打开连接，重新授权服务，然后从设置 → 资料库 → 在线同步手动运行同步。**<br><br>

在线同步仅在应用处于前台时运行，因此同步大型库可能需要一段时间。为了加速，请保持 Flacbox 打开，将设备插入电源，并启用**设置 → 屏幕 → 始终活跃**。对于非常大的库，请在桌面版应用上运行同步，并使用**备份和恢复**将结果传输到 iOS。

<div class="hx:mt-6"></div>

Flacbox adalah **pemutar musik hi-res** untuk **iPhone, iPad, dan Mac** yang melakukan streaming dan mengunduh lagu dari iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP, WD My Cloud Home, dan server WebDAV, SMB, atau DLNA lainnya. Dibangun di atas codec sistem dan library **FFmpeg** yang disertakan, ia menambahkan dukungan untuk format yang tidak dapat ditangani oleh pemutar default Apple — WMA, OGG, OPUS, APE, DSD — dan memberikan kontrol audiofil atas sample rate output, routing multi-saluran, koreksi pitch, dan durasi buffer IO. FAQ ini menjawab pertanyaan yang paling sering dikirimkan pengguna. Untuk panduan lebih mendalam, lihat [Panduan Pengguna Flacbox](/docs/guide/flacbox/) atau telusuri [artikel Cara Penggunaan](/docs/howto/) kami.

{{% details title="Bagaimana cara kerja Flacbox?" closed="true" %}}

Anda dapat mengunggah seluruh koleksi musik Anda ke layanan cloud seperti **Dropbox, Google Drive, OneDrive, iCloud Drive, MEGA, Box, pCloud,** atau NAS pribadi, dan memutar musik langsung dari cloud dengan kontrol penuh. Untuk mendengarkan secara offline, gunakan manajer unduhan bawaan untuk menyimpan musik ke perangkat Anda. File disimpan di penyimpanan Anda persis seperti yang Anda taruh di sana — Flacbox hanya memainkannya.<br><br>

**Ya — Flacbox gratis digunakan dengan beberapa keterbatasan**, yang dapat dihapus dengan meningkatkan ke versi Premium.<br><br>

Anda dapat memilih antara **pembelian seumur hidup satu kali** atau **dua opsi berlangganan (bulanan atau tahunan)**. Pembelian dan langganan Premium dibagikan di antara iOS dan Mac melalui iCloud — untuk menyinkronkan pembelian Anda, pastikan iCloud diaktifkan di kedua perangkat, buka aplikasi di iOS, dan tunggu satu menit hingga sinkronisasi selesai.<br><br>

{{% details title="Apa perbedaan antara Flacbox dan Evermusic?" closed="true" %}}
**Flacbox adalah pemutar yang berfokus pada audiofil dengan dukungan format bertenaga FFmpeg, sample rate yang dapat dikonfigurasi, output multi-saluran, dan koreksi pitch; Evermusic adalah pemutar mainstream dengan crossfade, gapless, audio spasial, dan kumpulan fitur cloud yang lebih luas.**<br><br>

**Flacbox** dibangun untuk mendukung semua format audio iOS default bersama dengan banyak format tambahan yang tidak didukung secara native di iPhone, seperti **WMA, OGG, OPUS, APE, DSD, DSF, DFF, TTA, MPC, WV,** dan lainnya. Ia menggunakan engine audio khusus (codec sistem + FFmpeg) untuk menangani hampir setiap format, dan menawarkan fitur kelas audiofil seperti **sample rate output audio yang dapat disesuaikan (44,1 – 96 kHz), output multi-saluran (Mono → 5.1 → SDDS), penyetelan buffer IO, dan koreksi pitch (-1000 hingga +1000)**.<br><br>

**Evermusic** mendukung format audio Apple default dan menyertakan fitur ramah konsumen seperti **pemutaran crossfade, pemutaran gapless, audio spasial, equalizer audio, kontrol kecepatan pemutaran,** dan perangkat streaming cloud yang lebih luas. Jika Anda terutama menggunakan MP3, AAC, ALAC, atau FLAC dan menginginkan crossfade, gapless, atau audio spasial, **Evermusic** mungkin menjadi pilihan yang lebih baik. Jika Anda membutuhkan kompatibilitas luas dengan berbagai format audio — terutama DSD atau WMA — dan menginginkan kontrol halus atas rantai output audio, **Flacbox** adalah pilihan yang tepat.<br><br>

{{% details title="Apakah Flacbox aman?" closed="true" %}}
**Ya — Flacbox hanya menggunakan SDK resmi dan koneksi OAuth yang aman untuk berinteraksi dengan layanan cloud yang terhubung, tidak pernah melihat kata sandi Anda, dan hanya menyimpan token akses terenkripsi di iOS Keychain.**<br><br>

Saat Anda memasukkan login dan kata sandi, aplikasi menampilkan halaman otorisasi resmi yang disediakan oleh penyedia layanan cloud, dan seluruh proses otorisasi terjadi di luar Flacbox. Penyedia cloud kemudian mengirimkan **auth-token** ke aplikasi setelah otorisasi berhasil, dan token tersebut digunakan untuk melakukan panggilan API.<br><br>

Auth-token adalah kunci digital yang memungkinkan aplikasi pihak ketiga berinteraksi dengan penyimpanan cloud. Token disimpan di perangkat Anda di **penyimpanan sistem aman Apple (Keychain)**, yang dienkripsi saat tidak aktif dan dilindungi oleh kode sandi perangkat atau kunci biometrik Anda. File yang diunduh disimpan di direktori **Documents** aplikasi; Anda dapat menghapusnya kapan saja menggunakan manajer file bawaan.<br><br>

Aplikasi **tidak berbagi informasi apa pun** dari akun cloud yang terhubung dengan Everappz, pengiklan, atau pihak ketiga mana pun. Anda dapat mencabut akses ke akun cloud Anda kapan saja dengan membuka halaman pengaturan akun di browser web Anda. Untuk mencabut auth-token, masuk ke akun cloud Anda di web, navigasikan ke halaman keamanan atau aplikasi yang terhubung, dan hapus entri untuk Flacbox.<br><br>

Anda juga dapat memutuskan koneksi akun cloud di dalam aplikasi itu sendiri — ketika Anda melakukannya, auth-token segera dihapus dari perangkat Anda. Menghapus instalasi aplikasi menghapus semua data yang diunduh dan token secara otomatis.<br><br>

{{% details title="Bagaimana cara membuat daftar putar di Flacbox?" closed="true" %}}
**Untuk membuat daftar putar, buka tab Daftar Putar, ketuk "+" atau menu "...", pilih "Daftar Putar Baru," beri nama, lalu pilih lagu yang ingin ditambahkan dari Perpustakaan, File dalam Aplikasi Ini, File di Perangkat Ini, atau Koneksi.**<br><br>

- Ketuk tombol **+** atau tombol **"..."** di sudut kanan atas dan pilih **Daftar Putar Baru**.<br>

- Pilih lagu yang ingin Anda tambahkan ke daftar putar, lalu ketuk **Selesai**.<br><br>

Secara default, setiap lagu hanya dapat ditambahkan ke daftar putar satu kali. Anda nanti dapat mengubah urutan lagu, mengganti nama daftar putar, mengedit gambar sampulnya, mengarsipkannya sebagai ZIP, dan mengekspornya ke M3U, CSV, atau TXT.<br><br>

{{% details title="Layanan cloud, server media, dan protokol apa yang didukung Flacbox?" closed="true" %}}
**Flacbox mendukung hampir setiap layanan penyimpanan cloud populer, server media yang dihosting sendiri, protokol berbagi file, dan penyimpanan objek yang kompatibel dengan S3 — semua dari satu layar Sambungkan ke penyimpanan cloud.**<br><br>

**Penyimpanan cloud pribadi:** iCloud Drive · Google Drive · Dropbox · OneDrive (Pribadi dan Bisnis) · Box · MEGA · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘).<br><br>

**Server media yang dihosting sendiri dan API server musik:** **Plex** · **Jellyfin** · **Emby** · **Subsonic** (dan setiap server yang kompatibel dengan Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · **Navidrome** · Nextcloud (dan ownCloud melalui API bersama) · **Synology Drive** · **QNAP**.<br><br>

**NAS dan protokol berbagi file:** **SMB** (SMB1, SMB2, Auto) · **WebDAV** (HTTP / HTTPS) · **FTP / FTPS** · **SFTP** (autentikasi kata sandi atau kunci publik) · **NFS** · **DLNA / UPnP** (pemutaran dan unduhan). Berfungsi dengan Apple Time Capsule, Synology, QNAP, WD My Cloud Home, Buffalo, host Linux Samba / NFS / SSH mana pun, atau folder bersama di Mac atau PC Windows Anda.<br><br>

**Penyimpanan objek yang kompatibel dengan S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces, dan layanan lain yang mengekspos endpoint S3-API.<br><br>

**Penemuan jaringan lokal:** bagian **Perangkat yang Tersedia** secara otomatis mencantumkan setiap layanan Bonjour / mDNS di jaringan Wi-Fi Anda — server Plex, Jellyfin, Emby, Synology, QNAP, router AirPort dengan drive yang terpasang, Time Capsule — sehingga Anda dapat mengetuk untuk terhubung tanpa mengetik alamat IP.<br><br>

Setiap koneksi menggunakan **SDK resmi atau protokol terbuka** dari layanan, dengan otorisasi berbasis OAuth jika didukung. Anda dapat menghubungkan beberapa akun dari layanan yang sama (misalnya, dua akun Google Drive, Dropbox pribadi di samping akun kerja, atau server Plex dan Jellyfin). Pengguna Premium dapat menambahkan layanan dalam jumlah tak terbatas; versi gratis dibatasi hingga tiga.<br><br>

{{% details title="Bagaimana cara mengaktifkan mode offline di Flacbox?" closed="true" %}}
**Hubungkan layanan cloud, temukan folder musik, ketuk "Lebih banyak tindakan → Mengaktifkan mode offline" — folder tersebut dan file baru yang ditambahkan ke dalamnya akan diunduh secara otomatis ke File Lokal → Folder offline.**<br><br>

- **Hubungkan Layanan Cloud:** buka tab **Koneksi**. Pilih **Sambungkan layanan cloud** dan ikuti petunjuknya.<br>
- **Navigasikan ke Folder Musik:** buka layanan cloud yang terhubung dan temukan folder musik Anda.<br>
- **Aktifkan Mode Offline:** ketuk tombol **"..."** di sebelah nama folder dan pilih **Mengaktifkan mode offline**.<br>
- **Unduh Konten:** folder yang dipilih dan semua isinya akan diunduh ke **File Lokal → Folder offline**.<br>
- **Pembaruan Otomatis:** aplikasi terus memindai perubahan. File baru yang ditambahkan ke folder online secara otomatis diunduh.<br>
- **Konfigurasikan Batas Waktu Pemindaian:** buka **Pengaturan → Manajer File → Folder offline → Interval Waktu** untuk mengatur frekuensi pemindaian.<br>
- **Sinkronisasi Manual:** untuk menyinkronkan secara manual, buka **Pengaturan → Manajer File → Folder offline → Folder offline yang disinkronkan**. Ketuk **"..."** dan pilih **Sinkronkan**.<br><br>

{{% details title="Bagaimana cara memutar musik yang diunduh secara lokal di iPhone?" closed="true" %}}
**Buka File Lokal, gulir ke "File di iPhone Ini," ketuk "Buka File…" atau "Buka Folder…", dan pilih file audio atau folder yang ingin Anda putar.**<br><br>

Aplikasi memindai konten folder, menambahkan file audio yang ditemukan ke antrean pemutar, dan memutarnya langsung dari lokasi aslinya tanpa menyalin apa pun ke bundel aplikasi.<br><br>

**Menambahkan Folder ke Favorit untuk Akses Cepat**<br>
Buka layar **File Lokal**, gulir ke **Akses Cepat**, dan ketuk **Favorit** untuk mengakses layar **File Favorit**. Ketuk tombol tindakan lainnya **"..."** di sudut kanan atas dan pilih **Tambah Folder**. Navigasikan ke folder yang diinginkan dan ketuk **Buka** untuk mengonfirmasi. Folder Anda akan ditambahkan ke **File Favorit**, memberi Anda akses cepat ke musik Anda tanpa mengulangi langkah-langkah tersebut.<br><br>

**Mengimpor File Lokal ke Perpustakaan Musik**<br>
Jika Anda lebih suka mengatur file lokal Anda dalam perpustakaan musik, buka layar **Perpustakaan Musik**, ketuk tombol tiga titik di sudut kanan atas, dan pilih **+ Tambah Musik**. Pilih item menu **File Lokal**, gulir ke bagian **File di iPhone Ini**, dan ketuk **Buka File…**. Pilih file yang ingin Anda tambahkan dan ketuk **Buka** untuk mengonfirmasi. Aplikasi memindai file yang dipilih dan menambahkannya ke perpustakaan musik Anda, mengaturnya berdasarkan metadata seperti artis, album, dan genre.<br><br>

**Menambahkan File Lokal ke Daftar Putar**<br>
Untuk menambahkan file lokal ke daftar putar, buka layar **Daftar Putar** dan ketuk tombol lainnya di sudut kanan atas. Pilih **+ Daftar Putar Baru**, masukkan nama untuk daftar putar baru Anda, dan di layar berikutnya, pilih opsi **File Lokal**. Gulir ke bagian **File di iPhone Ini** dan ketuk **Buka File…**. Pilih file audio yang ingin Anda tambahkan dan ketuk **Buka** untuk mengonfirmasi. File akan ditambahkan ke daftar putar Anda, di mana Anda dapat mengubah urutan lagu dan melakukan tindakan lain menggunakan tombol lainnya.<br><br>

{{% details title="Bagaimana cara melihat lirik lagu di Flacbox?" closed="true" %}}
**Buka pemutar audio layar penuh, ketuk "..." → Lihat Komentar, lalu geser untuk beralih antara mode Komentar, Lirik Tertanam, dan File LRC.**<br><br>

1. Mulai memutar trek.<br>
2. Buka pemutar audio layar penuh.<br>
3. Ketuk tombol **Lebih banyak tindakan** di sudut kanan atas.<br>
4. Pilih **Lihat Komentar**.<br>
5. Geser ke kanan untuk beralih antara mode **Komentar**, **Lirik Tertanam**, dan **File LRC**.<br><br>

**Mode:**<br>
- **Komentar** — konten dari tag COMMENT.<br>
- **Lirik Tertanam** — lirik dari tag LYRICS; mendukung teks berwaktu dalam format LRC. Edit lirik dengan editor tag eksternal seperti Evertag, atau unduh file LRC dari situs seperti Lyricsify dan tempelkan ke dalam tag.<br>
- **File LRC** — tempatkan file `.lrc` dengan nama dasar yang sama di folder yang sama dengan file audio; aplikasi akan mengambilnya secara otomatis.<br><br>

{{% details title="Bagaimana cara mentransfer musik ke Flacbox dari komputer saya?" closed="true" %}}
**Anda dapat mentransfer musik ke Flacbox melalui SMB, WebDAV, FTP / SFTP, DLNA, Wi-Fi Drive, atau melalui Berbagi File iTunes / Finder — metode apa pun yang mengekspos file ke perangkat Anda akan berhasil.**<br><br>

Untuk menghubungkan komputer menggunakan protokol **SMB**, ketuk **Sambungkan layanan cloud → SMB**. Masukkan alamat IP komputer dan nama folder bersama di bidang URL menggunakan format `smb://alamat-ip-komputer/nama-folder-bersama`, masukkan login dan kata sandi Anda, lalu ketuk **Selesai**. Jika koneksi berhasil, berbagi akan muncul di bagian **Layanan Cloud** di layar Koneksi.<br><br>

[Tutorial lengkap: transfer file Anda dari komputer ke iPhone menggunakan protokol SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

Untuk **WebDAV**, semua langkah sama kecuali untuk bidang URL — gunakan `http://nama-server` atau `https://nama-server` jika server mendukung SSL. WebDAV bekerja dengan baik dengan Synology, QNAP, Nextcloud, ownCloud, dan server lain yang mengekspos endpoint WebDAV.<br><br>

[Tutorial lengkap: cara menghubungkan penyimpanan NAS menggunakan WebDAV dan mendengarkan musik di iPhone atau Mac Anda](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac/)<br><br>

**Wi-Fi Drive** adalah teknologi populer yang memungkinkan Anda mentransfer file dari komputer ke perangkat iOS secara nirkabel menggunakan browser web desktop mana pun. Untuk menggunakan fitur ini, perangkat dan komputer Anda harus berada di jaringan Wi-Fi yang sama. Buka **Koneksi → Komputer → Sambungkan menggunakan Wi-Fi** dan aktifkan server. Kemudian buka browser desktop dan masukkan URL yang ditampilkan oleh aplikasi. Anda dapat menyeret dan melepas file dari komputer ke halaman web yang terbuka dan file tersebut akan muncul di perangkat Anda hampir seketika.<br><br>

**Berbagi File iTunes / Finder** adalah opsi lain, mentransfer file dari komputer ke perangkat melalui kabel Lightning atau USB-C. Hubungkan perangkat ke komputer, buka iTunes (atau Finder di macOS Catalina dan versi lebih baru), pilih perangkat, temukan **Flacbox** di daftar aplikasi, dan salin file ke folder bersamanya.<br><br>

{{% details title="Apakah Flacbox mendukung FTP, FTPS, SFTP, dan NFS?" closed="true" %}}

{{% details title="Apakah Flacbox memutar file DSD, DSF, dan DFF?" closed="true" %}}

Untuk hasil terbaik dengan DSD, alihkan **Codec Audio** ke **FFmpeg** di **Pengaturan → Pemutar Audio → Umum** dan pilih **Sample Rate Output Audio** sebesar **88,2 kHz** atau **96 kHz** yang sesuai dengan DAC Anda. {{% details title="Apakah Flacbox mendukung audio Hi-Res (24-bit / 96 kHz)?" closed="true" %}}

Di **Pengaturan → Pemutar Audio → Umum**, alihkan **Codec Audio** ke **FFmpeg**, lalu pilih **Sample Rate Output Audio** dan jumlah saluran yang Anda inginkan. Pasangkan ini dengan **DAC eksternal atau antarmuka audio USB / Lightning** yang kompatibel untuk mendapatkan output bit-perfect dari iPhone, iPad, atau Mac Anda.

Untuk mengganti sampul tertentu secara manual, ketuk **"..." → Edit Gambar** pada album, daftar putar, atau trek dan pilih gambar baru dari perpustakaan foto Anda, aplikasi Files, atau salah satu layanan cloud yang terhubung. Untuk kontrol lebih mendalam atas artwork yang tersimpan di dalam file audio itu sendiri, gunakan **Mengedit tag audio** atau aplikasi pendamping **Evertag** kami.
{{% /details %}}

{{% details title="Bagaimana cara menghapus lagu dari Flacbox tanpa menghapusnya dari penyimpanan cloud saya?" closed="true" %}}
**Pada trek mana pun, ketuk "..." → Hapus dari Perpustakaan Musik — ini menghapus lagu dari database perpustakaan Anda tetapi membiarkan file tidak tersentuh di penyimpanan cloud dan File Lokal.**<br><br>

Untuk juga menghapus file dari cloud atau perangkat lokal, pilih **Hapus dari Layanan Cloud** atau **Hapus dari File Lokal**. Tindakan destruktif ini tidak dapat dibatalkan, jadi berhati-hatilah saat Anda memiliki beberapa file yang dipilih.
{{% /details %}}

{{% details title="Bagaimana cara mencadangkan dan memulihkan perpustakaan musik Flacbox saya?" closed="true" %}}
**Buka Pengaturan → Cadangan & Pulihkan, pilih apa yang akan disertakan (Database, Sampul Album, Pengaturan), ketuk 'Cadangkan Data Aplikasi,' dan simpan file cadangan — buka di perangkat lain untuk memulihkan.**<br><br>

Cadangan berisi entri perpustakaan musik, daftar putar, peringkat, favorit, pengaturan, dan cache sampul album. Ini tidak menyertakan file audio yang diunduh secara offline, untuk menjaga ukuran file tetap dapat dikelola. Pindahkan file cadangan ke perangkat baru melalui iCloud Drive, AirDrop, atau layanan cloud yang terhubung, lalu buka di Flacbox untuk diterapkan.<br><br>

{{% details title="Bagaimana cara berbagi Flacbox Premium dengan keluarga saya?" closed="true" %}}
**Semua paket Flacbox Premium — seumur hidup, bulanan, dan tahunan — bekerja dengan Apple Family Sharing, sehingga siapa pun dalam grup keluarga Anda dapat menginstal Flacbox dan menggunakan Premium tanpa biaya tambahan.**<br><br>

Siapkan Family Sharing di **Pengaturan → Keluarga** iOS / macOS, lalu mintalah setiap anggota keluarga menginstal Flacbox dari App Store dan menjalankannya sekali saat masuk ke Apple ID mereka sendiri. Premium dikenali secara otomatis dalam satu menit. Paket yang sama dibagikan antara iPhone, iPad, dan Mac untuk setiap anggota keluarga.
{{% /details %}}

{{% details title="Bagaimana cara membatalkan langganan Flacbox Premium saya?" closed="true" %}}
**Buka Pengaturan iOS atau macOS → [nama Anda] → Langganan, temukan Flacbox, dan ketuk Batalkan Langganan — fitur Premium Anda tetap aktif hingga akhir periode penagihan saat ini.**<br><br>

Pembelian dalam aplikasi seumur hidup bukan merupakan langganan dan tidak perlu dibatalkan. Untuk pengembalian dana, gunakan halaman **Laporkan Masalah** Apple (`reportaproblem.apple.com`) — pengembalian dana dikeluarkan oleh Apple, bukan oleh Everappz.
{{% /details %}}

{{% details title="Bagaimana cara melindungi Flacbox dengan kode sandi?" closed="true" %}}
**Buka Pengaturan → Kode Sandi, ketuk Aktifkan, dan pilih kode 4 digit — Anda akan diminta untuk memasukkannya setiap kali aplikasi diluncurkan.**<br><br>

Flacbox menggunakan kode sandi numerik 4 digit yang tetap. Kode sandi mencegah siapa pun yang memiliki akses ke perangkat Anda membuka Flacbox dan menelusuri akun cloud yang terhubung, file yang diunduh, dan perpustakaan. Gabungkan dengan iOS Face ID / Touch ID di perangkat untuk perlindungan ekstra.
{{% /details %}}

{{% details title="Bagaimana cara mengaktifkan mode gelap di Flacbox?" closed="true" %}}
**Buka Pengaturan → Personalisasi → Skema Warna, lalu pilih Gelap, Terang, atau Default (yang mengikuti tampilan sistem Anda).**<br><br>

Anda juga dapat memilih ikon aplikasi alternatif di **Pengaturan → Personalisasi → Ikon Aplikasi** (Premium), dan memilih sampul album yang diburamkan sebagai latar belakang aplikasi di bawah **Gaya Latar Belakang**.
{{% /details %}}

{{% details title="Bagaimana cara mengubah kecepatan pemutaran di Flacbox?" closed="true" %}}
**Buka pemutar audio, ketuk kontrol Kecepatan, dan seret slider — kecepatan dapat disesuaikan dari 0,02× hingga 3,00×.**<br><br>

Anda juga dapat mengubah kecepatan default di **Pengaturan → Pemutar Audio → Kecepatan Pemutaran**. Ketuk ikon konfigurasi di sudut kanan atas untuk beralih ke **mode presisi** untuk penyesuaian yang sangat halus — berguna untuk buku audio, ceramah, dan konten pembelajaran bahasa.
{{% /details %}}

{{% details title="Bagaimana cara mengatur timer tidur di Flacbox?" closed="true" %}}
**Buka Pengaturan → Pemutar Audio → Timer Tidur, aktifkan, dan pilih berapa lama Anda ingin musik diputar sebelum berhenti.**<br><br>

Anda juga dapat mengaktifkan tombol **Timer Tidur** langsung di layar pemutar audio dengan menambahkannya melalui **Pengaturan → Pemutar Audio → Personalisasi → Tindakan Layar Utama**. Ketuk ikon konfigurasi di sudut kanan atas layar Timer Tidur untuk mengaktifkan penyesuaian menit demi menit yang presisi.
{{% /details %}}

{{% details title="Bisakah saya menggunakan Flacbox tanpa koneksi internet?" closed="true" %}}
**Ya — setelah Anda mengunduh musik atau mengaktifkan mode offline untuk folder, Flacbox memainkan segalanya sepenuhnya offline.**<br><br>

Konten offline berada di bawah **File Lokal** dan terus berfungsi dalam mode pesawat, di penerbangan, dan di mana saja tanpa Wi-Fi atau data seluler. Trek yang hanya ada di cloud (yang belum Anda unduh) akan berwarna abu-abu hingga Anda mendapatkan kembali koneksi. Untuk perjalanan, aktifkan **Mode Offline** untuk folder yang relevan atau unduh album dan daftar putar tertentu sebelum Anda pergi.
{{% /details %}}

{{% details title="Apakah Flacbox melakukan scrobble ke Last.fm?" closed="true" %}}
**Ya — Flacbox mendukung scrobbling Last.fm, sehingga setiap trek yang Anda putar secara otomatis dicatat ke profil Last.fm Anda.**<br><br>

Hubungkan akun Last.fm Anda dari bagian **Koneksi → Layanan Lainnya**. Setelah terhubung, semua statistik pemutaran dikirim ke Last.fm di latar belakang. Anda kemudian dapat mengunjungi profil Last.fm Anda untuk melihat tren mendengarkan, artis teratas mingguan, dan rekomendasi yang dipersonalisasi.<br><br>

{{% details title="Bagaimana cara mengubah bahasa antarmuka Flacbox?" closed="true" %}}
**Buka Pengaturan → Bahasa, pilih dari lebih dari 120 bahasa yang didukung, lalu restart aplikasi agar perubahan diterapkan.**<br><br>

Aplikasi mendukung lokalisasi termasuk Inggris, Prancis, Jerman, Spanyol, Italia, Portugis, Rusia, Ukraina, Polandia, Belanda, Arab, Ibrani, Hindi, Jepang, Korea, Mandarin (Sederhana dan Tradisional), Vietnam, Turki, dan banyak lagi. Pilih **Default** untuk secara otomatis mengikuti pengaturan bahasa perangkat. Setelah memilih bahasa baru, keluar sepenuhnya dari Flacbox dan buka kembali sehingga setiap layar digambar ulang dengan terjemahan baru.

{{% details title="Bagaimana cara mencari lagu, album, atau artis di Flacbox?" closed="true" %}}
**Ketuk ikon kaca pembesar di daftar mana pun — Perpustakaan Musik, Daftar Putar, Album, Artis, Genre, atau di dalam folder — dan ketik nama untuk memfilter hasil secara instan.**<br><br>

Pencarian bersifat lokal dan berfungsi sepenuhnya offline terhadap database perpustakaan Anda, sehingga hasil muncul saat Anda mengetik bahkan di jaringan yang lambat. Anda juga dapat mencari di dalam daftar putar, album, atau folder tertentu untuk menemukan satu trek di antara ratusan. Trek, album, artis, genre, komposer, dan folder semuanya dapat dicari.

{{% details title="Apakah Flacbox bekerja dengan headphone Bluetooth, AirPods, dan DAC eksternal?" closed="true" %}}
**Ya — Flacbox memutar melalui output audio mana pun yang dapat digunakan iPhone, iPad, atau Mac Anda: speaker bawaan, headphone berkabel, perangkat Bluetooth (AirPods, Beats, Sony, Bose, dll.), dan DAC USB / Lightning.**<br><br>

Untuk output Hi-Res ke DAC eksternal, alihkan codec audio ke **FFmpeg** di **Pengaturan → Pemutar Audio → Umum** dan pilih **Sample Rate Output Audio** dan jumlah saluran yang sesuai. Kontrol on-stem AirPods (putar / jeda, berikutnya / sebelumnya) bekerja seperti yang diharapkan, dan kontrol layar kunci dapat dikustomisasi di **Pengaturan → Pemutar Audio → Personalisasi → Kontrol Pemutaran di Layar Kunci**.
{{% /details %}}

{{% details title="Bagaimana cara menggunakan flash drive USB atau kartu SD dengan Flacbox di iPhone atau iPad?" closed="true" %}}
**Colokkan drive ke iPhone atau iPad Anda melalui adaptor Lightning-to-USB atau USB-C, lalu di Flacbox buka File Lokal → File di iPhone Ini → Buka Folder, navigasikan ke drive, dan pilih folder musik.**<br><br>

Flacbox memutar file langsung dari drive tanpa menyalinnya ke penyimpanan internal, yang berguna untuk perpustakaan lossless yang sangat besar. Anda juga dapat mengimpor trek ke perpustakaan musik atau menambahkannya ke daftar putar dengan cara yang sama. Aplikasi mendukung **pembaca kartu bersertifikat Apple** dan **iXpand Flash Drives** — untuk iXpand, masukkan ke port Lightning dan drive muncul secara otomatis di layar Koneksi. Lihat instruksi langkah demi langkah lengkap di [Cara menghubungkan flash drive USB ke iPhone dan mendengarkan musik](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).

{{% details title="Mengapa musik cloud saya tidak tersinkronisasi di Flacbox?" closed="true" %}}
**Sebagian besar masalah sinkronisasi disebabkan oleh token auth yang kedaluwarsa, aplikasi yang berjalan di latar belakang, atau tidak ada koneksi internet aktif — buka Koneksi, otorisasi ulang layanan, lalu jalankan sinkronisasi secara manual dari Pengaturan → Perpustakaan → Sinkronisasi Online.**<br><br>

Sinkronisasi online hanya berjalan ketika aplikasi ada di latar depan, sehingga menyinkronkan perpustakaan besar bisa memakan waktu. Untuk mempercepatnya, jaga Flacbox tetap terbuka, sambungkan perangkat Anda ke daya, dan aktifkan **Pengaturan → Layar → Selalu Aktif**. Untuk perpustakaan yang sangat besar, jalankan sinkronisasi pada versi desktop aplikasi dan transfer hasilnya ke iOS menggunakan **Cadangan & Pulihkan**.

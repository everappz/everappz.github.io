Flacbox 是適用於 **iPhone、iPad 和 Mac** 的 **hi-res 音樂播放器**，可從 iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Box、pCloud、Yandex Disk、Synology、QNAP、WD My Cloud Home 以及任何其他 WebDAV、SMB 或 DLNA 伺服器串流和下載曲目。基於系統編解碼器和內建 **FFmpeg** 程式庫建構，它增加了對 Apple 預設播放器無法處理的格式的支援——WMA、OGG、OPUS、APE、DSD——並為發燒友提供對輸出取樣率、多聲道路由、音調校正和 IO 緩衝區持續時間的控制。本 FAQ 回答了使用者最常發送的問題。如需更深入的指南，請參閱 [Flacbox 使用者指南](/docs/guide/flacbox/) 或瀏覽我們的 [操作文章](/docs/howto/)。

{{% details title="Flacbox 如何運作？" closed="true" %}}
**Flacbox 是一款 hi-res 音樂播放器，讓您可以像管理一般檔案一樣管理音訊曲目**，並從任何雲端服務或 NAS 進行串流——無需 iTunes 同步。<br><br>

您可以將整個音樂收藏上傳到 **Dropbox、Google Drive、OneDrive、iCloud Drive、MEGA、Box、pCloud** 等雲端服務或個人 NAS，並直接從雲端播放音樂，擁有完全的控制權。若要離線收聽，請使用內建下載管理器將音樂儲存到裝置。檔案以您放置的方式儲存在您的儲存空間中——Flacbox 只是簡單地播放它們。<br><br>

{{% details title="Flacbox 和 Evermusic 有什麼區別？" closed="true" %}}
**Flacbox 是專注於發燒友的播放器，具有 FFmpeg 驅動的格式支援、可配置的取樣率、多聲道輸出和音調校正；Evermusic 是主流播放器，具有交叉淡入淡出、無縫播放、空間音訊和更廣泛的雲端功能集。**<br><br>

**Flacbox** 旨在支援所有預設 iOS 音訊格式，以及 iPhone 上原生不支援的許多附加格式，如 **WMA、OGG、OPUS、APE、DSD、DSF、DFF、TTA、MPC、WV** 等。它使用自訂音訊引擎（系統編解碼器 + FFmpeg）處理幾乎每種格式，並提供發燒級功能，如**可調音訊輸出取樣率（44.1 – 96 kHz）、多聲道輸出（Mono → 5.1 → SDDS）、IO 緩衝區調整和音調校正（-1000 至 +1000）**。<br><br>

**Evermusic** 支援預設 Apple 音訊格式，並包含消費者友善的功能，如**交叉淡入淡出播放、無縫播放、空間音訊、音訊等化器、播放速度控制**和更廣泛的雲端串流工具。如果您主要使用 MP3、AAC、ALAC 或 FLAC 並希望使用交叉淡入淡出、無縫或空間音訊，**Evermusic** 可能是更好的選擇。如果您需要與更廣泛的音訊格式相容——特別是 DSD 或 WMA——並希望對音訊輸出鏈進行精細控制，**Flacbox** 是正確的選擇。<br><br>

[了解更多關於差異的資訊](/docs/faq/evermusic/what-is-the-difference-between-evermusic-and-flacbox/)

{{% details title="Flacbox 安全嗎？" closed="true" %}}
**是的——Flacbox 僅使用官方 SDK 和安全的 OAuth 連接與連接的雲端服務互動，從不看到您的密碼，並且僅在 iOS Keychain 中儲存加密的存取權杖。**<br><br>

當您輸入登入名稱和密碼時，應用程式會顯示雲端服務提供者提供的官方授權頁面，整個授權過程在 Flacbox 之外進行。成功授權後，雲端提供者會向應用程式發送 **auth-token**，該權杖用於進行 API 呼叫。<br><br>

auth-token 是允許第三方應用程式與雲端儲存互動的數位金鑰。權杖儲存在裝置的 **Apple 安全系統儲存（Keychain）**中，靜態加密，並受裝置密碼或生物識別鎖保護。下載的檔案儲存在應用程式的 **Documents** 目錄中；您可以隨時使用內建檔案管理器刪除它們。<br><br>

應用程式**不會與 Everappz、廣告商或任何第三方共享**來自已連接雲端帳戶的任何資訊。您可以隨時透過在網路瀏覽器中開啟帳戶設定頁面來撤銷對您雲端帳戶的存取權限。要撤銷 auth-token，請在網路上登入您的雲端帳戶，導航到安全性或已連接應用程式頁面，然後刪除 Flacbox 的條目。<br><br>

您也可以在應用程式內部斷開雲端帳戶連接——這樣做時，auth-token 會立即從您的裝置中刪除。解除安裝應用程式會自動刪除所有已下載的資料和權杖。<br><br>

{{% details title="Flacbox 支援哪些雲端服務、媒體伺服器和協定？" closed="true" %}}
**Flacbox 支援幾乎所有流行的雲端儲存服務、自託管媒體伺服器、檔案共享協定和 S3 相容物件儲存——全部透過一個「連接到雲端儲存」畫面完成。**<br><br>

**個人雲端儲存：** iCloud Drive · Google Drive · Dropbox · OneDrive（個人和商業）· Box · MEGA · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive（阿里雲盤）· Baidu Pan（百度網盤）。<br><br>

**自託管媒體伺服器和音樂伺服器 API：** **Plex** · **Jellyfin** · **Emby** · **Subsonic**（以及每個 Subsonic 相容伺服器——Airsonic、Funkwhale、Gonic、Logitech Media Server、Ampache）· **Navidrome** · Nextcloud（以及透過共享 API 的 ownCloud）· **Synology Drive** · **QNAP**。<br><br>

**NAS 和檔案共享協定：** **SMB**（SMB1、SMB2、Auto）· **WebDAV**（HTTP / HTTPS）· **FTP / FTPS** · **SFTP**（密碼或公開金鑰驗證）· **NFS** · **DLNA / UPnP**（播放和下載）。適用於 Apple Time Capsule、Synology、QNAP、WD My Cloud Home、Buffalo、任何 Linux Samba / NFS / SSH 主機，或 Mac 或 Windows PC 上的共享資料夾。<br><br>

**S3 相容物件儲存：** AWS S3、Backblaze B2、Wasabi、Cloudflare R2、MinIO、DigitalOcean Spaces，以及任何其他 S3-API 端點。<br><br>

**本地網路探索：** **可用設備**部分自動列出 Wi-Fi 網路上的每個 Bonjour / mDNS 服務——Plex、Jellyfin、Emby 伺服器、Synology、QNAP、附有連接磁碟的 AirPort 路由器、Time Capsule——這樣您可以點選連接而無需輸入 IP 位址。<br><br>

每個連接都使用服務的**官方 SDK 或開放協定**，在支援的情況下使用基於 OAuth 的授權。您可以連接同一服務的多個帳戶（例如，兩個 Google Drive 帳戶、個人 Dropbox 和工作 Dropbox，或同時連接 Plex 和 Jellyfin 伺服器）。Premium 使用者可以新增無限數量的服務；免費版本限制為三個。<br><br>

{{% details title="如何在 iPhone 上播放本機下載的音樂？" closed="true" %}}
**開啟本機檔案，捲動到「此 iPhone 上的檔案」，點選「開啟檔案…」或「開啟資料夾…」，然後選擇要播放的音訊檔案或資料夾。**<br><br>

應用程式掃描資料夾內容，將找到的音訊檔案新增到播放器佇列，並直接從其原始位置播放，無需將任何內容複製到應用程式套件。<br><br>

**將資料夾新增到最愛項目以便快速存取**<br>
開啟**本機檔案**畫面，捲動到**快速存取**，點選**最愛項目**以存取**最愛檔案**畫面。點選右上角的更多操作按鈕 **「...」** 並選擇**新增資料夾**。導航到所需資料夾並點選**開啟**確認。您的資料夾將被新增到**最愛檔案**，讓您無需重複步驟即可快速存取音樂。<br><br>

**將本機檔案匯入音樂庫**<br>
如果您希望在音樂庫中整理本機檔案，開啟**音樂庫**畫面，點選右上角的三點按鈕，選擇 **+ 新增音樂**。選擇**本機檔案**選單項目，捲動到**此 iPhone 上的檔案**部分，點選**開啟檔案…**。選擇要新增的檔案並點選**開啟**確認。應用程式掃描所選檔案並將其新增到音樂庫，按藝術家、專輯和類型等中繼資料進行整理。<br><br>

**將本機檔案新增到播放列表**<br>
要將本機檔案新增到播放列表，開啟**播放列表**畫面並點選右上角的更多按鈕。選擇 **+ 新建播放列表**，輸入新播放列表的名稱，在下一個畫面上選擇**本機檔案**選項。捲動到**此 iPhone 上的檔案**部分並點選**開啟檔案…**。選擇要新增的音訊檔案並點選**開啟**確認。檔案將被新增到播放列表，您可以使用更多按鈕變更歌曲順序並執行其他操作。<br><br>

{{% details title="如何在 Flacbox 中檢視歌詞？" closed="true" %}}
**開啟全螢幕音訊播放器，點選「...」→ 檢視評論，然後滑動在評論、嵌入歌詞和 LRC 檔案模式之間切換。**<br><br>

1. 開始播放曲目。<br>
2. 開啟全螢幕音訊播放器。<br>
3. 點選右上角的**更多操作**按鈕。<br>
4. 選擇**檢視評論**。<br>
5. 向右滑動在**評論**、**嵌入歌詞**和 **LRC 檔案**模式之間切換。<br><br>

**模式：**<br>
- **評論** — 來自 COMMENT 標籤的內容。<br>
- **嵌入歌詞** — 來自 LYRICS 標籤的歌詞；支援 LRC 格式的定時文字。使用外部標籤編輯器（如 Evertag）編輯歌詞，或從 Lyricsify 等網站下載 LRC 檔案並貼上到標籤中。<br>

許多發燒級 DAC 傾向於 **88.2 kHz** 或 **96 kHz** 配合**立體聲**輸出。將**音訊輸出首選 IO 緩衝區持續時間**降低到約 **5 ms（0.005 s）**以實現低延遲，如果聽到故障，則稍微增加。Flacbox 適用於大多數 class-compliant USB DAC 和 Lightning DAC，包括 AudioQuest、iFi、Chord、FiiO、Topping 等品牌的產品。
{{% /details %}}

{{% details title="Flacbox 可以播放 WMA、OGG、OPUS 和 APE 檔案嗎？" closed="true" %}}
**是的——Flacbox 透過內建 FFmpeg 引擎播放 WMA、OGG、OPUS、APE、TTA、MPC、WV 和許多其他不常見格式**，即使 iOS 原生不支援它們。<br><br>

不需要轉換：將檔案放入 iCloud Drive、Dropbox、Google Drive 或您的 NAS，在 Flacbox 中連接該服務，檔案就可以播放了。在有嵌入標籤、專輯封面和歌詞的地方都會讀取。
{{% /details %}}

{{% details title="Flacbox 支援多聲道或 5.1 環繞聲輸出嗎？" closed="true" %}}
**是的——Flacbox 允許您在設定 → 音訊播放器 → 一般 → 音訊輸出聲道數中選擇 Mono、Stereo、Center/Left/Right、Center/Left/Right/Surround、ITU BS.775-1、5.1 Surround 和 SDDS**（需要 FFmpeg 編解碼器）。<br><br>

這對外接多聲道 DAC 和 AV 接收器很有用。大多數使用者應堅持使用 **Stereo**，除非他們有連接的特定多聲道硬體，因為 iPhone 的內建揚聲器和大多數耳機僅為立體聲。
{{% /details %}}

{{% details title="Flacbox 支援音調校正嗎？" closed="true" %}}
**是的——Flacbox 提供從 -1000 到 +1000 音分的寬範圍音調校正**，可在**設定 → 音訊播放器 → 音調校正**中配置。<br><br>

使用**精確模式**（點選右上角的設定圖示）進行更精細的調整。音調校正在 **FFmpeg** 編解碼器路徑中可用，因此請確保在**設定 → 音訊播放器 → 一般 → 音訊編解碼器**下選擇了 **FFmpeg** 才能存取它。
{{% /details %}}

{{% details title="如何將 M3U、M3U8 或 CUE 播放列表匯入 Flacbox？" closed="true" %}}

應用程式解析播放列表檔案，在您的儲存空間上定位每個引用的曲目，並在音樂庫中建立真實的播放列表。支援的檔案副檔名為 **M3U**、**M3U8** 和 **CUE**。確保播放列表檔案中的路徑與音訊檔案實際儲存的位置匹配。<br><br>

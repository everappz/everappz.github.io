Flacbox è un **lettore musicale hi-res** per **iPhone, iPad e Mac** che esegue lo streaming e scarica brani da iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex Disk, Synology, QNAP, WD My Cloud Home e qualsiasi altro server WebDAV, SMB o DLNA. Basato sui codec di sistema e sulla libreria **FFmpeg** inclusa, aggiunge il supporto per i formati che il lettore predefinito di Apple non può gestire — WMA, OGG, OPUS, APE, DSD — e fornisce agli audiofili il controllo sulla frequenza di campionamento di output, il routing multicanale, la correzione dell'intonazione e la durata del buffer IO. Questa FAQ risponde alle domande che gli utenti inviano più spesso. Per guide più approfondite, consulta la [Guida utente di Flacbox](/docs/guide/flacbox/) o sfoglia i nostri [articoli pratici](/docs/howto/).

{{% details title="Qual è la differenza tra Flacbox ed Evermusic?" closed="true" %}}
**Flacbox è il lettore focalizzato sugli audiofili con supporto di formati basato su FFmpeg, sample rate configurabile, output multicanale e correzione dell'intonazione; Evermusic è il lettore mainstream con crossfade, gapless, audio spaziale e un set di funzionalità cloud più ampio.**<br><br>

**Flacbox** è progettato per supportare tutti i formati audio iOS predefiniti insieme a molti formati aggiuntivi non supportati nativamente su iPhone, come **WMA, OGG, OPUS, APE, DSD, DSF, DFF, TTA, MPC, WV,** e altri. Utilizza un motore audio personalizzato (codec di sistema + FFmpeg) per gestire quasi ogni formato, e offre funzionalità di qualità audiofila come **sample rate di output audio regolabile (44,1 – 96 kHz), output multicanale (Mono → 5.1 → SDDS), ottimizzazione buffer IO e correzione dell'intonazione (-1000 a +1000)**.<br><br>

**Evermusic** supporta i formati audio Apple predefiniti e include funzionalità user-friendly come **riproduzione crossfade, riproduzione gapless, audio spaziale, equalizzatore audio, controllo della velocità di riproduzione,** e strumenti di streaming cloud più ampi. Se usi principalmente MP3, AAC, ALAC o FLAC e vuoi crossfade, gapless o audio spaziale, **Evermusic** potrebbe essere la scelta migliore. Se hai bisogno di ampia compatibilità con una gamma più vasta di formati audio — specialmente DSD o WMA — e vuoi un controllo preciso sulla catena di output audio, **Flacbox** è la scelta giusta.<br><br>

{{% details title="Flacbox è sicuro?" closed="true" %}}
**Sì — Flacbox utilizza solo SDK ufficiali e connessioni OAuth sicure per interagire con i servizi cloud collegati, non vede mai la tua password e archivia solo token di accesso crittografati nel Keychain di iOS.**<br><br>

Quando inserisci il tuo login e la password, l'applicazione mostra la pagina di autorizzazione ufficiale fornita dal provider del servizio cloud, e l'intero processo di autorizzazione avviene al di fuori di Flacbox. Il provider cloud invia quindi un **auth-token** all'applicazione dopo l'autorizzazione riuscita, e quel token viene utilizzato per effettuare chiamate API.<br><br>

Un auth-token è una chiave digitale che consente alle applicazioni di terze parti di interagire con l'archiviazione cloud. Il token viene archiviato sul tuo dispositivo nell'**archivio di sistema sicuro di Apple (Keychain)**, che è crittografato a riposo e protetto dal codice di accesso del tuo dispositivo o dal blocco biometrico. I file scaricati vengono archiviati nella directory **Documenti** dell'app; puoi rimuoverli in qualsiasi momento usando il gestore file integrato.<br><br>

L'applicazione **non condivide alcuna informazione** dagli account cloud collegati con Everappz, inserzionisti o terze parti. Puoi revocare l'accesso al tuo account cloud in qualsiasi momento aprendo la pagina delle impostazioni dell'account nel tuo browser web. Per revocare un auth-token, accedi al tuo account cloud sul web, vai alla pagina della sicurezza o delle app collegate e rimuovi la voce per Flacbox.<br><br>

Puoi anche disconnettere l'account cloud all'interno dell'applicazione stessa — quando lo fai, l'auth-token viene immediatamente eliminato dal tuo dispositivo. La disinstallazione dell'applicazione elimina automaticamente tutti i dati scaricati e i token.<br><br>

{{% details title="Quali servizi cloud, server multimediali e protocolli supporta Flacbox?" closed="true" %}}
**Flacbox supporta praticamente ogni popolare servizio di archiviazione cloud, server multimediale self-hosted, protocollo di condivisione file e archivio oggetti compatibile con S3 — tutto da un'unica schermata Connetti all'archiviazione cloud.**<br><br>

**Archiviazione cloud personale:** iCloud Drive · Google Drive · Dropbox · OneDrive (Personale e Business) · Box · MEGA · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘).<br><br>

**Server multimediali self-hosted e API server musicali:** **Plex** · **Jellyfin** · **Emby** · **Subsonic** (e ogni server compatibile con Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · **Navidrome** · Nextcloud (e ownCloud tramite l'API condivisa) · **Synology Drive** · **QNAP**.<br><br>

**NAS e protocolli di condivisione file:** **SMB** (SMB1, SMB2, Auto) · **WebDAV** (HTTP / HTTPS) · **FTP / FTPS** · **SFTP** (autenticazione con password o chiave pubblica) · **NFS** · **DLNA / UPnP** (riproduzione e download). Funziona con Apple Time Capsule, Synology, QNAP, WD My Cloud Home, Buffalo, qualsiasi host Linux Samba / NFS / SSH, o una cartella condivisa sul tuo Mac o PC Windows.<br><br>

**Archiviazione oggetti compatibile con S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces e qualsiasi altro servizio che espone un endpoint S3-API.<br><br>

**Rilevamento della rete locale:** la sezione **Dispositivi disponibili** elenca automaticamente ogni servizio Bonjour / mDNS sulla tua rete Wi-Fi — server Plex, Jellyfin, Emby, Synology, QNAP, router AirPort con unità collegate, Time Capsule — in modo da poter toccare per connettersi senza digitare un indirizzo IP.<br><br>

Ogni connessione utilizza l'**SDK ufficiale o il protocollo aperto** del servizio, con autorizzazione basata su OAuth dove supportato. Puoi collegare più account dello stesso servizio (ad esempio, due account Google Drive, un Dropbox personale accanto a uno lavorativo, o sia un server Plex che Jellyfin). Gli utenti Premium possono aggiungere un numero illimitato di servizi; la versione gratuita è limitata a tre.<br><br>

{{% details title="Come abilito la modalità offline in Flacbox?" closed="true" %}}

{{% details title="Come riproduco la musica scaricata localmente su iPhone?" closed="true" %}}
**Apri File Locali, scorri fino a 'File su questo iPhone', tocca 'Apri File...' o 'Apri Cartella...', e seleziona i file audio o la cartella che vuoi riprodurre.**<br><br>

L'app scansiona il contenuto della cartella, aggiunge i file audio trovati alla coda del lettore e li riproduce direttamente dalla loro posizione originale senza copiare nulla nel pacchetto applicativo.<br><br>

**Aggiungere una Cartella ai Preferiti per Accesso Rapido**<br>
Apri la schermata **File Locali**, scorri fino ad **Accesso Rapido** e tocca **Preferiti** per accedere alla schermata **File Preferiti**. Tocca il pulsante altre azioni **"..."** nell'angolo in alto a destra e seleziona **Aggiungi Cartella**. Vai alla cartella desiderata e tocca **Apri** per confermare. La tua cartella verrà aggiunta a **File Preferiti**, dandoti accesso rapido alla tua musica senza ripetere i passaggi.<br><br>

**Importare File Locali nella Libreria Musicale**<br>
Se preferisci organizzare i tuoi file locali all'interno della tua libreria musicale, apri la schermata **Libreria Musicale**, tocca il pulsante a tre punti nell'angolo in alto a destra e seleziona **+ Aggiungi Musica**. Scegli la voce di menu **File Locali**, scorri fino alla sezione **File su questo iPhone** e tocca **Apri File...**. Seleziona i file che vuoi aggiungere e tocca **Apri** per confermare. L'app scansiona i file selezionati e li aggiunge alla tua libreria musicale, organizzandoli per metadati come artista, album e genere.<br><br>

**Aggiungere File Locali a una Playlist**<br>
Per aggiungere file locali a una playlist, apri la schermata **Playlist** e tocca il pulsante altro nell'angolo in alto a destra. Seleziona **+ Nuova Playlist**, inserisci un nome per la tua nuova playlist, e nella schermata successiva seleziona l'opzione **File Locali**. Scorri fino alla sezione **File su questo iPhone** e tocca **Apri File...**. Seleziona i file audio che vuoi aggiungere e tocca **Apri** per confermare. I file verranno aggiunti alla tua playlist, dove puoi cambiare l'ordine dei brani ed eseguire altre azioni usando il pulsante altro.<br><br>

{{% details title="Come trasferisco musica su Flacbox dal mio computer?" closed="true" %}}
**Puoi trasferire musica su Flacbox tramite SMB, WebDAV, FTP / SFTP, DLNA, Wi-Fi Drive o tramite Condivisione File iTunes / Finder — qualsiasi metodo che esponga i file al tuo dispositivo funziona.**<br><br>

Se la connessione ha successo, la condivisione apparirà nella sezione **Servizi Cloud** della schermata Connessioni.<br><br>

Per usare questa funzione, il tuo dispositivo e il computer devono essere sulla stessa rete Wi-Fi. Apri **Connessioni → Computer → Connetti usando Wi-Fi** e abilita il server. Poi apri un browser desktop e inserisci l'URL mostrato dall'app. Puoi trascinare e rilasciare file dal tuo computer sulla pagina web aperta e appariranno sul tuo dispositivo quasi immediatamente.<br><br>

Connetti il dispositivo al computer, apri iTunes (o Finder su macOS Catalina e versioni successive), seleziona il dispositivo, trova **Flacbox** nell'elenco delle app e copia i file nella sua cartella condivisa.<br><br>

Gli acquisti in-app a vita non sono abbonamenti e non devono essere annullati. Per i rimborsi, usa la pagina **Segnala un Problema** di Apple (`reportaproblem.apple.com`) — i rimborsi vengono emessi da Apple, non da Everappz.
{{% /details %}}

{{% details title="Come proteggo Flacbox con un codice di accesso?" closed="true" %}}
**Apri Impostazioni → Codice di Accesso, tocca Abilita e scegli un codice a 4 cifre — ti verrà chiesto di inserirlo ogni volta che l'app si avvia.**<br><br>

Flacbox usa un codice di accesso numerico fisso a 4 cifre. Il codice di accesso impedisce a chiunque abbia accesso al tuo dispositivo di aprire Flacbox e sfogliare i tuoi account cloud collegati, file scaricati e libreria. Combinalo con iOS Face ID / Touch ID sul dispositivo per una protezione extra.
{{% /details %}}

{{% details title="Come abilito la modalità scura in Flacbox?" closed="true" %}}
**Apri Impostazioni → Personalizzazione → Schema Colori, poi scegli Scuro, Chiaro o Predefinito (che segue l'aspetto del tuo sistema).**<br><br>

Puoi anche scegliere icone alternative dell'app in **Impostazioni → Personalizzazione → Icona Applicazione** (Premium), e scegliere una copertina dell'album sfocata come sfondo dell'app in **Stile Sfondo**.

{{% details title="Come cambio la velocità di riproduzione in Flacbox?" closed="true" %}}
**Apri il lettore audio, tocca il controllo Velocità e trascina il cursore — la velocità è regolabile da 0,02× a 3,00×.**<br><br>

Puoi anche cambiare la velocità predefinita in **Impostazioni → Lettore Audio → Velocità di Riproduzione**. Tocca l'icona di configurazione nell'angolo in alto a destra per passare alla **modalità precisa** per regolazioni molto fini — utile per audiolibri, conferenze e contenuti di apprendimento delle lingue.
{{% /details %}}

{{% details title="Come imposto un timer di sospensione in Flacbox?" closed="true" %}}
**Apri Impostazioni → Lettore Audio → Timer di Sospensione, attivalo e scegli per quanto tempo vuoi che la musica venga riprodotta prima di fermarsi.**<br><br>

Puoi anche abilitare il pulsante **Timer di Sospensione** direttamente sulla schermata del lettore audio aggiungendolo tramite **Impostazioni → Lettore Audio → Personalizzazione → Azioni della Schermata Principale**. Tocca l'icona di configurazione nell'angolo in alto a destra della schermata del Timer di Sospensione per abilitare regolazioni precise minuto per minuto.
{{% /details %}}

{{% details title="Posso usare Flacbox senza una connessione internet?" closed="true" %}}
**Sì — una volta che hai scaricato musica o abilitato la modalità offline per una cartella, Flacbox riproduce tutto completamente offline.**<br><br>

Il contenuto offline si trova in **File Locali** e continua a funzionare in modalità aereo, in volo e ovunque senza Wi-Fi o dati cellulare. Le tracce solo cloud (quelle che non hai scaricato) saranno disabilitate fino a quando non recuperi una connessione. Per i viaggi, abilita la **Modalità Offline** per le cartelle rilevanti o scarica album e playlist specifici prima di partire.
{{% /details %}}

{{% details title="Flacbox fa lo scrobbling su Last.fm?" closed="true" %}}

Collega il tuo account Last.fm dalla sezione **Connessioni → Altri Servizi**. Una volta collegato, tutte le statistiche di riproduzione vengono inviate a Last.fm in background. Puoi poi visitare il tuo profilo Last.fm per vedere le tendenze di ascolto, i migliori artisti settimanali e le raccomandazioni personalizzate.<br><br>

[Leggi la guida di configurazione completa](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm)

{{% details title="Come cambio la lingua dell'interfaccia di Flacbox?" closed="true" %}}
**Apri Impostazioni → Lingua, scegli tra oltre 120 lingue supportate, poi riavvia l'app affinché il cambiamento abbia effetto.**<br><br>

L'app supporta localizzazioni tra cui inglese, francese, tedesco, spagnolo, italiano, portoghese, russo, ucraino, polacco, olandese, arabo, ebraico, hindi, giapponese, coreano, cinese (semplificato e tradizionale), vietnamita, turco e molti altri. Scegli **Predefinito** per seguire automaticamente l'impostazione della lingua del dispositivo. Dopo aver selezionato una nuova lingua, esci completamente da Flacbox e riaprilo in modo che ogni schermata venga ridisegnata con le nuove traduzioni.

{{% details title="Come cerco un brano, un album o un artista in Flacbox?" closed="true" %}}
**Tocca l'icona della lente di ingrandimento in qualsiasi elenco — Libreria Musicale, Playlist, Album, Artisti, Generi o all'interno di una cartella — e digita un nome per filtrare i risultati istantaneamente.**<br><br>

La ricerca è locale e funziona completamente offline rispetto al database della tua libreria, quindi i risultati appaiono mentre digiti anche su reti lente. Puoi anche cercare all'interno di una specifica playlist, album o cartella per trovare una singola traccia tra centinaia. Tracce, album, artisti, generi, compositori e cartelle sono tutti ricercabili.

{{% details title="Flacbox funziona con cuffie Bluetooth, AirPods e DAC esterni?" closed="true" %}}
**Sì — Flacbox riproduce tramite qualsiasi output audio che il tuo iPhone, iPad o Mac può usare: altoparlanti integrati, cuffie cablate, dispositivi Bluetooth (AirPods, Beats, Sony, Bose, ecc.) e DAC USB / Lightning.**<br><br>

Per l'output Hi-Res su un DAC esterno, passa il codec audio a **FFmpeg** in **Impostazioni → Lettore Audio → Generale** e scegli il **Sample Rate di Output Audio** e il numero di canali appropriati. I controlli sullo stelo degli AirPods (riproduci / pausa, successivo / precedente) funzionano come previsto, e i controlli della schermata di blocco possono essere personalizzati in **Impostazioni → Lettore Audio → Personalizzazione → Controlli di Riproduzione sulla Schermata di Blocco**.
{{% /details %}}

{{% details title="Come uso un'unità flash USB o una scheda SD con Flacbox su iPhone o iPad?" closed="true" %}}
**Inserisci il drive nel tuo iPhone o iPad tramite l'adattatore Lightning-to-USB o USB-C, poi in Flacbox apri File Locali → File su questo iPhone → Apri Cartella, naviga all'unità e scegli la cartella musicale.**<br><br>

Flacbox riproduce i file direttamente dall'unità senza copiarli nell'archiviazione interna, il che è utile per librerie lossless molto grandi. Puoi anche importare tracce nella libreria musicale o aggiungerle alle playlist nello stesso modo. L'app supporta **lettori di schede certificati Apple** e **iXpand Flash Drives** — per iXpand, inseriscilo nella porta Lightning e il drive appare automaticamente nella schermata Connessioni. Vedi le istruzioni complete passo dopo passo in [Come collegare un'unità flash USB a iPhone e ascoltare musica](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).
{{% /details %}}

{{% details title="Perché la mia musica cloud non si sincronizza in Flacbox?" closed="true" %}}
**La maggior parte dei problemi di sincronizzazione è causata da un token di autenticazione scaduto, dall'app in background o dall'assenza di una connessione internet attiva — apri Connessioni, riautorizza il servizio, poi esegui manualmente la sincronizzazione da Impostazioni → Libreria → Sincronizzazione Online.**<br><br>

La sincronizzazione online viene eseguita solo quando l'app è in primo piano, quindi la sincronizzazione di una libreria di grandi dimensioni può richiedere del tempo. Per accelerarla, tieni Flacbox aperto, collega il dispositivo all'alimentazione e abilita **Impostazioni → Schermo → Sempre Attivo**. Per librerie molto grandi, esegui la sincronizzazione sulla versione desktop dell'app e trasferisci il risultato a iOS usando **Backup e Ripristino**.

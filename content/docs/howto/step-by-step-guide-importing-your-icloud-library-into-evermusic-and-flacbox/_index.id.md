**Ringkasan:** Anda dapat melakukan streaming perpustakaan musik iCloud Drive di Evermusic dan Flacbox tanpa mengunduh file ke perangkat Anda. Hubungkan iCloud Drive di aplikasi, aktifkan Sinkronisasi Musik Online untuk membangun perpustakaan Anda, konfigurasikan pembaca metadata untuk mengatur berdasarkan artis/album/genre, dan secara opsional aktifkan Mode Offline untuk mengunduh album untuk mendengarkan tanpa internet. Langkah-langkah ini juga berfungsi dengan Google Drive, Dropbox, OneDrive, dan layanan cloud lain yang didukung.

Cara Menyinkronkan Perpustakaan Musik iCloud Anda dengan Flacbox dan Evermusic untuk Pemutaran yang Lancar. Baru-baru ini, salah satu pelanggan kami menghubungi Tim Dukungan Everappz dengan kekhawatiran umum:

> "Halo Tim Dukungan Flacbox, saya baru-baru ini berlangganan Flacbox selama satu tahun, tetapi saya mengalami kesulitan yang signifikan dalam menyinkronkan musik saya dari iCloud dengan benar. Saya memiliki sekitar 60 album dari berbagai soundtrack yang tersimpan di iCloud saya, yang ingin saya integrasikan ke dalam aplikasi. Namun, tampaknya proses impor ke perpustakaan Flacbox tidak berjalan semulus yang saya harapkan. Membutuhkan waktu yang sangat lama bahkan untuk memuat satu album ke dalam aplikasi. Kemarin, saya menghabiskan beberapa jam mencoba berbagai metode untuk mengimpor album secara efisien, tetapi tidak menemukan solusi yang memuaskan. Saya akan menghargai panduan apa pun atau panduan langkah demi langkah tentang cara mengimpor musik berbasis iCloud (tanpa disimpan secara lokal di iPhone saya) dengan cara yang tidak menghasilkan waktu pemuatan yang berlebihan."

Bagi banyak pengguna dengan perpustakaan musik yang luas di iCloud, mengintegrasikan file ke Flacbox dan Evermusic terkadang dapat menghadirkan tantangan, terutama seputar kecepatan dan efisiensi sinkronisasi. Jika Anda mengalami masalah serupa, panduan ini akan memandu Anda melalui praktik terbaik untuk menyederhanakan proses impor, meminimalkan waktu pemuatan, dan menikmati musik Anda dengan mudah. Dalam panduan ini, kami akan menggunakan penyimpanan iCloud, tetapi langkah-langkah ini juga berlaku untuk layanan cloud lain yang didukung oleh aplikasi. Anda akan melihat tangkapan layar Mac dalam panduan ini, tetapi di iPhone, layar dan lokasi fitur sama.

## Impor Perpustakaan iCloud Anda

Mulailah dengan membuka [iCloud.com](https://icloud.com) dan membuat folder Music untuk mengatur perpustakaan Anda. Menggunakan komputer memudahkan, karena situs mendukung seret dan lepas.

![Folder Musik iCloud](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_141e817570c34e6a8c528b65a99a5ba4~mv2.png)

Tunggu hingga semua file sepenuhnya diunggah ke iCloud sebelum melanjutkan.

![Unggah iCloud](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_6f9f363bd64e4ca58190541fcc1ed28e~mv2.png)

Buka aplikasi dan navigasikan ke bagian **Koneksi**.

![Bagian Koneksi](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_ccfbcec8308d4974b630bd1a0868dfd4~mv2.png)

Ketuk **Hubungkan Layanan Cloud** dan pilih **iCloud Drive** dari daftar server cloud yang tersedia.

![Hubungkan Layanan Cloud](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_e0db66985abf49d7833fda296772ddb6~mv2.png)

Di layar berikutnya ketuk **Pilih folder** untuk membuka pemilih file sistem.

![Pilih Folder](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_ebeb7ebe890e4a9a876de22fbf445896~mv2.png)

Langkah kunci di sini adalah pertama memilih **iCloud Drive** di panel kiri, kemudian navigasikan ke folder Music yang dibuat di langkah sebelumnya, dan ketuk **Buka** untuk mengonfirmasi. Perhatikan bahwa file di folder Music menampilkan simbol awan, menunjukkan bahwa file tersebut tersimpan di server iCloud dan tidak diunduh ke perangkat Anda. Pengaturan ini memungkinkan kita melakukan streaming musik langsung dari iCloud tanpa mengunduhnya.

![Pemilihan iCloud Drive](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_944006734b3f43608c2deb24c56cd9d4~mv2.png)

Setelah folder cloud Anda ditambahkan ke aplikasi, folder tersebut akan muncul di bagian **Koneksi**. Ketuk untuk melihat konten folder dan menavigasi subfolder dan file. Gunakan menu opsi untuk mengganti nama, memindahkan, atau menghapus file dalam folder iCloud Drive yang terhubung.

![Koneksi](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_36f1e02d522b4140aa2ecc00e4068387~mv2.png)

Untuk memutar file audio yang tersimpan di cloud, cukup ketuk file apa pun di folder cloud yang terhubung untuk meluncurkan pemutar audio. Aplikasi akan melakukan streaming file tanpa mengunduhnya ke perangkat Anda.

![Pemutar Audio](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_27d2d6ec37c648fdb7e84e82d52eb38b~mv2.png)

Jika Anda ingin menambahkan file ke perpustakaan Musik dan mengatur trek berdasarkan tag Artis, Album, atau Genre, buka layar **Perpustakaan Musik**. Ketuk tombol **Lebih banyak tindakan** di sudut kanan atas dan pilih **Pengaturan** untuk mengakses pengaturan perpustakaan Musik.

![Pengaturan Perpustakaan Musik](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_cb6d59b3af5e466f9875f67199c5df94~mv2.png)

Di layar pengaturan perpustakaan Musik, beberapa opsi tersedia. Untuk saat ini, fokus pada **Sinkronisasi Musik Online**.

![Sinkronisasi Musik Online](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_cc7a80af9d394d57979549b0a5b9c91d~mv2.png)

Ketuk **Sinkronisasi Musik Online** untuk melanjutkan ke layar berikutnya. Pertama, aktifkan fitur ini. Di iPhone, Anda juga memiliki opsi untuk memilih jenis jaringan untuk sinkronisasi: hanya **Wi-Fi** atau **Wi-Fi dan data seluler**.

![Jenis Jaringan](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_a5dbaba46cf94db39cea44d0ab0a32e8~mv2.png)

Ketuk **Folder online yang disinkronkan** untuk memulai pemilihan folder. Ketika Anda memilih folder, aplikasi akan memindai konten folder, dan semua file audio yang ditemukan akan muncul di **Perpustakaan Musik**. Aplikasi hanya membuat tautan ke file media Anda yang terletak di server cloud.

![Folder Online yang Disinkronkan](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_c5e098f807574914a650ac637bad3b45~mv2.png)

Untuk saat ini, ketuk server iCloud yang terhubung dan pilih folder musik. Ketuk **Selesai** untuk mengonfirmasi pilihan Anda.

![Pilih Folder Musik](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_dc162bbb32764b359da4789fdb98294d~mv2.png)

Aplikasi akan mulai memindai folder yang Anda pilih, dan semua file yang ditemukan akan muncul di perpustakaan musik Anda. Anda juga dapat mengatur interval waktu untuk mengontrol seberapa sering aplikasi memindai pembaruan di server cloud Anda.

Dengan **Sinkronisasi Latar Belakang diaktifkan**, aplikasi terus memindai bahkan saat berjalan di latar belakang. Kami menyarankan mengaktifkan fitur ini untuk mempercepat proses; namun, perlu diingat bahwa jika aplikasi mengonsumsi terlalu banyak energi di latar belakang, iOS dapat menutupnya.

Untuk pemindaian yang lebih cepat, aktifkan Sinkronisasi Latar Belakang dan mulai pemutaran audio, karena memutar audio di latar belakang mencegah sistem iOS/macOS membongkar aplikasi. Atau, Anda dapat membiarkan aplikasi tetap terbuka selama proses sinkronisasi.

Ketuk **Mulai Sinkronisasi** untuk memulai dan lihat kemajuan sinkronisasi di bagian bawah layar ini.

Itu semua untuk sinkronisasi online; mari lanjutkan.

## Pembacaan Metadata

Proses penting lainnya adalah sinkronisasi metadata. Proses ini memuat metadata untuk trek yang ditambahkan ke perpustakaan musik, memungkinkan aplikasi mengelompokkannya berdasarkan artis, album, dan genre menggunakan data dari tag metadata.

Untuk mengkonfigurasi ini, buka layar **Pengaturan Perpustakaan Musik** dan ketuk **Pembacaan Metadata** untuk memperbarui pengaturan.

![Pembacaan Metadata](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_a74acfe948934fa48913bfcd8450b3ab~mv2.png)

**Mode yang Tersedia untuk Pembaca Metadata:**

- **Dinonaktifkan**: Pembaca metadata akan dinonaktifkan, dan nama file akan ditampilkan sebagai pengganti data dari tag audio.
- **Lagu Saat Ini**: Aplikasi akan membaca metadata hanya untuk lagu yang sedang diputar. Gunakan opsi ini jika Anda memiliki koneksi jaringan yang lambat untuk mencegah pembaca metadata mengirim banyak permintaan ke server cloud, yang dapat menyebabkan gangguan pemutaran.
- **Antrean Pemutar Audio**: Aplikasi akan membaca metadata untuk semua lagu dalam antrean pemutar audio.
- **Perpustakaan Musik**: Aplikasi akan membaca metadata untuk semua lagu di perpustakaan musik.

**Kecepatan pembacaan metadata**: Anda dapat menyesuaikan kecepatan pembacaan metadata untuk memuat data lebih cepat, tetapi perhatikan bahwa ini mungkin menggunakan lebih banyak energi. Opsi ini hanya tersedia di Mac. Jika Anda ingin memperbarui metadata perpustakaan musik Anda lebih cepat, pertimbangkan untuk menggunakan versi Mac dari aplikasi, dan kemudian gunakan fitur pencadangan/pemulihan data di pengaturan aplikasi untuk mentransfer perpustakaan musik dari versi desktop ke versi iOS.

**Penting**, pembaca metadata hanya memperbarui metadata di perpustakaan musik Anda dan tidak mengubah file yang disimpan di akun cloud atau penyimpanan lokal Anda. Jika Anda ingin mengedit metadata untuk file audio, Anda dapat melakukannya menggunakan editor tag bawaan, yang dapat diaktifkan dari tindakan yang sesuai di menu opsi.

Ketika sakelar **Pembacaan metadata di latar belakang** aktif, pembaca metadata bekerja dalam mode latar belakang. Namun, jika aplikasi mengonsumsi banyak energi selama pemutaran audio, sistem iOS dapat menangguhkannya. Untuk mempercepat pembacaan metadata, hubungkan perangkat iOS Anda ke pengisi daya, mulai pemutaran audio untuk menjaga aplikasi tetap aktif di latar belakang, dan aktifkan pembacaan metadata di latar belakang. Atau, Anda dapat membiarkan aplikasi terbuka dan mengatur **Layar Selalu Aktif** di Pengaturan aplikasi - menu Layar (khusus iOS). Ini akan menonaktifkan timer kunci otomatis, menjaga aplikasi tetap aktif secara terus-menerus.

**Penting**, jika Anda memiliki koleksi musik yang besar, disarankan untuk menggunakan versi desktop aplikasi untuk sinkronisasi metadata. Anda kemudian dapat menggunakan fitur pencadangan dan pemulihan data untuk mentransfer perpustakaan musik yang disinkronkan dari desktop, tersedia di pengaturan aplikasi.

Ketika **Normalisasi encoding metadata** diaktifkan, aplikasi akan secara otomatis menormalisasi encoding metadata untuk semua lagu di perpustakaan musik. Ini memperbaiki masalah dengan encoding yang rusak (misalnya, dari mengedit file di PC Windows) dan mencegah informasi yang salah ditampilkan saat trek diputar atau ditambahkan ke perpustakaan.

Tindakan **Muat ulang metadata** akan menandai semua file di perpustakaan musik Anda sebagai memiliki metadata yang hilang, memicu pembaca metadata untuk menyegarkan metadata untuk setiap file di perpustakaan Anda.

Ketuk **Mulai Pembacaan Metadata** untuk memulai pembaca metadata. Kemajuan operasi akan ditampilkan di bawah.

## Sinkronisasi Musik Offline

Jika Anda memutuskan nanti untuk membuat beberapa album tersedia secara offline di aplikasi, Anda memiliki dua opsi:

1. Gunakan tindakan **Unduh** untuk mengunduh seluruh konten folder ke perangkat Anda.
2. Gunakan **Mode Offline** yang lebih canggih. Dengan Mode Offline, aplikasi akan secara berkala memeriksa folder cloud untuk perubahan apa pun, dan jika file baru ditemukan, aplikasi akan mengunduh file yang hilang ke perangkat Anda secara otomatis.

Untuk mengaktifkan mode offline untuk folder, ketuk tombol **Lebih banyak tindakan** dan pilih item menu **Mengaktifkan mode offline**.

![Mengaktifkan mode offline](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_f77314386cab41349e0140311515f9f1~mv2.png)

Aplikasi akan menambahkan folder yang dipilih ke antrean transfer. Anda dapat memeriksa antrean di Pengaturan - Manajer File - Tugas Transfer.

![Antrean Transfer](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_12c26a5496ad480caccff79914de7289~mv2.png)

Semua file yang diunduh akan muncul di bagian **File Lokal - Folder offline**. Ketika Anda membuat perubahan pada folder online di server cloud (seperti menambahkan, menghapus, atau memperbarui file), aplikasi akan memeriksa pembaruan dan menyinkronkan salinan lokal folder ini di perangkat Anda.

Sekarang, mari kembali ke **Pengaturan Perpustakaan Musik** dan buka layar **Sinkronisasi Musik Offline**.

Di bagian **Folder offline yang disinkronkan**, Anda dapat:

- Memulai sinkronisasi folder offline secara manual
- Menampilkan folder offline di folder induknya
- Menonaktifkan mode offline untuk folder ini (yang akan menghapus semua salinan lokal file dari perangkat Anda)

![Folder Offline yang Disinkronkan](/docs/howto/step-by-step-guide-importing-your-icloud-library-into-evermusic-and-flacbox/21260c_bb941f44ec994437a05fc3bcdd121c91~mv2.png)

**Interval Waktu:** Anda dapat mengatur interval waktu seberapa sering aplikasi harus memeriksa folder offline untuk modifikasi.

**Mulai Pemindaian Folder Lokal:** Opsi ini memindai semua folder lokal yang terletak di direktori Documents aplikasi untuk menemukan file audio yang didukung. Semua file lokal ini ditambahkan dengan mulus ke perpustakaan musik Anda. Perhatikan bahwa file lokal yang disimpan di luar aplikasi ini harus ditambahkan ke perpustakaan musik secara manual, karena pembatasan keamanan iOS/macOS mencegah aplikasi mengakses file di luar direktori Documents aplikasi.

**Penting:** Disarankan untuk secara berkala memulai sinkronisasi musik offline agar perpustakaan musik Anda tetap diperbarui dengan file lokal Anda.

Itu semua untuk hari ini! Kami berharap panduan ini membantu Anda mengkonfigurasi sinkronisasi antara server cloud dan perangkat Anda.

## FAQ

{{% details title="Bisakah saya melakukan streaming musik iCloud tanpa mengunduh file ke iPhone saya?" closed="true" %}}
Ya. Ketika Anda menghubungkan iCloud Drive di Evermusic atau Flacbox dan menggunakan Sinkronisasi Musik Online, aplikasi membuat tautan ke file cloud Anda dan melakukan streaming sesuai permintaan. File tidak diunduh kecuali Anda secara eksplisit mengaktifkan Mode Offline.
{{% /details %}}

{{% details title="Mengapa impor musik iCloud lambat di Flacbox atau Evermusic?" closed="true" %}}
Impor lambat biasanya disebabkan oleh pembacaan metadata pada perpustakaan besar melalui koneksi seluler. Aktifkan Sinkronisasi Latar Belakang, mulai pemutaran audio untuk menjaga aplikasi tetap aktif, dan pertimbangkan menggunakan versi Mac untuk sinkronisasi awal koleksi besar.
{{% /details %}}

{{% details title="Apakah panduan ini berfungsi dengan layanan cloud selain iCloud?" closed="true" %}}
Ya. Langkah-langkah yang sama berlaku untuk Google Drive, Dropbox, OneDrive, SMB, WebDAV, dan semua layanan cloud lain yang didukung oleh Evermusic dan Flacbox.
{{% /details %}}

{{% details title="Bagaimana cara mentransfer perpustakaan musik saya dari Mac ke iPhone?" closed="true" %}}
Gunakan fitur pencadangan/pemulihan data di pengaturan aplikasi. Sinkronkan dan baca metadata di versi Mac terlebih dahulu, buat cadangan, lalu pulihkan di versi iOS. Ini adalah cara tercepat untuk mengatur perpustakaan besar di iPhone.
{{% /details %}}

{{% details title="Apakah pembaca metadata akan mengubah file audio asli saya?" closed="true" %}}
Tidak. Pembaca metadata hanya memperbarui informasi tampilan di perpustakaan musik Anda. Ini tidak mengubah file yang disimpan di akun cloud atau di perangkat Anda. Untuk mengedit tag file, gunakan editor tag bawaan.
{{% /details %}}

{{% details title="Bagaimana cara membuat album tersedia secara offline?" closed="true" %}}
Gunakan fitur Mode Offline. Ketuk **Lebih banyak tindakan** pada folder cloud mana pun dan pilih **Mengaktifkan mode offline**. Aplikasi mengunduh semua file dan menjaganya tetap sinkron dengan versi cloud secara otomatis.
{{% /details %}}

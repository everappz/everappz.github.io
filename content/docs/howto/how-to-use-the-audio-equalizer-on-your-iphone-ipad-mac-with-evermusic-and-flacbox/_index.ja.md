**要約：** Evermusic と Flacbox には、プリセット（Rock、Hip-Hop、Bass Booster など）、カスタムプリセット作成、音量ブースト用プリアンプを備えたプロフェッショナルな10バンドオーディオイコライザーが搭載されています。iPhone、iPad、Mac で利用可能です。

オーディオプレーヤーにおいて、適切なイコライザーを見つけることは、音楽リスニング体験に大きな違いをもたらします。私たちのオーディオアプリ、**Evermusic** と **Flacbox** は、プロフェッショナルな10バンドイコライザーと多彩な機能を備えたフル機能プレーヤーとして際立っています。

音楽ファイルとヘッドフォンが完璧な理想の世界では、イコライザーは不要に思えるかもしれません。しかし、私たちの日常生活には、理想的とは言えないリスニング環境が伴うことが多いです — 通勤、ワークアウト、騒がしい街中の散歩など。これらの状況は、標準的なヘッドフォンの制限と相まって、EQ 調整の力を必要とします。イコライザーを使えば、周波数バランスの微調整、不完全なヘッドフォンやスピーカーの補正、バックグラウンドノイズへの適応が可能です。パーティーの雰囲気のために低音と高音をブーストする場合でも、DJ セッション用にサウンドを微調整する場合でも、私たちのイコライザーはリスニング体験を向上させるクリエイティブなオーディオ強化を提供します。

## Evermusic と Flacbox のイコライザーはどのように動作しますか？

私たちのアプリには、特定の音声周波数を精密に制御できる強力な10バンドイコライザーが搭載されています。動作の仕組みは次のとおりです：

{{< figure src="/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox/21260c_f94a834da7c84eb8bdf2494086527bb6~mv2.png" width="400" >}}

**周波数制御：** イコライザーは 32 Hz から 16 kHz までの10の異なる周波数をカバーします。直感的なスライダーを使用して、各周波数のアンプゲインを調整します。

**プリセット：** Acoustic、Bass Booster、Hip-Hop、Rock など、さまざまな定義済みイコライザープリセットから選択できます。

**独自のプリセットを作成：** クリエイティブな気分ですか？あなた独自の好みやオーディオ機器に合わせたカスタムイコライザープリセットを作成して保存しましょう。

**プリアンプゲイン：** トラックの音量が足りない場合は、プリアンプスライダーを使用して出力をブーストし、より大きくクリアなサウンドを楽しみましょう。

**オーディオレベルインジケーター：** 曲の再生中に、出力レベルインジケーターを確認し、それに応じてプリアンプゲインを調整できます。

## Evermusic と Flacbox でイコライザーを有効にする

私たちのアプリでイコライザーを有効にしてカスタマイズするのはとても簡単です：

1. Evermusic または Flacbox で**オーディオプレーヤー**画面を開きます。  
2. 画面下部の**イコライザーアイコン**をタップします。  
3. 上部のスイッチを切り替えてイコライザーを有効にします。  
   （または、**設定 > オーディオプレーヤー > オーディオイコライザー**に移動します）

4. ポップアップメニューでプリセットオプションの中から選択します。

{{< figure src="/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox/21260c_165ec4812bde419cb824d130c581fd3f~mv2.png" width="400" >}}

5. 周波数スライダーを上下にドラッグして、デシベル単位で音量を調整し、サウンドを微調整します。

{{< figure src="/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox/21260c_bb5f312caf63466aac3e0d6cfd274d38~mv2.png" width="400" >}}

6. 将来のリスニングのために、独自のイコライザー設定を保存します。

{{< figure src="/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox/21260c_8aa45559788240a78806461d9c7d8bb5~mv2.png" width="400" >}}

7. さらに音量をブーストするには、**プリアンプ**スライダーを使用します。

{{< figure src="/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox/21260c_34869d69d565402da768f73e179b8f4e~mv2.jpeg" width="400" >}}

## プリセット設定の管理

イコライザー画面の左上隅にある**その他のアクション**（3つの点）ボタンをタップして、詳細オプションにアクセスします：

- **設定をインポート：** 保存されたイコライザーファイルを読み込みます。  
- **設定をエクスポート：** プリセットをファイルに保存します。  
- **設定を編集：** 保存されたプリセットを並べ替えて整理します。  
- **すべての設定をリセット：** デフォルトに戻し、すべてのカスタムプリセットを削除します。

## まとめ

Evermusic と Flacbox の多機能イコライザーで、音楽体験を向上させ、あらゆるシーンに合わせてオーディオをカスタマイズし、完璧なサウンドをお楽しみください。

## よくある質問

{{% details title="イコライザーはすべてのオーディオ形式に対応していますか？" closed="true" %}}
はい。Evermusic と Flacbox の10バンド EQ は、MP3、FLAC、AAC、WAV、ALAC、OGG、その他すべてのサポートされている形式に対応しています。
{{% /details %}}

{{% details title="EQ 設定はすべての曲に適用されますか？" closed="true" %}}
はい。イコライザーを有効にしてプリセットを選択すると、変更するかオフにするまで、すべての再生に適用されます。
{{% /details %}}

{{% details title="カスタムプリセットを複数作成できますか？" closed="true" %}}
はい。複数のカスタムプリセットを作成、保存し、切り替えることができます。エクスポート機能を使用してバックアップしてください。
{{% /details %}}

{{% details title="プリアンプは歪みを引き起こしますか？" closed="true" %}}
高く設定しすぎると歪みが生じる可能性があります。調整中にオーディオレベルインジケーターを確認してください。レベルがクリッピング（上限に達する）している場合は、プリアンプゲインをわずかに下げてください。
{{% /details %}}

{{% details title="イコライザーは Evermusic と Flacbox の両方で利用できますか？" closed="true" %}}
はい。両方のアプリに、プリセット、カスタムプリセット、プリアンプを備えた同じ10バンドイコライザーが搭載されています。
{{% /details %}}

{{% details title="EQ プリセットを他の人と共有できますか？" closed="true" %}}
はい。設定をエクスポートオプションを使用してプリセットをファイルに保存し、共有してください。相手は設定をインポートを使用してインポートできます。
{{% /details %}}

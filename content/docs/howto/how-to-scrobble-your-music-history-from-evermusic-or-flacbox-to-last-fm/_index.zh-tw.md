**簡述：** Evermusic 和 Flacbox 都內建了 Last.fm scrobbling 功能。在**連接**部分連接您的帳戶，您播放的每首曲目都會自動記錄——即使在離線時也是如此。設定不到一分鐘。

![如何將 Evermusic 或 Flacbox 的音樂歷史記錄 Scrobble 到 Last.fm](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_acb5cfbde21649a9841b86e236fd36de~mv2.png)

## 介紹

[**Last.fm**](http://Last.fm) 是一項強大的音樂追蹤和推薦服務。它可以讓您保留詳細的聆聽習慣歷史記錄，發現個人化的音樂推薦，並存取播放統計數據。

[**註冊是免費的**](https://www.last.fm/join)，而且很簡單——只需造訪 [Last.fm](https://www.last.fm) 並建立一個帳戶即可開始。

![範例圖片](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_c9eeca3c1c914ce0a4a3eef9a17e5ed5~mv2.png)

## 如何在 Evermusic 或 Flacbox 中啟用 Last.fm

1. **啟動** Evermusic 或 Flacbox，開啟**連接**部分（在 iPhone / iPad 上）或 Mac 上的 **All connections** 部分。
2. **捲動**到 **Other Services** 部分，點擊 **Last.fm**。
3. **輸入**您的 **Last.fm** 使用者名稱和密碼，然後點擊**連接**。請放心，應用程式不會儲存您的使用者名稱或密碼；它會在裝置鑰匙圈中安全地儲存存取權杖。此權杖用於透過安全連線使用官方 Last.fm API 更新您的聆聽歷史記錄。

![如何啟用 Last.fm](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_385ba38d08f14c7d9f076baced4b8b65~mv2.png)

## 連接後會發生什麼

- 連接後，應用程式將顯示來自您 Last.fm 個人資料的**最近 10 首已 scrobble 的曲目**。
- 開始播放任何歌曲，應用程式將在 Last.fm 上**自動更新您的聆聽歷史記錄**。
- **離線 Scrobbling**：如果您處於離線狀態，應用程式會在本機儲存您的播放歷史記錄，並在您重新上線時將其上傳到 Last.fm。

## 如何斷開 Last.fm 連接

您可以隨時斷開 Last.fm 帳戶的連接。這將阻止應用程式更新您的聆聽歷史記錄，並從裝置中移除存取權杖。只需點擊**斷開連接**按鈕。

![如何斷開 Last.fm 連接](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_5fde429d5afb4cafacbf7cfcc16de370~mv2.png)

## 享受播放統計數據和推薦

在使用應用程式數週或數月後，重新造訪 Last.fm 網站，探索您詳細的播放統計數據，並發現根據您的聆聽偏好量身打造的新音樂。

![播放統計數據和推薦](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_68ed03d282054d4f9718c2fc0c6fd48b~mv2.png)

## 結論

將音樂歷史記錄從 Evermusic 或 Flacbox scrobble 到 [Last.fm](http://Last.fm) 是提升音樂體驗的簡單而有效的方式。藉助播放統計數據、個人化推薦和離線 scrobbling 等功能，您可以更深入地了解自己的聆聽習慣，發現您會喜愛的音樂。今天就開始 scrobble，充分利用 [Last.fm](http://Last.fm) 提供的一切！

## 常見問題

{{% details title="Last.fm scrobbling 是免費的嗎？" closed="true" %}}
是的。Last.fm 提供免費帳戶，包括完整的 scrobbling、聆聽歷史記錄和基本推薦。付費的 Last.fm Pro 訂閱在 Last.fm 網站上增加了額外功能，但從 Evermusic 或 Flacbox 進行 scrobbling 並不需要。
{{% /details %}}

{{% details title="離線時 scrobbling 能運作嗎？" closed="true" %}}
能。Evermusic 和 Flacbox 都會在本機儲存您的播放歷史記錄。當您重新上線時，應用程式會自動將排隊的 scrobble 上傳到 Last.fm。
{{% /details %}}

{{% details title="應用程式會儲存我的 Last.fm 憑證嗎？" closed="true" %}}
不會。應用程式僅在裝置鑰匙圈中儲存安全的存取權杖。您的使用者名稱和密碼不會被儲存。
{{% /details %}}

{{% details title="我可以從 iPhone 和 Mac 同時 scrobble 嗎？" closed="true" %}}
可以。Evermusic 和 Flacbox 在 iPhone、iPad 和 Mac 上都支援 Last.fm scrobbling。在您想要追蹤播放的每個裝置上連接您的帳戶。
{{% /details %}}

{{% details title="如何在不刪除 Last.fm 帳戶的情況下停止 scrobbling？" closed="true" %}}
在 Evermusic 或 Flacbox 中開啟**連接**部分，點擊 Last.fm 旁邊的**斷開連接**。這將移除存取權杖並停止 scrobbling，同時保留您的 Last.fm 帳戶和歷史記錄。
{{% /details %}}

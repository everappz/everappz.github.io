**简述：** Evermusic 和 Flacbox 都内置了 Last.fm scrobbling 功能。在**连接**部分连接您的账户，您播放的每首曲目都会自动记录——即使在离线时也是如此。设置不到一分钟。

![如何将 Evermusic 或 Flacbox 的音乐历史记录 Scrobble 到 Last.fm](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_acb5cfbde21649a9841b86e236fd36de~mv2.png)

## 介绍

[**Last.fm**](http://Last.fm) 是一项强大的音乐追踪和推荐服务。它可以让您保留详细的收听习惯历史记录，发现个性化的音乐推荐，并访问播放统计数据。

[**注册是免费的**](https://www.last.fm/join)，而且很简单——只需访问 [Last.fm](https://www.last.fm) 并创建一个账户即可开始。

![示例图片](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_c9eeca3c1c914ce0a4a3eef9a17e5ed5~mv2.png)

## 如何在 Evermusic 或 Flacbox 中启用 Last.fm

1. **启动** Evermusic 或 Flacbox，打开**连接**部分（在 iPhone / iPad 上）或 Mac 上的 **All connections** 部分。
2. **滚动**到 **Other Services** 部分，点击 **Last.fm**。
3. **输入**您的 **Last.fm** 用户名和密码，然后点击**连接**。请放心，应用程序不会存储您的用户名或密码；它会在设备钥匙串中安全地保存访问令牌。此令牌用于通过安全连接使用官方 Last.fm API 更新您的收听历史记录。

![如何启用 Last.fm](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_385ba38d08f14c7d9f076baced4b8b65~mv2.png)

## 连接后会发生什么

- 连接后，应用程序将显示来自您 Last.fm 个人资料的**最近 10 首已 scrobble 的曲目**。
- 开始播放任何歌曲，应用程序将在 Last.fm 上**自动更新您的收听历史记录**。
- **离线 Scrobbling**：如果您处于离线状态，应用程序会在本地存储您的播放历史记录，并在您重新上线时将其上传到 Last.fm。

## 如何断开 Last.fm 连接

您可以随时断开 Last.fm 账户的连接。这将阻止应用程序更新您的收听历史记录，并从设备中删除访问令牌。只需点击**断开连接**按钮。

![如何断开 Last.fm 连接](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_5fde429d5afb4cafacbf7cfcc16de370~mv2.png)

## 享受播放统计数据和推荐

在使用应用程序数周或数月后，重新访问 Last.fm 网站，探索您详细的播放统计数据，并发现根据您的收听偏好量身定制的新音乐。

![播放统计数据和推荐](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm/21260c_68ed03d282054d4f9718c2fc0c6fd48b~mv2.png)

## 结论

将音乐历史记录从 Evermusic 或 Flacbox scrobble 到 [Last.fm](http://Last.fm) 是提升音乐体验的简单而有效的方式。借助播放统计数据、个性化推荐和离线 scrobbling 等功能，您可以更深入地了解自己的收听习惯，发现您会喜爱的音乐。今天就开始 scrobble，充分利用 [Last.fm](http://Last.fm) 提供的一切！

## 常见问题

{{% details title="Last.fm scrobbling 是免费的吗？" closed="true" %}}
是的。Last.fm 提供免费账户，包括完整的 scrobbling、收听历史记录和基本推荐。付费的 Last.fm Pro 订阅在 Last.fm 网站上增加了额外功能，但从 Evermusic 或 Flacbox 进行 scrobbling 并不需要。
{{% /details %}}

{{% details title="离线时 scrobbling 能工作吗？" closed="true" %}}
能。Evermusic 和 Flacbox 都会在本地存储您的播放历史记录。当您重新上线时，应用程序会自动将排队的 scrobble 上传到 Last.fm。
{{% /details %}}

{{% details title="应用程序会存储我的 Last.fm 凭据吗？" closed="true" %}}
不会。应用程序仅在设备钥匙串中保存安全的访问令牌。您的用户名和密码不会被存储。
{{% /details %}}

{{% details title="我可以从 iPhone 和 Mac 同时 scrobble 吗？" closed="true" %}}
可以。Evermusic 和 Flacbox 在 iPhone、iPad 和 Mac 上都支持 Last.fm scrobbling。在您想要跟踪播放的每个设备上连接您的账户。
{{% /details %}}

{{% details title="如何在不删除 Last.fm 账户的情况下停止 scrobbling？" closed="true" %}}
在 Evermusic 或 Flacbox 中打开**连接**部分，点击 Last.fm 旁边的**断开连接**。这将删除访问令牌并停止 scrobbling，同时保留您的 Last.fm 账户和历史记录。
{{% /details %}}

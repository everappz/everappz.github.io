**摘要：** 使用 Apple Music 內建的 **File > Library > Export Playlist** 功能將任何播放清單儲存為 M3U 檔案。然後在 Mac 上將其匯入 **Evermusic** 或 **Flacbox**。您也可以將播放清單封存為 ZIP 檔案，以便輕鬆傳輸到其他裝置。

macOS 上的 Apple Music 不僅允許您從 Apple Music 資料庫建立播放清單，還可以從直接位於裝置上的音訊檔案建立播放清單。  
如果您想將本機的 MP3、FLAC 或其他音訊檔案集合整理成播放清單，這非常適合。  

在本教學中，您將學習如何 **從 Mac 上的 Apple Music 匯出播放清單**，然後將其匯入 **Evermusic** 或 **Flacbox** 應用程式。  
這樣，您可以繼續使用進階功能聆聽您喜愛的播放清單——無需從頭重新建立。

## 從 Apple Music 匯出播放清單

首先在 Mac 上的 Apple Music 應用程式中開啟播放清單。

{{< cards cols="1">}}
  {{< card title="" subtitle="在 Apple Music 中開啟播放清單" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/1-apple-music-playlist.webp" >}}
{{< /cards >}}

從頂部選單前往 **File → Library → Export Playlist**。

{{< cards cols="1">}}
  {{< card title="" subtitle="從資料庫匯出播放清單" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/2-library-export-playlist.webp" >}}
{{< /cards >}}

選擇儲存 M3U 檔案的目標位置。  

> 我們建議將 M3U 檔案儲存在您的**音樂資料夾**中。 
> 這確保所有檔案路徑一致，Evermusic 或 Flacbox 中的匯入過程將順利進行。  
> 由於應用程式在 macOS 上以沙盒模式運行，**播放清單檔案**和**媒體檔案**都必須位於同一資料夾中才能成功匯入。

## 封存和傳輸播放清單

Evermusic 和 Flacbox 提供進階播放清單管理功能。  
其中之一是將播放清單封存為 **ZIP 檔案**的功能，其中包含播放清單檔案和所有曲目。  
這使得將播放清單傳輸到其他裝置變得快速且可靠。

只需從播放清單選單中選擇**更多動作 → 加入封存**。

{{< cards cols="1">}}
  {{< card title="" subtitle="開啟播放清單的更多動作" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/14-more-actions-playlist.webp" >}}
{{< /cards >}}

選擇**加入封存**後，等待應用程式處理播放清單。  
封存完成後，您將看到**成功提示**。點擊**顯示檔案**讓應用程式顯示新建立的 ZIP 封存。

{{< cards cols="1">}}
  {{< card title="" subtitle="封存完成" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/15-archiving-completed.webp" >}}
{{< /cards >}}

應用程式將開啟**匯出資料夾**，其中儲存了所有建立的封存。

{{< cards cols="1">}}
  {{< card title="" subtitle="開啟匯出資料夾" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/16-export-folder.webp" >}}
{{< /cards >}}

找到新建立的封存，點擊旁邊的**更多動作**按鈕，選擇**在 Finder 中顯示**。

{{< cards cols="1">}}
  {{< card title="" subtitle="對 ZIP 檔案使用更多動作" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/17-more-actions-menu-zip-file.webp" >}}
{{< /cards >}}

現在您將看到 Mac 上 **ZIP 檔案的實際位置**。  
此時，您可以輕鬆地將封存傳輸到其他裝置。  
但在此之前，讓我們仔細看看裡面的內容。雙擊檔案解壓縮。

{{< cards cols="1">}}
  {{< card title="" subtitle="在 Finder 中顯示 ZIP 檔案" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/18-zip-file-revealed-in-finder.webp" >}}
{{< /cards >}}

在裡面，您將找到**完整的播放清單內容**——播放清單中包含的所有音訊檔案以及 **M3U 播放清單檔案**。

{{< cards cols="1">}}
  {{< card title="" subtitle="解壓縮 ZIP 檔案" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/19-unarchived-zip-file.webp" >}}
{{< /cards >}}

最後，開啟 **M3U 檔案**檢查其內容。  
它的格式正確，因此當您將此 ZIP 封存移動到其他裝置並解壓縮時，您可以無縫匯入播放清單。  
應用程式將恢復播放清單，保持**正確的曲目順序**和**所有相關的媒體檔案**。

{{< cards cols="1">}}
  {{< card title="" subtitle="檢視 M3U 播放清單內容" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/20-m3u-playlist-content.webp" >}}

## 結論

在 Mac 上匯出 Apple Music 播放清單並在 **Evermusic** 或 **Flacbox** 中播放是一個簡單的過程。  
透過將播放清單儲存為 **M3U 檔案**並將其匯入這些應用程式，您可以獲得強大的功能，例如：

- FLAC、MP3、OGG 和高解析度音訊播放  
- 中繼資料編輯和專輯封面支援  
- 雲端儲存和網路磁碟機整合  
- 播放清單封存和跨裝置傳輸  

使用此工作流程，您不再需要手動重新建立播放清單。  
相反，享受更多控制、更好功能和跨裝置攜帶性的無縫音樂體驗。

## 常見問題

{{% details title="Apple Music 以什麼格式匯出播放清單？" closed="true" %}}
Apple Music 以 M3U 格式匯出播放清單，這是大多數音樂播放器支援的標準播放清單格式，包括 Evermusic、Flacbox、VLC 和 foobar2000。
{{% /details %}}

{{% details title="為什麼 M3U 檔案和音訊檔案必須在同一資料夾中？" closed="true" %}}
Evermusic 和 Flacbox 在 macOS 沙盒模式下運行，將檔案存取限制在您明確授權的資料夾中。將 M3U 檔案和音訊檔案保存在同一資料夾中可確保應用程式在匯入期間能夠讀取兩者。
{{% /details %}}

{{% details title="我可以在 Mac 和 iPhone 之間傳輸播放清單嗎？" closed="true" %}}
可以。使用播放清單封存功能建立包含播放清單和所有曲目的 ZIP 檔案。透過 AirDrop、iCloud Drive 或任何其他方法將 ZIP 傳輸到 iPhone，然後在 iOS 上的 Evermusic 或 Flacbox 中匯入。
{{% /details %}}

{{% details title="這適用於 Apple Music 串流曲目嗎？" closed="true" %}}
此方法適用於您新增到 Apple Music 的本機音訊檔案。Apple Music 中受 DRM 保護的串流曲目無法匯出為 M3U 播放清單。
{{% /details %}}

{{% details title="Evermusic 和 Flacbox 支援哪些音訊格式？" closed="true" %}}
兩個應用程式都支援廣泛的格式，包括 MP3、FLAC、AAC、WAV、OGG、AIFF、ALAC、WMA、APE 等。它們還支援無損格式的高解析度音訊播放。
{{% /details %}}

**In breve:** Usa la funzione integrata **File > Library > Export Playlist** di Apple Music per salvare qualsiasi playlist come file M3U. Poi importala in **Evermusic** o **Flacbox** su Mac. Puoi anche archiviare le playlist come file ZIP per un facile trasferimento su altri dispositivi.

Apple Music su macOS ti permette di creare playlist non solo dalla tua libreria Apple Music ma anche da file audio situati direttamente sul tuo dispositivo.  
Questo è perfetto se vuoi organizzare la tua collezione locale di file MP3, FLAC o altri file audio in playlist.  

In questo tutorial, imparerai come **esportare playlist da Apple Music sul tuo Mac** e poi importarle nell'app **Evermusic** o **Flacbox**.  
In questo modo, puoi continuare ad ascoltare le tue playlist preferite con funzionalità avanzate — senza bisogno di ricrearle da zero.

## Esportare una playlist da Apple Music

Inizia aprendo la playlist nell'app Apple Music sul tuo Mac.

{{< cards cols="1">}}
  {{< card title="" subtitle="Apri la playlist in Apple Music" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/1-apple-music-playlist.webp" >}}
{{< /cards >}}

Vai su **File → Library → Export Playlist** dal menu in alto.

{{< cards cols="1">}}
  {{< card title="" subtitle="Esporta la playlist dalla tua Libreria" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/2-library-export-playlist.webp" >}}
{{< /cards >}}

Scegli la destinazione dove verrà salvato il file M3U.  

> Consigliamo di salvare il file M3U nella tua **cartella Musica**. 
> Questo assicura che tutti i percorsi dei file siano coerenti, e il processo di importazione in Evermusic o Flacbox funzionerà senza problemi.  
> Poiché le app funzionano in modalità sandbox su macOS, sia il **file della playlist** che i **file multimediali** devono essere nella stessa cartella per un'importazione riuscita.

## Archiviazione e trasferimento delle playlist

Evermusic e Flacbox offrono funzionalità avanzate di gestione delle playlist.  
Una di queste è la possibilità di archiviare una playlist in un **file ZIP**, che include il file della playlist e tutti i brani.  
Questo rende il trasferimento delle playlist su un altro dispositivo rapido e affidabile.

Scegli semplicemente **Altre azioni → Aggiungi all'archivio** dal menu della playlist.

{{< cards cols="1">}}
  {{< card title="" subtitle="Apri altre azioni per la playlist" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/14-more-actions-playlist.webp" >}}
{{< /cards >}}

Dopo aver selezionato **Aggiungi all'archivio**, attendi un momento mentre l'app elabora la playlist.  
Una volta completata l'archiviazione, vedrai un **avviso di successo**. Tocca **Mostra file** per far mostrare all'app l'archivio ZIP appena creato.

{{< cards cols="1">}}
  {{< card title="" subtitle="Archiviazione completata" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/15-archiving-completed.webp" >}}
{{< /cards >}}

L'app aprirà quindi la **cartella Esportazione**, dove sono archiviati tutti gli archivi creati.

{{< cards cols="1">}}
  {{< card title="" subtitle="Apri la cartella di esportazione" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/16-export-folder.webp" >}}
{{< /cards >}}

Individua il tuo archivio appena creato, tocca il pulsante **Altre azioni** accanto ad esso e scegli **Mostra nel Finder**.

{{< cards cols="1">}}
  {{< card title="" subtitle="Usa Altre azioni sul file ZIP" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/17-more-actions-menu-zip-file.webp" >}}
{{< /cards >}}

Ora vedrai la **posizione reale del file ZIP** sul tuo Mac.  
A questo punto, puoi facilmente trasferire l'archivio su un altro dispositivo.  
Ma prima, diamo un'occhiata più da vicino a cosa c'è dentro. Fai doppio clic sul file per estrarlo.

{{< cards cols="1">}}
  {{< card title="" subtitle="Mostra il file ZIP nel Finder" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/18-zip-file-revealed-in-finder.webp" >}}
{{< /cards >}}

All'interno, troverai il **contenuto completo della playlist** — tutti i file audio inclusi nella playlist, così come il **file playlist M3U**.

{{< cards cols="1">}}
  {{< card title="" subtitle="Estrai il file ZIP" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/19-unarchived-zip-file.webp" >}}
{{< /cards >}}

Infine, apri il **file M3U** per esaminarne il contenuto.  
È formattato correttamente, quindi quando sposti questo archivio ZIP su un altro dispositivo e lo estrai, potrai importare la playlist senza problemi.  
L'app ripristinerà la playlist con il **corretto ordine dei brani** e **tutti i file multimediali associati**.

{{< cards cols="1">}}
  {{< card title="" subtitle="Visualizza il contenuto della playlist M3U" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/20-m3u-playlist-content.webp" >}}

Esportare le playlist di Apple Music e riprodurle in **Evermusic** o **Flacbox** sul tuo Mac è un processo semplice.

## Domande frequenti

{{% details title="Quale formato di playlist esporta Apple Music?" closed="true" %}}
Apple Music esporta le playlist in formato M3U, che è un formato playlist standard supportato dalla maggior parte dei lettori musicali inclusi Evermusic, Flacbox, VLC e foobar2000.
{{% /details %}}

{{% details title="Perché il file M3U e i file audio devono essere nella stessa cartella?" closed="true" %}}
Evermusic e Flacbox funzionano in modalità sandbox di macOS, che limita l'accesso ai file alle cartelle a cui concedi esplicitamente il permesso. Mantenere il file M3U e i file audio nella stessa cartella assicura che l'app possa leggere entrambi durante l'importazione.
{{% /details %}}

{{% details title="Posso trasferire playlist tra Mac e iPhone?" closed="true" %}}
Sì. Usa la funzione di archiviazione playlist per creare un file ZIP contenente la playlist e tutti i brani. Trasferisci lo ZIP sul tuo iPhone tramite AirDrop, iCloud Drive o qualsiasi altro metodo, poi importalo in Evermusic o Flacbox su iOS.
{{% /details %}}

{{% details title="Funziona con i brani in streaming di Apple Music?" closed="true" %}}
Questo metodo funziona con i file audio locali che hai aggiunto ad Apple Music. I brani in streaming protetti da DRM di Apple Music non possono essere esportati come playlist M3U.
{{% /details %}}

{{% details title="Quali formati audio supportano Evermusic e Flacbox?" closed="true" %}}
Entrambe le app supportano un'ampia gamma di formati tra cui MP3, FLAC, AAC, WAV, OGG, AIFF, ALAC, WMA, APE e altro. Supportano anche la riproduzione audio hi-res per i formati lossless.
{{% /details %}}

**Zusammenfassung:** Verwenden Sie die integrierte Funktion **File > Library > Export Playlist** von Apple Music, um jede Playlist als M3U-Datei zu speichern. Importieren Sie sie dann in **Evermusic** oder **Flacbox** auf dem Mac. Sie können Playlists auch als ZIP-Dateien archivieren, um sie einfach auf andere Geräte zu übertragen.

Apple Music auf macOS ermöglicht es Ihnen, Playlists nicht nur aus Ihrer Apple Music-Bibliothek, sondern auch aus Audiodateien zu erstellen, die sich direkt auf Ihrem Gerät befinden.  
Dies ist ideal, wenn Sie Ihre lokale Sammlung von MP3-, FLAC- oder anderen Audiodateien in Playlists organisieren möchten.  

In diesem Tutorial erfahren Sie, wie Sie **Playlists aus Apple Music auf Ihrem Mac exportieren** und dann in die **Evermusic**- oder **Flacbox**-App importieren.  
So können Sie Ihre Lieblings-Playlists mit erweiterten Funktionen weiter anhören — ohne sie von Grund auf neu erstellen zu müssen.

## Eine Playlist aus Apple Music exportieren

Öffnen Sie zunächst die Playlist in der Apple Music-App auf Ihrem Mac.

  {{< card title="" subtitle="Öffnen Sie die Playlist in Apple Music" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/1-apple-music-playlist.webp" >}}
{{< /cards >}}

Gehen Sie zu **File → Library → Export Playlist** im oberen Menü.

{{< cards cols="1">}}
  {{< card title="" subtitle="Exportieren Sie die Playlist aus Ihrer Bibliothek" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/2-library-export-playlist.webp" >}}
{{< /cards >}}

Wählen Sie den Speicherort, an dem die M3U-Datei gespeichert wird.  

> Wir empfehlen, die M3U-Datei in Ihrem **Musik-Ordner** zu speichern. 
> Dies stellt sicher, dass alle Dateipfade konsistent sind und der Importvorgang in Evermusic oder Flacbox reibungslos funktioniert.  
> Da die Apps im Sandbox-Modus auf macOS laufen, müssen sowohl die **Playlist-Datei** als auch die **Mediendateien** im selben Ordner sein, damit der Import erfolgreich ist.

## Archivierung und Übertragung von Playlists

Evermusic und Flacbox bieten erweiterte Playlist-Verwaltungsfunktionen.  
Eine davon ist die Möglichkeit, eine Playlist in einer **ZIP-Datei** zu archivieren, die die Playlist-Datei und alle Titel enthält.  
Dies macht die Übertragung von Playlists auf ein anderes Gerät schnell und zuverlässig.

Wählen Sie einfach **Weitere Aktionen → Zum Archiv hinzufügen** aus dem Playlist-Menü.

{{< cards cols="1">}}
  {{< card title="" subtitle="Öffnen Sie weitere Aktionen für die Playlist" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/14-more-actions-playlist.webp" >}}
{{< /cards >}}

Nachdem Sie **Zum Archiv hinzufügen** ausgewählt haben, warten Sie einen kurzen Moment, während die App die Playlist verarbeitet.  
Sobald die Archivierung abgeschlossen ist, sehen Sie eine **Erfolgsmeldung**. Tippen Sie auf **Datei anzeigen**, damit die App das neu erstellte ZIP-Archiv anzeigt.

{{< cards cols="1">}}
  {{< card title="" subtitle="Archivierung abgeschlossen" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/15-archiving-completed.webp" >}}
{{< /cards >}}

Die App öffnet dann den **Export-Ordner**, in dem alle erstellten Archive gespeichert sind.

{{< cards cols="1">}}
  {{< card title="" subtitle="Öffnen Sie den Export-Ordner" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/16-export-folder.webp" >}}
{{< /cards >}}

Suchen Sie Ihr neu erstelltes Archiv, tippen Sie auf die Schaltfläche **Weitere Aktionen** daneben und wählen Sie **Im Finder anzeigen**.

{{< cards cols="1">}}
  {{< card title="" subtitle="Verwenden Sie Weitere Aktionen für die ZIP-Datei" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/17-more-actions-menu-zip-file.webp" >}}
{{< /cards >}}

Sie sehen nun den **tatsächlichen Speicherort der ZIP-Datei** auf Ihrem Mac.  
An diesem Punkt können Sie das Archiv einfach auf ein anderes Gerät übertragen.  
Aber vorher werfen wir einen genaueren Blick auf den Inhalt. Doppelklicken Sie auf die Datei, um sie zu entpacken.

{{< cards cols="1">}}
  {{< card title="" subtitle="ZIP-Datei im Finder anzeigen" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/18-zip-file-revealed-in-finder.webp" >}}
{{< /cards >}}

Darin finden Sie den **vollständigen Playlist-Inhalt** — alle in der Playlist enthaltenen Audiodateien sowie die **M3U-Playlist-Datei**.

{{< cards cols="1">}}
  {{< card title="" subtitle="ZIP-Datei entpacken" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/19-unarchived-zip-file.webp" >}}
{{< /cards >}}

Öffnen Sie schließlich die **M3U-Datei**, um ihren Inhalt zu überprüfen.  
Sie ist korrekt formatiert, sodass Sie beim Übertragen dieses ZIP-Archivs auf ein anderes Gerät und Entpacken die Playlist nahtlos importieren können.  
Die App stellt die Playlist mit der **korrekten Titelreihenfolge** und **allen zugehörigen Mediendateien** wieder her.

{{< cards cols="1">}}
  {{< card title="" subtitle="M3U-Playlist-Inhalt anzeigen" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/20-m3u-playlist-content.webp" >}}
{{< /cards >}}

## Fazit

Das Exportieren von Apple Music-Playlists und deren Wiedergabe in **Evermusic** oder **Flacbox** auf Ihrem Mac ist ein unkomplizierter Prozess.  
Indem Sie Ihre Playlists als **M3U-Dateien** speichern und in diese Apps importieren, erhalten Sie Zugang zu leistungsstarken Funktionen wie:

- FLAC-, MP3-, OGG- und Hi-Res-Audiowiedergabe  
- Metadaten-Bearbeitung und Album-Artwork-Unterstützung  
- Cloud-Speicher- und Netzlaufwerk-Integration  
- Playlist-Archivierung und geräteübergreifende Übertragung  

Mit diesem Workflow müssen Sie Playlists nicht mehr manuell neu erstellen.  
Genießen Sie stattdessen ein nahtloses Musikerlebnis mit mehr Kontrolle, besseren Funktionen und geräteübergreifender Portabilität.

## Häufig gestellte Fragen

{{% details title="Welches Playlist-Format exportiert Apple Music?" closed="true" %}}
Apple Music exportiert Playlists im M3U-Format, einem Standard-Playlist-Format, das von den meisten Musikplayern unterstützt wird, darunter Evermusic, Flacbox, VLC und foobar2000.
{{% /details %}}

{{% details title="Warum müssen die M3U-Datei und die Audiodateien im selben Ordner sein?" closed="true" %}}
Evermusic und Flacbox laufen im macOS-Sandbox-Modus, der den Dateizugriff auf Ordner beschränkt, denen Sie explizit die Berechtigung erteilen. Das Aufbewahren der M3U-Datei und der Audiodateien im selben Ordner stellt sicher, dass die App beim Import beide lesen kann.
{{% /details %}}

{{% details title="Kann ich Playlists zwischen Mac und iPhone übertragen?" closed="true" %}}
Ja. Verwenden Sie die Playlist-Archivierungsfunktion, um eine ZIP-Datei mit der Playlist und allen Titeln zu erstellen. Übertragen Sie die ZIP-Datei per AirDrop, iCloud Drive oder einer anderen Methode auf Ihr iPhone und importieren Sie sie dann in Evermusic oder Flacbox auf iOS.
{{% /details %}}

{{% details title="Funktioniert das mit Apple Music-Streaming-Titeln?" closed="true" %}}
Diese Methode funktioniert mit lokalen Audiodateien, die Sie zu Apple Music hinzugefügt haben. DRM-geschützte Streaming-Titel von Apple Music können nicht als M3U-Playlists exportiert werden.
{{% /details %}}

{{% details title="Welche Audioformate unterstützen Evermusic und Flacbox?" closed="true" %}}
Beide Apps unterstützen eine breite Palette von Formaten, darunter MP3, FLAC, AAC, WAV, OGG, AIFF, ALAC, WMA, APE und mehr. Sie unterstützen auch Hi-Res-Audiowiedergabe für verlustfreie Formate.
{{% /details %}}

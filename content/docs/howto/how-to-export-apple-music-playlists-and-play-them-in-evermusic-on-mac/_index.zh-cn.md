**摘要：** 使用 Apple Music 内置的 **File > Library > Export Playlist** 功能将任何播放列表保存为 M3U 文件。然后在 Mac 上将其导入 **Evermusic** 或 **Flacbox**。您还可以将播放列表归档为 ZIP 文件，以便轻松传输到其他设备。

macOS 上的 Apple Music 不仅允许您从 Apple Music 资料库创建播放列表，还可以从直接位于设备上的音频文件创建播放列表。  
如果您想将本地的 MP3、FLAC 或其他音频文件集合整理成播放列表，这非常完美。  

在本教程中，您将学习如何 **从 Mac 上的 Apple Music 导出播放列表**，然后将其导入 **Evermusic** 或 **Flacbox** 应用。  
这样，您可以继续使用高级功能收听您喜爱的播放列表——无需从头开始重新创建。

## 从 Apple Music 导出播放列表

首先在 Mac 上的 Apple Music 应用中打开播放列表。

{{< cards cols="1">}}
  {{< card title="" subtitle="在 Apple Music 中打开播放列表" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/1-apple-music-playlist.webp" >}}
{{< /cards >}}

从顶部菜单转到 **File → Library → Export Playlist**。

{{< cards cols="1">}}
  {{< card title="" subtitle="从资料库导出播放列表" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/2-library-export-playlist.webp" >}}
{{< /cards >}}

选择保存 M3U 文件的目标位置。  

> 我们建议将 M3U 文件保存在您的**音乐文件夹**中。 
> 这确保所有文件路径一致，Evermusic 或 Flacbox 中的导入过程将顺利进行。  
> 由于应用在 macOS 上以沙盒模式运行，**播放列表文件**和**媒体文件**都必须位于同一文件夹中才能成功导入。

{{< cards cols="1">}}
  {{< card title="" subtitle="选择 M3U 文件的保存位置" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/3-select-m3u-destination.webp" >}}
{{< /cards >}}

## 归档和传输播放列表

Evermusic 和 Flacbox 提供高级播放列表管理功能。  
其中之一是将播放列表归档为 **ZIP 文件**的功能，其中包含播放列表文件和所有曲目。  
这使得将播放列表传输到其他设备变得快速且可靠。

只需从播放列表菜单中选择**更多操作 → 添加到归档**。

{{< cards cols="1">}}
  {{< card title="" subtitle="打开播放列表的更多操作" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/14-more-actions-playlist.webp" >}}
{{< /cards >}}

选择**添加到归档**后，等待应用处理播放列表。  
归档完成后，您将看到**成功提示**。点击**显示文件**让应用显示新创建的 ZIP 归档。

{{< cards cols="1">}}
  {{< card title="" subtitle="归档完成" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/15-archiving-completed.webp" >}}
{{< /cards >}}

应用将打开**导出文件夹**，其中存储了所有创建的归档。

{{< cards cols="1">}}
  {{< card title="" subtitle="打开导出文件夹" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/16-export-folder.webp" >}}
{{< /cards >}}

找到新创建的归档，点击旁边的**更多操作**按钮，选择**在 Finder 中显示**。

{{< cards cols="1">}}
  {{< card title="" subtitle="对 ZIP 文件使用更多操作" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/17-more-actions-menu-zip-file.webp" >}}
{{< /cards >}}

现在您将看到 Mac 上 **ZIP 文件的实际位置**。  
此时，您可以轻松地将归档传输到其他设备。  
但在此之前，让我们仔细看看里面的内容。双击文件解压。

{{< cards cols="1">}}
  {{< card title="" subtitle="在 Finder 中显示 ZIP 文件" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/18-zip-file-revealed-in-finder.webp" >}}
{{< /cards >}}

在里面，您将找到**完整的播放列表内容**——播放列表中包含的所有音频文件以及 **M3U 播放列表文件**。

{{< cards cols="1">}}
  {{< card title="" subtitle="解压 ZIP 文件" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/19-unarchived-zip-file.webp" >}}
{{< /cards >}}

最后，打开 **M3U 文件**检查其内容。  
它的格式正确，因此当您将此 ZIP 归档移动到其他设备并解压时，您可以无缝导入播放列表。  
应用将恢复播放列表，保持**正确的曲目顺序**和**所有关联的媒体文件**。

  {{< card title="" subtitle="查看 M3U 播放列表内容" image="/docs/howto/how-to-export-apple-music-playlists-and-play-them-in-evermusic-on-mac/20-m3u-playlist-content.webp" >}}

## 结论

在 Mac 上导出 Apple Music 播放列表并在 **Evermusic** 或 **Flacbox** 中播放是一个简单的过程。  
通过将播放列表保存为 **M3U 文件**并将其导入这些应用，您可以获得强大的功能，例如：

- FLAC、MP3、OGG 和高分辨率音频播放  
- 元数据编辑和专辑封面支持  
- 云存储和网络驱动器集成  
- 播放列表归档和跨设备传输  

使用此工作流程，您不再需要手动重新创建播放列表。  
相反，享受更多控制、更好功能和跨设备便携性的无缝音乐体验。

## 常见问题

{{% details title="Apple Music 以什么格式导出播放列表？" closed="true" %}}
Apple Music 以 M3U 格式导出播放列表，这是大多数音乐播放器支持的标准播放列表格式，包括 Evermusic、Flacbox、VLC 和 foobar2000。
{{% /details %}}

{{% details title="为什么 M3U 文件和音频文件必须在同一文件夹中？" closed="true" %}}
Evermusic 和 Flacbox 在 macOS 沙盒模式下运行，将文件访问限制在您明确授权的文件夹中。将 M3U 文件和音频文件保存在同一文件夹中可确保应用在导入期间能够读取两者。
{{% /details %}}

{{% details title="我可以在 Mac 和 iPhone 之间传输播放列表吗？" closed="true" %}}
可以。使用播放列表归档功能创建包含播放列表和所有曲目的 ZIP 文件。通过 AirDrop、iCloud Drive 或任何其他方法将 ZIP 传输到 iPhone，然后在 iOS 上的 Evermusic 或 Flacbox 中导入。
{{% /details %}}

{{% details title="这适用于 Apple Music 流媒体曲目吗？" closed="true" %}}
此方法适用于您添加到 Apple Music 的本地音频文件。Apple Music 中受 DRM 保护的流媒体曲目无法导出为 M3U 播放列表。
{{% /details %}}

{{% details title="Evermusic 和 Flacbox 支持哪些音频格式？" closed="true" %}}
两个应用都支持广泛的格式，包括 MP3、FLAC、AAC、WAV、OGG、AIFF、ALAC、WMA、APE 等。它们还支持无损格式的高分辨率音频播放。
{{% /details %}}

## Komentar Berwaktu
Tingkatkan komentar Anda ke level berikutnya dengan menyisipkan penanda waktu. Menggunakan format [MM:SS.SS], Anda dapat memastikan bahwa komentar Anda disorot dengan tepat saat pemutar audio mencapai interval waktu tertentu selama pemutaran. Ini adalah cara dinamis untuk berinteraksi dengan musik Anda, memberikan konteks dan wawasan pada momen-momen kunci.

[ti: Lorem Ipsum]

## Beralih antara Komentar / Lirik Tersemat / File LRC
Setelah Anda membuka layar Komentar, Anda dapat beralih antara tiga mode dengan menggeser ke kanan: Komentar, Lirik Tersemat, dan File LRC.

1. **Mode Komentar:** Menampilkan konten dari bidang KOMENTAR file audio Anda.
2. **Mode Lirik:** Menampilkan lirik tersemat dari file audio Anda. Jika file audio Anda memiliki lirik di tag, lirik tersebut akan muncul di sini. Mode ini juga mendukung teks berwaktu dalam format LRC. Untuk mengedit tag lirik, gunakan editor tag audio eksternal seperti Evertag. Anda dapat menemukan lirik format LRC secara gratis di situs web seperti [Lyricsify](https://www.lyricsify.com). Salin lirik yang diperlukan ke tag Anda dan simpan untuk ditampilkan nanti di layar ini.
3. **Mode File LRC:** Alih-alih mengedit file audio, Anda dapat menempatkan file LRC di folder yang sama dengan file audio asli. Kedua file harus memiliki nama yang sama tetapi ekstensi yang berbeda. Saat Anda menggeser ke halaman ketiga di layar Komentar, aplikasi akan mencari file LRC di direktori yang sama dan menampilkan kontennya.

Pengaturan ini memastikan Anda dapat dengan mudah melihat komentar, lirik, dan file LRC untuk trek audio Anda.

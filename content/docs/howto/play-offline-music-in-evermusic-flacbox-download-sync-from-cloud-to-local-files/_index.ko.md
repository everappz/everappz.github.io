**요약:** Evermusic과 Flacbox를 사용하면 클라우드 스토리지(Google Drive, Dropbox, OneDrive 등)에서 iPhone 또는 iPad로 음악을 다운로드하여 오프라인으로 재생할 수 있습니다. 직접 다운로드, 자동 동기화가 포함된 오프라인 모드, 오디오 플레이어 캐시 등 세 가지 방법을 사용할 수 있습니다. 이 가이드에서는 세 가지 방법을 단계별로 설명합니다.

음악 스트리밍이 지배적인 세상에서도 좋아하는 트랙을 오프라인으로 사용할 수 있다는 것은 여전히 큰 가치가 있습니다. Evermusic과 Flacbox는 클라우드 스토리지에서 로컬 파일로 음악을 다운로드하고 동기화할 수 있는 강력한 앱으로, 언제 어디서나 음악에 액세스할 수 있습니다. 이 가이드에서는 Evermusic과 Flacbox를 사용하여 인터넷 연결 없이도 음악 라이브러리에 접근할 수 있도록 오프라인 음악을 재생하는 단계를 안내합니다.

저희 음악 앱은 음악을 오프라인으로 사용할 수 있게 하는 두 가지 주요 방법을 제공합니다: 간단한 다운로드와 자동 동기화를 위한 오프라인 모드 활성화.

## 오프라인 재생을 위한 음악 다운로드

1. **연결하기 탭 열기:**
   - Evermusic 또는 Flacbox를 실행합니다.
   - "연결하기" 탭으로 이동하여 연결된 모든 클라우드 서비스를 확인합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **클라우드 서비스 열기:**
   - 액세스하려는 클라우드 서비스의 아이콘을 탭합니다.
   - 다운로드하려는 오디오 파일이 포함된 폴더를 탐색합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **음악 다운로드:**
   - 폴더 이름 옆의 점 세 개를 탭하여 추가 작업 메뉴를 엽니다.
   - **다운로드** 옵션을 선택하여 오프라인 재생을 위한 폴더 다운로드를 시작합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - 폴더가 앱의 문서 디렉토리에 다운로드됩니다.
   - 다운로드 진행 상황을 확인하려면 로컬 파일 화면의 왼쪽 상단에 있는 회전 화살표 아이콘을 탭하여 **파일 전송** 화면으로 이동합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - 다운로드된 모든 파일은 문서 - 다운로드 폴더 또는 앱 설정에서 구성한 사용자 지정 다운로드 폴더에서 찾을 수 있습니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## 자동 동기화를 위한 오프라인 모드 활성화

1. **오프라인 모드 활성화:**
   - 단순히 다운로드하는 대신 오프라인 모드를 활성화하여 클라우드 서비스에서 변경 사항(예: 파일 업데이트 또는 새 파일 추가)이 발생하면 로컬 파일을 자동으로 업데이트합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - 오프라인 모드는 단방향 동기화를 수행하며, 클라우드 서비스의 변경 사항만 로컬 사본에 반영합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **수동 동기화:**
   - 점 세 개 버튼을 통해 접근할 수 있는 추가 작업 메뉴에서 **동기화하다** 작업을 선택하여 오프라인 폴더를 수동으로 동기화합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **재생 목록/앨범/아티스트/장르에 대한 오프라인 모드:**
   - 재생 목록에 대해 오프라인 모드를 활성화하려면 재생 목록을 열고 오프라인 모드 작업 옆의 확인란을 선택합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - 앱이 재생 목록의 모든 트랙을 스캔하고 로컬 파일 - 오프라인 폴더 - 재생 목록 - [재생 목록 이름] 폴더에 다운로드합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **다운로드 설정:**
   - 앱 설정에서 다운로드 속도를 높이기 위해 최대 병렬 작업 수를 조정하고, 백그라운드 전송을 활성화하며, 다운로드 폴더 위치를 변경할 수 있습니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - 설정 - 파일 관리자 - 동기화된 오프라인 폴더를 통해 오프라인 폴더를 관리합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## 동기화된 오프라인 폴더 관리

1. **폴더 관리:**
   - 동기화된 오프라인 폴더 섹션에서 동기화를 시작 및 중지하고, 포함된 디렉토리에서 폴더를 표시하거나, 오프라인 모드를 비활성화할 수 있습니다(이렇게 하면 장치에서 파일의 로컬 사본이 제거됩니다).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **동기화 간격 설정:**
   - 오프라인 폴더의 동기화 간격(기본값: 1일)을 구성하여 앱을 열 때 자동으로 업데이트를 확인하고 로컬 사본을 동기화합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **음악 라이브러리 통합:**
   - 다운로드된 모든 파일은 음악 라이브러리에 추가됩니다. 음악 라이브러리 섹션의 오른쪽 상단에 있는 추가 작업 메뉴를 탭하고 **설정**을 선택하여 음악 라이브러리 설정을 관리합니다.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## 오디오 플레이어 캐시 사용

클라우드 파일을 다운로드하는 마지막 옵션은 오디오 플레이어 캐시를 사용하는 것입니다. 작동 방식은 다음과 같습니다: 앱 설정에서 오디오 플레이어 캐시를 활성화합니다(오디오 플레이어 > 오디오 플레이어 캐시).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- 캐시 크기를 설정하면 이전 파일이 자동으로 제거되고 새 파일이 로컬 파일 > 오디오 플레이어 폴더에 다운로드됩니다. 클라우드에서 음악을 재생하기 시작하면 앱이 자동으로 대기열의 다음 트랙을 미리 다운로드하여 트랙 간의 원활한 전환을 보장합니다.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **오디오 플레이어 캐시 활성화:**
   - 설정 - 오디오 플레이어 - 오디오 플레이어 캐시로 이동하여 캐시를 활성화합니다.
   - 캐시 크기를 설정하면 이전 파일이 자동으로 제거되고 새 파일이 로컬 파일 - 오디오 플레이어 폴더에 다운로드됩니다.
   - 클라우드에서 음악을 재생할 때 앱이 원활한 전환을 위해 다음 트랙을 미리 다운로드합니다.

2. **캐시 위치 확인:**
   - 설정 화면에서 **폴더에서 보기**를 선택하여 오디오 플레이어 캐시 위치를 확인합니다.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## 중복 파일 방지

앱은 파일을 다운로드하는 여러 옵션을 제공합니다: 오프라인 모드, 직접 다운로드, 오디오 플레이어 캐시. 중복을 방지하려면 한 가지 방법만 사용하세요. 또한 트랙이 여러 컬렉션(예: 앨범 또는 재생 목록)에 있고 두 컬렉션 모두에 대해 오프라인 모드를 활성화하면 오프라인 모드가 각 컬렉션에 대해 독립적으로 작동하므로 트랙이 두 번 다운로드됩니다.

1. **여러 사본 관리:**
   - **폴더에서 보기** 옵션을 사용하여 파일을 찾고 중복을 관리하여 동일한 파일의 여러 사본을 방지합니다.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## 문제 해결 및 팁

1. **파일의 여러 사본:**
   - 여러 다운로드 모드(예: 오디오 플레이어 캐시, 폴더 다운로드, 클라우드 폴더의 오프라인 모드, 재생 목록, 앨범, 아티스트)를 활성화하면 중복 파일이 발생할 수 있습니다. **폴더에서 보기**를 사용하여 이러한 파일을 찾고 관리합니다.

2. **백그라운드 전송:**
   - 앱이 활성화되지 않은 상태에서도 다운로드가 계속되도록 설정에서 백그라운드 전송을 활성화합니다.

3. **동기화 간격:**
   - 설정에서 동기화 간격을 조정하여 음악 라이브러리가 클라우드 서비스의 최신 변경 사항과 항상 동기화되도록 합니다.

이 상세한 단계를 따르면 Evermusic과 Flacbox를 사용하여 로컬 및 클라우드 저장 음악을 원활하게 관리하고 오프라인으로 재생할 수 있습니다. 정리되고 언제든지 재생할 준비가 된 음악 라이브러리에 중단 없이 액세스하세요.

{{% details title="클라우드 스토리지에서 iPhone으로 음악을 자동으로 동기화할 수 있나요?" closed="true" %}}

{{% details title="오프라인 모드는 장치의 저장 공간을 많이 사용하나요?" closed="true" %}}

{{% /details %}}

{{% details title="오프라인 재생에 지원되는 오디오 형식은 무엇인가요?" closed="true" %}}

{{% /details %}}

{{% details title="앱을 닫으면 오프라인 음악이 계속 재생되나요?" closed="true" %}}
예. 다운로드된 파일은 장치에 로컬로 저장되며 인터넷 연결에 관계없이 앱의 오디오 플레이어를 통해 재생됩니다. 백그라운드 재생이 완전히 지원됩니다.
{{% /details %}}

{{% /details %}}

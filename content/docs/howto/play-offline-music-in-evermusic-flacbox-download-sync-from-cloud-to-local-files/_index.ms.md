**TL;DR:** Evermusic dan Flacbox membolehkan anda memuat turun muzik dari storan awan (Google Drive, Dropbox, OneDrive, dan lain-lain) ke iPhone atau iPad anda untuk main balik luar talian. Anda boleh menggunakan tiga kaedah: muat turun terus, mod luar talian dengan penyelerasan automatik, atau cache pemain audio. Panduan ini merangkumi ketiga-tiga pendekatan langkah demi langkah.

Dalam dunia di mana penstriman muzik mendominasi, masih ada nilai besar dalam mempunyai trek kegemaran anda tersedia secara luar talian. Evermusic dan Flacbox adalah aplikasi berkuasa yang membolehkan anda memuat turun dan menyelaraskan muzik dari storan awan anda terus ke fail tempatan anda, memastikan anda mempunyai akses kepada muzik anda pada bila-bila masa, di mana sahaja. Dalam panduan ini, kami akan membimbing anda melalui langkah-langkah untuk memainkan muzik luar talian dengan mudah menggunakan Evermusic dan Flacbox, menjadikan pustaka muzik anda boleh diakses walaupun tanpa sambungan internet.

Aplikasi muzik kami menawarkan dua kaedah utama untuk menjadikan muzik tersedia secara luar talian: memuat turun secara mudah dan mengaktifkan mod luar talian untuk penyelerasan automatik.

## Memuat Turun Muzik untuk Main Balik Luar Talian

1. **Buka Tab Sambungan:**
   - Lancarkan Evermusic atau Flacbox.
   - Navigasi ke tab "Sambungan" untuk melihat semua perkhidmatan awan yang disambungkan.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **Buka Perkhidmatan Awan:**
   - Ketik ikon perkhidmatan awan yang anda ingin akses.
   - Layari ke folder yang mengandungi fail audio yang anda ingin muat turun.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **Muat Turun Muzik:**
   - Ketik tiga titik di sebelah nama folder untuk membuka menu tindakan tambahan.
   - Pilih pilihan **Muat Turun** untuk mula memuat turun folder untuk main balik luar talian.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Folder akan dimuat turun ke direktori Dokumen aplikasi.
   - Untuk menyemak kemajuan muat turun, pergi ke skrin **Pemindahan Fail** dengan mengetik ikon anak panah berputar di sudut kiri atas skrin Fail Tempatan.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - Anda boleh menemui semua fail yang dimuat turun dalam folder Dokumen - Muat Turun atau folder muat turun tersuai anda jika dikonfigurasi dalam tetapan aplikasi.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## Mengaktifkan Mod Luar Talian untuk Penyelerasan Automatik

1. **Aktifkan Mod Luar Talian:**
   - Daripada hanya memuat turun, aktifkan mod luar talian untuk mengemas kini fail tempatan secara automatik apabila perubahan berlaku pada perkhidmatan awan (contohnya, kemas kini fail atau fail baharu ditambah).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Mod luar talian melakukan penyelerasan sehala, hanya mencerminkan perubahan perkhidmatan awan dalam salinan tempatan anda.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **Penyelerasan Manual:**
   - Selaraskan folder luar talian anda secara manual dengan memilih tindakan **Menyelaraskan** dari menu tindakan tambahan, yang boleh diakses melalui butang tiga titik.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **Mod Luar Talian untuk Senarai Main/Album/Artis/Genre:**
   - Untuk mengaktifkan mod luar talian untuk senarai main, buka senarai main dan tandakan kotak berhampiran tindakan mod luar talian.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - Aplikasi akan mengimbas semua trek dalam senarai main dan memuat turunnya ke folder Fail Tempatan - Folder Luar Talian - Senarai Main - [Nama Senarai Main].

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **Tetapan untuk Muat Turun:**
   - Dalam tetapan aplikasi, anda boleh melaraskan bilangan maksimum tugas selari untuk mempercepatkan muat turun, mengaktifkan pemindahan latar belakang, dan menukar lokasi folder muat turun.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - Urus folder luar talian melalui Tetapan - Pengurus Fail - Folder luar talian yang diselaraskan.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## Mengurus Folder Luar Talian yang Diselaraskan

1. **Pengurusan Folder:**
   - Dalam bahagian Folder luar talian yang diselaraskan, anda boleh memulakan dan menghentikan penyelerasan, menunjukkan folder dalam direktori yang mengandunginya, atau menyahaktifkan mod luar talian (yang membuang salinan tempatan fail dari peranti anda).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **Tetapkan Selang Penyelerasan:**
   - Konfigurasikan selang penyelerasan untuk folder luar talian (lalai ialah 1 hari) untuk menyemak kemas kini dan menyelaraskan salinan tempatan secara automatik apabila aplikasi dibuka.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **Integrasi Pustaka Muzik:**
   - Semua fail yang dimuat turun ditambahkan ke pustaka muzik. Urus tetapan pustaka muzik dengan mengetik menu tindakan tambahan di sudut kanan atas bahagian Pustaka Muzik dan memilih **Tetapan**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## Menggunakan Cache Pemain Audio

Pilihan terakhir untuk memuat turun fail awan ialah menggunakan cache pemain audio. Begini cara ia berfungsi: aktifkan cache pemain audio dalam tetapan aplikasi (Pemain Audio > Cache Pemain Audio).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- Tetapkan saiz cache supaya fail lama dibuang secara automatik, dan fail baharu dimuat turun ke folder Fail Tempatan > Pemain Audio. Apabila anda mula memainkan muzik dari awan, aplikasi akan secara automatik memuat turun terlebih dahulu trek seterusnya dalam barisan, memastikan peralihan yang lancar antara trek.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **Aktifkan Cache Pemain Audio:**
   - Pergi ke Tetapan - Pemain Audio - Cache Pemain Audio untuk mengaktifkan cache.
   - Tetapkan saiz cache supaya fail lama dibuang secara automatik dan fail baharu dimuat turun ke folder Fail Tempatan - Pemain Audio.
   - Apabila memainkan muzik dari awan, aplikasi akan memuat turun terlebih dahulu trek seterusnya untuk memastikan peralihan yang lancar.

2. **Semak Lokasi Cache:**
   - Sahkan lokasi cache pemain audio dengan memilih **Tunjukkan dalam Folder** pada skrin tetapan.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## Mengelakkan Fail Pendua

Aplikasi menawarkan beberapa pilihan untuk memuat turun fail: Mod Luar Talian, Muat Turun Terus, dan Cache Pemain Audio. Untuk mengelakkan pendua, gunakan hanya satu kaedah. Selain itu, jika trek berada dalam pelbagai koleksi (seperti album atau senarai main) dan anda mengaktifkan Mod Luar Talian untuk kedua-duanya, trek akan dimuat turun dua kali, kerana Mod Luar Talian berfungsi secara bebas untuk setiap koleksi.

1. **Urus Pelbagai Salinan:**
   - Elakkan mempunyai pelbagai salinan fail yang sama dengan menggunakan pilihan **Tunjukkan dalam Folder** untuk mencari fail dan mengurus pendua.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## Penyelesaian Masalah dan Petua

1. **Pelbagai Salinan Fail:**
   - Ketahui bahawa mengaktifkan pelbagai mod muat turun (contohnya, cache pemain audio, muat turun folder, mod luar talian untuk folder awan, senarai main, album, artis) boleh menghasilkan fail pendua. Gunakan **Tunjukkan dalam Folder** untuk mencari dan mengurus fail ini.

2. **Pemindahan Latar Belakang:**
   - Pastikan pemindahan latar belakang diaktifkan dalam tetapan untuk memastikan muat turun terus berjalan walaupun aplikasi tidak aktif.

3. **Selang Penyelerasan:**
   - Laraskan selang penyelerasan dalam tetapan untuk memastikan pustaka muzik anda sentiasa dikemas kini dengan perubahan terkini dari perkhidmatan awan anda.

Dengan mengikuti langkah-langkah terperinci ini, anda boleh mengurus dan memainkan muzik tempatan dan yang disimpan di awan secara luar talian dengan lancar menggunakan Evermusic dan Flacbox. Nikmati akses tanpa gangguan ke pustaka muzik anda, tersusun dan sedia untuk dimainkan bila-bila masa anda mahu.

## Soalan Lazim

{{% details title="Apakah perkhidmatan awan yang disokong oleh Evermusic dan Flacbox?" closed="true" %}}
Kedua-dua aplikasi menyokong Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk, dan penyedia storan awan utama yang lain. Anda boleh menyambungkan pelbagai perkhidmatan pada masa yang sama.
{{% /details %}}

{{% details title="Bolehkah saya menyelaraskan muzik secara automatik dari storan awan ke iPhone saya?" closed="true" %}}
Ya. Aktifkan Mod Luar Talian untuk mana-mana folder, senarai main, album, atau artis. Aplikasi melakukan penyelerasan sehala dari awan ke peranti pada selang yang boleh dikonfigurasikan (lalai: sekali sehari).
{{% /details %}}

{{% details title="Adakah mod luar talian menggunakan banyak storan pada peranti saya?" closed="true" %}}
Penggunaan storan bergantung pada saiz koleksi muzik anda dan format fail. Anda boleh mengawalnya dengan memilih folder tertentu untuk diselaraskan, menetapkan had saiz cache, dan memantau storan dalam tetapan aplikasi.
{{% /details %}}

{{% details title="Apakah format audio yang disokong untuk main balik luar talian?" closed="true" %}}
Evermusic dan Flacbox menyokong MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, WMA, dan banyak format lain. Flacbox dioptimumkan untuk format tanpa rugi seperti FLAC dan ALAC.
{{% /details %}}

{{% details title="Adakah muzik luar talian saya akan terus dimainkan jika saya menutup aplikasi?" closed="true" %}}
Ya. Fail yang dimuat turun disimpan secara tempatan pada peranti anda dan dimainkan melalui pemain audio aplikasi tanpa mengira sambungan internet. Main balik latar belakang disokong sepenuhnya.
{{% /details %}}

{{% details title="Bagaimana saya membebaskan ruang yang digunakan oleh muzik luar talian?" closed="true" %}}
Nyahaktifkan Mod Luar Talian untuk folder tertentu dalam Tetapan > Pengurus Fail > Folder luar talian yang diselaraskan. Ini membuang salinan tempatan dari peranti anda. Anda juga boleh mengosongkan cache pemain audio atau memadam fail yang dimuat turun secara manual.
{{% /details %}}

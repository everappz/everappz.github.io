**Περίληψη:** Το Evermusic και το Flacbox σας επιτρέπουν να κατεβάζετε μουσική από αποθηκευτικό χώρο cloud (Google Drive, Dropbox, OneDrive και άλλα) στο iPhone ή το iPad σας για αναπαραγωγή εκτός σύνδεσης. Μπορείτε να χρησιμοποιήσετε τρεις μεθόδους: απευθείας λήψη, λειτουργία εκτός σύνδεσης με αυτόματο συγχρονισμό ή προσωρινή μνήμη αναπαραγωγέα ήχου. Αυτός ο οδηγός καλύπτει και τις τρεις προσεγγίσεις βήμα προς βήμα.

Σε έναν κόσμο όπου η ροή μουσικής κυριαρχεί, εξακολουθεί να υπάρχει μεγάλη αξία στο να έχετε τα αγαπημένα σας κομμάτια διαθέσιμα εκτός σύνδεσης. Το Evermusic και το Flacbox είναι ισχυρές εφαρμογές που σας επιτρέπουν να κατεβάζετε και να συγχρονίζετε μουσική από τον αποθηκευτικό χώρο cloud απευθείας στα τοπικά σας αρχεία, διασφαλίζοντας ότι έχετε πρόσβαση στα τραγούδια σας οποτεδήποτε και οπουδήποτε. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στα βήματα για να αναπαράγετε εύκολα μουσική εκτός σύνδεσης χρησιμοποιώντας το Evermusic και το Flacbox, κάνοντας τη μουσική βιβλιοθήκη σας προσβάσιμη ακόμα και χωρίς σύνδεση στο internet.

Οι εφαρμογές μουσικής μας προσφέρουν δύο κύριες μεθόδους για να κάνετε τη μουσική διαθέσιμη εκτός σύνδεσης: απλή λήψη και ενεργοποίηση της λειτουργίας εκτός σύνδεσης για αυτόματο συγχρονισμό.

## Λήψη μουσικής για αναπαραγωγή εκτός σύνδεσης

1. **Ανοίξτε την καρτέλα Συνδέσεις:**
   - Εκκινήστε το Evermusic ή το Flacbox.
   - Μεταβείτε στην καρτέλα "**Συνδέσεις**" για να δείτε όλες τις συνδεδεμένες υπηρεσίες cloud.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **Ανοίξτε την υπηρεσία cloud:**
   - Πατήστε το εικονίδιο της υπηρεσίας cloud στην οποία θέλετε να αποκτήσετε πρόσβαση.
   - Περιηγηθείτε στον φάκελο που περιέχει τα αρχεία ήχου που θέλετε να κατεβάσετε.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **Λήψη μουσικής:**
   - Πατήστε τις τρεις τελείες δίπλα στο όνομα του φακέλου για να ανοίξετε το μενού περισσότερων ενεργειών.
   - Επιλέξτε την επιλογή **Λήψη** για να ξεκινήσετε τη λήψη του φακέλου για αναπαραγωγή εκτός σύνδεσης.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Ο φάκελος θα ληφθεί στον κατάλογο **Έγγραφα** της εφαρμογής.
   - Για να ελέγξετε την πρόοδο λήψης, μεταβείτε στην οθόνη **Μεταφορές αρχείων** πατώντας το εικονίδιο με τα περιστρεφόμενα βέλη στην επάνω αριστερή γωνία της οθόνης τοπικών αρχείων.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - Μπορείτε να βρείτε όλα τα ληφθέντα αρχεία στον φάκελο **Έγγραφα** - **Λήψεις** ή στον προσαρμοσμένο φάκελο λήψεων σας, εάν έχει ρυθμιστεί στις ρυθμίσεις της εφαρμογής.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## Ενεργοποίηση λειτουργίας εκτός σύνδεσης για αυτόματο συγχρονισμό

1. **Ενεργοποιήστε τη λειτουργία εκτός σύνδεσης:**
   - Αντί να κάνετε απλώς λήψη, ενεργοποιήστε τη λειτουργία εκτός σύνδεσης για αυτόματη ενημέρωση των τοπικών αρχείων όταν γίνονται αλλαγές στην υπηρεσία cloud (π.χ. ενημερώσεις αρχείων ή προσθήκη νέων αρχείων).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Η λειτουργία εκτός σύνδεσης εκτελεί μονόδρομο συγχρονισμό, αντικατοπτρίζοντας μόνο τις αλλαγές της υπηρεσίας cloud στο τοπικό σας αντίγραφο.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **Χειροκίνητος συγχρονισμός:**
   - Συγχρονίστε χειροκίνητα τον φάκελο εκτός σύνδεσης επιλέγοντας την ενέργεια **Συγχρονισμός** από το μενού περισσότερων ενεργειών, προσβάσιμο μέσω του κουμπιού με τις τρεις τελείες.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **Λειτουργία εκτός σύνδεσης για λίστες αναπαραγωγής/άλμπουμ/καλλιτέχνες/είδη:**
   - Για να ενεργοποιήσετε τη λειτουργία εκτός σύνδεσης για τις λίστες αναπαραγωγής, ανοίξτε τη λίστα αναπαραγωγής και επιλέξτε το πλαίσιο δίπλα στην ενέργεια λειτουργίας εκτός σύνδεσης.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - Η εφαρμογή θα σαρώσει όλα τα κομμάτια στη λίστα αναπαραγωγής και θα τα κατεβάσει στον φάκελο Τοπικά αρχεία - Εκτός σύνδεσης φάκελοι - Λίστες αναπαραγωγής - [Όνομα λίστας αναπαραγωγής].

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **Ρυθμίσεις για λήψεις:**
   - Στις ρυθμίσεις της εφαρμογής, μπορείτε να ρυθμίσετε τον μέγιστο αριθμό παράλληλων εργασιών για να επιταχύνετε τις λήψεις, να ενεργοποιήσετε τις μεταφορές στο παρασκήνιο και να αλλάξετε τη θέση του φακέλου λήψης.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - Διαχειριστείτε τους φακέλους εκτός σύνδεσης μέσω **Ρυθμίσεις** - Διαχείριση αρχείων - **Συγχρονισμένοι εκτός σύνδεσης φάκελοι**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## Διαχείριση συγχρονισμένων εκτός σύνδεσης φακέλων

1. **Διαχείριση φακέλων:**
   - Στην ενότητα **Συγχρονισμένοι εκτός σύνδεσης φάκελοι**, μπορείτε να ξεκινήσετε και να σταματήσετε τον συγχρονισμό, να εμφανίσετε έναν φάκελο στον γονικό κατάλογό του ή να απενεργοποιήσετε τη λειτουργία εκτός σύνδεσης (η οποία αφαιρεί τα τοπικά αντίγραφα των αρχείων από τη συσκευή σας).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **Ορισμός διαστήματος συγχρονισμού:**
   - Ρυθμίστε το διάστημα συγχρονισμού για τους φακέλους εκτός σύνδεσης (προεπιλογή 1 ημέρα) για αυτόματο έλεγχο ενημερώσεων και συγχρονισμό τοπικών αντιγράφων κατά το άνοιγμα της εφαρμογής.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **Ενσωμάτωση μουσικής βιβλιοθήκης:**
   - Όλα τα ληφθέντα αρχεία προστίθενται στη μουσική βιβλιοθήκη. Διαχειριστείτε τις ρυθμίσεις της μουσικής βιβλιοθήκης πατώντας στο μενού περισσότερων ενεργειών στην επάνω δεξιά γωνία της ενότητας μουσικής βιβλιοθήκης και επιλέγοντας **Ρυθμίσεις**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## Χρήση προσωρινής μνήμης αναπαραγωγέα ήχου

Η τελευταία επιλογή για λήψη αρχείων cloud είναι η χρήση της προσωρινής μνήμης αναπαραγωγέα ήχου. Δείτε πώς λειτουργεί: ενεργοποιήστε την προσωρινή μνήμη αναπαραγωγέα ήχου στις ρυθμίσεις της εφαρμογής (Αναπαραγωγέας ήχου > Προσωρινή μνήμη αναπαραγωγέα ήχου).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- Ορίστε το μέγεθος της προσωρινής μνήμης ώστε τα παλιά αρχεία να αφαιρούνται αυτόματα και τα νέα αρχεία να λαμβάνονται στον φάκελο Τοπικά αρχεία > Αναπαραγωγέας ήχου. Όταν ξεκινήσετε την αναπαραγωγή μουσικής από το cloud, η εφαρμογή θα κατεβάσει αυτόματα εκ των προτέρων το επόμενο κομμάτι στην ουρά, εξασφαλίζοντας ομαλές μεταβάσεις μεταξύ κομματιών.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **Ενεργοποίηση προσωρινής μνήμης αναπαραγωγέα ήχου:**
   - Μεταβείτε στις **Ρυθμίσεις** - Αναπαραγωγέας ήχου - Προσωρινή μνήμη αναπαραγωγέα ήχου για να ενεργοποιήσετε την προσωρινή μνήμη.
   - Ορίστε το μέγεθος της προσωρινής μνήμης ώστε τα παλιά αρχεία να αφαιρούνται αυτόματα και τα νέα αρχεία να λαμβάνονται στον φάκελο Τοπικά αρχεία - Αναπαραγωγέας ήχου.
   - Κατά την αναπαραγωγή μουσικής από το cloud, η εφαρμογή θα κατεβάσει εκ των προτέρων το επόμενο κομμάτι για ομαλές μεταβάσεις.

2. **Έλεγχος θέσης προσωρινής μνήμης:**
   - Επαληθεύστε τη θέση της προσωρινής μνήμης αναπαραγωγέα ήχου επιλέγοντας **Εμφάνιση στον φάκελο** στην οθόνη ρυθμίσεων.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## Αποφυγή διπλότυπων αρχείων

Η εφαρμογή προσφέρει πολλές επιλογές για λήψη αρχείων: Λειτουργία εκτός σύνδεσης, Απευθείας λήψη και Προσωρινή μνήμη αναπαραγωγέα ήχου. Για να αποφύγετε τα διπλότυπα, χρησιμοποιήστε μόνο μία μέθοδο. Επιπλέον, εάν ένα κομμάτι βρίσκεται σε πολλές συλλογές (όπως άλμπουμ ή λίστες αναπαραγωγής) και ενεργοποιήσετε τη Λειτουργία εκτός σύνδεσης και για τις δύο, το κομμάτι θα ληφθεί δύο φορές, καθώς η Λειτουργία εκτός σύνδεσης λειτουργεί ανεξάρτητα για κάθε συλλογή.

1. **Διαχείριση πολλαπλών αντιγράφων:**
   - Αποφύγετε πολλαπλά αντίγραφα του ίδιου αρχείου χρησιμοποιώντας την επιλογή **Εμφάνιση στον φάκελο** για να εντοπίσετε αρχεία και να διαχειριστείτε τα διπλότυπα.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## Αντιμετώπιση προβλημάτων και συμβουλές

1. **Πολλαπλά αντίγραφα αρχείων:**
   - Λάβετε υπόψη ότι η ενεργοποίηση πολλαπλών λειτουργιών λήψης (π.χ. προσωρινή μνήμη αναπαραγωγέα ήχου, λήψη φακέλων, λειτουργία εκτός σύνδεσης για φακέλους cloud, λίστες αναπαραγωγής, άλμπουμ, καλλιτέχνες) μπορεί να οδηγήσει σε διπλότυπα αρχεία. Χρησιμοποιήστε **Εμφάνιση στον φάκελο** για να εντοπίσετε και να διαχειριστείτε αυτά τα αρχεία.

2. **Μεταφορές στο παρασκήνιο:**
   - Βεβαιωθείτε ότι οι μεταφορές στο παρασκήνιο είναι ενεργοποιημένες στις ρυθμίσεις για να συνεχίζονται οι λήψεις ακόμα και όταν η εφαρμογή δεν είναι ενεργή.

3. **Διαστήματα συγχρονισμού:**
   - Ρυθμίστε τα διαστήματα συγχρονισμού στις ρυθμίσεις για να διασφαλίσετε ότι η μουσική βιβλιοθήκη σας είναι πάντα ενημερωμένη με τις τελευταίες αλλαγές από την υπηρεσία cloud σας.

Ακολουθώντας αυτά τα λεπτομερή βήματα, μπορείτε να διαχειρίζεστε και να αναπαράγετε απρόσκοπτα τη μουσική σας τοπικά και από το cloud εκτός σύνδεσης χρησιμοποιώντας το Evermusic και το Flacbox. Απολαύστε αδιάκοπη πρόσβαση στη μουσική βιβλιοθήκη σας, οργανωμένη και έτοιμη για αναπαραγωγή όποτε θέλετε.

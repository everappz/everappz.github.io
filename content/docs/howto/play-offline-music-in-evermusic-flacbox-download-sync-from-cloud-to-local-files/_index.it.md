**Riepilogo:** Evermusic e Flacbox ti permettono di scaricare musica dall'archiviazione cloud (Google Drive, Dropbox, OneDrive e altro) sul tuo iPhone o iPad per la riproduzione offline. Puoi utilizzare tre metodi: download diretto, modalità offline con sincronizzazione automatica o cache del lettore audio. Questa guida copre tutti e tre gli approcci passo dopo passo.

In un mondo in cui lo streaming musicale domina, c'è ancora un grande valore nell'avere i tuoi brani preferiti disponibili offline. Evermusic e Flacbox sono applicazioni potenti che ti permettono di scaricare e sincronizzare la musica dalla tua archiviazione cloud direttamente nei tuoi file locali, assicurandoti l'accesso alla tua musica in qualsiasi momento e ovunque. In questa guida, ti guideremo attraverso i passaggi per riprodurre facilmente musica offline utilizzando Evermusic e Flacbox, rendendo la tua libreria musicale accessibile anche senza connessione a internet.

Le nostre app musicali offrono due metodi principali per rendere la musica disponibile offline: il download semplice e l'abilitazione della modalità offline per la sincronizzazione automatica.

## Scaricare musica per la riproduzione offline

1. **Apri la scheda Connessioni:**
   - Avvia Evermusic o Flacbox.
   - Vai alla scheda "**Connessioni**" per vedere tutti i servizi cloud connessi.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **Apri il servizio cloud:**
   - Tocca l'icona del servizio cloud a cui vuoi accedere.
   - Naviga nella cartella contenente i file audio che vuoi scaricare.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **Scarica la musica:**
   - Tocca i tre puntini accanto al nome della cartella per aprire il menu delle azioni aggiuntive.
   - Seleziona l'opzione **Scaricare** per iniziare a scaricare la cartella per la riproduzione offline.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - La cartella verrà scaricata nella directory **Documenti** dell'app.
   - Per controllare l'avanzamento del download, vai alla schermata **Trasferimenti file** toccando l'icona delle frecce rotanti nell'angolo superiore sinistro della schermata dei file locali.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - Puoi trovare tutti i file scaricati nella cartella **Documenti** - **Scaricare** o nella tua cartella di download personalizzata se configurata nelle impostazioni dell'app.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## Abilitare la modalità offline per la sincronizzazione automatica

1. **Attiva la modalità offline:**
   - Invece di limitarsi a scaricare, abilita la modalità offline per aggiornare automaticamente i file locali quando si verificano modifiche nel servizio cloud (ad esempio, aggiornamenti di file o nuovi file aggiunti).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - La modalità offline esegue una sincronizzazione unidirezionale, riflettendo solo le modifiche del servizio cloud nella tua copia locale.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **Sincronizzazione manuale:**
   - Sincronizza manualmente la tua cartella offline selezionando l'azione **Sincronizzare** dal menu delle azioni aggiuntive, accessibile tramite il pulsante con i tre puntini.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **Modalità offline per playlist/album/artisti/generi:**
   - Per abilitare la modalità offline per le playlist, apri la playlist e seleziona la casella accanto all'azione della modalità offline.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - L'app scansionerà tutti i brani nella playlist e li scaricherà nella cartella File locali - Cartelle offline - Playlist - [Nome Playlist].

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **Impostazioni per i download:**
   - Nelle impostazioni dell'app, puoi regolare il numero massimo di attività parallele per velocizzare i download, abilitare i trasferimenti in background e cambiare la posizione della cartella di download.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - Gestisci le cartelle offline tramite **Impostazioni** - Gestore file - **Cartelle offline sincronizzate**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## Gestire le cartelle offline sincronizzate

1. **Gestione delle cartelle:**
   - Nella sezione **Cartelle offline sincronizzate**, puoi avviare e interrompere la sincronizzazione, mostrare una cartella nella sua directory contenitore o disabilitare la modalità offline (che rimuove le copie locali dei file dal tuo dispositivo).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **Impostare l'intervallo di sincronizzazione:**
   - Configura l'intervallo di sincronizzazione per le cartelle offline (predefinito 1 giorno) per controllare automaticamente gli aggiornamenti e sincronizzare le copie locali all'apertura dell'app.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **Integrazione della libreria musicale:**
   - Tutti i file scaricati vengono aggiunti alla libreria musicale. Gestisci le impostazioni della libreria musicale toccando il menu delle azioni aggiuntive nell'angolo superiore destro della sezione della libreria musicale e selezionando **Impostazioni**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## Utilizzare la cache del lettore audio

L'ultima opzione per scaricare file cloud è utilizzare la cache del lettore audio. Ecco come funziona: abilita la cache del lettore audio nelle impostazioni dell'app (Lettore audio > Cache del lettore audio).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- Imposta la dimensione della cache in modo che i file vecchi vengano rimossi automaticamente e i nuovi file vengano scaricati nella cartella File locali > Lettore audio. Quando inizi a riprodurre musica dal cloud, l'app scaricherà automaticamente in anticipo il brano successivo nella coda, garantendo transizioni fluide tra i brani.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **Abilitare la cache del lettore audio:**
   - Vai su **Impostazioni** - Lettore audio - Cache del lettore audio per abilitare la cache.
   - Imposta la dimensione della cache in modo che i file vecchi vengano rimossi automaticamente e i nuovi file vengano scaricati nella cartella File locali - Lettore audio.
   - Durante la riproduzione di musica dal cloud, l'app scaricherà in anticipo il brano successivo per garantire transizioni fluide.

2. **Verificare la posizione della cache:**
   - Verifica la posizione della cache del lettore audio selezionando **Mostra nella cartella** nella schermata delle impostazioni.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## Evitare file duplicati

L'app offre diverse opzioni per scaricare file: Modalità offline, Download diretto e Cache del lettore audio. Per evitare duplicati, usa solo un metodo. Inoltre, se un brano si trova in più collezioni (come album o playlist) e abiliti la Modalità offline per entrambe, il brano verrà scaricato due volte, poiché la Modalità offline funziona in modo indipendente per ogni collezione.

1. **Gestire copie multiple:**
   - Evita di avere più copie dello stesso file utilizzando l'opzione **Mostra nella cartella** per localizzare i file e gestire i duplicati.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## Risoluzione dei problemi e suggerimenti

1. **Copie multiple di file:**
   - Tieni presente che l'abilitazione di più modalità di download (ad esempio, cache del lettore audio, download di cartelle, modalità offline per cartelle cloud, playlist, album, artisti) può risultare in file duplicati. Usa **Mostra nella cartella** per localizzare e gestire questi file.

2. **Trasferimenti in background:**
   - Assicurati che i trasferimenti in background siano abilitati nelle impostazioni per mantenere i download in esecuzione anche quando l'app non è attiva.

3. **Intervalli di sincronizzazione:**
   - Regola gli intervalli di sincronizzazione nelle impostazioni per assicurarti che la tua libreria musicale sia sempre aggiornata con le ultime modifiche dal tuo servizio cloud.

Seguendo questi passaggi dettagliati, puoi gestire e riprodurre senza problemi la tua musica locale e archiviata nel cloud offline utilizzando Evermusic e Flacbox. Goditi l'accesso ininterrotto alla tua libreria musicale, organizzata e pronta per la riproduzione quando vuoi.

## Domande frequenti

{{% details title="Quali servizi cloud supportano Evermusic e Flacbox?" closed="true" %}}
Entrambe le app supportano Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk e altri importanti fornitori di archiviazione cloud. Puoi connettere più servizi contemporaneamente.
{{% /details %}}

{{% details title="Posso sincronizzare automaticamente la musica dall'archiviazione cloud al mio iPhone?" closed="true" %}}
Sì. Abilita la Modalità offline per qualsiasi cartella, playlist, album o artista. L'app esegue una sincronizzazione unidirezionale dal cloud al dispositivo a un intervallo configurabile (predefinito: una volta al giorno).
{{% /details %}}

{{% details title="La modalità offline utilizza molto spazio di archiviazione sul mio dispositivo?" closed="true" %}}
L'utilizzo dello spazio di archiviazione dipende dalla dimensione della tua collezione musicale e dai formati dei file. Puoi controllarlo scegliendo cartelle specifiche da sincronizzare, impostando limiti di dimensione della cache e monitorando lo spazio di archiviazione nelle impostazioni dell'app.
{{% /details %}}

{{% details title="Quali formati audio sono supportati per la riproduzione offline?" closed="true" %}}
Evermusic e Flacbox supportano MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, WMA e molti altri formati. Flacbox è ottimizzato per formati lossless come FLAC e ALAC.
{{% /details %}}

{{% details title="La mia musica offline continuerà a riprodursi se chiudo l'app?" closed="true" %}}
Sì. I file scaricati sono archiviati localmente sul tuo dispositivo e vengono riprodotti tramite il lettore audio dell'app indipendentemente dalla connettività internet. La riproduzione in background è completamente supportata.
{{% /details %}}

{{% details title="Come posso liberare spazio occupato dalla musica offline?" closed="true" %}}
Disabilita la Modalità offline per cartelle specifiche in **Impostazioni** > Gestore file > **Cartelle offline sincronizzate**. Questo rimuove le copie locali dal tuo dispositivo. Puoi anche svuotare la cache del lettore audio o eliminare manualmente i file scaricati.
{{% /details %}}

**Résumé :** Evermusic et Flacbox vous permettent de télécharger de la musique depuis le stockage cloud (Google Drive, Dropbox, OneDrive et plus) vers votre iPhone ou iPad pour une lecture hors ligne. Vous pouvez utiliser trois méthodes : téléchargement direct, mode hors ligne avec synchronisation automatique ou cache du lecteur audio. Ce guide couvre les trois approches étape par étape.

Dans un monde où le streaming musical domine, il y a encore une grande valeur à avoir vos morceaux préférés disponibles hors ligne. Evermusic et Flacbox sont des applications puissantes qui vous permettent de télécharger et de synchroniser la musique de votre stockage cloud directement vers vos fichiers locaux, vous garantissant un accès à vos morceaux à tout moment et en tout lieu. Dans ce guide, nous vous guiderons à travers les étapes pour écouter facilement de la musique hors ligne avec Evermusic et Flacbox, rendant votre bibliothèque musicale accessible même sans connexion Internet.

Nos applications musicales offrent deux méthodes principales pour rendre la musique disponible hors ligne : le téléchargement simple et l'activation du mode hors ligne pour la synchronisation automatique.

## Télécharger de la musique pour la lecture hors ligne

1. **Ouvrez l'onglet Connexions :**
   - Lancez Evermusic ou Flacbox.
   - Accédez à l'onglet « **Connexions** » pour voir tous les services cloud connectés.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **Ouvrez le service cloud :**
   - Appuyez sur l'icône du service cloud auquel vous souhaitez accéder.
   - Parcourez le dossier contenant les fichiers audio que vous souhaitez télécharger.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **Télécharger la musique :**
   - Appuyez sur les trois points à côté du nom du dossier pour ouvrir le menu d'actions supplémentaires.
   - Sélectionnez l'option **Télécharger** pour commencer le téléchargement du dossier pour la lecture hors ligne.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Le dossier sera téléchargé dans le répertoire **Documents** de l'application.
   - Pour vérifier la progression du téléchargement, accédez à l'écran **Transferts de fichiers** en appuyant sur l'icône des flèches rotatives dans le coin supérieur gauche de l'écran des fichiers locaux.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - Vous pouvez trouver tous les fichiers téléchargés dans le dossier **Documents** - **Téléchargements** ou dans votre dossier de téléchargement personnalisé s'il est configuré dans les paramètres de l'application.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## Activer le mode hors ligne pour la synchronisation automatique

1. **Activer le mode hors ligne :**
   - Au lieu de simplement télécharger, activez le mode hors ligne pour mettre à jour automatiquement les fichiers locaux lorsque des modifications surviennent sur le service cloud (par exemple, mises à jour de fichiers ou ajout de nouveaux fichiers).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Le mode hors ligne effectue une synchronisation unidirectionnelle, reflétant uniquement les modifications du service cloud dans votre copie locale.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **Synchronisation manuelle :**
   - Synchronisez manuellement votre dossier hors ligne en sélectionnant l'action **Synchroniser** dans le menu d'actions supplémentaires, accessible via le bouton à trois points.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **Mode hors ligne pour les listes de lecture/albums/artistes/genres :**
   - Pour activer le mode hors ligne pour les listes de lecture, ouvrez la liste de lecture et cochez la case à côté de l'action du mode hors ligne.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - L'application analysera toutes les pistes de la liste de lecture et les téléchargera dans le dossier Fichiers locaux - Dossiers hors ligne - Listes de lecture - [Nom de la liste de lecture].

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **Paramètres pour les téléchargements :**
   - Dans les paramètres de l'application, vous pouvez ajuster le nombre maximum de tâches parallèles pour accélérer les téléchargements, activer les transferts en arrière-plan et modifier l'emplacement du dossier de téléchargement.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - Gérez les dossiers hors ligne via **Paramètres** - Gestionnaire de fichiers - **Dossiers hors ligne synchronisés**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## Gérer les dossiers hors ligne synchronisés

1. **Gestion des dossiers :**
   - Dans la section **Dossiers hors ligne synchronisés**, vous pouvez démarrer et arrêter la synchronisation, afficher un dossier dans son répertoire parent ou désactiver le mode hors ligne (ce qui supprime les copies locales des fichiers de votre appareil).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **Définir l'intervalle de synchronisation :**
   - Configurez l'intervalle de synchronisation pour les dossiers hors ligne (par défaut 1 jour) pour vérifier automatiquement les mises à jour et synchroniser les copies locales à l'ouverture de l'application.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **Intégration de la bibliothèque musicale :**
   - Tous les fichiers téléchargés sont ajoutés à la bibliothèque musicale. Gérez les paramètres de la bibliothèque musicale en appuyant sur le menu d'actions supplémentaires dans le coin supérieur droit de la section de la bibliothèque musicale et en sélectionnant **Paramètres**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## Utiliser le cache du lecteur audio

La dernière option pour télécharger des fichiers cloud est d'utiliser le cache du lecteur audio. Voici comment cela fonctionne : activez le cache du lecteur audio dans les paramètres de l'application (Lecteur audio > Cache du lecteur audio).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- Définissez la taille du cache pour que les anciens fichiers soient automatiquement supprimés et que les nouveaux fichiers soient téléchargés dans le dossier Fichiers locaux > Lecteur audio. Lorsque vous commencez à écouter de la musique depuis le cloud, l'application pré-téléchargera automatiquement la piste suivante dans la file d'attente, assurant des transitions fluides entre les pistes.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **Activer le cache du lecteur audio :**
   - Accédez à **Paramètres** - Lecteur audio - Cache du lecteur audio pour activer le cache.
   - Définissez la taille du cache pour que les anciens fichiers soient automatiquement supprimés et que les nouveaux fichiers soient téléchargés dans le dossier Fichiers locaux - Lecteur audio.
   - Lors de la lecture de musique depuis le cloud, l'application pré-téléchargera la piste suivante pour assurer des transitions fluides.

2. **Vérifier l'emplacement du cache :**
   - Vérifiez l'emplacement du cache du lecteur audio en sélectionnant **Afficher dans le dossier** sur l'écran des paramètres.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## Éviter les fichiers en double

L'application offre plusieurs options pour télécharger des fichiers : Mode hors ligne, Téléchargement direct et Cache du lecteur audio. Pour éviter les doublons, n'utilisez qu'une seule méthode. De plus, si une piste se trouve dans plusieurs collections (comme des albums ou des listes de lecture) et que vous activez le Mode hors ligne pour les deux, la piste sera téléchargée deux fois, car le Mode hors ligne fonctionne indépendamment pour chaque collection.

1. **Gérer les copies multiples :**
   - Évitez d'avoir plusieurs copies du même fichier en utilisant l'option **Afficher dans le dossier** pour localiser les fichiers et gérer les doublons.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## Dépannage et conseils

1. **Copies multiples de fichiers :**
   - Sachez que l'activation de plusieurs modes de téléchargement (par exemple, cache du lecteur audio, téléchargement de dossiers, mode hors ligne pour les dossiers cloud, listes de lecture, albums, artistes) peut entraîner des fichiers en double. Utilisez **Afficher dans le dossier** pour localiser et gérer ces fichiers.

2. **Transferts en arrière-plan :**
   - Assurez-vous que les transferts en arrière-plan sont activés dans les paramètres pour que les téléchargements continuent même lorsque l'application n'est pas active.

3. **Intervalles de synchronisation :**
   - Ajustez les intervalles de synchronisation dans les paramètres pour vous assurer que votre bibliothèque musicale est toujours à jour avec les dernières modifications de votre service cloud.

En suivant ces étapes détaillées, vous pouvez gérer et écouter sans problème votre musique locale et stockée dans le cloud hors ligne avec Evermusic et Flacbox. Profitez d'un accès ininterrompu à votre bibliothèque musicale, organisée et prête à être écoutée quand vous le souhaitez.

## Questions fréquemment posées

{{% details title="Quels services cloud Evermusic et Flacbox prennent-ils en charge ?" closed="true" %}}
Les deux applications prennent en charge Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk et d'autres grands fournisseurs de stockage cloud. Vous pouvez connecter plusieurs services à la fois.
{{% /details %}}

{{% details title="Puis-je synchroniser automatiquement la musique depuis le stockage cloud vers mon iPhone ?" closed="true" %}}
Oui. Activez le Mode hors ligne pour n'importe quel dossier, liste de lecture, album ou artiste. L'application effectue une synchronisation unidirectionnelle du cloud vers l'appareil à un intervalle configurable (par défaut : une fois par jour).
{{% /details %}}

{{% details title="Le mode hors ligne utilise-t-il beaucoup d'espace de stockage sur mon appareil ?" closed="true" %}}
L'utilisation du stockage dépend de la taille de votre collection musicale et des formats de fichiers. Vous pouvez contrôler cela en choisissant des dossiers spécifiques à synchroniser, en définissant des limites de taille de cache et en surveillant le stockage dans les paramètres de l'application.
{{% /details %}}

{{% details title="Quels formats audio sont pris en charge pour la lecture hors ligne ?" closed="true" %}}
Evermusic et Flacbox prennent en charge MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, WMA et de nombreux autres formats. Flacbox est optimisé pour les formats sans perte comme FLAC et ALAC.
{{% /details %}}

{{% details title="Ma musique hors ligne continuera-t-elle à jouer si je ferme l'application ?" closed="true" %}}
Oui. Les fichiers téléchargés sont stockés localement sur votre appareil et sont lus via le lecteur audio de l'application indépendamment de la connectivité Internet. La lecture en arrière-plan est entièrement prise en charge.
{{% /details %}}

{{% details title="Comment libérer de l'espace occupé par la musique hors ligne ?" closed="true" %}}
Désactivez le Mode hors ligne pour des dossiers spécifiques dans **Paramètres** > Gestionnaire de fichiers > **Dossiers hors ligne synchronisés**. Cela supprime les copies locales de votre appareil. Vous pouvez également vider le cache du lecteur audio ou supprimer manuellement les fichiers téléchargés.
{{% /details %}}

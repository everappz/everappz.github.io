**要約:** EvermusicとFlacboxを使用すると、クラウドストレージ（Google Drive、Dropbox、OneDriveなど）からiPhoneやiPadに音楽をダウンロードしてオフライン再生できます。直接ダウンロード、自動同期付きオフラインモード、オーディオプレーヤーキャッシュの3つの方法を使用できます。このガイドでは、3つのアプローチすべてをステップバイステップで説明します。

音楽ストリーミングが主流の世界でも、お気に入りの曲をオフラインで利用できることには大きな価値があります。EvermusicとFlacboxは、クラウドストレージから直接ローカルファイルに音楽をダウンロードして同期できる強力なアプリで、いつでもどこでも音楽にアクセスできます。このガイドでは、EvermusicとFlacboxを使用してオフライン音楽を簡単に再生する手順を説明し、インターネット接続なしでも音楽ライブラリにアクセスできるようにします。

私たちの音楽アプリは、音楽をオフラインで利用可能にする2つの主要な方法を提供しています：シンプルなダウンロードと、自動同期のためのオフラインモードの有効化です。

## オフライン再生のための音楽ダウンロード

1. **接続タブを開く：**
   - EvermusicまたはFlacboxを起動します。
   - 「**接続**」タブに移動して、接続されているすべてのクラウドサービスを確認します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **クラウドサービスを開く：**
   - アクセスしたいクラウドサービスのアイコンをタップします。
   - ダウンロードしたいオーディオファイルが含まれるフォルダーに移動します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **音楽をダウンロード：**
   - フォルダー名の横にある3つのドットをタップして、その他のアクションメニューを開きます。
   - **ダウンロード**オプションを選択して、オフライン再生用にフォルダーのダウンロードを開始します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - フォルダーはアプリの**ドキュメント**ディレクトリにダウンロードされます。
   - ダウンロードの進行状況を確認するには、ローカルファイル画面の左上隅にある回転する矢印アイコンをタップして**ファイル転送**画面に移動します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - ダウンロードしたすべてのファイルは、**ドキュメント** - **ダウンロード**フォルダー、またはアプリの設定で構成されている場合はカスタムダウンロードフォルダーにあります。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## 自動同期のためのオフラインモードの有効化

1. **オフラインモードを有効にする：**
   - 単にダウンロードするのではなく、クラウドサービスで変更が発生した場合（ファイルの更新や新しいファイルの追加など）にローカルファイルを自動的に更新するオフラインモードを有効にします。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - オフラインモードは一方向の同期を実行し、クラウドサービスの変更のみをローカルコピーに反映します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **手動同期：**
   - 3つのドットボタンからアクセスできるその他のアクションメニューから**同期する**アクションを選択して、オフラインフォルダーを手動で同期します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **プレイリスト/アルバム/アーティスト/ジャンルのオフラインモード：**
   - プレイリストのオフラインモードを有効にするには、プレイリストを開き、オフラインモードアクションの横にあるチェックボックスをオンにします。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - アプリはプレイリスト内のすべてのトラックをスキャンし、ローカルファイル - オフラインフォルダー - プレイリスト - [プレイリスト名]フォルダーにダウンロードします。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **ダウンロードの設定：**
   - アプリの設定で、ダウンロードを高速化するための並列タスクの最大数の調整、バックグラウンド転送の有効化、ダウンロードフォルダーの場所の変更ができます。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - **設定** - ファイルマネージャー - **同期されたオフラインフォルダー**からオフラインフォルダーを管理します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## 同期されたオフラインフォルダーの管理

1. **フォルダー管理：**
   - **同期されたオフラインフォルダー**セクションでは、同期の開始と停止、フォルダーの親ディレクトリへの表示、またはオフラインモードの無効化（デバイスからファイルのローカルコピーを削除）ができます。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **同期間隔の設定：**
   - オフラインフォルダーの同期間隔を設定（デフォルトは1日）して、アプリを開いたときに自動的に更新を確認しローカルコピーを同期するようにします。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **音楽ライブラリの統合：**
   - ダウンロードされたすべてのファイルは音楽ライブラリに追加されます。音楽ライブラリセクションの右上隅にあるその他のアクションメニューをタップし、**設定**を選択して音楽ライブラリの設定を管理します。

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## オーディオプレーヤーキャッシュの使用

クラウドファイルをダウンロードする最後のオプションは、オーディオプレーヤーキャッシュを使用することです。仕組みは次のとおりです：アプリの設定でオーディオプレーヤーキャッシュを有効にします（オーディオプレーヤー > オーディオプレーヤーキャッシュ）。

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- キャッシュサイズを設定して、古いファイルが自動的に削除され、新しいファイルがローカルファイル > オーディオプレーヤーフォルダーにダウンロードされるようにします。クラウドから音楽の再生を開始すると、アプリは自動的にキュー内の次のトラックを事前にダウンロードし、トラック間のスムーズな移行を保証します。

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **オーディオプレーヤーキャッシュを有効にする：**
   - **設定** - オーディオプレーヤー - オーディオプレーヤーキャッシュに移動してキャッシュを有効にします。
   - キャッシュサイズを設定して、古いファイルが自動的に削除され、新しいファイルがローカルファイル - オーディオプレーヤーフォルダーにダウンロードされるようにします。
   - クラウドから音楽を再生する際、アプリはスムーズな移行を確保するために次のトラックを事前にダウンロードします。

2. **キャッシュの場所を確認：**
   - 設定画面で**フォルダーで表示**を選択して、オーディオプレーヤーキャッシュの場所を確認します。

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## 重複ファイルの回避

アプリはファイルをダウンロードするためのいくつかのオプションを提供しています：オフラインモード、直接ダウンロード、オーディオプレーヤーキャッシュ。重複を避けるために、1つの方法のみを使用してください。また、トラックが複数のコレクション（アルバムやプレイリストなど）に含まれており、両方でオフラインモードを有効にすると、オフラインモードは各コレクションに対して独立して機能するため、トラックは2回ダウンロードされます。

1. **複数のコピーの管理：**
   - **フォルダーで表示**オプションを使用してファイルを見つけ、重複を管理することで、同じファイルの複数のコピーを持つことを避けてください。

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## トラブルシューティングとヒント

1. **ファイルの複数のコピー：**
   - 複数のダウンロードモード（オーディオプレーヤーキャッシュ、フォルダーダウンロード、クラウドフォルダー、プレイリスト、アルバム、アーティストのオフラインモードなど）を有効にすると、重複ファイルが発生する可能性があることに注意してください。**フォルダーで表示**を使用して、これらのファイルを見つけて管理してください。

2. **バックグラウンド転送：**
   - アプリがアクティブでないときでもダウンロードが継続するように、設定でバックグラウンド転送が有効になっていることを確認してください。

3. **同期間隔：**
   - 音楽ライブラリがクラウドサービスからの最新の変更で常に最新の状態になるように、設定で同期間隔を調整してください。

これらの詳細な手順に従うことで、EvermusicとFlacboxを使用してローカルおよびクラウドに保存された音楽をオフラインでシームレスに管理および再生できます。整理されていつでも再生準備ができている音楽ライブラリへの途切れないアクセスをお楽しみください。

## よくある質問

{{% details title="EvermusicとFlacboxはどのクラウドサービスに対応していますか？" closed="true" %}}
両方のアプリはGoogle Drive、Dropbox、OneDrive、Box、MEGA、Yandex.Disk、その他の主要なクラウドストレージプロバイダーに対応しています。複数のサービスを同時に接続できます。
{{% /details %}}

{{% details title="クラウドストレージからiPhoneに自動的に音楽を同期できますか？" closed="true" %}}
はい。任意のフォルダー、プレイリスト、アルバム、またはアーティストのオフラインモードを有効にします。アプリは設定可能な間隔（デフォルト：1日1回）でクラウドからデバイスへの一方向同期を実行します。
{{% /details %}}

{{% details title="オフラインモードはデバイスのストレージを大量に使用しますか？" closed="true" %}}
ストレージの使用量は、音楽コレクションのサイズとファイル形式によって異なります。同期する特定のフォルダーを選択し、キャッシュサイズの制限を設定し、アプリの設定でストレージを監視することで制御できます。
{{% /details %}}

{{% details title="オフライン再生にはどのオーディオ形式がサポートされていますか？" closed="true" %}}
EvermusicとFlacboxはMP3、FLAC、AAC、ALAC、WAV、AIFF、OGG、WMA、その他多くの形式をサポートしています。FlacboxはFLACやALACなどのロスレス形式に最適化されています。
{{% /details %}}

{{% details title="アプリを閉じてもオフライン音楽は再生され続けますか？" closed="true" %}}
はい。ダウンロードされたファイルはデバイスにローカルに保存され、インターネット接続に関係なくアプリのオーディオプレーヤーで再生されます。バックグラウンド再生は完全にサポートされています。
{{% /details %}}

{{% details title="オフライン音楽が占めるスペースを解放するにはどうすればよいですか？" closed="true" %}}
**設定** > ファイルマネージャー > **同期されたオフラインフォルダー**で特定のフォルダーのオフラインモードを無効にします。これにより、デバイスからローカルコピーが削除されます。オーディオプレーヤーキャッシュをクリアしたり、ダウンロードしたファイルを手動で削除することもできます。
{{% /details %}}

**Ringkasan:** Evermusic dan Flacbox memungkinkan Anda mengunduh musik dari penyimpanan cloud (Google Drive, Dropbox, OneDrive, dan lainnya) ke iPhone atau iPad Anda untuk pemutaran offline. Anda dapat menggunakan tiga metode: unduh langsung, mode offline dengan sinkronisasi otomatis, atau cache pemutar audio. Panduan ini mencakup ketiga pendekatan langkah demi langkah.

Di dunia di mana streaming musik mendominasi, masih ada nilai besar dalam memiliki lagu favorit Anda tersedia secara offline. Evermusic dan Flacbox adalah aplikasi yang kuat yang memungkinkan Anda mengunduh dan menyinkronkan musik dari penyimpanan cloud Anda langsung ke file lokal Anda, memastikan Anda memiliki akses ke musik Anda kapan saja dan di mana saja. Dalam panduan ini, kami akan memandu Anda melalui langkah-langkah untuk memutar musik offline dengan mudah menggunakan Evermusic dan Flacbox, membuat perpustakaan musik Anda dapat diakses bahkan tanpa koneksi internet.

Aplikasi musik kami menawarkan dua metode utama untuk membuat musik tersedia secara offline: pengunduhan sederhana dan mengaktifkan mode offline untuk sinkronisasi otomatis.

## Mengunduh Musik untuk Pemutaran Offline

1. **Buka tab Koneksi:**
   - Luncurkan Evermusic atau Flacbox.
   - Navigasi ke tab "**Koneksi**" untuk melihat semua layanan cloud yang terhubung.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_58ca45ebd97542e7ba481981ebee453e~mv2.png)

2. **Buka layanan cloud:**
   - Ketuk ikon layanan cloud yang ingin Anda akses.
   - Telusuri folder yang berisi file audio yang ingin Anda unduh.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7d043f62fe25429ba3382edd59eb96a0~mv2.png)

3. **Unduh musik:**
   - Ketuk tiga titik di samping nama folder untuk membuka menu tindakan lebih lanjut.
   - Pilih opsi **Unduh** untuk mulai mengunduh folder untuk pemutaran offline.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Folder akan diunduh ke direktori **Dokumen** aplikasi.
   - Untuk memeriksa kemajuan unduhan, buka layar **Transfer File** dengan mengetuk ikon panah berputar di pojok kiri atas layar file lokal.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_7296a80d22884cecb27092637c683999~mv2.png)

   - Anda dapat menemukan semua file yang diunduh di folder **Dokumen** - **Unduh** atau folder unduhan kustom Anda jika dikonfigurasi di pengaturan aplikasi.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_38c1e7c5d6f44efa955f72c60c62b8b2~mv2.png)

## Mengaktifkan Mode Offline untuk Sinkronisasi Otomatis

1. **Aktifkan mode offline:**
   - Alih-alih hanya mengunduh, aktifkan mode offline untuk memperbarui file lokal secara otomatis ketika terjadi perubahan di layanan cloud (misalnya, pembaruan file atau file baru ditambahkan).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_df16a0fb0bdf40e2b4f40bd78bdaaaee~mv2.png)

   - Mode offline melakukan sinkronisasi satu arah, hanya mencerminkan perubahan layanan cloud di salinan lokal Anda.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_5f7738dc14d94e27b7cbe840cfe23192~mv2.png)

2. **Sinkronisasi manual:**
   - Sinkronkan folder offline Anda secara manual dengan memilih tindakan **Sinkronkan** dari menu tindakan lebih lanjut, yang dapat diakses melalui tombol tiga titik.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_86ad3d2da0184c45b9b669f2e1091d6e~mv2.png)

3. **Mode offline untuk daftar putar/album/artis/genre:**
   - Untuk mengaktifkan mode offline untuk daftar putar, buka daftar putar dan centang kotak di dekat tindakan mode offline.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_820bbb34b486464e87c90147f7d480ae~mv2.png)

   - Aplikasi akan memindai semua trek di daftar putar dan mengunduhnya ke folder File Lokal - Folder offline - Daftar Putar - [Nama Daftar Putar].

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3ab4d66930db49bb8dce0c0031d3bcf2~mv2.png)

4. **Pengaturan untuk unduhan:**
   - Di pengaturan aplikasi, Anda dapat menyesuaikan jumlah maksimum tugas paralel untuk mempercepat unduhan, mengaktifkan transfer latar belakang, dan mengubah lokasi folder unduhan.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_3bc96980d2bc4b788fb3f8175793916a~mv2.png)

   - Kelola folder offline melalui **Pengaturan** - Pengelola File - **Folder offline yang disinkronkan**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_832a86aa14604bebb8dba67691ee2e5c~mv2.png)

## Mengelola Folder Offline yang Disinkronkan

1. **Pengelolaan folder:**
   - Di bagian **Folder offline yang disinkronkan**, Anda dapat memulai dan menghentikan sinkronisasi, menampilkan folder di direktori induknya, atau menonaktifkan mode offline (yang menghapus salinan lokal file dari perangkat Anda).

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0d9a712bc5ae448aa7f307926e8efdf5~mv2.png)

2. **Atur interval sinkronisasi:**
   - Konfigurasikan interval sinkronisasi untuk folder offline (default adalah 1 hari) untuk secara otomatis memeriksa pembaruan dan menyinkronkan salinan lokal saat aplikasi dibuka.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_a8249546a7d34fadabbda0b4252beb56~mv2.png)

3. **Integrasi perpustakaan musik:**
   - Semua file yang diunduh ditambahkan ke perpustakaan musik. Kelola pengaturan perpustakaan musik dengan mengetuk menu tindakan lebih lanjut di pojok kanan atas bagian perpustakaan musik dan memilih **Pengaturan**.

   ![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_8b0071adba8a4219bebb8b19967e584e~mv2.png)

## Menggunakan Cache Pemutar Audio

Opsi terakhir untuk mengunduh file cloud adalah menggunakan cache pemutar audio. Begini cara kerjanya: aktifkan cache pemutar audio di pengaturan aplikasi (Pemutar Audio > Cache Pemutar Audio).

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_0094e312fa294552b61571faea729b2d~mv2.png)

- Atur ukuran cache sehingga file lama dihapus secara otomatis, dan file baru diunduh ke folder File Lokal > Pemutar Audio. Saat Anda mulai memutar musik dari cloud, aplikasi akan secara otomatis mengunduh terlebih dahulu trek berikutnya dalam antrian, memastikan transisi yang mulus antar trek.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eae8d1b29c64474db88078e38182b617~mv2.png)

1. **Aktifkan cache pemutar audio:**
   - Buka **Pengaturan** - Pemutar Audio - Cache Pemutar Audio untuk mengaktifkan cache.
   - Atur ukuran cache sehingga file lama dihapus secara otomatis dan file baru diunduh ke folder File Lokal - Pemutar Audio.
   - Saat memutar musik dari cloud, aplikasi akan mengunduh terlebih dahulu trek berikutnya untuk memastikan transisi yang mulus.

2. **Periksa lokasi cache:**
   - Verifikasi lokasi cache pemutar audio dengan memilih **Tampilkan di Folder** di layar pengaturan.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_c92f2b1d0a404415b6b168eb32a6cc93~mv2.png)

## Menghindari File Duplikat

Aplikasi menawarkan beberapa opsi untuk mengunduh file: Mode Offline, Unduh Langsung, dan Cache Pemutar Audio. Untuk menghindari duplikat, gunakan hanya satu metode. Selain itu, jika trek ada di beberapa koleksi (seperti album atau daftar putar) dan Anda mengaktifkan Mode Offline untuk keduanya, trek akan diunduh dua kali, karena Mode Offline berfungsi secara independen untuk setiap koleksi.

1. **Kelola salinan ganda:**
   - Hindari memiliki beberapa salinan file yang sama dengan menggunakan opsi **Tampilkan di Folder** untuk menemukan file dan mengelola duplikat.

![Image](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/21260c_eca9e05a545e4cbe99ba76c0a998b880~mv2.png)

## Pemecahan Masalah dan Tips

1. **Salinan file ganda:**
   - Ketahui bahwa mengaktifkan beberapa mode unduhan (misalnya, cache pemutar audio, unduh folder, mode offline untuk folder cloud, daftar putar, album, artis) dapat menghasilkan file duplikat. Gunakan **Tampilkan di Folder** untuk menemukan dan mengelola file-file ini.

2. **Transfer latar belakang:**
   - Pastikan transfer latar belakang diaktifkan di pengaturan agar unduhan tetap berjalan bahkan saat aplikasi tidak aktif.

3. **Interval sinkronisasi:**
   - Sesuaikan interval sinkronisasi di pengaturan untuk memastikan perpustakaan musik Anda selalu diperbarui dengan perubahan terbaru dari layanan cloud Anda.

Dengan mengikuti langkah-langkah terperinci ini, Anda dapat mengelola dan memutar musik lokal dan yang disimpan di cloud secara offline dengan mulus menggunakan Evermusic dan Flacbox. Nikmati akses tanpa gangguan ke perpustakaan musik Anda, terorganisir dan siap diputar kapan pun Anda mau.

## Pertanyaan yang Sering Diajukan

{{% details title="Layanan cloud apa yang didukung Evermusic dan Flacbox?" closed="true" %}}
Kedua aplikasi mendukung Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk, dan penyedia penyimpanan cloud utama lainnya. Anda dapat menghubungkan beberapa layanan sekaligus.
{{% /details %}}

{{% details title="Bisakah saya menyinkronkan musik secara otomatis dari penyimpanan cloud ke iPhone saya?" closed="true" %}}
Ya. Aktifkan Mode Offline untuk folder, daftar putar, album, atau artis apa pun. Aplikasi melakukan sinkronisasi satu arah dari cloud ke perangkat pada interval yang dapat dikonfigurasi (default: sekali sehari).
{{% /details %}}

{{% details title="Apakah mode offline menggunakan banyak penyimpanan di perangkat saya?" closed="true" %}}
Penggunaan penyimpanan tergantung pada ukuran koleksi musik Anda dan format file. Anda dapat mengontrol ini dengan memilih folder tertentu untuk disinkronkan, menetapkan batas ukuran cache, dan memantau penyimpanan di pengaturan aplikasi.
{{% /details %}}

{{% details title="Format audio apa yang didukung untuk pemutaran offline?" closed="true" %}}
Evermusic dan Flacbox mendukung MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, WMA, dan banyak format lainnya. Flacbox dioptimalkan untuk format lossless seperti FLAC dan ALAC.
{{% /details %}}

{{% details title="Apakah musik offline saya akan terus diputar jika saya menutup aplikasi?" closed="true" %}}
Ya. File yang diunduh disimpan secara lokal di perangkat Anda dan diputar melalui pemutar audio aplikasi terlepas dari konektivitas internet. Pemutaran latar belakang didukung sepenuhnya.
{{% /details %}}

{{% details title="Bagaimana cara membebaskan ruang yang digunakan oleh musik offline?" closed="true" %}}
Nonaktifkan Mode Offline untuk folder tertentu di **Pengaturan** > Pengelola File > **Folder offline yang disinkronkan**. Ini menghapus salinan lokal dari perangkat Anda. Anda juga dapat menghapus cache pemutar audio atau menghapus file yang diunduh secara manual.
{{% /details %}}

**摘要：** Evermusic可作为iOS和macOS上功能齐全的有声书播放器。通过iTunes、WiFi或云存储传输有声书，然后使用播放速度控制、跳转按钮、音频书签、继续播放和离线下载功能，获得无缝的听书体验。

## 简介

在我们快节奏的世界中，有声书已成为书籍爱好者在旅途中享受文学的最爱。如果您使用Evermusic，那您很幸运，因为这款多功能音乐播放器还可以让您收听有声书。本指南将向您展示如何在Evermusic中收听有声书并为搜索引擎优化您的内容。

## 第1步：整理您的有声书

在开始使用Evermusic探索有声书世界之前，整理好有声书收藏至关重要。一个有序的资料库可以让您轻松找到并享受喜爱的故事。考虑按作者、类型或系列对有声书进行排序，以保持一切井然有序。

## 第2步：将有声书传输到Evermusic

要在Evermusic中享受有声书，您需要确保它们在应用程序中可以访问。以下是操作方法：

### 对于本地存储的有声书：

1. 将iPhone或iPad连接到电脑。
2. 启动iTunes（或macOS Catalina或更高版本上的Finder）。
3. 在iTunes或Finder中选择您的设备。
4. 导航到"Evermusic"部分。
5. 将有声书文件或文件夹拖放到"Evermusic"部分。

您可以在[这里](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)了解更多关于iTunes文件共享的信息。

您也可以通过WiFi将有声书传输到应用程序，如[这里](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)所述。

### 对于云端存储的有声书：

如果您的有声书存储在iCloud Drive、Google Drive或Dropbox等云服务中，请确保它们可以从Evermusic中访问。您可以在Evermusic应用程序中关联您的云存储账户，如[这里](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag)所述。

## 第3步：访问您的有声书

Evermusic启动并运行后，是时候开始收听有声书了：

1. 在Evermusic中点击"音乐库"或"本地文件"选项卡。
2. 导航到存储有声书的文件夹或位置。
3. 浏览有声书文件并选择您想开始收听的文件。

## 第4步：播放控制

Evermusic为有声书提供直观的播放控制：

**播放/暂停：** 点击播放按钮开始播放，需要时暂停。

**前进/后退：** 使用跳转按钮在有声书中前进或后退。

**跳转时间按钮：** 您还可以通过进入设置 > 音频播放器 > 个性化 > 主屏幕按钮 > 跳转时间来激活跳转时间按钮。在设置 > 音频播放器 > 个性化 > 跳转时间按钮 > 时间间隔中配置跳转时间间隔。要在锁屏上启用跳转时间按钮，请进入设置 > 音频播放器 > 个性化 > 锁屏按钮 > 跳转时间。

## 第5步：调整播放速度

Evermusic允许您控制播放速度，让您以喜欢的节奏收听。设置 > 音频播放器 > 播放速度

## 第6步：调整播放控制

Evermusic允许您在应用程序设置中自定义播放控制。设置 > 音频播放器 > 个性化 > 主屏幕按钮 和 设置 > 音频播放器 > 个性化 > 锁屏按钮

## 第7步：创建音频书签

Evermusic提供了一个便捷功能，允许您创建音频书签。这些书签不仅存储在应用程序的数据库中，还存储在文件的元数据中。这意味着如果您更换新设备，音频书签也会自动转移。

以下是在Evermusic中创建音频书签的方法：

1. 在Evermusic中打开音频播放器界面，选择要创建书签的歌曲。
2. 点击"更多操作"按钮（通常由三个点表示），然后选择"书签"菜单项。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_093fee004f344a08bcd96a8264312420~mv2.jpeg" alt="音频播放器选项界面" caption="从更多操作访问书签" width="400" >}}

3. 在下一个界面中，您会在导航栏中看到一个分段控件。此控件允许您在当前歌曲的音频书签和库中所有歌曲之间切换。要创建新的音频书签，点击"添加书签"。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_1d8f0c73656247449fee3a2c5ec6121d~mv2.png" alt="音频书签界面" caption="管理和查看音频书签" width="400" >}}

4. 在下一个界面中，您可以为书签指定标题并选择音频轨道中要放置书签的特定时间。填写完详细信息后，点击"完成"保存书签。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_33cc8dccee224a85963aa488e171a842~mv2.png" alt="书签编辑界面" caption="设置书签标题和位置" width="400" >}}

5. 如果您需要编辑或管理已创建的书签，请点击右上角的"编辑"按钮。从这里，您可以根据需要删除或重新排列书签。完成更改后，点击"完成"以应用。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_f4ffeac5be96410da521298d13db1a92~mv2.png" alt="移动书签界面" caption="编辑和重新排列书签" width="400" >}}

## 第8步：继续播放

Evermusic还提供便捷的"继续播放"功能，在应用程序启动之间保存您的收听进度。此功能让您只需轻轻一点即可轻松从上次中断的地方继续。使用此功能前，请确保已在应用程序设置中启用。

操作方法如下：

1. 打开Evermusic并导航到设置，进入"音频播放器" > "一般"。
2. 查找"保存播放位置"和"保存音频播放器状态"选项。

**保存播放位置：** 此选项保存单个文件的播放进度，并在主屏幕上显示此进度。

**保存音频播放器状态：** 此选项保存并恢复文件夹、播放列表、专辑、艺术家、流派和作曲家的播放器状态。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_bc458d615c9a4194a7086ee725d25104~mv2.png" alt="音频播放器设置界面" caption="继续播放设置" width="400" >}}

启用这两个功能后，在Evermusic中打开任意文件夹。您会在屏幕顶部看到"继续播放"按钮，以及上次保存的歌曲播放位置。

要恢复有声书播放进度，只需点击"继续播放"按钮。如果要恢复单个文件的播放，请点击该特定文件。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_7510b209723b4215a450cd20e34318a1~mv2.png" alt="音频播放器播放进度界面" caption="从上次位置继续播放" width="400" >}}

## 第9步：创建有声书播放列表

为了更加方便和有条理，考虑在Evermusic中为有声书创建播放列表。您可以按类型、作者或系列对有声书进行分组。方法如下：

1. 在Evermusic中点击"播放列表"选项卡。
2. 选择"创建播放列表"并命名（例如，"悬疑有声书"）。
3. 将所需的有声书文件添加到播放列表中。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_b0eb088bf4954e579314dd30e9ad442e~mv2.jpeg" alt="播放列表详情界面" caption="有声书播放列表便于整理" width="400" >}}

## 第10步：享受离线收听

Evermusic在支持离线收听方面表现出色，让您无需互联网连接即可享受有声书。要启用离线访问：

1. 点击要下载的有声书或播放列表旁边的下载图标。
2. 等待有声书下载到您的设备。

现在，即使在离线状态下，您也可以收听有声书。

## 常见问题

{{% details title="Evermusic支持哪些有声书格式？" closed="true" %}}
Evermusic支持MP3、M4A、M4B、FLAC、WAV、AIFF、OGG和其他常见音频格式。任何可以在Evermusic中播放的音频文件都可以作为有声书使用。
{{% /details %}}

{{% details title="我可以从云存储收听有声书吗？" closed="true" %}}
可以。Evermusic连接到iCloud Drive、Google Drive、Dropbox、OneDrive、Box和WebDAV服务器。您可以直接流式播放有声书或下载后离线收听。
{{% /details %}}

{{% details title="我的书签会转移到新设备吗？" closed="true" %}}
会的。Evermusic将音频书签保存在文件的元数据中，因此当您将文件移至新设备时，书签会自动转移。
{{% /details %}}

{{% details title="Evermusic会记住我停止收听的位置吗？" closed="true" %}}
会的。在设置 > 音频播放器 > 一般中启用"保存播放位置"和"保存音频播放器状态"。应用程序会在会话之间保存并恢复您的确切位置。
{{% /details %}}

{{% details title="我可以调整有声书播放速度吗？" closed="true" %}}
可以。进入设置 > 音频播放器 > 播放速度来设置您喜欢的速度。您可以加快或减慢朗读速度以匹配您的听书偏好。
{{% /details %}}

{{% details title="如何将有声书传输到Evermusic？" closed="true" %}}
您可以通过iTunes/Finder文件共享、WiFi Drive（应用内置）或在Evermusic中连接云存储账户来传输文件。
{{% /details %}}

**摘要：** Evermusic可作為iOS和macOS上功能齊全的有聲書播放器。透過iTunes、WiFi或雲端儲存傳輸有聲書，然後使用播放速度控制、跳轉按鈕、音訊書籤、繼續播放和離線下載功能，獲得無縫的聽書體驗。

## 簡介

在我們快節奏的世界中，有聲書已成為書籍愛好者在旅途中享受文學的最愛。如果您使用Evermusic，那您很幸運，因為這款多功能音樂播放器還可以讓您收聽有聲書。本指南將向您展示如何在Evermusic中收聽有聲書並為搜尋引擎最佳化您的內容。

## 第1步：整理您的有聲書

在開始使用Evermusic探索有聲書世界之前，整理好有聲書收藏至關重要。一個有序的資料庫可以讓您輕鬆找到並享受喜愛的故事。考慮按作者、類型或系列對有聲書進行排序，以保持一切井然有序。

## 第2步：將有聲書傳輸到Evermusic

要在Evermusic中享受有聲書，您需要確保它們在應用程式中可以存取。以下是操作方法：

### 對於本機儲存的有聲書：

1. 將iPhone或iPad連接到電腦。
2. 啟動iTunes（或macOS Catalina或更高版本上的Finder）。
3. 在iTunes或Finder中選擇您的裝置。
4. 導覽到「Evermusic」部分。
5. 將有聲書檔案或資料夾拖放到「Evermusic」部分。

您可以在[這裡](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)了解更多關於iTunes檔案共享的資訊。

您也可以透過WiFi將有聲書傳輸到應用程式，如[這裡](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)所述。

### 對於雲端儲存的有聲書：

如果您的有聲書儲存在iCloud Drive、Google Drive或Dropbox等雲端服務中，請確保它們可以從Evermusic中存取。您可以在Evermusic應用程式中關聯您的雲端儲存帳戶，如[這裡](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag)所述。

## 第3步：存取您的有聲書

Evermusic啟動並運行後，是時候開始收聽有聲書了：

1. 在Evermusic中點擊「音樂庫」或「本機檔案」標籤頁。
2. 導覽到儲存有聲書的資料夾或位置。
3. 瀏覽有聲書檔案並選擇您想開始收聽的檔案。

## 第4步：播放控制

Evermusic為有聲書提供直覺的播放控制：

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_89ad5edc3e8247fc811f988ac6e17051~mv2.png" alt="Evermusic播放器介面" caption="Evermusic播放器與有聲書控制" width="400" >}}

**播放/暫停：** 點擊播放按鈕開始播放，需要時暫停。

**前進/後退：** 使用跳轉按鈕在有聲書中前進或後退。

**跳轉時間按鈕：** 您還可以透過進入設定 > 音訊播放器 > 個人化 > 主畫面按鈕 > 跳轉時間來啟用跳轉時間按鈕。在設定 > 音訊播放器 > 個人化 > 跳轉時間按鈕 > 時間間隔中設定跳轉時間間隔。要在鎖定畫面上啟用跳轉時間按鈕，請進入設定 > 音訊播放器 > 個人化 > 鎖定畫面按鈕 > 跳轉時間。

## 第5步：調整播放速度

Evermusic允許您控制播放速度，讓您以喜歡的節奏收聽。設定 > 音訊播放器 > 播放速度

## 第6步：調整播放控制

Evermusic允許您在應用程式設定中自訂播放控制。設定 > 音訊播放器 > 個人化 > 主畫面按鈕 和 設定 > 音訊播放器 > 個人化 > 鎖定畫面按鈕

## 第7步：建立音訊書籤

Evermusic提供了一個便捷功能，允許您建立音訊書籤。這些書籤不僅儲存在應用程式的資料庫中，還儲存在檔案的中繼資料中。這意味著如果您更換新裝置，音訊書籤也會自動轉移。

以下是在Evermusic中建立音訊書籤的方法：

1. 在Evermusic中開啟音訊播放器介面，選擇要建立書籤的歌曲。
2. 點擊「更多操作」按鈕（通常由三個點表示），然後選擇「書籤」選單項目。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_093fee004f344a08bcd96a8264312420~mv2.jpeg" alt="音訊播放器選項介面" caption="從更多操作存取書籤" width="400" >}}

3. 在下一個介面中，您會在導覽列中看到一個分段控制項。此控制項允許您在目前歌曲的音訊書籤和資料庫中所有歌曲之間切換。要建立新的音訊書籤，點擊「新增書籤」。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_1d8f0c73656247449fee3a2c5ec6121d~mv2.png" alt="音訊書籤介面" caption="管理和檢視音訊書籤" width="400" >}}

4. 在下一個介面中，您可以為書籤指定標題並選擇音訊軌道中要放置書籤的特定時間。填寫完詳細資訊後，點擊「完成」儲存書籤。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_33cc8dccee224a85963aa488e171a842~mv2.png" alt="書籤編輯介面" caption="設定書籤標題和位置" width="400" >}}

5. 如果您需要編輯或管理已建立的書籤，請點擊右上角的「編輯」按鈕。從這裡，您可以根據需要刪除或重新排列書籤。完成變更後，點擊「完成」以套用。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_f4ffeac5be96410da521298d13db1a92~mv2.png" alt="移動書籤介面" caption="編輯和重新排列書籤" width="400" >}}

## 第8步：繼續播放

Evermusic還提供便捷的「繼續播放」功能，在應用程式啟動之間儲存您的收聽進度。此功能讓您只需輕輕一點即可輕鬆從上次中斷的地方繼續。使用此功能前，請確保已在應用程式設定中啟用。

操作方法如下：

1. 開啟Evermusic並導覽到設定，進入「音訊播放器」>「一般」。
2. 尋找「儲存播放位置」和「儲存音訊播放器狀態」選項。

**儲存播放位置：** 此選項儲存個別檔案的播放進度，並在主畫面上顯示此進度。

**儲存音訊播放器狀態：** 此選項儲存並還原資料夾、播放列表、專輯、藝人、類型和作曲家的播放器狀態。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_bc458d615c9a4194a7086ee725d25104~mv2.png" alt="音訊播放器設定介面" caption="繼續播放設定" width="400" >}}

啟用這兩個功能後，在Evermusic中開啟任意資料夾。您會在畫面頂部看到「繼續播放」按鈕，以及上次儲存的歌曲播放位置。

要還原有聲書播放進度，只需點擊「繼續播放」按鈕。如果要還原個別檔案的播放，請點擊該特定檔案。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_7510b209723b4215a450cd20e34318a1~mv2.png" alt="音訊播放器播放進度介面" caption="從上次位置繼續播放" width="400" >}}

## 第9步：建立有聲書播放列表

為了更加方便和有條理，考慮在Evermusic中為有聲書建立播放列表。您可以按類型、作者或系列對有聲書進行分組。方法如下：

1. 在Evermusic中點擊「播放列表」標籤頁。
2. 選擇「建立播放列表」並命名（例如，「懸疑有聲書」）。
3. 將所需的有聲書檔案新增到播放列表中。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_b0eb088bf4954e579314dd30e9ad442e~mv2.jpeg" alt="播放列表詳情介面" caption="有聲書播放列表便於整理" width="400" >}}

## 第10步：享受離線收聽

Evermusic在支援離線收聽方面表現出色，讓您無需網路連線即可享受有聲書。要啟用離線存取：

1. 點擊要下載的有聲書或播放列表旁邊的下載圖示。
2. 等待有聲書下載到您的裝置。

現在，即使在離線狀態下，您也可以收聽有聲書。

## 常見問題

{{% details title="Evermusic支援哪些有聲書格式？" closed="true" %}}
Evermusic支援MP3、M4A、M4B、FLAC、WAV、AIFF、OGG和其他常見音訊格式。任何可以在Evermusic中播放的音訊檔案都可以作為有聲書使用。
{{% /details %}}

{{% details title="我可以從雲端儲存收聽有聲書嗎？" closed="true" %}}
可以。Evermusic連接到iCloud Drive、Google Drive、Dropbox、OneDrive、Box和WebDAV伺服器。您可以直接串流播放有聲書或下載後離線收聽。
{{% /details %}}

{{% details title="我的書籤會轉移到新裝置嗎？" closed="true" %}}
會的。Evermusic將音訊書籤儲存在檔案的中繼資料中，因此當您將檔案移至新裝置時，書籤會自動轉移。
{{% /details %}}

{{% details title="Evermusic會記住我停止收聽的位置嗎？" closed="true" %}}
會的。在設定 > 音訊播放器 > 一般中啟用「儲存播放位置」和「儲存音訊播放器狀態」。應用程式會在工作階段之間儲存並還原您的確切位置。
{{% /details %}}

{{% details title="我可以調整有聲書播放速度嗎？" closed="true" %}}
可以。進入設定 > 音訊播放器 > 播放速度來設定您喜歡的速度。您可以加快或減慢朗讀速度以符合您的聽書偏好。
{{% /details %}}

{{% details title="如何將有聲書傳輸到Evermusic？" closed="true" %}}
您可以透過iTunes/Finder檔案共享、WiFi Drive（應用程式內建）或在Evermusic中連接雲端儲存帳戶來傳輸檔案。
{{% /details %}}

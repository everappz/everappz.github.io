**要約：** EvermusicはiOSとmacOSでフル機能のオーディオブックプレーヤーとしても機能します。iTunes、WiFi、またはクラウドストレージ経由でオーディオブックを転送し、再生速度コントロール、スキップタイムボタン、オーディオブックマーク、再生の続行、オフラインダウンロードを使用して、シームレスなリスニング体験をお楽しみください。

## はじめに

ペースの速い現代社会において、オーディオブックは移動中に文学を楽しみたい本好きの方々のお気に入りとなっています。Evermusicをお使いなら、この多機能な音楽プレーヤーでオーディオブックも聴くことができるので、とても幸運です。このガイドでは、Evermusicでオーディオブックを聴く方法と、検索エンジン向けにコンテンツを最適化する方法をご紹介します。

## ステップ1：オーディオブックを整理する

Evermusicでオーディオブックの世界に飛び込む前に、オーディオブックのコレクションを整理することが重要です。整理されたライブラリにより、お気に入りの物語を簡単に見つけて楽しむことができます。すべてをきちんと整理するために、著者、ジャンル、またはシリーズごとにオーディオブックをソートすることを検討してください。

## ステップ2：オーディオブックをEvermusicに転送する

Evermusicでオーディオブックを楽しむには、アプリ内でアクセスできるようにする必要があります。方法は以下の通りです：

### ローカルに保存されたオーディオブックの場合：

1. iPhoneまたはiPadをコンピュータに接続します。
2. iTunes（またはmacOS Catalina以降ではFinder）を起動します。
3. iTunesまたはFinder内でデバイスを選択します。
4. 「Evermusic」セクションに移動します。
5. オーディオブックのファイルまたはフォルダを「Evermusic」セクションにドラッグ＆ドロップします。

iTunesファイル共有について詳しくは[こちら](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)をご覧ください。

[こちら](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)で説明されているように、WiFiを使用してアプリにオーディオブックを転送することもできます。

### クラウドに保存されたオーディオブックの場合：

オーディオブックがiCloud Drive、Google Drive、Dropboxなどのクラウドサービスに保存されている場合は、Evermusic内からアクセスできることを確認してください。[こちら](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag)で説明されているように、Evermusicアプリ内でクラウドストレージアカウントをリンクできます。

## ステップ3：オーディオブックにアクセスする

Evermusicが起動して準備ができたら、オーディオブックの聴取を始めましょう：

1. Evermusic内の「ミュージックライブラリ」または「ローカルファイル」タブをタップします。
2. オーディオブックが保存されているフォルダまたは場所に移動します。
3. オーディオブックファイルを閲覧し、聴き始めたいものを選択します。

## ステップ4：再生コントロール

Evermusicはオーディオブック用の直感的な再生コントロールを提供します：

**再生/一時停止：** 再生ボタンをタップして再生を開始し、必要に応じて一時停止します。

**前へスキップ/後ろへスキップ：** スキップボタンを使用して、オーディオブック内で前後に移動します。

**スキップタイムボタン：** 設定 > オーディオプレーヤー > パーソナライズ > メイン画面のボタン > スキップタイムに移動して、スキップタイムボタンを有効にすることもできます。設定 > オーディオプレーヤー > パーソナライズ > スキップタイムボタン > 時間間隔でスキップタイムの間隔を設定します。ロック画面でスキップタイムボタンを有効にするには、設定 > オーディオプレーヤー > パーソナライズ > ロック画面のボタン > スキップタイムに移動します。

## ステップ5：再生速度を調整する

Evermusicでは再生速度をコントロールでき、お好みのペースで聴くことができます。設定 > オーディオプレーヤー > 再生速度

## ステップ6：再生コントロールを調整する

Evermusicでは、アプリケーション設定で再生コントロールをカスタマイズできます。設定 > オーディオプレーヤー > パーソナライズ > メイン画面のボタン および 設定 > オーディオプレーヤー > パーソナライズ > ロック画面のボタン

## ステップ7：オーディオブックマークを作成する

Evermusicには、オーディオブックマークを作成できる便利な機能があります。これらのブックマークはアプリケーションのデータベースだけでなく、ファイルのメタデータにも保存されます。つまり、新しいデバイスに切り替えても、オーディオブックマークもシームレスに転送されます。

Evermusicでオーディオブックマークを作成する方法は以下の通りです：

1. ブックマークを作成したい曲を選択して、Evermusic内のオーディオプレーヤー画面を開きます。
2. 通常3つのドットで表される「その他のアクション」ボタンをタップし、「ブックマーク」メニュー項目を選択します。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_093fee004f344a08bcd96a8264312420~mv2.jpeg" alt="オーディオプレーヤーオプション画面" caption="その他のアクションからブックマークにアクセス" width="400" >}}

3. 次の画面では、ナビゲーションバーにセグメントコントロールが表示されます。このコントロールにより、現在の曲とライブラリ内のすべての曲のオーディオブックマークを切り替えることができます。新しいオーディオブックマークを作成するには、「ブックマークを追加」をタップします。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_1d8f0c73656247449fee3a2c5ec6121d~mv2.png" alt="オーディオブックマーク画面" caption="オーディオブックマークの管理と表示" width="400" >}}

4. 次の画面で、ブックマークにタイトルを付け、ブックマークを配置したいオーディオトラック内の特定の時間を選択できます。詳細を入力したら、「完了」をタップしてブックマークを保存します。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_33cc8dccee224a85963aa488e171a842~mv2.png" alt="ブックマーク編集画面" caption="ブックマークのタイトルと位置を設定" width="400" >}}

5. 作成したブックマークを編集または管理する必要がある場合は、右上隅の「編集」ボタンをタップします。ここから、必要に応じてブックマークを削除または並べ替えることができます。変更を行ったら、「完了」をタップして適用します。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_f4ffeac5be96410da521298d13db1a92~mv2.png" alt="ブックマーク移動画面" caption="ブックマークの編集と並べ替え" width="400" >}}

## ステップ8：再生を続行する

Evermusicには便利な「再生を続行」機能もあり、アプリの起動間でリスニングの進行状況を保存します。この機能により、ワンタップで簡単に中断した箇所から再開できます。この機能を使用する前に、アプリケーション設定で有効になっていることを確認してください。

方法は以下の通りです：

1. Evermusicを開き、「オーディオプレーヤー」>「一般」に移動して設定に移動します。
2. 「再生位置を保存」と「オーディオプレーヤーの状態を保存」というオプションを探します。

**再生位置を保存：** このオプションは個々のファイルの再生進行状況を保存し、メイン画面にこの進行状況を表示します。

**オーディオプレーヤーの状態を保存：** このオプションはフォルダ、プレイリスト、アルバム、アーティスト、ジャンル、作曲家のプレーヤー状態を保存および復元します。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_bc458d615c9a4194a7086ee725d25104~mv2.png" alt="オーディオプレーヤー設定画面" caption="再生続行の設定" width="400" >}}

これらの機能を両方有効にしたら、Evermusic内の任意のフォルダを開きます。画面上部に「再生を続行」ボタンと、最後に保存された曲の再生位置が表示されます。

オーディオブックの再生進行状況を復元するには、「再生を続行」ボタンをタップするだけです。個々のファイルの再生を復元したい場合は、その特定のファイルをタップします。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_7510b209723b4215a450cd20e34318a1~mv2.png" alt="オーディオプレーヤー再生進行画面" caption="最後の位置から再生を続行" width="400" >}}

## ステップ9：オーディオブックのプレイリストを作成する

さらなる利便性と整理のために、Evermusic内でオーディオブック用のプレイリストの作成を検討してください。ジャンル、著者、またはシリーズごとにオーディオブックをグループ化できます。方法は以下の通りです：

1. Evermusic内の「プレイリスト」タブをタップします。
2. 「プレイリストを作成」を選択し、名前を付けます（例：「ミステリーオーディオブック」）。
3. 希望するオーディオブックファイルをプレイリストに追加します。

{{< figure src="/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic/21260c_b0eb088bf4954e579314dd30e9ad442e~mv2.jpeg" alt="プレイリスト詳細画面" caption="簡単整理のためのオーディオブックプレイリスト" width="400" >}}

## ステップ10：オフラインリスニングを楽しむ

Evermusicはオフラインリスニングのサポートに優れており、インターネット接続なしでオーディオブックを楽しむことができます。オフラインアクセスを有効にするには：

1. ダウンロードしたいオーディオブックまたはプレイリストの横にあるダウンロードアイコンをタップします。
2. オーディオブックがデバイスにダウンロードされるのを待ちます。

これで、オフラインの時でもオーディオブックを聴くことができます。

## まとめ

この包括的なガイドでは、Evermusic体験にオーディオブックをシームレスに統合する手順を専門的にご案内しました。オーディオブックコレクションの整理から再生コントロールの習得、便利なオーディオブックマークの作成まで、Evermusicはオーディオブックを簡単に楽しむことを可能にします。

Evermusicなら、オーディオブック体験が便利なだけでなく、お好みに合わせて高度にカスタマイズできることを信頼できます。印象的な箇所をブックマークしたり、再生コントロールをカスタマイズしたり、中断した箇所から簡単に再開したりと、Evermusicはあなたをサポートします。

さらに、「再生を続行」機能により、最も忙しい時でも物語の中で自分の場所を失うことはありません。ワンタップするだけで、物語の核心に戻ることができます。

パーソナライズされたオーディオブックプレイリストを作成する追加の利便性も見逃さないでください。ジャンル、著者、またはシリーズごとにお気に入りを分類できます。そしてオフラインの時でも、Evermusicの堅牢なサポートにより、オーディオブックは常に手元にあります。

時間が貴重な世界において、Evermusicとオーディオブックは共に物語の楽しさをより身近にします。文学の冒険を楽しみ、読書習慣を向上させ、Evermusicで移動中の時間を最大限に活用しましょう。

## よくある質問

{{% details title="Evermusicはどのオーディオブック形式をサポートしていますか？" closed="true" %}}
EvermusicはMP3、M4A、M4B、FLAC、WAV、AIFF、OGGなど、一般的なオーディオ形式をサポートしています。Evermusicで再生できるオーディオファイルであれば、オーディオブックとして使用できます。
{{% /details %}}

{{% details title="クラウドストレージからオーディオブックを聴くことはできますか？" closed="true" %}}
はい。EvermusicはiCloud Drive、Google Drive、Dropbox、OneDrive、Box、WebDAVサーバーに接続できます。オーディオブックを直接ストリーミングしたり、オフラインリスニング用にダウンロードしたりできます。
{{% /details %}}

{{% details title="ブックマークは新しいデバイスに転送されますか？" closed="true" %}}
はい。Evermusicはオーディオブックマークをファイルのメタデータに保存するため、ファイルを新しいデバイスに移動すると自動的に転送されます。
{{% /details %}}

{{% details title="Evermusicは聴取を中断した場所を記憶しますか？" closed="true" %}}
はい。設定 > オーディオプレーヤー > 一般で「再生位置を保存」と「オーディオプレーヤーの状態を保存」を有効にしてください。アプリはセッション間で正確な位置を保存および復元します。
{{% /details %}}

{{% details title="オーディオブックの再生速度を調整できますか？" closed="true" %}}
はい。設定 > オーディオプレーヤー > 再生速度に移動して、お好みの速度を設定してください。リスニングの好みに合わせて、ナレーションを速くしたり遅くしたりできます。
{{% /details %}}

{{% details title="オーディオブックをEvermusicに転送するにはどうすればよいですか？" closed="true" %}}
iTunes/Finderのファイル共有、WiFi Drive（アプリ内蔵）、またはEvermusic内でクラウドストレージアカウントを接続することでファイルを転送できます。
{{% /details %}}

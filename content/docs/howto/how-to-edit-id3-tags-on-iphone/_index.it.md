## Introduzione

In questo tutorial, impareremo come modificare i tag audio su iPhone o Mac utilizzando l'editor di tag integrato nell'app Evermusic o Flacbox. Evermusic e Flacbox ti permettono di connetterti ai tuoi servizi di archiviazione cloud come Dropbox, Google Drive, OneDrive, e persino al tuo computer di casa, TimeCapsule o NAS. Con la possibilità di scaricare brani, album e artisti per la riproduzione offline, la tua musica è sempre a portata di mano.

Quando importi brani nella tua libreria musicale, vengono raggruppati intelligentemente per artisti, album e generi. Questo rende fondamentale mantenere aggiornati i metadati dei tuoi brani. Con metadati accurati, puoi trovare qualsiasi brano, album o artista nella tua libreria musicale.

Mentre molte app desktop offrono la modifica dei metadati, Evermusic e Flacbox portano la semplicità al livello successivo includendo un editor di tag ID3. Ora puoi usare un'unica app per costruire la tua libreria musicale, goderti le tue tracce e correggere i tag audio.

## Completamento automatico intelligente  

Puoi anche utilizzare la funzione di completamento automatico intelligente toccando il pulsante 'Ricerca automatica tag audio'. Questo aiuta a correggere errori di ortografia e aggiungere informazioni mancanti. Immagina di ricordare solo l'artista e il nome della traccia ma non il nome dell'album — compila semplicemente i campi conosciuti (gli errori sono consentiti) e tocca il pulsante 'Ricerca automatica tag audio'. L'app utilizzerà un database online per trovare i tag mancanti e riempire le lacune.

## Caricamento automatico  

Quando chiudi l'editor dei tag, l'app caricherà automaticamente il file audio modificato sul cloud, assicurando che la tua libreria rimanga aggiornata.

### Ricerca automatica tag audio

Questa azione attiva il motore di ricerca intelligente dei tag, che trova e compila i metadati completi per il tuo file audio basandosi sulle informazioni esistenti.  
L'app utilizza il database MusicBrainz — uno dei database di tag più completi — con oltre **50 milioni** di tracce.

### Cerca copertina album

Usa i metadati per cercare sul web la copertina corretta dell'album. Una volta trovata, salva l'immagine nelle tue **Foto** usando il menu contestuale del sistema. Poi, torna all'editor tag, tocca l'icona della fotocamera, vai alla **Libreria foto** e seleziona l'immagine salvata. L'app la imposterà come copertina del tuo file audio. Puoi regolare il ridimensionamento delle immagini di copertina nelle impostazioni dell'app.

### Normalizza codifica

Questa azione normalizzerà la codifica del testo di tutti i tag nei metadati del file audio. È particolarmente utile se stai passando da un PC Windows, dove i file possono usare codifiche diverse che appaiono come caratteri illeggibili o corrotti su Mac.

### Elimina tag audio

Cancella tutti i campi dei metadati di un file. Utile quando si ricomincia da zero. Tocca **Salva** per confermare.

## Impostazioni editor tag

Naviga su **Impostazioni → Editor tag** per personalizzare il comportamento.

### Ridimensionamento copertina album

Seleziona come le copertine degli album devono essere ridimensionate quando vengono salvate nei file audio. Puoi disabilitare il ridimensionamento per mantenere la dimensione originale dell'immagine, ma tieni presente che alcuni lettori potrebbero non supportare file di illustrazioni grandi.

### Aggiorna file online

Puoi controllare come l'app aggiorna i metadati per i file audio archiviati nei servizi cloud:

- **Mostra messaggio di conferma**  
Chiedi prima di applicare modifiche ai metadati dei file cloud.

- **Aggiorna automaticamente i metadati del file**  
Salva le modifiche ai metadati nel file cloud automaticamente dopo la modifica.

- **Non aggiornare i metadati del file**  
Salta l'aggiornamento dei file cloud — le modifiche verranno applicate solo localmente.

### Modifica file online

Scegli cosa succede alle copie locali scaricate dei file cloud dopo la modifica:

- **Elimina sempre il file scaricato**  
  Rimuovi la copia locale dopo aver salvato le modifiche.

- **Non eliminare il file scaricato**  
  Mantieni il file scaricato sul dispositivo dopo la modifica.

### Pulsanti schermata principale

Personalizza quali azioni appaiono nella schermata principale dell'editor tag. Abilita o disabilita pulsanti specifici per adattare l'interfaccia al tuo flusso di lavoro.

## FAQ

{{% details title="Quali formati audio posso modificare?" closed="true" %}}
Evermusic e Flacbox supportano la modifica dei tag per MP3, FLAC, AAC, OGG e altri formati audio comuni. Evertag supporta oltre 30 formati inclusi WAV, AIFF, WMA e APE.
{{% /details %}}

{{% details title="Posso modificare i tag dei file archiviati nei servizi cloud?" closed="true" %}}
Sì. Collega il tuo account Dropbox, Google Drive, OneDrive o altro cloud. L'app scarica il file, ti permette di modificare i tag e carica automaticamente il file modificato sul cloud.
{{% /details %}}

{{% details title="Qual è la differenza tra Evermusic/Flacbox ed Evertag?" closed="true" %}}
Evermusic e Flacbox sono lettori musicali con un editor di tag base integrato. Evertag è un editor di tag dedicato che supporta oltre 120 tag audio, modifica in batch e oltre 30 formati -- ideale per gestire grandi librerie.
{{% /details %}}

{{% details title="La funzione di ricerca automatica richiede una connessione internet?" closed="true" %}}
Sì. La funzione di ricerca automatica tag audio interroga il database online MusicBrainz per trovare e compilare i metadati. È richiesta una connessione internet attiva per questa funzione.
{{% /details %}}

{{% details title="La modifica dei tag cambierà la qualità audio?" closed="true" %}}
No. La modifica dei tag modifica solo i metadati incorporati nel file. I dati audio stessi rimangono intatti -- non si verifica alcuna ricodifica.
{{% /details %}}

**摘要：** 使用 Evermusic 或 Flacbox 內建的標籤編輯器，在 iPhone 或 Mac 上編輯 ID3 標籤——適用於雲端和本機檔案。需要批次編輯或 120 多個標籤欄位？請改用 [Evertag](https://apps.apple.com/app/apple-store/id1450763230?pt=95781850&ct=everappzcom&mt=8)。

## 簡介

在本教學中，我們將學習如何使用 Evermusic 或 Flacbox 應用程式中內建的標籤編輯器，在 iPhone 或 Mac 上編輯音訊標籤。Evermusic 和 Flacbox 讓您可以連接 Dropbox、Google Drive、OneDrive 等雲端儲存服務，甚至連接您的家用電腦、TimeCapsule 或 NAS。您可以下載歌曲、專輯和藝人以便離線播放，讓您的音樂隨時隨地觸手可及。

當您將歌曲匯入音樂資料庫時，它們會按藝人、專輯和類型智慧分組。因此，保持歌曲元數據的更新至關重要。有了準確的元數據，您可以在音樂資料庫中找到任何歌曲、專輯或藝人。

雖然許多桌面應用程式提供元數據編輯功能，但 Evermusic 和 Flacbox 透過內建 ID3 標籤編輯器將簡便性提升到了新的層次。現在，您可以使用一個應用程式來建立音樂資料庫、享受音樂並修正音訊標籤。

在「標籤編輯器」畫面上，修改標題、藝人、專輯、年份、備註、曲目編號、類型等元數據欄位。

## 智慧自動完成  

您還可以透過點擊「自動搜尋音訊標籤」按鈕來使用智慧自動完成功能。這有助於修正拼寫錯誤並添加缺失的資訊。假設您只記得藝人和曲目名稱，但不記得專輯名稱——只需填入已知的欄位（允許有錯誤），然後點擊「自動搜尋音訊標籤」按鈕。應用程式將使用線上資料庫為您的曲目尋找缺失的標籤並填補空白。

## 自動上傳  

當您關閉標籤編輯器時，應用程式會自動將編輯後的音訊檔案上傳回雲端，確保您的資料庫保持最新。

額外的編輯選項可透過封面下方的工具列使用。

### 自動搜尋音訊標籤

此操作會啟動智慧標籤搜尋引擎，根據現有資訊為您的音訊檔案尋找並填入完整的元數據。  
應用程式使用 MusicBrainz 資料庫——最全面的標籤資料庫之一——擁有超過 **5,000 萬**首曲目。

### 搜尋專輯封面

使用元數據在網路上搜尋正確的專輯封面。找到後，使用系統右鍵選單將圖片儲存到**照片**中。然後返回標籤編輯器，點擊相機圖示，前往**照片圖庫**，選擇已儲存的圖片。應用程式會將其設定為您音訊檔案的封面。您可以在應用程式的設定中調整封面圖片的縮放方式。

### 標準化編碼

此操作會標準化音訊檔案元數據中所有標籤的文字編碼。如果您從 Windows 電腦轉換過來，這尤其有用，因為檔案可能使用不同的編碼，導致在 Mac 上顯示為無法辨識或亂碼的字元。

### 刪除音訊標籤

清除檔案的所有元數據欄位。在需要從頭開始時很有用。點擊**儲存**以確認。

## 標籤編輯器設定

前往**設定 → 標籤編輯器**自訂行為。

### 專輯封面縮放

選擇將專輯封面儲存到音訊檔案時的縮放方式。您可以停用縮放以保留原始圖片大小，但請注意某些播放器可能不支援大型封面檔案。

### 更新線上檔案

您可以控制應用程式如何更新儲存在雲端服務中的音訊檔案元數據：

- **顯示確認訊息**  
在將元數據變更套用到雲端檔案之前詢問確認。

- **自動更新檔案的元數據**  
編輯後自動將元數據變更儲存到雲端檔案。

- **不更新檔案的元數據**  
略過更新雲端檔案——變更僅在本機套用。

### 編輯線上檔案

選擇編輯後如何處理雲端檔案的本機下載副本：

- **始終刪除已下載的檔案**  
  儲存變更後移除本機副本。

- **不刪除已下載的檔案**  
  編輯後將已下載的檔案保留在裝置上。

### 主畫面按鈕

自訂標籤編輯器主畫面上顯示的操作。啟用或停用特定按鈕以根據您的工作流程自訂介面。

## 結論

使用 Evermusic 和 Flacbox 簡化您的音樂資料庫管理和標籤編輯。享受流暢的體驗，充分利用您儲存在雲端的音樂。立即試用，輕鬆重新探索您的音樂收藏！

## 常見問題

{{% details title="我可以編輯哪些音訊格式的標籤？" closed="true" %}}
Evermusic 和 Flacbox 支援編輯 MP3、FLAC、AAC、OGG 及其他常見音訊格式的標籤。Evertag 支援 30 多種格式，包括 WAV、AIFF、WMA 和 APE。
{{% /details %}}

{{% details title="我可以編輯儲存在雲端服務中的檔案標籤嗎？" closed="true" %}}
可以。連接您的 Dropbox、Google Drive、OneDrive 或其他雲端帳號。應用程式會下載檔案，讓您編輯標籤，並自動將修改後的檔案上傳回雲端。
{{% /details %}}

{{% details title="Evermusic/Flacbox 和 Evertag 有什麼區別？" closed="true" %}}
Evermusic 和 Flacbox 是內建基本標籤編輯器的音樂播放器。Evertag 是專用的標籤編輯器，支援 120 多個音訊標籤、批次編輯和 30 多種格式——非常適合管理大型資料庫。
{{% /details %}}

{{% details title="自動搜尋功能需要網路連線嗎？" closed="true" %}}
是的。自動搜尋音訊標籤功能會查詢 MusicBrainz 線上資料庫以尋找並填入元數據。此功能需要有效的網路連線。
{{% /details %}}

{{% details title="編輯標籤會影響我的音質嗎？" closed="true" %}}
不會。標籤編輯僅修改嵌入在檔案中的元數據。音訊資料本身保持不變——不會進行重新編碼。
{{% /details %}}

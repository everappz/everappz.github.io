**요약:** Evermusic 또는 Flacbox의 내장 태그 편집기를 사용하여 iPhone 또는 Mac에서 ID3 태그를 편집하세요 -- 클라우드 파일과 로컬 파일 모두 가능합니다. 일괄 편집이나 120개 이상의 태그 필드가 필요하세요? 대신 [Evertag](https://apps.apple.com/app/apple-store/id1450763230?pt=95781850&ct=everappzcom&mt=8)를 사용하세요.

## 소개

이 튜토리얼에서는 Evermusic 또는 Flacbox 앱의 내장 태그 편집기를 사용하여 iPhone 또는 Mac에서 오디오 태그를 편집하는 방법을 배웁니다. Evermusic과 Flacbox를 사용하면 Dropbox, Google Drive, OneDrive 같은 클라우드 스토리지 서비스는 물론 가정용 컴퓨터, TimeCapsule 또는 NAS에도 연결할 수 있습니다. 노래, 앨범, 아티스트를 오프라인 재생용으로 다운로드할 수 있어 음악이 항상 손끝에 있습니다.

음악 라이브러리에 노래를 가져오면 아티스트, 앨범, 장르별로 지능적으로 그룹화됩니다. 이것은 노래의 메타데이터를 최신 상태로 유지하는 것을 매우 중요하게 만듭니다. 정확한 메타데이터로 음악 라이브러리에서 어떤 노래, 앨범 또는 아티스트든 찾을 수 있습니다.

많은 데스크톱 앱이 메타데이터 편집을 제공하지만, Evermusic과 Flacbox는 ID3 태그 편집기를 포함하여 단순함을 한 단계 끌어올립니다. 이제 하나의 앱으로 음악 라이브러리를 구축하고, 트랙을 즐기고, 오디오 태그를 수정할 수 있습니다.

## 스마트 자동완성  

'오디오 태그 자동 검색' 버튼을 탭하여 스마트 자동완성 기능을 활용할 수도 있습니다. 이 기능은 맞춤법 오류를 수정하고 누락된 정보를 추가하는 데 도움이 됩니다. 아티스트와 트랙 이름은 기억하지만 앨범 이름이 기억나지 않는 경우를 상상해 보세요 — 알고 있는 필드를 채우고(실수는 허용됩니다) '오디오 태그 자동 검색' 버튼을 탭하면 됩니다. 앱이 온라인 데이터베이스를 사용하여 누락된 태그를 찾고 빈칸을 채웁니다.

## 앨범 커버 편집

앨범 커버를 변경하려면:

1. 아트워크 캐러셀에서 **카메라 아이콘**을 탭합니다.  
2. 이미지 위치 선택: 로컬 파일, 클라우드, 다운로드 또는 사진 라이브러리.  
3. 커버 아트로 적용할 이미지를 선택합니다.

## 태그 편집기의 추가 작업

아트워크 보기 아래의 도구 모음을 통해 추가 편집 옵션을 사용할 수 있습니다.

{{< cards cols="1">}}
{{< card title="" subtitle="추가 작업 메뉴" image="/docs/howto/how-to-edit-id3-tags-on-iphone/tags_edito_more_actions.webp" >}}

### 오디오 태그 자동 검색

이 작업은 스마트 태그 검색 엔진을 활성화하여 기존 정보를 기반으로 오디오 파일의 전체 메타데이터를 찾아 채웁니다.  
앱은 MusicBrainz 데이터베이스 — 가장 포괄적인 태그 데이터베이스 중 하나 — 를 사용하며 **5천만** 개 이상의 트랙을 보유하고 있습니다.

### 앨범 아트워크 검색

메타데이터를 사용하여 웹에서 올바른 앨범 아트워크를 검색합니다. 찾으면 시스템 컨텍스트 메뉴를 사용하여 이미지를 **사진**에 저장합니다. 그런 다음 태그 편집기로 돌아가 카메라 아이콘을 탭하고 **사진 라이브러리**로 이동하여 저장된 이미지를 선택합니다. 앱이 이를 오디오 파일의 커버로 설정합니다. 앱 설정에서 커버 이미지 크기 조정 방법을 조정할 수 있습니다.

### 인코딩 정규화

이 작업은 오디오 파일의 메타데이터에 있는 모든 태그의 텍스트 인코딩을 정규화합니다. Windows PC에서 전환하는 경우 특히 유용하며, 파일이 Mac에서 읽을 수 없거나 깨진 문자로 나타나는 다른 인코딩을 사용할 수 있습니다.

### 오디오 태그 삭제

파일의 모든 메타데이터 필드를 지웁니다. 처음부터 시작할 때 유용합니다. **저장**을 탭하여 확인합니다.

## 태그 편집기 설정

**설정 → 태그 편집기**로 이동하여 동작을 사용자 정의합니다.

### 앨범 커버 크기 조정

오디오 파일에 저장할 때 앨범 커버의 크기 조정 방법을 선택합니다. 원본 이미지 크기를 유지하기 위해 크기 조정을 비활성화할 수 있지만, 일부 플레이어가 큰 아트워크 파일을 지원하지 않을 수 있습니다.

### 온라인 파일 업데이트

클라우드 서비스에 저장된 오디오 파일의 메타데이터를 앱이 업데이트하는 방법을 제어할 수 있습니다:

- **확인 메시지 표시**  
클라우드 파일에 메타데이터 변경 사항을 적용하기 전에 확인합니다.

- **파일의 메타데이터 자동 업데이트**  
편집 후 클라우드 파일에 메타데이터 변경 사항을 자동으로 저장합니다.

- **파일의 메타데이터를 업데이트하지 않음**  
클라우드 파일 업데이트를 건너뜁니다 — 변경 사항은 로컬에서만 적용됩니다.

### 온라인 파일 편집

편집 후 클라우드 파일의 로컬 다운로드 복사본에 어떤 일이 일어나는지 선택합니다:

- **다운로드된 파일 항상 삭제**  
  변경 사항을 저장한 후 로컬 복사본을 제거합니다.

- **다운로드된 파일을 삭제하지 않음**  
  편집 후 기기에 다운로드된 파일을 유지합니다.

### 메인 화면 버튼

태그 편집기의 메인 화면에 표시되는 작업을 사용자 정의합니다. 워크플로에 맞게 인터페이스를 조정하기 위해 특정 버튼을 활성화하거나 비활성화합니다.

## 결론

Evermusic과 Flacbox로 음악 라이브러리 관리와 태그 편집을 간소화하세요. 최적화된 경험을 즐기고 클라우드에 저장된 음악을 최대한 활용하세요. 오늘 사용해 보고 음악 컬렉션을 쉽게 재발견하세요!

## FAQ

{{% details title="어떤 오디오 형식의 태그를 편집할 수 있나요?" closed="true" %}}
Evermusic과 Flacbox는 MP3, FLAC, AAC, OGG 및 기타 일반적인 오디오 형식의 태그 편집을 지원합니다. Evertag는 WAV, AIFF, WMA, APE를 포함한 30개 이상의 형식을 지원합니다.
{{% /details %}}

{{% details title="클라우드 서비스에 저장된 파일의 태그를 편집할 수 있나요?" closed="true" %}}
네. Dropbox, Google Drive, OneDrive 또는 기타 클라우드 계정을 연결하세요. 앱이 파일을 다운로드하고, 태그를 편집할 수 있게 하며, 수정된 파일을 자동으로 클라우드에 다시 업로드합니다.
{{% /details %}}

{{% details title="Evermusic/Flacbox와 Evertag의 차이점은 무엇인가요?" closed="true" %}}
Evermusic과 Flacbox는 기본 태그 편집기가 내장된 음악 플레이어입니다. Evertag는 120개 이상의 오디오 태그, 일괄 편집, 30개 이상의 형식을 지원하는 전용 태그 편집기로, 대규모 라이브러리 관리에 이상적입니다.
{{% /details %}}

{{% details title="자동 검색 기능에 인터넷 연결이 필요한가요?" closed="true" %}}
네. 오디오 태그 자동 검색 기능은 메타데이터를 찾고 채우기 위해 MusicBrainz 온라인 데이터베이스에 쿼리합니다. 이 기능에는 활성 인터넷 연결이 필요합니다.
{{% /details %}}

{{% details title="태그를 편집하면 오디오 품질이 변하나요?" closed="true" %}}
아니요. 태그 편집은 파일에 포함된 메타데이터만 수정합니다. 오디오 데이터 자체는 변경되지 않습니다 — 재인코딩이 발생하지 않습니다.
{{% /details %}}

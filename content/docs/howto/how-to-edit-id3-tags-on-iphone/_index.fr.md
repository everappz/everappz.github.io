## Introduction

Dans ce tutoriel, nous apprendrons comment modifier vos tags audio sur iPhone ou Mac en utilisant l'éditeur de tags intégré dans l'application Evermusic ou Flacbox. Evermusic et Flacbox vous permettent de vous connecter à vos services de stockage cloud comme Dropbox, Google Drive, OneDrive, et même votre ordinateur personnel, TimeCapsule ou NAS. Avec la possibilité de télécharger des chansons, albums et artistes pour une écoute hors ligne, votre musique est toujours à portée de main.

Lorsque vous importez des chansons dans votre bibliothèque musicale, elles sont regroupées intelligemment par artistes, albums et genres. Il est donc vital de garder les métadonnées de vos chansons à jour. Avec des métadonnées précises, vous pouvez localiser n'importe quelle chanson, album ou artiste dans votre bibliothèque musicale.

Alors que de nombreuses applications de bureau offrent l'édition de métadonnées, Evermusic et Flacbox portent la simplicité au niveau supérieur en incluant un éditeur de tags ID3. Maintenant, vous pouvez utiliser une seule application pour construire votre bibliothèque musicale, profiter de vos pistes et corriger les tags audio.

## Choisissez 'Modifier les tags audio'  

Dans le menu contextuel, sélectionnez 'Modifier les tags audio.'

## Téléchargement et édition  

Attendez que le fichier soit téléchargé dans le cache et que l'écran d'édition apparaisse.

## Modifier les tags  

Sur l'écran 'Éditeur de tags', modifiez les champs de métadonnées tels que Titre, Artiste, Album, Année, Commentaire, Numéro de piste, Genre.   

Une fois l'édition terminée, appuyez sur le bouton 'Enregistrer' pour sauvegarder vos modifications.

## Autocomplétion intelligente  

Vous pouvez également utiliser la fonction d'autocomplétion intelligente en appuyant sur le bouton 'Recherche automatique des tags audio'. Cela aide à corriger les erreurs d'orthographe et à ajouter les informations manquantes. Imaginez que vous ne vous souvenez que de l'artiste et du nom de la piste mais pas du nom de l'album — remplissez simplement les champs connus (les erreurs sont autorisées) et appuyez sur le bouton 'Recherche automatique des tags audio'. L'application utilisera une base de données en ligne pour trouver les tags manquants et combler les lacunes.

## Téléversement automatique  

Lorsque vous fermez l'éditeur de tags, l'application téléversera automatiquement le fichier audio modifié vers le cloud, garantissant que votre bibliothèque reste à jour.

### Recherche automatique des tags audio

Cette action active le moteur de recherche intelligent de tags, qui trouve et remplit les métadonnées complètes de votre fichier audio en fonction des informations existantes.  
L'application utilise la base de données MusicBrainz — l'une des bases de données de tags les plus complètes — avec plus de **50 millions** de pistes.

### Rechercher la pochette d'album

Utilisez les métadonnées pour rechercher sur le web la bonne pochette d'album. Une fois trouvée, enregistrez l'image dans vos **Photos** en utilisant le menu contextuel du système. Ensuite, retournez à l'éditeur de tags, appuyez sur l'icône de caméra, allez dans **Photothèque** et sélectionnez l'image enregistrée. L'application la définira comme pochette de votre fichier audio. Vous pouvez ajuster la mise à l'échelle des pochettes dans les paramètres de l'application.

### Normaliser l'encodage

Cette action normalisera l'encodage du texte de tous les tags dans les métadonnées du fichier audio. C'est particulièrement utile si vous passez d'un PC Windows, où les fichiers peuvent utiliser différents encodages qui apparaissent comme des caractères illisibles ou déformés sur un Mac.

### Supprimer les tags audio

Effacer tous les champs de métadonnées d'un fichier. Utile pour repartir de zéro. Appuyez sur **Enregistrer** pour confirmer.

## Paramètres de l'éditeur de tags

Naviguez vers **Réglages → Éditeur de tags** pour personnaliser le comportement.

### Mise à l'échelle de la pochette d'album

Sélectionnez comment les pochettes d'album doivent être mises à l'échelle lors de l'enregistrement dans les fichiers audio. Vous pouvez désactiver la mise à l'échelle pour conserver la taille d'image originale, mais sachez que certains lecteurs peuvent ne pas prendre en charge les fichiers d'illustrations volumineux.

### Mettre à jour les fichiers en ligne

Vous pouvez contrôler comment l'application met à jour les métadonnées des fichiers audio stockés dans les services cloud :

- **Afficher un message de confirmation**  
Demander avant d'appliquer les modifications de métadonnées aux fichiers cloud.

- **Mettre à jour automatiquement les métadonnées du fichier**  
Enregistrer les modifications de métadonnées dans le fichier cloud automatiquement après l'édition.

- **Ne pas mettre à jour les métadonnées du fichier**  
Ignorer la mise à jour des fichiers cloud — les modifications s'appliqueront uniquement localement.

### Modifier les fichiers en ligne

Choisissez ce qui se passe avec les copies locales téléchargées des fichiers cloud après l'édition :

- **Toujours supprimer le fichier téléchargé**  
  Supprimer la copie locale après l'enregistrement des modifications.

- **Ne pas supprimer le fichier téléchargé**  
  Conserver le fichier téléchargé sur votre appareil après l'édition.

### Boutons de l'écran principal

Personnalisez les actions qui apparaissent sur l'écran principal de l'éditeur de tags. Activez ou désactivez des boutons spécifiques pour adapter l'interface à votre flux de travail.

## FAQ

{{% details title="Quels formats audio puis-je modifier ?" closed="true" %}}
Evermusic et Flacbox prennent en charge l'édition de tags pour MP3, FLAC, AAC, OGG et d'autres formats audio courants. Evertag prend en charge plus de 30 formats, y compris WAV, AIFF, WMA et APE.
{{% /details %}}

{{% details title="Puis-je modifier les tags de fichiers stockés dans des services cloud ?" closed="true" %}}
Oui. Connectez votre compte Dropbox, Google Drive, OneDrive ou autre cloud. L'application télécharge le fichier, vous permet de modifier les tags et téléverse automatiquement le fichier modifié vers le cloud.
{{% /details %}}

{{% details title="Quelle est la différence entre Evermusic/Flacbox et Evertag ?" closed="true" %}}
Evermusic et Flacbox sont des lecteurs de musique avec un éditeur de tags basique intégré. Evertag est un éditeur de tags dédié prenant en charge plus de 120 tags audio, l'édition par lots et plus de 30 formats -- idéal pour gérer de grandes bibliothèques.
{{% /details %}}

{{% details title="La fonction de recherche automatique nécessite-t-elle une connexion internet ?" closed="true" %}}
Oui. La fonction de recherche automatique des tags audio interroge la base de données en ligne MusicBrainz pour trouver et remplir les métadonnées. Une connexion internet active est requise pour cette fonctionnalité.
{{% /details %}}

{{% details title="La modification des tags changera-t-elle la qualité audio ?" closed="true" %}}
Non. L'édition de tags ne modifie que les métadonnées intégrées dans le fichier. Les données audio elles-mêmes restent intactes -- aucun réencodage ne se produit.
{{% /details %}}

**要約:** Evermusic、Flacbox、またはEvertagのWi-Fi Driveを使って、パソコンからiPhoneやiPadに音楽を転送できます。iTunesは必要ありません。両方のデバイスが同じWi-Fiネットワークに接続されている必要があります。ウェブブラウザまたはWebDAV（Mac Finder / Windows エクスプローラー）を使って転送できます。

Wi-Fi Driveは、iTunesを必要とせずに、パソコンからiPhoneやiPadに音楽コレクションをシームレスに転送するための究極のソリューションです。この手軽な方法により、ローカルWi-Fi接続を使って複数のオーディオファイルやフォルダ全体を簡単にダウンロードまたはアップロードできます。これがスムーズに動作するためには、パソコンとiOSデバイスの両方が同じWi-Fiネットワークに接続されている必要があります。

オーディオファイルをワイヤレスで転送する2つの簡単な方法をご紹介します。

## デスクトップウェブブラウザ経由でオーディオファイルを転送する

1. iOSデバイスでEvermusic、Flacbox、またはEvertagなどのお好みのアプリを起動します。
2. アプリ内で「接続」>「コンピュータ」>「Wi-Fiで接続」に移動します。
3. セキュリティを強化するために、ユーザー名とパスワードを入力できます（オプション）。
4. 「Wi-Fi Driveを開始」をタップして接続を開始します。

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. アプリ内で生成されたウェブブラウザのURLをコピーします。
6. デスクトップで互換性のあるウェブブラウザを開きます（Safari、Google Chrome、Opera、Yandex Browser、FireFox）。
7. 手順5のURLをデスクトップブラウザのアドレスフィールドに入力します。

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. 接続が確立されると、ファイルマネージャーのウェブアプリが表示されます。コピー、移動、削除、アップロード、ダウンロードなどの一般的な操作を行って、パソコンからデバイスのローカルストレージにファイルを転送できます。ドラッグ＆ドロップまたはアップロードボタンを使って、シームレスにiPhoneに音楽を転送しましょう。

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. 終了するには、完了したらiPhoneで「Wi-Fi Driveを停止」をタップします。

**注意:** デスクトップブラウザの最新バージョンを使用していること、およびJavaScriptが有効になっていることを確認してください。接続の問題を解決できます。

## Mac FinderまたはWindowsエクスプローラーでオーディオファイルを転送する

### macOSをお使いの場合

1. Finderアイコンを右クリックし、メニューから「サーバへ接続...」を選択します。

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. iOSアプリで提供されるサーバーURLを入力します（「このURLをWebDAVアプリケーションに入力してください」というテキストの下にあります）。

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. 「接続する」ボタンをタップします。
4. 接続が確立されると、Mac FinderにiPhoneの内容が表示されます。

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. ドラッグ＆ドロップ、移動、名前変更などのコマンドを使って、iOSデバイス上のファイルを管理しましょう。

### Windowsユーザーの場合

1. エクスプローラーを開き、左サイドバーの「PC」を右クリックします。

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. ドロップダウンメニューから「ネットワークドライブの割り当て」を選択します。
3. 「フォルダー」フィールドに、iOSアプリで提供されるサーバーURLを入力します（「このURLをWebDAVアプリケーションに入力してください」というテキストの下にあります）。
4. ドライブ文字を選択し、「完了」ボタンをクリックします。

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. 接続が成功すれば、次のウィンドウでiOSデバイス上のファイルを管理できます。

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## まとめ

Wi-Fi Driveがあれば、iTunesに苦労する日々は終わりです。ローカルWi-Fiネットワークの利便性で、iPhoneやiPadへの音楽コレクションの転送をシームレスかつ効率的に行えます。面倒にさよならを告げ、手間いらずの音楽管理を始めましょう！

## よくある質問

{{% details title="Wi-Fi Driveで転送できるオーディオフォーマットは？" closed="true" %}}
Wi-Fi Driveはあらゆるファイルタイプを転送できます。EvermusucとFlacboxは、MP3、FLAC、AAC、WAV、AIFF、OGG、WMA、その他多くのオーディオフォーマットの再生をサポートしています。
{{% /details %}}

{{% details title="iPhoneに音楽を入れるのにiTunesは必要ですか？" closed="true" %}}
いいえ。Wi-Fi Driveはローカルの Wi-Fiネットワークを通じて直接音楽を転送します。iTunesは必要ありません。
{{% /details %}}

{{% details title="音楽フォルダ全体を一度に転送できますか？" closed="true" %}}
はい。ウェブブラウザ方式では、ネストされたサブフォルダを含むフォルダ全体のアップロードに対応しています。
{{% /details %}}

{{% details title="音楽の転送は安全ですか？" closed="true" %}}
Wi-Fi Driveはローカルネットワーク上でのみ動作します。追加のセキュリティのために、ユーザー名とパスワードを設定することもできます。
{{% /details %}}

{{% details title="音楽用のWi-Fi Driveに対応しているアプリは？" closed="true" %}}
Evermusic、Flacbox、Evertagのすべてに、パソコンからオーディオファイルを転送するためのWi-Fi Driveが搭載されています。
{{% /details %}}

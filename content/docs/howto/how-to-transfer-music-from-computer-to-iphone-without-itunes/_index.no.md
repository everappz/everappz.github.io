**Kort fortalt:** Bruk Wi-Fi Drive i Evermusic, Flacbox eller Evertag for å overføre musikk fra datamaskinen til iPhone eller iPad. Ingen iTunes nødvendig. Begge enhetene må være på samme Wi-Fi-nettverk. Overfør via en nettleser eller WebDAV (Mac Finder / Windows Filutforsker).

Du kan se en videoopplæring fra [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) eller lese tekstversjonen nedenfor.
<br><br>

Wi-Fi Drive er den ultimate løsningen for sømløs overføring av musikksamlingen din fra datamaskinen til iPhone eller iPad uten behov for iTunes. Denne problemfrie metoden lar deg enkelt laste ned eller laste opp flere lydfiler og til og med hele mapper ved hjelp av din lokale Wi-Fi-tilkobling. Både datamaskinen og iOS-enheten din bør være koblet til det samme Wi-Fi-nettverket for at dette skal fungere perfekt.

Her er to enkle måter å overføre lydfilene dine trådløst på.

## Overføre lydfiler via nettleser på skrivebordet

1. Start din foretrukne app, som Evermusic, Flacbox eller Evertag på iOS-enheten din.
2. Naviger til "Tilkoblinger" > "Datamaskin" > "Koble til med Wi-Fi" i appen.
3. For ekstra sikkerhet kan du angi brukernavn og passord (valgfritt).
4. Trykk på "Start Wi-Fi Drive" for å starte tilkoblingen.

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. Kopier nettleser-URLen som genereres i appen.
6. Åpne en kompatibel nettleser på skrivebordet ditt (Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Skriv inn URLen fra trinn 5 i adressefeltet i nettleseren på skrivebordet.

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Når tilkoblingen er etablert, vises en filbehandler-webapp. Du kan utføre vanlige operasjoner som Kopier, Flytt, Slett, Last opp og Last ned filer fra datamaskinen til enhetens lokale lagring. Bruk dra og slipp eller Last opp-knappen for å sømløst overføre musikk til iPhonen din.

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. For å avslutte, trykk på "Stopp Wi-Fi Drive" på iPhonen din når du er ferdig.

**Merk:** Sørg for at du bruker den nyeste versjonen av nettleseren på skrivebordet, og at JavaScript er aktivert for å løse eventuelle tilkoblingsproblemer.

## Overføre lydfiler med Mac Finder eller Windows Filutforsker

### Hvis du bruker macOS

1. Høyreklikk på Finder-ikonet og velg "Koble til server..." fra menyen.

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Skriv inn server-URLen som er oppgitt i iOS-appen (finnes under teksten "Skriv inn denne URLen i WebDAV-applikasjonen").

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Trykk på "Koble til"-knappen.
4. Når tilkoblingen er etablert, vil du se innholdet på iPhonen din i Mac Finder.

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Bruk dra og slipp, Flytt, Gi nytt navn og andre kommandoer for å administrere filene på iOS-enheten din.

### For Windows-brukere

1. Åpne Filutforsker og høyreklikk på "Denne PCen" i venstre sidefelt.

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Velg "Tilordne nettverksstasjon" fra rullegardinmenyen.
3. I "Mappe"-feltet, skriv inn server-URLen som er oppgitt i iOS-appen (finnes under teksten "Skriv inn denne URLen i WebDAV-applikasjonen").
4. Velg en stasjonsbokstav og klikk på "Fullfør"-knappen.

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Hvis tilkoblingen er vellykket, kan du administrere filer på iOS-enheten din i det neste vinduet.

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Konklusjon

Med Wi-Fi Drive er dagene med å slite med iTunes over. Nyt en sømløs og effektiv måte å overføre musikksamlingen din til iPhone eller iPad, alt med bekvemmeligheten av ditt lokale Wi-Fi-nettverk. Si farvel til bryet og hei til enkel musikkadministrasjon!

## Ofte stilte spørsmål

{{% details title="Hvilke lydformater kan jeg overføre med Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive overfører alle filtyper. Evermusic og Flacbox støtter MP3, FLAC, AAC, WAV, AIFF, OGG, WMA og mange andre lydformater for avspilling.
{{% /details %}}

{{% details title="Trenger jeg iTunes for å legge musikk på iPhonen min?" closed="true" %}}
Nei. Wi-Fi Drive overfører musikk direkte over ditt lokale Wi-Fi-nettverk. iTunes er ikke nødvendig.
{{% /details %}}

{{% details title="Kan jeg overføre hele musikkmapper på en gang?" closed="true" %}}
Ja. Nettlesermetoden støtter opplasting av hele mapper, inkludert nestede undermapper.
{{% /details %}}

{{% details title="Er musikkoverføringen min sikker?" closed="true" %}}
Wi-Fi Drive kjører kun på ditt lokale nettverk. Du kan også angi brukernavn og passord for ekstra sikkerhet.
{{% /details %}}

{{% details title="Hvilke apper støtter Wi-Fi Drive for musikk?" closed="true" %}}
Evermusic, Flacbox og Evertag inkluderer alle Wi-Fi Drive for overføring av lydfiler fra datamaskinen din.
{{% /details %}}

**Sažetak:** Koristite Wi-Fi Drive u Evermusic, Flacbox ili Evertag za prijenos glazbe s računala na iPhone ili iPad. iTunes nije potreban. Oba uređaja moraju biti na istoj Wi-Fi mreži. Prenesite putem web preglednika ili WebDAV-a (Mac Finder / Windows Preglednik datoteka).

Možete pogledati video vodič od [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) ili pročitati tekstualnu verziju u nastavku.

Wi-Fi Drive je ultimativno rješenje za besprijekoran prijenos vaše glazbene kolekcije s računala na iPhone ili iPad bez potrebe za iTunes. Ova jednostavna metoda omogućuje vam preuzimanje ili prijenos više audio datoteka, pa čak i cijelih mapa koristeći lokalnu Wi-Fi vezu. Vaše računalo i iOS uređaj moraju biti spojeni na istu Wi-Fi mrežu kako bi ovo savršeno funkcioniralo.

Evo dva jednostavna načina za bežični prijenos vaših audio datoteka.

## Prijenos audio datoteka putem web preglednika na računalu

1. Pokrenite svoju omiljenu aplikaciju, kao što je Evermusic, Flacbox ili Evertag na svom iOS uređaju.
2. Navigirajte do "Povezivanja" > "Računalo" > "Poveži putem Wi-Fi" unutar aplikacije.
3. Za dodatnu sigurnost možete unijeti korisničko ime i lozinku (neobavezno).
4. Dodirnite "Pokreni Wi-Fi Drive" za pokretanje veze.

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. Kopirajte URL web preglednika generiran unutar aplikacije.
6. Na svom računalu otvorite kompatibilan web preglednik (Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Unesite URL iz koraka 5 u adresno polje preglednika na računalu.

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Nakon uspostavljanja veze pojavit će se web aplikacija za upravljanje datotekama. Moći ćete izvršavati uobičajene operacije poput Kopiranja, Premještanja, Brisanja, Prijenosa i Preuzimanja datoteka s računala na lokalni prostor vašeg uređaja. Koristite povuci i ispusti ili gumb za Prijenos kako biste besprijekorno prenijeli glazbu na svoj iPhone.

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. Za završetak dodirnite "Zaustavi Wi-Fi Drive" na svom iPhoneu kada završite.

**Napomena:** Provjerite koristite li najnoviju verziju preglednika na računalu i da je JavaScript omogućen za rješavanje mogućih problema s vezom.

## Prijenos audio datoteka s Mac Finderom ili Windows Preglednikom datoteka

### Ako koristite macOS

1. Desnom tipkom miša kliknite na ikonu Finder i odaberite "Spoji se na server..." iz izbornika.

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Unesite URL servera naveden u iOS aplikaciji (nalazi se ispod teksta "Unesite ovaj URL u WebDAV aplikaciju").

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Dodirnite gumb "Povezati".
4. Nakon uspostavljanja veze vidjet ćete sadržaj svog iPhonea u Mac Finderu.

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Koristite povuci i ispusti, naredbe Premjesti, Preimenuj i druge za upravljanje datotekama na svom iOS uređaju.

### Za Windows korisnike

1. Otvorite Preglednik datoteka i desnom tipkom miša kliknite na "Ovaj PC" u lijevoj bočnoj traci.

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Odaberite "Mapiraj mrežni pogon" iz padajućeg izbornika.
3. U polje "Mapa" unesite URL servera naveden u iOS aplikaciji (nalazi se ispod teksta "Unesite ovaj URL u WebDAV aplikaciju").
4. Odaberite slovo pogona i kliknite gumb "Završi".

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Ako je veza uspješna, možete upravljati datotekama na svom iOS uređaju u sljedećem prozoru.

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Zaključak

S Wi-Fi Driveom, dani borbe s iTunesom su prošlost. Uživajte u besprijekornom i učinkovitom načinu prijenosa vaše glazbene kolekcije na iPhone ili iPad, sve uz praktičnost vaše lokalne Wi-Fi mreže. Recite zbogom problemima i pozdravite jednostavno upravljanje glazbom!

## Često postavljana pitanja

{{% details title="Koje audio formate mogu prenijeti s Wi-Fi Driveom?" closed="true" %}}
Wi-Fi Drive prenosi bilo koji tip datoteke. Evermusic i Flacbox podržavaju reprodukciju MP3, FLAC, AAC, WAV, AIFF, OGG, WMA i mnogih drugih audio formata.
{{% /details %}}

{{% details title="Trebam li iTunes za stavljanje glazbe na svoj iPhone?" closed="true" %}}
Ne. Wi-Fi Drive prenosi glazbu izravno putem vaše lokalne Wi-Fi mreže. iTunes nije potreban.
{{% /details %}}

{{% details title="Mogu li prenijeti cijele glazbene mape odjednom?" closed="true" %}}
Da. Metoda putem web preglednika podržava prijenos cijelih mapa, uključujući ugniježđene podmape.
{{% /details %}}

{{% details title="Je li moj prijenos glazbe siguran?" closed="true" %}}
Wi-Fi Drive radi samo na vašoj lokalnoj mreži. Također možete postaviti korisničko ime i lozinku za dodatnu sigurnost.
{{% /details %}}

{{% details title="Koje aplikacije podržavaju Wi-Fi Drive za glazbu?" closed="true" %}}
Evermusic, Flacbox i Evertag svi uključuju Wi-Fi Drive za prijenos audio datoteka s vašeg računala.
{{% /details %}}

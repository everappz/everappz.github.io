**Resum:** Utilitza Wi-Fi Drive a Evermusic, Flacbox o Evertag per transferir música des del teu ordinador al teu iPhone o iPad. No cal iTunes. Ambdós dispositius han d'estar a la mateixa xarxa Wi-Fi. Transfereix mitjançant un navegador web o WebDAV (Mac Finder / Windows Explorador de fitxers).

Pots veure un videotutorial de [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) o llegir la versió en text a continuació.

Wi-Fi Drive és la solució definitiva per transferir la teva col·lecció de música de l'ordinador al teu iPhone o iPad sense necessitat d'iTunes. Aquest mètode senzill et permet descarregar o pujar múltiples fitxers d'àudio i fins i tot carpetes senceres utilitzant la teva connexió Wi-Fi local. Tant l'ordinador com el dispositiu iOS han d'estar connectats a la mateixa xarxa Wi-Fi perquè funcioni perfectament.

Aquí tens dues maneres senzilles de transferir els teus fitxers d'àudio sense fils.

## Transferir fitxers d'àudio mitjançant el navegador web d'escriptori

1. Obre la teva aplicació preferida, com Evermusic, Flacbox o Evertag al teu dispositiu iOS.
2. Navega a "Connexions" > "Ordinador" > "Connectar mitjançant Wi-Fi" dins de l'aplicació.
3. Per a més seguretat, pots introduir un nom d'usuari i una contrasenya (opcional).
4. Toca "Iniciar Wi-Fi Drive" per iniciar la connexió.

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. Copia l'URL del navegador web generat dins de l'aplicació.
6. Al teu escriptori, obre un navegador web compatible (Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Introdueix l'URL del pas 5 al camp d'adreces del teu navegador d'escriptori.

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Un cop establerta la connexió, apareixerà una aplicació web de gestió de fitxers. Podràs realitzar operacions comunes com Copiar, Moure, Eliminar, Pujar i Descarregar fitxers des del teu ordinador a l'emmagatzematge local del teu dispositiu. Utilitza arrossegar i deixar anar o el botó de Pujar per transferir música al teu iPhone sense problemes.

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. Per acabar, toca "Aturar Wi-Fi Drive" al teu iPhone quan hagis acabat.

**Nota:** Assegura't d'utilitzar l'última versió del teu navegador d'escriptori i que JavaScript estigui activat per resoldre qualsevol problema de connexió.

## Transferir fitxers d'àudio amb Mac Finder o Windows Explorador de fitxers

### Si utilitzes macOS

1. Fes clic dret a la icona de Finder i selecciona "Connectar al servidor..." des del menú.

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Introdueix l'URL del servidor proporcionat dins de l'aplicació iOS (que es troba sota el text "Introdueix aquesta URL a l'aplicació WebDAV").

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Toca el botó "Connectar".
4. Un cop establerta la connexió, veuràs el contingut del teu iPhone al Mac Finder.

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Utilitza arrossegar i deixar anar, les comandes Moure, Reanomenar i més per gestionar els teus fitxers al teu dispositiu iOS.

### Per a usuaris de Windows

1. Obre l'Explorador de fitxers i fes clic dret a "Aquest PC" a la barra lateral esquerra.

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Selecciona "Connectar unitat de xarxa" del menú desplegable.
3. Al camp "Carpeta", introdueix l'URL del servidor proporcionat dins de l'aplicació iOS (que es troba sota el text "Introdueix aquesta URL a l'aplicació WebDAV").
4. Tria una lletra d'unitat i fes clic al botó "Finalitzar".

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Si la connexió és correcta, pots gestionar fitxers al teu dispositiu iOS a la finestra següent.

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusió

Amb Wi-Fi Drive, s'han acabat els dies de lluitar amb iTunes. Gaudeix d'una manera fluida i eficient de transferir la teva col·lecció de música al teu iPhone o iPad, tot amb la comoditat de la teva xarxa Wi-Fi local. Digues adéu als problemes i hola a la gestió de música sense esforç!

## Preguntes freqüents

{{% details title="Quins formats d'àudio puc transferir amb Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive transfereix qualsevol tipus de fitxer. Evermusic i Flacbox admeten la reproducció de MP3, FLAC, AAC, WAV, AIFF, OGG, WMA i molts altres formats d'àudio.
{{% /details %}}

{{% details title="Necessito iTunes per posar música al meu iPhone?" closed="true" %}}
No. Wi-Fi Drive transfereix música directament a través de la teva xarxa Wi-Fi local. No cal iTunes.
{{% /details %}}

{{% details title="Puc transferir carpetes de música senceres d'una vegada?" closed="true" %}}
Sí. El mètode del navegador web admet la pujada de carpetes senceres, incloent subcarpetes imbricades.
{{% /details %}}

{{% details title="La transferència de música és segura?" closed="true" %}}
Wi-Fi Drive funciona només a la teva xarxa local. També pots establir un nom d'usuari i una contrasenya per a seguretat addicional.
{{% /details %}}

{{% details title="Quines aplicacions admeten Wi-Fi Drive per a música?" closed="true" %}}
Evermusic, Flacbox i Evertag inclouen Wi-Fi Drive per transferir fitxers d'àudio des del teu ordinador.
{{% /details %}}

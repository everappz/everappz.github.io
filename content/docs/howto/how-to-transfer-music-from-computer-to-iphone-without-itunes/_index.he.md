**תקציר:** השתמשו ב-Wi-Fi Drive ב-Evermusic, Flacbox או Evertag כדי להעביר מוזיקה מהמחשב לאייפון או לאייפד. אין צורך ב-iTunes. שני המכשירים חייבים להיות על אותה רשת Wi-Fi. העבירו דרך דפדפן אינטרנט או WebDAV (Mac Finder / Windows סייר הקבצים).

תוכלו לצפות בסרטון הדרכה מ-[**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) או לקרוא את גרסת הטקסט למטה.
<br><br>

Wi-Fi Drive הוא הפתרון האולטימטיבי להעברה חלקה של אוסף המוזיקה שלכם מהמחשב לאייפון או לאייפד ללא צורך ב-iTunes. שיטה פשוטה זו מאפשרת לכם להוריד או להעלות מספר קבצי שמע ואפילו תיקיות שלמות באמצעות חיבור ה-Wi-Fi המקומי שלכם. גם המחשב וגם מכשיר ה-iOS שלכם צריכים להיות מחוברים לאותה רשת Wi-Fi כדי שזה יעבוד בצורה מושלמת.

הנה שתי דרכים פשוטות להעברת קבצי השמע שלכם באופן אלחוטי.

## העברת קבצי שמע דרך דפדפן אינטרנט שולחני

1. הפעילו את האפליקציה המועדפת עליכם, כגון Evermusic, Flacbox או Evertag במכשיר ה-iOS שלכם.
2. נווטו אל "חיבורים" > "מחשב" > "התחבר באמצעות Wi-Fi" בתוך האפליקציה.
3. לאבטחה נוספת, תוכלו להזין שם משתמש וסיסמה (אופציונלי).
4. הקישו על "הפעל Wi-Fi Drive" כדי ליזום את החיבור.

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. העתיקו את כתובת ה-URL של דפדפן האינטרנט שנוצרה בתוך האפליקציה.
6. על שולחן העבודה, פתחו דפדפן אינטרנט תואם (Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. הזינו את כתובת ה-URL משלב 5 בשדה הכתובת של דפדפן שולחן העבודה שלכם.

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. לאחר שהחיבור נוצר, תופיע אפליקציית ניהול קבצים מבוססת אינטרנט. תוכלו לבצע פעולות נפוצות כמו העתקה, העברה, מחיקה, העלאה והורדה של קבצים מהמחשב לאחסון המקומי של המכשיר. השתמשו בגרור ושחרר או בלחצן ההעלאה כדי להעביר מוזיקה בצורה חלקה לאייפון.

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. לסיום, הקישו על "עצור Wi-Fi Drive" באייפון כשסיימתם.

**הערה:** וודאו שאתם משתמשים בגרסה האחרונה של דפדפן שולחן העבודה שלכם, ושה-JavaScript מופעל כדי לפתור בעיות חיבור.

## העברת קבצי שמע עם Mac Finder או Windows סייר הקבצים

### אם אתם משתמשים ב-macOS

1. לחצו לחיצה ימנית על סמל Finder ובחרו "התחבר לשרת..." מהתפריט.

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. הזינו את כתובת ה-URL של השרת המסופקת באפליקציית ה-iOS (נמצאת מתחת לטקסט "הזינו כתובת URL זו באפליקציית WebDAV").

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. הקישו על כפתור "לחבר".
4. לאחר שהחיבור נוצר, תראו את תוכן האייפון שלכם ב-Mac Finder.

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. השתמשו בגרור ושחרר, פקודות העברה, שינוי שם ועוד כדי לנהל את הקבצים שלכם במכשיר ה-iOS.

### למשתמשי Windows

1. פתחו את סייר הקבצים ולחצו לחיצה ימנית על "מחשב זה" בסרגל הצד השמאלי.

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. בחרו "מפה כונן רשת" מהתפריט הנפתח.
3. בשדה "תיקייה", הזינו את כתובת ה-URL של השרת המסופקת באפליקציית ה-iOS (נמצאת מתחת לטקסט "הזינו כתובת URL זו באפליקציית WebDAV").
4. בחרו אות כונן ולחצו על כפתור "סיום".

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. אם החיבור הצליח, תוכלו לנהל קבצים במכשיר ה-iOS בחלון הבא.

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## סיכום

עם Wi-Fi Drive, הימים של ההתמודדות עם iTunes נגמרו. תהנו מדרך חלקה ויעילה להעביר את אוסף המוזיקה שלכם לאייפון או לאייפד, הכל בנוחות רשת ה-Wi-Fi המקומית שלכם. אמרו שלום לטרחה וברוכים הבאים לניהול מוזיקה ללא מאמץ!

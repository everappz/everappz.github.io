**Кратко:** Используйте Wi-Fi Drive в Evermusic, Flacbox или Evertag для переноса музыки с компьютера на iPhone или iPad. iTunes не нужен. Оба устройства должны быть подключены к одной сети Wi-Fi. Передавайте файлы через веб-браузер или WebDAV (Mac Finder / Проводник Windows).

Wi-Fi Drive — это лучшее решение для беспрепятственного переноса вашей музыкальной коллекции с компьютера на iPhone или iPad без необходимости в iTunes. Этот простой метод позволяет легко загружать или выгружать множество аудиофайлов и даже целые папки через локальное Wi-Fi-соединение. Для корректной работы компьютер и iOS-устройство должны быть подключены к одной сети Wi-Fi.

Вот два простых способа беспроводной передачи аудиофайлов.

## Передача аудиофайлов через веб-браузер на компьютере

1. Запустите предпочитаемое приложение, например Evermusic, Flacbox или Evertag, на вашем iOS-устройстве.
2. Перейдите в "Подключения" > "Компьютер" > "Подключиться по Wi-Fi" в приложении.
3. Для дополнительной безопасности вы можете ввести имя пользователя и пароль (необязательно).
4. Нажмите "Запустить Wi-Fi Drive", чтобы инициировать подключение.

   ![Evermusic Wi-Fi Drive: Start](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg)

5. Скопируйте URL-адрес веб-браузера, сгенерированный в приложении.
6. На компьютере откройте совместимый веб-браузер (Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Введите URL-адрес из шага 5 в адресную строку браузера на компьютере.

   ![Evermusic Wi-Fi Drive: Browser Screen](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. После установления соединения появится веб-приложение файлового менеджера. Вы сможете выполнять стандартные операции: Копировать, Перемещать, Удалять, Загружать и Скачивать файлы с компьютера в локальное хранилище устройства. Используйте перетаскивание или кнопку «Загрузить», чтобы легко перенести музыку на iPhone.

   ![Evermusic Wi-Fi Drive: File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

9. По завершении нажмите "Остановить Wi-Fi Drive" на iPhone.

**Примечание:** Убедитесь, что вы используете последнюю версию браузера на компьютере и что JavaScript включён для устранения проблем с подключением.

## Передача аудиофайлов через Mac Finder или Проводник Windows

### Если вы используете macOS

1. Щёлкните правой кнопкой мыши по значку Finder и выберите "Подключиться к серверу..." в меню.

   ![Evermusic Wi-Fi Drive: Mac Connect To Server...](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Введите URL-адрес сервера, указанный в iOS-приложении (находится под текстом "Введите этот URL в приложении WebDAV").

   ![Evermusic Wi-Fi Drive: Mac Enter Server Address](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Нажмите кнопку "Подключить".
4. После установления соединения вы увидите содержимое вашего iPhone в Mac Finder.

   ![Evermusic Wi-Fi Drive: Mac File Manager](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Используйте перетаскивание, команды «Переместить», «Переименовать» и другие для управления файлами на iOS-устройстве.

### Для пользователей Windows

1. Откройте Проводник и щёлкните правой кнопкой мыши по "Этот компьютер" на левой боковой панели.

   ![Evermusic Wi-Fi Drive: Windows Add Connection](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Выберите "Подключить сетевой диск" в выпадающем меню.
3. В поле "Папка" введите URL-адрес сервера, указанный в iOS-приложении (находится под текстом "Введите этот URL в приложении WebDAV").
4. Выберите букву диска и нажмите кнопку "Готово".

   ![Evermusic Wi-Fi Drive: Windows Map Network Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Если подключение успешно, вы сможете управлять файлами на iOS-устройстве в следующем окне.

   ![Evermusic Wi-Fi Drive: Windows Network Drive Mapped](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

{{% /details %}}

{{% details title="Можно ли передать целые папки с музыкой за раз?" closed="true" %}}
Да. Метод через веб-браузер поддерживает загрузку целых папок, включая вложенные подпапки.
{{% /details %}}

{{% details title="Безопасна ли передача музыки?" closed="true" %}}
Wi-Fi Drive работает только в вашей локальной сети. Вы также можете установить имя пользователя и пароль для дополнительной безопасности.
{{% /details %}}

**En bref :** Spotify ne vous permet pas de changer les pochettes d'albums pour les pistes en streaming, mais vous pouvez mettre à jour les illustrations des fichiers locaux. Activez la fonctionnalité Fichiers locaux de Spotify, copiez votre musique dans le dossier Spotify, puis utilisez l'application gratuite Evertag pour modifier les pochettes d'albums et les métadonnées. ## Introduction

Vous souhaitez personnaliser votre collection musicale Spotify avec des pochettes d'albums personnalisées ? Bien que Spotify ne vous permette pas de changer les pochettes de son catalogue de streaming, vous *pouvez* mettre à jour les pochettes d'albums pour les **fichiers musicaux locaux** ajoutés à votre bibliothèque.

Dans ce guide étape par étape, vous apprendrez comment **changer les illustrations des pistes locales** dans Spotify en utilisant les applications **mobiles (iOS)** et **de bureau (Mac)**. Que vous corrigiez des illustrations incorrectes ou ajoutiez des visuels personnalisés à vos MP3, ce guide vous accompagnera tout au long du processus.

Pour faciliter les choses, nous montrerons comment modifier les pochettes d'albums en utilisant l'application **Evertag** — un puissant éditeur de métadonnées audio qui prend en charge **plus de 120 tags audio** et **plus de 30 formats de fichiers**. C'est l'outil idéal pour mettre à jour les tags ID3 et les pochettes d'albums sur votre iPhone ou Mac.

## Modes d'édition

Evertag prend en charge deux modes d'édition :

- **Mode fichier unique**
  - Faites glisser vers la gauche ou la droite dans le carrousel d'illustrations
  - Modifiez les tags de base comme Titre, Artiste, Album, Genre, Année, etc.

- **Mode par lots**
  - Sélectionnez et modifiez plusieurs fichiers à la fois
  - Appuyez sur **Modifier plusieurs fichiers simultanément** en bas de l'écran

## Mode fichier unique : tags étendus

Pour révéler **plus de 120 tags supplémentaires**, faites défiler jusqu'en bas de l'écran de l'éditeur de tags et appuyez sur **Afficher les tags étendus**.

Les tags incluent :
- Tags MusicBrainz
- Paroles
- Classifications de contenu
- Compositeur, Interprète, Compilation
- BPM, Note, Numéros de disque/piste

## Mode par lots : modifier plusieurs fichiers

Vous pouvez démarrer le mode par lots depuis :

1. **Le gestionnaire de fichiers**
   - Appuyez sur **Plus d'actions (•••)** > **Sélectionner des fichiers** > **Modifier les tags audio**

2. **L'éditeur de tags**
   - Appuyez sur un fichier > faites défiler vers le bas > appuyez sur **Modifier les fichiers simultanément**

Une fois vos modifications terminées, appuyez sur **Enregistrer**.

## FAQ

{{% details title="Puis-je changer les pochettes d'albums pour les pistes en streaming de Spotify ?" closed="true" %}}
Non. Spotify ne permet pas de changer les illustrations des pistes de son catalogue de streaming. Vous ne pouvez modifier les pochettes d'albums que pour les fichiers locaux ajoutés à votre bibliothèque Spotify.
{{% /details %}}

{{% details title="Evertag est-il gratuit ?" closed="true" %}}
Oui. Evertag est gratuit à télécharger et à utiliser sur iOS et macOS. Il prend en charge plus de 120 tags audio et plus de 30 formats de fichiers.
{{% /details %}}

{{% details title="Quels formats audio Evertag prend-il en charge ?" closed="true" %}}
Evertag prend en charge plus de 30 formats dont MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, WMA et d'autres.
{{% /details %}}

{{% details title="Pourquoi Spotify n'affiche-t-il pas ma pochette d'album mise à jour ?" closed="true" %}}
Forcez la fermeture de l'application Spotify et rouvrez-la. Spotify met en cache les métadonnées et nécessite un redémarrage pour détecter les modifications des fichiers locaux.
{{% /details %}}

{{% details title="Puis-je modifier les tags de plusieurs fichiers à la fois ?" closed="true" %}}
Oui. Evertag prend en charge la modification par lots. Sélectionnez plusieurs fichiers et appuyez sur «Modifier plusieurs fichiers simultanément» pour mettre à jour les tags et illustrations de toutes les pistes sélectionnées en une seule fois.
{{% /details %}}

{{% details title="Dois-je copier les fichiers dans le dossier Spotify ?" closed="true" %}}
Oui. Spotify ne lit les fichiers locaux que depuis son dossier dédié. Copiez ou déplacez vos fichiers musicaux dans le dossier Spotify sur votre appareil, puis activez l'interrupteur Fichiers audio locaux dans les paramètres de Spotify.
{{% /details %}}

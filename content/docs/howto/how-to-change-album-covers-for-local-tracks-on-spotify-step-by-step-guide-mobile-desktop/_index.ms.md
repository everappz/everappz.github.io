## Mod Pengeditan

Evertag menyokong dua mod pengeditan:

- **Mod Fail Tunggal**
  - Leret ke kiri atau kanan melalui karusel karya seni
  - Edit tag teras seperti Tajuk, Artis, Album, Genre, Tahun, dll.

- **Mod Kelompok**
  - Pilih dan edit berbilang fail sekaligus
  - Ketik **Edit beberapa fail secara serentak** di bahagian bawah skrin

## Mod Fail Tunggal: Tag Lanjutan

Untuk mendedahkan **lebih 120 tag tambahan**, tatal ke bahagian bawah skrin editor tag dan ketik **Tunjukkan Tag Lanjutan**.

Tag termasuk:
- Tag MusicBrainz
- Lirik
- Penilaian Kandungan
- Komposer, Pemain, Kompilasi
- BPM, Penilaian, Nombor Cakera/Trek

## Mod Kelompok: Edit Berbilang Fail

Anda boleh memulakan mod kelompok dari:

1. **Pengurus Fail**
   - Ketik **Tindakan lanjut (•••)** > **Pilih fail** > **Edit tag audio**

2. **Editor Tag**
   - Ketik fail > tatal ke bawah > ketik **Edit fail secara serentak**

Selepas pengeditan selesai, ketik **Simpan**.

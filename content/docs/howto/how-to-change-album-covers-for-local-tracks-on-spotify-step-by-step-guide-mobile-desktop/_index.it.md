## Modalità di modifica

Evertag supporta due modalità di modifica:

- **Modalità file singolo**
  - Scorri a sinistra o a destra nel carosello delle copertine
  - Modifica i tag principali come Titolo, Artista, Album, Genere, Anno, ecc.

- **Modalità batch**
  - Seleziona e modifica più file contemporaneamente
  - Tocca **Modifica più file simultaneamente** nella parte inferiore dello schermo

## Modalità file singolo: tag estesi

Per rivelare **oltre 120 tag aggiuntivi**, scorri fino in fondo alla schermata dell'editor dei tag e tocca **Mostra tag estesi**.

I tag includono:
- Tag MusicBrainz
- Testi
- Classificazioni contenuto
- Compositore, Esecutore, Compilation
- BPM, Valutazione, Numeri disco/traccia

## Modalità batch: modifica più file

Puoi avviare la modalità batch da:

1. **Gestore file**
   - Tocca **Altre azioni (•••)** > **Seleziona file** > **Modifica tag audio**

2. **Editor dei tag**
   - Tocca un file > scorri verso il basso > tocca **Modifica file simultaneamente**

Una volta completate le modifiche, tocca **Salva**.

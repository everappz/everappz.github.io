## Mode Pengeditan

Evertag mendukung dua mode pengeditan:

- **Mode File Tunggal**
  - Geser ke kiri atau kanan melalui carousel artwork
  - Edit tag inti seperti Judul, Artis, Album, Genre, Tahun, dll.

- **Mode Batch**
  - Pilih dan edit beberapa file sekaligus
  - Ketuk **Edit beberapa file secara bersamaan** di bagian bawah layar

## Mode File Tunggal: Tag Diperluas

Untuk mengungkapkan **lebih dari 120 tag tambahan**, gulir ke bagian bawah layar editor tag dan ketuk **Tampilkan Tag Diperluas**.

Tag meliputi:
- Tag MusicBrainz
- Lirik
- Rating Konten
- Komposer, Pemain, Kompilasi
- BPM, Rating, Nomor Disc/Trek

## Mode Batch: Edit Beberapa File

Anda dapat memulai mode batch dari:

1. **Manajer File**
   - Ketuk **Tindakan lainnya (•••)** > **Pilih file** > **Edit tag audio**

2. **Editor Tag**
   - Ketuk file > gulir ke bawah > ketuk **Edit file secara bersamaan**

Setelah pengeditan selesai, ketuk **Simpan**.

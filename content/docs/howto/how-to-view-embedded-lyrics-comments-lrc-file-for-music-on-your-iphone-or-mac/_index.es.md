**Resumen:** Evermusic para iPhone y Mac muestra letras incrustadas, etiquetas de comentarios y archivos .lrc externos para cualquier pista de audio. Abre el reproductor, toca Más acciones y luego selecciona Comentarios para ver letras en tres modos: Comentarios, Letras incrustadas y Archivo LRC.

Descubre cómo ver fácilmente las letras incrustadas de tu música en tu iPhone o Mac usando la app **Evermusic**. Ya sea texto, letras, comentarios o archivos LRC, esta guía te mostrará paso a paso cómo acceder y disfrutar de las letras de tus canciones sin problemas.

## Ver letras y comentarios en Evermusic

1. Comienza a reproducir un archivo de audio tocándolo.

2. Abre el reproductor de audio a pantalla completa.  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_8d67c59984f2433d869fe9b8bb56c7be~mv2.png" width="400" >}}

3. Toca el botón **Más acciones** en la esquina superior derecha.  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_a5e7b21c26844b21ab966f40313fcfe6~mv2.png" width="400" >}}

4. Selecciona el elemento del menú **Comentarios**.  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_1031bd68b3d74f35aef6559ffc44aed4~mv2.png" width="400" >}}

5. En la pantalla de **Comentarios de la pista**, desliza para cambiar entre tres modos de visualización:
   - **Comentarios**
   - **Letras incrustadas**
   - **Archivo LRC**

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_b1b435feea43489cacc2cd32793dc1e3~mv2.png" width="400" >}}

## Entendiendo los modos de visualización

- **Modo Comentarios**:  
  Muestra el contenido del campo `COMMENT` de tu archivo de audio.

- **Modo Letras**:  
  Muestra letras incrustadas de las etiquetas. El texto temporizado en formato LRC es compatible aquí. Para editar letras, usa un editor de etiquetas como **Evertag**. También puedes encontrar contenido LRC en sitios web como *Lyricsify*, copiarlo en tus campos de etiquetas y verlo en esta pantalla.

- **Modo Archivo LRC**:  
  En lugar de etiquetar el archivo de audio, coloca un archivo `.lrc` con el **mismo nombre de archivo** en la misma carpeta que el archivo de audio. La app lo detectará y mostrará automáticamente en esta pantalla.

## Habilitar acceso rápido a comentarios desde el reproductor de audio

Puedes cambiar entre las vistas **Portada**, **Cola** y **Comentarios** usando el interruptor de vista en la pantalla del reproductor de audio.

1. Ve a **Ajustes > Reproductor de audio > Personalización**
2. Desplázate hasta **Botones en la pantalla principal**
3. Habilita el elemento del menú **Comentarios**

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_752ed0cb70114be2be2993819b94f2df~mv2.png" width="400" >}}

Regresa al reproductor principal y toca el **botón de cola de audio** varias veces para recorrer las vistas y llegar a la pantalla de **Comentarios**.

## Conclusión

Ver letras incrustadas, comentarios o archivos `.lrc` sincronizados en **Evermusic** es muy fácil. Ya sea que edites etiquetas o uses archivos de letras externos, la app hace que sea conveniente explorar los metadatos y las letras de tu música. ## Preguntas frecuentes

{{% details title="¿Cómo veo las letras incrustadas en mi iPhone?" closed="true" %}}
Abre Evermusic, reproduce una canción, toca Más acciones en el reproductor a pantalla completa y selecciona Comentarios. Desliza a la pestaña Letras incrustadas para ver las letras almacenadas en las etiquetas del archivo de audio.
{{% /details %}}

{{% details title="¿Qué es un archivo LRC y cómo funciona?" closed="true" %}}
Un archivo LRC es un archivo de texto que contiene letras de canciones con marcas de tiempo. Cuando se coloca en la misma carpeta que un archivo de audio con el mismo nombre de archivo, Evermusic lo lee y muestra letras sincronizadas que se desplazan durante la reproducción.
{{% /details %}}

{{% details title="¿Puedo agregar letras a mis archivos de música en iPhone?" closed="true" %}}
Sí. Usa la app Evertag para editar etiquetas ID3 y agregar o actualizar letras incrustadas directamente en tu iPhone. Puedes pegar texto temporizado en formato LRC para letras sincronizadas.
{{% /details %}}

{{% details title="¿Evermusic soporta letras sincronizadas (temporizadas)?" closed="true" %}}
Sí. Evermusic soporta letras temporizadas en formato LRC, tanto cuando están incrustadas en etiquetas de audio como cuando se proporcionan como un archivo `.lrc` separado.
{{% /details %}}

{{% details title="¿Qué formatos de audio soportan letras incrustadas?" closed="true" %}}
MP3, FLAC, AAC, M4A, OGG y la mayoría de otros formatos comunes soportan letras incrustadas a través de sus respectivos estándares de etiquetas.
{{% /details %}}

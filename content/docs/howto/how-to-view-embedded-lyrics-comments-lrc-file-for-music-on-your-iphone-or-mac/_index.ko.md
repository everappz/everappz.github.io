**요약:** iPhone 및 Mac용 Evermusic은 모든 오디오 트랙의 내장 가사, 댓글 태그 및 외부 .lrc 파일을 표시합니다. 플레이어를 열고 **추가 작업**을 탭한 다음 **댓글**을 선택하면 댓글, 내장 가사, LRC 파일의 세 가지 모드로 가사를 볼 수 있습니다.

**Evermusic** 앱을 사용하여 iPhone 또는 Mac에서 음악의 내장 가사를 쉽게 보는 방법을 알아보세요. 텍스트, 가사, 댓글 또는 LRC 파일 등 이 가이드는 노래 가사에 접근하고 즐기는 방법을 단계별로 보여줍니다.

## Evermusic에서 가사 및 댓글 보기

## 표시 모드 이해하기

- **댓글 모드**:  
  오디오 파일의 `COMMENT` 필드 내용을 표시합니다.

- **가사 모드**:  
  태그에서 내장 가사를 표시합니다. LRC 형식의 시간 동기화 텍스트가 여기서 지원됩니다. 가사를 편집하려면 **Evertag**와 같은 태그 편집기를 사용하세요. *Lyricsify*와 같은 웹사이트에서 LRC 콘텐츠를 찾아 태그 필드에 복사하고 이 화면에서 볼 수도 있습니다.

- **LRC 파일 모드**:  
  오디오 파일에 태그를 지정하는 대신 오디오 파일과 동일한 폴더에 **같은 파일 이름**의 `.lrc` 파일을 배치합니다. 앱이 자동으로 감지하여 이 화면에 표시합니다.

## 오디오 플레이어에서 빠른 댓글 접근 활성화

오디오 플레이어 화면의 보기 전환 버튼을 사용하여 **아트워크**, **대기열** 및 **댓글** 보기를 전환할 수 있습니다.

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_ec5cdb0786f246fe85198cc14564d0ee~mv2.png" width="400" >}}

1. **설정 > 오디오 플레이어 > 개인화**로 이동합니다
2. **메인 화면의 버튼**까지 스크롤합니다
3. **댓글** 메뉴 항목을 활성화합니다

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_752ed0cb70114be2be2993819b94f2df~mv2.png" width="400" >}}

메인 플레이어로 돌아가서 **오디오 대기열 버튼**을 여러 번 탭하여 보기를 전환하고 **댓글** 화면에 도달합니다.

## 결론

**Evermusic**에서 내장 가사, 댓글 또는 동기화된 `.lrc` 파일을 보는 것은 간단합니다. 태그를 편집하든 외부 가사 파일을 사용하든 앱은 음악의 메타데이터와 가사를 탐색하기 편리하게 해줍니다. 향상된 제어를 위해 청취 중 더 빠른 접근을 위해 **빠른 댓글 보기**를 활성화하는 것을 잊지 마세요.

## FAQ

{{% details title="iPhone에서 내장 가사를 어떻게 볼 수 있나요?" closed="true" %}}
Evermusic을 열고 노래를 재생한 후 전체 화면 플레이어에서 추가 작업을 탭하고 댓글을 선택합니다. 내장 가사 탭으로 스와이프하여 오디오 파일의 태그에 저장된 가사를 봅니다.
{{% /details %}}

{{% details title="LRC 파일이란 무엇이며 어떻게 작동하나요?" closed="true" %}}
LRC 파일은 타임스탬프가 포함된 노래 가사를 담고 있는 텍스트 파일입니다. 같은 파일 이름으로 오디오 파일과 같은 폴더에 배치하면 Evermusic이 이를 읽어 재생 중 스크롤되는 동기화된 가사를 표시합니다.
{{% /details %}}

{{% details title="iPhone에서 음악 파일에 가사를 추가할 수 있나요?" closed="true" %}}
네. Evertag 앱을 사용하여 ID3 태그를 편집하고 iPhone에서 직접 내장 가사를 추가하거나 업데이트할 수 있습니다. 동기화된 가사를 위해 시간 동기화 LRC 형식 텍스트를 붙여넣을 수 있습니다.
{{% /details %}}

{{% details title="Evermusic은 동기화된(시간 동기화) 가사를 지원하나요?" closed="true" %}}
네. Evermusic은 LRC 형식의 시간 동기화 가사를 지원하며, 오디오 태그에 내장된 경우와 별도의 `.lrc` 파일로 제공되는 경우 모두 지원합니다.
{{% /details %}}

{{% details title="어떤 오디오 형식이 내장 가사를 지원하나요?" closed="true" %}}
MP3, FLAC, AAC, M4A, OGG 및 대부분의 다른 일반적인 형식은 해당 태그 표준을 통해 내장 가사를 지원합니다.
{{% /details %}}

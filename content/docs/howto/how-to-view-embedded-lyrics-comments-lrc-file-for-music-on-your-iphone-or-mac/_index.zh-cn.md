**TL;DR:** Evermusic for iPhone 和 Mac 可显示任何音频曲目的嵌入式歌词、评论标签和外部 .lrc 文件。打开播放器，点击**更多操作**，然后选择**评论**，即可以三种模式查看歌词：评论、嵌入式歌词和 LRC 文件。

了解如何使用 **Evermusic** 应用在 iPhone 或 Mac 上轻松查看音乐的嵌入式歌词。无论是文本、歌词、评论还是 LRC 文件，本指南都将逐步向您展示如何访问和欣赏您的歌词。

## 在 Evermusic 中查看歌词和评论

1. 点击音频文件开始播放。

2. 打开全屏音频播放器。

3. 点击右上角的**更多操作**按钮。  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_a5e7b21c26844b21ab966f40313fcfe6~mv2.png" width="400" >}}

4. 选择**评论**菜单项。

## 了解显示模式

- **评论模式**：  
  显示音频文件 `COMMENT` 字段的内容。

- **歌词模式**：  
  显示标签中的嵌入式歌词。此处支持 LRC 格式的定时文本。要编辑歌词，请使用 **Evertag** 等标签编辑器。您还可以在 *Lyricsify* 等网站上找到 LRC 内容，将其复制到标签字段中，然后在此屏幕上查看。

- **LRC 文件模式**：  
  无需为音频文件添加标签，只需将与音频文件**同名**的 `.lrc` 文件放在同一文件夹中。应用将自动检测并在此屏幕上显示。

您可以使用音频播放器屏幕上的视图切换器在**封面**、**队列**和**评论**视图之间切换。

1. 前往**设置 > 音频播放器 > 个性化**

## 总结

在 **Evermusic** 中查看嵌入式歌词、评论或同步的 `.lrc` 文件非常简单。无论您是编辑标签还是使用外部歌词文件，该应用都能让您方便地探索音乐的元数据和歌词。为了获得更好的控制，别忘了启用**快速评论视图**，以便在听音乐时更快地访问。

## 常见问题

{{% details title="如何在 iPhone 上查看嵌入式歌词？" closed="true" %}}
打开 Evermusic，播放一首歌曲，在全屏播放器中点击更多操作，然后选择评论。滑动到嵌入式歌词标签页，查看存储在音频文件标签中的歌词。
{{% /details %}}

{{% details title="什么是 LRC 文件，它是如何工作的？" closed="true" %}}
LRC 文件是包含定时歌词的文本文件。当与音频文件放在同一文件夹中且具有相同的文件名时，Evermusic 会读取它并显示在播放过程中滚动的同步歌词。
{{% /details %}}

{{% details title="我可以在 iPhone 上为音乐文件添加歌词吗？" closed="true" %}}
可以。使用 Evertag 应用编辑 ID3 标签，直接在 iPhone 上添加或更新嵌入式歌词。您可以粘贴定时 LRC 格式的文本以获取同步歌词。
{{% /details %}}

{{% details title="Evermusic 支持同步（定时）歌词吗？" closed="true" %}}
支持。Evermusic 支持 LRC 格式的定时歌词，无论是嵌入在音频标签中还是作为单独的 `.lrc` 文件提供。
{{% /details %}}

{{% details title="哪些音频格式支持嵌入式歌词？" closed="true" %}}
MP3、FLAC、AAC、M4A、OGG 和大多数其他常见格式都通过各自的标签标准支持嵌入式歌词。
{{% /details %}}

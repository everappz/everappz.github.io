**要約:** iPhoneおよびMac用のEvermusicは、任意のオーディオトラックの埋め込み歌詞、コメントタグ、外部.lrcファイルを表示します。プレーヤーを開き、**その他のアクション**をタップしてから**コメント**を選択すると、コメント、埋め込み歌詞、LRCファイルの3つのモードで歌詞を表示できます。

**Evermusic**アプリを使用して、iPhoneまたはMacで音楽の埋め込み歌詞を簡単に表示する方法をご紹介します。テキスト、歌詞、コメント、LRCファイルなど、このガイドでは曲の歌詞にアクセスして楽しむ方法をステップバイステップでお見せします。

## Evermusicで歌詞とコメントを表示する

1. オーディオファイルをタップして再生を開始します。

2. フルスクリーンのオーディオプレーヤーを開きます。  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_8d67c59984f2433d869fe9b8bb56c7be~mv2.png" width="400" >}}

3. 右上隅の**その他のアクション**ボタンをタップします。  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_a5e7b21c26844b21ab966f40313fcfe6~mv2.png" width="400" >}}

4. **コメント**メニュー項目を選択します。  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_1031bd68b3d74f35aef6559ffc44aed4~mv2.png" width="400" >}}

5. **トラックコメント**画面で、スワイプして3つの表示モードを切り替えます：
   - **コメント**
   - **埋め込み歌詞**
   - **LRCファイル**

## 表示モードの理解

- **コメントモード**：  
  オーディオファイルの`COMMENT`フィールドの内容を表示します。

- **歌詞モード**：  
  タグから埋め込み歌詞を表示します。LRC形式のタイムド テキストがここでサポートされています。歌詞を編集するには、**Evertag**のようなタグエディタを使用してください。*Lyricsify*などのウェブサイトでLRCコンテンツを見つけ、タグフィールドにコピーして、この画面で表示することもできます。

- **LRCファイルモード**：  
  オーディオファイルにタグ付けする代わりに、オーディオファイルと同じフォルダに**同じファイル名**の`.lrc`ファイルを配置します。アプリが自動的に検出し、この画面に表示します。

## オーディオプレーヤーからのクイックコメントアクセスを有効にする

オーディオプレーヤー画面のビュー切り替えボタンを使用して、**アートワーク**、**キュー**、**コメント**のビューを切り替えることができます。

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_ec5cdb0786f246fe85198cc14564d0ee~mv2.png" width="400" >}}

1. **設定 > オーディオプレーヤー > パーソナライズ**に移動します
2. **メイン画面のボタン**までスクロールします
3. **コメント**メニュー項目を有効にします

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_752ed0cb70114be2be2993819b94f2df~mv2.png" width="400" >}}

メインプレーヤーに戻り、**オーディオキューボタン**を数回タップしてビューを切り替え、**コメント**画面に到達します。

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_d17d8626b2164614a31d75b96c650ec8~mv2.png" width="400" >}}

## まとめ

**Evermusic**で埋め込み歌詞、コメント、同期された`.lrc`ファイルを表示するのは簡単です。タグを編集する場合でも外部の歌詞ファイルを使用する場合でも、アプリは音楽のメタデータや歌詞の探索を便利にします。より高度な操作のために、リスニング中の素早いアクセスのために**クイックコメントビュー**を有効にすることをお忘れなく。

## よくある質問

{{% details title="iPhoneで埋め込み歌詞を表示するにはどうすればいいですか？" closed="true" %}}
Evermusicを開き、曲を再生し、フルスクリーンプレーヤーでその他のアクションをタップしてコメントを選択します。埋め込み歌詞タブにスワイプして、オーディオファイルのタグに保存された歌詞を表示します。
{{% /details %}}

{{% details title="LRCファイルとは何ですか？どのように機能しますか？" closed="true" %}}
LRCファイルは、タイムスタンプ付きの歌詞を含むテキストファイルです。同じファイル名でオーディオファイルと同じフォルダに配置すると、Evermusicがそれを読み取り、再生中にスクロールする同期歌詞を表示します。
{{% /details %}}

{{% details title="iPhoneで音楽ファイルに歌詞を追加できますか？" closed="true" %}}
はい。Evertagアプリを使用してID3タグを編集し、iPhone上で直接埋め込み歌詞を追加または更新できます。同期歌詞のためにタイムドLRC形式のテキストを貼り付けることができます。
{{% /details %}}

{{% details title="Evermusicは同期（タイムド）歌詞をサポートしていますか？" closed="true" %}}
はい。EvermusicはLRC形式のタイムド歌詞をサポートしています。オーディオタグに埋め込まれた場合と、別の`.lrc`ファイルとして提供された場合の両方に対応しています。
{{% /details %}}

{{% details title="どのオーディオ形式が埋め込み歌詞をサポートしていますか？" closed="true" %}}
MP3、FLAC、AAC、M4A、OGG、およびその他のほとんどの一般的な形式は、それぞれのタグ標準を通じて埋め込み歌詞をサポートしています。
{{% /details %}}

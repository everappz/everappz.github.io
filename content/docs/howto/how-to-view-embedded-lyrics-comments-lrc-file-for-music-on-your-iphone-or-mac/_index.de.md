**Zusammenfassung:** Evermusic für iPhone und Mac zeigt eingebettete Songtexte, Kommentar-Tags und externe .lrc-Dateien für jeden Audiotitel an. Öffnen Sie den Player, tippen Sie auf Weitere Aktionen und wählen Sie dann Kommentare, um Songtexte in drei Modi anzuzeigen: Kommentare, Eingebettete Songtexte und LRC-Datei.

Entdecken Sie, wie Sie eingebettete Songtexte für Ihre Musik auf Ihrem iPhone oder Mac mit der **Evermusic**-App einfach anzeigen können. Ob Text, Songtexte, Kommentare oder LRC-Dateien – dieser Leitfaden zeigt Ihnen Schritt für Schritt, wie Sie auf Ihre Songtexte zugreifen und sie nahtlos genießen können.

## Songtexte und Kommentare in Evermusic anzeigen

1. Beginnen Sie mit der Wiedergabe einer Audiodatei, indem Sie darauf tippen.

3. Tippen Sie auf die Schaltfläche **Weitere Aktionen** in der oberen rechten Ecke.

## Die Anzeigemodi verstehen

- **Kommentare-Modus**:  
  Zeigt den Inhalt aus dem `COMMENT`-Feld Ihrer Audiodatei an.

- **Songtexte-Modus**:  
  Zeigt eingebettete Songtexte aus Tags an. Zeitgesteuerter Text im LRC-Format wird hier unterstützt. Um Songtexte zu bearbeiten, verwenden Sie einen Tag-Editor wie **Evertag**. Sie können LRC-Inhalte auch auf Websites wie *Lyricsify* finden, in Ihre Tag-Felder kopieren und auf diesem Bildschirm anzeigen.

- **LRC-Datei-Modus**:  
  Anstatt die Audiodatei zu taggen, platzieren Sie eine `.lrc`-Datei mit dem **gleichen Dateinamen** im selben Ordner wie die Audiodatei. Die App erkennt und zeigt sie automatisch auf diesem Bildschirm an.

## Schnellzugriff auf Kommentare vom Audio-Player aktivieren

Sie können zwischen den Ansichten **Artwork**, **Warteschlange** und **Kommentare** mit dem Ansichtsumschalter auf dem Audio-Player-Bildschirm wechseln.

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_ec5cdb0786f246fe85198cc14564d0ee~mv2.png" width="400" >}}

1. Gehen Sie zu **Einstellungen > Audio-Player > Personalisierung**
2. Scrollen Sie zu **Schaltflächen auf dem Hauptbildschirm**
3. Aktivieren Sie den Menüpunkt **Kommentare**

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_752ed0cb70114be2be2993819b94f2df~mv2.png" width="400" >}}

Kehren Sie zum Hauptplayer zurück und tippen Sie mehrmals auf die **Audio-Warteschlangen-Schaltfläche**, um die Ansichten zu durchlaufen und den **Kommentare**-Bildschirm zu erreichen.

{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_d17d8626b2164614a31d75b96c650ec8~mv2.png" width="400" >}}

## Fazit

Das Anzeigen von eingebetteten Songtexten, Kommentaren oder synchronisierten `.lrc`-Dateien in **Evermusic** ist mühelos. Ob Sie Tags bearbeiten oder externe Songtextdateien verwenden, die App macht es bequem, die Metadaten und Songtexte Ihrer Musik zu erkunden. Für verbesserte Kontrolle vergessen Sie nicht, die **Schnelle Kommentaransicht** für schnelleren Zugriff während des Hörens zu aktivieren.

## Häufig gestellte Fragen

{{% details title="Wie zeige ich eingebettete Songtexte auf meinem iPhone an?" closed="true" %}}
Öffnen Sie Evermusic, spielen Sie einen Song ab, tippen Sie im Vollbild-Player auf Weitere Aktionen und wählen Sie Kommentare. Wischen Sie zur Registerkarte Eingebettete Songtexte, um die in den Tags der Audiodatei gespeicherten Songtexte zu sehen.
{{% /details %}}

{{% details title="Was ist eine LRC-Datei und wie funktioniert sie?" closed="true" %}}
Eine LRC-Datei ist eine Textdatei mit zeitgestempelten Songtexten. Wenn sie im selben Ordner wie eine Audiodatei mit dem gleichen Dateinamen platziert wird, liest Evermusic sie und zeigt synchronisierte Songtexte an, die während der Wiedergabe scrollen.
{{% /details %}}

{{% details title="Kann ich Songtexte zu meinen Musikdateien auf dem iPhone hinzufügen?" closed="true" %}}
Ja. Verwenden Sie die Evertag-App, um ID3-Tags zu bearbeiten und eingebettete Songtexte direkt auf Ihrem iPhone hinzuzufügen oder zu aktualisieren. Sie können zeitgesteuerten Text im LRC-Format für synchronisierte Songtexte einfügen.
{{% /details %}}

{{% details title="Unterstützt Evermusic synchronisierte (zeitgesteuerte) Songtexte?" closed="true" %}}
Ja. Evermusic unterstützt zeitgesteuerte Songtexte im LRC-Format, sowohl wenn sie in Audio-Tags eingebettet sind als auch wenn sie als separate `.lrc`-Datei bereitgestellt werden.
{{% /details %}}

{{% details title="Welche Audioformate unterstützen eingebettete Songtexte?" closed="true" %}}
MP3, FLAC, AAC, M4A, OGG und die meisten anderen gängigen Formate unterstützen eingebettete Songtexte über ihre jeweiligen Tag-Standards.
{{% /details %}}

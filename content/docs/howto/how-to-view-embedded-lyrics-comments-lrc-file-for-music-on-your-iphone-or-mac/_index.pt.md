**TL;DR:** Evermusic para iPhone e Mac exibe letras incorporadas, tags de comentários e arquivos .lrc externos para qualquer faixa de áudio. Abra o reprodutor, toque em **Mais ações** e selecione **Comentários** para visualizar letras em três modos: Comentários, Letras Incorporadas e Arquivo LRC.

Descubra como visualizar facilmente letras incorporadas para suas músicas no iPhone ou Mac usando o aplicativo **Evermusic**. Seja texto, letras, comentários ou arquivos LRC, este guia mostrará passo a passo como acessar e aproveitar suas letras de músicas perfeitamente.

## Visualizar letras e comentários no Evermusic

1. Comece a reproduzir um arquivo de áudio tocando nele.

2. Abra o reprodutor de áudio em tela cheia.

3. Toque no botão **Mais ações** no canto superior direito.  
{{< figure src="/docs/howto/how-to-view-embedded-lyrics-comments-lrc-file-for-music-on-your-iphone-or-mac/21260c_a5e7b21c26844b21ab966f40313fcfe6~mv2.png" width="400" >}}

4. Selecione o item de menu **Comentários**.

5. Na tela **Comentários da Faixa**, deslize para alternar entre três modos de exibição:
   - **Comentários**
   - **Letras Incorporadas**

## Entendendo os modos de exibição

- **Modo Comentários**:  
  Mostra o conteúdo do campo `COMMENT` do seu arquivo de áudio.

- **Modo Letras**:  
  Exibe letras incorporadas das tags. Texto cronometrado no formato LRC é suportado aqui. Para editar letras, use um editor de tags como o **Evertag**. Você também pode encontrar conteúdo LRC em sites como *Lyricsify*, copiá-lo para os campos de tags e visualizá-lo nesta tela.

- **Modo Arquivo LRC**:  
  Em vez de adicionar tags ao arquivo de áudio, coloque um arquivo `.lrc` com o **mesmo nome de arquivo** na mesma pasta do arquivo de áudio. O aplicativo detectará e exibirá automaticamente nesta tela.

## Ativar acesso rápido aos comentários pelo reprodutor de áudio

Você pode alternar entre as visualizações **Capa**, **Fila** e **Comentários** usando o alternador de visualização na tela do reprodutor de áudio.

1. Vá para **Configurações > Reprodutor de Áudio > Personalização**
2. Role até **Botões na Tela Principal**
3. Ative o item de menu **Comentários**

Retorne ao reprodutor principal e toque no **botão de fila de áudio** várias vezes para alternar visualizações e chegar à tela **Comentários**.

## Conclusão

Visualizar letras incorporadas, comentários ou arquivos `.lrc` sincronizados no **Evermusic** é simples. Seja editando tags ou usando arquivos de letras externos, o aplicativo facilita a exploração dos metadados e letras da sua música. Para maior controle, não esqueça de ativar a **Visualização Rápida de Comentários** para acesso mais rápido enquanto ouve.

## Perguntas frequentes

{{% details title="Como visualizo letras incorporadas no meu iPhone?" closed="true" %}}
Abra o Evermusic, reproduza uma música, toque em Mais ações no reprodutor em tela cheia e selecione Comentários. Deslize até a aba Letras Incorporadas para ver as letras armazenadas nas tags do arquivo de áudio.
{{% /details %}}

{{% details title="O que é um arquivo LRC e como funciona?" closed="true" %}}
Um arquivo LRC é um arquivo de texto contendo letras de músicas cronometradas. Quando colocado na mesma pasta de um arquivo de áudio com o mesmo nome, o Evermusic o lê e exibe letras sincronizadas que rolam durante a reprodução.
{{% /details %}}

{{% details title="Posso adicionar letras aos meus arquivos de música no iPhone?" closed="true" %}}
Sim. Use o aplicativo Evertag para editar tags ID3 e adicionar ou atualizar letras incorporadas diretamente no seu iPhone. Você pode colar texto no formato LRC cronometrado para letras sincronizadas.
{{% /details %}}

{{% details title="O Evermusic suporta letras sincronizadas (cronometradas)?" closed="true" %}}
Sim. O Evermusic suporta letras cronometradas no formato LRC, tanto quando incorporadas nas tags de áudio quanto quando fornecidas como um arquivo `.lrc` separado.
{{% /details %}}

{{% details title="Quais formatos de áudio suportam letras incorporadas?" closed="true" %}}
MP3, FLAC, AAC, M4A, OGG e a maioria dos outros formatos comuns suportam letras incorporadas por meio de seus respectivos padrões de tags.
{{% /details %}}

**要約：** SoundCloud公式iOSアプリはChromecastをサポートしていません。Soundyは、SoundCloudの音楽をGoogle ChromecastやAirPlayデバイスにストリーミングできる無料のiOSアプリです。10バンドイコライザー、プレイリスト、速度制御、オーディオブックマークも含まれています。

![Soundy：SoundCloudとChromecast](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_5e389cfe328f483d9475c1e59e2a4983~mv2.jpeg)

SoundCloudユーザーはiOSアプリにChromecastサポートがいつ追加されるかを熱心に尋ねていますが、これまでSoundCloudからの公式アップデートはありません。多くのiPhoneユーザーは、お気に入りのSoundCloudトラックをChromecastにストリーミングできずにいます。

しかし、解決策があります！Soundyは、Chromecastサポートが組み込まれたフル機能のSoundCloudオーディオプレーヤーです。最も素晴らしいことに、無料でiPhoneやiPadですぐに使えます。Soundyを使えば、Google ChromecastやAirPlayを通じてSoundCloudをテレビにシームレスに接続できます。このチュートリアルでは、Soundyの素晴らしい機能のいくつかを紹介し、iOSデバイスを使用してSoundCloudからGoogle Chromecastに音楽をストリーミングするプロセスをご案内します。

## ステップ1：Soundyをインストール

Soundyは App Storeから直接ダウンロードできます。イコライザー、速度制御、オーディオブックマーク、スリープタイマーなど、さまざまな便利な機能を提供しています。しかし、その際立った機能は間違いなくGoogle Chromecastサポートです。

## ステップ2：SoundCloudアカウントを接続（オプション）

![Soundy：アカウント接続](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_58d8d9c048c04662a33ebc6e43f8548e~mv2.png)

Soundyをインストールしたら、SoundCloudアカウントを接続することを選択できます。これにより、すべてのデバイスでSoundCloudの音楽にアクセスできます。

- アカウントタブでは、アップロードしたすべてのトラックが見つかります。
- お気に入りタブには、すべてのお気に入り音楽が表示されます。
- プレイリストタブには、自分のプレイリストと他のユーザーが作成した「いいね」したプレイリストが含まれています。画面上部のセグメントコントロールを使用して、自分のプレイリストと「いいね」したプレイリストを切り替えることができます。

ただし、SoundCloudアカウントなしでもプレイリストを作成したりお気に入りを管理したりできます。すべての変更はアプリ内にローカルに保存されます。

## ステップ3：プレイリストとお気に入りを管理

![Soundy：マイアカウント](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_23207ae56da242e2bb5f8bbe3a168004~mv2.png)

Soundyでは、プレイリストの編集、曲の順序の変更、トラックのお気に入りへの追加が自由にできます。友達との音楽共有は、オプションボタンをタップして共有メニューを選択するだけの簡単さです。

## ステップ4：新しい音楽を発見

![Soundy：SoundCloud検索](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_f79ced674597479a8158d8561ce585fe~mv2.png)

特定の曲やアルバムが決まっている場合は、Soundy内で簡単に検索できます。トップチャートタブを開き、画面上部の検索フィールドを使用します。お気に入りの曲、アルバム、プレイリストを検索できます。検索を絞り込むには、セグメントコントロールでコンテンツタイプを選択し、検索ボタンをタップします。気に入ったトラックが見つかったら、プレイリストやお気に入りに追加したり、オプションボタンで友達と共有したりできます。

## ステップ5：再生をカスタマイズ

![Soundy：新しいブックマーク](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_f9a88d7a8ffa4aefad5d2fad6a79ef49~mv2.png)

Soundyは、特にオーディオブック愛好家向けの便利な再生機能を提供しています。再生速度を0.2xから2.0xに変更したり、オーディオブックマークを作成して再生位置を保存し、中断したところから聴き続けたりできます。

## ステップ6：イコライザーを楽しむ

![Soundy：イコライザー](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_309abcf435674abb8aaa2fc99181d3c1~mv2.png)

Soundyの際立った機能の1つは、iTunesプリセット付きの10バンドイコライザーです。EQ周波数を-12から+12デシベルに調整してオーディオ体験を微調整できます。アコースティック、バスブースター、ジャズなどの組み込みプリセットから選択できます。

## ステップ7：Chromecastにストリーミング

![Soundy：Chromecastセットアップ](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_2cadb94e9a4d487a8154bb3ceb9f94fc~mv2.jpg)

最後に、SoundCloudの音楽をChromecastにストリーミングする方法をご紹介します。

- まず、[こちら](https://support.google.com/chromecast/answer/6260600?hl=en)の手順に従ってChromecastをセットアップします。
- Soundyアプリでプレーヤーを起動し、プレーヤー画面を開きます。
- 音量インジケーターの近くにあるキャストボタンを探します。
- キャストボタンをタップし、リストから利用可能なChromecastデバイスを選択します。

![Soundy：ロック画面](/docs/howto/soundy-stream-your-music-from-soundcloud-to-chromecast/21260c_77f092dea1794459bcd59f23c6d3988d~mv2.jpg)

接続が確立されると、接続されたGoogle Castデバイスで音楽の再生が開始されます。プレーヤー画面の音量スライダーを使用してChromecastの音量を制御でき、iPhoneのロック画面で曲を切り替えたりスクラバーを使用したりすることもできます。

Soundyで、お気に入りのSoundCloudトラックをChromecastに手軽にストリーミングしましょう！

## よくある質問

{{% details title="SoundCloud公式アプリはiOSでChromecastをサポートしていますか？" closed="true" %}}
いいえ。現時点では、SoundCloud公式iOSアプリはGoogle Chromecastをサポートしていません。Soundyは、iPhoneとiPadでSoundCloudにChromecastサポートを追加する無料の代替アプリです。
{{% /details %}}

{{% details title="Soundyは無料でダウンロードできますか？" closed="true" %}}
はい。SoundyはApp Storeから無料でダウンロードでき、追加費用なしでChromecastサポートが含まれています。
{{% /details %}}

{{% details title="SoundCloudアカウントなしでSoundyを使用できますか？" closed="true" %}}
はい。SoundCloudアカウントを接続せずに、音楽の検索、ローカルプレイリストの作成、Chromecastへのストリーミングができます。アカウントを接続すると、アップロードしたトラック、お気に入り、プレイリストにアクセスできます。
{{% /details %}}

{{% details title="SoundyはAirPlayをサポートしていますか？" closed="true" %}}
はい。Google Chromecastに加えて、SoundyはApple TVおよび互換性のあるスピーカーへのAirPlayストリーミングをサポートしています。
{{% /details %}}

{{% details title="Soundyはどのようなオーディオ品質機能を提供していますか？" closed="true" %}}
Soundyには、iTunesプリセット付きの10バンドイコライザー、0.2xから2.0xの再生速度制御、オーディオブックマーク、スリープタイマーが含まれています。
{{% /details %}}

{{% details title="iPhoneのロック画面からChromecastの再生を制御できますか？" closed="true" %}}
はい。Chromecastに接続すると、iPhoneのロック画面から直接、曲の切り替え、スクラバーの調整、再生の制御ができます。
{{% /details %}}

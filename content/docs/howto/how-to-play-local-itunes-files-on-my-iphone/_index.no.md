**Sammendrag:** Overfør musikk fra datamaskinen til iPhone ved hjelp av en av tre metoder: **iTunes/Finder File Sharing** (USB-kabel), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (trådløst, ingen kabel nødvendig), eller **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (strøm direkte uten å kopiere). Spill deretter med [Evermusic](/products/evermusic) eller [Flacbox](/products/flacbox).

Å administrere musikkfilene dine mellom datamaskinen og iOS-enheter har aldri vært enklere. Vi har beskrevet tre enkle metoder -- iTunes File Sharing, WiFi Drive og SMB-protokollen -- for å sikre en jevn musikkoverføring og sømløs avspilling.

## iTunes File Sharing

iTunes File Sharing gjør det enkelt å kopiere lydfiler mellom datamaskinen og iOS-appene dine på iPhone, iPad eller iPod touch. Denne tilnærmingen er perfekt for de som ønsker å nyte sine favorittalbum, artister og spor fra Mac/PC på iOS-enheten sin.

**Før du begynner**

- Alternativt kan du bruke [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) til å trådløst kopiere eller flytte lydfiler mellom datamaskinen og apper som Evermusic og Flacbox. Sørg for at datamaskinen og iOS-enheten er koblet til samme WiFi-nettverk, og du kan enkelt overføre filer mellom disse appene.
- En annen teknologi å vurdere er [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), som lar deg strømme musikk direkte fra hjemmedatamaskinen til iOS-enheten, uten å måtte kopiere lydfiler.

## macOS Catalina eller nyere

Med macOS Catalina eller en nyere versjon kan du bruke Finder til å dele filer mellom iOS- og iPadOS-enhetene dine og Macen.

**Krav**  

- macOS Catalina eller nyere
- En iPhone, iPad eller iPod touch med den nyeste iOS- eller iPadOS-versjonen
- En iOS- eller iPadOS-app som er kompatibel med File Sharing

### Hvordan bruke Finder for iTunes File Sharing macOS Catalina eller nyere

1. Åpne et Finder-vindu.
2. Koble iPhone, iPad eller iPod touch til datamaskinen med en USB-kabel.
3. Velg enheten din i Finder.
4. Klikk på "Filer"-fanen for å se en liste over apper som kan dele filer. Hvis du ikke ser denne delen, betyr det at enheten din ikke støtter fildelingsapper.
5. Klikk på trekanten ved siden av en app for å se tilgjengelige filer for deling.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Kopiere filer fra Mac til enhet

1. Velg filene du vil kopiere, og sørg for at de er kompatible med appen. Se appens brukerveiledning for støttede formater.
2. Dra og slipp filene til app-ikonet på enheten din. Finder tar seg av kopieringen av filene til enheten.

#### Kopiere filer fra enhet til Mac

1. Velg filene du ønsker å kopiere.
2. Dra filene til en mappe på datamaskinen, og Finder utfører kopieringsoperasjonen.

## macOS eldre enn Catalina

Hvis du bruker en eldre macOS-versjon, kan du bruke iTunes for File Sharing.

**Krav**  

- Sørg for at du har den nyeste iTunes-versjonen.
- iPhone, iPad eller iPod touch bør kjøre den nyeste iOS-versjonen.
- Hvis du bruker en Windows-PC, sørg for at den er oppdatert til minst Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Hvordan bruke iTunes File Sharing for macOS-versjoner eldre enn Catalina 

1. Åpne iTunes på Mac eller PC.
2. Koble iPhone eller iPad til datamaskinen via USB lightning-kabel.
3. Klikk på iOS-enheten din i iTunes.
4. Naviger til "Apper"-delen i venstre sidefelt.
5. Rull ned til "File Sharing"-delen nederst på siden.
6. Velg appen for å se hvilke filer som er tilgjengelige for deling på enheten din.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Kopiere filer fra datamaskinen til appen   

1. Åpne iTunes og velg appen fra listen i File Sharing-delen.
2. For å kopiere filer til enheten, dra og slipp dem fra en mappe eller et vindu til "Dokumenter"-listen.
3. Alternativt kan du klikke "Legg til" i "Dokumenter"-listen i iTunes, finne lydfilene på datamaskinen og klikke "Legg til." iTunes kopierer disse musikkfilene til "Dokumenter"-mappen i den valgte appen på enheten. Sørg for å velge lydfiler som er kompatible med appen; se brukerveiledningen for støttede formater.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Kopiere lydfiler fra en iOS-app til datamaskinen   

1. Velg appen fra listen i iTunes' File Sharing-del.
2. For å kopiere filer til datamaskinen, dra dem fra "Dokumenter"-listen til en mappe eller et vindu på datamaskinen.
3. Alternativt kan du velge lydfilene du vil kopiere til datamaskinen fra "Dokumenter"-listen, klikke "Lagre til" og velge målmappen på datamaskinen.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Tilgang til delte lydfiler   

Når du har kopiert filer fra datamaskinen til iOS-enheten, kan du enkelt spille dem av.

- Åpne appen du valgte for File Sharing, som Evermusic eller Flacbox.
- Trykk på "Lokale filer"-delen for å se listen over tilgjengelige filer.
- Trykk på en fil for å starte avspilling.

## Slette delte filer fra iOS-enheten   

For å fjerne lydfiler som er kopiert fra datamaskinen til en iOS-app med File Sharing, følg disse trinnene:

#### Bruke iTunes  

1. Åpne iTunes på Mac/PC.
2. Koble iOS-enheten til datamaskinen med USB-kabelen.
3. Velg enheten din i iTunes.
4. Klikk "Apper" i venstre sidefelt og rull ned til File Sharing-delen.
5. Velg appen med filen du vil slette, som Evermusic eller Flacbox.
6. Velg lydfilen du ønsker å fjerne fra "Dokumenter"-listen, og trykk deretter "Slette"-tasten på tastaturet.
7. Bekreft slettingen i iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Bruke den innebygde filbehandleren  

1. Åpne "Lokale filer"-delen.
2. Trykk på "Flere handlinger"-knappen øverst til høyre, og velg "Velge"-menyelementet.
3. Velg filene du vil slette.
4. Trykk på "Slette"-handlingen. Hvis denne handlingen ikke er umiddelbart synlig, trykk på "Flere handlinger"-menyen.
5. Bekreft slettingen i popup-meldingen.

## FAQ

{{% details title="Hva er den raskeste måten å overføre musikk til iPhone?" closed="true" %}}
iTunes/Finder File Sharing over USB er den raskeste metoden for store musikkbiblioteker. For mindre overføringer er WiFi Drive mer praktisk siden det ikke krever kabel.
{{% /details %}}

{{% details title="Kan jeg overføre FLAC-filer til iPhonen min?" closed="true" %}}
Ja. Både Evermusic og Flacbox aksepterer FLAC-filer via iTunes File Sharing, WiFi Drive eller SMB. Flacbox anbefales for tapsfrie formater.
{{% /details %}}

{{% details title="Trenger jeg iTunes på macOS Catalina eller nyere?" closed="true" %}}
Nei. Apple erstattet iTunes med Finder for enhetsadministrasjon fra og med macOS Catalina. Bruk Finders Filer-fane for fildeling.
{{% /details %}}

{{% details title="Kan jeg strømme musikk uten å kopiere filer til iPhonen?" closed="true" %}}
Ja. Bruk SMB-protokollen til å strømme musikk direkte fra Mac eller PC. Dette sparer enhetslagring og holder biblioteket på datamaskinen.
{{% /details %}}

{{% details title="Hvilken app bør jeg bruke -- Evermusic eller Flacbox?" closed="true" %}}
Bruk Evermusic for standardformater som MP3, WAV og AAC. Velg Flacbox hvis biblioteket ditt inkluderer tapsfrie formater som FLAC, DSD eller OGG.
{{% /details %}}

**Ringkasan:** Transfer musik dari komputer ke iPhone menggunakan salah satu dari tiga metode: **Berbagi File iTunes/Finder** (kabel USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (nirkabel, tanpa kabel), atau **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (streaming langsung tanpa menyalin). Kemudian putar dengan [Evermusic](/products/evermusic) atau [Flacbox](/products/flacbox).

Mengelola file musik antara komputer dan perangkat iOS tidak pernah semudah ini. Kami telah menguraikan tiga metode mudah – Berbagi File iTunes, WiFi Drive, dan protokol SMB – untuk memastikan transfer musik yang lancar dan pemutaran yang mulus.

## Berbagi File iTunes

Berbagi File iTunes memungkinkan Anda menyalin file audio dengan mudah antara komputer dan aplikasi iOS di iPhone, iPad, atau iPod touch. Pendekatan ini sempurna bagi mereka yang ingin menikmati album, artis, dan lagu favorit dari Mac/PC di perangkat iOS mereka.

**Sebelum Anda Mulai**

- Sebagai alternatif, Anda dapat menggunakan [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) untuk menyalin atau memindahkan file audio secara nirkabel antara komputer dan aplikasi seperti Evermusic dan Flacbox. Cukup pastikan komputer dan perangkat iOS Anda terhubung ke jaringan WiFi yang sama, dan Anda dapat dengan mudah mentransfer file antar aplikasi ini.
- Teknologi lain yang perlu dipertimbangkan adalah [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), yang memungkinkan Anda streaming musik langsung dari komputer rumah ke perangkat iOS, menghilangkan kebutuhan untuk menyalin file audio.

## macOS Catalina atau Lebih Baru

Dengan macOS Catalina atau versi yang lebih baru, Anda dapat menggunakan Finder untuk berbagi file antara perangkat iOS dan iPadOS dengan Mac Anda.

**Persyaratan**  

- macOS Catalina atau lebih baru
- iPhone, iPad, atau iPod touch dengan versi iOS atau iPadOS terbaru
- Aplikasi iOS atau iPadOS yang kompatibel dengan Berbagi File

### Cara Menggunakan Finder untuk Berbagi File iTunes di macOS Catalina atau Lebih Baru

1. Buka jendela Finder.
2. Hubungkan iPhone, iPad, atau iPod touch ke komputer menggunakan kabel USB.
3. Pilih perangkat Anda di Finder.
4. Klik tab "File" untuk melihat daftar aplikasi yang mendukung berbagi file. Jika Anda tidak melihat bagian ini, berarti perangkat Anda tidak mendukung aplikasi berbagi file.
5. Klik segitiga di samping aplikasi untuk melihat file yang tersedia untuk dibagikan.

![Cara Menggunakan Finder untuk Berbagi File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Menyalin File dari Mac ke Perangkat

1. Pilih file yang ingin Anda salin, pastikan file tersebut kompatibel dengan aplikasi. Lihat panduan pengguna aplikasi untuk format yang didukung.
2. Seret dan lepas file ke ikon aplikasi di perangkat Anda. Finder akan menangani penyalinan file ke perangkat Anda.

#### Menyalin File dari Perangkat ke Mac

1. Pilih file yang ingin Anda salin.
2. Seret file ke folder di komputer Anda, dan Finder akan melakukan operasi penyalinan.

## macOS Lebih Lama dari Catalina

Jika Anda menggunakan versi macOS yang lebih lama, Anda dapat menggunakan iTunes untuk Berbagi File.

**Persyaratan**  

- Pastikan Anda memiliki versi iTunes terbaru.
- iPhone, iPad, atau iPod touch Anda harus menjalankan versi iOS terbaru.
- Jika Anda menggunakan PC Windows, pastikan sudah diperbarui setidaknya ke Windows 7.

![Berbagi File iTunes](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Cara Menggunakan Berbagi File iTunes untuk Versi macOS Lebih Lama dari Catalina

1. Buka iTunes di Mac atau PC Anda.
2. Hubungkan iPhone atau iPad ke komputer melalui kabel USB lightning.
3. Klik perangkat iOS Anda di iTunes.
4. Navigasi ke bagian "Aplikasi" di sidebar kiri.
5. Gulir ke bawah ke bagian "Berbagi File" di bagian bawah halaman.
6. Pilih aplikasi untuk melihat file mana yang tersedia untuk dibagikan di perangkat Anda.

![berbagi file itunes pilih aplikasi](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Menyalin File dari Komputer ke Aplikasi   

1. Buka iTunes dan pilih aplikasi dari daftar di bagian Berbagi File.
2. Untuk menyalin file ke perangkat Anda, cukup seret dan lepas dari folder atau jendela ke daftar "Dokumen".
3. Sebagai alternatif, klik "Tambah" di daftar "Dokumen" di iTunes, temukan file audio di komputer Anda, dan klik "Tambah". iTunes akan menyalin file musik ini ke direktori "Dokumen" aplikasi yang dipilih di perangkat Anda. Pastikan Anda memilih file audio yang kompatibel dengan aplikasi; lihat panduan pengguna untuk format yang didukung.

![berbagi file itunes pindahkan file](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Menyalin File Audio dari Aplikasi iOS ke Komputer   

1. Pilih aplikasi dari daftar di bagian Berbagi File iTunes.
2. Untuk menyalin file ke komputer, seret dari daftar "Dokumen" ke folder atau jendela di komputer Anda.
3. Sebagai alternatif, pilih file audio yang ingin Anda salin ke komputer dari daftar "Dokumen", klik "Simpan ke", dan pilih folder tujuan di komputer Anda.

![Berbagi File iTunes: Menyalin File Audio dari Aplikasi iOS ke Komputer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Mengakses File Audio yang Dibagikan   

Setelah Anda berhasil menyalin file dari komputer ke perangkat iOS, Anda dapat memutarnya dengan mudah.

- Buka aplikasi yang Anda pilih untuk Berbagi File, seperti Evermusic atau Flacbox.
- Ketuk bagian "File lokal" untuk melihat daftar file yang tersedia.
- Cukup ketuk file mana saja untuk mulai memutar.

## Menghapus File yang Dibagikan dari Perangkat iOS   

Untuk menghapus file audio yang disalin dari komputer ke aplikasi iOS menggunakan Berbagi File, ikuti langkah-langkah berikut:

#### Menggunakan iTunes  

1. Buka iTunes di Mac/PC Anda.
2. Hubungkan perangkat iOS ke komputer menggunakan kabel USB.
3. Pilih perangkat Anda di iTunes.
4. Klik "Aplikasi" di sidebar kiri dan gulir ke bawah ke bagian Berbagi File.
5. Pilih aplikasi dengan file yang ingin Anda hapus, seperti Evermusic atau Flacbox.
6. Pilih file audio yang ingin Anda hapus dari daftar "Dokumen", lalu tekan tombol "Hapus" di keyboard Anda.
7. Konfirmasi penghapusan di iTunes.

![Berbagi File iTunes: Hapus File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Menggunakan Pengelola File Bawaan  

1. Buka bagian "File lokal".
2. Ketuk tombol "Lebih banyak tindakan" di sudut kanan atas, dan pilih item menu "Pilih".
3. Pilih file yang ingin Anda hapus.
4. Ketuk tindakan "Hapus". Jika tindakan ini tidak langsung terlihat, ketuk menu "Lebih banyak tindakan".
5. Konfirmasi penghapusan di pesan pop-up.

## FAQ

{{% details title="Apa cara tercepat untuk mentransfer musik ke iPhone?" closed="true" %}}
Berbagi File iTunes/Finder melalui USB adalah metode tercepat untuk perpustakaan musik besar. Untuk transfer yang lebih kecil, WiFi Drive lebih nyaman karena tidak memerlukan kabel.
{{% /details %}}

{{% details title="Bisakah saya mentransfer file FLAC ke iPhone saya?" closed="true" %}}
Ya. Baik Evermusic maupun Flacbox menerima file FLAC melalui Berbagi File iTunes, WiFi Drive, atau SMB. Flacbox direkomendasikan untuk format lossless.
{{% /details %}}

{{% details title="Apakah saya memerlukan iTunes di macOS Catalina atau lebih baru?" closed="true" %}}
Tidak. Apple mengganti iTunes dengan Finder untuk manajemen perangkat mulai dari macOS Catalina. Gunakan tab File di Finder untuk berbagi file.
{{% /details %}}

{{% details title="Bisakah saya streaming musik tanpa menyalin file ke iPhone saya?" closed="true" %}}
Ya. Gunakan protokol SMB untuk streaming musik langsung dari Mac atau PC Anda. Ini menghemat penyimpanan perangkat dan menjaga perpustakaan Anda tetap di komputer.
{{% /details %}}

{{% details title="Aplikasi mana yang harus saya gunakan -- Evermusic atau Flacbox?" closed="true" %}}
Gunakan Evermusic untuk format standar seperti MP3, WAV, dan AAC. Pilih Flacbox jika perpustakaan Anda mencakup format lossless seperti FLAC, DSD, atau OGG.
{{% /details %}}

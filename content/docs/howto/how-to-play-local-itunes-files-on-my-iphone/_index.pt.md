**Resumo:** Transfira música do seu computador para o iPhone usando um dos três métodos: **iTunes/Finder File Sharing** (cabo USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (sem fio, sem cabo necessário), ou **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (transmita diretamente sem copiar). Depois reproduza com [Evermusic](/products/evermusic) ou [Flacbox](/products/flacbox).

Gerenciar seus arquivos de música entre o computador e dispositivos iOS nunca foi tão simples. Descrevemos três métodos fáceis -- iTunes File Sharing, WiFi Drive e protocolo SMB -- para garantir uma transferência de música suave e reprodução contínua.

## iTunes File Sharing

O iTunes File Sharing permite copiar facilmente arquivos de áudio entre o computador e seus aplicativos iOS no iPhone, iPad ou iPod touch. Esta abordagem é perfeita para quem deseja aproveitar seus álbuns, artistas e faixas favoritas do Mac/PC no dispositivo iOS.

**Antes de começar**

- Alternativamente, você pode usar o [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) para copiar ou mover arquivos de áudio sem fio entre o computador e aplicativos como Evermusic e Flacbox. Basta garantir que o computador e o dispositivo iOS estejam conectados à mesma rede WiFi, e você poderá transferir arquivos facilmente entre esses aplicativos.
- Outra tecnologia a considerar é o [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), que permite transmitir música diretamente do computador doméstico para o dispositivo iOS, eliminando a necessidade de copiar arquivos de áudio.

## macOS Catalina ou posterior

Com macOS Catalina ou uma versão mais recente, você pode usar o Finder para compartilhar arquivos entre seus dispositivos iOS e iPadOS e o Mac.

**Requisitos**  

- macOS Catalina ou posterior
- Um iPhone, iPad ou iPod touch com a versão mais recente do iOS ou iPadOS
- Um aplicativo iOS ou iPadOS compatível com File Sharing

### Como usar o Finder para iTunes File Sharing macOS Catalina ou posterior

1. Abra uma janela do Finder.
2. Conecte seu iPhone, iPad ou iPod touch ao computador usando um cabo USB.
3. Selecione seu dispositivo no Finder.
4. Clique na aba "Arquivos" para ver uma lista de aplicativos capazes de compartilhar arquivos. Se você não vir esta seção, significa que seu dispositivo não suporta aplicativos de compartilhamento de arquivos.
5. Clique no triângulo ao lado de um aplicativo para ver os arquivos disponíveis para compartilhamento.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Copiando arquivos do Mac para o dispositivo

1. Selecione os arquivos que deseja copiar, garantindo que sejam compatíveis com o aplicativo. Consulte o guia do usuário do aplicativo para formatos suportados.
2. Arraste e solte os arquivos no ícone do aplicativo no seu dispositivo. O Finder cuidará de copiar os arquivos para o dispositivo.

#### Copiando arquivos do dispositivo para o Mac

1. Selecione os arquivos que deseja copiar.
2. Arraste os arquivos para uma pasta no computador, e o Finder realizará a operação de cópia.

## macOS anterior ao Catalina

Se você está usando uma versão mais antiga do macOS, pode usar o iTunes para File Sharing.

**Requisitos**  

- Certifique-se de ter a versão mais recente do iTunes.
- Seu iPhone, iPad ou iPod touch deve estar executando a versão mais recente do iOS.
- Se estiver usando um PC Windows, certifique-se de que esteja atualizado para pelo menos o Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Como usar o iTunes File Sharing para versões do macOS anteriores ao Catalina 

1. Abra o iTunes no Mac ou PC.
2. Conecte seu iPhone ou iPad ao computador via cabo USB lightning.
3. Clique no seu dispositivo iOS no iTunes.
4. Navegue até a seção "Apps" na barra lateral esquerda.
5. Role para baixo até a seção "File Sharing" na parte inferior da página.
6. Selecione o aplicativo para ver quais arquivos estão disponíveis para compartilhamento no dispositivo.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Copiando arquivos do computador para o aplicativo   

1. Abra o iTunes e escolha o aplicativo da lista na seção File Sharing.
2. Para copiar arquivos para o dispositivo, simplesmente arraste e solte de uma pasta ou janela para a lista "Documentos".
3. Alternativamente, clique em "Adicionar" na lista "Documentos" no iTunes, localize os arquivos de áudio no computador e clique em "Adicionar". O iTunes copiará esses arquivos de música para o diretório "Documentos" do aplicativo escolhido no dispositivo. Certifique-se de escolher arquivos de áudio compatíveis com o aplicativo; consulte o guia do usuário para formatos suportados.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Copiando arquivos de áudio de um aplicativo iOS para o computador   

1. Selecione o aplicativo da lista na seção File Sharing do iTunes.
2. Para copiar arquivos para o computador, arraste-os da lista "Documentos" para uma pasta ou janela no computador.
3. Alternativamente, selecione os arquivos de áudio que deseja copiar para o computador da lista "Documentos", clique em "Salvar em" e escolha a pasta de destino no computador.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Acessando arquivos de áudio compartilhados   

Depois de copiar com sucesso os arquivos do computador para o dispositivo iOS, você pode reproduzi-los facilmente.

- Abra o aplicativo selecionado para File Sharing, como Evermusic ou Flacbox.
- Toque na seção "Arquivos locais" para ver a lista de arquivos disponíveis.
- Basta tocar em qualquer arquivo para começar a reproduzir.

## Excluindo arquivos compartilhados do dispositivo iOS   

Para remover arquivos de áudio copiados do computador para um aplicativo iOS usando File Sharing, siga estes passos:

#### Usando o iTunes  

1. Abra o iTunes no Mac/PC.
2. Conecte o dispositivo iOS ao computador usando o cabo USB.
3. Selecione seu dispositivo no iTunes.
4. Clique em "Apps" na barra lateral esquerda e role até a seção File Sharing.
5. Escolha o aplicativo com o arquivo que deseja excluir, como Evermusic ou Flacbox.
6. Selecione o arquivo de áudio que deseja remover da lista "Documentos" e pressione a tecla "Excluir" no teclado.
7. Confirme a exclusão no iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Usando o gerenciador de arquivos integrado  

1. Abra a seção "Arquivos locais".
2. Toque no botão "Mais ações" no canto superior direito e selecione o item de menu "Selecionar".
3. Escolha os arquivos que deseja excluir.
4. Toque na ação "Excluir". Se esta ação não estiver imediatamente visível, toque no menu "Mais ações".
5. Confirme a exclusão na mensagem pop-up.

## FAQ

{{% details title="Qual é a maneira mais rápida de transferir música para o iPhone?" closed="true" %}}
iTunes/Finder File Sharing via USB é o método mais rápido para grandes bibliotecas de música. Para transferências menores, o WiFi Drive é mais conveniente, pois não requer cabo.
{{% /details %}}

{{% details title="Posso transferir arquivos FLAC para o meu iPhone?" closed="true" %}}
Sim. Tanto o Evermusic quanto o Flacbox aceitam arquivos FLAC via iTunes File Sharing, WiFi Drive ou SMB. O Flacbox é recomendado para formatos sem perdas.
{{% /details %}}

{{% details title="Preciso do iTunes no macOS Catalina ou posterior?" closed="true" %}}
Não. A Apple substituiu o iTunes pelo Finder para gerenciamento de dispositivos a partir do macOS Catalina. Use a aba Arquivos do Finder para compartilhamento de arquivos.
{{% /details %}}

{{% details title="Posso transmitir música sem copiar arquivos para o meu iPhone?" closed="true" %}}
Sim. Use o protocolo SMB para transmitir música diretamente do Mac ou PC. Isso economiza armazenamento do dispositivo e mantém sua biblioteca no computador.
{{% /details %}}

{{% details title="Qual aplicativo devo usar -- Evermusic ou Flacbox?" closed="true" %}}
Use o Evermusic para formatos padrão como MP3, WAV e AAC. Escolha o Flacbox se sua biblioteca inclui formatos sem perdas como FLAC, DSD ou OGG.
{{% /details %}}

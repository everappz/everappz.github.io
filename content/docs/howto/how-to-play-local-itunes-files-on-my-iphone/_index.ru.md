**Кратко:** Перенесите музыку с компьютера на iPhone одним из трёх способов: **iTunes/Finder File Sharing** (USB-кабель), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (беспроводной, кабель не нужен), или **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (потоковое воспроизведение напрямую без копирования). Затем воспроизводите с помощью [Evermusic](/products/evermusic) или [Flacbox](/products/flacbox).

Управление музыкальными файлами между компьютером и устройствами iOS ещё никогда не было таким простым. Мы описали три простых метода -- iTunes File Sharing, WiFi Drive и протокол SMB -- для обеспечения плавного переноса музыки и бесперебойного воспроизведения.

## iTunes File Sharing

iTunes File Sharing позволяет легко копировать аудиофайлы между компьютером и приложениями iOS на iPhone, iPad или iPod touch. Этот подход идеально подходит для тех, кто хочет наслаждаться любимыми альбомами, исполнителями и треками с Mac/ПК на своём устройстве iOS.

**Перед началом**

- Также вы можете использовать [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) для беспроводного копирования или перемещения аудиофайлов между компьютером и приложениями, такими как Evermusic и Flacbox. Просто убедитесь, что компьютер и устройство iOS подключены к одной сети WiFi, и вы сможете легко передавать файлы между этими приложениями.
- Ещё одна технология, которую стоит рассмотреть -- [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), позволяющая напрямую транслировать музыку с домашнего компьютера на устройство iOS без необходимости копирования аудиофайлов.

## macOS Catalina или новее

С macOS Catalina или более новой версией вы можете использовать Finder для обмена файлами между устройствами iOS и iPadOS и вашим Mac.

**Требования**  

- macOS Catalina или новее
- iPhone, iPad или iPod touch с последней версией iOS или iPadOS
- Приложение iOS или iPadOS, совместимое с File Sharing

### Как использовать Finder для iTunes File Sharing macOS Catalina или новее

1. Откройте окно Finder.
2. Подключите iPhone, iPad или iPod touch к компьютеру с помощью USB-кабеля.
3. Выберите устройство в Finder.
4. Нажмите на вкладку «Файлы», чтобы увидеть список приложений, поддерживающих общий доступ к файлам. Если вы не видите этот раздел, значит, ваше устройство не поддерживает приложения для обмена файлами.
5. Нажмите на треугольник рядом с приложением, чтобы просмотреть доступные для общего доступа файлы.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Копирование файлов с Mac на устройство

1. Выберите файлы, которые хотите скопировать, убедившись, что они совместимы с приложением. Обратитесь к руководству пользователя приложения для получения информации о поддерживаемых форматах.
2. Перетащите файлы на значок приложения на вашем устройстве. Finder скопирует файлы на устройство.

#### Копирование файлов с устройства на Mac

1. Выберите файлы, которые хотите скопировать.
2. Перетащите файлы в папку на компьютере, и Finder выполнит операцию копирования.

## macOS старше Catalina

Если вы используете более старую версию macOS, вы можете использовать iTunes для File Sharing.

**Требования**  

- Убедитесь, что у вас установлена последняя версия iTunes.
- На iPhone, iPad или iPod touch должна быть установлена последняя версия iOS.
- Если вы используете ПК с Windows, убедитесь, что он обновлён как минимум до Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Как использовать iTunes File Sharing для версий macOS старше Catalina 

1. Откройте iTunes на Mac или ПК.
2. Подключите iPhone или iPad к компьютеру через USB lightning-кабель.
3. Нажмите на устройство iOS в iTunes.
4. Перейдите в раздел «Программы» в левой боковой панели.
5. Прокрутите вниз до раздела «File Sharing» в нижней части страницы.
6. Выберите приложение, чтобы увидеть, какие файлы доступны для общего доступа на устройстве.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Копирование файлов с компьютера в приложение   

1. Откройте iTunes и выберите приложение из списка в разделе File Sharing.
2. Чтобы скопировать файлы на устройство, просто перетащите их из папки или окна в список «Документы».
3. Также можно нажать «Добавить» в списке «Документы» в iTunes, найти аудиофайлы на компьютере и нажать «Добавить». iTunes скопирует эти музыкальные файлы в каталог «Документы» выбранного приложения на устройстве. Убедитесь, что выбранные аудиофайлы совместимы с приложением; обратитесь к руководству пользователя для получения информации о поддерживаемых форматах.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Копирование аудиофайлов из приложения iOS на компьютер   

1. Выберите приложение из списка в разделе File Sharing iTunes.
2. Чтобы скопировать файлы на компьютер, перетащите их из списка «Документы» в папку или окно на компьютере.
3. Также можно выбрать аудиофайлы, которые хотите скопировать на компьютер, из списка «Документы», нажать «Сохранить в» и выбрать папку назначения на компьютере.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Доступ к общим аудиофайлам   

После успешного копирования файлов с компьютера на устройство iOS вы можете легко их воспроизвести.

- Откройте приложение, выбранное для File Sharing, например Evermusic или Flacbox.
- Нажмите на раздел «Локальные файлы», чтобы просмотреть список доступных файлов.
- Просто нажмите на любой файл, чтобы начать воспроизведение.

## Удаление общих файлов с устройства iOS   

Чтобы удалить аудиофайлы, скопированные с компьютера в приложение iOS с помощью File Sharing, выполните следующие действия:

#### Использование iTunes  

1. Откройте iTunes на Mac/ПК.
2. Подключите устройство iOS к компьютеру с помощью USB-кабеля.
3. Выберите устройство в iTunes.
4. Нажмите «Программы» в левой боковой панели и прокрутите вниз до раздела File Sharing.
5. Выберите приложение с файлом, который хотите удалить, например Evermusic или Flacbox.
6. Выберите аудиофайл, который хотите удалить, из списка «Документы», затем нажмите клавишу «Удалить» на клавиатуре.
7. Подтвердите удаление в iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Использование встроенного файлового менеджера  

1. Откройте раздел «Локальные файлы».
2. Нажмите кнопку «Другие действия» в правом верхнем углу и выберите пункт меню «Выбрать».
3. Выберите файлы, которые хотите удалить.
4. Нажмите действие «Удалить». Если это действие не отображается сразу, нажмите меню «Другие действия».
5. Подтвердите удаление во всплывающем сообщении.

## FAQ

{{% details title="Какой самый быстрый способ перенести музыку на iPhone?" closed="true" %}}
iTunes/Finder File Sharing через USB -- самый быстрый метод для больших музыкальных библиотек. Для небольших передач WiFi Drive удобнее, так как не требует кабеля.
{{% /details %}}

{{% details title="Можно ли перенести файлы FLAC на iPhone?" closed="true" %}}
Да. И Evermusic, и Flacbox принимают файлы FLAC через iTunes File Sharing, WiFi Drive или SMB. Для форматов без потерь рекомендуется Flacbox.
{{% /details %}}

{{% details title="Нужен ли iTunes на macOS Catalina или новее?" closed="true" %}}
Нет. Apple заменила iTunes на Finder для управления устройствами начиная с macOS Catalina. Используйте вкладку «Файлы» в Finder для обмена файлами.
{{% /details %}}

{{% details title="Можно ли транслировать музыку без копирования файлов на iPhone?" closed="true" %}}
Да. Используйте протокол SMB для потокового воспроизведения музыки напрямую с Mac или ПК. Это экономит место на устройстве и сохраняет библиотеку на компьютере.
{{% /details %}}

{{% details title="Какое приложение выбрать -- Evermusic или Flacbox?" closed="true" %}}
Используйте Evermusic для стандартных форматов, таких как MP3, WAV и AAC. Выберите Flacbox, если ваша библиотека включает форматы без потерь, такие как FLAC, DSD или OGG.
{{% /details %}}

**Коротко:** Перенесіть музику з комп'ютера на iPhone одним із трьох способів: **iTunes/Finder File Sharing** (USB-кабель), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (бездротовий, кабель не потрібен), або **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (потокове відтворення напряму без копіювання). Потім відтворюйте за допомогою [Evermusic](/products/evermusic) або [Flacbox](/products/flacbox).

Керування музичними файлами між комп'ютером та пристроями iOS ще ніколи не було таким простим. Ми описали три прості методи -- iTunes File Sharing, WiFi Drive та протокол SMB -- для забезпечення плавного перенесення музики та безперебійного відтворення.

## iTunes File Sharing

iTunes File Sharing дозволяє легко копіювати аудіофайли між комп'ютером та додатками iOS на iPhone, iPad або iPod touch. Цей підхід ідеально підходить для тих, хто хоче насолоджуватися улюбленими альбомами, виконавцями та треками з Mac/ПК на своєму пристрої iOS.

**Перед початком**

- Також ви можете використовувати [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) для бездротового копіювання або переміщення аудіофайлів між комп'ютером та додатками, такими як Evermusic та Flacbox. Просто переконайтеся, що комп'ютер та пристрій iOS підключені до однієї мережі WiFi, і ви зможете легко передавати файли між цими додатками.
- Ще одна технологія, яку варто розглянути -- [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), що дозволяє напряму транслювати музику з домашнього комп'ютера на пристрій iOS без необхідності копіювання аудіофайлів.

## macOS Catalina або новіша

З macOS Catalina або новішою версією ви можете використовувати Finder для обміну файлами між пристроями iOS та iPadOS та вашим Mac.

**Вимоги**  

- macOS Catalina або новіша
- iPhone, iPad або iPod touch з останньою версією iOS або iPadOS
- Додаток iOS або iPadOS, сумісний з File Sharing

### Як використовувати Finder для iTunes File Sharing macOS Catalina або новіша

1. Відкрийте вікно Finder.
2. Підключіть iPhone, iPad або iPod touch до комп'ютера за допомогою USB-кабелю.
3. Виберіть свій пристрій у Finder.
4. Натисніть на вкладку «Файли», щоб побачити список додатків, які підтримують спільний доступ до файлів. Якщо ви не бачите цей розділ, ваш пристрій не підтримує додатки для обміну файлами.
5. Натисніть на трикутник біля додатка, щоб переглянути доступні для спільного доступу файли.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Копіювання файлів з Mac на пристрій

1. Виберіть файли, які хочете скопіювати, переконавшись, що вони сумісні з додатком. Зверніться до посібника користувача додатка для отримання інформації про підтримувані формати.
2. Перетягніть файли на значок додатка на вашому пристрої. Finder скопіює файли на пристрій.

#### Копіювання файлів з пристрою на Mac

1. Виберіть файли, які хочете скопіювати.
2. Перетягніть файли до папки на комп'ютері, і Finder виконає операцію копіювання.

## macOS старша за Catalina

Якщо ви використовуєте старішу версію macOS, ви можете використовувати iTunes для File Sharing.

**Вимоги**  

- Переконайтеся, що у вас встановлена остання версія iTunes.
- На iPhone, iPad або iPod touch має бути встановлена остання версія iOS.
- Якщо ви використовуєте ПК з Windows, переконайтеся, що він оновлений щонайменше до Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Як використовувати iTunes File Sharing для версій macOS старших за Catalina 

1. Відкрийте iTunes на Mac або ПК.
2. Підключіть iPhone або iPad до комп'ютера через USB lightning-кабель.
3. Натисніть на пристрій iOS в iTunes.
4. Перейдіть до розділу «Програми» в лівій бічній панелі.
5. Прокрутіть вниз до розділу «File Sharing» в нижній частині сторінки.
6. Виберіть додаток, щоб побачити, які файли доступні для спільного доступу на пристрої.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Копіювання файлів з комп'ютера в додаток   

1. Відкрийте iTunes та виберіть додаток зі списку в розділі File Sharing.
2. Щоб скопіювати файли на пристрій, просто перетягніть їх з папки або вікна до списку «Документи».
3. Також можна натиснути «Додати» в списку «Документи» в iTunes, знайти аудіофайли на комп'ютері та натиснути «Додати». iTunes скопіює ці музичні файли до каталогу «Документи» вибраного додатка на пристрої. Переконайтеся, що вибрані аудіофайли сумісні з додатком; зверніться до посібника користувача для отримання інформації про підтримувані формати.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Копіювання аудіофайлів з додатка iOS на комп'ютер   

1. Виберіть додаток зі списку в розділі File Sharing iTunes.
2. Щоб скопіювати файли на комп'ютер, перетягніть їх зі списку «Документи» до папки або вікна на комп'ютері.
3. Також можна вибрати аудіофайли, які хочете скопіювати на комп'ютер, зі списку «Документи», натиснути «Зберегти в» та вибрати папку призначення на комп'ютері.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Доступ до спільних аудіофайлів   

Після успішного копіювання файлів з комп'ютера на пристрій iOS ви можете легко їх відтворити.

- Відкрийте додаток, вибраний для File Sharing, наприклад Evermusic або Flacbox.
- Натисніть на розділ «Локальні файли», щоб переглянути список доступних файлів.
- Просто натисніть на будь-який файл, щоб розпочати відтворення.

## Видалення спільних файлів з пристрою iOS   

Щоб видалити аудіофайли, скопійовані з комп'ютера в додаток iOS за допомогою File Sharing, виконайте такі кроки:

#### Використання iTunes  

1. Відкрийте iTunes на Mac/ПК.
2. Підключіть пристрій iOS до комп'ютера за допомогою USB-кабелю.
3. Виберіть пристрій в iTunes.
4. Натисніть «Програми» в лівій бічній панелі та прокрутіть вниз до розділу File Sharing.
5. Виберіть додаток з файлом, який хочете видалити, наприклад Evermusic або Flacbox.
6. Виберіть аудіофайл, який хочете видалити, зі списку «Документи», потім натисніть клавішу «Видалити» на клавіатурі.
7. Підтвердіть видалення в iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Використання вбудованого файлового менеджера  

1. Відкрийте розділ «Локальні файли».
2. Натисніть кнопку «Більше дій» у правому верхньому куті та виберіть пункт меню «Вибрати».
3. Виберіть файли, які хочете видалити.
4. Натисніть дію «Видалити». Якщо ця дія не відображається відразу, натисніть меню «Більше дій».
5. Підтвердіть видалення у спливаючому повідомленні.

## FAQ

{{% details title="Який найшвидший спосіб перенести музику на iPhone?" closed="true" %}}
iTunes/Finder File Sharing через USB -- найшвидший метод для великих музичних бібліотек. Для невеликих передач WiFi Drive зручніший, оскільки не потребує кабелю.
{{% /details %}}

{{% details title="Чи можна перенести файли FLAC на iPhone?" closed="true" %}}
Так. І Evermusic, і Flacbox приймають файли FLAC через iTunes File Sharing, WiFi Drive або SMB. Для форматів без втрат рекомендується Flacbox.
{{% /details %}}

{{% details title="Чи потрібен iTunes на macOS Catalina або новішій?" closed="true" %}}
Ні. Apple замінила iTunes на Finder для керування пристроями починаючи з macOS Catalina. Використовуйте вкладку «Файли» у Finder для обміну файлами.
{{% /details %}}

{{% details title="Чи можна транслювати музику без копіювання файлів на iPhone?" closed="true" %}}
Так. Використовуйте протокол SMB для потокового відтворення музики напряму з Mac або ПК. Це економить місце на пристрої та зберігає бібліотеку на комп'ютері.
{{% /details %}}

{{% details title="Який додаток обрати -- Evermusic чи Flacbox?" closed="true" %}}
Використовуйте Evermusic для стандартних форматів, таких як MP3, WAV та AAC. Виберіть Flacbox, якщо ваша бібліотека містить формати без втрат, такі як FLAC, DSD або OGG.
{{% /details %}}

**Sammanfattning:** Överför musik från din dator till iPhone med en av tre metoder: **iTunes/Finder File Sharing** (USB-kabel), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (trådlöst, ingen kabel behövs), eller **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (strömma direkt utan att kopiera). Spela sedan med [Evermusic](/products/evermusic) eller [Flacbox](/products/flacbox).

Att hantera dina musikfiler mellan din dator och iOS-enheter har aldrig varit enklare. Vi har beskrivit tre enkla metoder -- iTunes File Sharing, WiFi Drive och SMB-protokollet -- för att säkerställa en smidig musiköverföring och sömlös uppspelning.

## iTunes File Sharing

iTunes File Sharing gör det möjligt att enkelt kopiera ljudfiler mellan din dator och dina iOS-appar på din iPhone, iPad eller iPod touch. Detta tillvägagångssätt är perfekt för dem som vill njuta av sina favoritalbum, artister och låtar från sin Mac/PC på sin iOS-enhet.

**Innan du börjar**

- Alternativt kan du använda [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) för att trådlöst kopiera eller flytta ljudfiler mellan din dator och appar som Evermusic och Flacbox. Se till att din dator och iOS-enhet är anslutna till samma WiFi-nätverk, och du kan enkelt överföra filer mellan dessa appar.
- En annan teknik att överväga är [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), som låter dig strömma musik direkt från din hemdator till din iOS-enhet utan att behöva kopiera ljudfiler.

## macOS Catalina eller senare

Med macOS Catalina eller en nyare version kan du använda Finder för att dela filer mellan dina iOS- och iPadOS-enheter och din Mac.

**Krav**  

- macOS Catalina eller senare
- En iPhone, iPad eller iPod touch med den senaste iOS- eller iPadOS-versionen
- En iOS- eller iPadOS-app som är kompatibel med File Sharing

### Hur man använder Finder för iTunes File Sharing macOS Catalina eller senare

1. Öppna ett Finder-fönster.
2. Anslut din iPhone, iPad eller iPod touch till datorn med en USB-kabel.
3. Välj din enhet i Finder.
4. Klicka på fliken "Filer" för att se en lista över appar som kan dela filer. Om du inte ser detta avsnitt betyder det att din enhet inte stöder fildelningsappar.
5. Klicka på triangeln bredvid en app för att se tillgängliga filer för delning.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Kopiera filer från din Mac till enhet

1. Välj filerna du vill kopiera och se till att de är kompatibla med appen. Se appens användarguide för format som stöds.
2. Dra och släpp filerna till appikonen på din enhet. Finder tar hand om kopieringen av filerna till din enhet.

#### Kopiera filer från din enhet till Mac

1. Välj filerna du vill kopiera.
2. Dra filerna till en mapp på din dator, och Finder utför kopieringsoperationen.

## macOS äldre än Catalina

Om du använder en äldre macOS-version kan du använda iTunes för File Sharing.

**Krav**  

- Se till att du har den senaste versionen av iTunes.
- Din iPhone, iPad eller iPod touch bör köra den senaste iOS-versionen.
- Om du använder en Windows-PC, se till att den är uppdaterad till minst Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Hur man använder iTunes File Sharing för macOS-versioner äldre än Catalina 

1. Öppna iTunes på din Mac eller PC.
2. Anslut din iPhone eller iPad till datorn via USB lightning-kabel.
3. Klicka på din iOS-enhet i iTunes.
4. Navigera till avsnittet "Appar" i det vänstra sidofältet.
5. Scrolla ner till avsnittet "File Sharing" längst ner på sidan.
6. Välj appen för att se vilka filer som är tillgängliga för delning på din enhet.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Kopiera filer från din dator till appen   

1. Öppna iTunes och välj appen från listan i avsnittet File Sharing.
2. För att kopiera filer till din enhet, dra och släpp dem från en mapp eller ett fönster till listan "Dokument".
3. Alternativt kan du klicka på "Lägg till" i listan "Dokument" i iTunes, hitta ljudfilerna på din dator och klicka "Lägg till." iTunes kopierar dessa musikfiler till katalogen "Dokument" i den valda appen på din enhet. Se till att du väljer ljudfiler som är kompatibla med appen; se användarguiden för format som stöds.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Kopiera ljudfiler från en iOS-app till din dator   

1. Välj appen från listan i iTunes File Sharing-avsnitt.
2. För att kopiera filer till din dator, dra dem från listan "Dokument" till en mapp eller ett fönster på din dator.
3. Alternativt kan du välja ljudfilerna du vill kopiera till din dator från listan "Dokument", klicka "Spara till" och välja målmappen på din dator.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Åtkomst till delade ljudfiler   

När du framgångsrikt har kopierat filer från din dator till din iOS-enhet kan du enkelt spela dem.

- Öppna appen du valde för File Sharing, såsom Evermusic eller Flacbox.
- Tryck på avsnittet "Lokala filer" för att se listan över tillgängliga filer.
- Tryck på valfri fil för att börja spela.

## Ta bort delade filer från din iOS-enhet   

För att ta bort ljudfiler som kopierats från din dator till en iOS-app med File Sharing, följ dessa steg:

#### Använda iTunes  

1. Öppna iTunes på din Mac/PC.
2. Anslut din iOS-enhet till datorn med USB-kabeln.
3. Välj din enhet i iTunes.
4. Klicka på "Appar" i det vänstra sidofältet och scrolla ner till avsnittet File Sharing.
5. Välj appen med filen du vill ta bort, som Evermusic eller Flacbox.
6. Välj ljudfilen du vill ta bort från listan "Dokument" och tryck sedan på "Ta bort"-tangenten på tangentbordet.
7. Bekräfta borttagningen i iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Använda den inbyggda filhanteraren  

1. Öppna avsnittet "Lokala filer".
2. Tryck på knappen "Fler åtgärder" i det övre högra hörnet och välj menyalternativet "Välja".
3. Välj filerna du vill ta bort.
4. Tryck på åtgärden "Ta bort". Om denna åtgärd inte är omedelbart synlig, tryck på menyn "Fler åtgärder".
5. Bekräfta borttagningen i popup-meddelandet.

## FAQ

{{% details title="Vad är det snabbaste sättet att överföra musik till iPhone?" closed="true" %}}
iTunes/Finder File Sharing via USB är den snabbaste metoden för stora musikbibliotek. För mindre överföringar är WiFi Drive bekvämare eftersom det inte kräver kabel.
{{% /details %}}

{{% details title="Kan jag överföra FLAC-filer till min iPhone?" closed="true" %}}
Ja. Både Evermusic och Flacbox accepterar FLAC-filer via iTunes File Sharing, WiFi Drive eller SMB. Flacbox rekommenderas för förlustfria format.
{{% /details %}}

{{% details title="Behöver jag iTunes på macOS Catalina eller senare?" closed="true" %}}
Nej. Apple ersatte iTunes med Finder för enhetshantering från och med macOS Catalina. Använd Finders Filer-flik för fildelning.
{{% /details %}}

{{% details title="Kan jag strömma musik utan att kopiera filer till min iPhone?" closed="true" %}}
Ja. Använd SMB-protokollet för att strömma musik direkt från din Mac eller PC. Detta sparar enhetslagring och behåller ditt bibliotek på datorn.
{{% /details %}}

{{% details title="Vilken app ska jag använda -- Evermusic eller Flacbox?" closed="true" %}}
Använd Evermusic för standardformat som MP3, WAV och AAC. Välj Flacbox om ditt bibliotek innehåller förlustfria format som FLAC, DSD eller OGG.
{{% /details %}}

**Podsumowanie:** Przenieś muzykę z komputera na iPhone za pomocą jednej z trzech metod: **iTunes/Finder File Sharing** (kabel USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (bezprzewodowo, bez kabla), lub **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (strumieniuj bezpośrednio bez kopiowania). Następnie odtwarzaj za pomocą [Evermusic](/products/evermusic) lub [Flacbox](/products/flacbox).

Zarządzanie plikami muzycznymi między komputerem a urządzeniami iOS nigdy nie było prostsze. Przedstawiliśmy trzy łatwe metody -- iTunes File Sharing, WiFi Drive i protokół SMB -- aby zapewnić płynny transfer muzyki i bezproblemowe odtwarzanie.

## iTunes File Sharing

iTunes File Sharing umożliwia bezproblemowe kopiowanie plików audio między komputerem a aplikacjami iOS na iPhonie, iPadzie lub iPodzie touch. To podejście jest idealne dla osób, które chcą cieszyć się ulubionymi albumami, artystami i utworami z komputera Mac/PC na swoim urządzeniu iOS.

**Zanim zaczniesz**

- Alternatywnie możesz użyć [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) do bezprzewodowego kopiowania lub przenoszenia plików audio między komputerem a aplikacjami takimi jak Evermusic i Flacbox. Upewnij się, że komputer i urządzenie iOS są podłączone do tej samej sieci WiFi, a będziesz mógł łatwo przesyłać pliki między tymi aplikacjami.
- Inną technologią wartą rozważenia jest [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), która pozwala bezpośrednio strumieniować muzykę z domowego komputera na urządzenie iOS, eliminując potrzebę kopiowania plików audio.

## macOS Catalina lub nowszy

Z macOS Catalina lub nowszą wersją możesz używać Findera do udostępniania plików między urządzeniami iOS i iPadOS a komputerem Mac.

**Wymagania**  

- macOS Catalina lub nowszy
- iPhone, iPad lub iPod touch z najnowszą wersją iOS lub iPadOS
- Aplikacja iOS lub iPadOS kompatybilna z File Sharing

### Jak używać Findera do iTunes File Sharing macOS Catalina lub nowszy

1. Otwórz okno Findera.
2. Podłącz iPhone'a, iPada lub iPoda touch do komputera za pomocą kabla USB.
3. Wybierz swoje urządzenie w Finderze.
4. Kliknij zakładkę "Pliki", aby zobaczyć listę aplikacji obsługujących udostępnianie plików. Jeśli nie widzisz tej sekcji, oznacza to, że Twoje urządzenie nie obsługuje aplikacji do udostępniania plików.
5. Kliknij trójkąt obok aplikacji, aby wyświetlić pliki dostępne do udostępnienia.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Kopiowanie plików z komputera Mac na urządzenie

1. Wybierz pliki, które chcesz skopiować, upewniając się, że są kompatybilne z aplikacją. Sprawdź przewodnik użytkownika aplikacji, aby poznać obsługiwane formaty.
2. Przeciągnij i upuść pliki na ikonę aplikacji na urządzeniu. Finder skopiuje pliki na Twoje urządzenie.

#### Kopiowanie plików z urządzenia na komputer Mac

1. Wybierz pliki, które chcesz skopiować.
2. Przeciągnij pliki do folderu na komputerze, a Finder wykona operację kopiowania.

## macOS starszy niż Catalina

Jeśli używasz starszej wersji macOS, możesz użyć iTunes do File Sharing.

**Wymagania**  

- Upewnij się, że masz najnowszą wersję iTunes.
- iPhone, iPad lub iPod touch powinien mieć najnowszą wersję iOS.
- Jeśli używasz komputera z Windows, upewnij się, że jest zaktualizowany do co najmniej Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Jak używać iTunes File Sharing dla wersji macOS starszych niż Catalina 

1. Otwórz iTunes na komputerze Mac lub PC.
2. Podłącz iPhone'a lub iPada do komputera za pomocą kabla USB lightning.
3. Kliknij na swoje urządzenie iOS w iTunes.
4. Przejdź do sekcji "Aplikacje" w lewym pasku bocznym.
5. Przewiń w dół do sekcji "File Sharing" na dole strony.
6. Wybierz aplikację, aby zobaczyć, które pliki są dostępne do udostępnienia na urządzeniu.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Kopiowanie plików z komputera do aplikacji   

1. Otwórz iTunes i wybierz aplikację z listy w sekcji File Sharing.
2. Aby skopiować pliki na urządzenie, po prostu przeciągnij i upuść je z folderu lub okna do listy "Dokumenty".
3. Alternatywnie kliknij "Dodaj" na liście "Dokumenty" w iTunes, znajdź pliki audio na komputerze i kliknij "Dodaj". iTunes skopiuje te pliki muzyczne do katalogu "Dokumenty" wybranej aplikacji na urządzeniu. Upewnij się, że wybierasz pliki audio kompatybilne z aplikacją; sprawdź przewodnik użytkownika, aby poznać obsługiwane formaty.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Kopiowanie plików audio z aplikacji iOS na komputer   

1. Wybierz aplikację z listy w sekcji File Sharing iTunes.
2. Aby skopiować pliki na komputer, przeciągnij je z listy "Dokumenty" do folderu lub okna na komputerze.
3. Alternatywnie wybierz pliki audio, które chcesz skopiować na komputer z listy "Dokumenty", kliknij "Zapisz w" i wybierz folder docelowy na komputerze.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Dostęp do udostępnionych plików audio   

Po pomyślnym skopiowaniu plików z komputera na urządzenie iOS możesz je łatwo odtwarzać.

- Otwórz aplikację wybraną do File Sharing, taką jak Evermusic lub Flacbox.
- Dotknij sekcji "Pliki lokalne", aby wyświetlić listę dostępnych plików.
- Po prostu dotknij dowolnego pliku, aby rozpocząć odtwarzanie.

## Usuwanie udostępnionych plików z urządzenia iOS   

Aby usunąć pliki audio skopiowane z komputera do aplikacji iOS za pomocą File Sharing, wykonaj następujące kroki:

#### Używanie iTunes  

1. Otwórz iTunes na komputerze Mac/PC.
2. Podłącz urządzenie iOS do komputera za pomocą kabla USB.
3. Wybierz swoje urządzenie w iTunes.
4. Kliknij "Aplikacje" w lewym pasku bocznym i przewiń w dół do sekcji File Sharing.
5. Wybierz aplikację z plikiem, który chcesz usunąć, np. Evermusic lub Flacbox.
6. Wybierz plik audio, który chcesz usunąć z listy "Dokumenty", a następnie naciśnij klawisz "Usunąć" na klawiaturze.
7. Potwierdź usunięcie w iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Używanie wbudowanego menedżera plików  

1. Otwórz sekcję "Pliki lokalne".
2. Dotknij przycisku "Więcej akcji" w prawym górnym rogu i wybierz element menu "Wybrać".
3. Wybierz pliki, które chcesz usunąć.
4. Dotknij akcji "Usunąć". Jeśli ta akcja nie jest od razu widoczna, dotknij menu "Więcej akcji".
5. Potwierdź usunięcie w wyskakującym komunikacie.

## FAQ

{{% details title="Jaki jest najszybszy sposób przesyłania muzyki na iPhone?" closed="true" %}}
iTunes/Finder File Sharing przez USB to najszybsza metoda dla dużych bibliotek muzycznych. Dla mniejszych transferów WiFi Drive jest wygodniejszy, ponieważ nie wymaga kabla.
{{% /details %}}

{{% details title="Czy mogę przesłać pliki FLAC na mojego iPhone'a?" closed="true" %}}
Tak. Zarówno Evermusic, jak i Flacbox przyjmują pliki FLAC przez iTunes File Sharing, WiFi Drive lub SMB. Flacbox jest zalecany dla formatów bezstratnych.
{{% /details %}}

{{% details title="Czy potrzebuję iTunes na macOS Catalina lub nowszym?" closed="true" %}}
Nie. Apple zastąpił iTunes Finderem do zarządzania urządzeniami począwszy od macOS Catalina. Użyj zakładki Pliki w Finderze do udostępniania plików.
{{% /details %}}

{{% details title="Czy mogę strumieniować muzykę bez kopiowania plików na iPhone'a?" closed="true" %}}
Tak. Użyj protokołu SMB, aby strumieniować muzykę bezpośrednio z komputera Mac lub PC. Oszczędza to pamięć urządzenia i utrzymuje bibliotekę na komputerze.
{{% /details %}}

{{% details title="Której aplikacji powinienem używać -- Evermusic czy Flacbox?" closed="true" %}}
Używaj Evermusic dla standardowych formatów, takich jak MP3, WAV i AAC. Wybierz Flacbox, jeśli Twoja biblioteka zawiera formaty bezstratne, takie jak FLAC, DSD lub OGG.
{{% /details %}}

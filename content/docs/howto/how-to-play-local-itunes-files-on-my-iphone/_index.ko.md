**요약:** 세 가지 방법 중 하나를 사용하여 컴퓨터에서 iPhone으로 음악을 전송하세요: **iTunes/Finder 파일 공유** (USB 케이블), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (무선, 케이블 불필요), 또는 **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (복사 없이 직접 스트리밍). 그런 다음 [Evermusic](/products/evermusic) 또는 [Flacbox](/products/flacbox)로 재생하세요.

컴퓨터와 iOS 기기 간의 음악 파일 관리가 그 어느 때보다 간편해졌습니다. iTunes 파일 공유, WiFi Drive, SMB 프로토콜 등 세 가지 쉬운 방법을 안내하여 원활한 음악 전송과 끊김 없는 재생을 보장합니다.

## iTunes 파일 공유

iTunes 파일 공유를 사용하면 컴퓨터와 iPhone, iPad 또는 iPod touch의 iOS 앱 간에 오디오 파일을 손쉽게 복사할 수 있습니다. 이 방법은 Mac/PC에 있는 좋아하는 앨범, 아티스트, 트랙을 iOS 기기에서 즐기고 싶은 분들에게 적합합니다.

**시작하기 전에**

- 또는 [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)를 사용하여 컴퓨터와 Evermusic 및 Flacbox 같은 앱 간에 무선으로 오디오 파일을 복사하거나 이동할 수 있습니다. 컴퓨터와 iOS 기기가 동일한 WiFi 네트워크에 연결되어 있으면 이러한 앱 간에 파일을 쉽게 전송할 수 있습니다.
- 또 다른 기술로 [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)를 고려해 볼 수 있으며, 이를 통해 가정용 컴퓨터에서 iOS 기기로 직접 음악을 스트리밍할 수 있어 오디오 파일을 복사할 필요가 없습니다.

## macOS Catalina 이상

macOS Catalina 이상 버전에서는 Finder를 사용하여 iOS 및 iPadOS 기기와 Mac 간에 파일을 공유할 수 있습니다.

**요구 사항**  

- macOS Catalina 이상
- 최신 iOS 또는 iPadOS 버전이 설치된 iPhone, iPad 또는 iPod touch
- 파일 공유와 호환되는 iOS 또는 iPadOS 앱

### macOS Catalina 이상에서 Finder를 사용한 iTunes 파일 공유 방법

1. Finder 창을 엽니다.
2. USB 케이블을 사용하여 iPhone, iPad 또는 iPod touch를 컴퓨터에 연결합니다.
3. Finder에서 기기를 선택합니다.
4. "파일" 탭을 클릭하여 파일 공유가 가능한 앱 목록을 확인합니다. 이 섹션이 보이지 않으면 기기가 파일 공유 앱을 지원하지 않는 것입니다.
5. 앱 옆의 삼각형을 클릭하여 공유 가능한 파일을 확인합니다.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Mac에서 기기로 파일 복사

1. 복사할 파일을 선택하고, 해당 앱과 호환되는지 확인합니다. 지원되는 형식은 앱의 사용 설명서를 참조하세요.
2. 파일을 기기의 앱 아이콘으로 드래그 앤 드롭합니다. Finder가 파일을 기기로 복사합니다.

#### 기기에서 Mac으로 파일 복사

1. 복사할 파일을 선택합니다.
2. 파일을 컴퓨터의 폴더로 드래그하면 Finder가 복사 작업을 수행합니다.

## macOS Catalina 이전 버전

이전 macOS 버전을 사용하는 경우 iTunes를 사용하여 파일을 공유할 수 있습니다.

**요구 사항**  

- 최신 버전의 iTunes가 설치되어 있는지 확인하세요.
- iPhone, iPad 또는 iPod touch에 최신 iOS 버전이 설치되어 있어야 합니다.
- Windows PC를 사용하는 경우 Windows 7 이상으로 업데이트되어 있는지 확인하세요.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### macOS Catalina 이전 버전에서 iTunes 파일 공유 사용 방법 

1. Mac 또는 PC에서 iTunes를 엽니다.
2. USB 라이트닝 케이블을 사용하여 iPhone 또는 iPad를 컴퓨터에 연결합니다.
3. iTunes에서 iOS 기기를 클릭합니다.
4. 왼쪽 사이드바에서 "앱" 섹션으로 이동합니다.
5. 페이지 하단의 "파일 공유" 섹션까지 스크롤합니다.
6. 앱을 선택하여 기기에서 공유 가능한 파일을 확인합니다.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### 컴퓨터에서 앱으로 파일 복사   

1. iTunes를 열고 파일 공유 섹션의 목록에서 앱을 선택합니다.
2. 기기에 파일을 복사하려면 폴더나 창에서 "문서" 목록으로 파일을 드래그 앤 드롭합니다.
3. 또는 iTunes의 "문서" 목록에서 "추가"를 클릭하고 컴퓨터에서 오디오 파일을 찾은 다음 "추가"를 클릭합니다. iTunes가 이 음악 파일을 기기에서 선택한 앱의 "문서" 디렉토리로 복사합니다. 앱과 호환되는 오디오 파일을 선택해야 합니다. 지원되는 형식은 사용 설명서를 참조하세요.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### iOS 앱에서 컴퓨터로 오디오 파일 복사   

1. iTunes 파일 공유 섹션의 목록에서 앱을 선택합니다.
2. 컴퓨터로 파일을 복사하려면 "문서" 목록에서 컴퓨터의 폴더나 창으로 파일을 드래그합니다.
3. 또는 "문서" 목록에서 컴퓨터로 복사할 오디오 파일을 선택하고 "저장"을 클릭한 다음 컴퓨터에서 대상 폴더를 선택합니다.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 공유된 오디오 파일 접근   

컴퓨터에서 iOS 기기로 파일을 성공적으로 복사한 후 쉽게 재생할 수 있습니다.

- 파일 공유에 선택한 앱(예: Evermusic 또는 Flacbox)을 엽니다.
- "로컬 파일" 섹션을 탭하여 사용 가능한 파일 목록을 확인합니다.
- 파일을 탭하면 재생이 시작됩니다.

## iOS 기기에서 공유 파일 삭제   

파일 공유를 사용하여 컴퓨터에서 iOS 앱으로 복사한 오디오 파일을 삭제하려면 다음 단계를 따르세요:

#### iTunes 사용  

1. Mac/PC에서 iTunes를 엽니다.
2. USB 케이블을 사용하여 iOS 기기를 컴퓨터에 연결합니다.
3. iTunes에서 기기를 선택합니다.
4. 왼쪽 사이드바에서 "앱"을 클릭하고 파일 공유 섹션까지 스크롤합니다.
5. 삭제할 파일이 있는 앱(예: Evermusic 또는 Flacbox)을 선택합니다.
6. "문서" 목록에서 삭제할 오디오 파일을 선택한 다음 키보드에서 "삭제하기" 키를 누릅니다.
7. iTunes에서 삭제를 확인합니다.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### 내장 파일 관리자 사용  

1. "로컬 파일" 섹션을 엽니다.
2. 오른쪽 상단의 "추가 작업" 버튼을 탭하고 "선택하다" 메뉴 항목을 선택합니다.
3. 삭제할 파일을 선택합니다.
4. "삭제하기" 작업을 탭합니다. 이 작업이 바로 보이지 않으면 "추가 작업" 메뉴를 탭하세요.
5. 팝업 메시지에서 삭제를 확인합니다.

{{< figure src="/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Evermusic: Select Files" caption="Evermusic: Select Files" width="400" >}}

## 보너스 팁   

SMB 프로토콜을 사용하여 컴퓨터를 연결하면 MAC/PC에서 iOS 기기로 직접 음악을 스트리밍할 수도 있습니다. 자세한 안내는 여기를 참조하세요: [SMB를 사용하여 MAC 또는 PC에서 iPhone으로 음악을 스트리밍하는 방법](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).

## FAQ

{{% details title="iPhone으로 음악을 전송하는 가장 빠른 방법은 무엇인가요?" closed="true" %}}
USB를 통한 iTunes/Finder 파일 공유가 대용량 음악 라이브러리에 가장 빠른 방법입니다. 소량의 전송에는 케이블이 필요 없는 WiFi Drive가 더 편리합니다.
{{% /details %}}

{{% details title="iPhone에 FLAC 파일을 전송할 수 있나요?" closed="true" %}}
네. Evermusic과 Flacbox 모두 iTunes 파일 공유, WiFi Drive 또는 SMB를 통해 FLAC 파일을 지원합니다. 무손실 형식에는 Flacbox를 권장합니다.
{{% /details %}}

{{% details title="macOS Catalina 이상에서 iTunes가 필요한가요?" closed="true" %}}
아니요. Apple은 macOS Catalina부터 기기 관리를 위해 iTunes를 Finder로 대체했습니다. 파일 공유에는 Finder의 파일 탭을 사용하세요.
{{% /details %}}

{{% details title="파일을 iPhone에 복사하지 않고 음악을 스트리밍할 수 있나요?" closed="true" %}}
네. SMB 프로토콜을 사용하여 Mac 또는 PC에서 직접 음악을 스트리밍할 수 있습니다. 이렇게 하면 기기 저장 공간을 절약하고 라이브러리를 컴퓨터에 유지할 수 있습니다.
{{% /details %}}

{{% details title="Evermusic과 Flacbox 중 어떤 앱을 사용해야 하나요?" closed="true" %}}
MP3, WAV, AAC 같은 표준 형식에는 Evermusic을 사용하세요. 라이브러리에 FLAC, DSD 또는 OGG 같은 무손실 형식이 포함되어 있다면 Flacbox를 선택하세요.
{{% /details %}}

**Rezumat:** Transferați muzică de pe computer pe iPhone folosind una dintre cele trei metode: **iTunes/Finder File Sharing** (cablu USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (wireless, fără cablu necesar), sau **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (streaming direct fără copiere). Apoi redați cu [Evermusic](/products/evermusic) sau [Flacbox](/products/flacbox).

Gestionarea fișierelor muzicale între computer și dispozitivele iOS nu a fost niciodată mai simplă. Am descris trei metode ușoare -- iTunes File Sharing, WiFi Drive și protocolul SMB -- pentru a asigura un transfer muzical lin și o redare fără întreruperi.

## iTunes File Sharing

iTunes File Sharing vă permite să copiați fără efort fișiere audio între computer și aplicațiile iOS de pe iPhone, iPad sau iPod touch. Această abordare este perfectă pentru cei care doresc să se bucure de albumele, artiștii și piesele lor preferate de pe Mac/PC pe dispozitivul iOS.

**Înainte de a începe**

- Alternativ, puteți folosi [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) pentru a copia sau muta wireless fișiere audio între computer și aplicații precum Evermusic și Flacbox. Asigurați-vă că computerul și dispozitivul iOS sunt conectate la aceeași rețea WiFi, și puteți transfera ușor fișiere între aceste aplicații.
- O altă tehnologie de luat în considerare este [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), care vă permite să transmiteți muzică direct de pe computerul de acasă pe dispozitivul iOS, eliminând necesitatea de a copia fișiere audio.

## macOS Catalina sau ulterior

Cu macOS Catalina sau o versiune mai nouă, puteți folosi Finder pentru a partaja fișiere între dispozitivele iOS și iPadOS și Mac-ul dvs.

**Cerințe**  

- macOS Catalina sau ulterior
- Un iPhone, iPad sau iPod touch cu cea mai recentă versiune iOS sau iPadOS
- O aplicație iOS sau iPadOS compatibilă cu File Sharing

### Cum să folosiți Finder pentru iTunes File Sharing macOS Catalina sau ulterior

1. Deschideți o fereastră Finder.
2. Conectați iPhone-ul, iPad-ul sau iPod touch-ul la computer folosind un cablu USB.
3. Selectați dispozitivul în Finder.
4. Faceți clic pe fila "Fișiere" pentru a vedea o listă de aplicații capabile să partajeze fișiere. Dacă nu vedeți această secțiune, înseamnă că dispozitivul nu suportă aplicații de partajare fișiere.
5. Faceți clic pe triunghiul de lângă o aplicație pentru a vedea fișierele disponibile pentru partajare.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Copierea fișierelor de pe Mac pe dispozitiv

1. Selectați fișierele pe care doriți să le copiați, asigurându-vă că sunt compatibile cu aplicația. Consultați ghidul utilizatorului aplicației pentru formatele suportate.
2. Trageți și plasați fișierele pe pictograma aplicației de pe dispozitiv. Finder se va ocupa de copierea fișierelor pe dispozitiv.

#### Copierea fișierelor de pe dispozitiv pe Mac

1. Selectați fișierele pe care doriți să le copiați.
2. Trageți fișierele într-un folder de pe computer, iar Finder va efectua operațiunea de copiere.

## macOS anterior Catalina

Dacă folosiți o versiune mai veche de macOS, puteți folosi iTunes pentru File Sharing.

**Cerințe**  

- Asigurați-vă că aveți cea mai recentă versiune de iTunes.
- iPhone-ul, iPad-ul sau iPod touch-ul trebuie să ruleze cea mai recentă versiune iOS.
- Dacă folosiți un PC Windows, asigurați-vă că este actualizat cel puțin la Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Cum să folosiți iTunes File Sharing pentru versiuni macOS anterioare Catalina 

1. Deschideți iTunes pe Mac sau PC.
2. Conectați iPhone-ul sau iPad-ul la computer prin cablu USB lightning.
3. Faceți clic pe dispozitivul iOS din iTunes.
4. Navigați la secțiunea "Aplicații" din bara laterală stângă.
5. Derulați în jos până la secțiunea "File Sharing" din partea de jos a paginii.
6. Selectați aplicația pentru a vedea ce fișiere sunt disponibile pentru partajare pe dispozitiv.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Copierea fișierelor de pe computer în aplicație   

1. Deschideți iTunes și alegeți aplicația din lista din secțiunea File Sharing.
2. Pentru a copia fișiere pe dispozitiv, pur și simplu trageți și plasați-le dintr-un folder sau fereastră în lista "Documente".
3. Alternativ, faceți clic pe "Adăugare" în lista "Documente" din iTunes, localizați fișierele audio pe computer și faceți clic pe "Adăugare". iTunes va copia aceste fișiere muzicale în directorul "Documente" al aplicației alese de pe dispozitiv. Asigurați-vă că alegeți fișiere audio compatibile cu aplicația; consultați ghidul utilizatorului pentru formatele suportate.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Copierea fișierelor audio dintr-o aplicație iOS pe computer   

1. Selectați aplicația din lista din secțiunea File Sharing a iTunes.
2. Pentru a copia fișiere pe computer, trageți-le din lista "Documente" într-un folder sau fereastră de pe computer.
3. Alternativ, selectați fișierele audio pe care doriți să le copiați pe computer din lista "Documente", faceți clic pe "Salvare în" și alegeți folderul de destinație de pe computer.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Accesarea fișierelor audio partajate   

După ce ați copiat cu succes fișierele de pe computer pe dispozitivul iOS, le puteți reda cu ușurință.

- Deschideți aplicația selectată pentru File Sharing, precum Evermusic sau Flacbox.
- Atingeți secțiunea "Fișiere locale" pentru a vedea lista fișierelor disponibile.
- Atingeți orice fișier pentru a începe redarea.

## Ștergerea fișierelor partajate de pe dispozitivul iOS   

Pentru a elimina fișierele audio copiate de pe computer într-o aplicație iOS folosind File Sharing, urmați acești pași:

#### Folosind iTunes  

1. Deschideți iTunes pe Mac/PC.
2. Conectați dispozitivul iOS la computer folosind cablul USB.
3. Selectați dispozitivul în iTunes.
4. Faceți clic pe "Aplicații" în bara laterală stângă și derulați în jos până la secțiunea File Sharing.
5. Alegeți aplicația cu fișierul pe care doriți să-l ștergeți, precum Evermusic sau Flacbox.
6. Selectați fișierul audio pe care doriți să-l eliminați din lista "Documente", apoi apăsați tasta "Șterge" de pe tastatură.
7. Confirmați ștergerea în iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Folosind managerul de fișiere integrat  

1. Deschideți secțiunea "Fișiere locale".
2. Atingeți butonul "Mai multe acțiuni" din colțul din dreapta sus și selectați elementul de meniu "Selectați".
3. Alegeți fișierele pe care doriți să le ștergeți.
4. Atingeți acțiunea "Șterge". Dacă această acțiune nu este vizibilă imediat, atingeți meniul "Mai multe acțiuni".
5. Confirmați ștergerea în mesajul pop-up.

## FAQ

{{% details title="Care este cea mai rapidă modalitate de a transfera muzică pe iPhone?" closed="true" %}}
iTunes/Finder File Sharing prin USB este cea mai rapidă metodă pentru biblioteci muzicale mari. Pentru transferuri mai mici, WiFi Drive este mai convenabil deoarece nu necesită cablu.
{{% /details %}}

{{% details title="Pot transfera fișiere FLAC pe iPhone-ul meu?" closed="true" %}}
Da. Atât Evermusic cât și Flacbox acceptă fișiere FLAC prin iTunes File Sharing, WiFi Drive sau SMB. Flacbox este recomandat pentru formatele fără pierderi.
{{% /details %}}

{{% details title="Am nevoie de iTunes pe macOS Catalina sau ulterior?" closed="true" %}}
Nu. Apple a înlocuit iTunes cu Finder pentru gestionarea dispozitivelor începând cu macOS Catalina. Folosiți fila Fișiere din Finder pentru partajarea fișierelor.
{{% /details %}}

{{% details title="Pot transmite muzică fără a copia fișiere pe iPhone?" closed="true" %}}
Da. Folosiți protocolul SMB pentru a transmite muzică direct de pe Mac sau PC. Aceasta economisește spațiul de stocare al dispozitivului și păstrează biblioteca pe computer.
{{% /details %}}

{{% details title="Ce aplicație ar trebui să folosesc -- Evermusic sau Flacbox?" closed="true" %}}
Folosiți Evermusic pentru formate standard precum MP3, WAV și AAC. Alegeți Flacbox dacă biblioteca dvs. include formate fără pierderi precum FLAC, DSD sau OGG.
{{% /details %}}

**สรุป:** ถ่ายโอนเพลงจากคอมพิวเตอร์ไปยัง iPhone โดยใช้หนึ่งในสามวิธี: **iTunes/Finder File Sharing** (สาย USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (ไร้สาย ไม่ต้องใช้สาย), หรือ **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (สตรีมโดยตรงโดยไม่ต้องคัดลอก) จากนั้นเล่นด้วย [Evermusic](/products/evermusic) หรือ [Flacbox](/products/flacbox)

การจัดการไฟล์เพลงระหว่างคอมพิวเตอร์และอุปกรณ์ iOS ไม่เคยง่ายขนาดนี้มาก่อน เราได้สรุปสามวิธีง่าย ๆ -- iTunes File Sharing, WiFi Drive และโปรโตคอล SMB -- เพื่อให้มั่นใจว่าการถ่ายโอนเพลงราบรื่นและการเล่นไร้รอยต่อ

## iTunes File Sharing

iTunes File Sharing ช่วยให้คุณคัดลอกไฟล์เสียงระหว่างคอมพิวเตอร์กับแอป iOS บน iPhone, iPad หรือ iPod touch ได้อย่างง่ายดาย วิธีนี้เหมาะสำหรับผู้ที่ต้องการเพลิดเพลินกับอัลบั้ม ศิลปิน และเพลงโปรดจาก Mac/PC บนอุปกรณ์ iOS

**ก่อนเริ่มต้น**

- หรือคุณสามารถใช้ [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) เพื่อคัดลอกหรือย้ายไฟล์เสียงแบบไร้สายระหว่างคอมพิวเตอร์กับแอปเช่น Evermusic และ Flacbox เพียงตรวจสอบให้แน่ใจว่าคอมพิวเตอร์และอุปกรณ์ iOS เชื่อมต่อกับเครือข่าย WiFi เดียวกัน และคุณจะสามารถถ่ายโอนไฟล์ระหว่างแอปเหล่านี้ได้อย่างง่ายดาย
- อีกเทคโนโลยีหนึ่งที่ควรพิจารณาคือ [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/) ซึ่งช่วยให้คุณสตรีมเพลงจากคอมพิวเตอร์ที่บ้านไปยังอุปกรณ์ iOS ได้โดยตรง โดยไม่ต้องคัดลอกไฟล์เสียง

## macOS Catalina หรือใหม่กว่า

ด้วย macOS Catalina หรือเวอร์ชันที่ใหม่กว่า คุณสามารถใช้ Finder เพื่อแชร์ไฟล์ระหว่างอุปกรณ์ iOS และ iPadOS กับ Mac ของคุณ

**ข้อกำหนด**  

- macOS Catalina หรือใหม่กว่า
- iPhone, iPad หรือ iPod touch ที่มี iOS หรือ iPadOS เวอร์ชันล่าสุด
- แอป iOS หรือ iPadOS ที่รองรับ File Sharing

### วิธีใช้ Finder สำหรับ iTunes File Sharing macOS Catalina หรือใหม่กว่า

1. เปิดหน้าต่าง Finder
2. เชื่อมต่อ iPhone, iPad หรือ iPod touch กับคอมพิวเตอร์โดยใช้สาย USB
3. เลือกอุปกรณ์ของคุณใน Finder
4. คลิกแท็บ "ไฟล์" เพื่อดูรายชื่อแอปที่สามารถแชร์ไฟล์ได้ หากคุณไม่เห็นส่วนนี้ หมายความว่าอุปกรณ์ของคุณไม่รองรับแอปแชร์ไฟล์
5. คลิกสามเหลี่ยมข้างแอปเพื่อดูไฟล์ที่พร้อมแชร์

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### คัดลอกไฟล์จาก Mac ไปยังอุปกรณ์

1. เลือกไฟล์ที่ต้องการคัดลอก โดยตรวจสอบให้แน่ใจว่าเข้ากันได้กับแอป ดูคู่มือผู้ใช้ของแอปสำหรับรูปแบบที่รองรับ
2. ลากและวางไฟล์ลงบนไอคอนแอปบนอุปกรณ์ของคุณ Finder จะคัดลอกไฟล์ไปยังอุปกรณ์

#### คัดลอกไฟล์จากอุปกรณ์ไปยัง Mac

1. เลือกไฟล์ที่ต้องการคัดลอก
2. ลากไฟล์ไปยังโฟลเดอร์บนคอมพิวเตอร์ และ Finder จะทำการคัดลอก

## macOS เก่ากว่า Catalina

หากคุณใช้ macOS เวอร์ชันเก่า คุณสามารถใช้ iTunes สำหรับ File Sharing

**ข้อกำหนด**  

- ตรวจสอบให้แน่ใจว่าคุณมี iTunes เวอร์ชันล่าสุด
- iPhone, iPad หรือ iPod touch ควรใช้ iOS เวอร์ชันล่าสุด
- หากคุณใช้ Windows PC ตรวจสอบให้แน่ใจว่าอัปเดตเป็นอย่างน้อย Windows 7

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### วิธีใช้ iTunes File Sharing สำหรับ macOS เวอร์ชันเก่ากว่า Catalina 

1. เปิด iTunes บน Mac หรือ PC
2. เชื่อมต่อ iPhone หรือ iPad กับคอมพิวเตอร์ผ่านสาย USB lightning
3. คลิกที่อุปกรณ์ iOS ของคุณใน iTunes
4. ไปที่ส่วน "แอป" ในแถบด้านข้างซ้าย
5. เลื่อนลงไปที่ส่วน "File Sharing" ด้านล่างของหน้า
6. เลือกแอปเพื่อดูว่าไฟล์ใดบ้างที่พร้อมแชร์บนอุปกรณ์ของคุณ

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### คัดลอกไฟล์จากคอมพิวเตอร์ไปยังแอป   

1. เปิด iTunes และเลือกแอปจากรายการในส่วน File Sharing
2. หากต้องการคัดลอกไฟล์ไปยังอุปกรณ์ ให้ลากและวางจากโฟลเดอร์หรือหน้าต่างไปยังรายการ "เอกสาร"
3. หรือคลิก "เพิ่ม" ในรายการ "เอกสาร" ใน iTunes ค้นหาไฟล์เสียงบนคอมพิวเตอร์ และคลิก "เพิ่ม" iTunes จะคัดลอกไฟล์เพลงเหล่านี้ไปยังไดเรกทอรี "เอกสาร" ของแอปที่เลือกบนอุปกรณ์ ตรวจสอบให้แน่ใจว่าเลือกไฟล์เสียงที่เข้ากันได้กับแอป ดูคู่มือผู้ใช้สำหรับรูปแบบที่รองรับ

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### คัดลอกไฟล์เสียงจากแอป iOS ไปยังคอมพิวเตอร์   

1. เลือกแอปจากรายการในส่วน File Sharing ของ iTunes
2. หากต้องการคัดลอกไฟล์ไปยังคอมพิวเตอร์ ให้ลากจากรายการ "เอกสาร" ไปยังโฟลเดอร์หรือหน้าต่างบนคอมพิวเตอร์
3. หรือเลือกไฟล์เสียงที่ต้องการคัดลอกไปยังคอมพิวเตอร์จากรายการ "เอกสาร" คลิก "บันทึกไปยัง" และเลือกโฟลเดอร์ปลายทางบนคอมพิวเตอร์

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## การเข้าถึงไฟล์เสียงที่แชร์   

เมื่อคุณคัดลอกไฟล์จากคอมพิวเตอร์ไปยังอุปกรณ์ iOS สำเร็จแล้ว คุณสามารถเล่นได้อย่างง่ายดาย

- เปิดแอปที่คุณเลือกสำหรับ File Sharing เช่น Evermusic หรือ Flacbox
- แตะที่ส่วน "ไฟล์ในเครื่อง" เพื่อดูรายการไฟล์ที่มีอยู่
- แตะที่ไฟล์ใดก็ได้เพื่อเริ่มเล่น

## การลบไฟล์ที่แชร์จากอุปกรณ์ iOS   

หากต้องการลบไฟล์เสียงที่คัดลอกจากคอมพิวเตอร์ไปยังแอป iOS โดยใช้ File Sharing ให้ทำตามขั้นตอนเหล่านี้:

#### ใช้ iTunes  

1. เปิด iTunes บน Mac/PC
2. เชื่อมต่ออุปกรณ์ iOS กับคอมพิวเตอร์โดยใช้สาย USB
3. เลือกอุปกรณ์ของคุณใน iTunes
4. คลิก "แอป" ในแถบด้านข้างซ้ายและเลื่อนลงไปที่ส่วน File Sharing
5. เลือกแอปที่มีไฟล์ที่ต้องการลบ เช่น Evermusic หรือ Flacbox
6. เลือกไฟล์เสียงที่ต้องการลบจากรายการ "เอกสาร" จากนั้นกดปุ่ม "ลบ" บนแป้นพิมพ์
7. ยืนยันการลบใน iTunes

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### ใช้ตัวจัดการไฟล์ในตัว  

1. เปิดส่วน "ไฟล์ในเครื่อง"
2. แตะปุ่ม "ดำเนินการเพิ่มเติม" ที่มุมขวาบน และเลือกรายการเมนู "เลือก"
3. เลือกไฟล์ที่ต้องการลบ
4. แตะการดำเนินการ "ลบ" หากการดำเนินการนี้ไม่ปรากฏทันที ให้แตะเมนู "ดำเนินการเพิ่มเติม"
5. ยืนยันการลบในข้อความป๊อปอัป

{{< figure src="/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Evermusic: Select Files" caption="Evermusic: Select Files" width="400" >}}

## เคล็ดลับพิเศษ   

คุณยังสามารถเชื่อมต่อคอมพิวเตอร์โดยใช้โปรโตคอล SMB เพื่อสตรีมเพลงจาก MAC/PC ไปยังอุปกรณ์ iOS ได้โดยตรง ดูคำแนะนำโดยละเอียดที่นี่: [วิธีสตรีมเพลงจาก MAC หรือ PC ไปยัง iPhone โดยใช้ SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)

**In breve:** Trasferisci musica dal tuo computer all'iPhone usando uno dei tre metodi: **Condivisione file iTunes/Finder** (cavo USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (wireless, senza cavo), o **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (streaming diretto senza copiare). Poi riproduci con [Evermusic](/products/evermusic) o [Flacbox](/products/flacbox).

Gestire i tuoi file musicali tra il computer e i dispositivi iOS non è mai stato così semplice. Abbiamo descritto tre metodi facili: condivisione file di iTunes, WiFi Drive e protocollo SMB, per garantire un trasferimento musicale fluido e una riproduzione senza interruzioni.

## Condivisione file di iTunes

La condivisione file di iTunes ti permette di copiare senza sforzo file audio tra il tuo computer e le tue app iOS su iPhone, iPad o iPod touch. Questo approccio è perfetto per chi vuole godersi i propri album, artisti e brani preferiti dal Mac/PC sul proprio dispositivo iOS.

**Prima di iniziare**

- In alternativa, puoi usare [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) per copiare o spostare file audio in modalità wireless tra il tuo computer e app come Evermusic e Flacbox. Assicurati semplicemente che il tuo computer e il dispositivo iOS siano connessi alla stessa rete WiFi, e potrai facilmente trasferire file tra queste app.
- Un'altra tecnologia da considerare è [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), che ti permette di riprodurre musica in streaming direttamente dal tuo computer di casa al tuo dispositivo iOS, eliminando la necessità di copiare file audio.

## macOS Catalina o successivo

Con macOS Catalina o una versione più recente, puoi usare il Finder per condividere file tra i tuoi dispositivi iOS e iPadOS e il tuo Mac.

**Requisiti**  

- macOS Catalina o successivo
- Un iPhone, iPad o iPod touch con l'ultima versione di iOS o iPadOS
- Un'app iOS o iPadOS compatibile con la condivisione file

### Come usare il Finder per la condivisione file di iTunes su macOS Catalina o successivo

1. Apri una finestra del Finder.
2. Collega il tuo iPhone, iPad o iPod touch al computer usando un cavo USB.
3. Seleziona il tuo dispositivo nel Finder.
4. Clicca sulla scheda "File" per vedere un elenco di app in grado di condividere file. Se non vedi questa sezione, significa che il tuo dispositivo non supporta le app di condivisione file.
5. Clicca sul triangolo accanto a un'app per visualizzare i file disponibili per la condivisione.

![Come usare il Finder per la condivisione file](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Copiare file dal Mac al dispositivo

1. Seleziona i file che vuoi copiare, assicurandoti che siano compatibili con l'app. Consulta la guida utente dell'app per i formati supportati.
2. Trascina e rilascia i file sull'icona dell'app sul tuo dispositivo. Il Finder si occuperà di copiare i file sul tuo dispositivo.

#### Copiare file dal dispositivo al Mac

1. Seleziona i file che vuoi copiare.
2. Trascina i file in una cartella sul tuo computer, e il Finder eseguirà l'operazione di copia.

## macOS precedente a Catalina

Se stai usando una versione precedente di macOS, puoi usare iTunes per la condivisione file.

**Requisiti**  

- Assicurati di avere l'ultima versione di iTunes.
- Il tuo iPhone, iPad o iPod touch dovrebbe avere l'ultima versione di iOS.
- Se usi un PC Windows, assicurati che sia aggiornato almeno a Windows 7.

![Condivisione file di iTunes](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Come usare la condivisione file di iTunes per versioni di macOS precedenti a Catalina

1. Apri iTunes sul tuo Mac o PC.
2. Collega il tuo iPhone o iPad al computer tramite cavo USB lightning.
3. Clicca sul tuo dispositivo iOS in iTunes.
4. Naviga alla sezione "App" nella barra laterale sinistra.
5. Scorri verso il basso fino alla sezione "Condivisione file" in fondo alla pagina.
6. Seleziona l'app per vedere quali file sono disponibili per la condivisione sul tuo dispositivo.

![condivisione file itunes seleziona app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Copiare file dal computer all'app   

1. Apri iTunes e scegli l'app dall'elenco nella sezione Condivisione file.
2. Per copiare file sul tuo dispositivo, trascinali semplicemente da una cartella o finestra nell'elenco "Documenti".
3. In alternativa, clicca "Aggiungi" nell'elenco "Documenti" in iTunes, individua i file audio sul tuo computer e clicca "Aggiungi". iTunes copierà questi file musicali nella directory "Documenti" dell'app scelta sul tuo dispositivo. Assicurati di scegliere file audio compatibili con l'app; consulta la guida utente per i formati supportati.

![condivisione file itunes sposta file](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Copiare file audio da un'app iOS al computer   

1. Seleziona l'app dall'elenco nella sezione Condivisione file di iTunes.
2. Per copiare file sul computer, trascinali dall'elenco "Documenti" in una cartella o finestra sul tuo computer.
3. In alternativa, seleziona i file audio che vuoi copiare sul computer dall'elenco "Documenti", clicca "Salva in" e scegli la cartella di destinazione sul tuo computer.

![Condivisione file di iTunes: Copiare file audio da un'app iOS al computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Accesso ai file audio condivisi   

Una volta copiati con successo i file dal computer al dispositivo iOS, puoi riprodurli con facilità.

- Apri l'app che hai selezionato per la condivisione file, come Evermusic o Flacbox.
- Tocca la sezione "File locali" per visualizzare l'elenco dei file disponibili.
- Tocca semplicemente qualsiasi file per iniziare la riproduzione.

## Eliminazione dei file condivisi dal dispositivo iOS   

Per rimuovere file audio copiati dal computer a un'app iOS tramite la condivisione file, segui questi passaggi:

#### Usando iTunes  

1. Apri iTunes sul tuo Mac/PC.
2. Collega il tuo dispositivo iOS al computer usando il cavo USB.
3. Seleziona il tuo dispositivo in iTunes.
4. Clicca "App" nella barra laterale sinistra e scorri verso il basso fino alla sezione Condivisione file.
5. Scegli l'app con il file che vuoi eliminare, come Evermusic o Flacbox.
6. Seleziona il file audio che vuoi rimuovere dall'elenco "Documenti", poi premi il tasto "Eliminare" sulla tastiera.
7. Conferma l'eliminazione in iTunes.

![Condivisione file di iTunes: Elimina file](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Usando il file manager integrato  

1. Apri la sezione "File locali".
2. Tocca il pulsante "Altre azioni" nell'angolo in alto a destra e seleziona la voce di menu "Selezionare".
3. Scegli i file che vuoi eliminare.
4. Tocca l'azione "Eliminare". Se questa azione non è immediatamente visibile, tocca il menu "Altre azioni".
5. Conferma l'eliminazione nel messaggio pop-up.

## FAQ

{{% details title="Qual è il modo più veloce per trasferire musica sull'iPhone?" closed="true" %}}
La condivisione file iTunes/Finder tramite USB è il metodo più veloce per grandi librerie musicali. Per trasferimenti più piccoli, WiFi Drive è più comodo poiché non richiede un cavo.
{{% /details %}}

{{% details title="Posso trasferire file FLAC sul mio iPhone?" closed="true" %}}
Sì. Sia Evermusic che Flacbox accettano file FLAC tramite condivisione file iTunes, WiFi Drive o SMB. Flacbox è consigliato per i formati lossless.
{{% /details %}}

{{% details title="Ho bisogno di iTunes su macOS Catalina o successivo?" closed="true" %}}
No. Apple ha sostituito iTunes con il Finder per la gestione dei dispositivi a partire da macOS Catalina. Usa la scheda File del Finder per la condivisione file.
{{% /details %}}

{{% details title="Posso riprodurre musica in streaming senza copiare file sul mio iPhone?" closed="true" %}}
Sì. Usa il protocollo SMB per riprodurre musica in streaming direttamente dal tuo Mac o PC. Questo risparmia spazio di archiviazione sul dispositivo e mantiene la tua libreria sul computer.
{{% /details %}}

{{% details title="Quale app dovrei usare -- Evermusic o Flacbox?" closed="true" %}}
Usa Evermusic per formati standard come MP3, WAV e AAC. Scegli Flacbox se la tua libreria include formati lossless come FLAC, DSD o OGG.
{{% /details %}}

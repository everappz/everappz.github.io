**En bref :** Transférez de la musique de votre ordinateur vers l'iPhone en utilisant l'une des trois méthodes : **Partage de fichiers iTunes/Finder** (câble USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (sans fil, sans câble), ou **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (streaming direct sans copie). Puis lisez avec [Evermusic](/products/evermusic) ou [Flacbox](/products/flacbox).

La gestion de vos fichiers musicaux entre votre ordinateur et vos appareils iOS n'a jamais été aussi simple. Nous avons décrit trois méthodes faciles : le partage de fichiers iTunes, WiFi Drive et le protocole SMB, pour garantir un transfert musical fluide et une lecture sans interruption.

## Partage de fichiers iTunes

Le partage de fichiers iTunes vous permet de copier sans effort des fichiers audio entre votre ordinateur et vos applications iOS sur votre iPhone, iPad ou iPod touch. Cette approche est parfaite pour ceux qui souhaitent profiter de leurs albums, artistes et morceaux préférés depuis leur Mac/PC sur leur appareil iOS.

**Avant de commencer**

- Alternativement, vous pouvez utiliser [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) pour copier ou déplacer sans fil des fichiers audio entre votre ordinateur et des applications comme Evermusic et Flacbox. Assurez-vous simplement que votre ordinateur et votre appareil iOS sont connectés au même réseau WiFi, et vous pourrez facilement transférer des fichiers entre ces applications.
- Une autre technologie à considérer est [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), qui vous permet de diffuser de la musique directement depuis votre ordinateur personnel vers votre appareil iOS, éliminant le besoin de copier des fichiers audio.

## macOS Catalina ou ultérieur

Avec macOS Catalina ou une version plus récente, vous pouvez utiliser le Finder pour partager des fichiers entre vos appareils iOS et iPadOS et votre Mac.

**Prérequis**  

- macOS Catalina ou ultérieur
- Un iPhone, iPad ou iPod touch avec la dernière version d'iOS ou iPadOS
- Une application iOS ou iPadOS compatible avec le partage de fichiers

### Comment utiliser le Finder pour le partage de fichiers iTunes sur macOS Catalina ou ultérieur

1. Ouvrez une fenêtre Finder.
2. Connectez votre iPhone, iPad ou iPod touch à votre ordinateur à l'aide d'un câble USB.
3. Sélectionnez votre appareil dans le Finder.
4. Cliquez sur l'onglet "Fichiers" pour voir la liste des applications capables de partager des fichiers. Si vous ne voyez pas cette section, cela signifie que votre appareil ne prend pas en charge les applications de partage de fichiers.
5. Cliquez sur le triangle à côté d'une application pour voir les fichiers disponibles au partage.

![Comment utiliser le Finder pour le partage de fichiers](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Copier des fichiers de votre Mac vers l'appareil

1. Sélectionnez les fichiers que vous souhaitez copier, en vous assurant qu'ils sont compatibles avec l'application. Consultez le guide d'utilisation de l'application pour les formats pris en charge.
2. Glissez-déposez les fichiers sur l'icône de l'application sur votre appareil. Le Finder se chargera de copier les fichiers sur votre appareil.

#### Copier des fichiers de votre appareil vers le Mac

1. Sélectionnez les fichiers que vous souhaitez copier.
2. Faites glisser les fichiers vers un dossier sur votre ordinateur, et le Finder effectuera l'opération de copie.

## macOS antérieur à Catalina

Si vous utilisez une version plus ancienne de macOS, vous pouvez utiliser iTunes pour le partage de fichiers.

**Prérequis**  

- Assurez-vous d'avoir la dernière version d'iTunes.
- Votre iPhone, iPad ou iPod touch doit exécuter la dernière version d'iOS.
- Si vous utilisez un PC Windows, assurez-vous qu'il est mis à jour au moins vers Windows 7.

![Partage de fichiers iTunes](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Comment utiliser le partage de fichiers iTunes pour les versions de macOS antérieures à Catalina

1. Ouvrez iTunes sur votre Mac ou PC.
2. Connectez votre iPhone ou iPad à votre ordinateur via un câble USB lightning.
3. Cliquez sur votre appareil iOS dans iTunes.
4. Naviguez vers la section "Apps" dans la barre latérale gauche.
5. Faites défiler vers le bas jusqu'à la section "Partage de fichiers" en bas de la page.
6. Sélectionnez l'application pour voir quels fichiers sont disponibles au partage sur votre appareil.

![partage de fichiers itunes sélectionner l'application](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Copier des fichiers de votre ordinateur vers l'application   

1. Ouvrez iTunes et choisissez l'application dans la liste de la section Partage de fichiers.
2. Pour copier des fichiers sur votre appareil, glissez-déposez-les simplement depuis un dossier ou une fenêtre dans la liste "Documents".
3. Alternativement, cliquez sur "Ajouter" dans la liste "Documents" dans iTunes, localisez les fichiers audio sur votre ordinateur et cliquez sur "Ajouter". iTunes copiera ces fichiers musicaux dans le répertoire "Documents" de l'application choisie sur votre appareil. Assurez-vous de choisir des fichiers audio compatibles avec l'application ; consultez le guide d'utilisation pour les formats pris en charge.

![partage de fichiers itunes déplacer des fichiers](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Copier des fichiers audio d'une application iOS vers votre ordinateur   

1. Sélectionnez l'application dans la liste de la section Partage de fichiers d'iTunes.
2. Pour copier des fichiers sur votre ordinateur, faites-les glisser de la liste "Documents" vers un dossier ou une fenêtre sur votre ordinateur.
3. Alternativement, sélectionnez les fichiers audio que vous souhaitez copier sur votre ordinateur depuis la liste "Documents", cliquez sur "Enregistrer dans" et choisissez le dossier de destination sur votre ordinateur.

![Partage de fichiers iTunes : Copier des fichiers audio d'une application iOS vers votre ordinateur](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Accéder aux fichiers audio partagés   

Une fois que vous avez copié avec succès des fichiers de votre ordinateur vers votre appareil iOS, vous pouvez les lire facilement.

- Ouvrez l'application que vous avez sélectionnée pour le partage de fichiers, comme Evermusic ou Flacbox.
- Appuyez sur la section "Fichiers locaux" pour voir la liste des fichiers disponibles.
- Appuyez simplement sur n'importe quel fichier pour commencer la lecture.

## Supprimer des fichiers partagés de votre appareil iOS   

Pour supprimer des fichiers audio copiés depuis votre ordinateur vers une application iOS via le partage de fichiers, suivez ces étapes :

#### En utilisant iTunes  

1. Ouvrez iTunes sur votre Mac/PC.
2. Connectez votre appareil iOS à votre ordinateur à l'aide du câble USB.
3. Sélectionnez votre appareil dans iTunes.
4. Cliquez sur "Apps" dans la barre latérale gauche et faites défiler vers le bas jusqu'à la section Partage de fichiers.
5. Choisissez l'application contenant le fichier que vous souhaitez supprimer, comme Evermusic ou Flacbox.
6. Sélectionnez le fichier audio que vous souhaitez supprimer de la liste "Documents", puis appuyez sur la touche "Supprimer" de votre clavier.
7. Confirmez la suppression dans iTunes.

![Partage de fichiers iTunes : Supprimer un fichier](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### En utilisant le gestionnaire de fichiers intégré  

1. Ouvrez la section "Fichiers locaux".
2. Appuyez sur le bouton "Plus d'actions" dans le coin supérieur droit, et sélectionnez l'élément de menu "Sélectionner".
3. Choisissez les fichiers que vous souhaitez supprimer.
4. Appuyez sur l'action "Supprimer". Si cette action n'est pas immédiatement visible, appuyez sur le menu "Plus d'actions".
5. Confirmez la suppression dans le message contextuel.

## FAQ

{{% details title="Quelle est la façon la plus rapide de transférer de la musique vers l'iPhone ?" closed="true" %}}
Le partage de fichiers iTunes/Finder via USB est la méthode la plus rapide pour les grandes bibliothèques musicales. Pour les transferts plus petits, WiFi Drive est plus pratique car il ne nécessite pas de câble.
{{% /details %}}

{{% details title="Puis-je transférer des fichiers FLAC vers mon iPhone ?" closed="true" %}}
Oui. Evermusic et Flacbox acceptent tous deux les fichiers FLAC via le partage de fichiers iTunes, WiFi Drive ou SMB. Flacbox est recommandé pour les formats sans perte.
{{% /details %}}

{{% details title="Ai-je besoin d'iTunes sur macOS Catalina ou ultérieur ?" closed="true" %}}
Non. Apple a remplacé iTunes par le Finder pour la gestion des appareils à partir de macOS Catalina. Utilisez l'onglet Fichiers du Finder pour le partage de fichiers.
{{% /details %}}

{{% details title="Puis-je diffuser de la musique sans copier de fichiers sur mon iPhone ?" closed="true" %}}
Oui. Utilisez le protocole SMB pour diffuser de la musique directement depuis votre Mac ou PC. Cela économise de l'espace de stockage sur l'appareil et garde votre bibliothèque sur votre ordinateur.
{{% /details %}}

{{% details title="Quelle application devrais-je utiliser -- Evermusic ou Flacbox ?" closed="true" %}}
Utilisez Evermusic pour les formats standard comme MP3, WAV et AAC. Choisissez Flacbox si votre bibliothèque comprend des formats sans perte comme FLAC, DSD ou OGG.
{{% /details %}}

**简述：** 使用以下三种方法之一将音乐从电脑传输到iPhone：**iTunes/Finder File Sharing**（USB数据线）、**[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)**（无线，无需数据线）或 **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)**（直接流媒体播放，无需复制）。然后使用 [Evermusic](/products/evermusic) 或 [Flacbox](/products/flacbox) 播放。

在电脑和iOS设备之间管理音乐文件从未如此简单。我们介绍了三种简便方法——iTunes File Sharing、WiFi Drive和SMB协议——确保音乐传输顺畅、播放无缝。

## iTunes File Sharing

iTunes File Sharing让您可以轻松地在电脑和iPhone、iPad或iPod touch上的iOS应用之间复制音频文件。这种方法非常适合想要在iOS设备上享受Mac/PC中喜爱的专辑、艺术家和曲目的用户。

**开始之前**

- 您也可以使用 [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) 在电脑和Evermusic、Flacbox等应用之间无线复制或移动音频文件。只需确保电脑和iOS设备连接到同一WiFi网络，即可轻松在这些应用之间传输文件。
- 另一种值得考虑的技术是 [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)，它允许您从家用电脑直接向iOS设备流媒体播放音乐，无需复制音频文件。

## macOS Catalina或更高版本

使用macOS Catalina或更新版本，您可以使用Finder在iOS和iPadOS设备与Mac之间共享文件。

**要求**  

- macOS Catalina或更高版本
- 运行最新iOS或iPadOS版本的iPhone、iPad或iPod touch
- 与File Sharing兼容的iOS或iPadOS应用

### 如何在macOS Catalina或更高版本中使用Finder进行iTunes File Sharing

1. 打开Finder窗口。
2. 使用USB数据线将iPhone、iPad或iPod touch连接到电脑。
3. 在Finder中选择您的设备。
4. 点击"文件"选项卡查看能够共享文件的应用列表。如果您看不到此部分，说明您的设备不支持文件共享应用。
5. 点击应用旁边的三角形查看可共享的文件。

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### 从Mac复制文件到设备

1. 选择要复制的文件，确保它们与应用兼容。请参阅应用的用户指南了解支持的格式。
2. 将文件拖放到设备上的应用图标。Finder将把文件复制到您的设备。

#### 从设备复制文件到Mac

1. 选择要复制的文件。
2. 将文件拖到电脑上的文件夹中，Finder将执行复制操作。

## macOS Catalina之前的版本

如果您使用的是较旧的macOS版本，可以使用iTunes进行File Sharing。

**要求**  

- 确保您拥有最新版本的iTunes。
- 您的iPhone、iPad或iPod touch应运行最新的iOS版本。
- 如果您使用的是Windows PC，请确保已更新到至少Windows 7。

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### 如何在Catalina之前的macOS版本中使用iTunes File Sharing 

1. 在Mac或PC上打开iTunes。
2. 通过USB lightning数据线将iPhone或iPad连接到电脑。
3. 在iTunes中点击您的iOS设备。
4. 导航到左侧边栏中的"应用"部分。
5. 向下滚动到页面底部的"File Sharing"部分。
6. 选择应用以查看设备上可共享的文件。

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### 从电脑复制文件到应用   

1. 打开iTunes并从File Sharing部分的列表中选择应用。
2. 要将文件复制到设备，只需从文件夹或窗口拖放到"文档"列表中。
3. 或者，在iTunes的"文档"列表中点击"添加"，在电脑上找到音频文件，然后点击"添加"。iTunes将把这些音乐文件复制到设备上所选应用的"文档"目录中。确保选择与应用兼容的音频文件；请参阅用户指南了解支持的格式。

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### 从iOS应用复制音频文件到电脑   

1. 从iTunes File Sharing部分的列表中选择应用。
2. 要将文件复制到电脑，将它们从"文档"列表拖到电脑上的文件夹或窗口。
3. 或者，从"文档"列表中选择要复制到电脑的音频文件，点击"保存到"，然后选择电脑上的目标文件夹。

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 访问共享的音频文件   

成功将文件从电脑复制到iOS设备后，您可以轻松播放它们。

- 打开您选择用于File Sharing的应用，如Evermusic或Flacbox。
- 点击"本地文件"部分查看可用文件列表。
- 点击任意文件即可开始播放。

## 从iOS设备删除共享文件   

要删除使用File Sharing从电脑复制到iOS应用的音频文件，请按以下步骤操作：

#### 使用iTunes  

1. 在Mac/PC上打开iTunes。
2. 使用USB数据线将iOS设备连接到电脑。
3. 在iTunes中选择您的设备。
4. 点击左侧边栏中的"应用"，向下滚动到File Sharing部分。
5. 选择包含要删除文件的应用，如Evermusic或Flacbox。
6. 从"文档"列表中选择要删除的音频文件，然后按键盘上的"删除"键。
7. 在iTunes中确认删除。

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### 使用内置文件管理器  

1. 打开"本地文件"部分。
2. 点击右上角的"更多操作"按钮，选择"选择"菜单项。
3. 选择要删除的文件。
4. 点击"删除"操作。如果此操作未立即显示，请点击"更多操作"菜单。
5. 在弹出消息中确认删除。

## FAQ

{{% details title="将音乐传输到iPhone最快的方法是什么？" closed="true" %}}
通过USB的iTunes/Finder File Sharing是大型音乐库最快的方法。对于较小的传输，WiFi Drive更方便，因为不需要数据线。
{{% /details %}}

{{% details title="我可以将FLAC文件传输到iPhone吗？" closed="true" %}}
可以。Evermusic和Flacbox都通过iTunes File Sharing、WiFi Drive或SMB接受FLAC文件。对于无损格式，推荐使用Flacbox。
{{% /details %}}

{{% details title="在macOS Catalina或更高版本上需要iTunes吗？" closed="true" %}}
不需要。Apple从macOS Catalina开始用Finder替代了iTunes进行设备管理。使用Finder的"文件"选项卡进行文件共享。
{{% /details %}}

{{% details title="我可以在不将文件复制到iPhone的情况下流媒体播放音乐吗？" closed="true" %}}
可以。使用SMB协议直接从Mac或PC流媒体播放音乐。这样可以节省设备存储空间，并将音乐库保留在电脑上。
{{% /details %}}

{{% details title="我应该使用哪个应用——Evermusic还是Flacbox？" closed="true" %}}
对于MP3、WAV和AAC等标准格式，使用Evermusic。如果您的音乐库包含FLAC、DSD或OGG等无损格式，选择Flacbox。
{{% /details %}}

**Ringkasan:** Pindahkan muzik dari komputer anda ke iPhone menggunakan salah satu daripada tiga kaedah: **iTunes/Finder File Sharing** (kabel USB), **[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)** (tanpa wayar, tiada kabel diperlukan), atau **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)** (strim terus tanpa menyalin). Kemudian mainkan dengan [Evermusic](/products/evermusic) atau [Flacbox](/products/flacbox).

Mengurus fail muzik anda antara komputer dan peranti iOS tidak pernah semudah ini. Kami telah menggariskan tiga kaedah mudah -- iTunes File Sharing, WiFi Drive, dan protokol SMB -- untuk memastikan pemindahan muzik yang lancar dan main balik yang sempurna.

## iTunes File Sharing

iTunes File Sharing membolehkan anda menyalin fail audio dengan mudah antara komputer dan aplikasi iOS di iPhone, iPad, atau iPod touch anda. Pendekatan ini sesuai untuk mereka yang ingin menikmati album, artis, dan lagu kegemaran dari Mac/PC mereka pada peranti iOS.

**Sebelum Anda Mula**

- Sebagai alternatif, anda boleh menggunakan [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) untuk menyalin atau memindahkan fail audio secara tanpa wayar antara komputer anda dan aplikasi seperti Evermusic dan Flacbox. Pastikan komputer dan peranti iOS anda disambungkan ke rangkaian WiFi yang sama, dan anda boleh memindahkan fail dengan mudah antara aplikasi ini.
- Teknologi lain yang boleh dipertimbangkan ialah [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/), yang membolehkan anda menstrim muzik terus dari komputer rumah anda ke peranti iOS, tanpa perlu menyalin fail audio.

## macOS Catalina atau Lebih Baharu

Dengan macOS Catalina atau versi yang lebih baharu, anda boleh menggunakan Finder untuk berkongsi fail antara peranti iOS dan iPadOS anda dengan Mac.

**Keperluan**  

- macOS Catalina atau lebih baharu
- iPhone, iPad, atau iPod touch dengan versi iOS atau iPadOS terkini
- Aplikasi iOS atau iPadOS yang serasi dengan File Sharing

### Cara Menggunakan Finder untuk iTunes File Sharing macOS Catalina atau Lebih Baharu

1. Buka tetingkap Finder.
2. Sambungkan iPhone, iPad, atau iPod touch anda ke komputer menggunakan kabel USB.
3. Pilih peranti anda dalam Finder.
4. Klik tab "Fail" untuk melihat senarai aplikasi yang mampu berkongsi fail. Jika anda tidak melihat bahagian ini, peranti anda tidak menyokong aplikasi perkongsian fail.
5. Klik segitiga di sebelah aplikasi untuk melihat fail yang tersedia untuk dikongsi.

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Menyalin Fail dari Mac ke Peranti

1. Pilih fail yang ingin disalin, pastikan ia serasi dengan aplikasi. Rujuk panduan pengguna aplikasi untuk format yang disokong.
2. Seret dan lepaskan fail ke ikon aplikasi pada peranti anda. Finder akan menyalin fail ke peranti anda.

#### Menyalin Fail dari Peranti ke Mac

1. Pilih fail yang ingin disalin.
2. Seret fail ke folder di komputer anda, dan Finder akan melaksanakan operasi penyalinan.

## macOS Lebih Lama daripada Catalina

Jika anda menggunakan versi macOS yang lebih lama, anda boleh menggunakan iTunes untuk File Sharing.

**Keperluan**  

- Pastikan anda mempunyai versi iTunes terkini.
- iPhone, iPad, atau iPod touch anda perlu menjalankan versi iOS terkini.
- Jika anda menggunakan PC Windows, pastikan ia dikemas kini sekurang-kurangnya ke Windows 7.

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Cara Menggunakan iTunes File Sharing untuk Versi macOS Lebih Lama daripada Catalina 

1. Buka iTunes pada Mac atau PC anda.
2. Sambungkan iPhone atau iPad anda ke komputer melalui kabel USB lightning.
3. Klik pada peranti iOS anda dalam iTunes.
4. Navigasi ke bahagian "Aplikasi" di bar sisi kiri.
5. Tatal ke bawah ke bahagian "File Sharing" di bahagian bawah halaman.
6. Pilih aplikasi untuk melihat fail yang tersedia untuk dikongsi pada peranti anda.

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### Menyalin Fail dari Komputer ke Aplikasi   

1. Buka iTunes dan pilih aplikasi dari senarai dalam bahagian File Sharing.
2. Untuk menyalin fail ke peranti anda, seret dan lepaskan dari folder atau tetingkap ke senarai "Dokumen".
3. Sebagai alternatif, klik "Tambah" dalam senarai "Dokumen" di iTunes, cari fail audio pada komputer anda, dan klik "Tambah." iTunes akan menyalin fail muzik ini ke direktori "Dokumen" aplikasi yang dipilih pada peranti anda. Pastikan anda memilih fail audio yang serasi dengan aplikasi; rujuk panduan pengguna untuk format yang disokong.

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### Menyalin Fail Audio dari Aplikasi iOS ke Komputer   

1. Pilih aplikasi dari senarai dalam bahagian File Sharing iTunes.
2. Untuk menyalin fail ke komputer anda, seret dari senarai "Dokumen" ke folder atau tetingkap pada komputer anda.
3. Sebagai alternatif, pilih fail audio yang ingin disalin ke komputer anda dari senarai "Dokumen", klik "Simpan ke," dan pilih folder destinasi pada komputer anda.

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Mengakses Fail Audio yang Dikongsi   

Setelah berjaya menyalin fail dari komputer ke peranti iOS anda, anda boleh memainkannya dengan mudah.

- Buka aplikasi yang anda pilih untuk File Sharing, seperti Evermusic atau Flacbox.
- Ketik pada bahagian "Fail tempatan" untuk melihat senarai fail yang tersedia.
- Ketik pada mana-mana fail untuk mula memainkan.

## Memadam Fail Dikongsi dari Peranti iOS Anda   

Untuk membuang fail audio yang disalin dari komputer ke aplikasi iOS menggunakan File Sharing, ikuti langkah berikut:

#### Menggunakan iTunes  

1. Buka iTunes pada Mac/PC anda.
2. Sambungkan peranti iOS anda ke komputer menggunakan kabel USB.
3. Pilih peranti anda dalam iTunes.
4. Klik "Aplikasi" di bar sisi kiri dan tatal ke bawah ke bahagian File Sharing.
5. Pilih aplikasi yang mengandungi fail yang ingin dipadam, seperti Evermusic atau Flacbox.
6. Pilih fail audio yang ingin dibuang dari senarai "Dokumen", kemudian tekan kekunci "Padam" pada papan kekunci anda.
7. Sahkan pemadaman dalam iTunes.

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### Menggunakan Pengurus Fail Terbina Dalam  

1. Buka bahagian "Fail Tempatan".
2. Ketik butang "Lebih banyak tindakan" di penjuru kanan atas, dan pilih item menu "Pilih".
3. Pilih fail yang ingin dipadam.
4. Ketik tindakan "Padam". Jika tindakan ini tidak kelihatan serta-merta, ketik menu "Lebih banyak tindakan".
5. Sahkan pemadaman dalam mesej timbul.

## FAQ

{{% details title="Apakah cara terpantas untuk memindahkan muzik ke iPhone?" closed="true" %}}
iTunes/Finder File Sharing melalui USB adalah kaedah terpantas untuk perpustakaan muzik yang besar. Untuk pemindahan yang lebih kecil, WiFi Drive lebih mudah kerana tidak memerlukan kabel.
{{% /details %}}

{{% details title="Bolehkah saya memindahkan fail FLAC ke iPhone saya?" closed="true" %}}
Ya. Kedua-dua Evermusic dan Flacbox menerima fail FLAC melalui iTunes File Sharing, WiFi Drive, atau SMB. Flacbox disyorkan untuk format tanpa kehilangan.
{{% /details %}}

{{% details title="Adakah saya memerlukan iTunes pada macOS Catalina atau lebih baharu?" closed="true" %}}
Tidak. Apple menggantikan iTunes dengan Finder untuk pengurusan peranti bermula dengan macOS Catalina. Gunakan tab Fail Finder untuk perkongsian fail.
{{% /details %}}

{{% details title="Bolehkah saya menstrim muzik tanpa menyalin fail ke iPhone saya?" closed="true" %}}
Ya. Gunakan protokol SMB untuk menstrim muzik terus dari Mac atau PC anda. Ini menjimatkan storan peranti dan mengekalkan perpustakaan anda pada komputer.
{{% /details %}}

{{% details title="Aplikasi mana yang patut saya gunakan -- Evermusic atau Flacbox?" closed="true" %}}
Gunakan Evermusic untuk format standard seperti MP3, WAV, dan AAC. Pilih Flacbox jika perpustakaan anda termasuk format tanpa kehilangan seperti FLAC, DSD, atau OGG.
{{% /details %}}

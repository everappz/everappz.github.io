**簡述：** 使用以下三種方法之一將音樂從電腦傳輸到iPhone：**iTunes/Finder File Sharing**（USB傳輸線）、**[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)**（無線，無需傳輸線）或 **[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)**（直接串流播放，無需複製）。然後使用 [Evermusic](/products/evermusic) 或 [Flacbox](/products/flacbox) 播放。

在電腦和iOS裝置之間管理音樂檔案從未如此簡單。我們介紹了三種簡便方法——iTunes File Sharing、WiFi Drive和SMB協定——確保音樂傳輸順暢、播放無縫。

## iTunes File Sharing

iTunes File Sharing讓您可以輕鬆地在電腦和iPhone、iPad或iPod touch上的iOS應用程式之間複製音訊檔案。這種方法非常適合想要在iOS裝置上享受Mac/PC中喜愛的專輯、藝術家和曲目的使用者。

**開始之前**

- 您也可以使用 [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes) 在電腦和Evermusic、Flacbox等應用程式之間無線複製或移動音訊檔案。只需確保電腦和iOS裝置連接到同一WiFi網路，即可輕鬆在這些應用程式之間傳輸檔案。
- 另一種值得考慮的技術是 [SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)，它允許您從家用電腦直接向iOS裝置串流播放音樂，無需複製音訊檔案。

## macOS Catalina或更高版本

使用macOS Catalina或更新版本，您可以使用Finder在iOS和iPadOS裝置與Mac之間共享檔案。

**需求**  

- macOS Catalina或更高版本
- 執行最新iOS或iPadOS版本的iPhone、iPad或iPod touch
- 與File Sharing相容的iOS或iPadOS應用程式

### 如何在macOS Catalina或更高版本中使用Finder進行iTunes File Sharing

1. 開啟Finder視窗。
2. 使用USB傳輸線將iPhone、iPad或iPod touch連接到電腦。
3. 在Finder中選擇您的裝置。
4. 點擊「檔案」標籤頁查看能夠共享檔案的應用程式列表。如果您看不到此部分，表示您的裝置不支援檔案共享應用程式。
5. 點擊應用程式旁邊的三角形查看可共享的檔案。

![How to Use Finder for File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### 從Mac複製檔案到裝置

1. 選擇要複製的檔案，確保它們與應用程式相容。請參閱應用程式的使用者指南了解支援的格式。
2. 將檔案拖放到裝置上的應用程式圖示。Finder將把檔案複製到您的裝置。

#### 從裝置複製檔案到Mac

1. 選擇要複製的檔案。
2. 將檔案拖到電腦上的資料夾中，Finder將執行複製操作。

## macOS Catalina之前的版本

如果您使用的是較舊的macOS版本，可以使用iTunes進行File Sharing。

**需求**  

- 確保您擁有最新版本的iTunes。
- 您的iPhone、iPad或iPod touch應執行最新的iOS版本。
- 如果您使用的是Windows PC，請確保已更新到至少Windows 7。

![iTunes File Sharing](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### 如何在Catalina之前的macOS版本中使用iTunes File Sharing 

1. 在Mac或PC上開啟iTunes。
2. 透過USB lightning傳輸線將iPhone或iPad連接到電腦。
3. 在iTunes中點擊您的iOS裝置。
4. 導覽至左側邊欄中的「應用程式」部分。
5. 向下捲動到頁面底部的「File Sharing」部分。
6. 選擇應用程式以查看裝置上可共享的檔案。

![itunes file sharing select app](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### 從電腦複製檔案到應用程式   

1. 開啟iTunes並從File Sharing部分的列表中選擇應用程式。
2. 要將檔案複製到裝置，只需從資料夾或視窗拖放到「文件」列表中。
3. 或者，在iTunes的「文件」列表中點擊「新增」，在電腦上找到音訊檔案，然後點擊「新增」。iTunes將把這些音樂檔案複製到裝置上所選應用程式的「文件」目錄中。確保選擇與應用程式相容的音訊檔案；請參閱使用者指南了解支援的格式。

![itunes file sharing move files](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### 從iOS應用程式複製音訊檔案到電腦   

1. 從iTunes File Sharing部分的列表中選擇應用程式。
2. 要將檔案複製到電腦，將它們從「文件」列表拖到電腦上的資料夾或視窗。
3. 或者，從「文件」列表中選擇要複製到電腦的音訊檔案，點擊「儲存到」，然後選擇電腦上的目標資料夾。

![iTunes File Sharing: Copying Audio Files from an iOS App to Your Computer](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 存取共享的音訊檔案   

成功將檔案從電腦複製到iOS裝置後，您可以輕鬆播放它們。

- 開啟您選擇用於File Sharing的應用程式，如Evermusic或Flacbox。
- 點擊「本機檔案」部分查看可用檔案列表。
- 點擊任意檔案即可開始播放。

## 從iOS裝置刪除共享檔案   

要刪除使用File Sharing從電腦複製到iOS應用程式的音訊檔案，請按以下步驟操作：

#### 使用iTunes  

1. 在Mac/PC上開啟iTunes。
2. 使用USB傳輸線將iOS裝置連接到電腦。
3. 在iTunes中選擇您的裝置。
4. 點擊左側邊欄中的「應用程式」，向下捲動到File Sharing部分。
5. 選擇包含要刪除檔案的應用程式，如Evermusic或Flacbox。
6. 從「文件」列表中選擇要刪除的音訊檔案，然後按鍵盤上的「刪除」鍵。
7. 在iTunes中確認刪除。

![iTunes File Sharing: Delete File](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### 使用內建檔案管理器  

1. 開啟「本機檔案」部分。
2. 點擊右上角的「更多操作」按鈕，選擇「選擇」選單項目。
3. 選擇要刪除的檔案。
4. 點擊「刪除」操作。如果此操作未立即顯示，請點擊「更多操作」選單。
5. 在彈出訊息中確認刪除。

## FAQ

{{% details title="將音樂傳輸到iPhone最快的方法是什麼？" closed="true" %}}
透過USB的iTunes/Finder File Sharing是大型音樂庫最快的方法。對於較小的傳輸，WiFi Drive更方便，因為不需要傳輸線。
{{% /details %}}

{{% details title="我可以將FLAC檔案傳輸到iPhone嗎？" closed="true" %}}
可以。Evermusic和Flacbox都透過iTunes File Sharing、WiFi Drive或SMB接受FLAC檔案。對於無損格式，推薦使用Flacbox。
{{% /details %}}

{{% details title="在macOS Catalina或更高版本上需要iTunes嗎？" closed="true" %}}
不需要。Apple從macOS Catalina開始用Finder替代了iTunes進行裝置管理。使用Finder的「檔案」標籤頁進行檔案共享。
{{% /details %}}

{{% details title="我可以在不將檔案複製到iPhone的情況下串流播放音樂嗎？" closed="true" %}}
可以。使用SMB協定直接從Mac或PC串流播放音樂。這樣可以節省裝置儲存空間，並將音樂庫保留在電腦上。
{{% /details %}}

{{% details title="我應該使用哪個應用程式——Evermusic還是Flacbox？" closed="true" %}}
對於MP3、WAV和AAC等標準格式，使用Evermusic。如果您的音樂庫包含FLAC、DSD或OGG等無損格式，選擇Flacbox。
{{% /details %}}

**要約：** 3つの方法のいずれかを使用して、コンピュータからiPhoneに音楽を転送します：**iTunes/Finderファイル共有**（USBケーブル）、**[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)**（ワイヤレス、ケーブル不要）、または**[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)**（コピーせずに直接ストリーミング）。その後、[Evermusic](/products/evermusic)または[Flacbox](/products/flacbox)で再生します。

コンピュータとiOSデバイス間の音楽ファイル管理がこれまでになく簡単になりました。スムーズな音楽転送とシームレスな再生を確保するために、3つの簡単な方法 – iTunesファイル共有、WiFi Drive、SMBプロトコル – を紹介します。

## iTunesファイル共有

iTunesファイル共有を使うと、コンピュータとiPhone、iPad、iPod touch上のiOSアプリ間でオーディオファイルを簡単にコピーできます。このアプローチは、Mac/PCからiOSデバイスでお気に入りのアルバム、アーティスト、楽曲を楽しみたい方に最適です。

**始める前に**

- 代替手段として、[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)を使用して、コンピュータとEvermusicやFlacboxなどのアプリ間でワイヤレスにオーディオファイルをコピーまたは移動できます。コンピュータとiOSデバイスが同じWiFiネットワークに接続されていることを確認するだけで、これらのアプリ間で簡単にファイルを転送できます。
- もう一つの技術として[SMB](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)があります。これにより、自宅のコンピュータからiOSデバイスに直接音楽をストリーミングでき、オーディオファイルのコピーが不要になります。

## macOS Catalina以降

macOS Catalina以降のバージョンでは、FinderをiOSおよびiPadOSデバイスとMac間でファイルを共有するために使用できます。

**要件**  

- macOS Catalina以降
- 最新のiOSまたはiPadOSバージョンを搭載したiPhone、iPad、またはiPod touch
- ファイル共有に対応したiOSまたはiPadOSアプリ

### macOS Catalina以降でFinderを使用してiTunesファイル共有を行う方法

1. Finderウィンドウを開きます。
2. USBケーブルを使用して、iPhone、iPad、またはiPod touchをコンピュータに接続します。
3. Finderでデバイスを選択します。
4. 「ファイル」タブをクリックして、ファイル共有が可能なアプリのリストを表示します。このセクションが表示されない場合は、デバイスがファイル共有アプリをサポートしていないことを意味します。
5. アプリの横にある三角形をクリックして、共有可能なファイルを表示します。

![Finderを使用してファイル共有を行う方法](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

#### Macからデバイスにファイルをコピーする

1. コピーしたいファイルを選択し、アプリと互換性があることを確認します。サポートされているフォーマットについてはアプリのユーザーガイドを参照してください。
2. デバイス上のアプリアイコンにファイルをドラッグ＆ドロップします。Finderがデバイスへのファイルコピーを処理します。

#### デバイスからMacにファイルをコピーする

1. コピーしたいファイルを選択します。
2. コンピュータ上のフォルダにファイルをドラッグすると、Finderがコピー操作を実行します。

## macOS Catalina以前

古いバージョンのmacOSを使用している場合は、iTunesをファイル共有に使用できます。

**要件**  

- 最新バージョンのiTunesがインストールされていることを確認してください。
- iPhone、iPad、またはiPod touchが最新のiOSバージョンを実行している必要があります。
- Windows PCを使用している場合は、少なくともWindows 7に更新されていることを確認してください。

![iTunesファイル共有](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

### Catalina以前のmacOSバージョンでiTunesファイル共有を使用する方法

1. MacまたはPCでiTunesを開きます。
2. USB lightningケーブルを使って、iPhoneまたはiPadをコンピュータに接続します。
3. iTunesでiOSデバイスをクリックします。
4. 左サイドバーの「アプリ」セクションに移動します。
5. ページ下部の「ファイル共有」セクションまでスクロールします。
6. アプリを選択して、デバイス上で共有可能なファイルを確認します。

![iTunesファイル共有 アプリを選択](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

#### コンピュータからアプリにファイルをコピーする   

1. iTunesを開き、ファイル共有セクションのリストからアプリを選択します。
2. デバイスにファイルをコピーするには、フォルダまたはウィンドウから「ドキュメント」リストにドラッグ＆ドロップします。
3. または、iTunesの「ドキュメント」リストで「追加」をクリックし、コンピュータ上のオーディオファイルを見つけて「追加」をクリックします。iTunesがこれらの音楽ファイルをデバイス上の選択したアプリの「ドキュメント」ディレクトリにコピーします。アプリと互換性のあるオーディオファイルを選択してください。サポートされているフォーマットについてはユーザーガイドを参照してください。

![iTunesファイル共有 ファイルを移動](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

#### iOSアプリからコンピュータにオーディオファイルをコピーする   

1. iTunesのファイル共有セクションのリストからアプリを選択します。
2. コンピュータにファイルをコピーするには、「ドキュメント」リストからコンピュータ上のフォルダまたはウィンドウにドラッグします。
3. または、「ドキュメント」リストからコンピュータにコピーしたいオーディオファイルを選択し、「保存先」をクリックして、コンピュータ上の保存先フォルダを選択します。

![iTunesファイル共有：iOSアプリからコンピュータにオーディオファイルをコピー](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 共有オーディオファイルへのアクセス   

コンピュータからiOSデバイスにファイルを正常にコピーしたら、簡単に再生できます。

- ファイル共有用に選択したアプリ（EvermusicやFlacboxなど）を開きます。
- 「ローカルファイル」セクションをタップして、利用可能なファイルのリストを表示します。
- 任意のファイルをタップするだけで再生が始まります。

## iOSデバイスから共有ファイルを削除する   

ファイル共有を使用してコンピュータからiOSアプリにコピーしたオーディオファイルを削除するには、以下の手順に従ってください：

#### iTunesを使用する場合  

1. Mac/PCでiTunesを開きます。
2. USBケーブルを使用してiOSデバイスをコンピュータに接続します。
3. iTunesでデバイスを選択します。
4. 左サイドバーの「アプリ」をクリックし、ファイル共有セクションまでスクロールします。
5. 削除したいファイルがあるアプリ（EvermusicやFlacboxなど）を選択します。
6. 「ドキュメント」リストから削除したいオーディオファイルを選択し、キーボードの「削除する」キーを押します。
7. iTunesで削除を確認します。

![iTunesファイル共有：ファイルを削除](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

#### 内蔵ファイルマネージャーを使用する場合  

1. 「ローカルファイル」セクションを開きます。
2. 右上隅の「その他のアクション」ボタンをタップし、「選択する」メニュー項目を選択します。
3. 削除したいファイルを選択します。
4. 「削除する」アクションをタップします。このアクションがすぐに表示されない場合は、「その他のアクション」メニューをタップします。
5. ポップアップメッセージで削除を確認します。

## FAQ

{{% details title="iPhoneに音楽を転送する最も速い方法は？" closed="true" %}}
USB経由のiTunes/Finderファイル共有は、大きな音楽ライブラリに最も速い方法です。小さな転送の場合は、ケーブルが不要なWiFi Driveがより便利です。
{{% /details %}}

{{% details title="FLACファイルをiPhoneに転送できますか？" closed="true" %}}
はい。EvermusicとFlacboxの両方がiTunesファイル共有、WiFi Drive、またはSMBを通じてFLACファイルを受け入れます。ロスレスフォーマットにはFlacboxが推奨されます。
{{% /details %}}

{{% details title="macOS Catalina以降でiTunesは必要ですか？" closed="true" %}}
いいえ。AppleはmacOS Catalinaからデバイス管理のためにiTunesをFinderに置き換えました。ファイル共有にはFinderの「ファイル」タブを使用してください。
{{% /details %}}

{{% details title="iPhoneにファイルをコピーせずに音楽をストリーミングできますか？" closed="true" %}}
はい。SMBプロトコルを使用して、MacまたはPCから直接音楽をストリーミングできます。これによりデバイスのストレージを節約し、ライブラリをコンピュータに保持できます。
{{% /details %}}

{{% details title="どのアプリを使うべきですか -- EvermusicそれともFlacbox？" closed="true" %}}
MP3、WAV、AACなどの標準フォーマットにはEvermusicを使用してください。ライブラリにFLAC、DSD、OGGなどのロスレスフォーマットが含まれている場合はFlacboxを選択してください。
{{% /details %}}

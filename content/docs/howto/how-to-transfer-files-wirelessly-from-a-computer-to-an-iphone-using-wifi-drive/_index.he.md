**בקצרה:** Wi-Fi Drive מאפשר להעביר קבצים מכל מחשב לאייפון או אייפד שלכם דרך Wi-Fi -- ללא iTunes או כבלים. השתמשו בדפדפן אינטרנט, Mac Finder או Windows File Explorer. שני המכשירים חייבים להיות על אותה רשת Wi-Fi.

Wi-Fi Drive הוא הדרך הקלה ביותר להעביר קבצים מהמחשב לאייפון או אייפד ללא iTunes. תוכלו בקלות להוריד או להעלות קבצים מרובים או אפילו תיקיות באמצעות חיבור Wi-Fi מקומי. שני המכשירים צריכים להיות מחוברים לאותה רשת Wi-Fi.

תוכלו להעביר קבצים אלחוטית באמצעות דפדפן אינטרנט שולחני או לקוח WebDAV כמו Mac Finder או Windows File Explorer.

## העברת קבצים מהמחשב אלחוטית באמצעות דפדפן אינטרנט שולחני

1. פתחו אחת מהאפליקציות שלנו כגון [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. הפעילו את האפליקציה > עברו ל"חיבורים" > "מחשב" > "התחבר באמצעות Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="מסך חיבור Wi-Fi Drive" width="400" >}}

1. הזינו שם משתמש וסיסמה כדי להפוך את החיבור למאובטח יותר, אך שלב זה הוא אופציונלי.
2. הקישו על "הפעל Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="מסך Wi-Fi Drive פעיל" width="400" >}}

5. העתיקו את כתובת ה-URL של הדפדפן.
6. פתחו דפדפן אינטרנט על שולחן העבודה (דפדפנים נתמכים: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. הזינו את כתובת ה-URL משלב 5 בשדה הכתובת של דפדפן שולחן העבודה.

![מסך דפדפן Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. לאחר שהחיבור נוצר, תראו את אפליקציית מנהל הקבצים באינטרנט. כל הפעולות הנפוצות נתמכות. תוכלו להעתיק, להעביר, למחוק, להעלות ולהוריד את הקבצים שלכם מהמחשב לאחסון המקומי של המכשיר.<br><br>השתמשו בגרור ושחרר או בכפתור ההעלאה כדי להעביר מוזיקה לאייפון שלכם.<br><br>בנוסף, אם המכשיר שלכם מחובר ל-Mac באמצעות כבל, זה יכול להאיץ את העברת הקבצים. במקרה זה, ה-Mac יקים שרת וירטואלי מקוון, אך החיבור ישתמש בכבל למהירויות העברה מהירות יותר.

![מסך העלאת קבצים Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. הקישו על "עצור Wi-Fi Drive" באייפון כשסיימתם.

*הערה: ודאו שאתם משתמשים בגרסה העדכנית ביותר של דפדפן שולחן העבודה ושה-JavaScript אינו מושבת. זה יפתור כל בעיה אם לא ניתן ליצור את החיבור.*

## העברת קבצי אודיו באמצעות Mac Finder או Windows File Explorer

**אם אתם משתמשים ב-macOS:**

1. לחצו לחיצה ימנית על סמל ה-Finder ובחרו בפריט התפריט התחבר לשרת....

![Finder התחבר לשרת...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. הזינו את כתובת ה-URL של השרת מאפליקציית iOS (כתובת ה-URL של השרת נמצאת מתחת לטקסט "הזינו כתובת URL זו באפליקציית WebDAV").

![Finder הזנת כתובת שרת](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. הקישו על כפתור ההתחברות.
4. לאחר שהחיבור נוצר, תראו את תוכן האייפון שלכם ב-Mac Finder.

![Finder מחובר ל-Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. השתמשו בגרור ושחרר, העברה ושינוי שם כדי לנהל קבצים במכשיר שלכם.

**אם אתם משתמשים ב-Windows:**

1. פתחו את File Explorer ולחצו לחיצה ימנית על פריט התפריט "מחשב זה" בסרגל הצד השמאלי.

![Windows התחברות לשרת Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. בחרו "מפה כונן רשת" מהתפריט הנפתח.
3. בשדה הטקסט "תיקייה" הזינו את כתובת ה-URL של השרת מאפליקציית iOS (כתובת ה-URL של השרת נמצאת מתחת לטקסט "הזינו כתובת URL זו באפליקציית WebDAV").
4. בחרו את אות הכונן ולחצו על כפתור "סיום".

![Windows התחברות לשרת Wi-Fi Drive הזנת שם דיסק](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. אם החיבור הצליח, תוכלו לנהל קבצים במכשיר ה-iOS שלכם בחלון הבא.

![Windows מחובר לשרת Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## סיכום

עם Wi-Fi Drive, תוכלו בקלות ואלחוטית להעביר קבצים בין המחשב למכשיר ה-iOS שלכם, ולשפר את יכולות ניהול ושיתוף הקבצים שלכם.

אין צורך ב-iTunes!

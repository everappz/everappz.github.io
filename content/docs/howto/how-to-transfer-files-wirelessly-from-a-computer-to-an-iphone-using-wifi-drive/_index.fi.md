**Tiivistelmä:** Wi-Fi Drive mahdollistaa tiedostojen siirron mistä tahansa tietokoneesta iPhoneen tai iPadiin Wi-Fin kautta -- ilman iTunesia tai kaapeleita. Käytä verkkoselainta, Mac Finderia tai Windows File Exploreria. Molempien laitteiden on oltava samassa Wi-Fi-verkossa.

Wi-Fi Drive on helpoin tapa siirtää tiedostoja tietokoneelta iPhoneen tai iPadiin ilman iTunesia. Voit helposti ladata tai lähettää useita tiedostoja tai jopa kansioita paikallisen Wi-Fi-yhteyden kautta. Molempien laitteiden tulee olla yhteydessä samaan Wi-Fi-verkkoon.

Voit siirtää tiedostoja langattomasti käyttämällä työpöydän verkkoselainta tai WebDAV-asiakasohjelmaa, kuten Mac Finderia tai Windows File Exploreria.

Voit katsoa video-oppaan käyttäjältä [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) tai lukea tekstiversion alla.

## Tiedostojen siirtäminen tietokoneelta langattomasti työpöydän verkkoselaimen avulla

1. Avaa jokin sovelluksistamme, kuten [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Käynnistä sovellus > siirry kohtaan "Yhteydet" > "Tietokone" > "Yhdistä Wi-Fin kautta".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive -yhteysnäyttö" width="400" >}}

1. Syötä käyttäjätunnus ja salasana tehdäksesi yhteydestä turvallisemman, mutta tämä vaihe on valinnainen.
2. Napauta "Käynnistä Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive käynnistetty -näyttö" width="400" >}}

5. Kopioi verkkoselaimen URL-osoite.
6. Avaa verkkoselain työpöydälläsi (tuetut selaimet: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Syötä URL-osoite vaiheesta 5 työpöydän selaimen osoitekenttään.

![Wi-Fi Drive -selainnäyttö](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Kun yhteys on muodostettu, näet tiedostonhallinnan verkkosovelluksen. Kaikki yleiset toiminnot ovat tuettuja. Voit kopioida, siirtää, poistaa, lähettää ja ladata tiedostoja tietokoneelta laitteesi paikalliseen tallennustilaan.<br><br>Käytä vedä ja pudota -toimintoa tai Lähetä-painiketta musiikin siirtämiseen iPhoneen.<br><br>Lisäksi, jos laitteesi on yhdistetty Maciin kaapelilla, tämä voi nopeuttaa tiedostosiirtoja. Tässä tapauksessa Mac perustaa virtuaalisen verkkopalvelimen, mutta yhteys käyttää kaapelia nopeampiin siirtonopeuksiin.

![Wi-Fi Drive tiedostojen lähetysnäyttö](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Napauta "Pysäytä Wi-Fi Drive" iPhonessa kun olet valmis.

*Huomautus: varmista, että käytät uusinta versiota työpöydän selaimesta ja että JavaScript ei ole poistettu käytöstä. Tämä korjaa mahdolliset ongelmat, jos yhteyttä ei voida muodostaa.*

## Äänitiedostojen siirtäminen Mac Finderin tai Windows File Explorerin avulla

**Jos käytät macOS:a:**

1. Napsauta Finder-kuvaketta hiiren oikealla painikkeella ja valitse Yhdistä palvelimeen... -valikkokohta.

![Finder Yhdistä palvelimeen...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Syötä palvelimen URL-osoite iOS-sovelluksesta (palvelimen URL-osoite sijaitsee tekstin "Syötä tämä URL-osoite WebDAV-sovellukseen" alla).

![Finder Syötä palvelimen osoite](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Napauta Yhdistä-painiketta.
4. Kun yhteys on muodostettu, näet iPhonesi sisällön Mac Finderissa.

![Finder yhdistetty Wi-Fi Driveen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Käytä vedä ja pudota, siirrä ja nimeä uudelleen -komentoja hallitaksesi tiedostoja laitteellasi.

**Jos käytät Windowsia:**

1. Avaa File Explorer ja napsauta hiiren oikealla painikkeella "Tämä tietokone" -valikkokohtaa vasemmassa sivupalkissa.

![Windows Yhdistä Wi-Fi Drive -palvelimeen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Valitse "Yhdistä verkkoasema" avattavasta valikosta.
3. "Kansio"-tekstikenttään syötä palvelimen URL-osoite iOS-sovelluksesta (palvelimen URL-osoite sijaitsee tekstin "Syötä tämä URL-osoite WebDAV-sovellukseen" alla).
4. Valitse aseman kirjain ja napsauta "Valmis"-painiketta.

![Windows Yhdistä Wi-Fi Drive -palvelimeen Syötä levyn nimi](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Jos yhteys onnistuu, voit hallita iOS-laitteesi tiedostoja seuraavassa ikkunassa.

![Windows yhdistetty Wi-Fi Drive -palvelimeen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Yhteenveto

Wi-Fi Driven avulla voit helposti ja langattomasti siirtää tiedostoja tietokoneesi ja iOS-laitteesi välillä, parantaen tiedostonhallinta- ja jakamismahdollisuuksiasi.

iTunesia ei tarvita!

## Usein kysytyt kysymykset

{{% details title="Tarvitsenko iTunesin tiedostojen siirtämiseen iPhoneeni?" closed="true" %}}
Ei. Wi-Fi Drive siirtää tiedostot suoraan paikallisen Wi-Fi-verkon kautta. iTunesia ei tarvita.
{{% /details %}}

{{% details title="Mitkä sovellukset tukevat Wi-Fi Drivea?" closed="true" %}}
Wi-Fi Drive on saatavilla Evermusicissa, Flacboxissa, Evertagissa ja Evervideossa iOS:lle.
{{% /details %}}

{{% details title="Pitääkö molempien laitteiden olla samassa Wi-Fi-verkossa?" closed="true" %}}
Kyllä. Tietokoneesi ja iPhonen tai iPadin on oltava yhteydessä samaan paikalliseen Wi-Fi-verkkoon, jotta Wi-Fi Drive toimii.
{{% /details %}}

{{% details title="Voinko siirtää kokonaisia kansioita, ei vain yksittäisiä tiedostoja?" closed="true" %}}
Kyllä. Wi-Fi Drive tukee kokonaisten kansioiden lähettämistä ja lataamista verkkoselaimen käyttöliittymän kautta.
{{% /details %}}

{{% details title="Toimiiko Wi-Fi Drive Windowsin kanssa?" closed="true" %}}
Kyllä. Voit käyttää mitä tahansa verkkoselainta Windowsissa tai yhdistää Windows File Explorerin kautta WebDAV-protokollalla.
{{% /details %}}

{{% details title="Voinko käyttää USB-kaapelia siirron nopeuttamiseen?" closed="true" %}}
Kyllä. Jos iPhonesi on yhdistetty Maciin USB:n kautta Wi-Fi Driven ollessa käynnissä, siirto käyttää kaapeliyhteyttä nopeampiin siirtonopeuksiin.
{{% /details %}}

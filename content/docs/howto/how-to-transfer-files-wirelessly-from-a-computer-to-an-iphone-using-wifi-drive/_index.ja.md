**要約:** Wi-Fi Driveを使えば、どのパソコンからでもiPhoneやiPadにWi-Fi経由でファイルを転送できます -- iTunesやケーブルは不要です。Webブラウザ、Mac Finder、またはWindows File Explorerを使用してください。両方のデバイスが同じWi-Fiネットワークに接続されている必要があります。

Wi-Fi Driveは、iTunesなしでパソコンからiPhoneやiPadにファイルを転送する最も簡単な方法です。ローカルWi-Fi接続を使用して、複数のファイルやフォルダを簡単にダウンロードまたはアップロードできます。両方のデバイスが同じWi-Fiネットワークに接続されている必要があります。

デスクトップWebブラウザまたはMac FinderやWindows File ExplorerなどのWebDAVクライアントを使用して、ワイヤレスでファイルを転送できます。

## デスクトップWebブラウザを使ってパソコンからワイヤレスでファイルを転送する

1. [Evermusic](/products/evermusic)、[Flacbox](/products/flacbox)、[Evertag](/products/evertag)、[Evervideo](/products/evervideo)などのアプリを開きます。

2. アプリを起動 > 「接続」 > 「コンピュータ」 > 「Wi-Fiで接続」に移動します。
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive接続画面" width="400" >}}

1. 接続をより安全にするためにユーザー名とパスワードを入力しますが、このステップはオプションです。
2. 「Wi-Fi Driveを開始」をタップします。

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive開始画面" width="400" >}}

5. WebブラウザのURLをコピーします。
6. デスクトップでWebブラウザを開きます（対応ブラウザ：Safari、Google Chrome、Opera、Yandex Browser、FireFox）。
7. ステップ5のURLをデスクトップブラウザのアドレスフィールドに入力します。

![Wi-Fi Driveブラウザ画面](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. 接続が確立されると、ファイルマネージャーWebアプリが表示されます。一般的な操作はすべてサポートされています。パソコンからデバイスのローカルストレージにファイルをコピー、移動、削除、アップロード、ダウンロードできます。<br><br>ドラッグ＆ドロップまたはアップロードボタンを使用して、iPhoneに音楽を転送します。<br><br>さらに、デバイスがケーブルでMacに接続されている場合、ファイル転送を高速化できます。この場合、Macは仮想オンラインサーバーを設定しますが、接続はケーブルを使用してより高速な転送速度を実現します。

![Wi-Fi Driveファイルアップロード画面](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. 完了したら、iPhoneで「Wi-Fi Driveを停止」をタップします。

*注意：デスクトップブラウザの最新バージョンを使用し、JavaScriptが無効になっていないことを確認してください。接続が確立できない場合の問題を解決できます。*

## Mac FinderまたはWindows File Explorerでオーディオファイルを転送する

**macOSを使用している場合：**

1. Finderアイコンを右クリックし、サーバへ接続...メニュー項目を選択します。

![Finder サーバへ接続...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. iOSアプリからサーバーURLを入力します（サーバーURLは「このURLをWebDAVアプリケーションに入力してください」というテキストの下にあります）。

![Finder サーバーアドレスを入力](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. 接続ボタンをタップします。
4. 接続が確立されると、Mac FinderでiPhoneの内容が表示されます。

![Finder Wi-Fi Driveに接続](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. ドラッグ＆ドロップ、移動、名前変更コマンドを使用して、デバイス上のファイルを管理します。

**Windowsを使用している場合：**

1. File Explorerを開き、左サイドバーの「PC」メニュー項目を右クリックします。

![Windows Wi-Fi Driveサーバーに接続](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. ドロップダウンメニューで「ネットワークドライブの割り当て」を選択します。
3. 「フォルダー」テキストフィールドにiOSアプリからサーバーURLを入力します（サーバーURLは「このURLをWebDAVアプリケーションに入力してください」というテキストの下にあります）。
4. ドライブ文字を選択し、「完了」ボタンをクリックします。

![Windows Wi-Fi Driveサーバーに接続 ディスク名を入力](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. 接続が成功すると、次のウィンドウでiOSデバイス上のファイルを管理できます。

![Windows Wi-Fi Driveサーバーに接続済み](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## まとめ

Wi-Fi Driveを使えば、パソコンとiOSデバイス間で簡単にワイヤレスでファイルを転送でき、ファイル管理と共有の機能が向上します。

iTunesは不要です！

## よくある質問

{{% details title="iPhoneにファイルを転送するのにiTunesは必要ですか？" closed="true" %}}
いいえ。Wi-Fi Driveはローカルの Wi-Fiネットワークを介してファイルを直接転送します。iTunesは必要ありません。
{{% /details %}}

{{% details title="どのアプリがWi-Fi Driveをサポートしていますか？" closed="true" %}}
Wi-Fi DriveはiOS用のEvermusic、Flacbox、Evertag、Evervideoで利用できます。
{{% /details %}}

{{% details title="両方のデバイスが同じWi-Fiネットワークに接続されている必要がありますか？" closed="true" %}}
はい。Wi-Fi Driveが動作するには、パソコンとiPhoneまたはiPadが同じローカルWi-Fiネットワークに接続されている必要があります。
{{% /details %}}

{{% details title="個別のファイルだけでなく、フォルダ全体を転送できますか？" closed="true" %}}
はい。Wi-Fi DriveはWebブラウザインターフェースを通じてフォルダ全体のアップロードとダウンロードをサポートしています。
{{% /details %}}

{{% details title="Wi-Fi DriveはWindowsで動作しますか？" closed="true" %}}
はい。Windows上の任意のWebブラウザを使用するか、WebDAVプロトコルを使用してWindows File Explorer経由で接続できます。
{{% /details %}}

{{% details title="USBケーブルを使って転送を高速化できますか？" closed="true" %}}
はい。Wi-Fi Driveの実行中にiPhoneがUSBでMacに接続されている場合、転送はケーブル接続を使用してより高速な速度を実現します。
{{% /details %}}

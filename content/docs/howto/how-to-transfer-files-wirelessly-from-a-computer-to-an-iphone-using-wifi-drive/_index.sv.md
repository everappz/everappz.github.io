**TL;DR:** Wi-Fi Drive låter dig överföra filer från vilken dator som helst till din iPhone eller iPad via Wi-Fi -- ingen iTunes eller kablar krävs. Använd en webbläsare, Mac Finder eller Windows File Explorer. Båda enheterna måste vara på samma Wi-Fi-nätverk.

Wi-Fi Drive är det enklaste sättet att överföra dina filer från datorn till en iPhone eller iPad utan iTunes. Du kan enkelt ladda ner eller ladda upp flera filer eller till och med mappar via en lokal Wi-Fi-anslutning. Båda enheterna bör vara anslutna till samma Wi-Fi-nätverk.

Du kan överföra filer trådlöst med en webbläsare på datorn eller en WebDAV-klient som Mac Finder eller Windows File Explorer.

Du kan titta på en videohandledning från [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) eller läsa textversionen nedan.

## Överför filer från datorn trådlöst med en webbläsare

1. Öppna någon av våra appar som [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Starta appen > gå till "Anslutningar" > "Dator" > "Anslut via Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Ange användarnamn och lösenord för att göra anslutningen säkrare, men detta steg är valfritt.
2. Tryck på "Starta Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Kopiera webbläsarens URL.
6. Öppna en webbläsare på din dator (webbläsare som stöds: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Ange URL:en från steg 5 i adressfältet i din dators webbläsare.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. När anslutningen är upprättad ser du filhanterarens webbapp. Alla vanliga operationer stöds. Du kan kopiera, flytta, ta bort, ladda upp och ladda ner dina filer från datorn till enhetens lokala lagring.<br><br>Använd dra och släpp eller knappen Ladda upp för att överföra musik till din iPhone.<br><br>Om din enhet dessutom är ansluten till din Mac med en kabel kan detta snabba upp filöverföringarna. I det fallet kommer din Mac att konfigurera en virtuell onlineserver, men anslutningen kommer att använda kabeln för snabbare överföringshastigheter.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Tryck på "Stoppa Wi-Fi Drive" på iPhone när du är klar.

*Observera: se till att du använder den senaste versionen av datorns webbläsare och att JavaScript inte är inaktiverat. Detta löser eventuella problem om anslutningen inte kan upprättas.*

## Överför ljudfiler med Mac Finder eller Windows File Explorer

**Om du använder macOS:**

1. Högerklicka på Finder-ikonen och välj menyalternativet Anslut till server....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Ange server-URL:en från iOS-appen (server-URL:en finns under texten "Ange denna URL i WebDAV-applikationen").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Tryck på knappen Anslut.
4. När anslutningen är upprättad ser du innehållet på din iPhone i Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Använd dra och släpp, Flytta och Byt namn-kommandon för att hantera filer på din enhet.

**Om du använder Windows:**

1. Öppna Utforskaren och högerklicka på menyalternativet "Den här datorn" i det vänstra sidofältet.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Välj "Anslut nätverksenhet" i rullgardinsmenyn.
3. I textfältet "Mapp" anger du server-URL:en från iOS-appen (server-URL:en finns under texten "Ange denna URL i WebDAV-applikationen").
4. Välj enhetsbokstav och klicka på knappen "Slutför".

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Om anslutningen lyckas kan du hantera filer på din iOS-enhet i nästa fönster.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Slutsats

Med Wi-Fi Drive kan du enkelt och trådlöst överföra filer mellan din dator och iOS-enhet, vilket förbättrar dina filhanterings- och delningsmöjligheter.

Ingen iTunes krävs!

## Vanliga frågor

{{% details title="Behöver jag iTunes för att överföra filer till min iPhone?" closed="true" %}}
Nej. Wi-Fi Drive överför filer direkt via ditt lokala Wi-Fi-nätverk. iTunes krävs inte.
{{% /details %}}

{{% details title="Vilka appar stöder Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive finns i Evermusic, Flacbox, Evertag och Evervideo för iOS.
{{% /details %}}

{{% details title="Måste båda enheterna vara på samma Wi-Fi-nätverk?" closed="true" %}}
Ja. Din dator och iPhone eller iPad måste vara anslutna till samma lokala Wi-Fi-nätverk för att Wi-Fi Drive ska fungera.
{{% /details %}}

{{% details title="Kan jag överföra hela mappar, inte bara enskilda filer?" closed="true" %}}
Ja. Wi-Fi Drive stöder uppladdning och nedladdning av hela mappar via webbläsargränssnittet.
{{% /details %}}

{{% details title="Fungerar Wi-Fi Drive med Windows?" closed="true" %}}
Ja. Du kan använda vilken webbläsare som helst på Windows eller ansluta via Windows File Explorer med WebDAV-protokollet.
{{% /details %}}

{{% details title="Kan jag använda en USB-kabel för att snabba upp överföringen?" closed="true" %}}
Ja. Om din iPhone är ansluten till din Mac via USB medan Wi-Fi Drive körs kommer överföringen att använda kabelanslutningen för snabbare hastigheter.
{{% /details %}}

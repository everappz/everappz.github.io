**簡介：** Wi-Fi Drive讓您可以透過Wi-Fi從任何電腦將檔案傳輸到iPhone或iPad——無需iTunes或傳輸線。使用網頁瀏覽器、Mac Finder或Windows File Explorer。兩台裝置必須在同一個Wi-Fi網路上。

Wi-Fi Drive是無需iTunes將檔案從電腦傳輸到iPhone或iPad的最簡單方式。您可以使用本地Wi-Fi連線輕鬆下載或上傳多個檔案甚至資料夾。兩台裝置應連接到同一個Wi-Fi網路。

您可以使用桌面網頁瀏覽器或WebDAV用戶端（如Mac Finder或Windows File Explorer）無線傳輸檔案。

## 使用桌面網頁瀏覽器從電腦無線傳輸檔案

1. 開啟我們的任何應用程式，如[Evermusic](/products/evermusic)、[Flacbox](/products/flacbox)、[Evertag](/products/evertag)、[Evervideo](/products/evervideo)。

2. 啟動應用程式 > 前往「連接」>「電腦」>「使用Wi-Fi連接」。
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. 輸入使用者名稱和密碼以使連線更安全，但此步驟是可選的。
2. 點擊「啟動Wi-Fi Drive」。

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. 複製網頁瀏覽器URL。
6. 在電腦上開啟網頁瀏覽器（支援的瀏覽器：Safari、Google Chrome、Opera、Yandex Browser、FireFox）。
7. 將第5步的URL輸入到電腦瀏覽器的網址列中。

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. 連線建立後，您將看到檔案管理器網頁應用程式。支援所有常見操作。您可以從電腦複製、移動、刪除、上傳和下載檔案到裝置的本地儲存空間。<br><br>使用拖放或上傳按鈕將音樂傳輸到iPhone。<br><br>此外，如果您的裝置透過傳輸線連接到Mac，這可以加快檔案傳輸速度。在這種情況下，Mac將設定一個虛擬線上伺服器，但連線將使用傳輸線以獲得更快的傳輸速度。

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. 完成後在iPhone上點擊「停止Wi-Fi Drive」。

*注意：請確保您使用的是最新版本的桌面瀏覽器，並且JavaScript未被停用。如果無法建立連線，這將解決任何問題。*

## 使用Mac Finder或Windows File Explorer傳輸音訊檔案

**如果您使用macOS：**

1. 右鍵點擊Finder圖示，選擇「連接伺服器...」選單項目。

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. 輸入iOS應用程式中的伺服器URL（伺服器URL位於「在WebDAV應用程式中輸入此URL」文字下方）。

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. 點擊連接按鈕。
4. 連線建立後，您將在Mac Finder中看到iPhone的內容。

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. 使用拖放、移動和重新命名命令管理裝置上的檔案。

**如果您使用Windows：**

1. 開啟檔案總管，右鍵點擊左側邊欄中的「本機」選單項目。

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. 在下拉選單中選擇「連線網路磁碟機」。
3. 在「資料夾」文字欄位中輸入iOS應用程式的伺服器URL（伺服器URL位於「在WebDAV應用程式中輸入此URL」文字下方）。
4. 選擇磁碟機代號，點擊「完成」按鈕。

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. 如果連線成功，您可以在下一個視窗中管理iOS裝置上的檔案。

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## 結論

使用Wi-Fi Drive，您可以輕鬆地在電腦和iOS裝置之間無線傳輸檔案，增強檔案管理和共享能力。

無需iTunes！

## 常見問題

{{% details title="我需要iTunes才能將檔案傳輸到iPhone嗎？" closed="true" %}}
不需要。Wi-Fi Drive透過本地Wi-Fi網路直接傳輸檔案。不需要iTunes。
{{% /details %}}

{{% details title="哪些應用程式支援Wi-Fi Drive？" closed="true" %}}
Wi-Fi Drive在iOS版Evermusic、Flacbox、Evertag和Evervideo中可用。
{{% /details %}}

{{% details title="兩台裝置需要在同一個Wi-Fi網路上嗎？" closed="true" %}}
是的。您的電腦和iPhone或iPad必須連接到同一個本地Wi-Fi網路，Wi-Fi Drive才能運作。
{{% /details %}}

{{% details title="我可以傳輸整個資料夾，而不僅僅是單個檔案嗎？" closed="true" %}}
可以。Wi-Fi Drive支援透過網頁瀏覽器介面上傳和下載整個資料夾。
{{% /details %}}

{{% details title="Wi-Fi Drive可以在Windows上使用嗎？" closed="true" %}}
可以。您可以在Windows上使用任何網頁瀏覽器，或透過WebDAV協定使用Windows File Explorer連接。
{{% /details %}}

{{% details title="我可以使用USB傳輸線加快傳輸速度嗎？" closed="true" %}}
可以。如果在Wi-Fi Drive執行時iPhone透過USB連接到Mac，傳輸將使用有線連接以獲得更快的速度。
{{% /details %}}

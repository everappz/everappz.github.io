**TL;DR:** Wi-Fi Drive позволяет передавать файлы с любого компьютера на iPhone или iPad по Wi-Fi -- без iTunes и кабелей. Используйте веб-браузер, Mac Finder или Windows File Explorer. Оба устройства должны быть в одной сети Wi-Fi.

Wi-Fi Drive -- это самый простой способ передачи файлов с компьютера на iPhone или iPad без iTunes. Вы можете легко загружать или отправлять несколько файлов и даже папок через локальное соединение Wi-Fi. Оба устройства должны быть подключены к одной сети Wi-Fi.

Вы можете передавать файлы по беспроводной сети с помощью настольного веб-браузера или клиента WebDAV, такого как Mac Finder или Windows File Explorer.

## Передача файлов с компьютера по беспроводной сети через настольный веб-браузер

1. Откройте любое из наших приложений, например [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Запустите приложение > перейдите в "Подключения" > "Компьютер" > "Подключиться через Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Введите имя пользователя и пароль, чтобы сделать подключение более безопасным, но этот шаг необязателен.
2. Нажмите "Запустить Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Скопируйте URL веб-браузера.
6. Откройте веб-браузер на компьютере (поддерживаемые браузеры: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Введите URL из шага 5 в адресную строку браузера на компьютере.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. После установки соединения вы увидите веб-приложение файлового менеджера. Поддерживаются все стандартные операции. Вы можете копировать, перемещать, удалять, загружать и скачивать файлы с компьютера в локальное хранилище устройства.<br><br>Используйте перетаскивание или кнопку «Загрузить» для передачи музыки на iPhone.<br><br>Кроме того, если ваше устройство подключено к Mac с помощью кабеля, это может ускорить передачу файлов. В этом случае Mac настроит виртуальный онлайн-сервер, но соединение будет использовать кабель для более высокой скорости передачи.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. По завершении нажмите "Остановить Wi-Fi Drive" на iPhone.

*Примечание: убедитесь, что вы используете последнюю версию настольного браузера и JavaScript не отключен. Это решит любые проблемы, если соединение не может быть установлено.*

## Передача аудиофайлов с помощью Mac Finder или Windows File Explorer

**Если вы используете macOS:**

1. Щелкните правой кнопкой мыши по значку Finder и выберите пункт меню «Подключиться к серверу...».

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Введите URL сервера из приложения iOS (URL сервера расположен под текстом «Введите этот URL в приложении WebDAV»).

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Нажмите кнопку «Подключить».
4. После установки соединения вы увидите содержимое вашего iPhone в Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Используйте перетаскивание, команды «Переместить» и «Переименовать» для управления файлами на устройстве.

**Если вы используете Windows:**

1. Откройте Проводник и щелкните правой кнопкой мыши по пункту «Этот компьютер» на левой боковой панели.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Выберите «Подключить сетевой диск» в выпадающем меню.
3. В текстовом поле «Папка» введите URL сервера из приложения iOS (URL сервера расположен под текстом «Введите этот URL в приложении WebDAV»).
4. Выберите букву диска и нажмите кнопку «Готово».

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Если подключение успешно, вы можете управлять файлами на вашем iOS-устройстве в следующем окне.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Заключение

С Wi-Fi Drive вы можете легко и без проводов передавать файлы между компьютером и iOS-устройством, расширяя возможности управления файлами и обмена ими.

iTunes не требуется!

## Часто задаваемые вопросы

{{% details title="Нужен ли мне iTunes для передачи файлов на iPhone?" closed="true" %}}
Нет. Wi-Fi Drive передает файлы напрямую через вашу локальную сеть Wi-Fi. iTunes не требуется.
{{% /details %}}

{{% details title="Какие приложения поддерживают Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive доступен в Evermusic, Flacbox, Evertag и Evervideo для iOS.
{{% /details %}}

{{% details title="Оба устройства должны быть в одной сети Wi-Fi?" closed="true" %}}
Да. Ваш компьютер и iPhone или iPad должны быть подключены к одной локальной сети Wi-Fi для работы Wi-Fi Drive.
{{% /details %}}

{{% details title="Можно ли передавать целые папки, а не только отдельные файлы?" closed="true" %}}
Да. Wi-Fi Drive поддерживает загрузку и скачивание целых папок через интерфейс веб-браузера.
{{% /details %}}

{{% details title="Работает ли Wi-Fi Drive с Windows?" closed="true" %}}
Да. Вы можете использовать любой веб-браузер в Windows или подключиться через Windows File Explorer с помощью протокола WebDAV.
{{% /details %}}

{{% details title="Можно ли использовать USB-кабель для ускорения передачи?" closed="true" %}}
Да. Если ваш iPhone подключен к Mac через USB во время работы Wi-Fi Drive, передача будет использовать кабельное соединение для более высокой скорости.
{{% /details %}}

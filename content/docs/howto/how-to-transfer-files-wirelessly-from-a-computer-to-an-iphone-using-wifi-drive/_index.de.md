**Kurzfassung:** Wi-Fi Drive ermöglicht es Ihnen, Dateien von jedem Computer auf Ihr iPhone oder iPad über Wi-Fi zu übertragen -- ohne iTunes oder Kabel. Verwenden Sie einen Webbrowser, Mac Finder oder Windows File Explorer. Beide Geräte müssen mit demselben Wi-Fi-Netzwerk verbunden sein.

Wi-Fi Drive ist der einfachste Weg, Ihre Dateien vom Computer auf ein iPhone oder iPad ohne iTunes zu übertragen. Sie können ganz einfach mehrere Dateien oder sogar Ordner über eine lokale Wi-Fi-Verbindung herunterladen oder hochladen. Beide Geräte sollten mit demselben Wi-Fi-Netzwerk verbunden sein.

Sie können Dateien drahtlos über einen Desktop-Webbrowser oder einen WebDAV-Client wie Mac Finder oder Windows File Explorer übertragen.

Sie können ein Video-Tutorial von [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) ansehen oder die Textversion unten lesen.

## Dateien vom Computer drahtlos mit einem Desktop-Webbrowser übertragen

1. Öffnen Sie eine unserer Apps wie [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Starten Sie die App > gehen Sie zu "Verbindungen" > "Computer" > "Über Wi-Fi verbinden".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Verbindungsbildschirm" width="400" >}}

1. Geben Sie Benutzername und Passwort ein, um die Verbindung sicherer zu machen, aber dieser Schritt ist optional.
2. Tippen Sie auf "Wi-Fi Drive starten".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive gestartet Bildschirm" width="400" >}}

5. Kopieren Sie die Webbrowser-URL.
6. Öffnen Sie einen Webbrowser auf Ihrem Desktop (unterstützte Browser: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Geben Sie die URL aus Schritt 5 in das Adressfeld Ihres Desktop-Browsers ein.

![Wi-Fi Drive Browser-Bildschirm](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Nach dem Verbindungsaufbau sehen Sie die Dateimanager-Web-App. Alle gängigen Operationen werden unterstützt. Sie können Dateien vom Computer in den lokalen Speicher Ihres Geräts kopieren, verschieben, löschen, hochladen und herunterladen.<br><br>Verwenden Sie Drag & Drop oder die Upload-Schaltfläche, um Musik auf Ihr iPhone zu übertragen.<br><br>Wenn Ihr Gerät zusätzlich über ein Kabel mit Ihrem Mac verbunden ist, kann dies die Dateiübertragung beschleunigen. In diesem Fall richtet Ihr Mac einen virtuellen Online-Server ein, aber die Verbindung nutzt das Kabel für schnellere Übertragungsgeschwindigkeiten.

![Wi-Fi Drive Dateien hochladen Bildschirm](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Tippen Sie auf dem iPhone auf "Wi-Fi Drive stoppen", wenn Sie fertig sind.

*Hinweis: Stellen Sie sicher, dass Sie die neueste Version des Desktop-Browsers verwenden und JavaScript nicht deaktiviert ist. Dies behebt alle Probleme, falls die Verbindung nicht hergestellt werden kann.*

## Audiodateien mit Mac Finder oder Windows File Explorer übertragen

**Wenn Sie macOS verwenden:**

1. Klicken Sie mit der rechten Maustaste auf das Finder-Symbol und wählen Sie den Menüpunkt "Mit Server verbinden...".

![Finder Mit Server verbinden...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Geben Sie die Server-URL aus der iOS-App ein (die Server-URL befindet sich unter dem Text "Geben Sie diese URL in der WebDAV-Anwendung ein").

![Finder Serveradresse eingeben](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Tippen Sie auf die Verbinden-Schaltfläche.
4. Nach dem Verbindungsaufbau sehen Sie den Inhalt Ihres iPhones im Mac Finder.

![Finder verbunden mit Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Verwenden Sie Drag & Drop, Verschieben und Umbenennen, um Dateien auf Ihrem Gerät zu verwalten.

**Wenn Sie Windows verwenden:**

1. Öffnen Sie den File Explorer und klicken Sie mit der rechten Maustaste auf den Menüpunkt "Dieser PC" in der linken Seitenleiste.

![Windows Mit Wi-Fi Drive Server verbinden](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Wählen Sie "Netzlaufwerk verbinden" im Dropdown-Menü.
3. Geben Sie im Textfeld "Ordner" die Server-URL aus der iOS-App ein (die Server-URL befindet sich unter dem Text "Geben Sie diese URL in der WebDAV-Anwendung ein").
4. Wählen Sie den Laufwerksbuchstaben und klicken Sie auf die Schaltfläche "Fertig stellen".

![Windows Mit Wi-Fi Drive Server verbinden Laufwerksname eingeben](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Wenn die Verbindung erfolgreich ist, können Sie im nächsten Fenster Dateien auf Ihrem iOS-Gerät verwalten.

![Windows verbunden mit Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Fazit

Mit Wi-Fi Drive können Sie einfach und drahtlos Dateien zwischen Ihrem Computer und iOS-Gerät übertragen und so Ihre Dateiverwaltungs- und Freigabemöglichkeiten verbessern.

Kein iTunes erforderlich!

## Häufig gestellte Fragen

{{% details title="Brauche ich iTunes, um Dateien auf mein iPhone zu übertragen?" closed="true" %}}
Nein. Wi-Fi Drive überträgt Dateien direkt über Ihr lokales Wi-Fi-Netzwerk. iTunes ist nicht erforderlich.
{{% /details %}}

{{% details title="Welche Apps unterstützen Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive ist in Evermusic, Flacbox, Evertag und Evervideo für iOS verfügbar.
{{% /details %}}

{{% details title="Müssen beide Geräte im selben Wi-Fi-Netzwerk sein?" closed="true" %}}
Ja. Ihr Computer und iPhone oder iPad müssen mit demselben lokalen Wi-Fi-Netzwerk verbunden sein, damit Wi-Fi Drive funktioniert.
{{% /details %}}

{{% details title="Kann ich ganze Ordner übertragen, nicht nur einzelne Dateien?" closed="true" %}}
Ja. Wi-Fi Drive unterstützt das Hoch- und Herunterladen ganzer Ordner über die Webbrowser-Oberfläche.
{{% /details %}}

{{% details title="Funktioniert Wi-Fi Drive mit Windows?" closed="true" %}}
Ja. Sie können jeden Webbrowser unter Windows verwenden oder sich über den Windows File Explorer mit dem WebDAV-Protokoll verbinden.
{{% /details %}}

{{% details title="Kann ich ein USB-Kabel verwenden, um die Übertragung zu beschleunigen?" closed="true" %}}
Ja. Wenn Ihr iPhone über USB mit Ihrem Mac verbunden ist, während Wi-Fi Drive läuft, nutzt die Übertragung die Kabelverbindung für schnellere Geschwindigkeiten.
{{% /details %}}

**TL;DR:** Wi-Fi Drive umożliwia przesyłanie plików z dowolnego komputera na iPhone lub iPad przez Wi-Fi -- bez iTunes ani kabli. Użyj przeglądarki internetowej, Mac Finder lub Windows File Explorer. Oba urządzenia muszą być w tej samej sieci Wi-Fi.

Wi-Fi Drive to najłatwiejszy sposób na przesyłanie plików z komputera na iPhone lub iPad bez iTunes. Możesz łatwo pobierać lub przesyłać wiele plików, a nawet folderów, korzystając z lokalnego połączenia Wi-Fi. Oba urządzenia powinny być połączone z tą samą siecią Wi-Fi.

Możesz przesyłać pliki bezprzewodowo za pomocą przeglądarki internetowej na komputerze lub klienta WebDAV, takiego jak Mac Finder lub Windows File Explorer.

## Bezprzewodowe przesyłanie plików z komputera za pomocą przeglądarki internetowej

1. Otwórz dowolną z naszych aplikacji, taką jak [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Uruchom aplikację > przejdź do "Połączenia" > "Komputer" > "Połącz przez Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Wprowadź nazwę użytkownika i hasło, aby połączenie było bezpieczniejsze, ale ten krok jest opcjonalny.
2. Dotknij "Uruchom Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Skopiuj adres URL przeglądarki.
6. Otwórz przeglądarkę internetową na komputerze (obsługiwane przeglądarki: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Wprowadź adres URL z kroku 5 w polu adresu przeglądarki na komputerze.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Po nawiązaniu połączenia zobaczysz aplikację webową menedżera plików. Obsługiwane są wszystkie typowe operacje. Możesz kopiować, przenosić, usuwać, przesyłać i pobierać pliki z komputera do pamięci lokalnej urządzenia.<br><br>Użyj przeciągnij i upuść lub przycisku Prześlij, aby przesłać muzykę na iPhone.<br><br>Dodatkowo, jeśli urządzenie jest podłączone do Maca kablem, może to przyspieszyć transfer plików. W takim przypadku Mac skonfiguruje wirtualny serwer online, ale połączenie będzie korzystać z kabla dla szybszego transferu.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Po zakończeniu dotknij "Zatrzymaj Wi-Fi Drive" na iPhone.

*Uwaga: upewnij się, że używasz najnowszej wersji przeglądarki na komputerze i że JavaScript nie jest wyłączony. Rozwiąże to wszelkie problemy, jeśli połączenie nie może zostać nawiązane.*

## Przesyłanie plików audio za pomocą Mac Finder lub Windows File Explorer

**Jeśli używasz macOS:**

1. Kliknij prawym przyciskiem myszy ikonę Finder i wybierz element menu Połącz z serwerem....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Wprowadź adres URL serwera z aplikacji iOS (adres URL serwera znajduje się pod tekstem "Wprowadź ten adres URL w aplikacji WebDAV").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Dotknij przycisku Połącz.
4. Po nawiązaniu połączenia zobaczysz zawartość swojego iPhone w Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Użyj przeciągnij i upuść, poleceń Przenieś i Zmień nazwę, aby zarządzać plikami na urządzeniu.

**Jeśli używasz Windows:**

1. Otwórz Eksplorator plików i kliknij prawym przyciskiem myszy element menu „Ten komputer" na lewym pasku bocznym.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Wybierz "Mapuj dysk sieciowy" z menu rozwijanego.
3. W polu tekstowym "Folder" wprowadź adres URL serwera z aplikacji iOS (adres URL serwera znajduje się pod tekstem "Wprowadź ten adres URL w aplikacji WebDAV").
4. Wybierz literę dysku i kliknij przycisk "Zakończ".

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Jeśli połączenie zakończy się sukcesem, możesz zarządzać plikami znajdującymi się na urządzeniu iOS w następnym oknie.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Podsumowanie

Dzięki Wi-Fi Drive możesz łatwo i bezprzewodowo przesyłać pliki między komputerem a urządzeniem iOS, zwiększając możliwości zarządzania i udostępniania plików.

iTunes nie jest wymagany!

## Często zadawane pytania

{{% details title="Czy potrzebuję iTunes, aby przesyłać pliki na iPhone?" closed="true" %}}
Nie. Wi-Fi Drive przesyła pliki bezpośrednio przez lokalną sieć Wi-Fi. iTunes nie jest wymagany.
{{% /details %}}

{{% details title="Które aplikacje obsługują Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive jest dostępny w Evermusic, Flacbox, Evertag i Evervideo dla iOS.
{{% /details %}}

{{% details title="Czy oba urządzenia muszą być w tej samej sieci Wi-Fi?" closed="true" %}}
Tak. Komputer i iPhone lub iPad muszą być połączone z tą samą lokalną siecią Wi-Fi, aby Wi-Fi Drive działał.
{{% /details %}}

{{% details title="Czy mogę przesyłać całe foldery, a nie tylko pojedyncze pliki?" closed="true" %}}
Tak. Wi-Fi Drive obsługuje przesyłanie i pobieranie całych folderów przez interfejs przeglądarki internetowej.
{{% /details %}}

{{% details title="Czy Wi-Fi Drive działa z Windows?" closed="true" %}}
Tak. Możesz użyć dowolnej przeglądarki internetowej w systemie Windows lub połączyć się przez Windows File Explorer za pomocą protokołu WebDAV.
{{% /details %}}

{{% details title="Czy mogę użyć kabla USB, aby przyspieszyć transfer?" closed="true" %}}
Tak. Jeśli iPhone jest podłączony do Maca przez USB, gdy Wi-Fi Drive jest uruchomiony, transfer będzie korzystać z połączenia kablowego dla szybszych prędkości.
{{% /details %}}

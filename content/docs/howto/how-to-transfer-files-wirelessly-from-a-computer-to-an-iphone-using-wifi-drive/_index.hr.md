**Ukratko:** Wi-Fi Drive omogućuje prijenos datoteka s bilo kojeg računala na vaš iPhone ili iPad putem Wi-Fi -- bez iTunes ili kabela. Koristite web preglednik, Mac Finder ili Windows File Explorer. Oba uređaja moraju biti na istoj Wi-Fi mreži.

Wi-Fi Drive je najjednostavniji način za prijenos datoteka s računala na iPhone ili iPad bez iTunes. Možete jednostavno preuzimati ili slati više datoteka ili čak mapa koristeći lokalnu Wi-Fi vezu. Oba uređaja trebaju biti spojena na istu Wi-Fi mrežu.

Datoteke možete bežično prenositi koristeći desktop web preglednik ili WebDAV klijent poput Mac Finder ili Windows File Explorer.

Možete pogledati video vodič od [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) ili pročitati tekstualnu verziju u nastavku.

## Prijenos datoteka s računala bežično pomoću desktop web preglednika

1. Otvorite bilo koju od naših aplikacija poput [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Pokrenite aplikaciju > idite na "Povezivanja" > "Računalo" > "Poveži putem Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive zaslon za povezivanje" width="400" >}}

1. Unesite korisničko ime i lozinku kako biste vezu učinili sigurnijom, ali ovaj korak je opcionalan.
2. Dodirnite "Pokreni Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive pokrenuti zaslon" width="400" >}}

5. Kopirajte URL web preglednika.
6. Otvorite web preglednik na svom desktopu (podržani preglednici: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Unesite URL iz koraka 5 u adresno polje svog desktop preglednika.

![Wi-Fi Drive zaslon preglednika](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Nakon uspostavljanja veze vidjet ćete web aplikaciju za upravljanje datotekama. Sve uobičajene operacije su podržane. Možete kopirati, premjestiti, izbrisati, prenijeti i preuzeti svoje datoteke s računala na lokalno spremište vašeg uređaja.<br><br>Koristite povuci i ispusti ili gumb za prijenos za prebacivanje glazbe na vaš iPhone.<br><br>Dodatno, ako je vaš uređaj spojen na Mac kablom, to može ubrzati prijenos datoteka. U tom slučaju, vaš Mac će postaviti virtualni mrežni poslužitelj, ali veza će koristiti kabel za brže brzine prijenosa.

![Wi-Fi Drive zaslon za prijenos datoteka](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Dodirnite "Zaustavi Wi-Fi Drive" na iPhoneu kad završite.

*Napomena: provjerite koristite li najnoviju verziju desktop preglednika i da JavaScript nije onemogućen. To će riješiti sve probleme ako se veza ne može uspostaviti.*

## Prijenos audio datoteka pomoću Mac Finder ili Windows File Explorer

**Ako koristite macOS:**

1. Desnom tipkom miša kliknite na ikonu Finder i odaberite stavku izbornika Poveži se s poslužiteljem....

![Finder Poveži se s poslužiteljem...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Unesite URL poslužitelja iz iOS aplikacije (URL poslužitelja nalazi se ispod teksta "Unesite ovaj URL u WebDAV aplikaciju").

![Finder Unos adrese poslužitelja](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Dodirnite gumb Poveži.
4. Nakon uspostavljanja veze vidjet ćete sadržaj svog iPhonea u Mac Finderu.

![Finder povezan s Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Koristite povuci i ispusti, premjesti i preimenuj naredbe za upravljanje datotekama na vašem uređaju.

**Ako koristite Windows:**

1. Otvorite File Explorer i desnom tipkom miša kliknite na stavku izbornika "Ovo računalo" u lijevoj bočnoj traci.

![Windows Povezivanje s Wi-Fi Drive poslužiteljem](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Odaberite "Mapiraj mrežni pogon" u padajućem izborniku.
3. U tekstualno polje "Mapa" unesite URL poslužitelja iz iOS aplikacije (URL poslužitelja nalazi se ispod teksta "Unesite ovaj URL u WebDAV aplikaciju").
4. Odaberite slovo pogona i kliknite gumb "Završi".

![Windows Povezivanje s Wi-Fi Drive poslužiteljem Unos naziva diska](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Ako je veza uspješna, možete upravljati datotekama na svom iOS uređaju u sljedećem prozoru.

![Windows povezan s Wi-Fi Drive poslužiteljem](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Zaključak

S Wi-Fi Drive možete jednostavno i bežično prenositi datoteke između svog računala i iOS uređaja, poboljšavajući svoje mogućnosti upravljanja i dijeljenja datoteka.

iTunes nije potreban!

## Često postavljana pitanja

{{% details title="Trebam li iTunes za prijenos datoteka na svoj iPhone?" closed="true" %}}
Ne. Wi-Fi Drive prenosi datoteke izravno putem vaše lokalne Wi-Fi mreže. iTunes nije potreban.
{{% /details %}}

{{% details title="Koje aplikacije podržavaju Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive je dostupan u Evermusic, Flacbox, Evertag i Evervideo za iOS.
{{% /details %}}

{{% details title="Moraju li oba uređaja biti na istoj Wi-Fi mreži?" closed="true" %}}
Da. Vaše računalo i iPhone ili iPad moraju biti spojeni na istu lokalnu Wi-Fi mrežu da bi Wi-Fi Drive radio.
{{% /details %}}

{{% details title="Mogu li prenositi cijele mape, a ne samo pojedinačne datoteke?" closed="true" %}}
Da. Wi-Fi Drive podržava prijenos i preuzimanje cijelih mapa putem sučelja web preglednika.
{{% /details %}}

{{% details title="Radi li Wi-Fi Drive s Windowsom?" closed="true" %}}
Da. Možete koristiti bilo koji web preglednik na Windowsu ili se povezati putem Windows File Explorer koristeći WebDAV protokol.
{{% /details %}}

{{% details title="Mogu li koristiti USB kabel za ubrzavanje prijenosa?" closed="true" %}}
Da. Ako je vaš iPhone spojen na Mac putem USB dok Wi-Fi Drive radi, prijenos će koristiti kabelsku vezu za brže brzine.
{{% /details %}}

**Resumen:** Wi-Fi Drive le permite transferir archivos desde cualquier ordenador a su iPhone o iPad a través de Wi-Fi -- sin necesidad de iTunes ni cables. Use un navegador web, Mac Finder o Windows File Explorer. Ambos dispositivos deben estar en la misma red Wi-Fi.

Wi-Fi Drive es la forma más fácil de transferir sus archivos del ordenador a un iPhone o iPad sin iTunes. Puede descargar o subir fácilmente múltiples archivos o incluso carpetas usando una conexión Wi-Fi local. Ambos dispositivos deben estar conectados a la misma red Wi-Fi.

Puede transferir archivos de forma inalámbrica usando un navegador web de escritorio o un cliente WebDAV como Mac Finder o Windows File Explorer.

## Transferir archivos del ordenador de forma inalámbrica con un navegador web de escritorio

1. Abra cualquiera de nuestras aplicaciones como [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Inicie la aplicación > vaya a "Conexiones" > "Ordenador" > "Conectar mediante Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Pantalla de conexión Wi-Fi Drive" width="400" >}}

1. Introduzca nombre de usuario y contraseña para hacer la conexión más segura, pero este paso es opcional.
2. Toque "Iniciar Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Pantalla Wi-Fi Drive iniciado" width="400" >}}

5. Copie la URL del navegador web.
6. Abra un navegador web en su escritorio (navegadores compatibles: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Introduzca la URL del paso 5 en el campo de dirección de su navegador de escritorio.

![Pantalla del navegador Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Una vez establecida la conexión, verá la aplicación web del gestor de archivos. Todas las operaciones comunes están soportadas. Puede copiar, mover, eliminar, subir y descargar sus archivos del ordenador al almacenamiento local de su dispositivo.<br><br>Use arrastrar y soltar o el botón de subida para transferir música a su iPhone.<br><br>Además, si su dispositivo está conectado a su Mac mediante un cable, esto puede acelerar las transferencias de archivos. En este caso, su Mac configurará un servidor virtual en línea, pero la conexión utilizará el cable para velocidades de transferencia más rápidas.

![Pantalla de subida de archivos Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Toque "Detener Wi-Fi Drive" en el iPhone cuando haya terminado.

*Nota: asegúrese de usar la última versión del navegador de escritorio y de que JavaScript no esté desactivado. Esto solucionará cualquier problema si no se puede establecer la conexión.*

## Transferir archivos de audio con Mac Finder o Windows File Explorer

**Si usa macOS:**

1. Haga clic derecho en el icono del Finder y seleccione el elemento de menú Conectar al servidor....

![Finder Conectar al servidor...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Introduzca la URL del servidor desde la aplicación iOS (la URL del servidor se encuentra debajo del texto "Introduzca esta URL en la aplicación WebDAV").

![Finder Introducir dirección del servidor](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Toque el botón Conectar.
4. Una vez establecida la conexión, verá el contenido de su iPhone en Mac Finder.

![Finder conectado a Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Use arrastrar y soltar, mover y renombrar para gestionar archivos en su dispositivo.

**Si usa Windows:**

1. Abra File Explorer y haga clic derecho en el elemento de menú "Este equipo" en la barra lateral izquierda.

![Windows Conectar al servidor Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Seleccione "Conectar a unidad de red" en el menú desplegable.
3. En el campo de texto "Carpeta" introduzca la URL del servidor desde la aplicación iOS (la URL del servidor se encuentra debajo del texto "Introduzca esta URL en la aplicación WebDAV").
4. Seleccione la letra de unidad y haga clic en el botón "Finalizar".

![Windows Conectar al servidor Wi-Fi Drive Introducir nombre del disco](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Si la conexión es exitosa, puede gestionar los archivos de su dispositivo iOS en la siguiente ventana.

![Windows conectado al servidor Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusión

Con Wi-Fi Drive, puede transferir archivos fácilmente y de forma inalámbrica entre su ordenador y dispositivo iOS, mejorando sus capacidades de gestión y compartición de archivos.

¡No se necesita iTunes!

## Preguntas frecuentes

{{% details title="¿Necesito iTunes para transferir archivos a mi iPhone?" closed="true" %}}
No. Wi-Fi Drive transfiere archivos directamente a través de su red Wi-Fi local. No se necesita iTunes.
{{% /details %}}

{{% details title="¿Qué aplicaciones soportan Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive está disponible en Evermusic, Flacbox, Evertag y Evervideo para iOS.
{{% /details %}}

{{% details title="¿Deben ambos dispositivos estar en la misma red Wi-Fi?" closed="true" %}}
Sí. Su ordenador y iPhone o iPad deben estar conectados a la misma red Wi-Fi local para que Wi-Fi Drive funcione.
{{% /details %}}

{{% details title="¿Puedo transferir carpetas enteras, no solo archivos individuales?" closed="true" %}}
Sí. Wi-Fi Drive admite la subida y descarga de carpetas enteras a través de la interfaz del navegador web.
{{% /details %}}

{{% details title="¿Funciona Wi-Fi Drive con Windows?" closed="true" %}}
Sí. Puede usar cualquier navegador web en Windows o conectarse a través de Windows File Explorer usando el protocolo WebDAV.
{{% /details %}}

{{% details title="¿Puedo usar un cable USB para acelerar la transferencia?" closed="true" %}}
Sí. Si su iPhone está conectado a su Mac vía USB mientras Wi-Fi Drive está funcionando, la transferencia usará la conexión por cable para velocidades más rápidas.
{{% /details %}}

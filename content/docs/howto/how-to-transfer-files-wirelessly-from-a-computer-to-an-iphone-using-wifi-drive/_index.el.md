**Περίληψη:** Το Wi-Fi Drive σας επιτρέπει να μεταφέρετε αρχεία από οποιονδήποτε υπολογιστή στο iPhone ή iPad σας μέσω Wi-Fi -- χωρίς iTunes ή καλώδια. Χρησιμοποιήστε ένα πρόγραμμα περιήγησης, Mac Finder ή Windows File Explorer. Και οι δύο συσκευές πρέπει να είναι στο ίδιο δίκτυο Wi-Fi.

Το Wi-Fi Drive είναι ο ευκολότερος τρόπος να μεταφέρετε τα αρχεία σας από τον υπολογιστή σε iPhone ή iPad χωρίς iTunes. Μπορείτε εύκολα να κατεβάσετε ή να ανεβάσετε πολλαπλά αρχεία ή ακόμα και φακέλους χρησιμοποιώντας τοπική σύνδεση Wi-Fi. Και οι δύο συσκευές πρέπει να είναι συνδεδεμένες στο ίδιο δίκτυο Wi-Fi.

Μπορείτε να μεταφέρετε αρχεία ασύρματα χρησιμοποιώντας ένα πρόγραμμα περιήγησης επιφάνειας εργασίας ή ένα πρόγραμμα-πελάτη WebDAV όπως Mac Finder ή Windows File Explorer.

## Μεταφορά αρχείων από τον υπολογιστή ασύρματα με πρόγραμμα περιήγησης επιφάνειας εργασίας

1. Ανοίξτε οποιαδήποτε από τις εφαρμογές μας όπως [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Εκκινήστε την εφαρμογή > μεταβείτε στις "Συνδέσεις" > "Υπολογιστής" > "Σύνδεση μέσω Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Οθόνη σύνδεσης Wi-Fi Drive" width="400" >}}

1. Εισάγετε όνομα χρήστη και κωδικό πρόσβασης για πιο ασφαλή σύνδεση, αλλά αυτό το βήμα είναι προαιρετικό.
2. Πατήστε "Εκκίνηση Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Οθόνη εκκίνησης Wi-Fi Drive" width="400" >}}

5. Αντιγράψτε τη διεύθυνση URL του προγράμματος περιήγησης.
6. Ανοίξτε ένα πρόγραμμα περιήγησης στην επιφάνεια εργασίας σας (υποστηριζόμενα προγράμματα περιήγησης: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Εισάγετε τη διεύθυνση URL από το βήμα 5 στο πεδίο διεύθυνσης του προγράμματος περιήγησης.

![Οθόνη προγράμματος περιήγησης Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Μετά τη σύνδεση θα δείτε τη διαδικτυακή εφαρμογή διαχείρισης αρχείων. Υποστηρίζονται όλες οι κοινές λειτουργίες. Μπορείτε να αντιγράψετε, μετακινήσετε, διαγράψετε, ανεβάσετε και κατεβάσετε τα αρχεία σας από τον υπολογιστή στον τοπικό αποθηκευτικό χώρο της συσκευής σας.<br><br>Χρησιμοποιήστε μεταφορά και απόθεση ή το κουμπί ανεβάσματος για να μεταφέρετε μουσική στο iPhone σας.<br><br>Επιπλέον, αν η συσκευή σας είναι συνδεδεμένη στο Mac σας με καλώδιο, αυτό μπορεί να επιταχύνει τη μεταφορά αρχείων. Σε αυτή την περίπτωση, ο Mac σας θα δημιουργήσει έναν εικονικό διακομιστή, αλλά η σύνδεση θα χρησιμοποιεί το καλώδιο για ταχύτερες ταχύτητες μεταφοράς.

![Οθόνη ανεβάσματος αρχείων Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Πατήστε "Διακοπή Wi-Fi Drive" στο iPhone όταν τελειώσετε.

*Σημείωση: βεβαιωθείτε ότι χρησιμοποιείτε την τελευταία έκδοση του προγράμματος περιήγησης και ότι η JavaScript δεν είναι απενεργοποιημένη. Αυτό θα επιλύσει τυχόν προβλήματα αν δεν μπορεί να δημιουργηθεί η σύνδεση.*

## Μεταφορά αρχείων ήχου με Mac Finder ή Windows File Explorer

**Αν χρησιμοποιείτε macOS:**

1. Κάντε δεξί κλικ στο εικονίδιο του Finder και επιλέξτε το στοιχείο μενού Σύνδεση σε διακομιστή....

![Finder Σύνδεση σε διακομιστή...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Εισάγετε τη διεύθυνση URL του διακομιστή από την εφαρμογή iOS (η διεύθυνση URL του διακομιστή βρίσκεται κάτω από το κείμενο "Εισάγετε αυτή τη διεύθυνση URL στην εφαρμογή WebDAV").

![Finder Εισαγωγή διεύθυνσης διακομιστή](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Πατήστε το κουμπί Σύνδεση.
4. Μετά τη σύνδεση θα δείτε τα περιεχόμενα του iPhone σας στο Mac Finder.

![Finder συνδεδεμένο στο Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Χρησιμοποιήστε μεταφορά και απόθεση, μετακίνηση και μετονομασία για να διαχειριστείτε αρχεία στη συσκευή σας.

**Αν χρησιμοποιείτε Windows:**

1. Ανοίξτε τον File Explorer και κάντε δεξί κλικ στο στοιχείο μενού "Αυτός ο υπολογιστής" στην αριστερή πλαϊνή γραμμή.

![Windows Σύνδεση σε διακομιστή Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Επιλέξτε "Αντιστοίχιση μονάδας δίσκου δικτύου" στο αναπτυσσόμενο μενού.
3. Στο πεδίο κειμένου "Φάκελος" εισάγετε τη διεύθυνση URL του διακομιστή από την εφαρμογή iOS (η διεύθυνση URL του διακομιστή βρίσκεται κάτω από το κείμενο "Εισάγετε αυτή τη διεύθυνση URL στην εφαρμογή WebDAV").
4. Επιλέξτε το γράμμα μονάδας δίσκου και κάντε κλικ στο κουμπί "Τέλος".

![Windows Σύνδεση σε διακομιστή Wi-Fi Drive Εισαγωγή ονόματος δίσκου](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Αν η σύνδεση είναι επιτυχής, μπορείτε να διαχειριστείτε τα αρχεία στη συσκευή iOS σας στο επόμενο παράθυρο.

![Windows συνδεδεμένο σε διακομιστή Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Συμπέρασμα

Με το Wi-Fi Drive, μπορείτε εύκολα και ασύρματα να μεταφέρετε αρχεία μεταξύ του υπολογιστή σας και της συσκευής iOS, βελτιώνοντας τις δυνατότητες διαχείρισης και κοινής χρήσης αρχείων.

Δεν απαιτείται iTunes!

{{% /details %}}

{{% /details %}}

{{% details title="Λειτουργεί το Wi-Fi Drive με Windows;" closed="true" %}}
Ναι. Μπορείτε να χρησιμοποιήσετε οποιοδήποτε πρόγραμμα περιήγησης στα Windows ή να συνδεθείτε μέσω Windows File Explorer χρησιμοποιώντας το πρωτόκολλο WebDAV.
{{% /details %}}

{{% details title="Μπορώ να χρησιμοποιήσω καλώδιο USB για να επιταχύνω τη μεταφορά;" closed="true" %}}
Ναι. Αν το iPhone σας είναι συνδεδεμένο στο Mac σας μέσω USB ενώ λειτουργεί το Wi-Fi Drive, η μεταφορά θα χρησιμοποιεί τη σύνδεση καλωδίου για ταχύτερες ταχύτητες.

**Kort sagt:** Wi-Fi Drive lader dig overføre filer fra enhver computer til din iPhone eller iPad via Wi-Fi -- ingen iTunes eller kabler nødvendige. Brug en webbrowser, Mac Finder eller Windows File Explorer. Begge enheder skal være på det samme Wi-Fi-netværk.

Wi-Fi Drive er den nemmeste måde at overføre dine filer fra computeren til en iPhone eller iPad uden iTunes. Du kan nemt downloade eller uploade flere filer eller endda mapper ved hjælp af en lokal Wi-Fi-forbindelse. Begge enheder skal være forbundet til det samme Wi-Fi-netværk.

Du kan overføre filer trådløst ved hjælp af en desktop webbrowser eller WebDAV-klient som Mac Finder eller Windows File Explorer.

Du kan se en videovejledning fra [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) eller læse tekstversionen nedenfor.

## Overfør filer fra computeren trådløst med en desktop webbrowser

1. Åbn en af vores apps som [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Start appen > gå til "Forbindelser" > "Computer" > "Forbind via Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive forbindelsesskærm" width="400" >}}

1. Indtast brugernavn og adgangskode for at gøre forbindelsen mere sikker, men dette trin er valgfrit.
2. Tryk på "Start Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive startet skærm" width="400" >}}

5. Kopiér webbrowser-URL'en.
6. Åbn en webbrowser på din desktop (understøttede browsere: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Indtast URL'en fra trin 5 i adressefeltet i din desktop browser.

![Wi-Fi Drive browserskærm](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Når forbindelsen er etableret, vil du se filhåndterings-webappen. Alle almindelige operationer er understøttet. Du kan kopiere, flytte, slette, uploade og downloade dine filer fra computeren til det lokale lager på din enhed.<br><br>Brug træk og slip eller Upload-knappen til at overføre musik til din iPhone.<br><br>Derudover, hvis din enhed er forbundet til din Mac med et kabel, kan dette fremskynde filoverførsler. I dette tilfælde vil din Mac opsætte en virtuel online server, men forbindelsen vil bruge kablet til hurtigere overførselshastigheder.

![Wi-Fi Drive upload filer skærm](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Tryk på "Stop Wi-Fi Drive" på iPhonen, når du er færdig.

*Bemærk: sørg for at du bruger den nyeste version af desktop browseren, og at JavaScript ikke er deaktiveret. Dette vil løse eventuelle problemer, hvis forbindelsen ikke kan etableres.*

## Overfør lydfiler med Mac Finder eller Windows File Explorer

**Hvis du bruger macOS:**

1. Højreklik på Finder-ikonet og vælg menupunktet Opret forbindelse til server....

![Finder Opret forbindelse til server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Indtast server-URL'en fra iOS-appen (server-URL'en findes under teksten "Indtast denne URL i WebDAV-applikation").

![Finder Indtast serveradresse](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Tryk på Forbind-knappen.
4. Når forbindelsen er etableret, vil du se indholdet af din iPhone i Mac Finder.

![Finder forbundet til Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Brug træk og slip, flyt og omdøb kommandoer til at administrere filer på din enhed.

**Hvis du bruger Windows:**

1. Åbn File Explorer og højreklik på menupunktet "Denne computer" i venstre sidepanel.

![Windows Forbind til Wi-Fi Drive server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Vælg "Tilknyt netværksdrev" i rullemenuen.
3. I tekstfeltet "Mappe" indtast server-URL'en fra iOS-appen (server-URL'en findes under teksten "Indtast denne URL i WebDAV-applikation").
4. Vælg drevbogstavet og klik på "Udfør"-knappen.

![Windows Forbind til Wi-Fi Drive server Indtast disknavn](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Hvis forbindelsen er succesfuld, kan du administrere filer på din iOS-enhed i det næste vindue.

![Windows forbundet til Wi-Fi Drive server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Konklusion

Med Wi-Fi Drive kan du nemt og trådløst overføre filer mellem din computer og iOS-enhed, hvilket forbedrer dine filhåndterings- og delingsmuligheder.

Ingen iTunes nødvendig!

## Ofte stillede spørgsmål

{{% details title="Har jeg brug for iTunes til at overføre filer til min iPhone?" closed="true" %}}
Nej. Wi-Fi Drive overfører filer direkte over dit lokale Wi-Fi-netværk. iTunes er ikke nødvendig.
{{% /details %}}

{{% details title="Hvilke apps understøtter Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive er tilgængelig i Evermusic, Flacbox, Evertag og Evervideo til iOS.
{{% /details %}}

{{% details title="Skal begge enheder være på det samme Wi-Fi-netværk?" closed="true" %}}
Ja. Din computer og iPhone eller iPad skal være forbundet til det samme lokale Wi-Fi-netværk, for at Wi-Fi Drive kan fungere.
{{% /details %}}

{{% details title="Kan jeg overføre hele mapper, ikke kun individuelle filer?" closed="true" %}}
Ja. Wi-Fi Drive understøtter upload og download af hele mapper gennem webbrowsergrænsefladen.
{{% /details %}}

{{% details title="Fungerer Wi-Fi Drive med Windows?" closed="true" %}}
Ja. Du kan bruge enhver webbrowser på Windows eller forbinde via Windows File Explorer ved hjælp af WebDAV-protokollen.
{{% /details %}}

{{% details title="Kan jeg bruge et USB-kabel til at fremskynde overførslen?" closed="true" %}}
Ja. Hvis din iPhone er forbundet til din Mac via USB, mens Wi-Fi Drive kører, vil overførslen bruge kabelforbindelsen til hurtigere hastigheder.
{{% /details %}}

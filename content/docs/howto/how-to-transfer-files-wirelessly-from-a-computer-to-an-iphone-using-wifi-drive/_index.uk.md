**Коротко:** Wi-Fi Drive дозволяє передавати файли з будь-якого комп'ютера на iPhone або iPad через Wi-Fi -- без iTunes та кабелів. Використовуйте веб-браузер, Mac Finder або Windows File Explorer. Обидва пристрої повинні бути в одній мережі Wi-Fi.

Wi-Fi Drive -- це найпростіший спосіб передачі файлів з комп'ютера на iPhone або iPad без iTunes. Ви можете легко завантажувати або відправляти кілька файлів і навіть папок через локальне з'єднання Wi-Fi. Обидва пристрої повинні бути підключені до однієї мережі Wi-Fi.

Ви можете передавати файли бездротово за допомогою настільного веб-браузера або клієнта WebDAV, такого як Mac Finder або Windows File Explorer.

## Бездротова передача файлів з комп'ютера через настільний веб-браузер

1. Відкрийте будь-який з наших додатків, наприклад [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Запустіть додаток > перейдіть до "З'єднання" > "Комп'ютер" > "Підключитися через Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Введіть ім'я користувача та пароль, щоб зробити з'єднання більш безпечним, але цей крок є необов'язковим.
2. Натисніть "Запустити Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Скопіюйте URL веб-браузера.
6. Відкрийте веб-браузер на комп'ютері (підтримувані браузери: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Введіть URL з кроку 5 в адресний рядок браузера на комп'ютері.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Після встановлення з'єднання ви побачите веб-додаток файлового менеджера. Підтримуються всі стандартні операції. Ви можете копіювати, переміщувати, видаляти, завантажувати та скачувати файли з комп'ютера до локального сховища пристрою.<br><br>Використовуйте перетягування або кнопку «Завантажити» для передачі музики на iPhone.<br><br>Крім того, якщо ваш пристрій підключений до Mac за допомогою кабелю, це може прискорити передачу файлів. У цьому випадку Mac налаштує віртуальний онлайн-сервер, але з'єднання буде використовувати кабель для швидшої передачі.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Після завершення натисніть "Зупинити Wi-Fi Drive" на iPhone.

*Примітка: переконайтеся, що ви використовуєте останню версію настільного браузера і JavaScript не вимкнено. Це вирішить будь-які проблеми, якщо з'єднання не може бути встановлено.*

## Передача аудіофайлів за допомогою Mac Finder або Windows File Explorer

**Якщо ви використовуєте macOS:**

1. Клацніть правою кнопкою миші по значку Finder і виберіть пункт меню «Підключитися до сервера...».

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Введіть URL сервера з додатку iOS (URL сервера розташований під текстом «Введіть цей URL у додатку WebDAV»).

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Натисніть кнопку «Підключити».
4. Після встановлення з'єднання ви побачите вміст вашого iPhone в Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Використовуйте перетягування, команди «Перемістити» та «Перейменувати» для керування файлами на пристрої.

**Якщо ви використовуєте Windows:**

1. Відкрийте Провідник і клацніть правою кнопкою миші по пункту «Цей ПК» на лівій бічній панелі.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Виберіть «Підключити мережевий диск» у випадаючому меню.
3. У текстовому полі «Папка» введіть URL сервера з додатку iOS (URL сервера розташований під текстом «Введіть цей URL у додатку WebDAV»).
4. Виберіть букву диска і натисніть кнопку «Готово».

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Якщо підключення успішне, ви можете керувати файлами на вашому iOS-пристрої в наступному вікні.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Висновок

З Wi-Fi Drive ви можете легко та бездротово передавати файли між комп'ютером та iOS-пристроєм, покращуючи можливості керування файлами та обміну ними.

iTunes не потрібен!

## Поширені запитання

{{% details title="Чи потрібен мені iTunes для передачі файлів на iPhone?" closed="true" %}}
Ні. Wi-Fi Drive передає файли безпосередньо через вашу локальну мережу Wi-Fi. iTunes не потрібен.
{{% /details %}}

{{% details title="Які додатки підтримують Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive доступний в Evermusic, Flacbox, Evertag та Evervideo для iOS.
{{% /details %}}

{{% details title="Чи повинні обидва пристрої бути в одній мережі Wi-Fi?" closed="true" %}}
Так. Ваш комп'ютер та iPhone або iPad повинні бути підключені до однієї локальної мережі Wi-Fi для роботи Wi-Fi Drive.
{{% /details %}}

{{% details title="Чи можу я передавати цілі папки, а не лише окремі файли?" closed="true" %}}
Так. Wi-Fi Drive підтримує завантаження та скачування цілих папок через інтерфейс веб-браузера.
{{% /details %}}

{{% details title="Чи працює Wi-Fi Drive з Windows?" closed="true" %}}
Так. Ви можете використовувати будь-який веб-браузер у Windows або підключитися через Windows File Explorer за допомогою протоколу WebDAV.
{{% /details %}}

{{% details title="Чи можу я використовувати USB-кабель для прискорення передачі?" closed="true" %}}
Так. Якщо ваш iPhone підключений до Mac через USB під час роботи Wi-Fi Drive, передача буде використовувати кабельне з'єднання для більшої швидкості.
{{% /details %}}

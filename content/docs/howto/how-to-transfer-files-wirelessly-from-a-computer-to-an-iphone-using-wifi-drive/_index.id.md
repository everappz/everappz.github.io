**Ringkasan:** Wi-Fi Drive memungkinkan Anda mentransfer file dari komputer mana pun ke iPhone atau iPad melalui Wi-Fi -- tanpa iTunes atau kabel. Gunakan browser web, Mac Finder, atau Windows File Explorer. Kedua perangkat harus terhubung ke jaringan Wi-Fi yang sama.

Wi-Fi Drive adalah cara termudah untuk mentransfer file dari komputer ke iPhone atau iPad tanpa iTunes. Anda dapat dengan mudah mengunduh atau mengunggah beberapa file atau bahkan folder menggunakan koneksi Wi-Fi lokal. Kedua perangkat harus terhubung ke jaringan Wi-Fi yang sama.

Anda dapat mentransfer file secara nirkabel menggunakan browser web desktop atau klien WebDAV seperti Mac Finder atau Windows File Explorer.

Anda dapat menonton tutorial video dari [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) atau membaca versi teks di bawah ini.

## Mentransfer file dari komputer secara nirkabel dengan browser web desktop

1. Buka salah satu aplikasi kami seperti [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Jalankan aplikasi > buka "Koneksi" > "Komputer" > "Hubungkan menggunakan Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Layar koneksi Wi-Fi Drive" width="400" >}}

1. Masukkan nama pengguna dan kata sandi untuk membuat koneksi lebih aman tetapi langkah ini opsional.
2. Ketuk "Mulai Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Layar Wi-Fi Drive dimulai" width="400" >}}

5. Salin URL browser web.
6. Buka browser web di desktop Anda (browser yang didukung: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Masukkan URL dari langkah 5 ke kolom alamat browser desktop Anda.

![Layar browser Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Setelah koneksi terbuat, Anda akan melihat aplikasi web pengelola file. Semua operasi umum didukung. Anda dapat menyalin, memindahkan, menghapus, mengunggah, dan mengunduh file dari komputer ke penyimpanan lokal perangkat Anda.<br><br>Gunakan seret dan lepas atau tombol Unggah untuk mentransfer musik ke iPhone Anda.<br><br>Selain itu, jika perangkat Anda terhubung ke Mac menggunakan kabel, ini dapat mempercepat transfer file. Dalam hal ini, Mac Anda akan menyiapkan server virtual online, tetapi koneksi akan menggunakan kabel untuk kecepatan transfer yang lebih cepat.

![Layar unggah file Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Ketuk "Hentikan Wi-Fi Drive" di iPhone setelah selesai.

*Catatan: pastikan Anda menggunakan versi terbaru browser desktop dan JavaScript tidak dinonaktifkan. Ini akan memperbaiki masalah apa pun jika koneksi tidak dapat dibuat.*

## Mentransfer file audio dengan Mac Finder atau Windows File Explorer

**Jika Anda menggunakan macOS:**

1. Klik kanan pada ikon Finder dan pilih item menu Hubungkan ke Server....

![Finder Hubungkan ke Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Masukkan URL server dari aplikasi iOS (URL server terletak di bawah teks "Masukkan URL ini di aplikasi WebDAV").

![Finder Masukkan alamat server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Ketuk tombol Hubungkan.
4. Setelah koneksi terbuat, Anda akan melihat konten iPhone Anda di Mac Finder.

![Finder terhubung ke Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Gunakan seret dan lepas, pindahkan, dan ganti nama untuk mengelola file di perangkat Anda.

**Jika Anda menggunakan Windows:**

1. Buka File Explorer dan klik kanan pada item menu "Komputer Ini" di bilah sisi kiri.

![Windows Hubungkan ke server Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Pilih "Petakan Drive Jaringan" di menu tarik-turun.
3. Di kolom teks "Folder" masukkan URL server dari aplikasi iOS (URL server terletak di bawah teks "Masukkan URL ini di aplikasi WebDAV").
4. Pilih huruf Drive dan klik tombol "Selesai".

![Windows Hubungkan ke server Wi-Fi Drive Masukkan nama disk](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Jika koneksi berhasil, Anda dapat mengelola file di perangkat iOS Anda di jendela berikutnya.

![Windows terhubung ke server Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Kesimpulan

Dengan Wi-Fi Drive, Anda dapat dengan mudah dan secara nirkabel mentransfer file antara komputer dan perangkat iOS Anda, meningkatkan kemampuan pengelolaan dan berbagi file Anda.

Tidak perlu iTunes!

## Pertanyaan yang Sering Diajukan

{{% details title="Apakah saya perlu iTunes untuk mentransfer file ke iPhone saya?" closed="true" %}}
Tidak. Wi-Fi Drive mentransfer file langsung melalui jaringan Wi-Fi lokal Anda. iTunes tidak diperlukan.
{{% /details %}}

{{% details title="Aplikasi mana yang mendukung Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive tersedia di Evermusic, Flacbox, Evertag, dan Evervideo untuk iOS.
{{% /details %}}

{{% details title="Apakah kedua perangkat harus berada di jaringan Wi-Fi yang sama?" closed="true" %}}
Ya. Komputer dan iPhone atau iPad Anda harus terhubung ke jaringan Wi-Fi lokal yang sama agar Wi-Fi Drive dapat berfungsi.
{{% /details %}}

{{% details title="Bisakah saya mentransfer seluruh folder, bukan hanya file individual?" closed="true" %}}
Ya. Wi-Fi Drive mendukung pengunggahan dan pengunduhan seluruh folder melalui antarmuka browser web.
{{% /details %}}

{{% details title="Apakah Wi-Fi Drive bekerja dengan Windows?" closed="true" %}}
Ya. Anda dapat menggunakan browser web apa pun di Windows atau terhubung melalui Windows File Explorer menggunakan protokol WebDAV.
{{% /details %}}

{{% details title="Bisakah saya menggunakan kabel USB untuk mempercepat transfer?" closed="true" %}}
Ya. Jika iPhone Anda terhubung ke Mac melalui USB saat Wi-Fi Drive berjalan, transfer akan menggunakan koneksi kabel untuk kecepatan yang lebih cepat.
{{% /details %}}

**요약:** Wi-Fi 드라이브를 사용하면 Wi-Fi를 통해 모든 컴퓨터에서 iPhone 또는 iPad로 파일을 전송할 수 있습니다. iTunes나 케이블이 필요 없습니다. 웹 브라우저, Mac Finder 또는 Windows File Explorer를 사용하세요. 두 기기 모두 동일한 Wi-Fi 네트워크에 연결되어 있어야 합니다.

Wi-Fi 드라이브는 iTunes 없이 컴퓨터에서 iPhone 또는 iPad로 파일을 전송하는 가장 쉬운 방법입니다. 로컬 Wi-Fi 연결을 사용하여 여러 파일이나 폴더를 쉽게 다운로드하거나 업로드할 수 있습니다. 두 기기 모두 동일한 Wi-Fi 네트워크에 연결되어 있어야 합니다.

데스크톱 웹 브라우저 또는 Mac Finder나 Windows File Explorer와 같은 WebDAV 클라이언트를 사용하여 무선으로 파일을 전송할 수 있습니다.

## 데스크톱 웹 브라우저를 사용하여 컴퓨터에서 무선으로 파일 전송

1. [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo)와 같은 앱을 엽니다.

2. 앱을 실행하고 "연결하기" > "컴퓨터" > "Wi-Fi를 사용하여 연결"로 이동합니다.
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. 연결을 더 안전하게 하려면 사용자 이름과 비밀번호를 입력하세요. 이 단계는 선택 사항입니다.
2. "Wi-Fi 드라이브 시작"을 탭합니다.

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. 웹 브라우저 URL을 복사합니다.
6. 데스크톱에서 웹 브라우저를 엽니다(지원되는 브라우저: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. 5단계의 URL을 데스크톱 브라우저의 주소 필드에 입력합니다.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. 연결이 설정되면 파일 관리자 웹 앱이 표시됩니다. 모든 일반적인 작업이 지원됩니다. 컴퓨터에서 기기의 로컬 저장소로 파일을 복사, 이동, 삭제, 업로드 및 다운로드할 수 있습니다.<br><br>드래그 앤 드롭 또는 업로드 버튼을 사용하여 iPhone으로 음악을 전송하세요.<br><br>또한, 기기가 케이블로 Mac에 연결되어 있으면 파일 전송 속도가 빨라질 수 있습니다. 이 경우 Mac이 가상 온라인 서버를 설정하지만 연결은 케이블을 사용하여 더 빠른 전송 속도를 제공합니다.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. 완료되면 iPhone에서 "Wi-Fi 드라이브 중지"를 탭합니다.

*참고: 최신 버전의 데스크톱 브라우저를 사용하고 JavaScript가 비활성화되지 않았는지 확인하세요. 이렇게 하면 연결을 설정할 수 없는 경우 문제가 해결됩니다.*

## Mac Finder 또는 Windows File Explorer로 오디오 파일 전송

**macOS를 사용하는 경우:**

1. Finder 아이콘을 마우스 오른쪽 버튼으로 클릭하고 서버에 연결... 메뉴 항목을 선택합니다.

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. iOS 앱에서 서버 URL을 입력합니다(서버 URL은 "WebDAV 애플리케이션에서 이 URL을 입력하세요" 텍스트 아래에 있습니다).

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. 연결 버튼을 탭합니다.
4. 연결이 설정되면 Mac Finder에서 iPhone의 내용을 볼 수 있습니다.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. 드래그 앤 드롭, 이동 및 이름 변경 명령을 사용하여 기기의 파일을 관리합니다.

**Windows를 사용하는 경우:**

1. 파일 탐색기를 열고 왼쪽 사이드바에서 '내 PC' 메뉴 항목을 마우스 오른쪽 버튼으로 클릭합니다.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. 드롭다운 메뉴에서 "네트워크 드라이브 연결"을 선택합니다.
3. "폴더" 텍스트 필드에 iOS 앱의 서버 URL을 입력합니다(서버 URL은 "WebDAV 애플리케이션에서 이 URL을 입력하세요" 텍스트 아래에 있습니다).
4. 드라이브 문자를 선택하고 "마침" 버튼을 클릭합니다.

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. 연결이 성공하면 다음 창에서 iOS 기기에 있는 파일을 관리할 수 있습니다.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## 결론

Wi-Fi 드라이브를 사용하면 컴퓨터와 iOS 기기 간에 쉽고 무선으로 파일을 전송하여 파일 관리 및 공유 기능을 향상시킬 수 있습니다.

iTunes가 필요 없습니다!

{{% /details %}}

{{% /details %}}

{{% details title="USB 케이블을 사용하여 전송 속도를 높일 수 있습니까?" closed="true" %}}
예. Wi-Fi 드라이브가 실행되는 동안 iPhone이 USB를 통해 Mac에 연결되어 있으면 케이블 연결을 사용하여 더 빠른 속도로 전송됩니다.

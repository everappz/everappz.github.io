**TL;DR:** Wi-Fi Drive lar deg overføre filer fra hvilken som helst datamaskin til din iPhone eller iPad over Wi-Fi -- ingen iTunes eller kabler nødvendig. Bruk en nettleser, Mac Finder eller Windows File Explorer. Begge enhetene må være på samme Wi-Fi-nettverk.

Wi-Fi Drive er den enkleste måten å overføre filer fra datamaskinen til en iPhone eller iPad uten iTunes. Du kan enkelt laste ned eller laste opp flere filer eller til og med mapper ved hjelp av en lokal Wi-Fi-tilkobling. Begge enhetene bør være koblet til samme Wi-Fi-nettverk.

Du kan overføre filer trådløst ved hjelp av en stasjonær nettleser eller en WebDAV-klient som Mac Finder eller Windows File Explorer.

Du kan se en videoopplæring fra [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) eller lese tekstversjonen nedenfor.

## Overfør filer fra datamaskinen trådløst med en stasjonær nettleser

1. Åpne en av appene våre som [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Start appen > gå til "Tilkoblinger" > "Datamaskin" > "Koble til via Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Skriv inn brukernavn og passord for å gjøre tilkoblingen sikrere, men dette trinnet er valgfritt.
2. Trykk på "Start Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Kopier nettleser-URL-en.
6. Åpne en nettleser på skrivebordet ditt (støttede nettlesere: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Skriv inn URL-en fra trinn 5 i adressefeltet i skrivebordsnettleseren din.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Etter at tilkoblingen er opprettet, vil du se filbehandler-webappen. Alle vanlige operasjoner støttes. Du kan kopiere, flytte, slette, laste opp og laste ned filene dine fra datamaskinen til den lokale lagringen på enheten din.<br><br>Bruk dra og slipp eller Last opp-knappen for å overføre musikk til din iPhone.<br><br>I tillegg, hvis enheten din er koblet til Mac-en din med en kabel, kan dette øke filoverføringshastigheten. I dette tilfellet vil Mac-en din sette opp en virtuell nettserver, men tilkoblingen vil bruke kabelen for raskere overføringshastigheter.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Trykk på "Stopp Wi-Fi Drive" på iPhone når du er ferdig.

*Merk: sørg for at du bruker den nyeste versjonen av skrivebordsnettleseren og at JavaScript ikke er deaktivert. Dette vil løse eventuelle problemer hvis tilkoblingen ikke kan opprettes.*

## Overfør lydfiler med Mac Finder eller Windows File Explorer

**Hvis du bruker macOS:**

1. Høyreklikk på Finder-ikonet og velg menyelementet Koble til server....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Skriv inn server-URL-en fra iOS-appen (server-URL-en er plassert under teksten "Skriv inn denne URL-en i WebDAV-applikasjonen").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Trykk på Koble til-knappen.
4. Etter at tilkoblingen er opprettet, vil du se innholdet på din iPhone i Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Bruk dra og slipp, Flytt og Gi nytt navn-kommandoer for å administrere filer på enheten din.

**Hvis du bruker Windows:**

1. Åpne Filutforsker og høyreklikk på menyelementet 'Denne PC-en' i venstre sidefelt.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Velg "Koble til nettverksstasjon" fra rullegardinmenyen.
3. I tekstfeltet "Mappe" skriver du inn server-URL-en fra iOS-appen (server-URL-en er plassert under teksten "Skriv inn denne URL-en i WebDAV-applikasjonen").
4. Velg stasjonsbokstaven og klikk på "Fullfør"-knappen.

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Hvis tilkoblingen er vellykket, kan du administrere filer som finnes på iOS-enheten din i neste vindu.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Konklusjon

Med Wi-Fi Drive kan du enkelt og trådløst overføre filer mellom datamaskinen og iOS-enheten din, noe som forbedrer dine filhåndterings- og delingsmuligheter.

Ingen iTunes er nødvendig!

## Vanlige spørsmål

{{% details title="Trenger jeg iTunes for å overføre filer til min iPhone?" closed="true" %}}
Nei. Wi-Fi Drive overfører filer direkte over ditt lokale Wi-Fi-nettverk. iTunes er ikke nødvendig.
{{% /details %}}

{{% details title="Hvilke apper støtter Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive er tilgjengelig i Evermusic, Flacbox, Evertag og Evervideo for iOS.
{{% /details %}}

{{% details title="Må begge enhetene være på samme Wi-Fi-nettverk?" closed="true" %}}
Ja. Datamaskinen og iPhone eller iPad må være koblet til samme lokale Wi-Fi-nettverk for at Wi-Fi Drive skal fungere.
{{% /details %}}

{{% details title="Kan jeg overføre hele mapper, ikke bare individuelle filer?" closed="true" %}}
Ja. Wi-Fi Drive støtter opplasting og nedlasting av hele mapper gjennom nettlesergrensesnittet.
{{% /details %}}

{{% details title="Fungerer Wi-Fi Drive med Windows?" closed="true" %}}
Ja. Du kan bruke hvilken som helst nettleser på Windows eller koble til via Windows File Explorer ved hjelp av WebDAV-protokollen.
{{% /details %}}

{{% details title="Kan jeg bruke en USB-kabel for å øke overføringshastigheten?" closed="true" %}}
Ja. Hvis din iPhone er koblet til Mac-en din via USB mens Wi-Fi Drive kjører, vil overføringen bruke kabeltilkoblingen for raskere hastigheter.
{{% /details %}}

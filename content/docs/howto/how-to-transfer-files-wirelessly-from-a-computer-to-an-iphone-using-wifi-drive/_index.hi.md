**संक्षेप में:** Wi-Fi Drive आपको किसी भी कंप्यूटर से अपने iPhone या iPad में Wi-Fi के माध्यम से फ़ाइलें ट्रांसफर करने देता है -- बिना iTunes या केबल के। वेब ब्राउज़र, Mac Finder या Windows File Explorer का उपयोग करें। दोनों डिवाइस एक ही Wi-Fi नेटवर्क पर होने चाहिए।

Wi-Fi Drive बिना iTunes के कंप्यूटर से iPhone या iPad में फ़ाइलें ट्रांसफर करने का सबसे आसान तरीका है। आप स्थानीय Wi-Fi कनेक्शन का उपयोग करके आसानी से कई फ़ाइलें या फ़ोल्डर भी डाउनलोड या अपलोड कर सकते हैं। दोनों डिवाइस एक ही Wi-Fi नेटवर्क से जुड़े होने चाहिए।

आप डेस्कटॉप वेब ब्राउज़र या WebDAV क्लाइंट जैसे Mac Finder या Windows File Explorer का उपयोग करके वायरलेस तरीके से फ़ाइलें ट्रांसफर कर सकते हैं।

आप [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) का वीडियो ट्यूटोरियल देख सकते हैं या नीचे टेक्स्ट संस्करण पढ़ सकते हैं।

## डेस्कटॉप वेब ब्राउज़र के साथ कंप्यूटर से वायरलेस तरीके से फ़ाइलें ट्रांसफर करें

1. हमारी किसी भी ऐप को खोलें जैसे [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo)।

2. ऐप लॉन्च करें > "संपर्क" > "कंप्यूटर" > "Wi-Fi से कनेक्ट करें" पर जाएं।
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive कनेक्ट स्क्रीन" width="400" >}}

1. कनेक्शन को अधिक सुरक्षित बनाने के लिए यूज़रनेम और पासवर्ड दर्ज करें लेकिन यह चरण वैकल्पिक है।
2. "Wi-Fi Drive शुरू करें" पर टैप करें।

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive शुरू स्क्रीन" width="400" >}}

5. वेब ब्राउज़र URL कॉपी करें।
6. अपने डेस्कटॉप पर एक वेब ब्राउज़र खोलें (समर्थित ब्राउज़र: Safari, Google Chrome, Opera, Yandex Browser, FireFox)।
7. चरण 5 से URL को अपने डेस्कटॉप ब्राउज़र के एड्रेस फ़ील्ड में दर्ज करें।

![Wi-Fi Drive ब्राउज़र स्क्रीन](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. कनेक्शन स्थापित होने के बाद आपको फ़ाइल मैनेजर वेब ऐप दिखाई देगा। सभी सामान्य ऑपरेशन समर्थित हैं। आप कंप्यूटर से अपने डिवाइस के लोकल स्टोरेज में फ़ाइलें कॉपी, मूव, डिलीट, अपलोड और डाउनलोड कर सकते हैं।<br><br>अपने iPhone में संगीत ट्रांसफर करने के लिए ड्रैग और ड्रॉप या अपलोड बटन का उपयोग करें।<br><br>इसके अतिरिक्त, यदि आपका डिवाइस केबल का उपयोग करके आपके Mac से जुड़ा है, तो यह फ़ाइल ट्रांसफर को तेज़ कर सकता है। इस स्थिति में, आपका Mac एक वर्चुअल ऑनलाइन सर्वर सेट करेगा, लेकिन कनेक्शन तेज़ ट्रांसफर स्पीड के लिए केबल का उपयोग करेगा।

![Wi-Fi Drive फ़ाइलें अपलोड स्क्रीन](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. जब आप समाप्त कर लें तो iPhone पर "Wi-Fi Drive बंद करें" पर टैप करें।

*नोट: कृपया सुनिश्चित करें कि आप डेस्कटॉप ब्राउज़र का नवीनतम संस्करण उपयोग कर रहे हैं और JavaScript अक्षम नहीं है। यदि कनेक्शन स्थापित नहीं हो पा रहा है तो यह किसी भी समस्या को ठीक कर देगा।*

## Mac Finder या Windows File Explorer के साथ ऑडियो फ़ाइलें ट्रांसफर करें

**यदि आप macOS का उपयोग करते हैं:**

1. Finder आइकन पर राइट-क्लिक करें और सर्वर से कनेक्ट करें... मेनू आइटम चुनें।

![Finder सर्वर से कनेक्ट करें...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. iOS ऐप से सर्वर URL दर्ज करें (सर्वर URL "इस URL को WebDAV एप्लिकेशन में दर्ज करें" टेक्स्ट के नीचे स्थित है)।

![Finder सर्वर एड्रेस दर्ज करें](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. कनेक्ट बटन पर टैप करें।
4. कनेक्शन स्थापित होने के बाद आपको Mac Finder में अपने iPhone की सामग्री दिखाई देगी।

![Finder Wi-Fi Drive से कनेक्टेड](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. अपने डिवाइस पर फ़ाइलें प्रबंधित करने के लिए ड्रैग और ड्रॉप, मूव और रीनेम कमांड का उपयोग करें।

**यदि आप Windows का उपयोग करते हैं:**

1. File Explorer खोलें और बाएं साइडबार में "यह कंप्यूटर" मेनू आइटम पर राइट-क्लिक करें।

![Windows Wi-Fi Drive सर्वर से कनेक्ट करें](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. ड्रॉप-डाउन मेनू में "नेटवर्क ड्राइव मैप करें" चुनें।
3. "फ़ोल्डर" टेक्स्ट फ़ील्ड में iOS ऐप से सर्वर URL दर्ज करें (सर्वर URL "इस URL को WebDAV एप्लिकेशन में दर्ज करें" टेक्स्ट के नीचे स्थित है)।
4. ड्राइव लेटर चुनें और "समाप्त" बटन पर क्लिक करें।

![Windows Wi-Fi Drive सर्वर से कनेक्ट करें डिस्क नाम दर्ज करें](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. यदि कनेक्शन सफल है तो आप अगली विंडो में अपने iOS डिवाइस पर स्थित फ़ाइलों का प्रबंधन कर सकते हैं।

![Windows Wi-Fi Drive सर्वर से कनेक्टेड](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## निष्कर्ष

Wi-Fi Drive के साथ, आप आसानी से और वायरलेस तरीके से अपने कंप्यूटर और iOS डिवाइस के बीच फ़ाइलें ट्रांसफर कर सकते हैं, जिससे आपकी फ़ाइल प्रबंधन और शेयरिंग क्षमताएं बढ़ती हैं।

iTunes की कोई आवश्यकता नहीं!

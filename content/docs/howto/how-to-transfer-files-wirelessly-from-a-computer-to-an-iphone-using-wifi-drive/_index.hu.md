**Összefoglalás:** A Wi-Fi Drive lehetővé teszi fájlok átvitelét bármely számítógépről iPhone-ra vagy iPadre Wi-Fi-n keresztül -- iTunes vagy kábelek nélkül. Használjon webböngészőt, Mac Findert vagy Windows File Explorert. Mindkét eszköznek ugyanazon a Wi-Fi hálózaton kell lennie.

A Wi-Fi Drive a legegyszerűbb módja a fájlok átvitelének számítógépről iPhone-ra vagy iPadre iTunes nélkül. Könnyedén letölthet vagy feltölthet több fájlt vagy akár mappákat helyi Wi-Fi kapcsolaton keresztül. Mindkét eszköznek ugyanahhoz a Wi-Fi hálózathoz kell csatlakoznia.

Fájlokat vezeték nélkül vihet át asztali webböngésző vagy WebDAV kliens, például Mac Finder vagy Windows File Explorer segítségével.

Megtekinthet egy videós útmutatót a [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) csatornáról, vagy elolvashatja az alábbi szöveges verziót.

## Fájlok átvitele számítógépről vezeték nélkül asztali webböngészővel

1. Nyissa meg bármelyik alkalmazásunkat, például az [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo) alkalmazást.

2. Indítsa el az alkalmazást > lépjen a "Kapcsolatok" > "Számítógép" > "Csatlakozás Wi-Fi-n keresztül" menüpontra.
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive csatlakozási képernyő" width="400" >}}

1. Adjon meg felhasználónevet és jelszót a biztonságosabb kapcsolathoz, de ez a lépés opcionális.
2. Érintse meg a "Wi-Fi Drive indítása" gombot.

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive elindítva képernyő" width="400" >}}

5. Másolja ki a webböngésző URL-jét.
6. Nyisson meg egy webböngészőt az asztalán (támogatott böngészők: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Írja be az 5. lépés URL-jét az asztali böngésző címmezőjébe.

![Wi-Fi Drive böngésző képernyő](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. A kapcsolat létrehozása után megjelenik a fájlkezelő webalkalmazás. Minden gyakori művelet támogatott. Fájlokat másolhat, mozgathat, törölhet, feltölthet és letölthet a számítógépről az eszköz helyi tárolójába.<br><br>Használja a húzd és ejtsd módszert vagy a Feltöltés gombot zenék iPhone-ra való átviteléhez.<br><br>Ezenkívül, ha eszköze kábellel csatlakozik a Machez, ez felgyorsíthatja a fájlátvitelt. Ebben az esetben a Mac virtuális online szervert állít be, de a kapcsolat a kábelt használja a gyorsabb átviteli sebességek érdekében.

![Wi-Fi Drive fájlok feltöltése képernyő](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Ha végzett, érintse meg a "Wi-Fi Drive leállítása" gombot az iPhone-on.

*Megjegyzés: győződjön meg arról, hogy az asztali böngésző legújabb verzióját használja, és a JavaScript nincs letiltva. Ez megoldja az esetleges problémákat, ha a kapcsolat nem hozható létre.*

## Hangfájlok átvitele Mac Finder vagy Windows File Explorer segítségével

**Ha macOS-t használ:**

1. Kattintson jobb gombbal a Finder ikonra, és válassza a Csatlakozás szerverhez... menüpontot.

![Finder Csatlakozás szerverhez...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Írja be a szerver URL-jét az iOS alkalmazásból (a szerver URL a "Írja be ezt az URL-t a WebDAV alkalmazásba" szöveg alatt található).

![Finder Szervercím megadása](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Érintse meg a Csatlakozás gombot.
4. A kapcsolat létrehozása után megjelenik az iPhone tartalma a Mac Finderben.

![Finder csatlakozva a Wi-Fi Drive-hoz](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Használja a húzd és ejtsd, mozgatás és átnevezés parancsokat a fájlok kezeléséhez az eszközön.

**Ha Windows-t használ:**

1. Nyissa meg a File Explorert, és kattintson jobb gombbal az "Ez a gép" menüpontra a bal oldali oldalsávban.

![Windows Csatlakozás Wi-Fi Drive szerverhez](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Válassza a "Hálózati meghajtó csatlakoztatása" lehetőséget a legördülő menüből.
3. A "Mappa" szövegmezőbe írja be a szerver URL-jét az iOS alkalmazásból (a szerver URL a "Írja be ezt az URL-t a WebDAV alkalmazásba" szöveg alatt található).
4. Válassza ki a meghajtó betűjelét, és kattintson a "Befejezés" gombra.

![Windows Csatlakozás Wi-Fi Drive szerverhez Lemeznév megadása](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Ha a kapcsolat sikeres, a következő ablakban kezelheti az iOS eszközön lévő fájlokat.

![Windows csatlakozva a Wi-Fi Drive szerverhez](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Összegzés

A Wi-Fi Drive segítségével egyszerűen és vezeték nélkül vihet át fájlokat számítógépe és iOS eszköze között, javítva fájlkezelési és megosztási lehetőségeit.

Nincs szükség iTunes-ra!

## Gyakran ismételt kérdések

{{% details title="Szükségem van iTunes-ra fájlok átviteléhez az iPhone-ra?" closed="true" %}}
Nem. A Wi-Fi Drive közvetlenül a helyi Wi-Fi hálózaton keresztül viszi át a fájlokat. Nincs szükség iTunes-ra.
{{% /details %}}

{{% details title="Mely alkalmazások támogatják a Wi-Fi Drive-ot?" closed="true" %}}
A Wi-Fi Drive elérhető az Evermusic, Flacbox, Evertag és Evervideo alkalmazásokban iOS-re.
{{% /details %}}

{{% details title="Mindkét eszköznek ugyanazon a Wi-Fi hálózaton kell lennie?" closed="true" %}}
Igen. A számítógépnek és az iPhone-nak vagy iPadnek ugyanahhoz a helyi Wi-Fi hálózathoz kell csatlakoznia, hogy a Wi-Fi Drive működjön.
{{% /details %}}

{{% details title="Átvihetek teljes mappákat, nem csak egyedi fájlokat?" closed="true" %}}
Igen. A Wi-Fi Drive támogatja teljes mappák feltöltését és letöltését a webböngésző felületén keresztül.
{{% /details %}}

{{% details title="Működik a Wi-Fi Drive Windows-zal?" closed="true" %}}
Igen. Bármely webböngészőt használhatja Windows-on, vagy csatlakozhat a Windows File Explorer segítségével a WebDAV protokollon keresztül.
{{% /details %}}

{{% details title="Használhatok USB kábelt az átvitel gyorsításához?" closed="true" %}}
Igen. Ha az iPhone USB-n keresztül csatlakozik a Machez, miközben a Wi-Fi Drive fut, az átvitel a kábelkapcsolatot használja a gyorsabb sebesség érdekében.
{{% /details %}}

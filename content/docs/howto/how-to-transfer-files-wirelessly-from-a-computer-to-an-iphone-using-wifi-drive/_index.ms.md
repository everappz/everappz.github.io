**TL;DR:** Wi-Fi Drive membolehkan anda memindahkan fail dari mana-mana komputer ke iPhone atau iPad anda melalui Wi-Fi -- tanpa memerlukan iTunes atau kabel. Gunakan pelayar web, Mac Finder, atau Windows File Explorer. Kedua-dua peranti mesti berada dalam rangkaian Wi-Fi yang sama.

Wi-Fi Drive ialah cara paling mudah untuk memindahkan fail anda dari komputer ke iPhone atau iPad tanpa iTunes. Anda boleh memuat turun atau memuat naik berbilang fail atau folder dengan mudah menggunakan sambungan Wi-Fi tempatan. Kedua-dua peranti harus disambungkan ke rangkaian Wi-Fi yang sama.

Anda boleh memindahkan fail secara tanpa wayar menggunakan pelayar web desktop atau klien WebDAV seperti Mac Finder atau Windows File Explorer.

Anda boleh menonton tutorial video daripada [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) atau membaca versi teks di bawah.

## Pindahkan fail dari komputer secara tanpa wayar dengan pelayar web desktop

1. Buka mana-mana aplikasi kami seperti [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Lancarkan aplikasi > pergi ke "Sambungan" > "Komputer" > "Sambung menggunakan Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Masukkan nama pengguna dan kata laluan untuk menjadikan sambungan lebih selamat tetapi langkah ini adalah pilihan.
2. Ketik "Mulakan Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Salin URL pelayar web.
6. Buka pelayar web di desktop anda (pelayar yang disokong: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Masukkan URL dari langkah 5 ke medan alamat pelayar desktop anda.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Selepas sambungan diwujudkan, anda akan melihat aplikasi web pengurus fail. Semua operasi biasa disokong. Anda boleh Menyalin, Memindahkan, Memadam, Memuat Naik, dan Memuat Turun fail anda dari komputer ke storan tempatan peranti anda.<br><br>Gunakan seret dan lepas atau butang Muat Naik untuk memindahkan muzik ke iPhone anda.<br><br>Selain itu, jika peranti anda disambungkan ke Mac anda menggunakan kabel, ini boleh mempercepatkan pemindahan fail. Dalam kes ini, Mac anda akan menyediakan pelayan dalam talian maya, tetapi sambungan akan menggunakan kabel untuk kelajuan pemindahan yang lebih pantas.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Ketik "Hentikan Wi-Fi Drive" pada iPhone apabila anda selesai.

*Nota: sila pastikan anda menggunakan versi terkini pelayar desktop dan JavaScript tidak dinyahaktifkan. Ini akan membetulkan sebarang masalah jika sambungan tidak dapat diwujudkan.*

## Pindahkan fail audio dengan Mac Finder atau Windows File Explorer

**Jika anda menggunakan macOS:**

1. Klik kanan pada ikon Finder dan pilih item menu Sambung ke Pelayan....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Masukkan URL pelayan dari aplikasi iOS (URL pelayan terletak di bawah teks "Masukkan URL ini dalam aplikasi WebDAV").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Ketik butang Sambung.
4. Selepas sambungan diwujudkan, anda akan melihat kandungan iPhone anda dalam Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Gunakan seret dan lepas, arahan Pindahkan, dan Namakan Semula untuk mengurus fail pada peranti anda.

**Jika anda menggunakan Windows:**

1. Buka File Explorer dan klik kanan pada item menu 'PC Ini' di bar sisi kiri.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Pilih "Petakan Pemacu Rangkaian" pada menu lungsur.
3. Pada medan teks "Folder" masukkan URL pelayan dari aplikasi iOS (URL pelayan terletak di bawah teks "Masukkan URL ini dalam aplikasi WebDAV").
4. Pilih huruf Pemacu dan klik butang "Selesai".

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Jika sambungan berjaya, anda boleh mengurus fail yang terletak pada peranti iOS anda pada tetingkap seterusnya.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Kesimpulan

Dengan Wi-Fi Drive, anda boleh memindahkan fail antara komputer dan peranti iOS anda dengan mudah dan tanpa wayar, meningkatkan keupayaan pengurusan dan perkongsian fail anda.

iTunes tidak diperlukan!

## Soalan Lazim

{{% details title="Adakah saya memerlukan iTunes untuk memindahkan fail ke iPhone saya?" closed="true" %}}
Tidak. Wi-Fi Drive memindahkan fail terus melalui rangkaian Wi-Fi tempatan anda. iTunes tidak diperlukan.
{{% /details %}}

{{% details title="Aplikasi mana yang menyokong Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive tersedia dalam Evermusic, Flacbox, Evertag, dan Evervideo untuk iOS.
{{% /details %}}

{{% details title="Adakah kedua-dua peranti perlu berada dalam rangkaian Wi-Fi yang sama?" closed="true" %}}
Ya. Komputer dan iPhone atau iPad anda mesti disambungkan ke rangkaian Wi-Fi tempatan yang sama untuk Wi-Fi Drive berfungsi.
{{% /details %}}

{{% details title="Bolehkah saya memindahkan keseluruhan folder, bukan hanya fail individu?" closed="true" %}}
Ya. Wi-Fi Drive menyokong memuat naik dan memuat turun keseluruhan folder melalui antara muka pelayar web.
{{% /details %}}

{{% details title="Adakah Wi-Fi Drive berfungsi dengan Windows?" closed="true" %}}
Ya. Anda boleh menggunakan mana-mana pelayar web pada Windows atau menyambung melalui Windows File Explorer menggunakan protokol WebDAV.
{{% /details %}}

{{% details title="Bolehkah saya menggunakan kabel USB untuk mempercepatkan pemindahan?" closed="true" %}}
Ya. Jika iPhone anda disambungkan ke Mac anda melalui USB semasa Wi-Fi Drive berjalan, pemindahan akan menggunakan sambungan kabel untuk kelajuan yang lebih pantas.
{{% /details %}}

**In breve:** Wi-Fi Drive ti permette di trasferire file da qualsiasi computer al tuo iPhone o iPad tramite Wi-Fi -- senza iTunes o cavi. Usa un browser web, Mac Finder o Windows File Explorer. Entrambi i dispositivi devono essere sulla stessa rete Wi-Fi.

Wi-Fi Drive è il modo più semplice per trasferire i tuoi file dal computer a un iPhone o iPad senza iTunes. Puoi facilmente scaricare o caricare più file o persino cartelle utilizzando una connessione Wi-Fi locale. Entrambi i dispositivi devono essere connessi alla stessa rete Wi-Fi.

Puoi trasferire file in modalità wireless utilizzando un browser web desktop o un client WebDAV come Mac Finder o Windows File Explorer.

## Trasferire file dal computer in modalità wireless con un browser web desktop

1. Apri una delle nostre app come [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Avvia l'app > vai su "Connessioni" > "Computer" > "Connetti tramite Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Schermata di connessione Wi-Fi Drive" width="400" >}}

1. Inserisci nome utente e password per rendere la connessione più sicura, ma questo passaggio è opzionale.
2. Tocca "Avvia Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Schermata Wi-Fi Drive avviato" width="400" >}}

5. Copia l'URL del browser web.
6. Apri un browser web sul tuo desktop (browser supportati: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Inserisci l'URL dal passaggio 5 nel campo indirizzo del tuo browser desktop.

![Schermata browser Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Dopo aver stabilito la connessione, vedrai l'app web del gestore file. Tutte le operazioni comuni sono supportate. Puoi copiare, spostare, eliminare, caricare e scaricare i tuoi file dal computer allo storage locale del tuo dispositivo.<br><br>Usa il trascinamento o il pulsante Carica per trasferire musica sul tuo iPhone.<br><br>Inoltre, se il tuo dispositivo è collegato al Mac tramite cavo, questo può velocizzare i trasferimenti di file. In questo caso, il tuo Mac configurerà un server virtuale online, ma la connessione utilizzerà il cavo per velocità di trasferimento più elevate.

![Schermata caricamento file Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Tocca "Ferma Wi-Fi Drive" sull'iPhone quando hai finito.

*Nota: assicurati di utilizzare l'ultima versione del browser desktop e che JavaScript non sia disabilitato. Questo risolverà eventuali problemi se la connessione non può essere stabilita.*

## Trasferire file audio con Mac Finder o Windows File Explorer

**Se usi macOS:**

1. Fai clic destro sull'icona del Finder e seleziona la voce di menu Connetti al server....

![Finder Connetti al server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Inserisci l'URL del server dall'app iOS (l'URL del server si trova sotto il testo "Inserisci questo URL nell'applicazione WebDAV").

![Finder Inserire indirizzo del server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Tocca il pulsante Connetti.
4. Dopo aver stabilito la connessione, vedrai il contenuto del tuo iPhone in Mac Finder.

![Finder connesso a Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Usa trascinamento, sposta e rinomina per gestire i file sul tuo dispositivo.

**Se usi Windows:**

1. Apri File Explorer e fai clic destro sulla voce di menu "Questo PC" nella barra laterale sinistra.

![Windows Connessione al server Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Seleziona "Connetti unità di rete" nel menu a discesa.
3. Nel campo di testo "Cartella" inserisci l'URL del server dall'app iOS (l'URL del server si trova sotto il testo "Inserisci questo URL nell'applicazione WebDAV").
4. Seleziona la lettera dell'unità e fai clic sul pulsante "Fine".

![Windows Connessione al server Wi-Fi Drive Inserire nome disco](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Se la connessione ha successo, puoi gestire i file del tuo dispositivo iOS nella finestra successiva.

![Windows connesso al server Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusione

Con Wi-Fi Drive, puoi facilmente e in modalità wireless trasferire file tra il tuo computer e il dispositivo iOS, migliorando le tue capacità di gestione e condivisione dei file.

Non è necessario iTunes!

## Domande frequenti

{{% details title="Ho bisogno di iTunes per trasferire file sul mio iPhone?" closed="true" %}}
No. Wi-Fi Drive trasferisce i file direttamente tramite la tua rete Wi-Fi locale. iTunes non è necessario.
{{% /details %}}

{{% details title="Quali app supportano Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive è disponibile in Evermusic, Flacbox, Evertag ed Evervideo per iOS.
{{% /details %}}

{{% details title="Entrambi i dispositivi devono essere sulla stessa rete Wi-Fi?" closed="true" %}}
Sì. Il tuo computer e iPhone o iPad devono essere connessi alla stessa rete Wi-Fi locale affinché Wi-Fi Drive funzioni.
{{% /details %}}

{{% details title="Posso trasferire intere cartelle, non solo singoli file?" closed="true" %}}
Sì. Wi-Fi Drive supporta il caricamento e lo scaricamento di intere cartelle tramite l'interfaccia del browser web.
{{% /details %}}

{{% details title="Wi-Fi Drive funziona con Windows?" closed="true" %}}
Sì. Puoi usare qualsiasi browser web su Windows o connetterti tramite Windows File Explorer utilizzando il protocollo WebDAV.
{{% /details %}}

{{% details title="Posso usare un cavo USB per velocizzare il trasferimento?" closed="true" %}}
Sì. Se il tuo iPhone è collegato al Mac tramite USB mentre Wi-Fi Drive è in esecuzione, il trasferimento utilizzerà la connessione via cavo per velocità più elevate.
{{% /details %}}

**TL;DR:** Wi-Fi Drive vă permite să transferați fișiere de pe orice computer pe iPhone sau iPad prin Wi-Fi -- fără iTunes sau cabluri. Utilizați un browser web, Mac Finder sau Windows File Explorer. Ambele dispozitive trebuie să fie pe aceeași rețea Wi-Fi.

Wi-Fi Drive este cel mai simplu mod de a transfera fișierele de pe computer pe un iPhone sau iPad fără iTunes. Puteți descărca sau încărca cu ușurință mai multe fișiere sau chiar foldere folosind o conexiune Wi-Fi locală. Ambele dispozitive trebuie să fie conectate la aceeași rețea Wi-Fi.

Puteți transfera fișiere wireless folosind un browser web de desktop sau un client WebDAV precum Mac Finder sau Windows File Explorer.

## Transferați fișiere de pe computer wireless cu un browser web de desktop

1. Deschideți oricare dintre aplicațiile noastre precum [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Lansați aplicația > mergeți la "Conexiuni" > "Computer" > "Conectare prin Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Introduceți numele de utilizator și parola pentru a face conexiunea mai sigură, dar acest pas este opțional.
2. Atingeți "Pornire Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Copiați URL-ul browserului web.
6. Deschideți un browser web pe desktop (browsere acceptate: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Introduceți URL-ul de la pasul 5 în câmpul de adresă al browserului de desktop.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. După stabilirea conexiunii, veți vedea aplicația web de gestionare a fișierelor. Toate operațiunile comune sunt acceptate. Puteți Copia, Muta, Șterge, Încărca și Descărca fișierele de pe computer în stocarea locală a dispozitivului.<br><br>Utilizați glisare și plasare sau butonul Încărcare pentru a transfera muzica pe iPhone.<br><br>În plus, dacă dispozitivul este conectat la Mac printr-un cablu, acest lucru poate accelera transferurile de fișiere. În acest caz, Mac-ul va configura un server online virtual, dar conexiunea va utiliza cablul pentru viteze de transfer mai rapide.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Atingeți "Oprire Wi-Fi Drive" pe iPhone când ați terminat.

*Notă: asigurați-vă că utilizați cea mai recentă versiune a browserului de desktop și că JavaScript nu este dezactivat. Aceasta va rezolva orice probleme dacă conexiunea nu poate fi stabilită.*

## Transferați fișiere audio cu Mac Finder sau Windows File Explorer

**Dacă utilizați macOS:**

1. Faceți clic dreapta pe pictograma Finder și selectați elementul de meniu Conectare la server....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Introduceți URL-ul serverului din aplicația iOS (URL-ul serverului se află sub textul "Introduceți acest URL în aplicația WebDAV").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Atingeți butonul Conectare.
4. După stabilirea conexiunii, veți vedea conținutul iPhone-ului în Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Utilizați glisare și plasare, comenzile Mutare și Redenumire pentru a gestiona fișierele de pe dispozitiv.

**Dacă utilizați Windows:**

1. Deschideți Explorer și faceți clic dreapta pe elementul de meniu „Acest PC" din bara laterală stângă.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Selectați "Mapare unitate de rețea" din meniul derulant.
3. În câmpul de text "Folder" introduceți URL-ul serverului din aplicația iOS (URL-ul serverului se află sub textul "Introduceți acest URL în aplicația WebDAV").
4. Selectați litera unității și faceți clic pe butonul "Finalizare".

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Dacă conexiunea este reușită, puteți gestiona fișierele aflate pe dispozitivul iOS în fereastra următoare.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Concluzie

Cu Wi-Fi Drive, puteți transfera ușor și wireless fișiere între computer și dispozitivul iOS, îmbunătățindu-vă capacitățile de gestionare și partajare a fișierelor.

iTunes nu este necesar!

## Întrebări frecvente

{{% details title="Am nevoie de iTunes pentru a transfera fișiere pe iPhone?" closed="true" %}}
Nu. Wi-Fi Drive transferă fișierele direct prin rețeaua Wi-Fi locală. iTunes nu este necesar.
{{% /details %}}

{{% details title="Ce aplicații acceptă Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive este disponibil în Evermusic, Flacbox, Evertag și Evervideo pentru iOS.
{{% /details %}}

{{% details title="Trebuie ambele dispozitive să fie pe aceeași rețea Wi-Fi?" closed="true" %}}
Da. Computerul și iPhone-ul sau iPad-ul trebuie să fie conectate la aceeași rețea Wi-Fi locală pentru ca Wi-Fi Drive să funcționeze.
{{% /details %}}

{{% details title="Pot transfera foldere întregi, nu doar fișiere individuale?" closed="true" %}}
Da. Wi-Fi Drive acceptă încărcarea și descărcarea folderelor întregi prin interfața browserului web.
{{% /details %}}

{{% details title="Funcționează Wi-Fi Drive cu Windows?" closed="true" %}}
Da. Puteți utiliza orice browser web pe Windows sau vă puteți conecta prin Windows File Explorer folosind protocolul WebDAV.
{{% /details %}}

{{% details title="Pot folosi un cablu USB pentru a accelera transferul?" closed="true" %}}
Da. Dacă iPhone-ul este conectat la Mac prin USB în timp ce Wi-Fi Drive rulează, transferul va utiliza conexiunea prin cablu pentru viteze mai rapide.
{{% /details %}}

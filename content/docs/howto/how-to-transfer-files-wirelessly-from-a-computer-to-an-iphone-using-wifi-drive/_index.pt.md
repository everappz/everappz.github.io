**TL;DR:** Wi-Fi Drive permite transferir arquivos de qualquer computador para seu iPhone ou iPad via Wi-Fi -- sem necessidade de iTunes ou cabos. Use um navegador web, Mac Finder ou Windows File Explorer. Ambos os dispositivos devem estar na mesma rede Wi-Fi.

Wi-Fi Drive é a maneira mais fácil de transferir seus arquivos do computador para um iPhone ou iPad sem iTunes. Você pode facilmente baixar ou enviar vários arquivos ou até pastas usando uma conexão Wi-Fi local. Ambos os dispositivos devem estar conectados à mesma rede Wi-Fi.

Você pode transferir arquivos sem fio usando um navegador web de desktop ou um cliente WebDAV como Mac Finder ou Windows File Explorer.

## Transferir arquivos do computador sem fio com um navegador web de desktop

1. Abra qualquer um dos nossos aplicativos como [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Inicie o aplicativo > vá para "Conexões" > "Computador" > "Conectar usando Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. Digite nome de usuário e senha para tornar a conexão mais segura, mas esta etapa é opcional.
2. Toque em "Iniciar Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. Copie a URL do navegador web.
6. Abra um navegador web no seu desktop (navegadores suportados: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Digite a URL do passo 5 no campo de endereço do seu navegador de desktop.

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Após a conexão ser estabelecida, você verá o aplicativo web gerenciador de arquivos. Todas as operações comuns são suportadas. Você pode Copiar, Mover, Excluir, Enviar e Baixar seus arquivos do computador para o armazenamento local do seu dispositivo.<br><br>Use arrastar e soltar ou o botão Enviar para transferir músicas para o seu iPhone.<br><br>Além disso, se o seu dispositivo estiver conectado ao Mac usando um cabo, isso pode acelerar as transferências de arquivos. Neste caso, o Mac configurará um servidor online virtual, mas a conexão utilizará o cabo para velocidades de transferência mais rápidas.

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Toque em "Parar Wi-Fi Drive" no iPhone quando terminar.

*Nota: certifique-se de que está usando a versão mais recente do navegador de desktop e que o JavaScript não está desativado. Isso resolverá quaisquer problemas se a conexão não puder ser estabelecida.*

## Transferir arquivos de áudio com Mac Finder ou Windows File Explorer

**Se você usa macOS:**

1. Clique com o botão direito no ícone do Finder e selecione o item de menu Conectar ao Servidor....

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Digite a URL do servidor do aplicativo iOS (a URL do servidor está localizada abaixo do texto "Digite esta URL no aplicativo WebDAV").

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Toque no botão Conectar.
4. Após a conexão ser estabelecida, você verá o conteúdo do seu iPhone no Mac Finder.

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Use arrastar e soltar, comandos Mover e Renomear para gerenciar arquivos no seu dispositivo.

**Se você usa Windows:**

1. Abra o Explorador de Arquivos e clique com o botão direito no item de menu 'Este Computador' na barra lateral esquerda.

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Selecione "Mapear Unidade de Rede" no menu suspenso.
3. No campo de texto "Pasta" digite a URL do servidor do aplicativo iOS (a URL do servidor está localizada abaixo do texto "Digite esta URL no aplicativo WebDAV").
4. Selecione a letra da unidade e clique no botão "Concluir".

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Se a conexão for bem-sucedida, você pode gerenciar arquivos localizados no seu dispositivo iOS na próxima janela.

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusão

Com Wi-Fi Drive, você pode transferir arquivos entre seu computador e dispositivo iOS de forma fácil e sem fio, aprimorando suas capacidades de gerenciamento e compartilhamento de arquivos.

iTunes não é necessário!

## Perguntas Frequentes

{{% details title="Preciso do iTunes para transferir arquivos para meu iPhone?" closed="true" %}}
Não. Wi-Fi Drive transfere arquivos diretamente pela sua rede Wi-Fi local. iTunes não é necessário.
{{% /details %}}

{{% details title="Quais aplicativos suportam Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive está disponível no Evermusic, Flacbox, Evertag e Evervideo para iOS.
{{% /details %}}

{{% details title="Ambos os dispositivos precisam estar na mesma rede Wi-Fi?" closed="true" %}}
Sim. Seu computador e iPhone ou iPad devem estar conectados à mesma rede Wi-Fi local para que o Wi-Fi Drive funcione.
{{% /details %}}

{{% details title="Posso transferir pastas inteiras, não apenas arquivos individuais?" closed="true" %}}
Sim. Wi-Fi Drive suporta o envio e download de pastas inteiras através da interface do navegador web.
{{% /details %}}

{{% details title="Wi-Fi Drive funciona com Windows?" closed="true" %}}
Sim. Você pode usar qualquer navegador web no Windows ou conectar via Windows File Explorer usando o protocolo WebDAV.
{{% /details %}}

{{% details title="Posso usar um cabo USB para acelerar a transferência?" closed="true" %}}
Sim. Se o seu iPhone estiver conectado ao Mac via USB enquanto o Wi-Fi Drive estiver em execução, a transferência usará a conexão por cabo para velocidades mais rápidas.
{{% /details %}}

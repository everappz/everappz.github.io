**Shrnutí:** Wi-Fi Drive umožňuje přenášet soubory z jakéhokoli počítače do iPhonu nebo iPadu přes Wi-Fi -- bez iTunes nebo kabelů. Použijte webový prohlížeč, Mac Finder nebo Windows File Explorer. Obě zařízení musí být připojena ke stejné Wi-Fi síti.

Wi-Fi Drive je nejjednodušší způsob, jak přenést soubory z počítače do iPhonu nebo iPadu bez iTunes. Můžete snadno stahovat nebo nahrávat více souborů nebo dokonce složek pomocí lokálního Wi-Fi připojení. Obě zařízení by měla být připojena ke stejné Wi-Fi síti.

Soubory můžete bezdrátově přenášet pomocí desktopového webového prohlížeče nebo WebDAV klienta, jako je Mac Finder nebo Windows File Explorer.

Můžete se podívat na video tutoriál od [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) nebo si přečíst textovou verzi níže.

## Přenos souborů z počítače bezdrátově pomocí desktopového webového prohlížeče

1. Otevřete některou z našich aplikací, například [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Spusťte aplikaci > přejděte na "Připojení" > "Počítač" > "Připojit přes Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Obrazovka připojení Wi-Fi Drive" width="400" >}}

1. Zadejte uživatelské jméno a heslo pro bezpečnější připojení, ale tento krok je volitelný.
2. Klepněte na "Spustit Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Obrazovka spuštěného Wi-Fi Drive" width="400" >}}

5. Zkopírujte URL adresu webového prohlížeče.
6. Otevřete webový prohlížeč na svém počítači (podporované prohlížeče: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Zadejte URL z kroku 5 do adresního pole desktopového prohlížeče.

![Obrazovka prohlížeče Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Po navázání připojení uvidíte webovou aplikaci správce souborů. Všechny běžné operace jsou podporovány. Můžete kopírovat, přesouvat, mazat, nahrávat a stahovat soubory z počítače do lokálního úložiště vašeho zařízení.<br><br>Použijte přetažení nebo tlačítko nahrávání pro přenos hudby do iPhonu.<br><br>Navíc, pokud je vaše zařízení připojeno k Macu pomocí kabelu, může to urychlit přenos souborů. V tomto případě Mac nastaví virtuální online server, ale připojení využije kabel pro rychlejší přenosové rychlosti.

![Obrazovka nahrávání souborů Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Po dokončení klepněte na iPhonu na "Zastavit Wi-Fi Drive".

*Poznámka: ujistěte se, že používáte nejnovější verzi desktopového prohlížeče a že JavaScript není zakázán. To vyřeší případné problémy, pokud nelze navázat připojení.*

## Přenos zvukových souborů pomocí Mac Finder nebo Windows File Explorer

**Pokud používáte macOS:**

1. Klikněte pravým tlačítkem na ikonu Finder a vyberte položku nabídky Připojit k serveru....

![Finder Připojit k serveru...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Zadejte URL serveru z iOS aplikace (URL serveru se nachází pod textem "Zadejte tuto URL v aplikaci WebDAV").

![Finder Zadání adresy serveru](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Klepněte na tlačítko Připojit.
4. Po navázání připojení uvidíte obsah iPhonu v Mac Finder.

![Finder připojený k Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Použijte přetažení, přesun a přejmenování pro správu souborů na vašem zařízení.

**Pokud používáte Windows:**

1. Otevřete File Explorer a klikněte pravým tlačítkem na položku nabídky "Tento počítač" v levém postranním panelu.

![Windows Připojení k serveru Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. V rozbalovací nabídce vyberte "Připojit síťovou jednotku".
3. Do textového pole "Složka" zadejte URL serveru z iOS aplikace (URL serveru se nachází pod textem "Zadejte tuto URL v aplikaci WebDAV").
4. Vyberte písmeno jednotky a klikněte na tlačítko "Dokončit".

![Windows Připojení k serveru Wi-Fi Drive Zadání názvu disku](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Pokud je připojení úspěšné, můžete v dalším okně spravovat soubory na vašem iOS zařízení.

![Windows připojeno k serveru Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Závěr

S Wi-Fi Drive můžete snadno a bezdrátově přenášet soubory mezi počítačem a iOS zařízením, čímž vylepšíte své možnosti správy a sdílení souborů.

iTunes není potřeba!

## Často kladené dotazy

{{% details title="Potřebuji iTunes k přenosu souborů do iPhonu?" closed="true" %}}
Ne. Wi-Fi Drive přenáší soubory přímo přes vaši lokální Wi-Fi síť. iTunes není potřeba.
{{% /details %}}

{{% details title="Které aplikace podporují Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive je k dispozici v Evermusic, Flacbox, Evertag a Evervideo pro iOS.
{{% /details %}}

{{% details title="Musí být obě zařízení na stejné Wi-Fi síti?" closed="true" %}}
Ano. Váš počítač a iPhone nebo iPad musí být připojeny ke stejné lokální Wi-Fi síti, aby Wi-Fi Drive fungoval.
{{% /details %}}

{{% details title="Mohu přenášet celé složky, nejen jednotlivé soubory?" closed="true" %}}
Ano. Wi-Fi Drive podporuje nahrávání a stahování celých složek prostřednictvím rozhraní webového prohlížeče.
{{% /details %}}

{{% details title="Funguje Wi-Fi Drive s Windows?" closed="true" %}}
Ano. Můžete použít jakýkoli webový prohlížeč ve Windows nebo se připojit přes Windows File Explorer pomocí protokolu WebDAV.
{{% /details %}}

{{% details title="Mohu použít USB kabel pro urychlení přenosu?" closed="true" %}}
Ano. Pokud je váš iPhone připojen k Macu přes USB, zatímco Wi-Fi Drive běží, přenos využije kabelové připojení pro rychlejší přenosové rychlosti.
{{% /details %}}

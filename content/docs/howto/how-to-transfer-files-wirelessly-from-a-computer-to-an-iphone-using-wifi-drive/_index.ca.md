**Resum:** Wi-Fi Drive us permet transferir fitxers des de qualsevol ordinador al vostre iPhone o iPad a través de Wi-Fi -- sense necessitat d'iTunes ni cables. Utilitzeu un navegador web, Mac Finder o Windows File Explorer. Ambdós dispositius han d'estar connectats a la mateixa xarxa Wi-Fi.

Wi-Fi Drive és la manera més fàcil de transferir els vostres fitxers de l'ordinador a un iPhone o iPad sense iTunes. Podeu descarregar o pujar fàcilment múltiples fitxers o fins i tot carpetes mitjançant una connexió Wi-Fi local. Ambdós dispositius han d'estar connectats a la mateixa xarxa Wi-Fi.

Podeu transferir fitxers sense fil utilitzant un navegador web d'escriptori o un client WebDAV com Mac Finder o Windows File Explorer.

Podeu veure un vídeo tutorial de [**TECHGUYPH**](https://www.youtube.com/channel/UCgpf09gGFE_c_3pPTtTpnzg) o llegir la versió de text a continuació.

## Transferir fitxers de l'ordinador sense fil amb un navegador web d'escriptori

1. Obriu qualsevol de les nostres aplicacions com [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Inicieu l'aplicació > aneu a "Connexions" > "Ordinador" > "Connectar mitjançant Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Pantalla de connexió Wi-Fi Drive" width="400" >}}

1. Introduïu el nom d'usuari i la contrasenya per fer la connexió més segura, però aquest pas és opcional.
2. Toqueu "Iniciar Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Pantalla Wi-Fi Drive iniciat" width="400" >}}

5. Copieu l'URL del navegador web.
6. Obriu un navegador web al vostre escriptori (navegadors compatibles: Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Introduïu l'URL del pas 5 al camp d'adreça del navegador d'escriptori.

![Pantalla del navegador Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Un cop establerta la connexió, veureu l'aplicació web de gestió de fitxers. Totes les operacions comunes estan suportades. Podeu copiar, moure, eliminar, pujar i descarregar els vostres fitxers de l'ordinador a l'emmagatzematge local del vostre dispositiu.<br><br>Utilitzeu arrossegar i deixar anar o el botó de pujar per transferir música al vostre iPhone.<br><br>A més, si el vostre dispositiu està connectat al Mac mitjançant un cable, això pot accelerar les transferències de fitxers. En aquest cas, el Mac configurarà un servidor virtual en línia, però la connexió utilitzarà el cable per a velocitats de transferència més ràpides.

![Pantalla de pujar fitxers Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Toqueu "Aturar Wi-Fi Drive" a l'iPhone quan hàgiu acabat.

*Nota: assegureu-vos que utilitzeu l'última versió del navegador d'escriptori i que JavaScript no estigui desactivat. Això resoldrà qualsevol problema si no es pot establir la connexió.*

## Transferir fitxers d'àudio amb Mac Finder o Windows File Explorer

**Si utilitzeu macOS:**

1. Feu clic dret a la icona del Finder i seleccioneu l'element de menú Connectar al servidor....

![Finder Connectar al servidor...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Introduïu l'URL del servidor des de l'aplicació iOS (l'URL del servidor es troba a sota del text "Introduïu aquest URL a l'aplicació WebDAV").

![Finder Introduir adreça del servidor](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Toqueu el botó Connectar.
4. Un cop establerta la connexió, veureu el contingut del vostre iPhone al Mac Finder.

![Finder connectat a Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Utilitzeu arrossegar i deixar anar, moure i reanomenar per gestionar els fitxers del vostre dispositiu.

**Si utilitzeu Windows:**

1. Obriu File Explorer i feu clic dret a l'element de menú "Aquest ordinador" a la barra lateral esquerra.

![Windows Connectar al servidor Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Seleccioneu "Assignar unitat de xarxa" al menú desplegable.
3. Al camp de text "Carpeta" introduïu l'URL del servidor des de l'aplicació iOS (l'URL del servidor es troba a sota del text "Introduïu aquest URL a l'aplicació WebDAV").
4. Seleccioneu la lletra de la unitat i feu clic al botó "Finalitzar".

![Windows Connectar al servidor Wi-Fi Drive Introduir nom del disc](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Si la connexió és correcta, podeu gestionar els fitxers del vostre dispositiu iOS a la finestra següent.

![Windows connectat al servidor Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusió

Amb Wi-Fi Drive, podeu transferir fitxers fàcilment i sense fil entre el vostre ordinador i el dispositiu iOS, millorant les vostres capacitats de gestió i compartició de fitxers.

No cal iTunes!

## Preguntes freqüents

{{% details title="Necessito iTunes per transferir fitxers al meu iPhone?" closed="true" %}}
No. Wi-Fi Drive transfereix fitxers directament a través de la vostra xarxa Wi-Fi local. No cal iTunes.
{{% /details %}}

{{% details title="Quines aplicacions admeten Wi-Fi Drive?" closed="true" %}}
Wi-Fi Drive està disponible a Evermusic, Flacbox, Evertag i Evervideo per a iOS.
{{% /details %}}

{{% details title="Ambdós dispositius han d'estar a la mateixa xarxa Wi-Fi?" closed="true" %}}
Sí. El vostre ordinador i iPhone o iPad han d'estar connectats a la mateixa xarxa Wi-Fi local perquè Wi-Fi Drive funcioni.
{{% /details %}}

{{% details title="Puc transferir carpetes senceres, no només fitxers individuals?" closed="true" %}}
Sí. Wi-Fi Drive admet la pujada i descàrrega de carpetes senceres a través de la interfície del navegador web.
{{% /details %}}

{{% details title="Wi-Fi Drive funciona amb Windows?" closed="true" %}}
Sí. Podeu utilitzar qualsevol navegador web a Windows o connectar-vos mitjançant Windows File Explorer utilitzant el protocol WebDAV.
{{% /details %}}

{{% details title="Puc utilitzar un cable USB per accelerar la transferència?" closed="true" %}}
Sí. Si el vostre iPhone està connectat al Mac via USB mentre Wi-Fi Drive està en funcionament, la transferència utilitzarà la connexió per cable per a velocitats més ràpides.
{{% /details %}}

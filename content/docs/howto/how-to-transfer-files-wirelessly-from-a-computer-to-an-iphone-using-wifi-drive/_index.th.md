**สรุป:** Wi-Fi Drive ช่วยให้คุณถ่ายโอนไฟล์จากคอมพิวเตอร์เครื่องใดก็ได้ไปยัง iPhone หรือ iPad ผ่าน Wi-Fi -- ไม่ต้องใช้ iTunes หรือสาย ใช้เว็บเบราว์เซอร์, Mac Finder หรือ Windows File Explorer อุปกรณ์ทั้งสองต้องอยู่ในเครือข่าย Wi-Fi เดียวกัน

Wi-Fi Drive เป็นวิธีที่ง่ายที่สุดในการถ่ายโอนไฟล์จากคอมพิวเตอร์ไปยัง iPhone หรือ iPad โดยไม่ต้องใช้ iTunes คุณสามารถดาวน์โหลดหรืออัปโหลดไฟล์หลายไฟล์หรือแม้แต่โฟลเดอร์ได้อย่างง่ายดายโดยใช้การเชื่อมต่อ Wi-Fi ในเครื่อง อุปกรณ์ทั้งสองควรเชื่อมต่อกับเครือข่าย Wi-Fi เดียวกัน

คุณสามารถถ่ายโอนไฟล์แบบไร้สายโดยใช้เว็บเบราว์เซอร์บนเดสก์ท็อปหรือไคลเอนต์ WebDAV เช่น Mac Finder หรือ Windows File Explorer

## ถ่ายโอนไฟล์จากคอมพิวเตอร์แบบไร้สายด้วยเว็บเบราว์เซอร์บนเดสก์ท็อป

1. เปิดแอปใดก็ได้ของเรา เช่น [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo)

2. เปิดแอป > ไปที่ "การเชื่อมต่อ" > "คอมพิวเตอร์" > "เชื่อมต่อโดยใช้ Wi-Fi"
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. ป้อนชื่อผู้ใช้และรหัสผ่านเพื่อทำให้การเชื่อมต่อปลอดภัยมากขึ้น แต่ขั้นตอนนี้เป็นทางเลือก
2. แตะ "เริ่ม Wi-Fi Drive"

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. คัดลอก URL ของเว็บเบราว์เซอร์
6. เปิดเว็บเบราว์เซอร์บนเดสก์ท็อปของคุณ (เบราว์เซอร์ที่รองรับ: Safari, Google Chrome, Opera, Yandex Browser, FireFox)
7. ป้อน URL จากขั้นตอนที่ 5 ในช่องที่อยู่ของเบราว์เซอร์เดสก์ท็อป

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. หลังจากสร้างการเชื่อมต่อแล้ว คุณจะเห็นเว็บแอปจัดการไฟล์ รองรับการดำเนินการทั่วไปทั้งหมด คุณสามารถคัดลอก ย้าย ลบ อัปโหลด และดาวน์โหลดไฟล์จากคอมพิวเตอร์ไปยังที่เก็บข้อมูลภายในของอุปกรณ์<br><br>ใช้การลากและวางหรือปุ่มอัปโหลดเพื่อถ่ายโอนเพลงไปยัง iPhone ของคุณ<br><br>นอกจากนี้ หากอุปกรณ์ของคุณเชื่อมต่อกับ Mac โดยใช้สาย สามารถเพิ่มความเร็วในการถ่ายโอนไฟล์ได้ ในกรณีนี้ Mac ของคุณจะตั้งค่าเซิร์ฟเวอร์ออนไลน์เสมือน แต่การเชื่อมต่อจะใช้สายเพื่อความเร็วในการถ่ายโอนที่เร็วขึ้น

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. แตะ "หยุด Wi-Fi Drive" บน iPhone เมื่อเสร็จสิ้น

*หมายเหตุ: โปรดตรวจสอบให้แน่ใจว่าคุณใช้เวอร์ชันล่าสุดของเบราว์เซอร์เดสก์ท็อปและ JavaScript ไม่ได้ถูกปิดใช้งาน สิ่งนี้จะแก้ไขปัญหาใดๆ หากไม่สามารถสร้างการเชื่อมต่อได้*

## ถ่ายโอนไฟล์เสียงด้วย Mac Finder หรือ Windows File Explorer

**หากคุณใช้ macOS:**

1. คลิกขวาที่ไอคอน Finder และเลือกรายการเมนู เชื่อมต่อกับเซิร์ฟเวอร์...

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. ป้อน URL เซิร์ฟเวอร์จากแอป iOS (URL เซิร์ฟเวอร์อยู่ใต้ข้อความ "ป้อน URL นี้ในแอปพลิเคชัน WebDAV")

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. แตะปุ่มเชื่อมต่อ
4. หลังจากสร้างการเชื่อมต่อแล้ว คุณจะเห็นเนื้อหาของ iPhone ใน Mac Finder

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. ใช้การลากและวาง คำสั่งย้าย และเปลี่ยนชื่อเพื่อจัดการไฟล์บนอุปกรณ์ของคุณ

**หากคุณใช้ Windows:**

1. เปิด File Explorer และคลิกขวาที่รายการเมนู 'พีซีเครื่องนี้' บนแถบด้านข้างซ้าย

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. เลือก "แมปไดรฟ์เครือข่าย" จากเมนูดรอปดาวน์
3. ในช่องข้อความ "โฟลเดอร์" ให้ป้อน URL เซิร์ฟเวอร์จากแอป iOS (URL เซิร์ฟเวอร์อยู่ใต้ข้อความ "ป้อน URL นี้ในแอปพลิเคชัน WebDAV")
4. เลือกตัวอักษรไดรฟ์และคลิกปุ่ม "เสร็จสิ้น"

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. หากการเชื่อมต่อสำเร็จ คุณสามารถจัดการไฟล์ที่อยู่บนอุปกรณ์ iOS ในหน้าต่างถัดไป

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## สรุป

ด้วย Wi-Fi Drive คุณสามารถถ่ายโอนไฟล์ระหว่างคอมพิวเตอร์และอุปกรณ์ iOS ได้อย่างง่ายดายและไร้สาย เพิ่มความสามารถในการจัดการและแชร์ไฟล์

ไม่ต้องใช้ iTunes!

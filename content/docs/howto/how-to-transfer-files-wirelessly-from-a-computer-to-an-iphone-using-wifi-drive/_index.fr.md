**En bref :** Wi-Fi Drive vous permet de transférer des fichiers depuis n'importe quel ordinateur vers votre iPhone ou iPad via Wi-Fi -- sans iTunes ni câbles. Utilisez un navigateur web, Mac Finder ou Windows File Explorer. Les deux appareils doivent être sur le même réseau Wi-Fi.

Wi-Fi Drive est le moyen le plus simple de transférer vos fichiers de l'ordinateur vers un iPhone ou iPad sans iTunes. Vous pouvez facilement télécharger ou envoyer plusieurs fichiers ou même des dossiers en utilisant une connexion Wi-Fi locale. Les deux appareils doivent être connectés au même réseau Wi-Fi.

Vous pouvez transférer des fichiers sans fil en utilisant un navigateur web de bureau ou un client WebDAV comme Mac Finder ou Windows File Explorer.

## Transférer des fichiers de l'ordinateur sans fil avec un navigateur web de bureau

1. Ouvrez l'une de nos applications comme [Evermusic](/products/evermusic), [Flacbox](/products/flacbox), [Evertag](/products/evertag), [Evervideo](/products/evervideo).

2. Lancez l'application > allez dans "Connexions" > "Ordinateur" > "Se connecter via Wi-Fi".
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Écran de connexion Wi-Fi Drive" width="400" >}}

1. Entrez un nom d'utilisateur et un mot de passe pour rendre la connexion plus sécurisée, mais cette étape est facultative.
2. Appuyez sur "Démarrer Wi-Fi Drive".

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Écran Wi-Fi Drive démarré" width="400" >}}

5. Copiez l'URL du navigateur web.
6. Ouvrez un navigateur web sur votre bureau (navigateurs pris en charge : Safari, Google Chrome, Opera, Yandex Browser, FireFox).
7. Entrez l'URL de l'étape 5 dans le champ d'adresse de votre navigateur de bureau.

![Écran du navigateur Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. Une fois la connexion établie, vous verrez l'application web de gestion de fichiers. Toutes les opérations courantes sont prises en charge. Vous pouvez copier, déplacer, supprimer, envoyer et télécharger vos fichiers de l'ordinateur vers le stockage local de votre appareil.<br><br>Utilisez le glisser-déposer ou le bouton d'envoi pour transférer de la musique vers votre iPhone.<br><br>De plus, si votre appareil est connecté à votre Mac par un câble, cela peut accélérer les transferts de fichiers. Dans ce cas, votre Mac configurera un serveur virtuel en ligne, mais la connexion utilisera le câble pour des vitesses de transfert plus rapides.

![Écran d'envoi de fichiers Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. Appuyez sur "Arrêter Wi-Fi Drive" sur l'iPhone lorsque vous avez terminé.

*Remarque : assurez-vous d'utiliser la dernière version du navigateur de bureau et que JavaScript n'est pas désactivé. Cela résoudra tout problème si la connexion ne peut pas être établie.*

## Transférer des fichiers audio avec Mac Finder ou Windows File Explorer

**Si vous utilisez macOS :**

1. Faites un clic droit sur l'icône du Finder et sélectionnez l'élément de menu Se connecter au serveur....

![Finder Se connecter au serveur...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. Entrez l'URL du serveur depuis l'application iOS (l'URL du serveur se trouve sous le texte "Entrez cette URL dans l'application WebDAV").

![Finder Entrer l'adresse du serveur](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. Appuyez sur le bouton Se connecter.
4. Une fois la connexion établie, vous verrez le contenu de votre iPhone dans Mac Finder.

![Finder connecté à Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. Utilisez le glisser-déposer, déplacer et renommer pour gérer les fichiers sur votre appareil.

**Si vous utilisez Windows :**

1. Ouvrez File Explorer et faites un clic droit sur l'élément de menu "Ce PC" dans la barre latérale gauche.

![Windows Se connecter au serveur Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. Sélectionnez "Connecter un lecteur réseau" dans le menu déroulant.
3. Dans le champ de texte "Dossier", entrez l'URL du serveur depuis l'application iOS (l'URL du serveur se trouve sous le texte "Entrez cette URL dans l'application WebDAV").
4. Sélectionnez la lettre du lecteur et cliquez sur le bouton "Terminer".

![Windows Se connecter au serveur Wi-Fi Drive Entrer le nom du disque](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. Si la connexion est réussie, vous pouvez gérer les fichiers de votre appareil iOS dans la fenêtre suivante.

![Windows connecté au serveur Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## Conclusion

Avec Wi-Fi Drive, vous pouvez facilement et sans fil transférer des fichiers entre votre ordinateur et votre appareil iOS, améliorant vos capacités de gestion et de partage de fichiers.

iTunes n'est pas nécessaire !

## Questions fréquemment posées

{{% details title="Ai-je besoin d'iTunes pour transférer des fichiers vers mon iPhone ?" closed="true" %}}
Non. Wi-Fi Drive transfère les fichiers directement via votre réseau Wi-Fi local. iTunes n'est pas nécessaire.
{{% /details %}}

{{% details title="Quelles applications prennent en charge Wi-Fi Drive ?" closed="true" %}}
Wi-Fi Drive est disponible dans Evermusic, Flacbox, Evertag et Evervideo pour iOS.
{{% /details %}}

{{% details title="Les deux appareils doivent-ils être sur le même réseau Wi-Fi ?" closed="true" %}}
Oui. Votre ordinateur et votre iPhone ou iPad doivent être connectés au même réseau Wi-Fi local pour que Wi-Fi Drive fonctionne.
{{% /details %}}

{{% details title="Puis-je transférer des dossiers entiers, pas seulement des fichiers individuels ?" closed="true" %}}
Oui. Wi-Fi Drive prend en charge l'envoi et le téléchargement de dossiers entiers via l'interface du navigateur web.
{{% /details %}}

{{% details title="Wi-Fi Drive fonctionne-t-il avec Windows ?" closed="true" %}}
Oui. Vous pouvez utiliser n'importe quel navigateur web sur Windows ou vous connecter via Windows File Explorer en utilisant le protocole WebDAV.
{{% /details %}}

{{% details title="Puis-je utiliser un câble USB pour accélérer le transfert ?" closed="true" %}}
Oui. Si votre iPhone est connecté à votre Mac via USB pendant que Wi-Fi Drive fonctionne, le transfert utilisera la connexion par câble pour des vitesses plus rapides.
{{% /details %}}

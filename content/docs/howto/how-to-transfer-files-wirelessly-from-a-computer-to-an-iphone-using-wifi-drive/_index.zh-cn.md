**简介：** Wi-Fi Drive让您可以通过Wi-Fi从任何电脑将文件传输到iPhone或iPad——无需iTunes或数据线。使用网页浏览器、Mac Finder或Windows File Explorer。两台设备必须在同一个Wi-Fi网络上。

Wi-Fi Drive是无需iTunes将文件从电脑传输到iPhone或iPad的最简单方式。您可以使用本地Wi-Fi连接轻松下载或上传多个文件甚至文件夹。两台设备应连接到同一个Wi-Fi网络。

您可以使用桌面网页浏览器或WebDAV客户端（如Mac Finder或Windows File Explorer）无线传输文件。

## 使用桌面网页浏览器从电脑无线传输文件

1. 打开我们的任何应用程序，如[Evermusic](/products/evermusic)、[Flacbox](/products/flacbox)、[Evertag](/products/evertag)、[Evervideo](/products/evervideo)。

2. 启动应用 > 进入"连接" > "电脑" > "使用Wi-Fi连接"。
   
{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_467016d0322c45e1bda9c3b5cfd90019~mv2.jpeg" alt="Wi-Fi Drive Connect Screen" width="400" >}}

1. 输入用户名和密码以使连接更安全，但此步骤是可选的。
2. 点击"启动Wi-Fi Drive"。

{{< figure src="/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_0327236a38b84bca802973aab99838e8~mv2.jpeg" alt="Wi-Fi Drive Started Screen" width="400" >}}

5. 复制网页浏览器URL。
6. 在电脑上打开网页浏览器（支持的浏览器：Safari、Google Chrome、Opera、Yandex Browser、FireFox）。
7. 将第5步的URL输入到电脑浏览器的地址栏中。

![Wi-Fi Drive Browser Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_7c7af1b6788a4fab9a6a866f1bbcc1b9~mv2.png)

8. 连接建立后，您将看到文件管理器网页应用。支持所有常见操作。您可以从电脑复制、移动、删除、上传和下载文件到设备的本地存储。<br><br>使用拖放或上传按钮将音乐传输到iPhone。<br><br>此外，如果您的设备通过数据线连接到Mac，这可以加快文件传输速度。在这种情况下，Mac将设置一个虚拟在线服务器，但连接将使用数据线以获得更快的传输速度。

![Wi-Fi Drive Upload Files Screen](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_8699b90283c540cda6052b2480f1b99c~mv2.png)

1. 完成后在iPhone上点击"停止Wi-Fi Drive"。

*注意：请确保您使用的是最新版本的桌面浏览器，并且JavaScript未被禁用。如果无法建立连接，这将解决任何问题。*

## 使用Mac Finder或Windows File Explorer传输音频文件

**如果您使用macOS：**

1. 右键点击Finder图标，选择"连接服务器..."菜单项。

![Finder Connect to Server...](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b8ba587ed13c44688127bfac0e733335~mv2.png)

2. 输入iOS应用中的服务器URL（服务器URL位于"在WebDAV应用程序中输入此URL"文字下方）。

![Finder Enter Server Address](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_a50e06395c734fb290a209518782d8c0~mv2.png)

3. 点击连接按钮。
4. 连接建立后，您将在Mac Finder中看到iPhone的内容。

![Finder Connected Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e388c17010904e7c9d124803d44799a9~mv2.png)

5. 使用拖放、移动和重命名命令管理设备上的文件。

**如果您使用Windows：**

1. 打开文件资源管理器，右键点击左侧边栏中的"此电脑"菜单项。

![Windows Connect to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_bb1eda9e49fa4f678f444299eaa20d94~mv2.png)

2. 在下拉菜单中选择"映射网络驱动器"。
3. 在"文件夹"文本字段中输入iOS应用的服务器URL（服务器URL位于"在WebDAV应用程序中输入此URL"文字下方）。
4. 选择驱动器号，点击"完成"按钮。

![Windows Connect to Wi-Fi Drive Server Enter Disk Name](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_e46eaf51c98e427abc4f1fad1969eee6~mv2.png)

5. 如果连接成功，您可以在下一个窗口中管理iOS设备上的文件。

![Windows Connected to Wi-Fi Drive Server](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/21260c_b413a15cf410434cbb53c8f3efa4d3cc~mv2.png)

## 结论

使用Wi-Fi Drive，您可以轻松地在电脑和iOS设备之间无线传输文件，增强文件管理和共享能力。

无需iTunes！

## 常见问题

{{% details title="我需要iTunes才能将文件传输到iPhone吗？" closed="true" %}}
不需要。Wi-Fi Drive通过本地Wi-Fi网络直接传输文件。不需要iTunes。
{{% /details %}}

{{% details title="哪些应用支持Wi-Fi Drive？" closed="true" %}}
Wi-Fi Drive在iOS版Evermusic、Flacbox、Evertag和Evervideo中可用。
{{% /details %}}

{{% details title="两台设备需要在同一个Wi-Fi网络上吗？" closed="true" %}}
是的。您的电脑和iPhone或iPad必须连接到同一个本地Wi-Fi网络，Wi-Fi Drive才能工作。
{{% /details %}}

{{% details title="我可以传输整个文件夹，而不仅仅是单个文件吗？" closed="true" %}}
可以。Wi-Fi Drive支持通过网页浏览器界面上传和下载整个文件夹。
{{% /details %}}

{{% details title="Wi-Fi Drive可以在Windows上使用吗？" closed="true" %}}
可以。您可以在Windows上使用任何网页浏览器，或通过WebDAV协议使用Windows File Explorer连接。
{{% /details %}}

{{% details title="我可以使用USB数据线加快传输速度吗？" closed="true" %}}
可以。如果在Wi-Fi Drive运行时iPhone通过USB连接到Mac，传输将使用有线连接以获得更快的速度。
{{% /details %}}

**要点：** Evermusicには6つのリアルタイムオーディオエフェクトが含まれています。**音量ノーマライズ、コンプレッサー、リバーブ、クロスフィード、ディレイ、ディストーション** です。プレーヤーの **⋯（その他）メニュー > オーディオエフェクト** から、または **設定 > オーディオプレーヤー > オーディオエフェクト** から開きます。エフェクトをタップし、スイッチを（右上で）**オン** にして、**プリセット** を選び、必要に応じて **詳細モード** を開いてスライダーを細かく調整します。各エフェクトは独立して動作し、再エンコードなしで、再生するすべて（ローカルファイル、クラウドストリーム、インターネットラジオ）にリアルタイムで適用されます。

## Evermusicのオーディオエフェクトとは？

オーディオエフェクトは、再生時に音の性格を変えます。Evermusicはこれらを再生エンジン内の**ネイティブなリアルタイム処理ノード**として実行するので、ファイルを変更したり再エンコードしたりすることなく、あらゆるソース（ローカルファイル、クラウドドライブ、メディアサーバー、インターネットラジオ）に適用されます。エフェクトをオフにすれば、元の音がすぐに戻ります。

6つのエフェクトがあり、それぞれが**独立**しています。単一のマスタースイッチはないので、1つ、複数、またはすべてを同時に動かすことができます。

1. **音量ノーマライズ** — すべてのトラックを一定のラウドネスに保ちます。
2. **コンプレッサー** — トラック内の大きい部分と小さい部分を均します。
3. **リバーブ** — 小さな部屋から大聖堂まで、空間の広がりを加えます。
4. **クロスフィード** — ヘッドフォンを本物のスピーカーのように聴こえさせます。
5. **ディレイ** — タイトなスラップから長いアンビエントの余韻まで、エコーを加えます。
6. **ディストーション** — 楽しみのために、ざらつきとローファイな性格を加えます。

音量ノーマライズとコンプレッサーは**一貫性と明瞭さ**に関するものです。リバーブ、ディレイ、ディストーションは**クリエイティブなエフェクト**です。クロスフィードは**ヘッドフォンの快適さ**のためのツールです。これらを合わせると、Evermusicはポケットの中の小さなスタジオになります。

## オーディオエフェクトの開き方

オーディオエフェクト画面に到達する方法は2つあります。

**プレーヤーから（最速）：**

1. **再生中／プレーヤー** 画面を開きます。
2. **⋯（その他）** ボタンをタップします。
3. **オーディオエフェクト** をタップします。

**設定から：**

1. **設定** タブに移動します。
2. **オーディオプレーヤー** をタップします。
3. **オーディオエフェクト** をタップします。

どちらの方法でも、**オーディオエフェクト** の一覧に到達します。ここには6つのエフェクトすべてが次の順序で表示されます：**音量ノーマライズ、コンプレッサー、リバーブ、クロスフィード、ディレイ、ディストーション**。いずれかをタップするとそのエディタが開きます。

## 各エフェクトエディタの動作

すべてのエフェクトエディタは同じシンプルなパターンに従うので、1つを覚えればすべてわかります。

- **有効化スイッチ（右上）。** エフェクトを **オン** または **オフ** にします。すべてのエフェクトはデフォルトでオフです。オフのとき、コントロールは薄暗く表示されます。
- **シンプル／詳細切り替え（右上）。** *シンプル*モードは、わかりやすい説明付きのプリセット一覧だけを表示します。ワンタップで良い音を得る最も簡単な方法です。*詳細*モードは、細かく調整するスライダーを追加します。
- **プリセット選択。** 縦向きではプリセットの「バブル」の列、横向きではプリセットの列が表示されます。出発点を選び、好みに応じて調整します。
- **スライダー（詳細モード）。** 各スライダーは現在の値を表示し、デフォルトに戻すための小さな**リセット**ボタン（円形の矢印）があります。いずれかのスライダーを調整すると、エフェクトは**手動**状態に切り替わるので、プリセットから離れたことが常にわかります。

変更は自動的に保存されます。以下は各エフェクトが何をするのか、そしてどう設定するのかです。

## 音量ノーマライズ

**機能：** 一部の曲は他の曲より大きくマスタリングされているため、何度も音量に手を伸ばすことになります。音量ノーマライズは各トラックの実際に知覚されるラウドネスを測定し、一定のターゲットに向けて緩やかに揃えるので、すべてがほぼ同じ音量で再生されます。放送グレードの **EBU R128** ラウドネス規格（ITU-R BS.1770）を使用し、あらゆるソースでリアルタイムに動作し、ReplayGainとは異なり、**ファイルにラウドネスタグを必要とせず**、オーディオを一切変更しません。

**プリセット：** ライト、標準、強、ナイト。

**詳細コントロール：**

- **ターゲットラウドネス** — すべてのトラックが揃えられるラウドネスで、LUFSで表示されます。高い（たとえば −14 LUFS）と全体的にすべてが大きく再生され、低い（−23 LUFS）と静かで落ち着いた音になります。
- **最大ブースト** — 静かなトラックをどれだけ増幅できるかをdBで制限します。値が高いほど、柔らかい録音をターゲットに近づけます。

**使い方：** オンにして、ストリーミング風のラウドネスには**標準**を、一定で静かな夜の遅い時間のリスニングには**ナイト**を選びます。古い録音と新しい録音が混在するシャッフルプレイリストに最適です。

## コンプレッサー

**機能：** 1曲の中で、静かな部分は柔らかすぎ、大きな部分は大きすぎることがあります。コンプレッサーはこれらを近づけるので、車の中、ランニング中、その他騒がしい場所でも曲全体が聴きやすくなります。これはAppleの `AUDynamicsProcessor` を基に構築された本格的なダイナミクスプロセッサです。

**プリセット：** トランスペアレント、ソフト、標準、ヘビー、ボイス／ポッドキャスト、古い録音、深夜、映画のセリフ、ストリーミング一致、最大ラウドネス。

**詳細コントロール（7つのスライダー）：**

- **スレッショルド** — コンプレッションが始まるレベル。低いほど強く押さえます。
- **ヘッドルーム** — ハードリミッティングが作動する前の、スレッショルドを超える余地。
- **エクスパンション比** — ごく静かな音（背景ノイズなど）をどれだけ強く引き下げるか。
- **エクスパンションスレッショルド** — そのゲーティングが始まるレベル。
- **アタック** — 突然の大きなピークにエフェクトがどれだけ速く反応するか。
- **リリース** — 大きな部分が過ぎた後、どれだけ速く解放するか。
- **マスターゲイン** — 処理後に適用される最終的なメイクアップブースト。

**使い方：** ほとんどのリスニングでは、オンにして**標準**を選びます。話し言葉のコンテンツには**ボイス／ポッドキャスト**または**映画のセリフ**を、静かなリスニングには**深夜**を、最も大きく最も均一な結果には**最大ラウドネス**を選びます。

## リバーブ

**機能：** 親密な部屋から壮大なホールや大聖堂まで、自然なエコーの余韻とともに、音楽が実際の空間で鳴っているように聴こえさせます。Appleの `AVAudioUnitReverb` を基に構築されています。

**プリセット（13種）：** スモールルーム、ミディアムルーム、ラージルーム、ミディアムホール、ラージホール、プレート、ミディアムチェンバー、ラージチェンバー、大聖堂、およびいくつかのホールとルームのバリエーション。

**詳細コントロール：**

- **ミックス** — リバーブをどれだけ混ぜるか。0%（ドライ、元の音）から100%（選んだ空間に完全に）まで。

**使い方：** オンにして、空間を選び（たとえば温かく広がりのある感じには**ミディアムホール**）、**ミックス**は控えめに保ちます。少しで十分な効果があります。近接マイクや「ドライ」な録音に空気感を加えるのに使いましょう。

## クロスフィード

**機能：** ヘッドフォンでは左右のチャンネルが完全に分離したままなので、特に極端にパンを振った古いステレオミックスでは、音楽が頭の中に閉じ込められたように感じられることがあります。クロスフィードは、部屋の中でスピーカーを自然に聴くのと同じように、各チャンネルの小さくフィルタリングされた量をもう一方に混ぜるので、音がより自然になり、長時間のリスニングでも疲れにくくなります。よく知られた **Bauer stereophonic-to-binaural（bs2b）** アルゴリズムを基に構築されています。

**プリセット（6種）：** サトル、Chu Moy、強、Jan Meier、スピーカーライク、ヴィンテージステレオ。

**詳細コントロール：**

- **カットオフ** — チャンネル間の漏れが減衰し始める場所。値が低いほど、より温かく際立った効果になります。
- **フィードレベル** — 一方のチャンネルがもう一方にどれだけ漏れるか。値が高いほどスピーカーらしく聴こえます。

**使い方：** これは**ヘッドフォン**用のエフェクトです。スピーカーにはオフのままにしてください。オンにして、**Chu Moy** または **Jan Meier**（どちらもオーディオファイルのお気に入り）、あるいは1960年代・1970年代の極端にパンを振った録音には**ヴィンテージステレオ**を試してください。

## ディレイ（エコー）

**機能：** 山でのエコーのように音を繰り返します。少しだと音楽をより豊かに感じさせ、多いと各音の後に明瞭でリズミカルなエコーを残します。Appleの `AVAudioUnitDelay` を基に構築されています。

**プリセット（10種）：** スラップバック、ダブラー、ショートエコー、標準、テープエコー、ブライトエコー、ロングエコー、ダブ、スペイシャス、アンビエント。

**詳細コントロール：**

- **ディレイタイム** — 元の音とそのエコーの間の間隔。短いとタイトなスラップ、長いと明瞭な繰り返しになります。
- **フィードバック** — エコーが消えるまでに何回繰り返すか。
- **トーン** — エコーから明るさをフィルタリングして、より温かいテープのような音にします。
- **ミックス** — エコーをどれだけ混ぜるか。

**使い方：** オンにして、繊細な深みには**スラップバック**または**テープエコー**から、長く広がりのある余韻には**アンビエント**や**ダブ**から始めてください。

## ディストーション

**機能：** 壊れたスピーカーやローファイな送信のように、音楽を粗くざらついた音にします。これは忠実度のツールではなく、楽しみのためのクリエイティブなエフェクトなので、控えめに使ってください。Appleの `AVAudioUnitDistortion` を基に構築されています。

**プリセット（22種）：** Bit Brush、Broken Speaker、Cellphone Concert、Radio Tower、Alien Chatter、Cosmic Interferenceなど、多数を含みます。

**詳細コントロール：**

- **プリゲイン** — 信号がどれだけ強くディストーションを駆動するか。高いほど攻撃的になります。
- **ミックス** — クリーンな音にディストーションをどれだけ混ぜるか。

**使い方：** オンにして、性格のあるプリセットを選び、大きく壊れた音が欲しくない限り**ミックス**は低く保ちます。エレクトロニックや実験的なトラックで楽しめます。

## エフェクトの構築方法

Evermusicのエフェクトは、最新の **AVAudioEngine** 処理チェーン内で動作します。各エフェクトは信号経路に配置されたネイティブなオーディオノードであり、オンにしたときだけアクティブになります。そうでなければゼロコストでバイパスされます。リバーブ、ディレイ、ディストーションはAppleの組み込みオーディオユニット（`AVAudioUnitReverb`、`AVAudioUnitDelay`、`AVAudioUnitDistortion`）を使用します。コンプレッサーはAppleの `AUDynamicsProcessor` を使用します。クロスフィードはオープンソースの **bs2b** アルゴリズムに基づくカスタムノードです。そして音量ノーマライズはリアルタイムの **EBU R128** ラウドネスノードです。

エフェクトは再生エンジンそのものの一部であるため、次のことが可能です。

- クラウドストリームやライブラジオを含め、再生するすべてに**リアルタイム**で適用されます。
- ファイルを**決して変更したり再エンコードしたりしません**。エフェクトをオフにすれば元の音が戻ります。
- エフェクトごとに、セッション間で**設定を記憶します**。
- それぞれが独立しているため、自由に組み合わせられます。

これらはEvermusicの **10バンドグラフィックイコライザー** および **ギャップレス再生** とも連携して動作するので、音色を整え、ラウドネスを揃え、切り替わりをシームレスに保つことを一度にすべて行えます。

## ヒント

- **シンプルモードから始めましょう。** まずプリセットを選び、細かく調整したいときだけ詳細モードを開きます。
- リバーブ、ディレイ、ディストーションでは **少ないほど良い** です。音楽的な結果を得るには、ミックスを控えめに保ちましょう。
- **クロスフィードはヘッドフォン用** であり、スピーカー用ではありません。
- **音量ノーマライズ＋コンプレッサー** を組み合わせると、混在するプレイリストや車内リスニングで最も一貫した聴きやすい結果が得られます。
- デフォルトに戻したい場合、すべてのスライダーに**リセット**ボタンがあります。

{{% details title="Evermusicにはどんなオーディオエフェクトがありますか？" closed="true" %}}
6つのリアルタイムエフェクト：音量ノーマライズ（EBU R128ラウドネス調整）、コンプレッサー（ダイナミクス）、リバーブ（空間とエコーの余韻）、クロスフィード（自然なヘッドフォンの定位）、ディレイ（エコー）、ディストーション（ローファイなざらつき）。それぞれ独立しており、単独でも組み合わせても使えます。
{{% /details %}}

{{% details title="エフェクトは私のオーディオファイルを変更したり損なったりしますか？" closed="true" %}}
いいえ。すべてのエフェクトは再生中にのみリアルタイムで適用されます。ファイルを変更したり再エンコードしたりすることは決してありません。エフェクトをオフにすれば、元の音がすぐに戻ります。
{{% /details %}}

{{% details title="複数のエフェクトを同時に使えますか？" closed="true" %}}
はい。各エフェクトは独立しており、マスタースイッチはないので、任意の組み合わせを有効にできます。たとえば、一貫した快適なリスニングには音量ノーマライズとコンプレッサー、ヘッドフォンにはリバーブとクロスフィードなどです。
{{% /details %}}

{{% details title="クロスフィードとは何ですか、使うべきですか？" closed="true" %}}
クロスフィードは、各ステレオチャンネルの小さくフィルタリングされた量をもう一方に混ぜるので、ヘッドフォンが本物のスピーカーのように聴こえ、極端にパンを振ったミックスの「頭の中」感を軽減します。これはヘッドフォン用のエフェクトです（スピーカーにはオフのままにしてください）。Bauer stereophonic-to-binaural（bs2b）アルゴリズムを基に構築されており、Chu MoyやJan Meierなどのプリセットが含まれます。
{{% /details %}}

{{% details title="音量ノーマライズとは何ですか、ReplayGainとどう違いますか？" closed="true" %}}
音量ノーマライズは、EBU R128規格で知覚ラウドネスを測定し、ターゲットに向けて揃えることで、すべてのトラックを一定のラウドネスに保ちます。ReplayGainとは異なり、ファイルにラウドネスタグを必要とせず、オーディオを変更しません。クラウドストリームやインターネットラジオを含め、あらゆるソースでライブに動作します。プリセット：ライト、標準、強、ナイト。
{{% /details %}}

{{% details title="シンプルモードと詳細モードの違いは何ですか？" closed="true" %}}
シンプルモードはわかりやすい説明付きのプリセット一覧を表示するので、ワンタップで良い音を得られます。詳細モードはパラメータのスライダー（たとえばリバーブのミックスや、コンプレッサーの7つのコントロール）を追加して、精密な微調整を可能にします。各エフェクトエディタの右上のモードボタンで両者を切り替えます。
{{% /details %}}

{{% details title="なぜエフェクトのコントロールがグレーアウトしているのですか？" closed="true" %}}
エフェクトがオフになっています。エディタの右上にあるエフェクトのスイッチをオンにすると、コントロールが有効になります。すべてのエフェクトはデフォルトでオフです。
{{% /details %}}

{{% details title="エフェクトはストリーミングやCarPlayで機能しますか？" closed="true" %}}
はい。エフェクトは再生エンジン内で動作するので、ローカルファイル、クラウドドライブ、メディアサーバー、インターネットラジオに適用され、CarPlay再生中も動作し続けます。
{{% /details %}}

**Riepilogo:** Evermusic e Flacbox possono archiviare qualsiasi playlist, album, artista o genere in un file ZIP con una playlist M3U, la copertina dell'album e tutti i file audio. Trasferisci lo ZIP su un altro dispositivo, estrailo e importa l'M3U per ricostruire la playlist istantaneamente.

Questa guida completa ti accompagnerà nel processo di archiviazione (ZIP) di playlist, album, artisti e generi in **Evermusic** e **Flacbox** e nel loro trasferimento su un altro dispositivo.

## Istruzioni passo passo

### 1. Archiviare una playlist

**1. Apri la sezione Playlist:** Avvia l'app e naviga alla sezione playlist.

**2. Seleziona una playlist:** Scegli la playlist che desideri archiviare.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. Accedi a Altre azioni:** Tocca il pulsante **Altre azioni** (tre puntini) per visualizzare tutte le opzioni disponibili.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. Scegli «Aggiungi all'archivio».**

- L'app:
  - Scaricherà tutti i file cloud associati alla playlist.
  - Copierà tutti i file locali dalla playlist in una cartella locale nella directory **Esporta**.
  - Salverà la copertina dell'album della playlist come file separato.
  - Salverà la struttura della playlist in un file M3U.
  - Comprimerà la cartella in un archivio ZIP.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. Monitorare l'avanzamento dell'archiviazione

- Apri la sezione **File locali** e vai a **Trasferimenti file** per monitorare l'avanzamento del download dei file della playlist.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- Una volta completata l'archiviazione, apparirà un **Avviso informativo**.  
  Puoi:
  - Chiudere l'avviso.
  - Selezionare **Mostra file** per localizzare l'archivio nell'app.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. Visualizzare l'archivio creato

- L'app aprirà la cartella **Documenti locali > Esporta** contenente l'archivio appena creato.
- Tocca l'archivio per:
  - Estrarlo nella directory corrente.
  - Visualizzarne il contenuto, inclusi tutti i brani della playlist e il file M3U.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. Utilizzare l'archivio creato

- Tocca **Altre azioni** sull'archivio per opzioni come:
  - **Carica / Copia:** Trasferisci l'archivio su un altro dispositivo o archiviazione cloud. Puoi anche utilizzare la funzione Wi-Fi Drive come descritto qui: [Come trasferire file in modalità wireless da un computer a un iPhone utilizzando WiFi-Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **Mostra nel Finder:** Localizza l'archivio sul tuo Mac (solo per utenti Mac).

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

Toccando il file M3U generato, puoi visualizzare tutti i brani della playlist insieme ai loro percorsi relativi, garantendo una navigazione facile e un ripristino perfetto della playlist.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. Trasferire e importare l'archivio su un nuovo dispositivo

1. Sposta o carica l'archivio sul nuovo dispositivo.
2. Apri l'app sul nuovo dispositivo e localizza l'archivio nella directory locale.
3. Tocca l'archivio per estrarne il contenuto.
4. Naviga alla sezione **Playlist**:
   - Tocca **Altre azioni** e seleziona **Importa playlist da file**.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- Scegli **File in questa applicazione.**

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- Naviga alla cartella estratta.
- Seleziona il file M3U.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. Ricostruzione della playlist

- L'app analizzerà il file M3U e ricostruirà la playlist, ripristinando tutti i brani e mantenendo la struttura originale.

![Immagine](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. Conclusione

Seguendo questa guida, puoi archiviare e trasferire in modo efficiente le tue playlist, album, artisti o generi tra dispositivi utilizzando Evermusic o Flacbox. Questo metodo garantisce che le tue playlist rimangano intatte e accessibili ovunque tu vada.

## Domande frequenti

{{% details title="Cosa è incluso nell'archivio ZIP?" closed="true" %}}
L'archivio contiene tutti i file audio, un file playlist M3U che preserva l'ordine dei brani e la copertina dell'album della playlist salvata come file immagine separato.
{{% /details %}}

{{% details title="L'archiviazione funziona con i file cloud?" closed="true" %}}
Sì. L'app scarica automaticamente tutti i file archiviati nel cloud prima di aggiungerli all'archivio. Puoi monitorare l'avanzamento del download nella sezione trasferimenti file.
{{% /details %}}

{{% details title="Posso archiviare anche album, artisti e generi?" closed="true" %}}
Sì. L'opzione «Aggiungi all'archivio» è disponibile per playlist, album, artisti e generi. Il processo è identico per tutti.
{{% /details %}}

{{% details title="Come trasferisco l'archivio su un altro dispositivo?" closed="true" %}}
Puoi caricare lo ZIP su un servizio di archiviazione cloud (Google Drive, Dropbox, ecc.), utilizzare AirDrop o trasferire in modalità wireless tramite la funzione integrata Wi-Fi Drive in Evermusic e Flacbox.
{{% /details %}}

{{% details title="La struttura della playlist verrà preservata dopo il trasferimento?" closed="true" %}}
Sì. Il file M3U memorizza la struttura della playlist con percorsi relativi. Dopo l'estrazione sul nuovo dispositivo, importa il file M3U per ricostruire la playlist con tutti i brani nell'ordine originale.
{{% /details %}}

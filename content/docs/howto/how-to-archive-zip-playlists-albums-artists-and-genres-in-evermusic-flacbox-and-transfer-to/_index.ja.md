**要約:** Evermusic と Flacbox は、任意のプレイリスト、アルバム、アーティスト、またはジャンルを M3U プレイリスト、アルバムアートワーク、およびすべてのオーディオファイルとともに ZIP ファイルにアーカイブできます。ZIP を別のデバイスに転送し、展開して M3U をインポートすると、プレイリストを即座に再構築できます。

この包括的なガイドでは、**Evermusic** と **Flacbox** でプレイリスト、アルバム、アーティスト、ジャンルをアーカイブ（ZIP）し、別のデバイスに転送するプロセスを説明します。

## ステップバイステップの手順

### 1. プレイリストのアーカイブ

**1. プレイリストセクションを開く：** アプリを起動し、プレイリストセクションに移動します。

**2. プレイリストを選択：** アーカイブしたいプレイリストを選択します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. その他のアクションにアクセス：** **その他のアクション**ボタン（3つの点）をタップして、利用可能なすべてのオプションを表示します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. 「アーカイブに追加」を選択します。**

- アプリは以下を行います：
  - プレイリストに関連するすべてのクラウドファイルをダウンロードします。
  - プレイリストのすべてのローカルファイルを**エクスポート**ディレクトリのローカルフォルダにコピーします。
  - プレイリストのアルバムアートワークを別ファイルとして保存します。
  - プレイリストの構造を M3U ファイルに保存します。
  - フォルダを ZIP アーカイブに圧縮します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. アーカイブの進捗を監視する

- **ローカルファイル**セクションを開き、**ファイル転送**に移動して、プレイリストファイルのダウンロード進捗を追跡します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- アーカイブが完了すると、**情報アラート**が表示されます。  
  以下のことができます：
  - アラートを閉じる。
  - **ファイルを表示**を選択して、アプリ内でアーカイブを見つける。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. 作成されたアーカイブを表示する

- アプリは、新しく作成されたアーカイブを含む**ローカルドキュメント > エクスポート**フォルダを開きます。
- アーカイブをタップして：
  - 現在のディレクトリに展開します。
  - プレイリストの全トラックと M3U ファイルを含む内容を表示します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. 作成されたアーカイブを使用する

- アーカイブの**その他のアクション**をタップして、以下のオプションを表示します：
  - **アップロード / コピー：** アーカイブを別のデバイスまたはクラウドストレージに転送します。ここで説明されている Wi-Fi Drive 機能も使用できます：[WiFi-Driveを使用してコンピューターからiPhoneにワイヤレスでファイルを転送する方法](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **Finderで表示：** Mac上でアーカイブを見つけます（Macユーザーのみ）。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

生成された M3U ファイルをタップすると、すべてのプレイリストトラックとその相対パスを表示でき、簡単なナビゲーションとシームレスなプレイリストの復元が可能になります。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. 新しいデバイスでアーカイブを転送してインポートする

1. アーカイブを新しいデバイスに移動またはアップロードします。
2. 新しいデバイスでアプリを開き、ローカルディレクトリでアーカイブを見つけます。
3. アーカイブをタップして内容を展開します。
4. **プレイリスト**セクションに移動します：
   - **その他のアクション**をタップし、**ファイルからプレイリストをインポート**を選択します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- **このアプリケーションのファイル**を選択します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- 展開されたフォルダに移動します。
- M3U ファイルを選択します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. プレイリストの再構築

- アプリは M3U ファイルを解析し、プレイリストを再構築して、すべてのトラックを復元し、元の構造を維持します。

![画像](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. まとめ

このガイドに従うことで、Evermusic または Flacbox を使用して、デバイス間でプレイリスト、アルバム、アーティスト、またはジャンルを効率的にアーカイブおよび転送できます。この方法により、プレイリストはどこに行っても完全な状態でアクセス可能に保たれます。

## よくある質問

{{% details title="ZIPアーカイブには何が含まれますか？" closed="true" %}}
アーカイブには、すべてのオーディオファイル、トラック順序を保持する M3U プレイリストファイル、および別の画像ファイルとして保存されたプレイリストのアルバムアートワークが含まれます。
{{% /details %}}

{{% details title="クラウドファイルでもアーカイブは機能しますか？" closed="true" %}}
はい。アプリはアーカイブに追加する前に、クラウドに保存されたすべてのファイルを自動的にダウンロードします。ダウンロードの進捗はファイル転送セクションで監視できます。
{{% /details %}}

{{% details title="アルバム、アーティスト、ジャンルもアーカイブできますか？" closed="true" %}}
はい。「アーカイブに追加」オプションは、プレイリスト、アルバム、アーティスト、ジャンルで利用できます。プロセスはすべて同じです。
{{% /details %}}

{{% details title="アーカイブを別のデバイスに転送するには？" closed="true" %}}
ZIP をクラウドストレージ（Google Drive、Dropbox など）にアップロードしたり、AirDrop を使用したり、Evermusic と Flacbox の組み込み Wi-Fi Drive 機能を使用してワイヤレスで転送できます。
{{% /details %}}

{{% details title="転送後もプレイリストの構造は保持されますか？" closed="true" %}}
はい。M3U ファイルは相対パスでプレイリストの構造を保存します。新しいデバイスで展開した後、M3U ファイルをインポートして、元の順序ですべてのトラックを含むプレイリストを再構築できます。
{{% /details %}}

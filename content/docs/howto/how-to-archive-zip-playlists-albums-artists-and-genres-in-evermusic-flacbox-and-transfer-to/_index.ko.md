**요약:** Evermusic과 Flacbox는 모든 재생 목록, 앨범, 아티스트 또는 장르를 M3U 재생 목록, 앨범 아트워크 및 모든 오디오 파일과 함께 ZIP 파일로 아카이브할 수 있습니다. ZIP을 다른 기기로 전송하고, 압축을 풀고, M3U를 가져와서 재생 목록을 즉시 재구축하세요.

이 종합 가이드는 **Evermusic** 및 **Flacbox**에서 재생 목록, 앨범, 아티스트, 장르를 아카이브(ZIP)하고 다른 기기로 전송하는 과정을 안내합니다.

## 단계별 안내

### 1. 재생 목록 아카이브하기

**1. 재생 목록 섹션 열기:** 앱을 실행하고 재생 목록 섹션으로 이동합니다.

**2. 재생 목록 선택:** 아카이브할 재생 목록을 선택합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. 추가 작업 접근:** **추가 작업** 버튼(점 세 개)을 탭하여 사용 가능한 모든 옵션을 표시합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. '아카이브에 추가'를 선택합니다.**

- 앱이 수행하는 작업:
  - 재생 목록과 관련된 모든 클라우드 파일을 다운로드합니다.
  - 재생 목록의 모든 로컬 파일을 **내보내기** 디렉토리의 로컬 폴더로 복사합니다.
  - 재생 목록의 앨범 아트워크를 별도 파일로 저장합니다.
  - 재생 목록 구조를 M3U 파일로 저장합니다.
  - 폴더를 ZIP 아카이브로 압축합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. 아카이브 진행 상황 모니터링

- **로컬 파일** 섹션을 열고 **파일 전송**으로 이동하여 재생 목록 파일의 다운로드 진행 상황을 추적합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- 아카이브가 완료되면 **정보 알림**이 표시됩니다.  
  다음을 수행할 수 있습니다:
  - 알림을 닫습니다.
  - **파일 보기**를 선택하여 앱에서 아카이브를 찾습니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. 생성된 아카이브 보기

- 앱이 새로 생성된 아카이브가 포함된 **로컬 문서 > 내보내기** 폴더를 엽니다.
- 아카이브를 탭하여:
  - 현재 디렉토리에 압축을 풉니다.
  - 모든 재생 목록 트랙과 M3U 파일을 포함한 내용을 봅니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. 생성된 아카이브 사용하기

- 아카이브에서 **추가 작업**을 탭하여 다음 옵션을 확인합니다:
  - **업로드 / 복사:** 아카이브를 다른 기기나 클라우드 스토리지로 전송합니다. 여기에 설명된 Wi-Fi Drive 기능도 사용할 수 있습니다: [WiFi-Drive를 사용하여 컴퓨터에서 iPhone으로 무선으로 파일 전송하는 방법](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **Finder에서 보기:** Mac에서 아카이브를 찾습니다(Mac 사용자 전용).

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

생성된 M3U 파일을 탭하면 모든 재생 목록 트랙과 상대 경로를 볼 수 있어 쉬운 탐색과 원활한 재생 목록 복원이 가능합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. 새 기기에서 아카이브 전송 및 가져오기

1. 아카이브를 새 기기로 이동하거나 업로드합니다.
2. 새 기기에서 앱을 열고 로컬 디렉토리에서 아카이브를 찾습니다.
3. 아카이브를 탭하여 내용을 압축 해제합니다.
4. **재생 목록** 섹션으로 이동합니다:
   - **추가 작업**을 탭하고 **파일에서 재생 목록 가져오기**를 선택합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- **이 애플리케이션의 파일**을 선택합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- 압축 해제된 폴더로 이동합니다.
- M3U 파일을 선택합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. 재생 목록 재구축

- 앱이 M3U 파일을 분석하고 재생 목록을 재구축하여 모든 트랙을 복원하고 원래 구조를 유지합니다.

![이미지](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. 결론

이 가이드를 따르면 Evermusic 또는 Flacbox를 사용하여 기기 간에 재생 목록, 앨범, 아티스트 또는 장르를 효율적으로 아카이브하고 전송할 수 있습니다. 이 방법은 어디를 가든 재생 목록이 온전하고 접근 가능하게 유지되도록 보장합니다.

## 자주 묻는 질문

{{% details title="ZIP 아카이브에는 무엇이 포함되나요?" closed="true" %}}
아카이브에는 모든 오디오 파일, 트랙 순서를 유지하는 M3U 재생 목록 파일, 별도의 이미지 파일로 저장된 재생 목록의 앨범 아트워크가 포함됩니다.
{{% /details %}}

{{% details title="클라우드 파일에서도 아카이브가 작동하나요?" closed="true" %}}
예. 앱은 아카이브에 추가하기 전에 클라우드에 저장된 모든 파일을 자동으로 다운로드합니다. 파일 전송 섹션에서 다운로드 진행 상황을 모니터링할 수 있습니다.
{{% /details %}}

{{% details title="앨범, 아티스트, 장르도 아카이브할 수 있나요?" closed="true" %}}
예. '아카이브에 추가' 옵션은 재생 목록, 앨범, 아티스트, 장르에서 사용할 수 있습니다. 모든 항목에 대해 프로세스가 동일합니다.
{{% /details %}}

{{% details title="아카이브를 다른 기기로 어떻게 전송하나요?" closed="true" %}}
ZIP을 클라우드 스토리지(Google Drive, Dropbox 등)에 업로드하거나, AirDrop을 사용하거나, Evermusic 및 Flacbox의 내장 Wi-Fi Drive 기능을 통해 무선으로 전송할 수 있습니다.
{{% /details %}}

{{% details title="전송 후에도 재생 목록 구조가 유지되나요?" closed="true" %}}
예. M3U 파일은 상대 경로와 함께 재생 목록 구조를 저장합니다. 새 기기에서 압축 해제 후 M3U 파일을 가져와서 원래 순서의 모든 트랙으로 재생 목록을 재구축할 수 있습니다.
{{% /details %}}

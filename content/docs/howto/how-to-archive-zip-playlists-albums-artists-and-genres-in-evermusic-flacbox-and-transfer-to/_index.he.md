**תקציר:** Evermusic ו-Flacbox יכולים לארכב כל רשימת השמעה, אלבום, אמן או ז'אנר לקובץ ZIP עם רשימת השמעה M3U, עטיפת אלבום וכל קובצי השמע. העבירו את ה-ZIP למכשיר אחר, חלצו אותו וייבאו את ה-M3U לבניית רשימת ההשמעה מחדש מיידית.

מדריך מקיף זה ילווה אתכם בתהליך ארכוב (ZIP) רשימות השמעה, אלבומים, אמנים וז'אנרים ב-**Evermusic** ו-**Flacbox** והעברתם למכשיר אחר.

## הוראות שלב אחר שלב

### 1. ארכוב רשימת השמעה

**1. פתחו את קטע רשימות השמעה:** הפעילו את האפליקציה ונווטו לקטע רשימות ההשמעה.

**2. בחרו רשימת השמעה:** בחרו את רשימת ההשמעה שברצונכם לארכב.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. גישה לעוד פעולות:** הקישו על כפתור **עוד פעולות** (שלוש נקודות) כדי להציג את כל האפשרויות הזמינות.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. בחרו «הוסף לארכיון».**

- האפליקציה תבצע:
  - הורדת כל קובצי הענן המשויכים לרשימת ההשמעה.
  - העתקת כל הקבצים המקומיים מרשימת ההשמעה לתיקייה מקומית בספריית **ייצוא**.
  - שמירת עטיפת האלבום של רשימת ההשמעה כקובץ נפרד.
  - שמירת מבנה רשימת ההשמעה בקובץ M3U.
  - דחיסת התיקייה לארכיון ZIP.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. מעקב אחר התקדמות הארכוב

- פתחו את קטע **קבצים מקומיים** ועברו ל**העברות קבצים** כדי לעקוב אחר התקדמות הורדת קובצי רשימת ההשמעה.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- לאחר השלמת הארכוב, תופיע **התראת מידע**.  
  ניתן:
  - לדחות את ההתראה.
  - לבחור **הצג קובץ** כדי לאתר את הארכיון באפליקציה.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. צפייה בארכיון שנוצר

- האפליקציה תפתח את תיקיית **מסמכים מקומיים > ייצוא** המכילה את הארכיון שנוצר לאחרונה.
- הקישו על הארכיון כדי:
  - לחלץ אותו לספרייה הנוכחית.
  - לצפות בתוכנו, כולל כל שירי רשימת ההשמעה וקובץ ה-M3U.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. שימוש בארכיון שנוצר

- הקישו על **עוד פעולות** בארכיון לאפשרויות כמו:
  - **העלאה / העתקה:** העבירו את הארכיון למכשיר אחר או אחסון ענן. ניתן גם להשתמש בתכונת Wi-Fi Drive כמתואר כאן: [כיצד להעביר קבצים באופן אלחוטי ממחשב ל-iPhone באמצעות WiFi-Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **הצג ב-Finder:** אתרו את הארכיון ב-Mac שלכם (למשתמשי Mac בלבד).

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

בהקשה על קובץ ה-M3U שנוצר, תוכלו לצפות בכל שירי רשימת ההשמעה יחד עם הנתיבים היחסיים שלהם, מה שמבטיח ניווט קל ושחזור חלק של רשימת ההשמעה.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. העברה וייבוא הארכיון במכשיר חדש

1. העבירו או העלו את הארכיון למכשיר החדש.
2. פתחו את האפליקציה במכשיר החדש ואתרו את הארכיון בספרייה המקומית.
3. הקישו על הארכיון כדי לחלץ את תוכנו.
4. נווטו לקטע **רשימות השמעה**:
   - הקישו על **עוד פעולות** ובחרו **ייבוא רשימת השמעה מקובץ**.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- בחרו **קבצים באפליקציה זו.**

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- נווטו לתיקייה שחולצה.
- בחרו את קובץ ה-M3U.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. בנייה מחדש של רשימת ההשמעה

- האפליקציה תנתח את קובץ ה-M3U ותבנה מחדש את רשימת ההשמעה, תשחזר את כל השירים ותשמור על המבנה המקורי.

![תמונה](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. סיכום

על ידי מעקב אחר מדריך זה, תוכלו לארכב ולהעביר ביעילות את רשימות ההשמעה, האלבומים, האמנים או הז'אנרים שלכם בין מכשירים באמצעות Evermusic או Flacbox. שיטה זו מבטיחה שרשימות ההשמעה שלכם יישארו שלמות ונגישות בכל מקום שתלכו.

## שאלות נפוצות

{{% details title="מה כלול בארכיון ה-ZIP?" closed="true" %}}
הארכיון מכיל את כל קובצי השמע, קובץ רשימת השמעה M3U ששומר על סדר השירים, ועטיפת האלבום של רשימת ההשמעה שנשמרת כקובץ תמונה נפרד.
{{% /details %}}

{{% details title="האם הארכוב עובד עם קובצי ענן?" closed="true" %}}
כן. האפליקציה מורידה אוטומטית את כל הקבצים המאוחסנים בענן לפני הוספתם לארכיון. ניתן לעקוב אחר התקדמות ההורדה בקטע העברות קבצים.
{{% /details %}}

{{% details title="האם אני יכול לארכב גם אלבומים, אמנים וז'אנרים?" closed="true" %}}
כן. האפשרות «הוסף לארכיון» זמינה עבור רשימות השמעה, אלבומים, אמנים וז'אנרים. התהליך זהה לכולם.
{{% /details %}}

{{% details title="כיצד להעביר את הארכיון למכשיר אחר?" closed="true" %}}
ניתן להעלות את ה-ZIP לאחסון ענן (Google Drive, Dropbox וכו'), להשתמש ב-AirDrop, או להעביר באופן אלחוטי דרך תכונת Wi-Fi Drive המובנית ב-Evermusic ו-Flacbox.
{{% /details %}}

{{% details title="האם מבנה רשימת ההשמעה יישמר לאחר ההעברה?" closed="true" %}}
כן. קובץ ה-M3U שומר את מבנה רשימת ההשמעה עם נתיבים יחסיים. לאחר חילוץ במכשיר החדש, ייבאו את קובץ ה-M3U כדי לבנות מחדש את רשימת ההשמעה עם כל השירים בסדר המקורי.
{{% /details %}}

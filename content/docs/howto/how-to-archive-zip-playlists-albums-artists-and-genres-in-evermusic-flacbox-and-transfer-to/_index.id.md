**Ringkasan:** Evermusic dan Flacbox dapat mengarsipkan daftar putar, album, artis, atau genre apa pun ke dalam file ZIP dengan daftar putar M3U, sampul album, dan semua file audio. Transfer ZIP ke perangkat lain, ekstrak, dan impor M3U untuk membangun ulang daftar putar secara instan.

Panduan lengkap ini akan memandu Anda melalui proses pengarsipan (ZIP) daftar putar, album, artis, dan genre di **Evermusic** dan **Flacbox** serta mentransfernya ke perangkat lain.

## Petunjuk Langkah demi Langkah

### 1. Mengarsipkan Daftar Putar

**1. Buka Bagian Daftar Putar:** Luncurkan aplikasi dan navigasi ke bagian daftar putar.

**2. Pilih Daftar Putar:** Pilih daftar putar yang ingin Anda arsipkan.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. Akses Lebih banyak tindakan:** Ketuk tombol **Lebih banyak tindakan** (tiga titik) untuk menampilkan semua opsi yang tersedia.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. Pilih «Tambahkan ke Arsip».**

- Aplikasi akan:
  - Mengunduh semua file cloud yang terkait dengan daftar putar.
  - Menyalin semua file lokal dari daftar putar ke folder lokal di direktori **Ekspor**.
  - Menyimpan sampul album daftar putar sebagai file terpisah.
  - Menyimpan struktur daftar putar dalam file M3U.
  - Mengompresi folder menjadi arsip ZIP.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. Memantau Kemajuan Pengarsipan

- Buka bagian **File Lokal** dan buka **Transfer File** untuk melacak kemajuan unduhan file daftar putar.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- Setelah pengarsipan selesai, **Peringatan Info** akan muncul.  
  Anda dapat:
  - Menutup peringatan.
  - Memilih **Tampilkan file** untuk menemukan arsip di aplikasi.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. Melihat Arsip yang Dibuat

- Aplikasi akan membuka folder **Dokumen Lokal > Ekspor** yang berisi arsip yang baru dibuat.
- Ketuk arsip untuk:
  - Mengekstraknya ke direktori saat ini.
  - Melihat isinya, termasuk semua trek daftar putar dan file M3U.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. Menggunakan Arsip yang Dibuat

- Ketuk **Lebih banyak tindakan** pada arsip untuk opsi seperti:
  - **Unggah / Salin:** Transfer arsip ke perangkat lain atau penyimpanan cloud. Anda juga dapat menggunakan fitur Wi-Fi Drive seperti yang dijelaskan di sini: [Cara Mentransfer File Secara Nirkabel dari Komputer ke iPhone Menggunakan WiFi-Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **Tampilkan di Finder:** Temukan arsip di Mac Anda (khusus pengguna Mac).

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

Dengan mengetuk file M3U yang dihasilkan, Anda dapat melihat semua trek daftar putar beserta jalur relatifnya, memastikan navigasi mudah dan pemulihan daftar putar yang mulus.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. Mentransfer dan Mengimpor Arsip di Perangkat Baru

1. Pindahkan atau unggah arsip ke perangkat baru.
2. Buka aplikasi di perangkat baru dan temukan arsip di direktori lokal.
3. Ketuk arsip untuk mengekstrak isinya.
4. Navigasi ke bagian **Daftar Putar**:
   - Ketuk **Lebih banyak tindakan** dan pilih **Impor Daftar Putar dari File**.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- Pilih **File di Aplikasi Ini.**

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- Navigasi ke folder yang diekstrak.
- Pilih file M3U.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. Membangun Ulang Daftar Putar

- Aplikasi akan mengurai file M3U dan membangun ulang daftar putar, memulihkan semua trek dan mempertahankan struktur asli.

![Gambar](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. Kesimpulan

Dengan mengikuti panduan ini, Anda dapat mengarsipkan dan mentransfer daftar putar, album, artis, atau genre Anda secara efisien antar perangkat menggunakan Evermusic atau Flacbox. Metode ini memastikan daftar putar Anda tetap utuh dan dapat diakses ke mana pun Anda pergi.

## FAQ

{{% details title="Apa saja yang termasuk dalam arsip ZIP?" closed="true" %}}
Arsip berisi semua file audio, file daftar putar M3U yang mempertahankan urutan trek, dan sampul album daftar putar yang disimpan sebagai file gambar terpisah.
{{% /details %}}

{{% details title="Apakah pengarsipan berfungsi dengan file cloud?" closed="true" %}}
Ya. Aplikasi secara otomatis mengunduh semua file yang disimpan di cloud sebelum menambahkannya ke arsip. Anda dapat memantau kemajuan unduhan di bagian transfer file.
{{% /details %}}

{{% details title="Bisakah saya mengarsipkan album, artis, dan genre juga?" closed="true" %}}
Ya. Opsi «Tambahkan ke Arsip» tersedia untuk daftar putar, album, artis, dan genre. Prosesnya identik untuk semuanya.
{{% /details %}}

{{% details title="Bagaimana cara mentransfer arsip ke perangkat lain?" closed="true" %}}
Anda dapat mengunggah ZIP ke penyimpanan cloud (Google Drive, Dropbox, dll.), menggunakan AirDrop, atau mentransfer secara nirkabel melalui fitur Wi-Fi Drive bawaan di Evermusic dan Flacbox.
{{% /details %}}

{{% details title="Apakah struktur daftar putar akan terjaga setelah transfer?" closed="true" %}}
Ya. File M3U menyimpan struktur daftar putar dengan jalur relatif. Setelah mengekstrak di perangkat baru, impor file M3U untuk membangun ulang daftar putar dengan semua trek dalam urutan asli.
{{% /details %}}

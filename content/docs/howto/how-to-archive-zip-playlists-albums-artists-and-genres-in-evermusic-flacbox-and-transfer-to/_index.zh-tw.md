**摘要：** Evermusic 和 Flacbox 可以將任何播放列表、專輯、藝術家或類型封存為 ZIP 檔案，其中包含 M3U 播放列表、專輯封面和所有音訊檔案。將 ZIP 傳輸到另一台裝置，解壓縮並匯入 M3U 即可立即重建播放列表。

本完整指南將引導您完成在 **Evermusic** 和 **Flacbox** 中封存（ZIP）播放列表、專輯、藝術家和類型並將其傳輸到另一台裝置的過程。

## 逐步說明

### 1. 封存播放列表

**1. 開啟播放列表區段：** 啟動應用程式並導覽至播放列表區段。

**2. 選擇播放列表：** 選擇您要封存的播放列表。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. 存取更多操作：** 點選 **更多操作** 按鈕（三個點）以顯示所有可用選項。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. 選擇「加入封存」。**

- 應用程式將：
  - 下載與播放列表相關的所有雲端檔案。
  - 將播放列表中的所有本機檔案複製到 **匯出** 目錄中的本機資料夾。
  - 將播放列表的專輯封面儲存為單獨的檔案。
  - 將播放列表結構儲存在 M3U 檔案中。
  - 將資料夾壓縮為 ZIP 封存檔。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. 監控封存進度

- 開啟 **本機檔案** 區段並前往 **檔案傳輸** 以追蹤播放列表檔案的下載進度。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- 封存完成後，將出現 **資訊提醒**。  
  您可以：
  - 關閉提醒。
  - 選擇 **顯示檔案** 以在應用程式中找到封存檔。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. 檢視已建立的封存檔

- 應用程式將開啟包含新建立封存檔的 **本機文件 > 匯出** 資料夾。
- 點選封存檔以：
  - 解壓縮到目前的目錄。
  - 檢視其內容，包括所有播放列表曲目和 M3U 檔案。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. 使用已建立的封存檔

- 點選封存檔上的 **更多操作** 以獲取選項：
  - **上傳 / 複製：** 將封存檔傳輸到另一台裝置或雲端儲存空間。您還可以使用 Wi-Fi Drive 功能，如此處所述：[如何使用 WiFi-Drive 從電腦無線傳輸檔案到 iPhone](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **在 Finder 中顯示：** 在您的 Mac 上找到封存檔（僅限 Mac 使用者）。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

透過點選產生的 M3U 檔案，您可以檢視所有播放列表曲目及其相對路徑，確保輕鬆導覽和無縫的播放列表恢復。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. 在新裝置上傳輸和匯入封存檔

1. 將封存檔移動或上傳到新裝置。
2. 在新裝置上開啟應用程式並在本機目錄中找到封存檔。
3. 點選封存檔以解壓縮其內容。
4. 導覽至 **播放列表** 區段：
   - 點選 **更多操作** 並選擇 **從檔案匯入播放列表**。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- 選擇 **此應用程式中的檔案。**

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- 導覽至解壓縮後的資料夾。
- 選擇 M3U 檔案。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. 重建播放列表

- 應用程式將解析 M3U 檔案並重建播放列表，恢復所有曲目並維持原始結構。

![圖片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. 總結

按照本指南，您可以使用 Evermusic 或 Flacbox 高效地在裝置之間封存和傳輸您的播放列表、專輯、藝術家或類型。此方法確保您的播放列表無論您走到哪裡都保持完整且可存取。

## 常見問題

{{% details title="ZIP 封存檔中包含什麼？" closed="true" %}}
封存檔包含所有音訊檔案、保留曲目順序的 M3U 播放列表檔案，以及作為單獨影像檔案儲存的播放列表專輯封面。
{{% /details %}}

{{% details title="封存是否適用於雲端檔案？" closed="true" %}}
是的。應用程式會在將檔案加入封存檔之前自動下載所有雲端儲存的檔案。您可以在檔案傳輸區段監控下載進度。
{{% /details %}}

{{% details title="我可以封存專輯、藝術家和類型嗎？" closed="true" %}}
是的。「加入封存」選項適用於播放列表、專輯、藝術家和類型。所有項目的過程完全相同。
{{% /details %}}

{{% details title="如何將封存檔傳輸到另一台裝置？" closed="true" %}}
您可以將 ZIP 上傳到雲端儲存空間（Google Drive、Dropbox 等），使用 AirDrop，或透過 Evermusic 和 Flacbox 內建的 Wi-Fi Drive 功能無線傳輸。
{{% /details %}}

{{% details title="傳輸後播放列表結構是否會保留？" closed="true" %}}
是的。M3U 檔案使用相對路徑儲存播放列表結構。在新裝置上解壓縮後，匯入 M3U 檔案即可以原始順序重建包含所有曲目的播放列表。
{{% /details %}}

**摘要：** Evermusic 和 Flacbox 可以将任何播放列表、专辑、艺术家或流派归档为 ZIP 文件，其中包含 M3U 播放列表、专辑封面和所有音频文件。将 ZIP 传输到另一台设备，解压缩并导入 M3U 即可立即重建播放列表。

本全面指南将引导您完成在 **Evermusic** 和 **Flacbox** 中归档（ZIP）播放列表、专辑、艺术家和流派并将其传输到另一台设备的过程。

## 分步说明

### 1. 归档播放列表

**1. 打开播放列表部分：** 启动应用并导航到播放列表部分。

**2. 选择播放列表：** 选择您要归档的播放列表。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. 访问更多操作：** 点击 **更多操作** 按钮（三个点）以显示所有可用选项。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. 选择「添加到归档」。**

- 应用将：
  - 下载与播放列表关联的所有云文件。
  - 将播放列表中的所有本地文件复制到 **导出** 目录中的本地文件夹。
  - 将播放列表的专辑封面保存为单独的文件。
  - 将播放列表结构保存在 M3U 文件中。
  - 将文件夹压缩为 ZIP 归档。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. 监控归档进度

- 打开 **本地文件** 部分并前往 **文件传输** 以跟踪播放列表文件的下载进度。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- 归档完成后，将出现 **信息提醒**。  
  您可以：
  - 关闭提醒。
  - 选择 **显示文件** 以在应用中找到归档文件。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. 查看已创建的归档

- 应用将打开包含新创建归档的 **本地文档 > 导出** 文件夹。
- 点击归档以：
  - 解压到当前目录。
  - 查看其内容，包括所有播放列表曲目和 M3U 文件。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. 使用已创建的归档

- 点击归档上的 **更多操作** 以获取选项：
  - **上传 / 复制：** 将归档传输到另一台设备或云存储。您还可以使用 Wi-Fi Drive 功能，如此处所述：[如何使用 WiFi-Drive 从计算机无线传输文件到 iPhone](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **在 Finder 中显示：** 在您的 Mac 上找到归档（仅限 Mac 用户）。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

通过点击生成的 M3U 文件，您可以查看所有播放列表曲目及其相对路径，确保轻松导航和无缝的播放列表恢复。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. 在新设备上传输和导入归档

1. 将归档移动或上传到新设备。
2. 在新设备上打开应用并在本地目录中找到归档。
3. 点击归档以解压其内容。
4. 导航到 **播放列表** 部分：
   - 点击 **更多操作** 并选择 **从文件导入播放列表**。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- 选择 **此应用中的文件。**

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- 导航到解压后的文件夹。
- 选择 M3U 文件。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. 重建播放列表

- 应用将解析 M3U 文件并重建播放列表，恢复所有曲目并保持原始结构。

![图片](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. 总结

按照本指南，您可以使用 Evermusic 或 Flacbox 高效地在设备之间归档和传输您的播放列表、专辑、艺术家或流派。此方法确保您的播放列表无论您走到哪里都保持完整且可访问。

## 常见问题

{{% details title="ZIP 归档中包含什么？" closed="true" %}}
归档包含所有音频文件、保留曲目顺序的 M3U 播放列表文件，以及作为单独图像文件保存的播放列表专辑封面。
{{% /details %}}

{{% details title="归档是否适用于云文件？" closed="true" %}}
是的。应用会在将文件添加到归档之前自动下载所有云存储的文件。您可以在文件传输部分监控下载进度。
{{% /details %}}

{{% details title="我可以归档专辑、艺术家和流派吗？" closed="true" %}}
是的。「添加到归档」选项适用于播放列表、专辑、艺术家和流派。所有项目的过程完全相同。
{{% /details %}}

{{% details title="如何将归档传输到另一台设备？" closed="true" %}}
您可以将 ZIP 上传到云存储（Google Drive、Dropbox 等），使用 AirDrop，或通过 Evermusic 和 Flacbox 内置的 Wi-Fi Drive 功能无线传输。
{{% /details %}}

{{% details title="传输后播放列表结构是否会保留？" closed="true" %}}
是的。M3U 文件使用相对路径存储播放列表结构。在新设备上解压后，导入 M3U 文件即可以原始顺序重建包含所有曲目的播放列表。
{{% /details %}}

**Resumo:** Evermusic e Flacbox podem arquivar qualquer lista de reprodução, álbum, artista ou gênero em um arquivo ZIP com uma lista de reprodução M3U, capa do álbum e todos os arquivos de áudio. Transfira o ZIP para outro dispositivo, descompacte-o e importe o M3U para reconstruir a lista de reprodução instantaneamente.

Este guia completo irá orientá-lo através do processo de arquivamento (ZIP) de listas de reprodução, álbuns, artistas e gêneros no **Evermusic** e **Flacbox** e sua transferência para outro dispositivo.

## Instruções passo a passo

### 1. Arquivando uma lista de reprodução

**1. Abra a seção Listas de reprodução:** Inicie o aplicativo e navegue até a seção de listas de reprodução.

**2. Selecione uma lista de reprodução:** Escolha a lista de reprodução que deseja arquivar.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_fdea8fb2cbb9400babf3a192089127d6~mv2.png)

**3. Acesse Mais ações:** Toque no botão **Mais ações** (três pontos) para exibir todas as opções disponíveis.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e420b168becb46efa4dd83f31761113f~mv2.png)

**4. Escolha «Adicionar ao arquivo».**

- O aplicativo irá:
  - Baixar todos os arquivos na nuvem associados à lista de reprodução.
  - Copiar todos os arquivos locais da lista de reprodução para uma pasta local no diretório **Exportar**.
  - Salvar a capa do álbum da lista de reprodução como arquivo separado.
  - Salvar a estrutura da lista de reprodução em um arquivo M3U.
  - Comprimir a pasta em um arquivo ZIP.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_67b5c504e56e4e1fbf5560e9506354bb~mv2.png)

### 2. Monitorando o progresso do arquivamento

- Abra a seção **Arquivos locais** e vá para **Transferências de arquivos** para acompanhar o progresso do download dos arquivos da lista de reprodução.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_e02bec716db149cba5a28bcb2f28c092~mv2.png)

- Quando o arquivamento estiver concluído, um **Alerta informativo** aparecerá.  
  Você pode:
  - Dispensar o alerta.
  - Selecionar **Mostrar arquivo** para localizar o arquivo no aplicativo.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2e21ca73f2034a388a0bb7896721c1c6~mv2.png)

### 3. Visualizando o arquivo criado

- O aplicativo abrirá a pasta **Documentos locais > Exportar** contendo o arquivo recém-criado.
- Toque no arquivo para:
  - Descompactá-lo no diretório atual.
  - Ver seu conteúdo, incluindo todas as faixas da lista de reprodução e o arquivo M3U.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_abcb83ddae774817a10b552d6b402faf~mv2.png)

### 4. Usando o arquivo criado

- Toque em **Mais ações** no arquivo para opções como:
  - **Enviar / Copiar:** Transfira o arquivo para outro dispositivo ou armazenamento na nuvem. Você também pode usar o recurso Wi-Fi Drive conforme descrito aqui: [Como transferir arquivos sem fio de um computador para um iPhone usando WiFi-Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)
  - **Mostrar no Finder:** Localize o arquivo no seu Mac (somente para usuários de Mac).

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_6a67e1837cfb4227a63bd1dd863c1fed~mv2.png)

Ao tocar no arquivo M3U gerado, você pode ver todas as faixas da lista de reprodução junto com seus caminhos relativos, garantindo navegação fácil e restauração perfeita da lista de reprodução.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_23a2f7975dbf4df9a96aa58ad0d20636~mv2.png)

### 5. Transferindo e importando o arquivo em um novo dispositivo

1. Mova ou envie o arquivo para o novo dispositivo.
2. Abra o aplicativo no novo dispositivo e localize o arquivo no diretório local.
3. Toque no arquivo para descompactar seu conteúdo.
4. Navegue até a seção **Listas de reprodução**:
   - Toque em **Mais ações** e selecione **Importar lista de reprodução de arquivo**.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_2f25598602f94a76b06d10e78ec48f94~mv2.png)

- Escolha **Arquivos neste aplicativo.**

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_bee3177d4a304c07baebc2e1f6a89025~mv2.png)

- Navegue até a pasta descompactada.
- Selecione o arquivo M3U.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_7cb52c7988c54365b565987198083524~mv2.png)

### 6. Reconstruindo a lista de reprodução

- O aplicativo analisará o arquivo M3U e reconstruirá a lista de reprodução, restaurando todas as faixas e mantendo a estrutura original.

![Imagem](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to/21260c_75ab446b3d3345fc8325ab6c30916beb~mv2.png)

### 7. Conclusão

Seguindo este guia, você pode arquivar e transferir eficientemente suas listas de reprodução, álbuns, artistas ou gêneros entre dispositivos usando Evermusic ou Flacbox. Este método garante que suas listas de reprodução permaneçam intactas e acessíveis onde quer que você vá.

## Perguntas frequentes

{{% details title="O que está incluído no arquivo ZIP?" closed="true" %}}
O arquivo contém todos os arquivos de áudio, um arquivo de lista de reprodução M3U que preserva a ordem das faixas e a capa do álbum da lista de reprodução salva como arquivo de imagem separado.
{{% /details %}}

{{% details title="O arquivamento funciona com arquivos na nuvem?" closed="true" %}}
Sim. O aplicativo baixa automaticamente todos os arquivos armazenados na nuvem antes de adicioná-los ao arquivo. Você pode monitorar o progresso do download na seção de transferências de arquivos.
{{% /details %}}

{{% details title="Posso arquivar álbuns, artistas e gêneros também?" closed="true" %}}
Sim. A opção «Adicionar ao arquivo» está disponível para listas de reprodução, álbuns, artistas e gêneros. O processo é idêntico para todos.
{{% /details %}}

{{% details title="Como transfiro o arquivo para outro dispositivo?" closed="true" %}}
Você pode enviar o ZIP para armazenamento na nuvem (Google Drive, Dropbox, etc.), usar AirDrop ou transferir sem fio através do recurso Wi-Fi Drive integrado no Evermusic e Flacbox.
{{% /details %}}

{{% details title="A estrutura da lista de reprodução será preservada após a transferência?" closed="true" %}}
Sim. O arquivo M3U armazena a estrutura da lista de reprodução com caminhos relativos. Após descompactar no novo dispositivo, importe o arquivo M3U para reconstruir a lista de reprodução com todas as faixas na ordem original.
{{% /details %}}

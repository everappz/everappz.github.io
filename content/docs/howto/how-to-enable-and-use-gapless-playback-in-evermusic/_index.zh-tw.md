**重點速覽：** 開啟 **設定 > 音訊播放器 > 無縫播放**，將開關撥到 **開啟**。從此以後，歌曲之間播放時不再有停頓、喀噠聲或滴答聲。Evermusic 會在目前曲目仍在播放時預先緩衝並解碼下一首，然後在連續的緩衝區上於音訊取樣之間完成交接，因此切換真正做到無縫。這是精確到取樣的真正無縫播放，而不是交叉淡入淡出。

## 什麼是無縫播放？

無縫播放消除了通常出現在兩首曲目之間的短暫靜音。開啟後，一首歌的最後一個音符會直接流入下一首的第一個音符，**沒有停頓、沒有喀噠聲、也沒有滴答聲**。

它對那些被母帶處理成一個連續整體來聆聽的音樂最為重要：

- **現場錄音和演唱會**，掌聲和觀眾噪音在歌曲之間延續。
- **DJ 混音和連續 set**，一首曲目按節拍匹配進入下一首。
- **古典作品**，各樂章本應連貫銜接。
- **概念專輯**，各曲目按設計直接淡出或交叉淡入到彼此（例如 *The Dark Side of the Moon* 或 *Abbey Road*）。

如果沒有無縫播放，這些專輯會在每個曲目邊界被一個微小的間隙打斷，破壞藝術家所期望的連貫性。

## 如何在 Evermusic 中啟用無縫播放

無縫播放**預設關閉**，因此你只需開啟一次，它便會保持開啟。

1. 開啟 **Evermusic**。
2. 前往 **設定** 標籤頁。
3. 點按 **音訊播放器**。
4. 點按 **無縫播放**。
5. 將 **無縫播放** 開關撥到 **開啟**。

就這樣。變更會立即儲存，並套用於你接下來播放的所有內容。

> **注意：** 開啟無縫播放後，**交叉淡入淡出會自動關閉**。這兩項功能作用相反：交叉淡入淡出會將一首曲目的結尾與下一首的開頭重疊並混合，而無縫播放則完整保留原始音訊，只是移除它們之間的間隙。你只能二擇一，不能同時使用。

## 如何使用無縫播放

啟用後，無需再做任何操作，它就是這樣運作。為獲得最佳體驗：

- **按順序播放整張專輯或連續的播放清單。** 把整張專輯加入佇列，按下播放，讓它從頭到尾播完。
- **保持曲目原定的順序。** 無縫在相鄰曲目之間才有意義，因此對概念專輯或現場 set 來說，隨機播放的意義不大。
- **本機檔案與雲端檔案同樣適用。** 無論你的音樂儲存在裝置上、雲端硬碟上還是媒體伺服器上，Evermusic 都會提前開始準備下一首，從而讓交接保持無縫。對於遠端來源，它只是稍微提早開始緩衝。
- **無損與有損格式均適用**，包括 FLAC、Apple Lossless (ALAC)、MP3、AAC 等。

## 無縫播放在 Evermusic 中如何運作

下面用淺白的話說明底層發生了什麼。

Evermusic 的播放引擎會**同時保持兩首曲目處於執行狀態**：你正在聽的那一首（*目前*項目）和排在它之後的那一首（*下一個*項目）。

1. **下一首提前準備。** 目前曲目仍在播放時，Evermusic 會在背景擷取、解碼並**預先緩衝**下一首。等到目前曲目結束時，下一首已經解碼完畢、隨時可播——不會出現「載入中」的停頓。
2. **輸出從不停止。** 引擎的算繪迴圈持續從共用緩衝區取出音訊取樣並送往喇叭或耳機。這個迴圈不會在曲目邊界停下。
3. **交接發生在取樣之間。** 當目前曲目到達它的最後一個取樣時，Evermusic 會**在播放器內部**將音源切換到下一首，而不是在音訊串流內部切換。輸出緩衝區持續不間斷地流動，因此切換發生在兩個音訊取樣之間的空隙裡——小到耳朵根本無法察覺。

由於切換發生在取樣層面、且緩衝區從不暫停，所以既沒有需要插入的靜音，也沒有需要在邊界重新啟動的解碼器。這正是消除喀噠聲、滴答聲和間隙的原因。

## 為什麼它是真正的無縫播放

有些應用程式只是*模擬*無縫播放。Evermusic 的是貨真價實的，區別就在這裡：

- **它精確到取樣，而不是交叉淡入淡出。** 交叉淡入淡出透過將兩首曲目重疊並淡化在一起來掩蓋間隙，這會改變你在邊界處聽到的音訊。無縫播放則將兩首曲目的每一個取樣都完全保留為母帶時的樣子，只是移除它們之間的靜音。
- **沒有解碼器重新啟動造成的間隙。** 許多「無縫」實作在開啟並解碼下一個檔案時仍會短暫停頓。Evermusic *提前*解碼下一首，因此在邊界處無需等待。
- **不插入任何靜音。** 某些編碼器和播放器會在曲目之間加入幾毫秒的填充。Evermusic 的連續緩衝區交接意味著播放時不會加入任何填充。
- **不重新編碼任何內容。** 你的音訊絲毫不動。無縫是透過曲目*如何*被排程和緩衝來實現的，而不是透過處理或重新壓縮聲音。
- **隨處可用。** 由於它內建於核心播放引擎，無縫播放適用於本機檔案、雲端硬碟、媒體伺服器、無損與有損格式——在所有這些來源上都能得到同樣無縫的效果。

結果就是：現場專輯、按節拍匹配的 DJ set 或概念唱片，都能以其本應呈現的方式播放——作為一段連續的音樂整體。

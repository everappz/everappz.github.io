**요약:** iPhone 또는 iPad용 Evermusic 앱을 사용하여 SMB를 통해 로컬 네트워크에서 Mac 또는 Windows PC의 음악을 스트리밍하세요. 동기화 불필요, 복사 불필요 -- 컴퓨터에서 파일 공유를 활성화하고, 앱에서 연결한 다음 재생하면 됩니다. 설정은 5분 이내로 완료됩니다.

MAC이나 PC에 가득 찬 음악의 바다에서 허우적대고 있고, iPhone이나 iPad에서 번거로움 없이 즐기고 싶으신가요? Evermusic이 정답입니다. Evermusic을 사용하면 SMB 프로토콜로 컴퓨터를 연결하고, 기기 용량 걱정이나 동기화 번거로움 없이 좋아하는 음악을 스트리밍하는 것이 놀라울 정도로 간단합니다. 시작하기 위한 단계별 가이드입니다:

## 1단계: 컴퓨터에서 SMB 프로토콜 활성화하기

![Evermusic - SMB 지원 - Mac 공유 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_4c8f67e6cad0498080909244213f84af~mv2.png)

### MAC을 사용하는 경우

- 시스템 환경설정 -> 공유를 엽니다.
- 파일 공유 서비스를 활성화합니다.
- "공유 폴더" 섹션에서 음악 폴더를 추가하고, 사용자를 선택한 다음 권한 수준(읽기 및 쓰기 또는 읽기 전용)을 설정합니다.
- 편의를 위해 음악 폴더에 "모든 사용자: 읽기 전용"을 선택하면 Evermusic에서 쉽게 접근할 수 있습니다.
- 다음 단계를 위해 컴퓨터의 URL(smb://192.168.xx.xx)을 기억해 두세요.

![Evermusic - SMB 지원 - 파일 공유 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_32c05fd0930a4ec28256afe40c0ba8a5~mv2.png)

- "옵션"을 탭하고 "SMB를 사용하여 파일 및 폴더 공유"를 활성화합니다.
- 사용 가능한 계정에 대해 "Windows 파일 공유"를 활성화합니다.

![Evermusic - SMB 지원 - 파일 및 폴더 공유 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_26acaa067aae40788465c1698b0458dc~mv2.png)

### Windows PC를 사용하는 경우

![Evermusic - SMB 지원 - Windows에서 파일 공유](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_c503c5d0d1f44daeb14d5a4cadfe9ac2~mv2.png)

- 음악 폴더를 마우스 오른쪽 버튼으로 클릭합니다.
- "속성"을 선택합니다.
- "공유" 탭을 엽니다.
- "공유..."를 클릭합니다.
- 공유할 사람을 선택하고 권한 수준을 설정합니다.
- MAC과 마찬가지로 선택한 음악 폴더에 "모든 사용자: 읽기"를 선택할 수 있습니다.
- "완료됨"을 클릭하여 설정을 저장합니다.

![Evermusic - SMB 지원 - Windows에서 공유된 폴더](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_350e2dca694b41bcade8f455acc4e481~mv2.png)

## 2단계: 컴퓨터 자동으로 추가하기

- 이제 Evermusic을 열고 "연결하기" 탭(이전 앱 버전을 사용하는 경우 "네트워크")으로 이동합니다.
- "사용 가능한 장치"(이전 앱 버전을 사용하는 경우 "사용 가능한 공유") 섹션에서 컴퓨터가 보이고, 이전 단계에서 "모든 사용자: 읽기 전용"을 선택했다면, 컴퓨터를 탭하면 자동으로 연결됩니다.
- 이것이 작동하지 않으면 수동으로 컴퓨터를 추가할 수 있습니다.

![Evermusic - SMB 지원 - 계정 연결 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_b1a1b89d7756458c952957fc2dd05582~mv2.jpg)

## 3단계: 컴퓨터 수동으로 추가하기

- "클라우드 서비스 연결"(이전 앱 버전을 사용하는 경우 "계정 추가")을 탭합니다.
- 다음 화면에서 사용 가능한 서버 목록에서 "SMB"를 선택합니다.
- "SMB" 설정 화면에서:
  - 공유 폴더 경로와 함께 서버 URL을 입력합니다. 서버 이름 또는 서버 IP를 입력할 수 있습니다. 예:
  
    ```
    smb://ameleshko.local/Music/
    smb://192.168.0.102/Music/
    smb://192.168.0.102/
    ```
  
  - 로그인과 비밀번호를 입력하거나, 이전 단계에서 "모든 사용자: 읽기 전용"을 선택한 경우 이 필드를 비워 둡니다.
  - "WORKGROUP" 필드는 선택 사항이며 Active Directory 도메인이 있는 경우 사용해야 합니다.

![Evermusic - SMB 지원 - 자격 증명 입력 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_9e043c2fa28d4932a7e7fb7e01df6923~mv2.jpg)

SMB 계정을 연결하면 "클라우드 서비스"("계정") 섹션에 나타납니다. 연결된 계정을 탭하여 열고, 음악 폴더로 이동한 다음 오디오 파일을 탭하여 플레이어를 시작합니다.

![Evermusic - SMB 지원 - 연결된 폴더 열기 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_d517e0d9a8ae4d5d973f0b42e396dc71~mv2.jpg)

Evermusic으로 iPhone 또는 iPad에서 음악 컬렉션을 원활하게 즐기세요.

![Evermusic - SMB 지원 - 오디오 플레이어 화면](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_fa2058e0ed9d48e0921b7229e5747f02~mv2.jpg)

## 4단계: SMB2 해결 방법

폴더 탐색이나 특수 문자(ü, ö, é 등)가 포함된 파일 재생에 문제가 발생하면 SMB2 프로토콜과 관련이 있을 수 있습니다. 이 문제를 해결하기 위해 적극적으로 작업하고 있습니다.

임시 해결책으로 서버와 앱 설정에서 SMB 1을 활성화해 보세요. 또는 시스템 파일 열기 메뉴를 사용하여 SMB 서버에 연결할 수 있습니다. 방법은 다음과 같습니다:

1. "로컬 파일"로 이동합니다.
2. "이 기기의 파일" 섹션까지 아래로 스크롤하고 "파일 열기..." 또는 "폴더 열기..."를 탭합니다.
3. 서버를 찾아 필요한 파일이나 폴더를 선택합니다.
4. "열기"를 탭하여 선택을 확인합니다.

## 5단계: WebDAV 해결 방법

또한 지원되는 경우 WebDAV 또는 DLNA 프로토콜을 사용하여 NAS에 연결해 볼 수 있습니다.

이 단계를 따르면 파일 이름의 특수 문자와 관련된 문제를 우회하고 미디어 파일을 계속 즐길 수 있습니다.

추신: iTunes 파일 공유를 사용하여 MAC/PC에서 iPhone으로 오디오 파일을 전송하고 로컬 오디오 파일을 재생할 수도 있습니다. 이 기능에 대한 자세한 내용은 가이드를 참조하세요: [iPhone에서 iTunes 파일 재생하는 방법](/docs/howto/how-to-play-local-itunes-files-on-my-iphone).

## 자주 묻는 질문

{{% details title="iTunes 없이 PC에서 iPhone으로 음악을 스트리밍할 수 있나요?" closed="true" %}}
네. Evermusic은 로컬 Wi-Fi 네트워크에서 SMB를 통해 PC에 연결합니다. iTunes가 필요 없습니다. PC에서 파일 공유를 활성화하고 앱에서 연결하기만 하면 됩니다.
{{% /details %}}

{{% details title="SMB 스트리밍은 모바일 데이터를 사용하나요?" closed="true" %}}
아니요. SMB는 로컬 Wi-Fi 네트워크를 통해 작동합니다. 인터넷 연결이나 모바일 데이터가 필요하지 않습니다.
{{% /details %}}

{{% details title="Evermusic은 SMB를 통해 어떤 오디오 형식을 지원하나요?" closed="true" %}}
Evermusic은 MP3, FLAC, AAC, WAV, AIFF, OGG, WMA, ALAC 및 기타 일반적인 오디오 형식을 지원합니다. 파일은 SMB 공유에서 직접 재생됩니다.
{{% /details %}}

{{% details title="NAS에서 iPhone으로 음악을 스트리밍할 수 있나요?" closed="true" %}}
네. NAS가 SMB를 지원하는 경우(Synology, QNAP, WD My Cloud을 포함한 대부분이 지원), 이 가이드의 동일한 단계를 사용하여 연결할 수 있습니다.
{{% /details %}}

{{% details title="스트리밍하는 동안 컴퓨터를 켜 놓아야 하나요?" closed="true" %}}
네. Evermusic은 컴퓨터에서 직접 파일을 스트리밍하므로 컴퓨터가 켜져 있고 iPhone과 동일한 네트워크에 연결되어 있어야 합니다.
{{% /details %}}

{{% details title="SMB 스트리밍에 파일 크기 제한이 있나요?" closed="true" %}}
아니요. Evermusic은 SMB를 통해 모든 크기의 파일을 스트리밍합니다. 큰 무손실 파일(FLAC, WAV)도 문제 없이 작동합니다.
{{% /details %}}

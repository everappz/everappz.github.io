**สรุป:** ใช้แอป Evermusic สำหรับ iPhone หรือ iPad เพื่อสตรีมเพลงจาก Mac หรือ Windows PC ผ่านเครือข่ายท้องถิ่นโดยใช้ SMB ไม่ต้องซิงค์ ไม่ต้องคัดลอก -- เพียงเปิดการแชร์ไฟล์บนคอมพิวเตอร์ เชื่อมต่อในแอป แล้วเล่น การตั้งค่าใช้เวลาไม่ถึง 5 นาที

คุณกำลังจมอยู่ในทะเลเพลงบน MAC หรือ PC และต้องการเพลิดเพลินกับมันอย่างสะดวกบน iPhone หรือ iPad หรือไม่? ไม่ต้องมองหาที่อื่นนอกจาก Evermusic ด้วย Evermusic การเชื่อมต่อคอมพิวเตอร์โดยใช้โปรโตคอล SMB และสตรีมเพลงโปรดของคุณเป็นเรื่องง่ายอย่างเหลือเชื่อ โดยไม่ต้องกังวลเรื่องพื้นที่จัดเก็บเพิ่มเติมหรือปัญหาการซิงโครไนซ์ นี่คือคู่มือทีละขั้นตอนสำหรับเริ่มต้น:

## ขั้นตอนที่ 1: เปิดใช้งานโปรโตคอล SMB บนคอมพิวเตอร์

![Evermusic - รองรับ SMB - หน้าจอแชร์ Mac](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_4c8f67e6cad0498080909244213f84af~mv2.png)

### หากคุณใช้ MAC

- เปิด System Preferences -> Sharing
- เปิดใช้งานบริการ File Sharing
- ในส่วน "Shared Folders" เพิ่มโฟลเดอร์เพลงของคุณ เลือกผู้ใช้ และตั้งระดับสิทธิ์ (Read & Write หรือ Read Only)
- เพื่อความสะดวกเพิ่มเติม คุณสามารถเลือก "Everyone: Read Only" สำหรับโฟลเดอร์เพลง ทำให้เข้าถึงได้ง่ายใน Evermusic
- อย่าลืมจดจำ URL ของคอมพิวเตอร์ (smb://192.168.xx.xx) สำหรับขั้นตอนถัดไป

![Evermusic - รองรับ SMB - หน้าจอแชร์ไฟล์](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_32c05fd0930a4ec28256afe40c0ba8a5~mv2.png)

- แตะ "Options" และเปิดใช้งาน "Share files and folders using SMB"
- เปิดใช้งาน "Windows File Sharing" สำหรับบัญชีที่มีอยู่

![Evermusic - รองรับ SMB - หน้าจอแชร์ไฟล์และโฟลเดอร์](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_26acaa067aae40788465c1698b0458dc~mv2.png)

### หากคุณใช้ Windows PC

![Evermusic - รองรับ SMB - แชร์ไฟล์บน Windows](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_c503c5d0d1f44daeb14d5a4cadfe9ac2~mv2.png)

- คลิกขวาที่โฟลเดอร์เพลงของคุณ
- เลือก "Properties"
- เปิดแท็บ "Sharing"
- คลิก "Share..."
- เลือกคนที่จะแชร์ด้วยและตั้งระดับสิทธิ์ของพวกเขา
- เช่นเดียวกับ MAC คุณสามารถเลือก "Everyone: Read" สำหรับโฟลเดอร์เพลงที่เลือก
- คลิก "เสร็จสิ้น" เพื่อบันทึกการตั้งค่า

![Evermusic - รองรับ SMB - โฟลเดอร์ที่แชร์บน Windows](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_350e2dca694b41bcade8f455acc4e481~mv2.png)

## ขั้นตอนที่ 2: เพิ่มคอมพิวเตอร์โดยอัตโนมัติ

- ตอนนี้ เปิด Evermusic และไปที่แท็บ "การเชื่อมต่อ" ("เครือข่าย" หากคุณใช้แอปเวอร์ชันเก่า)
- หากคุณเห็นคอมพิวเตอร์ในส่วน "อุปกรณ์ที่มีให้บริการ" ("แชร์ที่มีอยู่" หากคุณใช้แอปเวอร์ชันเก่า) และเลือก "Everyone: Read Only" ในขั้นตอนก่อนหน้า เพียงแตะที่คอมพิวเตอร์แล้วจะเชื่อมต่อโดยอัตโนมัติ
- หากไม่เกิดขึ้น คุณสามารถเพิ่มคอมพิวเตอร์ด้วยตนเอง

![Evermusic - รองรับ SMB - หน้าจอเชื่อมต่อบัญชี](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_b1a1b89d7756458c952957fc2dd05582~mv2.jpg)

## ขั้นตอนที่ 3: เพิ่มคอมพิวเตอร์ด้วยตนเอง

- แตะ "เชื่อมต่อบริการคลาวด์" ("เพิ่มบัญชี" หากคุณใช้แอปเวอร์ชันเก่า)
- เลือก "SMB" จากรายการเซิร์ฟเวอร์ที่มีอยู่บนหน้าจอถัดไป
- บนหน้าจอการตั้งค่า "SMB":
  - ป้อน URL ของเซิร์ฟเวอร์พร้อมเส้นทางโฟลเดอร์ที่แชร์ คุณสามารถป้อนชื่อเซิร์ฟเวอร์หรือ IP ของเซิร์ฟเวอร์ ตัวอย่างเช่น:
  
    ```
    smb://ameleshko.local/Music/
    smb://192.168.0.102/Music/
    smb://192.168.0.102/
    ```
  
  - ป้อนชื่อผู้ใช้และรหัสผ่านหรือเว้นว่างหากคุณเลือก "Everyone: Read Only" ในขั้นตอนก่อนหน้า
  - ช่อง "WORKGROUP" เป็นทางเลือกและควรใช้หากคุณมีโดเมน Active Directory

![Evermusic - รองรับ SMB - หน้าจอป้อนข้อมูลรับรอง](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_9e043c2fa28d4932a7e7fb7e01df6923~mv2.jpg)

เมื่อคุณเชื่อมต่อบัญชี SMB แล้ว จะปรากฏในส่วน "บริการคลาวด์" ("บัญชี") เปิดบัญชีที่เชื่อมต่อโดยแตะ นำทางไปยังโฟลเดอร์เพลง และแตะไฟล์เสียงใดก็ได้เพื่อเริ่มเครื่องเล่น

![Evermusic - รองรับ SMB - หน้าจอเปิดโฟลเดอร์ที่เชื่อมต่อ](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_d517e0d9a8ae4d5d973f0b42e396dc71~mv2.jpg)

เพลิดเพลินกับคอลเลกชันเพลงของคุณอย่างราบรื่นบน iPhone หรือ iPad ด้วย Evermusic

![Evermusic - รองรับ SMB - หน้าจอเครื่องเล่นเสียง](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/21260c_fa2058e0ed9d48e0921b7229e5747f02~mv2.jpg)

## ขั้นตอนที่ 4: วิธีแก้ปัญหา SMB2

หากคุณพบปัญหาในการเรียกดูโฟลเดอร์หรือเล่นไฟล์ที่มีสัญลักษณ์พิเศษ (เช่น ü, ö, é) อาจเกี่ยวข้องกับโปรโตคอล SMB2 เรากำลังทำงานอย่างแข็งขันเพื่อแก้ไขปัญหานี้

เป็นวิธีแก้ปัญหาชั่วคราว กรุณาลองเปิดใช้งาน SMB 1 บนเซิร์ฟเวอร์และในการตั้งค่าแอป หรือคุณสามารถเชื่อมต่อกับเซิร์ฟเวอร์ SMB โดยใช้เมนูเปิดไฟล์ของระบบ วิธีการดังนี้:

1. นำทางไปยัง "ไฟล์ในเครื่อง"
2. เลื่อนลงไปที่ส่วน "ไฟล์บนอุปกรณ์นี้" แล้วแตะ "เปิดไฟล์..." หรือ "เปิดโฟลเดอร์..."
3. ค้นหาเซิร์ฟเวอร์ของคุณและเลือกไฟล์หรือโฟลเดอร์ที่คุณต้องการ
4. แตะ "เปิด" เพื่อยืนยันการเลือก

## ขั้นตอนที่ 5: วิธีแก้ปัญหา WebDAV

นอกจากนี้ คุณสามารถลองเชื่อมต่อกับ NAS ของคุณโดยใช้โปรโตคอล WebDAV หรือ DLNA หากรองรับ

โดยทำตามขั้นตอนเหล่านี้ คุณสามารถหลีกเลี่ยงปัญหาที่เกี่ยวข้องกับสัญลักษณ์พิเศษในชื่อไฟล์และเพลิดเพลินกับไฟล์สื่อของคุณต่อไป

ป.ล. คุณยังสามารถถ่ายโอนไฟล์เสียงจาก MAC/PC ไปยัง iPhone โดยใช้การแชร์ไฟล์ iTunes และเล่นไฟล์เสียงในเครื่อง เรียนรู้เพิ่มเติมเกี่ยวกับฟีเจอร์นี้ในคู่มือของเรา: [วิธีเล่นไฟล์ iTunes บน iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)

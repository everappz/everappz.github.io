## Installationsinstruktioner för WD My Cloud Home i Evermusic

Följ dessa steg för att ansluta din WD My Cloud Home till Evermusic:

1. **Ladda ner Evermusic**  
   Hämta den gratis Evermusic-appen från [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198).

2. **Starta appen**  
   Öppna Evermusic på din iPhone eller iPad.

3. **Hoppa över introduktionsskärmar**  
   Svep genom introduktions-/handledningssidorna tills du når den sista installationsskärmen.

4. **Anslut till molnlagring**  
   Öppna **Anslutningar**, tryck på **Anslut till molnlagring**, tryck på **WD My Cloud Home** från listan över tillgängliga molnlagringstjänster.

5. **Ange dina inloggningsuppgifter**  
   Logga in med de inloggningsuppgifter du skapade när du registrerade din WD My Cloud Home-enhet på [home.mycloud.com](https://home.mycloud.com/).

När det är klart kommer din WD My Cloud Home att vara ansluten till Evermusic och redo för musikströmning eller nedladdningar.

## Hur man spelar musik på iPhone från WD MyCloud EX2 Ultra, WD MyCloud Mirror eller annan WD MyCloud-lagring

När du lägger till en anslutning i appen via **Anslutningar > Anslut molntjänst > My Cloud Home** är den utformad för att officiellt stödja WD MyCloud Home-enheter.

Om du dock har lyckats ansluta din WD MyCloud EX2 Ultra, WD MyCloud Mirror eller annan WD MyCloud-lagring, finns här en lösning som hjälper dig att använda den.

För att få det att fungera, skapa en mapp i appens filhanterare efter anslutning och ladda upp dina musikfiler dit. Det verkar som att WD:s server bara tillåter åtkomst till app-skapade mappar i ett "sandlådeläge" för dessa enheter, så denna mapp kommer bara vara tillgänglig från Evermusic-appen.

Använd Evermusics filhanterare för att ladda upp dina låtar till denna mapp. När filerna har laddats upp kan du ta bort de lokala kopiorna från din enhet och lyssna direkt från din NAS.

**Steg:** 

1. **Anslut till WD MyCloud-lagring:** Gå till *Anslutningar > Anslut molntjänst > My Cloud Home*.
2. **Skapa en mapp:** På den anslutna molntjänsten (synlig bara i appen).
3. **Ladda upp musikfiler:** Ladda upp dina lokala ljudfiler från enheten till denna mapp.
4. **Ta bort lokala filer:** När de har laddats upp, ta bort originalen från din enhet om det behövs.
5. **Börja strömma:** Spela upp alla uppladdade mediefiler direkt från mappen.

## FAQ

{{% details title="Är Evermusic gratis att använda med WD My Cloud Home?" closed="true" %}}
Evermusic är gratis att ladda ner med kärnfunktioner inklusive equalizer, molnströmning och offline-uppspelning. Gratisversionen stöder upp till 3 molnanslutningar. Uppgradering till Premium tar bort begränsningar på molnkonton, spellistor och offline-mappar.
{{% /details %}}

{{% details title="Kan jag lyssna på musik offline från min NAS?" closed="true" %}}
Ja. Evermusic låter dig ladda ner låtar från WD My Cloud Home till din iPhone för offline-uppspelning. Detta är användbart när du reser eller har begränsad internetåtkomst.
{{% /details %}}

{{% details title="Stöder Evermusic förlustfria ljudformat från WD My Cloud?" closed="true" %}}
Ja. Evermusic stöder FLAC, ALAC, WAV, AIFF och andra förlustfria format. Du kan strömma eller ladda ner högkvalitativa ljudfiler från din NAS utan formatkonvertering.
{{% /details %}}

{{% details title="Kan jag använda WD MyCloud EX2 Ultra med Evermusic?" closed="true" %}}
Ja, med en lösning. Anslut via My Cloud Home-alternativet, skapa en mapp med Evermusics filhanterare och ladda upp dina musikfiler dit. På grund av sandlådeläget är bara filer i app-skapade mappar tillgängliga.
{{% /details %}}

{{% details title="Hur mycket musik kan jag lagra på WD My Cloud Home?" closed="true" %}}
WD My Cloud Home stöder upp till 8 TB lagring. Vid typiska bithastigheter kan detta rymma hundratusentals låtar, inklusive stora förlustfria musikbibliotek.
{{% /details %}}

{{% details title="Är anslutningen mellan Evermusic och WD My Cloud Home säker?" closed="true" %}}
Ja. Evermusic använder en säker anslutning och Western Digitals officiella API för att komma åt din NAS. Dina data och inloggningsuppgifter skyddas under överföring.
{{% /details %}}

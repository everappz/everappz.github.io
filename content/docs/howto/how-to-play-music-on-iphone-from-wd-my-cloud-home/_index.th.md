## 8 TB ของเพลงบน iPhone ของคุณ?

คุณเคยฝันที่จะมีคลังเพลงขนาดใหญ่ 8 TB อยู่ปลายนิ้วบน iPhone หรือไม่? ด้วย Evermusic และ WD My Cloud Home ไม่เพียงเป็นไปได้แต่ยังง่ายอย่างน่าประหลาดใจ หากคุณมีคอลเลกชันเพลงอันมีค่าของคุณ รวมถึงอัลบั้มโปรดและการบันทึกพิเศษ และกำลังสงสัยว่าจะเพลิดเพลินกับมันบน iPhone ได้อย่างไร ลงทุนเพียง 2 นาทีเพื่อเรียนรู้วิธีสตรีมเพลงจาก WD My Cloud Home ไปยัง iPhone ของคุณอย่างง่ายดาย

{{< figure src="/docs/howto/how-to-play-music-on-iphone-from-wd-my-cloud-home/21260c_78c8759201c947c794ab1882db805e7b~mv2_d_2838_1978_s_2.jpg" alt="Evermusic + My Cloud Home: เข้าสู่ระบบ" width="600" >}}

มีเซิร์ฟเวอร์ NAS ส่วนบุคคลหลากหลายในตลาด แต่ละเครื่องมีข้อดีและข้อเสียของตัวเอง บางเครื่องอาจซับซ้อนกว่า ในขณะที่บางเครื่องใช้งานง่าย Western Digital ได้เปิดตัวหนึ่งในโซลูชันการจัดเก็บข้อมูลเครือข่ายส่วนบุคคลที่ง่ายที่สุด นั่นคือ WD My Cloud Home

วัตถุประสงค์หลักของอุปกรณ์จัดเก็บข้อมูลนี้คือทำให้การตั้งค่าและการใช้งานง่ายที่สุดสำหรับลูกค้าทุกคน โดยไม่จำเป็นต้องมีการกำหนดค่าที่ซับซ้อนหรือโปรไฟล์การตั้งค่า การตั้งค่าอุปกรณ์นี้ใช้เวลาเพียง 3 นาที คุณสามารถซื้อ WD My Cloud Home ได้โดยตรงจากเว็บไซต์ของพวกเขา [ที่นี่](https://www.westerndigital.com/products/cloud-storage/wd-my-cloud-home/)

เมื่อคุณได้รับแล้ว เพียงเสียบอุปกรณ์เข้ากับแหล่งจ่ายไฟและเชื่อมต่อสายเครือข่าย เยี่ยมชมหน้าหลัก [ลิงก์](https://home.mycloud.com/) เพื่อสร้างบัญชีของคุณ ตอนนี้อุปกรณ์ของคุณเชื่อมต่อกับอินเทอร์เน็ตแล้ว ทำให้คุณสามารถเข้าถึงไฟล์ทั้งหมดจากที่ไหนก็ได้โดยใช้เว็บเบราว์เซอร์ ขั้นตอนต่อไปคือการคัดลอกเพลงจากคอมพิวเตอร์ ซีดี หรือดีวีดีไปยังอุปกรณ์ My Cloud Home ซึ่งอาจใช้เวลาสักหน่อยแต่ทำเพียงครั้งเดียว

{{< figure src="/docs/howto/how-to-play-music-on-iphone-from-wd-my-cloud-home/21260c_29344e6912a34a26ba17bcda89864a2c~mv2.jpg" alt="Evermusic + My Cloud Home: เพิ่มที่เก็บข้อมูลคลาวด์" width="400" >}}

ตอนนี้คุณอาจสงสัยว่าเพลงทั้งหมดนั้นจะเข้าถึงได้บน iPhone อย่างไร? คำตอบอยู่ที่ Evermusic เครื่องเล่นเพลงเริ่มต้นสำหรับผู้ใช้หลายพันคนที่เบื่อหน่ายกับบริการสมัครสมาชิกเช่น Apple Music, Spotify, Plex และอื่นๆ ที่กินเงินในกระเป๋าของคุณทุกเดือน โดยคุณภาพเสียงไม่ดีเสมอไป ตอนนี้ถึงคราวของคุณที่จะตัดสินใจว่าต้องการจ่ายเงินต่อไปสำหรับฟีเจอร์ที่คุณสามารถเพลิดเพลินได้ฟรีหรือไม่ ด้วย Evermusic คุณสามารถสร้างบริการสตรีมเพลงไม่จำกัดของคุณเอง

## คำแนะนำการตั้งค่า WD My Cloud Home ใน Evermusic

ทำตามขั้นตอนเหล่านี้เพื่อเชื่อมต่อ WD My Cloud Home กับ Evermusic:

1. **ดาวน์โหลด Evermusic**  
   ดาวน์โหลดแอป Evermusic ฟรีจาก [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198)

2. **เปิดแอป**  
   เปิด Evermusic บน iPhone หรือ iPad ของคุณ

3. **ข้ามหน้าจอแนะนำ**  
   ปัดผ่านหน้าแนะนำ/บทเรียนจนกว่าจะถึงหน้าจอตั้งค่าสุดท้าย

4. **เชื่อมต่อกับที่เก็บข้อมูลคลาวด์**  
   เปิด **การเชื่อมต่อ** แตะ **เชื่อมต่อกับที่เก็บข้อมูลคลาวด์** แตะ **WD My Cloud Home** จากรายการบริการจัดเก็บข้อมูลคลาวด์ที่มี

5. **ป้อนข้อมูลรับรอง**  
   เข้าสู่ระบบโดยใช้ข้อมูลรับรองที่คุณสร้างเมื่อลงทะเบียนอุปกรณ์ WD My Cloud Home ที่ [home.mycloud.com](https://home.mycloud.com/)

เมื่อเสร็จสิ้น WD My Cloud Home ของคุณจะเชื่อมต่อกับ Evermusic และพร้อมสำหรับการสตรีมหรือดาวน์โหลดเพลง

Evermusic ใช้การเชื่อมต่อที่ปลอดภัยและ API อย่างเป็นทางการที่พัฒนาโดยวิศวกรของ Western Digital เพื่อรับประกันความปลอดภัยของข้อมูลและการสื่อสารของคุณ คุณสามารถฟังเพลงออนไลน์ได้โดยตรงจากอุปกรณ์ NAS ที่บ้านหรือดาวน์โหลดเพลงสำหรับการเล่นแบบออฟไลน์เมื่อคุณไม่มีการเชื่อมต่ออินเทอร์เน็ต

คุณจะมีการควบคุมเต็มรูปแบบเหนือไฟล์ของคุณ ความสามารถในการแก้ไขแท็ก เปลี่ยนความเร็วในการเล่น และเข้าถึงฟีเจอร์อื่นๆ อีกมากมาย

## วิธีเล่นเพลงบน iPhone จาก WD MyCloud EX2 Ultra, WD MyCloud Mirror หรือที่เก็บข้อมูล WD MyCloud อื่นๆ

เมื่อคุณเพิ่มการเชื่อมต่อในแอปผ่าน **การเชื่อมต่อ > เชื่อมต่อบริการคลาวด์ > My Cloud Home** มันถูกออกแบบมาเพื่อรองรับอุปกรณ์ WD MyCloud Home อย่างเป็นทางการ

อย่างไรก็ตาม หากคุณสามารถเชื่อมต่อ WD MyCloud EX2 Ultra, WD MyCloud Mirror หรือที่เก็บข้อมูล WD MyCloud อื่นๆ ได้ นี่คือวิธีแก้ไขเพื่อช่วยให้คุณใช้งานได้

เพื่อให้ใช้งานได้ เพียงสร้างโฟลเดอร์ในตัวจัดการไฟล์ของแอปหลังจากเชื่อมต่อและอัปโหลดไฟล์เพลงของคุณไปที่นั่น ดูเหมือนว่าเซิร์ฟเวอร์ของ WD อนุญาตให้เข้าถึงเฉพาะโฟลเดอร์ที่สร้างโดยแอปใน "โหมดแซนด์บ็อกซ์" สำหรับอุปกรณ์เหล่านี้ ดังนั้นโฟลเดอร์นี้จะเข้าถึงได้จากแอป Evermusic เท่านั้น

กรุณาใช้ตัวจัดการไฟล์ของ Evermusic เพื่ออัปโหลดเพลงของคุณไปยังโฟลเดอร์นี้ เมื่ออัปโหลดไฟล์แล้ว คุณสามารถลบสำเนาในเครื่องออกจากอุปกรณ์และฟังโดยตรงจาก NAS ของคุณ

**ขั้นตอน:** 

1. **เชื่อมต่อกับที่เก็บข้อมูล WD MyCloud:** ไปที่ *การเชื่อมต่อ > เชื่อมต่อบริการคลาวด์ > My Cloud Home*
2. **สร้างโฟลเดอร์:** บนบริการคลาวด์ที่เชื่อมต่อ (มองเห็นได้เฉพาะในแอป)
3. **อัปโหลดไฟล์เพลง:** อัปโหลดไฟล์เสียงในเครื่องจากอุปกรณ์ของคุณไปยังโฟลเดอร์นี้
4. **ลบไฟล์ในเครื่อง:** เมื่ออัปโหลดแล้ว ลบไฟล์ต้นฉบับออกจากอุปกรณ์หากจำเป็น
5. **เริ่มสตรีม:** เล่นไฟล์มีเดียที่อัปโหลดโดยตรงจากโฟลเดอร์

## สรุป

หากคุณเคยฝันที่จะมีคลังเพลงทั้งหมดของคุณ สูงสุด 8 TB บน iPhone ตอนนี้คุณสามารถทำให้เป็นจริงได้แล้ว! ด้วย WD My Cloud Home และแอป Evermusic คุณสามารถสตรีมหรือดาวน์โหลดคอลเลกชันทั้งหมดของคุณได้อย่างง่ายดาย โดยไม่ต้องสมัครสมาชิกหรือค่าธรรมเนียมรายเดือน การตั้งค่าทำได้รวดเร็ว และเมื่อเชื่อมต่อแล้ว คุณจะเพลิดเพลินกับการเล่นคุณภาพสูง การควบคุมไฟล์อย่างเต็มรูปแบบ และแม้แต่การตั้งค่าแบบกำหนดเองเช่นอีควอไลเซอร์

ขอบคุณที่สำรวจคู่มือนี้ -- ตอนนี้ ดำดิ่งสู่คอลเลกชันเพลงส่วนตัวของคุณจากทุกที่ ทุกเวลา ด้วย Evermusic และ WD My Cloud เพลิดเพลินกับอิสรภาพของคลังเพลงของคุณเอง ไร้ความยุ่งยาก!

## Petunjuk Pengaturan untuk WD My Cloud Home di Evermusic

Ikuti langkah-langkah ini untuk menghubungkan WD My Cloud Home Anda ke Evermusic:

1. **Unduh Evermusic**  
   Dapatkan aplikasi Evermusic gratis dari [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198).

2. **Buka Aplikasi**  
   Buka Evermusic di iPhone atau iPad Anda.

3. **Lewati Layar Perkenalan**  
   Geser melalui halaman perkenalan/tutorial hingga Anda mencapai layar pengaturan terakhir.

4. **Hubungkan ke Penyimpanan Cloud**  
   Buka **Koneksi**, ketuk **Hubungkan ke Penyimpanan Cloud**, ketuk **WD My Cloud Home** dari daftar layanan penyimpanan cloud yang tersedia.

5. **Masukkan Kredensial Anda**  
   Masuk menggunakan kredensial yang Anda buat saat mendaftarkan perangkat WD My Cloud Home Anda di [home.mycloud.com](https://home.mycloud.com/).

Setelah selesai, WD My Cloud Home Anda akan terhubung ke Evermusic dan siap untuk streaming atau mengunduh musik.

## Cara Memutar Musik di iPhone dari WD MyCloud EX2 Ultra, WD MyCloud Mirror, atau Penyimpanan WD MyCloud Lainnya

Ketika Anda menambahkan koneksi di aplikasi melalui **Koneksi > Hubungkan Layanan Cloud > My Cloud Home**, ini dirancang untuk secara resmi mendukung perangkat WD MyCloud Home.

Namun, jika Anda berhasil menghubungkan WD MyCloud EX2 Ultra, WD MyCloud Mirror, atau penyimpanan WD MyCloud lainnya, berikut adalah solusi alternatif untuk membantu Anda menggunakannya.

Agar ini berfungsi, cukup buat folder di pengelola file aplikasi setelah terhubung dan unggah file musik Anda ke sana. Tampaknya server WD hanya mengizinkan akses ke folder yang dibuat aplikasi dalam "mode sandbox" untuk perangkat ini, jadi folder ini hanya akan dapat diakses dari aplikasi Evermusic.

Silakan gunakan pengelola file Evermusic untuk mengunggah lagu-lagu Anda ke folder ini. Setelah file diunggah, Anda dapat menghapus salinan lokal dari perangkat Anda dan mendengarkan langsung dari NAS Anda.

**Langkah-langkah:**

1. **Hubungkan ke Penyimpanan WD MyCloud:** Buka *Koneksi > Hubungkan Layanan Cloud > My Cloud Home*.
2. **Buat Folder:** Di layanan cloud yang terhubung (hanya terlihat di aplikasi).
3. **Unggah File Musik:** Unggah file audio lokal dari perangkat Anda ke folder ini.
4. **Hapus File Lokal:** Setelah diunggah, hapus file asli dari perangkat Anda jika diperlukan.
5. **Mulai Streaming:** Putar file media yang diunggah langsung dari folder.

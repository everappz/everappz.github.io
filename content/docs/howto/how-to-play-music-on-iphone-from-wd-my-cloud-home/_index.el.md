## 8 TB μουσικής στο iPhone σας;

Ονειρευτήκατε ποτέ να έχετε μια τεράστια μουσική βιβλιοθήκη 8 TB στα χέρια σας στο iPhone; Λοιπόν, δεν είναι μόνο δυνατό αλλά και εκπληκτικά εύκολο με το Evermusic και το WD My Cloud Home. Αν έχετε την αγαπημένη σας μουσική συλλογή, συμπεριλαμβανομένων αγαπημένων άλμπουμ και αποκλειστικών ηχογραφήσεων, και αναρωτιέστε πώς να την απολαύσετε απρόσκοπτα στο iPhone σας, απλά επενδύστε 2 λεπτά από τον χρόνο σας για να μάθετε πώς να κάνετε εύκολα streaming μουσικής από το WD My Cloud Home στο iPhone σας.

{{< figure src="/docs/howto/how-to-play-music-on-iphone-from-wd-my-cloud-home/21260c_78c8759201c947c794ab1882db805e7b~mv2_d_2838_1978_s_2.jpg" alt="Evermusic + My Cloud Home: Sign In" width="600" >}}

Υπάρχει μια σειρά προσωπικών διακομιστών NAS διαθέσιμων στην αγορά, ο καθένας με τα δικά του πλεονεκτήματα και μειονεκτήματα. Ορισμένοι μπορεί να είναι πιο πολύπλοκοι, ενώ άλλοι είναι φιλικοί προς τον χρήστη. Η Western Digital εισήγαγε μια από τις πιο απλές λύσεις προσωπικής δικτυακής αποθήκευσης, το WD My Cloud Home.

Ο κύριος στόχος αυτής της συσκευής αποθήκευσης είναι να κάνει την εγκατάσταση και τη χρήση όσο πιο απλή γίνεται για όλους τους πελάτες, εξαλείφοντας την ανάγκη για περίπλοκες διαμορφώσεις ή προφίλ εγκατάστασης. Η εγκατάσταση αυτής της συσκευής χρειάζεται μόλις 3 λεπτά. Μπορείτε να αγοράσετε το WD My Cloud Home απευθείας από τον ιστότοπό τους [εδώ](https://www.westerndigital.com/products/cloud-storage/wd-my-cloud-home/).

Μόλις το λάβετε, απλά συνδέστε τη συσκευή σε μια πηγή τροφοδοσίας και συνδέστε ένα καλώδιο δικτύου. Επισκεφτείτε την αρχική σελίδα [σύνδεσμος](https://home.mycloud.com/) για να δημιουργήσετε τον λογαριασμό σας. Η συσκευή σας είναι πλέον συνδεδεμένη στο διαδίκτυο, δίνοντάς σας πρόσβαση σε όλα τα αρχεία σας από οπουδήποτε χρησιμοποιώντας τον περιηγητή σας. Το επόμενο βήμα περιλαμβάνει την αντιγραφή της μουσικής σας από τον υπολογιστή, CD ή DVD στη συσκευή My Cloud Home, κάτι που μπορεί να πάρει λίγο χρόνο αλλά είναι μια εφάπαξ προσπάθεια.

{{< figure src="/docs/howto/how-to-play-music-on-iphone-from-wd-my-cloud-home/21260c_29344e6912a34a26ba17bcda89864a2c~mv2.jpg" alt="Evermusic + My Cloud Home: Add Cloud Storage" width="400" >}}

Τώρα, ίσως αναρωτιέστε, πώς μπορεί όλη αυτή η μουσική να είναι προσβάσιμη στο iPhone σας; Η απάντηση βρίσκεται στο Evermusic, τον προεπιλεγμένο μουσικό αναπαραγωγέα για χιλιάδες χρήστες που έχουν κουραστεί από υπηρεσίες βασισμένες σε συνδρομές όπως Apple Music, Spotify, Plex και άλλες που τρώνε το πορτοφόλι σας κάθε μήνα, με ποιότητα ήχου που δεν είναι πάντα στο ύψος της. Τώρα, είναι η σειρά σας να αποφασίσετε αν θέλετε να συνεχίσετε να πληρώνετε για λειτουργίες που μπορείτε να απολαύσετε δωρεάν. Με το Evermusic, μπορείτε να δημιουργήσετε τη δική σας απεριόριστη υπηρεσία streaming μουσικής.

## Οδηγίες εγκατάστασης για το WD My Cloud Home στο Evermusic

Ακολουθήστε αυτά τα βήματα για να συνδέσετε το WD My Cloud Home με το Evermusic:

1. **Λήψη του Evermusic**  
   Κατεβάστε τη δωρεάν εφαρμογή Evermusic από το [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198).

2. **Εκκίνηση της εφαρμογής**  
   Ανοίξτε το Evermusic στο iPhone ή iPad σας.

3. **Παράλειψη των εισαγωγικών οθονών**  
   Σύρετε μέσα από τις σελίδες εισαγωγής/εκμάθησης μέχρι να φτάσετε στην τελική οθόνη εγκατάστασης.

4. **Σύνδεση σε αποθήκευση cloud**  
   Ανοίξτε **Συνδέσεις**, πατήστε **Σύνδεση σε αποθήκευση cloud**, πατήστε **WD My Cloud Home** από τη λίστα των διαθέσιμων υπηρεσιών αποθήκευσης cloud.

5. **Εισαγωγή των διαπιστευτηρίων σας**  
   Συνδεθείτε χρησιμοποιώντας τα διαπιστευτήρια που δημιουργήσατε κατά την εγγραφή της συσκευής WD My Cloud Home στο [home.mycloud.com](https://home.mycloud.com/).

Μόλις ολοκληρωθεί, το WD My Cloud Home θα συνδεθεί με το Evermusic και θα είναι έτοιμο για streaming ή λήψη μουσικής.

{{< figure src="/docs/howto/how-to-play-music-on-iphone-from-wd-my-cloud-home/21260c_d6295ba7da90419ca5a50be103f78a0b~mv2.jpg" alt="Evermusic + My Cloud Home: Enter Credentials" width="400" >}}

Το Evermusic χρησιμοποιεί ασφαλή σύνδεση και το επίσημο API που αναπτύχθηκε από τους μηχανικούς της Western Digital, εξασφαλίζοντας την ασφάλεια των δεδομένων και της επικοινωνίας σας. Μπορείτε να ακούτε τη μουσική σας online απευθείας από τη συσκευή NAS στο σπίτι σας ή να κατεβάζετε κομμάτια για αναπαραγωγή εκτός σύνδεσης όταν δεν έχετε σύνδεση στο διαδίκτυο.

Θα έχετε πλήρη έλεγχο στα αρχεία σας, τη δυνατότητα επεξεργασίας ετικετών, αλλαγής ταχύτητας αναπαραγωγής και πρόσβαση σε πολυάριθμες άλλες λειτουργίες.

## Πώς να αναπαράγετε μουσική στο iPhone από WD MyCloud EX2 Ultra, WD MyCloud Mirror ή οποιαδήποτε άλλη αποθήκευση WD MyCloud

Όταν προσθέτετε μια σύνδεση στην εφαρμογή μέσω **Συνδέσεις > Σύνδεση υπηρεσίας Cloud > My Cloud Home**, είναι σχεδιασμένη για να υποστηρίζει επίσημα συσκευές WD MyCloud Home.

Ωστόσο, αν καταφέρατε να συνδέσετε το WD MyCloud EX2 Ultra, WD MyCloud Mirror ή οποιαδήποτε άλλη αποθήκευση WD MyCloud, εδώ είναι μια εναλλακτική λύση για να σας βοηθήσει να το χρησιμοποιήσετε.

Για να λειτουργήσει αυτό, απλά δημιουργήστε έναν φάκελο στον διαχειριστή αρχείων της εφαρμογής μετά τη σύνδεση και ανεβάστε τα αρχεία μουσικής σας εκεί. Φαίνεται ότι ο διακομιστής της WD επιτρέπει πρόσβαση μόνο σε φακέλους που δημιουργήθηκαν από την εφαρμογή σε "λειτουργία sandbox" για αυτές τις συσκευές, οπότε αυτός ο φάκελος θα είναι προσβάσιμος μόνο από την εφαρμογή Evermusic.

Παρακαλώ χρησιμοποιήστε τον διαχειριστή αρχείων του Evermusic για να ανεβάσετε τα τραγούδια σας σε αυτόν τον φάκελο. Μόλις ανεβούν τα αρχεία, μπορείτε να διαγράψετε τα τοπικά αντίγραφα από τη συσκευή σας και να ακούτε απευθείας από το NAS σας.

**Βήματα:**

1. **Σύνδεση στην αποθήκευση WD MyCloud:** Μεταβείτε στο *Συνδέσεις > Σύνδεση υπηρεσίας Cloud > My Cloud Home*.
2. **Δημιουργία φακέλου:** Στη συνδεδεμένη υπηρεσία cloud (ορατό μόνο στην εφαρμογή).
3. **Μεταφόρτωση αρχείων μουσικής:** Ανεβάστε τα τοπικά αρχεία ήχου από τη συσκευή σας σε αυτόν τον φάκελο.
4. **Διαγραφή τοπικών αρχείων:** Μόλις ανεβούν, διαγράψτε τα πρωτότυπα από τη συσκευή σας αν χρειάζεται.
5. **Έναρξη streaming:** Αναπαράγετε οποιοδήποτε ανεβασμένο αρχείο πολυμέσων απευθείας από τον φάκελο.

Με το WD My Cloud Home και την εφαρμογή Evermusic, μπορείτε να κάνετε streaming ή λήψη ολόκληρης της συλλογής σας αβίαστα, χωρίς την ανάγκη συνδρομών ή μηνιαίων τελών. Η εγκατάσταση είναι γρήγορη, και μόλις συνδεθείτε, θα απολαύσετε αναπαραγωγή υψηλής ποιότητας, πλήρη έλεγχο στα αρχεία σας και ακόμα προσαρμοσμένες ρυθμίσεις όπως ισοσταθμιστή.

Ευχαριστούμε που εξερευνήσατε αυτόν τον οδηγό -- τώρα, βουτήξτε στην προσωπική σας μουσική συλλογή από οπουδήποτε, οποτεδήποτε, με το Evermusic και το WD My Cloud. Απολαύστε την ελευθερία της δικής σας μουσικής βιβλιοθήκης, χωρίς ταλαιπωρία!

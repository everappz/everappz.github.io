## iPhoneに8 TBの音楽？

iPhoneで8 TBの巨大な音楽ライブラリを手元に持つことを夢見たことはありませんか？EvermusicとWD My Cloud Homeを使えば、それは可能なだけでなく、驚くほど簡単です。お気に入りのアルバムや限定録音を含む大切な音楽コレクションがあり、iPhoneでシームレスに楽しむ方法を考えているなら、たった2分の時間を投資して、WD My Cloud HomeからiPhoneに簡単に音楽をストリーミングする方法を学びましょう。

市場にはさまざまなパーソナルNASサーバーがあり、それぞれに長所と短所があります。より複雑なものもあれば、使いやすいものもあります。Western Digitalは最もシンプルなパーソナルネットワーク接続ストレージソリューションの1つ、WD My Cloud Homeを発表しました。

このストレージデバイスの主な目的は、すべてのお客様にとってセットアップと使用を可能な限り簡単にすることであり、複雑な設定やセットアッププロファイルの必要性を排除しています。このデバイスのセットアップにはわずか3分しかかかりません。WD My Cloud Homeはウェブサイトから直接購入できます[こちら](https://www.westerndigital.com/products/cloud-storage/wd-my-cloud-home/)。

届いたら、デバイスを電源に接続し、ネットワークケーブルを接続するだけです。ホームページ[リンク](https://home.mycloud.com/)にアクセスしてアカウントを作成してください。デバイスがインターネットに接続され、ウェブブラウザを使用してどこからでもすべてのファイルにアクセスできるようになります。次のステップは、コンピュータ、CD、またはDVDからMy Cloud Homeデバイスに音楽をコピーすることです。時間がかかるかもしれませんが、一度きりの作業です。

さて、その大量の音楽をiPhoneでどうやってアクセスできるのかと疑問に思うかもしれません。答えはEvermusicにあります。Apple Music、Spotify、Plexなどのサブスクリプションベースのサービスに疲れた何千人ものユーザーのためのデフォルト音楽プレーヤーです。これらのサービスは毎月財布を圧迫し、音質も常に最高とは限りません。今こそ、無料で楽しめる機能にお金を払い続けるかどうかを決める番です。Evermusicを使えば、自分だけの無制限音楽ストリーミングサービスを構築できます。

## EvermusicでのWD My Cloud Homeセットアップ手順

WD My Cloud HomeをEvermusicに接続するには、以下の手順に従ってください：

1. **Evermusicをダウンロード**  
   [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198)から無料のEvermusicアプリを入手します。

2. **アプリを起動**  
   iPhoneまたはiPadでEvermusicを開きます。

3. **イントロ画面をスキップ**  
   紹介/チュートリアルページをスワイプして、最後のセットアップ画面に到達します。

4. **クラウドストレージに接続**  
   **接続**を開き、**クラウドストレージに接続**をタップし、利用可能なクラウドストレージサービスのリストから**WD My Cloud Home**をタップします。

5. **認証情報を入力**  
   [home.mycloud.com](https://home.mycloud.com/)でWD My Cloud Homeデバイスを登録した際に作成した認証情報を使用してログインします。

完了すると、WD My Cloud HomeがEvermusicに接続され、音楽のストリーミングやダウンロードの準備が整います。

Evermusicは安全な接続とWestern Digitalのエンジニアが開発した公式APIを使用しており、データと通信の安全性を確保しています。自宅のNASデバイスから直接オンラインで音楽を聴いたり、インターネット接続がない時のためにトラックをダウンロードしてオフライン再生することができます。

## WD MyCloud EX2 Ultra、WD MyCloud Mirror、またはその他のWD MyCloudストレージからiPhoneで音楽を再生する方法

アプリで**接続 > クラウドサービスに接続 > My Cloud Home**からコネクションを追加する場合、WD MyCloud Homeデバイスを公式にサポートするように設計されています。

ただし、WD MyCloud EX2 Ultra、WD MyCloud Mirror、またはその他のWD MyCloudストレージの接続に成功した場合は、使用するための回避策があります。

これを機能させるには、接続後にアプリのファイルマネージャーでフォルダを作成し、音楽ファイルをそこにアップロードするだけです。WDのサーバーはこれらのデバイスの「サンドボックスモード」でアプリが作成したフォルダへのアクセスのみを許可しているようなので、このフォルダはEvermusicアプリからのみアクセス可能です。

Evermusicのファイルマネージャーを使用して、このフォルダに曲をアップロードしてください。ファイルがアップロードされたら、デバイスからローカルコピーを削除し、NASから直接聴くことができます。

**手順：**

1. **WD MyCloudストレージに接続：** *接続 > クラウドサービスに接続 > My Cloud Home*に移動します。
2. **フォルダを作成：** 接続されたクラウドサービス上に（アプリ内でのみ表示）。
3. **音楽ファイルをアップロード：** デバイスからこのフォルダにローカルオーディオファイルをアップロードします。
4. **ローカルファイルを削除：** アップロード後、必要に応じてデバイスからオリジナルを削除します。
5. **ストリーミングを開始：** フォルダからアップロードされたメディアファイルを直接再生します。

## Evermusic中WD My Cloud Home的設定說明

按照以下步驟將WD My Cloud Home連接到Evermusic：

1. **下載Evermusic**  
   從[App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198)取得免費的Evermusic應用程式。

2. **啟動應用程式**  
   在iPhone或iPad上開啟Evermusic。

3. **跳過介紹畫面**  
   滑過介紹/教學頁面，直到到達最後的設定畫面。

4. **連接到雲端儲存**  
   開啟**連接**，點選**連接到雲端儲存**，從可用的雲端儲存服務清單中點選**WD My Cloud Home**。

5. **輸入您的憑證**  
   使用您在[home.mycloud.com](https://home.mycloud.com/)註冊WD My Cloud Home裝置時建立的憑證登入。

完成後，您的WD My Cloud Home將連接到Evermusic，準備好進行音樂串流或下載。

## 如何在iPhone上播放WD MyCloud EX2 Ultra、WD MyCloud Mirror或任何其他WD MyCloud儲存中的音樂

當您透過**連接 > 連接雲端服務 > My Cloud Home**在應用程式中新增連線時，它旨在正式支援WD MyCloud Home裝置。

但是，如果您已經能夠連接WD MyCloud EX2 Ultra、WD MyCloud Mirror或任何其他WD MyCloud儲存，這裡有一個替代方法幫助您使用它。

要使其運作，只需在連接後在應用程式的檔案管理器中建立一個資料夾並將音樂檔案上傳到那裡。看起來WD的伺服器只允許在「沙盒模式」下存取由應用程式建立的資料夾，因此該資料夾只能從Evermusic應用程式存取。

請使用Evermusic的檔案管理器將歌曲上傳到此資料夾。檔案上傳後，您可以從裝置中刪除本機副本並直接從NAS收聽。

**步驟：** 

1. **連接到WD MyCloud儲存：** 前往*連接 > 連接雲端服務 > My Cloud Home*。
2. **建立資料夾：** 在已連接的雲端服務上（僅在應用程式中可見）。
3. **上傳音樂檔案：** 將裝置上的本機音訊檔案上傳到此資料夾。
4. **刪除本機檔案：** 上傳後，如需要，從裝置中刪除原始檔案。
5. **開始串流：** 直接從資料夾播放任何上傳的媒體檔案。

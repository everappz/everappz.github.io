## Arahan Persediaan untuk WD My Cloud Home dalam Evermusic

Ikuti langkah-langkah ini untuk menyambungkan WD My Cloud Home anda ke Evermusic:

1. **Muat Turun Evermusic**  
   Dapatkan aplikasi Evermusic percuma dari [App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198).

2. **Lancarkan Aplikasi**  
   Buka Evermusic di iPhone atau iPad anda.

3. **Langkau Skrin Pengenalan**  
   Leret melalui halaman pengenalan/tutorial sehingga anda sampai ke skrin persediaan terakhir.

4. **Sambung ke Storan Awan**  
   Buka **Sambungan**, ketik **Sambung ke Storan Awan**, ketik **WD My Cloud Home** dari senarai perkhidmatan storan awan yang tersedia.

5. **Masukkan Kelayakan Anda**  
   Log masuk menggunakan kelayakan yang anda buat semasa mendaftarkan peranti WD My Cloud Home anda di [home.mycloud.com](https://home.mycloud.com/).

Setelah selesai, WD My Cloud Home anda akan disambungkan ke Evermusic dan sedia untuk penstriman atau muat turun muzik.

## Cara Memainkan Muzik di iPhone dari WD MyCloud EX2 Ultra, WD MyCloud Mirror, atau Mana-mana Storan WD MyCloud Lain

Apabila anda menambah sambungan dalam aplikasi melalui **Sambungan > Sambung Perkhidmatan Awan > My Cloud Home**, ia direka untuk menyokong peranti WD MyCloud Home secara rasmi.

Walau bagaimanapun, jika anda telah berjaya menyambungkan WD MyCloud EX2 Ultra, WD MyCloud Mirror, atau mana-mana Storan WD MyCloud lain, berikut adalah penyelesaian untuk membantu anda menggunakannya.

Untuk menjadikannya berfungsi, cukup buat folder dalam pengurus fail aplikasi selepas menyambung dan muat naik fail muzik anda ke sana. Nampaknya pelayan WD hanya membenarkan akses ke folder yang dibuat oleh aplikasi dalam "mod kotak pasir" untuk peranti-peranti ini, jadi folder ini hanya boleh diakses dari aplikasi Evermusic.

Sila gunakan pengurus fail Evermusic untuk memuat naik lagu anda ke folder ini. Setelah fail dimuat naik, anda boleh memadamkan salinan tempatan dari peranti anda dan mendengar terus dari NAS anda.

**Langkah-langkah:** 

1. **Sambung ke Storan WD MyCloud:** Pergi ke *Sambungan > Sambung Perkhidmatan Awan > My Cloud Home*.
2. **Buat Folder:** Pada perkhidmatan awan yang disambungkan (hanya kelihatan dalam aplikasi).
3. **Muat Naik Fail Muzik:** Muat naik fail audio tempatan anda dari peranti ke folder ini.
4. **Padam Fail Tempatan:** Setelah dimuat naik, padamkan fail asal dari peranti anda jika perlu.
5. **Mula Menstrim:** Main mana-mana fail media yang dimuat naik terus dari folder.

## FAQ

Evermusic percuma untuk dimuat turun dengan ciri-ciri teras termasuk penyama, penstriman awan, dan main balik luar talian. Versi percuma menyokong sehingga 3 sambungan awan. Menaik taraf ke Premium menghapuskan had pada akaun awan, senarai main, dan folder luar talian.
{{% /details %}}

{{% details title="Bolehkah saya mendengar muzik secara luar talian dari NAS saya?" closed="true" %}}
Ya. Evermusic membolehkan anda memuat turun trek dari WD My Cloud Home ke iPhone anda untuk main balik luar talian. Ini berguna apabila anda melancong atau mempunyai akses internet yang terhad.
{{% /details %}}

{{% details title="Adakah Evermusic menyokong format audio tanpa rugi dari WD My Cloud?" closed="true" %}}
Ya. Evermusic menyokong FLAC, ALAC, WAV, AIFF, dan format tanpa rugi lain. Anda boleh menstrim atau memuat turun fail audio berkualiti tinggi dari NAS anda tanpa penukaran format.
{{% /details %}}

{{% details title="Bolehkah saya menggunakan WD MyCloud EX2 Ultra dengan Evermusic?" closed="true" %}}
Ya, dengan penyelesaian. Sambung melalui pilihan My Cloud Home, buat folder menggunakan pengurus fail Evermusic, dan muat naik fail muzik anda ke sana. Disebabkan mod kotak pasir, hanya fail dalam folder yang dibuat oleh aplikasi yang boleh diakses.
{{% /details %}}

{{% details title="Berapa banyak muzik yang boleh disimpan di WD My Cloud Home?" closed="true" %}}
WD My Cloud Home menyokong sehingga 8 TB storan. Pada kadar bit biasa, ini boleh memuatkan beratus-ratus ribu lagu, termasuk pustaka muzik tanpa rugi yang besar.
{{% /details %}}

{{% details title="Adakah sambungan antara Evermusic dan WD My Cloud Home selamat?" closed="true" %}}
Ya. Evermusic menggunakan sambungan selamat dan API rasmi Western Digital untuk mengakses NAS anda. Data dan kelayakan log masuk anda dilindungi semasa penghantaran.

## Instructions de configuration pour WD My Cloud Home dans Evermusic

Suivez ces étapes pour connecter votre WD My Cloud Home à Evermusic :

1. **Télécharger Evermusic**  
   Obtenez l'application gratuite Evermusic depuis l'[App Store](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198).

2. **Lancer l'application**  
   Ouvrez Evermusic sur votre iPhone ou iPad.

3. **Passer les écrans d'introduction**  
   Faites défiler les pages d'introduction/tutoriel jusqu'à atteindre l'écran de configuration final.

4. **Se connecter au stockage cloud**  
   Ouvrez **Connexions**, appuyez sur **Se connecter au stockage cloud**, appuyez sur **WD My Cloud Home** dans la liste des services de stockage cloud disponibles.

5. **Entrer vos identifiants**  
   Connectez-vous avec les identifiants que vous avez créés lors de l'enregistrement de votre appareil WD My Cloud Home sur [home.mycloud.com](https://home.mycloud.com/).

Une fois terminé, votre WD My Cloud Home sera connecté à Evermusic et prêt pour le streaming ou le téléchargement de musique.

## Comment lire de la musique sur iPhone depuis WD MyCloud EX2 Ultra, WD MyCloud Mirror ou tout autre stockage WD MyCloud

Lorsque vous ajoutez une connexion dans l'application via **Connexions > Connecter un service cloud > My Cloud Home**, elle est conçue pour prendre en charge officiellement les appareils WD MyCloud Home.

Cependant, si vous avez réussi à connecter votre WD MyCloud EX2 Ultra, WD MyCloud Mirror ou tout autre stockage WD MyCloud, voici une solution de contournement pour vous aider à l'utiliser.

Pour que cela fonctionne, créez simplement un dossier dans le gestionnaire de fichiers de l'application après la connexion et téléversez vos fichiers musicaux dedans. Il semble que le serveur de WD ne permette l'accès qu'aux dossiers créés par l'application en "mode sandbox" pour ces appareils, donc ce dossier ne sera accessible que depuis l'application Evermusic.

Veuillez utiliser le gestionnaire de fichiers d'Evermusic pour téléverser vos chansons dans ce dossier. Une fois les fichiers téléversés, vous pouvez supprimer les copies locales de votre appareil et écouter directement depuis votre NAS.

**Étapes :**

1. **Se connecter au stockage WD MyCloud :** Allez dans *Connexions > Connecter un service cloud > My Cloud Home*.
2. **Créer un dossier :** Sur le service cloud connecté (visible uniquement dans l'application).
3. **Téléverser des fichiers musicaux :** Téléversez vos fichiers audio locaux depuis votre appareil vers ce dossier.
4. **Supprimer les fichiers locaux :** Une fois téléversés, supprimez les originaux de votre appareil si nécessaire.
5. **Commencer le streaming :** Lisez n'importe quel fichier multimédia téléversé directement depuis le dossier.

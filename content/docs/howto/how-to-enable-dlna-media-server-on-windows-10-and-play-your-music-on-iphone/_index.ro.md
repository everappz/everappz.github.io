**Pe scurt:** Windows 10 are un server DLNA integrat. Activează-l în setările Rețea și Partajare, apoi folosește aplicația gratuită **Evermusic** pe iPhone pentru a transmite întreaga ta bibliotecă muzicală prin Wi-Fi. Nu este nevoie de software server terț.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Copertă" caption="Streaming muzică DLNA către iPhone folosind Windows 10 și Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) este un instrument puternic care îți permite să transmiți cu ușurință diverse conținuturi media, inclusiv muzică, între dispozitivele compatibile DLNA din rețeaua ta. Vestea bună este că Windows 10, și versiunile anterioare, vin cu o funcție DLNA integrată, eliminând nevoia de servere media terțe. Iată cum să activezi serverul media DLNA pe Windows 10 și să te bucuri de streaming muzical pe iPhone.

## Cum să activezi serverul media DLNA pe Windows 10

1. Fă clic pe butonul 'Start'.  
2. Selectează pictograma 'Setări'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Configurare server" caption="Deschide Setările Windows" width="500" >}}

3. Pe ecranul 'Setări Windows', alege 'Rețea și Internet'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Setări Windows" caption="Selectează Rețea și Internet" width="500" >}}

4. Sub 'Rețea', selectează 'Centru rețea și partajare'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Centru partajare" caption="Deschide Centru rețea și partajare" width="500" >}}

5. Pe ecranul 'Centru rețea și partajare', fă clic pe 'Modificare setări de partajare complexe' din meniul din stânga.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Setări de partajare complexe" caption="Modifică setările de partajare complexe" width="500" >}}

6. Pe ecranul 'Setări de partajare complexe', derulează în jos până la secțiunea 'Toate rețelele' și extinde-o făcând clic pe săgeată.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Activare descoperire" caption="Extinde setările tuturor rețelelor" width="500" >}}

7. Fă clic pe 'Activare streaming media' pentru a activa serverul DLNA.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Activare server" caption="Activează serverul de streaming media" width="500" >}}

8. Dă un nume bibliotecii tale media și selectează dispozitivele cărora li se permite accesul.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Nume bibliotecă media" caption="Denumește biblioteca ta media DLNA" width="500" >}}

9. Fă clic pe 'OK' pentru a confirma. Acum, folderele tale personale precum Muzică, Imagini și Videoclipuri vor fi vizibile pentru orice dispozitiv de streaming cu suport UPnP.

## Cum să dezactivezi serverul media DLNA pe Windows 10

1. Fă clic pe 'Start' și tastează 'services' în câmpul de căutare.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Servicii Windows" caption="Deschide Servicii Windows" width="500" >}}

2. Pe ecranul 'Servicii', derulează pentru a găsi 'Windows Media Player Network Sharing Service'.  
3. Fă dublu clic și setează 'Tipul de pornire' la 'Manual'.  
4. Oprește serviciul făcând clic pe butonul 'Oprire'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="Oprire serviciu DLNA" caption="Dezactivează serviciul de partajare rețea DLNA" width="500" >}}

## Cum să redai muzică de pe serverul DLNA pe iPhone

1. Instalează aplicația gratuită **Evermusic** din App Store:  
   [Aplicația Evermusic](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Deschide fila 'Conexiuni' și atinge 'Dispozitive disponibile' în secțiunea 'Rețea locală'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Conexiuni Evermusic" caption="Evermusic: ecranul Conexiuni" width="500" >}}

3. Așteaptă câteva secunde până se încarcă lista de dispozitive și atinge serverul Windows Media Player DLNA (ex.: 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Dispozitive disponibile" caption="Servere DLNA disponibile în Evermusic" width="500" >}}

4. Vei vedea o listă de foldere disponibile pe serverul media.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Foldere Evermusic" caption="Răsfoiește folderele partajate de pe serverul DLNA" width="500" >}}

5. Deschide orice folder care conține fișiere audio.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Conținut folder" caption="Vezi conținutul folderelor DLNA partajate" width="500" >}}

6. Atinge orice fișier pentru a porni playerul audio.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Player audio" caption="Redă fișier audio de pe DLNA în Evermusic" width="500" >}}

7. Pentru a îmbunătăți experiența audio, atinge pictograma 'Egalizator' lângă indicatorul de volum din partea de jos a ecranului pentru a activa egalizatorul stil iPod cu preamplificator.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Egalizator" caption="Folosește egalizatorul integrat pentru redare îmbunătățită" width="500" >}}

## Concluzie

Cu serverul media DLNA pe Windows 10 și Evermusic pe iPhone, te poți bucura de streaming muzical fără întreruperi de la computer la dispozitivul mobil. Spune adio limitărilor de stocare și bun venit muzicii la cerere!

## Întrebări frecvente

{{% details title="Trebuie să instalez software server pe Windows 10?" closed="true" %}}
Nu. Windows 10 include un server media DLNA integrat. Trebuie doar să activezi streaming-ul media în setările Centrului de rețea și partajare. Nu este necesar software terț.
{{% /details %}}

{{% details title="iPhone-ul meu trebuie să fie pe aceeași rețea Wi-Fi?" closed="true" %}}
Da. Streaming-ul DLNA funcționează prin rețeaua ta locală. Atât PC-ul cu Windows 10, cât și iPhone-ul trebuie să fie conectate la aceeași rețea Wi-Fi pentru ca Evermusic să descopere serverul DLNA.
{{% /details %}}

{{% details title="Ce formate audio pot transmite prin DLNA?" closed="true" %}}
Serverul Windows DLNA partajează fișiere din folderul Muzică indiferent de format. Evermusic suportă MP3, FLAC, AAC, WAV, OGG, AIFF și multe alte formate, astfel încât poți reda practic orice fișier audio de pe server.
{{% /details %}}

{{% details title="Pot folosi Flacbox în loc de Evermusic?" closed="true" %}}
Da. Flacbox suportă de asemenea navigarea și redarea DLNA/UPnP. Poți folosi oricare dintre cele două aplicații pentru a descoperi și reda muzică de pe serverul tău Windows DLNA.
{{% /details %}}

{{% details title="Streaming-ul DLNA folosește date mobile?" closed="true" %}}
Nu. DLNA funcționează exclusiv pe rețeaua ta Wi-Fi locală. Nu folosește date mobile. Cu toate acestea, ambele dispozitive trebuie să rămână conectate la aceeași rețea în timpul redării.
{{% /details %}}

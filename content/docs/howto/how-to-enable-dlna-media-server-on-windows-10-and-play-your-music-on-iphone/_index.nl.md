**Samenvatting:** Windows 10 heeft een ingebouwde DLNA-server. Schakel deze in bij Netwerk- en deelcentrum-instellingen en gebruik vervolgens de gratis **Evermusic**-app op je iPhone om je hele muziekbibliotheek te streamen via Wi-Fi. Geen serversoftware van derden nodig.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Voorblad" caption="DLNA-muziekstreaming naar iPhone met Windows 10 en Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) is een krachtig hulpmiddel waarmee je moeiteloos diverse media-inhoud, waaronder muziek, kunt streamen tussen DLNA-ondersteunde apparaten op je netwerk. Het goede nieuws is dat Windows 10, en eerdere versies, een ingebouwde DLNA-functie bevatten, waardoor mediaservers van derden overbodig worden. Hier lees je hoe je DLNA Media Server inschakelt op Windows 10 en muziekstreaming op je iPhone kunt genieten.

## Hoe DLNA Media Server inschakelen op Windows 10

1. Klik op de knop 'Start'.  
2. Selecteer het pictogram 'Instellingen'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Server instellen" caption="Open Windows Instellingen" width="500" >}}

3. Kies op het scherm 'Windows Instellingen' de optie 'Netwerk en internet'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Windows Instellingen" caption="Selecteer Netwerk en internet" width="500" >}}

4. Selecteer onder 'Netwerk' het 'Netwerkcentrum'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Netwerkcentrum" caption="Open Netwerkcentrum" width="500" >}}

5. Klik op het scherm 'Netwerkcentrum' op 'Geavanceerde instellingen voor delen wijzigen' in het linkermenu.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Geavanceerde instellingen voor delen" caption="Geavanceerde instellingen voor delen wijzigen" width="500" >}}

6. Scrol op het scherm 'Geavanceerde instellingen voor delen' naar het gedeelte 'Alle netwerken' en vouw het uit door op de pijl te klikken.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Detectie inschakelen" caption="Alle netwerkinstellingen uitvouwen" width="500" >}}

7. Klik op 'Mediastreaming inschakelen' om de DLNA-server te activeren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Server inschakelen" caption="Mediastreaming-server inschakelen" width="500" >}}

8. Geef je mediabibliotheek een naam en selecteer de apparaten die toegang mogen hebben.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Naam mediabibliotheek" caption="Geef je DLNA-mediabibliotheek een naam" width="500" >}}

9. Klik op 'OK' om te bevestigen. Nu zijn je persoonlijke mappen zoals Muziek, Afbeeldingen en Video's zichtbaar voor elk streamingapparaat met UPnP-ondersteuning.

## Hoe DLNA Media Server uitschakelen op Windows 10

1. Klik op 'Start' en typ 'services' in het zoekveld.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Windows Services" caption="Open Windows Services" width="500" >}}

2. Scrol op het scherm 'Services' naar beneden om 'Windows Media Player Network Sharing Service' te vinden.  
3. Dubbelklik erop en stel het 'Opstarttype' in op 'Handmatig'.  
4. Stop de service door op de knop 'Stoppen' te klikken.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="DLNA-service stoppen" caption="DLNA-netwerkdelingsservice uitschakelen" width="500" >}}

## Hoe muziek afspelen van DLNA-server op iPhone

1. Installeer de gratis **Evermusic**-app uit de App Store:  
   [Evermusic App](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Open het tabblad 'Verbindingen' en tik op 'Beschikbare apparaten' in het gedeelte 'Lokaal netwerk'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Evermusic Verbindingen" caption="Evermusic: Verbindingenscherm" width="500" >}}

3. Wacht een paar seconden terwijl de apparatenlijst wordt geladen en tik op de Windows Media Player DLNA-server (bijv. 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Beschikbare apparaten" caption="Beschikbare DLNA-servers in Evermusic" width="500" >}}

4. Je ziet een lijst met mappen die beschikbaar zijn op de mediaserver.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Evermusic Mappen" caption="Blader door gedeelde mappen van de DLNA-server" width="500" >}}

5. Open een map met audiobestanden.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Mapinhoud" caption="Bekijk de inhoud van gedeelde DLNA-mappen" width="500" >}}

6. Tik op een bestand om de audiospeler te starten.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Audiospeler" caption="Audiobestand afspelen van DLNA in Evermusic" width="500" >}}

7. Om je audio-ervaring te verbeteren, tik op het 'Equalizer'-pictogram bij de volume-indicator onderaan het scherm om de iPod-stijl equalizer met voorversterker in te schakelen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Equalizer" caption="Gebruik de ingebouwde equalizer voor verbeterde weergave" width="500" >}}

## Conclusie

Met DLNA Media Server op Windows 10 en Evermusic op je iPhone kun je genieten van naadloze muziekstreaming van je computer naar je mobiele apparaat. Zeg vaarwel tegen opslagbeperkingen en hallo tegen muziek op aanvraag!

## Veelgestelde vragen

{{% details title="Moet ik serversoftware installeren op Windows 10?" closed="true" %}}
Nee. Windows 10 bevat een ingebouwde DLNA-mediaserver. Je hoeft alleen mediastreaming in te schakelen in de instellingen van het Netwerkcentrum. Geen software van derden vereist.
{{% /details %}}

{{% details title="Moet mijn iPhone op hetzelfde Wi-Fi-netwerk zijn?" closed="true" %}}
Ja. DLNA-streaming werkt via je lokale netwerk. Zowel je Windows 10-pc als je iPhone moeten verbonden zijn met hetzelfde Wi-Fi-netwerk zodat Evermusic de DLNA-server kan ontdekken.
{{% /details %}}

{{% details title="Welke audioformaten kan ik streamen via DLNA?" closed="true" %}}
De Windows DLNA-server deelt bestanden uit je Muziekmap ongeacht het formaat. Evermusic ondersteunt MP3, FLAC, AAC, WAV, OGG, AIFF en vele andere formaten, zodat je vrijwel elk audiobestand van de server kunt afspelen.
{{% /details %}}

{{% details title="Kan ik Flacbox gebruiken in plaats van Evermusic?" closed="true" %}}
Ja. Flacbox ondersteunt ook DLNA/UPnP-browsen en afspelen. Je kunt beide apps gebruiken om muziek te ontdekken en af te spelen van je Windows DLNA-server.
{{% /details %}}

{{% details title="Gebruikt DLNA-streaming mobiele data?" closed="true" %}}
Nee. DLNA werkt volledig op je lokale Wi-Fi-netwerk. Het gebruikt geen mobiele data. Beide apparaten moeten echter verbonden blijven met hetzelfde netwerk tijdens het afspelen.
{{% /details %}}

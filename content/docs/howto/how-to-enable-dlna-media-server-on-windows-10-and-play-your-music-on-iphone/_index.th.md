**สรุป:** Windows 10 มีเซิร์ฟเวอร์ DLNA ในตัว เปิดใช้งานในการตั้งค่าเครือข่ายและการแชร์ จากนั้นใช้แอป **Evermusic** ฟรีบน iPhone เพื่อสตรีมคลังเพลงทั้งหมดผ่าน Wi-Fi ไม่ต้องใช้ซอฟต์แวร์เซิร์ฟเวอร์จากบุคคลที่สาม

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: หน้าปก" caption="สตรีมเพลง DLNA ไปยัง iPhone โดยใช้ Windows 10 และ Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) เป็นเครื่องมือที่ทรงพลังที่ช่วยให้คุณสตรีมเนื้อหาสื่อต่างๆ รวมถึงเพลง ระหว่างอุปกรณ์ที่รองรับ DLNA ในเครือข่ายของคุณได้อย่างง่ายดาย ข่าวดีคือ Windows 10 และเวอร์ชันก่อนหน้ามีคุณสมบัติ DLNA ในตัว ทำให้ไม่จำเป็นต้องใช้เซิร์ฟเวอร์มีเดียจากบุคคลที่สาม นี่คือวิธีเปิดใช้งาน DLNA Media Server บน Windows 10 และเพลิดเพลินกับการสตรีมเพลงบน iPhone

## วิธีเปิดใช้งาน DLNA Media Server บน Windows 10

1. คลิกปุ่ม 'Start'  
2. เลือกไอคอน 'Settings'

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="ตั้งค่าเซิร์ฟเวอร์" caption="เปิดการตั้งค่า Windows" width="500" >}}

3. บนหน้าจอ 'Windows Settings' เลือก 'Network & Internet'

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="การตั้งค่า Windows" caption="เลือก Network & Internet" width="500" >}}

4. ภายใต้ 'Network' เลือก 'Network and Sharing Center'

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="ศูนย์การแชร์" caption="เปิด Network and Sharing Center" width="500" >}}

5. บนหน้าจอ 'Network and Sharing Center' คลิก 'Change advanced sharing settings' ในเมนูด้านซ้าย

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="การตั้งค่าการแชร์ขั้นสูง" caption="เปลี่ยนการตั้งค่าการแชร์ขั้นสูง" width="500" >}}

6. บนหน้าจอ 'Advanced sharing settings' เลื่อนลงไปที่ส่วน 'All Networks' และขยายโดยคลิกลูกศร

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="เปิดการค้นหา" caption="ขยายการตั้งค่าเครือข่ายทั้งหมด" width="500" >}}

7. คลิก 'Turn on media streaming' เพื่อเปิดใช้งานเซิร์ฟเวอร์ DLNA

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="เปิดเซิร์ฟเวอร์" caption="เปิดใช้งานเซิร์ฟเวอร์สตรีมมีเดีย" width="500" >}}

8. ตั้งชื่อคลังสื่อของคุณและเลือกอุปกรณ์ที่อนุญาตให้เข้าถึง

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="ชื่อคลังสื่อ" caption="ตั้งชื่อคลังสื่อ DLNA ของคุณ" width="500" >}}

9. คลิก 'OK' เพื่อยืนยัน ตอนนี้โฟลเดอร์ส่วนตัวของคุณ เช่น เพลง รูปภาพ และวิดีโอ จะมองเห็นได้จากอุปกรณ์สตรีมมิ่งที่รองรับ UPnP

## วิธีปิดใช้งาน DLNA Media Server บน Windows 10

1. คลิก 'Start' และพิมพ์ 'services' ในช่องค้นหา

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="บริการ Windows" caption="เปิดบริการ Windows" width="500" >}}

2. บนหน้าจอ 'Services' เลื่อนลงเพื่อค้นหา 'Windows Media Player Network Sharing Service'  
3. ดับเบิลคลิกและตั้งค่า 'Startup type' เป็น 'Manual'  
4. หยุดบริการโดยคลิกปุ่ม 'Stop'

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="หยุดบริการ DLNA" caption="ปิดใช้งานบริการแชร์เครือข่าย DLNA" width="500" >}}

## วิธีเล่นเพลงจากเซิร์ฟเวอร์ DLNA บน iPhone

1. ติดตั้งแอป **Evermusic** ฟรีจาก App Store:  
   [แอป Evermusic](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. เปิดแท็บ 'การเชื่อมต่อ' แล้วแตะที่ 'อุปกรณ์ที่พร้อมใช้งาน' ในส่วน 'เครือข่ายท้องถิ่น'

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="การเชื่อมต่อ Evermusic" caption="Evermusic: หน้าจอการเชื่อมต่อ" width="500" >}}

3. รอสักครู่ขณะที่รายการอุปกรณ์โหลด แล้วแตะเซิร์ฟเวอร์ Windows Media Player DLNA (เช่น 'MSEDGEWIN10: My Windows Library')

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="อุปกรณ์ที่พร้อมใช้งาน" caption="เซิร์ฟเวอร์ DLNA ที่พร้อมใช้งานใน Evermusic" width="500" >}}

4. คุณจะเห็นรายการโฟลเดอร์ที่มีอยู่บนเซิร์ฟเวอร์มีเดีย

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="โฟลเดอร์ Evermusic" caption="เรียกดูโฟลเดอร์ที่แชร์จากเซิร์ฟเวอร์ DLNA" width="500" >}}

5. เปิดโฟลเดอร์ที่มีไฟล์เสียง

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="เนื้อหาโฟลเดอร์" caption="ดูเนื้อหาของโฟลเดอร์ DLNA ที่แชร์" width="500" >}}

6. แตะไฟล์ใดก็ได้เพื่อเริ่มเครื่องเล่นเสียง

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="เครื่องเล่นเสียง" caption="เล่นไฟล์เสียงจาก DLNA ใน Evermusic" width="500" >}}

7. เพื่อเพิ่มประสบการณ์เสียง แตะไอคอน 'อีควอไลเซอร์' ใกล้ตัวบ่งชี้ระดับเสียงที่ด้านล่างของหน้าจอเพื่อเปิดใช้งานอีควอไลเซอร์สไตล์ iPod พร้อมพรีแอมพลิไฟเออร์

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="อีควอไลเซอร์" caption="ใช้อีควอไลเซอร์ในตัวเพื่อการเล่นที่ดีขึ้น" width="500" >}}

## สรุป

ด้วย DLNA Media Server บน Windows 10 และ Evermusic บน iPhone คุณสามารถเพลิดเพลินกับการสตรีมเพลงอย่างราบรื่นจากคอมพิวเตอร์ไปยังอุปกรณ์มือถือ ลาก่อนข้อจำกัดพื้นที่จัดเก็บ สวัสดีเพลงตามต้องการ!

## คำถามที่พบบ่อย

{{% details title="ฉันต้องติดตั้งซอฟต์แวร์เซิร์ฟเวอร์บน Windows 10 หรือไม่?" closed="true" %}}
ไม่ Windows 10 มีเซิร์ฟเวอร์มีเดีย DLNA ในตัว คุณเพียงแค่ต้องเปิดใช้งานการสตรีมมีเดียในการตั้งค่า Network and Sharing Center ไม่ต้องใช้ซอฟต์แวร์จากบุคคลที่สาม
{{% /details %}}

{{% details title="iPhone ต้องอยู่ในเครือข่าย Wi-Fi เดียวกันหรือไม่?" closed="true" %}}
ใช่ การสตรีม DLNA ทำงานผ่านเครือข่ายท้องถิ่นของคุณ ทั้ง PC Windows 10 และ iPhone ต้องเชื่อมต่อกับเครือข่าย Wi-Fi เดียวกันเพื่อให้ Evermusic ค้นพบเซิร์ฟเวอร์ DLNA
{{% /details %}}

{{% details title="ฉันสามารถสตรีมรูปแบบเสียงอะไรได้บ้างผ่าน DLNA?" closed="true" %}}
เซิร์ฟเวอร์ Windows DLNA แชร์ไฟล์จากโฟลเดอร์เพลงโดยไม่คำนึงถึงรูปแบบ Evermusic รองรับ MP3, FLAC, AAC, WAV, OGG, AIFF และรูปแบบอื่นๆ อีกมากมาย ดังนั้นคุณสามารถเล่นไฟล์เสียงเกือบทุกชนิดจากเซิร์ฟเวอร์ได้
{{% /details %}}

{{% details title="ฉันสามารถใช้ Flacbox แทน Evermusic ได้หรือไม่?" closed="true" %}}
ได้ Flacbox ยังรองรับการเรียกดูและเล่น DLNA/UPnP คุณสามารถใช้แอปใดแอปหนึ่งเพื่อค้นหาและเล่นเพลงจากเซิร์ฟเวอร์ Windows DLNA ของคุณ
{{% /details %}}

{{% details title="การสตรีม DLNA ใช้ข้อมูลมือถือหรือไม่?" closed="true" %}}
ไม่ DLNA ทำงานบนเครือข่าย Wi-Fi ท้องถิ่นเท่านั้น ไม่ใช้ข้อมูลมือถือ อย่างไรก็ตาม อุปกรณ์ทั้งสองต้องเชื่อมต่อกับเครือข่ายเดียวกันระหว่างการเล่น
{{% /details %}}

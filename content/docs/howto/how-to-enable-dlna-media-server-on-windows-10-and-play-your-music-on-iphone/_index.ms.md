**Ringkasan:** Windows 10 mempunyai pelayan DLNA terbina dalam. Aktifkan ia dalam tetapan Rangkaian dan Perkongsian, kemudian gunakan aplikasi percuma **Evermusic** pada iPhone anda untuk menstrim seluruh pustaka muzik melalui Wi-Fi. Tiada perisian pelayan pihak ketiga diperlukan.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Muka Depan" caption="Penstriman muzik DLNA ke iPhone menggunakan Windows 10 dan Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) ialah alat berkuasa yang membolehkan anda menstrim pelbagai kandungan media dengan mudah, termasuk muzik, antara peranti yang menyokong DLNA pada rangkaian anda. Berita baiknya ialah Windows 10, dan versi sebelumnya, dilengkapi dengan ciri DLNA terbina dalam, menghapuskan keperluan pelayan media pihak ketiga. Berikut cara mengaktifkan Pelayan Media DLNA pada Windows 10 dan menikmati penstriman muzik pada iPhone anda.

## Cara Mengaktifkan Pelayan Media DLNA pada Windows 10

1. Klik butang 'Start'.  
2. Pilih ikon 'Tetapan'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Persediaan Pelayan" caption="Buka Tetapan Windows" width="500" >}}

3. Pada skrin 'Tetapan Windows', pilih 'Rangkaian & Internet'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Tetapan Windows" caption="Pilih Rangkaian & Internet" width="500" >}}

4. Di bawah 'Rangkaian', pilih 'Pusat Rangkaian dan Perkongsian'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Pusat Perkongsian" caption="Buka Pusat Rangkaian dan Perkongsian" width="500" >}}

5. Pada skrin 'Pusat Rangkaian dan Perkongsian', klik 'Tukar tetapan perkongsian lanjutan' di menu kiri.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Tetapan Perkongsian Lanjutan" caption="Tukar tetapan perkongsian lanjutan" width="500" >}}

6. Pada skrin 'Tetapan perkongsian lanjutan', tatal ke bawah ke bahagian 'Semua Rangkaian' dan kembangkan dengan mengklik anak panah.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Hidupkan Penemuan" caption="Kembangkan tetapan semua rangkaian" width="500" >}}

7. Klik 'Hidupkan penstriman media' untuk mengaktifkan pelayan DLNA.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Hidupkan Pelayan" caption="Aktifkan pelayan penstriman media" width="500" >}}

8. Berikan nama kepada pustaka media anda dan pilih peranti yang dibenarkan untuk mengaksesnya.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Nama Pustaka Media" caption="Namakan pustaka media DLNA anda" width="500" >}}

9. Klik 'OK' untuk mengesahkan. Kini, folder peribadi anda seperti Muzik, Gambar dan Video akan kelihatan kepada mana-mana peranti penstriman dengan sokongan UPnP.

## Cara Menyahaktifkan Pelayan Media DLNA pada Windows 10

1. Klik 'Start' dan taip 'services' dalam medan carian.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Perkhidmatan Windows" caption="Buka Perkhidmatan Windows" width="500" >}}

2. Pada skrin 'Perkhidmatan', tatal ke bawah untuk mencari 'Windows Media Player Network Sharing Service'.  
3. Klik dua kali dan tetapkan 'Jenis permulaan' kepada 'Manual'.  
4. Hentikan perkhidmatan dengan mengklik butang 'Berhenti'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="Hentikan Perkhidmatan DLNA" caption="Nyahaktifkan perkhidmatan perkongsian rangkaian DLNA" width="500" >}}

## Cara Memainkan Muzik dari Pelayan DLNA pada iPhone

1. Pasang aplikasi percuma **Evermusic** dari App Store:  
   [Aplikasi Evermusic](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Buka tab 'Sambungan' dan ketik pada item 'Peranti tersedia' dalam bahagian 'Rangkaian Tempatan'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Sambungan Evermusic" caption="Evermusic: skrin Sambungan" width="500" >}}

3. Tunggu beberapa saat sementara senarai peranti dimuat dan ketik pada pelayan Windows Media Player DLNA (cth., 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Peranti Tersedia" caption="Pelayan DLNA tersedia dalam Evermusic" width="500" >}}

4. Anda akan melihat senarai folder yang tersedia pada pelayan media.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Folder Evermusic" caption="Layari folder yang dikongsi dari pelayan DLNA" width="500" >}}

5. Buka mana-mana folder yang mengandungi fail audio.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Kandungan Folder" caption="Lihat kandungan folder DLNA yang dikongsi" width="500" >}}

6. Ketik pada mana-mana fail untuk memulakan pemain audio.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Pemain Audio" caption="Mainkan fail audio dari DLNA dalam Evermusic" width="500" >}}

7. Untuk meningkatkan pengalaman audio anda, ketik ikon 'Penyama' berhampiran penunjuk kelantangan di bahagian bawah skrin untuk mengaktifkan penyama gaya iPod dengan praamplifier.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Penyama" caption="Gunakan penyama terbina dalam untuk main balik yang lebih baik" width="500" >}}

## Kesimpulan

Dengan Pelayan Media DLNA pada Windows 10 dan Evermusic pada iPhone anda, anda boleh menikmati penstriman muzik yang lancar dari komputer ke peranti mudah alih anda. Ucapkan selamat tinggal kepada had storan dan selamat datang muzik atas permintaan!

## Soalan Lazim

{{% details title="Adakah saya perlu memasang perisian pelayan pada Windows 10?" closed="true" %}}
Tidak. Windows 10 sudah menyertakan pelayan media DLNA terbina dalam. Anda hanya perlu mengaktifkan penstriman media dalam tetapan Pusat Rangkaian dan Perkongsian. Tiada perisian pihak ketiga diperlukan.
{{% /details %}}

{{% details title="Adakah iPhone saya perlu berada pada rangkaian Wi-Fi yang sama?" closed="true" %}}
Ya. Penstriman DLNA berfungsi melalui rangkaian tempatan anda. Kedua-dua PC Windows 10 dan iPhone anda mesti disambungkan ke rangkaian Wi-Fi yang sama untuk Evermusic menemui pelayan DLNA.
{{% /details %}}

{{% details title="Format audio apa yang boleh saya strim melalui DLNA?" closed="true" %}}
Pelayan Windows DLNA berkongsi fail dari folder Muzik anda tanpa mengira format. Evermusic menyokong MP3, FLAC, AAC, WAV, OGG, AIFF dan banyak format lain, jadi anda boleh memainkan hampir mana-mana fail audio dari pelayan.
{{% /details %}}

{{% details title="Bolehkah saya menggunakan Flacbox sebagai ganti Evermusic?" closed="true" %}}
Ya. Flacbox juga menyokong pelayaran dan main balik DLNA/UPnP. Anda boleh menggunakan mana-mana aplikasi untuk menemui dan memainkan muzik dari pelayan Windows DLNA anda.
{{% /details %}}

{{% details title="Adakah penstriman DLNA menggunakan data mudah alih?" closed="true" %}}
Tidak. DLNA beroperasi sepenuhnya pada rangkaian Wi-Fi tempatan anda. Ia tidak menggunakan sebarang data mudah alih. Walau bagaimanapun, kedua-dua peranti mesti kekal disambungkan ke rangkaian yang sama semasa main balik.
{{% /details %}}

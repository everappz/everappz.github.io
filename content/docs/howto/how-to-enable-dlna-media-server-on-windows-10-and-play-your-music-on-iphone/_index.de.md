**Kurzfassung:** Windows 10 hat einen integrierten DLNA-Server. Aktivieren Sie ihn in den Netzwerk- und Freigabeeinstellungen und verwenden Sie dann die kostenlose **Evermusic**-App auf Ihrem iPhone, um Ihre gesamte Musikbibliothek über Wi-Fi zu streamen. Keine Drittanbieter-Serversoftware erforderlich.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Titelseite" caption="DLNA-Musikstreaming zum iPhone mit Windows 10 und Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) ist ein leistungsstarkes Tool, mit dem Sie verschiedene Medieninhalte, einschließlich Musik, mühelos zwischen DLNA-fähigen Geräten in Ihrem Netzwerk streamen können. Die gute Nachricht ist, dass Windows 10 und frühere Versionen eine integrierte DLNA-Funktion mitbringen, wodurch Medienserver von Drittanbietern überflüssig werden. Hier erfahren Sie, wie Sie den DLNA Media Server unter Windows 10 aktivieren und Musikstreaming auf Ihrem iPhone genießen.

## So aktivieren Sie den DLNA Media Server unter Windows 10

1. Klicken Sie auf die Schaltfläche 'Start'.  
2. Wählen Sie das Symbol 'Einstellungen'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Server einrichten" caption="Windows-Einstellungen öffnen" width="500" >}}

3. Wählen Sie auf dem Bildschirm 'Windows-Einstellungen' den Punkt 'Netzwerk und Internet'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Windows-Einstellungen" caption="Netzwerk und Internet auswählen" width="500" >}}

4. Wählen Sie unter 'Netzwerk' das 'Netzwerk- und Freigabecenter'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Freigabecenter" caption="Netzwerk- und Freigabecenter öffnen" width="500" >}}

5. Klicken Sie auf dem Bildschirm 'Netzwerk- und Freigabecenter' auf 'Erweiterte Freigabeeinstellungen ändern' im linken Menü.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Erweiterte Freigabeeinstellungen" caption="Erweiterte Freigabeeinstellungen ändern" width="500" >}}

6. Scrollen Sie auf dem Bildschirm 'Erweiterte Freigabeeinstellungen' zum Abschnitt 'Alle Netzwerke' und erweitern Sie ihn durch Klicken auf den Pfeil.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Erkennung aktivieren" caption="Alle Netzwerkeinstellungen erweitern" width="500" >}}

7. Klicken Sie auf 'Medienstreaming aktivieren', um den DLNA-Server zu aktivieren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Server aktivieren" caption="Medienstreaming-Server aktivieren" width="500" >}}

8. Geben Sie Ihrer Medienbibliothek einen Namen und wählen Sie die Geräte aus, die darauf zugreifen dürfen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Name der Medienbibliothek" caption="Benennen Sie Ihre DLNA-Medienbibliothek" width="500" >}}

9. Klicken Sie auf 'OK', um den Vorgang zu bestätigen. Nun sind Ihre persönlichen Ordner wie Musik, Bilder und Videos für alle Streaming-Geräte mit UPnP-Unterstützung sichtbar.

## So deaktivieren Sie den DLNA Media Server unter Windows 10

1. Klicken Sie auf 'Start' und geben Sie 'services' in das Suchfeld ein.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Windows-Dienste" caption="Windows-Dienste öffnen" width="500" >}}

2. Scrollen Sie auf dem Bildschirm 'Dienste' nach unten, um den 'Windows Media Player-Netzwerkfreigabedienst' zu finden.  
3. Doppelklicken Sie darauf und setzen Sie den 'Starttyp' auf 'Manuell'.  
4. Stoppen Sie den Dienst, indem Sie auf die Schaltfläche 'Beenden' klicken.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="DLNA-Dienst stoppen" caption="DLNA-Netzwerkfreigabedienst deaktivieren" width="500" >}}

## So spielen Sie Musik vom DLNA-Server auf dem iPhone ab

1. Installieren Sie die kostenlose **Evermusic**-App aus dem App Store:  
   [Evermusic App](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Öffnen Sie den Tab 'Verbindungen' und tippen Sie auf 'Verfügbare Geräte' im Abschnitt 'Lokales Netzwerk'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Evermusic Verbindungen" caption="Evermusic: Verbindungsbildschirm" width="500" >}}

3. Warten Sie einige Sekunden, während die Geräteliste geladen wird, und tippen Sie auf den Windows Media Player DLNA-Server (z.B. 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Verfügbare Geräte" caption="Verfügbare DLNA-Server in Evermusic" width="500" >}}

4. Sie sehen eine Liste der auf dem Medienserver verfügbaren Ordner.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Evermusic Ordner" caption="Freigegebene Ordner vom DLNA-Server durchsuchen" width="500" >}}

5. Öffnen Sie einen beliebigen Ordner mit Audiodateien.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Ordnerinhalt" caption="Inhalte freigegebener DLNA-Ordner anzeigen" width="500" >}}

6. Tippen Sie auf eine beliebige Datei, um den Audioplayer zu starten.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Audioplayer" caption="Audiodatei von DLNA in Evermusic abspielen" width="500" >}}

7. Um Ihr Klangerlebnis zu verbessern, tippen Sie auf das 'Equalizer'-Symbol neben der Lautstärkeanzeige am unteren Bildschirmrand, um den iPod-ähnlichen Equalizer mit Vorverstärker zu aktivieren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Equalizer" caption="Verwenden Sie den integrierten Equalizer für verbesserte Wiedergabe" width="500" >}}

## Fazit

Mit dem DLNA Media Server unter Windows 10 und Evermusic auf Ihrem iPhone können Sie nahtloses Musikstreaming von Ihrem Computer auf Ihr Mobilgerät genießen. Verabschieden Sie sich von Speicherbeschränkungen und begrüßen Sie Musik auf Abruf!

## Häufig gestellte Fragen

{{% details title="Muss ich Serversoftware auf Windows 10 installieren?" closed="true" %}}
Nein. Windows 10 enthält einen integrierten DLNA-Medienserver. Sie müssen nur das Medienstreaming in den Einstellungen des Netzwerk- und Freigabecenters aktivieren. Keine Drittanbieter-Software erforderlich.
{{% /details %}}

{{% details title="Muss mein iPhone im selben Wi-Fi-Netzwerk sein?" closed="true" %}}
Ja. DLNA-Streaming funktioniert über Ihr lokales Netzwerk. Sowohl Ihr Windows 10-PC als auch Ihr iPhone müssen mit demselben Wi-Fi-Netzwerk verbunden sein, damit Evermusic den DLNA-Server erkennen kann.
{{% /details %}}

{{% details title="Welche Audioformate kann ich über DLNA streamen?" closed="true" %}}
Der Windows DLNA-Server gibt Dateien aus Ihrem Musikordner unabhängig vom Format frei. Evermusic unterstützt MP3, FLAC, AAC, WAV, OGG, AIFF und viele weitere Formate, sodass Sie praktisch jede Audiodatei vom Server abspielen können.
{{% /details %}}

{{% details title="Kann ich Flacbox anstelle von Evermusic verwenden?" closed="true" %}}
Ja. Flacbox unterstützt ebenfalls DLNA/UPnP-Browsing und -Wiedergabe. Sie können beide Apps verwenden, um Musik von Ihrem Windows DLNA-Server zu entdecken und abzuspielen.
{{% /details %}}

{{% details title="Verbraucht DLNA-Streaming mobile Daten?" closed="true" %}}
Nein. DLNA arbeitet ausschließlich über Ihr lokales Wi-Fi-Netzwerk. Es verbraucht keine mobilen Daten. Beide Geräte müssen jedoch während der Wiedergabe mit demselben Netzwerk verbunden bleiben.
{{% /details %}}

**Kort fortalt:** Windows 10 har en innebygd DLNA-server. Aktiver den i Nettverks- og delingsinnstillinger, og bruk deretter den gratis **Evermusic**-appen på iPhone for å strømme hele musikkbiblioteket ditt over Wi-Fi. Ingen tredjeparts serverprogramvare nødvendig.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Forside" caption="DLNA-musikkstrømming til iPhone med Windows 10 og Evermusic" width="800" >}}

DLNA (Digital Living Network Alliance) er et kraftig verktøy som lar deg enkelt strømme ulike medieinnhold, inkludert musikk, mellom DLNA-støttede enheter på nettverket ditt. Den gode nyheten er at Windows 10, og tidligere versjoner, kommer med en innebygd DLNA-funksjon, noe som eliminerer behovet for tredjeparts medieservere. Her er hvordan du aktiverer DLNA Media Server på Windows 10 og nyter musikkstrømming på iPhone.

## Slik aktiverer du DLNA Media Server på Windows 10

1. Klikk på 'Start'-knappen.  
2. Velg 'Innstillinger'-ikonet.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Serveroppsett" caption="Åpne Windows-innstillinger" width="500" >}}

3. På 'Windows-innstillinger'-skjermen velger du 'Nettverk og Internett'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Windows-innstillinger" caption="Velg Nettverk og Internett" width="500" >}}

4. Under 'Nettverk' velger du 'Nettverks- og delingssenter'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Delingssenter" caption="Åpne Nettverks- og delingssenter" width="500" >}}

5. På 'Nettverks- og delingssenter'-skjermen klikker du på 'Endre avanserte delingsinnstillinger' i venstremenyen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Avanserte delingsinnstillinger" caption="Endre avanserte delingsinnstillinger" width="500" >}}

6. På 'Avanserte delingsinnstillinger'-skjermen blar du ned til 'Alle nettverk'-seksjonen og utvider den ved å klikke på pilen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Slå på oppdagelse" caption="Utvid alle nettverksinnstillinger" width="500" >}}

7. Klikk 'Slå på mediestrømming' for å aktivere DLNA-serveren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Slå på server" caption="Aktiver mediestrømmingsserver" width="500" >}}

8. Gi mediebiblioteket ditt et navn og velg enhetene som har tilgang til det.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Mediebiblioteksnavn" caption="Gi DLNA-mediebiblioteket ditt et navn" width="500" >}}

9. Klikk 'OK' for å bekrefte. Nå vil dine personlige mapper som Musikk, Bilder og Videoer være synlige for alle strømmingsenheter med UPnP-støtte.

## Slik deaktiverer du DLNA Media Server på Windows 10

1. Klikk 'Start' og skriv 'services' i søkefeltet.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Windows-tjenester" caption="Åpne Windows-tjenester" width="500" >}}

2. På 'Tjenester'-skjermen blar du ned for å finne 'Windows Media Player Network Sharing Service'.  
3. Dobbeltklikk på den og sett 'Oppstartstype' til 'Manuell'.  
4. Stopp tjenesten ved å klikke på 'Stopp'-knappen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="Stopp DLNA-tjeneste" caption="Deaktiver DLNA-nettverksdelingstjeneste" width="500" >}}

## Slik spiller du musikk fra DLNA-server på iPhone

1. Installer den gratis **Evermusic**-appen fra App Store:  
   [Evermusic App](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Åpne 'Tilkoblinger'-fanen og trykk på 'Tilgjengelige enheter' i 'Lokalt nettverk'-seksjonen.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Evermusic-tilkoblinger" caption="Evermusic: Tilkoblingsskjerm" width="500" >}}

3. Vent noen sekunder mens enhetslisten lastes og trykk på Windows Media Player DLNA-serveren (f.eks. 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Tilgjengelige enheter" caption="Tilgjengelige DLNA-servere i Evermusic" width="500" >}}

4. Du vil se en liste over mapper tilgjengelige på medieserveren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Evermusic-mapper" caption="Bla gjennom delte mapper fra DLNA-server" width="500" >}}

5. Åpne en mappe som inneholder lydfiler.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Mappeinnhold" caption="Se innholdet i delte DLNA-mapper" width="500" >}}

6. Trykk på en fil for å starte lydspilleren.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Lydspiller" caption="Spill av lydfil fra DLNA i Evermusic" width="500" >}}

7. For å forbedre lydopplevelsen, trykk på 'Equalizer'-ikonet nær volumindikatoren nederst på skjermen for å aktivere iPod-stil equalizeren med forforsterker.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Equalizer" caption="Bruk den innebygde equalizeren for forbedret avspilling" width="500" >}}

## Konklusjon

Med DLNA Media Server på Windows 10 og Evermusic på iPhone kan du nyte sømløs musikkstrømming fra datamaskinen til mobilenheten din. Si farvel til lagringsbegrensninger og hei til musikk på forespørsel!

## Vanlige spørsmål

{{% details title="Trenger jeg å installere serverprogramvare på Windows 10?" closed="true" %}}
Nei. Windows 10 inkluderer en innebygd DLNA-medieserver. Du trenger bare å aktivere mediestrømming i Nettverks- og delingssenter-innstillingene. Ingen tredjeparts programvare nødvendig.
{{% /details %}}

{{% details title="Må iPhonen min være på samme Wi-Fi-nettverk?" closed="true" %}}
Ja. DLNA-strømming fungerer over ditt lokale nettverk. Både Windows 10-PCen og iPhonen din må være koblet til samme Wi-Fi-nettverk for at Evermusic skal oppdage DLNA-serveren.
{{% /details %}}

{{% details title="Hvilke lydformater kan jeg strømme via DLNA?" closed="true" %}}
Windows DLNA-serveren deler filer fra Musikk-mappen uavhengig av format. Evermusic støtter MP3, FLAC, AAC, WAV, OGG, AIFF og mange andre formater, slik at du kan spille av praktisk talt alle lydfiler fra serveren.
{{% /details %}}

{{% details title="Kan jeg bruke Flacbox i stedet for Evermusic?" closed="true" %}}
Ja. Flacbox støtter også DLNA/UPnP-surfing og avspilling. Du kan bruke begge appene til å oppdage og spille musikk fra Windows DLNA-serveren din.
{{% /details %}}

{{% details title="Bruker DLNA-strømming mobildata?" closed="true" %}}
Nei. DLNA opererer utelukkende på ditt lokale Wi-Fi-nettverk. Det bruker ingen mobildata. Begge enhetene må imidlertid forbli koblet til samme nettverk under avspilling.
{{% /details %}}

**Περίληψη:** Τα Windows 10 διαθέτουν ενσωματωμένο DLNA server. Ενεργοποιήστε τον στις ρυθμίσεις Δικτύου και Κοινής Χρήσης, και στη συνέχεια χρησιμοποιήστε τη δωρεάν εφαρμογή **Evermusic** στο iPhone σας για να κάνετε streaming ολόκληρης της μουσικής βιβλιοθήκης σας μέσω Wi-Fi. Δεν απαιτείται λογισμικό server τρίτων.

{{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e902db0c9a4c45af9772ce000ef5891e~mv2.jpg" alt="Evermusic + Windows 10 DLNA: Εξώφυλλο" caption="Streaming μουσικής μέσω DLNA στο iPhone χρησιμοποιώντας Windows 10 και Evermusic" width="800" >}}

Το DLNA (Digital Living Network Alliance) είναι ένα ισχυρό εργαλείο που σας επιτρέπει να κάνετε εύκολα streaming διάφορου πολυμεσικού περιεχομένου, συμπεριλαμβανομένης της μουσικής, μεταξύ συσκευών με υποστήριξη DLNA στο δίκτυό σας. Τα καλά νέα είναι ότι τα Windows 10, καθώς και προηγούμενες εκδόσεις, διαθέτουν ενσωματωμένη λειτουργία DLNA, εξαλείφοντας την ανάγκη για media servers τρίτων. Δείτε πώς να ενεργοποιήσετε τον DLNA Media Server στα Windows 10 και να απολαύσετε streaming μουσικής στο iPhone σας.

## Πώς να ενεργοποιήσετε τον DLNA Media Server στα Windows 10

1. Κάντε κλικ στο κουμπί 'Έναρξη'.  
2. Επιλέξτε το εικονίδιο 'Ρυθμίσεις'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_c3a96f130f03415a82559099461d4134~mv2.jpg" alt="Ρύθμιση Server" caption="Ανοίξτε τις Ρυθμίσεις Windows" width="500" >}}

3. Στην οθόνη 'Ρυθμίσεις Windows', επιλέξτε 'Δίκτυο και Internet'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_1a9a141499644b399f3b71ffab5952b1~mv2.jpg" alt="Ρυθμίσεις Windows" caption="Επιλέξτε Δίκτυο και Internet" width="500" >}}

4. Στην ενότητα 'Δίκτυο', επιλέξτε 'Κέντρο δικτύου και κοινής χρήσης'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_65d1517f23c54a33a2c65dcdf391e518~mv2.jpg" alt="Κέντρο κοινής χρήσης" caption="Ανοίξτε το Κέντρο δικτύου και κοινής χρήσης" width="500" >}}

5. Στην οθόνη 'Κέντρο δικτύου και κοινής χρήσης', κάντε κλικ στο 'Αλλαγή σύνθετων ρυθμίσεων κοινής χρήσης' στο αριστερό μενού.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_4f1cdb9a11554cf098dede594121f395~mv2.jpg" alt="Σύνθετες ρυθμίσεις κοινής χρήσης" caption="Αλλαγή σύνθετων ρυθμίσεων κοινής χρήσης" width="500" >}}

6. Στην οθόνη 'Σύνθετες ρυθμίσεις κοινής χρήσης', κάντε κύλιση προς τα κάτω στην ενότητα 'Όλα τα δίκτυα' και αναπτύξτε την κάνοντας κλικ στο βέλος.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_e7f28734af9e43d2ae949abce24f772e~mv2.jpg" alt="Ενεργοποίηση ανίχνευσης" caption="Ανάπτυξη ρυθμίσεων όλων των δικτύων" width="500" >}}

7. Κάντε κλικ στο 'Ενεργοποίηση streaming πολυμέσων' για να ενεργοποιήσετε τον DLNA server.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_cbbd6a55ea484b6f861500dda6a87c10~mv2.jpg" alt="Ενεργοποίηση Server" caption="Ενεργοποίηση server streaming πολυμέσων" width="500" >}}

8. Δώστε ένα όνομα στη βιβλιοθήκη πολυμέσων σας και επιλέξτε τις συσκευές που επιτρέπεται να έχουν πρόσβαση.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_a6ef0f439bec43f4a669dc5b9e4fe69d~mv2.jpg" alt="Όνομα βιβλιοθήκης πολυμέσων" caption="Ονομάστε τη βιβλιοθήκη πολυμέσων DLNA" width="500" >}}

9. Κάντε κλικ στο 'OK' για επιβεβαίωση. Τώρα, οι προσωπικοί σας φάκελοι όπως Μουσική, Εικόνες και Βίντεο θα είναι ορατοί σε οποιαδήποτε συσκευή streaming με υποστήριξη UPnP.

## Πώς να απενεργοποιήσετε τον DLNA Media Server στα Windows 10

1. Κάντε κλικ στο 'Έναρξη' και πληκτρολογήστε 'services' στο πεδίο αναζήτησης.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51f9a32815df49d098d7aa0a87ea2670~mv2.jpg" alt="Υπηρεσίες Windows" caption="Ανοίξτε τις Υπηρεσίες Windows" width="500" >}}

2. Στην οθόνη 'Υπηρεσίες', κάντε κύλιση προς τα κάτω για να βρείτε το 'Windows Media Player Network Sharing Service'.  
3. Κάντε διπλό κλικ σε αυτό και ορίστε τον 'Τύπο εκκίνησης' σε 'Μη αυτόματα'.  
4. Σταματήστε την υπηρεσία κάνοντας κλικ στο κουμπί 'Διακοπή'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_5cd01e7879b6466390737e63415faa9c~mv2.jpg" alt="Διακοπή υπηρεσίας DLNA" caption="Απενεργοποίηση υπηρεσίας κοινής χρήσης δικτύου DLNA" width="500" >}}

## Πώς να ακούσετε μουσική από DLNA Server στο iPhone

1. Εγκαταστήστε τη δωρεάν εφαρμογή **Evermusic** από το App Store:  
   [Εφαρμογή Evermusic](https://apps.apple.com/us/app/evermusic-offline-music-player-cloud-streamer/id885367198?ls=1)

2. Ανοίξτε την καρτέλα 'Συνδέσεις' και πατήστε στο στοιχείο 'Διαθέσιμες συσκευές' στην ενότητα 'Τοπικό δίκτυο'.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_51ea5a3f90d745b188d1a0ca539116e9~mv2.jpg" alt="Συνδέσεις Evermusic" caption="Evermusic: Οθόνη Συνδέσεων" width="500" >}}

3. Περιμένετε λίγα δευτερόλεπτα μέχρι να φορτωθεί η λίστα συσκευών και πατήστε στον Windows Media Player DLNA server (π.χ. 'MSEDGEWIN10: My Windows Library').

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_281a535137cb409a9721da4a07581546~mv2.jpg" alt="Διαθέσιμες συσκευές" caption="Διαθέσιμοι DLNA servers στο Evermusic" width="500" >}}

4. Θα δείτε μια λίστα φακέλων που είναι διαθέσιμοι στον media server.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_45b5b6b0435945d1b9914ee1acb71698~mv2.jpg" alt="Φάκελοι Evermusic" caption="Περιήγηση σε κοινόχρηστους φακέλους από τον DLNA server" width="500" >}}

5. Ανοίξτε οποιονδήποτε φάκελο που περιέχει αρχεία ήχου.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_dc15136bfd8c4382a23fede8aaf630e5~mv2.jpg" alt="Περιεχόμενο φακέλου" caption="Προβολή περιεχομένων κοινόχρηστων φακέλων DLNA" width="500" >}}

6. Πατήστε σε οποιοδήποτε αρχείο για να ξεκινήσετε τον audio player.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_2f9ba14a07bb4b84b94b2613affeeafa~mv2.jpg" alt="Audio Player" caption="Αναπαραγωγή αρχείου ήχου από DLNA στο Evermusic" width="500" >}}

7. Για να βελτιώσετε την ακουστική εμπειρία σας, πατήστε το εικονίδιο 'Equalizer' κοντά στην ένδειξη έντασης στο κάτω μέρος της οθόνης για να ενεργοποιήσετε τον equalizer τύπου iPod με προενισχυτή.

   {{< figure src="/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone/21260c_7184e2d07347462aa788052e25074ad2~mv2.jpg" alt="Equalizer" caption="Χρησιμοποιήστε τον ενσωματωμένο equalizer για βελτιωμένη αναπαραγωγή" width="500" >}}

## Συμπέρασμα

Με τον DLNA Media Server στα Windows 10 και το Evermusic στο iPhone σας, μπορείτε να απολαύσετε απρόσκοπτο streaming μουσικής από τον υπολογιστή σας στην κινητή σας συσκευή. Πείτε αντίο στους περιορισμούς αποθήκευσης και καλωσορίστε τη μουσική κατ' απαίτηση!

## Συχνές ερωτήσεις

{{% details title="Χρειάζεται να εγκαταστήσω λογισμικό server στα Windows 10;" closed="true" %}}
Όχι. Τα Windows 10 περιλαμβάνουν ενσωματωμένο DLNA media server. Χρειάζεται μόνο να ενεργοποιήσετε το streaming πολυμέσων στις ρυθμίσεις του Κέντρου δικτύου και κοινής χρήσης. Δεν απαιτείται λογισμικό τρίτων.
{{% /details %}}

{{% details title="Πρέπει το iPhone μου να είναι στο ίδιο δίκτυο Wi-Fi;" closed="true" %}}
Ναι. Το streaming DLNA λειτουργεί μέσω του τοπικού σας δικτύου. Τόσο ο υπολογιστής σας με Windows 10 όσο και το iPhone σας πρέπει να είναι συνδεδεμένα στο ίδιο δίκτυο Wi-Fi για να ανακαλύψει το Evermusic τον DLNA server.
{{% /details %}}

{{% details title="Ποιες μορφές ήχου μπορώ να κάνω stream μέσω DLNA;" closed="true" %}}
Ο Windows DLNA server μοιράζεται αρχεία από τον φάκελο Μουσική ανεξάρτητα από τη μορφή. Το Evermusic υποστηρίζει MP3, FLAC, AAC, WAV, OGG, AIFF και πολλές άλλες μορφές, ώστε να μπορείτε να αναπαράγετε σχεδόν οποιοδήποτε αρχείο ήχου από τον server.
{{% /details %}}

{{% details title="Μπορώ να χρησιμοποιήσω το Flacbox αντί του Evermusic;" closed="true" %}}
Ναι. Το Flacbox υποστηρίζει επίσης περιήγηση και αναπαραγωγή DLNA/UPnP. Μπορείτε να χρησιμοποιήσετε οποιαδήποτε από τις δύο εφαρμογές για να ανακαλύψετε και να αναπαράγετε μουσική από τον Windows DLNA server σας.
{{% /details %}}

{{% details title="Θα χρησιμοποιήσει το DLNA streaming δεδομένα κινητού;" closed="true" %}}
Όχι. Το DLNA λειτουργεί αποκλειστικά στο τοπικό σας δίκτυο Wi-Fi. Δεν χρησιμοποιεί δεδομένα κινητού. Ωστόσο, και οι δύο συσκευές πρέπει να παραμένουν συνδεδεμένες στο ίδιο δίκτυο κατά τη διάρκεια της αναπαραγωγής.
{{% /details %}}

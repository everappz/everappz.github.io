**בקצרה:** העלה את קבצי המוזיקה או המדיה שלך לכל שירות ענן נתמך (Google Drive, Dropbox, OneDrive ועוד), ואז חבר את השירות הזה בתוך Evermusic, Flacbox או Evertag כדי להזרים או להוריד את הקבצים שלך ישירות ב-iPhone, iPad או Mac.

## סיכום

חיבור שירות הענן שלך לאפליקציות שלנו מייעל את הגישה לקבצים ואת ניהולם, ומקל יותר מאי פעם ליהנות מהקבצים שלך במכשירים שונים.

אמור שלום למגבלות אחסון ושלום לנוחות!

## שאלות נפוצות

{{% details title="אילו שירותי ענן נתמכים?" closed="true" %}}
Evermusic, Flacbox ו-Evertag תומכים ב-Google Drive, Dropbox, OneDrive, Box, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud וספקי ענן אחרים. אתה יכול גם לחבר שרתי WebDAV, SMB ו-FTP מותאמים אישית.
{{% /details %}}

{{% details title="האם אני יכול להזרים מוזיקה ישירות מהענן בלי להוריד?" closed="true" %}}
כן. כל שלוש האפליקציות תומכות בהזרמת קבצי אודיו ישירות מאחסון הענן המחובר שלך. אתה יכול גם להוריד קבצים להשמעה לא מקוונת כשאין לך גישה לאינטרנט.
{{% /details %}}

{{% details title="האם יש מגבלת גודל קובץ או אחסון באפליקציה?" closed="true" %}}
האפליקציות אינן מטילות מגבלות גודל קובץ או אחסון משלהן. האחסון הזמין שלך תלוי בתוכנית שירות הענן שלך ובאחסון המקומי של המכשיר שלך לקבצים שהורדו.
{{% /details %}}

{{% details title="האם אני יכול לחבר מספר חשבונות ענן בו-זמנית?" closed="true" %}}
כן. אתה יכול לחבר מספר שירותי ענן ומספר חשבונות מאותו ספק בו-זמנית. כל החשבונות המחוברים מופיעים בלשונית חיבורים למעבר קל.
{{% /details %}}

{{% details title="האם אני צריך להעלות מחדש קבצים אם אני עובר לאפליקציה אחרת?" closed="true" %}}
לא. מכיוון שהקבצים שלך מאוחסנים בענן, אתה יכול לחבר את אותו חשבון ענן ל-Evermusic, Flacbox או Evertag בלי להעלות מחדש דבר. כל אפליקציה ניגשת לאותם קבצים מאחסון הענן שלך.
{{% /details %}}

{{% details title="האם נתוני חשבון הענן שלי מאובטחים?" closed="true" %}}
כן. האפליקציה משתמשת רק ב-SDK רשמיים וחיבורים מוצפנים לאינטראקציה עם שירותי ענן. שם המשתמש והסיסמה שלך לעולם אינם נשמרים על ידי האפליקציה. בעת ההתחברות, האפליקציה מציגה את דף ההרשאה הרשמי שמספק שירות הענן. לאחר הרשאה מוצלחת, ספק הענן שולח אסימון אימות לאפליקציה, שנשמר בצורה מאובטחת ב-Keychain של המכשיר. אסימון זה משמש לכל בקשות ה-API.<br><br>
האפליקציה אינה משתפת שום מידע מחשבון הענן שלך. אתה יכול לבטל גישה בכל עת מדף ההגדרות של חשבון הענן שלך בדפדפן אינטרנט, או לנתק את החשבון בתוך האפליקציה.
{{% /details %}}

{{% details title="כיצד אני מנתק שירות ענן או משנה את הגדרותיו?" closed="true" %}}
אתר את אחסון הענן בלשונית **חיבורים** של האפליקציה והקש על כפתור **...** שלידו. תראה את האפשרויות הבאות:<br>
- **שינוי שם** -- שנה את שם התצוגה של שירות הענן<br>
- **הגדרות** -- שנה הגדרות או אשר מחדש אם האסימון פג<br>
- **לנתק** -- הסר את החיבור לחלוטין. פעולה זו מסירה את כל השירים משירות ענן זה מספריית המוזיקה של האפליקציה, אך הקבצים נשארים בשרת
{{% /details %}}

{{% details title="כיצד אני מבטל את גישת האפליקציה לחשבון הענן שלי?" closed="true" %}}
התחבר לחשבון הענן שלך בדפדפן אינטרנט ופתח את דף הגדרות החשבון או האבטחה. מצא את רשימת אפליקציות צד שלישי המחוברות והסר את האפליקציה שאתה כבר לא רוצה לאשר. אתה יכול גם לנתק את חשבון הענן בתוך האפליקציה -- זה מסיר את אסימון האימות מהמכשיר שלך. אם תמחק את האפליקציה לחלוטין, כל הנתונים שהורדו ואסימוני הגישה יוסרו אוטומטית.
{{% /details %}}

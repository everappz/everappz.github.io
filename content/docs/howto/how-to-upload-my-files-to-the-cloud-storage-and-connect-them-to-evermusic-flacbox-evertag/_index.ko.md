**요약:** 음악이나 미디어 파일을 지원되는 클라우드 서비스(Google Drive, Dropbox, OneDrive 등)에 업로드한 다음, Evermusic, Flacbox 또는 Evertag 내에서 해당 서비스를 연결하여 iPhone, iPad 또는 Mac에서 직접 파일을 스트리밍하거나 다운로드하세요.

## 1단계: 클라우드 스토리지 계정 가입

아직 가입하지 않았다면, Google Drive, Dropbox, OneDrive 등의 클라우드 서비스에 가입하세요.

지원되는 모든 계정을 연결할 수 있습니다: Dropbox, Box, Google Drive, OneDrive, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud 등.

{{< figure src="/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag/21260c_3433be90b41f4fe7988383929751b5ec~mv2.png" alt="Evermusic: Go to Google Drive" width="400" >}}

클라우드 제공업체에서 Mac 또는 PC용 공식 애플리케이션을 다운로드하세요.

## 3단계: 클라우드에 파일 업로드

데스크톱 앱이나 웹 인터페이스를 사용하여 클라우드 계정에 파일을 이동하세요.

## 4단계: 앱에 클라우드 스토리지 연결

업로드가 완료되면 iPhone, iPad 또는 Mac에서 Evermusic, Flacbox 또는 Evertag을 여세요.

## 5단계: 앱에서 파일 관리

연결되면 클라우드 파일을 탐색하고 앱 내 파일 관리자를 사용하여 관리할 수 있습니다. **추가 작업** 버튼 `...`을 탭하여 파일을 이동, 이름 변경, 삭제 또는 다운로드할 수 있습니다.

## 결론

클라우드 서비스를 앱에 연결하면 파일 접근과 관리가 간소화되어, 다양한 기기에서 파일을 더 쉽게 즐길 수 있습니다.

저장 공간 제한에 작별을 고하고 편리함을 만끽하세요!

## 자주 묻는 질문

{{% details title="어떤 클라우드 서비스가 지원되나요?" closed="true" %}}
Evermusic, Flacbox, Evertag은 Google Drive, Dropbox, OneDrive, Box, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud 및 기타 클라우드 제공업체를 지원합니다. 사용자 정의 WebDAV, SMB, FTP 서버도 연결할 수 있습니다.
{{% /details %}}

{{% details title="다운로드하지 않고 클라우드에서 직접 음악을 스트리밍할 수 있나요?" closed="true" %}}
네. 세 가지 앱 모두 연결된 클라우드 스토리지에서 직접 오디오 파일 스트리밍을 지원합니다. 인터넷에 접속할 수 없을 때 오프라인 재생을 위해 파일을 다운로드할 수도 있습니다.
{{% /details %}}

{{% details title="앱에 파일 크기나 저장 용량 제한이 있나요?" closed="true" %}}
앱 자체에는 파일 크기나 저장 용량 제한이 없습니다. 사용 가능한 저장 공간은 클라우드 서비스 요금제와 다운로드한 파일을 위한 기기의 로컬 저장 공간에 따라 달라집니다.
{{% /details %}}

{{% details title="여러 클라우드 계정을 동시에 연결할 수 있나요?" closed="true" %}}
네. 여러 클라우드 서비스와 동일한 제공업체의 여러 계정을 동시에 연결할 수 있습니다. 연결된 모든 계정은 연결하기 탭에 표시되어 쉽게 전환할 수 있습니다.
{{% /details %}}

{{% details title="다른 앱으로 전환하면 파일을 다시 업로드해야 하나요?" closed="true" %}}
아니요. 파일이 클라우드에 저장되어 있으므로 다시 업로드할 필요 없이 동일한 클라우드 계정을 Evermusic, Flacbox 또는 Evertag에 연결할 수 있습니다. 각 앱은 클라우드 스토리지에서 동일한 파일에 접근합니다.
{{% /details %}}

{{% details title="내 클라우드 계정 데이터는 안전한가요?" closed="true" %}}
네. 앱은 클라우드 서비스와 상호 작용하기 위해 공식 SDK와 암호화된 연결만 사용합니다. 로그인 정보와 비밀번호는 앱에 저장되지 않습니다. 로그인 시 앱은 클라우드 서비스에서 제공하는 공식 인증 페이지를 표시합니다. 인증이 완료되면 클라우드 제공업체가 앱에 인증 토큰을 보내며, 이 토큰은 기기 키체인에 안전하게 저장됩니다. 이 토큰은 모든 API 요청에 사용됩니다.<br><br>
앱은 클라우드 계정의 어떤 정보도 공유하지 않습니다. 웹 브라우저의 클라우드 계정 설정 페이지에서 언제든지 접근 권한을 취소하거나, 앱 내에서 계정 연결을 해제할 수 있습니다.
{{% /details %}}

{{% details title="클라우드 서비스를 연결 해제하거나 설정을 변경하려면 어떻게 하나요?" closed="true" %}}
앱의 연결하기 탭에서 클라우드 스토리지를 찾아 옆에 있는 **...** 버튼을 탭하세요. 다음 옵션이 표시됩니다:<br>
- **이름 변경** -- 클라우드 서비스의 표시 이름 변경<br>
- **설정** -- 설정 수정 또는 토큰이 만료된 경우 재인증<br>
- **연결 끊기** -- 연결을 완전히 제거합니다. 이 클라우드 서비스의 모든 노래가 앱의 음악 라이브러리에서 제거되지만, 파일은 서버에 남아 있습니다
{{% /details %}}

{{% details title="앱의 클라우드 계정 접근 권한을 취소하려면 어떻게 하나요?" closed="true" %}}
웹 브라우저에서 클라우드 계정에 로그인하고 계정 설정 또는 보안 페이지를 여세요. 연결된 타사 앱 목록을 찾아 더 이상 권한을 부여하고 싶지 않은 앱을 제거하세요. 앱 내에서도 클라우드 계정 연결을 해제할 수 있습니다. 이렇게 하면 기기에서 인증 토큰이 제거됩니다. 앱을 완전히 삭제하면 다운로드된 모든 데이터와 접근 토큰이 자동으로 제거됩니다.
{{% /details %}}

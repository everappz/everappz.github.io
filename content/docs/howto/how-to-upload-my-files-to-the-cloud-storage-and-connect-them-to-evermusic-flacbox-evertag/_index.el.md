## Συχνές ερωτήσεις

{{% details title="Ποιες υπηρεσίες cloud υποστηρίζονται;" closed="true" %}}
Τα Evermusic, Flacbox και Evertag υποστηρίζουν Google Drive, Dropbox, OneDrive, Box, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud και άλλους παρόχους cloud. Μπορείτε επίσης να συνδέσετε προσαρμοσμένους διακομιστές WebDAV, SMB και FTP.
{{% /details %}}

{{% details title="Μπορώ να κάνω streaming μουσικής απευθείας από το cloud χωρίς λήψη;" closed="true" %}}
Ναι. Και οι τρεις εφαρμογές υποστηρίζουν streaming αρχείων ήχου απευθείας από τον συνδεδεμένο αποθηκευτικό χώρο cloud σας. Μπορείτε επίσης να κατεβάσετε αρχεία για αναπαραγωγή εκτός σύνδεσης όταν δεν έχετε πρόσβαση στο internet.
{{% /details %}}

{{% details title="Υπάρχει όριο μεγέθους αρχείου ή αποθήκευσης στην εφαρμογή;" closed="true" %}}
Οι εφαρμογές δεν επιβάλλουν δικά τους όρια μεγέθους αρχείου ή αποθήκευσης. Ο διαθέσιμος αποθηκευτικός χώρος εξαρτάται από το πλάνο της υπηρεσίας cloud σας και τον τοπικό αποθηκευτικό χώρο της συσκευής σας για τα ληφθέντα αρχεία.
{{% /details %}}

{{% details title="Μπορώ να συνδέσω πολλαπλούς λογαριασμούς cloud ταυτόχρονα;" closed="true" %}}
Ναι. Μπορείτε να συνδέσετε πολλαπλές υπηρεσίες cloud και πολλαπλούς λογαριασμούς από τον ίδιο πάροχο ταυτόχρονα. Όλοι οι συνδεδεμένοι λογαριασμοί εμφανίζονται στην καρτέλα Συνδέσεις για εύκολη εναλλαγή.
{{% /details %}}

{{% details title="Χρειάζεται να ανεβάσω ξανά τα αρχεία αν αλλάξω εφαρμογή;" closed="true" %}}
Όχι. Δεδομένου ότι τα αρχεία σας αποθηκεύονται στο cloud, μπορείτε να συνδέσετε τον ίδιο λογαριασμό cloud στο Evermusic, Flacbox ή Evertag χωρίς να ανεβάσετε τίποτα ξανά. Κάθε εφαρμογή έχει πρόσβαση στα ίδια αρχεία από τον αποθηκευτικό χώρο cloud σας.
{{% /details %}}

{{% details title="Είναι ασφαλή τα δεδομένα του λογαριασμού cloud μου;" closed="true" %}}
Ναι. Η εφαρμογή χρησιμοποιεί μόνο επίσημα SDK και κρυπτογραφημένες συνδέσεις για την αλληλεπίδραση με τις υπηρεσίες cloud. Το όνομα χρήστη και ο κωδικός πρόσβασής σας δεν αποθηκεύονται ποτέ από την εφαρμογή. Κατά τη σύνδεση, η εφαρμογή εμφανίζει την επίσημη σελίδα εξουσιοδότησης που παρέχεται από την υπηρεσία cloud. Μετά την επιτυχή εξουσιοδότηση, ο πάροχος cloud στέλνει ένα token ελέγχου ταυτότητας στην εφαρμογή, το οποίο αποθηκεύεται με ασφάλεια στο Keychain της συσκευής. Αυτό το token χρησιμοποιείται για όλα τα αιτήματα API.<br><br>
Η εφαρμογή δεν μοιράζεται καμία πληροφορία από τον λογαριασμό cloud σας. Μπορείτε να ανακαλέσετε την πρόσβαση ανά πάσα στιγμή από τη σελίδα ρυθμίσεων του λογαριασμού cloud σας σε ένα πρόγραμμα περιήγησης ιστού ή να αποσυνδέσετε τον λογαριασμό μέσα στην εφαρμογή.
{{% /details %}}

{{% details title="Πώς αποσυνδέω μια υπηρεσία cloud ή αλλάζω τη διαμόρφωσή της;" closed="true" %}}
Εντοπίστε τον αποθηκευτικό χώρο cloud στην καρτέλα **Συνδέσεις** της εφαρμογής και πατήστε το κουμπί **...** δίπλα του. Θα δείτε αυτές τις επιλογές:<br>
- **Μετονομασία** -- αλλαγή του εμφανιζόμενου ονόματος της υπηρεσίας cloud<br>
- **Ρυθμίσεις** -- τροποποίηση διαμόρφωσης ή επανεξουσιοδότηση εάν το token έχει λήξει<br>
- **Αποσύνδεση** -- αφαίρεση της σύνδεσης εντελώς. Αυτό αφαιρεί όλα τα τραγούδια αυτής της υπηρεσίας cloud από τη μουσική βιβλιοθήκη της εφαρμογής, αλλά τα αρχεία παραμένουν στον διακομιστή
{{% /details %}}

{{% details title="Πώς ανακαλώ την πρόσβαση της εφαρμογής στον λογαριασμό cloud μου;" closed="true" %}}
Συνδεθείτε στον λογαριασμό cloud σας σε ένα πρόγραμμα περιήγησης ιστού και ανοίξτε τη σελίδα ρυθμίσεων λογαριασμού ή ασφάλειας. Βρείτε τη λίστα με τις συνδεδεμένες εφαρμογές τρίτων και αφαιρέστε την εφαρμογή που δεν θέλετε πλέον να εξουσιοδοτήσετε. Μπορείτε επίσης να αποσυνδέσετε τον λογαριασμό cloud μέσα στην εφαρμογή -- αυτό αφαιρεί το token ελέγχου ταυτότητας από τη συσκευή σας. Αν διαγράψετε την εφαρμογή εντελώς, όλα τα ληφθέντα δεδομένα και τα token πρόσβασης αφαιρούνται αυτόματα.
{{% /details %}}

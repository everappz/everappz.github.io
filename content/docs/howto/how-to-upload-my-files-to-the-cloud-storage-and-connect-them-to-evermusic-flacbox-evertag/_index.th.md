**สรุป:** อัปโหลดไฟล์เพลงหรือสื่อของคุณไปยังบริการคลาวด์ที่รองรับ (Google Drive, Dropbox, OneDrive และอื่นๆ) จากนั้นเชื่อมต่อบริการนั้นใน Evermusic, Flacbox หรือ Evertag เพื่อสตรีมหรือดาวน์โหลดไฟล์โดยตรงบน iPhone, iPad หรือ Mac

## ขั้นตอนที่ 1: สมัครบัญชีที่เก็บข้อมูลบนคลาวด์

หากคุณยังไม่ได้สมัคร ให้สมัครบริการคลาวด์ เช่น Google Drive, Dropbox, OneDrive เป็นต้น

{{< figure src="/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag/21260c_09c03b8445784d23b632534939b328dd~mv2.png" alt="Evermusic: Create your Google Account" width="400" >}}

คุณสามารถเชื่อมต่อบัญชีที่รองรับใดก็ได้: Dropbox, Box, Google Drive, OneDrive, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud และอื่นๆ

ดาวน์โหลดแอปพลิเคชันอย่างเป็นทางการสำหรับ Mac หรือ PC จากผู้ให้บริการคลาวด์ของคุณ

ใช้แอปเดสก์ท็อปหรืออินเทอร์เฟซเว็บเพื่อย้ายไฟล์ไปยังบัญชีคลาวด์ของคุณ

## บทสรุป

การเชื่อมต่อบริการคลาวด์กับแอปของเราช่วยให้การเข้าถึงและจัดการไฟล์ง่ายขึ้น ทำให้คุณเพลิดเพลินกับไฟล์บนอุปกรณ์ต่างๆ ได้ง่ายกว่าที่เคย

บอกลาข้อจำกัดของพื้นที่เก็บข้อมูลและต้อนรับความสะดวกสบาย!

## คำถามที่พบบ่อย

{{% details title="บริการคลาวด์ใดที่รองรับ?" closed="true" %}}
Evermusic, Flacbox และ Evertag รองรับ Google Drive, Dropbox, OneDrive, Box, MediaFire, Yandex.Disk, MEGA, MyDrive, pCloud และผู้ให้บริการคลาวด์อื่นๆ คุณยังสามารถเชื่อมต่อเซิร์ฟเวอร์ WebDAV, SMB และ FTP แบบกำหนดเองได้
{{% /details %}}

{{% details title="ฉันสามารถสตรีมเพลงจากคลาวด์โดยตรงโดยไม่ต้องดาวน์โหลดได้หรือไม่?" closed="true" %}}
ได้ แอปทั้งสามรองรับการสตรีมไฟล์เสียงโดยตรงจากที่เก็บข้อมูลบนคลาวด์ที่เชื่อมต่อ คุณยังสามารถดาวน์โหลดไฟล์เพื่อเล่นแบบออฟไลน์เมื่อคุณไม่มีการเข้าถึงอินเทอร์เน็ต
{{% /details %}}

{{% details title="มีข้อจำกัดขนาดไฟล์หรือพื้นที่เก็บข้อมูลในแอปหรือไม่?" closed="true" %}}
แอปไม่ได้กำหนดข้อจำกัดขนาดไฟล์หรือพื้นที่เก็บข้อมูลของตัวเอง พื้นที่เก็บข้อมูลที่มีอยู่ขึ้นอยู่กับแผนบริการคลาวด์ของคุณและพื้นที่เก็บข้อมูลภายในเครื่องของอุปกรณ์สำหรับไฟล์ที่ดาวน์โหลด
{{% /details %}}

{{% details title="ฉันสามารถเชื่อมต่อหลายบัญชีคลาวด์พร้อมกันได้หรือไม่?" closed="true" %}}
ได้ คุณสามารถเชื่อมต่อบริการคลาวด์หลายรายการและหลายบัญชีจากผู้ให้บริการเดียวกันพร้อมกันได้ บัญชีที่เชื่อมต่อทั้งหมดจะปรากฏในแท็บการเชื่อมต่อเพื่อสลับได้ง่าย
{{% /details %}}

{{% details title="ฉันต้องอัปโหลดไฟล์อีกครั้งหรือไม่หากเปลี่ยนไปใช้แอปอื่น?" closed="true" %}}
ไม่ต้อง เนื่องจากไฟล์ของคุณถูกเก็บไว้ในคลาวด์ คุณสามารถเชื่อมต่อบัญชีคลาวด์เดียวกันกับ Evermusic, Flacbox หรือ Evertag โดยไม่ต้องอัปโหลดซ้ำ แต่ละแอปเข้าถึงไฟล์เดียวกันจากที่เก็บข้อมูลบนคลาวด์ของคุณ
{{% /details %}}

{{% details title="ข้อมูลบัญชีคลาวด์ของฉันปลอดภัยหรือไม่?" closed="true" %}}
ปลอดภัย แอปใช้เฉพาะ SDK อย่างเป็นทางการและการเชื่อมต่อที่เข้ารหัสเพื่อโต้ตอบกับบริการคลาวด์ ข้อมูลเข้าสู่ระบบและรหัสผ่านของคุณจะไม่ถูกเก็บโดยแอป เมื่อคุณเข้าสู่ระบบ แอปจะแสดงหน้าอนุญาตอย่างเป็นทางการที่จัดทำโดยบริการคลาวด์ หลังจากอนุญาตสำเร็จ ผู้ให้บริการคลาวด์จะส่งโทเค็นการอนุญาตไปยังแอป ซึ่งจะถูกเก็บอย่างปลอดภัยใน Keychain ของอุปกรณ์ โทเค็นนี้ใช้สำหรับคำขอ API ทั้งหมด<br><br>
แอปไม่แชร์ข้อมูลใดๆ จากบัญชีคลาวด์ของคุณ คุณสามารถเพิกถอนการเข้าถึงได้ทุกเมื่อจากหน้าการตั้งค่าบัญชีคลาวด์ในเว็บเบราว์เซอร์ หรือตัดการเชื่อมต่อบัญชีในแอป
{{% /details %}}

{{% details title="ฉันจะตัดการเชื่อมต่อบริการคลาวด์หรือเปลี่ยนการกำหนดค่าได้อย่างไร?" closed="true" %}}
ค้นหาที่เก็บข้อมูลบนคลาวด์ในแท็บการเชื่อมต่อของแอปแล้วแตะปุ่ม **...** ข้างๆ คุณจะเห็นตัวเลือกเหล่านี้:<br>
- **เปลี่ยนชื่อ** -- เปลี่ยนชื่อที่แสดงของบริการคลาวด์<br>
- **การตั้งค่า** -- แก้ไขการกำหนดค่าหรืออนุญาตอีกครั้งหากโทเค็นหมดอายุ<br>
- **ตัดการเชื่อมต่อ** -- ลบการเชื่อมต่อทั้งหมด การดำเนินการนี้จะลบเพลงทั้งหมดจากบริการคลาวด์นี้ออกจากคลังเพลงของแอป แต่ไฟล์ยังคงอยู่บนเซิร์ฟเวอร์
{{% /details %}}

{{% details title="ฉันจะเพิกถอนการเข้าถึงแอปไปยังบัญชีคลาวด์ของฉันได้อย่างไร?" closed="true" %}}
เข้าสู่ระบบบัญชีคลาวด์ของคุณในเว็บเบราว์เซอร์แล้วเปิดหน้าการตั้งค่าบัญชีหรือหน้าความปลอดภัย ค้นหารายการแอปของบุคคลที่สามที่เชื่อมต่อแล้วลบแอปที่คุณไม่ต้องการอนุญาตอีกต่อไป คุณยังสามารถตัดการเชื่อมต่อบัญชีคลาวด์ในแอป ซึ่งจะลบโทเค็นการอนุญาตออกจากอุปกรณ์ของคุณ หากคุณลบแอปทั้งหมด ข้อมูลที่ดาวน์โหลดทั้งหมดและโทเค็นการเข้าถึงจะถูกลบโดยอัตโนมัติ
{{% /details %}}

**בקצרה:** השתמשו באפליקציית **Soundy** כדי לייצא כל רשימת השמעה, מועדפים או שיתופים חוזרים מ-SoundCloud כקובץ M3U. העבירו את הקובץ ל-Mac שלכם באמצעות AirDrop ופתחו אותו בכל נגן מוזיקה תואם.

{{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1d33da9a468b4b6aad57393aae61f763~mv2_d_4500_3450_s_4_2.jpg" alt="ייצוא רשימות השמעה ב-Soundy" caption="ייצוא רשימות השמעה מ-SoundCloud לפורמט M3U עם Soundy" width="800" >}}

## כיצד לייצא רשימת השמעה מ-SoundCloud לפורמט M3U

1. התקינו את [Soundy](http://bit.ly/soundyappstore).  
2. התחברו לחשבון ה-SoundCloud שלכם.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="מסך ההתחברות של Soundy" caption="התחברו לחשבון ה-SoundCloud שלכם ב-Soundy" width="400" >}}

3. בחרו 'רשימות השמעה' במסך 'הפרופיל שלי'.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="מסך הפרופיל שלי ב-Soundy" caption="נווטו אל רשימות ההשמעה" width="400" >}}

4. לחצו על כפתור 'עוד' ברשימת ההשמעה שברצונכם לייצא.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="מסך רשימות ההשמעה שלי ב-Soundy" caption="גשו לאפשרויות רשימת ההשמעה" width="400" >}}

5. בחרו 'ייצוא ל-M3U' מהתפריט הנפתח.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="מסך האפשרויות של Soundy" caption="ייצוא רשימת השמעה לפורמט M3U" width="400" >}}

6. המתינו מספר שניות בזמן שמחוון הפעילות פועל.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="מסך הטעינה של Soundy" caption="הייצוא מתבצע..." width="400" >}}

7. לאחר השלמת הייצוא, יופיע דיאלוג 'פתח באמצעות' הסטנדרטי כדי שתוכלו לבחור יעד לרשימת ההשמעה M3U שנוצרה.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="מסך הייצוא של Soundy" caption="שמרו או שתפו את רשימת ההשמעה M3U שלכם" width="400" >}}

8. תוכלו להעביר את רשימת ההשמעה ל-Mac שלכם באמצעות AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="מסך AirDrop של Soundy" caption="העבירו רשימת השמעה M3U באמצעות AirDrop" width="400" >}}

9. פתחו את רשימת ההשמעה בכל תוכנה שתומכת ברשימות השמעה מסוג M3U (לדוגמה, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="מסך נגן VOX" caption="השתמשו ברשימת ההשמעה שלכם ב-VOX או בכל אפליקציה תואמת" width="400" >}}

## כיצד לייצא מועדפים מ-SoundCloud לרשימת השמעה M3U

תוכלו לייצא יותר מרשימות השמעה בלבד — Soundy תומכת גם בייצוא מועדפים, שירים שהועלו ושיתופים חוזרים מהפרופיל שלכם או מכל פרופיל ציבורי ב-SoundCloud.

לייצוא מועדפים:

1. פתחו את לשונית **המועדפים** ולחצו על כפתור **עוד** בפינה הימנית העליונה.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="מסך המועדפים של Soundy" caption="גשו לאפשרויות בלשונית המועדפים" width="400" >}}

2. בחרו את הפעולה **ייצוא ל-M3U** כדי להתחיל את תהליך הייצוא.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="מסך ייצוא ל-M3U של Soundy" caption="ייצוא מועדפים מ-SoundCloud ל-M3U" width="400" >}}

זה הכל למדריך זה. תוכלו למצוא מידע נוסף על אפליקציית Soundy [כאן](/products/soundy).

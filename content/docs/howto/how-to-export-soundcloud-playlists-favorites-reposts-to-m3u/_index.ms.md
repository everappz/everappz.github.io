## Cara Mengeksport Senarai Main SoundCloud ke Format M3U

1. Pasang [Soundy](http://bit.ly/soundyappstore).  
2. Log masuk ke akaun SoundCloud anda.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Skrin Log Masuk Soundy" caption="Log masuk ke akaun SoundCloud anda dalam Soundy" width="400" >}}

3. Pilih 'Senarai Main' di skrin 'Profil Saya'.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Skrin Profil Saya Soundy" caption="Navigasi ke Senarai Main" width="400" >}}

4. Ketik butang 'Lagi' pada senarai main yang ingin anda eksport.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Skrin Senarai Main Saya Soundy" caption="Akses pilihan senarai main" width="400" >}}

5. Pilih 'Eksport ke M3U' dalam menu lungsur.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Skrin Pilihan Soundy" caption="Eksport senarai main ke format M3U" width="400" >}}

6. Tunggu beberapa saat sementara penunjuk aktiviti berjalan.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Skrin Pemuatan Soundy" caption="Eksport sedang berjalan..." width="400" >}}

7. Setelah eksport selesai, dialog standard 'Buka Dalam' akan muncul supaya anda boleh memilih destinasi untuk senarai main M3U yang dijana.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Skrin Eksport Soundy" caption="Simpan atau kongsi senarai main M3U anda" width="400" >}}

8. Anda boleh memindahkan senarai main ke Mac anda menggunakan AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Skrin AirDrop Soundy" caption="Pindahkan senarai main M3U menggunakan AirDrop" width="400" >}}

9. Buka senarai main dalam mana-mana program yang menyokong senarai main M3U (contohnya, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Skrin pemain VOX" caption="Gunakan senarai main anda dalam VOX atau mana-mana aplikasi yang serasi" width="400" >}}

## Cara Mengeksport Kegemaran SoundCloud ke Senarai Main M3U

Anda boleh mengeksport lebih daripada sekadar senarai main — Soundy juga menyokong eksport kegemaran, lagu yang dimuat naik, dan repost daripada profil anda sendiri atau mana-mana profil awam SoundCloud.

Untuk mengeksport kegemaran:

1. Buka tab **Kegemaran** dan ketik butang **Lagi** di penjuru kanan atas.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Skrin Kegemaran Soundy" caption="Akses pilihan dalam tab Kegemaran" width="400" >}}

2. Pilih tindakan **Eksport ke M3U** untuk memulakan proses eksport.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Skrin Eksport ke M3U Soundy" caption="Eksport kegemaran SoundCloud ke M3U" width="400" >}}

Itu sahaja untuk tutorial ini. Anda boleh mendapatkan maklumat lanjut tentang aplikasi Soundy [di sini](/products/soundy).

## Soalan Lazim

{{% details title="Apakah itu fail M3U?" closed="true" %}}
M3U ialah format senarai main standard yang disokong oleh kebanyakan pemain muzik. Ia mengandungi senarai laluan fail atau URL yang menunjuk ke trek audio. Apabila anda mengeksport senarai main SoundCloud ke M3U, fail tersebut mengandungi URL penstriman untuk setiap trek.
{{% /details %}}

{{% details title="Bolehkah saya mengeksport senarai main daripada pengguna SoundCloud lain?" closed="true" %}}
Ya. Soundy boleh mengeksport senarai main, kegemaran, dan repost daripada mana-mana profil awam SoundCloud, bukan hanya akaun anda sendiri.
{{% /details %}}

{{% details title="Pemain muzik mana yang menyokong senarai main M3U?" closed="true" %}}
Kebanyakan pemain muzik menyokong M3U, termasuk VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic, dan Flacbox.
{{% /details %}}

{{% details title="Bolehkah saya mengeksport repost ke M3U juga?" closed="true" %}}
Ya. Soundy menyokong eksport senarai main, kegemaran, lagu yang dimuat naik, dan repost. Prosesnya sama: ketik butang Lagi dan pilih Eksport ke M3U.
{{% /details %}}

{{% details title="Adakah had bilangan trek yang boleh saya eksport?" closed="true" %}}
Soundy mengeksport semua trek dalam senarai main atau koleksi yang dipilih. Tiada had buatan yang dikenakan oleh aplikasi.
{{% /details %}}

## Hur man exporterar en SoundCloud-spellista till M3U-format

1. Installera [Soundy](http://bit.ly/soundyappstore).  
2. Logga in på ditt SoundCloud-konto.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Inloggningsskärm i Soundy" caption="Logga in på ditt SoundCloud-konto i Soundy" width="400" >}}

3. Välj "Playlists" på skärmen "My Profile".

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="My Profile-skärmen i Soundy" caption="Navigera till Playlists" width="400" >}}

4. Tryck på knappen "More" på spellistan du vill exportera.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="My Playlists-skärmen i Soundy" caption="Öppna alternativ för spellistan" width="400" >}}

5. Välj "Export to M3U" i rullgardinsmenyn.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Alternativskärmen i Soundy" caption="Exportera spellistan till M3U-format" width="400" >}}

6. Vänta några sekunder medan aktivitetsindikatorn körs.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Laddningsskärm i Soundy" caption="Export pågår..." width="400" >}}

7. När exporten är klar visas standarddialogen "Open In" där du kan välja en destination för den genererade M3U-spellistan.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Exportskärmen i Soundy" caption="Spara eller dela din M3U-spellista" width="400" >}}

8. Du kan överföra spellistan till din Mac med AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="AirDrop-skärmen i Soundy" caption="Överför M3U-spellistan med AirDrop" width="400" >}}

9. Öppna spellistan i valfritt program som stöder M3U-spellistor (t.ex. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX-spelarens skärm" caption="Använd din spellista i VOX eller valfri kompatibel app" width="400" >}}

## Hur man exporterar SoundCloud-favoriter till en M3U-spellista

Du kan exportera mer än bara spellistor — Soundy stöder även export av favoriter, uppladdade låtar och reposter från din egen eller valfri offentlig SoundCloud-profil.

För att exportera favoriter:

1. Öppna fliken **Favorites** och tryck på knappen **More** i det övre högra hörnet.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Favoritskärmen i Soundy" caption="Öppna alternativ i fliken Favorites" width="400" >}}

2. Välj åtgärden **Export to M3U** för att starta exportprocessen.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Export To M3U-skärmen i Soundy" caption="Exportera SoundCloud-favoriter till M3U" width="400" >}}

Det var allt för denna guide. Mer information om appen Soundy hittar du [här](/products/soundy).

## Vanliga frågor

{{% details title="Vad är en M3U-fil?" closed="true" %}}
M3U är ett standardformat för spellistor som stöds av de flesta musikspelare. Det innehåller en lista med filsökvägar eller URL:er som pekar på ljudspår. När du exporterar en SoundCloud-spellista till M3U innehåller filen strömmnings-URL:er för varje spår.
{{% /details %}}

{{% details title="Kan jag exportera spellistor från andra SoundCloud-användare?" closed="true" %}}
Ja. Soundy kan exportera spellistor, favoriter och reposter från valfri offentlig SoundCloud-profil, inte bara ditt eget konto.
{{% /details %}}

{{% details title="Vilka musikspelare stöder M3U-spellistor?" closed="true" %}}
De flesta musikspelare stöder M3U, inklusive VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic och Flacbox.
{{% /details %}}

{{% details title="Kan jag exportera reposter till M3U också?" closed="true" %}}
Ja. Soundy stöder export av spellistor, favoriter, uppladdade låtar och reposter. Processen är densamma: tryck på knappen More och välj Export to M3U.
{{% /details %}}

{{% details title="Finns det en gräns för hur många spår jag kan exportera?" closed="true" %}}
Soundy exporterar alla spår i den valda spellistan eller samlingen. Det finns ingen artificiell begränsning i appen.
{{% /details %}}

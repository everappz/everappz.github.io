**W skrócie:** Użyj aplikacji **Soundy**, aby wyeksportować dowolną playlistę SoundCloud, ulubione lub reposty jako plik M3U. Prześlij plik przez AirDrop na swojego Mac i otwórz go w dowolnym kompatybilnym odtwarzaczu muzyki.

## Jak wyeksportować playlistę SoundCloud do formatu M3U

1. Zainstaluj [Soundy](http://bit.ly/soundyappstore).  
2. Zaloguj się na swoje konto SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Ekran logowania Soundy" caption="Zaloguj się na swoje konto SoundCloud w Soundy" width="400" >}}

3. Wybierz „Playlisty" na ekranie „Mój profil".

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Ekran Mój profil Soundy" caption="Przejdź do Playlist" width="400" >}}

4. Naciśnij przycisk „Więcej" przy playliście, którą chcesz wyeksportować.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Ekran Moje playlisty Soundy" caption="Dostęp do opcji playlisty" width="400" >}}

5. Wybierz „Eksportuj do M3U" z menu rozwijanego.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Ekran opcji Soundy" caption="Eksportuj playlistę do formatu M3U" width="400" >}}

6. Poczekaj kilka sekund, aż wskaźnik aktywności zakończy pracę.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Ekran ładowania Soundy" caption="Eksport w toku..." width="400" >}}

7. Po zakończeniu eksportu pojawi się standardowe okno dialogowe „Otwórz w", w którym możesz wybrać miejsce docelowe dla wygenerowanej playlisty M3U.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Ekran eksportu Soundy" caption="Zapisz lub udostępnij swoją playlistę M3U" width="400" >}}

8. Możesz przesłać playlistę na swojego Mac za pomocą AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Ekran AirDrop Soundy" caption="Prześlij playlistę M3U za pomocą AirDrop" width="400" >}}

9. Otwórz playlistę w dowolnym programie obsługującym playlisty M3U (np. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Ekran odtwarzacza VOX" caption="Użyj swojej playlisty w VOX lub dowolnej kompatybilnej aplikacji" width="400" >}}

## Jak wyeksportować ulubione z SoundCloud do playlisty M3U

Możesz eksportować nie tylko playlisty — Soundy obsługuje również eksport ulubionych, przesłanych utworów i repostów z własnego lub dowolnego publicznego profilu SoundCloud.

Aby wyeksportować ulubione:

1. Otwórz kartę **Ulubione** i naciśnij przycisk **Więcej** w prawym górnym rogu.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Ekran ulubionych Soundy" caption="Dostęp do opcji w karcie Ulubione" width="400" >}}

2. Wybierz akcję **Eksportuj do M3U**, aby rozpocząć proces eksportu.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Ekran Eksportuj do M3U Soundy" caption="Eksportuj ulubione z SoundCloud do M3U" width="400" >}}

To wszystko w tym poradniku. Więcej informacji o aplikacji Soundy znajdziesz [tutaj](/products/soundy).

## Często zadawane pytania

{{% details title="Co to jest plik M3U?" closed="true" %}}
M3U to standardowy format playlisty obsługiwany przez większość odtwarzaczy muzyki. Zawiera listę ścieżek plików lub adresów URL wskazujących na utwory audio. Gdy eksportujesz playlistę SoundCloud do M3U, plik zawiera adresy URL streamingu dla każdego utworu.
{{% /details %}}

{{% details title="Czy mogę eksportować playlisty innych użytkowników SoundCloud?" closed="true" %}}
Tak. Soundy może eksportować playlisty, ulubione i reposty z dowolnego publicznego profilu SoundCloud, nie tylko z własnego konta.
{{% /details %}}

{{% details title="Które odtwarzacze muzyki obsługują playlisty M3U?" closed="true" %}}
Większość odtwarzaczy muzyki obsługuje M3U, w tym VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic i Flacbox.
{{% /details %}}

{{% details title="Czy mogę również eksportować reposty do M3U?" closed="true" %}}
Tak. Soundy obsługuje eksport playlist, ulubionych, przesłanych utworów i repostów. Proces jest taki sam: naciśnij przycisk Więcej i wybierz Eksportuj do M3U.
{{% /details %}}

{{% details title="Czy istnieje limit liczby utworów, które mogę wyeksportować?" closed="true" %}}
Soundy eksportuje wszystkie utwory z wybranej playlisty lub kolekcji. Aplikacja nie narzuca żadnego sztucznego limitu.
{{% /details %}}

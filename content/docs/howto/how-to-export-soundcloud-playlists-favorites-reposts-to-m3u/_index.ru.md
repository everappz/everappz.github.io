## Как экспортировать плейлист SoundCloud в формат M3U

1. Установите [Soundy](http://bit.ly/soundyappstore).  
2. Войдите в свой аккаунт SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Экран входа в Soundy" caption="Войдите в свой аккаунт SoundCloud в Soundy" width="400" >}}

3. Выберите «Playlists» на экране «My Profile».

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Экран My Profile в Soundy" caption="Перейдите к Playlists" width="400" >}}

4. Нажмите кнопку «More» на плейлисте, который хотите экспортировать.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Экран My Playlists в Soundy" caption="Откройте параметры плейлиста" width="400" >}}

5. Выберите «Export to M3U» в выпадающем меню.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Экран параметров Soundy" caption="Экспортируйте плейлист в формат M3U" width="400" >}}

6. Подождите несколько секунд, пока работает индикатор активности.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Экран загрузки Soundy" caption="Экспорт выполняется..." width="400" >}}

7. После завершения экспорта появится стандартный диалог «Open In», в котором можно выбрать место сохранения сгенерированного плейлиста M3U.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Экран экспорта Soundy" caption="Сохраните или поделитесь плейлистом M3U" width="400" >}}

8. Вы можете передать плейлист на Mac с помощью AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Экран AirDrop в Soundy" caption="Передайте плейлист M3U через AirDrop" width="400" >}}

9. Откройте плейлист в любой программе, поддерживающей плейлисты M3U (например, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Экран плеера VOX" caption="Используйте плейлист в VOX или любом совместимом приложении" width="400" >}}

## Как экспортировать избранное SoundCloud в плейлист M3U

Вы можете экспортировать не только плейлисты — Soundy также поддерживает экспорт избранного, загруженных треков и репостов из вашего или любого публичного профиля SoundCloud.

Чтобы экспортировать избранное:

1. Откройте вкладку **Favorites** и нажмите кнопку **More** в правом верхнем углу.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Экран Favorites в Soundy" caption="Откройте параметры на вкладке Favorites" width="400" >}}

2. Выберите действие **Export to M3U**, чтобы начать процесс экспорта.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Экран Export To M3U в Soundy" caption="Экспортируйте избранное SoundCloud в M3U" width="400" >}}

На этом руководство завершено. Больше информации о приложении Soundy можно найти [здесь](/products/soundy).

## Часто задаваемые вопросы

{{% details title="Что такое файл M3U?" closed="true" %}}
M3U — это стандартный формат плейлистов, поддерживаемый большинством музыкальных плееров. Он содержит список путей к файлам или URL-адресов, указывающих на аудиотреки. При экспорте плейлиста SoundCloud в M3U файл содержит URL-адреса потоковой передачи для каждого трека.
{{% /details %}}

{{% details title="Могу ли я экспортировать плейлисты других пользователей SoundCloud?" closed="true" %}}
Да. Soundy может экспортировать плейлисты, избранное и репосты из любого публичного профиля SoundCloud, а не только из вашего аккаунта.
{{% /details %}}

{{% details title="Какие музыкальные плееры поддерживают плейлисты M3U?" closed="true" %}}
Большинство музыкальных плееров поддерживают M3U, включая VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic и Flacbox.
{{% /details %}}

{{% details title="Можно ли экспортировать репосты в M3U?" closed="true" %}}
Да. Soundy поддерживает экспорт плейлистов, избранного, загруженных треков и репостов. Процесс одинаковый: нажмите кнопку More и выберите Export to M3U.
{{% /details %}}

{{% details title="Есть ли ограничение на количество экспортируемых треков?" closed="true" %}}
Soundy экспортирует все треки из выбранного плейлиста или коллекции. Приложение не устанавливает искусственных ограничений.
{{% /details %}}

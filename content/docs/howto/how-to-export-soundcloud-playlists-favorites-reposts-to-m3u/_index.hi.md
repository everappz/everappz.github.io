**संक्षेप में:** किसी भी SoundCloud प्लेलिस्ट, पसंदीदा या रीपोस्ट को M3U फ़ाइल के रूप में निर्यात करने के लिए **Soundy** ऐप का उपयोग करें। फ़ाइल को AirDrop के माध्यम से अपने Mac पर स्थानांतरित करें और किसी भी संगत संगीत प्लेयर में खोलें।

## SoundCloud प्लेलिस्ट को M3U प्रारूप में कैसे निर्यात करें

1. [Soundy](http://bit.ly/soundyappstore) इंस्टॉल करें।  
2. अपने SoundCloud खाते में लॉगिन करें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy लॉगिन स्क्रीन" caption="Soundy में अपने SoundCloud खाते में लॉगिन करें" width="400" >}}

3. 'मेरी प्रोफ़ाइल' स्क्रीन में 'प्लेलिस्ट' चुनें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy मेरी प्रोफ़ाइल स्क्रीन" caption="प्लेलिस्ट पर जाएँ" width="400" >}}

4. जिस प्लेलिस्ट को निर्यात करना चाहते हैं उस पर 'अधिक' बटन टैप करें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy मेरी प्लेलिस्ट स्क्रीन" caption="प्लेलिस्ट विकल्प एक्सेस करें" width="400" >}}

5. ड्रॉप-डाउन मेनू में 'Export to M3U' चुनें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy विकल्प स्क्रीन" caption="प्लेलिस्ट को M3U प्रारूप में निर्यात करें" width="400" >}}

6. गतिविधि संकेतक चलने के दौरान कुछ सेकंड प्रतीक्षा करें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy लोडिंग स्क्रीन" caption="निर्यात प्रगति पर है..." width="400" >}}

7. निर्यात पूरा होने पर, मानक 'Open In' संवाद दिखाई देगा ताकि आप जनरेट की गई M3U प्लेलिस्ट के लिए गंतव्य चुन सकें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy निर्यात स्क्रीन" caption="अपनी M3U प्लेलिस्ट सहेजें या साझा करें" width="400" >}}

8. आप AirDrop का उपयोग करके प्लेलिस्ट को अपने Mac पर स्थानांतरित कर सकते हैं।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop स्क्रीन" caption="AirDrop का उपयोग करके M3U प्लेलिस्ट स्थानांतरित करें" width="400" >}}

9. M3U प्लेलिस्ट का समर्थन करने वाले किसी भी प्रोग्राम में प्लेलिस्ट खोलें (जैसे, VOX, WinAmp, Windows Media Player, iTunes)।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX प्लेयर स्क्रीन" caption="VOX या किसी भी संगत ऐप में अपनी प्लेलिस्ट का उपयोग करें" width="400" >}}

## SoundCloud पसंदीदा को M3U प्लेलिस्ट में कैसे निर्यात करें

आप केवल प्लेलिस्ट ही नहीं निर्यात कर सकते — Soundy आपके पसंदीदा, अपलोड किए गए गाने और अपनी या किसी भी सार्वजनिक SoundCloud प्रोफ़ाइल से रीपोस्ट निर्यात करने का भी समर्थन करता है।

पसंदीदा निर्यात करने के लिए:

1. **पसंदीदा** टैब खोलें और ऊपरी-दाएँ कोने में **अधिक** बटन टैप करें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy पसंदीदा स्क्रीन" caption="पसंदीदा टैब में विकल्प एक्सेस करें" width="400" >}}

2. निर्यात प्रक्रिया शुरू करने के लिए **Export to M3U** क्रिया चुनें।

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy M3U में निर्यात स्क्रीन" caption="SoundCloud पसंदीदा को M3U में निर्यात करें" width="400" >}}

इस ट्यूटोरियल के लिए बस इतना ही। आप Soundy ऐप के बारे में अधिक जानकारी [यहाँ](/products/soundy) पा सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

{{% details title="M3U फ़ाइल क्या है?" closed="true" %}}
M3U एक मानक प्लेलिस्ट प्रारूप है जो अधिकांश संगीत प्लेयर द्वारा समर्थित है। इसमें ऑडियो ट्रैक की ओर इशारा करने वाले फ़ाइल पथ या URL की सूची होती है। जब आप SoundCloud प्लेलिस्ट को M3U में निर्यात करते हैं, तो फ़ाइल में प्रत्येक ट्रैक के लिए स्ट्रीमिंग URL होते हैं।
{{% /details %}}

{{% details title="क्या मैं अन्य SoundCloud उपयोगकर्ताओं की प्लेलिस्ट निर्यात कर सकता हूँ?" closed="true" %}}
हाँ। Soundy किसी भी सार्वजनिक SoundCloud प्रोफ़ाइल से प्लेलिस्ट, पसंदीदा और रीपोस्ट निर्यात कर सकता है, न कि केवल आपके अपने खाते से।
{{% /details %}}

{{% details title="कौन से संगीत प्लेयर M3U प्लेलिस्ट का समर्थन करते हैं?" closed="true" %}}
अधिकांश संगीत प्लेयर M3U का समर्थन करते हैं, जिनमें VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic और Flacbox शामिल हैं।
{{% /details %}}

{{% details title="क्या मैं रीपोस्ट भी M3U में निर्यात कर सकता हूँ?" closed="true" %}}
हाँ। Soundy प्लेलिस्ट, पसंदीदा, अपलोड किए गए गाने और रीपोस्ट निर्यात करने का समर्थन करता है। प्रक्रिया वही है: अधिक बटन टैप करें और Export to M3U चुनें।
{{% /details %}}

{{% details title="क्या मैं कितने ट्रैक निर्यात कर सकता हूँ इसकी कोई सीमा है?" closed="true" %}}
Soundy चयनित प्लेलिस्ट या संग्रह में सभी ट्रैक निर्यात करता है। ऐप द्वारा कोई कृत्रिम सीमा नहीं लगाई गई है।
{{% /details %}}

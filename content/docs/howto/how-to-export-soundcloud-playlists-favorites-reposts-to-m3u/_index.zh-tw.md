**簡要概述：** 使用 **Soundy** 應用程式將任何 SoundCloud 播放列表、收藏或轉發匯出為 M3U 檔案。透過 AirDrop 將檔案傳輸到您的 Mac，然後在任何相容的音樂播放器中開啟。

## 如何將 SoundCloud 播放列表匯出為 M3U 格式

1. 安裝 [Soundy](http://bit.ly/soundyappstore)。
2. 登入您的 SoundCloud 帳戶。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy 登入畫面" caption="在 Soundy 中登入您的 SoundCloud 帳戶" width="400" >}}

3. 在「我的個人檔案」畫面中選擇「播放列表」。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy 我的個人檔案畫面" caption="導覽至播放列表" width="400" >}}

4. 點擊要匯出的播放列表上的「更多」按鈕。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy 我的播放列表畫面" caption="存取播放列表選項" width="400" >}}

5. 在下拉選單中選擇「匯出為 M3U」。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy 選項畫面" caption="將播放列表匯出為 M3U 格式" width="400" >}}

6. 等待幾秒鐘，活動指示器正在執行。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy 載入畫面" caption="正在匯出..." width="400" >}}

7. 匯出完成後，將出現標準的「開啟方式」對話框，您可以為產生的 M3U 播放列表選擇目標位置。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy 匯出畫面" caption="儲存或分享您的 M3U 播放列表" width="400" >}}

8. 您可以使用 AirDrop 將播放列表傳輸到您的 Mac。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop 畫面" caption="使用 AirDrop 傳輸 M3U 播放列表" width="400" >}}

9. 在任何支援 M3U 播放列表的程式中開啟（例如 VOX、WinAmp、Windows Media Player、iTunes）。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX 播放器畫面" caption="在 VOX 或任何相容的應用程式中使用您的播放列表" width="400" >}}

## 如何將 SoundCloud 收藏匯出為 M3U 播放列表

您不僅可以匯出播放列表——Soundy 還支援從您自己的或任何公開的 SoundCloud 個人檔案中匯出收藏、已上傳的歌曲和轉發。

要匯出收藏：

1. 開啟**收藏**標籤頁，點擊右上角的**更多**按鈕。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy 收藏畫面" caption="在收藏標籤頁中存取選項" width="400" >}}

2. 選擇**匯出為 M3U** 動作以開始匯出過程。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy 匯出為 M3U 畫面" caption="將 SoundCloud 收藏匯出為 M3U" width="400" >}}

本教學到此結束。您可以在[此處](/products/soundy)找到有關 Soundy 應用程式的更多資訊。

## 常見問題

{{% details title="什麼是 M3U 檔案？" closed="true" %}}
M3U 是大多數音樂播放器支援的標準播放列表格式。它包含指向音訊曲目的檔案路徑或 URL 列表。當您將 SoundCloud 播放列表匯出為 M3U 時，該檔案包含每個曲目的串流 URL。
{{% /details %}}

{{% details title="我可以匯出其他 SoundCloud 使用者的播放列表嗎？" closed="true" %}}
可以。Soundy 可以從任何公開的 SoundCloud 個人檔案匯出播放列表、收藏和轉發，不僅限於您自己的帳戶。
{{% /details %}}

{{% details title="哪些音樂播放器支援 M3U 播放列表？" closed="true" %}}
大多數音樂播放器都支援 M3U，包括 VLC、WinAmp、Windows Media Player、iTunes、foobar2000、Evermusic 和 Flacbox。
{{% /details %}}

{{% details title="我也可以將轉發匯出為 M3U 嗎？" closed="true" %}}
可以。Soundy 支援匯出播放列表、收藏、已上傳的歌曲和轉發。操作方法相同：點擊「更多」按鈕並選擇「匯出為 M3U」。
{{% /details %}}

{{% details title="匯出的曲目數量有限制嗎？" closed="true" %}}
Soundy 會匯出所選播放列表或合集中的所有曲目。該應用程式沒有設置人為限制。
{{% /details %}}

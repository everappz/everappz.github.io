**En bref :** Utilisez l'application **Soundy** pour exporter n'importe quelle playlist, vos favoris ou reposts SoundCloud sous forme de fichier M3U. Transférez le fichier sur votre Mac via AirDrop et ouvrez-le dans n'importe quel lecteur de musique compatible.

## Comment exporter une playlist SoundCloud au format M3U

1. Installez [Soundy](http://bit.ly/soundyappstore).  
2. Connectez-vous à votre compte SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Écran de connexion Soundy" caption="Connectez-vous à votre compte SoundCloud dans Soundy" width="400" >}}

3. Sélectionnez « Playlists » dans l'écran « Mon profil ».

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Écran Mon profil de Soundy" caption="Accédez aux Playlists" width="400" >}}

4. Appuyez sur le bouton « Plus » sur la playlist que vous souhaitez exporter.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Écran Mes playlists de Soundy" caption="Accédez aux options de la playlist" width="400" >}}

5. Sélectionnez « Exporter en M3U » dans le menu déroulant.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Écran des options de Soundy" caption="Exporter la playlist au format M3U" width="400" >}}

6. Attendez quelques secondes pendant que l'indicateur d'activité tourne.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Écran de chargement de Soundy" caption="Export en cours..." width="400" >}}

7. Une fois l'export terminé, la boîte de dialogue standard « Ouvrir dans » apparaîtra pour vous permettre de sélectionner une destination pour la playlist M3U générée.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Écran d'export de Soundy" caption="Enregistrez ou partagez votre playlist M3U" width="400" >}}

8. Vous pouvez transférer la playlist sur votre Mac en utilisant AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Écran AirDrop de Soundy" caption="Transférez la playlist M3U via AirDrop" width="400" >}}

9. Ouvrez la playlist dans n'importe quel programme prenant en charge les playlists M3U (par exemple, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Écran du lecteur VOX" caption="Utilisez votre playlist dans VOX ou toute application compatible" width="400" >}}

## Comment exporter les favoris SoundCloud en playlist M3U

Vous pouvez exporter bien plus que des playlists — Soundy prend également en charge l'export de vos favoris, chansons téléchargées et reposts depuis votre propre profil ou tout profil public SoundCloud.

Pour exporter les favoris :

1. Ouvrez l'onglet **Favoris** et appuyez sur le bouton **Plus** dans le coin supérieur droit.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Écran des favoris de Soundy" caption="Accédez aux options dans l'onglet Favoris" width="400" >}}

2. Sélectionnez l'action **Exporter en M3U** pour lancer le processus d'export.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Écran Exporter en M3U de Soundy" caption="Exporter les favoris SoundCloud en M3U" width="400" >}}

C'est tout pour ce tutoriel. Vous pouvez trouver plus d'informations sur l'application Soundy [ici](/products/soundy).

## Questions fréquemment posées

{{% details title="Qu'est-ce qu'un fichier M3U ?" closed="true" %}}
M3U est un format de playlist standard pris en charge par la plupart des lecteurs de musique. Il contient une liste de chemins de fichiers ou d'URL pointant vers des pistes audio. Lorsque vous exportez une playlist SoundCloud en M3U, le fichier contient les URL de streaming de chaque piste.
{{% /details %}}

{{% details title="Puis-je exporter des playlists d'autres utilisateurs SoundCloud ?" closed="true" %}}
Oui. Soundy peut exporter les playlists, favoris et reposts de n'importe quel profil public SoundCloud, pas seulement de votre propre compte.
{{% /details %}}

{{% details title="Quels lecteurs de musique prennent en charge les playlists M3U ?" closed="true" %}}
La plupart des lecteurs de musique prennent en charge le format M3U, notamment VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic et Flacbox.
{{% /details %}}

{{% details title="Puis-je aussi exporter les reposts en M3U ?" closed="true" %}}
Oui. Soundy prend en charge l'export des playlists, favoris, chansons téléchargées et reposts. Le processus est le même : appuyez sur le bouton Plus et sélectionnez Exporter en M3U.
{{% /details %}}

{{% details title="Y a-t-il une limite au nombre de pistes exportables ?" closed="true" %}}
Soundy exporte toutes les pistes de la playlist ou collection sélectionnée. L'application n'impose aucune limite artificielle.
{{% /details %}}

**요약:** **Soundy** 앱을 사용하여 SoundCloud 플레이리스트, 좋아요 또는 리포스트를 M3U 파일로 내보낼 수 있습니다. AirDrop을 통해 파일을 Mac으로 전송하고 호환되는 음악 플레이어에서 열어보세요.

{{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1d33da9a468b4b6aad57393aae61f763~mv2_d_4500_3450_s_4_2.jpg" alt="Soundy 플레이리스트 내보내기" caption="Soundy로 SoundCloud 플레이리스트를 M3U 형식으로 내보내기" width="800" >}}

## SoundCloud 플레이리스트를 M3U 형식으로 내보내는 방법

1. [Soundy](http://bit.ly/soundyappstore)를 설치합니다.  
2. SoundCloud 계정에 로그인합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy 로그인 화면" caption="Soundy에서 SoundCloud 계정에 로그인" width="400" >}}

3. '내 프로필' 화면에서 '플레이리스트'를 선택합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy 내 프로필 화면" caption="플레이리스트로 이동" width="400" >}}

4. 내보내려는 플레이리스트의 '더보기' 버튼을 탭합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy 내 플레이리스트 화면" caption="플레이리스트 옵션에 접근" width="400" >}}

5. 드롭다운 메뉴에서 'M3U로 내보내기'를 선택합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy 옵션 화면" caption="플레이리스트를 M3U 형식으로 내보내기" width="400" >}}

6. 활동 표시기가 실행되는 동안 몇 초간 기다립니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy 로딩 화면" caption="내보내기 진행 중..." width="400" >}}

7. 내보내기가 완료되면 표준 '열기' 대화상자가 나타나 생성된 M3U 플레이리스트의 저장 위치를 선택할 수 있습니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy 내보내기 화면" caption="M3U 플레이리스트 저장 또는 공유" width="400" >}}

8. AirDrop을 사용하여 플레이리스트를 Mac으로 전송할 수 있습니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop 화면" caption="AirDrop으로 M3U 플레이리스트 전송" width="400" >}}

9. M3U 플레이리스트를 지원하는 프로그램에서 플레이리스트를 엽니다 (예: VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX 플레이어 화면" caption="VOX 또는 호환되는 앱에서 플레이리스트 사용" width="400" >}}

## SoundCloud 좋아요를 M3U 플레이리스트로 내보내는 방법

플레이리스트뿐만 아니라 Soundy는 자신의 프로필이나 공개된 SoundCloud 프로필에서 좋아요, 업로드한 곡, 리포스트를 내보내는 것도 지원합니다.

좋아요를 내보내려면:

1. **좋아요** 탭을 열고 오른쪽 상단의 **더보기** 버튼을 탭합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy 좋아요 화면" caption="좋아요 탭에서 옵션에 접근" width="400" >}}

2. **M3U로 내보내기** 작업을 선택하여 내보내기 프로세스를 시작합니다.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy M3U 내보내기 화면" caption="SoundCloud 좋아요를 M3U로 내보내기" width="400" >}}

이 튜토리얼은 여기까지입니다. Soundy 앱에 대한 자세한 정보는 [여기](/products/soundy)에서 확인할 수 있습니다.

## 자주 묻는 질문

{{% details title="M3U 파일이란 무엇인가요?" closed="true" %}}
M3U는 대부분의 음악 플레이어에서 지원하는 표준 플레이리스트 형식입니다. 오디오 트랙을 가리키는 파일 경로 또는 URL 목록이 포함되어 있습니다. SoundCloud 플레이리스트를 M3U로 내보내면 파일에 각 트랙의 스트리밍 URL이 포함됩니다.
{{% /details %}}

{{% details title="다른 SoundCloud 사용자의 플레이리스트도 내보낼 수 있나요?" closed="true" %}}
네. Soundy는 자신의 계정뿐만 아니라 공개된 모든 SoundCloud 프로필에서 플레이리스트, 좋아요, 리포스트를 내보낼 수 있습니다.
{{% /details %}}

{{% details title="M3U 플레이리스트를 지원하는 음악 플레이어는?" closed="true" %}}
대부분의 음악 플레이어가 M3U를 지원합니다. VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic, Flacbox 등이 있습니다.
{{% /details %}}

{{% details title="리포스트도 M3U로 내보낼 수 있나요?" closed="true" %}}
네. Soundy는 플레이리스트, 좋아요, 업로드한 곡, 리포스트 내보내기를 지원합니다. 과정은 동일합니다: 더보기 버튼을 탭하고 M3U로 내보내기를 선택하세요.
{{% /details %}}

{{% details title="내보낼 수 있는 트랙 수에 제한이 있나요?" closed="true" %}}
Soundy는 선택한 플레이리스트 또는 컬렉션의 모든 트랙을 내보냅니다. 앱에서 부과하는 인위적인 제한은 없습니다.
{{% /details %}}

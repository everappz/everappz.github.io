**要約:** **Soundy**アプリを使用して、SoundCloudのプレイリスト、お気に入り、またはリポストをM3Uファイルとしてエクスポートできます。AirDropでファイルをMacに転送し、互換性のある音楽プレーヤーで開きましょう。

## SoundCloudのプレイリストをM3U形式にエクスポートする方法

1. [Soundy](http://bit.ly/soundyappstore)をインストールします。  
2. SoundCloudアカウントにログインします。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundyログイン画面" caption="SoundyでSoundCloudアカウントにログイン" width="400" >}}

3. 「マイプロフィール」画面で「プレイリスト」を選択します。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundyマイプロフィール画面" caption="プレイリストに移動" width="400" >}}

4. エクスポートしたいプレイリストの「その他」ボタンをタップします。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundyマイプレイリスト画面" caption="プレイリストのオプションにアクセス" width="400" >}}

5. ドロップダウンメニューで「M3Uにエクスポート」を選択します。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundyオプション画面" caption="プレイリストをM3U形式にエクスポート" width="400" >}}

6. アクティビティインジケーターが動作している間、数秒お待ちください。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy読み込み画面" caption="エクスポート中..." width="400" >}}

7. エクスポートが完了すると、標準の「共有」ダイアログが表示され、生成されたM3Uプレイリストの保存先を選択できます。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundyエクスポート画面" caption="M3Uプレイリストを保存または共有" width="400" >}}

8. AirDropを使用してプレイリストをMacに転送できます。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop画面" caption="AirDropでM3Uプレイリストを転送" width="400" >}}

9. M3Uプレイリストをサポートする任意のプログラムでプレイリストを開きます（例：VOX、WinAmp、Windows Media Player、iTunes）。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOXプレーヤー画面" caption="VOXまたは互換性のあるアプリでプレイリストを使用" width="400" >}}

## SoundCloudのお気に入りをM3Uプレイリストにエクスポートする方法

プレイリストだけでなく、Soundyはお気に入り、アップロードした曲、リポストを自分のプロフィールや公開されている任意のSoundCloudプロフィールからエクスポートすることもできます。

お気に入りをエクスポートするには：

1. **お気に入り**タブを開き、右上の**その他**ボタンをタップします。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundyお気に入り画面" caption="お気に入りタブのオプションにアクセス" width="400" >}}

2. **M3Uにエクスポート**アクションを選択してエクスポートプロセスを開始します。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="SoundyのM3Uエクスポート画面" caption="SoundCloudのお気に入りをM3Uにエクスポート" width="400" >}}

このチュートリアルは以上です。Soundyアプリの詳細については[こちら](/products/soundy)をご覧ください。

## よくある質問

{{% details title="M3Uファイルとは何ですか？" closed="true" %}}
M3Uは、ほとんどの音楽プレーヤーでサポートされている標準的なプレイリスト形式です。オーディオトラックを指すファイルパスまたはURLのリストが含まれています。SoundCloudのプレイリストをM3Uにエクスポートすると、ファイルには各トラックのストリーミングURLが含まれます。
{{% /details %}}

{{% details title="他のSoundCloudユーザーのプレイリストをエクスポートできますか？" closed="true" %}}
はい。Soundyは、自分のアカウントだけでなく、公開されている任意のSoundCloudプロフィールからプレイリスト、お気に入り、リポストをエクスポートできます。
{{% /details %}}

{{% details title="M3Uプレイリストをサポートする音楽プレーヤーは？" closed="true" %}}
ほとんどの音楽プレーヤーがM3Uをサポートしています。VLC、WinAmp、Windows Media Player、iTunes、foobar2000、Evermusic、Flacboxなどがあります。
{{% /details %}}

{{% details title="リポストもM3Uにエクスポートできますか？" closed="true" %}}
はい。Soundyはプレイリスト、お気に入り、アップロードした曲、リポストのエクスポートをサポートしています。手順は同じです：「その他」ボタンをタップし、「M3Uにエクスポート」を選択します。
{{% /details %}}

{{% details title="エクスポートできるトラック数に制限はありますか？" closed="true" %}}
Soundyは選択したプレイリストまたはコレクション内のすべてのトラックをエクスポートします。アプリによる人為的な制限はありません。
{{% /details %}}

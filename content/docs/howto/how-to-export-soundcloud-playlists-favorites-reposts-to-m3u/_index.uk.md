**Коротко:** Використовуйте додаток **Soundy**, щоб експортувати будь-який плейлист, вподобання або репости SoundCloud у файл M3U. Перенесіть файл через AirDrop на ваш Mac і відкрийте його в будь-якому сумісному музичному програвачі.

## Як експортувати плейлист SoundCloud у формат M3U

1. Встановіть [Soundy](http://bit.ly/soundyappstore).
2. Увійдіть у свій обліковий запис SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Екран входу в Soundy" caption="Увійдіть у свій обліковий запис SoundCloud у Soundy" width="400" >}}

3. Виберіть «Плейлисти» на екрані «Мій профіль».

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Екран «Мій профіль» у Soundy" caption="Перейдіть до плейлистів" width="400" >}}

4. Натисніть кнопку «Більше» на плейлисті, який ви хочете експортувати.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Екран «Мої плейлисти» в Soundy" caption="Відкрийте параметри плейлиста" width="400" >}}

5. Виберіть «Експорт у M3U» у випадаючому меню.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Екран параметрів Soundy" caption="Експортуйте плейлист у формат M3U" width="400" >}}

6. Зачекайте кілька секунд, поки працює індикатор активності.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Екран завантаження Soundy" caption="Експорт виконується..." width="400" >}}

7. Після завершення експорту з'явиться стандартне діалогове вікно «Відкрити в», де ви зможете вибрати місце призначення для створеного плейлиста M3U.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Екран експорту Soundy" caption="Збережіть або поділіться своїм плейлистом M3U" width="400" >}}

8. Ви можете перенести плейлист на свій Mac за допомогою AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Екран AirDrop у Soundy" caption="Перенесіть плейлист M3U за допомогою AirDrop" width="400" >}}

9. Відкрийте плейлист у будь-якій програмі, що підтримує плейлисти M3U (наприклад, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Екран програвача VOX" caption="Використовуйте свій плейлист у VOX або будь-якому сумісному додатку" width="400" >}}

## Як експортувати вподобання SoundCloud у плейлист M3U

Ви можете експортувати не лише плейлисти — Soundy також підтримує експорт вподобань, завантажених пісень і репостів з вашого власного або будь-якого публічного профілю SoundCloud.

Щоб експортувати вподобання:

1. Відкрийте вкладку **Вподобання** та натисніть кнопку **Більше** у верхньому правому куті.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Екран вподобань у Soundy" caption="Відкрийте параметри на вкладці вподобань" width="400" >}}

2. Виберіть дію **Експорт у M3U**, щоб розпочати процес експорту.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Екран «Експорт у M3U» в Soundy" caption="Експортуйте вподобання SoundCloud у M3U" width="400" >}}

На цьому цей посібник завершено. Більше інформації про додаток Soundy можна знайти [тут](/products/soundy).

## Часті запитання

{{% details title="Що таке файл M3U?" closed="true" %}}
M3U — це стандартний формат плейлиста, який підтримується більшістю музичних програвачів. Він містить список шляхів до файлів або URL-адрес, що вказують на аудіотреки. Коли ви експортуєте плейлист SoundCloud у M3U, файл містить URL-адреси потокового відтворення для кожного треку.
{{% /details %}}

{{% details title="Чи можу я експортувати плейлисти інших користувачів SoundCloud?" closed="true" %}}
Так. Soundy може експортувати плейлисти, вподобання та репости з будь-якого публічного профілю SoundCloud, а не лише з вашого власного облікового запису.
{{% /details %}}

{{% details title="Які музичні програвачі підтримують плейлисти M3U?" closed="true" %}}
Більшість музичних програвачів підтримують M3U, зокрема VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic та Flacbox.
{{% /details %}}

{{% details title="Чи можу я також експортувати репости у M3U?" closed="true" %}}
Так. Soundy підтримує експорт плейлистів, вподобань, завантажених пісень і репостів. Процес однаковий: натисніть кнопку «Більше» і виберіть «Експорт у M3U».
{{% /details %}}

{{% details title="Чи є обмеження на кількість треків, які можна експортувати?" closed="true" %}}
Soundy експортує всі треки у вибраному плейлисті або колекції. Додаток не встановлює штучних обмежень.
{{% /details %}}

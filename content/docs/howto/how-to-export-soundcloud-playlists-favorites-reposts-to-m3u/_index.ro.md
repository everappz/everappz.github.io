**Pe scurt:** Folosiți aplicația **Soundy** pentru a exporta orice playlist SoundCloud, favorite sau repostări ca fișier M3U. Transferați fișierul prin AirDrop pe Mac și deschideți-l în orice player muzical compatibil.

## Cum să exportați un playlist SoundCloud în format M3U

1. Instalați [Soundy](http://bit.ly/soundyappstore).  
2. Conectați-vă la contul SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Ecranul de autentificare Soundy" caption="Conectați-vă la contul SoundCloud în Soundy" width="400" >}}

3. Selectați „Playlists" în ecranul „My Profile".

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Ecranul My Profile din Soundy" caption="Navigați la Playlists" width="400" >}}

4. Apăsați butonul „More" de pe playlistul pe care doriți să-l exportați.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Ecranul My Playlists din Soundy" caption="Accesați opțiunile playlistului" width="400" >}}

5. Selectați „Export to M3U" din meniul derulant.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Ecranul de opțiuni Soundy" caption="Exportați playlistul în format M3U" width="400" >}}

6. Așteptați câteva secunde în timp ce indicatorul de activitate rulează.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Ecranul de încărcare Soundy" caption="Export în curs..." width="400" >}}

7. Odată ce exportul este finalizat, va apărea dialogul standard „Open In" pentru a selecta o destinație pentru playlistul M3U generat.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Ecranul de export Soundy" caption="Salvați sau partajați playlistul M3U" width="400" >}}

8. Puteți transfera playlistul pe Mac folosind AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Ecranul AirDrop din Soundy" caption="Transferați playlistul M3U folosind AirDrop" width="400" >}}

9. Deschideți playlistul în orice program care acceptă playlisturi M3U (de ex., VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Ecranul playerului VOX" caption="Folosiți playlistul în VOX sau orice aplicație compatibilă" width="400" >}}

## Cum să exportați favoritele SoundCloud într-un playlist M3U

Puteți exporta mai mult decât playlisturi — Soundy acceptă și exportul favoritelor, melodiilor încărcate și repostărilor din propriul profil sau din orice profil public SoundCloud.

Pentru a exporta favoritele:

1. Deschideți fila **Favorites** și apăsați butonul **More** din colțul din dreapta sus.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Ecranul Favorites din Soundy" caption="Accesați opțiunile din fila Favorites" width="400" >}}

2. Selectați acțiunea **Export to M3U** pentru a începe procesul de export.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Ecranul Export To M3U din Soundy" caption="Exportați favoritele SoundCloud în M3U" width="400" >}}

Asta e tot pentru acest tutorial. Puteți găsi mai multe informații despre aplicația Soundy [aici](/products/soundy).

## Întrebări frecvente

{{% details title="Ce este un fișier M3U?" closed="true" %}}
M3U este un format standard de playlist acceptat de majoritatea playerelor muzicale. Conține o listă de căi de fișiere sau URL-uri care indică piesele audio. Când exportați un playlist SoundCloud în M3U, fișierul conține URL-uri de streaming pentru fiecare piesă.
{{% /details %}}

{{% details title="Pot exporta playlisturi de la alți utilizatori SoundCloud?" closed="true" %}}
Da. Soundy poate exporta playlisturi, favorite și repostări din orice profil public SoundCloud, nu doar din contul dumneavoastră.
{{% /details %}}

{{% details title="Care playere muzicale acceptă playlisturi M3U?" closed="true" %}}
Majoritatea playerelor muzicale acceptă M3U, inclusiv VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic și Flacbox.
{{% /details %}}

{{% details title="Pot exporta și repostările în M3U?" closed="true" %}}
Da. Soundy acceptă exportul playlisturilor, favoritelor, melodiilor încărcate și repostărilor. Procesul este același: apăsați butonul More și selectați Export to M3U.
{{% /details %}}

{{% details title="Există o limită pentru numărul de piese pe care le pot exporta?" closed="true" %}}
Soundy exportă toate piesele din playlistul sau colecția selectată. Nu există o limită artificială impusă de aplicație.
{{% /details %}}

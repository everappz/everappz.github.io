**简要概述：** 使用 **Soundy** 应用将任何 SoundCloud 播放列表、收藏或转发导出为 M3U 文件。通过 AirDrop 将文件传输到您的 Mac，然后在任何兼容的音乐播放器中打开。

## 如何将 SoundCloud 播放列表导出为 M3U 格式

1. 安装 [Soundy](http://bit.ly/soundyappstore)。
2. 登录您的 SoundCloud 账户。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy 登录界面" caption="在 Soundy 中登录您的 SoundCloud 账户" width="400" >}}

3. 在"我的资料"界面中选择"播放列表"。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy 我的资料界面" caption="导航到播放列表" width="400" >}}

4. 点击要导出的播放列表上的"更多"按钮。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy 我的播放列表界面" caption="访问播放列表选项" width="400" >}}

5. 在下拉菜单中选择"导出为 M3U"。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy 选项界面" caption="将播放列表导出为 M3U 格式" width="400" >}}

6. 等待几秒钟，活动指示器正在运行。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy 加载界面" caption="正在导出..." width="400" >}}

7. 导出完成后，将出现标准的"打开方式"对话框，您可以为生成的 M3U 播放列表选择目标位置。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy 导出界面" caption="保存或分享您的 M3U 播放列表" width="400" >}}

8. 您可以使用 AirDrop 将播放列表传输到您的 Mac。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop 界面" caption="使用 AirDrop 传输 M3U 播放列表" width="400" >}}

9. 在任何支持 M3U 播放列表的程序中打开（例如 VOX、WinAmp、Windows Media Player、iTunes）。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX 播放器界面" caption="在 VOX 或任何兼容的应用中使用您的播放列表" width="400" >}}

## 如何将 SoundCloud 收藏导出为 M3U 播放列表

您不仅可以导出播放列表——Soundy 还支持从您自己的或任何公开的 SoundCloud 个人资料中导出收藏、已上传的歌曲和转发。

要导出收藏：

1. 打开**收藏**标签页，点击右上角的**更多**按钮。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy 收藏界面" caption="在收藏标签页中访问选项" width="400" >}}

2. 选择**导出为 M3U** 操作以开始导出过程。

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy 导出为 M3U 界面" caption="将 SoundCloud 收藏导出为 M3U" width="400" >}}

本教程到此结束。您可以在[此处](/products/soundy)找到有关 Soundy 应用的更多信息。

## 常见问题

{{% details title="什么是 M3U 文件？" closed="true" %}}
M3U 是大多数音乐播放器支持的标准播放列表格式。它包含指向音频曲目的文件路径或 URL 列表。当您将 SoundCloud 播放列表导出为 M3U 时，该文件包含每个曲目的流媒体 URL。
{{% /details %}}

{{% details title="我可以导出其他 SoundCloud 用户的播放列表吗？" closed="true" %}}
可以。Soundy 可以从任何公开的 SoundCloud 个人资料导出播放列表、收藏和转发，不仅限于您自己的账户。
{{% /details %}}

{{% details title="哪些音乐播放器支持 M3U 播放列表？" closed="true" %}}
大多数音乐播放器都支持 M3U，包括 VLC、WinAmp、Windows Media Player、iTunes、foobar2000、Evermusic 和 Flacbox。
{{% /details %}}

{{% details title="我也可以将转发导出为 M3U 吗？" closed="true" %}}
可以。Soundy 支持导出播放列表、收藏、已上传的歌曲和转发。操作方法相同：点击"更多"按钮并选择"导出为 M3U"。
{{% /details %}}

{{% details title="导出的曲目数量有限制吗？" closed="true" %}}
Soundy 会导出所选播放列表或合集中的所有曲目。该应用没有设置人为限制。
{{% /details %}}

**Resumo:** Use o aplicativo **Soundy** para exportar qualquer playlist do SoundCloud, favoritos ou reposts como um arquivo M3U. Transfira o arquivo via AirDrop para o seu Mac e abra-o em qualquer reprodutor de música compatível.

## Como exportar uma playlist do SoundCloud para o formato M3U

1. Instale o [Soundy](http://bit.ly/soundyappstore).  
2. Faça login na sua conta do SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Tela de login do Soundy" caption="Faça login na sua conta do SoundCloud no Soundy" width="400" >}}

3. Selecione "Playlists" na tela "Meu Perfil".

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Tela Meu Perfil do Soundy" caption="Navegue até Playlists" width="400" >}}

4. Toque no botão "Mais" na playlist que deseja exportar.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Tela Minhas Playlists do Soundy" caption="Acesse as opções da playlist" width="400" >}}

5. Selecione "Exportar para M3U" no menu suspenso.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Tela de opções do Soundy" caption="Exportar playlist para o formato M3U" width="400" >}}

6. Aguarde alguns segundos enquanto o indicador de atividade é executado.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Tela de carregamento do Soundy" caption="Exportação em andamento..." width="400" >}}

7. Quando a exportação estiver concluída, a caixa de diálogo padrão "Abrir em" aparecerá para que você possa selecionar um destino para a playlist M3U gerada.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Tela de exportação do Soundy" caption="Salve ou compartilhe sua playlist M3U" width="400" >}}

8. Você pode transferir a playlist para o seu Mac usando o AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Tela do AirDrop no Soundy" caption="Transferir playlist M3U usando o AirDrop" width="400" >}}

9. Abra a playlist em qualquer programa que suporte playlists M3U (ex.: VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Tela do reprodutor VOX" caption="Use sua playlist no VOX ou em qualquer aplicativo compatível" width="400" >}}

## Como exportar favoritos do SoundCloud para uma playlist M3U

Você pode exportar mais do que apenas playlists — o Soundy também suporta a exportação de favoritos, músicas enviadas e reposts do seu próprio perfil ou de qualquer perfil público do SoundCloud.

Para exportar favoritos:

1. Abra a aba **Favoritos** e toque no botão **Mais** no canto superior direito.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Tela de favoritos do Soundy" caption="Acesse as opções na aba Favoritos" width="400" >}}

2. Selecione a ação **Exportar para M3U** para iniciar o processo de exportação.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Tela Exportar para M3U do Soundy" caption="Exportar favoritos do SoundCloud para M3U" width="400" >}}

Isso é tudo para este tutorial. Você pode encontrar mais informações sobre o aplicativo Soundy [aqui](/products/soundy).

## Perguntas frequentes

{{% details title="O que é um arquivo M3U?" closed="true" %}}
M3U é um formato padrão de playlist suportado pela maioria dos reprodutores de música. Ele contém uma lista de caminhos de arquivos ou URLs que apontam para faixas de áudio. Quando você exporta uma playlist do SoundCloud para M3U, o arquivo contém URLs de streaming para cada faixa.
{{% /details %}}

{{% details title="Posso exportar playlists de outros usuários do SoundCloud?" closed="true" %}}
Sim. O Soundy pode exportar playlists, favoritos e reposts de qualquer perfil público do SoundCloud, não apenas da sua própria conta.
{{% /details %}}

{{% details title="Quais reprodutores de música suportam playlists M3U?" closed="true" %}}
A maioria dos reprodutores de música suporta M3U, incluindo VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic e Flacbox.
{{% /details %}}

{{% details title="Posso exportar reposts para M3U também?" closed="true" %}}
Sim. O Soundy suporta a exportação de playlists, favoritos, músicas enviadas e reposts. O processo é o mesmo: toque no botão Mais e selecione Exportar para M3U.
{{% /details %}}

{{% details title="Existe um limite de quantas faixas posso exportar?" closed="true" %}}
O Soundy exporta todas as faixas da playlist ou coleção selecionada. Não há limite artificial imposto pelo aplicativo.
{{% /details %}}

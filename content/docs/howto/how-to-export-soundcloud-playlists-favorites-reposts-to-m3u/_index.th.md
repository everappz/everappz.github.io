**สรุปสั้น:** ใช้แอป **Soundy** เพื่อส่งออกเพลย์ลิสต์ รายการโปรด หรือรีโพสต์จาก SoundCloud เป็นไฟล์ M3U โอนไฟล์ผ่าน AirDrop ไปยัง Mac ของคุณและเปิดในโปรแกรมเล่นเพลงที่รองรับ

{{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1d33da9a468b4b6aad57393aae61f763~mv2_d_4500_3450_s_4_2.jpg" alt="ส่งออกเพลย์ลิสต์ด้วย Soundy" caption="ส่งออกเพลย์ลิสต์ SoundCloud เป็นรูปแบบ M3U ด้วย Soundy" width="800" >}}

## วิธีส่งออกเพลย์ลิสต์ SoundCloud เป็นรูปแบบ M3U

1. ติดตั้ง [Soundy](http://bit.ly/soundyappstore)
2. เข้าสู่ระบบบัญชี SoundCloud ของคุณ

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="หน้าจอเข้าสู่ระบบ Soundy" caption="เข้าสู่ระบบบัญชี SoundCloud ของคุณใน Soundy" width="400" >}}

3. เลือก 'เพลย์ลิสต์' ในหน้าจอ 'โปรไฟล์ของฉัน'

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="หน้าจอโปรไฟล์ของฉันใน Soundy" caption="ไปที่เพลย์ลิสต์" width="400" >}}

4. แตะปุ่ม 'เพิ่มเติม' บนเพลย์ลิสต์ที่คุณต้องการส่งออก

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="หน้าจอเพลย์ลิสต์ของฉันใน Soundy" caption="เข้าถึงตัวเลือกเพลย์ลิสต์" width="400" >}}

5. เลือก 'ส่งออกเป็น M3U' ในเมนูดรอปดาวน์

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="หน้าจอตัวเลือก Soundy" caption="ส่งออกเพลย์ลิสต์เป็นรูปแบบ M3U" width="400" >}}

6. รอสักครู่ขณะที่ตัวบ่งชี้กิจกรรมทำงาน

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="หน้าจอกำลังโหลดของ Soundy" caption="กำลังส่งออก..." width="400" >}}

7. เมื่อการส่งออกเสร็จสมบูรณ์ กล่องโต้ตอบ 'เปิดใน' มาตรฐานจะปรากฏขึ้นเพื่อให้คุณเลือกปลายทางสำหรับเพลย์ลิสต์ M3U ที่สร้างขึ้น

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="หน้าจอส่งออกของ Soundy" caption="บันทึกหรือแชร์เพลย์ลิสต์ M3U ของคุณ" width="400" >}}

8. คุณสามารถโอนเพลย์ลิสต์ไปยัง Mac ของคุณโดยใช้ AirDrop

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="หน้าจอ AirDrop ของ Soundy" caption="โอนเพลย์ลิสต์ M3U โดยใช้ AirDrop" width="400" >}}

9. เปิดเพลย์ลิสต์ในโปรแกรมใดก็ได้ที่รองรับเพลย์ลิสต์ M3U (เช่น VOX, WinAmp, Windows Media Player, iTunes)

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="หน้าจอโปรแกรมเล่น VOX" caption="ใช้เพลย์ลิสต์ของคุณใน VOX หรือแอปที่รองรับ" width="400" >}}

## วิธีส่งออกรายการโปรดจาก SoundCloud เป็นเพลย์ลิสต์ M3U

คุณสามารถส่งออกได้มากกว่าแค่เพลย์ลิสต์ — Soundy ยังรองรับการส่งออกรายการโปรด เพลงที่อัปโหลด และรีโพสต์จากโปรไฟล์ SoundCloud ของคุณเองหรือโปรไฟล์สาธารณะใดก็ได้

วิธีส่งออกรายการโปรด:

1. เปิดแท็บ **รายการโปรด** แล้วแตะปุ่ม **เพิ่มเติม** ที่มุมขวาบน

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="หน้าจอรายการโปรดของ Soundy" caption="เข้าถึงตัวเลือกในแท็บรายการโปรด" width="400" >}}

2. เลือกการดำเนินการ **ส่งออกเป็น M3U** เพื่อเริ่มกระบวนการส่งออก

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="หน้าจอส่งออกเป็น M3U ของ Soundy" caption="ส่งออกรายการโปรด SoundCloud เป็น M3U" width="400" >}}

นั่นคือทั้งหมดสำหรับบทช่วยสอนนี้ คุณสามารถหาข้อมูลเพิ่มเติมเกี่ยวกับแอป Soundy ได้[ที่นี่](/products/soundy)

{{% details title="ฉันสามารถส่งออกเพลย์ลิสต์จากผู้ใช้ SoundCloud คนอื่นได้หรือไม่?" closed="true" %}}
ได้ Soundy สามารถส่งออกเพลย์ลิสต์ รายการโปรด และรีโพสต์จากโปรไฟล์ SoundCloud สาธารณะใดก็ได้ ไม่ใช่แค่บัญชีของคุณเอง

{{% /details %}}

{{% /details %}}

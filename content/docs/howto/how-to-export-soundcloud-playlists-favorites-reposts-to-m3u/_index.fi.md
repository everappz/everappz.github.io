## Kuinka viedä SoundCloud-soittolista M3U-muotoon

1. Asenna [Soundy](http://bit.ly/soundyappstore).  
2. Kirjaudu SoundCloud-tilillesi.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy-kirjautumisnäyttö" caption="Kirjaudu SoundCloud-tilillesi Soundy-sovelluksessa" width="400" >}}

3. Valitse 'Soittolistat' 'Oma profiili' -näytössä.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy Oma profiili -näyttö" caption="Siirry Soittolistoihin" width="400" >}}

4. Napauta 'Lisää'-painiketta soittolistassa, jonka haluat viedä.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy Omat soittolistat -näyttö" caption="Avaa soittolistan asetukset" width="400" >}}

5. Valitse 'Vie M3U-muotoon' avautuvasta valikosta.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy-asetusnäyttö" caption="Vie soittolista M3U-muotoon" width="400" >}}

6. Odota muutama sekunti aktiviteetti-ilmaisimen pyöriessä.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy-latausnäyttö" caption="Vienti käynnissä..." width="400" >}}

7. Kun vienti on valmis, näyttöön tulee tavallinen 'Avaa sovelluksessa' -valintaikkuna, josta voit valita kohteen luodulle M3U-soittolistalle.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy-vientinäyttö" caption="Tallenna tai jaa M3U-soittolistasi" width="400" >}}

8. Voit siirtää soittolistan Mac-tietokoneellesi AirDropin avulla.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop -näyttö" caption="Siirrä M3U-soittolista AirDropilla" width="400" >}}

9. Avaa soittolista missä tahansa ohjelmassa, joka tukee M3U-soittolistoja (esim. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX-soittimen näyttö" caption="Käytä soittolistaasi VOX-sovelluksessa tai muussa yhteensopivassa sovelluksessa" width="400" >}}

## Kuinka viedä SoundCloud-suosikit M3U-soittolistaksi

Voit viedä muutakin kuin soittolistoja — Soundy tukee myös suosikkien, ladattujen kappaleiden ja uudelleenjakojen vientiä omasta tai mistä tahansa julkisesta SoundCloud-profiilista.

Suosikkien vienti:

1. Avaa **Suosikit**-välilehti ja napauta **Lisää**-painiketta oikeassa yläkulmassa.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy-suosikkinäyttö" caption="Avaa asetukset Suosikit-välilehdellä" width="400" >}}

2. Valitse **Vie M3U-muotoon** -toiminto aloittaaksesi vientiprosessin.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy Vie M3U-muotoon -näyttö" caption="Vie SoundCloud-suosikit M3U-muotoon" width="400" >}}

Siinä kaikki tästä ohjeesta. Lisätietoja Soundy-sovelluksesta löydät [täältä](/products/soundy).

## Usein kysytyt kysymykset

{{% details title="Mikä on M3U-tiedosto?" closed="true" %}}
M3U on vakiomuotoinen soittolistaformaatti, jota useimmat musiikkisoittimet tukevat. Se sisältää luettelon tiedostopoluista tai URL-osoitteista, jotka osoittavat ääniraitoihin. Kun viet SoundCloud-soittolistan M3U-muotoon, tiedosto sisältää suoratoistolinkit jokaiselle kappaleelle.
{{% /details %}}

{{% details title="Voinko viedä muiden SoundCloud-käyttäjien soittolistoja?" closed="true" %}}
Kyllä. Soundy voi viedä soittolistoja, suosikkeja ja uudelleenjakoja mistä tahansa julkisesta SoundCloud-profiilista, ei vain omasta tilistäsi.
{{% /details %}}

{{% details title="Mitkä musiikkisoittimet tukevat M3U-soittolistoja?" closed="true" %}}
Useimmat musiikkisoittimet tukevat M3U-muotoa, mukaan lukien VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic ja Flacbox.
{{% /details %}}

{{% details title="Voinko viedä myös uudelleenjaot M3U-muotoon?" closed="true" %}}
Kyllä. Soundy tukee soittolistojen, suosikkien, ladattujen kappaleiden ja uudelleenjakojen vientiä. Prosessi on sama: napauta Lisää-painiketta ja valitse Vie M3U-muotoon.
{{% /details %}}

{{% details title="Onko vietävien kappaleiden määrällä rajoitusta?" closed="true" %}}
Soundy vie kaikki valitun soittolistan tai kokoelman kappaleet. Sovellus ei aseta keinotekoisia rajoituksia.
{{% /details %}}

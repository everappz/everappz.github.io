**Ringkasan:** Gunakan aplikasi **Soundy** untuk mengekspor daftar putar, favorit, atau repost SoundCloud apa pun sebagai file M3U. Transfer file melalui AirDrop ke Mac Anda dan buka di pemutar musik yang kompatibel.

## Cara Mengekspor Daftar Putar SoundCloud ke Format M3U

1. Instal [Soundy](http://bit.ly/soundyappstore).  
2. Masuk ke akun SoundCloud Anda.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Layar masuk Soundy" caption="Masuk ke akun SoundCloud Anda di Soundy" width="400" >}}

3. Pilih 'Daftar Putar' di layar 'Profil Saya'.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Layar Profil Saya Soundy" caption="Navigasi ke Daftar Putar" width="400" >}}

4. Ketuk tombol 'Lainnya' pada daftar putar yang ingin Anda ekspor.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Layar Daftar Putar Saya Soundy" caption="Akses opsi daftar putar" width="400" >}}

5. Pilih 'Export to M3U' di menu tarik-turun.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Layar opsi Soundy" caption="Ekspor daftar putar ke format M3U" width="400" >}}

6. Tunggu beberapa detik sementara indikator aktivitas berjalan.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Layar pemuatan Soundy" caption="Ekspor sedang berlangsung..." width="400" >}}

7. Setelah ekspor selesai, dialog standar 'Open In' akan muncul sehingga Anda dapat memilih tujuan untuk daftar putar M3U yang dihasilkan.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Layar ekspor Soundy" caption="Simpan atau bagikan daftar putar M3U Anda" width="400" >}}

8. Anda dapat mentransfer daftar putar ke Mac Anda menggunakan AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Layar AirDrop Soundy" caption="Transfer daftar putar M3U menggunakan AirDrop" width="400" >}}

9. Buka daftar putar di program apa pun yang mendukung daftar putar M3U (misalnya, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Layar pemutar VOX" caption="Gunakan daftar putar Anda di VOX atau aplikasi kompatibel lainnya" width="400" >}}

## Cara Mengekspor Favorit SoundCloud ke Daftar Putar M3U

Anda dapat mengekspor lebih dari sekadar daftar putar — Soundy juga mendukung ekspor favorit, lagu yang diunggah, dan repost dari profil SoundCloud Anda sendiri atau profil publik mana pun.

Untuk mengekspor favorit:

1. Buka tab **Favorit** dan ketuk tombol **Lainnya** di sudut kanan atas.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Layar favorit Soundy" caption="Akses opsi di tab Favorit" width="400" >}}

2. Pilih tindakan **Export to M3U** untuk memulai proses ekspor.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Layar Ekspor ke M3U Soundy" caption="Ekspor favorit SoundCloud ke M3U" width="400" >}}

Itu saja untuk tutorial ini. Anda dapat menemukan informasi lebih lanjut tentang aplikasi Soundy [di sini](/products/soundy).

## Pertanyaan yang Sering Diajukan

{{% details title="Apa itu file M3U?" closed="true" %}}
M3U adalah format daftar putar standar yang didukung oleh sebagian besar pemutar musik. File ini berisi daftar jalur file atau URL yang mengarah ke trek audio. Ketika Anda mengekspor daftar putar SoundCloud ke M3U, file tersebut berisi URL streaming untuk setiap trek.
{{% /details %}}

{{% details title="Bisakah saya mengekspor daftar putar dari pengguna SoundCloud lain?" closed="true" %}}
Ya. Soundy dapat mengekspor daftar putar, favorit, dan repost dari profil SoundCloud publik mana pun, tidak hanya dari akun Anda sendiri.
{{% /details %}}

{{% details title="Pemutar musik mana yang mendukung daftar putar M3U?" closed="true" %}}
Sebagian besar pemutar musik mendukung M3U, termasuk VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic, dan Flacbox.
{{% /details %}}

{{% details title="Bisakah saya mengekspor repost ke M3U juga?" closed="true" %}}
Ya. Soundy mendukung ekspor daftar putar, favorit, lagu yang diunggah, dan repost. Prosesnya sama: ketuk tombol Lainnya dan pilih Export to M3U.
{{% /details %}}

{{% details title="Apakah ada batasan berapa banyak trek yang bisa saya ekspor?" closed="true" %}}
Soundy mengekspor semua trek dalam daftar putar atau koleksi yang dipilih. Tidak ada batasan buatan yang diberlakukan oleh aplikasi.
{{% /details %}}

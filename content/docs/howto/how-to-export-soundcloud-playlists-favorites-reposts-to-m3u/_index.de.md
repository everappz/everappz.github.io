**Kurzfassung:** Verwenden Sie die **Soundy**-App, um jede SoundCloud-Playlist, Favoriten oder Reposts als M3U-Datei zu exportieren. Übertragen Sie die Datei per AirDrop auf Ihren Mac und öffnen Sie sie in einem kompatiblen Musikplayer.

## So exportieren Sie eine SoundCloud-Playlist in das M3U-Format

1. Installieren Sie [Soundy](http://bit.ly/soundyappstore).  
2. Melden Sie sich bei Ihrem SoundCloud-Konto an.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy Anmeldebildschirm" caption="Melden Sie sich bei Ihrem SoundCloud-Konto in Soundy an" width="400" >}}

3. Wählen Sie „Playlists" auf dem Bildschirm „Mein Profil".

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy Mein-Profil-Bildschirm" caption="Navigieren Sie zu Playlists" width="400" >}}

4. Tippen Sie auf die Schaltfläche „Mehr" bei der Playlist, die Sie exportieren möchten.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy Meine-Playlists-Bildschirm" caption="Zugriff auf Playlist-Optionen" width="400" >}}

5. Wählen Sie „Nach M3U exportieren" im Dropdown-Menü.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy Optionen-Bildschirm" caption="Playlist in das M3U-Format exportieren" width="400" >}}

6. Warten Sie einige Sekunden, während die Aktivitätsanzeige läuft.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy Ladebildschirm" caption="Export wird durchgeführt..." width="400" >}}

7. Nach Abschluss des Exports erscheint der Standard-Dialog „Öffnen in", in dem Sie ein Ziel für die erstellte M3U-Playlist auswählen können.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy Export-Bildschirm" caption="Speichern oder teilen Sie Ihre M3U-Playlist" width="400" >}}

8. Sie können die Playlist per AirDrop auf Ihren Mac übertragen.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop-Bildschirm" caption="M3U-Playlist per AirDrop übertragen" width="400" >}}

9. Öffnen Sie die Playlist in einem Programm, das M3U-Playlists unterstützt (z. B. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX Player-Bildschirm" caption="Verwenden Sie Ihre Playlist in VOX oder einer anderen kompatiblen App" width="400" >}}

## So exportieren Sie SoundCloud-Favoriten in eine M3U-Playlist

Sie können mehr als nur Playlists exportieren — Soundy unterstützt auch den Export Ihrer Favoriten, hochgeladenen Songs und Reposts von Ihrem eigenen oder jedem öffentlichen SoundCloud-Profil.

So exportieren Sie Favoriten:

1. Öffnen Sie den Tab **Favoriten** und tippen Sie auf die Schaltfläche **Mehr** in der oberen rechten Ecke.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy Favoriten-Bildschirm" caption="Zugriff auf Optionen im Tab Favoriten" width="400" >}}

2. Wählen Sie die Aktion **Nach M3U exportieren**, um den Exportvorgang zu starten.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy Nach-M3U-exportieren-Bildschirm" caption="SoundCloud-Favoriten nach M3U exportieren" width="400" >}}

Das war alles für diese Anleitung. Weitere Informationen über die Soundy-App finden Sie [hier](/products/soundy).

## Häufig gestellte Fragen

{{% details title="Was ist eine M3U-Datei?" closed="true" %}}
M3U ist ein Standard-Playlistformat, das von den meisten Musikplayern unterstützt wird. Es enthält eine Liste von Dateipfaden oder URLs, die auf Audiotitel verweisen. Wenn Sie eine SoundCloud-Playlist nach M3U exportieren, enthält die Datei Streaming-URLs für jeden Titel.
{{% /details %}}

{{% details title="Kann ich Playlists von anderen SoundCloud-Benutzern exportieren?" closed="true" %}}
Ja. Soundy kann Playlists, Favoriten und Reposts von jedem öffentlichen SoundCloud-Profil exportieren, nicht nur von Ihrem eigenen Konto.
{{% /details %}}

{{% details title="Welche Musikplayer unterstützen M3U-Playlists?" closed="true" %}}
Die meisten Musikplayer unterstützen M3U, darunter VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic und Flacbox.
{{% /details %}}

{{% details title="Kann ich auch Reposts nach M3U exportieren?" closed="true" %}}
Ja. Soundy unterstützt den Export von Playlists, Favoriten, hochgeladenen Songs und Reposts. Der Vorgang ist derselbe: Tippen Sie auf die Schaltfläche Mehr und wählen Sie Nach M3U exportieren.
{{% /details %}}

{{% details title="Gibt es ein Limit für die Anzahl der Titel, die ich exportieren kann?" closed="true" %}}
Soundy exportiert alle Titel in der ausgewählten Playlist oder Sammlung. Die App setzt kein künstliches Limit.
{{% /details %}}

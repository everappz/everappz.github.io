**Resumen:** Usa la aplicación **Soundy** para exportar cualquier lista de reproducción, favoritos o reposts de SoundCloud como archivo M3U. Transfiere el archivo a tu Mac mediante AirDrop y ábrelo en cualquier reproductor de música compatible.

## Cómo exportar una lista de reproducción de SoundCloud a formato M3U

1. Instala [Soundy](http://bit.ly/soundyappstore).  
2. Inicia sesión en tu cuenta de SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Pantalla de inicio de sesión de Soundy" caption="Inicia sesión en tu cuenta de SoundCloud en Soundy" width="400" >}}

3. Selecciona 'Listas de reproducción' en la pantalla 'Mi perfil'.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Pantalla Mi perfil de Soundy" caption="Navega a Listas de reproducción" width="400" >}}

4. Toca el botón 'Más' en la lista de reproducción que deseas exportar.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Pantalla Mis listas de reproducción de Soundy" caption="Accede a las opciones de la lista de reproducción" width="400" >}}

5. Selecciona 'Exportar a M3U' en el menú desplegable.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Pantalla de opciones de Soundy" caption="Exportar lista de reproducción a formato M3U" width="400" >}}

6. Espera unos segundos mientras se ejecuta el indicador de actividad.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Pantalla de carga de Soundy" caption="Exportación en progreso..." width="400" >}}

7. Una vez completada la exportación, aparecerá el diálogo estándar 'Abrir en' para que puedas seleccionar un destino para la lista de reproducción M3U generada.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Pantalla de exportación de Soundy" caption="Guarda o comparte tu lista de reproducción M3U" width="400" >}}

8. Puedes transferir la lista de reproducción a tu Mac usando AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Pantalla de AirDrop de Soundy" caption="Transfiere la lista de reproducción M3U usando AirDrop" width="400" >}}

9. Abre la lista de reproducción en cualquier programa que soporte listas M3U (por ejemplo, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Pantalla del reproductor VOX" caption="Usa tu lista de reproducción en VOX o cualquier aplicación compatible" width="400" >}}

## Cómo exportar favoritos de SoundCloud a lista de reproducción M3U

Puedes exportar más que solo listas de reproducción — Soundy también permite exportar tus favoritos, canciones subidas y reposts desde tu propio perfil o cualquier perfil público de SoundCloud.

Para exportar favoritos:

1. Abre la pestaña **Favoritos** y toca el botón **Más** en la esquina superior derecha.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Pantalla de favoritos de Soundy" caption="Accede a las opciones en la pestaña Favoritos" width="400" >}}

2. Selecciona la acción **Exportar a M3U** para iniciar el proceso de exportación.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Pantalla Exportar a M3U de Soundy" caption="Exportar favoritos de SoundCloud a M3U" width="400" >}}

Eso es todo para este tutorial. Puedes encontrar más información sobre la aplicación Soundy [aquí](/products/soundy).

## Preguntas frecuentes

{{% details title="¿Qué es un archivo M3U?" closed="true" %}}
M3U es un formato estándar de lista de reproducción compatible con la mayoría de los reproductores de música. Contiene una lista de rutas de archivos o URLs que apuntan a pistas de audio. Cuando exportas una lista de reproducción de SoundCloud a M3U, el archivo contiene URLs de streaming para cada pista.
{{% /details %}}

{{% details title="¿Puedo exportar listas de reproducción de otros usuarios de SoundCloud?" closed="true" %}}
Sí. Soundy puede exportar listas de reproducción, favoritos y reposts de cualquier perfil público de SoundCloud, no solo de tu propia cuenta.
{{% /details %}}

{{% details title="¿Qué reproductores de música soportan listas M3U?" closed="true" %}}
La mayoría de los reproductores de música soportan M3U, incluyendo VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic y Flacbox.
{{% /details %}}

{{% details title="¿Puedo exportar reposts a M3U también?" closed="true" %}}
Sí. Soundy permite exportar listas de reproducción, favoritos, canciones subidas y reposts. El proceso es el mismo: toca el botón Más y selecciona Exportar a M3U.
{{% /details %}}

{{% details title="¿Hay un límite en la cantidad de pistas que puedo exportar?" closed="true" %}}
Soundy exporta todas las pistas de la lista de reproducción o colección seleccionada. La aplicación no impone ningún límite artificial.
{{% /details %}}

**Περίληψη:** Χρησιμοποιήστε την εφαρμογή **Soundy** για να εξάγετε οποιαδήποτε λίστα αναπαραγωγής, αγαπημένα ή reposts του SoundCloud ως αρχείο M3U. Μεταφέρετε το αρχείο μέσω AirDrop στο Mac σας και ανοίξτε το σε οποιοδήποτε συμβατό πρόγραμμα αναπαραγωγής μουσικής.

## Πώς να εξάγετε μια λίστα αναπαραγωγής SoundCloud σε μορφή M3U

1. Εγκαταστήστε το [Soundy](http://bit.ly/soundyappstore).  
2. Συνδεθείτε στον λογαριασμό σας στο SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Οθόνη σύνδεσης Soundy" caption="Συνδεθείτε στον λογαριασμό SoundCloud σας στο Soundy" width="400" >}}

3. Επιλέξτε «Λίστες αναπαραγωγής» στην οθόνη «Το προφίλ μου».

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Οθόνη Το προφίλ μου στο Soundy" caption="Πλοηγηθείτε στις Λίστες αναπαραγωγής" width="400" >}}

4. Πατήστε το κουμπί «Περισσότερα» στη λίστα αναπαραγωγής που θέλετε να εξάγετε.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Οθόνη Οι λίστες αναπαραγωγής μου στο Soundy" caption="Πρόσβαση στις επιλογές λίστας αναπαραγωγής" width="400" >}}

5. Επιλέξτε «Εξαγωγή σε M3U» στο αναπτυσσόμενο μενού.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Οθόνη επιλογών Soundy" caption="Εξαγωγή λίστας αναπαραγωγής σε μορφή M3U" width="400" >}}

6. Περιμένετε μερικά δευτερόλεπτα ενώ εκτελείται η ένδειξη δραστηριότητας.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Οθόνη φόρτωσης Soundy" caption="Η εξαγωγή βρίσκεται σε εξέλιξη..." width="400" >}}

7. Μόλις ολοκληρωθεί η εξαγωγή, θα εμφανιστεί το τυπικό παράθυρο διαλόγου «Άνοιγμα σε» για να επιλέξετε προορισμό για τη δημιουργημένη λίστα αναπαραγωγής M3U.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Οθόνη εξαγωγής Soundy" caption="Αποθηκεύστε ή μοιραστείτε τη λίστα αναπαραγωγής M3U" width="400" >}}

8. Μπορείτε να μεταφέρετε τη λίστα αναπαραγωγής στο Mac σας χρησιμοποιώντας το AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Οθόνη AirDrop στο Soundy" caption="Μεταφορά λίστας αναπαραγωγής M3U μέσω AirDrop" width="400" >}}

9. Ανοίξτε τη λίστα αναπαραγωγής σε οποιοδήποτε πρόγραμμα που υποστηρίζει λίστες αναπαραγωγής M3U (π.χ. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Οθόνη αναπαραγωγής VOX" caption="Χρησιμοποιήστε τη λίστα αναπαραγωγής σας στο VOX ή σε οποιαδήποτε συμβατή εφαρμογή" width="400" >}}

## Πώς να εξάγετε αγαπημένα SoundCloud σε λίστα αναπαραγωγής M3U

Μπορείτε να εξάγετε περισσότερα από απλές λίστες αναπαραγωγής — το Soundy υποστηρίζει επίσης την εξαγωγή των αγαπημένων σας, των ανεβασμένων τραγουδιών και των reposts από το δικό σας ή οποιοδήποτε δημόσιο προφίλ SoundCloud.

Για εξαγωγή αγαπημένων:

1. Ανοίξτε την καρτέλα **Αγαπημένα** και πατήστε το κουμπί **Περισσότερα** στην επάνω δεξιά γωνία.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Οθόνη αγαπημένων Soundy" caption="Πρόσβαση στις επιλογές στην καρτέλα Αγαπημένα" width="400" >}}

2. Επιλέξτε την ενέργεια **Εξαγωγή σε M3U** για να ξεκινήσει η διαδικασία εξαγωγής.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Οθόνη εξαγωγής σε M3U στο Soundy" caption="Εξαγωγή αγαπημένων SoundCloud σε M3U" width="400" >}}

Αυτά ήταν όλα για αυτόν τον οδηγό. Μπορείτε να βρείτε περισσότερες πληροφορίες σχετικά με την εφαρμογή Soundy [εδώ](/products/soundy).

## Συχνές ερωτήσεις

{{% details title="Τι είναι ένα αρχείο M3U;" closed="true" %}}
Το M3U είναι μια τυπική μορφή λίστας αναπαραγωγής που υποστηρίζεται από τα περισσότερα προγράμματα αναπαραγωγής μουσικής. Περιέχει μια λίστα διαδρομών αρχείων ή URL που δείχνουν σε κομμάτια ήχου. Όταν εξάγετε μια λίστα αναπαραγωγής SoundCloud σε M3U, το αρχείο περιέχει URL ροής για κάθε κομμάτι.
{{% /details %}}

{{% details title="Μπορώ να εξάγω λίστες αναπαραγωγής από άλλους χρήστες SoundCloud;" closed="true" %}}
Ναι. Το Soundy μπορεί να εξάγει λίστες αναπαραγωγής, αγαπημένα και reposts από οποιοδήποτε δημόσιο προφίλ SoundCloud, όχι μόνο από τον δικό σας λογαριασμό.
{{% /details %}}

{{% details title="Ποια προγράμματα αναπαραγωγής μουσικής υποστηρίζουν λίστες αναπαραγωγής M3U;" closed="true" %}}
Τα περισσότερα προγράμματα αναπαραγωγής μουσικής υποστηρίζουν M3U, συμπεριλαμβανομένων των VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic και Flacbox.
{{% /details %}}

{{% details title="Μπορώ να εξάγω και reposts σε M3U;" closed="true" %}}
Ναι. Το Soundy υποστηρίζει την εξαγωγή λιστών αναπαραγωγής, αγαπημένων, ανεβασμένων τραγουδιών και reposts. Η διαδικασία είναι η ίδια: πατήστε το κουμπί Περισσότερα και επιλέξτε Εξαγωγή σε M3U.
{{% /details %}}

{{% details title="Υπάρχει όριο στον αριθμό των κομματιών που μπορώ να εξάγω;" closed="true" %}}
Το Soundy εξάγει όλα τα κομμάτια στην επιλεγμένη λίστα αναπαραγωγής ή συλλογή. Δεν υπάρχει τεχνητό όριο από την εφαρμογή.
{{% /details %}}

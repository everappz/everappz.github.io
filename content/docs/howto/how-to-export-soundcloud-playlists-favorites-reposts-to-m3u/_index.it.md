**In breve:** Usa l'app **Soundy** per esportare qualsiasi playlist, preferiti o repost di SoundCloud come file M3U. Trasferisci il file tramite AirDrop sul tuo Mac e aprilo in qualsiasi lettore musicale compatibile.

## Come esportare una playlist di SoundCloud in formato M3U

1. Installa [Soundy](http://bit.ly/soundyappstore).  
2. Accedi al tuo account SoundCloud.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Schermata di accesso di Soundy" caption="Accedi al tuo account SoundCloud in Soundy" width="400" >}}

3. Seleziona 'Playlist' nella schermata 'Il mio profilo'.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Schermata Il mio profilo di Soundy" caption="Vai alle Playlist" width="400" >}}

4. Tocca il pulsante 'Altro' sulla playlist che desideri esportare.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Schermata Le mie playlist di Soundy" caption="Accedi alle opzioni della playlist" width="400" >}}

5. Seleziona 'Esporta in M3U' nel menu a tendina.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Schermata opzioni di Soundy" caption="Esporta la playlist in formato M3U" width="400" >}}

6. Attendi qualche secondo mentre l'indicatore di attività è in esecuzione.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Schermata di caricamento di Soundy" caption="Esportazione in corso..." width="400" >}}

7. Una volta completata l'esportazione, apparirà la finestra di dialogo standard 'Apri con' per selezionare una destinazione per la playlist M3U generata.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Schermata di esportazione di Soundy" caption="Salva o condividi la tua playlist M3U" width="400" >}}

8. Puoi trasferire la playlist sul tuo Mac utilizzando AirDrop.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Schermata AirDrop di Soundy" caption="Trasferisci la playlist M3U tramite AirDrop" width="400" >}}

9. Apri la playlist in qualsiasi programma che supporti le playlist M3U (ad esempio, VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="Schermata del lettore VOX" caption="Usa la tua playlist in VOX o in qualsiasi app compatibile" width="400" >}}

## Come esportare i preferiti di SoundCloud in una playlist M3U

Puoi esportare non solo le playlist — Soundy supporta anche l'esportazione dei tuoi preferiti, brani caricati e repost dal tuo profilo o da qualsiasi profilo pubblico di SoundCloud.

Per esportare i preferiti:

1. Apri la scheda **Preferiti** e tocca il pulsante **Altro** nell'angolo in alto a destra.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Schermata Preferiti di Soundy" caption="Accedi alle opzioni nella scheda Preferiti" width="400" >}}

2. Seleziona l'azione **Esporta in M3U** per avviare il processo di esportazione.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Schermata Esporta in M3U di Soundy" caption="Esporta i preferiti di SoundCloud in M3U" width="400" >}}

Questo è tutto per questo tutorial. Puoi trovare maggiori informazioni sull'app Soundy [qui](/products/soundy).

## Domande frequenti

{{% details title="Cos'è un file M3U?" closed="true" %}}
M3U è un formato di playlist standard supportato dalla maggior parte dei lettori musicali. Contiene un elenco di percorsi di file o URL che puntano a tracce audio. Quando esporti una playlist di SoundCloud in M3U, il file contiene gli URL di streaming per ogni traccia.
{{% /details %}}

{{% details title="Posso esportare playlist di altri utenti SoundCloud?" closed="true" %}}
Sì. Soundy può esportare playlist, preferiti e repost da qualsiasi profilo pubblico di SoundCloud, non solo dal tuo account.
{{% /details %}}

{{% details title="Quali lettori musicali supportano le playlist M3U?" closed="true" %}}
La maggior parte dei lettori musicali supporta M3U, tra cui VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic e Flacbox.
{{% /details %}}

{{% details title="Posso esportare anche i repost in M3U?" closed="true" %}}
Sì. Soundy supporta l'esportazione di playlist, preferiti, brani caricati e repost. Il processo è lo stesso: tocca il pulsante Altro e seleziona Esporta in M3U.
{{% /details %}}

{{% details title="C'è un limite al numero di tracce che posso esportare?" closed="true" %}}
Soundy esporta tutte le tracce nella playlist o nella raccolta selezionata. Non c'è alcun limite artificiale imposto dall'app.
{{% /details %}}

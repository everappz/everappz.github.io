**Röviden:** Használja a **Soundy** alkalmazást bármely SoundCloud lejátszási lista, kedvencek vagy újramegosztások M3U fájlként történő exportálásához. Vigye át a fájlt AirDrop segítségével Mac gépére, és nyissa meg bármely kompatibilis zenelejátszóban.

{{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1d33da9a468b4b6aad57393aae61f763~mv2_d_4500_3450_s_4_2.jpg" alt="Soundy lejátszási lista exportálás" caption="SoundCloud lejátszási listák exportálása M3U formátumba a Soundy segítségével" width="800" >}}

## Hogyan exportálható egy SoundCloud lejátszási lista M3U formátumba

1. Telepítse a [Soundy](http://bit.ly/soundyappstore) alkalmazást.  
2. Jelentkezzen be SoundCloud fiókjába.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c05275d6d7094a2788119a33593d8df1~mv2.jpg" alt="Soundy bejelentkezési képernyő" caption="Jelentkezzen be SoundCloud fiókjába a Soundy alkalmazásban" width="400" >}}

3. Válassza a 'Lejátszási listák' lehetőséget a 'Saját profil' képernyőn.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_083edbef3e0244fc82688e9dfbcc7ab3~mv2.jpg" alt="Soundy Saját profil képernyő" caption="Navigáljon a Lejátszási listákhoz" width="400" >}}

4. Koppintson a 'Továbbiak' gombra az exportálni kívánt lejátszási listán.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c02d2f9696954bb2b7aefacc28cce9fe~mv2.jpg" alt="Soundy Saját lejátszási listák képernyő" caption="Lejátszási lista beállítások elérése" width="400" >}}

5. Válassza az 'Export to M3U' lehetőséget a legördülő menüben.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_c03ed0eae8a743e3bfafaee84ed1b03d~mv2.jpg" alt="Soundy beállítások képernyő" caption="Lejátszási lista exportálása M3U formátumba" width="400" >}}

6. Várjon néhány másodpercet, amíg a tevékenységjelző fut.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_f35904d8256f429ab763cb66215485fc~mv2.jpg" alt="Soundy betöltési képernyő" caption="Exportálás folyamatban..." width="400" >}}

7. Az exportálás befejezése után megjelenik a szabványos 'Open In' párbeszédablak, ahol kiválaszthatja a létrehozott M3U lejátszási lista célhelyét.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1807ed3f45d34edd96a5ff1e5b8b309c~mv2.jpg" alt="Soundy exportálási képernyő" caption="Mentse el vagy ossza meg M3U lejátszási listáját" width="400" >}}

8. A lejátszási listát AirDrop segítségével átviheti Mac gépére.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_6e0406b61b7548a788e58b2582680a0f~mv2.jpg" alt="Soundy AirDrop képernyő" caption="M3U lejátszási lista átvitele AirDrop segítségével" width="400" >}}

9. Nyissa meg a lejátszási listát bármely M3U lejátszási listákat támogató programban (pl. VOX, WinAmp, Windows Media Player, iTunes).

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_74cf569b58c047b7916564c775f199fa~mv2.jpg" alt="VOX lejátszó képernyő" caption="Használja lejátszási listáját a VOX vagy bármely kompatibilis alkalmazásban" width="400" >}}

## Hogyan exportálhatók a SoundCloud kedvencek M3U lejátszási listába

Nem csak lejátszási listákat exportálhat — a Soundy támogatja kedvenceinek, feltöltött dalainak és újramegosztásainak exportálását is saját vagy bármely nyilvános SoundCloud profilról.

Kedvencek exportálásához:

1. Nyissa meg a **Kedvencek** lapot, és koppintson a jobb felső sarokban lévő **Továbbiak** gombra.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_1425541c40c9443a84feacabaadfba12~mv2.jpg" alt="Soundy kedvencek képernyő" caption="Beállítások elérése a Kedvencek lapon" width="400" >}}

2. Válassza az **Export to M3U** műveletet az exportálási folyamat elindításához.

   {{< figure src="/docs/howto/how-to-export-soundcloud-playlists-favorites-reposts-to-m3u/21260c_869482bcb58d49cf90972ee68ce08254~mv2.jpg" alt="Soundy Exportálás M3U-ba képernyő" caption="SoundCloud kedvencek exportálása M3U formátumba" width="400" >}}

Ennyi volt ehhez az útmutatóhoz. A Soundy alkalmazásról további információkat [itt](/products/soundy) találhat.

## Gyakran ismételt kérdések

{{% details title="Mi az M3U fájl?" closed="true" %}}
Az M3U egy szabványos lejátszási lista formátum, amelyet a legtöbb zenelejátszó támogat. Fájl útvonalak vagy URL-ek listáját tartalmazza, amelyek audió számokra mutatnak. Amikor egy SoundCloud lejátszási listát M3U formátumba exportál, a fájl streaming URL-eket tartalmaz minden egyes számhoz.
{{% /details %}}

{{% details title="Exportálhatok lejátszási listákat más SoundCloud felhasználóktól?" closed="true" %}}
Igen. A Soundy bármely nyilvános SoundCloud profilról képes exportálni lejátszási listákat, kedvenceket és újramegosztásokat, nem csak a saját fiókjából.
{{% /details %}}

{{% details title="Mely zenelejátszók támogatják az M3U lejátszási listákat?" closed="true" %}}
A legtöbb zenelejátszó támogatja az M3U formátumot, beleértve a VLC, WinAmp, Windows Media Player, iTunes, foobar2000, Evermusic és Flacbox alkalmazásokat.
{{% /details %}}

{{% details title="Exportálhatom az újramegosztásokat is M3U formátumba?" closed="true" %}}
Igen. A Soundy támogatja lejátszási listák, kedvencek, feltöltött dalok és újramegosztások exportálását. A folyamat ugyanaz: koppintson a Továbbiak gombra, és válassza az Export to M3U lehetőséget.
{{% /details %}}

{{% details title="Van korlátozás arra, hogy hány számot exportálhatok?" closed="true" %}}
A Soundy a kiválasztott lejátszási lista vagy gyűjtemény összes számát exportálja. Az alkalmazás nem szab mesterséges korlátot.
{{% /details %}}

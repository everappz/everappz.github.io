**摘要：** 將任何 Internet Archive URL 貼到 [archivetom3u.com](https://archivetom3u.com)，選擇音訊格式（MP3、FLAC、OGG），然後下載即可播放的 M3U 播放清單 -- 無需帳戶。然後將其匯入到 iPhone 或 Mac 上的 [Evermusic](https://apps.apple.com/app/apple-store/id885367198?pt=95781850&ct=everappzcom&mt=8) 中即可立即播放。

在這份逐步指南中，您將學習如何使用簡單的瀏覽器工具，從 [Internet Archive](https://archive.org) 的項目中產生和下載 M3U 播放清單 — 包括 **Live Music Archive**。

## 什麼是 Internet Archive？

[Internet Archive](https://archive.org) 是一個非營利數位圖書館，提供數百萬本免費書籍、電影、軟體、音樂等。其音訊收藏中包括 **Live Music Archive**，擁有數千個高品質的演唱會錄音，提供 **FLAC**、**24-bit FLAC**、**MP3** 和 **OGG** 等格式 — 全部完全免費存取和下載。

## 開啟 M3U 產生器工具

首先造訪 M3U 播放清單產生器網站：

👉 [https://archivetom3u.com](https://archivetom3u.com)

這個免費工具讓您貼上任何 Internet Archive 項目 URL 並立即產生可播放的 `.m3u` 檔案 — 無需註冊或安裝軟體。

## 工具如何運作

該工具使用 Internet Archive 官方中繼資料 API 來擷取曲目清單並產生播放清單檔案。網站不託管或鏡像任何媒體。您也可以使用內建播放器直接在瀏覽器中預覽曲目。

## 支援的音訊格式

您可以從以下格式中選擇：
- **VBR MP3**
- **FLAC**
- **24-BIT FLAC**
- **OGG VORBIS**

> ⚠️ 只有所選格式中可用的曲目才會包含在產生的播放清單中。

本網站不託管或重新分發任何內容。請確保您遵守與每個項目相關的著作權、授權和使用政策。使用風險自負。

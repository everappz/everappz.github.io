**תקציר:** השתמשו ב-Evermusic באייפון או אייפד שלכם כדי לגשת לקבצים המאוחסנים ב-Mac או PC Windows דרך הרשת המקומית שלכם באמצעות SMB. ללא כבלים, ללא iTunes, ללא צורך בהעלאה לענן. הפעילו שיתוף קבצים במחשב, התחברו באפליקציה, וגלשו או נגנו את הקבצים שלכם באופן אלחוטי.

האם יש לכם אוסף נרחב של קבצים גדולים ב-MAC או PC ואתם רוצים לגשת אליהם בקלות מהאייפון או האייפד? האפליקציות שלנו מספקות פתרון פשוט.

עקבו אחר הצעדים הבאים כדי לאפשר גישה חלקה בין המחשב שלכם למכשיר iOS באמצעות פרוטוקול SMB:

## שלב 1: הפעלת פרוטוקול SMB במחשב שלכם

**עבור MAC:**

1. פתחו את "העדפות מערכת" ב-MAC שלכם.
2. לחצו על "שיתוף".
3. הפעילו את שירות "שיתוף קבצים".
4. הוסיפו את תיקיית המוזיקה שלכם לסעיף "תיקיות משותפות". הוסיפו משתמש ובחרו את רמת ההרשאה (קריאה וכתיבה או קריאה בלבד). תוכלו לבחור "כולם: קריאה בלבד" עבור תיקיית המוזיקה שנוספה.

   ![מסך הגדרות Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_4c8f67e6cad0498080909244213f84af~mv2.png)

5. זכרו את כתובת ה-URL של המחשב (smb://192.168.xx.xx), שכן תשתמשו בה בשלבים הבאים.
6. לחצו על "אפשרויות" והפעילו "שיתוף קבצים ותיקיות באמצעות SMB".

   ![מסך שיתוף קבצים Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_32c05fd0930a4ec28256afe40c0ba8a5~mv2.png)

7. הפעילו "שיתוף קבצי Windows" עבור חשבונות זמינים.

   ![מסך שיתוף SMB ב-Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_26acaa067aae40788465c1698b0458dc~mv2.png)

**עבור Windows PC:**

1. לחצו לחיצה ימנית על תיקיית המוזיקה שלכם.
2. בחרו "מאפיינים".
3. נווטו ללשונית "שיתוף".
4. לחצו על "שתף..."
5. בחרו את האנשים שאיתם תרצו לשתף את התיקייה וציינו את רמת ההרשאה. תוכלו לבחור "כולם: קריאה" עבור תיקיית המוזיקה שנבחרה.

   ![מסך שיתוף SMB ב-Windows](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_c503c5d0d1f44daeb14d5a4cadfe9ac2~mv2.png)

6. לחצו "בוצע".
7. לחצו "בוצע" בחלון "שיתוף קבצים", וזכרו את נתיב התיקייה.

   ![תיקייה משותפת SMB ב-Windows](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_350e2dca694b41bcade8f455acc4e481~mv2.png)

1. פתחו את האפליקציה באייפון או אייפד.
2. עברו ללשונית "חיבורים".

*אם המחשב שלכם מופיע בסעיף "מכשירים זמינים":*

אם המחשב שלכם גלוי בסעיף "מכשירים זמינים" ובחרתם "כולם: קריאה בלבד" בשלב הקודם, פשוט הקישו על המחשב שלכם והוא יתחבר אוטומטית.

*אם המחשב שלכם לא מופיע אוטומטית:*

1. הקישו על "התחברות לשירות ענן".
2. בחרו "SMB" במסך "התחברות לשירות ענן".

3. במסך "חיבור SMB", הזינו את כתובת ה-URL של השרת עם נתיב התיקייה המשותפת. תוכלו להשתמש בשם השרת או בכתובת IP של השרת:

   ```
   smb://ameleshko.local/Music/ 
   smb://192.168.0.102/Music/ 
   smb://192.168.0.102/
   ```

4. הזינו את שם המשתמש והסיסמה שלכם או השאירו שדות אלה ריקים אם בחרתם "כולם: קריאה בלבד" בשלב הקודם.
5. שדה "WORKGROUP" הוא אופציונלי ויש להשתמש בו אם יש לכם Active Directory Domain.

7. פתחו את השירות המחובר ונווטו לתיקייה הרצויה.

## שלב 3: פתרון לתיקיות SMB2 עם תווים מיוחדים

לעיתים אתם עלולים להיתקל בבעיות עם תיקיות המכילות תווים מיוחדים בעת שימוש בפרוטוקול SMB2. הנה כמה צעדים לפתרון בעיה זו:

1. **הפעלת SMB 1:**  
   • כפתרון זמני, נסו להפעיל SMB 1 בשרת שלכם ובהגדרות האפליקציה. זה יכול לעזור לעקוף בעיות הקשורות לתווים מיוחדים בשמות תיקיות.

2. **שימוש בתפריט פתיחת קבצי מערכת:**  
   • נווטו ל"קבצים מקומיים".  
   • גללו מטה לסעיף "קבצים במכשיר זה".  
   • הקישו על "פתח קבצים..." או "פתח תיקיות...".  
   • אתרו את השרת שלכם ובחרו את הקבצים או התיקיות שאתם צריכים.  
   • הקישו "פתח" כדי לאשר את הבחירה שלכם.

3. **פרוטוקולים חלופיים:**  
   • אם הבעיה נמשכת, שקלו להתחבר ל-NAS שלכם באמצעות פרוטוקולי WebDAV או DLNA אם ה-NAS שלכם תומך באפשרויות אלה. פרוטוקולים אלה עשויים לטפל בתווים מיוחדים בצורה טובה יותר.

על ידי ביצוע צעדים אלה, תוכלו להקל על הבעיות עם תווים מיוחדים בשמות תיקיות בעת שימוש בפרוטוקול SMB2.

## סיכום

עם צעדים אלה, תוכלו לגשת בקלות לאוסף הקבצים הנרחב שלכם מה-MAC או PC באייפון או אייפד באמצעות האפליקציות שלנו.

## שאלות נפוצות

{{% details title="האם אני יכול לגשת לקבצים במחשב מהאייפון ללא iTunes?" closed="true" %}}
כן. Evermusic מתחבר למחשב שלכם דרך SMB ברשת ה-Wi-Fi המקומית. אין צורך בסנכרון iTunes או Finder. הפעילו שיתוף קבצים במחשב והתחברו ישירות מהאפליקציה.
{{% /details %}}

{{% details title="האם גישת קבצי SMB עובדת דרך האינטרנט?" closed="true" %}}
לא. SMB הוא פרוטוקול רשת מקומית. האייפון והמחשב שלכם חייבים להיות באותה רשת Wi-Fi. לגישה מרחוק, העלו קבצים לשירות ענן כמו Google Drive או Dropbox והתחברו אליו ב-Evermusic.
{{% /details %}}

{{% details title="לאילו סוגי קבצים אני יכול לגשת דרך SMB?" closed="true" %}}
Evermusic תומך ב-MP3, FLAC, AAC, WAV, AIFF, OGG, WMA, ALAC ופורמטי אודיו נוספים. תוכלו גם לגלוש ולנהל קבצים שאינם אודיו באמצעות מנהל הקבצים המובנה.
{{% /details %}}

{{% details title="האם אני יכול להעביר קבצים מ-NAS לאייפון באמצעות SMB?" closed="true" %}}
כן. רוב מכשירי NAS (Synology, QNAP, WD My Cloud ואחרים) תומכים ב-SMB. התחברו ל-NAS שלכם באמצעות אותם צעדים במדריך זה.
{{% /details %}}

{{% details title="האם אני צריך להעתיק קבצים לאייפון כדי לנגן אותם?" closed="true" %}}
לא. Evermusic מזרים קבצים ישירות מהמחשב או ה-NAS שלכם דרך הרשת. קבצים אינם מועתקים לאייפון אלא אם תבחרו להוריד אותם לנגינה במצב לא מקוון.
{{% /details %}}

{{% details title="האם שיתוף קבצי SMB בטוח?" closed="true" %}}
שיתוף קבצי SMB פועל רק ברשת המקומית שלכם. מכשירים אחרים ברשתות שונות לא יכולים לגשת לתיקיות המשותפות שלכם. לאבטחה נוספת, השתמשו בשם משתמש וסיסמה במקום גישה אנונימית (כולם).
{{% /details %}}

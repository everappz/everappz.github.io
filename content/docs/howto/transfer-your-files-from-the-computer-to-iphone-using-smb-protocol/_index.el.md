**Περίληψη:** Χρησιμοποιήστε το Evermusic στο iPhone ή iPad σας για πρόσβαση σε αρχεία αποθηκευμένα στον Mac ή Windows PC σας μέσω του τοπικού σας δικτύου με SMB. Χωρίς καλώδια, χωρίς iTunes, χωρίς ανέβασμα στο cloud. Ενεργοποιήστε την κοινή χρήση αρχείων στον υπολογιστή σας, συνδεθείτε στην εφαρμογή και περιηγηθείτε ή αναπαράγετε τα αρχεία σας ασύρματα.

Έχετε μια εκτεταμένη συλλογή μεγάλων αρχείων στον MAC ή PC σας και θέλετε να αποκτήσετε πρόσβαση σε αυτά εύκολα από το iPhone ή iPad σας; Οι εφαρμογές μας παρέχουν μια απλή λύση.

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε την απρόσκοπτη πρόσβαση μεταξύ του υπολογιστή σας και της συσκευής iOS χρησιμοποιώντας το πρωτόκολλο SMB:

## Βήμα 1: Ενεργοποίηση του πρωτοκόλλου SMB στον υπολογιστή σας

**Για MAC:**

1. Ανοίξτε τις "Προτιμήσεις Συστήματος" στον MAC σας.
2. Κάντε κλικ στο "Κοινή χρήση".
3. Ενεργοποιήστε την υπηρεσία "Κοινή χρήση αρχείων".
4. Προσθέστε τον φάκελο μουσικής σας στην ενότητα "Κοινόχρηστοι φάκελοι". Προσθέστε έναν χρήστη και επιλέξτε το επίπεδο δικαιωμάτων (Ανάγνωση & Εγγραφή ή Μόνο ανάγνωση). Μπορείτε να επιλέξετε "Όλοι: Μόνο ανάγνωση" για τον προστιθέμενο φάκελο μουσικής.

   ![Οθόνη ρυθμίσεων Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_4c8f67e6cad0498080909244213f84af~mv2.png)

5. Θυμηθείτε τη διεύθυνση URL του υπολογιστή (smb://192.168.xx.xx), καθώς θα τη χρησιμοποιήσετε στα επόμενα βήματα.
6. Κάντε κλικ στις "Επιλογές" και ενεργοποιήστε "Κοινή χρήση αρχείων και φακέλων μέσω SMB".

   ![Οθόνη κοινής χρήσης αρχείων Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_32c05fd0930a4ec28256afe40c0ba8a5~mv2.png)

7. Ενεργοποιήστε την "Κοινή χρήση αρχείων Windows" για τους διαθέσιμους λογαριασμούς.

   ![Οθόνη κοινής χρήσης SMB Mac](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_26acaa067aae40788465c1698b0458dc~mv2.png)

**Για Windows PC:**

1. Κάντε δεξί κλικ στον φάκελο μουσικής σας.
2. Επιλέξτε "Ιδιότητες".
3. Μεταβείτε στην καρτέλα "Κοινή χρήση".
4. Κάντε κλικ στο "Κοινή χρήση..."
5. Επιλέξτε τα άτομα με τα οποία θέλετε να μοιραστείτε τον φάκελο και καθορίστε το επίπεδο δικαιωμάτων. Μπορείτε να επιλέξετε "Όλοι: Ανάγνωση" για τον επιλεγμένο φάκελο μουσικής.

   ![Οθόνη κοινής χρήσης SMB Windows](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_c503c5d0d1f44daeb14d5a4cadfe9ac2~mv2.png)

6. Κάντε κλικ στο "Ολοκλήρωση".
7. Κάντε κλικ στο "Ολοκλήρωση" στο παράθυρο "Κοινή χρήση αρχείων" και θυμηθείτε τη διαδρομή του φακέλου.

   ![Κοινόχρηστος φάκελος SMB Windows](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_350e2dca694b41bcade8f455acc4e481~mv2.png)

## Βήμα 3: Αντιμετώπιση φακέλων SMB2 με ειδικούς χαρακτήρες

Μερικές φορές μπορεί να αντιμετωπίσετε προβλήματα με φακέλους που περιέχουν ειδικούς χαρακτήρες κατά τη χρήση του πρωτοκόλλου SMB2. Ακολουθούν ορισμένα βήματα για την επίλυση αυτού του προβλήματος:

1. **Ενεργοποίηση SMB 1:**  
   • Ως προσωρινή λύση, δοκιμάστε να ενεργοποιήσετε το SMB 1 στον διακομιστή σας και στις ρυθμίσεις της εφαρμογής. Αυτό μπορεί να βοηθήσει στην παράκαμψη των προβλημάτων που σχετίζονται με ειδικούς χαρακτήρες στα ονόματα φακέλων.

2. **Χρήση μενού ανοίγματος αρχείων συστήματος:**  
   • Μεταβείτε στα "Τοπικά αρχεία".  
   • Κάντε κύλιση προς τα κάτω στην ενότητα "Αρχεία σε αυτή τη συσκευή".  
   • Πατήστε "Άνοιγμα αρχείων..." ή "Άνοιγμα φακέλων...".  
   • Εντοπίστε τον διακομιστή σας και επιλέξτε τα αρχεία ή τους φακέλους που χρειάζεστε.  
   • Πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή σας.

3. **Εναλλακτικά πρωτόκολλα:**  
   • Εάν το πρόβλημα παραμένει, σκεφτείτε να συνδεθείτε στο NAS σας χρησιμοποιώντας πρωτόκολλα WebDAV ή DLNA εάν το NAS σας υποστηρίζει αυτές τις επιλογές. Αυτά τα πρωτόκολλα μπορεί να χειρίζονται τους ειδικούς χαρακτήρες πιο αποτελεσματικά.

Ακολουθώντας αυτά τα βήματα, μπορείτε να μετριάσετε τα προβλήματα με ειδικούς χαρακτήρες στα ονόματα φακέλων κατά τη χρήση του πρωτοκόλλου SMB2.

## Συμπέρασμα

Με αυτά τα βήματα, μπορείτε να αποκτήσετε εύκολα πρόσβαση στην τεράστια συλλογή αρχείων σας από τον MAC ή PC σας στο iPhone ή iPad σας χρησιμοποιώντας τις εφαρμογές μας.

## Συχνές ερωτήσεις

{{% details title="Μπορώ να αποκτήσω πρόσβαση σε αρχεία στον PC μου από το iPhone μου χωρίς iTunes;" closed="true" %}}
Ναι. Το Evermusic συνδέεται στον υπολογιστή σας μέσω SMB στο τοπικό σας δίκτυο Wi-Fi. Δεν απαιτείται συγχρονισμός iTunes ή Finder. Ενεργοποιήστε την κοινή χρήση αρχείων στον PC σας και συνδεθείτε απευθείας από την εφαρμογή.
{{% /details %}}

{{% details title="Λειτουργεί η πρόσβαση αρχείων SMB μέσω internet;" closed="true" %}}
Όχι. Το SMB είναι πρωτόκολλο τοπικού δικτύου. Το iPhone και ο υπολογιστής σας πρέπει να βρίσκονται στο ίδιο δίκτυο Wi-Fi. Για απομακρυσμένη πρόσβαση, ανεβάστε αρχεία σε υπηρεσία cloud όπως Google Drive ή Dropbox και συνδεθείτε σε αυτήν στο Evermusic.
{{% /details %}}

{{% details title="Σε ποιους τύπους αρχείων μπορώ να αποκτήσω πρόσβαση μέσω SMB;" closed="true" %}}
Το Evermusic υποστηρίζει MP3, FLAC, AAC, WAV, AIFF, OGG, WMA, ALAC και άλλες μορφές ήχου. Μπορείτε επίσης να περιηγηθείτε και να διαχειριστείτε αρχεία μη ήχου χρησιμοποιώντας τον ενσωματωμένο διαχειριστή αρχείων.
{{% /details %}}

{{% details title="Μπορώ να μεταφέρω αρχεία από NAS στο iPhone μου χρησιμοποιώντας SMB;" closed="true" %}}
Ναι. Οι περισσότερες συσκευές NAS (Synology, QNAP, WD My Cloud και άλλες) υποστηρίζουν SMB. Συνδεθείτε στο NAS σας χρησιμοποιώντας τα ίδια βήματα σε αυτόν τον οδηγό.
{{% /details %}}

{{% details title="Χρειάζεται να αντιγράψω αρχεία στο iPhone μου για να τα αναπαράγω;" closed="true" %}}
Όχι. Το Evermusic κάνει streaming αρχείων απευθείας από τον υπολογιστή ή το NAS σας μέσω του δικτύου. Τα αρχεία δεν αντιγράφονται στο iPhone σας εκτός αν επιλέξετε να τα κατεβάσετε για αναπαραγωγή εκτός σύνδεσης.
{{% /details %}}

{{% details title="Είναι ασφαλής η κοινή χρήση αρχείων SMB;" closed="true" %}}
Η κοινή χρήση αρχείων SMB λειτουργεί μόνο στο τοπικό σας δίκτυο. Άλλες συσκευές σε διαφορετικά δίκτυα δεν μπορούν να αποκτήσουν πρόσβαση στους κοινόχρηστους φακέλους σας. Για πρόσθετη ασφάλεια, χρησιμοποιήστε όνομα χρήστη και κωδικό πρόσβασης αντί για ανώνυμη (Όλοι) πρόσβαση.
{{% /details %}}

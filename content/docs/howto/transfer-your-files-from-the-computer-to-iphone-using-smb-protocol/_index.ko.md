**요약:** iPhone 또는 iPad에서 Evermusic을 사용하여 SMB를 통해 로컬 네트워크에서 Mac 또는 Windows PC에 저장된 파일에 액세스하세요. 케이블 없이, iTunes 없이, 클라우드 업로드 없이 가능합니다. 컴퓨터에서 파일 공유를 활성화하고, 앱에서 연결하고, 무선으로 파일을 탐색하거나 재생하세요.

MAC 또는 PC에 대용량 파일의 방대한 컬렉션이 있고 iPhone 또는 iPad에서 쉽게 액세스하고 싶으신가요? 저희 앱이 간단한 솔루션을 제공합니다.

SMB 프로토콜을 사용하여 컴퓨터와 iOS 장치 간의 원활한 액세스를 활성화하려면 다음 단계를 따르세요:

## 1단계: 컴퓨터에서 SMB 프로토콜 활성화

**MAC의 경우:**

1. MAC에서 "시스템 환경설정"을 엽니다.
2. "공유"를 클릭합니다.
3. "파일 공유" 서비스를 활성화합니다.
4. "공유 폴더" 섹션에 음악 폴더를 추가합니다. 사용자를 추가하고 권한 수준을 선택합니다(읽기 및 쓰기 또는 읽기 전용). 추가된 음악 폴더에 "모든 사람: 읽기 전용"을 선택할 수 있습니다.

   ![Mac 설정 화면](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_4c8f67e6cad0498080909244213f84af~mv2.png)

5. 컴퓨터 URL(smb://192.168.xx.xx)을 기억해 두세요. 다음 단계에서 사용합니다.
6. "옵션"을 클릭하고 "SMB를 사용하여 파일 및 폴더 공유"를 활성화합니다.

   ![Mac 파일 공유 화면](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_32c05fd0930a4ec28256afe40c0ba8a5~mv2.png)

7. 사용 가능한 계정에 대해 "Windows 파일 공유"를 활성화합니다.

   ![Mac SMB 공유 화면](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_26acaa067aae40788465c1698b0458dc~mv2.png)

**Windows PC의 경우:**

1. 음악 폴더를 마우스 오른쪽 버튼으로 클릭합니다.
2. "속성"을 선택합니다.
3. "공유" 탭으로 이동합니다.
4. "공유..."를 클릭합니다.
5. 폴더를 공유할 사람을 선택하고 권한 수준을 지정합니다. 선택한 음악 폴더에 "모든 사람: 읽기"를 선택할 수 있습니다.

   ![Windows SMB 공유 화면](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_c503c5d0d1f44daeb14d5a4cadfe9ac2~mv2.png)

6. "완료됨"을 클릭합니다.
7. "파일 공유" 창에서 "완료됨"을 클릭하고 폴더 경로를 기억해 둡니다.

   ![Windows SMB 공유 폴더](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_350e2dca694b41bcade8f455acc4e481~mv2.png)

## 2단계: iOS 장치 연결

1. iPhone 또는 iPad에서 앱을 엽니다.
2. "연결하기" 탭으로 이동합니다.

   {{< figure
  src="/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_1b1864a302194414a6ec4dc14f95bfaf~mv2.png"

>}}

*"사용 가능한 장치" 섹션에 컴퓨터가 표시되는 경우:*

컴퓨터가 "사용 가능한 장치" 섹션에 보이고 이전 단계에서 "모든 사람: 읽기 전용"을 선택한 경우, 컴퓨터를 탭하면 자동으로 연결됩니다.

*컴퓨터가 자동으로 표시되지 않는 경우:*

1. "클라우드 서비스 연결"을 탭합니다.
2. "클라우드 서비스 연결" 화면에서 "SMB"를 선택합니다.

   
{{< figure
  src="/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_fd5a7b81f9cf427e99ccb0024c0a686c~mv2.jpeg"

  caption="Evermusic 클라우드 서비스 연결 화면"
  width="400"
>}}

3. "SMB 연결" 화면에서 공유 폴더 경로와 함께 서버 URL을 입력합니다. 서버 이름 또는 서버 IP를 사용할 수 있습니다:

   ```
   smb://ameleshko.local/Music/ 
   smb://192.168.0.102/Music/ 
   smb://192.168.0.102/
   ```

4. 로그인과 비밀번호를 입력하거나 이전 단계에서 "모든 사람: 읽기 전용"을 선택한 경우 이 필드를 비워 둡니다.
5. "WORKGROUP" 필드는 선택 사항이며 Active Directory Domain이 있는 경우 사용해야 합니다.

  {{< figure
  src="/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_b6a9a4ad317447768f7f38b41bc07aca~mv2.png"

6. SMB 프로토콜을 사용하여 컴퓨터를 연결하면 "연결하기" 화면의 "클라우드 서비스" 섹션에 표시됩니다.
7. 연결된 서비스를 열고 원하는 폴더로 이동합니다.

  {{< figure
  src="/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/21260c_ed605ed873184662bbc26e75651cc8d8~mv2.png"

8. 내장 파일 관리자를 사용하여 필요에 따라 파일을 편집할 수 있습니다.

## 3단계: 특수 문자가 포함된 SMB2 폴더 해결 방법

SMB2 프로토콜을 사용할 때 특수 문자가 포함된 폴더에서 문제가 발생할 수 있습니다. 이 문제를 해결하기 위한 몇 가지 단계는 다음과 같습니다:

1. **SMB 1 활성화:**  
   • 임시 해결책으로 서버와 앱 설정에서 SMB 1을 활성화해 보세요. 이렇게 하면 폴더 이름의 특수 문자와 관련된 문제를 우회할 수 있습니다.

2. **시스템 파일 열기 메뉴 사용:**  
   • "로컬 파일"로 이동합니다.  
   • "이 장치의 파일" 섹션까지 아래로 스크롤합니다.  
   • "파일 열기..." 또는 "폴더 열기..."를 탭합니다.  
   • 서버를 찾아 필요한 파일 또는 폴더를 선택합니다.  
   • "열기"를 탭하여 선택을 확인합니다.

3. **대체 프로토콜:**  
   • 문제가 지속되면 NAS가 이러한 옵션을 지원하는 경우 WebDAV 또는 DLNA 프로토콜을 사용하여 NAS에 연결하는 것을 고려하세요. 이러한 프로토콜은 특수 문자를 더 잘 처리할 수 있습니다.

이 단계를 따르면 SMB2 프로토콜을 사용할 때 폴더 이름의 특수 문자 문제를 완화할 수 있습니다.

## 결론

이 단계를 통해 저희 앱을 사용하여 MAC 또는 PC의 방대한 파일 컬렉션을 iPhone 또는 iPad에서 쉽게 액세스할 수 있습니다.

## 자주 묻는 질문

{{% details title="iTunes 없이 iPhone에서 PC의 파일에 접근할 수 있나요?" closed="true" %}}
네. Evermusic은 로컬 Wi-Fi 네트워크에서 SMB를 통해 컴퓨터에 연결합니다. iTunes 또는 Finder 동기화가 필요하지 않습니다. PC에서 파일 공유를 활성화하고 앱에서 직접 연결하세요.
{{% /details %}}

{{% details title="SMB 파일 접근이 인터넷을 통해 작동하나요?" closed="true" %}}
아니요. SMB는 로컬 네트워크 프로토콜입니다. iPhone과 컴퓨터가 동일한 Wi-Fi 네트워크에 있어야 합니다. 원격 접근을 위해서는 Google Drive 또는 Dropbox와 같은 클라우드 서비스에 파일을 업로드하고 Evermusic에서 연결하세요.
{{% /details %}}

{{% details title="SMB를 통해 어떤 파일 유형에 접근할 수 있나요?" closed="true" %}}
Evermusic은 MP3, FLAC, AAC, WAV, AIFF, OGG, WMA, ALAC 및 기타 오디오 형식을 지원합니다. 내장 파일 관리자를 사용하여 비오디오 파일도 탐색하고 관리할 수 있습니다.
{{% /details %}}

{{% details title="SMB를 사용하여 NAS에서 iPhone으로 파일을 전송할 수 있나요?" closed="true" %}}
네. 대부분의 NAS 장치(Synology, QNAP, WD My Cloud 등)는 SMB를 지원합니다. 이 가이드의 동일한 단계를 사용하여 NAS에 연결하세요.
{{% /details %}}

{{% details title="파일을 재생하려면 iPhone에 복사해야 하나요?" closed="true" %}}
아니요. Evermusic은 네트워크를 통해 컴퓨터 또는 NAS에서 직접 파일을 스트리밍합니다. 오프라인 재생을 위해 다운로드하지 않는 한 파일은 iPhone에 복사되지 않습니다.
{{% /details %}}

{{% details title="SMB 파일 공유는 안전한가요?" closed="true" %}}
SMB 파일 공유는 로컬 네트워크에서만 작동합니다. 다른 네트워크의 다른 장치는 공유 폴더에 접근할 수 없습니다. 추가 보안을 위해 익명(모든 사람) 접근 대신 로그인과 비밀번호를 사용하세요.
{{% /details %}}

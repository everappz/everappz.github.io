**요약:** USB 케이블로 iPhone 또는 iPad를 Mac(또는 PC)에 연결하세요. macOS Catalina 이상에서는 Finder를 사용하세요. 이전 macOS 또는 Windows에서는 iTunes를 사용하세요. Evermusic, Flacbox 또는 Evertag과 같은 앱으로 파일을 드래그하면 즉시 전송됩니다.

iTunes 파일 공유는 컴퓨터와 iPhone, iPad 또는 iPod touch의 iOS 애플리케이션 간에 파일을 전송하는 편리한 방법을 제공합니다. Mac이나 PC에 좋아하는 앨범, 아티스트 또는 트랙 컬렉션이 있다면 Evermusic, Flacbox 또는 Evertag과 같은 앱으로 이 콘텐츠를 쉽게 복사할 수 있습니다.

다음은 iTunes 파일 공유를 사용하는 단계별 가이드입니다.

## macOS Catalina 이상의 경우

**사전 요구 사항**  
- macOS Catalina 또는 이후 버전
- 최신 iOS 또는 iPadOS 버전이 설치된 iOS 기기(iPhone, iPad 또는 iPod touch)
- 파일 공유와 호환되는 iOS 앱

### 단계  
1. Mac에서 Finder 창을 여세요.
2. USB 케이블을 사용하여 iOS 기기를 컴퓨터에 연결하세요.
3. Finder에서 기기를 선택하세요.
4. "파일" 탭을 클릭하여 파일 공유를 지원하는 앱 목록을 확인하세요. "파일" 섹션이 보이지 않으면 기기에 파일 공유를 지원하는 앱이 없다는 뜻입니다.
5. 앱 옆의 삼각형을 클릭하여 공유할 수 있는 파일을 확인하세요.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Mac에서 iOS 기기로 파일 복사  
1. 복사할 파일을 선택하고, 앱과 호환되는지 확인하세요(지원되는 형식은 앱의 사용자 가이드를 참조하세요).
2. 선택한 파일을 Finder에서 iOS 기기의 앱 아이콘으로 드래그 앤 드롭하세요. Finder가 파일을 기기에 복사합니다.

### iOS 기기에서 Mac으로 파일 복사  
1. iOS 기기에서 복사할 파일을 선택하세요.
2. 선택한 파일을 컴퓨터의 폴더나 창으로 드래그 앤 드롭하세요. Finder가 파일을 컴퓨터에 복사합니다.

## Catalina 이전 macOS 버전의 경우

**사전 요구 사항**  
- 최신 버전의 iTunes
- 최신 iOS 버전이 설치된 iOS 기기
- Windows 7 이상의 Windows PC(Windows를 사용하는 경우)

### 단계
1. Mac 또는 PC에서 iTunes를 여세요.
2. USB Lightning 케이블을 사용하여 iPhone 또는 iPad를 컴퓨터에 연결하세요.
3. iTunes에서 iOS 기기를 선택하세요.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. 왼쪽 사이드바에서 앱 섹션을 클릭하세요.
5. 페이지 하단의 파일 공유 섹션으로 스크롤하세요.
6. 앱을 선택하여 기기에서 공유할 수 있는 파일을 확인하세요.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### 컴퓨터에서 앱으로 파일 복사  
1. iTunes 왼쪽 사이드바에서 "앱" 섹션을 클릭하세요.
2. 하단의 "파일 공유" 섹션으로 스크롤하세요.
3. 앱을 선택하여 기기에서 공유할 수 있는 파일을 확인하세요.
4. 기기에 파일을 복사하려면 폴더나 창에서 iTunes의 "문서" 목록으로 드래그 앤 드롭하세요.
5. 또는 iTunes의 "문서" 목록에서 "추가"를 클릭하고, 컴퓨터에서 오디오 파일을 찾은 다음 "추가"를 클릭하세요. iTunes가 이 음악 파일을 기기의 앱 "문서" 디렉토리에 복사합니다.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### iOS 앱에서 컴퓨터로 파일 복사  
1. iTunes에서 "파일 공유" 섹션의 목록에서 앱을 선택하세요.
2. 컴퓨터로 파일을 복사하려면 "문서" 목록에서 컴퓨터의 폴더나 창으로 드래그 앤 드롭하세요.
3. 또는 "문서" 목록에서 컴퓨터로 복사할 오디오 파일을 선택하고 "다음에 저장"을 클릭하세요. 파일을 복사할 컴퓨터의 폴더를 찾고 "다음에 저장"을 클릭하세요.

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 공유 파일 열기  
컴퓨터에서 iOS 기기로 복사한 파일을 재생하려면 파일 공유용으로 지정된 앱(예: Evermusic 또는 Flacbox)을 열고 "로컬 파일" 탭을 탭한 다음 "문서" 폴더를 선택하세요. 재생할 수 있는 파일 목록이 표시됩니다.

## iOS 기기에서 공유 파일 삭제  

### iTunes 사용  
1. Mac/PC에서 iTunes를 여세요.
2. USB를 통해 iOS 기기를 컴퓨터에 연결하세요.
3. iTunes에서 기기를 선택하세요.
4. 왼쪽 사이드바에서 "앱"을 클릭하고 "파일 공유" 섹션으로 스크롤하세요.
5. 삭제할 파일이 포함된 앱을 선택하세요.
6. "문서" 목록에서 파일을 선택하고 키보드의 Delete 키를 누르세요.
7. iTunes에서 삭제를 확인하세요.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### 내장 파일 관리자 사용  
1. "로컬 파일" 섹션을 여세요.
2. 오른쪽 상단의 "추가 작업" 버튼을 탭하고 메뉴에서 "선택하다"를 선택하세요.
3. 삭제할 파일을 선택하세요.
4. "삭제하기" 작업을 탭하세요(보이지 않으면 "추가 작업" 메뉴를 탭하세요).
5. 삭제를 확인하세요.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

iTunes 파일 공유를 사용하면 컴퓨터와 iOS 앱 간의 파일을 쉽게 관리하여 음악 및 파일 공유 경험을 향상시킬 수 있습니다.

## 자주 묻는 질문

{{% details title="USB를 통해 파일을 전송하려면 인터넷 연결이 필요한가요?" closed="true" %}}
아니요. 파일 공유는 컴퓨터와 iOS 기기 간의 USB 케이블 연결을 통해 완전히 작동합니다. 인터넷은 필요하지 않습니다.
{{% /details %}}

{{% details title="Evermusic 또는 Flacbox에 어떤 파일 형식을 전송할 수 있나요?" closed="true" %}}
두 앱 모두 MP3, FLAC, AAC, WAV, AIFF, OGG, WMA 등 다양한 오디오 형식을 지원합니다. 지원되는 형식의 전체 목록은 앱의 문서를 확인하세요.
{{% /details %}}

{{% details title="Finder에서 파일 탭이 보이지 않는 이유는 무엇인가요?" closed="true" %}}
파일 탭은 기기에 파일 공유를 지원하는 앱이 하나 이상 설치되어 있을 때만 나타납니다. Evermusic, Flacbox 또는 Evertag을 설치한 다음 기기를 다시 연결하세요.
{{% /details %}}

{{% details title="USB 케이블 대신 무선으로 파일을 전송할 수 있나요?" closed="true" %}}
네. Evermusic과 Flacbox는 클라우드 저장소 서비스와 Wi-Fi 전송도 지원합니다. 그러나 Finder 또는 iTunes를 통한 USB 파일 공유는 일반적으로 대용량 음악 라이브러리에 더 빠릅니다.
{{% /details %}}

{{% details title="Finder를 통해 파일을 전송하면 기기의 기존 파일이 덮어쓰기되나요?" closed="true" %}}
아니요. 새 파일은 기존 파일과 함께 추가됩니다. 같은 이름의 파일이 이미 있는 경우 macOS가 새 파일의 이름을 자동으로 변경할 수 있습니다.
{{% /details %}}

{{% details title="이 방법은 Windows PC에서도 작동하나요?" closed="true" %}}
네. Windows에서는 iTunes를 사용하여 파일을 전송하세요. 위의 iTunes 섹션에서 설명한 것과 동일한 과정입니다. Microsoft Store 또는 Apple 웹사이트에서 iTunes를 설치하세요.
{{% /details %}}

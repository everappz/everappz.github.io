**Sammanfattning:** Anslut din iPhone eller iPad till din Mac (eller PC) med en USB-kabel. På macOS Catalina och senare använder du Finder. På äldre macOS eller Windows använder du iTunes. Dra filer till en app som Evermusic, Flacbox eller Evertag för att överföra dem direkt.

iTunes Fildelning erbjuder ett bekvämt sätt att överföra filer mellan din dator och iOS-applikationer på din iPhone, iPad eller iPod touch. Oavsett om du har en samling favoritalbum, artister eller spår på din Mac eller PC kan du enkelt kopiera detta innehåll till appar som Evermusic, Flacbox eller Evertag.

Här är en steg-för-steg-guide för att använda iTunes Fildelning.

## För macOS Catalina eller senare

**Förutsättningar**  
- macOS Catalina eller en senare version
- En iOS-enhet (iPhone, iPad eller iPod touch) med den senaste iOS- eller iPadOS-versionen
- En iOS-app som är kompatibel med Fildelning

### Steg  
1. Öppna ett Finder-fönster på din Mac.
2. Anslut din iOS-enhet till datorn med en USB-kabel.
3. Välj din enhet i Finder.
4. Klicka på fliken "Filer" för att visa en lista över appar som stöder fildelning. Om du inte ser avsnittet "Filer" betyder det att din enhet inte har några appar som stöder fildelning.
5. Klicka på triangeln bredvid en app för att visa filerna du kan dela.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Kopiera filer från din Mac till din iOS-enhet  
1. Välj filerna du vill kopiera och se till att de är kompatibla med appen (se appens användarhandbok för format som stöds).
2. Dra och släpp de valda filerna på appikonen på din iOS-enhet i Finder. Finder kopierar filerna till din enhet.

### Kopiera filer från din iOS-enhet till din Mac  
1. Välj filerna du vill kopiera från din iOS-enhet.
2. Dra och släpp de valda filerna till en mapp eller ett fönster på din dator. Finder kopierar filerna till din dator.

## För macOS-versioner före Catalina

**Förutsättningar**  
- Den senaste versionen av iTunes
- En iOS-enhet med den senaste iOS-versionen
- En Windows-dator med Windows 7 eller senare (om du använder Windows)

### Steg
1. Öppna iTunes på din Mac eller PC.
2. Anslut din iPhone eller iPad till datorn med en USB Lightning-kabel.
3. Välj din iOS-enhet i iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Klicka på avsnittet Appar i det vänstra sidofältet.
5. Bläddra till avsnittet Fildelning längst ner på sidan.
6. Välj appen för att se vilka filer som är tillgängliga för delning på din enhet.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Kopiera filer från din dator till appen  
1. I det vänstra sidofältet i iTunes klickar du på avsnittet "Appar".
2. Bläddra ner till avsnittet "Fildelning" längst ner.
3. Välj appen för att se tillgängliga filer för delning på din enhet.
4. För att kopiera filer till din enhet, dra och släpp dem från en mapp eller ett fönster till listan "Dokument" i iTunes.
5. Alternativt kan du klicka på "Lägg till" i listan "Dokument" i iTunes, leta reda på ljudfilerna på din dator och klicka på "Lägg till." iTunes kopierar dessa musikfiler till appens mapp "Dokument" på din enhet.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Kopiera filer från en iOS-app till din dator  
1. I iTunes väljer du appen från listan i avsnittet "Fildelning".
2. För att kopiera filer till din dator, dra och släpp dem från listan "Dokument" till en mapp eller ett fönster på din dator.
3. Du kan också välja de ljudfiler du vill kopiera till din dator från listan "Dokument" och klicka på "Spara till." Leta reda på mappen på din dator där du vill kopiera filerna och klicka på "Spara till."

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Öppna delade filer  
För att spela filer som kopierats från din dator till din iOS-enhet, öppna appen som är avsedd för Fildelning (t.ex. Evermusic eller Flacbox) och tryck på fliken "Lokala filer" och välj mappen "Dokument". Du ser en lista över tillgängliga filer som du kan spela.

## Ta bort delade filer från din iOS-enhet  

### Med iTunes  
1. Öppna iTunes på din Mac/PC.
2. Anslut din iOS-enhet till datorn via USB.
3. Välj din enhet i iTunes.
4. Klicka på "Appar" i det vänstra sidofältet och bläddra till avsnittet "Fildelning".
5. Välj appen som innehåller filen du vill ta bort.
6. Välj filen i listan "Dokument" och tryck på Delete-tangenten på tangentbordet.
7. Bekräfta borttagningen i iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Med den inbyggda filhanteraren  
1. Öppna avsnittet "Lokala filer".
2. Tryck på knappen "Fler åtgärder" i det övre högra hörnet och välj "Välja" från menyn.
3. Välj filerna du vill ta bort.
4. Tryck på åtgärden "Ta bort" (om den inte syns, tryck på menyn "Fler åtgärder").
5. Bekräfta borttagningen.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Med iTunes Fildelning kan du enkelt hantera filer mellan din dator och iOS-appar, vilket förbättrar din musik- och fildelningsupplevelse.

## Vanliga frågor

{{% details title="Behöver jag en internetanslutning för att överföra filer via USB?" closed="true" %}}
Nej. Fildelning fungerar helt via USB-kabelanslutningen mellan din dator och din iOS-enhet. Inget internet krävs.
{{% /details %}}

{{% details title="Vilka filformat kan jag överföra till Evermusic eller Flacbox?" closed="true" %}}
Båda apparna stöder ett brett utbud av ljudformat inklusive MP3, FLAC, AAC, WAV, AIFF, OGG, WMA och fler. Kontrollera appens dokumentation för den fullständiga listan över format som stöds.
{{% /details %}}

{{% details title="Varför ser jag inte fliken Filer i Finder?" closed="true" %}}
Fliken Filer visas bara när din enhet har minst en app installerad som stöder Fildelning. Installera Evermusic, Flacbox eller Evertag och anslut sedan din enhet igen.
{{% /details %}}

{{% details title="Kan jag överföra filer trådlöst istället för att använda en USB-kabel?" closed="true" %}}
Ja. Evermusic och Flacbox stöder också molnlagringstjänster och Wi-Fi-överföring. USB-fildelning genom Finder eller iTunes är dock vanligtvis snabbare för stora musikbibliotek.
{{% /details %}}

{{% details title="Kommer överföring av filer via Finder att skriva över befintliga filer på min enhet?" closed="true" %}}
Nej. Nya filer läggs till bredvid befintliga. Om en fil med samma namn redan finns kan macOS automatiskt byta namn på den nya filen.
{{% /details %}}

{{% details title="Fungerar den här metoden med Windows-datorer?" closed="true" %}}
Ja. På Windows använder du iTunes för att överföra filer. Processen är densamma som beskrivs i iTunes-avsnittet ovan. Installera iTunes från Microsoft Store eller Apples webbplats.
{{% /details %}}

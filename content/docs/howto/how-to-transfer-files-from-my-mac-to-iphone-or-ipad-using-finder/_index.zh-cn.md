**摘要：** 使用 USB 数据线将 iPhone 或 iPad 连接到 Mac（或 PC）。在 macOS Catalina 及更高版本上，使用 Finder。在较旧的 macOS 或 Windows 上，使用 iTunes。将文件拖入 Evermusic、Flacbox 或 Evertag 等应用即可立即传输。

iTunes 文件共享提供了一种在电脑和 iPhone、iPad 或 iPod touch 上的 iOS 应用之间传输文件的便捷方式。无论您在 Mac 或 PC 上拥有喜爱的专辑、艺术家或曲目的收藏，都可以轻松地将这些内容复制到 Evermusic、Flacbox 或 Evertag 等应用中。

以下是使用 iTunes 文件共享的分步指南。

## 适用于 macOS Catalina 或更高版本

**前提条件**  
- macOS Catalina 或更高版本
- 装有最新 iOS 或 iPadOS 版本的 iOS 设备（iPhone、iPad 或 iPod touch）
- 兼容文件共享的 iOS 应用

### 步骤  
1. 在 Mac 上打开 Finder 窗口。
2. 使用 USB 数据线将 iOS 设备连接到电脑。
3. 在 Finder 中选择您的设备。
4. 点击"文件"选项卡查看支持文件共享的应用列表。如果您没有看到"文件"部分，则表示您的设备没有支持文件共享的应用。
5. 点击应用旁边的三角形以查看可以共享的文件。

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### 从 Mac 复制文件到 iOS 设备  
1. 选择要复制的文件，确保它们与应用兼容（有关支持的格式，请参阅应用的用户指南）。
2. 将选定的文件拖放到 Finder 中 iOS 设备上的应用图标上。Finder 将把文件复制到您的设备上。

### 从 iOS 设备复制文件到 Mac  
1. 选择要从 iOS 设备复制的文件。
2. 将选定的文件拖放到电脑上的文件夹或窗口中。Finder 将把文件复制到您的电脑上。

## 适用于 Catalina 之前的 macOS 版本

**前提条件**  
- 最新版本的 iTunes
- 装有最新 iOS 版本的 iOS 设备
- 装有 Windows 7 或更高版本的 Windows PC（如果使用 Windows）

### 步骤
1. 在 Mac 或 PC 上打开 iTunes。
2. 使用 USB Lightning 数据线将 iPhone 或 iPad 连接到电脑。
3. 在 iTunes 中选择您的 iOS 设备。

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. 在左侧边栏中点击"应用"部分。
5. 滚动到页面底部的"文件共享"部分。
6. 选择应用以查看设备上可供共享的文件。

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### 从电脑复制文件到应用  
1. 在 iTunes 左侧边栏中，点击"应用"部分。
2. 向下滚动到底部的"文件共享"部分。
3. 选择应用以查看设备上可供共享的文件。
4. 要将文件复制到设备，请将它们从文件夹或窗口拖放到 iTunes 中的"文档"列表。
5. 或者，点击 iTunes 中"文档"列表中的"添加"，在电脑上找到音频文件，然后点击"添加"。iTunes 将把这些音乐文件复制到设备上应用的"文档"目录中。

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### 从 iOS 应用复制文件到电脑  
1. 在 iTunes 中，从"文件共享"部分的列表中选择应用。
2. 要将文件复制到电脑，请将它们从"文档"列表拖放到电脑上的文件夹或窗口中。
3. 您也可以从"文档"列表中选择要复制到电脑的音频文件，然后点击"存储到"。找到电脑上要复制文件的文件夹，然后点击"存储到"。

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 打开共享文件  
要播放从电脑复制到 iOS 设备的文件，请打开指定用于文件共享的应用（例如 Evermusic 或 Flacbox），点击"本地文件"选项卡，然后选择"文档"文件夹。您将看到可以播放的可用文件列表。

## 从 iOS 设备删除共享文件  

### 使用 iTunes  
1. 在 Mac/PC 上打开 iTunes。
2. 通过 USB 将 iOS 设备连接到电脑。
3. 在 iTunes 中选择您的设备。
4. 在左侧边栏中点击"应用"，滚动到"文件共享"部分。
5. 选择包含要删除文件的应用。
6. 在"文档"列表中选择文件，然后按键盘上的 Delete 键。
7. 在 iTunes 中确认删除。

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### 使用内置文件管理器  
1. 打开"本地文件"部分。
2. 点击右上角的"更多操作"按钮，从菜单中选择"选择"。
3. 选择要删除的文件。
4. 点击"删除"操作（如果不可见，请点击"更多操作"菜单）。
5. 确认删除。

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

通过 iTunes 文件共享，您可以轻松管理电脑和 iOS 应用之间的文件，提升您的音乐和文件共享体验。

## 常见问题

{{% details title="通过 USB 传输文件需要互联网连接吗？" closed="true" %}}
不需要。文件共享完全通过电脑和 iOS 设备之间的 USB 数据线连接运行。不需要互联网。
{{% /details %}}

{{% details title="我可以将哪些文件格式传输到 Evermusic 或 Flacbox？" closed="true" %}}
两款应用都支持多种音频格式，包括 MP3、FLAC、AAC、WAV、AIFF、OGG、WMA 等。请查看应用文档以获取支持格式的完整列表。
{{% /details %}}

{{% details title="为什么我在 Finder 中看不到文件选项卡？" closed="true" %}}
"文件"选项卡仅在设备上安装了至少一个支持文件共享的应用时才会出现。安装 Evermusic、Flacbox 或 Evertag，然后重新连接设备。
{{% /details %}}

{{% details title="我可以无线传输文件而不使用 USB 数据线吗？" closed="true" %}}
可以。Evermusic 和 Flacbox 还支持云存储服务和 Wi-Fi 传输。但是，通过 Finder 或 iTunes 进行的 USB 文件共享通常对于大型音乐库来说更快。
{{% /details %}}

{{% details title="通过 Finder 传输文件会覆盖设备上的现有文件吗？" closed="true" %}}
不会。新文件会与现有文件一起添加。如果已存在同名文件，macOS 可能会自动重命名新文件。
{{% /details %}}

{{% details title="此方法适用于 Windows PC 吗？" closed="true" %}}
是的。在 Windows 上，使用 iTunes 传输文件。过程与上面 iTunes 部分描述的相同。从 Microsoft Store 或 Apple 网站安装 iTunes。
{{% /details %}}

**In breve:** Collega il tuo iPhone o iPad al Mac (o PC) con un cavo USB. Su macOS Catalina e versioni successive, usa Finder. Su versioni precedenti di macOS o Windows, usa iTunes. Trascina i file in un'app come Evermusic, Flacbox o Evertag per trasferirli istantaneamente.

La condivisione file di iTunes offre un modo pratico per trasferire file tra il computer e le applicazioni iOS sul tuo iPhone, iPad o iPod touch. Che tu abbia una collezione di album, artisti o brani preferiti sul Mac o PC, puoi facilmente copiare questo contenuto in app come Evermusic, Flacbox o Evertag.

Ecco una guida passo passo per usare la condivisione file di iTunes.

## Per macOS Catalina o successivo

**Prerequisiti**  
- macOS Catalina o una versione successiva
- Un dispositivo iOS (iPhone, iPad o iPod touch) con l'ultima versione di iOS o iPadOS
- Un'app iOS compatibile con la condivisione file

### Passaggi  
1. Apri una finestra del Finder sul tuo Mac.
2. Collega il dispositivo iOS al computer usando un cavo USB.
3. Seleziona il tuo dispositivo nel Finder.
4. Fai clic sulla scheda "File" per visualizzare un elenco di app che supportano la condivisione file. Se non vedi una sezione "File", significa che il tuo dispositivo non ha app che supportano la condivisione file.
5. Fai clic sul triangolo accanto a un'app per mostrare i file che puoi condividere.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Copiare file dal Mac al dispositivo iOS  
1. Seleziona i file che desideri copiare, assicurandoti che siano compatibili con l'app (consulta la guida utente dell'app per i formati supportati).
2. Trascina e rilascia i file selezionati sull'icona dell'app sul tuo dispositivo iOS nel Finder. Il Finder copierà i file sul tuo dispositivo.

### Copiare file dal dispositivo iOS al Mac  
1. Seleziona i file che desideri copiare dal tuo dispositivo iOS.
2. Trascina e rilascia i file selezionati in una cartella o finestra sul tuo computer. Il Finder copierà i file sul tuo computer.

## Per versioni di macOS precedenti a Catalina

**Prerequisiti**  
- L'ultima versione di iTunes
- Un dispositivo iOS con l'ultima versione di iOS
- Un PC Windows con Windows 7 o successivo (se si usa Windows)

### Passaggi
1. Apri iTunes sul tuo Mac o PC.
2. Collega il tuo iPhone o iPad al computer usando un cavo USB Lightning.
3. Seleziona il tuo dispositivo iOS in iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Fai clic sulla sezione App nella barra laterale sinistra.
5. Scorri fino alla sezione Condivisione file in fondo alla pagina.
6. Seleziona l'app per vedere quali file sono disponibili per la condivisione sul tuo dispositivo.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Copiare file dal computer all'app  
1. Nella barra laterale sinistra di iTunes, fai clic sulla sezione "App".
2. Scorri fino alla sezione "Condivisione file" in fondo.
3. Seleziona l'app per vedere i file disponibili per la condivisione sul tuo dispositivo.
4. Per copiare file sul dispositivo, trascinali da una cartella o finestra nell'elenco "Documenti" in iTunes.
5. In alternativa, fai clic su "Aggiungi" nell'elenco "Documenti" in iTunes, individua i file audio sul computer e fai clic su "Aggiungi". iTunes copierà questi file musicali nella directory "Documenti" dell'app sul tuo dispositivo.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Copiare file da un'app iOS al computer  
1. In iTunes, seleziona l'app dall'elenco nella sezione "Condivisione file".
2. Per copiare file sul computer, trascinali dall'elenco "Documenti" in una cartella o finestra sul computer.
3. Puoi anche selezionare i file audio che desideri copiare sul computer dall'elenco "Documenti" e fare clic su "Salva in". Individua la cartella sul computer in cui desideri copiare i file e fai clic su "Salva in".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Aprire i file condivisi  
Per riprodurre i file copiati dal computer al dispositivo iOS, apri l'app designata per la condivisione file (ad es., Evermusic o Flacbox) e tocca la scheda "File locali" e scegli la cartella "Documenti". Vedrai un elenco di file disponibili che puoi riprodurre.

## Eliminare i file condivisi dal dispositivo iOS

### Con iTunes  
1. Apri iTunes sul tuo Mac/PC.
2. Collega il dispositivo iOS al computer tramite USB.
3. Seleziona il tuo dispositivo in iTunes.
4. Fai clic su "App" nella barra laterale sinistra e scorri fino alla sezione "Condivisione file".
5. Scegli l'app contenente il file che desideri eliminare.
6. Seleziona il file nell'elenco "Documenti" e premi il tasto Canc sulla tastiera.
7. Conferma l'eliminazione in iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Con il file manager integrato  
1. Apri la sezione "File locali".
2. Tocca il pulsante "Altre azioni" nell'angolo in alto a destra e scegli "Selezionare" dal menu.
3. Seleziona i file che desideri eliminare.
4. Tocca l'azione "Eliminare" (se non è visibile, tocca il menu "Altre azioni").
5. Conferma l'eliminazione.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Con la condivisione file di iTunes, puoi gestire facilmente i file tra il computer e le app iOS, migliorando la tua esperienza musicale e di condivisione file.

## Domande frequenti

{{% details title="Ho bisogno di una connessione internet per trasferire file tramite USB?" closed="true" %}}
No. La condivisione file funziona interamente tramite la connessione via cavo USB tra il computer e il dispositivo iOS. Non è necessario internet.
{{% /details %}}

{{% details title="Quali formati di file posso trasferire su Evermusic o Flacbox?" closed="true" %}}
Entrambe le app supportano un'ampia gamma di formati audio tra cui MP3, FLAC, AAC, WAV, AIFF, OGG, WMA e altri. Consulta la documentazione dell'app per l'elenco completo dei formati supportati.
{{% /details %}}

{{% details title="Perché non vedo la scheda File nel Finder?" closed="true" %}}
La scheda File appare solo quando il dispositivo ha almeno un'app installata che supporta la condivisione file. Installa Evermusic, Flacbox o Evertag, quindi ricollega il dispositivo.
{{% /details %}}

{{% details title="Posso trasferire file in modalità wireless invece di usare un cavo USB?" closed="true" %}}
Sì. Evermusic e Flacbox supportano anche servizi di archiviazione cloud e trasferimento Wi-Fi. Tuttavia, la condivisione file USB tramite Finder o iTunes è generalmente più veloce per grandi librerie musicali.
{{% /details %}}

{{% details title="Il trasferimento di file tramite Finder sovrascriverà i file esistenti sul mio dispositivo?" closed="true" %}}
No. I nuovi file vengono aggiunti accanto a quelli esistenti. Se esiste già un file con lo stesso nome, macOS potrebbe rinominare automaticamente il nuovo file.
{{% /details %}}

{{% details title="Questo metodo funziona con i PC Windows?" closed="true" %}}
Sì. Su Windows, usa iTunes per trasferire i file. Il processo è lo stesso descritto nella sezione iTunes sopra. Installa iTunes dal Microsoft Store o dal sito web di Apple.
{{% /details %}}

**בקצרה:** חברו את ה-iPhone או ה-iPad שלכם ל-Mac (או למחשב) באמצעות כבל USB. ב-macOS Catalina ואילך, השתמשו ב-Finder. בגרסאות ישנות יותר של macOS או ב-Windows, השתמשו ב-iTunes. גררו קבצים לאפליקציה כמו Evermusic, Flacbox או Evertag כדי להעביר אותם מיידית.

שיתוף קבצים של iTunes מציע דרך נוחה להעברת קבצים בין המחשב שלכם לאפליקציות iOS ב-iPhone, iPad או iPod touch. בין אם יש לכם אוסף של אלבומים, אמנים או שירים מועדפים ב-Mac או במחשב, תוכלו להעתיק בקלות תוכן זה לאפליקציות כמו Evermusic, Flacbox או Evertag.

הנה מדריך שלב אחר שלב לשימוש בשיתוף קבצים של iTunes.

## עבור macOS Catalina או מאוחר יותר

**דרישות מוקדמות**  
- macOS Catalina או גרסה מאוחרת יותר
- מכשיר iOS (iPhone, iPad או iPod touch) עם גרסת iOS או iPadOS העדכנית ביותר
- אפליקציית iOS תואמת שיתוף קבצים

### שלבים  
1. פתחו חלון Finder ב-Mac שלכם.
2. חברו את מכשיר ה-iOS למחשב באמצעות כבל USB.
3. בחרו את המכשיר שלכם ב-Finder.
4. לחצו על הכרטיסייה "קבצים" כדי להציג רשימת אפליקציות התומכות בשיתוף קבצים. אם אינכם רואים את קטע "קבצים", פירוש הדבר שאין במכשיר שלכם אפליקציות התומכות בשיתוף קבצים.
5. לחצו על המשולש ליד אפליקציה כדי לחשוף את הקבצים שאתם יכולים לשתף.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### העתקת קבצים מה-Mac למכשיר ה-iOS  
1. בחרו את הקבצים שברצונכם להעתיק, וודאו שהם תואמים לאפליקציה (עיינו במדריך למשתמש של האפליקציה לפורמטים נתמכים).
2. גררו ושחררו את הקבצים הנבחרים על סמל האפליקציה במכשיר ה-iOS ב-Finder. Finder יעתיק את הקבצים למכשיר שלכם.

### העתקת קבצים ממכשיר ה-iOS ל-Mac  
1. בחרו את הקבצים שברצונכם להעתיק ממכשיר ה-iOS.
2. גררו ושחררו את הקבצים הנבחרים לתיקייה או חלון במחשב שלכם. Finder יעתיק את הקבצים למחשב שלכם.

## עבור גרסאות macOS לפני Catalina

**דרישות מוקדמות**  
- הגרסה העדכנית ביותר של iTunes
- מכשיר iOS עם גרסת iOS העדכנית ביותר
- מחשב Windows עם Windows 7 או מאוחר יותר (אם משתמשים ב-Windows)

### שלבים
1. פתחו את iTunes ב-Mac או במחשב שלכם.
2. חברו את ה-iPhone או ה-iPad למחשב באמצעות כבל USB Lightning.
3. בחרו את מכשיר ה-iOS ב-iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. לחצו על קטע האפליקציות בסרגל הצד השמאלי.
5. גללו למטה לקטע שיתוף קבצים בתחתית העמוד.
6. בחרו את האפליקציה כדי לראות אילו קבצים זמינים לשיתוף במכשיר שלכם.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### העתקת קבצים מהמחשב לאפליקציה  
1. בסרגל הצד השמאלי של iTunes, לחצו על קטע "אפליקציות".
2. גללו למטה לקטע "שיתוף קבצים".
3. בחרו את האפליקציה כדי לראות את הקבצים הזמינים לשיתוף במכשיר שלכם.
4. כדי להעתיק קבצים למכשיר שלכם, גררו ושחררו אותם מתיקייה או חלון לרשימת "מסמכים" ב-iTunes.
5. לחלופין, לחצו על "הוסף" ברשימת "מסמכים" ב-iTunes, אתרו את קבצי השמע במחשב שלכם ולחצו על "הוסף". iTunes יעתיק את קבצי המוזיקה לספריית "מסמכים" של האפליקציה במכשיר שלכם.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### העתקת קבצים מאפליקציית iOS למחשב  
1. ב-iTunes, בחרו את האפליקציה מהרשימה בקטע "שיתוף קבצים".
2. כדי להעתיק קבצים למחשב שלכם, גררו ושחררו אותם מרשימת "מסמכים" לתיקייה או חלון במחשב שלכם.
3. תוכלו גם לבחור את קבצי השמע שברצונכם להעתיק למחשב מרשימת "מסמכים" וללחוץ על "שמור ב". אתרו את התיקייה במחשב שלכם שאליה תרצו להעתיק את הקבצים ולחצו על "שמור ב".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## פתיחת קבצים משותפים  
כדי לנגן קבצים שהועתקו מהמחשב למכשיר ה-iOS, פתחו את האפליקציה המיועדת לשיתוף קבצים (למשל, Evermusic או Flacbox) והקישו על הכרטיסייה "קבצים מקומיים" ובחרו בתיקיית "מסמכים". תראו רשימת קבצים זמינים שתוכלו לנגן.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_549f899dd8dc41b29efa7f548c826d15~mv2.png" alt="Evermusic Files Section" width="400" >}}

## מחיקת קבצים משותפים ממכשיר ה-iOS

### עם iTunes  
1. פתחו את iTunes ב-Mac/במחשב שלכם.
2. חברו את מכשיר ה-iOS למחשב דרך USB.
3. בחרו את המכשיר שלכם ב-iTunes.
4. לחצו על "אפליקציות" בסרגל הצד השמאלי וגללו לקטע "שיתוף קבצים".
5. בחרו את האפליקציה המכילה את הקובץ שברצונכם למחוק.
6. בחרו את הקובץ ברשימת "מסמכים" ולחצו על מקש Delete במקלדת.
7. אשרו את המחיקה ב-iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### עם מנהל הקבצים המובנה  
1. פתחו את קטע "קבצים מקומיים".
2. הקישו על כפתור "עוד פעולות" בפינה הימנית העליונה ובחרו "בחר" מהתפריט.
3. בחרו את הקבצים שברצונכם למחוק.
4. הקישו על פעולת "למחוק" (אם היא אינה גלויה, הקישו על תפריט "עוד פעולות").
5. אשרו את המחיקה.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

עם שיתוף קבצים של iTunes, תוכלו לנהל בקלות קבצים בין המחשב ואפליקציות iOS, ולשפר את חוויית המוזיקה ושיתוף הקבצים שלכם.

## שאלות נפוצות

{{% details title="האם אני צריך חיבור אינטרנט כדי להעביר קבצים דרך USB?" closed="true" %}}
לא. שיתוף קבצים פועל לחלוטין דרך חיבור כבל ה-USB בין המחשב למכשיר ה-iOS. אין צורך באינטרנט.
{{% /details %}}

{{% details title="אילו פורמטי קבצים אני יכול להעביר ל-Evermusic או Flacbox?" closed="true" %}}
שתי האפליקציות תומכות במגוון רחב של פורמטי שמע כולל MP3, FLAC, AAC, WAV, AIFF, OGG, WMA ועוד. בדקו את תיעוד האפליקציה לרשימה המלאה של פורמטים נתמכים.
{{% /details %}}

{{% details title="למה אני לא רואה את כרטיסיית הקבצים ב-Finder?" closed="true" %}}
כרטיסיית הקבצים מופיעה רק כאשר למכשיר שלכם מותקנת לפחות אפליקציה אחת התומכת בשיתוף קבצים. התקינו Evermusic, Flacbox או Evertag, ואז חברו מחדש את המכשיר שלכם.
{{% /details %}}

{{% details title="האם אני יכול להעביר קבצים באופן אלחוטי במקום להשתמש בכבל USB?" closed="true" %}}
כן. Evermusic ו-Flacbox תומכים גם בשירותי אחסון ענן והעברת Wi-Fi. עם זאת, שיתוף קבצים דרך USB באמצעות Finder או iTunes הוא בדרך כלל מהיר יותר עבור ספריות מוזיקה גדולות.
{{% /details %}}

{{% details title="האם העברת קבצים דרך Finder תדרוס קבצים קיימים במכשיר שלי?" closed="true" %}}
לא. קבצים חדשים מתווספים לצד הקיימים. אם כבר קיים קובץ עם אותו שם, macOS עשוי לשנות את שם הקובץ החדש באופן אוטומטי.
{{% /details %}}

{{% details title="האם שיטה זו עובדת עם מחשבי Windows?" closed="true" %}}
כן. ב-Windows, השתמשו ב-iTunes להעברת קבצים. התהליך זהה לזה המתואר בקטע iTunes למעלה. התקינו את iTunes מ-Microsoft Store או מאתר Apple.
{{% /details %}}

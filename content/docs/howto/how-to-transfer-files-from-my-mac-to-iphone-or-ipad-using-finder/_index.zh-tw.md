**摘要：** 使用 USB 傳輸線將 iPhone 或 iPad 連接到 Mac（或 PC）。在 macOS Catalina 及更新版本上，使用 Finder。在較舊的 macOS 或 Windows 上，使用 iTunes。將檔案拖入 Evermusic、Flacbox 或 Evertag 等應用程式即可立即傳輸。

iTunes 檔案共享提供了一種在電腦和 iPhone、iPad 或 iPod touch 上的 iOS 應用程式之間傳輸檔案的便捷方式。無論您在 Mac 或 PC 上擁有喜愛的專輯、藝人或曲目的收藏，都可以輕鬆地將這些內容複製到 Evermusic、Flacbox 或 Evertag 等應用程式中。

以下是使用 iTunes 檔案共享的逐步指南。

## 適用於 macOS Catalina 或更新版本

**先決條件**  
- macOS Catalina 或更新版本
- 裝有最新 iOS 或 iPadOS 版本的 iOS 裝置（iPhone、iPad 或 iPod touch）
- 相容檔案共享的 iOS 應用程式

### 步驟  
1. 在 Mac 上開啟 Finder 視窗。
2. 使用 USB 傳輸線將 iOS 裝置連接到電腦。
3. 在 Finder 中選擇您的裝置。
4. 點選「檔案」標籤頁查看支援檔案共享的應用程式列表。如果您沒有看到「檔案」部分，則表示您的裝置沒有支援檔案共享的應用程式。
5. 點選應用程式旁邊的三角形以查看可以共享的檔案。

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### 從 Mac 複製檔案到 iOS 裝置  
1. 選擇要複製的檔案，確保它們與應用程式相容（有關支援的格式，請參閱應用程式的使用者指南）。
2. 將選定的檔案拖放到 Finder 中 iOS 裝置上的應用程式圖示上。Finder 將把檔案複製到您的裝置上。

### 從 iOS 裝置複製檔案到 Mac  
1. 選擇要從 iOS 裝置複製的檔案。
2. 將選定的檔案拖放到電腦上的資料夾或視窗中。Finder 將把檔案複製到您的電腦上。

## 適用於 Catalina 之前的 macOS 版本

**先決條件**  
- 最新版本的 iTunes
- 裝有最新 iOS 版本的 iOS 裝置
- 裝有 Windows 7 或更新版本的 Windows PC（如果使用 Windows）

### 步驟
1. 在 Mac 或 PC 上開啟 iTunes。
2. 使用 USB Lightning 傳輸線將 iPhone 或 iPad 連接到電腦。
3. 在 iTunes 中選擇您的 iOS 裝置。

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. 在左側邊欄中點選「應用程式」部分。
5. 捲動到頁面底部的「檔案共享」部分。
6. 選擇應用程式以查看裝置上可供共享的檔案。

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### 從電腦複製檔案到應用程式  
1. 在 iTunes 左側邊欄中，點選「應用程式」部分。
2. 向下捲動到底部的「檔案共享」部分。
3. 選擇應用程式以查看裝置上可供共享的檔案。
4. 要將檔案複製到裝置，請將它們從資料夾或視窗拖放到 iTunes 中的「文件」列表。
5. 或者，點選 iTunes 中「文件」列表中的「加入」，在電腦上找到音訊檔案，然後點選「加入」。iTunes 將把這些音樂檔案複製到裝置上應用程式的「文件」目錄中。

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### 從 iOS 應用程式複製檔案到電腦  
1. 在 iTunes 中，從「檔案共享」部分的列表中選擇應用程式。
2. 要將檔案複製到電腦，請將它們從「文件」列表拖放到電腦上的資料夾或視窗中。
3. 您也可以從「文件」列表中選擇要複製到電腦的音訊檔案，然後點選「儲存到」。找到電腦上要複製檔案的資料夾，然後點選「儲存到」。

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 開啟共享檔案  
要播放從電腦複製到 iOS 裝置的檔案，請開啟指定用於檔案共享的應用程式（例如 Evermusic 或 Flacbox），點選「本機檔案」標籤頁，然後選擇「文件」資料夾。您將看到可以播放的可用檔案列表。

## 從 iOS 裝置刪除共享檔案  

### 使用 iTunes  
1. 在 Mac/PC 上開啟 iTunes。
2. 透過 USB 將 iOS 裝置連接到電腦。
3. 在 iTunes 中選擇您的裝置。
4. 在左側邊欄中點選「應用程式」，捲動到「檔案共享」部分。
5. 選擇包含要刪除檔案的應用程式。
6. 在「文件」列表中選擇檔案，然後按鍵盤上的 Delete 鍵。
7. 在 iTunes 中確認刪除。

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### 使用內建檔案管理器  
1. 開啟「本機檔案」部分。
2. 點選右上角的「更多操作」按鈕，從選單中選擇「選擇」。
3. 選擇要刪除的檔案。
4. 點選「刪除」操作（如果不可見，請點選「更多操作」選單）。
5. 確認刪除。

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

透過 iTunes 檔案共享，您可以輕鬆管理電腦和 iOS 應用程式之間的檔案，提升您的音樂和檔案共享體驗。

## 常見問題

{{% details title="透過 USB 傳輸檔案需要網路連線嗎？" closed="true" %}}
不需要。檔案共享完全透過電腦和 iOS 裝置之間的 USB 傳輸線連接運作。不需要網路。
{{% /details %}}

{{% details title="我可以將哪些檔案格式傳輸到 Evermusic 或 Flacbox？" closed="true" %}}
兩款應用程式都支援多種音訊格式，包括 MP3、FLAC、AAC、WAV、AIFF、OGG、WMA 等。請查看應用程式文件以取得支援格式的完整列表。
{{% /details %}}

{{% details title="為什麼我在 Finder 中看不到「檔案」標籤頁？" closed="true" %}}
「檔案」標籤頁僅在裝置上安裝了至少一個支援檔案共享的應用程式時才會出現。安裝 Evermusic、Flacbox 或 Evertag，然後重新連接裝置。
{{% /details %}}

{{% details title="我可以無線傳輸檔案而不使用 USB 傳輸線嗎？" closed="true" %}}
可以。Evermusic 和 Flacbox 還支援雲端儲存服務和 Wi-Fi 傳輸。但是，透過 Finder 或 iTunes 進行的 USB 檔案共享通常對於大型音樂庫來說更快。
{{% /details %}}

{{% details title="透過 Finder 傳輸檔案會覆蓋裝置上的現有檔案嗎？" closed="true" %}}
不會。新檔案會與現有檔案一起加入。如果已存在同名檔案，macOS 可能會自動重新命名新檔案。
{{% /details %}}

{{% details title="此方法適用於 Windows PC 嗎？" closed="true" %}}
是的。在 Windows 上，使用 iTunes 傳輸檔案。過程與上面 iTunes 部分描述的相同。從 Microsoft Store 或 Apple 網站安裝 iTunes。
{{% /details %}}

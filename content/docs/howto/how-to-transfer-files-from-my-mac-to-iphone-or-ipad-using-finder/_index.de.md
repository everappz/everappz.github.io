**Kurzfassung:** Verbinden Sie Ihr iPhone oder iPad mit Ihrem Mac (oder PC) über ein USB-Kabel. Unter macOS Catalina und neuer verwenden Sie Finder. Unter älteren macOS-Versionen oder Windows verwenden Sie iTunes. Ziehen Sie Dateien in eine App wie Evermusic, Flacbox oder Evertag, um sie sofort zu übertragen.

Die iTunes Dateifreigabe bietet eine bequeme Möglichkeit, Dateien zwischen Ihrem Computer und iOS-Anwendungen auf Ihrem iPhone, iPad oder iPod touch zu übertragen. Ob Sie eine Sammlung von Lieblingsalben, Künstlern oder Titeln auf Ihrem Mac oder PC haben, Sie können diesen Inhalt einfach in Apps wie Evermusic, Flacbox oder Evertag kopieren.

Hier ist eine Schritt-für-Schritt-Anleitung zur Verwendung der iTunes Dateifreigabe.

## Für macOS Catalina oder neuer

**Voraussetzungen**  
- macOS Catalina oder eine neuere Version
- Ein iOS-Gerät (iPhone, iPad oder iPod touch) mit der neuesten iOS- oder iPadOS-Version
- Eine iOS-App, die mit der Dateifreigabe kompatibel ist

### Schritte  
1. Öffnen Sie ein Finder-Fenster auf Ihrem Mac.
2. Verbinden Sie Ihr iOS-Gerät mit Ihrem Computer über ein USB-Kabel.
3. Wählen Sie Ihr Gerät im Finder aus.
4. Klicken Sie auf die Registerkarte "Dateien", um eine Liste der Apps anzuzeigen, die die Dateifreigabe unterstützen. Wenn Sie keinen Bereich "Dateien" sehen, bedeutet dies, dass Ihr Gerät keine Apps hat, die die Dateifreigabe unterstützen.
5. Klicken Sie auf das Dreieck neben einer App, um die Dateien anzuzeigen, die Sie freigeben können.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Dateien vom Mac auf Ihr iOS-Gerät kopieren  
1. Wählen Sie die Dateien aus, die Sie kopieren möchten, und stellen Sie sicher, dass sie mit der App kompatibel sind (siehe Benutzerhandbuch der App für unterstützte Formate).
2. Ziehen Sie die ausgewählten Dateien auf das App-Symbol auf Ihrem iOS-Gerät im Finder. Der Finder kopiert die Dateien auf Ihr Gerät.

### Dateien von Ihrem iOS-Gerät auf den Mac kopieren  
1. Wählen Sie die Dateien aus, die Sie von Ihrem iOS-Gerät kopieren möchten.
2. Ziehen Sie die ausgewählten Dateien in einen Ordner oder ein Fenster auf Ihrem Computer. Der Finder kopiert die Dateien auf Ihren Computer.

## Für macOS-Versionen vor Catalina

**Voraussetzungen**  
- Die neueste Version von iTunes
- Ein iOS-Gerät mit der neuesten iOS-Version
- Ein Windows-PC mit Windows 7 oder neuer (bei Verwendung von Windows)

### Schritte
1. Öffnen Sie iTunes auf Ihrem Mac oder PC.
2. Verbinden Sie Ihr iPhone oder iPad mit Ihrem Computer über ein USB Lightning-Kabel.
3. Wählen Sie Ihr iOS-Gerät in iTunes aus.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Klicken Sie auf den Bereich Apps in der linken Seitenleiste.
5. Scrollen Sie zum Bereich Dateifreigabe am unteren Rand der Seite.
6. Wählen Sie die App aus, um zu sehen, welche Dateien auf Ihrem Gerät zur Freigabe verfügbar sind.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Dateien von Ihrem Computer in die App kopieren  
1. Klicken Sie in der linken Seitenleiste von iTunes auf den Bereich "Apps".
2. Scrollen Sie nach unten zum Bereich "Dateifreigabe".
3. Wählen Sie die App aus, um die verfügbaren Dateien zur Freigabe auf Ihrem Gerät zu sehen.
4. Um Dateien auf Ihr Gerät zu kopieren, ziehen Sie sie aus einem Ordner oder Fenster in die Liste "Dokumente" in iTunes.
5. Alternativ klicken Sie auf "Hinzufügen" in der Liste "Dokumente" in iTunes, suchen Sie die Audiodateien auf Ihrem Computer und klicken Sie auf "Hinzufügen". iTunes kopiert diese Musikdateien in das "Dokumente"-Verzeichnis der App auf Ihrem Gerät.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Dateien von einer iOS-App auf Ihren Computer kopieren  
1. Wählen Sie in iTunes die App aus der Liste im Bereich "Dateifreigabe" aus.
2. Um Dateien auf Ihren Computer zu kopieren, ziehen Sie sie aus der Liste "Dokumente" in einen Ordner oder ein Fenster auf Ihrem Computer.
3. Sie können auch die Audiodateien auswählen, die Sie auf Ihren Computer kopieren möchten, aus der Liste "Dokumente" und auf "Sichern unter" klicken. Suchen Sie den Ordner auf Ihrem Computer, in den Sie die Dateien kopieren möchten, und klicken Sie auf "Sichern unter".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Freigegebene Dateien öffnen  
Um Dateien abzuspielen, die von Ihrem Computer auf Ihr iOS-Gerät kopiert wurden, öffnen Sie die für die Dateifreigabe vorgesehene App (z. B. Evermusic oder Flacbox) und tippen Sie auf die Registerkarte "Lokale Dateien" und wählen Sie den Ordner "Dokumente". Sie sehen eine Liste der verfügbaren Dateien, die Sie abspielen können.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_549f899dd8dc41b29efa7f548c826d15~mv2.png" alt="Evermusic Files Section" width="400" >}}

## Freigegebene Dateien von Ihrem iOS-Gerät löschen

### Mit iTunes  
1. Öffnen Sie iTunes auf Ihrem Mac/PC.
2. Verbinden Sie Ihr iOS-Gerät über USB mit dem Computer.
3. Wählen Sie Ihr Gerät in iTunes aus.
4. Klicken Sie auf "Apps" in der linken Seitenleiste und scrollen Sie zum Bereich "Dateifreigabe".
5. Wählen Sie die App aus, die die Datei enthält, die Sie löschen möchten.
6. Wählen Sie die Datei in der Liste "Dokumente" aus und drücken Sie die Entf-Taste auf Ihrer Tastatur.
7. Bestätigen Sie die Löschung in iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Mit dem integrierten Dateimanager  
1. Öffnen Sie den Bereich "Lokale Dateien".
2. Tippen Sie auf die Schaltfläche "Weitere Aktionen" in der oberen rechten Ecke und wählen Sie "Auswählen" aus dem Menü.
3. Wählen Sie die Dateien aus, die Sie löschen möchten.
4. Tippen Sie auf die Aktion "Löschen" (wenn sie nicht sichtbar ist, tippen Sie auf das Menü "Weitere Aktionen").
5. Bestätigen Sie die Löschung.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Mit der iTunes Dateifreigabe können Sie Dateien zwischen Ihrem Computer und iOS-Apps einfach verwalten und Ihr Musik- und Dateifreigabe-Erlebnis verbessern.

## Häufig gestellte Fragen

{{% details title="Benötige ich eine Internetverbindung, um Dateien über USB zu übertragen?" closed="true" %}}
Nein. Die Dateifreigabe funktioniert ausschließlich über die USB-Kabelverbindung zwischen Ihrem Computer und Ihrem iOS-Gerät. Kein Internet erforderlich.
{{% /details %}}

{{% details title="Welche Dateiformate kann ich an Evermusic oder Flacbox übertragen?" closed="true" %}}
Beide Apps unterstützen eine breite Palette von Audioformaten, darunter MP3, FLAC, AAC, WAV, AIFF, OGG, WMA und mehr. Die vollständige Liste der unterstützten Formate finden Sie in der App-Dokumentation.
{{% /details %}}

{{% details title="Warum sehe ich die Registerkarte Dateien nicht im Finder?" closed="true" %}}
Die Registerkarte Dateien wird nur angezeigt, wenn auf Ihrem Gerät mindestens eine App installiert ist, die die Dateifreigabe unterstützt. Installieren Sie Evermusic, Flacbox oder Evertag und verbinden Sie dann Ihr Gerät erneut.
{{% /details %}}

{{% details title="Kann ich Dateien drahtlos anstatt über ein USB-Kabel übertragen?" closed="true" %}}
Ja. Evermusic und Flacbox unterstützen auch Cloud-Speicherdienste und Wi-Fi-Übertragung. Die USB-Dateifreigabe über Finder oder iTunes ist jedoch in der Regel schneller für große Musikbibliotheken.
{{% /details %}}

{{% details title="Werden beim Übertragen von Dateien über Finder vorhandene Dateien auf meinem Gerät überschrieben?" closed="true" %}}
Nein. Neue Dateien werden neben den vorhandenen hinzugefügt. Wenn bereits eine Datei mit demselben Namen existiert, kann macOS die neue Datei automatisch umbenennen.
{{% /details %}}

{{% details title="Funktioniert diese Methode mit Windows-PCs?" closed="true" %}}
Ja. Unter Windows verwenden Sie iTunes zum Übertragen von Dateien. Der Vorgang ist derselbe wie im iTunes-Abschnitt oben beschrieben. Installieren Sie iTunes aus dem Microsoft Store oder von der Apple-Website.
{{% /details %}}

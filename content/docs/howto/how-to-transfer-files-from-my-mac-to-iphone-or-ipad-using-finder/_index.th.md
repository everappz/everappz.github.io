**สรุป:** เชื่อมต่อ iPhone หรือ iPad กับ Mac (หรือ PC) ด้วยสาย USB บน macOS Catalina และใหม่กว่า ใช้ Finder บน macOS เก่าหรือ Windows ใช้ iTunes ลากไฟล์ไปยังแอปเช่น Evermusic, Flacbox หรือ Evertag เพื่อถ่ายโอนทันที

การแชร์ไฟล์ iTunes เสนอวิธีที่สะดวกในการถ่ายโอนไฟล์ระหว่างคอมพิวเตอร์และแอปพลิเคชัน iOS บน iPhone, iPad หรือ iPod touch ไม่ว่าคุณจะมีคอลเลกชันอัลบั้ม ศิลปิน หรือแทร็กโปรดบน Mac หรือ PC คุณสามารถคัดลอกเนื้อหานี้ไปยังแอปเช่น Evermusic, Flacbox หรือ Evertag ได้อย่างง่ายดาย

นี่คือคู่มือทีละขั้นตอนในการใช้การแชร์ไฟล์ iTunes

## สำหรับ macOS Catalina หรือใหม่กว่า

**ข้อกำหนดเบื้องต้น**  
- macOS Catalina หรือเวอร์ชันที่ใหม่กว่า
- อุปกรณ์ iOS (iPhone, iPad หรือ iPod touch) ที่มีเวอร์ชัน iOS หรือ iPadOS ล่าสุด
- แอป iOS ที่เข้ากันได้กับการแชร์ไฟล์

### ขั้นตอน  
1. เปิดหน้าต่าง Finder บน Mac ของคุณ
2. เชื่อมต่ออุปกรณ์ iOS กับคอมพิวเตอร์โดยใช้สาย USB
3. เลือกอุปกรณ์ของคุณใน Finder
4. คลิกแท็บ "ไฟล์" เพื่อดูรายการแอปที่รองรับการแชร์ไฟล์ หากคุณไม่เห็นส่วน "ไฟล์" หมายความว่าอุปกรณ์ของคุณไม่มีแอปที่รองรับการแชร์ไฟล์
5. คลิกสามเหลี่ยมข้างแอปเพื่อดูไฟล์ที่คุณสามารถแชร์ได้

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### การคัดลอกไฟล์จาก Mac ไปยังอุปกรณ์ iOS  
1. เลือกไฟล์ที่คุณต้องการคัดลอก ตรวจสอบให้แน่ใจว่าเข้ากันได้กับแอป (ดูคู่มือผู้ใช้ของแอปสำหรับรูปแบบที่รองรับ)
2. ลากและวางไฟล์ที่เลือกลงบนไอคอนแอปบนอุปกรณ์ iOS ใน Finder Finder จะคัดลอกไฟล์ไปยังอุปกรณ์ของคุณ

### การคัดลอกไฟล์จากอุปกรณ์ iOS ไปยัง Mac  
1. เลือกไฟล์ที่คุณต้องการคัดลอกจากอุปกรณ์ iOS
2. ลากและวางไฟล์ที่เลือกไปยังโฟลเดอร์หรือหน้าต่างบนคอมพิวเตอร์ Finder จะคัดลอกไฟล์ไปยังคอมพิวเตอร์ของคุณ

## สำหรับ macOS เวอร์ชันก่อน Catalina

**ข้อกำหนดเบื้องต้น**  
- iTunes เวอร์ชันล่าสุด
- อุปกรณ์ iOS ที่มีเวอร์ชัน iOS ล่าสุด
- PC ที่ใช้ Windows 7 หรือใหม่กว่า (หากใช้ Windows)

### ขั้นตอน
1. เปิด iTunes บน Mac หรือ PC
2. เชื่อมต่อ iPhone หรือ iPad กับคอมพิวเตอร์โดยใช้สาย USB Lightning
3. เลือกอุปกรณ์ iOS ใน iTunes

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. คลิกส่วนแอปในแถบด้านข้างซ้าย
5. เลื่อนไปที่ส่วนการแชร์ไฟล์ที่ด้านล่างของหน้า
6. เลือกแอปเพื่อดูไฟล์ที่สามารถแชร์บนอุปกรณ์ได้

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### การคัดลอกไฟล์จากคอมพิวเตอร์ไปยังแอป  
1. ในแถบด้านข้างซ้ายของ iTunes คลิกส่วน "แอป"
2. เลื่อนลงไปที่ส่วน "การแชร์ไฟล์" ที่ด้านล่าง
3. เลือกแอปเพื่อดูไฟล์ที่มีสำหรับแชร์บนอุปกรณ์
4. หากต้องการคัดลอกไฟล์ไปยังอุปกรณ์ ลากและวางจากโฟลเดอร์หรือหน้าต่างไปยังรายการ "เอกสาร" ใน iTunes
5. หรือคลิก "เพิ่ม" ในรายการ "เอกสาร" ใน iTunes ค้นหาไฟล์เสียงบนคอมพิวเตอร์ แล้วคลิก "เพิ่ม" iTunes จะคัดลอกไฟล์เพลงเหล่านี้ไปยังไดเรกทอรี "เอกสาร" ของแอปบนอุปกรณ์

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### การคัดลอกไฟล์จากแอป iOS ไปยังคอมพิวเตอร์  
1. ใน iTunes เลือกแอปจากรายการในส่วน "การแชร์ไฟล์"
2. หากต้องการคัดลอกไฟล์ไปยังคอมพิวเตอร์ ลากและวางจากรายการ "เอกสาร" ไปยังโฟลเดอร์หรือหน้าต่างบนคอมพิวเตอร์
3. คุณยังสามารถเลือกไฟล์เสียงที่ต้องการคัดลอกไปยังคอมพิวเตอร์จากรายการ "เอกสาร" แล้วคลิก "บันทึกไปที่" ค้นหาโฟลเดอร์บนคอมพิวเตอร์ที่คุณต้องการคัดลอกไฟล์ แล้วคลิก "บันทึกไปที่"

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## การเปิดไฟล์ที่แชร์  
หากต้องการเล่นไฟล์ที่คัดลอกจากคอมพิวเตอร์ไปยังอุปกรณ์ iOS ให้เปิดแอปที่กำหนดสำหรับการแชร์ไฟล์ (เช่น Evermusic หรือ Flacbox) แล้วแตะที่แท็บ "ไฟล์ในเครื่อง" และเลือกโฟลเดอร์ "เอกสาร" คุณจะเห็นรายการไฟล์ที่สามารถเล่นได้

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_549f899dd8dc41b29efa7f548c826d15~mv2.png" alt="Evermusic Files Section" width="400" >}}

## การลบไฟล์ที่แชร์จากอุปกรณ์ iOS  

### ด้วย iTunes  
1. เปิด iTunes บน Mac/PC
2. เชื่อมต่ออุปกรณ์ iOS กับคอมพิวเตอร์ผ่าน USB
3. เลือกอุปกรณ์ใน iTunes
4. คลิก "แอป" ในแถบด้านข้างซ้ายและเลื่อนไปที่ส่วน "การแชร์ไฟล์"
5. เลือกแอปที่มีไฟล์ที่คุณต้องการลบ
6. เลือกไฟล์ในรายการ "เอกสาร" แล้วกดปุ่ม Delete บนแป้นพิมพ์
7. ยืนยันการลบใน iTunes

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### ด้วยตัวจัดการไฟล์ในตัว  
1. เปิดส่วน "ไฟล์ในเครื่อง"
2. แตะปุ่ม "ดำเนินการเพิ่มเติม" ที่มุมขวาบนและเลือก "เลือก" จากเมนู
3. เลือกไฟล์ที่คุณต้องการลบ
4. แตะการดำเนินการ "ลบ" (หากไม่เห็น ให้แตะเมนู "ดำเนินการเพิ่มเติม")
5. ยืนยันการลบ

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

ด้วยการแชร์ไฟล์ iTunes คุณสามารถจัดการไฟล์ระหว่างคอมพิวเตอร์และแอป iOS ได้อย่างง่ายดาย เพิ่มประสบการณ์การใช้งานเพลงและการแชร์ไฟล์ของคุณ

## คำถามที่พบบ่อย

{{% details title="ฉันต้องการการเชื่อมต่ออินเทอร์เน็ตเพื่อถ่ายโอนไฟล์ผ่าน USB หรือไม่?" closed="true" %}}
ไม่ การแชร์ไฟล์ทำงานผ่านการเชื่อมต่อสาย USB ระหว่างคอมพิวเตอร์และอุปกรณ์ iOS ทั้งหมด ไม่จำเป็นต้องใช้อินเทอร์เน็ต
{{% /details %}}

{{% details title="ฉันสามารถถ่ายโอนไฟล์รูปแบบใดไปยัง Evermusic หรือ Flacbox ได้บ้าง?" closed="true" %}}
ทั้งสองแอปรองรับรูปแบบเสียงหลากหลาย รวมถึง MP3, FLAC, AAC, WAV, AIFF, OGG, WMA และอื่นๆ ตรวจสอบเอกสารของแอปสำหรับรายการรูปแบบที่รองรับทั้งหมด
{{% /details %}}

{{% details title="ทำไมฉันไม่เห็นแท็บไฟล์ใน Finder?" closed="true" %}}
แท็บไฟล์จะปรากฏเมื่ออุปกรณ์มีแอปอย่างน้อยหนึ่งแอปที่รองรับการแชร์ไฟล์ ติดตั้ง Evermusic, Flacbox หรือ Evertag แล้วเชื่อมต่ออุปกรณ์อีกครั้ง
{{% /details %}}

{{% details title="ฉันสามารถถ่ายโอนไฟล์แบบไร้สายแทนการใช้สาย USB ได้หรือไม่?" closed="true" %}}
ได้ Evermusic และ Flacbox ยังรองรับบริการจัดเก็บข้อมูลบนคลาวด์และการถ่ายโอน Wi-Fi อย่างไรก็ตาม การแชร์ไฟล์ USB ผ่าน Finder หรือ iTunes มักจะเร็วกว่าสำหรับคลังเพลงขนาดใหญ่
{{% /details %}}

{{% details title="การถ่ายโอนไฟล์ผ่าน Finder จะเขียนทับไฟล์ที่มีอยู่ในอุปกรณ์หรือไม่?" closed="true" %}}
ไม่ ไฟล์ใหม่จะถูกเพิ่มควบคู่กับไฟล์ที่มีอยู่ หากมีไฟล์ชื่อเดียวกันอยู่แล้ว macOS อาจเปลี่ยนชื่อไฟล์ใหม่โดยอัตโนมัติ
{{% /details %}}

{{% details title="วิธีนี้ใช้ได้กับ PC ที่ใช้ Windows หรือไม่?" closed="true" %}}
ได้ บน Windows ใช้ iTunes เพื่อถ่ายโอนไฟล์ กระบวนการเหมือนกับที่อธิบายในส่วน iTunes ด้านบน ติดตั้ง iTunes จาก Microsoft Store หรือเว็บไซต์ของ Apple
{{% /details %}}

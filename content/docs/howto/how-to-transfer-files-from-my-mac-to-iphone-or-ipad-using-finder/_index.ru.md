**Кратко:** Подключите iPhone или iPad к Mac (или ПК) с помощью USB-кабеля. На macOS Catalina и более поздних версиях используйте Finder. На более старых версиях macOS или Windows используйте iTunes. Перетащите файлы в приложение, такое как Evermusic, Flacbox или Evertag, чтобы мгновенно перенести их.

Общий доступ к файлам iTunes предлагает удобный способ переноса файлов между компьютером и приложениями iOS на iPhone, iPad или iPod touch. Если у вас есть коллекция любимых альбомов, исполнителей или треков на Mac или ПК, вы можете легко скопировать этот контент в приложения, такие как Evermusic, Flacbox или Evertag.

Вот пошаговое руководство по использованию общего доступа к файлам iTunes.

## Для macOS Catalina или более поздней версии

**Предварительные требования**  
- macOS Catalina или более поздняя версия
- Устройство iOS (iPhone, iPad или iPod touch) с последней версией iOS или iPadOS
- Приложение iOS, совместимое с общим доступом к файлам

### Шаги  
1. Откройте окно Finder на Mac.
2. Подключите устройство iOS к компьютеру с помощью USB-кабеля.
3. Выберите своё устройство в Finder.
4. Нажмите вкладку «Файлы», чтобы просмотреть список приложений, поддерживающих общий доступ к файлам. Если вы не видите раздел «Файлы», это означает, что на вашем устройстве нет приложений, поддерживающих общий доступ к файлам.
5. Нажмите на треугольник рядом с приложением, чтобы увидеть файлы, которые можно передать.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Копирование файлов с Mac на устройство iOS  
1. Выберите файлы, которые хотите скопировать, убедившись, что они совместимы с приложением (поддерживаемые форматы см. в руководстве пользователя приложения).
2. Перетащите выбранные файлы на значок приложения на устройстве iOS в Finder. Finder скопирует файлы на ваше устройство.

### Копирование файлов с устройства iOS на Mac  
1. Выберите файлы, которые хотите скопировать с устройства iOS.
2. Перетащите выбранные файлы в папку или окно на компьютере. Finder скопирует файлы на ваш компьютер.

## Для версий macOS до Catalina

**Предварительные требования**  
- Последняя версия iTunes
- Устройство iOS с последней версией iOS
- ПК с Windows 7 или более поздней версией (при использовании Windows)

### Шаги
1. Откройте iTunes на Mac или ПК.
2. Подключите iPhone или iPad к компьютеру с помощью USB-кабеля Lightning.
3. Выберите устройство iOS в iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Нажмите раздел «Программы» на левой боковой панели.
5. Прокрутите до раздела «Общий доступ к файлам» в нижней части страницы.
6. Выберите приложение, чтобы увидеть, какие файлы доступны для передачи на устройстве.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Копирование файлов с компьютера в приложение  
1. На левой боковой панели iTunes нажмите раздел «Программы».
2. Прокрутите вниз до раздела «Общий доступ к файлам» внизу.
3. Выберите приложение, чтобы увидеть доступные файлы для передачи на устройстве.
4. Чтобы скопировать файлы на устройство, перетащите их из папки или окна в список «Документы» в iTunes.
5. Или нажмите «Добавить» в списке «Документы» в iTunes, найдите аудиофайлы на компьютере и нажмите «Добавить». iTunes скопирует эти музыкальные файлы в каталог «Документы» приложения на устройстве.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Копирование файлов из приложения iOS на компьютер  
1. В iTunes выберите приложение из списка в разделе «Общий доступ к файлам».
2. Чтобы скопировать файлы на компьютер, перетащите их из списка «Документы» в папку или окно на компьютере.
3. Вы также можете выбрать аудиофайлы, которые хотите скопировать на компьютер, из списка «Документы» и нажать «Сохранить в». Найдите папку на компьютере, в которую хотите скопировать файлы, и нажмите «Сохранить в».

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Открытие общих файлов  
Чтобы воспроизвести файлы, скопированные с компьютера на устройство iOS, откройте приложение, предназначенное для общего доступа к файлам (например, Evermusic или Flacbox), нажмите на вкладку «Локальные файлы» и выберите папку «Документы». Вы увидите список доступных файлов, которые можно воспроизвести.

## Удаление общих файлов с устройства iOS  

### С помощью iTunes  
1. Откройте iTunes на Mac/ПК.
2. Подключите устройство iOS к компьютеру через USB.
3. Выберите устройство в iTunes.
4. Нажмите «Программы» на левой боковой панели и прокрутите до раздела «Общий доступ к файлам».
5. Выберите приложение, содержащее файл, который хотите удалить.
6. Выберите файл в списке «Документы» и нажмите клавишу Delete на клавиатуре.
7. Подтвердите удаление в iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### С помощью встроенного файлового менеджера  
1. Откройте раздел «Локальные файлы».
2. Нажмите кнопку «Другие действия» в правом верхнем углу и выберите «Выбрать» из меню.
3. Выберите файлы, которые хотите удалить.
4. Нажмите действие «Удалить» (если оно не отображается, нажмите меню «Другие действия»).
5. Подтвердите удаление.

С помощью общего доступа к файлам iTunes вы можете легко управлять файлами между компьютером и приложениями iOS, улучшая свой опыт работы с музыкой и обмена файлами.

## Часто задаваемые вопросы

{{% details title="Нужно ли подключение к интернету для переноса файлов через USB?" closed="true" %}}
Нет. Общий доступ к файлам работает полностью через USB-кабельное соединение между компьютером и устройством iOS. Интернет не требуется.
{{% /details %}}

{{% details title="Какие форматы файлов можно перенести в Evermusic или Flacbox?" closed="true" %}}
Оба приложения поддерживают широкий спектр аудиоформатов, включая MP3, FLAC, AAC, WAV, AIFF, OGG, WMA и другие. Полный список поддерживаемых форматов см. в документации приложения.
{{% /details %}}

{{% details title="Почему я не вижу вкладку «Файлы» в Finder?" closed="true" %}}
Вкладка «Файлы» появляется только тогда, когда на устройстве установлено хотя бы одно приложение, поддерживающее общий доступ к файлам. Установите Evermusic, Flacbox или Evertag, затем повторно подключите устройство.
{{% /details %}}

{{% details title="Можно ли передавать файлы по беспроводной сети вместо USB-кабеля?" closed="true" %}}
Да. Evermusic и Flacbox также поддерживают облачные хранилища и передачу по Wi-Fi. Однако общий доступ к файлам через USB с помощью Finder или iTunes обычно быстрее для больших музыкальных библиотек.
{{% /details %}}

{{% details title="Перезапишет ли перенос файлов через Finder существующие файлы на моём устройстве?" closed="true" %}}
Нет. Новые файлы добавляются рядом с существующими. Если файл с таким же именем уже существует, macOS может автоматически переименовать новый файл.
{{% /details %}}

{{% details title="Работает ли этот метод с ПК на Windows?" closed="true" %}}
Да. В Windows используйте iTunes для переноса файлов. Процесс такой же, как описано в разделе iTunes выше. Установите iTunes из Microsoft Store или с сайта Apple.
{{% /details %}}

**Résumé:** Connectez votre iPhone ou iPad à votre Mac (ou PC) avec un câble USB. Sur macOS Catalina et versions ultérieures, utilisez Finder. Sur les anciennes versions de macOS ou Windows, utilisez iTunes. Glissez des fichiers dans une application comme Evermusic, Flacbox ou Evertag pour les transférer instantanément.

Le partage de fichiers iTunes offre un moyen pratique de transférer des fichiers entre votre ordinateur et les applications iOS sur votre iPhone, iPad ou iPod touch. Que vous ayez une collection d'albums, d'artistes ou de morceaux préférés sur votre Mac ou PC, vous pouvez facilement copier ce contenu vers des applications comme Evermusic, Flacbox ou Evertag.

Voici un guide étape par étape pour utiliser le partage de fichiers iTunes.

## Pour macOS Catalina ou ultérieur

**Prérequis**  
- macOS Catalina ou une version ultérieure
- Un appareil iOS (iPhone, iPad ou iPod touch) avec la dernière version d'iOS ou d'iPadOS
- Une application iOS compatible avec le partage de fichiers

### Étapes  
1. Ouvrez une fenêtre Finder sur votre Mac.
2. Connectez votre appareil iOS à votre ordinateur à l'aide d'un câble USB.
3. Sélectionnez votre appareil dans le Finder.
4. Cliquez sur l'onglet "Fichiers" pour voir la liste des applications prenant en charge le partage de fichiers. Si vous ne voyez pas de section "Fichiers", cela signifie que votre appareil n'a aucune application prenant en charge le partage de fichiers.
5. Cliquez sur le triangle à côté d'une application pour révéler les fichiers que vous pouvez partager.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Copier des fichiers de votre Mac vers votre appareil iOS  
1. Sélectionnez les fichiers que vous souhaitez copier, en vous assurant qu'ils sont compatibles avec l'application (consultez le guide de l'utilisateur de l'application pour les formats pris en charge).
2. Glissez et déposez les fichiers sélectionnés sur l'icône de l'application sur votre appareil iOS dans le Finder. Le Finder copiera les fichiers sur votre appareil.

### Copier des fichiers de votre appareil iOS vers votre Mac  
1. Sélectionnez les fichiers que vous souhaitez copier depuis votre appareil iOS.
2. Glissez et déposez les fichiers sélectionnés dans un dossier ou une fenêtre sur votre ordinateur. Le Finder copiera les fichiers sur votre ordinateur.

## Pour les versions de macOS antérieures à Catalina

**Prérequis**  
- La dernière version d'iTunes
- Un appareil iOS avec la dernière version d'iOS
- Un PC Windows avec Windows 7 ou ultérieur (si vous utilisez Windows)

### Étapes
1. Ouvrez iTunes sur votre Mac ou PC.
2. Connectez votre iPhone ou iPad à votre ordinateur à l'aide d'un câble USB Lightning.
3. Sélectionnez votre appareil iOS dans iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Cliquez sur la section Apps dans la barre latérale gauche.
5. Faites défiler jusqu'à la section Partage de fichiers en bas de la page.
6. Sélectionnez l'application pour voir quels fichiers sont disponibles pour le partage sur votre appareil.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Copier des fichiers de votre ordinateur vers l'application  
1. Dans la barre latérale gauche d'iTunes, cliquez sur la section "Apps".
2. Faites défiler jusqu'à la section "Partage de fichiers" en bas.
3. Sélectionnez l'application pour voir les fichiers disponibles pour le partage sur votre appareil.
4. Pour copier des fichiers sur votre appareil, glissez-les et déposez-les depuis un dossier ou une fenêtre dans la liste "Documents" dans iTunes.
5. Vous pouvez également cliquer sur "Ajouter" dans la liste "Documents" dans iTunes, localiser les fichiers audio sur votre ordinateur et cliquer sur "Ajouter". iTunes copiera ces fichiers musicaux dans le répertoire "Documents" de l'application sur votre appareil.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Copier des fichiers d'une application iOS vers votre ordinateur  
1. Dans iTunes, sélectionnez l'application dans la liste de la section "Partage de fichiers".
2. Pour copier des fichiers sur votre ordinateur, glissez-les et déposez-les depuis la liste "Documents" dans un dossier ou une fenêtre sur votre ordinateur.
3. Vous pouvez également sélectionner les fichiers audio que vous souhaitez copier sur votre ordinateur dans la liste "Documents" et cliquer sur "Enregistrer dans". Localisez le dossier sur votre ordinateur où vous souhaitez copier les fichiers et cliquez sur "Enregistrer dans".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Ouvrir les fichiers partagés  
Pour lire les fichiers copiés de votre ordinateur vers votre appareil iOS, ouvrez l'application désignée pour le partage de fichiers (par ex., Evermusic ou Flacbox) et appuyez sur l'onglet "Fichiers locaux" et choisissez le dossier "Documents". Vous verrez une liste de fichiers disponibles que vous pouvez lire.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_549f899dd8dc41b29efa7f548c826d15~mv2.png" alt="Evermusic Files Section" width="400" >}}

## Supprimer les fichiers partagés de votre appareil iOS

### Avec iTunes  
1. Ouvrez iTunes sur votre Mac/PC.
2. Connectez votre appareil iOS à l'ordinateur via USB.
3. Sélectionnez votre appareil dans iTunes.
4. Cliquez sur "Apps" dans la barre latérale gauche et faites défiler jusqu'à la section "Partage de fichiers".
5. Choisissez l'application contenant le fichier que vous souhaitez supprimer.
6. Sélectionnez le fichier dans la liste "Documents" et appuyez sur la touche Supprimer de votre clavier.
7. Confirmez la suppression dans iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Avec le gestionnaire de fichiers intégré  
1. Ouvrez la section "Fichiers locaux".
2. Appuyez sur le bouton "Plus d'actions" dans le coin supérieur droit et choisissez "Sélectionner" dans le menu.
3. Sélectionnez les fichiers que vous souhaitez supprimer.
4. Appuyez sur l'action "Supprimer" (si elle n'est pas visible, appuyez sur le menu "Plus d'actions").
5. Confirmez la suppression.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Avec le partage de fichiers iTunes, vous pouvez facilement gérer les fichiers entre votre ordinateur et les applications iOS, améliorant votre expérience musicale et de partage de fichiers.

## Questions fréquemment posées

{{% details title="Ai-je besoin d'une connexion internet pour transférer des fichiers via USB ?" closed="true" %}}
Non. Le partage de fichiers fonctionne entièrement via la connexion par câble USB entre votre ordinateur et votre appareil iOS. Aucun internet n'est requis.
{{% /details %}}

{{% details title="Quels formats de fichiers puis-je transférer vers Evermusic ou Flacbox ?" closed="true" %}}
Les deux applications prennent en charge une large gamme de formats audio, notamment MP3, FLAC, AAC, WAV, AIFF, OGG, WMA et plus encore. Consultez la documentation de l'application pour la liste complète des formats pris en charge.
{{% /details %}}

{{% details title="Pourquoi ne vois-je pas l'onglet Fichiers dans Finder ?" closed="true" %}}
L'onglet Fichiers n'apparaît que lorsque votre appareil a au moins une application installée qui prend en charge le partage de fichiers. Installez Evermusic, Flacbox ou Evertag, puis reconnectez votre appareil.
{{% /details %}}

{{% details title="Puis-je transférer des fichiers sans fil au lieu d'utiliser un câble USB ?" closed="true" %}}
Oui. Evermusic et Flacbox prennent également en charge les services de stockage cloud et le transfert Wi-Fi. Cependant, le partage de fichiers USB via Finder ou iTunes est généralement plus rapide pour les grandes bibliothèques musicales.
{{% /details %}}

{{% details title="Le transfert de fichiers via Finder écrasera-t-il les fichiers existants sur mon appareil ?" closed="true" %}}
Non. Les nouveaux fichiers sont ajoutés aux côtés des fichiers existants. Si un fichier portant le même nom existe déjà, macOS peut renommer automatiquement le nouveau fichier.
{{% /details %}}

{{% details title="Cette méthode fonctionne-t-elle avec les PC Windows ?" closed="true" %}}
Oui. Sous Windows, utilisez iTunes pour transférer des fichiers. Le processus est le même que celui décrit dans la section iTunes ci-dessus. Installez iTunes depuis le Microsoft Store ou le site web d'Apple.
{{% /details %}}

**Περίληψη:** Συνδέστε το iPhone ή iPad σας στο Mac (ή τον υπολογιστή) με ένα καλώδιο USB. Στο macOS Catalina και νεότερο, χρησιμοποιήστε το Finder. Σε παλαιότερο macOS ή Windows, χρησιμοποιήστε το iTunes. Σύρετε αρχεία σε μια εφαρμογή όπως Evermusic, Flacbox ή Evertag για να τα μεταφέρετε αμέσως.

Η κοινή χρήση αρχείων iTunes προσφέρει έναν βολικό τρόπο μεταφοράς αρχείων μεταξύ του υπολογιστή σας και των εφαρμογών iOS στο iPhone, iPad ή iPod touch. Είτε έχετε μια συλλογή αγαπημένων άλμπουμ, καλλιτεχνών ή κομματιών στο Mac ή τον υπολογιστή σας, μπορείτε εύκολα να αντιγράψετε αυτό το περιεχόμενο σε εφαρμογές όπως Evermusic, Flacbox ή Evertag.

Ακολουθεί ένας οδηγός βήμα προς βήμα για τη χρήση της κοινής χρήσης αρχείων iTunes.

## Για macOS Catalina ή νεότερο

**Προαπαιτούμενα**  
- macOS Catalina ή νεότερη έκδοση
- Μια συσκευή iOS (iPhone, iPad ή iPod touch) με την πιο πρόσφατη έκδοση iOS ή iPadOS
- Μια εφαρμογή iOS συμβατή με την κοινή χρήση αρχείων

### Βήματα  
1. Ανοίξτε ένα παράθυρο Finder στο Mac σας.
2. Συνδέστε τη συσκευή iOS στον υπολογιστή σας χρησιμοποιώντας ένα καλώδιο USB.
3. Επιλέξτε τη συσκευή σας στο Finder.
4. Κάντε κλικ στην καρτέλα "Αρχεία" για να δείτε μια λίστα εφαρμογών που υποστηρίζουν την κοινή χρήση αρχείων. Αν δεν βλέπετε την ενότητα "Αρχεία", σημαίνει ότι η συσκευή σας δεν έχει εφαρμογές που υποστηρίζουν την κοινή χρήση αρχείων.
5. Κάντε κλικ στο τρίγωνο δίπλα σε μια εφαρμογή για να εμφανίσετε τα αρχεία που μπορείτε να μοιραστείτε.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Αντιγραφή αρχείων από το Mac στη συσκευή iOS  
1. Επιλέξτε τα αρχεία που θέλετε να αντιγράψετε, βεβαιώνοντας ότι είναι συμβατά με την εφαρμογή (ανατρέξτε στον οδηγό χρήστη της εφαρμογής για τις υποστηριζόμενες μορφές).
2. Σύρετε και αποθέστε τα επιλεγμένα αρχεία στο εικονίδιο της εφαρμογής στη συσκευή iOS στο Finder. Το Finder θα αντιγράψει τα αρχεία στη συσκευή σας.

### Αντιγραφή αρχείων από τη συσκευή iOS στο Mac  
1. Επιλέξτε τα αρχεία που θέλετε να αντιγράψετε από τη συσκευή iOS.
2. Σύρετε και αποθέστε τα επιλεγμένα αρχεία σε έναν φάκελο ή παράθυρο στον υπολογιστή σας. Το Finder θα αντιγράψει τα αρχεία στον υπολογιστή σας.

## Για εκδόσεις macOS πριν από το Catalina

**Προαπαιτούμενα**  
- Η πιο πρόσφατη έκδοση του iTunes
- Μια συσκευή iOS με την πιο πρόσφατη έκδοση iOS
- Υπολογιστής Windows με Windows 7 ή νεότερο (αν χρησιμοποιείτε Windows)

### Βήματα
1. Ανοίξτε το iTunes στο Mac ή τον υπολογιστή σας.
2. Συνδέστε το iPhone ή iPad στον υπολογιστή σας χρησιμοποιώντας ένα καλώδιο USB Lightning.
3. Επιλέξτε τη συσκευή iOS στο iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Κάντε κλικ στην ενότητα Εφαρμογές στην αριστερή πλαϊνή γραμμή.
5. Μετακινηθείτε προς τα κάτω στην ενότητα Κοινή χρήση αρχείων στο κάτω μέρος της σελίδας.
6. Επιλέξτε την εφαρμογή για να δείτε ποια αρχεία είναι διαθέσιμα για κοινή χρήση στη συσκευή σας.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Αντιγραφή αρχείων από τον υπολογιστή στην εφαρμογή  
1. Στην αριστερή πλαϊνή γραμμή του iTunes, κάντε κλικ στην ενότητα "Εφαρμογές".
2. Μετακινηθείτε προς τα κάτω στην ενότητα "Κοινή χρήση αρχείων".
3. Επιλέξτε την εφαρμογή για να δείτε τα διαθέσιμα αρχεία για κοινή χρήση στη συσκευή σας.
4. Για να αντιγράψετε αρχεία στη συσκευή σας, σύρετε και αποθέστε τα από έναν φάκελο ή παράθυρο στη λίστα "Έγγραφα" στο iTunes.
5. Εναλλακτικά, κάντε κλικ στο "Προσθήκη" στη λίστα "Έγγραφα" στο iTunes, εντοπίστε τα αρχεία ήχου στον υπολογιστή σας και κάντε κλικ στο "Προσθήκη". Το iTunes θα αντιγράψει αυτά τα αρχεία μουσικής στον κατάλογο "Έγγραφα" της εφαρμογής στη συσκευή σας.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Αντιγραφή αρχείων από εφαρμογή iOS στον υπολογιστή σας  
1. Στο iTunes, επιλέξτε την εφαρμογή από τη λίστα στην ενότητα "Κοινή χρήση αρχείων".
2. Για να αντιγράψετε αρχεία στον υπολογιστή σας, σύρετε και αποθέστε τα από τη λίστα "Έγγραφα" σε έναν φάκελο ή παράθυρο στον υπολογιστή σας.
3. Μπορείτε επίσης να επιλέξετε τα αρχεία ήχου που θέλετε να αντιγράψετε στον υπολογιστή σας από τη λίστα "Έγγραφα" και να κάνετε κλικ στο "Αποθήκευση σε". Εντοπίστε τον φάκελο στον υπολογιστή σας όπου θέλετε να αντιγράψετε τα αρχεία και κάντε κλικ στο "Αποθήκευση σε".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Άνοιγμα κοινόχρηστων αρχείων  
Για να αναπαράγετε αρχεία που αντιγράφηκαν από τον υπολογιστή στη συσκευή iOS, ανοίξτε την εφαρμογή που προορίζεται για κοινή χρήση αρχείων (π.χ. Evermusic ή Flacbox) και πατήστε στην καρτέλα "Τοπικά αρχεία" και επιλέξτε τον φάκελο "Έγγραφα". Θα δείτε μια λίστα διαθέσιμων αρχείων που μπορείτε να αναπαράγετε.

## Διαγραφή κοινόχρηστων αρχείων από τη συσκευή iOS

### Με το iTunes  
1. Ανοίξτε το iTunes στο Mac/υπολογιστή σας.
2. Συνδέστε τη συσκευή iOS στον υπολογιστή μέσω USB.
3. Επιλέξτε τη συσκευή σας στο iTunes.
4. Κάντε κλικ στο "Εφαρμογές" στην αριστερή πλαϊνή γραμμή και μετακινηθείτε στην ενότητα "Κοινή χρήση αρχείων".
5. Επιλέξτε την εφαρμογή που περιέχει το αρχείο που θέλετε να διαγράψετε.
6. Επιλέξτε το αρχείο στη λίστα "Έγγραφα" και πατήστε το πλήκτρο Delete στο πληκτρολόγιό σας.
7. Επιβεβαιώστε τη διαγραφή στο iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Με τον ενσωματωμένο διαχειριστή αρχείων  
1. Ανοίξτε την ενότητα "Τοπικά αρχεία".
2. Πατήστε το κουμπί "Περισσότερες ενέργειες" στην πάνω δεξιά γωνία και επιλέξτε "Επιλογή" από το μενού.
3. Επιλέξτε τα αρχεία που θέλετε να διαγράψετε.
4. Πατήστε την ενέργεια "Διαγραφή" (αν δεν είναι ορατή, πατήστε το μενού "Περισσότερες ενέργειες").
5. Επιβεβαιώστε τη διαγραφή.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Με την κοινή χρήση αρχείων iTunes, μπορείτε εύκολα να διαχειριστείτε αρχεία μεταξύ του υπολογιστή σας και των εφαρμογών iOS, βελτιώνοντας την εμπειρία μουσικής και κοινής χρήσης αρχείων.

## Συχνές ερωτήσεις

{{% details title="Χρειάζομαι σύνδεση στο διαδίκτυο για να μεταφέρω αρχεία μέσω USB;" closed="true" %}}
Όχι. Η κοινή χρήση αρχείων λειτουργεί εξ ολοκλήρου μέσω της σύνδεσης καλωδίου USB μεταξύ του υπολογιστή σας και της συσκευής iOS. Δεν απαιτείται διαδίκτυο.
{{% /details %}}

{{% details title="Ποιες μορφές αρχείων μπορώ να μεταφέρω στο Evermusic ή το Flacbox;" closed="true" %}}
Και οι δύο εφαρμογές υποστηρίζουν μια ευρεία γκάμα μορφών ήχου, συμπεριλαμβανομένων MP3, FLAC, AAC, WAV, AIFF, OGG, WMA και άλλων. Ελέγξτε την τεκμηρίωση της εφαρμογής για την πλήρη λίστα υποστηριζόμενων μορφών.
{{% /details %}}

{{% details title="Γιατί δεν βλέπω την καρτέλα Αρχεία στο Finder;" closed="true" %}}
Η καρτέλα Αρχεία εμφανίζεται μόνο όταν η συσκευή σας έχει τουλάχιστον μία εγκατεστημένη εφαρμογή που υποστηρίζει την κοινή χρήση αρχείων. Εγκαταστήστε το Evermusic, Flacbox ή Evertag και μετά επανασυνδέστε τη συσκευή σας.
{{% /details %}}

{{% details title="Μπορώ να μεταφέρω αρχεία ασύρματα αντί να χρησιμοποιήσω καλώδιο USB;" closed="true" %}}
Ναι. Το Evermusic και το Flacbox υποστηρίζουν επίσης υπηρεσίες αποθήκευσης cloud και μεταφορά Wi-Fi. Ωστόσο, η κοινή χρήση αρχείων USB μέσω Finder ή iTunes είναι συνήθως ταχύτερη για μεγάλες μουσικές βιβλιοθήκες.
{{% /details %}}

{{% details title="Η μεταφορά αρχείων μέσω Finder θα αντικαταστήσει τα υπάρχοντα αρχεία στη συσκευή μου;" closed="true" %}}
Όχι. Τα νέα αρχεία προστίθενται δίπλα στα υπάρχοντα. Αν υπάρχει ήδη ένα αρχείο με το ίδιο όνομα, το macOS μπορεί να μετονομάσει αυτόματα το νέο αρχείο.
{{% /details %}}

{{% details title="Αυτή η μέθοδος λειτουργεί με υπολογιστές Windows;" closed="true" %}}
Ναι. Στα Windows, χρησιμοποιήστε το iTunes για να μεταφέρετε αρχεία. Η διαδικασία είναι η ίδια με αυτήν που περιγράφεται στην ενότητα iTunes παραπάνω. Εγκαταστήστε το iTunes από το Microsoft Store ή τον ιστότοπο της Apple.
{{% /details %}}

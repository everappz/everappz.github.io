**要約:** iPhoneまたはiPadをUSBケーブルでMac（またはPC）に接続します。macOS Catalina以降ではFinderを使用します。古いmacOSまたはWindowsではiTunesを使用します。Evermusic、Flacbox、Evertagなどのアプリにファイルをドラッグして、即座に転送します。

iTunesファイル共有は、コンピュータとiPhone、iPad、iPod touchのiOSアプリケーション間でファイルを転送する便利な方法を提供します。MacまたはPCにお気に入りのアルバム、アーティスト、トラックのコレクションがある場合、このコンテンツをEvermusic、Flacbox、Evertagなどのアプリに簡単にコピーできます。

以下はiTunesファイル共有を使用するためのステップバイステップガイドです。

## macOS Catalina以降の場合

**前提条件**  
- macOS Catalinaまたはそれ以降のバージョン
- 最新のiOSまたはiPadOSバージョンを搭載したiOSデバイス（iPhone、iPad、またはiPod touch）
- ファイル共有に対応したiOSアプリ

### 手順  
1. MacでFinderウィンドウを開きます。
2. USBケーブルを使用してiOSデバイスをコンピュータに接続します。
3. Finderでデバイスを選択します。
4. 「ファイル」タブをクリックして、ファイル共有をサポートするアプリのリストを表示します。「ファイル」セクションが表示されない場合は、デバイスにファイル共有をサポートするアプリがないことを意味します。
5. アプリの横にある三角形をクリックして、共有できるファイルを表示します。

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### MacからiOSデバイスにファイルをコピー  
1. コピーするファイルを選択し、アプリと互換性があることを確認します（サポートされている形式についてはアプリのユーザーガイドを参照してください）。
2. 選択したファイルをFinderのiOSデバイスのアプリアイコンにドラッグ＆ドロップします。Finderがファイルをデバイスにコピーします。

### iOSデバイスからMacにファイルをコピー  
1. iOSデバイスからコピーするファイルを選択します。
2. 選択したファイルをコンピュータのフォルダまたはウィンドウにドラッグ＆ドロップします。Finderがファイルをコンピュータにコピーします。

## Catalina以前のmacOSバージョンの場合

**前提条件**  
- 最新バージョンのiTunes
- 最新のiOSバージョンを搭載したiOSデバイス
- Windows 7以降のWindows PC（Windowsを使用する場合）

### 手順
1. MacまたはPCでiTunesを開きます。
2. USB Lightningケーブルを使用してiPhoneまたはiPadをコンピュータに接続します。
3. iTunesでiOSデバイスを選択します。

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. 左サイドバーのアプリセクションをクリックします。
5. ページ下部のファイル共有セクションまでスクロールします。
6. アプリを選択して、デバイスで共有可能なファイルを確認します。

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### コンピュータからアプリにファイルをコピー  
1. iTunesの左サイドバーで「アプリ」セクションをクリックします。
2. 下部の「ファイル共有」セクションまでスクロールします。
3. アプリを選択して、デバイスで共有可能なファイルを確認します。
4. デバイスにファイルをコピーするには、フォルダまたはウィンドウからiTunesの「ドキュメント」リストにドラッグ＆ドロップします。
5. または、iTunesの「ドキュメント」リストで「追加」をクリックし、コンピュータでオーディオファイルを見つけて「追加」をクリックします。iTunesがこれらの音楽ファイルをデバイス上のアプリの「ドキュメント」ディレクトリにコピーします。

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### iOSアプリからコンピュータにファイルをコピー  
1. iTunesで、「ファイル共有」セクションのリストからアプリを選択します。
2. コンピュータにファイルをコピーするには、「ドキュメント」リストからコンピュータのフォルダまたはウィンドウにドラッグ＆ドロップします。
3. 「ドキュメント」リストからコンピュータにコピーしたいオーディオファイルを選択し、「保存先」をクリックすることもできます。ファイルをコピーしたいコンピュータのフォルダを見つけて「保存先」をクリックします。

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## 共有ファイルを開く  
コンピュータからiOSデバイスにコピーしたファイルを再生するには、ファイル共有用のアプリ（例：EvermusicまたはFlacbox）を開き、「ローカルファイル」タブをタップして「ドキュメント」フォルダを選択します。再生可能なファイルのリストが表示されます。

## iOSデバイスから共有ファイルを削除

### iTunesを使用  
1. Mac/PCでiTunesを開きます。
2. USB経由でiOSデバイスをコンピュータに接続します。
3. iTunesでデバイスを選択します。
4. 左サイドバーの「アプリ」をクリックし、「ファイル共有」セクションまでスクロールします。
5. 削除したいファイルを含むアプリを選択します。
6. 「ドキュメント」リストでファイルを選択し、キーボードのDeleteキーを押します。
7. iTunesで削除を確認します。

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### 内蔵ファイルマネージャーを使用  
1. 「ローカルファイル」セクションを開きます。
2. 右上隅の「その他のアクション」ボタンをタップし、メニューから「選択する」を選びます。
3. 削除したいファイルを選択します。
4. 「削除する」アクションをタップします（表示されていない場合は、「その他のアクション」メニューをタップします）。
5. 削除を確認します。

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

iTunesファイル共有を使用すると、コンピュータとiOSアプリ間でファイルを簡単に管理でき、音楽やファイル共有の体験を向上させることができます。

## よくある質問

{{% details title="USBでファイルを転送するのにインターネット接続は必要ですか？" closed="true" %}}
いいえ。ファイル共有は、コンピュータとiOSデバイス間のUSBケーブル接続のみで動作します。インターネットは不要です。
{{% /details %}}

{{% details title="EvermusicまたはFlacboxにどのファイル形式を転送できますか？" closed="true" %}}
両方のアプリは、MP3、FLAC、AAC、WAV、AIFF、OGG、WMAなど、幅広いオーディオ形式をサポートしています。サポートされている形式の完全なリストについては、アプリのドキュメントを確認してください。
{{% /details %}}

{{% details title="Finderにファイルタブが表示されないのはなぜですか？" closed="true" %}}
ファイルタブは、デバイスにファイル共有をサポートするアプリが少なくとも1つインストールされている場合にのみ表示されます。Evermusic、Flacbox、またはEvertagをインストールしてから、デバイスを再接続してください。
{{% /details %}}

{{% details title="USBケーブルの代わりにワイヤレスでファイルを転送できますか？" closed="true" %}}
はい。EvermusicとFlacboxは、クラウドストレージサービスやWi-Fi転送もサポートしています。ただし、FinderまたはiTunesを使用したUSBファイル共有は、大規模な音楽ライブラリの場合、通常より高速です。
{{% /details %}}

{{% details title="Finderでファイルを転送すると、デバイス上の既存のファイルが上書きされますか？" closed="true" %}}
いいえ。新しいファイルは既存のファイルの横に追加されます。同じ名前のファイルがすでに存在する場合、macOSが新しいファイルの名前を自動的に変更する場合があります。
{{% /details %}}

{{% details title="この方法はWindows PCでも使えますか？" closed="true" %}}
はい。Windowsでは、iTunesを使用してファイルを転送します。プロセスは上記のiTunesセクションで説明したものと同じです。Microsoft StoreまたはAppleのウェブサイトからiTunesをインストールしてください。
{{% /details %}}

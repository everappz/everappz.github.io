**Resumo:** Conecte seu iPhone ou iPad ao Mac (ou PC) com um cabo USB. No macOS Catalina e posterior, use o Finder. No macOS mais antigo ou Windows, use o iTunes. Arraste arquivos para um aplicativo como Evermusic, Flacbox ou Evertag para transferi-los instantaneamente.

O Compartilhamento de Arquivos do iTunes oferece uma maneira conveniente de transferir arquivos entre seu computador e aplicativos iOS no iPhone, iPad ou iPod touch. Seja uma coleção de álbuns, artistas ou faixas favoritas no seu Mac ou PC, você pode copiar facilmente esse conteúdo para aplicativos como Evermusic, Flacbox ou Evertag.

Aqui está um guia passo a passo para usar o Compartilhamento de Arquivos do iTunes.

## Para macOS Catalina ou posterior

**Pré-requisitos**  
- macOS Catalina ou uma versão posterior
- Um dispositivo iOS (iPhone, iPad ou iPod touch) com a versão mais recente do iOS ou iPadOS
- Um aplicativo iOS compatível com Compartilhamento de Arquivos

### Passos  
1. Abra uma janela do Finder no seu Mac.
2. Conecte seu dispositivo iOS ao computador usando um cabo USB.
3. Selecione seu dispositivo no Finder.
4. Clique na aba "Arquivos" para ver uma lista de aplicativos que suportam compartilhamento de arquivos. Se você não vir uma seção "Arquivos", significa que seu dispositivo não tem nenhum aplicativo que suporte compartilhamento de arquivos.
5. Clique no triângulo ao lado de um aplicativo para revelar os arquivos que você pode compartilhar.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Copiando arquivos do Mac para o dispositivo iOS  
1. Selecione os arquivos que deseja copiar, garantindo que sejam compatíveis com o aplicativo (consulte o guia do usuário do aplicativo para formatos suportados).
2. Arraste e solte os arquivos selecionados no ícone do aplicativo no seu dispositivo iOS no Finder. O Finder copiará os arquivos para o seu dispositivo.

### Copiando arquivos do dispositivo iOS para o Mac  
1. Selecione os arquivos que deseja copiar do seu dispositivo iOS.
2. Arraste e solte os arquivos selecionados em uma pasta ou janela no seu computador. O Finder copiará os arquivos para o seu computador.

## Para versões do macOS anteriores ao Catalina

**Pré-requisitos**  
- A versão mais recente do iTunes
- Um dispositivo iOS com a versão mais recente do iOS
- Um PC Windows com Windows 7 ou posterior (se estiver usando Windows)

### Passos
1. Abra o iTunes no seu Mac ou PC.
2. Conecte seu iPhone ou iPad ao computador usando um cabo USB Lightning.
3. Selecione seu dispositivo iOS no iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Clique na seção Apps na barra lateral esquerda.
5. Role até a seção Compartilhamento de Arquivos na parte inferior da página.
6. Selecione o aplicativo para ver quais arquivos estão disponíveis para compartilhamento no seu dispositivo.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Copiando arquivos do computador para o aplicativo  
1. Na barra lateral esquerda do iTunes, clique na seção "Apps".
2. Role até a seção "Compartilhamento de Arquivos" na parte inferior.
3. Selecione o aplicativo para ver os arquivos disponíveis para compartilhamento no seu dispositivo.
4. Para copiar arquivos para seu dispositivo, arraste e solte-os de uma pasta ou janela na lista "Documentos" no iTunes.
5. Alternativamente, clique em "Adicionar" na lista "Documentos" no iTunes, localize os arquivos de áudio no seu computador e clique em "Adicionar". O iTunes copiará esses arquivos de música para o diretório "Documentos" do aplicativo no seu dispositivo.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Copiando arquivos de um aplicativo iOS para o computador  
1. No iTunes, selecione o aplicativo da lista na seção "Compartilhamento de Arquivos".
2. Para copiar arquivos para o computador, arraste e solte-os da lista "Documentos" para uma pasta ou janela no computador.
3. Você também pode selecionar os arquivos de áudio que deseja copiar para o computador na lista "Documentos" e clicar em "Salvar em". Localize a pasta no computador onde deseja copiar os arquivos e clique em "Salvar em".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Abrindo arquivos compartilhados  
Para reproduzir arquivos copiados do computador para o dispositivo iOS, abra o aplicativo designado para Compartilhamento de Arquivos (por exemplo, Evermusic ou Flacbox) e toque na aba "Arquivos locais" e escolha a pasta "Documentos". Você verá uma lista de arquivos disponíveis que pode reproduzir.

## Excluindo arquivos compartilhados do dispositivo iOS  

### Com o iTunes  
1. Abra o iTunes no seu Mac/PC.
2. Conecte seu dispositivo iOS ao computador via USB.
3. Selecione seu dispositivo no iTunes.
4. Clique em "Apps" na barra lateral esquerda e role até a seção "Compartilhamento de Arquivos".
5. Escolha o aplicativo que contém o arquivo que deseja excluir.
6. Selecione o arquivo na lista "Documentos" e pressione a tecla Delete no teclado.
7. Confirme a exclusão no iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Com o gerenciador de arquivos integrado  
1. Abra a seção "Arquivos locais".
2. Toque no botão "Mais ações" no canto superior direito e escolha "Selecionar" no menu.
3. Selecione os arquivos que deseja excluir.
4. Toque na ação "Excluir" (se não estiver visível, toque no menu "Mais ações").
5. Confirme a exclusão.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Com o Compartilhamento de Arquivos do iTunes, você pode gerenciar facilmente arquivos entre seu computador e aplicativos iOS, aprimorando sua experiência de música e compartilhamento de arquivos.

## Perguntas frequentes

{{% details title="Preciso de conexão com a internet para transferir arquivos via USB?" closed="true" %}}
Não. O Compartilhamento de Arquivos funciona inteiramente pela conexão do cabo USB entre o computador e o dispositivo iOS. Não é necessário internet.
{{% /details %}}

{{% details title="Quais formatos de arquivo posso transferir para o Evermusic ou Flacbox?" closed="true" %}}
Ambos os aplicativos suportam uma ampla variedade de formatos de áudio, incluindo MP3, FLAC, AAC, WAV, AIFF, OGG, WMA e mais. Consulte a documentação do aplicativo para a lista completa de formatos suportados.
{{% /details %}}

{{% details title="Por que não vejo a aba Arquivos no Finder?" closed="true" %}}
A aba Arquivos só aparece quando seu dispositivo tem pelo menos um aplicativo instalado que suporta Compartilhamento de Arquivos. Instale o Evermusic, Flacbox ou Evertag e reconecte seu dispositivo.
{{% /details %}}

{{% details title="Posso transferir arquivos sem fio em vez de usar um cabo USB?" closed="true" %}}
Sim. Evermusic e Flacbox também suportam serviços de armazenamento em nuvem e transferência Wi-Fi. No entanto, o compartilhamento de arquivos via USB pelo Finder ou iTunes é geralmente mais rápido para grandes bibliotecas de música.
{{% /details %}}

{{% details title="A transferência de arquivos pelo Finder substituirá arquivos existentes no meu dispositivo?" closed="true" %}}
Não. Novos arquivos são adicionados junto aos existentes. Se um arquivo com o mesmo nome já existir, o macOS pode renomear o novo arquivo automaticamente.
{{% /details %}}

{{% details title="Este método funciona com PCs Windows?" closed="true" %}}
Sim. No Windows, use o iTunes para transferir arquivos. O processo é o mesmo descrito na seção iTunes acima. Instale o iTunes na Microsoft Store ou no site da Apple.
{{% /details %}}

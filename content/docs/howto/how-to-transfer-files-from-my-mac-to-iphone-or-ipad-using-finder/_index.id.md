**Ringkasan:** Hubungkan iPhone atau iPad Anda ke Mac (atau PC) dengan kabel USB. Di macOS Catalina dan yang lebih baru, gunakan Finder. Di macOS lama atau Windows, gunakan iTunes. Seret file ke aplikasi seperti Evermusic, Flacbox, atau Evertag untuk mentransfernya secara instan.

Berbagi File iTunes menawarkan cara yang nyaman untuk mentransfer file antara komputer Anda dan aplikasi iOS di iPhone, iPad, atau iPod touch. Baik Anda memiliki koleksi album, artis, atau lagu favorit di Mac atau PC, Anda dapat dengan mudah menyalin konten ini ke aplikasi seperti Evermusic, Flacbox, atau Evertag.

Berikut panduan langkah demi langkah untuk menggunakan Berbagi File iTunes.

## Untuk macOS Catalina atau Lebih Baru

**Prasyarat**  
- macOS Catalina atau versi yang lebih baru
- Perangkat iOS (iPhone, iPad, atau iPod touch) dengan versi iOS atau iPadOS terbaru
- Aplikasi iOS yang kompatibel dengan Berbagi File

### Langkah-langkah  
1. Buka jendela Finder di Mac Anda.
2. Hubungkan perangkat iOS Anda ke komputer menggunakan kabel USB.
3. Pilih perangkat Anda di Finder.
4. Klik tab "File" untuk melihat daftar aplikasi yang mendukung berbagi file. Jika Anda tidak melihat bagian "File", berarti perangkat Anda tidak memiliki aplikasi yang mendukung berbagi file.
5. Klik segitiga di samping aplikasi untuk menampilkan file yang dapat Anda bagikan.

![MacOS Finder Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_ff2f6bb6f8674f598e0697e64594f707~mv2.png)

### Menyalin File dari Mac ke Perangkat iOS  
1. Pilih file yang ingin Anda salin, pastikan file tersebut kompatibel dengan aplikasi (lihat panduan pengguna aplikasi untuk format yang didukung).
2. Seret dan lepaskan file yang dipilih ke ikon aplikasi di perangkat iOS Anda di Finder. Finder akan menyalin file ke perangkat Anda.

### Menyalin File dari Perangkat iOS ke Mac  
1. Pilih file yang ingin Anda salin dari perangkat iOS Anda.
2. Seret dan lepaskan file yang dipilih ke folder atau jendela di komputer Anda. Finder akan menyalin file ke komputer Anda.

## Untuk Versi macOS Sebelum Catalina

**Prasyarat**  
- Versi terbaru iTunes
- Perangkat iOS dengan versi iOS terbaru
- PC Windows dengan Windows 7 atau yang lebih baru (jika menggunakan Windows)

### Langkah-langkah
1. Buka iTunes di Mac atau PC Anda.
2. Hubungkan iPhone atau iPad Anda ke komputer menggunakan kabel USB Lightning.
3. Pilih perangkat iOS Anda di iTunes.

![iTunes Window](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_deca296b9fe34c50bec8717646da7b34~mv2.png)

4. Klik bagian Aplikasi di sidebar kiri.
5. Gulir ke bawah ke bagian Berbagi File di bagian bawah halaman.
6. Pilih aplikasi untuk melihat file mana yang tersedia untuk dibagikan di perangkat Anda.

![iTunes Window Apps Section](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_5af90f153a0949e9914130b7c7c8fccc~mv2.png)

### Menyalin File dari Komputer ke Aplikasi  
1. Di sidebar kiri iTunes, klik bagian "Aplikasi".
2. Gulir ke bawah ke bagian "Berbagi File" di bagian bawah.
3. Pilih aplikasi untuk melihat file yang tersedia untuk dibagikan di perangkat Anda.
4. Untuk menyalin file ke perangkat Anda, seret dan lepaskan dari folder atau jendela ke daftar "Dokumen" di iTunes.
5. Atau, klik "Tambah" di daftar "Dokumen" di iTunes, temukan file audio di komputer Anda, dan klik "Tambah". iTunes akan menyalin file musik ini ke direktori "Dokumen" aplikasi di perangkat Anda.

![iTunes File Sharing Copy Files](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_3c8f3a4e03c8475ab183e33ff4c964ea~mv2.png)

### Menyalin File dari Aplikasi iOS ke Komputer  
1. Di iTunes, pilih aplikasi dari daftar di bagian "Berbagi File".
2. Untuk menyalin file ke komputer Anda, seret dan lepaskan dari daftar "Dokumen" ke folder atau jendela di komputer Anda.
3. Anda juga dapat memilih file audio yang ingin Anda salin ke komputer dari daftar "Dokumen" dan klik "Simpan ke". Temukan folder di komputer Anda tempat Anda ingin menyalin file dan klik "Simpan ke".

![iTunes File Sharing Copy Files To Computer](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_18c02708dcc542489c2ec68842bc925b~mv2.png)

## Membuka File Bersama  
Untuk memutar file yang disalin dari komputer ke perangkat iOS Anda, buka aplikasi yang ditunjuk untuk Berbagi File (misalnya, Evermusic atau Flacbox) dan ketuk tab "File Lokal" dan pilih folder "Dokumen". Anda akan melihat daftar file yang tersedia yang dapat Anda putar.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_549f899dd8dc41b29efa7f548c826d15~mv2.png" alt="Evermusic Files Section" width="400" >}}

## Menghapus File Bersama dari Perangkat iOS

### Dengan iTunes  
1. Buka iTunes di Mac/PC Anda.
2. Hubungkan perangkat iOS Anda ke komputer melalui USB.
3. Pilih perangkat Anda di iTunes.
4. Klik "Aplikasi" di sidebar kiri dan gulir ke bagian "Berbagi File".
5. Pilih aplikasi yang berisi file yang ingin Anda hapus.
6. Pilih file di daftar "Dokumen" dan tekan tombol Delete di keyboard Anda.
7. Konfirmasi penghapusan di iTunes.

![Deleting Shared Files from Your iOS Device with iTunes](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_1c6199eeac94435fa557b688c75c2fcd~mv2.png)

### Dengan pengelola file bawaan  
1. Buka bagian "File Lokal".
2. Ketuk tombol "Lebih banyak tindakan" di pojok kanan atas dan pilih "Pilih" dari menu.
3. Pilih file yang ingin Anda hapus.
4. Ketuk tindakan "Hapus" (jika tidak terlihat, ketuk menu "Lebih banyak tindakan").
5. Konfirmasi penghapusan.

{{< figure src="/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder/21260c_535a8879faf84b8bacf283cbde15e480~mv2.png" alt="Deleting Shared Files from Your iOS Device with Built-in File Manager" width="400" >}}

Dengan Berbagi File iTunes, Anda dapat dengan mudah mengelola file antara komputer dan aplikasi iOS, meningkatkan pengalaman musik dan berbagi file Anda.

## Pertanyaan yang Sering Diajukan

{{% details title="Apakah saya memerlukan koneksi internet untuk mentransfer file melalui USB?" closed="true" %}}
Tidak. Berbagi File bekerja sepenuhnya melalui koneksi kabel USB antara komputer dan perangkat iOS Anda. Tidak diperlukan internet.
{{% /details %}}

{{% details title="Format file apa yang dapat saya transfer ke Evermusic atau Flacbox?" closed="true" %}}
Kedua aplikasi mendukung berbagai format audio termasuk MP3, FLAC, AAC, WAV, AIFF, OGG, WMA, dan lainnya. Periksa dokumentasi aplikasi untuk daftar lengkap format yang didukung.
{{% /details %}}

{{% details title="Mengapa saya tidak melihat tab File di Finder?" closed="true" %}}
Tab File hanya muncul ketika perangkat Anda memiliki setidaknya satu aplikasi yang terinstal yang mendukung Berbagi File. Instal Evermusic, Flacbox, atau Evertag, lalu hubungkan kembali perangkat Anda.
{{% /details %}}

{{% details title="Bisakah saya mentransfer file secara nirkabel alih-alih menggunakan kabel USB?" closed="true" %}}
Ya. Evermusic dan Flacbox juga mendukung layanan penyimpanan cloud dan transfer Wi-Fi. Namun, berbagi file USB melalui Finder atau iTunes biasanya lebih cepat untuk perpustakaan musik yang besar.
{{% /details %}}

{{% details title="Apakah mentransfer file melalui Finder akan menimpa file yang ada di perangkat saya?" closed="true" %}}
Tidak. File baru ditambahkan bersama file yang sudah ada. Jika file dengan nama yang sama sudah ada, macOS mungkin secara otomatis mengganti nama file baru.
{{% /details %}}

{{% details title="Apakah metode ini berfungsi dengan PC Windows?" closed="true" %}}
Ya. Di Windows, gunakan iTunes untuk mentransfer file. Prosesnya sama seperti yang dijelaskan di bagian iTunes di atas. Instal iTunes dari Microsoft Store atau situs web Apple.
{{% /details %}}

iOS 11 dari Apple membawa dukungan untuk format audio FLAC ke aplikasi Files, tetapi file FLAC bisa cukup besar. Bayangkan memiliki fleksibilitas untuk memutar perpustakaan musik FLAC Anda sendiri dari cloud dan mengunduhnya kapan pun Anda mau. Di sinilah Flacbox hadir, menyediakan solusi komprehensif untuk impian pecinta musik ini.

Flacbox adalah pemutar musik kaya fitur yang mendukung berbagai format audio populer, termasuk FLAC, OGG, OPUS, MP3, dan WAV.

## Pemutar cloud

Dengan Flacbox, Anda dapat dengan mudah menghubungkan akun cloud favorit Anda dari layanan populer seperti Dropbox, Google Drive, OneDrive, Box, dan MEGA. Cukup buka folder yang berisi file audio Anda dan ketuk file mana saja untuk mulai memutar.

## Pengunduh audio

Ketika Anda ingin menikmati file audio lossless secara offline, unduh dan dengarkan musik FLAC Anda saat bepergian – di mobil, di pesawat, atau saat berjalan-jalan. Jika Anda membutuhkan lebih banyak ruang di perangkat, hapus cache audio lokal dan unduh lagi kapan pun Anda mau.

## Akses NAS dan komputer

Bagi mereka yang memiliki perpustakaan musik yang luas di NAS, Wi-Fi Disk, atau komputer rumah, hubungkan menggunakan protokol SMB. Sekarang, semua trek Anda tersedia di iPhone atau iPad Anda. Anda juga dapat mentransfer file menggunakan iTunes File Sharing atau fitur Wi-Fi Drive.

## Manajemen file dan daftar putar

Flacbox berfungsi sebagai pengelola file yang kuat dan pengunduh audio, memungkinkan Anda mengedit, memindahkan, dan menghapus file untuk menjaga perpustakaan musik Anda tetap terbaru. Buat daftar putar, ubah urutan lagu, dan buat daftar putar tersedia secara offline.

## Sinkronisasi pintar

Flacbox memiliki fungsi sinkronisasi pintar yang memindai semua akun cloud yang terhubung untuk file audio dan mengimpornya ke perpustakaan cloud Anda. Perpustakaan musik cloud Anda adalah database lokal yang ringkas dengan tautan ke file di penyimpanan cloud Anda. Ringan dan sangat cepat, mampu menambahkan lebih dari 20.000 trek ke perpustakaan musik Anda. Cukup pilih folder musik di pengaturan sinkronisasi dan tunggu aplikasi selesai memindai.

## Album, artis, genre, pencarian pintar

Setelah trek Anda ditambahkan ke perpustakaan musik, manfaatkan pembaca metadata yang memindai trek yang ditambahkan untuk metadata yang tersedia dan mengelompokkan semua lagu berdasarkan artis, album, dan genre. Cari album, artis, dan trek favorit Anda dengan mudah hanya dengan satu ketukan.

## Editor tag ID3

Untuk trek tanpa metadata atau ketika Anda perlu mengedit tag ID3, gunakan Editor Metadata. Fitur ini menyederhanakan pembersihan dan pengorganisasian perpustakaan musik Anda, menawarkan fitur koreksi otomatis dan pengisian otomatis untuk pengelolaan yang mudah.

## Equalizer 10-band, penguat bass, preamplifier

Flacbox menyertakan equalizer yang tangguh dengan penguat bass, preset gaya iTunes, dan preamplifier. Sesuaikan sistem suara Anda untuk mencapai lebih banyak desibel atau konfigurasikan untuk tingkat kualitas suara yang baru.

## Streaming nirkabel

Flacbox menyertakan streaming nirkabel, menjadikannya fitur wajib bagi pengguna Apple TV atau Google Chromecast, dan tersedia di semua aplikasi audio kami.

Yang terbaik, Flacbox sepenuhnya gratis dan kompatibel dengan iPhone dan iPad yang menjalankan iOS 8.0 dan yang lebih baru.

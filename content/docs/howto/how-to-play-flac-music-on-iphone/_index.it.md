iOS 11 di Apple ha introdotto il supporto per il formato audio FLAC nell'app File, ma i file FLAC possono essere piuttosto grandi. Immagina di avere la flessibilità di riprodurre la tua libreria musicale FLAC dal cloud e scaricarla quando vuoi. È qui che entra in gioco Flacbox, fornendo una soluzione completa per questo sogno degli amanti della musica.

Flacbox è un lettore musicale ricco di funzionalità che supporta un'ampia gamma di formati audio popolari, tra cui FLAC, OGG, OPUS, MP3 e WAV.

## Lettore cloud

Con Flacbox, puoi collegare facilmente i tuoi account cloud preferiti da servizi popolari come Dropbox, Google Drive, OneDrive, Box e MEGA. Basta aprire la cartella contenente i tuoi file audio e toccare qualsiasi file per iniziare la riproduzione.

## Downloader audio

Quando vuoi goderti i tuoi file audio lossless offline, scaricali e ascolta la tua musica FLAC in movimento – in auto, in aereo o durante una passeggiata. Se hai bisogno di più spazio sul tuo dispositivo, elimina la cache audio locale e scaricala di nuovo quando vuoi.

## Accesso NAS e computer

Per chi possiede librerie musicali estese su NAS, Wi-Fi Disk o computer di casa, collegati ad essi usando il protocollo SMB. Ora tutte le tue tracce sono disponibili sul tuo iPhone o iPad. Puoi anche trasferire file usando la condivisione file di iTunes o la funzione Wi-Fi Drive.

## Gestione file e playlist

Flacbox funziona come un potente gestore di file e downloader audio, permettendoti di modificare, spostare ed eliminare file per mantenere aggiornata la tua libreria musicale. Crea playlist, cambia l'ordine delle canzoni e rendi le playlist disponibili offline.

## Sincronizzazione intelligente

Flacbox dispone di una funzione di sincronizzazione intelligente che scansiona tutti gli account cloud collegati alla ricerca di file audio e li importa nella tua libreria cloud. La tua libreria musicale cloud è un database locale compatto con link ai file sul tuo cloud storage. È leggera e incredibilmente veloce, capace di aggiungere oltre 20.000 tracce alla tua libreria musicale. Seleziona semplicemente le cartelle musicali nelle impostazioni di sincronizzazione e attendi che l'app completi la scansione.

## Album, artisti, generi, ricerca intelligente

Una volta che le tue tracce sono state aggiunte alla libreria musicale, approfitta del lettore di metadati che scansiona le tracce aggiunte per i metadati disponibili e raggruppa tutte le canzoni per artista, album e genere. Cerca facilmente i tuoi album, artisti e tracce preferiti con un solo tocco.

## Editor tag ID3

Per le tracce senza metadati o quando devi modificare i tag ID3, usa l'Editor Metadati. Questa funzione semplifica la pulizia e l'organizzazione della tua libreria musicale, offrendo funzioni di correzione automatica e completamento automatico per una gestione senza sforzo.

## Equalizzatore a 10 bande, bass booster, preamplificatore

Flacbox include un robusto equalizzatore con bass booster, preset in stile iTunes e preamplificatore. Personalizza il tuo sistema audio per ottenere più decibel o configuralo per un nuovo livello di qualità del suono.

## Streaming wireless

Flacbox include lo streaming wireless, rendendolo una funzione indispensabile per gli utenti di Apple TV o Google Chromecast, ed è disponibile in tutte le nostre app audio.

Soprattutto, Flacbox è completamente gratuito e compatibile con iPhone e iPad con iOS 8.0 e versioni successive.

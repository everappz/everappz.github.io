**要約:** iCloud Driveに音楽をアップロードし、[Evermusic](/products/evermusic)（MP3/WAV用）または[Flacbox](/products/flacbox)（FLAC/DSD用）をインストールし、iCloud Driveフォルダを接続して、デバイスのストレージを使用せずに直接ストリーミングできます。

今日の世界では、音楽は私たちの生活の大きな部分を占めており、お気に入りの曲をいつでも利用できることは不可欠です。クラウド技術のおかげで、物理的な音楽コレクションを維持する必要はもうありません。iCloud Driveからわずか数回のタップで音楽をストリーミングできます。

iPhoneはクラウドに保存された曲のライブラリ全体にアクセスできます。丁寧に整理されたプレイリストでも、さまざまな曲のミックスでも、このガイドではiCloud Driveを使用してiPhoneをポータブル音楽プレーヤーに変える方法を紹介します。

iCloud Driveから音楽をストリーミングする方法を、簡単な手順とスムーズなプロセスのためのヒントとともに説明します。iCloud Driveに精通している方も初めての方も、このガイドの最後には、iPhoneでクラウドからお気に入りの曲を再生する方法がわかるでしょう。

iPhoneから音楽ストリーミングの便利さを楽しむ準備ができたら、始めましょう！

## はじめに：必要なアプリのインストール

iPhoneやMacでiCloud Driveの音楽を楽しむ前に、適切なアプリをインストールする必要があります。手順は以下の通りです：

- iOS向け：

1. App Storeにアクセスし、音楽がmp3やwavなどの標準的なオーディオ形式で保存されている場合は**Evermusic**をダウンロードします。dsdやflacのロスレス音楽がある場合は、**Flacbox**を選択してください。両方のアプリはiOSとMacOSで利用可能です。

- MacOS向け：

1. MacのApp Storeにアクセスし、音楽形式の好みに応じて**Evermusic**または**Flacbox**をインストールします。

iPhoneまたはMacにアプリをインストールしたら、次に進む準備ができています。

## iCloud Driveに音楽をアップロードする

MacまたはPCからiCloud Driveに音楽をアップロードするには、まずコンピュータのウェブブラウザで[iCloud.com](https://www.icloud.com)を開きます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_main.webp" alt="iCloud Driveメイン画面" width="800" >}}

Apple IDとパスワードでログインします。

正常にサインインすると、アカウントのiCloud.comメインダッシュボードが表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_logged_in.webp" alt="iCloud Driveインターフェースにログイン" width="800" >}}

**iCloud Drive**アイコンを見つけてクリックし、iCloud Driveの内容を開きます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_bubble.webp" alt="iCloud Driveショートカット" width="800" >}}

iCloud Driveに現在保存されているすべてのファイルとフォルダのリストが表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_folders_list.webp" alt="iCloud Driveのフォルダリスト" width="800" >}}

音楽を整理するには、新しいフォルダを作成するのがよいでしょう。**新規フォルダ**アイコンをクリックし、フォルダの名前を入力します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_enter_folder_name.webp" alt="iCloud Driveでフォルダ名を入力" width="800" >}}

新しいフォルダをクリックして開きます。現在空であることがわかります。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_opened_empty_folder.webp" alt="iCloud Driveで開いた空のフォルダ" width="800" >}}

次に**アップロード**ボタンをクリックします。ファイルピッカーが開きます。アップロードしたい音楽ファイルをコンピュータ上で選択し、**アップロード**をクリックしてプロセスを開始します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_selected_local_files.webp" alt="iCloudにアップロードする準備ができたローカルファイル" width="800" >}}

アップロードが完了するまで待ちます。進行状況はブラウザのインターフェースに直接表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_files_uploading_2.webp" alt="複数のファイルがiCloud Driveにアップロード中" width="800" >}}

ファイルがアップロードされたら、同じウェブインターフェースを使用してiCloud Drive内のファイルの名前変更、移動、削除ができます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_folder_rename.webp" alt="iCloud Driveでフォルダの名前を変更" width="800" >}}

## 音楽ストリーミング用にiCloud Driveを設定する

必要なアプリをインストールしたら、以下の手順に従ってiCloud Driveの音楽ストリーミングを設定します：

**iOS向け：**

1. アプリを開き、「接続」タブに移動します。
2. 「クラウドサービスを接続」をタップします。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_6120fb02234c4997a90734215cc4034a~mv2.webp" alt="クラウドサービスを接続" width="400" >}}

1. オプションから「iCloud Drive」を選択します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_a83af8de2e1a4c6dadf172a385f5353c~mv2.webp" alt="オプションメニューからiCloud Driveを選択" width="400" >}}

4. 下部のタブバーで「ブラウズ」をタップし、「場所」セクションまでスクロールします。「iCloud Drive」をタップして、iCloudアカウントにあるフォルダを開きます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_7f5d9769507943fb9956c22c76bed3d2~mv2.webp" alt="iCloud Driveフォルダを選択" width="400" >}}

5. オーディオファイルを含むiCloud Drive内のフォルダを選択します。「開く」をタップして選択を確認します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_e10a080568aa407d8e8fedde37234f7b~mv2.webp" alt="iCloud Driveフォルダ内のファイルを選択" width="400"  >}}

6. フォルダを選択すると、「クラウドサービス」セクションに表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_805d747135ae492e890257e80f7dedd6~mv2.webp" alt="接続画面の接続済みiCloud Driveフォルダ" width="400"  >}}

7. 追加したフォルダをタップしてコンテンツにアクセスし、iCloudから直接音楽の再生を開始します。アプリケーションはiCloudアカウントから直接オーディオファイルを再生し、デバイスにダウンロードせずにストレージをクリーンに保ちます。必要なファイルは三点メニューを使用してダウンロードできます。今すぐオーディオファイルをタップしてプレーヤーを起動してください。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_27090f1c30ff4c688d799dd4916e9ff8~mv2.webp" alt="接続済みiCloud Driveフォルダ内のファイルリスト" width="400"  >}}

8. タブバーの上にあるミニプレーヤーをタップするか、フルスクリーンプレーヤーを使用して、キュー、イコライザーなどの音楽プレーヤー設定を管理できます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_fb300231b3dd4e938bdc96b27c399374~mv2.webp" alt="オーディオプレーヤー画面" width="400"  >}}

**MacOS向け：**

1. アプリを起動し、左サイドバーの「すべての接続」をクリックします。
2. 「クラウドサービスを接続」を選択してクラウドサービスの追加を開始します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_6b0602b3bed2440986c0945585a5fbff~mv2.webp" alt="Mac画面のすべての接続メニュー項目" width="600"  >}}

3. オプションリストから「iCloud Drive」を選択します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_c003880182ec456bbb0403b0fb996a9e~mv2.webp" alt="オプションメニューからiCloud Driveを選択" width="600"  >}}

4. システムの「ファイル選択ダイアログ」が表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_0f7187489a194f8bb97f5554864a62cb~mv2.webp" alt="システムファイル選択ダイアログ" width="600"  >}}

5. ダイアログの上部にあるフォルダ選択をタップし、「iCloud Drive」に切り替えます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_2a750f91b74b47cc992e54c09315d424~mv2.webp" alt="メニューからiCloud Driveを選択" width="600"  >}}

6. iCloud Drive内のフォルダを選択し、「開く」をタップして選択を確認します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_4beca011a0dc4e47b9e7f51dc26baf5a~mv2.webp" alt="iCloud Driveフォルダ内のファイルを選択" width="600"  >}}

7. 選択したフォルダが「すべての接続」画面の「クラウドサービス」セクションに表示されます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_9e9a6e47129f4acd8c545bc5935dfe1c~mv2.webp" alt="接続済みiCloud Driveフォルダを含むすべての接続画面" width="600"  >}}

8. 追加したフォルダをタップして開きます。オーディオファイルをタップして再生を開始します。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_fd3f7b232d6042808cc01293e49fb104~mv2.webp" alt="iCloud Driveフォルダの内容" width="600"  >}}

9. 左サイドバーの「オーディオプレーヤー」をタップして、フルスクリーンオーディオプレーヤーを開きます。

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_0e45427ecccd4ba79b290f78bbdc3fd6~mv2.webp" alt="Macのフルスクリーンオーディオプレーヤー" width="600"  >}}

これらの手順に従うことで、デバイスにファイルをダウンロードすることなく、iCloud Driveから直接音楽を楽しむことができ、ストレージをクリーンに保ちながら音楽にアクセスできます。

## iCloud Driveからミュージックライブラリに曲を追加する

iCloud Driveから「ミュージックライブラリ」に曲を追加することもでき、アプリケーションがトラックのメタデータの読み取りを完了した後、すべてのトラックがアーティスト、アルバム、ジャンル、作曲者ごとにグループ化されます。

1. 「ミュージックライブラリ」に音楽を追加するには、左サイドバーの「すべてのミュージックライブラリ」メニュー項目を選択します。
2. 「ミュージックライブラリ」画面の右上隅にある三点をタップします。
3. 「+ 音楽を追加」メニュー項目を選択してファイル選択を開きます。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_db6d9e219ee1493983d59ad60c91fb92~mv2.webp" alt="ライブラリに音楽を追加" caption="ライブラリに音楽を追加" width="600" >}}

4. 次の画面で「接続」をタップし、接続済みのiCloud Driveフォルダを選択します。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_a2e104591b1c4ef7b8e54c4ac6df2cdb~mv2.webp" alt="音楽の場所を選択" caption="音楽の場所を選択" width="600" >}}

5. 次の画面で、ミュージックライブラリに追加したいファイルを選択します。フォルダを選択することもでき、その場合、アプリケーションが選択したフォルダをスキャンし、見つかったすべてのファイルがミュージックライブラリに追加されます。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_84aa1e97dc7e4361bdb2319a4544be39~mv2.webp" alt="フォルダ選択" caption="フォルダ選択" width="600" >}}

## iCloud Driveからプレイリストに曲を追加する

iCloudファイルをプレイリストに追加し、曲の順序を変更することもできます。

1. これを行うには、左サイドバーの「すべてのプレイリスト」メニュー項目をタップします。
2. 「すべてのプレイリスト」画面で、右上隅の三点ボタンをタップし、「+ 新しいプレイリスト」メニュー項目を選択して、新しいプレイリストの名前を入力します。
3. その後、「曲を追加」ダイアログが表示されます。
4. 「接続」メニュー項目をタップし、接続済みのiCloud Driveフォルダを選択します。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_10c3cee5586c4f8e9bdc17de5779e7c4~mv2.webp" alt="プレイリストに曲を追加" caption="プレイリストに曲を追加" width="600" >}}

5. 次の画面で、プレイリストに追加したいファイルを選択し、「完了」ボタンをタップして選択を確認します。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_361124a4c5d04b3e9077ad498900b645~mv2.webp" alt="プレイリスト用の曲を選択" caption="プレイリスト用の曲を選択" width="600" >}}

6. 右上隅の三点ボタンをタップして、作成したプレイリストで利用可能なすべてのアクションを表示します。

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_1506ad638e4347da8f93a0d411165f7a~mv2.webp" alt="iCloud Driveの曲が追加されたプレイリスト" caption="iCloud Driveの曲が追加されたプレイリスト" width="600" >}}

アプリケーションの設定で「オーディオプレーヤーの状態を保存」オプションを有効にしている場合、アプリの起動間ですべてのフォルダ/アルバム/プレイリストの再生位置が保存されます。再生位置を復元するには、プレイリストを開いて「再生を続ける」ボタンをタップします。

「すべて再生」 – 新しいオーディオプレーヤーキューを開始し、最初の曲から順番に再生されます（シャッフルモードが無効の場合）。

「すべてシャッフル」 – 新しいオーディオプレーヤーキューを開始し、再生開始前にこのプレイリストのすべての曲をシャッフルします。

## ストリーミングの問題を修正する

iCloudフォルダから音楽をストリーミングする際に問題が発生することがあります。この場合、オーディオプレーヤーキャッシュを有効にすることをお勧めします。この機能は、現在の曲が再生されている間に次の曲を事前にダウンロードし、中断のないスムーズな移行を確保します。オーディオプレーヤーキャッシュを有効にするには、設定 -> オーディオプレーヤー -> オーディオプレーヤーキャッシュ -> 有効化に進みます。キャッシュサイズも設定でき、古いトラックが削除され、デバイスの制限内にキャッシュが保たれます。

## FAQ

{{% details title="iCloud Driveからどのオーディオ形式をストリーミングできますか？" closed="true" %}}
EvermusicはMP3、WAV、AACなどの標準形式をサポートしています。FlacboxはFLAC、DSD、OGG、OPUSのサポートを追加します。音楽コレクションに合ったアプリを選択してください。
{{% /details %}}

{{% details title="iCloud Driveからのストリーミングはデバイスのストレージを使用しますか？" closed="true" %}}
いいえ。EvermusicとFlacboxの両方がiCloud Driveから直接オーディオをストリーミングし、デバイスにファイルをダウンロードしません。オフラインで聴くために個別のトラックをダウンロードすることもできます。
{{% /details %}}

{{% details title="iCloud Driveの音楽をオフラインで使用できますか？" closed="true" %}}
はい。任意のトラックの三点メニューをタップし、ダウンロードオプションを選択します。ファイルはオフライン再生用にローカルに保存されます。
{{% /details %}}

{{% details title="再生中に音楽が停止したりバッファリングしたりするのはなぜですか？" closed="true" %}}
これは通常、インターネット接続が遅いか不安定であることが原因です。設定でオーディオプレーヤーキャッシュを有効にして、次のトラックを事前にダウンロードし、中断を防止してください。
{{% /details %}}

{{% details title="iCloud Driveからの音楽ストリーミングは無料ですか？" closed="true" %}}
EvermusicとFlacboxはどちらも無料でダウンロードできます。iCloud Driveは5GBの無料ストレージを提供しています。より多くのスペースが必要な場合は、AppleからiCloudストレージプランをアップグレードできます。
{{% /details %}}

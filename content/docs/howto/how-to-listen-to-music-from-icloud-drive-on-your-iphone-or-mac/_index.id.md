**Ringkasan:** Unggah musik ke iCloud Drive, instal [Evermusic](/products/evermusic) (untuk MP3/WAV) atau [Flacbox](/products/flacbox) (untuk FLAC/DSD), hubungkan folder iCloud Drive Anda, dan streaming langsung tanpa menggunakan penyimpanan perangkat.

Di dunia saat ini, musik adalah bagian besar dari kehidupan kita, dan memiliki lagu favorit yang tersedia setiap saat adalah suatu keharusan. Berkat teknologi cloud, Anda tidak perlu lagi menyimpan koleksi musik fisik. Anda sekarang dapat streaming musik dari iCloud Drive hanya dengan beberapa ketukan.

iPhone Anda dapat mengakses seluruh perpustakaan lagu yang tersimpan di cloud. Baik Anda memiliki daftar putar yang terorganisir dengan rapi atau campuran berbagai lagu, panduan ini akan menunjukkan cara menggunakan iCloud Drive untuk mengubah iPhone Anda menjadi pemutar musik portabel.

Kami akan menjelaskan cara streaming musik dari iCloud Drive, dengan langkah-langkah mudah dan tips untuk membuat prosesnya lancar. Baik Anda sudah familiar dengan iCloud Drive atau baru mengenalnya, di akhir panduan ini, Anda akan tahu cara memutar lagu favorit dari cloud di iPhone Anda.

Jadi, jika Anda siap menikmati kemudahan streaming musik dari iPhone, mari kita mulai!

## Memulai: Menginstal Aplikasi yang Diperlukan

Sebelum Anda dapat mulai menikmati musik iCloud Drive di iPhone atau Mac, Anda perlu menginstal aplikasi yang tepat. Berikut langkah-langkahnya:

- Untuk iOS:

1. Buka App Store dan unduh **Evermusic** jika musik Anda disimpan dalam format audio standar seperti mp3 atau wav. Jika Anda memiliki musik lossless dalam format dsd atau flac, pilih **Flacbox**. Kedua aplikasi tersedia untuk iOS dan MacOS.

- Untuk MacOS:

1. Kunjungi App Store di Mac Anda dan instal **Evermusic** atau **Flacbox** berdasarkan preferensi format musik Anda.

## Mengunggah Musik ke iCloud Drive

Untuk mengunggah musik dari Mac atau PC ke iCloud Drive, mulailah dengan membuka [iCloud.com](https://www.icloud.com) di browser web di komputer Anda.

Masuk menggunakan Apple ID dan kata sandi Anda.

Setelah berhasil masuk, Anda akan melihat dasbor utama iCloud.com untuk akun Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_logged_in.webp" alt="Antarmuka iCloud Drive setelah masuk" width="800" >}}

Cari ikon **iCloud Drive** dan klik untuk membuka konten iCloud Drive Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_bubble.webp" alt="Pintasan iCloud Drive" width="800" >}}

Anda akan melihat daftar semua file dan folder yang saat ini tersimpan di iCloud Drive Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_folders_list.webp" alt="Daftar folder di iCloud Drive" width="800" >}}

Untuk mengorganisir musik Anda, sebaiknya buat folder baru. Klik ikon **Folder Baru** dan masukkan nama untuk folder Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/icloud_drive_enter_folder_name.webp" alt="Memasukkan nama folder di iCloud Drive" width="800" >}}

## Menyiapkan iCloud Drive untuk Streaming Musik

Sekarang setelah Anda menginstal aplikasi yang diperlukan, ikuti langkah-langkah berikut untuk menyiapkan iCloud Drive untuk streaming musik:

**Untuk iOS:**

1. Buka aplikasi dan navigasi ke tab «Koneksi».
2. Ketuk «Hubungkan layanan cloud».

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_6120fb02234c4997a90734215cc4034a~mv2.webp" alt="Hubungkan layanan cloud" width="400" >}}

1. Pilih «iCloud Drive» dari opsi yang tersedia.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_a83af8de2e1a4c6dadf172a385f5353c~mv2.webp" alt="Pilih iCloud Drive dari menu opsi" width="400" >}}

4. Ketuk «Telusuri» pada bilah tab bawah dan gulir ke bawah ke bagian «Lokasi». Ketuk «iCloud Drive» untuk membuka folder yang ada di akun iCloud Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_7f5d9769507943fb9956c22c76bed3d2~mv2.webp" alt="Pilih folder iCloud Drive" width="400" >}}

5. Pilih folder di iCloud Drive yang berisi file audio. Ketuk «Buka» untuk mengonfirmasi pilihan Anda.

{{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_e10a080568aa407d8e8fedde37234f7b~mv2.webp" alt="Pilih file di folder iCloud Drive" width="400"  >}}

## Menambahkan Lagu dari iCloud Drive ke Perpustakaan Musik

Anda juga dapat menambahkan lagu dari iCloud Drive ke «Perpustakaan Musik» dan semua lagu akan dikelompokkan berdasarkan Artis, Album, Genre, dan Komposer setelah aplikasi selesai membaca metadata lagu.

1. Untuk menambahkan musik ke «Perpustakaan Musik», pilih item menu «Seluruh Perpustakaan Musik» di bilah sisi kiri.
2. Ketuk tiga titik di sudut kanan atas layar «Perpustakaan Musik».
3. Pilih item menu «+ Tambah musik» untuk membuka pemilih file.

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_db6d9e219ee1493983d59ad60c91fb92~mv2.webp" alt="Tambah musik ke perpustakaan" caption="Tambah musik ke perpustakaan" width="600" >}}

4. Di layar berikutnya ketuk «Koneksi» dan pilih folder iCloud Drive yang terhubung.

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_a2e104591b1c4ef7b8e54c4ac6df2cdb~mv2.webp" alt="Pilih lokasi musik" caption="Pilih lokasi musik" width="600" >}}

5. Di layar berikutnya pilih file yang ingin Anda tambahkan ke perpustakaan musik. Anda juga dapat memilih folder dan dalam hal ini, aplikasi akan memindai folder yang dipilih dan semua file yang ditemukan akan ditambahkan ke perpustakaan musik.

## Menambahkan Lagu dari iCloud Drive ke Daftar Putar

Anda juga dapat menambahkan file iCloud ke daftar putar dan mengubah urutan lagu.

1. Untuk melakukannya, ketuk item menu «Semua Daftar Putar» di bilah sisi kiri.
2. Di layar «Semua Daftar Putar», ketuk tombol tiga titik di sudut kanan atas, pilih item menu «+ Daftar putar baru» dan masukkan nama untuk daftar putar baru.
3. Setelah itu, Anda akan melihat dialog «Tambah lagu».
4. Ketuk item menu «Koneksi» dan pilih folder iCloud Drive yang terhubung.

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_10c3cee5586c4f8e9bdc17de5779e7c4~mv2.webp" alt="Tambah lagu ke daftar putar" caption="Tambah lagu ke daftar putar" width="600" >}}

5. Di layar berikutnya pilih file yang ingin Anda tambahkan ke daftar putar dan ketuk «Selesai» untuk mengonfirmasi pilihan.

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_361124a4c5d04b3e9077ad498900b645~mv2.webp" alt="Pilih lagu untuk daftar putar" caption="Pilih lagu untuk daftar putar" width="600" >}}

6. Ketuk tombol tiga titik di sudut kanan atas untuk melihat semua tindakan yang tersedia untuk daftar putar yang dibuat.

   {{< figure src="/docs/howto/how-to-listen-to-music-from-icloud-drive-on-your-iphone-or-mac/21260c_1506ad638e4347da8f93a0d411165f7a~mv2.webp" alt="Daftar putar dengan lagu iCloud Drive yang ditambahkan" caption="Daftar putar dengan lagu iCloud Drive yang ditambahkan" width="600" >}}

Jika Anda telah mengaktifkan opsi «Simpan status pemutar audio» di Pengaturan aplikasi, posisi pemutaran akan disimpan untuk semua folder/album/daftar putar Anda di antara peluncuran aplikasi. Untuk memulihkan posisi pemutaran, buka daftar putar dan ketuk tombol «Lanjutkan pemutaran».

«Putar semua» – memulai antrean pemutar audio baru dan lagu akan diputar satu per satu dari lagu pertama (jika mode acak dinonaktifkan).

«Acak semua» – memulai antrean pemutar audio baru dan mengacak semua lagu dari daftar putar ini sebelum memulai pemutaran.

## Memperbaiki Masalah Streaming

Terkadang, Anda mungkin mengalami masalah saat streaming musik dari folder iCloud Anda. Dalam hal ini, kami merekomendasikan untuk mengaktifkan cache pemutar audio. Fitur ini akan mengunduh lagu berikutnya terlebih dahulu sementara yang saat ini sedang diputar, memastikan transisi yang mulus tanpa gangguan. Untuk mengaktifkan cache pemutar audio, buka Pengaturan -> Pemutar Audio -> Cache Pemutar Audio -> Aktifkan. Anda juga dapat mengatur ukuran cache sehingga lagu lama dihapus, menjaga cache tetap dalam batas di perangkat Anda.

## FAQ

{{% details title="Format audio apa yang bisa saya streaming dari iCloud Drive?" closed="true" %}}
Evermusic mendukung MP3, WAV, AAC, dan format standar lainnya. Flacbox menambahkan dukungan untuk FLAC, DSD, OGG, dan OPUS. Pilih aplikasi yang sesuai dengan koleksi musik Anda.
{{% /details %}}

{{% details title="Apakah streaming dari iCloud Drive menggunakan penyimpanan perangkat?" closed="true" %}}
Tidak. Baik Evermusic maupun Flacbox streaming audio langsung dari iCloud Drive Anda tanpa mengunduh file ke perangkat. Anda dapat secara opsional mengunduh lagu individual untuk mendengarkan offline.
{{% /details %}}

{{% details title="Bisakah saya menggunakan musik iCloud Drive secara offline?" closed="true" %}}
Ya. Ketuk menu tiga titik pada lagu apa pun dan pilih opsi unduh. File akan disimpan secara lokal untuk pemutaran offline.
{{% /details %}}

{{% details title="Mengapa musik saya berhenti atau buffering saat pemutaran?" closed="true" %}}
Ini biasanya disebabkan oleh koneksi internet yang lambat atau tidak stabil. Aktifkan cache pemutar audio di Pengaturan untuk mengunduh lagu berikutnya terlebih dahulu dan mencegah gangguan.
{{% /details %}}

{{% details title="Apakah streaming musik dari iCloud Drive gratis?" closed="true" %}}
Baik Evermusic maupun Flacbox gratis untuk diunduh. iCloud Drive menawarkan penyimpanan gratis 5 GB. Anda dapat meningkatkan paket penyimpanan iCloud melalui Apple jika membutuhkan lebih banyak ruang.
{{% /details %}}

**Ringkasan:** Gunakan aplikasi **Evertag** gratis untuk mengedit lirik tidak tersinkronisasi, peringkat konten, dan 120+ tag audio di iPhone atau Mac. Bekerja dengan file lokal dan yang disimpan di cloud, mendukung pengeditan batch, dan menyimpan lirik yang terlihat di Evermusic, Flacbox, dan pemutar lainnya.

## Pendahuluan

Dalam tutorial ini, Anda akan belajar cara mengedit **Lirik Tidak Tersinkronisasi** dan **Peringkat Konten Lirik** di tag audio file musik Anda. Setelah diperbarui, lirik ini akan terlihat di pemutar musik mana pun yang mendukung tag lirik seperti **Evermusic** atau **Flacbox**.

Untuk demonstrasi, kami akan menggunakan aplikasi **Evertag**. Aplikasi ini mendukung **120+ tag audio**, **30+ format audio**, dan menyertakan fitur **pengeditan batch** yang kuat. Jika Anda mencari alat manajemen tag lengkap untuk iPhone atau Mac, **Evertag** adalah pilihan terbaik Anda.

## Mode Pengeditan

Evertag menyediakan dua mode pengeditan:

- **Mode file tunggal**  
  - Navigasi antar file dengan menggeser ke kiri atau kanan pada karusel artwork.  

- **Mode batch**  
  - Edit beberapa file sekaligus dan terapkan metadata bersama.  
  - Untuk mengaktifkan, gulir ke bawah dan ketuk **Edit beberapa file secara bersamaan**.

## Mode File Tunggal

Secara default, aplikasi membuka editor tag dalam mode file tunggal dengan hanya opsi pengeditan utama yang diaktifkan. Dalam mode ini, Anda dapat mengedit bidang metadata paling umum, seperti: Judul Trek, Subjudul, Artis Album, Album, Artis, Komposer, Performer, Genre, Komentar, Ketukan Per Menit, Podcast, Kompilasi, Nomor Disk, Nomor Trek, Total Trek, Peringkat, Tahun.

Untuk mengakses semua tag yang tersedia, gulir ke bagian bawah layar dan ketuk opsi **Tampilkan Tag Diperluas**. Ini akan mengalihkan editor ke mode diperluas, memungkinkan Anda mengedit lebih dari **120 bidang metadata**, termasuk **tag MusicBrainz**, **lirik**, **peringkat konten**, dan lainnya.

## Mode Batch

Anda dapat masuk ke pengeditan batch dengan dua cara:

1. **Dari Pengelola File**  
   - Ketuk **Tindakan lainnya** (•••) di kanan atas.  
   - Ketuk **Pilih**, pilih beberapa file, lalu ketuk **Edit tag audio**.

2. **Dari Editor Tag**  
   - Buka file apa pun, gulir ke bawah, dan ketuk **Edit file secara bersamaan** untuk memuat semua file dari folder yang sama.

Setelah mengedit, ketuk **Simpan** untuk menerapkan perubahan.

## Edit Lirik

Berikut cara menambahkan atau memperbarui **Lirik Tidak Tersinkronisasi** yang tertanam di file audio Anda menggunakan aplikasi Evertag.

### Ketuk Bidang Teks "Lirik Tidak Tersinkronisasi"

Dalam mode **Tag Diperluas**, gulir ke bawah dan ketuk bidang teks **Lirik Tidak Tersinkronisasi**.

{{< cards cols="1">}}
{{< card title="" subtitle="Bidang Teks Lirik Tidak Tersinkronisasi" image="/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/advisory-rating-2.webp" >}}
{{< /cards >}}

> File audio yang mendukung **tag ID3** (seperti `.mp3` atau `.wav`) memungkinkan Anda menambahkan lirik dalam beberapa bahasa. Jika Anda mengedit file bertag ID3, Evertag mengaktifkan dukungan multi-bahasa penuh.  
> Untuk format lain (mis. FLAC, M4A), hanya satu entri lirik yang didukung. Setelah mengetuk bidang **Lirik Tidak Tersinkronisasi**, Anda akan dapat menambahkan atau mengedit konten lirik.

### Tambah Halaman Baru (Hanya Tag ID3)

Jika Anda mengedit tag ID3, layar berikutnya akan menampilkan tombol **Tambah Halaman Baru**. Ketuk untuk mulai menambahkan entri lirik baru.

### Atur Peringkat Konten Lirik

Opsional, perbarui **Peringkat Konten Lirik** menggunakan kontrol tersegmentasi. Pengaturan ini membantu pemutar musik dan perpustakaan menandai trek sebagai eksplisit atau bersih.

| Value | Symbol | Label           | Deskripsi                                           |
|-------|--------|------------------|-------------------------------------------------------|
| 0     | –      | Tidak Diperingkat        | Tidak ada peringkat konten yang ditetapkan.                |
| 1     | 🅒     | Lirik Bersih     | Konten diberi label bersih; aman untuk semua audiens. |
| 2     | 🅔     | Lirik Eksplisit  | Mengandung lirik eksplisit atau konten dewasa.           |

## Kesimpulan

Dengan mengikuti langkah-langkah ini, lirik Anda akan tertanam dengan benar dalam metadata file audio dan terlihat di aplikasi yang kompatibel seperti **Evermusic**, **Flacbox**, dan lainnya.

## Pertanyaan yang Sering Diajukan

{{% details title="Format audio apa yang didukung Evertag untuk pengeditan lirik?" closed="true" %}}
Evertag mendukung lebih dari 30 format audio, termasuk MP3, FLAC, WAV, M4A, OGG, AIFF, dan lainnya. Anda dapat mengedit lirik dan tag metadata lainnya di format apa pun ini.
{{% /details %}}

{{% details title="Bisakah saya menambahkan lirik dalam beberapa bahasa?" closed="true" %}}
Ya, tetapi hanya untuk file audio yang menggunakan tag ID3 (seperti MP3 dan WAV). Untuk format lain seperti FLAC atau M4A, hanya satu entri lirik yang didukung.
{{% /details %}}

{{% details title="Apakah Evertag mendukung pengeditan lirik batch?" closed="true" %}}
Ya. Anda dapat masuk ke mode batch untuk mengedit metadata di beberapa file sekaligus. Ini berguna untuk menerapkan peringkat konten lirik yang sama atau tag bersama lainnya ke seluruh album.
{{% /details %}}

{{% details title="Apakah lirik yang diedit akan muncul di Apple Music atau Spotify?" closed="true" %}}
Lirik yang diedit dengan Evertag tertanam dalam metadata file audio. Mereka akan muncul di pemutar musik mana pun yang membaca tag lirik tertanam, seperti Evermusic, Flacbox, VLC, dan foobar2000. Aplikasi streaming seperti Spotify dan Apple Music menggunakan basis data lirik mereka sendiri dan tidak membaca tag tertanam.
{{% /details %}}

{{% details title="Bisakah saya mengedit tag untuk file yang disimpan di penyimpanan cloud?" closed="true" %}}
Ya. Evertag mendukung koneksi ke layanan penyimpanan cloud. Aplikasi mengunduh file, memungkinkan Anda mengedit tag, dan mengunggah file yang diperbarui kembali ke cloud secara otomatis.
{{% /details %}}

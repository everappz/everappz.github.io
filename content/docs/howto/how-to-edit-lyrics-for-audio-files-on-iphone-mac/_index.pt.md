**Resumo:** Use o aplicativo gratuito **Evertag** para editar letras não sincronizadas, classificações de conteúdo e mais de 120 tags de áudio no iPhone ou Mac. Funciona com arquivos locais e armazenados na nuvem, suporta edição em lote e salva letras visíveis no Evermusic, Flacbox e outros reprodutores.

## Introdução

Neste tutorial, você aprenderá como editar **letras não sincronizadas** e a **classificação de conteúdo das letras** nas tags de áudio dos seus arquivos de música. Uma vez atualizadas, essas letras serão visíveis em qualquer reprodutor de música que suporte a tag de letras, como **Evermusic** ou **Flacbox**.

Para demonstração, usaremos o aplicativo **Evertag**. Ele suporta **mais de 120 tags de áudio**, **mais de 30 formatos de áudio** e inclui recursos poderosos de **edição em lote**. Se você está procurando uma ferramenta completa de gerenciamento de tags para iPhone ou Mac, o **Evertag** é sua melhor escolha.

## Conectar pasta local

Você pode editar tags de áudio para arquivos armazenados diretamente no seu dispositivo sem importá-los para o aplicativo.

### iOS

- **Vá para a seção "Arquivos locais"**, depois role para baixo até **"Arquivos neste dispositivo"**

- Toque em **"Conectar uma pasta"**.

- No seletor de pastas, escolha o diretório que deseja acessar e toque em **"Abrir"** para confirmar.

- Após adicionar a pasta, toque nela para ver os arquivos dentro.

## Modos de edição

O Evertag oferece dois modos de edição:

- **Modo de arquivo único**  
  - Navegue entre arquivos deslizando para a esquerda ou direita no carrossel de capas.  

- **Modo em lote**  
  - Edite vários arquivos de uma vez e aplique metadados compartilhados.  
  - Para ativar, role para baixo e toque em **Editar vários arquivos simultaneamente**.

## Modo de arquivo único

Por padrão, o aplicativo abre o editor de tags no modo de arquivo único com apenas as opções principais de edição habilitadas. Neste modo, você pode editar os campos de metadados mais comuns, como: Título da faixa, Subtítulo, Artista do álbum, Álbum, Artista, Compositor, Intérprete, Gênero, Comentário, Batidas por minuto, Podcast, Compilação, Número do disco, Número da faixa, Total de faixas, Classificação, Ano.

Para acessar todas as tags disponíveis, role até o final da tela e toque na opção **Mostrar tags estendidas**. Isso alternará o editor para o modo estendido, permitindo editar mais de **120 campos de metadados**, incluindo **tags MusicBrainz**, **letras**, **classificações de conteúdo** e mais.

## Modo em lote

Você pode entrar na edição em lote de duas formas:

1. **Do gerenciador de arquivos**  
   - Toque em **Mais ações** (•••) no canto superior direito.  
   - Toque em **Selecionar**, escolha vários arquivos e depois toque em **Editar tags de áudio**.

2. **Do editor de tags**  
   - Abra qualquer arquivo, role para baixo e toque em **Editar arquivos simultaneamente** para carregar todos os arquivos da mesma pasta.

Após a edição, toque em **Salvar** para aplicar as alterações.

## Editar letras

Veja como adicionar ou atualizar **letras não sincronizadas** incorporadas nos seus arquivos de áudio usando o aplicativo Evertag.

### Toque no campo de texto "Letras não sincronizadas"

No modo **Tags estendidas**, role para baixo e toque no campo de texto **Letras não sincronizadas**.

{{< cards cols="1">}}
{{< card title="" subtitle="Campo de texto de letras não sincronizadas" image="/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/advisory-rating-2.webp" >}}
{{< /cards >}}

> Arquivos de áudio que suportam **tags ID3** (como `.mp3` ou `.wav`) permitem adicionar letras em vários idiomas. Se você está editando um arquivo com tags ID3, o Evertag habilita suporte multilíngue completo.  
> Para outros formatos (ex.: FLAC, M4A), apenas uma única entrada de letras é suportada. Após tocar no campo **Letras não sincronizadas**, você poderá adicionar ou editar o conteúdo das letras.

### Adicionar uma nova página (apenas tags ID3)

Se você está editando tags ID3, a próxima tela mostrará um botão **Adicionar nova página**. Toque nele para começar a adicionar uma nova entrada de letras.

### Escolher idioma, comentário e conteúdo das letras

Na tela de entrada de letras, você poderá:
- Selecionar o **idioma** para suas letras (padrão é inglês)
- Adicionar um **comentário** opcional
- Inserir o **texto das letras** real

> Dica: Procurando letras de alta qualidade? Visite [lyricsify.com](https://www.lyricsify.com) para encontrar letras no formato LRC para milhares de faixas.

### Definir a classificação de conteúdo das letras

Opcionalmente, atualize a **Classificação de conteúdo das letras** usando o controle segmentado. Esta configuração ajuda reprodutores de música e bibliotecas a marcar faixas como explícitas ou limpas.

| Value | Symbol | Rótulo           | Descrição                                           |
|-------|--------|------------------|-------------------------------------------------------|
| 0     | –      | Não classificado        | Nenhuma classificação de conteúdo foi atribuída.                |
| 1     | 🅒     | Letras limpas     | Conteúdo rotulado como limpo; seguro para todos os públicos. |
| 2     | 🅔     | Letras explícitas  | Contém letras explícitas ou conteúdo adulto.           |

## Perguntas frequentes

{{% details title="Quais formatos de áudio o Evertag suporta para edição de letras?" closed="true" %}}
O Evertag suporta mais de 30 formatos de áudio, incluindo MP3, FLAC, WAV, M4A, OGG, AIFF e mais. Você pode editar letras e outras tags de metadados em qualquer um desses formatos.
{{% /details %}}

{{% details title="Posso adicionar letras em vários idiomas?" closed="true" %}}
Sim, mas apenas para arquivos de áudio que usam tags ID3 (como MP3 e WAV). Para outros formatos como FLAC ou M4A, apenas uma única entrada de letras é suportada.
{{% /details %}}

{{% details title="O Evertag suporta edição em lote de letras?" closed="true" %}}
Sim. Você pode entrar no modo em lote para editar metadados de vários arquivos de uma vez. Isso é útil para aplicar a mesma classificação de conteúdo de letras ou outras tags compartilhadas a um álbum inteiro.
{{% /details %}}

{{% details title="As letras editadas aparecerão no Apple Music ou Spotify?" closed="true" %}}
Letras editadas com o Evertag são incorporadas nos metadados do arquivo de áudio. Elas aparecerão em qualquer reprodutor de música que leia tags de letras incorporadas, como Evermusic, Flacbox, VLC e foobar2000. Aplicativos de streaming como Spotify e Apple Music usam seus próprios bancos de dados de letras e não leem tags incorporadas.
{{% /details %}}

{{% details title="Posso editar tags para arquivos armazenados na nuvem?" closed="true" %}}
Sim. O Evertag suporta conexão com serviços de armazenamento em nuvem. O aplicativo baixa o arquivo, permite que você edite as tags e faz o upload do arquivo atualizado de volta para a nuvem automaticamente.
{{% /details %}}

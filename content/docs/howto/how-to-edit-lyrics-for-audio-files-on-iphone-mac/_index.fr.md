**Résumé:** Utilisez l'application gratuite **Evertag** pour modifier les paroles non synchronisées, les classements consultatifs et plus de 120 tags audio sur iPhone ou Mac. Fonctionne avec les fichiers locaux et stockés dans le cloud, prend en charge l'édition par lots et enregistre les paroles visibles dans Evermusic, Flacbox et d'autres lecteurs.

## Introduction

Dans ce tutoriel, vous apprendrez à modifier les **paroles non synchronisées** et le **classement consultatif des paroles** dans les tags audio de vos fichiers musicaux. Une fois mises à jour, ces paroles seront visibles dans tout lecteur de musique prenant en charge le tag de paroles, comme **Evermusic** ou **Flacbox**.

Pour la démonstration, nous utiliserons l'application **Evertag**. Elle prend en charge **plus de 120 tags audio**, **plus de 30 formats audio** et inclut de puissantes fonctionnalités d'**édition par lots**. Si vous cherchez un outil complet de gestion des tags pour iPhone ou Mac, **Evertag** est votre meilleur choix.

## Connecter un dossier local

Vous pouvez modifier les tags audio des fichiers stockés directement sur votre appareil sans les importer dans l'application.

### iOS

- **Allez dans la section « Fichiers locaux »**, puis faites défiler jusqu'à **« Fichiers sur cet appareil »**

- Appuyez sur **« Connecter un dossier »**.

- Dans le sélecteur de dossiers, choisissez le répertoire auquel vous souhaitez accéder et appuyez sur **« Ouvrir »** pour confirmer.

- Après avoir ajouté le dossier, appuyez dessus pour voir les fichiers qu'il contient.

### macOS

- Faites défiler jusqu'à **Fichiers sur cet appareil** dans le menu de la barre latérale

## Modes d'édition

Evertag propose deux modes d'édition :

- **Mode fichier unique**  
  - Naviguez entre les fichiers en glissant à gauche ou à droite sur le carrousel d'illustrations.  

- **Mode par lots**  
  - Modifiez plusieurs fichiers à la fois et appliquez des métadonnées partagées.  
  - Pour activer, faites défiler vers le bas et appuyez sur **Modifier plusieurs fichiers simultanément**.

## Mode fichier unique

Par défaut, l'application ouvre l'éditeur de tags en mode fichier unique avec uniquement les options d'édition principales activées. Dans ce mode, vous pouvez modifier les champs de métadonnées les plus courants, tels que : Titre de la piste, Sous-titre, Artiste de l'album, Album, Artiste, Compositeur, Interprète, Genre, Commentaire, Battements par minute, Podcast, Compilation, Numéro de disque, Numéro de piste, Total des pistes, Note, Année.

Pour accéder à tous les tags disponibles, faites défiler jusqu'en bas de l'écran et appuyez sur l'option **Afficher les tags étendus**. Cela basculera l'éditeur en mode étendu, vous permettant de modifier plus de **120 champs de métadonnées**, y compris les **tags MusicBrainz**, les **paroles**, les **classements consultatifs** et plus encore.

## Mode par lots

Vous pouvez entrer en édition par lots de deux façons :

1. **Depuis le gestionnaire de fichiers**  
   - Appuyez sur **Plus d'actions** (•••) en haut à droite.  
   - Appuyez sur **Sélectionner**, choisissez plusieurs fichiers, puis appuyez sur **Modifier les tags audio**.

2. **Depuis l'éditeur de tags**  
   - Ouvrez n'importe quel fichier, faites défiler vers le bas et appuyez sur **Modifier les fichiers simultanément** pour charger tous les fichiers du même dossier.

Après l'édition, appuyez sur **Enregistrer** pour appliquer les modifications.

## Modifier les paroles

Voici comment ajouter ou mettre à jour les **paroles non synchronisées** intégrées dans vos fichiers audio à l'aide de l'application Evertag.

### Appuyez sur le champ de texte « Paroles non synchronisées »

En mode **Tags étendus**, faites défiler vers le bas et appuyez sur le champ de texte **Paroles non synchronisées**.

{{< cards cols="1">}}
{{< card title="" subtitle="Champ de texte des paroles non synchronisées" image="/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/advisory-rating-2.webp" >}}
{{< /cards >}}

> Les fichiers audio qui prennent en charge les **tags ID3** (tels que `.mp3` ou `.wav`) vous permettent d'ajouter des paroles dans plusieurs langues. Si vous modifiez un fichier avec des tags ID3, Evertag active la prise en charge multilingue complète.  
> Pour les autres formats (par exemple FLAC, M4A), seule une entrée de paroles unique est prise en charge. Après avoir appuyé sur le champ **Paroles non synchronisées**, vous pourrez ajouter ou modifier le contenu des paroles.

### Ajouter une nouvelle page (tags ID3 uniquement)

Si vous modifiez des tags ID3, l'écran suivant affichera un bouton **Ajouter une nouvelle page**. Appuyez dessus pour commencer à ajouter une nouvelle entrée de paroles.

- Ajouter un **commentaire** optionnel
- Entrer le **texte des paroles** réel

Enfin, sur l'écran de l'**Éditeur de tags**, appuyez sur **Enregistrer** pour écrire les tags mis à jour — y compris vos nouvelles paroles — dans le fichier.

### Définir le classement consultatif des paroles

Optionnellement, mettez à jour le **Classement consultatif des paroles** à l'aide du contrôle segmenté. Ce paramètre aide les lecteurs de musique et les bibliothèques à marquer les pistes comme explicites ou propres.

| Value | Symbol | Libellé           | Description                                           |
|-------|--------|------------------|-------------------------------------------------------|
| 0     | –      | Non classé        | Aucun classement consultatif n'a été attribué.                |
| 1     | 🅒     | Paroles propres     | Le contenu est étiqueté comme propre ; adapté à tous les publics. |
| 2     | 🅔     | Paroles explicites  | Contient des paroles explicites ou du contenu pour adultes.           |

## Conclusion

En suivant ces étapes, vos paroles seront correctement intégrées dans les métadonnées du fichier audio et visibles dans les applications compatibles comme **Evermusic**, **Flacbox** et d'autres.

## Questions fréquemment posées

{{% details title="Quels formats audio Evertag prend-il en charge pour l'édition de paroles ?" closed="true" %}}
Evertag prend en charge plus de 30 formats audio, y compris MP3, FLAC, WAV, M4A, OGG, AIFF et plus encore. Vous pouvez modifier les paroles et d'autres tags de métadonnées dans n'importe lequel de ces formats.
{{% /details %}}

{{% details title="Puis-je ajouter des paroles dans plusieurs langues ?" closed="true" %}}
Oui, mais uniquement pour les fichiers audio qui utilisent des tags ID3 (tels que MP3 et WAV). Pour les autres formats comme FLAC ou M4A, seule une entrée de paroles unique est prise en charge.
{{% /details %}}

{{% details title="Evertag prend-il en charge l'édition par lots des paroles ?" closed="true" %}}
Oui. Vous pouvez entrer en mode par lots pour modifier les métadonnées de plusieurs fichiers à la fois. C'est utile pour appliquer le même classement consultatif de paroles ou d'autres tags partagés à un album entier.
{{% /details %}}

{{% details title="Les paroles modifiées apparaîtront-elles dans Apple Music ou Spotify ?" closed="true" %}}
Les paroles modifiées avec Evertag sont intégrées dans les métadonnées du fichier audio. Elles apparaîtront dans tout lecteur de musique qui lit les tags de paroles intégrés, comme Evermusic, Flacbox, VLC et foobar2000. Les applications de streaming comme Spotify et Apple Music utilisent leurs propres bases de données de paroles et ne lisent pas les tags intégrés.
{{% /details %}}

{{% details title="Puis-je modifier les tags des fichiers stockés dans le cloud ?" closed="true" %}}
Oui. Evertag prend en charge la connexion aux services de stockage cloud. L'application télécharge le fichier, vous permet de modifier les tags et téléverse automatiquement le fichier mis à jour dans le cloud.
{{% /details %}}

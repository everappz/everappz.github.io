**Коротко:** Підключіть Synology NAS до Evermusic або Flacbox за допомогою нативного API Synology -- вручну через IP-адресу або автоматично через QuickConnect ID. QuickConnect дозволяє транслювати музику віддалено без переадресації портів. Обидва додатки підтримують FLAC, MP3, WAV та інші формати високої роздільності.

Якщо ви шукаєте зручний спосіб підключити Synology NAS та насолоджуватися музичною бібліотекою на iPhone або Mac, додатки Evermusic та Flacbox стануть ідеальним рішенням. Ці додатки підтримують широкий спектр аудіоформатів, включаючи FLAC, MP3 та WAV, що дозволяє легко транслювати та слухати високоякісну музику безпосередньо з Synology NAS.

У цьому посібнику ми покажемо, як підключити Synology NAS до додатку Evermusic або Flacbox за допомогою нативного API Synology та функції QuickConnect. Використовуючи прямий API Synology, ви можете безпечно отримати доступ до музичної бібліотеки за межами домашньої мережі без складних налаштувань, таких як WebDAV, SMB, DLNA. З QuickConnect ви зможете транслювати та керувати музикою з будь-якого місця, використовуючи iPhone або Mac.

## Крок 1: Налаштування дозволів спільної папки (необов'язково)

1. Відкрийте **Панель керування** та перейдіть до розділу **Спільна папка**.

![Спільна папка](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_599ebfa896164704ba4497524286ea58~mv2.png)

2. Виберіть папку **Music** та натисніть **Редагувати**.

3. На вкладці **Дозволи** налаштуйте дозволи. Увімкніть гостьовий доступ з правами «Лише читання», якщо вам потрібно лише слухати музику, або «Читання/Запис», якщо потрібно змінювати файли. Збережіть зміни.

![Дозволи](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_43b09e36fc284a43b867e588da32b314~mv2.png)

## Крок 2: Знайдіть IP-адресу Synology NAS

1. Відкрийте **Панель керування** та перейдіть на вкладку **Мережевий інтерфейс**.

![Мережевий інтерфейс](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_51839801a6f44035b08b3a4b7d33f142~mv2.png)

2. Або скористайтеся веб-браузером, щоб відвідати [find.synology.com](http://find.synology.com).

![Знайти Synology](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_5bfb1db8e04d4eddb8ff5238f8b214da~mv2.png)

3. Запишіть IP-адресу вашого Synology NAS (наприклад, 192.168.18.137).

## Крок 3: Знайдіть мережеві порти Synology NAS

Офіційну документацію Synology щодо мережевих портів DSM за замовчуванням можна знайти в цій [статті Центру знань Synology](https://kb.synology.com/en-global/DSM/tutorial/What_network_ports_are_used_by_Synology_services).

Synology DSM використовує такі порти за замовчуванням:
- **HTTP (Веб-доступ):** Порт **5000**
- **HTTPS (Безпечний веб-доступ):** Порт **5001**

Це порти за замовчуванням для доступу до інтерфейсу DSM.

![Мережеві порти](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_61d64239cd7e4bfab2530c32b5a8ceee~mv2.png)

## Крок 4: Увімкніть функцію QuickConnect ID

Synology QuickConnect ID — це унікальний ідентифікатор, який дозволяє віддалено отримати доступ до Synology NAS через інтернет без необхідності налаштовувати складні мережеві параметри, такі як переадресація портів. QuickConnect спрощує віддалений доступ, використовуючи сервери Synology для встановлення з'єднання між NAS та вашим пристроєм, таким як смартфон або комп'ютер, через QuickConnect ID.

**Як знайти або налаштувати QuickConnect ID:**

1. **Увійдіть у DSM.**
2. Перейдіть до **Панель керування > Зовнішній доступ > QuickConnect**.
3. **Увімкніть QuickConnect** та створіть або перегляньте ваш унікальний QuickConnect ID.

![QuickConnect](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_504d681f7e5848fabe0ee57fc80e770b~mv2.png)

## Крок 5: Підключення до Synology NAS на iPhone/Mac за допомогою Evermusic або Flacbox

[Evermusic](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198) та [Flacbox](https://apps.apple.com/us/app/flacbox-hi-res-music-player/id1097564256) — це додатки-музичні плеєри, розроблені для пристроїв iOS та macOS, кожен з яких пропонує унікальні функції та можливості для керування та насолоди музичною бібліотекою.

1. Відкрийте додаток Evermusic або Flacbox та перейдіть на вкладку **З'єднання**.

![З'єднання](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_d2dedf2cc0614bbe818f89e9d165411c~mv2.png)

2. Виберіть **Підключити хмарний сервіс** та оберіть **Synology Drive**.

![Synology Drive](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_eb5e8f1a02b64748a9fa23eee5df7e0d~mv2.jpg)

У вас є два варіанти підключення: **вручну** з використанням IP-адреси та порту сервера, або **автоматично** через QuickConnect ID.

### Ручне підключення

Для ручного методу вам знадобиться пряма IP-адреса та номер порту, отримані на попередніх кроках.

1. Введіть URL сервера, отриманий на кроці 2, у такому форматі: ПРОТОКОЛ://IP_АДРЕСА:НОМЕР_ПОРТУ
   - Використовуйте **порт 5000** для HTTP та **порт 5001** для HTTPS-з'єднань.

   Наприклад:
   - HTTP: [http://192.168.18.137:5000](http://192.168.18.137:5000)
   - HTTPS: [https://192.168.18.137:5001](https://192.168.18.137:5001)

2. Фактичний номер порту можна підтвердити на кроці 3 налаштування.
3. Введіть **логін** та **пароль** від Synology NAS.
4. Натисніть **Увійти**, щоб встановити з'єднання.

![Ручне підключення](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_8952ec16c92046fd81d62bff4139a97b~mv2.jpg)

### Автоматичне підключення

Для автоматичного підключення ви будете використовувати **QuickConnect ID** з кроку 4. Замість ручного введення IP-адреси та номера порту Synology NAS просто введіть **QuickConnect ID**. Додаток автоматично отримає необхідну інформацію для підключення.

Цей метод дозволяє отримати віддалений доступ до NAS навіть за межами домашньої мережі, щоб ви могли керувати файлами через інтернет без налаштування переадресації портів або статичних IP-адрес.

![Автоматичне підключення](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_68bd2a6bcbf844eea7248cbe1c1cb3fe~mv2.jpg)

## Двофакторна автентифікація

Починаючи з DSM 4.2, Synology запровадила **двоетапну верифікацію** для підвищення безпеки. Ця функція вимагає код **одноразового пароля (OTP)**, згенерований додатком-автентифікатором, на додаток до звичайних облікових даних. Якщо ви увімкнули двоетапну верифікацію, після введення імені користувача та пароля вам також потрібно буде ввести OTP для входу в сесію DSM.

Зверніть увагу, що після закінчення сесії додаток потребуватиме повторної авторизації вручну. Для повторної авторизації:

1. Перейдіть на вкладку **З'єднання** в додатку.
2. Натисніть кнопку **Більше дій** поруч із назвою сервера.
3. Виберіть **Налаштування** з меню, щоб ввести новий код автентифікації та завершити процес повторної авторизації.

Це гарантує безпеку ваших даних навіть при доступі до NAS з ненадійної мережі.

## Крок 6: Навігація та відтворення музики

1. Після підключення пристрій з'явиться у списку **Підключені пристрої**.

![Підключені пристрої](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_61446121c6b240f1a2c04ecd4c4badc0~mv2.jpg)

2. Перейдіть до папки **Music** та натисніть на будь-який аудіофайл, щоб розпочати відтворення.

![Відтворення музики](/docs/howto/how-to-connect-synology-nas-and-listen-to-music-on-your-iphone-or-mac/21260c_1702cbbfaa474d5b8c64fb9ca5e77dd4~mv2.jpg)

## Крок 7: Додайте підключену хмарну папку до музичної бібліотеки

1. Відкрийте розділ **Музична бібліотека** в додатку.
2. Виберіть **Додати музику** та оберіть **З'єднання**.
3. Виберіть підключений NAS-сервер та оберіть папку **Music**. Натисніть **Готово**.
4. Додаток просканує музичну папку та додасть підтримувані аудіофайли до музичної бібліотеки. Метадані будуть завантажені, а ваші треки будуть згруповані за альбомом, виконавцем та жанром.

Вдома чи в дорозі — насолоджуйтесь безперебійним високоякісним доступом до улюблених треків з будь-якого місця за допомогою QuickConnect. Скористайтеся гнучкістю та зручністю цих додатків і почніть легко керувати музичною колекцією на всіх своїх пристроях.

З безпечним віддаленим доступом через QuickConnect та підтримкою широкого спектру аудіоформатів ви ніколи не будете далеко від своєї музики. Тепер файли, збережені на NAS, лише на відстані одного натискання!

## FAQ

{{% details title="У чому різниця між ручним підключенням та QuickConnect?" closed="true" %}}
Ручне підключення використовує IP-адресу та порт NAS, що працює в локальній мережі. QuickConnect використовує службу ретрансляції Synology для встановлення з'єднання з будь-якого місця через інтернет без переадресації портів.
{{% /details %}}

{{% details title="Чи можу я транслювати музику з Synology NAS за межами домашньої мережі?" closed="true" %}}
Так. Увімкніть QuickConnect на Synology NAS та використовуйте QuickConnect ID в Evermusic або Flacbox для трансляції музики з будь-якого місця з підключенням до інтернету.
{{% /details %}}

{{% details title="Які аудіоформати підтримуються при трансляції з Synology NAS?" closed="true" %}}
Evermusic та Flacbox підтримують FLAC, MP3, AAC, WAV, ALAC, OGG, WMA, DSD та багато інших форматів. Усі підтримувані формати працюють при трансляції з Synology NAS.
{{% /details %}}

{{% details title="Чи потрібна двофакторна автентифікація для підключення?" closed="true" %}}
Ні, 2FA необов'язкова. Однак якщо ви увімкнули двоетапну верифікацію на Synology DSM, додаток запитає одноразовий пароль під час входу. Вам знадобиться повторна авторизація після закінчення сесії.
{{% /details %}}

{{% details title="Що краще використовувати для підключення: нативний API Synology, WebDAV чи SMB?" closed="true" %}}
Нативний API Synology з QuickConnect — найкращий вибір для віддаленого доступу. Для використання в локальній мережі SMB зазвичай є найшвидшим варіантом. WebDAV добре працює як для локального, так і для віддаленого доступу. Evermusic та Flacbox підтримують усі три протоколи.
{{% /details %}}

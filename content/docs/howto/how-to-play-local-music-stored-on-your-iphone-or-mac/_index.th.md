**สรุป:** ติดตั้ง [Evermusic](/products/evermusic) (MP3/WAV) หรือ [Flacbox](/products/flacbox) (FLAC/DSD) เปิดไฟล์เสียงหรือโฟลเดอร์ในเครื่องใดก็ได้ แล้วเริ่มเล่น เพิ่มโฟลเดอร์ไปยัง**รายการโปรด**เพื่อเข้าถึงอย่างรวดเร็ว นำเข้าแทร็กไปยังคลังเพลงของคุณ หรือสร้าง**เพลย์ลิสต์**

ในยุคดิจิทัล บริการสตรีมมิ่งได้เปลี่ยนวิธีที่เราฟังเพลง แต่มีบางอย่างที่พิเศษเกี่ยวกับเพลงที่เราสะสมมาหลายปีบน Mac, PC หรือไดรฟ์ภายนอก

เพลงในเครื่องที่รักเหล่านี้มีความหมายส่วนตัวและความทรงจำ โชคดีที่การเล่นเพลงเหล่านี้บน iPhone และ Mac ยังคงเป็นไปได้และสนุกสนาน ไม่ว่าคุณจะต้องการรำลึกความทรงจำหรือเพลิดเพลินกับคอลเลกชันพิเศษของคุณแบบออฟไลน์ คู่มือนี้จะแสดงวิธีการ

เราจะสำรวจวิธีการและเครื่องมือเพื่อเล่นเพลงในเครื่องของคุณบนอุปกรณ์ Apple อย่างราบรื่น เปลี่ยนอุปกรณ์ให้เป็นห้องแสดงคอนเสิร์ตส่วนตัวสำหรับเพลงโปรดของคุณ หากคุณพร้อมที่จะปลดล็อกศักยภาพของ iPhone และ Mac สำหรับเพลงในเครื่อง มาเริ่มการเดินทางทางดนตรีนี้ด้วยกัน!

## ขั้นตอนที่ 1: เริ่มต้นกับ Evermusic (สำหรับรูปแบบเสียงมาตรฐาน) หรือ Flacbox (สำหรับเพลงแบบไม่สูญเสียคุณภาพ)

เพื่อเริ่มต้นการเดินทางในโลกของเพลงในเครื่องบน iPhone และ Mac ให้เริ่มด้วยการติดตั้ง Evermusic (สำหรับรูปแบบเสียงมาตรฐานเช่น mp3 และ wav) หรือ Flacbox (สำหรับเพลงแบบไม่สูญเสียคุณภาพใน dsd และ flac) แอปทั้งสองมีให้ใช้งานบน iOS และ MacOS และคุณสามารถดาวน์โหลดได้ฟรี

## ขั้นตอนที่ 2: เล่นไฟล์ในเครื่องบนอุปกรณ์ของคุณ

หลังจากติดตั้งแอปพลิเคชันแล้ว ให้เปิดหน้าจอ "Local Files" และเลื่อนลงไปที่ส่วน "Files on this iPhone" จากตรงนั้น เลือก "Open files..." หากคุณต้องการเลือกไฟล์เสียงหลายไฟล์ หรือ "Open folder..." หากคุณต้องการเลือกโฟลเดอร์เพลง แอปจะสแกนเนื้อหาของโฟลเดอร์ และไฟล์เสียงทั้งหมดที่พบจะถูกเลือก

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

บนหน้าจอถัดไป แตะรายการ "Browse" บนแถบแท็บด้านล่างและเลื่อนไปที่ส่วน "Locations" เลือกรายการเมนู "On My iPhone"

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

นำทางไปยังโฟลเดอร์เพลงของคุณแล้วแตะ "Open" เพื่อยืนยันการเลือกโฟลเดอร์ปัจจุบัน

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

หรือหากคุณเลือก "Open files..." ตั้งแต่แรก ให้เลือกไฟล์ที่ต้องการเปิดแล้วแตะ "Open" เพื่อยืนยันการเลือกของคุณ

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

หลังจากแตะ "Open" เครื่องเล่นเสียงแบบเต็มหน้าจอจะปรากฏขึ้น และไฟล์จะถูกเพิ่มลงในคิวของเครื่องเล่น ไฟล์เหล่านี้จะเล่นโดยตรงจากตำแหน่งที่เลือกโดยไม่ถูกคัดลอกไปยังบันเดิลของแอปพลิเคชัน หากคุณต้องการคัดลอกไฟล์ไปยังแอปพลิเคชันทุกครั้งที่เปิด ให้ไปที่**การตั้งค่า** - "File manager" - "Copy files during opening" แล้วเลือก "Always copy files during opening"

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## ขั้นตอนที่ 3: เพิ่มโฟลเดอร์จากอุปกรณ์ของคุณไปยังรายการโปรดเพื่อเข้าถึงอย่างรวดเร็ว

คุณสามารถทำให้กระบวนการง่ายขึ้นโดยการเพิ่มโฟลเดอร์จากอุปกรณ์ของคุณไปยัง**รายการโปรด** ด้วยวิธีนี้ คุณไม่จำเป็นต้องทำซ้ำขั้นตอนข้างต้นทุกครั้งที่ต้องการเล่นเพลง ให้เปิดหน้าจอ "Local Files" และเลื่อนไปที่ส่วน "Quick access" แตะ**รายการโปรด**เพื่อเข้าถึงหน้าจอ "Favorite files"

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

แตะปุ่ม**ดำเนินการเพิ่มเติม** (แทนด้วยจุดสามจุด) ที่มุมขวาบนและเลือกรายการเมนู "Add folder"

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

แตะ "Files on this iPhone" เพื่อเปิดตัวเลือกไฟล์ของระบบ

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

นำทางไปยังโฟลเดอร์ที่มีไฟล์เสียงของคุณแล้วแตะ "Open" เพื่อยืนยันการเลือกของคุณ

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

โฟลเดอร์ของคุณจะถูกเพิ่มไปยัง "Favorite files" ทำให้คุณเข้าถึงเพลงของคุณได้อย่างรวดเร็ว

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## ขั้นตอนที่ 4: นำเข้าไฟล์ในเครื่องจาก iPhone ไปยังคลังเพลง

หากคุณต้องการทำมากกว่าแค่เล่นไฟล์เสียงบน iPhone หรือ iPad และต้องการจัดระเบียบภายในคลังเพลง ให้ทำตามขั้นตอนเหล่านี้:

1. เปิดหน้าจอ "Music library"

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_2f9f0ff8b353456399e0880c19345b60~mv2.png" alt="music library screen" width="400" >}}

2. แตะปุ่มจุดสามจุดที่มุมขวาบนแล้วเลือก "+ Add music"

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. บนหน้าจอถัดไป เลือกรายการเมนู "Local files"

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. เลื่อนลงไปที่ส่วน "Files on this iPhone" แล้วแตะ "Open files..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. เลือกไฟล์บนอุปกรณ์ของคุณแล้วแตะ "Open" เพื่อยืนยันการเลือกของคุณ

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## ขั้นตอนที่ 5: เพิ่มไฟล์ในเครื่องจาก iPhone ไปยังเพลย์ลิสต์

เพื่อเพิ่มไฟล์ในเครื่องจากอุปกรณ์ของคุณไปยัง**เพลย์ลิสต์** ให้ทำตามขั้นตอนเหล่านี้:

1. เปิดหน้าจอ**เพลย์ลิสต์**
2. แตะปุ่มเพิ่มเติมที่มุมขวาบน
3. เลือก "+ New playlist" จากเมนู

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. ป้อนชื่อสำหรับ**เพลย์ลิสต์**ใหม่ของคุณ

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. บนหน้าจอถัดไป เลือกตัวเลือก "Local files"

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. เลื่อนลงไปที่ส่วน "Files on this iPhone" แล้วแตะ "Open files..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. เลือกไฟล์เสียงที่ต้องการเพิ่มแล้วแตะ "Open" เพื่อยืนยันการเลือกของคุณ

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. ไฟล์จะถูกเพิ่มลงใน**เพลย์ลิสต์** และคุณสามารถเปลี่ยนลำดับเพลงและดำเนินการอื่นๆ โดยใช้ปุ่มเพิ่มเติมที่มุมขวาบน

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## สรุป

ด้วยขั้นตอนง่ายๆ เหล่านี้ คุณสามารถปลดล็อกศักยภาพเต็มที่ของ iPhone และ Mac ในฐานะแพลตฟอร์มที่ดีที่สุดสำหรับการเพลิดเพลินกับคอลเลกชันเพลงในเครื่องที่คุณรัก บอกลาข้อจำกัดของบริการสตรีมมิ่งและค้นพบมนต์เสน่ห์ของ**เพลย์ลิสต์**และเพลงที่คุณคัดสรรเองอีกครั้ง การเดินทางทางดนตรีของคุณอยู่แค่เพียงไม่กี่แตะ!

## FAQ

{{% details title="Evermusic และ Flacbox เล่นรูปแบบเสียงอะไรได้บ้าง?" closed="true" %}}
Evermusic เล่น MP3, WAV, AAC, M4A และรูปแบบมาตรฐานอื่นๆ Flacbox เพิ่มการรองรับ FLAC, DSD, OGG, OPUS, APE, WMA และ ALAC
{{% /details %}}

{{% details title="แอปเหล่านี้คัดลอกไฟล์ไปยังที่จัดเก็บของแอปหรือไม่?" closed="true" %}}
โดยค่าเริ่มต้น ไฟล์จะเล่นจากตำแหน่งเดิมโดยไม่ถูกคัดลอก หากต้องการเปลี่ยนพฤติกรรมนี้ ให้เปิดใช้งาน "Always copy files during opening" ใน**การตั้งค่า** > File manager
{{% /details %}}

{{% details title="ฉันสามารถจัดระเบียบเพลงในเครื่องตามศิลปินและอัลบั้มได้หรือไม่?" closed="true" %}}
ได้ นำเข้าไฟล์ไปยังคลังเพลง (ขั้นตอนที่ 4) แล้วแอปจะอ่านข้อมูลเมตาเพื่อจัดกลุ่มแทร็กตามศิลปิน อัลบั้ม แนวเพลง และผู้แต่ง
{{% /details %}}

{{% details title="ฉันจะโอนเพลงจากคอมพิวเตอร์ไปยัง iPhone ได้อย่างไร?" closed="true" %}}
ใช้ iTunes File Sharing (USB), WiFi Drive (ไร้สาย) หรือ SMB (สตรีมมิ่ง) ดูคู่มือโดยละเอียดของเรา: [โอนและเล่นไฟล์ในเครื่องบน iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/)
{{% /details %}}

{{% details title="Evermusic และ Flacbox ฟรีหรือไม่?" closed="true" %}}
ใช่ ทั้งสองแอปสามารถดาวน์โหลดได้ฟรีพร้อมคุณสมบัติหลักรวมถึงการเล่น อีควอไลเซอร์ และการสตรีมผ่านคลาวด์ เวอร์ชันฟรีมีข้อจำกัดบางประการ (จำนวน**เพลย์ลิสต์** บัญชีคลาวด์ โฟลเดอร์ออฟไลน์) การอัปเกรดเป็น Premium จะลบข้อจำกัดเหล่านี้
{{% /details %}}

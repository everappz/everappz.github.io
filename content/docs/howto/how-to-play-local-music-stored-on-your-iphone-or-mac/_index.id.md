**Ringkasan:** Instal [Evermusic](/products/evermusic) (MP3/WAV) atau [Flacbox](/products/flacbox) (FLAC/DSD), buka file audio lokal atau folder apa pun, dan mulai memutar. Tambahkan folder ke **Favorit** untuk akses cepat, impor trek ke Perpustakaan Musik Anda, atau buat daftar putar.

Di era digital, layanan streaming telah mengubah cara kita mendengarkan musik, tetapi ada sesuatu yang istimewa tentang musik yang telah kita kumpulkan selama bertahun-tahun di Mac, PC, atau drive eksternal kita.   

Lagu-lagu lokal yang berharga ini memiliki makna pribadi dan nostalgia. Untungnya, memutar musik ini di iPhone dan Mac Anda masih mungkin dan menyenangkan. Baik Anda ingin menghidupkan kembali kenangan atau menikmati koleksi unik Anda secara offline, panduan ini akan menunjukkan caranya.    

Kami akan menjelajahi metode dan alat untuk memutar musik lokal Anda dengan mulus di perangkat Apple Anda, mengubahnya menjadi ruang konser pribadi untuk melodi favorit Anda. Jika Anda siap untuk membuka potensi iPhone dan Mac Anda untuk musik lokal Anda, mari mulai perjalanan musikal ini bersama-sama!

## Langkah 1: Memulai dengan Evermusic (untuk Format Audio Standar) atau Flacbox (untuk Musik Lossless)

Untuk memulai perjalanan Anda ke dunia musik lokal di iPhone dan Mac Anda, mulailah dengan menginstal Evermusic (untuk format audio standar seperti mp3 dan wav) atau Flacbox (untuk musik lossless dalam format dsd dan flac). Kedua aplikasi ini tersedia untuk iOS dan macOS, dan Anda dapat mengunduhnya secara gratis.

## Langkah 2: Memutar File Lokal yang Ada di Perangkat Anda

Setelah Anda menginstal aplikasi, buka layar "File Lokal" dan gulir ke bawah ke bagian "File di iPhone ini". Dari sana, pilih "Buka file..." jika Anda perlu memilih beberapa file audio atau "Buka folder..." jika Anda ingin memilih folder musik. Aplikasi akan memindai konten folder, dan semua file audio yang ditemukan akan dipilih.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

Di layar berikutnya, ketuk item "Jelajahi" di bilah tab bawah dan gulir ke bagian "Lokasi". Pilih item menu "Di iPhone Saya".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

Navigasi ke folder musik Anda dan ketuk "Buka" untuk mengonfirmasi pemilihan folder saat ini.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

Atau, jika Anda memilih "Buka file..." pada awalnya, pilih file yang ingin Anda buka dan ketuk "Buka" untuk mengonfirmasi pilihan Anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

Setelah mengetuk "Buka", pemutar audio layar penuh akan muncul, dan file akan ditambahkan ke antrean pemutar. File-file ini akan diputar langsung dari lokasi yang dipilih tanpa disalin ke bundel aplikasi. Jika Anda lebih suka menyalin file ke aplikasi setiap kali Anda membukanya, navigasi ke **Pengaturan** - "Manajer file" - "Salin file saat membuka" dan pilih "Selalu salin file saat membuka".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## Langkah 3: Menambahkan Folder di Perangkat Anda ke Favorit untuk Akses Cepat

Anda dapat menyederhanakan prosesnya dengan menambahkan folder di perangkat Anda ke favorit. Dengan cara ini, Anda tidak perlu mengulangi langkah-langkah di atas setiap kali ingin memutar musik. Untuk melakukannya, buka layar "File Lokal" dan gulir ke bagian "Akses cepat". Ketuk **Favorit** untuk mengakses layar "File favorit".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

Ketuk tombol **Lebih banyak tindakan** (diwakili oleh tiga titik) di sudut kanan atas dan pilih item menu "Tambah folder".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

Ketuk "File di iPhone ini" untuk membuka pemilih file sistem.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

Navigasi ke dalam folder yang berisi file audio Anda dan ketuk "Buka" untuk mengonfirmasi pilihan Anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

Folder Anda akan ditambahkan ke "File favorit", memberikan Anda akses cepat ke musik Anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## Langkah 4: Mengimpor File Lokal di iPhone Anda ke Perpustakaan Musik

Jika Anda ingin melakukan lebih dari sekadar memutar file audio di iPhone atau iPad dan lebih suka mengaturnya dalam Perpustakaan Musik Anda, ikuti langkah-langkah berikut:

1. Buka layar "Perpustakaan Musik".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_2f9f0ff8b353456399e0880c19345b60~mv2.png" alt="music library screen" width="400" >}}

2. Ketuk tombol tiga titik di sudut kanan atas dan pilih "+ Tambah musik".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. Di layar berikutnya, pilih item menu "File lokal".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. Gulir ke bawah ke bagian "File di iPhone ini" dan ketuk "Buka file..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. Pilih file di perangkat Anda dan ketuk "Buka" untuk mengonfirmasi pilihan Anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## Langkah 5: Menambahkan File Lokal di iPhone Anda ke Daftar Putar

Untuk menambahkan file lokal di perangkat Anda ke daftar putar, ikuti langkah-langkah berikut:

1. Buka layar **Daftar Putar**.
2. Ketuk tombol lainnya di sudut kanan atas.
3. Pilih "+ Daftar putar baru" dari menu.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. Masukkan nama untuk daftar putar baru Anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. Di layar berikutnya, pilih opsi "File lokal".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. Gulir ke bawah ke bagian "File di iPhone ini" dan ketuk "Buka file..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. Pilih file audio yang ingin Anda tambahkan dan ketuk "Buka" untuk mengonfirmasi pilihan Anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. File akan ditambahkan ke daftar putar, dan Anda dapat mengubah urutan lagu dan melakukan tindakan lainnya menggunakan tombol lainnya di sudut kanan atas.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## FAQ

{{% details title="Format audio apa yang dapat diputar Evermusic dan Flacbox?" closed="true" %}}
Evermusic memutar MP3, WAV, AAC, M4A, dan format standar lainnya. Flacbox menambahkan dukungan untuk FLAC, DSD, OGG, OPUS, APE, WMA, dan ALAC.
{{% /details %}}

{{% details title="Apakah aplikasi ini menyalin file ke penyimpanan aplikasi?" closed="true" %}}
Secara default, file diputar dari lokasi aslinya tanpa disalin. Untuk mengubah perilaku ini, aktifkan "Selalu salin file saat membuka" di **Pengaturan** > Manajer file.
{{% /details %}}

{{% details title="Bisakah saya mengatur musik lokal berdasarkan artis dan album?" closed="true" %}}
Ya. Impor file ke Perpustakaan Musik (Langkah 4) dan aplikasi membaca metadata untuk mengelompokkan trek berdasarkan Artis, Album, Genre, dan Komposer.
{{% /details %}}

{{% details title="Bagaimana cara mentransfer musik dari komputer saya ke iPhone?" closed="true" %}}
Gunakan Berbagi File iTunes (USB), WiFi Drive (nirkabel), atau SMB (streaming). Lihat panduan terperinci kami: [Transfer dan Putar File Lokal di iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).
{{% /details %}}

{{% details title="Apakah Evermusic dan Flacbox gratis?" closed="true" %}}
Ya, kedua aplikasi gratis untuk diunduh dengan fitur inti termasuk pemutaran, equalizer, dan streaming cloud. Versi gratis memiliki beberapa batasan (jumlah daftar putar, akun cloud, folder offline). Upgrade ke Premium menghapus batasan ini.
{{% /details %}}

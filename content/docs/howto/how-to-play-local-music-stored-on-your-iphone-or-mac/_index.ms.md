## Langkah 1: Bermula dengan Evermusic (untuk Format Audio Standard) atau Flacbox (untuk Muzik Tanpa Rugi)

Untuk memulakan perjalanan anda ke dunia muzik tempatan di iPhone dan Mac, mulakan dengan memasang sama ada Evermusic (untuk format audio standard seperti mp3 dan wav) atau Flacbox (untuk muzik tanpa rugi dalam dsd dan flac). Kedua-dua aplikasi ini tersedia untuk iOS dan MacOS, dan anda boleh memuat turunnya secara percuma.

## Langkah 2: Memainkan Fail Tempatan yang Terdapat pada Peranti Anda

Setelah anda memasang aplikasi, buka skrin "Local Files" dan tatal ke bawah ke bahagian "Files on this iPhone". Dari situ, pilih "Open files..." jika anda perlu memilih beberapa fail audio atau "Open folder..." jika anda ingin memilih folder muzik. Aplikasi akan mengimbas kandungan folder, dan semua fail audio yang ditemui akan dipilih.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

Pada skrin seterusnya, ketik item "Browse" pada bar tab bawah dan tatal ke bahagian "Locations". Pilih item menu "On My iPhone".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

Navigasi ke folder muzik anda dan ketik "Open" untuk mengesahkan pemilihan folder semasa.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

Sebagai alternatif, jika anda memilih "Open files..." pada mulanya, pilih fail yang anda ingin buka dan ketik "Open" untuk mengesahkan pilihan anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

Selepas mengetik "Open," pemain audio skrin penuh akan muncul, dan fail akan ditambahkan ke barisan pemain. Fail-fail ini akan dimainkan terus dari lokasi yang dipilih tanpa disalin ke berkas aplikasi. Jika anda lebih suka menyalin fail ke aplikasi setiap kali anda membukanya, navigasi ke **Tetapan** - "File manager" - "Copy files during opening" dan pilih "Always copy files during opening."

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## Langkah 3: Menambah Folder pada Peranti Anda ke Kegemaran untuk Akses Pantas

Anda boleh memudahkan proses dengan menambah folder pada peranti anda ke **Kegemaran**. Dengan cara ini, anda tidak perlu mengulangi langkah-langkah di atas setiap kali anda ingin memainkan muzik. Untuk melakukan ini, buka skrin "Local Files" dan tatal ke bahagian "Quick access". Ketik **Kegemaran** untuk mengakses skrin "Favorite files".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

Ketik butang **Lebih banyak tindakan** (diwakili oleh tiga titik) di sudut kanan atas dan pilih item menu "Add folder".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

Ketik "Files on this iPhone" untuk membuka pemilih fail sistem.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

Navigasi ke dalam folder yang mengandungi fail audio anda dan ketik "Open" untuk mengesahkan pilihan anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

Folder anda akan ditambahkan ke "Favorite files," memberikan anda akses pantas ke muzik anda.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## Langkah 4: Mengimport Fail Tempatan pada iPhone Anda ke Pustaka Muzik

Jika anda ingin melakukan lebih daripada sekadar memainkan fail audio yang terdapat pada iPhone atau iPad anda dan lebih suka mengaturnya dalam pustaka Muzik anda, ikuti langkah-langkah ini:

1. Buka skrin "Music library".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_2f9f0ff8b353456399e0880c19345b60~mv2.png" alt="music library screen" width="400" >}}

2. Ketik butang tiga titik di sudut kanan atas dan pilih "+ Add music."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. Pada skrin seterusnya, pilih item menu "Local files".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. Tatal ke bawah ke bahagian "Files on this iPhone" dan ketik "Open files..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. Pilih fail yang terdapat pada peranti anda dan ketik "Open" untuk mengesahkan pilihan anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## Langkah 5: Menambah Fail Tempatan pada iPhone Anda ke Senarai Main

Untuk menambah fail tempatan pada peranti anda ke **Senarai Main**, ikuti langkah-langkah ini:

1. Buka skrin **Senarai Main**.
2. Ketik butang lagi di sudut kanan atas.
3. Pilih "+ New playlist" dari menu.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. Masukkan nama untuk **Senarai Main** baharu anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. Pada skrin seterusnya, pilih pilihan "Local files".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. Tatal ke bawah ke bahagian "Files on this iPhone" dan ketik "Open files..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. Pilih fail audio yang anda ingin tambah dan ketik "Open" untuk mengesahkan pilihan anda.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. Fail akan ditambahkan ke **Senarai Main**, dan anda boleh menukar susunan lagu dan melakukan tindakan lain menggunakan butang lagi di sudut kanan atas.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## FAQ

{{% details title="Apakah format audio yang boleh dimainkan oleh Evermusic dan Flacbox?" closed="true" %}}
Evermusic memainkan MP3, WAV, AAC, M4A, dan format standard lain. Flacbox menambah sokongan untuk FLAC, DSD, OGG, OPUS, APE, WMA, dan ALAC.
{{% /details %}}

{{% details title="Adakah aplikasi ini menyalin fail ke storan aplikasi?" closed="true" %}}
Secara lalai, fail dimainkan dari lokasi asal tanpa disalin. Untuk menukar tingkah laku ini, aktifkan "Always copy files during opening" dalam **Tetapan** > File manager.
{{% /details %}}

{{% details title="Bolehkah saya mengatur muzik tempatan mengikut artis dan album?" closed="true" %}}
Ya. Import fail ke Pustaka Muzik (Langkah 4) dan aplikasi membaca metadata untuk mengumpulkan trek mengikut Artis, Album, Genre, dan Komposer.
{{% /details %}}

{{% details title="Bagaimana cara memindahkan muzik dari komputer ke iPhone?" closed="true" %}}
Gunakan iTunes File Sharing (USB), WiFi Drive (tanpa wayar), atau SMB (penstriman). Lihat panduan terperinci kami: [Pindah dan Main Fail Tempatan di iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).
{{% /details %}}

{{% details title="Adakah Evermusic dan Flacbox percuma?" closed="true" %}}
Ya, kedua-dua aplikasi percuma untuk dimuat turun dengan ciri teras termasuk main balik, penyama, dan penstriman awan. Versi percuma mempunyai beberapa had (bilangan **Senarai Main**, akaun awan, folder luar talian). Menaik taraf ke Premium membuang had ini.
{{% /details %}}

**Περίληψη:** Εγκαταστήστε το [Evermusic](/products/evermusic) (MP3/WAV) ή το [Flacbox](/products/flacbox) (FLAC/DSD), ανοίξτε οποιοδήποτε τοπικό αρχείο ήχου ή φάκελο και ξεκινήστε την αναπαραγωγή. Προσθέστε φακέλους στα **Αγαπημένα** για γρήγορη πρόσβαση, εισαγάγετε κομμάτια στη Μουσική Βιβλιοθήκη σας ή δημιουργήστε λίστες αναπαραγωγής.

Στην ψηφιακή εποχή, οι υπηρεσίες streaming έχουν αλλάξει τον τρόπο που ακούμε μουσική, αλλά υπάρχει κάτι ιδιαίτερο στη μουσική που έχουμε συλλέξει με τα χρόνια στα Mac, PC ή εξωτερικούς δίσκους μας.   

Αυτές οι αγαπημένες τοπικές μελωδίες έχουν προσωπικό νόημα και νοσταλγία. Ευτυχώς, η αναπαραγωγή αυτής της μουσικής στο iPhone και Mac σας εξακολουθεί να είναι δυνατή και ευχάριστη. Είτε θέλετε να αναβιώσετε αναμνήσεις είτε να απολαύσετε τη μοναδική σας συλλογή εκτός σύνδεσης, αυτός ο οδηγός θα σας δείξει πώς.    

Θα εξερευνήσουμε μεθόδους και εργαλεία για να αναπαράγετε απρόσκοπτα την τοπική σας μουσική στις συσκευές Apple σας, μετατρέποντάς τες σε προσωπικές αίθουσες συναυλιών για τις αγαπημένες σας μελωδίες. Αν είστε έτοιμοι να ξεκλειδώσετε τις δυνατότητες του iPhone και Mac σας για την τοπική σας μουσική, ας ξεκινήσουμε αυτό το μουσικό ταξίδι μαζί!

## Βήμα 1: Ξεκινώντας με το Evermusic (για τυπικές μορφές ήχου) ή το Flacbox (για μουσική χωρίς απώλειες)

Για να ξεκινήσετε το ταξίδι σας στον κόσμο της τοπικής μουσικής στο iPhone και Mac σας, ξεκινήστε εγκαθιστώντας είτε το Evermusic (για τυπικές μορφές ήχου όπως mp3 και wav) είτε το Flacbox (για μουσική χωρίς απώλειες σε dsd και flac). Και οι δύο εφαρμογές είναι διαθέσιμες για iOS και macOS και μπορείτε να τις κατεβάσετε δωρεάν.

## Βήμα 2: Αναπαραγωγή τοπικών αρχείων στη συσκευή σας

Αφού εγκαταστήσετε την εφαρμογή, ανοίξτε την οθόνη "Τοπικά αρχεία" και μετακινηθείτε προς τα κάτω στην ενότητα "Αρχεία σε αυτό το iPhone". Από εκεί, επιλέξτε "Άνοιγμα αρχείων..." αν χρειάζεται να επιλέξετε πολλά αρχεία ήχου ή "Άνοιγμα φακέλου..." αν θέλετε να επιλέξετε ένα φάκελο μουσικής. Η εφαρμογή θα σαρώσει τα περιεχόμενα του φακέλου και όλα τα αρχεία ήχου που βρέθηκαν θα επιλεγούν.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

Στην επόμενη οθόνη, πατήστε το στοιχείο "Περιήγηση" στην κάτω γραμμή καρτελών και μετακινηθείτε στην ενότητα "Τοποθεσίες". Επιλέξτε το στοιχείο μενού "Στο iPhone μου".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

Πλοηγηθείτε στο φάκελο μουσικής σας και πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή του τρέχοντος φακέλου.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

Εναλλακτικά, αν επιλέξατε αρχικά "Άνοιγμα αρχείων...", επιλέξτε τα αρχεία που θέλετε να ανοίξετε και πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή σας.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

Αφού πατήσετε "Άνοιγμα", θα εμφανιστεί ένας αναπαραγωγέας ήχου πλήρους οθόνης και τα αρχεία θα προστεθούν στην ουρά αναπαραγωγής. Αυτά τα αρχεία θα αναπαραχθούν απευθείας από την επιλεγμένη τοποθεσία χωρίς να αντιγραφούν στο πακέτο της εφαρμογής. Αν προτιμάτε να αντιγράφετε τα αρχεία στην εφαρμογή κάθε φορά που τα ανοίγετε, μεταβείτε στις **Ρυθμίσεις** - "Διαχείριση αρχείων" - "Αντιγραφή αρχείων κατά το άνοιγμα" και επιλέξτε "Πάντα αντιγραφή αρχείων κατά το άνοιγμα".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## Βήμα 3: Προσθήκη φακέλου στη συσκευή σας στα Αγαπημένα για γρήγορη πρόσβαση

Μπορείτε να απλοποιήσετε τη διαδικασία προσθέτοντας ένα φάκελο στη συσκευή σας στα αγαπημένα σας. Με αυτόν τον τρόπο, δεν θα χρειάζεται να επαναλάβετε τα παραπάνω βήματα κάθε φορά που θέλετε να ακούσετε μουσική. Για να το κάνετε αυτό, ανοίξτε την οθόνη "Τοπικά αρχεία" και μεταβείτε στην ενότητα "Γρήγορη πρόσβαση". Πατήστε **Αγαπημένα** για να αποκτήσετε πρόσβαση στην οθόνη "Αγαπημένα αρχεία".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

Πατήστε το κουμπί **Περισσότερες ενέργειες** (αναπαρίσταται από τρεις τελείες) στην επάνω δεξιά γωνία και επιλέξτε το στοιχείο μενού "Προσθήκη φακέλου".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

Πατήστε "Αρχεία σε αυτό το iPhone" για να ανοίξετε τον επιλογέα αρχείων συστήματος.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

Πλοηγηθείτε μέσα στο φάκελο που περιέχει τα αρχεία ήχου σας και πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή σας.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

Ο φάκελός σας θα προστεθεί στα "Αγαπημένα αρχεία", παρέχοντάς σας γρήγορη πρόσβαση στη μουσική σας.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## Βήμα 4: Εισαγωγή τοπικών αρχείων στο iPhone σας στη Μουσική Βιβλιοθήκη

Αν θέλετε να κάνετε περισσότερα από το να αναπαράγετε απλά αρχεία ήχου στο iPhone ή iPad σας και προτιμάτε να τα οργανώσετε στη Μουσική βιβλιοθήκη σας, ακολουθήστε αυτά τα βήματα:

1. Ανοίξτε την οθόνη "Μουσική βιβλιοθήκη".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_2f9f0ff8b353456399e0880c19345b60~mv2.png" alt="music library screen" width="400" >}}

2. Πατήστε το κουμπί με τις τρεις τελείες στην επάνω δεξιά γωνία και επιλέξτε "+ Προσθήκη μουσικής".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. Στην επόμενη οθόνη, επιλέξτε το στοιχείο μενού "Τοπικά αρχεία".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. Μετακινηθείτε προς τα κάτω στην ενότητα "Αρχεία σε αυτό το iPhone" και πατήστε "Άνοιγμα αρχείων..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. Επιλέξτε τα αρχεία στη συσκευή σας και πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή σας.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## Βήμα 5: Προσθήκη τοπικών αρχείων στο iPhone σας σε λίστα αναπαραγωγής

Για να προσθέσετε τοπικά αρχεία στη συσκευή σας σε μια λίστα αναπαραγωγής, ακολουθήστε αυτά τα βήματα:

1. Ανοίξτε την οθόνη **Λίστες αναπαραγωγής**.
2. Πατήστε το κουμπί περισσότερων στην επάνω δεξιά γωνία.
3. Επιλέξτε "+ Νέα λίστα αναπαραγωγής" από το μενού.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. Εισαγάγετε ένα όνομα για τη νέα σας λίστα αναπαραγωγής.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. Στην επόμενη οθόνη, επιλέξτε την επιλογή "Τοπικά αρχεία".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. Μετακινηθείτε προς τα κάτω στην ενότητα "Αρχεία σε αυτό το iPhone" και πατήστε "Άνοιγμα αρχείων..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. Επιλέξτε τα αρχεία ήχου που θέλετε να προσθέσετε και πατήστε "Άνοιγμα" για να επιβεβαιώσετε την επιλογή σας.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. Τα αρχεία θα προστεθούν σε μια λίστα αναπαραγωγής και μπορείτε να αλλάξετε τη σειρά των τραγουδιών και να εκτελέσετε άλλες ενέργειες χρησιμοποιώντας το κουμπί περισσότερων στην επάνω δεξιά γωνία.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## Συμπέρασμα

Με αυτά τα απλά βήματα, μπορείτε να ξεκλειδώσετε το πλήρες δυναμικό του iPhone και Mac σας ως τις απόλυτες πλατφόρμες για να απολαύσετε την αγαπημένη σας τοπική μουσική συλλογή. Πείτε αντίο στους περιορισμούς των υπηρεσιών streaming και ανακαλύψτε ξανά τη μαγεία των προσωπικά επιμελημένων λιστών αναπαραγωγής και τραγουδιών σας. Το μουσικό σας ταξίδι απέχει μόνο λίγα πατήματα!

## Συχνές ερωτήσεις

{{% details title="Ποιες μορφές ήχου μπορούν να αναπαράγουν τα Evermusic και Flacbox;" closed="true" %}}
Το Evermusic αναπαράγει MP3, WAV, AAC, M4A και άλλες τυπικές μορφές. Το Flacbox προσθέτει υποστήριξη για FLAC, DSD, OGG, OPUS, APE, WMA και ALAC.
{{% /details %}}

{{% details title="Αντιγράφουν αυτές οι εφαρμογές αρχεία στον αποθηκευτικό χώρο της εφαρμογής;" closed="true" %}}
Από προεπιλογή, τα αρχεία αναπαράγονται από την αρχική τους τοποθεσία χωρίς αντιγραφή. Για να αλλάξετε αυτή τη συμπεριφορά, ενεργοποιήστε την "Πάντα αντιγραφή αρχείων κατά το άνοιγμα" στις **Ρυθμίσεις** > Διαχείριση αρχείων.
{{% /details %}}

{{% details title="Μπορώ να οργανώσω την τοπική μουσική ανά καλλιτέχνη και άλμπουμ;" closed="true" %}}
Ναι. Εισαγάγετε αρχεία στη Μουσική Βιβλιοθήκη (Βήμα 4) και η εφαρμογή διαβάζει τα μεταδεδομένα για να ομαδοποιήσει τα κομμάτια ανά Καλλιτέχνη, Άλμπουμ, Είδος και Συνθέτη.
{{% /details %}}

{{% details title="Πώς μεταφέρω μουσική από τον υπολογιστή μου στο iPhone;" closed="true" %}}
Χρησιμοποιήστε την Κοινή χρήση αρχείων iTunes (USB), WiFi Drive (ασύρματα) ή SMB (streaming). Δείτε τον αναλυτικό οδηγό μας: [Μεταφορά και αναπαραγωγή τοπικών αρχείων στο iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).
{{% /details %}}

{{% details title="Είναι δωρεάν τα Evermusic και Flacbox;" closed="true" %}}
Ναι, και οι δύο εφαρμογές είναι δωρεάν για λήψη με βασικές λειτουργίες συμπεριλαμβανομένων αναπαραγωγής, ισοσταθμιστή και cloud streaming. Οι δωρεάν εκδόσεις έχουν κάποιους περιορισμούς (αριθμός λιστών αναπαραγωγής, λογαριασμών cloud, φακέλων εκτός σύνδεσης). Η αναβάθμιση σε Premium αφαιρεί αυτούς τους περιορισμούς.
{{% /details %}}

**Riepilogo:** Installa [Evermusic](/products/evermusic) (MP3/WAV) o [Flacbox](/products/flacbox) (FLAC/DSD), apri qualsiasi file audio locale o cartella e inizia a riprodurre. Aggiungi cartelle ai **Preferiti** per un accesso rapido, importa brani nella tua Libreria Musicale o crea playlist.

Nell'era digitale, i servizi di streaming hanno cambiato il modo in cui ascoltiamo la musica, ma c'è qualcosa di speciale nella musica che abbiamo raccolto nel corso degli anni sui nostri Mac, PC o unità esterne.   

Queste amate melodie locali hanno un significato personale e nostalgia. Fortunatamente, riprodurre questa musica sul tuo iPhone e Mac è ancora possibile e piacevole. Che tu voglia rivivere i ricordi o goderti la tua collezione unica offline, questa guida ti mostrerà come fare.    

Esploreremo metodi e strumenti per riprodurre senza problemi la tua musica locale sui tuoi dispositivi Apple, trasformandoli in sale da concerto personali per le tue melodie preferite. Se sei pronto a sbloccare il potenziale del tuo iPhone e Mac per la tua musica locale, iniziamo insieme questo viaggio musicale!

## Passo 1: Iniziare con Evermusic (per formati audio standard) o Flacbox (per musica lossless)

Per iniziare il tuo viaggio nel mondo della musica locale sul tuo iPhone e Mac, inizia installando Evermusic (per formati audio standard come mp3 e wav) o Flacbox (per musica lossless in dsd e flac). Entrambe le app sono disponibili per iOS e macOS e puoi scaricarle gratuitamente.

## Passo 2: Riprodurre file locali sul tuo dispositivo

Una volta installata l'applicazione, apri la schermata "File locali" e scorri verso il basso fino alla sezione "File su questo iPhone". Da lì, scegli "Apri file..." se devi selezionare diversi file audio o "Apri cartella..." se vuoi scegliere una cartella musicale. L'app scansionerà il contenuto della cartella e tutti i file audio trovati saranno selezionati.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

Nella schermata successiva, tocca l'elemento "Sfoglia" nella barra delle schede inferiore e scorri fino alla sezione "Posizioni". Seleziona la voce di menu "Sul mio iPhone".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

Naviga fino alla tua cartella musicale e tocca "Apri" per confermare la selezione della cartella corrente.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

In alternativa, se hai scelto "Apri file..." inizialmente, seleziona i file che desideri aprire e tocca "Apri" per confermare la tua selezione.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

Dopo aver toccato "Apri", apparirà un lettore audio a schermo intero e i file saranno aggiunti alla coda del lettore. Questi file verranno riprodotti direttamente dalla posizione selezionata senza essere copiati nel bundle dell'applicazione. Se preferisci copiare i file nell'applicazione ogni volta che li apri, naviga su **Impostazioni** - "Gestore file" - "Copia file durante l'apertura" e seleziona "Copia sempre i file durante l'apertura".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## Passo 3: Aggiungere una cartella sul tuo dispositivo ai Preferiti per un accesso rapido

Puoi semplificare il processo aggiungendo una cartella sul tuo dispositivo ai tuoi preferiti. In questo modo, non dovrai ripetere i passaggi sopra ogni volta che vorrai riprodurre musica. Per farlo, apri la schermata "File locali" e scorri fino alla sezione "Accesso rapido". Tocca **Preferiti** per accedere alla schermata "File preferiti".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

Tocca il pulsante **Altre azioni** (rappresentato da tre puntini) nell'angolo in alto a destra e seleziona la voce di menu "Aggiungi cartella".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

Tocca "File su questo iPhone" per aprire il selettore file di sistema.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

Naviga all'interno della cartella contenente i tuoi file audio e tocca "Apri" per confermare la tua selezione.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

La tua cartella sarà aggiunta ai "File preferiti", offrendoti un accesso rapido alla tua musica.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## Passo 4: Importare file locali sul tuo iPhone nella Libreria Musicale

Se vuoi fare più che semplicemente riprodurre file audio sul tuo iPhone o iPad e preferisci organizzarli nella tua Libreria musicale, segui questi passaggi:

1. Apri la schermata "Libreria musicale".

2. Tocca il pulsante con i tre puntini nell'angolo in alto a destra e seleziona "+ Aggiungi musica".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. Nella schermata successiva, seleziona la voce di menu "File locali".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. Scorri verso il basso fino alla sezione "File su questo iPhone" e tocca "Apri file..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. Seleziona i file sul tuo dispositivo e tocca "Apri" per confermare la tua selezione.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## Passo 5: Aggiungere file locali sul tuo iPhone a una playlist

Per aggiungere file locali sul tuo dispositivo a una playlist, segui questi passaggi:

1. Apri la schermata **Playlist**.
2. Tocca il pulsante altro nell'angolo in alto a destra.
3. Seleziona "+ Nuova playlist" dal menu.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. Inserisci un nome per la tua nuova playlist.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. Nella schermata successiva, seleziona l'opzione "File locali".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. Scorri verso il basso fino alla sezione "File su questo iPhone" e tocca "Apri file..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. Seleziona i file audio che desideri aggiungere e tocca "Apri" per confermare la tua selezione.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. I file saranno aggiunti a una playlist e potrai cambiare l'ordine dei brani ed eseguire altre azioni usando il pulsante altro nell'angolo in alto a destra.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## FAQ

{{% details title="Quali formati audio possono riprodurre Evermusic e Flacbox?" closed="true" %}}
Evermusic riproduce MP3, WAV, AAC, M4A e altri formati standard. Flacbox aggiunge il supporto per FLAC, DSD, OGG, OPUS, APE, WMA e ALAC.
{{% /details %}}

{{% details title="Queste app copiano i file nella memoria dell'app?" closed="true" %}}
Per impostazione predefinita, i file vengono riprodotti dalla loro posizione originale senza essere copiati. Per modificare questo comportamento, abilita "Copia sempre i file durante l'apertura" in **Impostazioni** > Gestore file.
{{% /details %}}

{{% details title="Posso organizzare la musica locale per artista e album?" closed="true" %}}
Sì. Importa i file nella Libreria Musicale (Passo 4) e l'app legge i metadati per raggruppare i brani per Artista, Album, Genere e Compositore.
{{% /details %}}

{{% details title="Come trasferisco la musica dal mio computer all'iPhone?" closed="true" %}}
Usa la Condivisione File di iTunes (USB), WiFi Drive (wireless) o SMB (streaming). Consulta la nostra guida dettagliata: [Trasferire e riprodurre file locali su iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).
{{% /details %}}

{{% details title="Evermusic e Flacbox sono gratuiti?" closed="true" %}}
Sì, entrambe le app sono gratuite da scaricare con funzionalità principali tra cui riproduzione, equalizzatore e streaming cloud. Le versioni gratuite hanno alcune limitazioni (numero di playlist, account cloud, cartelle offline). L'aggiornamento a Premium rimuove queste limitazioni.
{{% /details %}}

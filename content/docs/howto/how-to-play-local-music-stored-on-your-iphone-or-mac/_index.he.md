**תקציר:** התקן את [Evermusic](/products/evermusic) (MP3/WAV) או [Flacbox](/products/flacbox) (FLAC/DSD), פתח כל קובץ אודיו מקומי או תיקייה, והתחל להשמיע. הוסף תיקיות ל**מועדפים** לגישה מהירה, ייבא רצועות לספריית המוזיקה שלך, או צור רשימות השמעה.

בעידן הדיגיטלי, שירותי הסטרימינג שינו את האופן שבו אנו מאזינים למוזיקה, אך יש משהו מיוחד במוזיקה שאספנו לאורך השנים על מחשבי Mac, PC או כוננים חיצוניים.   

המנגינות המקומיות האהובות הללו נושאות משמעות אישית ונוסטלגיה. למרבה המזל, השמעת מוזיקה זו ב-iPhone וב-Mac שלך עדיין אפשרית ומהנה. בין אם אתה רוצה לחיות מחדש זיכרונות או ליהנות מהאוסף הייחודי שלך במצב לא מקוון, מדריך זה יראה לך כיצד.    

נחקור שיטות וכלים להשמעה חלקה של המוזיקה המקומית שלך במכשירי Apple שלך, והפיכתם לאולמות קונצרטים אישיים למנגינות האהובות עליך. אם אתה מוכן לשחרר את הפוטנציאל של ה-iPhone וה-Mac שלך למוזיקה המקומית שלך, בואו נתחיל את המסע המוזיקלי הזה ביחד!

## שלב 1: תחילת העבודה עם Evermusic (לפורמטי אודיו סטנדרטיים) או Flacbox (למוזיקה ללא אובדן)

כדי להתחיל את המסע שלך בעולם המוזיקה המקומית ב-iPhone וב-Mac שלך, התחל בהתקנת Evermusic (לפורמטי אודיו סטנדרטיים כמו mp3 ו-wav) או Flacbox (למוזיקה ללא אובדן ב-dsd ו-flac). שתי האפליקציות זמינות ל-iOS ול-macOS, ותוכל להוריד אותן בחינם.

## שלב 2: השמעת קבצים מקומיים הנמצאים במכשיר שלך

לאחר שהתקנת את האפליקציה, פתח את מסך "קבצים מקומיים" וגלול למטה לסעיף "קבצים ב-iPhone זה". משם, בחר "פתח קבצים..." אם אתה צריך לבחור מספר קבצי אודיו או "פתח תיקייה..." אם אתה רוצה לבחור תיקיית מוזיקה. האפליקציה תסרוק את תוכן התיקייה, וכל קבצי האודיו שנמצאו ייבחרו.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ec30fd738d6b46eb9dba21f51c7f1f3e~mv2.png" alt="evermusic local files screen" width="400" >}}

במסך הבא, הקש על פריט "עיון" בשורת הלשוניות התחתונה וגלול לסעיף "מיקומים". בחר את פריט התפריט "ב-iPhone שלי".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_67b3f80c255b4be59588bfab8d978588~mv2.png" alt="system files picker" width="400" >}}

נווט לתיקיית המוזיקה שלך והקש "פתח" כדי לאשר את בחירת התיקייה הנוכחית.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40b43e5713954f10a2f8843abed491d7~mv2.png" alt="music folder content" width="400" >}}

לחלופין, אם בחרת "פתח קבצים..." בהתחלה, בחר את הקבצים שברצונך לפתוח והקש "פתח" כדי לאשר את הבחירה שלך.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_af4a7b0543444fe6adac9f33403c1153~mv2.png" alt="selected audio files" width="400" >}}

לאחר הקשה על "פתח", יופיע נגן אודיו במסך מלא, והקבצים יתווספו לתור הנגן. קבצים אלה יושמעו ישירות מהמיקום שנבחר מבלי להיות מועתקים לחבילת האפליקציה. אם אתה מעדיף להעתיק קבצים לאפליקציה בכל פעם שאתה פותח אותם, נווט אל **הגדרות** - "מנהל קבצים" - "העתק קבצים בעת פתיחה" ובחר "העתק תמיד קבצים בעת פתיחה".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_38eec9a335904d1899f47e89a6e78287~mv2.png" alt="audio player" width="400" >}}

## שלב 3: הוספת תיקייה הנמצאת במכשיר שלך למועדפים לגישה מהירה

תוכל לפשט את התהליך על ידי הוספת תיקייה הנמצאת במכשיר שלך למועדפים שלך. בדרך זו, לא תצטרך לחזור על השלבים לעיל בכל פעם שתרצה להשמיע מוזיקה. לשם כך, פתח את מסך "קבצים מקומיים" וגלול לסעיף "גישה מהירה". הקש על **מועדפים** כדי לגשת למסך "קבצים מועדפים".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_4ce5bbe2e4f5407f9f30fd520de2c7fa~mv2.png" alt="favorites, recents, quick access section" width="400" >}}

הקש על כפתור **עוד פעולות** (המיוצג על ידי שלוש נקודות) בפינה הימנית העליונה ובחר את פריט התפריט "הוסף תיקייה".

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_f662e350a6ac4b568913e0a8d2ab92e9~mv2.png" alt="add favorite folder" width="400" >}}

הקש על "קבצים ב-iPhone זה" כדי לפתוח את בורר הקבצים של המערכת.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5e2625989c6d4bf7ad45e747b42ef6ba~mv2.png" alt="select folder for favorites" width="400" >}}

נווט בתוך התיקייה המכילה את קבצי האודיו שלך והקש "פתח" כדי לאשר את הבחירה שלך.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_10affade5c9b4bdd95ddea26088a38cf~mv2.png" alt="files picker" width="400" >}}

התיקייה שלך תתווסף ל"קבצים מועדפים", ותספק לך גישה מהירה למוזיקה שלך.

{{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_e495cae3bfc7456fb13de1e3d13f0682~mv2.png" alt="folder added to favorites" width="400" >}}

## שלב 4: ייבוא קבצים מקומיים הנמצאים ב-iPhone שלך לספריית המוזיקה

אם אתה רוצה לעשות יותר מאשר רק להשמיע קבצי אודיו הנמצאים ב-iPhone או iPad שלך ומעדיף לארגן אותם בספריית המוזיקה שלך, פעל לפי השלבים הבאים:

1. פתח את מסך "ספריית המוזיקה".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_2f9f0ff8b353456399e0880c19345b60~mv2.png" alt="music library screen" width="400" >}}

2. הקש על כפתור שלוש הנקודות בפינה הימנית העליונה ובחר "+ הוסף מוזיקה".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_517389cdad0743b398749db38927df99~mv2.png" alt="more actions menu for music library" width="400" >}}

3. במסך הבא, בחר את פריט התפריט "קבצים מקומיים".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_ac4887b4731142c2b860605523fa1355~mv2.png" alt="local files chooser" width="400" >}}

4. גלול למטה לסעיף "קבצים ב-iPhone זה" והקש "פתח קבצים..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_382f77bc69ef4ece98ec7dcb383f9eba~mv2.png" alt="open files section" width="400" >}}

5. בחר את הקבצים הנמצאים במכשיר שלך והקש "פתח" כדי לאשר את הבחירה שלך.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_5799a1e4b52e49b1946a77970daf40c5~mv2.png" alt="selected audio files" width="400" >}}

## שלב 5: הוספת קבצים מקומיים הנמצאים ב-iPhone שלך לרשימת השמעה

כדי להוסיף קבצים מקומיים הנמצאים במכשיר שלך לרשימת השמעה, פעל לפי השלבים הבאים:

1. פתח את מסך **רשימות השמעה**.
2. הקש על כפתור עוד בפינה הימנית העליונה.
3. בחר "+ רשימת השמעה חדשה" מהתפריט.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_c7d893af6424434aa4f5ffff81170c07~mv2.png" alt="more actions for playlist" width="400" >}}

4. הזן שם לרשימת ההשמעה החדשה שלך.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_282c085da817422f90638b4312849658~mv2.png" alt="create new playlist - enter a name" width="400" >}}

5. במסך הבא, בחר באפשרות "קבצים מקומיים".

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d96d75b3ba334eb48665ae170aebffeb~mv2.png" alt="add songs to playlist - select sources" width="400" >}}

6. גלול למטה לסעיף "קבצים ב-iPhone זה" והקש "פתח קבצים..."

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_d20cf14ed00f402b819060a3b99ed61e~mv2.png" alt="local files chooser screen" width="400" >}}

7. בחר את קבצי האודיו שברצונך להוסיף והקש "פתח" כדי לאשר את הבחירה שלך.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_40bfea8b45b542d0a3f92f59203d461d~mv2.png" alt="selected audio files" width="400" >}}

8. הקבצים יתווספו לרשימת ההשמעה, ותוכל לשנות את סדר השירים ולבצע פעולות נוספות באמצעות כפתור עוד בפינה הימנית העליונה.

   {{< figure src="/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac/21260c_7a22420b2fcc43a59c745b93e7fb53e6~mv2.png" alt="playlist with local files screen" width="400" >}}

## סיכום

עם צעדים פשוטים אלה, תוכל לשחרר את הפוטנציאל המלא של ה-iPhone וה-Mac שלך כפלטפורמות האולטימטיביות ליהנות מאוסף המוזיקה המקומית האהוב שלך. היפרד ממגבלות שירותי הסטרימינג וגלה מחדש את הקסם של רשימות ההשמעה והשירים שאצרת באופן אישי. המסע המוזיקלי שלך נמצא במרחק הקשות ספורות בלבד!

## שאלות נפוצות

{{% details title="אילו פורמטי אודיו יכולים Evermusic ו-Flacbox להשמיע?" closed="true" %}}
Evermusic משמיע MP3, WAV, AAC, M4A ופורמטים סטנדרטיים אחרים. Flacbox מוסיף תמיכה ב-FLAC, DSD, OGG, OPUS, APE, WMA ו-ALAC.
{{% /details %}}

{{% details title="האם אפליקציות אלה מעתיקות קבצים לאחסון האפליקציה?" closed="true" %}}
כברירת מחדל, קבצים מושמעים מהמיקום המקורי שלהם ללא העתקה. כדי לשנות התנהגות זו, הפעל "העתק תמיד קבצים בעת פתיחה" ב**הגדרות** > מנהל קבצים.
{{% /details %}}

{{% details title="האם אוכל לארגן מוזיקה מקומית לפי אמן ואלבום?" closed="true" %}}
כן. ייבא קבצים לספריית המוזיקה (שלב 4) והאפליקציה קוראת מטא-נתונים כדי לקבץ רצועות לפי אמן, אלבום, ז'אנר ומלחין.
{{% /details %}}

{{% details title="כיצד אעביר מוזיקה מהמחשב שלי ל-iPhone?" closed="true" %}}
השתמש בשיתוף קבצי iTunes (USB), WiFi Drive (אלחוטי) או SMB (סטרימינג). ראה את המדריך המפורט שלנו: [העברה והשמעה של קבצים מקומיים ב-iPhone](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).
{{% /details %}}

{{% details title="האם Evermusic ו-Flacbox חינמיים?" closed="true" %}}
כן, שתי האפליקציות חינמיות להורדה עם תכונות ליבה הכוללות השמעה, אקולייזר וסטרימינג בענן. הגרסאות החינמיות כוללות מגבלות מסוימות (מספר רשימות השמעה, חשבונות ענן, תיקיות לא מקוונות). שדרוג ל-Premium מסיר מגבלות אלה.

**Samenvatting:** Met een promotiecode kun je een betaalde app gratis downloaden of in-app aankopen ontgrendelen. Op iOS: App Store > Accountpictogram > Wissel cadeaukaart of code in > voer code in. Op Mac: App Store > Account > Wissel cadeaukaart in > voer code in. Open vervolgens de app en herstel aankopen indien nodig.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="Promotiecode activeren op iPad" width="800" >}}

## Wat is een promotiecode?

Een promotiecode is als een gouden ticket waarmee je een betaalde app gratis kunt installeren of in-app aankopen kunt ontgrendelen zonder een cent uit te geven. Om er een te krijgen, vraag je het simpelweg aan de app-ontwikkelaar. Zo gebruik je het:

## Hoe wissel je een promotiecode in op iOS 11 of nieuwer

1. Open de App Store op je iPhone of iPad en tik op het tabblad "Vandaag" onderaan.

2. Tik rechtsboven op het "Account"-pictogram om je "Accountinstellingen" te openen.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Store hoofdpagina iPhone" caption="App Store hoofdpagina iPhone" width="400" >}}

3. Selecteer "Wissel cadeaukaart of code in" uit het menu.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Accountinstellingen scherm voor App Store iPhone" caption="Accountinstellingen scherm voor App Store iPhone" width="400" >}}

4. Kies "Je kunt je code ook handmatig invoeren."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Code inwisselen scherm voor iPhone" caption="Code inwisselen scherm voor iPhone" width="400" >}}

5. Voer je promotiecode in en tik op "Gereed" om het downloaden of activeren te starten.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Promotiecode invoeren voor iPhone" caption="Promotiecode invoeren voor iPhone" width="400" >}}

6. Na installatie verschijnt de app op je beginscherm. Tik om te openen.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Je hebt je code succesvol ingewisseld" caption="Je hebt je code succesvol ingewisseld" width="400" >}}

## Hoe wissel je een promotiecode in op iOS 10 of ouder

1. Open de App Store en ga naar het tabblad "Uitgelicht."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store scherm iPhone" caption="App Store op iOS 10" width="400" >}}

2. Scroll naar "Andere links."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store scherm iPhone Mijn Profiel" caption="Andere links sectie in iOS 10 App Store" width="400" >}}

3. Tik op "Inwisselen" en voer je Apple ID-wachtwoord in.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store wachtwoord invoeren" caption="Voer je wachtwoord in wanneer gevraagd" width="400" >}}

4. Je ziet vervolgens het inwisselscherm.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Code inwisselen scherm op iOS 10" caption="Code inwisselen scherm op iOS 10" width="400" >}}

5. Tik op "Je kunt je code ook handmatig invoeren" en voer je code in.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Handmatige code-invoer op iOS 10" caption="Handmatige code-invoer op iOS 10" width="400" >}}

6. Tik op "Inwisselen" en de app-installatie begint.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="App wordt gedownload na het inwisselen van de code" caption="App wordt gedownload na het inwisselen van de code" width="400" >}}

7. De app verschijnt op je beginscherm zodra deze is geinstalleerd.

8. Geniet van je app!

## Hoe gebruik je een promotiecode voor in-app aankopen

Volg dezelfde stappen hierboven om de code in te wisselen. Open daarna de app en navigeer naar **Instellingen > Herstel aankopen** om de premiuminhoud te activeren.

## Hoe wissel je een promotiecode in op Mac

1. Open de **App Store** op je Mac.

2. Klik op je **Account** en selecteer "Wissel cadeaukaart in."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store op Mac" caption="App Store op Mac" width="600" >}}

3. Voer je promotiecode in en klik op **Inwisselen**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Code inwisselen scherm op Mac" caption="Code inwisselen scherm op Mac" width="600" >}}

4. Open de app en ga naar **Instellingen > Herstel aankopen** om premiumfuncties te ontgrendelen.

Geniet van je gratis app of in-app upgrade!

## Veelgestelde vragen

{{% details title="Waar krijg ik een promotiecode?" closed="true" %}}
Promotiecodes worden verstrekt door app-ontwikkelaars voor recensies, weggeefacties of promoties. Neem rechtstreeks contact op met de ontwikkelaar om er een aan te vragen.
{{% /details %}}

{{% details title="Verlopen promotiecodes?" closed="true" %}}
Ja. Apple App Store-promotiecodes verlopen 28 dagen nadat ze zijn gegenereerd en kunnen slechts eenmaal worden ingewisseld.
{{% /details %}}

{{% details title="Kan ik een promotiecode in elk land gebruiken?" closed="true" %}}
Promotiecodes zijn regiospecifiek. De code moet overeenkomen met het App Store-land van je Apple ID.
{{% /details %}}

{{% details title="Hoe activeer ik in-app aankopen met een promotiecode?" closed="true" %}}
Na het inwisselen van de code in de App Store, open je de app en ga je naar Instellingen > Herstel aankopen. De premiuminhoud wordt automatisch ontgrendeld.
{{% /details %}}

{{% details title="De promotiecode zegt dat deze al is ingewisseld. Wat moet ik doen?" closed="true" %}}
Elke promotiecode kan slechts eenmaal worden gebruikt. Neem contact op met de ontwikkelaar om een nieuwe code aan te vragen.
{{% /details %}}

**En bref :** Un code promotionnel vous permet de télécharger une application payante gratuitement ou de débloquer des achats intégrés. Sur iOS : App Store > Icône du compte > Utiliser une carte cadeau ou un code > saisir le code. Sur Mac : App Store > Compte > Utiliser une carte cadeau > saisir le code. Ensuite, ouvrez l'application et restaurez les achats si nécessaire.

## Qu'est-ce qu'un code promotionnel ?

Un code promotionnel est comme un ticket en or qui vous permet d'installer une application payante gratuitement ou de débloquer des achats intégrés sans dépenser un centime. Pour en obtenir un, il suffit de le demander au développeur de l'application. Voici comment l'utiliser :

## Comment utiliser un code promotionnel sur iOS 11 ou version ultérieure

1. Ouvrez l'App Store sur votre iPhone ou iPad et appuyez sur l'onglet « Aujourd'hui » en bas de l'écran.

2. Dans le coin supérieur droit, appuyez sur l'icône « Compte » pour accéder aux « Réglages du compte ».

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Page principale de l'App Store sur iPhone" caption="Page principale de l'App Store sur iPhone" width="400" >}}

3. Sélectionnez « Utiliser une carte cadeau ou un code » dans le menu.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Écran des réglages du compte de l'App Store sur iPhone" caption="Écran des réglages du compte de l'App Store sur iPhone" width="400" >}}

4. Choisissez « Vous pouvez également saisir votre code manuellement ».

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Écran de saisie du code sur iPhone" caption="Écran de saisie du code sur iPhone" width="400" >}}

5. Saisissez votre code promotionnel et appuyez sur « Terminé » pour lancer le téléchargement ou l'activation.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Saisir le code promotionnel sur iPhone" caption="Saisir le code promotionnel sur iPhone" width="400" >}}

6. Une fois installée, l'application apparaîtra sur votre écran d'accueil. Appuyez pour l'ouvrir.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Vous avez utilisé votre code avec succès" caption="Vous avez utilisé votre code avec succès" width="400" >}}

## Comment utiliser un code promotionnel sur iOS 10 ou version antérieure

1. Ouvrez l'App Store et accédez à l'onglet « Sélection ».

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="Écran de l'App Store sur iOS 10 iPhone" caption="App Store sur iOS 10" width="400" >}}

2. Faites défiler jusqu'à « Autres liens ».

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="Écran de l'App Store sur iOS 10 iPhone Mon profil" caption="Section Autres liens dans l'App Store sur iOS 10" width="400" >}}

3. Appuyez sur « Utiliser » et saisissez votre mot de passe Apple ID.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="Saisie du mot de passe dans l'App Store sur iOS 10" caption="Saisir le mot de passe lorsque demandé" width="400" >}}

4. Vous verrez ensuite l'écran d'utilisation du code.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Écran d'utilisation du code sur iOS 10" caption="Écran d'utilisation du code sur iOS 10" width="400" >}}

5. Appuyez sur « Vous pouvez également saisir votre code manuellement » et entrez votre code.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Saisie manuelle du code sur iOS 10" caption="Saisie manuelle du code sur iOS 10" width="400" >}}

6. Appuyez sur « Utiliser » et l'installation de l'application commencera.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Téléchargement de l'application après utilisation du code" caption="Téléchargement de l'application après utilisation du code" width="400" >}}

7. L'application apparaîtra sur votre écran d'accueil une fois installée.

8. Profitez de votre application !

## Comment utiliser un code promotionnel pour les achats intégrés

Suivez les mêmes étapes ci-dessus pour utiliser le code. Ensuite, ouvrez l'application et accédez à **Paramètres > Restaurer les achats** pour activer le contenu premium.

## Comment utiliser un code promotionnel sur Mac

1. Ouvrez l'**App Store** sur votre Mac.

2. Cliquez sur votre **Compte** et sélectionnez « Utiliser une carte cadeau ».

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store sur Mac" caption="App Store sur Mac" width="600" >}}

3. Saisissez votre code promotionnel et cliquez sur **Utiliser**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Écran d'utilisation du code sur Mac" caption="Écran d'utilisation du code sur Mac" width="600" >}}

4. Ouvrez l'application et accédez à **Paramètres > Restaurer les achats** pour débloquer les fonctionnalités premium.

Profitez de votre application gratuite ou de votre mise à niveau intégrée !

## Questions fréquemment posées

{{% details title="Où puis-je obtenir un code promotionnel ?" closed="true" %}}
Les codes promotionnels sont fournis par les développeurs d'applications pour des évaluations, des concours ou des promotions. Contactez directement le développeur pour en demander un.
{{% /details %}}

{{% details title="Les codes promotionnels expirent-ils ?" closed="true" %}}
Oui. Les codes promotionnels de l'App Store expirent 28 jours après leur génération et ne peuvent être utilisés qu'une seule fois.
{{% /details %}}

{{% details title="Puis-je utiliser un code promotionnel dans n'importe quel pays ?" closed="true" %}}
Les codes promotionnels sont spécifiques à une région. Le code doit correspondre au pays de l'App Store de votre Apple ID.
{{% /details %}}

{{% details title="Comment activer les achats intégrés avec un code promotionnel ?" closed="true" %}}
Après avoir utilisé le code dans l'App Store, ouvrez l'application et accédez à Paramètres > Restaurer les achats. Le contenu premium sera débloqué automatiquement.
{{% /details %}}

{{% details title="Le code promotionnel indique qu'il a déjà été utilisé. Que dois-je faire ?" closed="true" %}}
Chaque code promotionnel ne peut être utilisé qu'une seule fois. Contactez le développeur pour demander un nouveau code.
{{% /details %}}

**Resumo:** Um código promocional permite baixar um aplicativo pago gratuitamente ou desbloquear compras no aplicativo. No iOS: App Store > ícone da Conta > Resgatar cartão-presente ou código > insira o código. No Mac: App Store > Conta > Resgatar cartão-presente > insira o código. Em seguida, abra o aplicativo e restaure as compras, se necessário.

## O que é um código promocional?

Um código promocional é como um bilhete dourado que permite instalar um aplicativo pago gratuitamente ou desbloquear compras no aplicativo sem gastar nada. Para obter um, basta pedir ao desenvolvedor do aplicativo. Veja como usá-lo:

## Como resgatar um código promocional no iOS 11 ou posterior

1. Abra a App Store no seu iPhone ou iPad e toque na aba "Hoje" na parte inferior.

2. No canto superior direito, toque no ícone "Conta" para acessar as "Configurações da conta".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Página principal da App Store iPhone" caption="Página principal da App Store iPhone" width="400" >}}

3. Selecione "Resgatar cartão-presente ou código" no menu.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Tela de configurações da conta da App Store iPhone" caption="Tela de configurações da conta da App Store iPhone" width="400" >}}

4. Escolha "Você também pode inserir seu código manualmente."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Tela de resgate de código para iPhone" caption="Tela de resgate de código para iPhone" width="400" >}}

5. Insira seu código promocional e toque em "Concluído" para iniciar o download ou a ativação.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Inserir código promocional para iPhone" caption="Inserir código promocional para iPhone" width="400" >}}

6. Após a instalação, o aplicativo aparecerá na tela inicial. Toque para abrir.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Você resgatou seu código com sucesso" caption="Você resgatou seu código com sucesso" width="400" >}}

## Como resgatar um código promocional no iOS 10 ou anterior

1. Abra a App Store e vá para a aba "Destaques".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="Tela da App Store iOS 10 iPhone" caption="App Store no iOS 10" width="400" >}}

2. Role até "Outros links."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="Tela da App Store iOS 10 iPhone Meu perfil" caption="Seção Outros links na App Store do iOS 10" width="400" >}}

3. Toque em "Resgatar" e insira a senha do seu Apple ID.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="App Store iOS 10 inserir senha" caption="Insira a senha quando solicitado" width="400" >}}

4. Em seguida, você verá a tela de resgate.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Tela de resgate de código no iOS 10" caption="Tela de resgate de código no iOS 10" width="400" >}}

5. Toque em "Você também pode inserir seu código manualmente" e digite seu código.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Inserção manual de código no iOS 10" caption="Inserção manual de código no iOS 10" width="400" >}}

6. Toque em "Resgatar" e a instalação do aplicativo começará.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Aplicativo sendo baixado após resgate do código" caption="Aplicativo sendo baixado após resgate do código" width="400" >}}

7. O aplicativo aparecerá na tela inicial após a instalação.

8. Aproveite seu aplicativo!

## Como usar um código promocional para compras no aplicativo

Siga os mesmos passos acima para resgatar o código. Depois, abra o aplicativo e navegue até **Configurações > Restaurar compras** para ativar o conteúdo premium.

## Como resgatar um código promocional no Mac

1. Abra a **App Store** no seu Mac.

2. Clique na sua **Conta** e selecione "Resgatar cartão-presente."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store no Mac" caption="App Store no Mac" width="600" >}}

3. Insira seu código promocional e clique em **Resgatar**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Tela de resgate de código no Mac" caption="Tela de resgate de código no Mac" width="600" >}}

4. Abra o aplicativo e vá para **Configurações > Restaurar compras** para desbloquear recursos premium.

Aproveite seu aplicativo grátis ou atualização no aplicativo!

## Perguntas frequentes

{{% details title="Onde consigo um código promocional?" closed="true" %}}
Códigos promocionais são fornecidos por desenvolvedores de aplicativos para avaliações, sorteios ou promoções. Entre em contato diretamente com o desenvolvedor para solicitar um.
{{% /details %}}

{{% details title="Os códigos promocionais expiram?" closed="true" %}}
Sim. Os códigos promocionais da Apple App Store expiram 28 dias após serem gerados e só podem ser resgatados uma vez.
{{% /details %}}

{{% details title="Posso usar um código promocional em qualquer país?" closed="true" %}}
Códigos promocionais são específicos por região. O código deve corresponder ao país da App Store do seu Apple ID.
{{% /details %}}

{{% details title="Como ativo compras no aplicativo com um código promocional?" closed="true" %}}
Após resgatar o código na App Store, abra o aplicativo e vá para Configurações > Restaurar compras. O conteúdo premium será desbloqueado automaticamente.
{{% /details %}}

{{% details title="O código promocional diz que já foi resgatado. O que devo fazer?" closed="true" %}}
Cada código promocional só pode ser usado uma vez. Entre em contato com o desenvolvedor para solicitar um novo código.
{{% /details %}}

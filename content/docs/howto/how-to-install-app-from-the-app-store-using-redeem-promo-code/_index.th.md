**สรุปสั้น:** รหัสโปรโมชันช่วยให้คุณดาวน์โหลดแอปแบบเสียเงินได้ฟรีหรือปลดล็อกการซื้อในแอป บน iOS: App Store > ไอคอนบัญชี > แลกบัตรของขวัญหรือรหัส > ป้อนรหัส บน Mac: App Store > บัญชี > แลกบัตรของขวัญ > ป้อนรหัส จากนั้นเปิดแอปและกู้คืนการซื้อหากจำเป็น

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="เปิดใช้งานรหัสโปรโมชันบน iPad" width="800" >}}

## รหัสโปรโมชันคืออะไร?

รหัสโปรโมชันเปรียบเสมือนตั๋วทองที่ช่วยให้คุณติดตั้งแอปแบบเสียเงินได้ฟรีหรือปลดล็อกการซื้อในแอปโดยไม่ต้องเสียเงินแม้แต่บาทเดียว เพื่อรับรหัส คุณเพียงแค่ขอจากนักพัฒนาแอป นี่คือวิธีใช้งาน:

## วิธีแลกรหัสโปรโมชันบน iOS 11 หรือใหม่กว่า

1. เปิด App Store บน iPhone หรือ iPad ของคุณแล้วแตะแท็บ "วันนี้" ที่ด้านล่าง

2. ที่มุมขวาบน แตะไอคอน "บัญชี" เพื่อเข้าถึง "การตั้งค่าบัญชี"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="หน้าหลัก App Store iPhone" caption="หน้าหลัก App Store iPhone" width="400" >}}

3. เลือก "แลกบัตรของขวัญหรือรหัส" จากเมนู

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="หน้าจอการตั้งค่าบัญชี App Store iPhone" caption="หน้าจอการตั้งค่าบัญชี App Store iPhone" width="400" >}}

4. เลือก "คุณยังสามารถป้อนรหัสด้วยตนเอง"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="หน้าจอแลกรหัสสำหรับ iPhone" caption="หน้าจอแลกรหัสสำหรับ iPhone" width="400" >}}

5. ป้อนรหัสโปรโมชันของคุณแล้วแตะ "เสร็จสิ้น" เพื่อเริ่มดาวน์โหลดหรือเปิดใช้งาน

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="ป้อนรหัสโปรโมชันสำหรับ iPhone" caption="ป้อนรหัสโปรโมชันสำหรับ iPhone" width="400" >}}

6. เมื่อติดตั้งแล้ว แอปจะปรากฏบนหน้าจอหลัก แตะเพื่อเปิด

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="คุณแลกรหัสสำเร็จแล้ว" caption="คุณแลกรหัสสำเร็จแล้ว" width="400" >}}

## วิธีแลกรหัสโปรโมชันบน iOS 10 หรือเก่ากว่า

1. เปิด App Store แล้วไปที่แท็บ "แนะนำ"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="หน้าจอ App Store iOS 10 iPhone" caption="App Store บน iOS 10" width="400" >}}

2. เลื่อนลงไปที่ "ลิงก์อื่นๆ"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="หน้าจอ App Store iOS 10 iPhone โปรไฟล์ของฉัน" caption="ส่วนลิงก์อื่นๆ ใน App Store iOS 10" width="400" >}}

3. แตะ "แลก" แล้วป้อนรหัสผ่าน Apple ID ของคุณ

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="App Store iOS 10 ป้อนรหัสผ่าน" caption="ป้อนรหัสผ่านเมื่อได้รับแจ้ง" width="400" >}}

4. จากนั้นคุณจะเห็นหน้าจอแลกรหัส

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="หน้าจอแลกรหัสบน iOS 10" caption="หน้าจอแลกรหัสบน iOS 10" width="400" >}}

5. แตะ "คุณยังสามารถป้อนรหัสด้วยตนเอง" แล้วป้อนรหัสของคุณ

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="การป้อนรหัสด้วยตนเองบน iOS 10" caption="การป้อนรหัสด้วยตนเองบน iOS 10" width="400" >}}

6. แตะ "แลก" แล้วการติดตั้งแอปจะเริ่มต้น

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="กำลังดาวน์โหลดแอปหลังจากแลกรหัส" caption="กำลังดาวน์โหลดแอปหลังจากแลกรหัส" width="400" >}}

7. แอปจะปรากฏบนหน้าจอหลักเมื่อติดตั้งเสร็จ

8. เพลิดเพลินกับแอปของคุณ!

## วิธีใช้รหัสโปรโมชันสำหรับการซื้อในแอป

ทำตามขั้นตอนเดียวกันข้างต้นเพื่อแลกรหัส จากนั้นเปิดแอปและไปที่ **การตั้งค่า > กู้คืนการซื้อ** เพื่อเปิดใช้งานเนื้อหาพรีเมียม

## วิธีแลกรหัสโปรโมชันบน Mac

1. เปิด **App Store** บน Mac ของคุณ

2. คลิกที่ **บัญชี** ของคุณแล้วเลือก "แลกบัตรของขวัญ"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store บน Mac" caption="App Store บน Mac" width="600" >}}

3. ป้อนรหัสโปรโมชันของคุณแล้วคลิก **แลก**

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="หน้าจอแลกรหัสบน Mac" caption="หน้าจอแลกรหัสบน Mac" width="600" >}}

4. เปิดแอปและไปที่ **การตั้งค่า > กู้คืนการซื้อ** เพื่อปลดล็อกคุณสมบัติพรีเมียม

เพลิดเพลินกับแอปฟรีหรือการอัปเกรดในแอปของคุณ!

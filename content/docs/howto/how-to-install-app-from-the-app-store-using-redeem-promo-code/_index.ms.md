**Ringkasan:** Kod promo membolehkan anda memuat turun aplikasi berbayar secara percuma atau membuka kunci pembelian dalam aplikasi. Pada iOS: App Store > Ikon Akaun > Tebus Kad Hadiah atau Kod > masukkan kod. Pada Mac: App Store > Akaun > Tebus Kad Hadiah > masukkan kod. Kemudian buka aplikasi dan pulihkan pembelian jika perlu.

## Apakah Kod Promo?

Kod Promo adalah seperti tiket emas yang membolehkan anda memasang aplikasi berbayar secara percuma atau membuka kunci pembelian dalam aplikasi tanpa mengeluarkan sebarang wang. Untuk mendapatkannya, anda hanya perlu meminta pembangun aplikasi. Berikut adalah cara menggunakannya:

## Cara Menebus Kod Promo pada iOS 11 atau Lebih Baharu

1. Buka App Store pada iPhone atau iPad anda dan ketik tab "Hari Ini" di bahagian bawah.

2. Di sudut kanan atas, ketik ikon "Akaun" untuk mengakses "Tetapan Akaun" anda.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Halaman Utama App Store iPhone" caption="Halaman Utama App Store iPhone" width="400" >}}

3. Pilih "Tebus Kad Hadiah atau Kod" dari menu.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Skrin Tetapan Akaun untuk App Store iPhone" caption="Skrin Tetapan Akaun untuk App Store iPhone" width="400" >}}

4. Pilih "Anda juga boleh memasukkan kod anda secara manual."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Skrin Tebus Kod untuk iPhone" caption="Skrin Tebus Kod untuk iPhone" width="400" >}}

5. Masukkan kod promo anda dan ketik "Selesai" untuk mula memuat turun atau mengaktifkan.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Masukkan Kod Promo Untuk iPhone" caption="Masukkan Kod Promo Untuk iPhone" width="400" >}}

6. Setelah dipasang, aplikasi akan muncul di Skrin Utama anda. Ketik untuk membuka.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Anda telah berjaya menebus kod anda" caption="Anda telah berjaya menebus kod anda" width="400" >}}

## Cara Menebus Kod Promo pada iOS 10 atau Lebih Lama

1. Buka App Store dan pergi ke tab "Pilihan."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="Skrin App Store iOS 10 iPhone" caption="App Store pada iOS 10" width="400" >}}

2. Tatal ke "Pautan lain."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="Skrin App Store iOS 10 iPhone Profil Saya" caption="Bahagian Pautan Lain di App Store iOS 10" width="400" >}}

3. Ketik "Tebus" dan masukkan kata laluan Apple ID anda.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="App Store iOS 10 Masukkan Kata Laluan" caption="Masukkan kata laluan apabila diminta" width="400" >}}

4. Anda akan melihat skrin Tebus.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Skrin Tebus Kod pada iOS 10" caption="Skrin Tebus Kod pada iOS 10" width="400" >}}

5. Ketik "Anda juga boleh memasukkan kod anda secara manual" dan masukkan kod anda.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Kemasukan Kod Manual pada iOS 10" caption="Kemasukan Kod Manual pada iOS 10" width="400" >}}

6. Ketik "Tebus" dan pemasangan aplikasi akan bermula.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Aplikasi dimuat turun selepas menebus kod" caption="Aplikasi dimuat turun selepas menebus kod" width="400" >}}

7. Aplikasi akan muncul di Skrin Utama anda setelah dipasang.

8. Nikmati aplikasi anda!

## Cara Menggunakan Kod Promo untuk Pembelian Dalam Aplikasi

Ikuti langkah yang sama di atas untuk menebus kod. Selepas itu, buka aplikasi dan navigasi ke **Tetapan > Pulihkan Pembelian** untuk mengaktifkan kandungan premium.

## Cara Menebus Kod Promo pada Mac

1. Buka **App Store** pada Mac anda.

2. Klik pada **Akaun** anda dan pilih "Tebus Kad Hadiah."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store pada Mac" caption="App Store pada Mac" width="600" >}}

3. Masukkan kod promo anda dan klik **Tebus**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Skrin tebus kod pada Mac" caption="Skrin tebus kod pada Mac" width="600" >}}

4. Buka aplikasi dan pergi ke **Tetapan > Pulihkan Pembelian** untuk membuka kunci ciri premium.

Nikmati aplikasi percuma atau peningkatan dalam aplikasi anda!

## Soalan Lazim

{{% details title="Di mana saya boleh mendapatkan kod promo?" closed="true" %}}
Kod promo disediakan oleh pembangun aplikasi untuk ulasan, pemberian hadiah, atau promosi. Hubungi pembangun secara terus untuk meminta satu.
{{% /details %}}

{{% details title="Adakah kod promo mempunyai tarikh luput?" closed="true" %}}
Ya. Kod promo Apple App Store tamat tempoh 28 hari selepas ia dijana dan hanya boleh ditebus sekali.
{{% /details %}}

{{% details title="Bolehkah saya menggunakan kod promo di mana-mana negara?" closed="true" %}}
Kod promo adalah khusus untuk rantau tertentu. Kod mesti sepadan dengan negara App Store Apple ID anda.
{{% /details %}}

{{% details title="Bagaimana cara mengaktifkan pembelian dalam aplikasi dengan kod promo?" closed="true" %}}
Selepas menebus kod di App Store, buka aplikasi dan pergi ke Tetapan > Pulihkan Pembelian. Kandungan premium akan dibuka kunci secara automatik.
{{% /details %}}

{{% details title="Kod promo mengatakan ia sudah ditebus. Apa yang perlu saya lakukan?" closed="true" %}}
Setiap kod promo hanya boleh digunakan sekali. Hubungi pembangun untuk meminta kod baharu.
{{% /details %}}

**Resumen:** Un código promocional te permite descargar una app de pago gratis o desbloquear compras dentro de la app. En iOS: App Store > Icono de cuenta > Canjear tarjeta regalo o código > introducir código. En Mac: App Store > Cuenta > Canjear tarjeta regalo > introducir código. Luego abre la app y restaura las compras si es necesario.

## ¿Qué es un código promocional?

Un código promocional es como un boleto dorado que te permite instalar una app de pago gratis o desbloquear compras dentro de la app sin gastar un centavo. Para obtener uno, simplemente contacta al desarrollador de la app. Así es cómo usarlo:

## Cómo canjear un código promocional en iOS 11 o posterior

1. Abre el App Store en tu iPhone o iPad y toca la pestaña "Hoy" en la parte inferior.

2. En la esquina superior derecha, toca el icono "Cuenta" para acceder a la "Configuración de cuenta."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Página principal del App Store iPhone" caption="Página principal del App Store iPhone" width="400" >}}

3. Selecciona "Canjear tarjeta regalo o código" del menú.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Pantalla de configuración de cuenta del App Store iPhone" caption="Pantalla de configuración de cuenta del App Store iPhone" width="400" >}}

4. Elige "También puedes introducir tu código manualmente."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Pantalla de canjear código para iPhone" caption="Pantalla de canjear código para iPhone" width="400" >}}

5. Introduce tu código promocional y toca "Hecho" para comenzar la descarga o activación.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Introducir código promocional para iPhone" caption="Introducir código promocional para iPhone" width="400" >}}

6. Una vez instalada, la app aparecerá en tu pantalla de inicio. Toca para abrir.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Has canjeado tu código exitosamente" caption="Has canjeado tu código exitosamente" width="400" >}}

## Cómo canjear un código promocional en iOS 10 o anterior

1. Abre el App Store y ve a la pestaña "Destacados."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="Pantalla del App Store iOS 10 iPhone" caption="App Store en iOS 10" width="400" >}}

2. Desplázate hasta "Otros enlaces."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="Pantalla del App Store iOS 10 iPhone Mi perfil" caption="Sección de otros enlaces en el App Store de iOS 10" width="400" >}}

3. Toca "Canjear" e introduce tu contraseña de Apple ID.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="App Store iOS 10 introducir contraseña" caption="Introduce la contraseña cuando se te solicite" width="400" >}}

4. Luego verás la pantalla de canje.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Pantalla de canjear código en iOS 10" caption="Pantalla de canjear código en iOS 10" width="400" >}}

5. Toca "También puedes introducir tu código manualmente" e introduce tu código.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Entrada manual de código en iOS 10" caption="Entrada manual de código en iOS 10" width="400" >}}

6. Toca "Canjear" y comenzará la instalación de la app.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="App descargándose después de canjear el código" caption="App descargándose después de canjear el código" width="400" >}}

7. La app aparecerá en tu pantalla de inicio una vez instalada.

8. ¡Disfruta de tu app!

## Cómo usar un código promocional para compras dentro de la app

Sigue los mismos pasos anteriores para canjear el código. Después, abre la app y navega a **Ajustes > Restaurar compras** para activar el contenido premium.

## Cómo canjear un código promocional en Mac

1. Abre el **App Store** en tu Mac.

2. Haz clic en tu **Cuenta** y selecciona "Canjear tarjeta regalo."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store en Mac" caption="App Store en Mac" width="600" >}}

3. Introduce tu código promocional y haz clic en **Canjear**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Pantalla de canjear código en Mac" caption="Pantalla de canjear código en Mac" width="600" >}}

4. Abre la app y ve a **Ajustes > Restaurar compras** para desbloquear las funciones premium.

¡Disfruta de tu app gratuita o de la mejora dentro de la app!

## Preguntas frecuentes

{{% details title="¿Dónde consigo un código promocional?" closed="true" %}}
Los códigos promocionales son proporcionados por los desarrolladores de apps para reseñas, sorteos o promociones. Contacta directamente al desarrollador para solicitar uno.
{{% /details %}}

{{% details title="¿Los códigos promocionales caducan?" closed="true" %}}
Sí. Los códigos promocionales del Apple App Store caducan 28 días después de su generación y solo pueden canjearse una vez.
{{% /details %}}

{{% details title="¿Puedo usar un código promocional en cualquier país?" closed="true" %}}
Los códigos promocionales son específicos de cada región. El código debe coincidir con el país del App Store de tu Apple ID.
{{% /details %}}

{{% details title="¿Cómo activo las compras dentro de la app con un código promocional?" closed="true" %}}
Después de canjear el código en el App Store, abre la app y ve a Ajustes > Restaurar compras. El contenido premium se desbloqueará automáticamente.
{{% /details %}}

{{% details title="El código promocional dice que ya ha sido canjeado. ¿Qué debo hacer?" closed="true" %}}
Cada código promocional solo puede usarse una vez. Contacta al desarrollador para solicitar un nuevo código.
{{% /details %}}

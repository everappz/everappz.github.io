**要約:** プロモコードを使用すると、有料アプリを無料でダウンロードしたり、アプリ内課金をアンロックしたりできます。iOSの場合：App Store > アカウントアイコン > ギフトカードまたはコードを使う > コードを入力。Macの場合：App Store > アカウント > ギフトカードを使う > コードを入力。その後、アプリを開き、必要に応じて購入を復元してください。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="iPadでプロモコードを有効にする" width="800" >}}

## プロモコードとは？

プロモコードは、有料アプリを無料でインストールしたり、一銭も使わずにアプリ内課金をアンロックしたりできるゴールデンチケットのようなものです。入手するには、アプリ開発者に依頼するだけです。使い方は以下の通りです：

## iOS 11以降でプロモコードを引き換える方法

1. iPhoneまたはiPadでApp Storeを開き、下部の「Today」タブをタップします。

2. 右上の「アカウント」アイコンをタップして「アカウント設定」にアクセスします。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Storeメインページ iPhone" caption="App Storeメインページ iPhone" width="400" >}}

3. メニューから「ギフトカードまたはコードを使う」を選択します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="App Store iPhoneのアカウント設定画面" caption="App Store iPhoneのアカウント設定画面" width="400" >}}

4. 「コードを手動で入力することもできます」を選択します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="iPhoneのコード引き換え画面" caption="iPhoneのコード引き換え画面" width="400" >}}

5. プロモコードを入力し、「完了」をタップしてダウンロードまたは有効化を開始します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="iPhoneでプロモコードを入力" caption="iPhoneでプロモコードを入力" width="400" >}}

6. インストール後、アプリはホーム画面に表示されます。タップして開きます。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="コードの引き換えに成功しました" caption="コードの引き換えに成功しました" width="400" >}}

## iOS 10以前でプロモコードを引き換える方法

1. App Storeを開き、「おすすめ」タブに移動します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store画面 iPhone" caption="iOS 10のApp Store" width="400" >}}

2. 「その他のリンク」までスクロールします。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store画面 iPhone マイプロフィール" caption="iOS 10 App Storeのその他のリンクセクション" width="400" >}}

3. 「コードを使う」をタップし、Apple IDのパスワードを入力します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Storeパスワード入力" caption="求められたらパスワードを入力" width="400" >}}

4. 引き換え画面が表示されます。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="iOS 10のコード引き換え画面" caption="iOS 10のコード引き換え画面" width="400" >}}

5. 「コードを手動で入力することもできます」をタップし、コードを入力します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="iOS 10での手動コード入力" caption="iOS 10での手動コード入力" width="400" >}}

6. 「コードを使う」をタップすると、アプリのインストールが開始されます。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="コード引き換え後にアプリをダウンロード中" caption="コード引き換え後にアプリをダウンロード中" width="400" >}}

7. インストール後、アプリはホーム画面に表示されます。

8. アプリをお楽しみください！

## アプリ内課金にプロモコードを使用する方法

上記と同じ手順に従ってコードを引き換えます。その後、アプリを開き、**設定 > 購入を復元**に移動してプレミアムコンテンツを有効にします。

## Macでプロモコードを引き換える方法

1. Macで**App Store**を開きます。

2. **アカウント**をクリックし、「ギフトカードを使う」を選択します。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="MacのApp Store" caption="MacのApp Store" width="600" >}}

3. プロモコードを入力し、**コードを使う**をクリックします。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Macのコード引き換え画面" caption="Macのコード引き換え画面" width="600" >}}

4. アプリを開き、**設定 > 購入を復元**に移動してプレミアム機能をアンロックします。

無料アプリまたはアプリ内アップグレードをお楽しみください！

## よくある質問

{{% details title="プロモコードはどこで入手できますか？" closed="true" %}}
プロモコードは、レビュー、プレゼント企画、またはプロモーションのためにアプリ開発者から提供されます。開発者に直接連絡してリクエストしてください。
{{% /details %}}

{{% details title="プロモコードには有効期限がありますか？" closed="true" %}}
はい。Apple App Storeのプロモコードは生成後28日で期限切れとなり、一度しか引き換えできません。
{{% /details %}}

{{% details title="どの国でもプロモコードを使用できますか？" closed="true" %}}
プロモコードは地域固有です。コードはApple IDのApp Store国と一致する必要があります。
{{% /details %}}

{{% details title="プロモコードでアプリ内課金を有効にするにはどうすればよいですか？" closed="true" %}}
App Storeでコードを引き換えた後、アプリを開いて設定 > 購入を復元に移動してください。プレミアムコンテンツが自動的にアンロックされます。
{{% /details %}}

{{% details title="プロモコードがすでに引き換え済みと表示されます。どうすればよいですか？" closed="true" %}}
各プロモコードは一度しか使用できません。新しいコードをリクエストするには、開発者に連絡してください。
{{% /details %}}

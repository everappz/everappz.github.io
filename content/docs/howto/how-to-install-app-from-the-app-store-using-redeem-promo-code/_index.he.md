**בקצרה:** קוד פרומו מאפשר לכם להוריד אפליקציה בתשלום בחינם או לפתוח רכישות בתוך האפליקציה. ב-iOS: App Store > סמל החשבון > ממש כרטיס מתנה או קוד > הזינו את הקוד. ב-Mac: App Store > חשבון > ממש כרטיס מתנה > הזינו את הקוד. לאחר מכן פתחו את האפליקציה ושחזרו רכישות במידת הצורך.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="הפעלת קוד פרומו ב-iPad" width="800" >}}

## מהו קוד פרומו?

קוד פרומו הוא כמו כרטיס זהב שמאפשר לכם להתקין אפליקציה בתשלום בחינם או לפתוח רכישות בתוך האפליקציה מבלי להוציא אגורה. כדי לקבל אחד, פשוט בקשו ממפתח האפליקציה. כך משתמשים בו:

## כיצד לממש קוד פרומו ב-iOS 11 ואילך

1. פתחו את App Store ב-iPhone או ב-iPad שלכם והקישו על הלשונית "היום" בתחתית המסך.

2. בפינה הימנית העליונה, הקישו על סמל "החשבון" כדי לגשת ל"הגדרות החשבון".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="עמוד ראשי של App Store ב-iPhone" caption="עמוד ראשי של App Store ב-iPhone" width="400" >}}

3. בחרו "ממש כרטיס מתנה או קוד" מהתפריט.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="מסך הגדרות חשבון ב-App Store ב-iPhone" caption="מסך הגדרות חשבון ב-App Store ב-iPhone" width="400" >}}

4. בחרו "ניתן גם להזין את הקוד ידנית".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="מסך מימוש קוד ב-iPhone" caption="מסך מימוש קוד ב-iPhone" width="400" >}}

5. הזינו את קוד הפרומו שלכם והקישו על "סיום" כדי להתחיל בהורדה או בהפעלה.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="הזנת קוד פרומו ב-iPhone" caption="הזנת קוד פרומו ב-iPhone" width="400" >}}

6. לאחר ההתקנה, האפליקציה תופיע במסך הבית שלכם. הקישו כדי לפתוח.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="מימשתם את הקוד בהצלחה" caption="מימשתם את הקוד בהצלחה" width="400" >}}

## כיצד לממש קוד פרומו ב-iOS 10 או גרסה קודמת

1. פתחו את App Store ועברו ללשונית "מומלצים".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="מסך App Store ב-iOS 10 iPhone" caption="App Store ב-iOS 10" width="400" >}}

2. גללו אל "קישורים נוספים".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="מסך App Store ב-iOS 10 iPhone הפרופיל שלי" caption="אזור קישורים נוספים ב-App Store ב-iOS 10" width="400" >}}

3. הקישו על "ממש" והזינו את סיסמת ה-Apple ID שלכם.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="הזנת סיסמה ב-App Store ב-iOS 10" caption="הזינו סיסמה כשתתבקשו" width="400" >}}

4. לאחר מכן תראו את מסך המימוש.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="מסך מימוש קוד ב-iOS 10" caption="מסך מימוש קוד ב-iOS 10" width="400" >}}

5. הקישו על "ניתן גם להזין את הקוד ידנית" והזינו את הקוד שלכם.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="הזנת קוד ידנית ב-iOS 10" caption="הזנת קוד ידנית ב-iOS 10" width="400" >}}

6. הקישו על "ממש" והתקנת האפליקציה תתחיל.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="האפליקציה מורדת לאחר מימוש הקוד" caption="האפליקציה מורדת לאחר מימוש הקוד" width="400" >}}

7. האפליקציה תופיע במסך הבית שלכם לאחר ההתקנה.

8. תהנו מהאפליקציה שלכם!

## כיצד להשתמש בקוד פרומו לרכישות בתוך האפליקציה

עקבו אחר אותם השלבים לעיל כדי לממש את הקוד. לאחר מכן, פתחו את האפליקציה ונווטו אל **הגדרות > שחזור רכישות** כדי להפעיל את התוכן הפרימיום.

## כיצד לממש קוד פרומו ב-Mac

1. פתחו את **App Store** ב-Mac שלכם.

2. לחצו על **החשבון** שלכם ובחרו "ממש כרטיס מתנה".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store ב-Mac" caption="App Store ב-Mac" width="600" >}}

3. הזינו את קוד הפרומו שלכם ולחצו על **ממש**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="מסך מימוש קוד ב-Mac" caption="מסך מימוש קוד ב-Mac" width="600" >}}

4. פתחו את האפליקציה ועברו אל **הגדרות > שחזור רכישות** כדי לפתוח תכונות פרימיום.

תהנו מהאפליקציה החינמית או משדרוג הרכישה בתוך האפליקציה!

## שאלות נפוצות

{{% details title="איפה אפשר להשיג קוד פרומו?" closed="true" %}}
קודי פרומו מסופקים על ידי מפתחי אפליקציות לסקירות, הגרלות או מבצעים. פנו ישירות למפתח כדי לבקש אחד.
{{% /details %}}

{{% details title="האם קודי פרומו פגים?" closed="true" %}}
כן. קודי פרומו של App Store פגים 28 ימים לאחר יצירתם וניתן לממש אותם פעם אחת בלבד.
{{% /details %}}

{{% details title="האם אפשר להשתמש בקוד פרומו בכל מדינה?" closed="true" %}}
קודי פרומו הם ספציפיים לאזור. הקוד חייב להתאים למדינת ה-App Store של ה-Apple ID שלכם.
{{% /details %}}

{{% details title="כיצד מפעילים רכישות בתוך האפליקציה עם קוד פרומו?" closed="true" %}}
לאחר מימוש הקוד ב-App Store, פתחו את האפליקציה ועברו אל הגדרות > שחזור רכישות. התוכן הפרימיום ייפתח אוטומטית.
{{% /details %}}

{{% details title="הקוד הפרומו מציין שכבר מומש. מה עלי לעשות?" closed="true" %}}
כל קוד פרומו ניתן לשימוש פעם אחת בלבד. פנו למפתח כדי לבקש קוד חדש.
{{% /details %}}

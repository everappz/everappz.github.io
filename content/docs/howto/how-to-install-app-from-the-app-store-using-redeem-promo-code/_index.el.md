**Συνοπτικά:** Ένας κωδικός προσφοράς σας επιτρέπει να κατεβάσετε μια επί πληρωμή εφαρμογή δωρεάν ή να ξεκλειδώσετε αγορές εντός εφαρμογής. Σε iOS: App Store > Εικονίδιο λογαριασμού > Εξαργύρωση δωροκάρτας ή κωδικού > εισαγάγετε τον κωδικό. Σε Mac: App Store > Λογαριασμός > Εξαργύρωση δωροκάρτας > εισαγάγετε τον κωδικό. Στη συνέχεια, ανοίξτε την εφαρμογή και επαναφέρετε τις αγορές αν χρειάζεται.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="Ενεργοποίηση κωδικού εξαργύρωσης σε iPad" width="800" >}}

## Τι είναι ένας κωδικός εξαργύρωσης;

Ένας κωδικός εξαργύρωσης είναι σαν ένα χρυσό εισιτήριο που σας επιτρέπει να εγκαταστήσετε μια επί πληρωμή εφαρμογή δωρεάν ή να ξεκλειδώσετε αγορές εντός εφαρμογής χωρίς να ξοδέψετε ούτε ένα λεπτό. Για να αποκτήσετε έναν, απλά ζητήστε τον από τον προγραμματιστή της εφαρμογής. Δείτε πώς να τον χρησιμοποιήσετε:

## Πώς να εξαργυρώσετε έναν κωδικό προσφοράς σε iOS 11 ή νεότερο

1. Ανοίξτε το App Store στο iPhone ή iPad σας και πατήστε την καρτέλα "Σήμερα" στο κάτω μέρος.

2. Στην επάνω δεξιά γωνία, πατήστε το εικονίδιο "Λογαριασμός" για να αποκτήσετε πρόσβαση στις "Ρυθμίσεις λογαριασμού."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Κύρια σελίδα App Store iPhone" caption="Κύρια σελίδα App Store iPhone" width="400" >}}

3. Επιλέξτε "Εξαργύρωση δωροκάρτας ή κωδικού" από το μενού.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Οθόνη ρυθμίσεων λογαριασμού App Store iPhone" caption="Οθόνη ρυθμίσεων λογαριασμού App Store iPhone" width="400" >}}

4. Επιλέξτε "Μπορείτε επίσης να εισαγάγετε τον κωδικό σας χειροκίνητα."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Οθόνη εξαργύρωσης κωδικού για iPhone" caption="Οθόνη εξαργύρωσης κωδικού για iPhone" width="400" >}}

5. Εισαγάγετε τον κωδικό εξαργύρωσης και πατήστε "Τέλος" για να ξεκινήσετε τη λήψη ή την ενεργοποίηση.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Εισαγωγή κωδικού εξαργύρωσης για iPhone" caption="Εισαγωγή κωδικού εξαργύρωσης για iPhone" width="400" >}}

6. Μετά την εγκατάσταση, η εφαρμογή θα εμφανιστεί στην αρχική οθόνη σας. Πατήστε για να ανοίξετε.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Εξαργυρώσατε επιτυχώς τον κωδικό σας" caption="Εξαργυρώσατε επιτυχώς τον κωδικό σας" width="400" >}}

## Πώς να εξαργυρώσετε έναν κωδικό προσφοράς σε iOS 10 ή παλαιότερο

1. Ανοίξτε το App Store και μεταβείτε στην καρτέλα "Προτεινόμενα."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 οθόνη App Store iPhone" caption="App Store σε iOS 10" width="400" >}}

2. Κάντε κύλιση στους "Άλλους συνδέσμους."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 οθόνη App Store iPhone Το προφίλ μου" caption="Ενότητα άλλων συνδέσμων στο App Store iOS 10" width="400" >}}

3. Πατήστε "Εξαργύρωση" και εισαγάγετε τον κωδικό πρόσβασης του Apple ID σας.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store εισαγωγή κωδικού πρόσβασης" caption="Εισαγάγετε τον κωδικό πρόσβασης όταν σας ζητηθεί" width="400" >}}

4. Στη συνέχεια θα δείτε την οθόνη εξαργύρωσης.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Οθόνη εξαργύρωσης κωδικού σε iOS 10" caption="Οθόνη εξαργύρωσης κωδικού σε iOS 10" width="400" >}}

5. Πατήστε "Μπορείτε επίσης να εισαγάγετε τον κωδικό σας χειροκίνητα" και εισαγάγετε τον κωδικό σας.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Χειροκίνητη εισαγωγή κωδικού σε iOS 10" caption="Χειροκίνητη εισαγωγή κωδικού σε iOS 10" width="400" >}}

6. Πατήστε "Εξαργύρωση" και η εγκατάσταση της εφαρμογής θα ξεκινήσει.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Η εφαρμογή κατεβαίνει μετά την εξαργύρωση του κωδικού" caption="Η εφαρμογή κατεβαίνει μετά την εξαργύρωση του κωδικού" width="400" >}}

7. Η εφαρμογή θα εμφανιστεί στην αρχική οθόνη σας μετά την εγκατάσταση.

8. Απολαύστε την εφαρμογή σας!

## Πώς να χρησιμοποιήσετε έναν κωδικό προσφοράς για αγορές εντός εφαρμογής

Ακολουθήστε τα ίδια βήματα παραπάνω για να εξαργυρώσετε τον κωδικό. Στη συνέχεια, ανοίξτε την εφαρμογή και μεταβείτε στις **Ρυθμίσεις > Επαναφορά αγορών** για να ενεργοποιήσετε το premium περιεχόμενο.

## Πώς να εξαργυρώσετε έναν κωδικό προσφοράς σε Mac

1. Ανοίξτε το **App Store** στο Mac σας.

2. Κάντε κλικ στον **Λογαριασμό** σας και επιλέξτε "Εξαργύρωση δωροκάρτας."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store σε Mac" caption="App Store σε Mac" width="600" >}}

3. Εισαγάγετε τον κωδικό εξαργύρωσης και κάντε κλικ στο **Εξαργύρωση**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Οθόνη εξαργύρωσης κωδικού σε Mac" caption="Οθόνη εξαργύρωσης κωδικού σε Mac" width="600" >}}

4. Ανοίξτε την εφαρμογή και μεταβείτε στις **Ρυθμίσεις > Επαναφορά αγορών** για να ξεκλειδώσετε τις premium λειτουργίες.

Απολαύστε τη δωρεάν εφαρμογή σας ή την αναβάθμιση εντός εφαρμογής!

## Συχνές ερωτήσεις

{{% details title="Πού μπορώ να βρω έναν κωδικό προσφοράς;" closed="true" %}}
Οι κωδικοί προσφοράς παρέχονται από προγραμματιστές εφαρμογών για κριτικές, κληρώσεις ή προωθητικές ενέργειες. Επικοινωνήστε απευθείας με τον προγραμματιστή για να ζητήσετε έναν.
{{% /details %}}

{{% details title="Λήγουν οι κωδικοί προσφοράς;" closed="true" %}}
Ναι. Οι κωδικοί προσφοράς του Apple App Store λήγουν 28 ημέρες μετά τη δημιουργία τους και μπορούν να εξαργυρωθούν μόνο μία φορά.
{{% /details %}}

{{% details title="Μπορώ να χρησιμοποιήσω έναν κωδικό προσφοράς σε οποιαδήποτε χώρα;" closed="true" %}}
Οι κωδικοί προσφοράς είναι συγκεκριμένοι ανά περιοχή. Ο κωδικός πρέπει να αντιστοιχεί στη χώρα του App Store του Apple ID σας.
{{% /details %}}

{{% details title="Πώς ενεργοποιώ αγορές εντός εφαρμογής με κωδικό προσφοράς;" closed="true" %}}
Μετά την εξαργύρωση του κωδικού στο App Store, ανοίξτε την εφαρμογή και μεταβείτε στις Ρυθμίσεις > Επαναφορά αγορών. Το premium περιεχόμενο θα ξεκλειδωθεί αυτόματα.
{{% /details %}}

{{% details title="Ο κωδικός προσφοράς λέει ότι έχει ήδη εξαργυρωθεί. Τι πρέπει να κάνω;" closed="true" %}}
Κάθε κωδικός προσφοράς μπορεί να χρησιμοποιηθεί μόνο μία φορά. Επικοινωνήστε με τον προγραμματιστή για να ζητήσετε νέο κωδικό.
{{% /details %}}

**簡要說明：** 促銷代碼可以讓您免費下載付費應用程式或解鎖應用程式內購買。在 iOS 上：App Store > 帳號圖示 > 兌換禮品卡或代碼 > 輸入代碼。在 Mac 上：App Store > 帳號 > 兌換禮品卡 > 輸入代碼。然後開啟應用程式，如有需要，恢復購買。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="在 iPad 上啟用兌換促銷代碼" width="800" >}}

## 什麼是兌換促銷代碼？

兌換促銷代碼就像一張金色門票，可以讓您免費安裝付費應用程式或解鎖應用程式內購買，無需花費任何費用。要取得促銷代碼，只需聯繫應用程式開發者即可。以下是使用方法：

## 如何在 iOS 11 或更新版本上兌換促銷代碼

1. 在您的 iPhone 或 iPad 上開啟 App Store，點按底部的「今天」標籤頁。

2. 在右上角，點按「帳號」圖示以存取「帳號設定」。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Store 主頁 iPhone" caption="App Store 主頁 iPhone" width="400" >}}

3. 從選單中選擇「兌換禮品卡或代碼」。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="App Store 帳號設定畫面 iPhone" caption="App Store 帳號設定畫面 iPhone" width="400" >}}

4. 選擇「您也可以手動輸入代碼。」

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="iPhone 兌換代碼畫面" caption="iPhone 兌換代碼畫面" width="400" >}}

5. 輸入您的兌換代碼，點按「完成」開始下載或啟用。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="為 iPhone 輸入兌換代碼" caption="為 iPhone 輸入兌換代碼" width="400" >}}

6. 安裝完成後，應用程式將出現在您的主畫面上。點按即可開啟。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="您已成功兌換代碼" caption="您已成功兌換代碼" width="400" >}}

## 如何在 iOS 10 或更早版本上兌換促銷代碼

1. 開啟 App Store，前往「精選」標籤頁。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store 畫面 iPhone" caption="iOS 10 上的 App Store" width="400" >}}

2. 捲動到「其他連結」。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store 畫面 iPhone 我的個人資料" caption="iOS 10 App Store 中的其他連結部分" width="400" >}}

3. 點按「兌換」並輸入您的 Apple ID 密碼。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store 輸入密碼" caption="在出現提示時輸入密碼" width="400" >}}

4. 然後您將看到兌換畫面。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="iOS 10 上的兌換代碼畫面" caption="iOS 10 上的兌換代碼畫面" width="400" >}}

5. 點按「您也可以手動輸入代碼」並輸入您的代碼。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="iOS 10 上手動輸入代碼" caption="iOS 10 上手動輸入代碼" width="400" >}}

6. 點按「兌換」，應用程式安裝將開始。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="兌換代碼後應用程式正在下載" caption="兌換代碼後應用程式正在下載" width="400" >}}

7. 應用程式安裝完成後將出現在您的主畫面上。

8. 盡情享用您的應用程式吧！

## 如何使用促銷代碼進行應用程式內購買

按照上述相同步驟兌換代碼。然後開啟應用程式，導覽至 **設定 > 恢復購買** 以啟用進階內容。

## 如何在 Mac 上兌換促銷代碼

1. 在您的 Mac 上開啟 **App Store**。

2. 點按您的 **帳號**，選擇「兌換禮品卡」。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="Mac 上的 App Store" caption="Mac 上的 App Store" width="600" >}}

3. 輸入您的兌換代碼，點按 **兌換**。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Mac 上的兌換代碼畫面" caption="Mac 上的兌換代碼畫面" width="600" >}}

4. 開啟應用程式，前往 **設定 > 恢復購買** 以解鎖進階功能。

盡情享用您的免費應用程式或應用程式內升級吧！

## 常見問題

{{% details title="我在哪裡可以取得促銷代碼？" closed="true" %}}
促銷代碼由應用程式開發者提供，用於評測、贈品活動或促銷。直接聯繫開發者即可申請。
{{% /details %}}

{{% details title="促銷代碼會過期嗎？" closed="true" %}}
會。Apple App Store 促銷代碼在產生後 28 天內過期，且只能兌換一次。
{{% /details %}}

{{% details title="我可以在任何國家使用促銷代碼嗎？" closed="true" %}}
促銷代碼有地區限制。代碼必須與您 Apple ID 的 App Store 國家/地區相符。
{{% /details %}}

{{% details title="如何使用促銷代碼啟用應用程式內購買？" closed="true" %}}
在 App Store 中兌換代碼後，開啟應用程式並前往 設定 > 恢復購買。進階內容將自動解鎖。
{{% /details %}}

{{% details title="促銷代碼顯示已被兌換。我該怎麼辦？" closed="true" %}}
每個促銷代碼只能使用一次。請聯繫開發者申請新代碼。
{{% /details %}}

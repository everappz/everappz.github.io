**요약:** 프로모 코드를 사용하면 유료 앱을 무료로 다운로드하거나 인앱 구매를 활성화할 수 있습니다. iOS: App Store > 계정 아이콘 > 기프트 카드 또는 코드 사용 > 코드 입력. Mac: App Store > 계정 > 기프트 카드 사용 > 코드 입력. 그런 다음 앱을 열고 필요한 경우 구매를 복원하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="iPad에서 프로모 코드 활성화" width="800" >}}

## 프로모 코드란 무엇인가요?

프로모 코드는 유료 앱을 무료로 설치하거나 인앱 구매를 무료로 활성화할 수 있는 특별한 코드입니다. 앱 개발자에게 요청하면 받을 수 있습니다. 사용 방법은 다음과 같습니다:

## iOS 11 이상에서 프로모 코드를 사용하는 방법

1. iPhone 또는 iPad에서 App Store를 열고 하단의 "투데이" 탭을 탭하세요.

2. 오른쪽 상단에서 "계정" 아이콘을 탭하여 "계정 설정"에 접근하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Store 메인 페이지 iPhone" caption="App Store 메인 페이지 iPhone" width="400" >}}

3. 메뉴에서 "기프트 카드 또는 코드 사용"을 선택하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="App Store iPhone 계정 설정 화면" caption="App Store iPhone 계정 설정 화면" width="400" >}}

4. "코드를 수동으로 입력할 수도 있습니다"를 선택하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="iPhone 코드 사용 화면" caption="iPhone 코드 사용 화면" width="400" >}}

5. 프로모 코드를 입력하고 "완료"를 탭하여 다운로드 또는 활성화를 시작하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="iPhone에서 프로모 코드 입력" caption="iPhone에서 프로모 코드 입력" width="400" >}}

6. 설치가 완료되면 홈 화면에 앱이 나타납니다. 탭하여 열어보세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="코드가 성공적으로 사용되었습니다" caption="코드가 성공적으로 사용되었습니다" width="400" >}}

## iOS 10 이하에서 프로모 코드를 사용하는 방법

1. App Store를 열고 "추천" 탭으로 이동하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store 화면 iPhone" caption="iOS 10의 App Store" width="400" >}}

2. "기타 링크"로 스크롤하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store 화면 iPhone 내 프로필" caption="iOS 10 App Store의 기타 링크 섹션" width="400" >}}

3. "코드 사용"을 탭하고 Apple ID 비밀번호를 입력하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store 비밀번호 입력" caption="메시지가 표시되면 비밀번호를 입력하세요" width="400" >}}

4. 코드 사용 화면이 표시됩니다.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="iOS 10의 코드 사용 화면" caption="iOS 10의 코드 사용 화면" width="400" >}}

5. "코드를 수동으로 입력할 수도 있습니다"를 탭하고 코드를 입력하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="iOS 10에서 수동 코드 입력" caption="iOS 10에서 수동 코드 입력" width="400" >}}

6. "코드 사용"을 탭하면 앱 설치가 시작됩니다.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="코드 사용 후 앱 다운로드 중" caption="코드 사용 후 앱 다운로드 중" width="400" >}}

7. 설치가 완료되면 홈 화면에 앱이 나타납니다.

8. 앱을 즐기세요!

## 인앱 구매에 프로모 코드를 사용하는 방법

위의 동일한 단계에 따라 코드를 사용하세요. 그런 다음 앱을 열고 **설정 > 구매 복원**으로 이동하여 프리미엄 콘텐츠를 활성화하세요.

## Mac에서 프로모 코드를 사용하는 방법

1. Mac에서 **App Store**를 여세요.

2. **계정**을 클릭하고 "기프트 카드 사용"을 선택하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="Mac의 App Store" caption="Mac의 App Store" width="600" >}}

3. 프로모 코드를 입력하고 **코드 사용**을 클릭하세요.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Mac의 코드 사용 화면" caption="Mac의 코드 사용 화면" width="600" >}}

4. 앱을 열고 **설정 > 구매 복원**으로 이동하여 프리미엄 기능을 활성화하세요.

무료 앱이나 인앱 업그레이드를 즐기세요!

## 자주 묻는 질문

{{% details title="프로모 코드는 어디서 받을 수 있나요?" closed="true" %}}
프로모 코드는 앱 개발자가 리뷰, 이벤트 또는 프로모션을 위해 제공합니다. 개발자에게 직접 연락하여 요청하세요.
{{% /details %}}

{{% details title="프로모 코드에 만료일이 있나요?" closed="true" %}}
네. Apple App Store 프로모 코드는 생성 후 28일 후에 만료되며 한 번만 사용할 수 있습니다.
{{% /details %}}

{{% details title="다른 국가에서 프로모 코드를 사용할 수 있나요?" closed="true" %}}
프로모 코드는 지역별로 제한됩니다. 코드는 Apple ID의 App Store 국가와 일치해야 합니다.
{{% /details %}}

{{% details title="프로모 코드로 인앱 구매를 어떻게 활성화하나요?" closed="true" %}}
App Store에서 코드를 사용한 후 앱을 열고 설정 > 구매 복원으로 이동하세요. 프리미엄 콘텐츠가 자동으로 활성화됩니다.
{{% /details %}}

{{% details title="프로모 코드가 이미 사용되었다고 표시됩니다. 어떻게 해야 하나요?" closed="true" %}}
각 프로모 코드는 한 번만 사용할 수 있습니다. 개발자에게 연락하여 새 코드를 요청하세요.
{{% /details %}}

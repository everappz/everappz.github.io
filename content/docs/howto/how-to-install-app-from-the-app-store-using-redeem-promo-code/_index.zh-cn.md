**简要说明：** 促销代码可以让您免费下载付费应用或解锁应用内购买。在 iOS 上：App Store > 账户图标 > 兑换礼品卡或代码 > 输入代码。在 Mac 上：App Store > 账户 > 兑换礼品卡 > 输入代码。然后打开应用，如有需要，恢复购买。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="在 iPad 上激活兑换促销代码" width="800" >}}

## 什么是兑换促销代码？

兑换促销代码就像一张金色门票，可以让您免费安装付费应用或解锁应用内购买，无需花费一分钱。要获取促销代码，只需联系应用开发者即可。以下是使用方法：

## 如何在 iOS 11 或更高版本上兑换促销代码

1. 在您的 iPhone 或 iPad 上打开 App Store，点击底部的"今天"标签。

2. 在右上角，点击"账户"图标以访问"账户设置"。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Store 主页 iPhone" caption="App Store 主页 iPhone" width="400" >}}

3. 从菜单中选择"兑换礼品卡或代码"。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="App Store 账户设置屏幕 iPhone" caption="App Store 账户设置屏幕 iPhone" width="400" >}}

4. 选择"您也可以手动输入代码。"

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="iPhone 兑换代码屏幕" caption="iPhone 兑换代码屏幕" width="400" >}}

5. 输入您的兑换代码，点击"完成"开始下载或激活。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="为 iPhone 输入兑换代码" caption="为 iPhone 输入兑换代码" width="400" >}}

6. 安装完成后，应用将出现在您的主屏幕上。点击即可打开。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="您已成功兑换代码" caption="您已成功兑换代码" width="400" >}}

## 如何在 iOS 10 或更早版本上兑换促销代码

1. 打开 App Store，进入"精选"标签。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store 屏幕 iPhone" caption="iOS 10 上的 App Store" width="400" >}}

2. 滚动到"其他链接"。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store 屏幕 iPhone 我的个人资料" caption="iOS 10 App Store 中的其他链接部分" width="400" >}}

3. 点击"兑换"并输入您的 Apple ID 密码。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store 输入密码" caption="在提示时输入密码" width="400" >}}

4. 然后您将看到兑换屏幕。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="iOS 10 上的兑换代码屏幕" caption="iOS 10 上的兑换代码屏幕" width="400" >}}

5. 点击"您也可以手动输入代码"并输入您的代码。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="iOS 10 上手动输入代码" caption="iOS 10 上手动输入代码" width="400" >}}

6. 点击"兑换"，应用安装将开始。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="兑换代码后应用正在下载" caption="兑换代码后应用正在下载" width="400" >}}

7. 应用安装完成后将出现在您的主屏幕上。

8. 尽情享用您的应用吧！

## 如何使用促销代码进行应用内购买

按照上述相同步骤兑换代码。然后打开应用，导航到 **设置 > 恢复购买** 以激活高级内容。

## 如何在 Mac 上兑换促销代码

1. 在您的 Mac 上打开 **App Store**。

2. 点击您的 **账户**，选择"兑换礼品卡"。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="Mac 上的 App Store" caption="Mac 上的 App Store" width="600" >}}

3. 输入您的兑换代码，点击 **兑换**。

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Mac 上的兑换代码屏幕" caption="Mac 上的兑换代码屏幕" width="600" >}}

4. 打开应用，进入 **设置 > 恢复购买** 以解锁高级功能。

尽情享用您的免费应用或应用内升级吧！

## 常见问题

{{% details title="我在哪里可以获取促销代码？" closed="true" %}}
促销代码由应用开发者提供，用于评测、赠品活动或促销。直接联系开发者即可申请。
{{% /details %}}

{{% details title="促销代码会过期吗？" closed="true" %}}
会。Apple App Store 促销代码在生成后 28 天内过期，且只能兑换一次。
{{% /details %}}

{{% details title="我可以在任何国家使用促销代码吗？" closed="true" %}}
促销代码有区域限制。代码必须与您 Apple ID 的 App Store 国家/地区匹配。
{{% /details %}}

{{% details title="如何使用促销代码激活应用内购买？" closed="true" %}}
在 App Store 中兑换代码后，打开应用并前往 设置 > 恢复购买。高级内容将自动解锁。
{{% /details %}}

{{% details title="促销代码显示已被兑换。我该怎么办？" closed="true" %}}
每个促销代码只能使用一次。请联系开发者申请新代码。
{{% /details %}}

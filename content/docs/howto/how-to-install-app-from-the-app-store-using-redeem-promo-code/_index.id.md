**Ringkasan:** Kode promo memungkinkan Anda mengunduh aplikasi berbayar secara gratis atau membuka pembelian dalam aplikasi. Di iOS: App Store > Ikon Akun > Tukarkan Kartu Hadiah atau Kode > masukkan kode. Di Mac: App Store > Akun > Tukarkan Kartu Hadiah > masukkan kode. Kemudian buka aplikasi dan pulihkan pembelian jika diperlukan.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="Aktifkan kode promo di iPad" width="800" >}}

## Apa itu Kode Promo?

Kode Promo seperti tiket emas yang memungkinkan Anda menginstal aplikasi berbayar secara gratis atau membuka pembelian dalam aplikasi tanpa mengeluarkan uang sepeser pun. Untuk mendapatkannya, cukup hubungi pengembang aplikasi. Berikut cara menggunakannya:

## Cara Menukarkan Kode Promo di iOS 11 atau Lebih Baru

1. Buka App Store di iPhone atau iPad Anda dan ketuk tab "Hari Ini" di bagian bawah.

2. Di pojok kanan atas, ketuk ikon "Akun" untuk mengakses "Pengaturan Akun."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="Halaman Utama App Store iPhone" caption="Halaman Utama App Store iPhone" width="400" >}}

3. Pilih "Tukarkan Kartu Hadiah atau Kode" dari menu.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Layar Pengaturan Akun untuk App Store iPhone" caption="Layar Pengaturan Akun untuk App Store iPhone" width="400" >}}

4. Pilih "Anda juga dapat memasukkan kode secara manual."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Layar Tukarkan Kode untuk iPhone" caption="Layar Tukarkan Kode untuk iPhone" width="400" >}}

5. Masukkan kode promo Anda dan ketuk "Selesai" untuk mulai mengunduh atau mengaktifkan.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Masukkan Kode Promo Untuk iPhone" caption="Masukkan Kode Promo Untuk iPhone" width="400" >}}

6. Setelah terinstal, aplikasi akan muncul di layar Beranda Anda. Ketuk untuk membuka.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Anda berhasil menukarkan kode Anda" caption="Anda berhasil menukarkan kode Anda" width="400" >}}

## Cara Menukarkan Kode Promo di iOS 10 atau Lebih Lama

1. Buka App Store dan buka tab "Unggulan."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="Layar App Store iOS 10 iPhone" caption="App Store di iOS 10" width="400" >}}

2. Gulir ke "Tautan lainnya."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="Layar App Store iOS 10 iPhone Profil Saya" caption="Bagian tautan lainnya di App Store iOS 10" width="400" >}}

3. Ketuk "Tukarkan" dan masukkan kata sandi Apple ID Anda.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="App Store iOS 10 masukkan kata sandi" caption="Masukkan kata sandi saat diminta" width="400" >}}

4. Anda kemudian akan melihat layar Tukarkan.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Layar Tukarkan Kode di iOS 10" caption="Layar Tukarkan Kode di iOS 10" width="400" >}}

5. Ketuk "Anda juga dapat memasukkan kode secara manual" dan masukkan kode Anda.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Entri kode manual di iOS 10" caption="Entri kode manual di iOS 10" width="400" >}}

6. Ketuk "Tukarkan" dan instalasi aplikasi akan dimulai.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Aplikasi mengunduh setelah menukarkan kode" caption="Aplikasi mengunduh setelah menukarkan kode" width="400" >}}

7. Aplikasi akan muncul di layar Beranda Anda setelah terinstal.

8. Nikmati aplikasi Anda!

## Cara Menggunakan Kode Promo untuk Pembelian Dalam Aplikasi

Ikuti langkah-langkah yang sama di atas untuk menukarkan kode. Setelah itu, buka aplikasi dan navigasi ke **Pengaturan > Pulihkan Pembelian** untuk mengaktifkan konten premium.

## Cara Menukarkan Kode Promo di Mac

1. Buka **App Store** di Mac Anda.

2. Klik **Akun** Anda dan pilih "Tukarkan Kartu Hadiah."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store di Mac" caption="App Store di Mac" width="600" >}}

3. Masukkan kode promo Anda dan klik **Tukarkan**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Layar tukarkan kode di Mac" caption="Layar tukarkan kode di Mac" width="600" >}}

4. Buka aplikasi dan buka **Pengaturan > Pulihkan Pembelian** untuk membuka fitur premium.

Nikmati aplikasi gratis atau peningkatan dalam aplikasi Anda!

## Pertanyaan yang Sering Diajukan

{{% details title="Di mana saya bisa mendapatkan kode promo?" closed="true" %}}
Kode promo disediakan oleh pengembang aplikasi untuk ulasan, giveaway, atau promosi. Hubungi pengembang secara langsung untuk memintanya.
{{% /details %}}

{{% details title="Apakah kode promo kedaluwarsa?" closed="true" %}}
Ya. Kode promo Apple App Store kedaluwarsa 28 hari setelah dibuat dan hanya dapat ditukarkan satu kali.
{{% /details %}}

{{% details title="Bisakah saya menggunakan kode promo di negara mana pun?" closed="true" %}}
Kode promo bersifat spesifik per wilayah. Kode harus sesuai dengan negara App Store dari Apple ID Anda.
{{% /details %}}

{{% details title="Bagaimana cara mengaktifkan pembelian dalam aplikasi dengan kode promo?" closed="true" %}}
Setelah menukarkan kode di App Store, buka aplikasi dan buka Pengaturan > Pulihkan Pembelian. Konten premium akan terbuka secara otomatis.
{{% /details %}}

{{% details title="Kode promo mengatakan sudah ditukarkan. Apa yang harus saya lakukan?" closed="true" %}}
Setiap kode promo hanya dapat digunakan satu kali. Hubungi pengembang untuk meminta kode baru.
{{% /details %}}

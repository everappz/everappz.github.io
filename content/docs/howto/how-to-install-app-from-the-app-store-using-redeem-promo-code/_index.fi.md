**Lyhyesti:** Kampanjakoodi mahdollistaa maksullisen sovelluksen lataamisen ilmaiseksi tai sovelluksen sisäisten ostojen avaamisen. iOS:llä: App Store > Tili-kuvake > Lunasta lahjakortti tai koodi > syötä koodi. Macilla: App Store > Tili > Lunasta lahjakortti > syötä koodi. Avaa sitten sovellus ja palauta ostot tarvittaessa.

## Mikä on lunastuskoodi?

Lunastuskoodi on kuin kultainen lippu, joka mahdollistaa maksullisen sovelluksen asentamisen ilmaiseksi tai sovelluksen sisäisten ostojen avaamisen ilman senttiäkään. Saadaksesi sellaisen, pyydä sitä sovelluksen kehittäjältä. Näin käytät sitä:

## Kuinka lunastaa kampanjakoodi iOS 11:ssä tai uudemmassa

1. Avaa App Store iPhonellasi tai iPadillasi ja napauta "Tänään"-välilehteä alareunassa.

2. Napauta oikeassa yläkulmassa "Tili"-kuvaketta päästäksesi "Tiliasetuksiin."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Storen pääsivu iPhone" caption="App Storen pääsivu iPhone" width="400" >}}

3. Valitse valikosta "Lunasta lahjakortti tai koodi".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Tiliasetukset-näyttö App Store iPhone" caption="Tiliasetukset-näyttö App Store iPhone" width="400" >}}

4. Valitse "Voit myös syöttää koodisi manuaalisesti."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Lunasta koodi -näyttö iPhonelle" caption="Lunasta koodi -näyttö iPhonelle" width="400" >}}

5. Syötä lunastuskoodisi ja napauta "Valmis" aloittaaksesi latauksen tai aktivoinnin.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Syötä lunastuskoodi iPhonelle" caption="Syötä lunastuskoodi iPhonelle" width="400" >}}

6. Asennuksen jälkeen sovellus ilmestyy kotinäytöllesi. Napauta avataksesi.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Olet lunastanut koodisi onnistuneesti" caption="Olet lunastanut koodisi onnistuneesti" width="400" >}}

## Kuinka lunastaa kampanjakoodi iOS 10:ssä tai aiemmassa

1. Avaa App Store ja siirry "Suositellut"-välilehteen.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store -näyttö iPhone" caption="App Store iOS 10:ssä" width="400" >}}

2. Vieritä kohtaan "Muut linkit."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store -näyttö iPhone Oma profiili" caption="Muut linkit -osio iOS 10 App Storessa" width="400" >}}

3. Napauta "Lunasta" ja syötä Apple ID -salasanasi.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store syötä salasana" caption="Syötä salasana pyydettäessä" width="400" >}}

4. Tämän jälkeen näet lunastusnäytön.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Lunasta koodi -näyttö iOS 10:ssä" caption="Lunasta koodi -näyttö iOS 10:ssä" width="400" >}}

5. Napauta "Voit myös syöttää koodisi manuaalisesti" ja syötä koodisi.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Manuaalinen koodin syöttö iOS 10:ssä" caption="Manuaalinen koodin syöttö iOS 10:ssä" width="400" >}}

6. Napauta "Lunasta" ja sovelluksen asennus alkaa.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="Sovellus latautuu koodin lunastamisen jälkeen" caption="Sovellus latautuu koodin lunastamisen jälkeen" width="400" >}}

7. Sovellus ilmestyy kotinäytöllesi asennuksen jälkeen.

8. Nauti sovelluksestasi!

## Kuinka käyttää kampanjakoodia sovelluksen sisäisiin ostoihin

Noudata samoja yllä olevia ohjeita koodin lunastamiseksi. Avaa sen jälkeen sovellus ja siirry kohtaan **Asetukset > Palauta ostot** aktivoidaksesi premium-sisällön.

## Kuinka lunastaa kampanjakoodi Macilla

1. Avaa **App Store** Macillasi.

2. Napsauta **Tiliäsi** ja valitse "Lunasta lahjakortti."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store Macilla" caption="App Store Macilla" width="600" >}}

3. Syötä lunastuskoodisi ja napsauta **Lunasta**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Lunasta koodi -näyttö Macilla" caption="Lunasta koodi -näyttö Macilla" width="600" >}}

4. Avaa sovellus ja siirry kohtaan **Asetukset > Palauta ostot** avataksesi premium-ominaisuudet.

Nauti ilmaisesta sovelluksestasi tai sovelluksen sisäisestä päivityksestä!

## Usein kysytyt kysymykset

{{% details title="Mistä saan kampanjakoodin?" closed="true" %}}
Kampanjakoodeja tarjoavat sovelluskehittäjät arvosteluihin, arvontoihin tai kampanjoihin. Ota suoraan yhteyttä kehittäjään pyytääksesi sellaisen.
{{% /details %}}

{{% details title="Vanhenevatko kampanjakoodit?" closed="true" %}}
Kyllä. Apple App Storen kampanjakoodit vanhenevat 28 päivää niiden luomisen jälkeen, ja ne voidaan lunastaa vain kerran.
{{% /details %}}

{{% details title="Voinko käyttää kampanjakoodia missä tahansa maassa?" closed="true" %}}
Kampanjakoodit ovat aluekohtaisia. Koodin on vastattava Apple ID:si App Store -maata.
{{% /details %}}

{{% details title="Kuinka aktivoin sovelluksen sisäiset ostot kampanjakoodilla?" closed="true" %}}
Koodin lunastamisen jälkeen App Storessa avaa sovellus ja siirry kohtaan Asetukset > Palauta ostot. Premium-sisältö avautuu automaattisesti.
{{% /details %}}

{{% details title="Kampanjakoodi sanoo, että se on jo lunastettu. Mitä minun pitäisi tehdä?" closed="true" %}}
Jokainen kampanjakoodi voidaan käyttää vain kerran. Ota yhteyttä kehittäjään pyytääksesi uuden koodin.
{{% /details %}}

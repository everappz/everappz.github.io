**Kurzfassung:** Mit einem Promo-Code können Sie eine kostenpflichtige App kostenlos herunterladen oder In-App-Käufe freischalten. Auf iOS: App Store > Kontosymbol > Geschenkkarte oder Code einlösen > Code eingeben. Auf dem Mac: App Store > Konto > Geschenkkarte einlösen > Code eingeben. Öffnen Sie dann die App und stellen Sie Käufe wieder her, falls erforderlich.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b9528a8ed58c4207be1e012627bce8d1~mv2.png" alt="Einlösecode auf dem iPad aktivieren" width="800" >}}

## Was ist ein Einlösecode?

Ein Einlösecode ist wie ein goldenes Ticket, mit dem Sie eine kostenpflichtige App kostenlos installieren oder In-App-Käufe freischalten können, ohne einen Cent auszugeben. Um einen zu erhalten, wenden Sie sich einfach an den App-Entwickler. So verwenden Sie ihn:

## So lösen Sie einen Promo-Code unter iOS 11 oder neuer ein

1. Öffnen Sie den App Store auf Ihrem iPhone oder iPad und tippen Sie unten auf den Tab "Heute".

2. Tippen Sie in der oberen rechten Ecke auf das Symbol "Konto", um auf Ihre "Kontoeinstellungen" zuzugreifen.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_24b5a1108741439ca30f19aa35723e58~mv2.png" alt="App Store Hauptseite iPhone" caption="App Store Hauptseite iPhone" width="400" >}}

3. Wählen Sie "Geschenkkarte oder Code einlösen" aus dem Menü.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_15f0bdd629114233b26ef05680f2accb~mv2.png" alt="Kontoeinstellungen im App Store iPhone" caption="Kontoeinstellungen im App Store iPhone" width="400" >}}

4. Wählen Sie "Sie können Ihren Code auch manuell eingeben."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_ed83453111534d3fb7aecacf77ccd2d6~mv2.png" alt="Code einlösen Bildschirm für iPhone" caption="Code einlösen Bildschirm für iPhone" width="400" >}}

5. Geben Sie Ihren Einlösecode ein und tippen Sie auf "Fertig", um den Download oder die Aktivierung zu starten.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_57722043a3384279914bc9eb49566e37~mv2.png" alt="Einlösecode eingeben für iPhone" caption="Einlösecode eingeben für iPhone" width="400" >}}

6. Nach der Installation erscheint die App auf Ihrem Startbildschirm. Tippen Sie zum Öffnen.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_826f889f680f4072aa59ef17dce2386c~mv2.png" alt="Sie haben Ihren Code erfolgreich eingelöst" caption="Sie haben Ihren Code erfolgreich eingelöst" width="400" >}}

## So lösen Sie einen Promo-Code unter iOS 10 oder früher ein

1. Öffnen Sie den App Store und gehen Sie zum Tab "Highlights".

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_c61ca4b7bb0c433c843539e0f627d898~mv2.png" alt="iOS 10 App Store Bildschirm iPhone" caption="App Store unter iOS 10" width="400" >}}

2. Scrollen Sie zu "Weitere Links."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_e1e70b3714ab4aada97faa762c81d448~mv2.png" alt="iOS 10 App Store Bildschirm iPhone Mein Profil" caption="Weitere Links im iOS 10 App Store" width="400" >}}

3. Tippen Sie auf "Einlösen" und geben Sie Ihr Apple ID-Passwort ein.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_5ab2d0b322bf4b1faa7a8e046b8b680f~mv2.png" alt="iOS 10 App Store Passwort eingeben" caption="Passwort eingeben, wenn Sie dazu aufgefordert werden" width="400" >}}

4. Anschließend sehen Sie den Einlösebildschirm.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_88566104fb6e44489a8f08ee58a30452~mv2.png" alt="Code einlösen Bildschirm unter iOS 10" caption="Code einlösen Bildschirm unter iOS 10" width="400" >}}

5. Tippen Sie auf "Sie können Ihren Code auch manuell eingeben" und geben Sie Ihren Code ein.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_26498a082a2a4003b0ca09effc325df1~mv2.png" alt="Manuelle Codeeingabe unter iOS 10" caption="Manuelle Codeeingabe unter iOS 10" width="400" >}}

6. Tippen Sie auf "Einlösen" und die App-Installation beginnt.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_b42ce94ba22146a19a40c97453de3745~mv2.png" alt="App wird nach Einlösung des Codes heruntergeladen" caption="App wird nach Einlösung des Codes heruntergeladen" width="400" >}}

7. Die App erscheint nach der Installation auf Ihrem Startbildschirm.

8. Viel Spaß mit Ihrer App!

## So verwenden Sie einen Promo-Code für In-App-Käufe

Folgen Sie den gleichen Schritten oben, um den Code einzulösen. Öffnen Sie danach die App und navigieren Sie zu **Einstellungen > Käufe wiederherstellen**, um die Premium-Inhalte zu aktivieren.

## So lösen Sie einen Promo-Code auf dem Mac ein

1. Öffnen Sie den **App Store** auf Ihrem Mac.

2. Klicken Sie auf Ihr **Konto** und wählen Sie "Geschenkkarte einlösen."

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_2dba830bbec644c084581c2170e4966e~mv2.png" alt="App Store auf dem Mac" caption="App Store auf dem Mac" width="600" >}}

3. Geben Sie Ihren Einlösecode ein und klicken Sie auf **Einlösen**.

{{< figure src="/docs/howto/how-to-install-app-from-the-app-store-using-redeem-promo-code/21260c_d7f132879a254873aaa8a65c688f59b3~mv2.png" alt="Code einlösen Bildschirm auf dem Mac" caption="Code einlösen Bildschirm auf dem Mac" width="600" >}}

4. Öffnen Sie die App und gehen Sie zu **Einstellungen > Käufe wiederherstellen**, um Premium-Funktionen freizuschalten.

Viel Spaß mit Ihrer kostenlosen App oder dem In-App-Upgrade!

## Häufig gestellte Fragen

{{% details title="Wo bekomme ich einen Promo-Code?" closed="true" %}}
Promo-Codes werden von App-Entwicklern für Rezensionen, Verlosungen oder Werbeaktionen bereitgestellt. Kontaktieren Sie den Entwickler direkt, um einen anzufordern.
{{% /details %}}

{{% details title="Laufen Promo-Codes ab?" closed="true" %}}
Ja. Apple App Store Promo-Codes laufen 28 Tage nach ihrer Erstellung ab und können nur einmal eingelöst werden.
{{% /details %}}

{{% details title="Kann ich einen Promo-Code in jedem Land verwenden?" closed="true" %}}
Promo-Codes sind regionsspezifisch. Der Code muss mit dem App Store-Land Ihrer Apple ID übereinstimmen.
{{% /details %}}

{{% details title="Wie aktiviere ich In-App-Käufe mit einem Promo-Code?" closed="true" %}}
Nach dem Einlösen des Codes im App Store öffnen Sie die App und gehen Sie zu Einstellungen > Käufe wiederherstellen. Die Premium-Inhalte werden automatisch freigeschaltet.
{{% /details %}}

{{% details title="Der Promo-Code sagt, er wurde bereits eingelöst. Was soll ich tun?" closed="true" %}}
Jeder Promo-Code kann nur einmal verwendet werden. Kontaktieren Sie den Entwickler, um einen neuen Code anzufordern.
{{% /details %}}

## 如何以不同速度播放 SoundCloud（5 个简单步骤）

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Soundy 应用图标" caption="Soundy 应用图标" width="100" >}}

### 1. 从 App Store 安装 Soundy  
   [**获取 Soundy**](/products/soundy)

### 2. 登录您的 SoundCloud 帐户  
   打开"我的帐户"屏幕，使用您的凭据或通过 Google 或 Facebook 登录。Soundy 使用官方 SoundCloud API，因此您的登录凭据安全可靠。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_044a5215e6b946018c90475147b03333~mv2.png" alt="Soundy：登录 SoundCloud" caption="通过官方 SoundCloud API 安全登录" width="400" >}}

### 3. 浏览您的帐户  
   访问您的曲目、播放列表、专辑、喜欢的内容和关注者。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy：我的帐户屏幕" caption="包含所有 SoundCloud 数据的我的帐户屏幕" width="400" >}}

### 4. 查找并播放曲目  
   找到您想收听的曲目、播客或有声书，然后点击播放。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy：喜欢的歌曲" caption="播放喜欢的歌曲、播客或有声书" width="400" >}}

### 5. 调整播放速度  
   在"播放器"屏幕上，点击"速度"按钮，以 0.1x 为增量在 0.2x 到 2x 之间调节。非常适合微调您的播客速度！

## 如何以不同速度播放 SoundCloud（5 個簡單步驟）

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Soundy 應用程式圖示" caption="Soundy 應用程式圖示" width="100" >}}

### 1. 從 App Store 安裝 Soundy  
   [**取得 Soundy**](/products/soundy)

### 2. 登入您的 SoundCloud 帳戶  
   開啟「我的帳戶」畫面，使用您的憑證或透過 Google 或 Facebook 登入。Soundy 使用官方 SoundCloud API，因此您的登入憑證安全可靠。

### 3. 瀏覽您的帳戶  
   存取您的曲目、播放清單、專輯、喜歡的內容和追蹤者。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy：我的帳戶畫面" caption="包含所有 SoundCloud 資料的我的帳戶畫面" width="400" >}}

### 4. 尋找並播放曲目  
   找到您想收聽的曲目、播客或有聲書，然後點擊播放。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy：喜歡的歌曲" caption="播放喜歡的歌曲、播客或有聲書" width="400" >}}

### 5. 調整播放速度  
   在「播放器」畫面上，點擊「速度」按鈕，以 0.1x 為增量在 0.2x 到 2x 之間調節。非常適合微調您的播客速度！

**要約:** SoundCloud公式iOSアプリには再生速度コントロール機能がありません。Soundyは無料の非公式SoundCloudプレーヤーで、0.1x刻みで0.2xから2xまで速度を調整できます -- ポッドキャストやオーディオブックに最適です。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_85ec03750940439a9fcdb6360e93a5f6~mv2.jpg" alt="Soundy: 再生速度を上げる" caption="Soundy: SoundCloudの再生速度を上げる" width="800" >}}

音楽発見が大好きな熱心なSoundCloudユーザーですか？SoundCloudは素晴らしいですが、オーディオブックやポッドキャストが好きなら、公式iOSクライアントに重要な機能がないことに気づいているでしょう：再生速度コントロールです。お気持ちはわかります！多くのSoundCloudユーザーがSoundCloudヘルプコミュニティセンターでこの機能をリクエストしていますが、まだ実現していません。

**Soundy**をご紹介します。この欠けている機能への解決策です。SoundyはiOS向けの最高の非公式SoundCloudプレーヤーで、完全に無料です。再生速度コントロール、イコライザー、オーディオブックマーク、スリープタイマーなど幅広い機能を備え、SoundCloudの体験を次のレベルへ引き上げます。

## SoundCloudを異なる速度でストリーミングする方法（5つの簡単なステップ）

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Soundyアプリアイコン" caption="Soundyアプリアイコン" width="100" >}}

### 1. App StoreからSoundyをインストール  
   [**Soundyを入手**](/products/soundy)

### 2. SoundCloudアカウントにログイン  
   「マイアカウント」画面を開き、認証情報またはGoogleやFacebook経由でサインインします。SoundyはSoundCloud公式APIを使用しているため、ログイン情報は安全に保護されます。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_044a5215e6b946018c90475147b03333~mv2.png" alt="Soundy: SoundCloudにログイン" caption="SoundCloud公式APIで安全にログイン" width="400" >}}

### 3. アカウントを閲覧  
   トラック、プレイリスト、アルバム、いいね、フォロワーにアクセスできます。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy: マイアカウント画面" caption="すべてのSoundCloudデータが表示されるマイアカウント画面" width="400" >}}

### 4. トラックを見つけて再生  
   聴きたいトラック、ポッドキャスト、またはオーディオブックを見つけ、タップして再生します。

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy: お気に入りの曲" caption="お気に入りの曲、ポッドキャスト、オーディオブックを再生" width="400" >}}

### 5. 再生速度を調整  
   「プレーヤー」画面で「速度」ボタンをタップし、0.1x刻みで0.2xから2xまで調整します。ポッドキャストの速度を細かく調整するのに最適です！

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_12b6077816a04189bbdf3342d9cf0c3b~mv2.png" alt="Soundy: 再生速度コントロール" caption="精密に再生速度を調整" width="400" >}}

**Ringkasan:** Aplikasi rasmi SoundCloud iOS tidak mempunyai kawalan kelajuan main balik. Soundy, pemain SoundCloud tidak rasmi yang percuma, membolehkan anda melaraskan kelajuan dari 0.2x hingga 2x dalam kenaikan 0.1x -- sesuai untuk podcast dan buku audio.

Adakah anda pengguna SoundCloud yang gemar menemui muzik? SoundCloud memang hebat, tetapi jika anda minat buku audio dan podcast, anda mungkin merasakan ketiadaan ciri utama dalam klien iOS rasmi: kawalan kelajuan main balik. Kami faham! Ramai pengguna SoundCloud telah meminta ciri ini di Pusat Komuniti Bantuan SoundCloud, tetapi ia masih belum dilaksanakan.

Memperkenalkan **Soundy**, penyelesaian anda untuk ciri yang hilang ini. Soundy ialah pemain SoundCloud tidak rasmi terbaik untuk iOS, dan ia PERCUMA sepenuhnya. Dengan pelbagai ciri seperti kawalan kelajuan main balik, penyama, penanda buku audio dan pemasa tidur, Soundy membawa pengalaman SoundCloud anda ke tahap seterusnya.

## Cara Strim SoundCloud pada Kelajuan Berbeza (5 Langkah Mudah)

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Ikon aplikasi Soundy" caption="Ikon aplikasi Soundy" width="100" >}}

### 1. Pasang Soundy dari App Store  
   [**Dapatkan Soundy**](/products/soundy)

### 2. Log Masuk ke Akaun SoundCloud Anda  
   Buka skrin 'Akaun Saya' dan log masuk menggunakan kelayakan anda atau melalui Google atau Facebook. Soundy menggunakan API rasmi SoundCloud, jadi kelayakan log masuk anda kekal selamat.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_044a5215e6b946018c90475147b03333~mv2.png" alt="Soundy: Log Masuk ke SoundCloud" caption="Log masuk dengan selamat menggunakan API rasmi SoundCloud" width="400" >}}

### 3. Layari Akaun Anda  
   Akses trek, senarai main, album, suka dan pengikut anda.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy: Skrin Akaun Saya" caption="Skrin Akaun Saya dengan semua data SoundCloud anda" width="400" >}}

### 4. Cari dan Mainkan Trek  
   Cari trek, podcast atau buku audio yang ingin anda dengar, kemudian ketik untuk main.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy: Lagu Disukai" caption="Mainkan lagu, podcast atau buku audio yang disukai" width="400" >}}

### 5. Laraskan Kelajuan Main Balik  
   Pada skrin 'Pemain', ketik butang 'Kelajuan' dan laraskan dari 0.2x hingga 2x menggunakan kenaikan 0.1x. Sesuai untuk melaraskan kelajuan podcast anda dengan tepat!

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_12b6077816a04189bbdf3342d9cf0c3b~mv2.png" alt="Soundy: Kawalan Kelajuan Main Balik" caption="Laraskan kelajuan main balik dengan tepat" width="400" >}}

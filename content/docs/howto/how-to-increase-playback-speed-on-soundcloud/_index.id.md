Apakah Anda pengguna setia SoundCloud yang suka menemukan musik? SoundCloud memang fantastis, tetapi jika Anda menyukai buku audio dan podcast, Anda mungkin merasakan tidak adanya fitur penting di klien resmi iOS: kontrol kecepatan pemutaran. Kami memahami Anda! Banyak pengguna SoundCloud telah meminta fitur ini di Pusat Komunitas Bantuan SoundCloud, tetapi belum terwujud.

Memperkenalkan **Soundy**, solusi Anda untuk fitur yang hilang ini. Soundy adalah pemutar SoundCloud tidak resmi terbaik untuk iOS, dan sepenuhnya GRATIS. Dengan berbagai fitur seperti kontrol kecepatan pemutaran, equalizer, penanda audio, dan timer tidur, Soundy membawa pengalaman SoundCloud Anda ke level berikutnya.

## Cara Streaming SoundCloud pada Kecepatan Berbeda (5 Langkah Mudah)

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Ikon aplikasi Soundy" caption="Ikon aplikasi Soundy" width="100" >}}

### 1. Instal Soundy dari App Store  
   [**Dapatkan Soundy**](/products/soundy)

### 2. Masuk ke Akun SoundCloud Anda  
   Buka layar 'Akun Saya' dan masuk menggunakan kredensial Anda atau melalui Google atau Facebook. Soundy menggunakan API resmi SoundCloud, sehingga kredensial login Anda tetap aman.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_044a5215e6b946018c90475147b03333~mv2.png" alt="Soundy: Masuk ke SoundCloud" caption="Masuk dengan aman menggunakan API resmi SoundCloud" width="400" >}}

### 3. Jelajahi Akun Anda  
   Akses trek, daftar putar, album, suka, dan pengikut Anda.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy: Layar Akun Saya" caption="Layar Akun Saya dengan semua data SoundCloud Anda" width="400" >}}

### 4. Temukan dan Putar Trek  
   Temukan trek, podcast, atau buku audio yang ingin Anda dengarkan, lalu ketuk untuk memutar.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy: Lagu yang Disukai" caption="Putar lagu, podcast, atau buku audio yang disukai" width="400" >}}

### 5. Atur Kecepatan Pemutaran  
   Di layar 'Pemutar', ketuk tombol 'Kecepatan' dan atur dari 0,2x hingga 2x menggunakan peningkatan 0,1x. Sempurna untuk menyesuaikan kecepatan podcast Anda!

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_12b6077816a04189bbdf3342d9cf0c3b~mv2.png" alt="Soundy: Kontrol Kecepatan Pemutaran" caption="Atur kecepatan pemutaran dengan presisi" width="400" >}}

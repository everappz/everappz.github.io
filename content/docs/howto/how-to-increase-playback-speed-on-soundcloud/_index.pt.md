Você é um usuário ávido do SoundCloud que adora descobrir músicas? O SoundCloud é fantástico, mas se você gosta de audiolivros e podcasts, provavelmente sentiu a ausência de um recurso essencial no cliente oficial para iOS: controle de velocidade de reprodução. Nós entendemos! Muitos usuários do SoundCloud solicitaram esse recurso no Centro de Ajuda da Comunidade SoundCloud, mas ele ainda não foi implementado.

Apresentamos o **Soundy**, sua solução para esse recurso ausente. O Soundy é o melhor reprodutor não oficial do SoundCloud para iOS e é totalmente GRATUITO. Com uma ampla gama de recursos como controle de velocidade de reprodução, equalizador, marcadores de áudio e timer de sono, o Soundy leva sua experiência no SoundCloud para o próximo nível.

## Como transmitir SoundCloud em diferentes velocidades (5 passos fáceis)

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_715334daa5ea4b44b6fee6d768b4c31c~mv2.png" alt="Ícone do aplicativo Soundy" caption="Ícone do aplicativo Soundy" width="100" >}}

### 1. Instale o Soundy pela App Store  
   [**Baixar Soundy**](/products/soundy)

### 2. Faça login na sua conta SoundCloud  
   Abra a tela 'Minha Conta' e faça login usando suas credenciais ou via Google ou Facebook. O Soundy usa a API oficial do SoundCloud, então suas credenciais de login permanecem seguras.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_044a5215e6b946018c90475147b03333~mv2.png" alt="Soundy: Login no SoundCloud" caption="Faça login com segurança usando a API oficial do SoundCloud" width="400" >}}

### 3. Navegue pela sua conta  
   Acesse suas faixas, playlists, álbuns, curtidas e seguidores.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_2342fb33e88c45bba99cd7462c5153a0~mv2.png" alt="Soundy: Tela Minha Conta" caption="Tela Minha Conta com todos os seus dados do SoundCloud" width="400" >}}

### 4. Encontre e reproduza uma faixa  
   Localize a faixa, podcast ou audiolivro que deseja ouvir e toque para reproduzir.

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_1a7aeb38c01d4c5cbdbce8a2e3c6b4ba~mv2.png" alt="Soundy: Músicas curtidas" caption="Reproduza músicas, podcasts ou audiolivros curtidos" width="400" >}}

### 5. Ajuste a velocidade de reprodução  
   Na tela 'Reprodutor', toque no botão 'Velocidade' e ajuste de 0,2x a 2x usando incrementos de 0,1x. Perfeito para ajustar a velocidade do seu podcast com precisão!

{{< figure src="/docs/howto/how-to-increase-playback-speed-on-soundcloud/21260c_12b6077816a04189bbdf3342d9cf0c3b~mv2.png" alt="Soundy: Controle de velocidade de reprodução" caption="Ajuste a velocidade de reprodução com precisão" width="400" >}}

**Περίληψη:** Μεταβείτε στο [myaccount.google.com](https://myaccount.google.com/) > Ασφάλεια > Εφαρμογές και υπηρεσίες τρίτων μερών. Κάντε κλικ στην εφαρμογή που θέλετε να αφαιρέσετε και επιλέξτε "Κατάργηση πρόσβασης" ή "Διαγραφή όλων των συνδέσεων." Επαναλάβετε για κάθε εφαρμογή.

Στη σημερινή ψηφιακή εποχή, συχνά συνδέουμε τους λογαριασμούς μας Google με διάφορες εφαρμογές και υπηρεσίες τρίτων μερών για μεγαλύτερη ευκολία. Αν και αυτό μπορεί να διευκολύνει τη ζωή, είναι απαραίτητο να διατηρείτε τον έλεγχο και την ασφάλεια του λογαριασμού σας Google. Μερικές φορές, μπορεί να θέλετε να αποσυνδέσετε μια εφαρμογή τρίτου μέρους που δεν χρησιμοποιείτε πλέον ή δεν εμπιστεύεστε. Σε αυτό το άρθρο, θα σας καθοδηγήσουμε στη διαδικασία αποσύνδεσης μιας εφαρμογής τρίτου μέρους από τον λογαριασμό σας Google, βοηθώντας σας να διασφαλίσετε την ασφάλειά σας στο διαδίκτυο.

## Γιατί να αποσυνδέσετε μια εφαρμογή τρίτου μέρους;

Υπάρχουν αρκετοί λόγοι για τους οποίους μπορεί να θέλετε να αποσυνδέσετε μια εφαρμογή τρίτου μέρους από τον λογαριασμό σας Google:

- Ανησυχίες ασφάλειας   
Εάν υποψιάζεστε ότι μια εφαρμογή τρίτου μέρους μπορεί να θέσει σε κίνδυνο την ασφάλεια του λογαριασμού σας ή να αποκτήσει πρόσβαση σε περισσότερες πληροφορίες από τις απαραίτητες, είναι καλύτερο να διακόψετε τη σύνδεση.

- Αχρησιμοποίητες εφαρμογές   
Με τον καιρό, μπορεί να έχετε συνδέσει διάφορες εφαρμογές που δεν χρησιμοποιείτε πλέον. Η αποσύνδεσή τους μειώνει την ακαταστασία και σας βοηθά να διατηρείτε καλύτερο έλεγχο του λογαριασμού σας.

- Ιδιωτικότητα   
Η αποσύνδεση εφαρμογών μπορεί να βοηθήσει στην προστασία της ιδιωτικότητάς σας περιορίζοντας τα δεδομένα που μοιράζονται με αυτές τις υπηρεσίες.

Τώρα, ας δούμε τα βήματα για την αποσύνδεση μιας εφαρμογής τρίτου μέρους από τον λογαριασμό σας Google.

## Βήματα για την αποσύνδεση μιας εφαρμογής τρίτου μέρους

### 1. Πρόσβαση στις ρυθμίσεις του λογαριασμού σας Google     
Ανοίξτε ένα πρόγραμμα περιήγησης και μεταβείτε στο [myaccount.google.com](https://myaccount.google.com/).

### 2. Σύνδεση   
Εάν δεν έχετε ήδη συνδεθεί, εισάγετε τα διαπιστευτήρια του λογαριασμού σας Google για πρόσβαση στις ρυθμίσεις του λογαριασμού σας.

### 3. Ασφάλεια   
Στο μενού στα αριστερά, επιλέξτε "Ασφάλεια." Αυτή η ενότητα σας επιτρέπει να διαχειρίζεστε τις ρυθμίσεις ασφάλειας του λογαριασμού σας Google.

### 4. Εφαρμογές τρίτων μερών με πρόσβαση στον λογαριασμό  
Κάντε κύλιση προς τα κάτω στην ενότητα με τίτλο "Εφαρμογές και υπηρεσίες τρίτων μερών" και κάντε κλικ σε αυτήν. Εδώ, θα βρείτε μια λίστα εφαρμογών που έχουν πρόσβαση στον λογαριασμό σας Google.

![Εφαρμογές τρίτων μερών με πρόσβαση στον λογαριασμό](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_c9a2599fb25b4325ac2de19396cadd56~mv2.png)

### 5. Ελέγξτε τις εφαρμογές τρίτων μερών  
Ελέγξτε προσεκτικά τη λίστα των συνδεδεμένων εφαρμογών. Αυτές οι εφαρμογές έχουν διαφορετικά επίπεδα πρόσβασης στα δεδομένα του λογαριασμού σας Google.

![Ελέγξτε τις εφαρμογές τρίτων μερών](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_676e0cc9fa4a46da97ef1cad418a3433~mv2.png)

### 6. Κατάργηση πρόσβασης  
Για να αποσυνδέσετε μια εφαρμογή, κάντε κλικ σε αυτήν. Θα εμφανιστεί ένα παράθυρο με λεπτομέρειες σχετικά με την πρόσβαση της εφαρμογής. Εδώ, θα βρείτε την επιλογή "Κατάργηση πρόσβασης", "Αποσύνδεση" ή "Διαγραφή όλων των συνδέσεων".

![Κατάργηση πρόσβασης](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_a3cb1e7cd9964857b527cbb76ef7ddb1~mv2.png)

### 7. Επιβεβαίωση κατάργησης    
Επιβεβαιώστε την απόφασή σας να αποσυνδέσετε την εφαρμογή. Η Google θα σας ζητήσει να επιβεβαιώσετε αυτήν την ενέργεια για να βεβαιωθεί ότι λαμβάνετε μια ενημερωμένη απόφαση.

![Επιβεβαίωση κατάργησης](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_07819d0a93f946b1b9f6c84803c53056~mv2.png)

### 8. Επαναλάβετε όσο χρειάζεται     
Εάν έχετε πολλές εφαρμογές που θέλετε να αποσυνδέσετε, επαναλάβετε τη διαδικασία για καθεμία.

Μετά την αποσύνδεση των εφαρμογών, είναι καλή πρακτική να ελέγξετε και να προσαρμόσετε τα δικαιώματα για τις εφαρμογές που αποφασίζετε να κρατήσετε συνδεδεμένες. Μπορείτε να το κάνετε αυτό κάνοντας κλικ σε κάθε εφαρμογή και ελέγχοντας τα δικαιώματα που έχει.

![Ελέγξτε και προσαρμόστε τα δικαιώματα για τις εφαρμογές που αποφασίζετε να κρατήσετε συνδεδεμένες](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_4a76c5eb3c7a413d8bf7126d81eba014~mv2.png)

## Συμπέρασμα

Ο έλεγχος της ασφάλειας και της ιδιωτικότητάς σας στο διαδίκτυο είναι ζωτικής σημασίας στο σημερινό ψηφιακό τοπίο. Η αποσύνδεση εφαρμογών τρίτων μερών που δεν χρησιμοποιείτε πλέον ή δεν εμπιστεύεστε είναι ένα βήμα προς τη σωστή κατεύθυνση. Ακολουθώντας τα απλά βήματα που περιγράφονται σε αυτόν τον οδηγό, μπορείτε να διαχειριστείτε αποτελεσματικά την πρόσβαση στον λογαριασμό σας Google και να διασφαλίσετε ότι οι προσωπικές σας πληροφορίες παραμένουν ασφαλείς.

Θυμηθείτε ότι ενώ οι εφαρμογές τρίτων μερών μπορούν να βελτιώσουν την ψηφιακή σας εμπειρία, είναι σημαντικό να είστε επιλεκτικοί σχετικά με τις εφαρμογές που συνδέετε στον λογαριασμό σας Google. Ο τακτικός έλεγχος και η αποσύνδεση εφαρμογών που δεν χρειάζεστε πλέον είναι ένας προληπτικός τρόπος για την προστασία της διαδικτυακής σας ταυτότητας και των δεδομένων σας.

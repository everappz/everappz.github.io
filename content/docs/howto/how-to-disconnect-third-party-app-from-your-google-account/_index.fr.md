**En bref :** Allez sur [myaccount.google.com](https://myaccount.google.com/) > Sécurité > Applications et services tiers. Cliquez sur l'application que vous souhaitez supprimer, puis sélectionnez « Supprimer l'accès » ou « Supprimer toutes les connexions ». Répétez pour chaque application.

À l'ère numérique actuelle, nous lions souvent nos comptes Google à diverses applications et services tiers pour plus de commodité. Bien que cela puisse faciliter la vie, il est essentiel de maintenir le contrôle et la sécurité de votre compte Google. Parfois, vous pouvez vouloir déconnecter une application tierce que vous n'utilisez plus ou en laquelle vous n'avez plus confiance. Dans cet article, nous vous guiderons à travers le processus de déconnexion d'une application tierce de votre compte Google, vous aidant à assurer votre sécurité en ligne.

## Pourquoi déconnecter une application tierce ?

Il existe plusieurs raisons pour lesquelles vous pourriez vouloir déconnecter une application tierce de votre compte Google :

- Préoccupations de sécurité   
Si vous soupçonnez qu'une application tierce pourrait compromettre la sécurité de votre compte ou accéder à plus d'informations que nécessaire, il est préférable de couper la connexion.

- Applications inutilisées   
Au fil du temps, vous avez peut-être connecté diverses applications que vous n'utilisez plus. Les déconnecter réduit l'encombrement et vous aide à mieux contrôler votre compte.

- Confidentialité   
Déconnecter des applications peut aider à protéger votre confidentialité en limitant les données partagées avec ces services.

Maintenant, plongeons dans les étapes pour déconnecter une application tierce de votre compte Google.

## Étapes pour déconnecter une application tierce

### 1. Accéder aux paramètres de votre compte Google     
Ouvrez un navigateur web et allez sur [myaccount.google.com](https://myaccount.google.com/).

### 2. Se connecter   
Si vous n'êtes pas encore connecté, entrez les identifiants de votre compte Google pour accéder aux paramètres de votre compte.

### 3. Sécurité   
Dans le menu de gauche, sélectionnez « Sécurité ». Cette section vous permet de gérer les paramètres de sécurité de votre compte Google.

### 4. Applications tierces ayant accès au compte  
Faites défiler vers le bas jusqu'à la section intitulée « Applications et services tiers » et cliquez dessus. Ici, vous trouverez une liste des applications qui ont accès à votre compte Google.

![Applications tierces ayant accès au compte](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_c9a2599fb25b4325ac2de19396cadd56~mv2.png)

### 5. Examiner les applications tierces  
Examinez attentivement la liste des applications connectées. Ces applications ont différents niveaux d'accès aux données de votre compte Google.

![Examiner les applications tierces](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_676e0cc9fa4a46da97ef1cad418a3433~mv2.png)

### 6. Supprimer l'accès  
Pour déconnecter une application, cliquez dessus. Une fenêtre apparaîtra avec des détails sur l'accès de l'application. Ici, vous trouverez l'option « Supprimer l'accès », « Déconnecter » ou « Supprimer toutes les connexions ».

![Supprimer l'accès](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_a3cb1e7cd9964857b527cbb76ef7ddb1~mv2.png)

### 7. Confirmer la suppression    
Confirmez votre décision de déconnecter l'application. Google vous demandera de confirmer cette action pour s'assurer que vous faites un choix éclairé.

![Confirmer la suppression](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_07819d0a93f946b1b9f6c84803c53056~mv2.png)

### 8. Répéter si nécessaire     
Si vous avez plusieurs applications à déconnecter, répétez le processus pour chacune.

Après avoir déconnecté les applications, il est recommandé de vérifier et d'ajuster les autorisations des applications que vous décidez de garder connectées. Vous pouvez le faire en cliquant sur chaque application et en vérifiant les autorisations dont elle dispose.

![Vérifier et ajuster les autorisations des applications que vous décidez de garder connectées](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account/21260c_4a76c5eb3c7a413d8bf7126d81eba014~mv2.png)

## Conclusion

Prendre le contrôle de votre sécurité et de votre confidentialité en ligne est vital dans le paysage numérique actuel. Déconnecter les applications tierces que vous n'utilisez plus ou en lesquelles vous n'avez plus confiance est un pas dans la bonne direction. En suivant les étapes simples décrites dans ce guide, vous pouvez gérer efficacement l'accès à votre compte Google et vous assurer que vos informations personnelles restent sécurisées.

N'oubliez pas que si les applications tierces peuvent améliorer votre expérience numérique, il est crucial d'être sélectif quant aux applications que vous connectez à votre compte Google. Vérifier régulièrement et déconnecter les applications dont vous n'avez plus besoin est un moyen proactif de protéger votre identité et vos données en ligne.

## FAQ

{{% details title="La déconnexion d'une application supprimera-t-elle mes données de cette application ?" closed="true" %}}
Non. La suppression de l'accès empêche uniquement l'application d'accéder à votre compte Google à l'avenir. Les données déjà partagées avec l'application peuvent encore exister sur leurs serveurs. Vérifiez les propres paramètres de confidentialité de l'application pour supprimer ces données.
{{% /details %}}

{{% details title="Puis-je reconnecter une application après l'avoir déconnectée ?" closed="true" %}}
Oui. Si vous avez de nouveau besoin de l'application, connectez-vous simplement avec Google lorsque vous y êtes invité. L'application demandera à nouveau des autorisations, et vous pourrez les examiner avant d'accorder l'accès.
{{% /details %}}

{{% details title="À quelle fréquence dois-je vérifier l'accès des applications tierces ?" closed="true" %}}
Vérifiez vos applications connectées tous les 3 à 6 mois, ou immédiatement après avoir cessé d'utiliser un service. Des audits réguliers aident à maintenir votre compte sécurisé.
{{% /details %}}

{{% details title="Cela affecte-t-il des applications comme Evermusic qui se connectent à Google Drive ?" closed="true" %}}
Oui. Si vous déconnectez une application comme Evermusic ou Flacbox de votre compte Google, elle perdra l'accès à vos fichiers Google Drive. Vous pouvez vous reconnecter à tout moment depuis l'application.
{{% /details %}}

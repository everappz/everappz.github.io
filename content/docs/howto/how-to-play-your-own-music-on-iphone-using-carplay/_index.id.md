## Masukkan File ke Aplikasi

Pertama, Anda perlu menambahkan file ke aplikasi. Ada beberapa cara untuk melakukannya:

### Hubungkan ke penyimpanan cloud

- Buka tab '**Koneksi**'
- Pilih 'Hubungkan ke penyimpanan cloud' dari menu
- Pilih layanan penyimpanan cloud dari daftar
- Masukkan kredensial Anda, dan ketuk '**Selesai**.'

Jika Anda menemui masalah, pastikan untuk memeriksa koneksi internet dan login/kata sandi Anda.
Pada versi Premium aplikasi, Anda dapat menambahkan jumlah layanan yang tidak terbatas.

### Layanan penyimpanan cloud yang didukung

Saat ini, aplikasi mendukung layanan penyimpanan cloud paling populer: iCloud Drive, Dropbox, OneDrive, Google Drive, MEGA, Synology Drive, SMB, WebDAV, DLNA, Yandex.Disk, Box, 阿里云盘, 百度网盘, pCloud, WD My Cloud Home, InfiniCLOUD, MediaFire, OpenDrive, HiDrive, Cloud Mail.ru, Put.io, MyDrive. ## Aktifkan CarPlay di iPhone

### Persyaratan
- iPhone menjalankan iOS 13.0 atau lebih baru
- Mobil dengan dukungan **Apple CarPlay** (USB atau nirkabel)
- **Siri** harus diaktifkan di iPhone Anda

### Atur CarPlay melalui USB

1. Nyalakan mobil Anda dan pastikan **Siri** diaktifkan di iPhone Anda.
2. Hubungkan iPhone ke mobil menggunakan **kabel Lightning ke USB**.
3. Saat diminta di iPhone, ketuk **Izinkan** untuk mengaktifkan CarPlay.
4. CarPlay akan diluncurkan secara otomatis di layar infotainment mobil Anda.

### Atur CarPlay Nirkabel

1. Nyalakan mobil Anda dan pastikan **Bluetooth** dan **Wi-Fi** diaktifkan.
2. Di iPhone Anda, buka:
   - **Pengaturan > Umum > CarPlay**
3. Pilih mobil Anda dari daftar kendaraan yang tersedia.
4. Ikuti instruksi yang ditampilkan di layar mobil Anda untuk menyelesaikan pemasangan.

### Opsional: Sesuaikan Tata Letak CarPlay Anda

- Buka **Pengaturan > Umum > CarPlay**
- Pilih mobil Anda dari daftar
- Ketuk **Sesuaikan** untuk mengatur ulang atau menyembunyikan aplikasi

### Tips Pemecahan Masalah

- Pastikan mobil Anda dalam **mode pemasangan** (lihat manual mobil Anda)
- Restart iPhone dan sistem mobil Anda jika koneksi gagal
- Pastikan firmware mobil Anda sudah diperbarui

## Perpustakaan

Tab **Perpustakaan** di Evermusic adalah pusat utama tempat semua musik Anda terorganisir. Setelah Anda menambahkan trek ke perpustakaan — baik dari penyimpanan cloud, file lokal, atau iTunes — aplikasi secara otomatis membaca metadata dan mengelompokkannya dengan rapi ke dalam kategori seperti **Album**, **Artis**, **Genre**, dan lainnya.

Layar ini memberi Anda akses cepat ke favorit, terbaru, daftar putar, tandai halaman, dan semua trek yang ditambahkan. Anda juga dapat melanjutkan pemutaran dari sesi terakhir, melihat lagu yang belum diputar, dan menjelajahi musik berdasarkan tag atau jenis sumber.

## Koneksi

Tab **Koneksi** adalah pusat utama Anda untuk mengakses dan mengelola semua layanan penyimpanan cloud yang terhubung dan perangkat jaringan lokal.

{{< cards cols="1">}}
{{< card title="" subtitle="Koneksi" image="/docs/howto/how-to-play-your-own-music-on-iphone-using-carplay/connections-with-images-3.webp" imageStyle="border-radius: clamp(14px, 2vw, 28px);" >}}
{{< /cards >}}

Dari sini, Anda dapat terhubung ke platform cloud populer seperti Dropbox, Google Drive, OneDrive, MEGA, iCloud Drive, dan bahkan drive jaringan seperti SMB, DLNA, dan WebDAV. Setelah terhubung, Anda dapat menelusuri, streaming, mengunduh, dan mengelola file langsung dari dalam aplikasi.

Layar **Koneksi** mencakup bagian berikut:

- **File Favorit** – Akses cepat ke folder atau file yang telah ditandai sebagai favorit untuk navigasi cepat, sangat berguna di CarPlay.
- **File yang Baru Dibuka** – Lihat file terbaru yang telah Anda akses dari layanan yang terhubung.
- **Penyimpanan Cloud** – Kelola semua layanan cloud dan drive jaringan lokal yang terhubung.

Anda dapat menghubungkan layanan cloud baru menggunakan opsi **"Hubungkan ke Penyimpanan Cloud"** di layar Koneksi utama. Setelah terhubung, Anda dapat menavigasi folder, menelusuri perpustakaan musik, dan bahkan streaming atau mengunduh trek saat bepergian. Jika folder sangat dalam, Anda dapat **menandainya sebagai Favorit** untuk akses mudah nanti — sangat membantu bagi pengguna CarPlay yang menginginkan navigasi cepat.

Anda juga dapat menggunakan **Wi-Fi Drive** atau **Berbagi File iTunes** untuk mengimpor musik dari komputer secara nirkabel atau melalui USB. Dukungan untuk menghubungkan ke perangkat NAS, drive eksternal, dan jaringan lokal memberi Anda kontrol penuh atas cara Anda menyimpan dan mengakses musik.

Untuk mempelajari lebih lanjut tentang semua cara menghubungkan dan mengelola perpustakaan cloud dan jaringan Anda, baca [Panduan Koneksi](/docs/guide/evermusic/evermusic-guide-connections/) lengkap.

## File Lokal

Bagian **File Lokal** adalah pusat utama Anda untuk mengelola file audio yang disimpan langsung di perangkat atau di dalam direktori **Dokumen** aplikasi Evermusic. Ini juga mencakup file offline yang diunduh dari penyimpanan cloud, file cache pemutar audio, dan folder yang telah Anda buat tersedia untuk pemutaran offline. Bagian ini memastikan Anda dapat menikmati perpustakaan musik bahkan tanpa koneksi internet.

Layar **File Lokal** diatur ke dalam bagian-bagian utama berikut:

- **File Favorit** – Pintasan ke folder atau file yang telah ditandai sebagai favorit. Sangat berguna untuk CarPlay atau akses cepat.
- **File yang Baru Dibuka** – Trek dan folder yang baru-baru ini dibuka dari penyimpanan lokal.
- **Dokumen** – Menampilkan semua file dan folder yang terletak di direktori Dokumen internal aplikasi. Termasuk item yang ditambahkan melalui unduhan cloud, Wi-Fi Drive, Berbagi File iTunes, atau transfer USB.
- **Folder offline** – Menampilkan semua folder yang telah ditandai untuk penggunaan offline (misalnya folder cloud, album, daftar putar, atau koleksi artis yang diunduh untuk pemutaran offline).

- **Pemutar Audio** – Folder cache yang digunakan untuk crossfade dan optimisasi kinerja. Dapat dinonaktifkan atau dibersihkan di pengaturan.

{{< cards cols="1">}}
{{< card title="" subtitle="Batas Kedalaman Konten" image="/docs/howto/how-to-play-your-own-music-on-iphone-using-carplay/content-depth-limit.webp" imageStyle="border-radius: clamp(14px, 2vw, 28px);" >}}

Saat menggunakan CarPlay, Anda mungkin menemui kesalahan **"Batas Kedalaman Konten"** — terutama jika perpustakaan musik Anda memiliki banyak folder yang sangat bertingkat.
Ini terjadi karena keterbatasan dalam sistem CarPlay, yang membatasi berapa banyak level folder yang dapat ditampilkan.

**Cara Memperbaikinya**

Untuk menghindari masalah ini dan mengakses musik Anda lebih cepat:

- **Tambahkan folder yang sangat bertingkat ke Favorit**
  Ini membuatnya mudah diakses langsung dari menu **Favorit** di CarPlay.

- **Sederhanakan struktur folder Anda**
  Jika memungkinkan, coba ratakan hierarki folder Anda untuk navigasi CarPlay yang lebih lancar.

Solusi ini memastikan pengalaman yang mulus saat menelusuri musik di mobil.

{{% details title="Mengapa saya melihat kesalahan Batas Kedalaman Konten di CarPlay?" closed="true" %}}
CarPlay membatasi berapa banyak level folder yang dapat ditampilkan. Jika musik Anda berada di folder yang sangat bertingkat, tambahkan folder tersebut ke Favorit sehingga Anda dapat mengaksesnya langsung dari menu Favorit di CarPlay.
{{% /details %}}

{{% details title="Apakah Evermusic atau Flacbox gratis digunakan dengan CarPlay?" closed="true" %}}
Kedua aplikasi gratis untuk diunduh dengan dukungan CarPlay penuh, equalizer, dan fitur pemutaran. Versi gratis memiliki batasan pada koneksi cloud (3), daftar putar (10), dan folder offline (1). Premium menghapus semua batasan.

**בקצרה:** הגדירו את פלט האודיו של Evermusic ל"מצב מעורב", התחילו להשמיע שיר, ואז פתחו את אפליקציית המצלמה והקליטו. המוזיקה ממשיכה לנגן ברקע. עובד עם TikTok, Instagram וכל אפליקציית מצלמה.

הרגע סיפרנו לכם על טריק קטן שמאפשר לכם להשמיע מוזיקה ברקע בזמן הקלטת וידאו באייפון. ייתכן שתמצאו את זה שימושי בפעם הבאה שתחליטו ליצור סרטון חדש ומגניב ל-TikTok.

## הורידו את האפליקציה

פתחו את App Store וחפשו את האפליקציה בשם [Evermusic](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198). הורידו אותה בחינם.

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_9430a2a876cf4747b996044a6493696d~mv2.jpg" alt="Evermusic ב-App Store" width="400" >}}

## ייבוא מוזיקה
הוסיפו מוזיקה באמצעות [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes), [שיתוף קבצי iTunes](/docs/howto/how-to-play-local-itunes-files-on-my-iphone) או ייבאו מוזיקה קיימת באייפון שלכם.

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_e1998cc648c24dbcb081c8ce76578715~mv2.jpg" alt="הוספת מוזיקה ל-Evermusic" width="400" >}}

## שינוי מצב פלט האודיו

פתחו את "הגדרות" האפליקציה - "נגן אודיו" - "כללי" - "מצב פלט אודיו" ושנו ל"מצב מעורב".

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_59fdf74049e6464f8e9611fdcf9360af~mv2.jpg" alt="מצב פלט אודיו - מעורב" width="400" >}}

## הפעלת הנגן
הפעילו את נגן האודיו על ידי לחיצה על כל שיר באפליקציה.

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_5b2c8042b10a4678a7e45b16406f8449~mv2.jpg" alt="נגן אודיו Evermusic" width="400" >}}

## פתחו את אפליקציית המצלמה

פתחו את אפליקציית המצלמה או כל אפליקציה אחרת והתחילו להקליט.   
הטריק הזה עובד על כל אייפון.

## שאלות נפוצות

{{% details title="האם מוזיקת הרקע מוקלטת לתוך הווידאו?" closed="true" %}}
המוזיקה מושמעת דרך הרמקול של האייפון, כך שהמיקרופון יקלוט אותה יחד עם רעשי סביבה אחרים. לאודיו נקי יותר, שקלו להשתמש ברמקול חיצוני הממוקם ליד המיקרופון.
{{% /details %}}

{{% details title="האם זה עובד עם TikTok ו-Instagram?" closed="true" %}}
כן. ברגע ש-Evermusic מוגדר למצב מעורב ושיר מתנגן, המוזיקה ממשיכה כשפותחים את TikTok, Instagram או כל אפליקציית מצלמה או הקלטה אחרת.
{{% /details %}}

{{% details title="מהו מצב מעורב ב-Evermusic?" closed="true" %}}
מצב מעורב הוא הגדרת פלט אודיו שמאפשרת ל-Evermusic לשתף את הפעלת האודיו עם אפליקציות אחרות. זה מונע מהמוזיקה להפסיק כאשר אפליקציה אחרת ניגשת למיקרופון או למצלמה.
{{% /details %}}

{{% details title="האם אפשר להשתמש ב-Flacbox במקום Evermusic לשם כך?" closed="true" %}}
כן. גם Flacbox תומך במצב פלט אודיו מעורב. השלבים זהים: הפעילו מצב מעורב בהגדרות, התחילו השמעה ופתחו את אפליקציית המצלמה.
{{% /details %}}

{{% details title="האם אפשר לשחק משחק בזמן שמוזיקה מ-Evermusic מתנגנת ברקע?" closed="true" %}}
כן. כאשר מצב מעורב מופעל, המוזיקה מ-Evermusic ממשיכה להתנגן כשפותחים כל משחק או אפליקציה. גם האודיו של המשחק וגם המוזיקה שלכם יושמעו בו-זמנית.
{{% /details %}}

**要約：** Evermusicのオーディオ出力を「混合モード」に設定し、曲の再生を開始してから、カメラアプリを開いて録画します。音楽はバックグラウンドで再生され続けます。TikTok、Instagram、その他すべてのカメラアプリで動作します。

iPhoneでビデオを録画しながらバックグラウンドで音楽を再生できる小さなテクニックをお伝えしました。次にTikTok用の新しいクールなビデオを作成する際に役立つかもしれません。

## アプリをダウンロード

App Storeを開き、[Evermusic](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198)というアプリを検索してください。無料でダウンロードできます。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_9430a2a876cf4747b996044a6493696d~mv2.jpg" alt="App StoreのEvermusic" width="400" >}}

## 音楽をインポート
[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)、[iTunesファイル共有](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)を使って音楽を追加するか、iPhone上の既存の音楽をインポートしてください。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_e1998cc648c24dbcb081c8ce76578715~mv2.jpg" alt="Evermusicに音楽を追加" width="400" >}}

## オーディオ出力モードを変更

アプリの「設定」-「オーディオプレーヤー」-「一般」-「オーディオ出力モード」を開き、「混合モード」に変更します。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_59fdf74049e6464f8e9611fdcf9360af~mv2.jpg" alt="オーディオ出力モード - 混合" width="400" >}}

## プレーヤーを開始
アプリ内の任意のトラックをタップしてオーディオプレーヤーを開始します。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_5b2c8042b10a4678a7e45b16406f8449~mv2.jpg" alt="Evermusicオーディオプレーヤー" width="400" >}}

## カメラアプリを開く

カメラアプリまたは他のアプリケーションを開いて録画を開始します。   
このテクニックはすべてのiPhoneで動作します。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_fff02309eeb74e7fa8d73d948ad6157f~mv2.jpg" alt="音楽再生中のiPhoneカメラ" width="400" >}}

## よくある質問

{{% details title="バックグラウンドミュージックはビデオに録音されますか？" closed="true" %}}
音楽はiPhoneのスピーカーを通じて再生されるため、マイクが周囲の音と一緒に拾います。よりクリアなオーディオを得るには、マイクの近くに外部スピーカーを設置することを検討してください。
{{% /details %}}

{{% details title="TikTokやInstagramでも動作しますか？" closed="true" %}}
はい。Evermusicが混合モードに設定され、トラックが再生されている状態で、TikTok、Instagram、その他のカメラや録画アプリを開いても音楽は継続されます。
{{% /details %}}

{{% details title="Evermusicの混合モードとは何ですか？" closed="true" %}}
混合モードは、Evermusicが他のアプリとオーディオセッションを共有できるようにするオーディオ出力設定です。これにより、別のアプリがマイクやカメラにアクセスしても音楽が停止するのを防ぎます。
{{% /details %}}

{{% details title="Evermusicの代わりにFlacboxを使用できますか？" closed="true" %}}
はい。Flacboxも混合オーディオ出力モードをサポートしています。手順は同じです：設定で混合モードを有効にし、再生を開始してからカメラアプリを開いてください。
{{% /details %}}

{{% details title="Evermusicの音楽をバックグラウンドで再生しながらゲームをプレイできますか？" closed="true" %}}
はい。混合モードを有効にすると、ゲームやアプリを開いてもEvermusicの音楽は再生され続けます。ゲームのオーディオと音楽が同時に再生されます。
{{% /details %}}

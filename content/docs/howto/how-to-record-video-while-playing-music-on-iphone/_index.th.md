**สรุป:** ตั้งค่าเอาต์พุตเสียงของ Evermusic เป็น "โหมดผสม" เริ่มเล่นเพลง จากนั้นเปิดแอปกล้องและบันทึก เพลงจะเล่นต่อในพื้นหลัง ใช้งานได้กับ TikTok, Instagram และแอปกล้องทุกตัว

เราเพิ่งบอกเคล็ดลับเล็กๆ ที่ช่วยให้คุณเล่นเพลงในพื้นหลังขณะถ่ายวิดีโอบน iPhone คุณอาจพบว่าสิ่งนี้มีประโยชน์ในครั้งต่อไปที่คุณต้องการสร้างวิดีโอเจ๋งๆ สำหรับ TikTok

## ดาวน์โหลดแอป

เปิด App Store และค้นหาแอปชื่อ [Evermusic](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198) ดาวน์โหลดได้ฟรี

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_9430a2a876cf4747b996044a6493696d~mv2.jpg" alt="Evermusic บน App Store" width="400" >}}

## นำเข้าเพลง
เพิ่มเพลงโดยใช้ [WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes), [การแชร์ไฟล์ iTunes](/docs/howto/how-to-play-local-itunes-files-on-my-iphone) หรือนำเข้าเพลงที่มีอยู่ใน iPhone ของคุณ

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_e1998cc648c24dbcb081c8ce76578715~mv2.jpg" alt="เพิ่มเพลงใน Evermusic" width="400" >}}

## เปลี่ยนโหมดเอาต์พุตเสียง

เปิด "การตั้งค่า" ของแอป - "เครื่องเล่นเสียง" - "ทั่วไป" - "โหมดเอาต์พุตเสียง" และเปลี่ยนเป็น "โหมดผสม"

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_59fdf74049e6464f8e9611fdcf9360af~mv2.jpg" alt="โหมดเอาต์พุตเสียง - ผสม" width="400" >}}

## เริ่มเครื่องเล่น
เริ่มเครื่องเล่นเสียงโดยแตะที่เพลงใดก็ได้ในแอป

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_5b2c8042b10a4678a7e45b16406f8449~mv2.jpg" alt="เครื่องเล่นเสียง Evermusic" width="400" >}}

## เปิดแอปกล้อง

เปิดแอปกล้องหรือแอปพลิเคชันอื่นๆ และเริ่มบันทึก   
เคล็ดลับนี้ใช้ได้กับ iPhone ทุกรุ่น

## คำถามที่พบบ่อย

{{% details title="เพลงพื้นหลังจะถูกบันทึกลงในวิดีโอหรือไม่?" closed="true" %}}
เพลงจะเล่นผ่านลำโพง iPhone ดังนั้นไมโครโฟนจะรับเสียงเพลงพร้อมกับเสียงรอบข้างอื่นๆ หากต้องการเสียงที่สะอาดขึ้น ลองใช้ลำโพงภายนอกวางใกล้ไมโครโฟน
{{% /details %}}

{{% details title="ใช้งานได้กับ TikTok และ Instagram หรือไม่?" closed="true" %}}
ได้ เมื่อตั้งค่า Evermusic เป็นโหมดผสมและเพลงกำลังเล่นอยู่ เพลงจะเล่นต่อเมื่อคุณเปิด TikTok, Instagram หรือแอปกล้องหรือบันทึกอื่นๆ
{{% /details %}}

{{% details title="โหมดผสมใน Evermusic คืออะไร?" closed="true" %}}
โหมดผสมเป็นการตั้งค่าเอาต์พุตเสียงที่ช่วยให้ Evermusic แชร์เซสชันเสียงกับแอปอื่นๆ ได้ ซึ่งป้องกันไม่ให้เพลงหยุดเมื่อแอปอื่นเข้าถึงไมโครโฟนหรือกล้อง
{{% /details %}}

{{% details title="สามารถใช้ Flacbox แทน Evermusic ได้หรือไม่?" closed="true" %}}
ได้ Flacbox รองรับโหมดเอาต์พุตเสียงผสมเช่นกัน ขั้นตอนเหมือนกัน: เปิดใช้งานโหมดผสมในการตั้งค่า เริ่มเล่นเพลง และเปิดแอปกล้อง
{{% /details %}}

{{% details title="สามารถเล่นเกมขณะที่เพลงจาก Evermusic เล่นในพื้นหลังได้หรือไม่?" closed="true" %}}
ได้ เมื่อเปิดใช้งานโหมดผสม เพลงจาก Evermusic จะเล่นต่อเมื่อคุณเปิดเกมหรือแอปใดๆ เสียงเกมและเพลงของคุณจะเล่นพร้อมกัน
{{% /details %}}

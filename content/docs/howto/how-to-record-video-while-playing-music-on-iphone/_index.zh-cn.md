**简介：** 将Evermusic的音频输出设置为"混合模式"，开始播放曲目，然后打开相机应用并录制。音乐将在后台继续播放。适用于TikTok、Instagram和任何相机应用。

我们刚刚分享了一个小技巧，可以让您在iPhone上录制视频时在后台播放音乐。下次您想为TikTok创建新的酷炫视频时，可能会发现这很有用。

## 下载应用

打开App Store并搜索名为[Evermusic](https://apps.apple.com/app/evermusic-cloud-music-player/id885367198)的应用。免费下载。

## 导入音乐
使用[WiFi Drive](/docs/howto/how-to-transfer-music-from-computer-to-iphone-without-itunes)、[iTunes文件共享](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)添加音乐，或导入iPhone上现有的音乐。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_e1998cc648c24dbcb081c8ce76578715~mv2.jpg" alt="将音乐添加到Evermusic" width="400" >}}

## 更改音频输出模式

打开应用"设置" - "音频播放器" - "一般" - "音频输出模式"，并将其更改为"混合模式"。

{{< figure src="/docs/howto/how-to-record-video-while-playing-music-on-iphone/21260c_59fdf74049e6464f8e9611fdcf9360af~mv2.jpg" alt="音频输出模式 - 混合" width="400" >}}

## 启动播放器
点击应用中的任何曲目以启动音频播放器。

## 打开相机应用

打开相机应用或任何其他应用并开始录制。   
此技巧适用于每一部iPhone。

## 常见问题

{{% details title="背景音乐会被录入视频吗？" closed="true" %}}
音乐通过iPhone扬声器播放，因此麦克风会将其与其他环境声音一起录入。要获得更清晰的音频，请考虑使用放置在麦克风附近的外部扬声器。
{{% /details %}}

{{% details title="这在TikTok和Instagram上有效吗？" closed="true" %}}
是的。一旦Evermusic设置为混合模式并且正在播放曲目，当您打开TikTok、Instagram或任何其他相机或录制应用时，音乐会继续播放。
{{% /details %}}

{{% details title="Evermusic中的混合模式是什么？" closed="true" %}}
混合模式是一种音频输出设置，允许Evermusic与其他应用共享音频会话。这可以防止当其他应用访问麦克风或相机时音乐停止。
{{% /details %}}

{{% details title="我可以使用Flacbox代替Evermusic吗？" closed="true" %}}
可以。Flacbox也支持混合音频输出模式。步骤相同：在设置中启用混合模式，开始播放，然后打开相机应用。
{{% /details %}}

{{% details title="Evermusic在后台播放音乐时可以玩游戏吗？" closed="true" %}}
可以。启用混合模式后，打开任何游戏或应用时，Evermusic的音乐会继续播放。游戏音频和您的音乐将同时播放。
{{% /details %}}

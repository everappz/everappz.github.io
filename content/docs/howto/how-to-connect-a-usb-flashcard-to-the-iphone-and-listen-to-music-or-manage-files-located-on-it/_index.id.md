**Ringkasan:** Hubungkan USB flash drive atau kartu SD ke iPhone Anda menggunakan adaptor Apple atau drive SanDisk iXpand, lalu gunakan Evermusic, Flacbox, atau Evertag untuk menelusuri, memutar, dan mengelola file audio Anda langsung dari penyimpanan eksternal.

Ada beberapa cara untuk menghubungkan USB flash atau kartu SD ke iPhone dan mendengarkan musik yang ada di dalamnya. Anda juga dapat mengelola file Anda dengan pengelola file bawaan di aplikasi kami Evermusic, Flacbox, Evertag.

Cara pertama adalah menghubungkan USB flash Anda menggunakan [Lightning to SD Card Camera Reader](https://www.google.com/search?q=Lightning+to+SD+Card+Camera+Reader) standar

![Pembaca kartu Apple Lightning 1](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e81edb4d52d8419790468aa37bc191ce~mv2.jpeg)

atau menghubungkan USB flash Anda menggunakan [Apple Lightning to USB 3 Camera Adapter](https://www.apple.com/shop/product/MX5J3AM/A/lightning-to-usb-3-camera-adapter)

![Pembaca kartu Apple Lightning 2](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_33bca1e41db84a0a9fedfcbe804830de~mv2.jpeg)

Dalam hal ini, buka aplikasi Evermusic, Flacbox, atau Evertag dan navigasi ke tab "File Lokal". Gulir ke bawah ke bagian "File di iPhone ini" dan ketuk "Buka file..." untuk mengaktifkan "Pemilih File Eksternal".

![Evermusic: Bagian file di iPhone ini](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_60686b5f3e0d43a0a169dfae25462e46~mv2.png)

Di layar "Pemilih File Eksternal", pindah ke layar utama dan gulir ke bawah bagian "Lokasi" untuk menemukan perangkat USB yang terhubung.

![Evermusic: Pilih aksesori USB yang terhubung](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e4cc6085cbe94e54b6d0ee5cbb654119~mv2.png)

Buka perangkat dan pilih file yang ingin Anda impor ke aplikasi.

![Evermusic: Pilih file di aksesori USB yang terhubung](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_f67ec02898ec4021aeff54a822defae1~mv2.png)

File yang diimpor terletak di perangkat USB Anda dan jika Anda ingin menyalinnya ke aplikasi, jawab "Salin file" pada pesan konfirmasi.

![Evermusic: Pesan konfirmasi salin file](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_3152e675eacf49b3bde5f4623b8ad666~mv2.png)

Tetapi ada juga opsi lain yang tersedia karena aplikasi kami mendukung drive SanDisk iXpand Flash secara native.

Anda dapat membeli SanDisk di sini: [SanDisk iXpand Mini](https://www.google.com/search?q=sandisk+ixpand+mini)

Saat ini, aplikasi mendukung drive SanDisk Flash dengan protokol iXpand V1, iXpand V2, iXpand V3, iXpand V6, iXpand V7. Anda dapat memeriksa apakah iXpand Flash Drive Anda didukung menggunakan langkah-langkah berikut:

- **Hubungkan** drive ke perangkat iOS
- Buka **Pengaturan**
- Ketuk **Umum**
- Ketuk **Tentang**
- Cari **iXpand Flash Drive**
- Ketuk **Temukan Aplikasi untuk Aksesori**
- Aplikasi seharusnya terdaftar di sana

![Layar pengaturan drive iXpand flash](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_97e7a70f78a54bada207e1f529b33b22~mv2.png)

Setelah Anda menemukan iXpand Flash Drive yang didukung, misalnya: [iXpand Mini Flash Drive](https://www.westerndigital.com/en-ap/products/usb-flash-drives/sandisk-ixpand-mini-usb-3-0?sku=SDIX40N-016G-GN6NN), masukkan ke port Lightning dan buka aplikasi. Anda akan melihat pesan "Perangkat terhubung" dan informasi perangkat.

Cukup buka layar "Koneksi" dan gulir ke bagian "Aksesori terhubung".

![Evermusic: Aksesori terhubung](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_8416a9ee12dc42039a030948790a73e5~mv2.png)

Ketuk ikon aksesori yang terhubung untuk membukanya dan mendapatkan daftar file dan folder yang ada di perangkat USB. Dalam beberapa kasus, aplikasi lain mungkin menggunakan drive Anda dan mengatur status sibuk. Dalam hal ini, Anda akan melihat pesan peringatan. Tidak ada yang serius - cukup tutup semua aplikasi lain yang menggunakan flash drive Anda atau cabut dan pasang kembali.

![Evermusic: Pengelola File](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_cb43f9e4b57c4fa597a6051289c41485~mv2.png)

Setelah Anda membuka folder apa pun yang berisi musik, Anda akan melihat semua file audio yang tersedia. Cukup ketuk file apa pun untuk memulai pemutaran. Anda dapat mengaktifkan equalizer di layar pemutar atau streaming konten audio ke AirPlay (Sonos, Apple TV) atau perangkat Google Chromecast.

Anda dapat menggunakan pengelola file bawaan untuk MENYALIN, MEMINDAHKAN, MENGGANTI NAMA, MENGHAPUS file Anda. Anda juga dapat menambahkan file audio ke perpustakaan musik di aplikasi Evermusic atau Flacbox. Dan tentu saja Anda dapat menyalin file audio dari flash drive ke memori iPhone Anda.

Nikmati kebebasan mengakses dan mengelola musik Anda dengan mudah menggunakan langkah-langkah sederhana ini, dan ucapkan selamat tinggal pada keterbatasan penyimpanan!

## FAQ

{{% details title="Adaptor USB mana yang berfungsi dengan iPhone untuk pemutaran musik?" closed="true" %}}
Lightning to SD Card Camera Reader dan Lightning to USB 3 Camera Adapter dari Apple keduanya berfungsi. Adaptor USB-C berfungsi pada iPhone yang lebih baru dengan port USB-C. Drive SanDisk iXpand Flash (V1-V7) juga didukung secara native oleh Evermusic, Flacbox, dan Evertag.
{{% /details %}}

{{% details title="Bisakah saya memutar musik langsung dari drive USB tanpa menyalin file?" closed="true" %}}
Ya. Dengan drive SanDisk iXpand, Anda dapat memutar musik langsung dari drive tanpa menyalin file ke iPhone Anda. Saat menggunakan adaptor Apple, file diimpor tetapi Anda dapat memilih apakah akan menyalinnya ke penyimpanan lokal.
{{% /details %}}

{{% details title="Format audio apa yang didukung dari drive USB?" closed="true" %}}
Evermusic dan Flacbox mendukung berbagai format termasuk FLAC, MP3, AAC, WAV, ALAC, OGG, WMA, dan lainnya. Semua format yang didukung berfungsi saat memutar dari penyimpanan USB.
{{% /details %}}

{{% details title="SanDisk iXpand saya menunjukkan kesalahan 'sibuk'. Apa yang harus saya lakukan?" closed="true" %}}
Aplikasi lain mungkin mengakses drive. Tutup semua aplikasi lain yang mungkin menggunakan flash drive, atau cabut dan masukkan kembali. Kemudian buka kembali Evermusic, Flacbox, atau Evertag.
{{% /details %}}

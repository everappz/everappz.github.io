**Riepilogo:** Collega una chiavetta USB o una scheda SD al tuo iPhone usando un adattatore Apple o un'unità SanDisk iXpand, quindi usa Evermusic, Flacbox o Evertag per sfogliare, riprodurre e gestire i tuoi file audio direttamente dall'archiviazione esterna.

Ci sono diversi modi per collegare una chiavetta USB o una scheda SD all'iPhone e ascoltare la musica su di essa. Puoi anche gestire i tuoi file con il file manager integrato nelle nostre app Evermusic, Flacbox, Evertag.

Il primo modo è collegare la tua chiavetta USB usando il [Lightning to SD Card Camera Reader](https://www.google.com/search?q=Lightning+to+SD+Card+Camera+Reader) standard

![Lettore di schede Apple Lightning 1](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e81edb4d52d8419790468aa37bc191ce~mv2.jpeg)

o collegare la tua chiavetta USB usando l'[Apple Lightning to USB 3 Camera Adapter](https://www.apple.com/shop/product/MX5J3AM/A/lightning-to-usb-3-camera-adapter)

![Lettore di schede Apple Lightning 2](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_33bca1e41db84a0a9fedfcbe804830de~mv2.jpeg)

In questo caso, apri l'app Evermusic, Flacbox o Evertag e vai alla scheda "File locali". Scorri verso il basso fino alla sezione "File su questo iPhone" e tocca "Apri file..." per attivare il "Selettore file esterni".

![Evermusic: Sezione file su questo iPhone](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_60686b5f3e0d43a0a169dfae25462e46~mv2.png)

Nella schermata "Selettore file esterni", vai alla schermata principale e scorri verso il basso nella sezione "Posizione" per trovare un dispositivo USB collegato.

![Evermusic: Seleziona accessorio USB collegato](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e4cc6085cbe94e54b6d0ee5cbb654119~mv2.png)

Apri il dispositivo e seleziona i file che vuoi importare nell'app.

![Evermusic: Seleziona file sull'accessorio USB collegato](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_f67ec02898ec4021aeff54a822defae1~mv2.png)

I file importati si trovano sul tuo dispositivo USB e se vuoi copiarli nell'app, rispondi "Copia file" al messaggio di conferma.

![Evermusic: Messaggio di conferma copia file](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_3152e675eacf49b3bde5f4623b8ad666~mv2.png)

Ma c'è anche un'altra opzione disponibile poiché le nostre app supportano nativamente le unità SanDisk iXpand Flash.

Puoi acquistare SanDisk qui: [SanDisk iXpand Mini](https://www.google.com/search?q=sandisk+ixpand+mini)

Attualmente, l'app supporta le unità SanDisk Flash con protocolli iXpand V1, iXpand V2, iXpand V3, iXpand V6, iXpand V7. Puoi verificare se il tuo iXpand Flash Drive è supportato seguendo questi passaggi:

- **Collega** l'unità al dispositivo iOS
- Apri **Impostazioni**
- Tocca **Generale**
- Tocca **Info**
- Cerca **iXpand Flash Drive**
- Tocca **Trova app per accessorio**
- L'app dovrebbe essere elencata lì

![Schermata impostazioni unità iXpand flash](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_97e7a70f78a54bada207e1f529b33b22~mv2.png)

Una volta trovata l'unità iXpand Flash Drive supportata, ad esempio: [iXpand Mini Flash Drive](https://www.westerndigital.com/en-ap/products/usb-flash-drives/sandisk-ixpand-mini-usb-3-0?sku=SDIX40N-016G-GN6NN), inseriscila nella porta Lightning e apri l'applicazione. Vedrai il messaggio "Dispositivo collegato" e le informazioni del dispositivo.

Basta aprire la schermata "Connessioni" e scorrere fino alla sezione "Accessori connessi".

![Evermusic: Accessori connessi](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_8416a9ee12dc42039a030948790a73e5~mv2.png)

Tocca l'icona dell'accessorio collegato per aprirlo e ottenere l'elenco dei file e delle cartelle presenti sul dispositivo USB. In alcuni casi, altre app potrebbero utilizzare la tua unità e impostare lo stato occupato. In questo caso, vedrai un messaggio di avviso. Niente di grave - chiudi semplicemente tutte le altre app che utilizzano la tua chiavetta o scollegala e ricollegala.

![Evermusic: File Manager](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_cb43f9e4b57c4fa597a6051289c41485~mv2.png)

Una volta aperta qualsiasi cartella con musica, vedrai tutti i file audio disponibili. Basta toccare qualsiasi file per avviare la riproduzione. Puoi attivare un equalizzatore nella schermata del lettore o trasmettere contenuti audio su AirPlay (Sonos, Apple TV) o un dispositivo Google Chromecast.

Puoi utilizzare il file manager integrato per COPIARE, SPOSTARE, RINOMINARE, ELIMINARE i tuoi file. Puoi anche aggiungere file audio alla libreria musicale nelle app Evermusic o Flacbox. E naturalmente puoi copiare file audio dalla chiavetta alla memoria del tuo iPhone.

Goditi la libertà di accedere e gestire la tua musica senza sforzo con questi semplici passaggi, e di' addio alle limitazioni di archiviazione!

## FAQ

{{% details title="Quali adattatori USB funzionano con iPhone per la riproduzione musicale?" closed="true" %}}
Sia il Lightning to SD Card Camera Reader che il Lightning to USB 3 Camera Adapter di Apple funzionano. Gli adattatori USB-C funzionano sui nuovi iPhone con porte USB-C. Anche le unità SanDisk iXpand Flash (V1-V7) sono supportate nativamente da Evermusic, Flacbox ed Evertag.
{{% /details %}}

{{% details title="Posso riprodurre musica direttamente dall'unità USB senza copiare i file?" closed="true" %}}
Sì. Con le unità SanDisk iXpand, puoi riprodurre musica direttamente dall'unità senza copiare file sul tuo iPhone. Quando usi adattatori Apple, i file vengono importati ma puoi scegliere se copiarli nell'archiviazione locale.
{{% /details %}}

{{% details title="Quali formati audio sono supportati dalle unità USB?" closed="true" %}}
Evermusic e Flacbox supportano un'ampia gamma di formati tra cui FLAC, MP3, AAC, WAV, ALAC, OGG, WMA e altri. Tutti i formati supportati funzionano durante la riproduzione dall'archiviazione USB.
{{% /details %}}

{{% details title="Il mio SanDisk iXpand mostra un errore 'occupato'. Cosa devo fare?" closed="true" %}}
Un'altra app potrebbe star accedendo all'unità. Chiudi tutte le altre app che potrebbero utilizzare la chiavetta, oppure scollegala e reinseriscila. Poi riapri Evermusic, Flacbox o Evertag.
{{% /details %}}

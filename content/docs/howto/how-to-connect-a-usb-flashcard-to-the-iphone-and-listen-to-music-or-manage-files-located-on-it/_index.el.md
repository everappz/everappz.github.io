**Περίληψη:** Συνδέστε ένα USB flash drive ή κάρτα SD στο iPhone σας χρησιμοποιώντας έναν προσαρμογέα Apple ή μονάδα SanDisk iXpand και στη συνέχεια χρησιμοποιήστε τα Evermusic, Flacbox ή Evertag για να περιηγηθείτε, να αναπαράγετε και να διαχειριστείτε τα αρχεία ήχου σας απευθείας από τον εξωτερικό αποθηκευτικό χώρο.

Υπάρχουν αρκετοί τρόποι για να συνδέσετε ένα USB flash ή κάρτα SD στο iPhone και να ακούσετε μουσική που βρίσκεται σε αυτό. Μπορείτε επίσης να διαχειριστείτε τα αρχεία σας με τον ενσωματωμένο διαχειριστή αρχείων στις εφαρμογές μας Evermusic, Flacbox, Evertag.

Ο πρώτος τρόπος είναι να συνδέσετε το USB flash σας χρησιμοποιώντας τον τυπικό [Lightning to SD Card Camera Reader](https://www.google.com/search?q=Lightning+to+SD+Card+Camera+Reader)

![Αναγνώστης καρτών Apple Lightning 1](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e81edb4d52d8419790468aa37bc191ce~mv2.jpeg)

ή να συνδέσετε το USB flash σας χρησιμοποιώντας τον [Apple Lightning to USB 3 Camera Adapter](https://www.apple.com/shop/product/MX5J3AM/A/lightning-to-usb-3-camera-adapter)

![Αναγνώστης καρτών Apple Lightning 2](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_33bca1e41db84a0a9fedfcbe804830de~mv2.jpeg)

Σε αυτή την περίπτωση, ανοίξτε την εφαρμογή Evermusic, Flacbox ή Evertag και μεταβείτε στην καρτέλα "Τοπικά αρχεία". Κάντε κύλιση προς τα κάτω στην ενότητα "Αρχεία σε αυτό το iPhone" και πατήστε "Άνοιγμα αρχείων..." για να ενεργοποιήσετε τον "Επιλογέα εξωτερικών αρχείων".

![Evermusic: Ενότητα αρχείων σε αυτό το iPhone](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_60686b5f3e0d43a0a169dfae25462e46~mv2.png)

Στην οθόνη "Επιλογέας εξωτερικών αρχείων", μεταβείτε στην αρχική οθόνη και κάντε κύλιση προς τα κάτω στην ενότητα "Τοποθεσία" για να βρείτε μια συνδεδεμένη συσκευή USB.

![Evermusic: Επιλογή συνδεδεμένου USB αξεσουάρ](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_e4cc6085cbe94e54b6d0ee5cbb654119~mv2.png)

Ανοίξτε τη συσκευή και επιλέξτε τα αρχεία που θέλετε να εισαγάγετε στην εφαρμογή.

![Evermusic: Επιλογή αρχείων σε συνδεδεμένο USB αξεσουάρ](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_f67ec02898ec4021aeff54a822defae1~mv2.png)

Τα εισαγόμενα αρχεία βρίσκονται στη συσκευή USB σας και αν θέλετε να τα αντιγράψετε στην εφαρμογή, απαντήστε "Αντιγραφή αρχείων" στο μήνυμα επιβεβαίωσης.

![Evermusic: Μήνυμα επιβεβαίωσης αντιγραφής αρχείων](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_3152e675eacf49b3bde5f4623b8ad666~mv2.png)

Υπάρχει όμως και μια άλλη επιλογή διαθέσιμη, καθώς οι εφαρμογές μας υποστηρίζουν εγγενώς τις μονάδες SanDisk iXpand Flash.

Μπορείτε να αγοράσετε SanDisk εδώ: [SanDisk iXpand Mini](https://www.google.com/search?q=sandisk+ixpand+mini)

Αυτή τη στιγμή, η εφαρμογή υποστηρίζει μονάδες SanDisk Flash με πρωτόκολλα iXpand V1, iXpand V2, iXpand V3, iXpand V6, iXpand V7. Μπορείτε να ελέγξετε αν η μονάδα iXpand Flash Drive σας υποστηρίζεται ακολουθώντας τα παρακάτω βήματα:

- **Συνδέστε** τη μονάδα στη συσκευή iOS
- Ανοίξτε τις **Ρυθμίσεις**
- Πατήστε **Γενικά**
- Πατήστε **Πληροφορίες**
- Αναζητήστε **iXpand Flash Drive**
- Πατήστε **Εύρεση εφαρμογής για αξεσουάρ**
- Η εφαρμογή θα πρέπει να αναφέρεται εκεί

![Οθόνη ρυθμίσεων μονάδας iXpand flash](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_97e7a70f78a54bada207e1f529b33b22~mv2.png)

Μόλις βρείτε την υποστηριζόμενη μονάδα iXpand Flash Drive, για παράδειγμα: [iXpand Mini Flash Drive](https://www.westerndigital.com/en-ap/products/usb-flash-drives/sandisk-ixpand-mini-usb-3-0?sku=SDIX40N-016G-GN6NN), εισάγετέ την στη θύρα Lightning και ανοίξτε την εφαρμογή. Θα δείτε το μήνυμα "Συσκευή συνδεδεμένη" και πληροφορίες συσκευής.

Απλά ανοίξτε την οθόνη "Συνδέσεις" και κάντε κύλιση στην ενότητα "Συνδεδεμένα αξεσουάρ".

![Evermusic: Συνδεδεμένα αξεσουάρ](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_8416a9ee12dc42039a030948790a73e5~mv2.png)

Πατήστε στο εικονίδιο του συνδεδεμένου αξεσουάρ για να το ανοίξετε και να λάβετε τη λίστα αρχείων και φακέλων που βρίσκονται στη συσκευή USB. Σε ορισμένες περιπτώσεις, άλλες εφαρμογές μπορεί να χρησιμοποιούν τη μονάδα σας και να ορίσουν κατάσταση απασχολημένη. Σε αυτή την περίπτωση, θα δείτε ένα μήνυμα προειδοποίησης. Τίποτα σοβαρό - απλά κλείστε όλες τις άλλες εφαρμογές που χρησιμοποιούν τη μονάδα flash σας ή αποσυνδέστε την και συνδέστε την ξανά.

![Evermusic: Διαχειριστής αρχείων](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it/21260c_cb43f9e4b57c4fa597a6051289c41485~mv2.png)

Μόλις ανοίξετε οποιονδήποτε φάκελο με μουσική, θα δείτε όλα τα διαθέσιμα αρχεία ήχου. Απλά πατήστε σε οποιοδήποτε αρχείο για να ξεκινήσετε την αναπαραγωγή. Μπορείτε να ενεργοποιήσετε έναν ισοσταθμιστή στην οθόνη του player ή να μεταδώσετε περιεχόμενο ήχου σε AirPlay (Sonos, Apple TV) ή συσκευή Google Chromecast.

Μπορείτε να χρησιμοποιήσετε τον ενσωματωμένο διαχειριστή αρχείων για ΑΝΤΙΓΡΑΦΗ, ΜΕΤΑΚΙΝΗΣΗ, ΜΕΤΟΝΟΜΑΣΙΑ, ΔΙΑΓΡΑΦΗ των αρχείων σας. Μπορείτε επίσης να προσθέσετε αρχεία ήχου στη μουσική βιβλιοθήκη στις εφαρμογές Evermusic ή Flacbox. Και φυσικά μπορείτε να αντιγράψετε αρχεία ήχου από τη μονάδα flash στη μνήμη του iPhone σας.

Απολαύστε την ελευθερία πρόσβασης και διαχείρισης της μουσικής σας χωρίς κόπο με αυτά τα απλά βήματα και πείτε αντίο στους περιορισμούς αποθήκευσης!

## Συχνές ερωτήσεις

{{% details title="Ποιοι προσαρμογείς USB λειτουργούν με το iPhone για αναπαραγωγή μουσικής;" closed="true" %}}
Τόσο ο Lightning to SD Card Camera Reader όσο και ο Lightning to USB 3 Camera Adapter της Apple λειτουργούν. Οι προσαρμογείς USB-C λειτουργούν σε νεότερα iPhone με θύρες USB-C. Οι μονάδες SanDisk iXpand Flash (V1-V7) υποστηρίζονται επίσης εγγενώς από τα Evermusic, Flacbox και Evertag.
{{% /details %}}

{{% details title="Μπορώ να αναπαράγω μουσική απευθείας από τη μονάδα USB χωρίς αντιγραφή αρχείων;" closed="true" %}}
Ναι. Με τις μονάδες SanDisk iXpand, μπορείτε να αναπαράγετε μουσική απευθείας από τη μονάδα χωρίς να αντιγράψετε αρχεία στο iPhone σας. Όταν χρησιμοποιείτε προσαρμογείς Apple, τα αρχεία εισάγονται αλλά μπορείτε να επιλέξετε αν θα τα αντιγράψετε στην τοπική αποθήκευση.
{{% /details %}}

{{% details title="Ποιες μορφές ήχου υποστηρίζονται από μονάδες USB;" closed="true" %}}
Τα Evermusic και Flacbox υποστηρίζουν ένα ευρύ φάσμα μορφών, συμπεριλαμβανομένων FLAC, MP3, AAC, WAV, ALAC, OGG, WMA και άλλων. Όλες οι υποστηριζόμενες μορφές λειτουργούν κατά την αναπαραγωγή από αποθήκευση USB.
{{% /details %}}

{{% details title="Το SanDisk iXpand μου εμφανίζει σφάλμα 'απασχολημένο'. Τι πρέπει να κάνω;" closed="true" %}}
Μια άλλη εφαρμογή μπορεί να έχει πρόσβαση στη μονάδα. Κλείστε όλες τις άλλες εφαρμογές που μπορεί να χρησιμοποιούν τη μονάδα flash ή αποσυνδέστε την και επανεισάγετε τη. Στη συνέχεια, ανοίξτε ξανά τα Evermusic, Flacbox ή Evertag.
{{% /details %}}

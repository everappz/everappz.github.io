**摘要：** Evermusic 和 Flacbox 在 iOS 17+ 和 macOS 14 Sonoma+ 上提供互動式正在播放小工具。您可以直接從 iPhone 主畫面或 Mac 桌面控制播放、跳過曲目、新增最愛項目和建立音訊書籤——無需開啟應用程式。

在 **Evermusic** 和 **Flacbox** 的最新更新中，我們很高興推出適用於 iOS 和 Mac 的**動態正在播放小工具**。這些小工具讓您可以直接從主畫面或桌面控制播放，快速存取播放、暫停、停止、下一首、上一首、快轉、倒轉、新增至最愛項目，甚至為特定曲目建立音訊書籤等指令。

## 動態小工具需求

動態小工具可在 iOS 17.0 或更高版本以及 macOS 14 Sonoma 或更高版本上使用。透過 macOS 14 Sonoma，Apple 升級了小工具功能，允許小工具直接放置在桌面上，提供即時互動性，讓您無需開啟應用程式即可控制音樂播放、建立書籤和勾選提醒事項。

## 在 macOS Sonoma 上新增動態小工具

要在 Mac 上新增小工具：

1. 右鍵點擊桌面並選擇**編輯小工具**選單項目。

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_aac9a1f8c7f14cf2bfff623db541f65d~mv2.png" width="400" >}}

2. 向下捲動找到 Evermusic 或 Flacbox，選擇您偏好的小工具大小（小、中、大）。

   - **小**：基本播放控制和新增至最愛項目操作  
   - **中**：包含額外的跳過時間按鈕  
   - **大**：包含額外的跳過時間按鈕和音訊書籤操作

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_ef50a88f9b1b4d12b7dbbcadccaa312d~mv2.png" width="400" >}}

新增後，小工具將出現在桌面上，您可以將其拖曳到任意位置。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_4430a891228d4477bd68ae3d45c401b6~mv2.png" width="400" >}}

## 在 iOS 上新增動態小工具

要在 iOS 上新增小工具：

1. 長按主畫面，然後點擊左上角的**加號（+）按鈕**。  
2. 選擇您的音樂應用程式 Evermusic 或 Flacbox，然後選擇小工具大小。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_62bf80c9017643799e18711fa743485c~mv2.png" width="400" >}}

新增後，小工具將顯示在主畫面上，讓您無需啟動應用程式即可控制播放。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_8544d74b82984c87af127deb6197c5b8~mv2.png" width="400" >}}

點擊**最愛項目**按鈕將曲目新增至最愛，或使用**書籤**按鈕（在大型小工具上可用）建立音訊書籤。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_0ecaa9a53a494ed7aff75d8efc7a7627~mv2.png" width="400" >}}

每次點擊**音訊書籤**按鈕，都會為目前歌曲和時間戳記建立一個新書籤。您可以稍後在應用程式內存取它——非常適合有聲書或長時間講座。

## 繼續播放功能

要啟用無縫播放繼續：

1. 在應用程式中，前往**設定 > Audio Player > 一般**並啟用**儲存音訊播放器狀態**。  
2. 啟用此設定後，如果應用程式被關閉或卸載，小工具將顯示一個活動的播放按鈕。點擊它將在背景重新開啟應用程式並從上次儲存的位置繼續播放。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_a036bcfa1d894557bdccab78d340e52e~mv2.png" width="400" >}}

此功能非常適合有聲書和長篇音訊內容，讓您可以從上次停止的地方繼續收聽。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_de1ce7b14cb84d36acb45e67f314d04b~mv2.png" width="400" >}}

## 常見問題

{{% details title="小工具無需開啟應用程式就能運作嗎？" closed="true" %}}
是的。在 iOS 17 和 macOS 14 Sonoma 上，小工具按鈕是互動式的，可直接控制播放。應用程式無需在前景執行。
{{% /details %}}

{{% details title="我應該選擇哪種大小的小工具？" closed="true" %}}
選擇小型用於基本的播放/暫停和最愛項目。如果需要跳過按鈕，選擇中型。如果還需要音訊書籤，選擇大型。
{{% /details %}}

{{% details title="我可以使用小工具繼續收聽有聲書嗎？" closed="true" %}}
可以。在設定中啟用「儲存音訊播放器狀態」，小工具將從您上次的位置繼續播放，即使應用程式已被關閉。
{{% /details %}}

{{% details title="iPad 上有這些小工具嗎？" closed="true" %}}
有。iPadOS 17 支援與 iPhone 相同的互動式小工具。
{{% /details %}}

{{% details title="Evermusic 和 Flacbox 都有這些小工具嗎？" closed="true" %}}
是的。正在播放小工具在 Evermusic 和 Flacbox 中均可使用，功能完全相同。
{{% /details %}}

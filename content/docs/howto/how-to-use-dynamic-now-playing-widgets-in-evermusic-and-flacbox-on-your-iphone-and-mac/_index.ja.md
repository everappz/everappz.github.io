**要約:** Evermusic と Flacbox は iOS 17+ と macOS 14 Sonoma+ でインタラクティブな再生中ウィジェットを提供しています。iPhone のホーム画面や Mac のデスクトップから直接、再生の制御、トラックのスキップ、お気に入りの追加、オーディオブックマークの作成が可能です — アプリを開く必要はありません。

**Evermusic** と **Flacbox** の最新アップデートで、iOS と Mac 向けの**ダイナミック再生中ウィジェット**を発表できることを嬉しく思います。これらのウィジェットにより、ホーム画面やデスクトップから直接再生を制御でき、再生、一時停止、停止、次のトラック、前のトラック、スキップフォワード、スキップバック、お気に入りに追加、さらには特定のトラックのオーディオブックマーク作成などのコマンドに素早くアクセスできます。

## ダイナミックウィジェットの要件

ダイナミックウィジェットは iOS 17.0 以降および macOS 14 Sonoma 以降で利用可能です。macOS 14 Sonoma で、Apple はウィジェット機能をアップグレードし、デスクトップ上に直接ウィジェットを配置できるようになりました。リアルタイムのインタラクティブ性を提供し、アプリを開かずに音楽の再生制御、ブックマークの作成、リマインダーの確認ができます。

## macOS Sonoma でダイナミックウィジェットを追加する

Mac でウィジェットを追加するには：

1. デスクトップを右クリックし、**ウィジェットを編集**メニュー項目を選択します。

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_aac9a1f8c7f14cf2bfff623db541f65d~mv2.png" width="400" >}}

2. 下にスクロールして Evermusic または Flacbox を見つけ、お好みのウィジェットサイズ（小、中、大）を選択します。

   - **小**：基本的な再生コントロールとお気に入りに追加アクション  
   - **中**：追加のスキップタイムボタンを含む  
   - **大**：追加のスキップタイムボタンとオーディオブックマークアクションを含む

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_ef50a88f9b1b4d12b7dbbcadccaa312d~mv2.png" width="400" >}}

追加すると、ウィジェットがデスクトップに表示され、任意の位置にドラッグできます。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_4430a891228d4477bd68ae3d45c401b6~mv2.png" width="400" >}}

## iOS でダイナミックウィジェットを追加する

iOS でウィジェットを追加するには：

1. ホーム画面を長押しし、左上隅の**プラス（+）ボタン**をタップします。  
2. 音楽アプリ、Evermusic または Flacbox を選択し、ウィジェットのサイズを選びます。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_62bf80c9017643799e18711fa743485c~mv2.png" width="400" >}}

追加後、ウィジェットがホーム画面に表示され、アプリを起動せずに再生を制御できます。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_8544d74b82984c87af127deb6197c5b8~mv2.png" width="400" >}}

**お気に入り**ボタンをタップしてトラックをお気に入りに追加するか、**ブックマーク**ボタン（大きなウィジェットで利用可能）を使用してオーディオブックマークを作成します。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_0ecaa9a53a494ed7aff75d8efc7a7627~mv2.png" width="400" >}}

**オーディオブックマーク**ボタンをタップするたびに、現在の曲とタイムスタンプに新しいブックマークが作成されます。後でアプリ内からアクセスできます — オーディオブックや長い講義に最適です。

## 再生継続機能

シームレスな再生継続を有効にするには：

1. アプリで、**設定 > オーディオプレーヤー > 一般**に移動し、**オーディオプレーヤーの状態を保存**を有効にします。  
2. この設定が有効な場合、アプリが閉じられたりアンロードされたりすると、ウィジェットにアクティブな再生ボタンが表示されます。タップすると、アプリがバックグラウンドで再度開かれ、最後に保存された位置から再生が再開されます。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_a036bcfa1d894557bdccab78d340e52e~mv2.png" width="400" >}}

この機能はオーディオブックや長時間のオーディオコンテンツに最適で、中断したところから正確に再開できます。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_de1ce7b14cb84d36acb45e67f314d04b~mv2.png" width="400" >}}

## よくある質問

{{% details title="アプリを開かずにウィジェットは機能しますか？" closed="true" %}}
はい。iOS 17 と macOS 14 Sonoma では、ウィジェットのボタンはインタラクティブで、直接再生を制御します。アプリがフォアグラウンドにある必要はありません。
{{% /details %}}

{{% details title="どのウィジェットサイズを選ぶべきですか？" closed="true" %}}
基本的な再生/一時停止とお気に入りには小を選びます。スキップボタンが必要な場合は中を選びます。オーディオブックマークも必要な場合は大を選びます。
{{% /details %}}

{{% details title="ウィジェットを使ってオーディオブックを再開できますか？" closed="true" %}}
はい。設定で「オーディオプレーヤーの状態を保存」を有効にすると、アプリが閉じられた後でも、ウィジェットは最後の位置から再生を再開します。
{{% /details %}}

{{% details title="ウィジェットは iPad で利用できますか？" closed="true" %}}
はい。iPadOS 17 は iPhone と同じインタラクティブウィジェットをサポートしています。
{{% /details %}}

{{% details title="Evermusic と Flacbox の両方にこれらのウィジェットがありますか？" closed="true" %}}
はい。再生中ウィジェットは Evermusic と Flacbox の両方で同一の機能で利用可能です。
{{% /details %}}

**摘要：** Evermusic 和 Flacbox 在 iOS 17+ 和 macOS 14 Sonoma+ 上提供交互式正在播放小组件。您可以直接从 iPhone 主屏幕或 Mac 桌面控制播放、跳过曲目、添加收藏夹和创建音频书签——无需打开应用。

在 **Evermusic** 和 **Flacbox** 的最新更新中，我们很高兴推出适用于 iOS 和 Mac 的**动态正在播放小组件**。这些小组件让您可以直接从主屏幕或桌面控制播放，快速访问播放、暂停、停止、下一首、上一首、快进、快退、添加到收藏夹，甚至为特定曲目创建音频书签等命令。

## 动态小组件要求

动态小组件可在 iOS 17.0 或更高版本以及 macOS 14 Sonoma 或更高版本上使用。通过 macOS 14 Sonoma，Apple 升级了小组件功能，允许小组件直接放置在桌面上，提供实时交互性，让您无需打开应用即可控制音乐播放、创建书签和勾选提醒事项。

## 在 macOS Sonoma 上添加动态小组件

要在 Mac 上添加小组件：

1. 右键点击桌面并选择**编辑小组件**菜单项。

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_aac9a1f8c7f14cf2bfff623db541f65d~mv2.png" width="400" >}}

2. 向下滚动找到 Evermusic 或 Flacbox，选择您偏好的小组件大小（小、中、大）。

   - **小**：基本播放控制和添加到收藏夹操作  
   - **中**：包含额外的跳过时间按钮  
   - **大**：包含额外的跳过时间按钮和音频书签操作

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_ef50a88f9b1b4d12b7dbbcadccaa312d~mv2.png" width="400" >}}

添加后，小组件将出现在桌面上，您可以将其拖动到任意位置。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_4430a891228d4477bd68ae3d45c401b6~mv2.png" width="400" >}}

## 在 iOS 上添加动态小组件

要在 iOS 上添加小组件：

1. 长按主屏幕，然后点击左上角的**加号（+）按钮**。  
2. 选择您的音乐应用 Evermusic 或 Flacbox，然后选择小组件大小。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_62bf80c9017643799e18711fa743485c~mv2.png" width="400" >}}

添加后，小组件将显示在主屏幕上，让您无需启动应用即可控制播放。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_8544d74b82984c87af127deb6197c5b8~mv2.png" width="400" >}}

点击**收藏夹**按钮将曲目添加到收藏夹，或使用**书签**按钮（在大号小组件上可用）创建音频书签。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_0ecaa9a53a494ed7aff75d8efc7a7627~mv2.png" width="400" >}}

每次点击**音频书签**按钮，都会为当前歌曲和时间戳创建一个新书签。您可以稍后在应用内访问它——非常适合有声书或长时间讲座。

## 继续播放功能

要启用无缝播放继续：

1. 在应用中，前往**设置 > Audio Player > 一般**并启用**保存音频播放器状态**。  
2. 启用此设置后，如果应用被关闭或卸载，小组件将显示一个活动的播放按钮。点击它将在后台重新打开应用并从上次保存的位置恢复播放。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_a036bcfa1d894557bdccab78d340e52e~mv2.png" width="400" >}}

此功能非常适合有声书和长篇音频内容，让您可以从上次停止的地方继续收听。

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_de1ce7b14cb84d36acb45e67f314d04b~mv2.png" width="400" >}}

## 常见问题

{{% details title="小组件无需打开应用就能工作吗？" closed="true" %}}
是的。在 iOS 17 和 macOS 14 Sonoma 上，小组件按钮是交互式的，可直接控制播放。应用无需在前台运行。
{{% /details %}}

{{% details title="我应该选择哪种大小的小组件？" closed="true" %}}
选择小号用于基本的播放/暂停和收藏夹。如果需要跳过按钮，选择中号。如果还需要音频书签，选择大号。
{{% /details %}}

{{% details title="我可以使用小组件继续收听有声书吗？" closed="true" %}}
可以。在设置中启用"保存音频播放器状态"，小组件将从您上次的位置恢复播放，即使应用已被关闭。
{{% /details %}}

{{% details title="iPad 上有这些小组件吗？" closed="true" %}}
有。iPadOS 17 支持与 iPhone 相同的交互式小组件。
{{% /details %}}

{{% details title="Evermusic 和 Flacbox 都有这些小组件吗？" closed="true" %}}
是的。正在播放小组件在 Evermusic 和 Flacbox 中均可使用，功能完全相同。
{{% /details %}}

**Περίληψη:** Το Evermusic και το Flacbox προσφέρουν διαδραστικά widgets Τώρα παίζει σε iOS 17+ και macOS 14 Sonoma+. Μπορείτε να ελέγχετε την αναπαραγωγή, να παραλείπετε κομμάτια, να προσθέτετε αγαπημένα και να δημιουργείτε σελιδοδείκτες ήχου απευθείας από την αρχική οθόνη του iPhone ή την επιφάνεια εργασίας του Mac — χωρίς να χρειάζεται να ανοίξετε την εφαρμογή.

Στην τελευταία ενημέρωση του **Evermusic** και του **Flacbox**, είμαστε ενθουσιασμένοι που παρουσιάζουμε τα **Δυναμικά widgets Τώρα παίζει** για iOS και Mac. Αυτά τα widgets σας επιτρέπουν να ελέγχετε την αναπαραγωγή απευθείας από την αρχική οθόνη ή την επιφάνεια εργασίας σας, δίνοντάς σας γρήγορη πρόσβαση σε εντολές όπως Αναπαραγωγή, Παύση, Διακοπή, Επόμενο κομμάτι, Προηγούμενο κομμάτι, Μετάβαση μπροστά, Μετάβαση πίσω, Προσθήκη στα Αγαπημένα, ακόμα και Δημιουργία σελιδοδεικτών ήχου για συγκεκριμένα κομμάτια.

## Απαιτήσεις για δυναμικά widgets

Τα δυναμικά widgets είναι διαθέσιμα σε iOS 17.0 ή νεότερο και macOS 14 Sonoma ή νεότερο. Με το macOS 14 Sonoma, η Apple αναβάθμισε τη λειτουργικότητα των widgets για να επιτρέπει widgets απευθείας στην επιφάνεια εργασίας, παρέχοντας διαδραστικότητα σε πραγματικό χρόνο, ώστε να μπορείτε να ελέγχετε την αναπαραγωγή μουσικής, να δημιουργείτε σελιδοδείκτες και να ελέγχετε υπενθυμίσεις χωρίς να ανοίξετε την εφαρμογή.

## Προσθήκη δυναμικών widgets στο macOS Sonoma

Για να προσθέσετε το widget στο Mac σας:

1. Κάντε δεξί κλικ στην επιφάνεια εργασίας και επιλέξτε το στοιχείο μενού **Επεξεργασία widgets**.

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_aac9a1f8c7f14cf2bfff623db541f65d~mv2.png" width="400" >}}

2. Κάντε κύλιση προς τα κάτω για να βρείτε το Evermusic ή Flacbox και επιλέξτε το προτιμώμενο μέγεθος widget (Μικρό, Μεσαίο, Μεγάλο).

   - **Μικρό**: Βασικά χειριστήρια αναπαραγωγής και ενέργεια προσθήκης στα αγαπημένα  
   - **Μεσαίο**: Περιλαμβάνει πρόσθετα κουμπιά μετάβασης χρόνου  
   - **Μεγάλο**: Περιλαμβάνει πρόσθετα κουμπιά μετάβασης χρόνου και ενέργεια σελιδοδεικτών ήχου

   {{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_ef50a88f9b1b4d12b7dbbcadccaa312d~mv2.png" width="400" >}}

Μόλις προστεθεί, το widget θα εμφανιστεί στην επιφάνεια εργασίας και μπορείτε να το σύρετε σε οποιαδήποτε θέση.

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_4430a891228d4477bd68ae3d45c401b6~mv2.png" width="400" >}}

## Προσθήκη δυναμικών widgets στο iOS

Για να προσθέσετε το widget στο iOS:

1. Πατήστε παρατεταμένα στην αρχική οθόνη και μετά πατήστε το **κουμπί συν (+)** στην επάνω αριστερή γωνία.  
2. Επιλέξτε την εφαρμογή μουσικής σας, Evermusic ή Flacbox, και επιλέξτε το μέγεθος widget.

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_62bf80c9017643799e18711fa743485c~mv2.png" width="400" >}}

Μετά την προσθήκη, το widget θα εμφανιστεί στην αρχική οθόνη σας, επιτρέποντάς σας να ελέγχετε την αναπαραγωγή χωρίς να εκκινήσετε την εφαρμογή.

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_8544d74b82984c87af127deb6197c5b8~mv2.png" width="400" >}}

Πατήστε το κουμπί **Αγαπημένα** για να προσθέσετε κομμάτια στα αγαπημένα σας, ή χρησιμοποιήστε το κουμπί **Σελιδοδείκτες** (διαθέσιμο στο μεγάλο widget) για να δημιουργήσετε σελιδοδείκτες ήχου.

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_0ecaa9a53a494ed7aff75d8efc7a7627~mv2.png" width="400" >}}

Κάθε φορά που πατάτε το κουμπί **Σελιδοδείκτης ήχου**, δημιουργείται ένας νέος σελιδοδείκτης για το τρέχον τραγούδι και τη χρονική σήμανση. Μπορείτε να τον προσπελάσετε αργότερα μέσα στην εφαρμογή — ιδανικό για ηχοβιβλία ή μεγάλες διαλέξεις.

## Λειτουργία συνέχισης αναπαραγωγής

Για να ενεργοποιήσετε την απρόσκοπτη συνέχιση αναπαραγωγής:

1. Στην εφαρμογή, μεταβείτε στο **Ρυθμίσεις > Αναπαραγωγέας ήχου > Γενικά** και ενεργοποιήστε **Αποθήκευση κατάστασης αναπαραγωγέα ήχου**.  
2. Με αυτή τη ρύθμιση ενεργοποιημένη, εάν η εφαρμογή κλείσει ή αποφορτωθεί, το widget θα εμφανίζει ένα ενεργό κουμπί αναπαραγωγής. Πατώντας το θα ανοίξει ξανά η εφαρμογή στο παρασκήνιο και θα συνεχίσει την αναπαραγωγή από την τελευταία αποθηκευμένη θέση.

{{< figure src="/docs/howto/how-to-use-dynamic-now-playing-widgets-in-evermusic-and-flacbox-on-your-iphone-and-mac/21260c_a036bcfa1d894557bdccab78d340e52e~mv2.png" width="400" >}}

Αυτή η λειτουργία είναι ιδανική για ηχοβιβλία και μεγάλης διάρκειας ηχητικό περιεχόμενο, επιτρέποντάς σας να συνεχίσετε ακριβώς από εκεί που σταματήσατε.

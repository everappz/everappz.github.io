**要約:** SoundyはiPhone用のSoundCloudプレーヤーで、本格的な10バンドイコライザー、20以上のプリセット（Jazz、Rock、Electronic、Hip Hopなど）、スリープタイマー、オーディオブックマーク、再生速度コントロール、そして優れた音質のためのオートノーマライザーを搭載しています。

SoundCloudは音楽発見で有名ですが、公式SoundCloudアプリの明らかな欠点はイコライザーがないことです。TuneShellなどの代替アプリはSoundCloud再生を提供していますが、イコライザーオプションは限られていることが多いです。そこで登場するのがSoundy、あなたが待ち望んでいたソリューションです。

Soundyは機能豊富なSoundCloudオーディオプレーヤーで、iTunesを彷彿とさせる堅牢な10バンドイコライザーを搭載し、Electronic、Jazz、Pop、Hip Hop、Rock、Bass Booster、Bass Reducerなどの人気プリセットを備えています。イコライザー以外にも、Soundyは簡単なプレイリスト管理、シャッフル、リピート、オーディオブックマーク、お気に入り、スリープタイマー、再生速度コントロールなど、トップクラスの音楽アプリに期待される多くの機能を提供しています。

しかし、Soundyの最大の魅力は間違いなくその本物の10バンドイコライザーです。ご存知ない方のために説明すると、イコライザーは低音、中音、高音の周波数をお好みに合わせて増減できます。10バンドイコライザーでは、広範な低音の強化や、狭い周波数範囲の精密な調整が可能です。

では、Soundyのイコライザーを本物たらしめるものは何でしょうか？疑似パラメトリックイコライザーを持つ他のアプリとは異なり、Soundyではコントロールノブをドラッグするだけで音量と周波数を簡単に選択できます。

## Soundyの10バンドイコライザーでできること

- サウンドスペクトラム全体の特定の周波数を微調整する。
- さまざまな音楽ジャンル、部屋の音響、または特定のスピーカーに合わせてサウンドをカスタマイズする。
- 一般的に使用される20以上のイコライザー設定プリセットから選択するか、手動で設定を調整する。

## Soundyの10バンドイコライザーの使い方

1. プレーヤー画面 > イコライザーに移動します。
2. ポップアップメニューからプリセットを選択するか、周波数スライダーをドラッグして特定の周波数の音量を調整します（デシベル単位）。

（注：イコライザーの設定は、変更するまですべての曲に適用されます）。

## 独自のイコライザープリセットも作成できます

1. プレーヤー画面 > イコライザーに移動します。
2. お好みに合わせてスライダーをドラッグして周波数設定を調整します。

Soundyは良い音がするだけではありません。卓越した音がします。独自のオートノーマライザーのおかげで、Soundyはクリッピングとダイナミクスの圧縮を防止します。これにより、SoundCloudトラックの比類のないリスニング体験が実現します。

## 追加機能

しかし、イコライザーはSoundyの唯一の優れた機能ではありません。以下も提供しています：

- オーディオブック用の再生速度コントロール。
- オーディオブックマークと最後の位置からの再生再開。
- 音楽をやさしくフェードアウトさせるスリープタイマー。デバイスのバッテリーを節約しながら、あなたを眠りに誘います。

## よくある質問

{{% details title="SoundCloudアプリにイコライザーはありますか？" closed="true" %}}
いいえ。公式SoundCloudアプリにはイコライザーが含まれていません。Soundyは20以上のプリセットを備えた本格的な10バンドイコライザーを追加するサードパーティのSoundCloudプレーヤーです。
{{% /details %}}

{{% details title="Soundyは無料ですか？" closed="true" %}}
SoundyはiPhone用のApp Storeで入手できます。現在の価格と無料トライアルオプションについてはApp Storeのリストをご確認ください。
{{% /details %}}

{{% details title="SoundCloudで低音を調整できますか？" closed="true" %}}
はい、Soundyで可能です。Bass Boosterプリセットを使用して素早く低音を強化するか、10バンドイコライザーの低周波数スライダーを手動でドラッグして精密な低音コントロールを行えます。
{{% /details %}}

{{% details title="Soundyはバックグラウンド再生をサポートしていますか？" closed="true" %}}
はい。Soundyは他のアプリを使用中やデバイスをロック中でもバックグラウンドでSoundCloudオーディオを再生します。
{{% /details %}}

{{% details title="Soundyのオートノーマライザーとは何ですか？" closed="true" %}}
オートノーマライザーは、クリッピングを防止しオーディオダイナミクスを保持するために再生音量を自動的に調整します。これにより、歪みのないトラック間で一貫した音量を実現します。
{{% /details %}}

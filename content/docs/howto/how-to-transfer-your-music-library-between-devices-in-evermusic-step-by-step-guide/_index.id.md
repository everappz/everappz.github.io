**Ringkasan:** Untuk mentransfer perpustakaan Evermusic Anda ke perangkat baru, buat cadangan di perangkat sumber, mulai Wi-Fi Drive, hubungkan perangkat kedua melalui jaringan yang sama, unduh cadangan dan file musik, lalu pulihkan dari cadangan. Seluruh proses memakan waktu sekitar 10 menit tergantung ukuran perpustakaan.

Dalam panduan ini, kami akan memandu Anda melalui transfer seluruh perpustakaan musik Anda — termasuk database, sampul album, dan pengaturan — dari satu perangkat (iPhone atau Mac) ke perangkat lain. Meskipun sinkronisasi otomatis perpustakaan musik dan daftar putar adalah fitur yang direncanakan untuk masa depan, proses ini saat ini harus dilakukan secara manual.

## Langkah 1: Buat Cadangan di Perangkat Pertama

1. **Buka aplikasi di perangkat pertama Anda** (yang memiliki perpustakaan musik, daftar putar, dan layanan cloud yang terhubung).
2. **Buka Pengaturan** dan pilih opsi **Cadangan dan Pemulihan**.

![Cadangan dan Pemulihan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_44dbabc06ba844c0b665f17ac01fec84~mv2.png)

3. Di layar **Cadangan dan Pemulihan**, pilih item yang akan disertakan dalam file cadangan Anda:
   - **Database** (mencakup perpustakaan musik dan daftar putar Anda)
   - **Sampul Album**
   - **Pengaturan**

Opsi-opsi ini diaktifkan secara default.

![Opsi Cadangan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_160d11e81b104384a5fef09ae196c260~mv2.png)

4. Ketuk **Cadangkan Data Aplikasi** untuk memulai proses.

![Cadangkan Data Aplikasi](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_5bab30ce6d7b4fcf8b67ad6bd18b5f21~mv2.png)

5. Setelah cadangan selesai, peringatan info akan muncul.

![Cadangan Selesai](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_cc751d3e710941479102d286d0504a80~mv2.png)

Ketuk **Tampilkan File** untuk menampilkan file cadangan di direktori **Dokumen**. File cadangan biasanya disimpan di folder **Backup**.

![Tampilkan File Cadangan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_d857f23f0ad94ae8851f93baed15aeb1~mv2.png)

## Langkah 2: Mulai Server Wi-Fi Drive

1. Buka bagian **Koneksi** di aplikasi.
2. Gulir ke bawah ke **Hubungkan Melalui Wi-Fi** dan ketuk.

![Hubungkan Melalui Wi-Fi](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_7241696cb6a54f8d95d15cb9592bbeed~mv2.png)

3. Di layar berikutnya, ketuk **Mulai Wi-Fi Drive**.

- Secara opsional, Anda dapat mengatur login dan kata sandi untuk keamanan, tetapi ini tidak diperlukan untuk jaringan rumah.

4. Setelah dimulai, Anda akan melihat alamat server yang tersedia. Ini dapat digunakan untuk browser desktop atau aplikasi WebDAV, tetapi dalam panduan ini, kita akan langsung ke langkah berikutnya.

![Wi-Fi Drive Dimulai](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_11ed60c4073640cc9aafda20cec8431c~mv2.png)

## Langkah 3: Hubungkan Perangkat Kedua Anda ke Perangkat Pertama

1. Buka aplikasi di perangkat kedua Anda (tempat Anda ingin mentransfer perpustakaan).
2. Pastikan kedua perangkat terhubung ke jaringan Wi-Fi yang sama.
3. Di perangkat kedua, buka tab **Koneksi** dan pilih **Perangkat yang Tersedia**.

![Perangkat yang Tersedia](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_4d9abdd7dd80409caabfeca89535754d~mv2.png)

4. Anda akan melihat koneksi WebDAV bernama **Evermusic** (server yang kita mulai di perangkat pertama). Ketuk untuk menghubungkan.

5. Setelah terhubung, Anda akan melihat semua folder dari perangkat pertama.

![Terhubung ke Perangkat Pertama](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_3075c7ee8dfb41f6a2496f8be0673d37~mv2.png)

## Langkah 4: Persiapkan Transfer File

1. Di perangkat kedua, buka **Pengaturan > Pengelola File** dan aktifkan **Simpan File yang Diunduh Ke - Tanya Setiap Kali**.

- Ini memastikan Anda dapat memilih folder tujuan untuk setiap unduhan.

2. Kembali ke tab **Koneksi** dan navigasi ke server WebDAV yang terhubung.

![Persiapkan Transfer File](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_fb325a9cc68d419389ee76cd19b821d5~mv2.png)

## Langkah 5: Transfer Cadangan dan File Musik

1. Buka folder **Backup** di server WebDAV yang terhubung.

![Folder Cadangan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_91254ba6f64649adbc8a759899a8618a~mv2.png)

2. Ketuk tombol **Lebih banyak tindakan** (tiga titik) di dekat file cadangan dan pilih **Unduh**.

3. Buat folder **Backup** di perangkat kedua untuk menyimpan file yang diunduh. Konfirmasi pilihan Anda dengan mengetuk **Selesai**.

![Unduh Cadangan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_ffd617db2ab64bbcb580b70aa94fe3df~mv2.png)

4. Transfer file musik tambahan:
   - Periksa folder **Unduh** atau folder relevan lainnya di server WebDAV.

![Transfer File Musik](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_6536aba207bc4ff98af139f7dc8a2f45~mv2.png)

- Gunakan opsi **Pilih** untuk memilih file, lalu ketuk **Lebih banyak tindakan > Unduh**. Simpan di folder yang sesuai di perangkat kedua untuk mempertahankan struktur direktori yang sama.

Tujuannya adalah mentransfer semua file dari perangkat pertama ke perangkat Anda saat ini, memastikan bahwa struktur folder tetap identik. Dengan cara ini, tautan di perpustakaan musik dan daftar putar Anda tetap utuh. Perhatikan bahwa file lokal yang disimpan di luar direktori Dokumen aplikasi di perangkat pertama Anda harus ditransfer secara terpisah. Karena aplikasi tidak dapat mengakses file-file ini dalam mode Wi-Fi Drive, Anda perlu menggunakan aplikasi File sistem untuk transfernya.

![Transfer Selesai](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_352edb4aeb584d53b8646d7bf779de02~mv2.png)

## Langkah 6: Pantau Kemajuan Transfer

1. Di perangkat kedua, buka bagian **File Lokal** (atau tab **Unduh** di iPad/Mac).

2. Ketuk tombol **Aktivitas Transfer** di sudut kiri atas untuk memantau antrean transfer.

![Pantau Transfer](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_82c35eccb18245fe8e1135ab86532a52~mv2.png)

## Langkah 7: Pulihkan Data dari Cadangan

1. Setelah file cadangan dan semua file audio yang diperlukan diunduh ke perangkat kedua, buka folder **Backup**.

2. Ketuk file cadangan, dan pesan konfirmasi akan muncul.

![Pulihkan Cadangan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_480c20fc3d664307b135881d04444fb5~mv2.png)

- **Catatan:** Pemulihan akan menggantikan semua data perpustakaan musik saat ini, daftar putar, pengaturan, dan sampul album dengan data cadangan.

3. Ketuk **Pulihkan** untuk memulai proses.

![Proses Pemulihan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_469647cffde34ae7a448f7928d1436a9~mv2.png)

4. Setelah pemulihan selesai, peringatan akan mengonfirmasi keberhasilan.

![Pemulihan Selesai](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_c6ca2dc848b64a26a30b511e9e4b79cd~mv2.png)

Periksa bagian **Daftar Putar** atau **Perpustakaan Musik** untuk memverifikasi pemulihan.

![Verifikasi Pemulihan](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide/21260c_ff3f96fe50f845b392ef24f2de03a794~mv2.png)

## Langkah 8: Indeks Ulang Perpustakaan Musik

1. Untuk hasil terbaik, indeks ulang perpustakaan Anda untuk memastikan semua file terdeteksi dengan sukses.

2. Buka **Pengaturan > Perpustakaan Musik > Sinkronisasi Musik Offline** dan pilih **Mulai Pemindaian Folder Lokal**.

Dengan mengikuti langkah-langkah ini, Anda akan berhasil mentransfer perpustakaan musik, daftar putar, dan pengaturan Anda ke perangkat lain. Jika Anda mengalami masalah, jangan ragu untuk menghubungi dukungan.

## Pertanyaan yang Sering Diajukan

{{% details title="Bisakah saya mentransfer perpustakaan Evermusic saya tanpa Wi-Fi?" closed="true" %}}
Wi-Fi Drive mengharuskan kedua perangkat berada di jaringan Wi-Fi yang sama. Saat ini tidak ada opsi transfer Bluetooth atau seluler. Anda dapat menggunakan AirDrop atau aplikasi File sebagai alternatif untuk memindahkan file cadangan dan folder musik secara manual antar perangkat.
{{% /details %}}

{{% details title="Apakah koneksi layanan cloud saya akan ditransfer bersama cadangan?" closed="true" %}}
Cadangan mencakup database, daftar putar, sampul album, dan pengaturan Anda. Kredensial login layanan cloud tidak disertakan karena alasan keamanan. Anda perlu menghubungkan kembali akun cloud Anda di perangkat baru setelah pemulihan.
{{% /details %}}

{{% details title="Apa yang terjadi pada perpustakaan saya yang ada di perangkat kedua?" closed="true" %}}
Memulihkan cadangan menggantikan semua data perpustakaan musik yang ada, daftar putar, pengaturan, dan sampul album di perangkat kedua. Buat cadangan terpisah dari perangkat kedua terlebih dahulu jika Anda ingin menyimpan datanya.
{{% /details %}}

{{% details title="Apakah proses ini berfungsi antara iPhone dan Mac?" closed="true" %}}
Ya. Evermusic mendukung transfer Wi-Fi Drive antara kombinasi apa pun dari iPhone, iPad, dan Mac. Kedua perangkat hanya perlu berada di jaringan Wi-Fi yang sama.
{{% /details %}}

{{% details title="Berapa lama waktu transfer?" closed="true" %}}
Waktu transfer tergantung pada ukuran perpustakaan musik Anda dan kecepatan Wi-Fi Anda. Perpustakaan tipikal beberapa gigabyte ditransfer dalam 5-15 menit melalui jaringan rumah standar.
{{% /details %}}

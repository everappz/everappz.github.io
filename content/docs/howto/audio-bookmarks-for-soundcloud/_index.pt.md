**Resumo:** Soundy permite criar marcadores de áudio em faixas do SoundCloud – ideal para marcar posições em audiolivros, podcasts e palestras.

Com o [Soundy](/products/soundy), você pode ouvir seus audiolivros, palestras e podcasts favoritos do SoundCloud – e criar marcadores de áudio para salvar sua posição ou destacar momentos importantes.

Para criar um marcador de áudio para o SoundCloud, siga os passos abaixo:

### 1. Pesquise uma faixa

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Pesquisando um podcast no Soundy usando o SoundCloud"
  caption="Pesquise podcasts ou faixas pelo SoundCloud dentro do Soundy"
  width="400"
>}}

Use a aba **Pesquisa** ou a aba **Minha conta** para encontrar suas faixas do SoundCloud.

### 2. Inicie a reprodução

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Tela de detalhes da playlist no Soundy"
  caption="Toque em uma faixa para abrir a tela do reprodutor"
  width="400"
>}}

Toque na faixa para abrir o reprodutor e começar a ouvir.

### 3. Abra o menu de Favoritos

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Abrindo marcadores na tela do reprodutor"
  caption="Toque no botão MAIS (...) e escolha «Favoritos»"
  width="400"
>}}

No reprodutor, toque no botão **...** no canto superior esquerdo e selecione **Favoritos**.

### 4. Adicione um novo marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Criando um novo marcador de áudio"
  caption="Defina um horário e um título opcional para o novo marcador"
  width="400"
>}}

Escolha um horário (o padrão é o tempo de reprodução atual) e edite o nome se necessário.

### 5. Salve o marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="Lista de marcadores de áudio"
  caption="Você pode criar vários marcadores por faixa"
  width="400"
>}}

Toque em **Concluído** para salvar o marcador e voltar à lista de marcadores.

### 6. Vá para um marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Tocando em um marcador para ir até aquele momento"
  caption="Toque no nome de um marcador para ir até aquele ponto no áudio"
  width="400"
>}}

Toque em qualquer título de marcador para retomar instantaneamente a reprodução a partir daquele momento.

### 7. Veja todas as faixas com marcadores

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Filtro do Soundy mostrando faixas com marcadores"
  caption="Use o controle segmentado para ver todas as faixas com marcadores salvos"
  width="400"
>}}

Use o filtro **Todas as músicas** para encontrar e gerenciar todos os seus áudios com marcadores.

## FAQ

{{% details title="O que são marcadores de áudio?" closed="true" %}}
Marcadores de áudio permitem salvar carimbos de tempo específicos dentro de uma faixa para que você possa voltar exatamente àquele momento mais tarde. São especialmente úteis para conteúdo de formato longo como audiolivros, podcasts e palestras.
{{% /details %}}

{{% details title="Posso criar marcadores de áudio no Evermusic ou Flacbox?" closed="true" %}}
Sim. Tanto o Evermusic quanto o Flacbox incluem um recurso gratuito de marcadores de áudio para faixas armazenadas em serviços de nuvem ou armazenamento local.
{{% /details %}}

{{% details title="Quantos marcadores posso criar por faixa?" closed="true" %}}
Não há limite fixo. Você pode criar vários marcadores por faixa e gerenciá-los a partir da lista de marcadores.
{{% /details %}}

**Resumen:** Soundy te permite crear marcadores de audio en pistas de SoundCloud -- ideal para marcar posiciones en audiolibros, podcasts y conferencias.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="Soundy app used for audio bookmarking on SoundCloud"
  caption="Soundy: Audio Bookmark For SoundCloud"
  width="600"
>}}

Con [Soundy](/products/soundy), puedes escuchar tus audiolibros, conferencias y podcasts favoritos de SoundCloud — y crear marcadores de audio para guardar tu posición o destacar momentos clave.

Para crear un marcador de audio para SoundCloud, sigue los pasos a continuación:

### 1. Buscar una pista

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Searching for a podcast in Soundy using SoundCloud"
  caption="Busca podcasts o pistas a través de SoundCloud dentro de Soundy"
  width="400"
>}}

Usa la pestaña **Buscar** o la pestaña **Mi cuenta** para encontrar tus pistas de SoundCloud.

### 2. Iniciar reproducción

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Playlist detail screen in Soundy"
  caption="Toca una pista para abrir la pantalla del reproductor"
  width="400"
>}}

Toca la pista para abrir el reproductor y comenzar a escuchar.

### 3. Abrir el menú de marcadores

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Opening bookmarks from the player screen"
  caption="Toca el botón MÁS (...) y elige 'Marcadores'"
  width="400"
>}}

En el reproductor, toca el botón **...** en la esquina superior izquierda, luego selecciona **Marcadores**.

### 4. Agregar un nuevo marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Creating a new audio bookmark"
  caption="Establece un tiempo y un título opcional para el nuevo marcador"
  width="400"
>}}

Elige un tiempo (el predeterminado es el tiempo de reproducción actual) y edita el nombre si es necesario.

### 5. Guardar el marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="List of audio bookmarks"
  caption="Puedes crear múltiples marcadores por pista"
  width="400"
>}}

Toca **Hecho** para guardar el marcador y volver a la lista de marcadores.

### 6. Saltar a un marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Tapping a bookmark to jump to that moment"
  caption="Toca el nombre de un marcador para saltar a ese punto en el audio"
  width="400"
>}}

Toca cualquier título de marcador para reanudar instantáneamente la reproducción desde ese momento.

### 7. Ver todas las pistas con marcadores

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy filter showing bookmarked tracks"
  caption="Usa el control segmentado para ver todas las pistas con marcadores guardados"
  width="400"
>}}

Usa el filtro **Todas las canciones** para encontrar y gestionar todo tu audio marcado.

## Preguntas frecuentes

{{% details title="¿Qué son los marcadores de audio?" closed="true" %}}
Los marcadores de audio te permiten guardar marcas de tiempo específicas dentro de una pista para que puedas volver a ese momento exacto más tarde. Son especialmente útiles para contenido de larga duración como audiolibros, podcasts y conferencias.
{{% /details %}}

{{% details title="¿Puedo crear marcadores de audio en Evermusic o Flacbox?" closed="true" %}}
Sí. Tanto Evermusic como Flacbox incluyen una función gratuita de marcadores de audio para pistas almacenadas en servicios en la nube o almacenamiento local.
{{% /details %}}

{{% details title="¿Cuántos marcadores puedo crear por pista?" closed="true" %}}
No hay un límite fijo. Puedes crear múltiples marcadores por pista y gestionarlos desde la lista de marcadores.
{{% /details %}}

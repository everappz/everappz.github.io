**摘要：** Soundy 允许您在 SoundCloud 曲目中创建音频书签——非常适合在有声书、播客和讲座中标记位置。

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="用于在 SoundCloud 上进行音频书签标记的 Soundy 应用"
  caption="Soundy：SoundCloud 音频书签"
  width="600"
>}}

使用 [Soundy](/products/soundy)，您可以收听 SoundCloud 上喜爱的有声书、讲座和播客，并创建音频书签来保存位置或标记关键时刻。

要为 SoundCloud 创建音频书签，请按照以下步骤操作：

### 1. 搜索曲目

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="在 Soundy 中使用 SoundCloud 搜索播客"
  caption="在 Soundy 中通过 SoundCloud 搜索播客或曲目"
  width="400"
>}}

使用**搜索**标签页或**我的账户**标签页查找您的 SoundCloud 曲目。

### 2. 开始播放

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Soundy 中的播放列表详情屏幕"
  caption="点击曲目以打开播放器屏幕"
  width="400"
>}}

点击曲目以打开播放器并开始收听。

### 3. 打开书签菜单

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="从播放器屏幕打开书签"
  caption="点击「更多」(...)按钮并选择「书签」"
  width="400"
>}}

在播放器中，点击左上角的 **...** 按钮，然后选择**书签**。

### 4. 添加新书签

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="创建新的音频书签"
  caption="为新书签设置时间和可选标题"
  width="400"
>}}

选择时间（默认为当前播放时间），如有需要可编辑名称。

### 5. 保存书签

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="音频书签列表"
  caption="您可以为每个曲目创建多个书签"
  width="400"
>}}

点击**完成**以保存书签并返回书签列表。

### 6. 跳转到书签

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="点击书签跳转到该时刻"
  caption="点击书签名称跳转到音频中的该位置"
  width="400"
>}}

点击任何书签标题，即可从该时刻立即恢复播放。

### 7. 查看所有带书签的曲目

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy 过滤器显示带书签的曲目"
  caption="使用分段控件查看所有已保存书签的曲目"
  width="400"
>}}

使用**所有歌曲**过滤器查找和管理所有带书签的音频。

## 常见问题

{{% details title="什么是音频书签？" closed="true" %}}
音频书签允许您在曲目中保存特定的时间戳，以便稍后跳转回确切的时刻。它们对于有声书、播客和讲座等长内容特别有用。
{{% /details %}}

{{% details title="我可以在 Evermusic 或 Flacbox 中创建音频书签吗？" closed="true" %}}
可以。Evermusic 和 Flacbox 都包含免费的音频书签功能，适用于存储在云服务或本地存储中的曲目。
{{% /details %}}

{{% details title="每个曲目可以创建多少个书签？" closed="true" %}}
没有固定限制。您可以为每个曲目创建多个书签，并从书签列表中管理它们。
{{% /details %}}

**En bref :** Soundy vous permet de créer des signets audio dans les pistes SoundCloud -- idéal pour marquer des positions dans les livres audio, podcasts et conférences.

  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"

>}}

Avec [Soundy](/products/soundy), vous pouvez écouter vos livres audio, conférences et podcasts préférés depuis SoundCloud — et créer des signets audio pour sauvegarder votre position ou mettre en évidence les moments clés.

Pour créer un signet audio pour SoundCloud, suivez les étapes ci-dessous :

### 1. Rechercher une piste

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Recherche d'un podcast dans Soundy via SoundCloud"
  caption="Recherchez des podcasts ou des pistes via SoundCloud dans Soundy"
  width="400"
>}}

Utilisez l'onglet **Recherche** ou l'onglet **Mon compte** pour trouver vos pistes SoundCloud.

### 2. Lancer la lecture

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Écran de détail de la playlist dans Soundy"
  caption="Appuyez sur une piste pour ouvrir l'écran du lecteur"
  width="400"
>}}

Appuyez sur la piste pour ouvrir le lecteur et commencer l'écoute.

### 3. Ouvrir le menu Signets

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Ouverture des signets depuis l'écran du lecteur"
  caption="Appuyez sur le bouton PLUS (...) et choisissez 'Signets'"
  width="400"
>}}

Dans le lecteur, appuyez sur le bouton **...** en haut à gauche, puis sélectionnez **Signets**.

### 4. Ajouter un nouveau signet

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Création d'un nouveau signet audio"
  caption="Définissez un temps et un titre optionnel pour le nouveau signet"
  width="400"
>}}

Choisissez un temps (par défaut le temps de lecture actuel) et modifiez le nom si nécessaire.

### 5. Enregistrer le signet

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="Liste des signets audio"
  caption="Vous pouvez créer plusieurs signets par piste"
  width="400"
>}}

Appuyez sur **Fait** pour enregistrer le signet et revenir à la liste des signets.

### 6. Accéder à un signet

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Appuyer sur un signet pour accéder à ce moment"
  caption="Appuyez sur le nom d'un signet pour accéder à ce point de l'audio"
  width="400"
>}}

Appuyez sur n'importe quel titre de signet pour reprendre instantanément la lecture à partir de ce moment.

### 7. Voir toutes les pistes avec des signets

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Filtre Soundy affichant les pistes avec signets"
  caption="Utilisez le contrôle segmenté pour voir toutes les pistes avec des signets enregistrés"
  width="400"
>}}

Utilisez le filtre **Tous les morceaux** pour trouver et gérer tous vos audios avec signets.

## FAQ

{{% details title="Que sont les signets audio ?" closed="true" %}}
Les signets audio vous permettent de sauvegarder des horodatages spécifiques dans une piste afin de pouvoir revenir à ce moment précis plus tard. Ils sont particulièrement utiles pour les contenus longs comme les livres audio, les podcasts et les conférences.
{{% /details %}}

{{% details title="Puis-je créer des signets audio dans Evermusic ou Flacbox ?" closed="true" %}}
Oui. Evermusic et Flacbox incluent tous deux une fonctionnalité gratuite de signets audio pour les pistes stockées dans des services cloud ou en stockage local.
{{% /details %}}

{{% details title="Combien de signets puis-je créer par piste ?" closed="true" %}}
Il n'y a pas de limite fixe. Vous pouvez créer plusieurs signets par piste et les gérer depuis la liste des signets.
{{% /details %}}

**สรุป:** Soundy ให้คุณสร้างบุ๊คมาร์คเสียงในเพลง SoundCloud -- เหมาะสำหรับการทำเครื่องหมายตำแหน่งในหนังสือเสียง พอดแคสต์ และการบรรยาย

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="แอป Soundy สำหรับบุ๊คมาร์คเสียงบน SoundCloud"
  caption="Soundy: บุ๊คมาร์คเสียงสำหรับ SoundCloud"
  width="600"
>}}

ด้วย [Soundy](/products/soundy) คุณสามารถฟังหนังสือเสียง การบรรยาย และพอดแคสต์ที่ชื่นชอบจาก SoundCloud -- และสร้างบุ๊คมาร์คเสียงเพื่อบันทึกตำแหน่งหรือไฮไลท์ช่วงเวลาสำคัญ

ในการสร้างบุ๊คมาร์คเสียงสำหรับ SoundCloud ให้ทำตามขั้นตอนด้านล่าง:

### 1. ค้นหาเพลง

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="ค้นหาพอดแคสต์ใน Soundy ผ่าน SoundCloud"
  caption="ค้นหาพอดแคสต์หรือเพลงผ่าน SoundCloud ใน Soundy"
  width="400"
>}}

ใช้แท็บ **ค้นหา** หรือแท็บ **บัญชีของฉัน** เพื่อค้นหาเพลง SoundCloud ของคุณ

### 2. เริ่มเล่น

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="หน้าจอรายละเอียดเพลย์ลิสต์ใน Soundy"
  caption="แตะที่เพลงเพื่อเปิดหน้าจอเครื่องเล่น"
  width="400"
>}}

แตะที่เพลงเพื่อเปิดเครื่องเล่นและเริ่มฟัง

### 3. เปิดเมนูบุ๊คมาร์ค

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="เปิดบุ๊คมาร์คจากหน้าจอเครื่องเล่น"
  caption="แตะปุ่ม «เพิ่มเติม» (...) แล้วเลือก «บุ๊คมาร์ค»"
  width="400"
>}}

ในเครื่องเล่น ให้แตะปุ่ม **...** ที่ด้านซ้ายบน จากนั้นเลือก **บุ๊คมาร์ค**

### 4. เพิ่มบุ๊คมาร์คใหม่

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="สร้างบุ๊คมาร์คเสียงใหม่"
  caption="ตั้งเวลาและชื่อเรื่องเสริมสำหรับบุ๊คมาร์คใหม่"
  width="400"
>}}

เลือกเวลา (ค่าเริ่มต้นคือเวลาเล่นปัจจุบัน) และแก้ไขชื่อหากต้องการ

### 5. บันทึกบุ๊คมาร์ค

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="รายการบุ๊คมาร์คเสียง"
  caption="คุณสามารถสร้างบุ๊คมาร์คหลายรายการต่อเพลง"
  width="400"
>}}

แตะ **เสร็จสิ้น** เพื่อบันทึกบุ๊คมาร์คและกลับไปยังรายการบุ๊คมาร์ค

### 6. ข้ามไปยังบุ๊คมาร์ค

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="แตะบุ๊คมาร์คเพื่อข้ามไปยังช่วงเวลานั้น"
  caption="แตะชื่อบุ๊คมาร์คเพื่อข้ามไปยังจุดนั้นในเสียง"
  width="400"
>}}

แตะชื่อบุ๊คมาร์คใดก็ได้เพื่อเล่นต่อจากช่วงเวลานั้นทันที

### 7. ดูเพลงทั้งหมดที่มีบุ๊คมาร์ค

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="ตัวกรอง Soundy แสดงเพลงที่มีบุ๊คมาร์ค"
  caption="ใช้ตัวควบคุมแบบแบ่งส่วนเพื่อดูเพลงทั้งหมดที่มีบุ๊คมาร์คที่บันทึกไว้"
  width="400"
>}}

ใช้ตัวกรอง **เพลงทั้งหมด** เพื่อค้นหาและจัดการเสียงที่มีบุ๊คมาร์คทั้งหมดของคุณ

หากคุณมีคำถาม โปรดแสดงความคิดเห็นด้านล่างหรือติดต่อฝ่ายสนับสนุน!

## คำถามที่พบบ่อย

{{% details title="บุ๊คมาร์คเสียงคืออะไร?" closed="true" %}}
บุ๊คมาร์คเสียงช่วยให้คุณบันทึกการประทับเวลาเฉพาะในเพลง เพื่อให้คุณสามารถกลับไปยังช่วงเวลานั้นได้ในภายหลัง มีประโยชน์อย่างยิ่งสำหรับเนื้อหาแบบยาว เช่น หนังสือเสียง พอดแคสต์ และการบรรยาย
{{% /details %}}

{{% details title="ฉันสามารถสร้างบุ๊คมาร์คเสียงใน Evermusic หรือ Flacbox ได้ไหม?" closed="true" %}}
ได้ ทั้ง Evermusic และ Flacbox มีฟีเจอร์บุ๊คมาร์คเสียงฟรีสำหรับเพลงที่เก็บไว้ในบริการคลาวด์หรือที่จัดเก็บในเครื่อง
{{% /details %}}

{{% details title="ฉันสามารถสร้างบุ๊คมาร์คได้กี่รายการต่อเพลง?" closed="true" %}}
ไม่มีข้อจำกัดตายตัว คุณสามารถสร้างบุ๊คมาร์คหลายรายการต่อเพลงและจัดการจากรายการบุ๊คมาร์ค
{{% /details %}}

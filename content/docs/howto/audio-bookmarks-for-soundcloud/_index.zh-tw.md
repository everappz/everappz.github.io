**摘要：** Soundy 允許您在 SoundCloud 曲目中建立音訊書籤——非常適合在有聲書、Podcast 和講座中標記位置。

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="用於在 SoundCloud 上進行音訊書籤標記的 Soundy 應用程式"
  caption="Soundy：SoundCloud 音訊書籤"
  width="600"
>}}

使用 [Soundy](/products/soundy)，您可以收聽 SoundCloud 上喜愛的有聲書、講座和 Podcast，並建立音訊書籤來儲存位置或標記關鍵時刻。

要為 SoundCloud 建立音訊書籤，請按照以下步驟操作：

### 1. 搜尋曲目

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="在 Soundy 中使用 SoundCloud 搜尋 Podcast"
  caption="在 Soundy 中透過 SoundCloud 搜尋 Podcast 或曲目"
  width="400"
>}}

使用**搜尋**標籤頁或**我的帳戶**標籤頁尋找您的 SoundCloud 曲目。

### 2. 開始播放

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Soundy 中的播放清單詳情畫面"
  caption="點擊曲目以開啟播放器畫面"
  width="400"
>}}

點擊曲目以開啟播放器並開始收聽。

### 3. 開啟書籤選單

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="從播放器畫面開啟書籤"
  caption="點擊「更多」(...)按鈕並選擇「書籤」"
  width="400"
>}}

在播放器中，點擊左上角的 **...** 按鈕，然後選擇**書籤**。

### 4. 新增書籤

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="建立新的音訊書籤"
  caption="為新書籤設定時間和選用標題"
  width="400"
>}}

選擇時間（預設為目前播放時間），如有需要可編輯名稱。

### 5. 儲存書籤

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="音訊書籤清單"
  caption="您可以為每個曲目建立多個書籤"
  width="400"
>}}

點擊**完成**以儲存書籤並返回書籤清單。

### 6. 跳轉至書籤

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="點擊書籤跳轉到該時刻"
  caption="點擊書籤名稱跳轉到音訊中的該位置"
  width="400"
>}}

點擊任何書籤標題，即可從該時刻立即恢復播放。

### 7. 檢視所有帶書籤的曲目

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy 篩選器顯示帶書籤的曲目"
  caption="使用分段控制項檢視所有已儲存書籤的曲目"
  width="400"
>}}

使用**所有歌曲**篩選器尋找和管理所有帶書籤的音訊。

## 常見問題

{{% details title="什麼是音訊書籤？" closed="true" %}}
音訊書籤允許您在曲目中儲存特定的時間戳記，以便稍後跳轉回確切的時刻。它們對於有聲書、Podcast 和講座等長篇內容特別有用。
{{% /details %}}

{{% details title="我可以在 Evermusic 或 Flacbox 中建立音訊書籤嗎？" closed="true" %}}
可以。Evermusic 和 Flacbox 都包含免費的音訊書籤功能，適用於儲存在雲端服務或本機儲存空間中的曲目。
{{% /details %}}

{{% details title="每個曲目可以建立多少個書籤？" closed="true" %}}
沒有固定限制。您可以為每個曲目建立多個書籤，並從書籤清單中管理它們。
{{% /details %}}

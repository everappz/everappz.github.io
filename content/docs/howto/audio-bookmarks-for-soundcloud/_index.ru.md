**Кратко:** Soundy позволяет создавать аудиозакладки в треках SoundCloud — идеально для отметки позиций в аудиокнигах, подкастах и лекциях.

{{< figure

  alt="Приложение Soundy для аудиозакладок в SoundCloud"
  caption="Soundy: аудиозакладки для SoundCloud"
  width="600"
>}}

С помощью [Soundy](/products/soundy) вы можете слушать любимые аудиокниги, лекции и подкасты из SoundCloud — и создавать аудиозакладки для сохранения позиции или выделения ключевых моментов.

Чтобы создать аудиозакладку для SoundCloud, выполните следующие шаги:

### 1. Найдите трек

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Поиск подкаста в Soundy через SoundCloud"
  caption="Ищите подкасты или треки через SoundCloud в Soundy"
  width="400"
>}}

Используйте вкладку **Поиск** или вкладку **Мой аккаунт** для поиска треков SoundCloud.

### 2. Начните воспроизведение

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Экран деталей плейлиста в Soundy"
  caption="Нажмите на трек, чтобы открыть экран проигрывателя"
  width="400"
>}}

Нажмите на трек, чтобы открыть проигрыватель и начать прослушивание.

### 3. Откройте меню закладок

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Открытие закладок с экрана проигрывателя"
  caption="Нажмите кнопку «Ещё» (...) и выберите «Закладки»"
  width="400"
>}}

В проигрывателе нажмите кнопку **...** в верхнем левом углу, затем выберите **Закладки**.

### 4. Добавьте новую закладку

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Создание новой аудиозакладки"
  caption="Установите время и необязательное название для новой закладки"
  width="400"
>}}

Выберите время (по умолчанию — текущее время воспроизведения) и при необходимости отредактируйте название.

### 5. Сохраните закладку

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="Список аудиозакладок"
  caption="Вы можете создать несколько закладок для одного трека"
  width="400"
>}}

Нажмите **Готово**, чтобы сохранить закладку и вернуться к списку закладок.

### 6. Перейдите к закладке

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Нажатие на закладку для перехода к этому моменту"
  caption="Нажмите на название закладки, чтобы перейти к этой точке аудио"
  width="400"
>}}

Нажмите на любое название закладки, чтобы мгновенно возобновить воспроизведение с этого момента.

### 7. Просмотрите все треки с закладками

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Фильтр Soundy, показывающий треки с закладками"
  caption="Используйте сегментированный элемент управления для просмотра всех треков с сохранёнными закладками"
  width="400"
>}}

Используйте фильтр **Все песни** для поиска и управления всеми аудиозаписями с закладками.

Если у вас есть вопросы, оставьте комментарий ниже или свяжитесь со службой поддержки!

## Часто задаваемые вопросы

{{% details title="Что такое аудиозакладки?" closed="true" %}}
Аудиозакладки позволяют сохранять определённые временные метки в треке, чтобы вы могли вернуться к этому точному моменту позже. Они особенно полезны для длинного контента, такого как аудиокниги, подкасты и лекции.
{{% /details %}}

{{% details title="Можно ли создавать аудиозакладки в Evermusic или Flacbox?" closed="true" %}}
Да. Как Evermusic, так и Flacbox включают бесплатную функцию аудиозакладок для треков, хранящихся в облачных сервисах или локальном хранилище.
{{% /details %}}

{{% details title="Сколько закладок можно создать для одного трека?" closed="true" %}}
Фиксированного ограничения нет. Вы можете создать несколько закладок для одного трека и управлять ими из списка закладок.
{{% /details %}}

**Resum:** Soundy et permet crear marcadors d'àudio en pistes de SoundCloud -- ideal per marcar posicions en audiollibres, podcasts i conferències.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="Soundy app used for audio bookmarking on SoundCloud"
  caption="Soundy: Audio Bookmark For SoundCloud"
  width="600"
>}}

Amb [Soundy](/products/soundy), pots escoltar els teus audiollibres, conferències i podcasts preferits de SoundCloud — i crear marcadors d'àudio per guardar la teva posició o destacar moments clau.

Per crear un marcador d'àudio per a SoundCloud, segueix els passos següents:

### 1. Cercar una pista

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Searching for a podcast in Soundy using SoundCloud"
  caption="Cerca podcasts o pistes via SoundCloud dins de Soundy"
  width="400"
>}}

Utilitza la pestanya **Cerca** o la pestanya **El meu compte** per trobar les teves pistes de SoundCloud.

### 2. Iniciar la reproducció

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Playlist detail screen in Soundy"
  caption="Toca una pista per obrir la pantalla del reproductor"
  width="400"
>}}

Toca la pista per obrir el reproductor i començar a escoltar.

### 3. Obrir el menú de marcadors

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Opening bookmarks from the player screen"
  caption="Toca el botó MÉS (...) i tria 'Marcadors'"
  width="400"
>}}

Al reproductor, toca el botó **...** a la part superior esquerra, després selecciona **Marcadors**.

### 4. Afegir un nou marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Creating a new audio bookmark"
  caption="Estableix un temps i un títol opcional per al nou marcador"
  width="400"
>}}

Tria un temps (per defecte és el temps de reproducció actual) i edita el nom si cal.

### 5. Guardar el marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="List of audio bookmarks"
  caption="Pots crear diversos marcadors per pista"
  width="400"
>}}

Toca **Fet** per guardar el marcador i tornar a la llista de marcadors.

### 6. Saltar a un marcador

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Tapping a bookmark to jump to that moment"
  caption="Toca el nom d'un marcador per saltar a aquell punt de l'àudio"
  width="400"
>}}

Toca qualsevol títol de marcador per reprendre instantàniament la reproducció des d'aquell moment.

### 7. Veure totes les pistes amb marcadors

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy filter showing bookmarked tracks"
  caption="Utilitza el control segmentat per veure totes les pistes amb marcadors guardats"
  width="400"
>}}

Utilitza el filtre **Totes les cançons** per trobar i gestionar tot el teu àudio marcat.

Si tens alguna pregunta, deixa un comentari a continuació o contacta amb el suport!

## Preguntes freqüents

{{% details title="Què són els marcadors d'àudio?" closed="true" %}}
Els marcadors d'àudio et permeten guardar marques de temps específiques dins d'una pista perquè puguis tornar a aquell moment exacte més tard. Són especialment útils per a contingut de llarga durada com audiollibres, podcasts i conferències.
{{% /details %}}

{{% details title="Puc crear marcadors d'àudio a Evermusic o Flacbox?" closed="true" %}}
Sí. Tant Evermusic com Flacbox inclouen una funció gratuïta de marcadors d'àudio per a pistes emmagatzemades en serveis al núvol o emmagatzematge local.
{{% /details %}}

{{% details title="Quants marcadors puc crear per pista?" closed="true" %}}
No hi ha un límit fix. Pots crear diversos marcadors per pista i gestionar-los des de la llista de marcadors.
{{% /details %}}

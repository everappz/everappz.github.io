**Коротко:** Soundy дозволяє створювати аудіозакладки в треках SoundCloud -- ідеально для позначення позицій в аудіокнигах, подкастах та лекціях.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="Додаток Soundy для створення аудіозакладок у SoundCloud"
  caption="Soundy: Аудіозакладки для SoundCloud"
  width="600"
>}}

З [Soundy](/products/soundy) ви можете слухати улюблені аудіокниги, лекції та подкасти з SoundCloud, а також створювати аудіозакладки для збереження позиції або виділення ключових моментів.

Щоб створити аудіозакладку для SoundCloud, виконайте наступні кроки:

### 1. Знайдіть трек

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Пошук подкасту в Soundy через SoundCloud"
  caption="Шукайте подкасти або треки через SoundCloud у Soundy"
  width="400"
>}}

Використовуйте вкладку **Пошук** або вкладку **Мій акаунт**, щоб знайти ваші треки SoundCloud.

### 2. Почніть відтворення

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Екран деталей плейлиста в Soundy"
  caption="Натисніть на трек, щоб відкрити екран програвача"
  width="400"
>}}

Натисніть на трек, щоб відкрити програвач і почати слухати.

### 3. Відкрийте меню закладок

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Відкриття закладок з екрану програвача"
  caption="Натисніть кнопку БІЛЬШЕ (...) та виберіть 'Закладки'"
  width="400"
>}}

У програвачі натисніть кнопку **...** у верхньому лівому куті, потім виберіть **Закладки**.

### 4. Додайте нову закладку

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Створення нової аудіозакладки"
  caption="Встановіть час та необов'язкову назву для нової закладки"
  width="400"
>}}

Виберіть час (за замовчуванням — поточний час відтворення) та відредагуйте назву за потреби.

### 5. Збережіть закладку

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="Список аудіозакладок"
  caption="Ви можете створити кілька закладок для кожного треку"
  width="400"
>}}

Натисніть **Готово**, щоб зберегти закладку та повернутися до списку закладок.

### 6. Перейдіть до закладки

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Натискання закладки для переходу до цього моменту"
  caption="Натисніть на назву закладки, щоб перейти до цієї точки в аудіо"
  width="400"
>}}

Натисніть на будь-яку назву закладки, щоб миттєво відновити відтворення з цього моменту.

### 7. Перегляньте всі треки із закладками

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Фільтр Soundy, що показує треки із закладками"
  caption="Використовуйте сегментований елемент керування для перегляду всіх треків із збереженими закладками"
  width="400"
>}}

Використовуйте фільтр **Усі пісні**, щоб знайти та керувати всіма аудіо із закладками.

Якщо у вас є запитання, залиште коментар нижче або зверніться до служби підтримки!

## Часті запитання

{{% details title="Що таке аудіозакладки?" closed="true" %}}
Аудіозакладки дозволяють зберігати конкретні мітки часу в треку, щоб пізніше повернутися до точного моменту. Вони особливо корисні для довгого контенту, такого як аудіокниги, подкасти та лекції.
{{% /details %}}

{{% details title="Чи можу я створювати аудіозакладки в Evermusic або Flacbox?" closed="true" %}}
Так. Evermusic і Flacbox включають безкоштовну функцію аудіозакладок для треків, збережених у хмарних сервісах або локальному сховищі.
{{% /details %}}

{{% details title="Скільки закладок можна створити для одного треку?" closed="true" %}}
Фіксованого обмеження немає. Ви можете створити кілька закладок для кожного треку та керувати ними зі списку закладок.
{{% /details %}}

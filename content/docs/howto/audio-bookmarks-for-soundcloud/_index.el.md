**Περίληψη:** Το Soundy σας επιτρέπει να δημιουργείτε σελιδοδείκτες ήχου σε κομμάτια SoundCloud -- ιδανικό για την επισήμανση θέσεων σε ηχητικά βιβλία, podcasts και διαλέξεις.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="Soundy app used for audio bookmarking on SoundCloud"
  caption="Soundy: Audio Bookmark For SoundCloud"
  width="600"
>}}

Με το [Soundy](/products/soundy), μπορείτε να ακούτε τα αγαπημένα σας ηχητικά βιβλία, διαλέξεις και podcasts από το SoundCloud — και να δημιουργείτε σελιδοδείκτες ήχου για να αποθηκεύσετε τη θέση σας ή να επισημάνετε σημαντικές στιγμές.

Για να δημιουργήσετε έναν σελιδοδείκτη ήχου για το SoundCloud, ακολουθήστε τα παρακάτω βήματα:

### 1. Αναζήτηση κομματιού

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Searching for a podcast in Soundy using SoundCloud"
  caption="Αναζητήστε podcasts ή κομμάτια μέσω SoundCloud μέσα στο Soundy"
  width="400"
>}}

Χρησιμοποιήστε την καρτέλα **Αναζήτηση** ή την καρτέλα **Ο Λογαριασμός μου** για να βρείτε τα κομμάτια σας στο SoundCloud.

### 2. Έναρξη αναπαραγωγής

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Playlist detail screen in Soundy"
  caption="Πατήστε σε ένα κομμάτι για να ανοίξετε την οθόνη αναπαραγωγής"
  width="400"
>}}

Πατήστε στο κομμάτι για να ανοίξετε τον αναπαραγωγέα και να αρχίσετε να ακούτε.

### 3. Άνοιγμα μενού σελιδοδεικτών

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Opening bookmarks from the player screen"
  caption="Πατήστε το κουμπί ΠΕΡΙΣΣΟΤΕΡΑ (...) και επιλέξτε 'Σελιδοδείκτες'"
  width="400"
>}}

Στον αναπαραγωγέα, πατήστε το κουμπί **...** στο επάνω αριστερά, στη συνέχεια επιλέξτε **Σελιδοδείκτες**.

### 4. Προσθήκη νέου σελιδοδείκτη

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Creating a new audio bookmark"
  caption="Ορίστε μια ώρα και προαιρετικό τίτλο για τον νέο σελιδοδείκτη"
  width="400"
>}}

Επιλέξτε μια ώρα (η προεπιλογή είναι ο τρέχων χρόνος αναπαραγωγής) και επεξεργαστείτε το όνομα αν χρειαστεί.

### 5. Αποθήκευση σελιδοδείκτη

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="List of audio bookmarks"
  caption="Μπορείτε να δημιουργήσετε πολλαπλούς σελιδοδείκτες ανά κομμάτι"
  width="400"
>}}

Πατήστε **Ολοκλήρωση** για να αποθηκεύσετε τον σελιδοδείκτη και να επιστρέψετε στη λίστα σελιδοδεικτών.

### 6. Μετάβαση σε σελιδοδείκτη

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Tapping a bookmark to jump to that moment"
  caption="Πατήστε σε ένα όνομα σελιδοδείκτη για να μεταβείτε σε εκείνο το σημείο στον ήχο"
  width="400"
>}}

Πατήστε οποιονδήποτε τίτλο σελιδοδείκτη για να συνεχίσετε αμέσως την αναπαραγωγή από εκείνη τη στιγμή.

### 7. Προβολή όλων των κομματιών με σελιδοδείκτες

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy filter showing bookmarked tracks"
  caption="Χρησιμοποιήστε το τμηματοποιημένο στοιχείο ελέγχου για να δείτε όλα τα κομμάτια με αποθηκευμένους σελιδοδείκτες"
  width="400"
>}}

Χρησιμοποιήστε το φίλτρο **Όλα τα τραγούδια** για να βρείτε και να διαχειριστείτε όλον τον ήχο με σελιδοδείκτες.

Αν έχετε οποιεσδήποτε ερωτήσεις, αφήστε ένα σχόλιο παρακάτω ή επικοινωνήστε με την υποστήριξη!

## Συχνές ερωτήσεις

{{% details title="Τι είναι οι σελιδοδείκτες ήχου;" closed="true" %}}
Οι σελιδοδείκτες ήχου σας επιτρέπουν να αποθηκεύσετε συγκεκριμένες χρονοσφραγίδες μέσα σε ένα κομμάτι ώστε να μπορείτε να επιστρέψετε σε εκείνη ακριβώς τη στιγμή αργότερα. Είναι ιδιαίτερα χρήσιμοι για περιεχόμενο μεγάλης διάρκειας όπως ηχητικά βιβλία, podcasts και διαλέξεις.
{{% /details %}}

{{% details title="Μπορώ να δημιουργήσω σελιδοδείκτες ήχου στο Evermusic ή στο Flacbox;" closed="true" %}}
Ναι. Τόσο το Evermusic όσο και το Flacbox περιλαμβάνουν δωρεάν δυνατότητα σελιδοδεικτών ήχου για κομμάτια αποθηκευμένα σε υπηρεσίες cloud ή τοπικό αποθηκευτικό χώρο.
{{% /details %}}

{{% details title="Πόσους σελιδοδείκτες μπορώ να δημιουργήσω ανά κομμάτι;" closed="true" %}}
Δεν υπάρχει σταθερό όριο. Μπορείτε να δημιουργήσετε πολλαπλούς σελιδοδείκτες ανά κομμάτι και να τους διαχειριστείτε από τη λίστα σελιδοδεικτών.
{{% /details %}}

**בקצרה:** Soundy מאפשר לכם ליצור סימניות אודיו בשירי SoundCloud -- אידיאלי לסימון מיקומים בספרי שמע, פודקאסטים והרצאות.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="אפליקציית Soundy לסימניות אודיו ב-SoundCloud"

  width="600"
>}}

עם [Soundy](/products/soundy), תוכלו להאזין לספרי השמע, ההרצאות והפודקאסטים האהובים עליכם מ-SoundCloud — וליצור סימניות אודיו כדי לשמור את המיקום שלכם או להדגיש רגעים מרכזיים.

כדי ליצור סימנייה אודיו ל-SoundCloud, עקבו אחר השלבים הבאים:

### 1. חיפוש שיר

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="חיפוש פודקאסט ב-Soundy באמצעות SoundCloud"
  caption="חפשו פודקאסטים או שירים דרך SoundCloud בתוך Soundy"
  width="400"
>}}

השתמשו בלשונית **חיפוש** או בלשונית **החשבון שלי** כדי למצוא את שירי ה-SoundCloud שלכם.

### 2. התחלת השמעה

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="מסך פרטי רשימת השמעה ב-Soundy"
  caption="הקישו על שיר כדי לפתוח את מסך הנגן"
  width="400"
>}}

הקישו על השיר כדי לפתוח את הנגן ולהתחיל להאזין.

### 3. פתיחת תפריט הסימניות

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="פתיחת סימניות ממסך הנגן"
  caption="הקישו על כפתור עוד (...) ובחרו 'סימניות'"
  width="400"
>}}

בנגן, הקישו על כפתור **...** בפינה השמאלית העליונה, ואז בחרו **סימניות**.

### 4. הוספת סימנייה חדשה

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="יצירת סימניית אודיו חדשה"
  caption="הגדירו זמן ושם אופציונלי לסימנייה החדשה"
  width="400"
>}}

בחרו זמן (ברירת המחדל היא זמן ההשמעה הנוכחי) וערכו את השם במידת הצורך.

### 5. שמירת הסימנייה

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="רשימת סימניות אודיו"
  caption="ניתן ליצור מספר סימניות לכל שיר"
  width="400"
>}}

הקישו על **בוצע** כדי לשמור את הסימנייה ולחזור לרשימת הסימניות.

### 6. מעבר לסימנייה

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="הקשה על סימנייה כדי לקפוץ לאותו רגע"
  caption="הקישו על שם הסימנייה כדי לקפוץ לנקודה זו באודיו"
  width="400"
>}}

הקישו על כותרת סימנייה כלשהי כדי לחדש מיד את ההשמעה מאותו רגע.

### 7. צפייה בכל השירים עם סימניות

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="מסנן Soundy המציג שירים עם סימניות"
  caption="השתמשו בפקד המקוטע כדי לצפות בכל השירים עם סימניות שמורות"
  width="400"
>}}

השתמשו במסנן **כל השירים** כדי למצוא ולנהל את כל האודיו עם הסימניות שלכם.

אם יש לכם שאלות, השאירו תגובה למטה או פנו לתמיכה!

{{% /details %}}

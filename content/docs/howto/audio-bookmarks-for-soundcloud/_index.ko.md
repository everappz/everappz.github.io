**요약:** Soundy를 사용하면 SoundCloud 트랙에 오디오 북마크를 만들 수 있습니다. 오디오북, 팟캐스트, 강의에서 위치를 표시하는 데 이상적입니다.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"

[Soundy](/products/soundy)를 사용하면 SoundCloud에서 좋아하는 오디오북, 강의, 팟캐스트를 들으면서 오디오 북마크를 만들어 재생 위치를 저장하거나 주요 순간을 표시할 수 있습니다.

SoundCloud의 오디오 북마크를 만들려면 아래 단계를 따르세요:

### 1. 트랙 검색하기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Soundy에서 SoundCloud를 사용하여 팟캐스트 검색"
  caption="Soundy 내에서 SoundCloud를 통해 팟캐스트 또는 트랙 검색"
  width="400"
>}}

**검색** 탭 또는 **내 계정** 탭을 사용하여 SoundCloud 트랙을 찾으세요.

### 2. 재생 시작하기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Soundy의 재생목록 상세 화면"
  caption="트랙을 탭하여 플레이어 화면 열기"
  width="400"
>}}

트랙을 탭하여 플레이어를 열고 듣기를 시작하세요.

### 3. 북마크 메뉴 열기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="플레이어 화면에서 북마크 열기"
  caption="MORE (...) 버튼을 탭하고 '북마크'를 선택"
  width="400"
>}}

플레이어에서 왼쪽 상단의 **...** 버튼을 탭한 다음 **북마크**를 선택합니다.

### 4. 새 북마크 추가하기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="새 오디오 북마크 만들기"
  caption="새 북마크의 시간과 선택적 제목 설정"
  width="400"
>}}

시간을 선택하고(기본값은 현재 재생 시간) 필요에 따라 이름을 편집합니다.

### 5. 북마크 저장하기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="오디오 북마크 목록"
  caption="트랙당 여러 북마크 생성 가능"
  width="400"
>}}

**완료됨**을 탭하여 북마크를 저장하고 북마크 목록으로 돌아갑니다.

### 6. 북마크로 이동하기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="북마크를 탭하여 해당 순간으로 이동"
  caption="북마크 이름을 탭하여 오디오의 해당 지점으로 이동"
  width="400"
>}}

아무 북마크 제목이나 탭하면 해당 순간부터 즉시 재생이 재개됩니다.

### 7. 북마크가 있는 모든 트랙 보기

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="북마크된 트랙을 보여주는 Soundy 필터"
  caption="세그먼트 컨트롤을 사용하여 저장된 북마크가 있는 모든 트랙 보기"
  width="400"
>}}

**전체 곡** 필터를 사용하여 북마크된 모든 오디오를 찾고 관리하세요.

## FAQ

{{% details title="오디오 북마크란 무엇인가요?" closed="true" %}}
오디오 북마크를 사용하면 트랙 내의 특정 타임스탬프를 저장하여 나중에 정확한 순간으로 돌아갈 수 있습니다. 오디오북, 팟캐스트, 강의와 같은 긴 콘텐츠에 특히 유용합니다.
{{% /details %}}

{{% details title="Evermusic이나 Flacbox에서도 오디오 북마크를 만들 수 있나요?" closed="true" %}}
네. Evermusic과 Flacbox 모두 클라우드 서비스 또는 로컬 저장소에 저장된 트랙에 대한 무료 오디오 북마크 기능을 포함하고 있습니다.
{{% /details %}}

{{% details title="트랙당 몇 개의 북마크를 만들 수 있나요?" closed="true" %}}
고정된 제한은 없습니다. 트랙당 여러 개의 북마크를 만들고 북마크 목록에서 관리할 수 있습니다.
{{% /details %}}

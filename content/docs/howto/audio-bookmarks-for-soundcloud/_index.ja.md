**要約:** Soundyを使えば、SoundCloudのトラックにオーディオブックマークを作成できます。オーディオブック、ポッドキャスト、講義の位置をマークするのに最適です。

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="SoundCloudでオーディオブックマークに使用されるSoundyアプリ"
  caption="Soundy：SoundCloudのオーディオブックマーク"
  width="600"
>}}

[Soundy](/products/soundy)を使えば、SoundCloudのお気に入りのオーディオブック、講義、ポッドキャストを聴きながら、オーディオブックマークを作成して再生位置を保存したり、重要な箇所をハイライトしたりできます。

SoundCloudのオーディオブックマークを作成するには、以下の手順に従ってください：

### 1. トラックを検索する

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="SoundyでSoundCloudを使ってポッドキャストを検索"
  caption="Soundy内でSoundCloudを使ってポッドキャストやトラックを検索"
  width="400"
>}}

**検索**タブまたは**マイアカウント**タブを使ってSoundCloudのトラックを見つけます。

### 2. 再生を開始する

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Soundyのプレイリスト詳細画面"
  caption="トラックをタップしてプレーヤー画面を開く"
  width="400"
>}}

トラックをタップしてプレーヤーを開き、リスニングを始めます。

### 3. ブックマークメニューを開く

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="プレーヤー画面からブックマークを開く"
  caption="MORE (...)ボタンをタップして「ブックマーク」を選択"
  width="400"
>}}

プレーヤーで、左上の**...**ボタンをタップし、**ブックマーク**を選択します。

### 4. 新しいブックマークを追加する

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="新しいオーディオブックマークを作成"
  caption="新しいブックマークの時間と任意のタイトルを設定"
  width="400"
>}}

時間を選択し（デフォルトは現在の再生時間）、必要に応じて名前を編集します。

### 5. ブックマークを保存する

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="オーディオブックマークのリスト"
  caption="トラックごとに複数のブックマークを作成可能"
  width="400"
>}}

**完了**をタップしてブックマークを保存し、ブックマークリストに戻ります。

### 6. ブックマークにジャンプする

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="ブックマークをタップしてその瞬間にジャンプ"
  caption="ブックマーク名をタップしてオーディオのその地点にジャンプ"
  width="400"
>}}

任意のブックマークタイトルをタップすると、その瞬間から即座に再生が再開されます。

### 7. ブックマーク付きの全トラックを表示する

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="ブックマーク付きトラックを表示するSoundyフィルター"
  caption="セグメントコントロールを使用して保存済みブックマークのあるすべてのトラックを表示"
  width="400"
>}}

**全曲**フィルターを使って、ブックマーク付きのすべてのオーディオを見つけて管理します。

## FAQ

{{% details title="オーディオブックマークとは何ですか？" closed="true" %}}
オーディオブックマークを使えば、トラック内の特定のタイムスタンプを保存し、後でその正確な瞬間に戻ることができます。オーディオブック、ポッドキャスト、講義などの長時間コンテンツに特に便利です。
{{% /details %}}

{{% details title="EvermusciやFlacboxでもオーディオブックマークを作成できますか？" closed="true" %}}
はい。EvermusciとFlacboxの両方に、クラウドサービスやローカルストレージに保存されたトラックのための無料オーディオブックマーク機能が含まれています。
{{% /details %}}

{{% details title="1つのトラックにいくつのブックマークを作成できますか？" closed="true" %}}
固定の制限はありません。トラックごとに複数のブックマークを作成し、ブックマークリストから管理できます。
{{% /details %}}

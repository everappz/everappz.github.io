**Sammanfattning:** Soundy låter dig skapa ljudbokmärken i SoundCloud-spår -- perfekt för att markera positioner i ljudböcker, poddar och föreläsningar.

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_617fb607236c4b239b64b8816b7d19fe~mv2.jpg"
  alt="Soundy-appen för ljudbokmärkning på SoundCloud"
  caption="Soundy: ljudbokmärken för SoundCloud"
  width="600"
>}}

Med [Soundy](/products/soundy) kan du lyssna på dina favoritljudböcker, föreläsningar och poddar från SoundCloud -- och skapa ljudbokmärken för att spara din position eller markera viktiga ögonblick.

Följ stegen nedan för att skapa ett ljudbokmärke för SoundCloud:

### 1. Sök efter ett spår

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_646b3e7a15994df08b111736acfc2d53~mv2.png"
  alt="Sökning efter en podd i Soundy via SoundCloud"
  caption="Sök efter poddar eller spår via SoundCloud i Soundy"
  width="400"
>}}

Använd fliken **Sök** eller fliken **Mitt konto** för att hitta dina SoundCloud-spår.

### 2. Starta uppspelning

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_058904e083cf40f0986c252ab743e031~mv2.png"
  alt="Skärm med spellistdetaljer i Soundy"
  caption="Tryck på ett spår för att öppna spelarskärmen"
  width="400"
>}}

Tryck på spåret för att öppna spelaren och börja lyssna.

### 3. Öppna bokmärkesmenyn

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_6e93c487869f4edb8652839ecd0e3ff5~mv2.png"
  alt="Öppna bokmärken från spelarskärmen"
  caption="Tryck på knappen «Mer» (...) och välj «Bokmärken»"
  width="400"
>}}

I spelaren trycker du på knappen **...** uppe till vänster och väljer sedan **Bokmärken**.

### 4. Lägg till ett nytt bokmärke

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_758a132ab32045398be2e1cef1ef99b3~mv2.png"
  alt="Skapa ett nytt ljudbokmärke"
  caption="Ställ in en tid och valfri titel för det nya bokmärket"
  width="400"
>}}

Välj en tid (standard är aktuell uppspelningstid) och redigera namnet vid behov.

### 5. Spara bokmärket

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_9e9a22ff2eba4fa2a269692ef6c93f42~mv2.png"
  alt="Lista över ljudbokmärken"
  caption="Du kan skapa flera bokmärken per spår"
  width="400"
>}}

Tryck på **Färdig** för att spara bokmärket och återgå till bokmärkeslistan.

### 6. Hoppa till ett bokmärke

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_3edb9c68d94f41e5a1e7cac68e689871~mv2.png"
  alt="Tryck på ett bokmärke för att hoppa till det ögonblicket"
  caption="Tryck på ett bokmärkesnamn för att hoppa till den punkten i ljudet"
  width="400"
>}}

Tryck på valfri bokmärkestitel för att omedelbart återuppta uppspelningen från det ögonblicket.

### 7. Visa alla spår med bokmärken

{{< figure
  src="/docs/howto/audio-bookmarks-for-soundcloud/21260c_de5ea7bf71894dfe9a70c90d8ebe7d40~mv2.png"
  alt="Soundy-filter som visar bokmärkta spår"
  caption="Använd den segmenterade kontrollen för att visa alla spår med sparade bokmärken"
  width="400"
>}}

Använd filtret **Alla låtar** för att hitta och hantera allt ditt bokmärkta ljud.

Om du har några frågor, lämna en kommentar nedan eller kontakta supporten!

## Vanliga frågor

{{% details title="Vad är ljudbokmärken?" closed="true" %}}
Ljudbokmärken låter dig spara specifika tidsstämplar i ett spår så att du kan hoppa tillbaka till exakt det ögonblicket senare. De är särskilt användbara för långt innehåll som ljudböcker, poddar och föreläsningar.
{{% /details %}}

{{% details title="Kan jag skapa ljudbokmärken i Evermusic eller Flacbox?" closed="true" %}}
Ja. Både Evermusic och Flacbox inkluderar en gratis funktion för ljudbokmärken för spår lagrade i molntjänster eller lokal lagring.
{{% /details %}}

{{% details title="Hur många bokmärken kan jag skapa per spår?" closed="true" %}}
Det finns ingen fast gräns. Du kan skapa flera bokmärken per spår och hantera dem från bokmärkeslistan.
{{% /details %}}

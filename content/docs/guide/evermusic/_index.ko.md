Evermusic는 iPhone, iPad, Mac을 완전한 스트리밍 및 오프라인 음악 시스템으로 바꿔주는 강력한 클라우드 음악 플레이어 및 라이브러리 관리자입니다. 이미 소유하고 있는 클라우드 계정 및 NAS와 함께 작동하며, 기기가 디코딩할 수 있는 거의 모든 오디오 형식을 재생하고, 심층적인 오디오 엔진(10밴드 이퀄라이저, 크로스페이드, 무갭 재생, 공간 오디오, 피치 보정)을 제공합니다.

두 가지 에디션으로 제공됩니다: Evermusic(무료, 광고 지원) 및 Evermusic Pro(유료, 광고 없음, 전체 기능). 무료 에디션의 평생, 월간, 연간 프리미엄 플랜은 동일한 기능 세트를 제공하며, 두 에디션 모두 iCloud 및 Family Sharing을 통해 iOS와 Mac 간에 구매를 공유합니다.

### 어디서나 나의 음악

Evermusic는 나만의 클라우드 음악 스트리밍 시스템을 구축할 수 있게 해줍니다 — Spotify처럼, 하지만 어떤 제한도 없이. 다양한 소스를 연결하고 온디맨드로 트랙을 스트리밍하거나 다운로드하세요:

- **개인 클라우드 계정:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘).
- **자체 호스팅 서버 및 미디어 라이브러리:** Plex · Jellyfin · Emby · Subsonic(모든 Subsonic 호환 서버 포함 — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud(공유 API를 통한 Owncloud 포함) · Synology Drive · QNAP.
- **NAS 및 파일 공유 프로토콜:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP / FTPS · SFTP (SSH 파일 전송 프로토콜, 비밀번호 또는 공개키 인증) · NFS · DLNA / UPnP (재생 및 다운로드). Apple Time Capsule, Synology, QNAP, WD My Cloud Home, Buffalo, 모든 Linux Samba / NFS / SSH 호스트, 또는 Mac이나 Windows PC의 공유 폴더와 함께 작동합니다.
- **S3 호환 오브젝트 스토리지:** AWS S3 · Backblaze B2 · Wasabi · Cloudflare R2 · MinIO · DigitalOcean Spaces · Linode Object Storage · IBM Cloud Object Storage · 모든 S3 API 엔드포인트.
- **로컬 네트워크 검색:** 사용 가능한 장치 섹션은 Wi-Fi의 모든 Bonjour / mDNS 장치를 자동으로 나열합니다 — Plex, Jellyfin, Emby 서버, Synology, QNAP, 드라이브가 연결된 AirPort 라우터, Time Capsule — IP를 입력하지 않고 탭하여 연결할 수 있습니다.
- **장치 및 외부 소스:** iPod / Apple Music 라이브러리(비 DRM 트랙), 시스템 파일 앱의 파일(다른 앱, 외부 SSD, 마운트된 폴더), Apple 인증 카드 리더 및 iXpand Flash Drive를 통한 USB 플래시 드라이브, Wi-Fi Drive(데스크톱 브라우저에서 드래그 앤 드롭), USB 케이블을 통한 iTunes / Finder File Sharing.

단 하나의 앱에 30개 이상의 소스 유형이 있습니다. 음악이 클라우드에 있으면, 모든 기기에서 동일한 라이브러리로 언제 어디서나 재생할 수 있습니다. 백업 및 복원을 통해 전체 라이브러리를 단일 파일로 스냅샷하고 몇 초 내에 다른 iPhone, iPad, 또는 Mac으로 가져올 수 있습니다.

### 부드러운 재생

Evermusic는 스마트 프리로딩과 구성 가능한 재생 캐시를 사용하여 느린 연결에서도 노래가 부드럽게 재생되도록 합니다. 스마트 스트리밍은 AVAssetResourceLoader를 사용하여 파일의 나머지 부분이 백그라운드에서 다운로드되는 동안 즉시 재생을 시작합니다. 오프라인으로 듣기 위해 앨범, 아티스트, 플레이리스트, 또는 개별 노래를 다운로드할 수도 있습니다 — 저장 공간이 부족하면 캐시된 파일을 삭제하고 클라우드에서 계속 스트리밍하면 됩니다.

### 쉬운 음악 정렬

Evermusic에 노래를 추가하면, 백그라운드 메타데이터 리더가 각 파일을 파싱하고 노래, 재생되지 않은 노래, 앨범, 앨범 아티스트, 아티스트, 장르, 작곡가별로 정렬된 깔끔한 음악 라이브러리를 구축합니다. 앨범에는 세 가지 하위 뷰가 있습니다: 모든 앨범, 독점 앨범, 솔로 앨범 — 컴필레이션과 다른 아티스트의 동명 앨범이 충돌하지 않도록 합니다.

### 노래 정보 쉽게 수정

노래 제목이나 앨범 정보가 잘못되었거나 누락된 경우 걱정하지 마세요. Evermusic에는 몇 초 만에 메타데이터를 수정할 수 있는 내장 태그 편집기가 있습니다. 제목, 아티스트, 앨범, 앨범 아티스트, 연도, 장르, 앨범 커버를 업데이트하고, 한 번의 탭으로 깨진 인코딩(키릴 문자나 아시아 텍스트가 깨져 보이는 경우)을 정규화할 수 있습니다. 편집기는 MusicBrainz를 사용하여 누락된 태그와 앨범 아트워크를 자동으로 찾습니다.

### 간편한 파일 전송

컴퓨터에서 iPhone 또는 iPad로 음악을 보내는 것은 쉽습니다. Evermusic는 SMB, WebDAV, FTP / FTPS, SFTP, NFS, DLNA, Wi-Fi Drive(브라우저에서 드래그 앤 드롭), iTunes / Finder File Sharing(USB 케이블), 연결된 모든 클라우드 계정에서 직접 다운로드를 지원합니다. 기기 저장 공간을 사용하지 않고도 Apple Time Capsule, WD My Cloud Home, Synology, QNAP, Buffalo, 또는 표준 프로토콜 중 하나를 노출하는 다른 NAS에 연결할 수 있습니다.

### 강력한 사운드 컨트롤

Evermusic에는 iPod 스타일 프리셋(Acoustic, Bass Booster, Classical, Dance, Electronic, Flat, Hip-Hop, Jazz, Latin, Loudness, Lounge, Piano, Pop, R&B, Rock, Small Speakers, Spoken Word, Treble Booster, Vocal Booster)이 있는 완전한 10밴드 오디오 이퀄라이저, 조용한 트랙을 위한 프리앰프, 내보내기 및 가져오기 가능한 커스텀 프리셋, RMS 기반 음량 계산이 포함되어 있습니다. 크로스페이드 재생(1–15초), 무갭 재생, 공간 오디오, 피치 보정, 384kHz 모노 또는 스테레오의 CoreAudio 출력을 추가하면 원하는 방식으로 사운드를 정확하게 조정할 수 있습니다.

### 오디오북에 최적

오디오북, 팟캐스트, 또는 강의를 듣는다면 Evermusic이 완벽합니다. 트랙당 여러 북마크를 추가하고, 마지막 위치에서 자동으로 재개하고, 재생 속도를 0.02×에서 3×로 변경하고(정밀 슬라이더 모드 포함), 사용자 정의 간격으로 수면 타이머를 설정하고, 같은 화면에서 댓글, 내장된 가사, LRC 파일을 볼 수 있습니다.

### 어디서나 작동

전화기나 태블릿에만 국한되지 않습니다. Evermusic는 Apple CarPlay(iOS 전용, 차량 UI에 최적화), AirPlay 2(여러 스피커에 동시 스트리밍), Google Cast / Chromecast, 잠금 화면 위젯, 지금 재생 중 컨트롤, Last.fm 스크로블링(재생을 Last.fm 계정으로 자동 전송하여 통계 및 추천 제공), Mac 키보드 단축키와도 작동합니다. 집에 있든, 차 안에 있든, 여행 중이든 음악은 항상 함께합니다.

### Evermusic 시작하기

이 가이드는 iPhone, iPad, 또는 Mac에서 Evermusic를 최대한 활용하는 데 도움이 됩니다. 클라우드에서 음악을 스트리밍하는 방법, 자체 저장소에 있는 라이브러리를 정리하는 방법, 플레이리스트 및 오프라인 폴더를 관리하는 방법, 오디오 엔진을 세밀하게 조정하는 방법, 오디오북을 듣는 방법을 배우세요. Evermusic는 하나의 쉬운 앱에서 음악 컬렉션에 대한 완전한 제어를 제공합니다.<br><br>

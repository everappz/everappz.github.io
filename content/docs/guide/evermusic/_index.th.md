Evermusic คือเครื่องเล่นเพลงคลาวด์และตัวจัดการไลบรารีที่ทรงพลัง ซึ่งเปลี่ยน iPhone, iPad หรือ Mac ของคุณให้กลายเป็นระบบเพลงสตรีมมิ่งและออฟไลน์ที่มีฟีเจอร์ครบครัน ทำงานร่วมกับบัญชีคลาวด์และ NAS ที่คุณมีอยู่แล้ว เล่นได้แทบทุกรูปแบบเสียงที่อุปกรณ์ของคุณรองรับ และมีเอ็นจินเสียงขั้นสูง (อีควอไลเซอร์ 10 แบนด์, ครอสเฟด, เกปเลส, เสียงสเปเชียล, การแก้ไขระดับเสียง)

มีให้บริการสองรุ่น: Evermusic (ฟรี มีโฆษณา) และ Evermusic Pro (ชำระเงิน ไม่มีโฆษณา ชุดฟีเจอร์เต็มรูปแบบ) แผนพรีเมียมแบบตลอดชีพ รายเดือน และรายปีในรุ่นฟรีจะปลดล็อกชุดฟีเจอร์เดียวกัน ทั้งสองรุ่นแชร์การซื้อระหว่าง iOS และ Mac ผ่าน iCloud และ Family Sharing

### เพลงของคุณทุกที่

Evermusic ให้คุณสร้างระบบสตรีมเพลงคลาวด์ของตัวเอง — เหมือน Spotify แต่ไม่มีข้อจำกัดใดๆ เชื่อมต่อแหล่งที่มาหลายรูปแบบและสตรีมหรือดาวน์โหลดเพลงตามต้องการ:

- **บัญชีคลาวด์ส่วนตัว:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘)
- **เซิร์ฟเวอร์ที่โฮสต์เองและไลบรารีมีเดีย:** Plex · Jellyfin · Emby · Subsonic (และทุกเซิร์ฟเวอร์ที่รองรับ Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud (และ Owncloud ผ่าน shared API) · Synology Drive · QNAP
- **NAS และโปรโตคอลแชร์ไฟล์:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP / FTPS · SFTP (SSH File Transfer Protocol, รหัสผ่านหรือ public-key auth) · NFS · DLNA / UPnP (เล่นและดาวน์โหลด) รองรับ Apple Time Capsule, Synology, QNAP, WD My Cloud Home, Buffalo, Linux Samba / NFS / SSH host ใดก็ได้ หรือโฟลเดอร์แชร์บน Mac หรือ Windows PC
- **Object storage ที่รองรับ S3:** AWS S3 · Backblaze B2 · Wasabi · Cloudflare R2 · MinIO · DigitalOcean Spaces · Linode Object Storage · IBM Cloud Object Storage · S3-API endpoint ใดก็ได้
- **การค้นพบในเครือข่ายท้องถิ่น:** ส่วน อุปกรณ์ที่มีให้บริการ จะแสดงรายการอุปกรณ์ Bonjour / mDNS ทุกเครื่องใน Wi-Fi ของคุณโดยอัตโนมัติ — เซิร์ฟเวอร์ Plex, Jellyfin, Emby, Synology, QNAP, เราเตอร์ AirPort ที่มีไดรฟ์เชื่อมต่อ, Time Capsule — เพื่อให้คุณเชื่อมต่อได้ด้วยการแตะโดยไม่ต้องพิมพ์ IP
- **อุปกรณ์และแหล่งภายนอก:** ไลบรารี iPod / Apple Music (เพลงที่ไม่มี DRM), ไฟล์ในแอป Files ของระบบ (แอปอื่นๆ, SSD ภายนอก, โฟลเดอร์ที่เมาต์), USB flash drive ผ่านเครื่องอ่านการ์ด Apple Certified และ iXpand Flash Drives, Wi-Fi Drive (ลากและวางในเบราว์เซอร์บนเดสก์ท็อป) และ iTunes / Finder File Sharing ผ่านสาย USB

นั่นคือแหล่งที่มากกว่า 30 ประเภทในแอปเดียว เมื่อเพลงของคุณอยู่บนคลาวด์ คุณสามารถเล่นได้ทุกที่ทุกเวลา ด้วยไลบรารีเดียวกันบนทุกอุปกรณ์ ฟีเจอร์สำรองและกู้คืนข้อมูลช่วยให้คุณบันทึกภาพรวมของทั้งไลบรารีเป็นไฟล์เดียวและนำเข้าบน iPhone, iPad หรือ Mac เครื่องอื่นได้ในไม่กี่วินาที

### การเล่นที่ราบรื่น

Evermusic ใช้การโหลดล่วงหน้าอัจฉริยะและแคชการเล่นที่กำหนดค่าได้ เพื่อให้เพลงเล่นได้อย่างราบรื่นแม้ในการเชื่อมต่อที่ช้า การสตรีมอัจฉริยะใช้ AVAssetResourceLoader เพื่อให้การเล่นเริ่มต้นทันทีในขณะที่ส่วนที่เหลือของไฟล์กำลังดาวน์โหลดในพื้นหลัง คุณยังสามารถดาวน์โหลดอัลบั้ม ศิลปิน เพลย์ลิสต์ หรือเพลงแต่ละเพลงเพื่อฟังแบบออฟไลน์ และหากพื้นที่เก็บข้อมูลเต็ม เพียงลบไฟล์ที่แคชไว้แล้วสตรีมต่อจากคลาวด์

### การจัดเรียงเพลงที่ง่ายดาย

เมื่อคุณเพิ่มเพลงลงใน Evermusic โปรแกรมอ่านข้อมูล metadata ในพื้นหลังจะวิเคราะห์ไฟล์แต่ละไฟล์และสร้างไลบรารีเพลงที่สะอาดตามหมวดหมู่ เพลง, เพลงที่ยังไม่ได้เล่น, อัลบั้ม, ศิลปินอัลบั้ม, ศิลปิน, แนวเพลง และนักแต่งเพลง อัลบั้มมีสามมุมมองย่อย: อัลบั้มทั้งหมด, อัลบั้มเอกสิทธิ์ และอัลบั้มเดี่ยว เพื่อไม่ให้คอมไพเลชั่นและอัลบั้มที่มีชื่อเดียวกันของศิลปินต่างกันปะปนกัน

### แก้ไขข้อมูลเพลงได้ง่าย

หากชื่อเพลงหรือข้อมูลอัลบั้มไม่ถูกต้องหรือหายไป ไม่ต้องกังวล Evermusic มีโปรแกรมแก้ไขแท็กในตัวที่ช่วยให้คุณแก้ไข metadata ได้ในไม่กี่วินาที คุณสามารถอัปเดตชื่อ ศิลปิน อัลบั้ม ศิลปินอัลบั้ม ปี แนวเพลง และปกอัลบั้ม รวมถึงทำให้การเข้ารหัสที่เสียหาย (ข้อความซีริลลิกหรือเอเชียที่แสดงผิดปกติ) เป็นปกติด้วยการแตะเพียงครั้งเดียว โปรแกรมแก้ไขใช้ MusicBrainz เพื่อค้นหาแท็กและภาพปกอัลบั้มที่หายไปโดยอัตโนมัติ

### การโอนไฟล์ที่ง่ายดาย

การส่งเพลงจากคอมพิวเตอร์ไปยัง iPhone หรือ iPad ของคุณเป็นเรื่องง่าย Evermusic รองรับ SMB, WebDAV, FTP / FTPS, SFTP, NFS, DLNA, Wi-Fi Drive (ลากและวางในเบราว์เซอร์), iTunes / Finder File Sharing (สาย USB) และการดาวน์โหลดโดยตรงจากบัญชีคลาวด์ที่เชื่อมต่อ คุณยังสามารถเชื่อมต่อ Apple Time Capsule, WD My Cloud Home, Synology, QNAP, Buffalo หรือ NAS อื่นๆ ที่รองรับโปรโตคอลมาตรฐานเหล่านี้ โดยไม่ต้องใช้พื้นที่เก็บข้อมูลของอุปกรณ์

### การควบคุมเสียงที่ทรงพลัง

Evermusic มีอีควอไลเซอร์เสียง 10 แบนด์พร้อมพรีเซ็ตแบบ iPod (Acoustic, Bass Booster, Classical, Dance, Electronic, Flat, Hip-Hop, Jazz, Latin, Loudness, Lounge, Piano, Pop, R&B, Rock, Small Speakers, Spoken Word, Treble Booster, Vocal Booster), พรีแอมปลิฟายเออร์สำหรับเพลงที่เบา, พรีเซ็ตที่กำหนดเองที่ส่งออกและนำเข้าได้ และการคำนวณระดับเสียงแบบ RMS เพิ่มครอสเฟดการเล่น (1–15 วินาที), เกปเลสการเล่น, เสียงสเปเชียล, การแก้ไขระดับเสียง และ CoreAudio output สูงสุด 384 kHz โมโนหรือสเตอริโอ แล้วคุณจะปรับแต่งเสียงได้ตามที่ต้องการ

### เหมาะสำหรับหนังสือเสียง

หากคุณฟังหนังสือเสียง พอดแคสต์ หรือการบรรยายที่บันทึกไว้ Evermusic เหมาะสมที่สุด คุณสามารถเพิ่มบุ๊คมาร์คหลายรายการต่อเพลง เล่นต่อจากตำแหน่งล่าสุดโดยอัตโนมัติ เปลี่ยนความเร็วการเล่นจาก 0.02× ถึง 3× (พร้อมโหมดตัวเลื่อนที่แม่นยำ), ตั้งตั้งเวลานอนด้วยช่วงเวลาที่กำหนดเอง และดูความคิดเห็น เนื้อเพลงที่ฝังไว้ และไฟล์ LRC บนหน้าจอเดียวกัน

### ใช้งานได้ทุกที่

คุณไม่จำกัดอยู่แค่โทรศัพท์หรือแท็บเล็ต Evermusic ยังทำงานร่วมกับ Apple CarPlay (iOS เท่านั้น ปรับแต่งสำหรับ UI ในรถ), AirPlay 2 (สตรีมไปยังลำโพงหลายตัวพร้อมกัน), Google Cast / Chromecast, วิดเจ็ตหน้าจอล็อค, ตัวควบคุม Now Playing, Last.fm scrobbling (ส่งข้อมูลการเล่นไปยังบัญชี Last.fm ของคุณโดยอัตโนมัติเพื่อสถิติและคำแนะนำ) และแป้นพิมพ์ลัด Mac ไม่ว่าคุณจะอยู่บ้าน ในรถ หรือเดินทาง เพลงของคุณอยู่กับคุณตลอดเวลา

### เริ่มต้นใช้งาน Evermusic

คู่มือนี้จะช่วยให้คุณได้รับประโยชน์สูงสุดจาก Evermusic บน iPhone, iPad หรือ Mac เรียนรู้วิธีสตรีมเพลงจากคลาวด์ จัดระเบียบไลบรารีบนพื้นที่เก็บข้อมูลของคุณเอง จัดการเพลย์ลิสต์และโฟลเดอร์ออฟไลน์ ปรับแต่งเอ็นจินเสียง และฟังหนังสือเสียง Evermusic ให้การควบคุมคอลเลคชันเพลงของคุณอย่างเต็มรูปแบบในแอปเดียวที่ใช้งานง่าย<br><br>

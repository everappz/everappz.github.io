## 概述

「播放列表」部分为您提供将曲目整理成列表的工具。它包括展示您所有已创建播放列表的内容视图、导航栏中提供各种播放列表相关操作的「...」按钮，以及带有「搜索」、「全部播放」和「随机播放全部」按钮的导航工具栏。此外，每个单独的播放列表在播放列表标题附近都有一个「...」按钮，提供一系列特定于该播放列表的操作。

## 创建播放列表

要创建新播放列表，请点击「+」按钮或导航栏右上角的「...」按钮，选择「新建播放列表」并为播放列表命名。命名后，点击「保存」。

{{< cards cols="1">}}
  {{< card title="" subtitle="创建新播放列表" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-add-new.webp" >}}
{{< /cards >}}

这会打开「添加歌曲」对话框，您可以在其中选择要添加到新播放列表的曲目。曲目按来源类型分类，您有几个选项：

- **音乐库**：来自您音乐库的曲目。
- **此应用程序中的文件**：存储在应用程序 Documents 目录中的音频文件。
- **此 iPhone/iPad/Mac 上的文件**：位于您设备上应用程序外部其他文件夹中的音频文件。
- **连接**：存储在已连接云存储服务中的在线文件。

默认情况下，您只能将一首曲目添加到播放列表一次。要允许播放列表中有重复歌曲，请在应用程序设置 - 音乐库 - 播放列表 - 播放列表中的重复项 - 启用中启用此功能。

## 导入播放列表

在 Evermusic 中，我们添加了 M3U 文件导入功能，这样您就不必手动创建播放列表。

{{< cards cols="1">}}
  {{< card title="" subtitle="从文件来源导入播放列表" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-import-from-files.webp" >}}
{{< /cards >}}

首先，转到「播放列表」部分。然后，点击右上角的「更多」按钮。从出现的菜单中，选择「导入播放列表」选项。

在下一个屏幕上，选择文件位置。支持的选项包括：

- 已连接的云存储
- 应用程序中的文件
- 您设备上的文件

让我们选择已连接的云存储并打开包含播放列表文件的文件夹。支持的播放列表文件扩展名包括 M3U、M3U8 和 CUE。选择播放列表文件并点击「完成」以确认您的选择。

应用程序将解析播放列表文件，创建曲目列表，并在存储中找到这些文件以编制最终播放列表，该播放列表将被导入到音乐库中。您的 M3U/CUE 文件包含媒体文件的正确路径至关重要，文件应位于您存储中的这些路径上。您可以在[此处](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox)阅读更多关于播放列表导入的信息。

## 播放列表详情屏幕

当您打开播放列表时，「播放列表详情屏幕」会出现。在此屏幕上，您会在右上角找到带有播放列表选项的「...」按钮，以及封面图片下方的三个按钮：「搜索」、「继续播放」、「全部播放」和「随机播放全部」。此外，还有一个「离线模式」复选框。

{{< cards cols="1">}}
  {{< card title="" subtitle="播放列表详情屏幕" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-detail-screen.webp" >}}
{{< /cards >}}

- **继续播放**：恢复此播放列表的播放位置。
- **搜索**：在当前播放列表中执行搜索。
- **全部播放**：将当前播放列表中的所有曲目添加到播放器队列。
- **随机播放全部**：类似于「全部播放」，但在将曲目添加到音频播放器队列前先对其随机排序。
- **离线模式**：将此播放列表中的所有曲目下载到本地文件。添加到播放列表的任何新项目都将自动下载。

## 播放列表屏幕中播放列表的更多操作

您可以通过点击播放列表标题附近的「...」按钮来访问播放列表的操作。以下是可用操作：

- **全部播放：** 将播放列表曲目添加到新的播放器队列。
- **下一首播放：** 将播放列表曲目添加到现有播放器队列顶部。
- **稍后播放：** 将播放列表曲目添加到现有播放器队列底部。
- **编辑图像：** 编辑播放列表的封面图片。
- **启用离线模式：** 为播放列表启用离线模式。在此场景中，现有和新曲目都将自动下载。详细了解离线模式请见[此处](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)。
- **导出歌曲列表：** 您可以将此播放列表导出为不同格式，如[此处](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)所述。
- **添加到存档：** 您可以将此播放列表存档，包括 m3u 文件、专辑封面和所有曲目，如[此处](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)所述。
- **添加歌曲：** 向当前播放列表添加更多歌曲。
- **重命名：** 重命名播放列表。
- **删除播放列表：** 从音乐库中删除播放列表。请注意，此操作无法撤销。

## 播放列表详情屏幕中播放列表的更多操作

您可以通过点击右上角的「...」按钮来访问播放列表的操作。以下是可用操作：

- **选择：** 激活曲目选择模式，适用于从播放列表中删除多首曲目或更改其顺序。
- **下一首播放：** 将播放列表曲目添加到现有播放器队列顶部。
- **稍后播放：** 将播放列表曲目添加到现有播放器队列底部。
- **添加歌曲：** 向播放列表添加新歌曲。
- **重新排列歌曲：** 使用拖放手动更改播放列表中歌曲的顺序。
- **重命名：** 重命名当前播放列表。
- **编辑图像：** 编辑当前播放列表的专辑封面。
- **导出歌曲列表：** 将此播放列表导出为不同格式。详细信息请见[此处](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。
- **添加到存档：** 压缩当前播放列表，包括所有曲目和 m3u 文件。详细信息请见[此处](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)。
- **排序：** 更改播放列表中曲目的顺序。排序选项包括「歌曲标题」、「歌曲编号」、「专辑」、「艺术家」、「专辑艺术家」、「流派」、「作曲家」、「评分」、「年份」、「每分钟节拍」、「时长」、「文件名」、「文件修改日期」、「文件创建日期」和「手动」。「手动」排序选项允许使用拖放手动重新排列歌曲。
- **搜索：** 在当前播放列表中搜索特定歌曲。
- **网格/列表：** 更改屏幕布局展示方式。
- **删除播放列表：** 从音乐库中删除播放列表。重要的是，此操作不会从您的存储中删除曲目，且无法撤销。

## 更改播放列表中的歌曲顺序

要更改播放列表中歌曲的顺序，请点击右上角的「...」按钮并选择「选择」以进入选择模式。使用每首曲目附近的重新排序控件和拖放手势上下移动它们。点击重新排序控件将把曲目移动到列表顶部。要退出选择模式并应用更改，请点击「完成」。

## 更改播放列表封面图片

要更改播放列表的封面图片，请点击右上角的「...」按钮并选择「编辑图像」。从可用来源选择图片并通过点击「完成」确认更改。

## 向播放列表添加歌曲

打开播放列表并点击右上角的「...」按钮，然后选择「添加歌曲」以打开对话框。选择您要添加的曲目并通过点击「完成」确认更改。

## 从播放列表中删除多首歌曲

打开播放列表，点击右上角的「...」按钮，然后选择「选择」以进入选择模式。选择您要删除的曲目并点击屏幕底部的「从播放列表中删除」按钮。通过点击「完成」确认更改。

## 曲目选项

播放列表中的每首曲目都有一个操作列表，通过点击「...」按钮访问。如果您看不到所有操作，请向下滚动查看。您可以从播放列表中删除曲目、下载它、编辑音频标签等。

{{< cards cols="1">}}
  {{< card title="" subtitle="播放列表中的曲目选项菜单" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-track-options.webp" >}}
{{< /cards >}}

- **下一首播放：** 将曲目添加到播放器队列顶部。
- **稍后播放：** 将曲目追加到播放器队列底部。
- **添加到播放列表：** 将曲目添加到播放列表。
- **添加到收藏夹：** 将曲目标记为收藏以便快速访问。
- **下载：** 使曲目可离线使用。它将出现在传输队列和音乐库「已下载音乐」部分的「本地文件」标签页中。
- **编辑音频标签：** 打开内置标签编辑器以更改曲目元数据。
- **在其中打开：** 导出曲目并在另一个应用程序中打开它。
- **在文件夹中显示：** 显示音频文件所在的文件夹。
- **在 Finder 中显示：** 对于从 Mac 导入的文件，此操作显示音频文件在 Mac 电脑上的所在文件夹。
- **从播放列表中删除：** 从播放列表中删除曲目。
- **从云服务删除：** 从播放列表和关联的云服务中删除曲目。请注意，此操作无法撤销。
- **从音乐库删除：** 从音乐库中删除曲目，文件在存储中保持不变。

## 辅助功能

我们的应用程序完全支持 VoiceOver 技术，确保每个组件都有精心设计的标签和描述。当 VoiceOver 激活时，应用程序将用户界面转换为文本模式，仅显示可访问且有用的元素，以提高导航速度和便利性。您也可以在设置 → 辅助功能 → 文本模式中激活文本模式。

使用 VoiceOver 调整播放列表中的曲目位置：

1. 打开播放列表并点击「更多」按钮。
2. 选择「更改歌曲顺序」。视图将切换到编辑模式。
3. 点击曲目标题附近的重新排序指示图标以使其获得焦点。
4. 快速双击重新排序指示图标。在第二次点击时，不要松开手指——按住直到听到表示单元格已准备好移动的声音。
5. 现在，您可以将单元格移动到新位置。

其他组件按预期工作，使用系统提供的 VoiceOver 模式。

## Обзор

Раздел Плейлисты предоставляет инструменты для организации треков в списки. Он включает вид контента с показом всех созданных плейлистов, кнопку «...» на панели навигации с различными действиями для плейлистов и панель инструментов навигации с кнопками «Поиск», «Воспроизвести всё» и «Перемешать». Кроме того, у каждого отдельного плейлиста есть кнопка «...» рядом с названием, предлагающая ряд действий, специфичных для этого плейлиста.

## Создание плейлиста

Чтобы создать новый плейлист, нажмите кнопку «+» или кнопку «...» в правом верхнем углу панели навигации, выберите «Новый плейлист» и присвойте имя плейлисту. После именования нажмите «Сохранить».

{{< cards cols="1">}}
  {{< card title="" subtitle="Создание нового плейлиста" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-add-new.webp" >}}
{{< /cards >}}

Откроется диалог «Добавить треки», где вы можете выбрать, какие треки добавить в новый плейлист. Треки разделены по типу источника, и у вас есть несколько вариантов:

- **Библиотека**: Треки из музыкальной библиотеки.
- **Файлы в этом приложении**: Аудиофайлы, хранящиеся в директории Документы приложения.
- **Файлы на этом iPhone/iPad/Mac**: Аудиофайлы в других папках на устройстве, вне приложения.
- **Подключения**: Онлайн-файлы, хранящиеся в подключённых облачных хранилищах.

По умолчанию вы можете добавить трек в плейлист только один раз. Чтобы разрешить повторяющиеся треки в плейлисте, включите эту функцию в Настройках приложения — Библиотека музыки — Плейлисты — Дубликаты в плейлисте — Включить.

## Импорт плейлиста

В Evermusic мы добавили функцию импорта файлов M3U, поэтому вам не нужно создавать плейлисты вручную.

{{< cards cols="1">}}
  {{< card title="" subtitle="Импорт плейлиста из источника файлов" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-import-from-files.webp" >}}
{{< /cards >}}

Сначала перейдите в раздел «Плейлисты». Затем нажмите кнопку «Ещё» в правом верхнем углу. В появившемся меню выберите опцию «Импортировать плейлист».

На следующем экране выберите расположение файла. Поддерживаемые варианты включают:

- Подключённое облачное хранилище
- Файлы в приложении
- Файлы на устройстве

Выберем подключённое облачное хранилище и откроем папку с файлом плейлиста. Поддерживаемые расширения файлов плейлиста включают M3U, M3U8 и CUE. Выберите файл плейлиста и нажмите «Готово» для подтверждения выбора.

Приложение проанализирует файл плейлиста, создаст список треков и найдёт эти файлы в хранилище для составления финального плейлиста, который будет импортирован в музыкальную библиотеку. Важно, чтобы ваш файл M3U/CUE содержал правильные пути к медиафайлам, и файлы должны находиться по этим путям в вашем хранилище. Подробнее об импорте плейлиста можно прочитать [здесь](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox).

## Экран деталей плейлиста

При открытии плейлиста появляется «Экран деталей плейлиста». На этом экране вы найдёте кнопку «...» в правом верхнем углу с опциями плейлиста и три кнопки под изображением обложки: «Поиск», «Продолжить воспроизведение», «Воспроизвести всё» и «Перемешать». Кроме того, есть флажок «Офлайн-режим».

{{< cards cols="1">}}
  {{< card title="" subtitle="Экран деталей плейлиста" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-detail-screen.webp" >}}
{{< /cards >}}

- **Продолжить воспроизведение**: Восстановить позицию воспроизведения для этого плейлиста.
- **Поиск**: Выполнить поиск в текущем плейлисте.
- **Воспроизвести всё**: Добавить все треки из текущего плейлиста в очередь плеера.
- **Перемешать**: Аналогично «Воспроизвести всё», но перемешивает треки перед добавлением в очередь аудиоплеера.
- **Офлайн-режим**: Загрузить все треки из этого плейлиста в локальные файлы. Любые новые элементы, добавленные в плейлист, будут автоматически загружены.

## Дополнительные действия для плейлиста на экране плейлистов

Вы можете получить доступ к действиям для плейлиста, нажав кнопку «...» рядом с названием плейлиста. Вот доступные действия:

- **Воспроизвести всё:** Добавляет треки плейлиста в новую очередь плеера.
- **Следующий в очереди:** Добавляет треки плейлиста в начало существующей очереди плеера.
- **Воспроизвести позже:** Добавляет треки плейлиста в конец существующей очереди плеера.
- **Изменить обложку:** Изменить изображение обложки плейлиста.
- **Включить офлайн режим:** Включает офлайн-режим для плейлиста. В этом случае как существующие, так и новые треки будут загружаться автоматически. Подробнее об офлайн-режиме можно прочитать [здесь](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).
- **Экспортировать список треков:** Вы можете экспортировать этот плейлист в различные форматы, как описано [здесь](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).
- **Добавить в архив:** Вы можете заархивировать этот плейлист, включая файл m3u, обложку альбома и все треки, как описано [здесь](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to).
- **Добавить треки:** Добавить больше треков в текущий плейлист.
- **Переименовать:** Переименовать плейлист.
- **Удалить плейлист:** Удалить плейлист из музыкальной библиотеки. Обратите внимание, что это действие нельзя отменить.

## Дополнительные действия для плейлиста на экране деталей плейлиста

Вы можете получить доступ к действиям для плейлиста, нажав кнопку «...» в правом верхнем углу. Вот доступные действия:

- **Выбрать:** Активирует режим выбора треков, полезный для удаления нескольких треков из плейлиста или изменения их порядка.
- **Следующий в очереди:** Добавляет треки плейлиста в начало существующей очереди плеера.
- **Воспроизвести позже:** Добавляет треки плейлиста в конец существующей очереди плеера.
- **Добавить треки:** Добавить новые треки в плейлист.
- **Изменить порядок треков:** Вручную изменить порядок треков в плейлисте с помощью перетаскивания.
- **Переименовать:** Переименовать текущий плейлист.
- **Изменить обложку:** Изменить обложку альбома для текущего плейлиста.
- **Экспортировать список треков:** Экспортировать этот плейлист в различные форматы. Подробнее можно прочитать [здесь](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).
- **Добавить в архив:** Сжать текущий плейлист, включая все треки и файл m3u. Подробнее можно прочитать [здесь](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to).
- **Сортировать:** Изменить порядок треков в плейлисте. Варианты сортировки включают «Название трека», «Номер трека», «Альбом», «Исполнитель», «Исполнитель альбома», «Жанр», «Композитор», «Рейтинг», «Год», «Удары в минуту», «Длительность», «Имя файла», «Дата изменения файла», «Дата создания файла» и «Вручную». Вариант сортировки «Вручную» позволяет вручную менять порядок треков с помощью перетаскивания.
- **Поиск:** Найти конкретный трек в текущем плейлисте.
- **Сетка/Список:** Изменить представление макета экрана.
- **Удалить плейлист:** Удалить плейлист из музыкальной библиотеки. Важно: это действие не удаляет треки из хранилища и не может быть отменено.

## Изменение порядка треков в плейлисте

Чтобы изменить порядок треков в плейлисте, нажмите кнопку «...» в правом верхнем углу и выберите «Выбрать», чтобы войти в режим выбора. Используйте элемент управления порядком и жесты перетаскивания рядом с каждым треком для перемещения вверх или вниз. Нажатие на элемент управления порядком переместит трек в начало списка. Чтобы выйти из режима выбора и применить изменения, нажмите «Готово».

## Изменение обложки плейлиста

Чтобы изменить изображение обложки плейлиста, нажмите кнопку «...» в правом верхнем углу и выберите «Изменить обложку». Выберите изображение из доступных источников и подтвердите изменения, нажав «Готово».

## Добавление треков в плейлист

Откройте плейлист и нажмите кнопку «...» в правом верхнем углу, затем выберите «Добавить треки», чтобы открыть диалог. Выберите треки, которые хотите добавить, и подтвердите изменения, нажав «Готово».

## Удаление нескольких треков из плейлиста

Откройте плейлист, нажмите кнопку «...» в правом верхнем углу и выберите «Выбрать», чтобы войти в режим выбора. Выберите треки, которые хотите удалить, и нажмите кнопку «Удалить из плейлиста» в нижней части экрана. Подтвердите изменения, нажав «Готово».

## Параметры трека

У каждого трека в плейлисте есть список действий, доступный нажатием кнопки «...». Если не видите все действия, прокрутите вниз для их просмотра. Вы можете удалить трек из плейлиста, загрузить его, редактировать аудиотеги и многое другое.

{{< cards cols="1">}}
  {{< card title="" subtitle="Меню параметров трека в плейлисте" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-track-options.webp" >}}
{{< /cards >}}

- **Следующий в очереди:** Добавляет трек в начало очереди плеера.
- **Воспроизвести позже:** Добавляет трек в конец очереди плеера.
- **Добавить в плейлист:** Добавляет трек в плейлист.
- **Добавить в избранное:** Отмечает трек как избранный для быстрого доступа.
- **Скачать:** Делает трек доступным офлайн. Он появится в очереди передачи и во вкладке «Локальные файлы» в разделе «Загруженная музыка» музыкальной библиотеки.
- **Редактировать теги:** Открывает встроенный редактор тегов для изменения метаданных трека.
- **Открыть в:** Экспортирует трек и открывает его в другом приложении.
- **Показать в папке:** Показывает папку, где находится аудиофайл.
- **Показать в Finder:** Для файлов, импортированных с Mac, это действие показывает папку, где аудиофайл расположен на компьютере Mac.
- **Удалить из плейлиста:** Удаляет трек из плейлиста.
- **Удалить из облачного сервиса:** Удаляет трек из плейлиста и связанного облачного сервиса. Обратите внимание, что это действие нельзя отменить.
- **Удалить из музыкальной библиотеки:** Удаляет трек из музыкальной библиотеки, оставляя файл в хранилище нетронутым.

## Доступность

Наше приложение полностью доступно с технологией VoiceOver, обеспечивая каждому компоненту хорошо продуманную метку и описание. Когда VoiceOver активен, приложение переводит пользовательский интерфейс в текстовый режим, отображая только доступные и полезные элементы для повышения скорости и удобства навигации. Вы также можете активировать текстовый режим в Настройки > Доступность > Текстовый режим.

Для настройки позиции трека в плейлисте с VoiceOver:

1. Откройте плейлист и нажмите кнопку «Ещё».
2. Выберите «Изменить порядок треков». Вид переключится в режим редактирования.
3. Нажмите на иконку индикатора порядка рядом с названием трека, чтобы передать ей фокус.
4. Быстро дважды нажмите на иконку индикатора порядка. При втором нажатии не отпускайте палец — удерживайте, пока не услышите звук, сигнализирующий о готовности ячейки к перемещению.
5. Теперь вы можете переместить ячейку в новую позицию.

Другие компоненты работают как ожидается, используя стандартные паттерны VoiceOver.

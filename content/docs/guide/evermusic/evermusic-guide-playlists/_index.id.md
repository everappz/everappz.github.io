## Ikhtisar

Bagian Daftar Putar menyediakan alat untuk mengorganisir trek Anda ke dalam daftar. Ini mencakup tampilan konten yang menampilkan semua daftar putar yang telah Anda buat, tombol "..." di bilah navigasi yang menawarkan berbagai tindakan terkait daftar putar, dan toolbar navigasi dengan tombol "Cari", "Putar semua", dan "Acak semua". Selain itu, setiap daftar putar memiliki tombol "..." di dekat judul daftar putar, yang menawarkan berbagai tindakan khusus untuk daftar putar tersebut.

## Membuat Daftar Putar

Untuk membuat daftar putar baru, ketuk tombol "+" atau tombol "..." di sudut kanan atas bilah navigasi, pilih "Daftar putar baru" dan berikan nama pada daftar putar Anda. Setelah memberi nama, ketuk "Simpan".

{{< cards cols="1">}}
  {{< card title="" subtitle="Buat Daftar Putar Baru" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-add-new.webp" >}}
{{< /cards >}}

Ini memunculkan dialog "Tambahkan lagu", di mana Anda dapat memilih trek mana yang akan ditambahkan ke daftar putar baru. Trek dikategorikan berdasarkan jenis sumber, dan Anda memiliki beberapa opsi:

- **Perpustakaan**: Trek dari perpustakaan musik Anda.
- **File di Aplikasi Ini**: File audio yang tersimpan di direktori Documents aplikasi.
- **File di iPhone/iPad/Mac Ini**: File audio yang terletak di folder lain di perangkat Anda, di luar aplikasi.
- **Koneksi**: File online yang tersimpan di layanan penyimpanan cloud yang terhubung.

Secara default, Anda hanya dapat menambahkan trek ke daftar putar satu kali. Untuk mengizinkan lagu duplikat dalam daftar putar, aktifkan fitur ini di Pengaturan aplikasi - Perpustakaan musik - Daftar Putar - Duplikat dalam daftar putar - Aktifkan.

## Mengimpor Daftar Putar

Di Evermusic, kami telah menambahkan fungsionalitas impor file M3U, sehingga Anda tidak perlu membuat daftar putar secara manual.

{{< cards cols="1">}}
  {{< card title="" subtitle="Impor Daftar Putar dari Sumber File" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-import-from-files.webp" >}}
{{< /cards >}}

Pertama, buka bagian 'Daftar Putar'. Kemudian, ketuk tombol 'Lebih' di sudut kanan atas. Dari menu yang muncul, pilih opsi 'Impor Daftar Putar'.

Di layar berikutnya, pilih lokasi file. Opsi yang didukung meliputi:

- Penyimpanan cloud yang terhubung
- File dalam aplikasi
- File di perangkat Anda

Mari pilih penyimpanan cloud yang terhubung dan buka folder yang berisi file daftar putar. Ekstensi file daftar putar yang didukung meliputi M3U, M3U8, dan CUE. Pilih file daftar putar dan ketuk 'Selesai' untuk mengonfirmasi pilihan Anda.

Aplikasi akan mengurai file daftar putar, membuat daftar trek, dan mencari file tersebut di penyimpanan untuk menyusun daftar putar akhir, yang akan diimpor ke perpustakaan musik. Penting bahwa file M3U/CUE Anda berisi jalur yang benar untuk file media, dan file harus berada di jalur tersebut di penyimpanan Anda. Anda dapat membaca lebih lanjut tentang impor daftar putar [di sini](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox).

## Layar Detail Daftar Putar

Saat Anda membuka daftar putar, "Layar detail daftar putar" muncul. Di layar ini, Anda akan menemukan tombol "..." di sudut kanan atas dengan opsi daftar putar dan tiga tombol di bawah gambar sampul: "Cari", "Lanjutkan pemutaran", "Putar semua", dan "Acak semua". Selain itu, ada kotak centang "Mode offline".

{{< cards cols="1">}}
  {{< card title="" subtitle="Layar Detail Daftar Putar" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-detail-screen.webp" >}}
{{< /cards >}}

- **Lanjutkan pemutaran**: Pulihkan posisi pemutaran untuk daftar putar ini.
- **Cari**: Lakukan pencarian dalam daftar putar saat ini.
- **Putar semua**: Tambahkan semua trek dari daftar putar saat ini ke antrean pemutar.
- **Acak Semua**: Mirip dengan "Putar semua", tetapi mengacak trek sebelum menambahkannya ke antrean pemutar audio.
- **Mode offline**: Unduh semua trek dari daftar putar ini ke file lokal. Item baru yang ditambahkan ke daftar putar akan diunduh secara otomatis.

## Lebih Banyak Tindakan untuk Daftar Putar di Layar Daftar Putar

Anda dapat mengakses tindakan untuk daftar putar dengan mengetuk tombol "..." di dekat judul daftar putar. Berikut adalah tindakan yang tersedia:

- **Putar semua:** Menambahkan trek daftar putar ke antrean pemutar baru.
- **Putar berikutnya:** Menambahkan trek daftar putar ke bagian atas antrean pemutar yang ada.
- **Putar nanti:** Menambahkan trek daftar putar ke bagian bawah antrean pemutar yang ada.
- **Edit gambar:** Edit gambar sampul daftar putar.
- **Aktifkan mode offline:** Mengaktifkan mode offline untuk daftar putar. Dalam skenario ini, trek yang ada dan baru akan diunduh secara otomatis. Baca lebih lanjut tentang mode offline [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).
- **Ekspor daftar lagu:** Anda dapat mengekspor daftar putar ini ke format yang berbeda seperti yang dijelaskan [di sini](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).
- **Tambahkan ke arsip:** Anda dapat mengarsipkan daftar putar ini termasuk file m3u, sampul album, dan semua trek seperti yang dijelaskan [di sini](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to).
- **Tambahkan lagu:** Tambahkan lebih banyak lagu ke daftar putar saat ini.
- **Ubah nama:** Ubah nama daftar putar.
- **Hapus daftar putar:** Hapus daftar putar dari Perpustakaan musik. Harap diperhatikan bahwa tindakan ini tidak dapat dibatalkan.

## Lebih Banyak Tindakan untuk Daftar Putar di Layar Detail Daftar Putar

Anda dapat mengakses tindakan untuk daftar putar dengan mengetuk tombol "..." di sudut kanan atas. Berikut adalah tindakan yang tersedia:

- **Pilih:** Mengaktifkan mode pemilihan trek, berguna untuk menghapus beberapa trek dari daftar putar atau mengubah urutannya.
- **Putar berikutnya:** Menambahkan trek daftar putar ke bagian atas antrean pemutar yang ada.
- **Putar nanti:** Menambahkan trek daftar putar ke bagian bawah antrean pemutar yang ada.
- **Tambahkan lagu:** Tambahkan lagu baru ke daftar putar.
- **Susun ulang lagu:** Ubah urutan lagu dalam daftar putar secara manual menggunakan seret dan lepas.
- **Ubah nama:** Ubah nama daftar putar saat ini.
- **Edit gambar:** Edit sampul album untuk daftar putar saat ini.
- **Ekspor daftar lagu:** Ekspor daftar putar ini ke format yang berbeda. Anda dapat membaca lebih lanjut [di sini](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).
- **Tambahkan ke arsip:** Zip daftar putar saat ini termasuk semua trek dan file m3u. Anda dapat membaca lebih lanjut [di sini](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to).
- **Urutkan:** Ubah urutan trek dalam daftar putar. Opsi pengurutan meliputi "Judul lagu", "Nomor lagu", "Album", "Artis", "Artis album", "Genre", "Komposer", "Rating", "Tahun", "Ketukan per menit", "Durasi", "Nama file", "Tanggal modifikasi file", "Tanggal pembuatan file", dan "Manual". Opsi sortir "Manual" memungkinkan pengurutan ulang lagu secara manual menggunakan seret dan lepas.
- **Cari:** Cari lagu tertentu dalam daftar putar saat ini.
- **Kisi/Daftar:** Ubah presentasi tata letak layar.
- **Hapus daftar putar:** Hapus daftar putar dari Perpustakaan musik. Yang penting, tindakan ini tidak menghapus trek dari penyimpanan Anda, dan tidak dapat dibatalkan.

## Mengubah Urutan Lagu dalam Daftar Putar

Untuk mengubah urutan lagu dalam daftar putar, ketuk tombol "..." di sudut kanan atas dan pilih "Pilih" untuk masuk ke mode pemilihan. Gunakan kontrol pengurutan ulang dan gerakan seret-dan-lepas di dekat setiap trek untuk memindahkannya ke atas atau ke bawah. Mengetuk kontrol pengurutan ulang akan memindahkan trek ke bagian atas daftar. Untuk keluar dari mode pemilihan dan menerapkan perubahan, ketuk "Selesai".

## Mengubah Gambar Sampul Daftar Putar

Untuk mengubah gambar sampul daftar putar, ketuk tombol "..." di sudut kanan atas dan pilih "Edit gambar". Pilih gambar dari sumber yang tersedia dan konfirmasikan perubahan dengan mengetuk "Selesai".

## Menambahkan Lagu ke Daftar Putar

Buka daftar putar dan ketuk tombol "..." di sudut kanan atas, kemudian pilih "Tambahkan lagu" untuk membuka dialog. Pilih trek yang ingin Anda tambahkan dan konfirmasikan perubahan dengan mengetuk "Selesai".

## Menghapus Beberapa Lagu dari Daftar Putar

Buka daftar putar, ketuk tombol "..." di sudut kanan atas, dan pilih "Pilih" untuk masuk ke mode pemilihan. Pilih trek yang ingin Anda hapus dan ketuk tombol "Hapus dari daftar putar" di bagian bawah layar. Konfirmasikan perubahan dengan mengetuk "Selesai".

## Opsi Trek

Setiap trek dalam daftar putar memiliki daftar tindakan, dapat diakses dengan mengetuk tombol "...". Jika Anda tidak dapat melihat semua tindakan, gulir ke bawah untuk melihatnya. Anda dapat menghapus trek dari daftar putar, mengunduhnya, mengedit tag audio, dan lainnya.

{{< cards cols="1">}}
  {{< card title="" subtitle="Menu Opsi Trek dalam Daftar Putar" image="/docs/guide/evermusic/evermusic-guide-playlists/img/playlists-track-options.webp" >}}
{{< /cards >}}

- **Putar berikutnya:** Menambahkan trek ke bagian atas antrean pemutar.
- **Putar nanti:** Menambahkan trek ke bagian bawah antrean pemutar.
- **Tambahkan ke daftar putar:** Menambahkan trek ke daftar putar.
- **Tambahkan ke favorit:** Menandai trek sebagai favorit untuk akses cepat.
- **Unduh:** Membuat trek tersedia secara offline. Akan muncul di antrean transfer dan tab "File Lokal" di bagian "Musik yang diunduh" dari Perpustakaan musik.
- **Mengedit tag audio:** Membuka editor tag bawaan untuk mengubah metadata trek.
- **Buka di:** Mengekspor trek dan membukanya di aplikasi lain.
- **Tampilkan di folder:** Mengungkapkan folder tempat file audio berada.
- **Tampilkan di Finder:** Untuk file yang diimpor dari Mac, tindakan ini mengungkapkan folder tempat file audio berada di komputer Mac Anda.
- **Hapus dari daftar putar:** Menghapus trek dari daftar putar.
- **Hapus dari layanan cloud:** Menghapus trek dari daftar putar dan layanan cloud terkait. Harap diperhatikan bahwa tindakan ini tidak dapat dibatalkan.
- **Hapus dari perpustakaan musik:** Menghapus trek dari perpustakaan musik, membiarkan file di penyimpanan tidak tersentuh.

## Aksesibilitas

Aplikasi kami sepenuhnya dapat diakses dengan teknologi VoiceOver, memastikan setiap komponen memiliki label dan deskripsi yang dirancang dengan baik. Saat VoiceOver aktif, aplikasi menerjemahkan antarmuka pengguna ke mode teks, hanya menampilkan elemen yang dapat diakses dan berguna untuk meningkatkan kecepatan navigasi dan kenyamanan. Anda juga dapat mengaktifkan mode teks di Pengaturan > Aksesibilitas > Mode Teks.

Untuk menyesuaikan posisi trek dalam daftar putar dengan VoiceOver:

1. Buka daftar putar dan ketuk tombol "Lebih".
2. Pilih "Ubah Urutan Lagu." Tampilan akan beralih ke mode pengeditan.
3. Ketuk ikon indikator pengurutan di dekat judul trek untuk memberikannya fokus.
4. Ketuk dua kali ikon indikator pengurutan dengan cepat. Pada ketukan kedua, jangan lepaskan jari Anda — tahan sampai Anda mendengar suara yang menandakan sel siap dipindahkan.
5. Sekarang, Anda dapat memindahkan sel ke posisi baru.

Komponen lain berfungsi seperti yang diharapkan, menggunakan pola VoiceOver yang disediakan sistem.

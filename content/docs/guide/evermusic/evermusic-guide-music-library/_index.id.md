Mengelola perpustakaan musik adalah hal yang mudah dengan Evermusic, di mana Anda dapat dengan mudah mengatur semua trek Anda. Anda memiliki dua pilihan untuk membangun perpustakaan musik: penambahan manual atau sinkronisasi otomatis.

## Penambahan Manual

Untuk menambahkan trek secara manual, ketuk item menu "Tambahkan musik" dan pilih folder/file dari layanan penyimpanan cloud yang terhubung atau file yang terletak di perangkat Anda. Ketika Anda menambahkan trek ke perpustakaan, hanya tautan ke trek tersebut yang dibuat, menjaga file aktual di lokasi aslinya untuk menghemat ruang disk yang berharga. Jika Anda ingin membuat trek tersedia offline, Anda dapat menggunakan tindakan unduh dari menu opsi atau mengaktifkan mode offline untuk playlist dan koleksi trek.

## Akses Cepat

Di bagian atas menu perpustakaan musik, Anda akan menemukan bagian akses cepat yang menyediakan tautan mudah ke favorit, terbaru, dan bookmark audio Anda.

### Favorit

Anda dapat menandai lagu sebagai favorit di layar pemutar audio atau menggunakan menu opsi.

### Terbaru

Bagian ini menampilkan semua trek yang baru diputar.

### Tandai Halaman

Anda dapat membuat bookmark audio saat lagu diputar dan mengelolanya di layar ini. Instruksi terperinci tentang cara bekerja dengan bookmark audio dapat ditemukan [di sini](/docs/guide/evermusic/evermusic-guide-music-library).

## Lanjutkan Pemutaran

Pulihkan antrean pemutar audio dari posisi tersimpan terakhir jika fitur ini diaktifkan di pengaturan aplikasi.

## Lokasi

Semua trek di perpustakaan Anda dikelompokkan dengan cermat berdasarkan jenis sumber dan tag musik.

### Musik online

Bagian ini menampilkan musik dari layanan penyimpanan cloud Anda.

### File di aplikasi ini

Di sini Anda dapat menemukan musik yang tersedia untuk pemutaran offline, bersumber dari file lokal Anda. Ini mencakup file di direktori dokumen aplikasi.

### File di iPhone/iPad/Mac ini

Kategori ini mencakup musik yang diimpor ke aplikasi dari perangkat Anda yang ditambahkan melalui dialog sistem 'Buka file…'.

### Musik iTunes

Bagian ini menampilkan musik yang diimpor ke perpustakaan iPod perangkat Anda. Harap perhatikan bahwa trek dari Apple Music tidak tersedia di sini.

## Kategori

Ketika Anda menambahkan trek ke perpustakaan musik, aplikasi secara otomatis membaca tag audio mereka dan mengaturnya ke dalam kategori seperti Lagu, Lagu Belum Diputar, Album, Artis Album, Artis, Genre, dan Komposer.

## Toolbar Atas

Terletak tepat di bawah bilah navigasi, toolbar atas menawarkan beberapa tindakan yang nyaman: "Cari," "Putar semua," "Acak semua," dan "Lanjutkan pemutaran." Anda dapat menampilkan atau menyembunyikan toolbar ini dengan gerakan gesek ke bawah yang sederhana.

## Menu Opsi

Setiap lagu di perpustakaan musik Anda memiliki menu dengan lebih banyak tindakan, diakses dengan mengetuk tombol tiga titik di dekat judul lagu. Tindakan ini bervariasi tergantung pada apakah itu satu lagu atau bagian dari koleksi.

### Untuk Lagu Individual

- **Putar berikutnya**: Menambahkan lagu ke bagian atas antrean pemutar.
- **Putar nanti**: Menambahkan lagu ke bagian bawah antrean pemutar.
- **Tambahkan ke daftar putar**: Menambahkan lagu ke daftar putar.
- **Tambahkan ke favorit**: Menandai lagu sebagai favorit untuk akses cepat.
- **Unduh**: Menyimpan lagu ke file lokal. Lagu akan muncul di tab "File Lokal" dan bagian "Musik offline".
- **Mengedit tag audio**: Membuka editor tag audio bawaan untuk memperbaiki metadata yang hilang; perhatikan bahwa ini akan mengubah lagu di penyimpanan Anda.
- **Tampilkan di folder**: Mengungkapkan folder di mana file audio disimpan.
- **Tampilkan di Finder**: Untuk file yang diimpor dari Mac Anda, tindakan ini mengungkapkan folder di mana file audio berada di Mac Anda.
- **Buka dengan**: Mengekspor file audio ke aplikasi lain.
- **Hapus dari layanan cloud**: Menghapus file dari perpustakaan musik dan penyimpanan cloud, dan harap perhatikan bahwa tindakan ini tidak dapat dibatalkan.
- **Hapus dari perpustakaan musik**: Menghapus lagu dari perpustakaan musik Anda, tetapi file tetap ada di penyimpanan. Jika sinkronisasi otomatis diaktifkan dan file ada di penyimpanan jarak jauh, lagu akan muncul kembali di perpustakaan Anda setelah operasi sinkronisasi.

### Untuk Koleksi Lagu

Untuk koleksi lagu seperti Album, Artis, Genre, atau Komposer, menu opsi menyertakan tindakan berikut.

- **Putar semua**: Mengganti antrean pemutar dengan lagu dari koleksi yang dipilih.
- **Putar berikutnya**: Menambahkan lagu dari koleksi ini ke bagian atas antrean pemutar.
- **Putar nanti**: Menambahkan lagu dari koleksi ini ke bagian bawah antrean pemutar.
- **Tambahkan ke daftar putar**: Menyertakan lagu dari koleksi ini dalam daftar putar, dengan opsi untuk membuat daftar putar baru.
- **Mengaktifkan mode offline**: Mengunduh lagu dari koleksi ini ke file lokal. Lagu yang diunduh ini akan muncul di tab "File Lokal" dan bagian "Musik offline" dari Perpustakaan Musik Anda. Jika item baru ditambahkan ke koleksi di server, item tersebut akan diunduh secara otomatis ke "File Lokal."
- **Edit gambar**: Memungkinkan Anda mengubah sampul album untuk koleksi lagu.
- **Hapus dari perpustakaan musik**: Menghapus koleksi lagu dari perpustakaan musik Anda. Namun, tindakan ini tidak menghapus file aktual dari penyimpanan. Jika sinkronisasi otomatis diaktifkan dan file ada di penyimpanan jarak jauh, file tersebut akan muncul kembali di perpustakaan Anda setelah operasi sinkronisasi.

## Pengelompokan Tag

Kategori ini membantu Anda mengatur trek berdasarkan tag musik: Lagu, Album, Artis Album, Artis, Genre, Komposer. Ketika Anda menambahkan trek ke perpustakaan musik, aplikasi membaca metadata trek dan mengelompokkannya berdasarkan kategori ini. Jika Anda tidak melihat semua album Anda, pastikan aplikasi telah memindai semua trek. Anda dapat memeriksa kemajuan pemindaian di Pengaturan -> Perpustakaan Musik -> Pembacaan Metadata -> Jumlah File yang Diproses di Perpustakaan Musik. Untuk file lokal, Anda juga dapat memindai ulang folder offline di Pengaturan -> Perpustakaan Musik -> Sinkronisasi Folder Offline -> Mulai Pemindaian Folder Lokal. Setelah pembaca metadata menyelesaikan semua operasi, Anda dapat melihat kategori berikut di perpustakaan musik Anda.

- **Lagu**: Semua lagu dikelompokkan berdasarkan tag TRACK_TITLE. Anda dapat memeriksa urutan pengurutan menggunakan menu Lebih Banyak Tindakan di sudut kanan atas.
- **Lagu belum diputar**: Semua lagu dikelompokkan berdasarkan tag TRACK_TITLE dan belum diputar.
- **Album**: Lagu dikelompokkan berdasarkan tag ALBUM_NAME, sedangkan tag artis, artis album, dan komposer dilewati. Jika Anda memiliki beberapa album dengan nama yang sama tetapi artis yang berbeda, pertimbangkan untuk menggunakan pengurutan Album Eksklusif yang dijelaskan di bawah.
- **Artis Album**: Lagu dikelompokkan hanya berdasarkan ALBUM_ARTIST_TAG.
- **Artis**: Lagu dikelompokkan hanya berdasarkan ARTIST_TAG.
- **Genre**: Lagu dikelompokkan berdasarkan tag GENRE.
- **Komposer**: Lagu dikelompokkan berdasarkan tag COMPOSER.

- **Lagu**: Menampilkan semua lagu di mana Artis/Artis Album/Komposer ini ditetapkan dalam tag audio.

#### Pembacaan Metadata

Ketika Anda menambahkan trek ke perpustakaan, pembaca metadata mulai bekerja. Proses latar belakang ini membaca semua metadata dari trek Anda dan mengaturnya berdasarkan Artis, Album, Genre, dan Komposer. Anda memiliki fleksibilitas untuk menyesuaikan kecepatan pembacaan metadata untuk memuat data lebih cepat, tetapi ketahuilah bahwa ini dapat menggunakan lebih banyak energi. Anda juga dapat menonaktifkan pembaca metadata dan menampilkan nama file alih-alih informasi tag.

Pentingnya, pembaca metadata hanya memperbarui metadata di perpustakaan musik Anda dan tidak mengubah file yang disimpan di akun cloud atau penyimpanan lokal Anda. Jika Anda ingin mengedit metadata untuk file audio, Anda dapat melakukannya menggunakan editor tag bawaan, yang dapat Anda aktifkan dari tindakan yang sesuai di menu opsi.

#### Mode yang Tersedia untuk Pembaca Metadata

- **Dinonaktifkan**: Pembaca metadata akan dinonaktifkan, dan nama file akan ditampilkan alih-alih data dari tag audio.
- **Lagu Saat Ini**: Aplikasi hanya akan membaca metadata untuk lagu yang sedang diputar. Gunakan opsi ini jika Anda memiliki koneksi jaringan yang lambat untuk mencegah pembaca metadata mengirim banyak permintaan ke server cloud, yang dapat menyebabkan gangguan pemutaran.
- **Antrean Pemutaran**: Aplikasi akan membaca metadata untuk semua lagu di antrean pemutar audio.
- **Perpustakaan**: Aplikasi akan membaca metadata untuk semua lagu di perpustakaan musik.

Ketika sakelar pembacaan Metadata di latar belakang aktif, pembaca metadata bekerja dalam mode latar belakang. Namun, harap perhatikan bahwa jika aplikasi menggunakan banyak energi selama pemutaran audio, sistem operasi iOS dapat menangguhkannya.

Jadi, jika Anda memiliki koleksi musik yang besar, disarankan untuk menggunakan versi desktop aplikasi untuk sinkronisasi metadata. Anda kemudian dapat menggunakan fitur backup dan restore data untuk mentransfer perpustakaan musik yang disinkronkan dari desktop, yang tersedia di pengaturan aplikasi.

Ketika Normalisasi pengkodean metadata diaktifkan, aplikasi akan secara otomatis menormalkan pengkodean metadata untuk semua lagu di perpustakaan musik. Ini memperbaiki masalah di mana pengkodean tag audio rusak (seperti setelah mengedit file di PC Windows) dan mencegah informasi yang salah ditampilkan saat trek sedang diputar atau ditambahkan ke perpustakaan.

Tindakan Muat ulang metadata akan menandai semua file di perpustakaan musik Anda sebagai memiliki metadata yang hilang, memicu pembaca metadata untuk menyegarkan metadata untuk setiap file di perpustakaan musik Anda.

Ketuk tindakan Mulai Pembacaan Metadata untuk memulai pembaca metadata. Kemajuan operasi akan ditampilkan di bawah.

#### Sinkronisasi Online

Sinkronisasi musik online otomatis memungkinkan Anda menambahkan trek dari layanan cloud yang terhubung ke perpustakaan musik secara otomatis. Untuk mengaktifkan fitur ini, buka pengaturan perpustakaan musik dan pilih folder sinkronisasi.

Dengan opsi ini diaktifkan, aplikasi memindai semua folder yang dipilih, mengidentifikasi file audio yang didukung, dan mengintegrasikannya dengan mulus ke perpustakaan Anda. Anda dapat memulai atau menghentikan sinkronisasi dengan mengetuk tindakan menu yang sesuai.

Sinkronisasi musik online hanya beroperasi ketika aplikasi berada di latar depan, yang berarti sinkronisasi mungkin memakan beberapa waktu. Untuk mempercepat prosesnya, biarkan aplikasi Anda terbuka, hubungkan ke sumber daya, dan aktifkan opsi Layar -> Selalu aktif di pengaturan aplikasi.

Alternatifnya, Anda dapat melakukan sinkronisasi musik online di versi desktop aplikasi dan mentransfer perpustakaan musik ke versi iOS menggunakan fitur backup restore data.

Anda juga dapat mengatur seberapa sering Anda ingin menyinkronkan perpustakaan musik online Anda. Jika Anda mengaturnya ke "Segera," sinkronisasi online akan dimulai setiap kali Anda membuka aplikasi.

#### Sinkronisasi Offline

Di sini Anda dapat mengkonfigurasi sinkronisasi musik offline.

##### Folder Offline yang Disinkronkan

Ketika Anda membuat folder cloud tersedia offline (melalui menu Lebih Banyak Tindakan), folder tersebut akan muncul di bagian File Lokal → Folder Offline. Aplikasi akan mengunduh isinya ke perangkat Anda. Jika Anda melakukan perubahan pada folder di cloud — seperti menambahkan, menghapus, atau memperbarui file — aplikasi akan mendeteksi perubahan tersebut dan memperbarui salinan lokal secara otomatis.

Di layar ini, Anda dapat memulai sinkronisasi folder offline secara manual, menampilkan folder offline di folder yang memuatnya, dan menonaktifkan mode offline untuk folder ini. Menonaktifkan mode offline akan menghapus semua salinan lokal file dari perangkat Anda.

##### Interval Waktu

Anda dapat mengatur interval waktu untuk seberapa sering aplikasi harus memeriksa folder offline untuk modifikasi.

##### Mulai Pemindaian Folder Lokal

Opsi ini memindai semua folder lokal yang terletak di direktori Documents aplikasi untuk menemukan file audio yang didukung. Semua file lokal ini ditambahkan dengan mulus ke perpustakaan musik Anda. File lokal yang terletak di perangkat Anda tetapi di luar aplikasi ini harus ditambahkan ke perpustakaan musik secara manual, karena aplikasi tidak memiliki akses ke file di luar direktori Documents aplikasi karena batasan keamanan iOS/MacOS.

##### Penting

Disarankan untuk secara berkala memulai sinkronisasi musik offline untuk menjaga perpustakaan musik Anda diperbarui dengan file lokal Anda.

#### Personalisasi

Di bagian ini, Anda dapat mengkonfigurasi gaya layar perpustakaan musik sesuai preferensi Anda. Tiga opsi tersedia: Menu polos, Menu berkelompok, Menu bertab.

#### Sampul Album

Di sini, Anda dapat memilih kualitas sampul album yang disimpan di perangkat Anda dan mengelola sampul album yang di-cache. Secara default, aplikasi akan memeriksa sampul album yang tertanam di trek Anda dan menampilkannya jika tersedia. Jika tidak ada karya album yang tertanam dan opsi 'Cari di folder' diaktifkan, aplikasi akan memeriksa folder yang memuatnya untuk gambar JPEG atau PNG dan menggunakannya sebagai karya album untuk semua trek di folder tersebut.

#### Daftar Putar

Anda dapat mengaktifkan opsi untuk menambahkan lagu yang sama ke daftar putar dua kali. Secara default, opsi ini dinonaktifkan.

#### Terbaru

Anda dapat mengelola daftar lagu yang baru diputar.

- **Hapus Daftar**: Anda dapat menghapus seluruh daftar lagu yang baru diputar.
- **Ubah Ukuran Daftar**: Anda dapat mengatur jumlah item yang harus muncul di daftar.
- **Ekspor Daftar Lagu**: Gunakan tindakan ini untuk mengekspor daftar lagu yang baru diputar dalam berbagai format: M3U, CSV, atau TXT. Instruksi terperinci tersedia di situs web kami [di sini](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).

#### Favorit

Anda dapat mengelola daftar lagu favorit Anda.

- **Pengeditan Serentak**: Aktifkan opsi ini untuk menambahkan lagu ke daftar favorit di perpustakaan musik dan bagian file secara bersamaan.
- **Hapus Daftar**: Anda dapat menghapus seluruh daftar lagu favorit.
- **Ekspor Daftar Lagu**: Mirip dengan bagian Terbaru, Anda dapat mengekspor daftar trek favorit Anda dalam berbagai format: M3U, CSV, atau TXT.

#### Hapus perpustakaan musik

Tindakan ini akan menghapus database perpustakaan musik, tetapi akan membiarkan file musik Anda tidak tersentuh.

#### Batas pemuatan konten

Secara default, aplikasi menggunakan paginasi untuk mengurangi waktu pemuatan konten. Namun, Anda dapat menonaktifkan opsi ini dan mengizinkan aplikasi memuat semua data yang tersedia sekaligus. Untuk melakukannya, buka pengaturan aplikasi, gulir ke bawah ke "Personalisasi" -> "Batas pemuatan konten" dan pilih "Dinonaktifkan".

#### Gaya Menu Utama

Anda dapat mengkonfigurasi gaya menu Perpustakaan Media. Opsi yang tersedia adalah Menu Polos dan Menu Berkelompok. Untuk mengubah ini, buka Pengaturan, lalu pergi ke Personalisasi, dan pilih Gaya Menu Utama.

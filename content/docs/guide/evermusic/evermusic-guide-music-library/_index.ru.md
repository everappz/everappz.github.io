Управление музыкальной библиотекой в Evermusic — простое и удобное занятие: вы можете легко организовать все свои треки. Есть два способа наполнить библиотеку: ручное добавление или автоматическая синхронизация.

## Ручное добавление

Для ручного добавления треков нажмите пункт меню «Добавить музыку» и выберите папки/файлы из подключённого облачного хранилища или файлы на устройстве. При добавлении треков в библиотеку создаются только ссылки на эти треки, а сами файлы остаются в исходных местах, экономя место на диске. Для офлайн-доступа используйте действие «Скачать» в меню параметров или включите офлайн-режим для плейлистов и коллекций треков.

## Быстрый доступ

В верхней части меню музыкальной библиотеки вы найдёте раздел быстрого доступа с удобными ссылками на избранное, недавние и аудиозакладки.

### Избранное

Вы можете отмечать песни как избранные на экране аудиоплеера или через меню параметров.

### Недавние

В этом разделе отображаются все недавно воспроизведённые треки.

### Закладки

Вы можете создавать аудиозакладки во время воспроизведения песни и управлять ими на этом экране. Подробные инструкции по работе с аудиозакладками можно найти [здесь](/docs/guide/evermusic/evermusic-guide-music-library).

## Продолжить воспроизведение

Восстановить очередь аудиоплеера с последней сохранённой позиции, если эта функция включена в настройках приложения.

## Категории

При добавлении треков в музыкальную библиотеку приложение автоматически считывает их аудиотеги и организует по категориям: Песни, Непрослушанные Песни, Альбомы, Исполнители Альбома, Исполнители, Жанры и Композиторы.

## Верхняя панель инструментов

Расположенная прямо под навигационной панелью, верхняя панель инструментов предлагает несколько удобных действий: «Поиск», «Воспроизвести всё», «Перемешать всё» и «Продолжить воспроизведение». Показать или скрыть панель можно простым жестом смахивания вниз.

## Меню параметров

Каждая песня в библиотеке имеет меню с дополнительными действиями, открываемое нажатием кнопки трёх точек рядом с названием. Набор действий зависит от того, это одиночная песня или часть коллекции.

{{< cards cols="1">}}
  {{< card title="" subtitle="Другие Действия для Элемента Библиотеки" image="/docs/guide/evermusic/evermusic-guide-music-library/img/music-library-item-more-actions.webp" >}}

### Для отдельных песен

- **Воспроизвести следующим**: добавляет песню в начало очереди плеера.
- **Воспроизвести позже**: добавляет песню в конец очереди плеера.
- **Добавить в плейлист**: добавляет песню в плейлист.
- **Добавить в избранное**: отмечает песню как избранную для быстрого доступа.
- **Скачать**: сохраняет песню в локальных файлах. Появляется во вкладке «Локальные файлы» и разделе «Офлайн-музыка».
- **Редактировать теги**: открывает встроенный редактор аудиотегов для исправления отсутствующих метаданных; это изменит файл в хранилище.
- **Показать в папке**: показывает папку, где хранится аудиофайл.
- **Показать в Finder**: для файлов с Mac открывает папку в Finder, где находится аудиофайл.
- **Открыть в**: экспортирует аудиофайл в другое приложение.
- **Удалить из облачного сервиса**: удаляет файл из библиотеки и облачного хранилища — это необратимо.
- **Удалить из музыкальной библиотеки**: удаляет песню из библиотеки, но файл остаётся в хранилище. При включённой автосинхронизации и наличии файла на удалённом хранилище он снова появится в библиотеке после синхронизации.

### Для коллекций песен

Для коллекций (Альбомы, Исполнители, Жанры, Композиторы) в меню параметров доступны следующие действия.

- **Воспроизвести всё**: заменяет очередь плеера треками выбранной коллекции.
- **Воспроизвести следующим**: добавляет треки коллекции в начало очереди.
- **Воспроизвести позже**: добавляет треки коллекции в конец очереди.
- **Добавить в плейлист**: включает треки коллекции в плейлист с возможностью создания нового.
- **Включить офлайн режим**: загружает треки коллекции в локальные файлы. Они появятся во вкладке «Локальные файлы» и разделе «Офлайн-музыка». При добавлении новых элементов на сервер они автоматически загружаются в «Локальные файлы».
- **Редактировать изображение**: позволяет изменить обложку альбома для коллекции.
- **Удалить из музыкальной библиотеки**: удаляет коллекцию из библиотеки. Фактические файлы в хранилище не удаляются. При автосинхронизации они снова появятся.

## Группировка тегов

Эти категории помогают организовать треки по музыкальным тегам: Песни, Альбомы, Исполнители Альбома, Исполнители, Жанры, Композиторы. При добавлении треков приложение считывает метаданные и группирует их по этим категориям. Если не все альбомы видны, убедитесь, что приложение просканировало все треки. Прогресс сканирования можно проверить в Настройки -> Библиотека -> Чтение Метаданных -> Количество Обработанных Файлов. Для локальных файлов можно повторно просканировать офлайн-папки в Настройки -> Библиотека -> Синхронизация Офлайн-Папок -> Запустить Сканирование Локальных Папок. После завершения работы читателя метаданных в библиотеке будут доступны следующие категории.

- **Песни**: Все песни сгруппированы по тегу TRACK_TITLE. Порядок сортировки можно проверить через меню «Другие действия» в правом верхнем углу.
- **Непрослушанные песни**: Все непрослушанные песни, сгруппированные по тегу TRACK_TITLE.
- **Альбомы**: Песни сгруппированы по тегу ALBUM_NAME; теги исполнителя, исполнителя альбома и композитора игнорируются. При наличии нескольких альбомов с одинаковым названием разных исполнителей рассмотрите сортировку «Эксклюзивные Альбомы».
- **Исполнители Альбома**: Песни сгруппированы только по ALBUM_ARTIST_TAG.
- **Исполнители**: Песни сгруппированы только по ARTIST_TAG.
- **Жанры**: Песни сгруппированы по тегу GENRE.
- **Композиторы**: Песни сгруппированы по тегу COMPOSER.

## Детали альбома

При открытии разделов Исполнитель, Исполнитель Альбома или Композитор отображается переключатель Песни/Все Альбомы/Эксклюзивные Альбомы/Сольные Альбомы.

- **Песни**: Отображает все песни, где данный Исполнитель/Исполнитель Альбома/Композитор указан в аудиотегах.
- **Все Альбомы**: Показывает компиляции и все альбомы, в которых присутствует исполнитель.
- **Эксклюзивные Альбомы**: Показывает альбомы, где указанный исполнитель — единственный исполнитель с этим названием альбома.
- **Сольные Альбомы**: Показывает альбомы только с треками указанного исполнителя, даже если другие исполнители имеют альбомы с тем же названием.

Нажмите пункт меню «Настройки», чтобы настроить параметры музыкальной библиотеки.

#### Чтение метаданных

При добавлении треков в библиотеку запускается читатель метаданных. Этот фоновый процесс читает все метаданные треков и организует их по Исполнителю, Альбому, Жанру и Композитору. Можно регулировать скорость чтения для более быстрой загрузки данных, но это может потреблять больше энергии. Также можно отключить читатель метаданных и отображать имена файлов вместо тегов.

Читатель метаданных обновляет только базу данных библиотеки и не изменяет файлы в облаке или локальном хранилище. Для редактирования метаданных аудиофайлов используйте встроенный редактор тегов из меню параметров.

#### Доступные режимы читателя метаданных

- **Деактивирован**: Читатель отключён, вместо тегов отображаются имена файлов.
- **Текущая песня**: Метаданные читаются только для текущей воспроизводимой песни. Используйте при медленном соединении во избежание многочисленных запросов к серверу.
- **Очередь воспроизведения**: Метаданные читаются для всех треков в очереди плеера.
- **Библиотека**: Метаданные читаются для всех треков в библиотеке.

При включённом фоновом чтении метаданных читатель продолжает работу в фоне. Если приложение потребляет много энергии при воспроизведении, iOS может его приостановить.

При наличии большой коллекции рекомендуется выполнять синхронизацию метаданных в настольной версии, а затем переносить библиотеку через функцию резервного копирования.

При включённой нормализации кодировки метаданных приложение автоматически нормализует кодировку для всех треков. Это исправляет проблемы с повреждёнными тегами (например, после редактирования на Windows).

Действие «Перезагрузить метаданные» помечает все файлы как требующие обновления, запуская повторное чтение.

Нажмите «Начать чтение метаданных» для запуска читателя. Прогресс будет отображаться ниже.

#### Онлайн-синхронизация

Автоматическая онлайн-синхронизация позволяет автоматически добавлять треки из подключённых облачных сервисов. Для включения перейдите в настройки библиотеки и выберите папки синхронизации.

При включении приложение сканирует выбранные папки, находит поддерживаемые аудиофайлы и добавляет их в библиотеку. Запустить или остановить синхронизацию можно через соответствующее действие в меню.

Онлайн-синхронизация работает только при активном приложении на переднем плане, поэтому для большой библиотеки может потребоваться время. Для ускорения оставьте приложение открытым, подключите к зарядке и включите «Экран → Всегда активен» в настройках.

Также можно выполнить синхронизацию в настольной версии и перенести библиотеку через функцию резервного копирования.

Частоту синхронизации можно настроить: при значении «Немедленно» синхронизация запускается каждый раз при открытии приложения.

#### Офлайн-синхронизация

Здесь настраивается синхронизация офлайн-музыки.

##### Синхронизированные офлайн папки

При переводе облачной папки в офлайн-режим (через меню «Другие действия») она появляется в разделе Локальные файлы → Офлайн папки. Содержимое скачивается на устройство. При изменениях в облаке (добавление, удаление, обновление файлов) приложение обнаруживает их и обновляет локальную копию.

На этом экране можно вручную запустить синхронизацию папки, показать её расположение или отключить офлайн-режим. Отключение удаляет все локальные копии файлов.

##### Интервал времени

Установите интервал, с которым приложение проверяет офлайн-папки на изменения.

##### Запустить сканирование локальных папок

Сканирует все локальные папки в директории «Документы» на наличие поддерживаемых аудиофайлов и добавляет их в библиотеку. Файлы за пределами директории «Документы» необходимо добавлять вручную из-за ограничений безопасности iOS/macOS.

##### Важно

Рекомендуется периодически запускать синхронизацию офлайн-музыки для поддержания актуальности библиотеки.

#### Персонализация

Здесь можно настроить стиль экрана библиотеки. Доступно три варианта: Обычное меню, Сгруппированное меню, Меню с вкладками.

#### Обложки альбомов

Выберите качество обложек, хранящихся на устройстве, и управляйте кэшированными обложками. По умолчанию приложение ищет встроенные обложки в треках и отображает их. При отсутствии встроенных обложек и включённой опции «Поиск в папке» приложение ищет изображения JPEG или PNG в той же папке.

#### Плейлисты

Включите возможность добавления одной и той же песни в плейлист дважды. По умолчанию отключено.

#### Недавние

Управляйте списком недавно воспроизведённых.

- **Удалить список**: Удалить весь список недавно воспроизведённых.
- **Изменить размер списка**: Задать количество элементов в списке.
- **Экспортировать список**: Экспортировать список в форматах M3U, CSV или TXT. Подробные инструкции [здесь](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).

#### Избранное

Управляйте списком избранных треков.

- **Одновременное редактирование**: При включении добавление в избранное происходит одновременно в библиотеке и в разделе файлов.
- **Удалить список**: Удалить весь список избранных.
- **Экспортировать список**: Экспортировать список в форматах M3U, CSV или TXT.

#### Удалить музыкальную библиотеку

Удаляет базу данных библиотеки, оставляя файлы нетронутыми.

#### Лимит загрузки содержимого

По умолчанию используется пагинация для ускорения загрузки. Можно отключить и загружать всё сразу: откройте настройки, перейдите в «Персонализация» → «Лимит загрузки содержимого» и выберите «Деактивировано».

#### Стиль главного меню

Настройте стиль меню Медиабиблиотеки: Обычное меню или Сгруппированное меню. Перейдите в Настройки → Персонализация → Стиль Главного Меню.

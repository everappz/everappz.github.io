Gestire la tua libreria musicale è semplicissimo con Evermusic, dove puoi organizzare senza sforzo tutte le tue tracce. Hai due opzioni per costruire la tua libreria musicale: aggiunta manuale o sincronizzazione automatica.

## Aggiunta Manuale

Per aggiungere tracce manualmente, tocca la voce di menu "Aggiungi musica" e seleziona cartelle/file dal servizio di storage cloud collegato o file situati sul tuo dispositivo. Quando aggiungi tracce alla libreria, vengono creati solo collegamenti a quelle tracce, preservando i file effettivi nelle loro posizioni originali per risparmiare spazio su disco prezioso. Se vuoi rendere le tracce disponibili offline, puoi usare l'azione di download dal menu delle opzioni o abilitare la modalità offline per playlist e collezioni di tracce.

## Accesso Rapido

Nella parte superiore del menu della libreria musicale, troverai una sezione di accesso rapido che fornisce comodi collegamenti ai tuoi preferiti, recenti e segnalibri audio.

### Preferiti

Puoi contrassegnare i brani come preferiti nella schermata del lettore audio o usando il menu delle opzioni.

### Recenti

Questa sezione mostra tutte le tracce riprodotte di recente.

### Segnalibri

Puoi creare segnalibri audio mentre un brano è in riproduzione e gestirli in questa schermata. Istruzioni dettagliate su come lavorare con i segnalibri audio si trovano [qui](/docs/guide/evermusic/evermusic-guide-music-library).

## Continua Riproduzione

Ripristina la coda del lettore audio dall'ultima posizione salvata se questa funzione è abilitata nelle impostazioni dell'applicazione.

## Posizioni

Tutte le tracce nella tua libreria sono raggruppate per tipo di sorgente e tag musicali.

### Musica online

Questa sezione mostra la musica dai tuoi servizi di storage cloud.

### File in questa applicazione

Qui puoi trovare la musica disponibile per la riproduzione offline, proveniente dai tuoi file locali. Questo include i file nella directory dei documenti dell'app.

### File su questo iPhone/iPad/Mac

Questa categoria include la musica importata nell'applicazione dal tuo dispositivo aggiunta attraverso il dialogo di sistema 'Apri file…'.

### Musica iTunes

Questa sezione mostra la musica importata nella libreria iPod del tuo dispositivo. Tieni presente che le tracce di Apple Music non sono disponibili qui.

Quando aggiungi tracce alla tua libreria musicale, l'app legge automaticamente i loro tag audio e le organizza in categorie come Brani, Brani non riprodotti, Album, Artisti album, Artisti, Generi e Compositori.

## Menu Opzioni

Ogni brano nella tua libreria musicale ha un menu con altre azioni, accessibile toccando il pulsante dei tre punti vicino al titolo del brano. Queste azioni variano a seconda che si tratti di un singolo brano o parte di una raccolta.

{{< cards cols="1">}}
  {{< card title="" subtitle="Altre Azioni per un Elemento della Libreria" image="/docs/guide/evermusic/evermusic-guide-music-library/img/music-library-item-more-actions.webp" >}}
{{< /cards >}}

### Per Brani Singoli

- **Riproduci successivo**: Aggiunge il brano in cima alla coda del lettore.
- **Riproduci più tardi**: Aggiunge il brano in fondo alla coda del lettore.
- **Aggiungi alla playlist**: Aggiunge il brano a una playlist.
- **Aggiungi ai preferiti**: Contrassegna il brano come preferito per un accesso rapido.
- **Scarica**: Salva il brano nei file locali. Appare nella scheda "File Locali" e nella sezione "Musica offline".
- **Modificare i tag audio**: Apre l'editor di tag audio integrato per correggere i metadati mancanti; nota che questo modificherà il brano nel tuo storage.
- **Mostra in cartella**: Rivela la cartella dove è archiviato il file audio.
- **Mostra in Finder**: Per i file importati dal tuo Mac, questa azione rivela la cartella dove si trova il file audio sul tuo Mac.
- **Apri con**: Esporta il file audio in un'altra app.
- **Elimina dal servizio cloud**: Rimuove il file dalla libreria musicale e dallo storage cloud, e tieni presente che questa azione è irreversibile.
- **Elimina dalla libreria musicale**: Elimina il brano dalla tua libreria musicale, ma il file rimane nel storage. Se la sincronizzazione automatica è abilitata e il file esiste nel storage remoto, riapparirà nella tua libreria dopo un'operazione di sincronizzazione.

### Per Raccolte di Brani

Per raccolte di brani come Album, Artisti, Generi o Compositori, il menu delle opzioni include le seguenti azioni.

- **Riproduci tutto**: Sostituisce la coda del lettore con i brani dalla raccolta selezionata.
- **Riproduci successivo**: Aggiunge i brani da questa raccolta in cima alla coda del lettore.
- **Riproduci più tardi**: Aggiunge i brani da questa raccolta in fondo alla coda del lettore.
- **Aggiungi alla playlist**: Include i brani da questa raccolta in una playlist, con la possibilità di creare una nuova playlist.
- **Abilita modalità offline**: Scarica i brani da questa raccolta nei file locali. Questi brani scaricati appariranno nella scheda "File Locali" e nella sezione "Musica offline" della tua Libreria musicale. Se nuovi elementi vengono aggiunti alla raccolta nel server, verranno scaricati automaticamente in "File Locali."
- **Modifica immagine**: Ti permette di cambiare la copertina dell'album per la raccolta di brani.
- **Elimina dalla libreria musicale**: Rimuove la raccolta di brani dalla tua libreria musicale. Tuttavia, questa azione non elimina i file effettivi dal storage. Se la sincronizzazione automatica è abilitata e i file esistono nel storage remoto, riappariranno nella tua libreria dopo un'operazione di sincronizzazione.

## Raggruppamento Tag

Queste categorie ti aiutano a organizzare le tue tracce per tag musicali: Brani, Album, Artisti album, Artisti, Generi, Compositori. Quando aggiungi tracce alla libreria musicale, l'app legge i metadati delle tracce e le raggruppa per queste categorie. Se non vedi tutti i tuoi album, assicurati che l'app abbia scansionato tutte le tracce. Puoi verificare il progresso della scansione in Impostazioni -> Libreria musicale -> Lettura Metadati -> Numero di File Elaborati nella Libreria musicale. Per i file locali, puoi anche ri-scansionare le cartelle offline in Impostazioni -> Libreria musicale -> Sincronizzazione Cartelle Offline -> Avvia Scansione Cartelle Locali. Dopo che il lettore di metadati ha completato tutte le operazioni, puoi vedere le seguenti categorie nella tua libreria musicale.

- **Brani**: Tutti i brani sono raggruppati per il tag TRACK_TITLE. Puoi verificare l'ordine di ordinamento usando il menu Altre Azioni nell'angolo in alto a destra.
- **Brani non riprodotti**: Tutti i brani sono raggruppati per il tag TRACK_TITLE e non ancora riprodotti.
- **Album**: I brani sono raggruppati per il tag ALBUM_NAME, mentre i tag artista, artista album e compositore vengono saltati. Se hai più album con lo stesso nome ma artisti diversi, considera l'utilizzo dell'ordinamento Album Esclusivi descritto di seguito.
- **Artisti Album**: I brani sono raggruppati solo per ALBUM_ARTIST_TAG.
- **Artisti**: I brani sono raggruppati solo per ARTIST_TAG.
- **Generi**: I brani sono raggruppati per il tag GENRE.
- **Compositori**: I brani sono raggruppati per il tag COMPOSER.

- **Brani**: Mostra tutti i brani in cui questo Artista/Artista Album/Compositore è impostato nei tag audio.

#### Lettura Metadati

Quando aggiungi tracce alla libreria, il lettore di metadati entra in azione. Questo processo in background legge tutti i metadati dalle tue tracce e le organizza per Artista, Album, Genere e Compositore. Hai la flessibilità di regolare la velocità di lettura dei metadati per caricare i dati più velocemente, ma tieni presente che questo potrebbe consumare più energia. Puoi anche disabilitare il lettore di metadati e visualizzare i nomi dei file invece delle informazioni sui tag.

È importante notare che il lettore di metadati aggiorna solo i metadati nella tua libreria musicale e non altera i file archiviati nel tuo account cloud o nel storage locale. Se desideri modificare i metadati per file audio, puoi farlo usando l'editor di tag integrato, che puoi attivare dalla corrispondente azione nel menu delle opzioni.

#### Modalità Disponibili per il Lettore di Metadati

- **Disattivato**: Il lettore di metadati sarà disattivato, e verranno mostrati i nomi dei file invece dei dati dai tag audio.
- **Brano Corrente**: L'applicazione leggerà i metadati solo per il brano attualmente in riproduzione. Usa questa opzione se hai una connessione di rete lenta per evitare che il lettore di metadati invii molte richieste al server cloud, il che potrebbe causare interruzioni della riproduzione.
- **Coda di Riproduzione**: L'app leggerà i metadati per tutti i brani nella coda del lettore audio.
- **Libreria**: L'app leggerà i metadati per tutti i brani nella libreria musicale.

Quando l'interruttore Lettura Metadati in background è attivo, il lettore di metadati funziona in modalità background. Tuttavia, tieni presente che se l'app consuma molta energia durante la riproduzione audio, il sistema operativo iOS potrebbe sospenderla.

Quindi, se hai una grande collezione musicale, è consigliabile usare la versione desktop dell'applicazione per la sincronizzazione dei metadati. Puoi quindi usare la funzione di backup e ripristino dei dati per trasferire la libreria musicale sincronizzata dal desktop, disponibile nelle impostazioni dell'app.

Quando Normalizza codifica metadati è abilitato, l'app normalizzerà automaticamente la codifica dei metadati per tutti i brani nella libreria musicale. Questo risolve i problemi in cui la codifica dei tag audio è rotta (come dopo aver modificato file su un PC Windows) e previene la visualizzazione di informazioni errate durante la riproduzione o l'aggiunta di una traccia alla libreria.

L'azione Ricaricare i metadati contrassegnerà tutti i file nella tua libreria musicale come aventi metadati mancanti, attivando il lettore di metadati per aggiornare i metadati per ogni file nella tua libreria musicale.

Tocca l'azione Avvia Lettura Metadati per avviare il lettore di metadati. Il progresso dell'operazione verrà visualizzato di seguito.

#### Sincronizzazione Online

La sincronizzazione automatica della musica online ti permette di aggiungere tracce dai servizi cloud collegati alla libreria musicale automaticamente. Per attivare questa funzione, vai alle impostazioni della libreria musicale e seleziona le cartelle di sincronizzazione.

Con questa opzione abilitata, l'applicazione scansiona tutte le cartelle selezionate, identifica i file audio supportati e li integra senza problemi nella tua libreria. Puoi avviare o fermare la sincronizzazione toccando l'azione di menu corrispondente.

La sincronizzazione della musica online funziona esclusivamente quando l'app è in primo piano, il che significa che la sincronizzazione potrebbe richiedere del tempo. Per accelerare il processo, lascia l'app aperta, collegala a una fonte di alimentazione e abilita l'opzione Schermo -> Sempre attivo nelle impostazioni dell'applicazione.

In alternativa, puoi eseguire la sincronizzazione della musica online sulla versione desktop dell'app e trasferire la libreria musicale alla versione iOS usando la funzione di backup e ripristino dei dati.

Puoi anche impostare la frequenza con cui vuoi sincronizzare la tua libreria musicale online. Se lo imposti su "Immediatamente," la sincronizzazione online inizierà ogni volta che apri l'applicazione.

#### Sincronizzazione Offline

Qui puoi configurare la sincronizzazione della musica offline.

##### Cartelle Offline Sincronizzate

Quando rendi disponibile offline una cartella cloud (tramite il menu Altre Azioni), essa apparirà nella sezione File Locali → Cartelle Offline. L'app scaricherà il suo contenuto sul tuo dispositivo. Se apporti modifiche alla cartella nel cloud — come aggiungere, rimuovere o aggiornare file — l'app rileverà queste modifiche e aggiornerà automaticamente la copia locale.

In questa schermata, puoi avviare manualmente la sincronizzazione delle cartelle offline, mostrare la cartella offline nella sua cartella contenitore e disabilitare la modalità offline per questa cartella. Disabilitare la modalità offline rimuoverà tutte le copie locali dei file dal tuo dispositivo.

##### Intervallo di Tempo

Puoi impostare l'intervallo di tempo per la frequenza con cui l'app dovrebbe verificare le modifiche nelle cartelle offline.

##### Avvia Scansione Cartelle Locali

Questa opzione scansiona tutte le cartelle locali situate nella directory Documents dell'applicazione per trovare i file audio supportati. Tutti questi file locali vengono aggiunti senza problemi alla tua libreria musicale. I file locali situati sul tuo dispositivo ma al di fuori di questa applicazione devono essere aggiunti manualmente alla libreria musicale, poiché l'app non ha accesso ai file al di fuori della directory Documents dell'applicazione a causa delle restrizioni di sicurezza di iOS/MacOS.

##### Importante

È consigliabile avviare periodicamente la sincronizzazione della musica offline per mantenere la tua libreria musicale aggiornata con i tuoi file locali.

#### Personalizzazione

In questa sezione, puoi configurare lo stile della schermata della libreria musicale secondo le tue preferenze. Sono disponibili tre opzioni: Menu semplice, Menu raggruppato, Menu a schede.

#### Copertine Album

Qui puoi scegliere la qualità delle copertine degli album memorizzate sul tuo dispositivo e gestire le copertine degli album memorizzate nella cache. Per impostazione predefinita, l'app verifica le copertine degli album incorporate nelle tue tracce e le mostra se disponibili. Se non ci sono copertine degli album incorporate e l'opzione 'Cerca nella cartella' è abilitata, l'app cercherà nella cartella contenitore immagini JPEG o PNG e le userà come copertina dell'album per tutte le tracce in quella cartella.

#### Playlist

Puoi abilitare l'opzione per aggiungere lo stesso brano a una playlist due volte. Per impostazione predefinita, questa opzione è disabilitata.

#### Recenti

Puoi gestire il tuo elenco di brani riprodotti di recente.

- **Elimina Elenco**: Puoi eliminare l'intero elenco dei brani riprodotti di recente.
- **Cambia Dimensione Elenco**: Puoi impostare il numero di elementi che dovrebbero apparire nell'elenco.
- **Esporta Elenco Brani**: Usa questa azione per esportare il tuo elenco di brani riprodotti di recente in diversi formati: M3U, CSV o TXT. Istruzioni dettagliate sono disponibili sul nostro sito web [qui](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).

#### Preferiti

Puoi gestire l'elenco dei tuoi brani preferiti.

- **Modifica Simultanea**: Abilita questa opzione per aggiungere un brano all'elenco dei preferiti sia nella libreria musicale che nella sezione dei file contemporaneamente.
- **Elimina Elenco**: Puoi eliminare l'intero elenco dei brani preferiti.
- **Esporta Elenco Brani**: Simile alla sezione Recenti, puoi esportare l'elenco delle tue tracce preferite in diversi formati: M3U, CSV o TXT.

#### Elimina libreria musicale

Questa azione cancellerà il database della libreria musicale, ma lascerà intatti i tuoi file musicali.

#### Limite di caricamento contenuti

Per impostazione predefinita, l'applicazione usa la paginazione per ridurre i tempi di caricamento dei contenuti. Tuttavia, puoi disabilitare questa opzione e consentire all'applicazione di caricare tutti i dati disponibili in una volta. Per farlo, apri le impostazioni dell'applicazione, scorri verso il basso su "Personalizzazione" -> "Limite di caricamento contenuti" e scegli "Disattivato".

#### Stile Menu Principale

Puoi configurare lo stile del menu Libreria multimediale. Le opzioni disponibili sono Menu semplice e Menu raggruppato. Per cambiare questo, apri Impostazioni, poi vai a Personalizzazione e seleziona Stile Menu Principale.

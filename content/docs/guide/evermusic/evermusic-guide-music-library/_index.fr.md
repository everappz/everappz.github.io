Gérer votre bibliothèque musicale est un jeu d'enfant avec Evermusic, où vous pouvez organiser toutes vos pistes sans effort. Vous avez deux options pour créer votre bibliothèque musicale : l'ajout manuel ou la synchronisation automatique.

## Ajout manuel

Pour ajouter des pistes manuellement, appuyez sur l'élément de menu « Ajouter de la musique » et sélectionnez des dossiers/fichiers depuis le service de stockage cloud connecté ou les fichiers situés sur votre appareil. Lorsque vous ajoutez des pistes à la bibliothèque, seuls des liens vers ces pistes sont créés, préservant les fichiers réels dans leurs emplacements d'origine pour économiser l'espace disque. Si vous souhaitez rendre les pistes disponibles hors ligne, vous pouvez utiliser l'action de téléchargement depuis le menu des options ou activer le mode hors ligne pour les playlists et les collections de pistes.

## Accès rapide

En haut du menu de la bibliothèque musicale, vous trouverez une section d'accès rapide offrant des liens pratiques vers vos favoris, vos récents et vos signets audio.

### Favoris

Vous pouvez marquer des chansons comme favoris sur l'écran du lecteur audio ou en utilisant le menu des options.

### Récents

Cette section affiche toutes les pistes récemment lues.

### Signets

Vous pouvez créer des signets audio pendant qu'une chanson est en cours de lecture et les gérer sur cet écran. Des instructions détaillées sur l'utilisation des signets audio peuvent être trouvées [ici](/docs/guide/evermusic/evermusic-guide-music-library).

## Continuer la lecture

Restaurez la file d'attente du lecteur audio depuis la dernière position sauvegardée si cette fonctionnalité est activée dans les paramètres de l'application.

## Emplacements

Toutes les pistes de votre bibliothèque sont soigneusement regroupées par types de sources et tags musicaux.

### Musique en ligne

Cette section présente la musique de vos services de stockage cloud.

### Fichiers dans cette application

Vous y trouverez la musique disponible pour la lecture hors ligne, provenant de vos fichiers locaux. Cela inclut les fichiers dans le répertoire Documents de l'application.

### Fichiers sur cet iPhone/iPad/Mac

Cette catégorie inclut la musique importée dans l'application depuis votre appareil, ajoutée via la boîte de dialogue système « Ouvrir les fichiers... ».

### Musique iTunes

Cette section affiche la musique importée dans la bibliothèque iPod de votre appareil. Veuillez noter que les pistes d'Apple Music ne sont pas disponibles ici.

## Catégories

Lorsque vous ajoutez des pistes à votre bibliothèque musicale, l'application lit automatiquement leurs tags audio et les organise en catégories telles que Chansons, Chansons non lues, Albums, Artistes d'album, Artistes, Genres et Compositeurs.

## Barre d'outils supérieure

Située juste sous la barre de navigation, la barre d'outils supérieure offre plusieurs actions pratiques : « Rechercher », « Tout lire », « Lecture aléatoire » et « Continuer la lecture ». Vous pouvez afficher ou masquer cette barre d'outils avec un simple geste de glissement vers le bas.

## Menu Options

Chaque chanson dans votre bibliothèque musicale dispose d'un menu avec plus d'actions, accessible en appuyant sur le bouton à trois points près du titre de la chanson. Ces actions varient selon qu'il s'agit d'une seule chanson ou d'une collection.

  {{< card title="" subtitle="Plus d'actions pour un élément de la bibliothèque" image="/docs/guide/evermusic/evermusic-guide-music-library/img/music-library-item-more-actions.webp" >}}
{{< /cards >}}

### Pour les chansons individuelles

- **Lire ensuite** : ajoute la chanson en haut de la file d'attente du lecteur.
- **Lire plus tard** : ajoute la chanson en bas de la file d'attente du lecteur.
- **Ajouter à une playlist** : ajoute la chanson dans une playlist.
- **Ajouter aux favoris** : marque la chanson comme favorite pour un accès rapide.
- **Télécharger** : sauvegarde la chanson dans les fichiers locaux. Elle apparaît dans l'onglet « Fichiers locaux » et la section « Musique hors ligne ».
- **Modifier les balises audio** : ouvre l'éditeur de tags audio intégré pour corriger les métadonnées manquantes ; notez que cela modifiera la chanson sur votre stockage.
- **Afficher dans le dossier** : révèle le dossier où le fichier audio est stocké.
- **Afficher dans le Finder** : pour les fichiers importés depuis votre Mac, cette action révèle le dossier où le fichier audio est situé sur votre Mac.
- **Ouvrir dans** : exporte le fichier audio vers une autre application.
- **Supprimer du service cloud** : supprime le fichier de la bibliothèque musicale et du stockage cloud ; veuillez noter que cette action est irréversible.
- **Supprimer de la bibliothèque musicale** : supprime la chanson de votre bibliothèque musicale, mais le fichier reste dans le stockage. Si la synchronisation automatique est activée et que le fichier existe sur le stockage distant, il réapparaîtra dans votre bibliothèque après une opération de synchronisation.

### Pour les collections de chansons

Pour les collections de chansons comme les Albums, les Artistes, les Genres ou les Compositeurs, le menu des options inclut les actions suivantes.

- **Tout lire** : remplace la file d'attente du lecteur par les chansons de la collection sélectionnée.
- **Lire ensuite** : ajoute les chansons de cette collection en haut de la file d'attente du lecteur.
- **Lire plus tard** : ajoute les chansons de cette collection en bas de la file d'attente du lecteur.
- **Ajouter à une playlist** : inclut les chansons de cette collection dans une playlist, avec la possibilité de créer une nouvelle playlist.
- **Activer le mode hors ligne** : télécharge les chansons de cette collection dans les fichiers locaux. Ces chansons téléchargées apparaîtront dans l'onglet « Fichiers locaux » et la section « Musique hors ligne » de votre bibliothèque musicale. Si de nouveaux éléments sont ajoutés à la collection sur le serveur, ils seront automatiquement téléchargés dans « Fichiers locaux ».
- **Modifier l'image** : vous permet de changer la pochette d'album pour la collection de chansons.
- **Supprimer de la bibliothèque musicale** : supprime la collection de chansons de votre bibliothèque musicale. Cependant, cette action ne supprime pas les fichiers réels du stockage. Si la synchronisation automatique est activée et que les fichiers existent sur le stockage distant, ils réapparaîtront dans votre bibliothèque après une opération de synchronisation.

## Regroupement par tags

Ces catégories vous aident à organiser vos pistes par tags musicaux : Chansons, Albums, Artistes d'album, Artistes, Genres, Compositeurs. Lorsque vous ajoutez des pistes à la bibliothèque musicale, l'application lit les métadonnées des pistes et les regroupe par ces catégories. Si vous ne voyez pas tous vos albums, assurez-vous que l'application a analysé toutes les pistes. Vous pouvez vérifier la progression de l'analyse dans Paramètres -> Bibliothèque musicale -> Lecture des métadonnées -> Nombre de fichiers traités dans la bibliothèque musicale. Pour les fichiers locaux, vous pouvez également rescanner les dossiers hors ligne dans Paramètres -> Bibliothèque musicale -> Synchronisation des dossiers hors ligne -> Démarrer l'analyse des dossiers locaux. Après que le lecteur de métadonnées ait terminé toutes les opérations, vous pouvez voir les catégories suivantes dans votre bibliothèque musicale.

- **Chansons** : toutes les chansons sont regroupées par le tag TRACK_TITLE. Vous pouvez vérifier l'ordre de tri en utilisant le menu Plus d'actions dans le coin supérieur droit.
- **Chansons non lues** : toutes les chansons sont regroupées par le tag TRACK_TITLE et n'ont pas été lues.
- **Albums** : les chansons sont regroupées par le tag ALBUM_NAME, tandis que les tags d'artiste, d'artiste d'album et de compositeur sont ignorés. Si vous avez plusieurs albums avec le même nom mais des artistes différents, envisagez d'utiliser le tri Albums exclusifs décrit ci-dessous.
- **Artistes d'album** : les chansons sont regroupées par le ALBUM_ARTIST_TAG uniquement.
- **Artistes** : les chansons sont regroupées par le ARTIST_TAG uniquement.
- **Genres** : les chansons sont regroupées par le tag GENRE.
- **Compositeurs** : les chansons sont regroupées par le tag COMPOSER.

- **Chansons** : affiche toutes les chansons où cet Artiste/Artiste d'album/Compositeur est défini dans les tags audio.
- **Tous les albums** : affiche les albums de compilation et tous les albums où l'artiste est présent.
- **Albums exclusifs** : affiche les albums où l'artiste spécifié est le seul artiste avec ce nom d'album.
- **Albums solo** : affiche les albums où seules les pistes de l'artiste spécifié sont affichées, même si d'autres artistes ont des albums du même nom.

#### Lecture des métadonnées

Lorsque vous ajoutez des pistes à la bibliothèque, le lecteur de métadonnées se met au travail. Ce processus en arrière-plan lit toutes les métadonnées de vos pistes et les organise par Artiste, Album, Genre et Compositeur. Vous avez la flexibilité d'ajuster la vitesse de lecture des métadonnées pour charger les données plus rapidement, mais soyez conscient que cela peut utiliser plus d'énergie. Vous pouvez également désactiver le lecteur de métadonnées et afficher les noms de fichiers à la place des informations de tag.

Surtout, le lecteur de métadonnées ne met à jour que les métadonnées dans votre bibliothèque musicale et ne modifie pas les fichiers stockés dans votre compte cloud ou stockage local. Si vous souhaitez modifier les métadonnées pour les fichiers audio, vous pouvez le faire en utilisant l'éditeur de tags intégré, que vous pouvez activer depuis l'action correspondante dans le menu des options.

#### Modes disponibles pour le lecteur de métadonnées

- **Désactivé** : le lecteur de métadonnées sera désactivé et les noms de fichiers seront affichés à la place des données des tags audio.
- **Chanson actuelle** : l'application lira les métadonnées uniquement pour la chanson en cours de lecture. Utilisez cette option si vous avez une connexion réseau lente pour éviter que le lecteur de métadonnées envoie de nombreuses requêtes au serveur cloud, ce qui pourrait provoquer des interruptions de lecture.
- **File d'attente de lecture** : l'application lira les métadonnées pour toutes les chansons dans la file d'attente du lecteur audio.
- **Bibliothèque** : l'application lira les métadonnées pour toutes les chansons dans la bibliothèque musicale.

Lorsque l'interrupteur Lecture des métadonnées en arrière-plan est activé, le lecteur de métadonnées fonctionne en mode arrière-plan. Cependant, veuillez noter que si l'application consomme beaucoup d'énergie pendant la lecture audio, le système d'exploitation iOS peut la suspendre.

Donc, si vous avez une grande collection musicale, il est conseillé d'utiliser la version de bureau de l'application pour la synchronisation des métadonnées. Vous pouvez ensuite utiliser la fonctionnalité de sauvegarde et de restauration des données pour transférer la bibliothèque musicale synchronisée depuis le bureau, disponible dans les paramètres de l'application.

Lorsque Normaliser l'encodage des métadonnées est activé, l'application normalisera automatiquement l'encodage des métadonnées pour toutes les chansons dans la bibliothèque musicale. Cela corrige les problèmes où l'encodage des tags audio est défectueux (par exemple après la modification de fichiers sur un PC Windows) et empêche l'affichage d'informations incorrectes pendant la lecture d'une piste ou son ajout à la bibliothèque.

L'action Recharger les métadonnées marquera tous les fichiers de votre bibliothèque musicale comme ayant des métadonnées manquantes, déclenchant le lecteur de métadonnées pour actualiser les métadonnées de chaque fichier dans votre bibliothèque musicale.

Appuyez sur l'action Démarrer la lecture des métadonnées pour démarrer le lecteur de métadonnées. La progression de l'opération sera affichée ci-dessous.

#### Synchronisation en ligne

La synchronisation automatique de la musique en ligne vous permet d'ajouter automatiquement des pistes des services cloud connectés à la bibliothèque musicale. Pour activer cette fonctionnalité, accédez aux paramètres de la bibliothèque musicale et sélectionnez les dossiers de synchronisation.

Avec cette option activée, l'application analyse tous les dossiers sélectionnés, identifie les fichiers audio pris en charge et les intègre de manière transparente dans votre bibliothèque. Vous pouvez démarrer ou arrêter la synchronisation en appuyant sur l'action de menu correspondante.

La synchronisation de la musique en ligne ne fonctionne que lorsque l'application est au premier plan, ce qui signifie que la synchronisation peut prendre un certain temps. Pour accélérer le processus, laissez votre application ouverte, connectez-la à une source d'alimentation et activez l'option Écran -> Toujours actif dans les paramètres de l'application.

Alternativement, vous pouvez effectuer la synchronisation de la musique en ligne sur la version de bureau de l'application et transférer la bibliothèque musicale vers la version iOS en utilisant la fonctionnalité de sauvegarde et restauration des données.

Vous pouvez également définir la fréquence de synchronisation de votre bibliothèque musicale en ligne. Si vous le définissez sur « Immédiatement », la synchronisation en ligne démarrera chaque fois que vous ouvrirez l'application.

#### Synchronisation hors ligne

Ici, vous pouvez configurer la synchronisation de la musique hors ligne.

##### Dossiers hors ligne synchronisés

Lorsque vous rendez un dossier cloud disponible hors ligne (via le menu Plus d'actions), il apparaîtra dans la section Fichiers locaux → Dossiers hors ligne. L'application téléchargera son contenu sur votre appareil. Si vous apportez des modifications au dossier dans le cloud — comme ajouter, supprimer ou mettre à jour des fichiers — l'application détectera ces modifications et mettra à jour la copie locale automatiquement.

Sur cet écran, vous pouvez démarrer manuellement la synchronisation des dossiers hors ligne, afficher le dossier hors ligne dans son dossier englobant et désactiver le mode hors ligne pour ce dossier. La désactivation du mode hors ligne supprimera toutes les copies locales des fichiers de votre appareil.

##### Intervalle de temps

Vous pouvez définir l'intervalle de temps pour lequel l'application doit vérifier les modifications dans les dossiers hors ligne.

##### Démarrer l'analyse des dossiers locaux

Cette option analyse tous les dossiers locaux situés dans le répertoire Documents de l'application pour trouver les fichiers audio pris en charge. Tous ces fichiers locaux sont ajoutés de manière transparente à votre bibliothèque musicale. Les fichiers locaux situés sur votre appareil mais en dehors de cette application doivent être ajoutés manuellement à la bibliothèque musicale, car l'application n'a pas accès aux fichiers en dehors du répertoire Documents de l'application en raison des restrictions de sécurité iOS/macOS.

##### Important

Il est conseillé de lancer périodiquement la synchronisation de la musique hors ligne pour maintenir votre bibliothèque musicale à jour avec vos fichiers locaux.

#### Personnalisation

Dans cette section, vous pouvez configurer le style de l'écran de la bibliothèque musicale selon vos préférences. Trois options sont disponibles : Menu simple, Menu groupé, Menu à onglets.

#### Pochettes d'album

Ici, vous pouvez choisir la qualité des pochettes d'album stockées sur votre appareil et gérer vos pochettes d'album mises en cache. Par défaut, l'application vérifiera les pochettes d'album intégrées dans vos pistes et les affichera si disponibles. S'il n'y a pas de pochettes intégrées et que l'option « Rechercher dans le dossier » est activée, l'application vérifiera le dossier englobant pour les images JPEG ou PNG et les utilisera comme pochettes d'album pour toutes les pistes dans ce dossier.

#### Listes de lecture

Vous pouvez activer l'option d'ajouter la même chanson à une playlist deux fois. Par défaut, cette option est désactivée.

#### Récents

Vous pouvez gérer votre liste de chansons récemment lues.

- **Supprimer la liste** : vous pouvez supprimer toute la liste des chansons récemment lues.
- **Modifier la taille de la liste** : vous pouvez définir le nombre d'éléments qui doivent apparaître dans la liste.
- **Exporter la liste de chansons** : utilisez cette action pour exporter votre liste de chansons récemment lues dans différents formats : M3U, CSV ou TXT. Des instructions détaillées sont disponibles sur notre site web [ici](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/).

#### Favoris

Vous pouvez gérer la liste de vos chansons favorites.

- **Édition simultanée** : activez cette option pour ajouter une chanson à la liste des favoris à la fois dans la bibliothèque musicale et la section des fichiers simultanément.
- **Supprimer la liste** : vous pouvez supprimer toute la liste des chansons favorites.
- **Exporter la liste de chansons** : similaire à la section Récents, vous pouvez exporter la liste de vos pistes favorites dans différents formats : M3U, CSV ou TXT.

#### Supprimer la bibliothèque musicale

Cette action effacera la base de données de la bibliothèque musicale, mais laissera vos fichiers musicaux intacts.

#### Limite de chargement du contenu

Par défaut, l'application utilise la pagination pour réduire le temps de chargement du contenu. Cependant, vous pouvez désactiver cette option et permettre à l'application de charger toutes les données disponibles en même temps. Pour ce faire, ouvrez les paramètres de l'application, faites défiler vers le bas jusqu'à « Personnalisation » -> « Limite de chargement du contenu » et choisissez « Désactivé ».

#### Style du menu principal

Vous pouvez configurer le style du menu Bibliothèque multimédia. Les options disponibles sont Menu simple et Menu groupé. Pour changer cela, ouvrez les Paramètres, puis allez dans Personnalisation et sélectionnez Style du menu principal.

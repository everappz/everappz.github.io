Evermusic adalah pemutar musik cloud yang powerful dan manajer perpustakaan yang mengubah iPhone, iPad, atau Mac Anda menjadi sistem streaming dan musik offline yang lengkap. Aplikasi ini bekerja dengan akun cloud dan NAS yang sudah Anda miliki, memutar hampir setiap format audio yang dapat didekode perangkat Anda, dan memberikan mesin audio canggih (equalizer 10-band, crossfade, gapless, spatial audio, koreksi pitch) di atasnya.

Tersedia dalam dua edisi: Evermusic (gratis, didukung iklan) dan Evermusic Pro (berbayar, tanpa iklan, fitur lengkap). Paket Premium seumur hidup, bulanan, dan tahunan pada edisi gratis membuka fitur yang sama; kedua edisi berbagi pembelian antara iOS dan Mac melalui iCloud dan Family Sharing.

### Musik Anda di Mana Saja

Evermusic memungkinkan Anda membangun sistem streaming musik cloud sendiri — seperti Spotify, tetapi tanpa batasan apapun. Hubungkan kombinasi sumber apapun dan streaming atau unduh lagu sesuai permintaan:

- **Akun cloud pribadi:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘).
- **Server self-hosted dan perpustakaan media:** Plex · Jellyfin · Emby · Subsonic (dan setiap server kompatibel Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud (dan Owncloud, melalui API bersama) · Synology Drive · QNAP.
- **NAS dan protokol berbagi file:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP / FTPS · SFTP (SSH File Transfer Protocol, autentikasi kata sandi atau kunci publik) · NFS · DLNA / UPnP (pemutaran dan pengunduhan). Bekerja dengan Apple Time Capsule, Synology, QNAP, WD My Cloud Home, Buffalo, host Linux Samba / NFS / SSH apapun, atau folder bersama di Mac atau PC Windows Anda.
- **Penyimpanan objek kompatibel S3:** AWS S3 · Backblaze B2 · Wasabi · Cloudflare R2 · MinIO · DigitalOcean Spaces · Linode Object Storage · IBM Cloud Object Storage · endpoint S3-API apapun.
- **Penemuan jaringan lokal:** bagian Perangkat yang Tersedia secara otomatis mencantumkan setiap perangkat Bonjour / mDNS di Wi-Fi Anda — server Plex, Jellyfin, Emby, Synology, QNAP, router AirPort dengan drive terpasang, Time Capsule — sehingga Anda dapat mengetuk untuk terhubung tanpa mengetik IP.
- **Perangkat dan sumber eksternal:** perpustakaan iPod / Apple Music (trek non-DRM), file di aplikasi Files sistem (aplikasi lain, SSD eksternal, folder terpasang), flash drive USB melalui pembaca kartu Apple Certified dan iXpand Flash Drive, Wi-Fi Drive (drag-and-drop di browser desktop), dan iTunes / Finder File Sharing melalui kabel USB.

Itu 30+ tipe sumber dalam satu aplikasi. Setelah musik Anda ada di cloud, Anda dapat memutarnya di mana saja, kapan saja, dengan perpustakaan yang sama di setiap perangkat. Cadangan & Pulihkan memungkinkan Anda mengambil snapshot seluruh perpustakaan ke satu file dan mengimpornya di iPhone, iPad, atau Mac lain dalam hitungan detik.

### Pemutaran Mulus

Evermusic menggunakan pra-pemuatan cerdas dan cache pemutaran yang dapat dikonfigurasi untuk membuat lagu berputar dengan mulus bahkan melalui koneksi lambat. Streaming cerdas menggunakan AVAssetResourceLoader sehingga pemutaran dimulai secara instan sementara sisa file diunduh di latar belakang. Anda juga dapat mengunduh Album, Artis, Daftar Putar, atau lagu individual untuk mendengarkan offline — dan jika penyimpanan penuh, cukup hapus file yang di-cache dan terus streaming dari cloud.

### Pengurutan Musik yang Mudah

Saat Anda menambahkan lagu ke Evermusic, pembaca metadata latar belakang mengurai setiap file dan membangun Perpustakaan Musik yang bersih, diurutkan berdasarkan Lagu, Lagu Belum Diputar, Album, Artis Album, Artis, Genre, dan Komposer. Album memiliki tiga sub-tampilan: Semua Album, Album Eksklusif, dan Album Solo sehingga kompilasi dan album bernama sama dari artis berbeda tidak bertabrakan.

### Perbaiki Info Lagu dengan Mudah

Jika judul lagu atau info album salah atau hilang, jangan khawatir. Evermusic menyertakan editor tag bawaan yang memungkinkan Anda memperbaiki metadata dalam hitungan detik. Anda dapat memperbarui judul, artis, album, artis album, tahun, genre, dan sampul album, plus menormalkan encoding yang rusak (teks Cyrillic atau Asia yang tampak berantakan) dengan satu ketukan. Editor menggunakan MusicBrainz untuk menemukan tag dan karya album yang hilang secara otomatis.

### Transfer File Sederhana

Mengirim musik dari komputer ke iPhone atau iPad Anda mudah. Evermusic mendukung SMB, WebDAV, FTP / FTPS, SFTP, NFS, DLNA, Wi-Fi Drive (drag-and-drop di browser), iTunes / Finder File Sharing (kabel USB), dan unduhan langsung dari akun cloud yang terhubung. Anda juga dapat menghubungkan Apple Time Capsule, WD My Cloud Home, Synology, QNAP, Buffalo, atau NAS lain yang mengekspos salah satu protokol standar tersebut, tanpa menggunakan penyimpanan perangkat.

### Kontrol Suara yang Powerful

Evermusic menyertakan equalizer audio 10-band penuh dengan preset gaya iPod (Acoustic, Bass Booster, Classical, Dance, Electronic, Flat, Hip-Hop, Jazz, Latin, Loudness, Lounge, Piano, Pop, R&B, Rock, Small Speakers, Spoken Word, Treble Booster, Vocal Booster), preamplifier untuk trek yang tenang, preset kustom yang dapat diekspor dan diimpor, dan perhitungan kenyaringan berbasis RMS. Tambahkan crossfade playback (1–15 detik), gapless playback, spatial audio, koreksi pitch, dan output CoreAudio hingga 384 kHz mono atau stereo, dan Anda dapat menyetel suara persis seperti yang Anda inginkan.

### Bagus untuk Audiobook

Jika Anda mendengarkan audiobook, podcast, atau kuliah yang direkam, Evermusic sangat cocok. Anda dapat menambahkan beberapa bookmark per trek, melanjutkan dari posisi terakhir secara otomatis, mengubah kecepatan pemutaran dari 0,02× hingga 3× (dengan mode slider presisi), mengatur sleep timer dengan interval kustom, dan melihat komentar, lirik tertanam, dan file LRC di layar yang sama.

### Bekerja di Mana Saja

Anda tidak terbatas pada ponsel atau tablet. Evermusic juga bekerja dengan Apple CarPlay (hanya iOS, dioptimalkan untuk UI mobil), AirPlay 2 (streaming ke beberapa speaker secara bersamaan), Google Cast / Chromecast, widget Layar Kunci, kontrol Now Playing, Last.fm scrobbling (kirim pemutaran otomatis ke akun Last.fm Anda untuk statistik dan rekomendasi), dan pintasan keyboard Mac. Jadi baik di rumah, di mobil, atau dalam perjalanan, musik Anda selalu bersama Anda.

### Memulai dengan Evermusic

Panduan ini akan membantu Anda mendapatkan yang terbaik dari Evermusic di iPhone, iPad, atau Mac Anda. Pelajari cara streaming musik dari cloud, mengatur perpustakaan yang ada di penyimpanan Anda sendiri, mengelola daftar putar dan folder offline, menyetel mesin audio, dan mendengarkan audiobook. Evermusic memberi Anda kendali penuh atas koleksi musik Anda dalam satu aplikasi yang mudah.<br><br>

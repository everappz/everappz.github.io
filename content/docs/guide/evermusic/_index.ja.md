Evermusicは、iPhone、iPad、またはMacをフル機能のストリーミングおよびオフライン音楽システムに変える強力なクラウド音楽プレーヤーとライブラリマネージャーです。すでにお持ちのクラウドアカウントとNASで動作し、デバイスがデコードできるほぼすべてのオーディオフォーマットを再生し、高度なオーディオエンジン（10バンドイコライザー、クロスフェード、ギャップレス、空間オーディオ、ピッチ補正）を提供します。

2つのエディションで利用可能：Evermusic（無料、広告あり）とEvermusic Pro（有料、広告なし、フル機能）。無料エディションの生涯、月次、年次プレミアムプランは同じ機能セットを解放し、両エディションともiCloudとFamily SharingでiOSとMac間で購入を共有します。

### どこでもあなたの音楽

Evermusicは、まるでSpotifyのような独自のクラウド音楽ストリーミングシステムを構築できます — ただし制限なしで。任意のソースの組み合わせを接続し、オンデマンドでストリーミングまたはダウンロード：

- **個人クラウドアカウント：** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘)
- **セルフホストサーバーとメディアライブラリ：** Plex · Jellyfin · Emby · Subsonic（すべてのSubsonic互換サーバー — Airsonic、Funkwhale、Gonic、Logitech Media Server、Ampache）· Navidrome · Nextcloud（共有APIによるOwnccloud）· Synology Drive · QNAP
- **NASとファイル共有プロトコル：** SMB (SMB1、SMB2、Auto) · WebDAV (HTTP / HTTPS) · FTP / FTPS · SFTP（SSHファイル転送プロトコル、パスワードまたは公開鍵認証）· NFS · DLNA / UPnP（再生とダウンロード）。Apple Time Capsule、Synology、QNAP、WD My Cloud Home、Buffalo、任意のLinux Samba / NFS / SSHホスト、またはMacやWindows PCの共有フォルダで動作。
- **S3互換オブジェクトストレージ：** AWS S3 · Backblaze B2 · Wasabi · Cloudflare R2 · MinIO · DigitalOcean Spaces · Linode Object Storage · IBM Cloud Object Storage · 任意のS3-APIエンドポイント
- **ローカルネットワーク検出：** 利用可能なデバイスセクションはWi-Fi上のすべてのBonjour / mDNSデバイスを自動リスト表示 — Plex、Jellyfin、Embyサーバー、Synology、QNAP、ドライブ接続済みAirPortルーター、Time Capsule — IPを入力せずにタップして接続できます。
- **デバイスと外部ソース：** iPod / Apple Musicライブラリ（非DRMトラック）、システムファイルアプリのファイル（他のアプリ、外部SSD、マウントされたフォルダ）、Apple認定カードリーダーとiXpand Flash DriveによるUSBフラッシュドライブ、Wi-Fi Drive（デスクトップブラウザでドラッグ&ドロップ）、USBケーブルでのiTunes / Finder ファイル共有

1つのアプリで30以上のソースタイプ。音楽がクラウドにあれば、すべてのデバイスで同じライブラリを使って、いつでもどこでも再生できます。バックアップと復元により、ライブラリ全体を1つのファイルにスナップショットし、数秒で別のiPhone、iPad、またはMacにインポートできます。

### スムーズな再生

Evermusicはスマートプリロードと設定可能な再生キャッシュを使用して、遅い接続でも曲をスムーズに再生します。スマートストリーミングはAVAssetResourceLoaderを使用するため、ファイルの残りがバックグラウンドでダウンロードされる間も再生が即座に開始されます。アルバム、アーティスト、プレイリスト、または個別の曲をダウンロードしてオフラインで聴くこともできます。ストレージが少なくなったら、キャッシュされたファイルを削除してクラウドからのストリーミングを続けるだけです。

### 簡単な音楽の並べ替え

Evermusicに曲を追加すると、バックグラウンドのメタデータリーダーが各ファイルを解析し、曲、未再生の曲、アルバム、アルバムアーティスト、アーティスト、ジャンル、作曲家でソートされたクリーンな音楽ライブラリを構築します。アルバムには3つのサブビューがあります：すべてのアルバム、エクスクルーシブアルバム、ソロアルバム。コンピレーションや異なるアーティストの同名アルバムが混在しないようにします。

### 曲情報の簡単な修正

曲のタイトルやアルバム情報が間違っていたり欠落していたりしても心配しないでください。Evermusicには、数秒でメタデータを修正できる組み込みタグエディターが含まれています。タイトル、アーティスト、アルバム、アルバムアーティスト、年、ジャンル、アルバムカバーを更新でき、ワンタップで壊れたエンコード（文字化けして表示されるキリル文字やアジア語テキスト）を正規化できます。エディターはMusicBrainzを使用して不足しているタグとアルバムアートワークを自動検索します。

### シンプルなファイル転送

コンピューターからiPhoneやiPadへの音楽送信は簡単です。EvermusicはSMB、WebDAV、FTP / FTPS、SFTP、NFS、DLNA、Wi-Fi Drive（ブラウザでのドラッグ&ドロップ）、iTunes / Finder ファイル共有（USBケーブル）、接続されたクラウドアカウントからの直接ダウンロードをサポートしています。Apple Time Capsule、WD My Cloud Home、Synology、QNAP、Buffalo、またはこれらの標準プロトコルのいずれかを公開する他のNASに、デバイスのストレージを使用せずに接続することもできます。

### 強力なサウンドコントロール

EvermusicにはiPodスタイルのプリセット（Acoustic、Bass Booster、Classical、Dance、Electronic、Flat、Hip-Hop、Jazz、Latin、Loudness、Lounge、Piano、Pop、R&B、Rock、Small Speakers、Spoken Word、Treble Booster、Vocal Booster）、静かなトラック用プリアンプ、エクスポート・インポート可能なカスタムプリセット、RMSベースのラウドネス計算を備えたフル10バンドオーディオイコライザーが含まれています。クロスフェード再生（1〜15秒）、ギャップレス再生、空間オーディオ、ピッチ補正、最大384 kHzモノまたはステレオのCoreAudio出力を追加すれば、好みのサウンドに正確に調整できます。

### オーディオブックに最適

オーディオブック、ポッドキャスト、または録音された講義を聴く場合、Evermusicは最適です。トラックごとに複数のブックマークを追加し、自動的に最後の位置から再開し、再生速度を0.02倍から3倍に変更（精密スライダーモード付き）、カスタム間隔でスリープタイマーを設定し、同じ画面でコメント、埋め込み歌詞、LRCファイルを表示できます。

### どこでも使える

スマートフォンやタブレットに限定されていません。Evermusicは、Apple CarPlay（iOSのみ、カーUI向けに最適化）、AirPlay 2（複数のスピーカーに同時ストリーミング）、Google Cast / Chromecast、ロック画面ウィジェット、Now Playing コントロール、Last.fmスクロブリング（統計と推薦のためにLast.fmアカウントに再生を自動送信）、Macキーボードショートカットでも動作します。自宅でも、車でも、旅行中でも、音楽はいつも一緒です。

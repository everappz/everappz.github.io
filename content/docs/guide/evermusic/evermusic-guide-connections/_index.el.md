Στην οθόνη Συνδέσεις μπορείτε να συνδέσετε κάθε πηγή που περιέχει τη μουσική σας — δημοφιλείς υπηρεσίες cloud, self-hosted media servers, το Mac ή PC σας, ένα προσωπικό NAS, Apple Time Capsule, WD My Cloud Home, ακόμα και USB flash drive — και να τα χρησιμοποιείτε όλα από μία ενιαία διεπαφή. Οι Συνδέσεις είναι επίσης εκεί όπου ρυθμίζετε τη Γρήγορη Πρόσβαση σε βαθιά ένθετους cloud φακέλους και όπου πιστοποιείτε το Last.fm για scrobbling.

Η οθόνη χωρίζεται σε σαφώς επισημασμένες ενότητες: Γρήγορη Πρόσβαση στην κορυφή (οι αγαπημένοι σας cloud φάκελοι), Cloud storage (οι λογαριασμοί που έχετε προσθέσει), Τοπικό δίκτυο (συσκευές που εντοπίστηκαν μέσω Bonjour), Υπολογιστής (Wi-Fi Drive, κοινή χρήση αρχείων iTunes, SMB), Εξωτερικά αξεσουάρ (συνδεδεμένες USB μονάδες flash) και Άλλες υπηρεσίες (Last.fm και παρόμοιες).

## Σύνδεση σε cloud storage

- Ανοίξτε την καρτέλα Συνδέσεις.
- Επιλέξτε 'Σύνδεση σε cloud storage' από το μενού.
- Επιλέξτε μια υπηρεσία cloud storage από τη λίστα.
- Συνδεθείτε στην επίσημη σελίδα εξουσιοδότησης του παρόχου (το Evermusic δεν βλέπει ποτέ τον κωδικό σας).
- Πατήστε 'Ολοκλήρωση'.

Αν αντιμετωπίσετε προβλήματα, ελέγξτε τη σύνδεσή σας στο διαδίκτυο και τα διαπιστευτήρια σύνδεσης, και βεβαιωθείτε ότι ο έλεγχος ταυτότητας δύο παραγόντων έχει ρυθμιστεί σωστά για αυτή την υπηρεσία.  
Στην Premium έκδοση της εφαρμογής μπορείτε να προσθέσετε απεριόριστο αριθμό υπηρεσιών. Οι δωρεάν χρήστες μπορούν να συνδέσουν έναν μόνο cloud λογαριασμό κάθε φορά.

## Υποστηριζόμενες υπηρεσίες cloud storage

Το Evermusic υποστηρίζει το πλήρες φάσμα δημοφιλών υπηρεσιών cloud και self-hosted. Οι δωρεάν χρήστες έχουν τον ίδιο κατάλογο παρόχων αλλά με τον περιορισμό ενός λογαριασμού· το Premium ξεκλειδώνει απεριόριστους λογαριασμούς.

- **Προσωπικοί cloud λογαριασμοί:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘).
- **Self-hosted servers και media libraries:** Plex · Jellyfin · Emby · Subsonic (και κάθε server συμβατός με Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud (και Owncloud μέσω κοινού API) · Synology Drive · QNAP.
- **NAS και πρωτόκολλα κοινής χρήσης αρχείων:** SMB (SMB1, SMB2, Auto), WebDAV (HTTP / HTTPS), FTP / FTPS, SFTP (κωδικός πρόσβασης ή έλεγχος ταυτότητας δημόσιου κλειδιού), NFS και DLNA (μόνο ανάγνωση — αναπαραγωγή και λήψη).
- **S3-compatible object storage:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces, Linode Object Storage, IBM Cloud Object Storage ή οποιοδήποτε S3-API endpoint.
- **Ανακάλυψη τοπικού δικτύου:** η ενότητα 'Διαθέσιμες συσκευές' παραθέτει αυτόματα τις συσκευές στο Wi-Fi σας που διαφημίζονται μέσω Bonjour / mDNS — Plex, Jellyfin, Emby servers, Synology, QNAP, WD My Cloud Home, Apple Time Capsule, AirPort routers κ.ά.

## Ασφάλεια και απόρρητο

Χρησιμοποιούμε μόνο επίσημο SDK και ασφαλή σύνδεση για αλληλεπίδραση με τις συνδεδεμένες υπηρεσίες cloud. Το login και ο κωδικός σας δεν είναι διαθέσιμα στην εφαρμογή. Όλα τα αιτήματα από την εφαρμογή προς την υπηρεσία cloud είναι κρυπτογραφημένα.

Όταν εισάγετε το login και τον κωδικό σας, η εφαρμογή εμφανίζει την επίσημη σελίδα εξουσιοδότησης που παρέχεται από τον πάροχο της υπηρεσίας cloud, και όλη η διαδικασία εξουσιοδότησης γίνεται εκτός εφαρμογής. Ο πάροχος της υπηρεσίας cloud στέλνει ένα auth token στην εφαρμογή μετά την επιτυχή εξουσιοδότηση, το οποίο χρησιμοποιείται για κλήσεις API.

Το auth token είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με το cloud storage. Αποθηκεύεται στη συσκευή σας σε ασφαλή αποθήκη συστήματος που ονομάζεται Keychain. Μπορείτε να κατεβάσετε αρχεία από τη συνδεδεμένη υπηρεσία cloud στη συσκευή σας, και αυτά τοποθετούνται στον κατάλογο 'Έγγραφα' της εφαρμογής. Μπορείτε να τα αφαιρέσετε ανά πάσα στιγμή μέσω του ενσωματωμένου διαχειριστή αρχείων.

Η εφαρμογή δεν μοιράζεται καμία πληροφορία από τον συνδεδεμένο cloud λογαριασμό. Μπορείτε να ανακαλέσετε την πρόσβαση ανά πάσα στιγμή από τη σελίδα ρυθμίσεων λογαριασμού στον web browser σας.

## Απόρριψη auth token

Συνδεθείτε στο λογαριασμό σας μέσω web browser και μεταβείτε στη σελίδα ρυθμίσεων. Εκεί μπορείτε να βρείτε όλες τις εφαρμογές τρίτων που είναι συνδεδεμένες με τον cloud λογαριασμό σας και να αφαιρέσετε όποια δεν θέλετε πλέον. Λεπτομερείς οδηγίες διαθέσιμες [εδώ](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account).

Μπορείτε επίσης να αποσυνδέσετε τους συνδεδεμένους cloud λογαριασμούς στην εφαρμογή και το auth token θα αφαιρεθεί και από τη συσκευή σας. Αν αφαιρέσετε την εφαρμογή από τη συσκευή σας, θα αφαιρεθούν και όλα τα ληφθέντα δεδομένα και τα access tokens.

## Αποσύνδεση cloud storage ή αλλαγή διαμόρφωσης

- Πρόσβαση στις Επιλογές Cloud Storage: εντοπίστε πρώτα το cloud storage που θέλετε να διαχειριστείτε στη διεπαφή της εφαρμογής.
- Πατήστε το κουμπί '...': δίπλα στον τίτλο της υπηρεσίας. Πατήστε το για πρόσβαση σε πρόσθετες επιλογές.
  - **Μετονομασία**: αν θέλετε να αλλάξετε το όνομα της υπηρεσίας cloud στη λίστα σας, επιλέξτε 'Μετονομασία'.
  - **Ρυθμίσεις**: για να τροποποιήσετε τη διαμόρφωση ή τα δεδομένα ελέγχου ταυτότητας για την υπηρεσία cloud, επιλέξτε 'Ρυθμίσεις'. Μερικές φορές θα χρειαστεί να επανεξουσιοδοτήσετε τη συνδεδεμένη υπηρεσία cloud αν έχει λήξει το auth token.
  - **Αποσύνδεση**: αν θέλετε να αποκόψετε εντελώς τη σύνδεση μεταξύ της εφαρμογής και της υπηρεσίας cloud, επιλέξτε 'Αποσύνδεση'. Σημειώστε ότι αυτό θα αφαιρέσει όλα τα τραγούδια που σχετίζονται με αυτή την υπηρεσία cloud από τη μουσική βιβλιοθήκη της εφαρμογής, αλλά θα παραμείνουν στον server.

## Σύνδεση σε Υπολογιστή ή NAS

Μπορείτε επίσης να συνδέσετε τον υπολογιστή σας, ένα προσωπικό NAS ή άλλες δικτυακές συσκευές χρησιμοποιώντας SMB, DLNA ή WebDAV.

## Σύνδεση σε υπολογιστή με χρήση SMB

- Πατήστε 'Σύνδεση σε cloud storage' → SMB.
- Εισάγετε τη διεύθυνση IP του υπολογιστή και το όνομα του κοινόχρηστου φακέλου στο πεδίο URL με τη μορφή smb://διεύθυνση-ip-υπολογιστή/όνομα-κοινόχρηστου-φακέλου
- Επιλέξτε έκδοση πρωτοκόλλου: Auto, SMB1, SMB2
- Εισάγετε login και κωδικό πρόσβασης (αν απαιτείται)
- Πατήστε 'Ολοκλήρωση'.

Αν η σύνδεση είναι επιτυχής, θα δείτε τον συνδεδεμένο χώρο αποθήκευσης στην ενότητα 'Cloud storage'.  
Ένας πλήρης οδηγός για τη σύνδεση Mac ή PC μέσω SMB είναι διαθέσιμος [εδώ](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).

## Wi-Fi Drive

Το Wi-Fi Drive είναι μια βολική τεχνολογία που επιτρέπει ασύρματες μεταφορές αρχείων από τον υπολογιστή σας στη συσκευή iOS μέσω ενός desktop browser.  
Για να χρησιμοποιήσετε αποτελεσματικά αυτή τη λειτουργία, βεβαιωθείτε ότι η συσκευή σας και ο υπολογιστής σας είναι συνδεδεμένα στο ίδιο Wi-Fi δίκτυο.  
Ακολουθεί οδηγός βήμα προς βήμα για τη χρήση του Wi-Fi Drive.

## Ενεργοποίηση Wi-Fi Drive

- Ανοίξτε την εφαρμογή και μεταβείτε στην καρτέλα 'Συνδέσεις'.
- Επιλέξτε 'Σύνδεση μέσω Wi-Fi' για πρόσβαση στην κύρια οθόνη Wi-Fi Drive.
- Πατήστε 'Εκκίνηση Wi-Fi Drive' για ενεργοποίηση.

## Πρόσβαση στο Wi-Fi Drive από τον Υπολογιστή σας

- Στον υπολογιστή σας (desktop ή laptop), ανοίξτε έναν web browser (όπως Chrome, Firefox ή Safari).
- Στη γραμμή διευθύνσεων του browser, εισάγετε το URL που παρέχεται από την εφαρμογή.

## Ασύρματη Μεταφορά Αρχείων

Μόλις ανοίξει η ιστοσελίδα που αντιστοιχεί στη συσκευή iOS στον browser, μπορείτε εύκολα να σύρετε και να αποθέσετε αρχεία από τον υπολογιστή σας στην ιστοσελίδα.  
Τα αρχεία που σύρετε και αποθέτετε θα μεταφερθούν στη συσκευή iOS και θα είναι προσβάσιμα εντός της εφαρμογής.

## Κοινή Χρήση Αρχείων iTunes

Η κοινή χρήση αρχείων iTunes είναι μια άλλη τεχνολογία που σας επιτρέπει να μεταφέρετε αρχεία από υπολογιστή σε συσκευή χρησιμοποιώντας την εφαρμογή Finder στο Mac σας και καλώδιο Lightning.  
- Απλώς συνδέστε μια συσκευή στον υπολογιστή χρησιμοποιώντας καλώδιο και εκκινήστε την εφαρμογή Finder στο Mac σας.
- Ανοίξτε 'Τοποθεσίες' → 'Η Συνδεδεμένη Συσκευή σας' → 'Αρχεία' → και βρείτε την εφαρμογή Evermusic.
- Πατήστε στο εικονίδιο της εφαρμογής για να δείτε όλους τους κοινόχρηστους φακέλους.
- Αντιγράψτε αρχεία από τον υπολογιστή στον κοινόχρηστο φάκελο στη συσκευή χρησιμοποιώντας drag-and-drop.

Λεπτομερείς οδηγίες για τη χρήση κοινής χρήσης αρχείων iTunes είναι διαθέσιμες [εδώ](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).

## Σύνδεση USB flashcard

Αν έχετε κάρτα SD, μπορείτε να τη συνδέσετε χρησιμοποιώντας αναγνώστη καρτών lightning. Η εφαρμογή υποστηρίζει επί του παρόντος Apple Certified αναγνώστες καρτών και iXpand Flash Drives. Αν έχετε iXpand Flash Drive — εισάγετέ το στη θύρα lightning και ανοίξτε την εφαρμογή. Θα δείτε μήνυμα 'Συνδέθηκε εξωτερική συσκευή' και πληροφορίες συσκευής. Απλώς πατήστε στο εικονίδιο flash drive για πρόσβαση στον φάκελο μουσικής και πατήστε σε οποιοδήποτε αρχείο ήχου για να ξεκινήσει η αναπαραγωγή. Λεπτομερείς οδηγίες για σύνδεση USB flashcard στο iPhone είναι διαθέσιμες [εδώ](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).

## Διαχειριστής Αρχείων

Μόλις συνδέσετε μια υπηρεσία cloud storage, πατήστε στο εικονίδιό της για να δείτε όλα τα διαθέσιμα αρχεία και φακέλους. Μπορείτε να χρησιμοποιήσετε τον ενσωματωμένο διαχειριστή αρχείων για να εργαστείτε με αυτά — λήψη, μετονομασία, μετακίνηση και άλλα. Όταν ξεκινάτε λήψη, το αρχείο εμφανίζεται στην ουρά μεταφοράς. Για να δείτε την ουρά μεταφοράς, μεταβείτε στην καρτέλα 'Τοπικά Αρχεία' και πατήστε τα περιστρεφόμενα βέλη στην επάνω αριστερή γωνία. Όλα τα ληφθέντα αρχεία και φάκελοι είναι διαθέσιμα στην ενότητα 'Τοπικά Αρχεία'.

## Επάνω Γραμμή Εργαλείων

Η επάνω γραμμή εργαλείων, που βρίσκεται κάτω από τη γραμμή πλοήγησης, προσφέρει πολλές χρήσιμες ενέργειες. Μπορείτε να την εμφανίσετε ή να την αποκρύψετε με χειρονομία σάρωσης προς τα κάτω. Ακολουθούν οι διαθέσιμες ενέργειες:

- **Αναζήτηση**: Αυτή η επιλογή σας επιτρέπει να κάνετε γρήγορη αναζήτηση εντός του τρέχοντος καταλόγου.
- **Συνέχεια Αναπαραγωγής**: Αν είναι ενεργοποιημένη στις ρυθμίσεις εφαρμογής, αυτή η λειτουργία επαναφέρει την ουρά audio player και την τελευταία θέση μέσου για τον τρέχοντα φάκελο.
- **Αναπαραγωγή όλων**: Με αυτή την ενέργεια, η εφαρμογή σαρώνει τον τρέχοντα φάκελο και τους υποφακέλους, προσθέτοντας όλα τα αρχεία ήχου σε νέα ουρά player.
- **Ανακάτεμα όλων**: Παρόμοιο με 'Αναπαραγωγή όλων', αλλά ανακατεύει τα αρχεία πριν τα προσθέσει στην ουρά audio player.

## Επιλογές Φακέλου

Όταν ανοίγετε έναν φάκελο στην εφαρμογή, θα βρείτε ένα σύνολο ενεργειών πατώντας το κουμπί '...' στην επάνω δεξιά γωνία της οθόνης.  
Ακολουθεί ανάλυση αυτών των ενεργειών:

- **Επιλογή**: Ενεργοποιήστε τη λειτουργία επιλογής αρχείων. Αυτή η λειτουργία σας επιτρέπει να επιλέξετε ένα ή περισσότερα αρχεία εντός του φακέλου.
- **Νέος Φάκελος**: Δημιουργήστε νέο φάκελο εντός του τρέχοντος φακέλου.
- **Ενεργοποίηση εκτός σύνδεσης λειτουργίας**: Ενεργοποιήστε τη λειτουργία εκτός σύνδεσης για τον τρέχοντα φάκελο. Παρακολουθεί ενεργά τον φάκελο για αλλαγές. Αν προσθέσετε νέα αρχεία online, η εφαρμογή τα κατεβάζει αυτόματα στη συσκευή σας.
- **Μεταφόρτωση Αρχείων**: Μεταφορτώστε αρχεία από τη συσκευή σας στον online φάκελο.
- **Αναζήτηση**: Αναζητήστε συγκεκριμένα αρχεία εντός του τρέχοντος φακέλου.
- **Ταξινόμηση**: Ταξινομήστε αρχεία κατά όνομα, μέγεθος ή ημερομηνία τροποποίησης.
- **Προβολή Πλέγματος/Λίστας**: Εναλλαγή μεταξύ προβολής πίνακα και προβολής μικρογραφιών.

## Επεξεργασία Online Αρχείων

Όταν χρειάζεστε να διαχειριστείτε πολλά αρχεία στο cloud storage σας στο Evermusic, μπορείτε να χρησιμοποιήσετε τη λειτουργία επιλογής. Ακολουθήστε τα εξής βήματα:

- **Ενεργοποίηση Λειτουργίας Επιλογής**: Ανοίξτε την εφαρμογή και μεταβείτε στην ενότητα με το cloud storage σας. Αναζητήστε το κουμπί '...' (αποσιώπηση) στην επάνω δεξιά γωνία. Πατήστε το και επιλέξτε 'Επιλογή' για ενεργοποίηση λειτουργίας επιλογής.
- **Επιλογή Αρχείων**: Θα εμφανιστούν πλαίσια ελέγχου δίπλα σε κάθε αρχείο ή φάκελο. Επιλέξτε ένα ή περισσότερα πατώντας τα πλαίσια.
- **Εκτέλεση Διαφόρων Ενεργειών**: Μόλις επιλέξετε τα αρχεία ή φακέλους που θέλετε να διαχειριστείτε, θα έχετε πρόσβαση σε διάφορες ενέργειες.

## Ενέργειες αρχείου

Κοντά στον τίτλο αρχείου θα παρατηρήσετε το σύμβολο αποσιώπησης '...' (τρεις τελείες) — αυτό είναι το μενού ενεργειών.  
Πατήστε το για να αποκαλύψετε τη λίστα διαθέσιμων ενεργειών:

- **Αναπαραγωγή επόμενου**: Εισάγει το επιλεγμένο αρχείο στην κορυφή της ουράς player σας.
- **Αναπαραγωγή αργότερα**: Αυτή η επιλογή προσθέτει το αρχείο στο κάτω μέρος της ουράς player σας.
- **Προσθήκη στη Μουσική Βιβλιοθήκη**: Αυτή η ενέργεια σας επιτρέπει να ενσωματώσετε το αρχείο στη μουσική βιβλιοθήκη σας.
- **Προσθήκη σε Λίστα Αναπαραγωγής**: Χρησιμοποιήστε αυτή την ενέργεια για να προσθέσετε το αρχείο σε υπάρχουσα λίστα αναπαραγωγής ή να δημιουργήσετε νέα.
- **Επεξεργασία ετικετών ήχου**: Με αυτή την επιλογή, μπορείτε να αποκτήσετε πρόσβαση στον ενσωματωμένο επεξεργαστή tags του Evermusic. Το αρχείο κατεβαίνει προσωρινά και στη συνέχεια μεταφορτώνεται αφού επιβεβαιώσετε τις αλλαγές.
- **Προσθήκη στα Αγαπημένα**: Αυτή η ενέργεια προσθέτει το αρχείο στη λίστα αγαπημένων αρχείων σας.
- **Λήψη**: Επιλέξτε αυτή την ενέργεια για να κατεβάσετε το αρχείο ή τον φάκελο στη συσκευή σας.
- **Μετονομασία**: Αυτή η επιλογή σας επιτρέπει να μετονομάσετε το αρχείο απευθείας στον απομακρυσμένο χώρο αποθήκευσης.
- **Μετακίνηση**: Επιλέξτε αυτή την ενέργεια για να μετακινήσετε το αρχείο σε διαφορετικό φάκελο στο cloud storage σας.
- **Άνοιγμα με**: Χρησιμοποιήστε αυτή την ενέργεια για να εξάγετε το αρχείο σε άλλη συμβατή εφαρμογή.
- **Διαγραφή**: Να είστε προσεκτικοί με αυτή την ενέργεια, καθώς αφαιρεί μόνιμα το αρχείο από το cloud storage σας. Αυτή η διαγραφή δεν μπορεί να αναιρεθεί.

## Ενέργειες φακέλου

Για κάθε φάκελο στο cloud storage σας, υπάρχουν διάφορες διαθέσιμες ενέργειες. Για να αποκτήσετε πρόσβαση στις επιλογές, πατήστε στο εικονίδιο αποσιώπησης '...' δίπλα στον τίτλο φακέλου. Ακολουθούν οι διαθέσιμες ενέργειες:

- **Αναπαραγωγή όλων**: Αντικαταστήστε την τρέχουσα ουρά player με όλα τα στοιχεία από τον επιλεγμένο φάκελο.
- **Αναπαραγωγή επόμενου**: Προσθέστε ολόκληρο τον φάκελο στην κορυφή της ουράς player.
- **Αναπαραγωγή αργότερα**: Προσαρτήστε τα περιεχόμενα του φακέλου στο κάτω μέρος της ουράς player.
- **Προσθήκη στη Μουσική Βιβλιοθήκη**: Αυτή η ενέργεια ενσωματώνει τα περιεχόμενα του φακέλου στη μουσική βιβλιοθήκη σας.
- **Προσθήκη σε Λίστα Αναπαραγωγής**: Μπορείτε να συμπεριλάβετε ολόκληρο τον φάκελο σε λίστα αναπαραγωγής. Το όνομα φακέλου θα αποδοθεί αυτόματα.
- **Προσθήκη στα Αγαπημένα**: Χρησιμοποιήστε αυτή την ενέργεια για να προσθέσετε τον φάκελο στη λίστα αγαπημένων αρχείων σας.
- **Ενεργοποίηση εκτός σύνδεσης λειτουργίας**: Σαρώνει συνεχώς για αλλαγές και τα νέα αρχεία κατεβαίνουν αυτόματα.
- **Λήψη**: Κατεβάστε όλα τα περιεχόμενα του φακέλου στη συσκευή σας για offline πρόσβαση.
- **Μετονομασία**: Μετονομάστε τον φάκελο απευθείας στον απομακρυσμένο χώρο αποθήκευσης.
- **Μετακίνηση**: Μετακινήστε τον φάκελο σε διαφορετική τοποθεσία στο cloud storage σας.
- **Διαγραφή**: Να είστε προσεκτικοί καθώς αυτή η ενέργεια αφαιρεί μόνιμα τον φάκελο και τα περιεχόμενά του. Δεν μπορεί να αναιρεθεί.


## Γρήγορη Πρόσβαση

Η ενότητα Γρήγορης Πρόσβασης βρίσκεται στην κορυφή της οθόνης. Σας δίνει γρήγορη πρόσβαση στους αγαπημένους σας φακέλους και πρόσφατα ανοιγμένα αρχεία από συνδεδεμένες υπηρεσίες cloud.
Κάθε φορά που ανοίγετε ένα αρχείο ή φάκελο από το cloud, προστίθεται στη λίστα 'Πρόσφατα Ανοιγμένα'. Για να διαγράψετε αυτή τη λίστα, ανοίξτε 'Πρόσφατα', πατήστε το κουμπί 'Περισσότερες Ενέργειες' και επιλέξτε 'Διαγραφή Λίστας'. Μπορείτε επίσης να σημειώσετε βαθιά ένθετους φακέλους ως Αγαπημένα για γρήγορη πρόσβαση.

## Άλλες Υπηρεσίες

Αυτή η ενότητα εμφανίζει επιπλέον λειτουργίες. Προς το παρόν, η εφαρμογή υποστηρίζει scrobbling Last.fm. Όταν συνδεθείτε, τα στατιστικά αναπαραγωγής σας αποστέλλονται αυτόματα στον λογαριασμό σας Last.fm. Μπορείτε να επισκεφθείτε το προφίλ σας αργότερα για να δείτε αναλυτικά στοιχεία ακρόασης και να λάβετε εξατομικευμένες συστάσεις μουσικής. Λεπτομερείς οδηγίες ρύθμισης είναι διαθέσιμες [εδώ](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm).

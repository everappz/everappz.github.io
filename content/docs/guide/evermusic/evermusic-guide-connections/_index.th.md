ในหน้าจอ การเชื่อมต่อ คุณสามารถเชื่อมต่อทุกแหล่งที่มีเพลงของคุณ — บริการคลาวด์ยอดนิยม เซิร์ฟเวอร์มีเดียที่โฮสต์เอง Mac หรือ PC ของคุณ NAS ส่วนตัว Apple Time Capsule, WD My Cloud Home แม้แต่ USB flash drive — และใช้งานทั้งหมดจากอินเทอร์เฟซเดียว การเชื่อมต่อยังเป็นที่ที่คุณตั้งค่าการเข้าถึงด่วนไปยังโฟลเดอร์คลาวด์ที่ซ้อนกันลึก และตรวจสอบสิทธิ์ Last.fm สำหรับการ scrobbling

หน้าจอแบ่งออกเป็นส่วนที่มีป้ายชัดเจน เพื่อให้รองรับตั้งแต่บัญชี iCloud Drive เดียวไปจนถึงไลบรารีที่กระจายอยู่ในหลายคลาวด์และอุปกรณ์ NAS: การเข้าถึงด่วนที่ด้านบน (โฟลเดอร์คลาวด์โปรดของคุณ), ที่เก็บข้อมูลคลาวด์ (บัญชีที่คุณเพิ่มไว้), เครือข่ายท้องถิ่น (อุปกรณ์ที่ค้นพบผ่าน Bonjour), คอมพิวเตอร์ (Wi-Fi Drive, iTunes File Sharing, SMB), อุปกรณ์ที่เชื่อมต่อ (USB flash drive ที่เชื่อมต่อ) และบริการอื่นๆ (Last.fm และที่คล้ายกัน)

## เชื่อมต่อกับที่เก็บข้อมูลคลาวด์

- เปิดแท็บ การเชื่อมต่อ
- เลือก เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ จากเมนู
- เลือกบริการที่เก็บข้อมูลคลาวด์จากรายการ
- ลงชื่อเข้าใช้บนหน้าการอนุมัติอย่างเป็นทางการของผู้ให้บริการ (Evermusic ไม่เคยเห็นรหัสผ่านของคุณ)
- แตะ เสร็จสิ้น

หากพบปัญหา ให้ตรวจสอบการเชื่อมต่ออินเทอร์เน็ตและข้อมูลรับรองการเข้าสู่ระบบ และตรวจสอบให้แน่ใจว่าการยืนยันตัวตนสองปัจจัยได้รับการกำหนดค่าอย่างถูกต้องสำหรับบริการนั้น  
ในเวอร์ชัน Premium ของแอป คุณสามารถเพิ่มบริการได้ไม่จำกัด ผู้ใช้ฟรีสามารถเชื่อมต่อบัญชีคลาวด์ได้ครั้งละหนึ่งบัญชี

## บริการที่เก็บข้อมูลคลาวด์ที่รองรับ

Evermusic รองรับบริการคลาวด์และที่โฮสต์เองยอดนิยมครบถ้วน ผู้ใช้ฟรีได้รับแคตตาล็อกผู้ให้บริการเดียวกัน แต่จำกัดหนึ่งบัญชี Premium จะปลดล็อกบัญชีไม่จำกัดและยกเลิกข้อจำกัดส่วนใหญ่

- **บัญชีคลาวด์ส่วนตัว:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · Yandex.Disk · pCloud · MediaFire · WD My Cloud Home · InfiniCLOUD (TeraCLOUD) · HiDrive · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Icedrive · Koofr · Proton Drive · Internxt · AliDrive (阿里云盘) · Baidu Pan (百度网盘)
- **เซิร์ฟเวอร์ที่โฮสต์เองและไลบรารีมีเดีย:** Plex · Jellyfin · Emby · Subsonic (และทุกเซิร์ฟเวอร์ที่รองรับ Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud (และ Owncloud ผ่าน shared API) · Synology Drive · QNAP
- **NAS และโปรโตคอลแชร์ไฟล์:** SMB (SMB1, SMB2, Auto), WebDAV (HTTP / HTTPS), FTP / FTPS, SFTP (รหัสผ่านหรือการยืนยันตัวตนด้วยกุญแจสาธารณะ), NFS และ DLNA (อ่านอย่างเดียว — เล่นและดาวน์โหลด)
- **Object storage ที่รองรับ S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces, Linode Object Storage, IBM Cloud Object Storage หรือ S3-API endpoint ใดก็ได้
- **การค้นพบในเครือข่ายท้องถิ่น:** ส่วน อุปกรณ์ที่มีให้บริการ จะแสดงรายการอุปกรณ์ทุกเครื่องใน Wi-Fi ของคุณที่ประกาศตัวเองผ่าน Bonjour / mDNS โดยอัตโนมัติ — เซิร์ฟเวอร์ Plex, Jellyfin, Emby, Synology, QNAP, WD My Cloud Home, Apple Time Capsule, เราเตอร์ AirPort ที่มีไดรฟ์เชื่อมต่อ เป็นต้น

## ความปลอดภัยและความเป็นส่วนตัว

เราใช้เฉพาะ SDK อย่างเป็นทางการและการเชื่อมต่อที่ปลอดภัยในการโต้ตอบกับบริการคลาวด์ที่เชื่อมต่อ ชื่อผู้ใช้และรหัสผ่านของคุณไม่สามารถเข้าถึงได้โดยแอปพลิเคชัน คำขอทั้งหมดจากแอปพลิเคชันไปยังบริการคลาวด์จะถูกเข้ารหัส

เมื่อคุณป้อนชื่อผู้ใช้และรหัสผ่าน แอปพลิเคชันจะแสดงหน้าการอนุมัติอย่างเป็นทางการที่ผู้ให้บริการคลาวด์จัดให้ และกระบวนการการอนุมัติทั้งหมดเกิดขึ้นนอกแอปพลิเคชัน ผู้ให้บริการคลาวด์จะส่ง auth token ไปยังแอปพลิเคชันหลังการอนุมัติสำเร็จ และ token นั้นจะถูกใช้เพื่อเรียก API

Auth-token คือกุญแจดิจิทัลที่อนุญาตให้แอปพลิเคชันของบุคคลที่สามโต้ตอบกับที่เก็บข้อมูลคลาวด์ Auth-token จะถูกเก็บไว้บนอุปกรณ์ของคุณในระบบจัดเก็บข้อมูลที่ปลอดภัยที่เรียกว่า Keychain คุณสามารถดาวน์โหลดไฟล์จากบริการคลาวด์ที่เชื่อมต่อไปยังอุปกรณ์ และไฟล์เหล่านั้นจะถูกวางไว้ในไดเรกทอรี "Documents" ของแอป คุณสามารถลบไฟล์เหล่านั้นได้ตลอดเวลาโดยใช้ตัวจัดการไฟล์ในตัว

แอปพลิเคชันไม่แชร์ข้อมูลใดๆ จากบัญชีคลาวด์ที่เชื่อมต่อ คุณสามารถเพิกถอนการเข้าถึงบัญชีคลาวด์ของคุณได้ตลอดเวลาโดยเปิดหน้าการตั้งค่าบัญชีในเว็บเบราว์เซอร์

## ปฏิเสธ auth-token

เข้าสู่ระบบบัญชีของคุณในเว็บเบราว์เซอร์และไปที่หน้าการตั้งค่า ที่นั่นคุณจะพบแอปของบุคคลที่สามทั้งหมดที่เชื่อมต่อกับบัญชีคลาวด์ของคุณ และสามารถลบออกได้หากไม่ต้องการใช้แอปพลิเคชันนั้นอีกต่อไป มีคำแนะนำโดยละเอียด [ที่นี่](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account)

คุณยังสามารถตัดการเชื่อมต่อบัญชีคลาวด์ที่เชื่อมต่อในแอปพลิเคชัน และ auth token จะถูกลบออกจากอุปกรณ์ของคุณด้วย หากคุณลบแอปพลิเคชันออกจากอุปกรณ์ ข้อมูลที่ดาวน์โหลดทั้งหมดและ access token จะถูกลบด้วย

## ตัดการเชื่อมต่อที่เก็บข้อมูลคลาวด์หรือเปลี่ยนการกำหนดค่า

- เข้าถึงตัวเลือกที่เก็บข้อมูลคลาวด์: ขั้นแรก ค้นหาที่เก็บข้อมูลคลาวด์ที่คุณต้องการจัดการภายในอินเทอร์เฟซของแอป
- แตะปุ่ม '...': ถัดจากชื่อบริการ คุณจะเห็นปุ่ม '...' แตะเพื่อเข้าถึงตัวเลือกเพิ่มเติม
  - **เปลี่ยนชื่อ**: หากต้องการเปลี่ยนชื่อบริการคลาวด์ที่แสดงในรายการ ให้เลือก 'เปลี่ยนชื่อ'
  - **การตั้งค่า**: หากต้องการแก้ไขการกำหนดค่าหรือข้อมูลการยืนยันตัวตนของบริการคลาวด์ ให้เลือก 'การตั้งค่า' บางครั้งคุณอาจต้องอนุมัติบริการคลาวด์ที่เชื่อมต่อใหม่หาก authorization token หมดอายุ
  - **ตัดการเชื่อมต่อ**: หากต้องการตัดการเชื่อมต่อระหว่างแอปและบริการคลาวด์ทั้งหมด ให้เลือก 'ตัดการเชื่อมต่อ' โปรดทราบว่าการเลือกตัวเลือกนี้จะลบเพลงทั้งหมดที่เกี่ยวข้องกับบริการคลาวด์นี้ออกจากไลบรารีเพลงของแอป แต่เพลงจะยังคงอยู่บนเซิร์ฟเวอร์

## เชื่อมต่อกับคอมพิวเตอร์หรือ NAS

คุณยังสามารถเชื่อมต่อคอมพิวเตอร์ NAS ส่วนตัว หรืออุปกรณ์เครือข่ายอื่นๆ โดยใช้โปรโตคอล SMB, DLNA หรือ WebDAV

## เชื่อมต่อกับคอมพิวเตอร์โดยใช้ SMB

- แตะ "เชื่อมต่อกับที่เก็บข้อมูลคลาวด์" → SMB
- ป้อนที่อยู่ IP ของคอมพิวเตอร์และชื่อโฟลเดอร์ที่แชร์ในช่อง URL โดยใช้รูปแบบ smb://computer-ip-address/shared-folder-name
- เลือกเวอร์ชันโปรโตคอล: Auto, SMB1, SMB2
- ป้อนชื่อผู้ใช้และรหัสผ่าน (หากต้องการ)
- แตะ "เสร็จสิ้น"

หากการเชื่อมต่อสำเร็จ คุณจะเห็นที่เก็บข้อมูลที่เชื่อมต่อในส่วน "ที่เก็บข้อมูลคลาวด์"  
บทช่วยสอนเต็มรูปแบบเกี่ยวกับวิธีเชื่อมต่อ MAC หรือ PC โดยใช้ SMB มีอยู่ [ที่นี่](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)

## อุปกรณ์ที่มีให้บริการ

ส่วนนี้แสดงอุปกรณ์ทั้งหมดในเครือข่ายท้องถิ่นของคุณที่คุณสามารถเชื่อมต่อผ่านแอปพลิเคชัน  
ทำตามขั้นตอนเหล่านี้เพื่อสร้างการเชื่อมต่อกับอุปกรณ์:

- เปิดแอปและไปที่ส่วน "อุปกรณ์ที่มีให้บริการ"
- แตะอุปกรณ์ที่คุณต้องการเชื่อมต่อจากรายการ
- หากจำเป็น ให้ป้อนรายละเอียดการเข้าสู่ระบบเพื่อดำเนินการเชื่อมต่อให้สมบูรณ์

## Wi-Fi Drive

Wi-Fi Drive เป็นเทคโนโลยีที่สะดวกซึ่งช่วยให้การโอนไฟล์แบบไร้สายจากคอมพิวเตอร์ไปยังอุปกรณ์ iOS ของคุณผ่านเว็บเบราว์เซอร์บนเดสก์ท็อป  
เพื่อใช้คุณสมบัตินี้อย่างมีประสิทธิภาพ ตรวจสอบให้แน่ใจว่าอุปกรณ์และคอมพิวเตอร์ของคุณเชื่อมต่อกับเครือข่าย Wi-Fi เดียวกัน  
นี่คือคู่มือทีละขั้นตอนในการใช้ Wi-Fi Drive

## เปิดใช้งาน Wi-Fi Drive

- เปิดแอปพลิเคชันและไปที่แท็บ "การเชื่อมต่อ"
- เลือก "เชื่อมต่อผ่าน Wi-Fi" เพื่อเข้าถึงหน้าจอหลักของ Wi-Fi Drive
- แตะ "เริ่ม Wi-Fi Drive" เพื่อเปิดใช้งาน Wi-Fi Drive

## เข้าถึง Wi-Fi Drive บนคอมพิวเตอร์ของคุณ

- บนคอมพิวเตอร์ของคุณ (เดสก์ท็อปหรือแล็ปท็อป) เปิดเว็บเบราว์เซอร์ (เช่น Chrome, Firefox หรือ Safari)
- ในแถบที่อยู่ของเบราว์เซอร์ ป้อน URL ที่แอปพลิเคชันให้มา

## ตัวจัดการไฟล์

เมื่อคุณเชื่อมต่อบริการที่เก็บข้อมูลคลาวด์แล้ว แตะที่ไอคอนเพื่อดูไฟล์และโฟลเดอร์ที่มีทั้งหมด คุณสามารถใช้ตัวจัดการไฟล์ในตัวในการทำงานกับไฟล์เหล่านี้ — ดาวน์โหลด เปลี่ยนชื่อ ย้าย และอื่นๆ เมื่อคุณเริ่มการดาวน์โหลด ไฟล์จะปรากฏในคิวการโอน หากต้องการดูคิวการโอน ไปที่แท็บ "ไฟล์ในเครื่อง" และแตะที่ลูกศรหมุนที่มุมบนซ้าย ไฟล์และโฟลเดอร์ที่ดาวน์โหลดทั้งหมดมีอยู่ในส่วน "ไฟล์ในเครื่อง"

## แถบเครื่องมือด้านบน

แถบเครื่องมือด้านบนที่อยู่ใต้แถบการนำทางอย่างสะดวก มีการดำเนินการที่มีประโยชน์หลายอย่างสำหรับการเข้าถึงที่ง่าย คุณสามารถแสดงหรือซ่อนแถบเครื่องมือนี้ได้โดยใช้ท่าทางปัดลง นี่คือการดำเนินการที่มีอยู่:

- **ค้นหา**: ตัวเลือกนี้ช่วยให้คุณค้นหาอย่างรวดเร็วในไดเรกทอรีปัจจุบัน ทำให้ง่ายต่อการค้นหาไฟล์หรือโฟลเดอร์เฉพาะ
- **เล่นต่อ**: หากเปิดใช้งานในการตั้งค่าแอปพลิเคชัน คุณสมบัตินี้จะกู้คืนคิวเครื่องเล่นเสียงและตำแหน่งมีเดียล่าสุดสำหรับโฟลเดอร์ปัจจุบัน เป็นวิธีที่สะดวกในการเล่นต่อจากที่คุณค้างไว้ในไลบรารีเพลงของคุณ
- **เล่นทั้งหมด**: การเลือกการดำเนินการนี้ แอปจะสแกนโฟลเดอร์ปัจจุบันและโฟลเดอร์ย่อยของมัน โดยเพิ่มไฟล์เสียงทั้งหมดที่พบลงในคิวเครื่องเล่นใหม่ ซึ่งมีประโยชน์เมื่อคุณต้องการเล่นเพลงทั้งหมดในไดเรกทอรีเฉพาะ
- **สับเปลี่ยนทั้งหมด**: คล้ายกับ "เล่นทั้งหมด" การดำเนินการนี้จะสแกนโฟลเดอร์ปัจจุบันและโฟลเดอร์ย่อย แต่จะสับเปลี่ยนไฟล์ก่อนเพิ่มลงในคิวเครื่องเล่นเสียง เป็นวิธีที่ยอดเยี่ยมในการเพลิดเพลินกับเพลงในลำดับสุ่มเพื่อความหลากหลาย

## ตัวเลือกโฟลเดอร์

เมื่อคุณเปิดโฟลเดอร์ในแอป คุณจะพบชุดการดำเนินการที่มีประโยชน์โดยแตะปุ่ม "..." ที่มุมบนขวาของหน้าจอ  
นี่คือรายละเอียดของการดำเนินการเหล่านี้:

- **เลือก**: เปิดใช้งานโหมดการเลือกไฟล์ โหมดนี้ช่วยให้คุณเลือกไฟล์หนึ่งหรือหลายไฟล์ภายในโฟลเดอร์ ทำให้ง่ายต่อการดำเนินการกับรายการที่เลือก
- **โฟลเดอร์ใหม่**: สร้างโฟลเดอร์ใหม่ภายในโฟลเดอร์ปัจจุบัน คุณสมบัตินี้ช่วยให้คุณจัดระเบียบไฟล์และรักษาไลบรารีให้มีโครงสร้างที่ดี
- **เปิดโหมดออฟไลน์**: เปิดใช้งานโหมดออฟไลน์สำหรับโฟลเดอร์ปัจจุบัน โหมดออฟไลน์ไปไกลกว่าการดาวน์โหลดธรรมดา โดยจะตรวจสอบการเปลี่ยนแปลงของโฟลเดอร์อย่างต่อเนื่อง หากคุณเพิ่มไฟล์ใหม่ลงในโฟลเดอร์ออนไลน์ แอปจะดาวน์โหลดไฟล์เหล่านั้นไปยังอุปกรณ์ของคุณโดยอัตโนมัติ ซึ่งทำให้ไลบรารีท้องถิ่นของคุณอัปเดตตามการเปลี่ยนแปลงบนเซิร์ฟเวอร์
- **อัปโหลดไฟล์**: อัปโหลดไฟล์จากอุปกรณ์ของคุณไปยังโฟลเดอร์ออนไลน์ การดำเนินการนี้ช่วยให้คุณโอนไฟล์ไปยังคลาวด์หรือเซิร์ฟเวอร์ ทำให้สามารถเข้าถึงได้จากทุกที่
- **ค้นหา**: ค้นหาไฟล์เฉพาะภายในโฟลเดอร์ปัจจุบัน ซึ่งมีประโยชน์อย่างยิ่งในการค้นหารายการเฉพาะในคอลเลกชันขนาดใหญ่อย่างรวดเร็ว
- **จัดเรียง**: จัดเรียงไฟล์ในโฟลเดอร์ตามเกณฑ์เช่น ชื่อ ขนาด หรือวันที่แก้ไข โหมดการจัดเรียงเริ่มต้นมักจะสะท้อนลำดับการจัดเรียงบนเซิร์ฟเวอร์ แต่คุณสามารถเปลี่ยนได้ตามความต้องการ
- **มุมมองกริด/รายการ**: สลับระหว่างโหมดการดูสองแบบ: มุมมองตารางและมุมมองภาพขนาดย่อ มุมมองตารางนำเสนอไฟล์ในรูปแบบรายการ ในขณะที่มุมมองภาพขนาดย่อแสดงการแสดงภาพของไฟล์ ทำให้ระบุเนื้อหาได้ง่ายขึ้นเพียงแวบเดียว

## การดำเนินการกับไฟล์

ใกล้ชื่อไฟล์ คุณจะเห็นสัญลักษณ์จุดไข่ปลา "..." (จุดสามจุด) — นี่คือเมนูการดำเนินการ  
แตะเพื่อแสดงรายการการดำเนินการที่มีอยู่:

- **เล่นต่อไป**: เลือกการดำเนินการนี้เพื่อแทรกไฟล์ที่เลือกไว้ที่ด้านบนของคิวเครื่องเล่นของคุณ ทำให้แน่ใจว่าจะเล่นทันทีหลังจากรายการที่กำลังเล่นอยู่
- **เล่นภายหลัง**: การเลือกตัวเลือกนี้จะเพิ่มไฟล์ไปที่ด้านล่างของคิวเครื่องเล่น ทำให้แน่ใจว่าจะเล่นหลังจากคิวที่มีอยู่
- **เพิ่มในไลบรารีเพลง**: การดำเนินการนี้ช่วยให้คุณรวมไฟล์เข้าในไลบรารีเพลงของคุณ ทำให้สามารถเข้าถึงได้ง่ายและจัดระเบียบอย่างดีตามศิลปิน อัลบั้ม แนวเพลง หรือนักแต่งเพลง
- **เพิ่มในเพลย์ลิสต์**: ใช้การดำเนินการนี้เพื่อเพิ่มไฟล์ลงในเพลย์ลิสต์ที่มีอยู่หรือสร้างใหม่
- **แก้ไขแท็กเสียง**: การเลือกตัวเลือกนี้ช่วยให้คุณเข้าถึงโปรแกรมแก้ไขแท็กในตัวของ Evermusic ซึ่งช่วยให้คุณแก้ไขแท็กเสียงสำหรับไฟล์ที่เลือก ไฟล์จะถูกดาวน์โหลดชั่วคราวไปยังไดเรกทอรีชั่วคราวแล้วอัปโหลดไปยังที่เก็บข้อมูลหลังจากที่คุณยืนยันการเปลี่ยนแปลง
- **เพิ่มในรายการโปรด**: การดำเนินการนี้เพิ่มไฟล์ลงในรายการไฟล์โปรดของคุณสำหรับการเข้าถึงที่รวดเร็วและสะดวก
- **ดาวน์โหลด**: เลือกการดำเนินการนี้เพื่อดาวน์โหลดไฟล์หรือโฟลเดอร์ไปยังอุปกรณ์ของคุณ ทำให้สามารถใช้งานแบบออฟไลน์ได้
- **เปลี่ยนชื่อ**: ตัวเลือกนี้อนุญาตให้คุณเปลี่ยนชื่อไฟล์โดยตรงบนที่เก็บข้อมูลระยะไกล ช่วยให้ตั้งชื่อไฟล์ตามที่กำหนดเองได้
- **ย้าย**: เลือกการดำเนินการนี้เพื่อย้ายไฟล์ไปยังโฟลเดอร์อื่นภายในที่เก็บข้อมูลคลาวด์ของคุณ ช่วยในการรักษาโครงสร้างไฟล์ที่เป็นระเบียบ
- **เปิดใน**: ใช้การดำเนินการนี้เพื่อส่งออกไฟล์ไปยังแอปอื่นที่เข้ากันได้ ไฟล์จะถูกดาวน์โหลดไปยังอุปกรณ์ของคุณ จากนั้นกล่องโต้ตอบแชร์จะปรากฏขึ้นสำหรับการแชร์หรือโต้ตอบเพิ่มเติม
- **ลบ**: ระวังการดำเนินการนี้เนื่องจากจะลบไฟล์ออกจากที่เก็บข้อมูลคลาวด์ของคุณอย่างถาวร การลบนี้ไม่สามารถยกเลิกได้

หากรายการการดำเนินการเกินพื้นที่หน้าจอที่มีอยู่ เพียงเลื่อนลงในเมนูการดำเนินการเพื่อเข้าถึงตัวเลือกเพิ่มเติม

## การดำเนินการกับโฟลเดอร์

สำหรับแต่ละโฟลเดอร์ในที่เก็บข้อมูลคลาวด์ของคุณ มีการดำเนินการต่างๆ ให้ใช้ได้ หากต้องการเข้าถึงตัวเลือกเหล่านี้ เพียงแตะที่ไอคอนจุดไข่ปลา "..." ที่อยู่ถัดจากชื่อโฟลเดอร์ หากคุณไม่เห็นการดำเนินการทั้งหมด ให้เลื่อนลงเพื่อแสดงตัวเลือกเพิ่มเติม นี่คือการดำเนินการที่มีอยู่:

- **เล่นทั้งหมด**: แทนที่คิวเครื่องเล่นปัจจุบันด้วยรายการทั้งหมดจากโฟลเดอร์ที่เลือก
- **เล่นต่อไป**: เพิ่มโฟลเดอร์ทั้งหมดไปที่ด้านบนของคิวเครื่องเล่น ทันทีหลังจากรายการที่กำลังเล่นอยู่
- **เล่นภายหลัง**: เพิ่มเนื้อหาโฟลเดอร์ไปที่ด้านล่างของคิวเครื่องเล่น
- **เพิ่มในไลบรารีเพลง**: การดำเนินการนี้รวมเนื้อหาของโฟลเดอร์เข้าในไลบรารีเพลงของคุณอย่างราบรื่น ทำให้สามารถเข้าถึงได้ง่ายและจัดระเบียบอย่างดีตามศิลปิน อัลบั้ม แนวเพลง หรือนักแต่งเพลง
- **เพิ่มในเพลย์ลิสต์**: คุณสามารถรวมโฟลเดอร์ทั้งหมดในเพลย์ลิสต์ คุณยังมีตัวเลือกในการสร้างเพลย์ลิสต์ใหม่ และชื่อโฟลเดอร์จะถูกกำหนดโดยอัตโนมัติ
- **เพิ่มในรายการโปรด**: ใช้การดำเนินการนี้เพื่อเพิ่มโฟลเดอร์ลงในรายการไฟล์โปรดของคุณสำหรับการเข้าถึงที่รวดเร็วและสะดวก
- **เปิดโหมดออฟไลน์**: การเปิดใช้งานโหมดออฟไลน์สำหรับโฟลเดอร์ที่เลือก แอปพลิเคชันจะไปไกลกว่าการดาวน์โหลดธรรมดา โดยสแกนการเปลี่ยนแปลงอย่างต่อเนื่อง และหากมีการเพิ่มไฟล์ใหม่ลงในโฟลเดอร์ออนไลน์ ไฟล์เหล่านั้นจะถูกดาวน์โหลดไปยังแอปโดยอัตโนมัติ
- **ดาวน์โหลด**: ดาวน์โหลดเนื้อหาทั้งหมดของโฟลเดอร์ไปยังอุปกรณ์ของคุณสำหรับการเข้าถึงแบบออฟไลน์
- **เปลี่ยนชื่อ**: เปลี่ยนชื่อโฟลเดอร์โดยตรงบนที่เก็บข้อมูลระยะไกล
- **ย้าย**: ย้ายโฟลเดอร์ไปยังตำแหน่งอื่นภายในที่เก็บข้อมูลคลาวด์ของคุณ
- **ลบ**: ระวังการดำเนินการนี้เนื่องจากจะลบโฟลเดอร์และเนื้อหาออกจากที่เก็บข้อมูลคลาวด์ของคุณอย่างถาวร การดำเนินการนี้ไม่สามารถยกเลิกได้


## การเข้าถึงด่วน

ส่วนการเข้าถึงด่วนอยู่ที่ด้านบนของหน้าจอ ให้การเข้าถึงไฟล์โปรดและที่เปิดล่าสุดจากบริการคลาวด์ที่เชื่อมต่ออย่างรวดเร็ว
ทุกครั้งที่คุณเปิดไฟล์หรือโฟลเดอร์จากคลาวด์ ไฟล์นั้นจะถูกเพิ่มในรายการ "เปิดล่าสุด" หากต้องการล้างรายการนี้ ให้เปิด "ล่าสุด" แตะปุ่ม "ดำเนินการเพิ่มเติม" และเลือก "ลบรายการ" คุณยังสามารถทำเครื่องหมายโฟลเดอร์ที่ซ้อนกันลึกเป็นรายการโปรดเพื่อเข้าถึงได้อย่างรวดเร็วโดยไม่ต้องค้นหาผ่านโครงสร้างไดเรกทอรี

## บริการอื่นๆ

ส่วนนี้แสดงคุณสมบัติพิเศษที่ช่วยเสริมประสบการณ์ของคุณ ปัจจุบัน แอปรองรับการ scrobbling ของ Last.fm เมื่อเชื่อมต่อแล้ว สถิติการเล่นของคุณจะถูกส่งไปยังบัญชี Last.fm ของคุณโดยอัตโนมัติ คุณสามารถเยี่ยมชมโปรไฟล์ Last.fm ของคุณในภายหลังเพื่อดูการวิเคราะห์การฟังและรับคำแนะนำเพลงส่วนตัว มีคำแนะนำการตั้งค่าโดยละเอียด [ที่นี่](/docs/howto/how-to-scrobble-your-music-history-from-evermusic-or-flacbox-to-last-fm)

Bagian File Lokal berfungsi sebagai pusat untuk mengelola file yang terletak di folder "Documents" aplikasi, serta file yang ditambahkan dari perangkat Anda, yang tersedia untuk pemutaran offline.

Manajer file bawaan ini memungkinkan Anda mengedit file dan menawarkan berbagai metode untuk mengimpor file audio ke dalam aplikasi.

## Toolbar Atas

Toolbar atas, yang terletak di bawah bilah navigasi, menawarkan beberapa tindakan:

- **Cari:** Lakukan pencarian di dalam folder saat ini.
- **Lanjutkan Pemutaran:** Jika diaktifkan di pengaturan aplikasi, fitur ini memulihkan antrean pemutar audio dan posisi media terakhir untuk folder saat ini, memungkinkan Anda melanjutkan dari pemutaran terakhir.
- **Putar semua:** Pindai folder saat ini dan sub-foldernya dan tambahkan file ke antrean pemutar dengan urutan pengurutan saat ini.
- **Acak Semua:** Mirip dengan "Putar semua," tetapi mengacak file sebelum menambahkannya ke antrean pemutar audio.

## Folder Khusus

Di dalam layar "File lokal", Anda akan menemukan beberapa folder khusus.

## Unduh

Semua file yang diunduh dari layanan penyimpanan cloud akan muncul di sini secara default. Anda dapat menyesuaikan perilaku ini di pengaturan aplikasi.

## Pemutar Audio

Ini adalah folder cache pemutar audio. Secara default, pemutar audio mengunduh beberapa lagu dalam antrean pemutar audio untuk menerapkan crossfade dan mengoptimalkan pemutaran. Anda dapat menonaktifkan cache pemutar audio di pengaturan aplikasi atau cukup hapus folder ini dari "File lokal."

## iCloud

Tempatkan file di folder ini, dan file tersebut akan disinkronkan di semua perangkat Anda yang terhubung ke akun iCloud yang sama.

## Folder offline

Ketika Anda membuat playlist, album, artis, atau folder online tersedia untuk penggunaan offline, semua file tersebut akan diunduh ke folder ini.  
Jika Anda perlu menonaktifkan mode offline untuk item tertentu, Anda dapat menggunakan menu lebih banyak tindakan atau menghapus folder yang sesuai yang terletak di "Folder offline."

## Dua Bagian yang Berbeda

Layar "File Lokal" diatur menjadi dua bagian yang berbeda.

## File di Aplikasi Ini

Menampilkan file dan folder yang disimpan di direktori Documents aplikasi dan iCloud Drive.

## File di iPhone/iPad/Mac ini

Menampilkan file yang terletak di perangkat Anda tetapi di aplikasi yang berbeda. Anda dapat mengimpornya ke aplikasi ini menggunakan pemilih file sistem. Untuk mengaktifkan pemilih, pilih "Buka file..." untuk memilih file atau "Buka folder..." untuk memilih folder. Instruksi terperinci tentang cara mengimpor musik lokal yang tersimpan di iPhone atau Mac Anda tersedia [di sini](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac). Anda juga dapat menghubungkan folder yang terletak di perangkat Anda dan memiliki akses cepat ke konten folder. Gunakan item menu "Hubungkan folder" dan pilih folder yang terletak di perangkat Anda. Ketuk "Selesai" dan aplikasi akan membuat tautan ke folder tersebut dengan akses baca/tulis dan Anda dapat mengelola file langsung dari aplikasi ini. Untuk memutuskan koneksi folder yang terletak di perangkat Anda, ketuk tombol "Lebih banyak tindakan" dan pilih "Putuskan Koneksi".

## Lebih Banyak Tindakan

Menu lebih banyak tindakan untuk folder yang saat ini dibuka terletak di sudut kanan atas dan menyediakan akses ke berbagai tindakan.

- **Pilih:** Beralih ke mode pemilihan untuk file dan folder.
- **Folder Baru:** Buat folder baru di dalam folder saat ini.
- **Cari:** Aktifkan fungsi pencarian untuk menemukan file atau folder tertentu.
- **Urutkan:** Atur file berdasarkan nama, ukuran, tanggal, atau menggunakan informasi metadata.
- **Daftar/Grid:** Ubah presentasi konten dari tampilan tabel ke tampilan grid.
- **Bantuan:** Akses instruksi bantuan.

## Mode Pemilihan

Jika Anda perlu mengedit beberapa file, aktifkan mode pemilihan dengan mengetuk tombol lebih banyak tindakan "..." di bilah navigasi di sudut kanan atas lalu pilih item menu "Pilih". Ini akan menampilkan kotak centang di dekat setiap file. Pilih file yang diinginkan dengan mengetuk kotak centangnya. Anda dapat melakukan tindakan berikut pada file yang dipilih.

- **Putar berikutnya:** Tambahkan file atau folder yang dipilih ke bagian atas antrean pemutar dengan urutan pengurutan saat ini.

## Menu Opsi

Untuk setiap file atau folder di aplikasi, beberapa tindakan tersedia, dapat diakses dengan mengetuk tombol lebih "...". Jika Anda tidak melihat semua tindakan, gulir ke bawah untuk melihatnya:

- **Putar semua:** Ganti antrean pemutar dengan item dari folder yang dipilih.
- **Putar berikutnya:** Tambahkan file atau folder ke bagian atas antrean pemutar.
- **Putar nanti:** Tambahkan file atau folder ke bagian bawah antrean pemutar.
- **Tambahkan ke Perpustakaan Musik:** Sertakan file atau folder yang dipilih di perpustakaan musik.
- **Tambahkan ke Daftar Putar:** Tambahkan file atau folder ke daftar putar. Jika Anda membuat daftar putar dari folder, nama daftar putar akan ditetapkan secara otomatis.
- **Tambahkan ke Favorit:** Tandai file atau folder yang dipilih sebagai favorit.
- **Mengedit tag audio:** Buka editor tag bawaan untuk mengubah tag audio untuk file yang dipilih.
- **Ganti Nama:** Ubah nama file atau folder yang dipilih.
- **Pindah:** Pindahkan file atau folder yang dipilih ke lokasi lain di perangkat.
- **Salin:** Buat salinan file atau folder yang dipilih.
- **Unggah:** Unggah file atau folder yang dipilih ke penyimpanan jarak jauh.
- **Tampilkan di Finder:** Untuk file yang diimpor dari Mac Anda, tindakan ini mengungkapkan direktori di Finder yang berisi file tersebut.
- **Buka dengan:** Ekspor file ke aplikasi lain.
- **Hapus:** Hapus file atau folder dari perangkat. Harap perhatikan bahwa tindakan ini tidak dapat dibatalkan dan Anda tidak dapat memulihkan file yang dihapus.

## Folder Offline

{{< cards cols="1">}}
  {{< card title="" subtitle="Menu Lebih Banyak Tindakan Folder Offline" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-offline-folders.webp" >}}
{{< /cards >}}

Mode offline adalah fitur praktis yang memungkinkan Anda mengakses musik favorit bahkan ketika Anda tidak terhubung ke internet. Ketika Anda mengaktifkan mode offline untuk album, artis, playlist, genre, atau folder jarak jauh manapun, semua file dalam koleksi tersebut akan diunduh secara otomatis ke perangkat Anda untuk pemutaran offline. Anda dapat mengakses file-file ini dengan mudah di bagian "Folder Offline" aplikasi.

Yang lebih nyaman lagi adalah ketika Anda menambahkan file baru ke server jarak jauh di folder yang telah Anda tandai untuk penggunaan offline, penambahan baru ini juga akan diunduh secara otomatis ke perangkat Anda. Ini memastikan bahwa Anda memiliki musik terbaru yang siap untuk dinikmati secara offline.

Jika Anda perlu menyinkronkan folder offline dari layanan cloud secara manual, berikut caranya: cukup ketuk tiga titik yang terletak di sudut kanan atas aplikasi, dan pilih opsi "Sinkronkan". Ini memungkinkan Anda memperbarui koleksi offline dengan trek terbaru dari layanan cloud sesuai keinginan Anda.

Bagi mereka yang ingin menyesuaikan pengalaman mereka lebih lanjut, Anda dapat menyesuaikan pengaturan batas waktu sinkronisasi di pengaturan aplikasi sesuai preferensi Anda. Dengan cara ini, Anda memiliki kontrol penuh atas kapan dan bagaimana koleksi offline Anda diperbarui, memastikan Anda selalu memiliki lagu favorit tersedia, bahkan tanpa koneksi internet.

Instruksi terperinci tentang Cara Memutar Musik Offline di Evermusic & Flacbox: Unduh & Sinkronisasi dari Cloud ke File Lokal tersedia [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Pengaturan Manajer File

Hampir setiap perilaku layar File Lokal — dari bandwidth jaringan hingga tempat landing unduhan hingga cara thumbnail di-cache — dapat dikonfigurasi di **Pengaturan → Manajer file**. Buka kapan pun Anda ingin menyesuaikan kecepatan transfer, menghemat ruang penyimpanan, atau membatasi aplikasi hanya ke Wi-Fi.

  {{< card title="" subtitle="Layar Pengaturan Manajer File" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-file-manager-settings.webp" >}}

Layar ini mengekspos setiap opsi yang dikelompokkan ke dalam bagian yang diberi label jelas:

### Transfer file (jenis jaringan)

Pilih bagaimana unduhan dan unggahan diizinkan menggunakan jaringan:

- **Wi-Fi:** Transfer hanya berjalan saat Anda terhubung ke jaringan Wi-Fi. Gunakan ini jika Anda menggunakan paket data seluler yang diukur atau ingin penggunaan data mobile yang dapat diprediksi.
- **Wi-Fi & Data Seluler:** Transfer berjalan melalui Wi-Fi saat tersedia dan beralih ke koneksi seluler Anda sebaliknya. Mungkin menggunakan paket data mobile Anda.

### Batas transfer file paralel

Mengatur jumlah maksimum file yang dapat ditransfer secara bersamaan. Defaultnya adalah nilai yang seimbang; naikkan untuk mempercepat unduhan folder besar, turunkan di jaringan lemah atau perangkat lama untuk menjaga transfer tetap stabil. Nilai yang lebih tinggi menggunakan lebih banyak bandwidth, CPU, dan baterai.

### Tugas transfer file

Membuka antrean transfer langsung — setiap tugas unggah/unduh yang sedang berlangsung dan selesai dengan bilah kemajuan, kecepatan, dan status. Anda dapat menjeda, melanjutkan, dan membatalkan tugas individual di sini. Antrean yang sama dapat dijangkau dari ikon panah berputar di sudut kiri atas layar File Lokal.

### Transfer latar belakang

Ketika diaktifkan, unduhan berlanjut bahkan setelah Anda meletakkan Evermusic di latar belakang atau mengunci perangkat Anda. Berguna untuk perpustakaan besar di jaringan lambat. Catatan: jika operasi menggunakan banyak energi, iOS dapat menangguhkan aplikasi untuk melindungi baterai.

### Simpan file yang diunduh ke

Tentukan tujuan default untuk unduhan cloud:

- **Unduh:** *(default)* Semua unduhan mendarat di folder **File Lokal → Unduh**.
- **Selalu tanya:** Evermusic meminta Anda memilih folder tujuan untuk setiap unduhan. Berguna jika Anda mengatur perpustakaan berdasarkan folder artis atau genre di perangkat.
- **Folder kustom:** Pilih folder tertentu sekali dan setiap unduhan akan mendarat di sana sampai Anda mengubah pengaturan.

### Folder offline yang disinkronkan

Menampilkan setiap folder yang telah Anda tandai sebagai "Tersedia offline" dan memungkinkan Anda mengelolanya. Ketuk folder untuk memicu sinkronisasi secara manual, menampilkan lokasinya, atau **Nonaktifkan mode offline** untuk berhenti memantaunya dan menghapus salinan lokal. Baca lebih lanjut tentang mode offline [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

### Interval waktu (sinkronisasi offline)

Memilih seberapa sering aplikasi memeriksa folder offline untuk perubahan:

- **Segera:** Folder offline disinkronkan setiap kali Anda membuka aplikasi.
- **15 / 30 / 60 menit:** Pemeriksaan berkala saat aplikasi aktif.
- **Manual:** Sinkronisasi hanya saat Anda mengetuk tindakan **Sinkronkan** pada folder.

### Tampilkan nama file lengkap

Ketika diaktifkan, manajer file menampilkan nama file lengkap termasuk ekstensi (mis. `Song Title - Artist.flac`). Nonaktifkan untuk menyembunyikan ekstensi dan menampilkan nama yang lebih pendek dan lebih bersih. Berguna saat bekerja dengan format audio campuran atau file non-audio.

### Edit file online

Mengalihkan seluruh aplikasi antara mode baca-tulis dan hanya baca untuk akun cloud yang terhubung. Ketika **mode hanya baca** diaktifkan, setiap tindakan destruktif (ganti nama, pindah, hapus, unggah, edit tag audio) disembunyikan dari antarmuka pengguna untuk file online — jaring pengaman yang bagus jika Anda khawatir tentang kehilangan data tidak sengaja di perpustakaan keluarga bersama atau cloud kerja. File lokal tetap dapat diedit sepenuhnya terlepas dari pengaturan ini.

### Salin file saat membuka

Mengontrol apa yang terjadi ketika Anda membuka file audio dari aplikasi lain (mis. melalui aplikasi Files, lampiran Mail, AirDrop, atau ekstensi Berbagi):

- **Selalu tanya:** Evermusic meminta setiap kali apakah akan menyalin file ke folder Documents aplikasi atau membukanya di tempat.
- **Selalu salin:** File diimpor ke bagian File Lokal → Unduh, menjadi salinan lokal permanen.
- **Buka di tempat:** File tetap di lokasi aslinya dan Evermusic mengaksesnya melalui referensi. Tautan menghilang jika sumber dihapus.

### Thumbnail untuk file

Mengelola thumbnail yang di-cache (artwork album, waveform audio, pratinjau gambar) yang ditampilkan di sebelah file:

- **Muat thumbnail untuk file online:** Aktifkan apakah thumbnail diambil untuk file di cloud. Nonaktifkan untuk mengurangi penggunaan data pada koneksi lambat.
- **Jenis jaringan (thumbnail):** Batasi pemuatan thumbnail ke Wi-Fi atau izinkan data seluler.
- **Tampilkan di folder:** Tampilkan folder cache thumbnail sehingga Anda dapat memeriksa apa yang tersimpan.
- **Hapus semua:** Hapus setiap thumbnail yang di-cache untuk mengklaim kembali ruang penyimpanan. Aplikasi akan meregenerasi thumbnail saat Anda membuka kembali folder.

### Hapus file sementara

Mengosongkan folder cache aplikasi (buffer audio, unduhan setengah selesai, pratinjau gambar, artwork yang didekode). Tombol menampilkan ukuran cache saat ini sehingga Anda tahu berapa banyak yang akan Anda dapatkan kembali. Aman untuk dijalankan kapan saja — Evermusic meregenerasi apa pun yang diperlukan pada pemutaran berikutnya.

## Personalisasi

Dengan membuka Pengaturan aplikasi dan masuk ke Personalisasi, Anda dapat mengkonfigurasi gaya layar File Lokal.  
Opsi yang tersedia adalah

- **Menu polos:** Menampilkan folder yang terletak di direktori Documents aplikasi.
- **Menu berkelompok:** Menampilkan konten layar dikelompokkan berdasarkan kategori.

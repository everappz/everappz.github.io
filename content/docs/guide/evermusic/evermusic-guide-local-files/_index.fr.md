La section Fichiers locaux sert de hub pour gérer les fichiers situés dans le dossier « Documents » de l'application, ainsi que les fichiers ajoutés depuis votre appareil, disponibles pour la lecture hors ligne.

Ce gestionnaire de fichiers intégré vous permet de modifier des fichiers et propose diverses méthodes pour importer des fichiers audio dans l'application.

## Barre d'outils supérieure

La barre d'outils supérieure, située sous la barre de navigation, offre plusieurs actions :

- **Rechercher :** effectuez une recherche dans le dossier actuel.
- **Continuer la lecture :** si activée dans les paramètres de l'application, cette fonctionnalité restaure la file d'attente du lecteur audio et la dernière position multimédia pour le dossier actuel, vous permettant de reprendre là où vous en étiez.
- **Tout lire :** analysez le dossier actuel et ses sous-dossiers et ajoutez les fichiers à la file d'attente du lecteur avec l'ordre de tri actuel.
- **Lecture aléatoire :** similaire à « Tout lire », mais mélange les fichiers avant de les ajouter à la file d'attente du lecteur audio.

Vous pouvez afficher ou masquer la barre d'outils supérieure en utilisant un geste de glissement vers le bas.

## Dossiers spéciaux

Dans l'écran « Fichiers locaux », vous rencontrerez plusieurs dossiers spéciaux.

## Téléchargements

Tous les fichiers téléchargés depuis le service de stockage cloud apparaîtront ici par défaut. Vous pouvez personnaliser ce comportement dans les paramètres de l'application.

## Lecteur audio

Il s'agit du dossier de cache du lecteur audio. Par défaut, le lecteur audio télécharge certaines chansons dans la file d'attente du lecteur audio pour appliquer le crossfade et optimiser la lecture. Vous pouvez désactiver le cache du lecteur audio dans les paramètres de l'application ou simplement supprimer ce dossier depuis « Fichiers locaux ».

## iCloud

Placez des fichiers dans ce dossier et ils se synchroniseront sur tous vos appareils connectés au même compte iCloud.

## Dossiers hors ligne

Lorsque vous rendez une playlist, un album, un artiste ou un dossier en ligne disponible hors ligne, tous ces fichiers seront téléchargés dans ce dossier.  
Si vous avez besoin de désactiver le mode hors ligne pour des éléments spécifiques, vous pouvez utiliser le menu Plus d'actions ou supprimer le dossier correspondant situé dans « Dossiers hors ligne ».

## Deux sections distinctes

L'écran « Fichiers locaux » est organisé en deux sections distinctes.

## Fichiers dans cette application

Affiche les fichiers et dossiers stockés dans le répertoire Documents de l'application et iCloud Drive.

## Fichiers sur cet iPhone/iPad/Mac

Affiche les fichiers situés sur votre appareil mais dans différentes applications. Vous pouvez les importer dans cette application en utilisant le sélecteur de fichiers système. Pour activer le sélecteur, choisissez « Ouvrir les fichiers... » pour sélectionner des fichiers ou « Ouvrir les dossiers... » pour sélectionner des dossiers. Des instructions détaillées sur la façon d'importer la musique locale stockée sur votre iPhone ou Mac sont disponibles [ici](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac). Vous pouvez également connecter un dossier situé sur votre appareil et avoir un accès rapide au contenu du dossier. Utilisez l'élément de menu « Connecter un dossier » et choisissez un dossier situé sur votre appareil. Appuyez sur « Terminé » et l'application créera un lien vers ce dossier avec un accès en lecture/écriture et vous pourrez gérer les fichiers directement depuis cette application. Pour déconnecter un dossier situé sur votre appareil, appuyez sur le bouton « Plus d'actions » et choisissez « Déconnecter ».

## Plus d'actions

Le menu Plus d'actions pour le dossier actuellement ouvert, situé dans le coin supérieur droit, donne accès à diverses actions.

- **Sélectionner :** passez en mode de sélection pour les fichiers et dossiers.
- **Nouveau dossier :** créez un nouveau dossier dans le dossier actuel.
- **Rechercher :** activez la fonction de recherche pour trouver un fichier ou dossier spécifique.
- **Trier :** organisez les fichiers par nom, taille, date ou en utilisant des informations de métadonnées.
- **Liste/Grille :** modifiez la présentation du contenu d'une vue tableau à une vue grille.
- **Aide :** accédez aux instructions d'aide.

## Mode de sélection

Si vous avez besoin de modifier plusieurs fichiers, activez le mode de sélection en appuyant sur le bouton Plus d'actions « ... » dans la barre de navigation dans le coin supérieur droit, puis choisissez l'élément de menu « Sélectionner ». Cela affichera des cases à cocher près de chaque fichier. Sélectionnez les fichiers souhaités en appuyant sur leurs cases à cocher. Vous pouvez effectuer les actions suivantes sur les fichiers sélectionnés.

{{< cards cols="1">}}
  {{< card title="" subtitle="Actions du mode de sélection pour les fichiers locaux" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-selection-mode.webp" >}}
{{< /cards >}}

- **Lire ensuite :** ajoutez les fichiers ou dossiers sélectionnés en haut de la file d'attente du lecteur avec l'ordre de tri actuel.
- **Lire plus tard :** ajoutez les fichiers ou dossiers sélectionnés en bas de la file d'attente du lecteur avec l'ordre de tri actuel.
- **Ajouter à la bibliothèque musicale :** incluez les fichiers ou dossiers sélectionnés dans la bibliothèque musicale.
- **Ajouter à une playlist :** ajoutez les fichiers ou dossiers sélectionnés à une playlist.
- **Copier :** créez une copie du fichier ou dossier sélectionné.
- **Téléverser :** téléversez le fichier ou dossier sélectionné vers un stockage distant.
- **Déplacer :** déplacez le fichier ou dossier sélectionné vers un autre emplacement sur l'appareil.
- **Renommer :** changez le nom du fichier ou dossier sélectionné.
- **Supprimer :** supprimez le fichier ou dossier sélectionné de l'appareil. Veuillez noter que cette action est irréversible.

## Menu Options

Pour chaque fichier ou dossier dans l'application, plusieurs actions sont disponibles, accessibles en appuyant sur le bouton Plus « ... ». Si vous ne voyez pas toutes les actions, faites défiler vers le bas pour les afficher :

- **Tout lire :** remplacez la file d'attente du lecteur par les éléments du dossier sélectionné.
- **Lire ensuite :** ajoutez le fichier ou dossier en haut de la file d'attente du lecteur.
- **Lire plus tard :** ajoutez le fichier ou dossier en bas de la file d'attente du lecteur.
- **Ajouter à la bibliothèque musicale :** incluez les fichiers ou dossiers sélectionnés dans la bibliothèque musicale.
- **Ajouter à une playlist :** ajoutez le fichier ou dossier à une playlist. Si vous créez une playlist à partir d'un dossier, un nom de playlist lui sera automatiquement attribué.
- **Ajouter aux favoris :** marquez les fichiers ou dossiers sélectionnés comme favoris.
- **Modifier les balises audio :** ouvrez l'éditeur de tags intégré pour modifier les tags audio du fichier sélectionné.
- **Renommer :** modifiez le nom du fichier ou dossier sélectionné.
- **Déplacer :** déplacez le fichier ou dossier sélectionné vers un autre emplacement sur l'appareil.
- **Copier :** créez une copie du fichier ou dossier sélectionné.
- **Téléverser :** téléversez le fichier ou dossier sélectionné vers un stockage distant.
- **Afficher dans le Finder :** pour les fichiers importés depuis votre Mac, cette action révèle le répertoire dans le Finder contenant le fichier.
- **Ouvrir dans :** exportez le fichier vers une autre application.
- **Supprimer :** supprimez le fichier ou dossier de l'appareil. Veuillez noter que cette action est irréversible et vous ne pouvez pas restaurer les fichiers supprimés.

## Dossiers hors ligne

{{< cards cols="1">}}
  {{< card title="" subtitle="Menu Plus d'actions du dossier hors ligne" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-offline-folders.webp" >}}
{{< /cards >}}

Le mode hors ligne est une fonctionnalité pratique qui vous permet d'accéder à votre musique préférée même lorsque vous n'êtes pas connecté à Internet. Lorsque vous activez le mode hors ligne pour un album, un artiste, une playlist, un genre ou un dossier distant, tous les fichiers de cette collection seront automatiquement téléchargés sur votre appareil pour une lecture hors ligne. Vous pouvez accéder facilement à ces fichiers dans la section « Dossiers hors ligne » de l'application.

Ce qui est encore plus pratique, c'est que lorsque vous ajoutez de nouveaux fichiers au serveur distant dans le dossier que vous avez marqué pour une utilisation hors ligne, ces nouvelles additions seront également automatiquement téléchargées sur votre appareil. Cela garantit que vous avez la musique la plus récente prête pour une écoute hors ligne.

Si vous avez besoin de synchroniser manuellement un dossier hors ligne depuis votre service cloud, voici comment procéder : appuyez simplement sur les trois points situés dans le coin supérieur droit de l'application et sélectionnez l'option « Synchroniser ». Cela vous permet de mettre à jour votre collection hors ligne avec les dernières pistes du service cloud à votre convenance.

Pour ceux qui souhaitent personnaliser davantage leur expérience, vous pouvez ajuster les paramètres du délai de synchronisation dans les paramètres de l'application selon vos préférences. Ainsi, vous avez un contrôle total sur quand et comment votre collection hors ligne est mise à jour, garantissant que vous avez toujours vos morceaux préférés disponibles, même sans connexion Internet.

Des instructions détaillées sur la façon de lire de la musique hors ligne dans Evermusic et Flacbox : télécharger et synchroniser du cloud vers les fichiers locaux sont disponibles [ici](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Paramètres du gestionnaire de fichiers

Presque tous les comportements de l'écran Fichiers locaux — de la bande passante réseau à l'emplacement des téléchargements en passant par la mise en cache des miniatures — sont configurables sous **Paramètres → Gestionnaire de fichiers**. Ouvrez-le chaque fois que vous souhaitez affiner la vitesse de transfert, économiser de l'espace de stockage ou restreindre l'application au Wi-Fi uniquement.

{{< cards cols="1">}}
  {{< card title="" subtitle="Écran des paramètres du gestionnaire de fichiers" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-file-manager-settings.webp" >}}
{{< /cards >}}

L'écran expose chaque option regroupée en sections clairement étiquetées :

### Transferts de fichiers (type de réseau)

Choisissez comment les téléchargements et téléversements sont autorisés à utiliser le réseau :

- **Wi-Fi :** les transferts ne s'exécutent que lorsque vous êtes connecté à un réseau Wi-Fi. Utilisez ceci si vous êtes sur un forfait cellulaire mesuré ou si vous souhaitez une utilisation prévisible des données mobiles.
- **Wi-Fi et données cellulaires :** les transferts s'exécutent via Wi-Fi lorsque disponible et basculent sur votre connexion cellulaire sinon. Peut utiliser votre forfait de données mobiles.

### Limite de transfert de fichiers parallèles

Définit le nombre maximum de fichiers pouvant être transférés simultanément. La valeur par défaut est équilibrée ; augmentez-la pour accélérer les téléchargements de grands dossiers, réduisez-la sur des réseaux faibles ou des appareils plus anciens pour maintenir des transferts stables. Une valeur plus élevée utilise plus de bande passante, de CPU et de batterie.

### Tâches de transfert de fichiers

Ouvre la file de transfert en direct — chaque tâche de téléversement/téléchargement en cours et terminée avec des barres de progression, des vitesses et des statuts. Vous pouvez mettre en pause, reprendre et annuler des tâches individuelles ici. La même file est accessible depuis l'icône de flèches rotatives dans le coin supérieur gauche de l'écran Fichiers locaux.

### Transferts en arrière-plan

Lorsqu'activé, les téléchargements continuent même après que vous ayez mis Evermusic en arrière-plan ou verrouillé votre appareil. Utile pour les grandes bibliothèques sur des réseaux lents. Remarque : si l'opération consomme une quantité significative d'énergie, iOS peut suspendre l'application pour protéger la durée de vie de la batterie.

### Enregistrer les fichiers téléchargés dans

Définissez votre destination par défaut pour les téléchargements cloud :

- **Téléchargements :** *(par défaut)* tous les téléchargements atterrissent dans le dossier **Fichiers locaux → Téléchargements**.
- **Toujours demander :** Evermusic vous invite à choisir le dossier de destination pour chaque téléchargement. Utile si vous organisez votre bibliothèque par dossiers d'artistes ou de genres sur l'appareil.
- **Dossier personnalisé :** choisissez un dossier spécifique une fois et chaque téléchargement y atterrira jusqu'à ce que vous changiez le paramètre.

### Dossiers hors ligne synchronisés

Affiche chaque dossier que vous avez marqué comme « Disponible hors ligne » et vous permet de les gérer. Appuyez sur un dossier pour déclencher manuellement une synchronisation, afficher son emplacement ou **Désactiver le mode hors ligne** pour arrêter de le surveiller et supprimer la copie locale. En savoir plus sur les modes hors ligne [ici](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

### Intervalle de temps (synchronisation hors ligne)

Sélectionne la fréquence à laquelle l'application vérifie les modifications dans les dossiers hors ligne :

- **Immédiatement :** les dossiers hors ligne sont synchronisés à chaque fois que vous ouvrez l'application.
- **15 / 30 / 60 minutes :** vérifications périodiques pendant que l'application est active.
- **Manuel :** synchronisation uniquement lorsque vous appuyez sur l'action **Synchroniser** sur un dossier.

### Afficher les noms de fichiers complets

Lorsqu'activé, le gestionnaire de fichiers affiche les noms de fichiers complets incluant les extensions (ex. `Titre de la chanson - Artiste.flac`). Désactivez pour masquer les extensions et afficher des noms plus courts et plus propres. Utile lorsque vous travaillez avec des formats audio mixtes ou des fichiers non audio.

### Modifier les fichiers en ligne

Bascule toute l'application entre les modes lecture-écriture et lecture seule pour les comptes cloud connectés. Lorsque le **mode lecture seule** est activé, chaque action destructive (renommer, déplacer, supprimer, téléverser, modifier les balises audio) est masquée de l'interface utilisateur pour les fichiers en ligne — un excellent filet de sécurité si vous craignez une perte accidentelle de données dans une bibliothèque familiale partagée ou un cloud de travail. Les fichiers locaux restent entièrement modifiables quel que soit ce paramètre.

### Copier les fichiers lors de l'ouverture

Contrôle ce qui se passe lorsque vous ouvrez des fichiers audio depuis une autre application (ex. via l'application Fichiers, pièces jointes Mail, AirDrop ou extensions de partage) :

- **Toujours demander :** Evermusic demande à chaque fois s'il faut copier le fichier dans le dossier Documents de l'application ou l'ouvrir en place.
- **Toujours copier :** les fichiers sont importés dans la section Fichiers locaux → Téléchargements, devenant des copies locales permanentes.
- **Ouvrir en place :** les fichiers restent à leur emplacement d'origine et Evermusic y accède par référence. Le lien disparaît si la source est supprimée.

### Miniatures pour les fichiers

Gère les miniatures mises en cache (pochettes d'album, formes d'onde audio, aperçus d'images) affichées à côté des fichiers :

- **Charger les miniatures pour les fichiers en ligne :** activez ou désactivez la récupération des miniatures pour les fichiers dans le cloud. Désactivez pour réduire l'utilisation des données sur les connexions lentes.
- **Type de réseau (miniatures) :** restreignez le chargement des miniatures au Wi-Fi ou autorisez les données cellulaires.
- **Afficher dans le dossier :** révélez le dossier de cache des miniatures pour inspecter ce qui est stocké.
- **Tout supprimer :** effacez chaque miniature mise en cache pour récupérer de l'espace de stockage. L'application régénérera les miniatures au fur et à mesure que vous rouvrirez les dossiers.

### Supprimer les fichiers temporaires

Vide le dossier de cache de l'application (tampons audio, téléchargements partiellement terminés, aperçus d'images, pochettes décodées). Le bouton affiche la taille actuelle du cache pour que vous sachiez combien vous récupérerez. Sans danger à exécuter à tout moment — Evermusic régénère ce dont il a besoin lors de la prochaine lecture.

## Personnalisation

En ouvrant les Paramètres de l'application et en allant dans Personnalisation, vous pouvez configurer le style de l'écran Fichiers locaux.  
Les options disponibles sont :

- **Menu simple :** affiche les dossiers situés dans le répertoire Documents de l'application.
- **Menu groupé :** affiche le contenu de l'écran regroupé par catégories.

Questo gestore file integrato consente di modificare i file e offre vari metodi per importare file audio nell'app.

## Barra degli strumenti superiore

La barra degli strumenti superiore, situata sotto la barra di navigazione, offre diverse azioni:

- **Cerca:** Esegui una ricerca all'interno della cartella corrente.
- **Continua Riproduzione:** Se abilitata nelle impostazioni dell'applicazione, questa funzione ripristina la coda del lettore audio e l'ultima posizione media per la cartella corrente, permettendoti di riprendere dall'ultima riproduzione.
- **Riproduci tutto:** Scansiona la cartella corrente e le sue sottocartelle e aggiungi file alla coda del lettore con l'ordine di ordinamento corrente.

## Cartelle Speciali

All'interno della schermata "File locali", incontrerai diverse cartelle speciali.

## Download

Tutti i file scaricati dal servizio di storage cloud appariranno qui per impostazione predefinita. Puoi personalizzare questo comportamento nelle impostazioni dell'applicazione.

## Lettore Audio

Questa è la cartella cache del lettore audio. Per impostazione predefinita, il lettore audio scarica alcuni brani nella coda del lettore audio per applicare il crossfade e ottimizzare la riproduzione. Puoi disabilitare la cache del lettore audio nelle impostazioni dell'app o semplicemente eliminare questa cartella dai "File locali."

## iCloud

Inserisci i file in questa cartella e verranno sincronizzati su tutti i tuoi dispositivi collegati allo stesso account iCloud.

## Cartelle offline

Quando rendi una playlist, un album, un artista o una cartella online disponibile per l'uso offline, tutti quei file verranno scaricati in questa cartella.  
Se devi disabilitare la modalità offline per elementi specifici, puoi usare il menu altre azioni o eliminare la cartella corrispondente situata in "Cartelle offline."

## Due Sezioni Distinte

La schermata "File Locali" è organizzata in due sezioni distinte.

Usa la voce di menu "Connetti una cartella" e scegli una cartella situata sul tuo dispositivo. Tocca "Fine" e l'app creerà un collegamento a quella cartella con accesso in lettura/scrittura e potrai gestire i file direttamente da questa app. Per disconnettere la cartella situata sul tuo dispositivo tocca il pulsante "Altre azioni" e scegli "Disconnetti".

## Altre Azioni

Il menu altre azioni per la cartella attualmente aperta si trova nell'angolo in alto a destra e fornisce accesso a varie azioni.

- **Seleziona:** Passa alla modalità di selezione per file e cartelle.
- **Nuova Cartella:** Crea una nuova cartella all'interno della cartella corrente.
- **Cerca:** Attiva la funzione di ricerca per trovare un file o una cartella specifici.
- **Ordina:** Organizza i file per nome, dimensione, data o usando informazioni sui metadati.
- **Lista/Griglia:** Cambia la presentazione dei contenuti da una tabella a una visualizzazione a griglia.
- **Aiuto:** Accedi alle istruzioni di aiuto.

## Modalità Selezione

Se devi modificare più file, attiva la modalità di selezione toccando il pulsante altre azioni "..." nella barra di navigazione nell'angolo in alto a destra e poi scegli la voce di menu "Seleziona". Questo mostrerà caselle di controllo vicino a ogni file. Seleziona i file desiderati toccando le loro caselle di controllo. Puoi eseguire le seguenti azioni sui file selezionati.

## Menu Opzioni

Per ogni file o cartella nell'app, sono disponibili diverse azioni, accessibili toccando il pulsante "...". Se non vedi tutte le azioni, scorri verso il basso per visualizzarle:

- **Riproduci tutto:** Sostituisci la coda del lettore con gli elementi dalla cartella selezionata.
- **Riproduci successivo:** Aggiungi il file o la cartella in cima alla coda del lettore.
- **Riproduci più tardi:** Aggiungi il file o la cartella in fondo alla coda del lettore.
- **Aggiungi alla Libreria musicale:** Includi i file o le cartelle selezionati nella libreria musicale.
- **Aggiungi a una Playlist:** Aggiungi il file o la cartella a una playlist. Se crei una playlist da una cartella, il nome della playlist verrà assegnato automaticamente.
- **Aggiungi ai Preferiti:** Contrassegna i file o le cartelle selezionati come preferiti.
- **Modificare i tag audio:** Apri l'editor di tag integrato per modificare i tag audio per il file selezionato.
- **Rinomina:** Modifica il nome del file o della cartella selezionati.
- **Sposta:** Sposta il file o la cartella selezionati in una posizione diversa sul dispositivo.
- **Copia:** Crea una copia del file o della cartella selezionati.
- **Carica:** Carica il file o la cartella selezionati sullo storage remoto.
- **Mostra in Finder:** Per i file importati dal tuo Mac, questa azione rivela la directory in Finder che contiene il file.
- **Apri con:** Esporta il file in un'altra app.
- **Elimina:** Elimina il file o la cartella dal dispositivo. Tieni presente che questa azione è irreversibile e non puoi ripristinare i file eliminati.

## Cartelle Offline

La modalità offline è una funzione pratica che ti permette di accedere alla tua musica preferita anche quando non sei connesso a internet. Quando abiliti la modalità offline per un album, un artista, una playlist, un genere o una cartella remota, tutti i file all'interno di quella raccolta verranno scaricati automaticamente sul tuo dispositivo per la riproduzione offline. Puoi accedere comodamente a questi file nella sezione "Cartelle Offline" dell'app.

Ancora più conveniente è il fatto che quando aggiungi nuovi file al server remoto nella cartella che hai contrassegnato per l'uso offline, anche queste nuove aggiunte verranno scaricate automaticamente sul tuo dispositivo. Questo assicura che tu abbia la musica più recente pronta per il godimento offline.

Se hai mai bisogno di sincronizzare manualmente una cartella offline dal tuo servizio cloud, ecco come puoi farlo: basta toccare i tre punti situati nell'angolo in alto a destra dell'app e selezionare l'opzione "Sincronizzare". Questo ti permette di aggiornare la tua raccolta offline con i brani più recenti dal servizio cloud quando ti è comodo.

Per coloro che vogliono personalizzare ulteriormente la propria esperienza, puoi regolare le impostazioni del timeout di sincronizzazione nelle impostazioni dell'applicazione per adattarle alle tue preferenze. In questo modo, hai il pieno controllo su quando e come la tua raccolta offline viene aggiornata, assicurandoti di avere sempre le tue melodie preferite disponibili, anche senza connessione internet.

Istruzioni dettagliate su Come Riprodurre Musica Offline in Evermusic & Flacbox: Download & Sincronizzazione dal Cloud ai File Locali sono disponibili [qui](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Impostazioni Gestore File

Quasi ogni comportamento della schermata File Locali — dalla larghezza di banda della rete a dove finiscono i download a come vengono memorizzate nella cache le miniature — è configurabile in **Impostazioni → Gestore file**. Aprilo quando vuoi ottimizzare la velocità di trasferimento, risparmiare spazio di archiviazione o limitare l'app solo al Wi-Fi.

{{< cards cols="1">}}
  {{< card title="" subtitle="Schermata Impostazioni Gestore File" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-file-manager-settings.webp" >}}
{{< /cards >}}

La schermata espone ogni opzione raggruppata in sezioni chiaramente etichettate:

### Trasferimenti file (tipo di rete)

Scegli come download e upload possono utilizzare la rete:

- **Wi-Fi:** I trasferimenti vengono eseguiti solo quando sei connesso a una rete Wi-Fi. Usa questa opzione se hai un piano dati cellulare a consumo o vuoi un utilizzo dei dati mobili prevedibile.
- **Wi-Fi e Dati Cellulari:** I trasferimenti vengono eseguiti tramite Wi-Fi quando disponibile e si ripiegano sulla connessione cellulare in caso contrario. Potrebbe utilizzare il tuo piano dati mobile.

### Limite trasferimento file paralleli

Imposta il numero massimo di file che possono essere trasferiti simultaneamente. Il valore predefinito è un valore bilanciato; aumentalo per velocizzare i download di cartelle grandi, abbassalo su reti deboli o dispositivi meno recenti per mantenere stabili i trasferimenti. Un valore più alto utilizza più larghezza di banda, CPU e batteria.

### Attività di trasferimento file

Apre la coda di trasferimento in tempo reale — ogni attività di upload/download in corso e completata con barre di avanzamento, velocità e stato. Puoi mettere in pausa, riprendere e annullare singole attività qui. La stessa coda è raggiungibile dall'icona delle frecce rotanti nell'angolo in alto a sinistra della schermata File Locali.

### Trasferimenti in background

Quando abilitato, i download continuano anche dopo aver messo Evermusic in background o bloccato il dispositivo. Utile per grandi librerie su reti lente. Nota: se l'operazione consuma una quantità significativa di energia, iOS potrebbe sospendere l'app per proteggere la batteria.

### Salva file scaricati in

Definisci la destinazione predefinita per i download cloud:

- **Download:** *(predefinito)* Tutti i download arrivano nella cartella **File Locali → Download**.
- **Chiedi sempre:** Evermusic ti chiede di scegliere la cartella di destinazione per ogni download. Utile se organizzi la tua libreria per cartelle di artisti o generi sul dispositivo.
- **Cartella personalizzata:** Scegli una cartella specifica una volta e ogni download arriverà lì finché non cambi l'impostazione.

### Cartelle offline sincronizzate

Mostra ogni cartella che hai contrassegnato come "Disponibile offline" e ti permette di gestirle. Tocca una cartella per attivare manualmente una sincronizzazione, mostrarne la posizione, o **Disabilitare la modalità offline** per smettere di monitorarla e rimuovere la copia locale. Leggi di più sulle modalità offline [qui](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

### Intervallo di tempo (sincronizzazione offline)

Seleziona la frequenza con cui l'app controlla le cartelle offline per le modifiche:

- **Immediatamente:** Le cartelle offline vengono sincronizzate ogni volta che apri l'app.
- **15 / 30 / 60 minuti:** Controlli periodici mentre l'app è attiva.
- **Manuale:** Sincronizza solo quando tocchi l'azione **Sincronizzare** su una cartella.

### Mostra nomi file completi

Quando abilitato, il gestore file mostra i nomi file completi incluse le estensioni (es. `Song Title - Artist.flac`). Disabilita per nascondere le estensioni e mostrare nomi più brevi e puliti. Utile quando si lavora con formati audio misti o file non audio.

### Modifica file online

Attiva l'intera app tra modalità lettura-scrittura e sola lettura per gli account cloud collegati. Quando è abilitata la **modalità sola lettura**, ogni azione distruttiva (rinominare, spostare, eliminare, caricare, modificare i tag audio) viene nascosta dall'interfaccia utente per i file online — un'ottima rete di sicurezza se sei preoccupato per la perdita accidentale di dati in una libreria familiare condivisa o nel cloud di lavoro. I file locali rimangono completamente modificabili indipendentemente da questa impostazione.

### Copia file all'apertura

Controlla cosa succede quando apri file audio da un'altra app (es. tramite l'app Files, allegati Mail, AirDrop o estensioni di Condivisione):

- **Chiedi sempre:** Evermusic chiede ogni volta se copiare il file nella cartella Documents dell'app o aprirlo in posizione.
- **Copia sempre:** I file vengono importati nella sezione File Locali → Download, diventando copie locali permanenti.
- **Apri in posizione:** I file rimangono nella loro posizione originale e Evermusic vi accede per riferimento. Il collegamento scompare se la sorgente viene rimossa.

### Miniature per i file

Gestisce le miniature memorizzate nella cache (artwork album, forme d'onda audio, anteprime immagini) mostrate accanto ai file:

- **Carica miniature per file online:** Attiva se le miniature vengono recuperate per i file nel cloud. Disabilita per ridurre l'utilizzo dei dati su connessioni lente.
- **Tipo di rete (miniature):** Limita il caricamento delle miniature al Wi-Fi o consenti i dati cellulari.
- **Mostra in cartella:** Rivela la cartella cache delle miniature in modo da poter ispezionare cosa è archiviato.
- **Elimina tutto:** Cancella ogni miniatura memorizzata nella cache per recuperare spazio di archiviazione. L'app rigenererà le miniature quando riapri le cartelle.

### Elimina file temporanei

Svuota la cartella cache dell'applicazione (buffer audio, download a metà, anteprime immagini, artwork decodificati). Il pulsante mostra la dimensione attuale della cache così sai quanto recupererai. Sicuro da eseguire in qualsiasi momento — Evermusic rigenera tutto ciò di cui ha bisogno alla prossima riproduzione.

## Personalizzazione

Aprendo le Impostazioni dell'applicazione e andando su Personalizzazione, puoi configurare lo stile della schermata File Locali.  
Le opzioni disponibili sono:

- **Menu semplice:** Mostra le cartelle situate nella directory Documents dell'app.
- **Menu raggruppato:** Mostra il contenuto della schermata raggruppato per categorie.

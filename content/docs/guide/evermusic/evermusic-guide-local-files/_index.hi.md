## Special Folders

## डाउनलोड्स

Cloud storage service से download की गई सभी files यहां default रूप से appear होंगी। आप application settings में इस behavior को customize कर सकते हैं।

## Audio Player

यह audio player cache folder है। Default रूप से, audio player crossfade apply करने और playback optimize करने के लिए audio player queue में कुछ songs download करता है। आप app settings में audio player cache disable कर सकते हैं या simply इस folder को "Local files" से delete कर सकते हैं।

## iCloud

Files इस folder में रखें, और वे same iCloud account से connected आपके सभी devices पर sync होंगी।

## Offline Folders

जब आप किसी playlist, album, artist, या online folder को offline use के लिए available बनाते हैं, तो वे सभी files इस folder में download होंगी।
अगर specific items के लिए offline mode disable करने की आवश्यकता है, तो आप more actions menu का उपयोग कर सकते हैं या "Offline folders" में located corresponding folder delete कर सकते हैं।

## Offline Folders

  {{< card title="" subtitle="Offline Folder अधिक क्रियाएँ मेनू" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-offline-folders.webp" >}}

Offline mode एक handy feature है जो आपको internet से connected न होने पर भी अपना favorite music access करने देता है। जब आप किसी album, artist, playlist, genre, या remote folder के लिए offline mode enable करते हैं, तो उस collection के भीतर सभी files automatically offline playback के लिए आपके device पर download हो जाती हैं। आप app के "Offline Folders" section में इन files को conveniently access कर सकते हैं।

जो और भी convenient है वह यह है कि जब आप जिस folder को offline use के लिए mark किया है उसमें remote server पर new files add करते हैं, तो वे new additions भी automatically आपके device पर download हो जाएंगी। यह सुनिश्चित करता है कि आपके पास offline enjoyment के लिए latest music ready है।

अगर आपको cloud service से offline folder को manually synchronize करने की आवश्यकता हो, तो इस तरह कर सकते हैं: बस app के top-right corner में three dots पर टैप करें, और "सिंक्रनाइज़ करें" option select करें। यह आपको अपनी offline collection को cloud service के latest tracks के साथ अपनी convenience पर update करने देता है।

जो लोग अपना experience और customize करना चाहते हैं उनके लिए, आप अपनी preferences के अनुसार application settings में synchronization timeout settings adjust कर सकते हैं। इस तरह, आपके पास पूरा control है कि आपकी offline collection कब और कैसे update होती है, यह सुनिश्चित करते हुए कि आपके पास internet connection के बिना भी हमेशा आपके favorite tunes available हैं।

Evermusic और Flacbox में Offline Music कैसे Play करें: Cloud से Local Files पर Download और Sync के बारे में विस्तृत निर्देश [यहां](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/) उपलब्ध हैं।

## File Manager सेटिंग्स

Local Files screen का लगभग हर behavior — network bandwidth से लेकर downloads कहां land करते हैं से thumbnail caching तक — **सेटिंग्स → File manager** के अंतर्गत configurable है। जब भी आप transfer speed fine-tune करना चाहें, storage space save करना चाहें, या app को Wi-Fi only तक restrict करना चाहें तो इसे खोलें।

{{< cards cols="1">}}
  {{< card title="" subtitle="File Manager सेटिंग्स स्क्रीन" image="/docs/guide/evermusic/evermusic-guide-local-files/img/local-files-file-manager-settings.webp" >}}

Screen clearly-labeled sections में grouped हर option expose करती है:

### File transfers (network type)

Choose करें कि downloads और uploads को network use करने की कैसे permission है:

- **Wi-Fi:** Transfers केवल तब run होते हैं जब आप Wi-Fi network से connected हों। इसका उपयोग करें अगर आप metered cellular plan पर हैं या predictable mobile-data usage चाहते हैं।
- **Wi-Fi और Cellular Data:** Transfers Wi-Fi available होने पर run होते हैं और otherwise आपके cellular connection पर fall back करते हैं। आपके mobile data plan का उपयोग हो सकता है।

### Parallel file transfer limit

एक साथ transfer की जा सकने वाली files की maximum number set करता है। Default एक balanced value है; large folders की downloads speed up करने के लिए इसे raise करें, transfers stable रखने के लिए weak networks या older devices पर lower करें। Higher value अधिक bandwidth, CPU, और battery use करती है।

### File transfer tasks

Live transfer queue खोलता है — progress bars, speeds, और status के साथ हर ongoing और completed upload/download task। आप यहां individual tasks pause, resume, और cancel कर सकते हैं। Same queue Local Files screen के top-left corner में spinning-arrows icon से भी reachable है।

### Background transfers

Enable होने पर, downloads तब भी continue रहते हैं जब आप Evermusic को background में put करते हैं या अपने device को lock करते हैं। Slow networks पर large libraries के लिए useful। Note: अगर operation significant amount of energy consume करती है, तो iOS battery life protect करने के लिए app suspend कर सकती है।

### Save downloaded files to

Cloud downloads के लिए अपना default destination define करें:

- **डाउनलोड्स:** *(default)* सभी downloads **Local Files → डाउनलोड्स** folder में land करते हैं।
- **Always ask:** Evermusic हर download के लिए destination folder pick करने के लिए prompt करता है। Useful अगर आप device पर artist या genre folders के अनुसार अपनी library organize करते हैं।
- **Custom folder:** एक specific folder एक बार pick करें और हर download वहां land करे जब तक आप setting change न करें।

### Synchronized offline folders

हर folder show करता है जिसे आपने "Available offline" के रूप में mark किया है और उन्हें manage करने देता है। Manually sync trigger करने, उसकी location show करने, या **Disable offline mode** करने के लिए folder टैप करें ताकि इसे watch करना बंद हो और local copy हट जाए। Offline modes के बारे में अधिक पढ़ें [यहां](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)।

### Time interval (offline sync)

App कितनी बार offline folders में changes check करती है यह select करता है:

- **तुरंत:** Offline folders हर बार जब आप app खोलते हैं sync होते हैं।
- **15 / 30 / 60 minutes:** App active रहने के दौरान periodic checks।
- **Manual:** केवल तब Sync करें जब आप folder पर **सिंक्रनाइज़ करें** action टैप करते हैं।

### Show full filenames

Enable होने पर, file manager complete filenames extensions सहित display करता है (जैसे `Song Title - Artist.flac`)। Extensions hide करने और shorter, cleaner names show करने के लिए disable करें। Mixed audio formats या non-audio files के साथ काम करते समय useful।

### Edit online files

Connected cloud accounts के लिए पूरे app को read-write और read-only mode के बीच toggle करता है। **Read-only mode** enable होने पर, हर destructive action (rename, move, delete, upload, edit audio tags) online files के लिए user interface से hidden है — यह एक great safety net है अगर आप shared family library या work cloud drive में accidental data loss से concerned हैं। Local files इस setting के बावजूद पूरी तरह editable रहती हैं।

### Copy files when opening

Control करता है कि जब आप किसी दूसरे app से audio files खोलते हैं (जैसे Files app, Mail attachments, AirDrop, या Share extensions के माध्यम से) क्या होता है:

- **Always ask:** Evermusic हर बार prompt करता है कि file को app की Documents folder में copy करना है या in place खोलना है।
- **Always copy:** Files Local Files → डाउनलोड्स section में import होती हैं, permanent local copies बनती हैं।
- **Open in place:** Files अपनी original location में रहती हैं और Evermusic उन्हें reference से access करता है। Source remove होने पर link disappear हो जाता है।

### Thumbnails for files

Files के बगल में shown cached thumbnails (album artwork, audio waveforms, image previews) manage करता है:

- **Load thumbnails for online files:** Toggle करें कि cloud में files के लिए thumbnails fetch होते हैं या नहीं। Slow connections पर data usage reduce करने के लिए disable करें।
- **Network type (thumbnails):** Thumbnail loading को Wi-Fi तक restrict करें या cellular data allow करें।
- **Show in folder:** Thumbnail cache folder reveal करें ताकि आप inspect कर सकें क्या stored है।
- **Delete all:** Storage space reclaim करने के लिए हर cached thumbnail wipe करें। App जरूरत पड़ने पर thumbnails regenerate करेगी जैसे आप folders re-open करते हैं।

### Delete temporary files

Application का cache folder empty करता है (audio buffers, half-completed downloads, image previews, decoded artwork)। Button current cache size show करता है ताकि आप जान सकें कितना reclaim होगा। Anytime safe to run — Evermusic next playback पर जो भी चाहिए regenerate करता है।

## Personalization

Application सेटिंग्स खोलकर और Personalization पर जाकर, आप Local Files screen style configure कर सकते हैं।
Available options हैं

- **Plain menu:** App की Documents directory में located folders show करता है।
- **Grouped menu:** Screen content को categories के अनुसार grouped show करता है।

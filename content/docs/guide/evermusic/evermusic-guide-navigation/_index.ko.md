Evermusic는 기본 Music 앱과 매우 유사한 직관적인 사용자 인터페이스를 제공합니다. 많은 인기 음악 앱과 차별화되는 점은 내장 파일 관리자로, 사용자가 오디오 파일을 편집하고 클라우드 스토리지와 원활하게 전송할 수 있는 기능을 제공합니다.

Evermusic의 기능은 두 가지 구별된 구성 요소로 나뉩니다: "음악 라이브러리"와 "재생 목록" 탭을 통해 편리하게 접근할 수 있는 음악 라이브러리, 그리고 "연결하기"와 "로컬 파일" 탭 아래에 있는 파일 섹션.

## 섹션

iPhone을 사용하든, iPad를 사용하든, Mac의 컴팩트 모드를 사용하든 모든 앱 기능은 화면 하단의 탭 바를 통해 쉽게 접근할 수 있습니다. iPad 및 Mac 사용자를 위해 화면 왼쪽에 동일한 메뉴가 있습니다. 이 신중한 구성은 모든 앱 기능을 쉽게 접근 가능한 섹션으로 분류하여 사용자 친화적이고 효율적인 경험을 보장합니다.

**연결하기:** 이 화면에서 Google Drive, MEGA, OneDrive, Dropbox와 같은 클라우드 스토리지 서비스는 물론 컴퓨터와 개인 NAS를 손쉽게 연결할 수 있습니다.

**재생 목록:** 여기에서 모든 플레이리스트를 관리하고 플레이리스트를 만들고, 편집하고, 제거하고, 노래 순서를 변경하고, 오프라인 모드를 활성화하고, 플레이어 대기열에 노래를 추가할 수 있습니다.

**라이브러리:** 이 섹션은 아티스트, 장르, 앨범별로 그룹화된 모든 사용 가능한 트랙을 깔끔하게 표시합니다. 음악 라이브러리에 노래를 수동 또는 자동으로 추가할 수 있으며 iPod 트랙과 다운로드한 음악이 포함됩니다.

**로컬 파일:** 전송 대기열 제어를 포함하여 다운로드된 파일을 손쉽게 찾고 관리합니다. 다양한 파일 관리 작업을 사용하여 이러한 로컬 파일을 편집할 수 있습니다. 전송 대기열에 접근하려면 '로컬 파일' 화면의 왼쪽 상단 모서리에 있는 회전 화살표 아이콘을 탭합니다.

로컬 파일 섹션은 두 가지 카테고리로 나뉩니다: 응용 프로그램의 문서 디렉토리 내의 로컬 파일을 표시하는 이 응용 프로그램의 파일, 응용 프로그램의 문서 디렉토리 외부의 기기에 있는 로컬 파일을 표시하는 이 iPad/iPhone/Mac의 파일.

**설정:** 크로스페이드 재생, 음악 라이브러리 동기화, 플레이어 캐시, 앨범 아트워크 캐시, 사용자 인터페이스, 지역화와 같은 인기 옵션을 포함한 응용 프로그램 설정을 수정합니다.

**빠른 접근:** 즐겨찾는 트랙과 최근 재생한 트랙 및 파일로 원활하게 이동합니다. Mac 및 iPad 사용자를 위해 전용 섹션이 있으며, iPhone 사용자는 화면 상단에 있는 '로컬 파일', '연결하기', '음악 라이브러리' 섹션 내에서 편리하게 찾을 수 있습니다.

## iPhone의 미니 플레이어

미니 플레이어 아이콘을 탭하여 전체 화면 플레이어를 활성화하고 아래로 스와이프하는 제스처를 사용하여 숨깁니다. iPad 및 Mac에서 미니 플레이어 화면은 화면 상단에 있으며 메인 메뉴를 통해 전체 화면 플레이어를 열 때 숨길 수 있습니다.

## 미니 플레이어 창 (Mac 전용)

Mac에서 미니 플레이어 창에 접근하려면 앱 창의 오른쪽 하단 모서리로 커서를 이동하고 가능한 가장 작은 크기로 조정합니다. 그런 다음 접기 버튼(아래 화살표로 표시됨)을 탭하여 미니 플레이어 창을 활성화합니다. 미니 플레이어 창을 항상 다른 창 위에 표시하려면 Mac의 상단 메뉴 바로 이동하여 '창'을 선택한 다음 '창을 항상 위에 표시'를 선택합니다. 이 기능은 중단 없이 오디오 강의를 듣는 데 유용합니다.

## 상단 도구 모음

탐색 바 바로 아래에 편리하게 위치한 상단 도구 모음은 여러 유용한 작업에 빠르게 접근할 수 있습니다.  
아래로 스와이프하는 간단한 제스처로 이 도구 모음을 쉽게 표시하거나 숨길 수 있습니다. 작업 목록은 다음과 같습니다.

- **검색:** 현재 컨텍스트 내에서 검색을 시작합니다.
- **재생 계속:** Evermusic는 마지막으로 저장된 위치에서 현재 디렉토리, 앨범, 아티스트, 또는 플레이리스트의 오디오 플레이어 상태를 손쉽게 복원하는 '재생 계속' 기능을 제공합니다. 이 기능은 응용 프로그램 설정에서 활성화된 경우 사용 가능합니다.
- **모두 재생:** 현재 정렬 순서를 유지하면서 현재 페이지의 모든 트랙을 오디오 플레이어 대기열에 쉽게 추가합니다.
- **모두 셔플:** 즐거운 청취 경험을 위해 추가하기 전에 셔플하여 현재 페이지의 모든 트랙을 오디오 플레이어 대기열에 추가합니다.

## 컨텍스트 메뉴

컨텍스트 메뉴는 기기 전반에 걸쳐 원활한 상호 작용을 위한 추가 옵션과 작업에 빠르게 접근할 수 있습니다.

### iOS의 컨텍스트 메뉴
**탭하고 누르기:** 셀, 미니 플레이어, 또는 컴팩트 플레이어를 탭하고 눌러 컨텍스트 메뉴를 표시합니다.

### macOS의 컨텍스트 메뉴
**마우스 오른쪽 버튼 클릭:** 셀, 미니 플레이어, 또는 컴팩트 플레이어를 마우스 오른쪽 버튼으로 클릭하여 컨텍스트 메뉴를 표시합니다.

## 접근성

앱은 VoiceOver 기술을 완벽하게 지원하여 모든 구성 요소에 잘 설계된 레이블과 설명이 있습니다. VoiceOver가 활성화되면 앱이 사용자 인터페이스를 텍스트 모드로 번역하여 탐색 속도와 편의성을 향상시키기 위해 접근 가능하고 유용한 요소만 표시합니다. 설정 > 접근성 > 텍스트 모드에서 텍스트 모드를 활성화할 수도 있습니다.

### VoiceOver로 슬라이더 조정하기

1. **슬라이더 선택:** VoiceOver가 슬라이더를 알려줄 때까지 왼쪽 또는 오른쪽으로 스와이프합니다.
2. **값 조정:** 슬라이더를 두 번 탭하고 누른 다음 위 또는 아래로 드래그하여 값을 더 빠르게 조정합니다. VoiceOver가 조정하는 동안 새 값을 알립니다.

### VoiceOver로 플레이리스트에서 트랙 위치 조정하기

1. 플레이리스트를 열고 "추가" 버튼을 탭합니다.
2. "노래 순서 변경"을 선택합니다. 뷰가 편집 모드로 전환됩니다.
3. 트랙 제목 옆의 재정렬 표시기 아이콘을 탭하여 포커스를 줍니다.
4. 재정렬 표시기 아이콘을 빠르게 두 번 탭합니다. 두 번째 탭 시 셀이 이동 준비가 되었음을 나타내는 소리가 들릴 때까지 손가락을 놓지 마세요.
5. 이제 셀을 새 위치로 이동할 수 있습니다.

다른 구성 요소는 시스템 제공 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

## 위젯

Evermusic에는 모든 Apple 플랫폼에서 앱의 주요 부분을 표시하는 네 가지 홈 화면/잠금 화면 위젯이 포함되어 있습니다 — Evermusic를 열지 않고도 재생을 보거나 제어할 수 있습니다:

- **지금 재생 중** — 현재 트랙 아트워크, 제목, 아티스트, 실시간 재생 컨트롤(재생/일시 중지, 다음, 이전). 대기열이 진행되면 실시간으로 업데이트됩니다.
- **즐겨찾기** — 즐겨찾는 노래를 한눈에 볼 수 있습니다. 항목을 탭하면 플레이어에서 바로 열립니다.
- **가사** — 현재 재생 중인 트랙의 가사(내장 또는 LRC), 더 큰 크기에서 줄별로 스크롤됩니다.
- **최근 재생** — 최근에 재생한 노래에 빠르게 접근하고 한 번 탭으로 재개합니다.

네 가지 위젯 모두 소형, 중형, 대형 크기로 제공되어 화면에 맞는 레이아웃을 선택할 수 있습니다.

### iPhone에서 위젯 추가 (홈 화면)

1. 아이콘이 흔들리기 시작할 때까지 홈 화면의 빈 영역을 길게 탭합니다.
2. 왼쪽 상단 모서리의 **+** (편집) 버튼을 탭한 다음 **위젯 추가**를 탭합니다.
3. **Evermusic** (또는 **Evermusic Pro**)을 검색하고 탭합니다.
4. 왼쪽/오른쪽으로 스와이프하여 지금 재생 중, 즐겨찾기, 가사, 또는 최근 재생 중에서 선택하고 크기를 선택합니다(소형 / 중형 / 대형).
5. **위젯 추가**를 탭하고 원하는 위치에 배치한 다음 **완료**를 탭합니다.

### iPhone에서 위젯 추가 (잠금 화면)

1. **사용자 지정**이 나타날 때까지 잠금 화면을 길게 탭한 다음 **사용자 지정 → 잠금 화면**을 탭합니다.
2. 시계 아래의 위젯 영역을 탭하고 목록에서 **Evermusic**를 선택합니다.
3. 위젯을 선택하고 **완료 → 배경화면 쌍으로 설정** (또는 **현재 배경화면으로 설정**)을 탭합니다.

### iPad에서 위젯 추가

iPad는 오늘 보기 위젯과 홈 화면 위젯을 모두 지원합니다:

1. 아이콘이 흔들릴 때까지 홈 화면을 길게 탭합니다.
2. 상단 모서리의 **+**를 탭하고 **Evermusic**를 검색하고 위젯과 크기를 선택한 다음 **위젯 추가**를 탭합니다.

iPadOS 15 이상에서는 여러 Evermusic 위젯을 세로로 쌓아 같은 슬롯에서 지금 재생 중, 즐겨찾기, 최근 재생 사이를 스와이프할 수 있습니다.

### Mac에서 위젯 추가

1. **데스크톱**을 마우스 오른쪽 버튼으로 클릭(macOS Sonoma 이상)하고 **위젯 편집**을 선택하거나 **알림 센터**를 열고 하단의 **위젯 편집**을 클릭합니다.
2. 위젯 갤러리에서 **Evermusic**를 검색합니다.
3. 위젯을 데스크톱 또는 알림 센터로 드래그하고 크기를 선택한 다음 **완료**를 클릭합니다.

### CarPlay 대시보드에 Evermusic 위젯 추가

**iOS 17**부터 CarPlay는 **CarPlay 대시보드**(지도, 캘린더, 지금 재생 중이 있는 다중 타일 화면)에서 호환되는 잠금 화면/홈 화면 위젯을 직접 지원합니다. Evermusic 위젯은 iPhone에 설치되면 자동으로 표시됩니다:

1. iPhone에서 **잠금 화면 레이아웃**에 Evermusic 위젯을 추가합니다(사용자 지정 → 잠금 화면 → 위젯 슬롯 → Evermusic). 위에 설명된 대로. 동일한 위젯 인스턴스가 CarPlay에 미러링됩니다.
2. USB 또는 무선 CarPlay를 통해 iPhone을 차량에 연결합니다.
3. CarPlay 화면에서 왼쪽 하단 모서리의 **대시보드** 아이콘(네 타일 격자)을 탭합니다.
4. 대시보드 타일 중 하나를 탭하고 누르면 편집 모드가 되고, 변경하려는 타일의 **+**를 탭합니다.
5. 사용 가능한 CarPlay 위젯 목록에서 **Evermusic**를 선택하고 지금 재생 중, 즐겨찾기, 가사, 또는 최근 재생을 선택합니다.
6. **완료**를 탭하면 위젯이 이후의 모든 CarPlay 세션에 대한 대시보드에 고정됩니다.

CarPlay 위젯은 음악이 변경되면 실시간으로 업데이트되며 운전 중 화면을 내려다 보지 않아도 되도록 큰 손가락에 친화적입니다.

## Apple CarPlay

Evermusic에는 차량 디스플레이에 최적화된 완전한 기능의 **Apple CarPlay** 인터페이스(iOS 전용)가 포함되어 있습니다. iPhone이 USB 또는 무선으로 호환되는 CarPlay 헤드 유닛에 연결되면 Evermusic가 Apple Music 및 Spotify와 함께 CarPlay 앱 격자에 나타나 도로에서 클라우드 라이브러리를 스트리밍할 준비가 됩니다.

### CarPlay에서 제공하는 기능

CarPlay UI는 Apple이 차량에 맞게 조정한 레이아웃에서 라이브러리의 가장 많이 사용되는 부분을 반영합니다. 메인 메뉴에서 다음을 탐색할 수 있습니다:

- **라이브러리** — 노래, 앨범, 앨범 아티스트, 아티스트, 장르, 작곡가, 최근 재생, 즐겨찾기.
- **재생 목록** — 만들거나 가져온 모든 플레이리스트(오프라인 동기화 플레이리스트 포함).
- **로컬 파일** — 앱의 문서 디렉토리에 있는 폴더와 오프라인 폴더.
- **연결하기** — 연결된 클라우드 계정으로 주차 중에 Google Drive, Dropbox, iCloud Drive, MEGA, Synology, 또는 다른 소스에서 직접 노래를 선택할 수 있습니다.

모든 트랙은 Evermusic의 완전한 오디오 엔진을 통해 재생됩니다 — **10밴드 이퀄라이저**, **크로스페이드**, **무갭 재생**, **공간 오디오**, **재생 속도 제어** 포함 — 따라서 차 안에서 듣는 음악은 추가 설정 없이 전화기에서와 동일하게 들립니다.

### 지금 재생 중 템플릿

표준 CarPlay **지금 재생 중** 템플릿은 모든 운전자가 기대하는 핸들에 친화적인 컨트롤을 제공합니다: 큰 아트워크, 재생/일시 중지, 이전/다음, 스크러버, **다음 재생** 대기열. **설정 → 오디오 플레이어 → 개인화 → 잠금 화면의 재생 컨트롤**에서 북마크 추가 및 즐겨찾기에 추가를 잠금 화면/CarPlay 작업 세트에 활성화하여 눈을 도로에서 떼지 않고도 트랙을 즐겨찾기로 표시할 수 있습니다.

### 안전 운전 팁

CarPlay는 도로에 주의를 유지하도록 설계되었으며 Evermusic도 동일한 원칙을 따릅니다. 안전을 유지하기 위해 다음 습관을 사용하세요:

- **집에서 미리 플레이리스트를 만드세요.** 도로에서 수십 개의 앨범을 탐색하는 것은 주의를 산만하게 합니다. 미리 몇 개의 "드라이브" 플레이리스트를 만들고 각각에 **오프라인 모드**를 활성화하여 터널 및 농촌 지역에서 신호가 없어도 재생되도록 합니다.
- **"연결 시 재생 일시 중지" 활성화** **설정 → CarPlay → 연결 시 재생 일시 중지**. CarPlay가 연결되는 순간 Evermusic를 중지하여 헤드폰 수준의 갑작스러운 볼륨 폭발을 방지합니다.
- **음성 제어를 사용하세요.** 핸들의 **음성 명령** 버튼을 누르거나 *"헤이 Siri, Evermusic에서 내 드라이브 플레이리스트 재생"*이라고 말하세요 — Siri 인텐트 지원으로 화면을 터치하지 않고 재생을 시작하고, 트랙을 건너뛰고, 볼륨을 조정할 수 있습니다.
- **지금 재생 중 위젯을 CarPlay 대시보드에 고정**하세요(위 섹션 참조). 그러면 일시 중지하거나 건너뛰기 위해 전체 앱을 열 필요가 없습니다 — 대부분의 작업은 대시보드에서 한 번 탭으로 수행됩니다.
- **메타데이터 리더 속도를 낮추세요.** 큰 라이브러리와 함께 셀룰러 데이터를 사용하는 경우 **설정 → 음악 라이브러리 → 메타데이터 읽기**를 **현재 노래**로 설정하여 운전 중에 앱이 백그라운드에서 수천 개의 클라우드 요청을 가져오지 않도록 합니다.
- **라이브러리가 큰 경우 CarPlay에서 이미지를 비활성화합니다.** **설정 → CarPlay → 이미지 표시 → 꺼짐**은 느린 헤드 유닛에서 목록을 빠르게 유지하고 시각적 혼란을 줄입니다.
- **운전 중 가사 위젯을 읽지 마세요.** 빨간 신호등이나 동승자와 함께 있을 때는 괜찮지만, 그렇지 않으면 도로에 주의를 기울이고 대신 듣는 것이 좋습니다.

운전 중에 문제가 발생하면 가장 안전한 방법은 항상 차를 세우고 iPhone을 직접 사용하는 것입니다 — 속도를 내면서 헤드 유닛을 조작하지 마세요.

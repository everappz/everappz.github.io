Evervideo 提供简洁直观的界面，对于使用过 iOS 音乐或媒体播放器的人来说十分熟悉——同时增加了真正的内置文件管理器，让您可以在不离开应用的情况下编辑、重命名、移动和整理云存储及设备上的视频文件。

与大多数媒体应用不同，Evervideo 将云账户、NAS 共享、媒体服务器和本地文件整合到单一的「文件」标签页中——无需在不同屏幕之间切换。这使得将视频从 Plex 服务器移动到 iCloud Drive 文件夹再到 iPhone Documents 文件夹成为一个屏幕、一次操作的事情。

## 标签页

无论您使用 iPhone、iPad 还是 Mac 上的紧凑窗口模式，所有应用功能都可以从屏幕底部的标签栏访问。在 iPad 和 Mac 上，同一菜单在左侧边栏中也可用，以便为视频内容保留更多屏幕空间。

- **最近使用** — Evervideo 默认在此打开。列出您最近观看的每个视频，带有缩略图和每个文件的观看进度，可一键恢复任意视频。
- **收藏夹** — 您标记为收藏的每个视频和文件夹，每个条目都有快速更多操作按钮。
- **媒体资料库** — 您的媒体资料库，按专辑、流派、播放列表、最近使用和收藏夹分组。您还可以浏览来自 Photos 资料库（所有视频、短、中、长、屏幕录制和您的相册）和 Music 资料库（专辑、流派、播放列表）的视频。
- **播放列表** — 在一处管理所有视频播放列表：创建、重命名、编辑封面、重新排序、启用离线模式、导入 M3U / M3U8 / CUE 文件，以及导出到 M3U / CSV / TXT。
- **文件** — 多合一文件管理器。「文件」标签页整合了大多数应用会分成「连接」和「本地文件」的内容：云存储（Google Drive、Dropbox、OneDrive、iCloud Drive、MEGA、Box、pCloud、Yandex Disk 等）、SMB / WebDAV / FTP / SFTP / NFS / DLNA 上的 NAS 共享、自托管媒体服务器（Plex、Jellyfin、Emby、Subsonic、Navidrome）、RTSP IP 摄像头流、S3 兼容对象存储、Documents 文件夹、此 iPhone / iPad / Mac 上的文件、传输队列和可用设备部分。左上角带有旋转箭头的传输图标可即时访问活动下载和上传。
- **设置** — 更改应用设置：媒体播放器（画中画、硬件解码器、视频均衡器、视频缩放、旋转）、音频（均衡器、采样率、声道、IO 缓冲）、字幕（主 / 副轨道、字体、外部文件）、媒体资料库、文件管理器、主屏幕小组件、个性化、语言、密码和备份与恢复。
- **快速访问** — 位于「文件」和「媒体资料库」屏幕顶部，可在每个上下文中快速访问最近使用和收藏夹。

## 紧凑媒体播放器

Evervideo 包含一个独特的紧凑媒体播放器，在您浏览资料库、文件管理器、设置或任何其他标签页时始终显示在应用顶部。点击紧凑播放器可将其展开为全屏媒体播放器；在全屏播放器上点击并向下滑动可将其折叠回紧凑视图，而无需停止播放。

这意味着您可以在浏览资料库寻找下一个播放内容、管理下载或调整应用设置时继续观看视频——永不停止播放或失去位置。

在 macOS 上，紧凑播放器也可以作为单独的浮动迷你播放器窗口分离，始终置于您正在使用的其他所有应用之上——非常适合教程、讲座或在工作时在旁边播放电影。

## 画中画（PiP）

在 iPhone 和 iPad 上，Evervideo 完全支持画中画（PiP）：当您切换到另一个应用时，视频继续在可移动的浮动窗口中播放。点击全屏播放器上的 PiP 图标手动进入 PiP 模式；将浮动窗口拖到任何角落；用两指捏合来调整大小；点击一次调出基本播放 / 暂停 / 跳过控制；点击小展开按钮返回完整的 Evervideo 应用。

PiP 适用于 Evervideo 播放的所有视频格式，包括云端流媒体文件和 RTSP 流，并在手机锁定时继续运行。

## 更多操作

屏幕上几乎每个内容项都有一个「更多操作」按钮（「⋯」三点图标）。点击它可打开上下文敏感菜单，其中包含该项目的所有可用操作——播放下一个、稍后播放、添加到播放列表、添加到收藏夹、编辑标签、下载、分享、重命名、移动等。长列表可垂直滚动，让您无需挤占主界面就能访问不常用的操作。

## 顶部工具栏

顶部工具栏位于导航栏正下方，可快速访问每个屏幕最常用的操作。您可以通过简单的向下滑动手势显示或隐藏它。

- **搜索** — 在当前屏幕（文件夹、播放列表、专辑、流派、资料库或队列）中开始搜索。
- **继续播放** — Evervideo 可以恢复当前目录、专辑、流派或播放列表的播放器状态和上次位置。当在设置中启用**保存音频 / 媒体播放器状态**时，此选项会出现。
- **播放全部** — 将当前页面的所有视频添加到播放器队列，保留当前排序顺序。
- **随机播放** — 将当前页面的所有视频随机添加到播放器队列。非常适合重新发现长期未看的视频集。

## 上下文菜单

上下文菜单提供对其他选项和操作的快速访问，实现跨设备的无缝交互。

### iOS / iPadOS 上的上下文菜单

在单元格、紧凑播放器或任何缩略图上长按以显示上下文菜单。您可以在**设置 → 个性化 → 上下文菜单样式**中更改上下文菜单的呈现样式。

### macOS 上的上下文菜单

右键点击单元格、紧凑播放器或视频缩略图以显示上下文菜单。在全屏播放器中，右键点击正在播放的视频也有效。

## 主屏幕小组件

Evervideo 在 iPhone、iPad 和 Mac 上支持小组件，让您一目了然地看到正在播放的视频，并一键恢复播放。

首先，在应用中启用小组件：

1. 打开**设置 → 小组件 → 启用小组件**（小组件更新使用少量能量，因此开关默认关闭）。

然后在设备上添加小组件：

**在 iPhone 或 iPad 上**
1. 长按**主屏幕**或**锁定屏幕**的空白区域。
2. 点击顶部角落的 **+** 按钮，搜索 **Evervideo**。
3. 选择小组件大小，点击**添加小组件**。

**在 Mac 上**
1. 点击菜单栏中的日期和时间以打开**通知中心**，然后点击底部的**编辑小组件**。（您也可以按住 Control 键点击桌面，选择**编辑小组件**直接放置在桌面上。）
2. 在小组件库中搜索 **Evervideo**。
3. 点击小组件大小，或将其拖到通知中心或桌面。

小组件在每个平台上显示当前视频的标题、缩略图和基本控件。

## 无障碍功能

Evervideo 完全支持 **VoiceOver**：每个组件都有描述性标签和合理的无障碍提示。VoiceOver 激活时，应用自动切换到**文字模式**——含有大量图像的装饰性元素被隐藏，只显示有意义的控件，提高屏幕阅读器用户的导航速度和清晰度。您也可以在**设置 → 无障碍 → 文字模式**中手动启用文字模式。

### 使用 VoiceOver 调整滑块

1. **选择滑块** — 向左或向右滑动，直到 VoiceOver 宣布它。
2. **调整值** — 双击并按住滑块，然后向上或向下拖动以快速更改值。VoiceOver 会在您操作时宣布每个新值。

### 使用 VoiceOver 调整播放列表中的曲目位置

1. 打开播放列表，点击**更多**按钮。
2. 选择**重新排列歌曲**。列表切换到编辑模式。
3. 点击曲目标题旁边的重新排序指示器图标以聚焦。
4. 快速**双击**重新排序指示器。第二次点击时不要松开手指——按住直到听到声音表示单元格可以移动。
5. 现在可以将单元格移动到新位置。

其他组件按预期工作，使用系统提供的 VoiceOver 模式。

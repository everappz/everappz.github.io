Evervideo 提供簡潔直觀的介面，對於使用過 iOS 音樂或媒體播放器的人來說十分熟悉——同時增加了真正的內建檔案管理員，讓您可以在不離開應用程式的情況下編輯、重新命名、移動和整理雲端儲存及裝置上的視訊檔案。

與大多數媒體應用程式不同，Evervideo 將雲端帳戶、NAS 共享、媒體伺服器和本機檔案整合到單一的「檔案」標籤頁中——無需在不同畫面之間切換。這使得將視訊從 Plex 伺服器移動到 iCloud Drive 資料夾再到 iPhone Documents 資料夾成為一個畫面、一次操作的事情。

## 標籤頁

無論您使用 iPhone、iPad 還是 Mac 上的緊湊視窗模式，所有應用程式功能都可以從畫面底部的標籤列存取。在 iPad 和 Mac 上，同一選單在左側邊欄中也可用，以便為視訊內容保留更多畫面空間。

- **最近使用的** — Evervideo 預設在此開啟。列出您最近觀看的每個視訊，附帶縮圖和每個檔案的觀看進度，可一鍵恢復任意視訊。
- **最愛項目** — 您標記為最愛的每個視訊和資料夾，每個項目都有快速更多操作按鈕。
- **媒體資料庫** — 您的媒體資料庫，按專輯、流派、播放清單、最近使用的和最愛項目分組。您還可以瀏覽來自 Photos 資料庫（所有視訊、短、中、長、螢幕錄製和您的相簿）和 Music 資料庫（專輯、流派、播放清單）的視訊。
- **播放清單** — 在一處管理所有視訊播放清單：建立、重新命名、編輯封面、重新排序、啟用離線模式、匯入 M3U / M3U8 / CUE 檔案，以及匯出到 M3U / CSV / TXT。
- **檔案** — 多合一檔案管理員。「檔案」標籤頁整合了大多數應用程式會分成「連接」和「本機檔案」的內容：雲端儲存（Google Drive、Dropbox、OneDrive、iCloud Drive、MEGA、Box、pCloud、Yandex Disk 等）、SMB / WebDAV / FTP / SFTP / NFS / DLNA 上的 NAS 共享、自行託管媒體伺服器（Plex、Jellyfin、Emby、Subsonic、Navidrome）、RTSP IP 攝影機串流、S3 相容物件儲存、Documents 資料夾、此 iPhone / iPad / Mac 上的檔案、傳輸佇列和可用設備部分。左上角帶有旋轉箭頭的傳輸圖示可即時存取活動下載和上傳。
- **設定** — 變更應用程式設定：媒體播放器（子母畫面、硬體解碼器、視訊等化器、視訊縮放、旋轉）、音訊（等化器、取樣率、聲道、IO 緩衝）、字幕（主 / 副軌道、字型、外部檔案）、媒體資料庫、檔案管理員、主畫面小工具、個人化、語言、密碼和備份與還原。
- **快速存取** — 位於「檔案」和「媒體資料庫」畫面頂部，可在每個上下文中快速存取最近使用的和最愛項目。

## 緊湊媒體播放器

Evervideo 包含一個獨特的緊湊媒體播放器，在您瀏覽資料庫、檔案管理員、設定或任何其他標籤頁時始終顯示在應用程式頂部。點選緊湊播放器可將其展開為全螢幕媒體播放器；在全螢幕播放器上點選並向下滑動可將其折疊回緊湊視圖，而無需停止播放。

這意味著您可以在瀏覽資料庫尋找下一個播放內容、管理下載或調整應用程式設定時繼續觀看視訊——永不停止播放或失去位置。

在 macOS 上，緊湊播放器也可以作為單獨的浮動迷你播放器視窗分離，始終置於您正在使用的其他所有應用程式之上——非常適合教學影片、講座或在工作時在旁邊播放電影。

## 子母畫面（PiP）

在 iPhone 和 iPad 上，Evervideo 完全支援子母畫面（PiP）：當您切換到另一個應用程式時，視訊繼續在可移動的浮動視窗中播放。點選全螢幕播放器上的 PiP 圖示手動進入 PiP 模式；將浮動視窗拖到任何角落；用兩指捏合來調整大小；點選一次調出基本播放 / 暫停 / 跳過控制；點選小展開按鈕返回完整的 Evervideo 應用程式。

PiP 適用於 Evervideo 播放的所有視訊格式，包括雲端串流檔案和 RTSP 串流，並在手機鎖定時繼續執行。

## 更多操作

畫面上幾乎每個內容項目都有一個「更多操作」按鈕（「⋯」三點圖示）。點選它可開啟上下文敏感選單，其中包含該項目的所有可用操作——播放下一個、稍後播放、加入播放清單、加入最愛、編輯標籤、下載、分享、重新命名、移動等。長清單可垂直捲動，讓您無需擠占主介面就能存取不常用的操作。

## 頂部工具列

頂部工具列位於導覽列正下方，可快速存取每個畫面最常用的操作。您可以透過簡單的向下滑動手勢顯示或隱藏它。

- **搜尋** — 在目前畫面（資料夾、播放清單、專輯、流派、資料庫或佇列）中開始搜尋。
- **繼續播放** — Evervideo 可以恢復目前目錄、專輯、流派或播放清單的播放器狀態和上次位置。當在設定中啟用**儲存音訊 / 媒體播放器狀態**時，此選項會出現。
- **播放全部** — 將目前頁面的所有視訊加入播放器佇列，保留目前排序順序。
- **隨機播放** — 將目前頁面的所有視訊隨機加入播放器佇列。非常適合重新發現長期未看的視訊集。

## 上下文選單

上下文選單提供對其他選項和操作的快速存取，實現跨裝置的無縫互動。

### iOS / iPadOS 上的上下文選單

在儲存格、緊湊播放器或任何縮圖上長按以顯示上下文選單。您可以在**設定 → 個人化 → 上下文選單樣式**中變更上下文選單的呈現樣式。

### macOS 上的上下文選單

右鍵點選儲存格、緊湊播放器或視訊縮圖以顯示上下文選單。在全螢幕播放器中，右鍵點選正在播放的視訊也有效。

## 主畫面小工具

Evervideo 在 iPhone、iPad 和 Mac 上支援小工具，讓您一目瞭然地看到正在播放的視訊，並一鍵恢復播放。

首先，在應用程式中啟用小工具：

1. 開啟**設定 → 小工具 → 啟用小工具**（小工具更新使用少量能量，因此開關預設關閉）。

然後在裝置上新增小工具：

**在 iPhone 或 iPad 上**
1. 長按**主畫面**或**鎖定螢幕**的空白區域。
2. 點選頂部角落的 **+** 按鈕，搜尋 **Evervideo**。
3. 選擇小工具大小，點選**新增小工具**。

**在 Mac 上**
1. 點選選單列中的日期和時間以開啟**通知中心**，然後點選底部的**編輯小工具**。（您也可以按住 Control 鍵點選桌面，選擇**編輯小工具**直接放置在桌面上。）
2. 在小工具庫中搜尋 **Evervideo**。
3. 點選小工具大小，或將其拖到通知中心或桌面。

小工具在每個平台上顯示目前視訊的標題、縮圖和基本控制項。

## 無障礙功能

Evervideo 完全支援 **VoiceOver**：每個元件都有描述性標籤和合理的無障礙提示。VoiceOver 啟動時，應用程式自動切換到**文字模式**——含有大量影像的裝飾性元素被隱藏，只顯示有意義的控制項，提升螢幕閱讀器使用者的導覽速度和清晰度。您也可以在**設定 → 無障礙 → 文字模式**中手動啟用文字模式。

### 使用 VoiceOver 調整滑桿

1. **選擇滑桿** — 向左或向右滑動，直到 VoiceOver 宣告它。
2. **調整值** — 雙擊並按住滑桿，然後向上或向下拖動以快速變更值。VoiceOver 會在您操作時宣告每個新值。

### 使用 VoiceOver 調整播放清單中的曲目位置

1. 開啟播放清單，點選**更多**按鈕。
2. 選擇**重新排列歌曲**。清單切換到編輯模式。
3. 點選曲目標題旁邊的重新排序指示器圖示以聚焦。
4. 快速**雙擊**重新排序指示器。第二次點選時不要放開手指——按住直到聽到聲音表示儲存格可以移動。
5. 現在可以將儲存格移動到新位置。

其他元件按預期工作，使用系統提供的 VoiceOver 模式。

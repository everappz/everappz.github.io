Evervideo menawarkan antarmuka yang bersih dan intuitif yang terasa familiar bagi siapa saja yang pernah menggunakan pemutar musik atau media di iOS — tetapi menambahkan manajer file bawaan yang sesungguhnya sehingga Anda dapat mengedit, mengganti nama, memindahkan, dan mengatur file video di penyimpanan cloud dan di perangkat Anda tanpa meninggalkan aplikasi.

Tidak seperti kebanyakan aplikasi media, Evervideo menggabungkan akun cloud, berbagi NAS, server media, dan file lokal Anda ke dalam satu tab File yang terpadu — sehingga Anda tidak perlu berpindah antara layar terpisah. Hal ini membuat pemindahan video dari server Plex, ke folder iCloud Drive, ke folder Dokumen iPhone Anda menjadi operasi satu layar, satu ketuk.

## Tab

Baik Anda menggunakan iPhone, iPad, atau mode jendela kompak di Mac, semua fitur aplikasi dapat diakses dari bilah tab di bagian bawah layar. Di iPad dan Mac, menu yang sama tersedia di sidebar kiri sehingga Anda dapat menjaga lebih banyak layar untuk konten video.

- **Terbaru** — Evervideo dibuka di sini secara default. Daftar setiap video yang baru saja Anda tonton, dengan thumbnail dan kemajuan pemutaran per file, sehingga Anda dapat melanjutkan salah satunya dengan satu ketuk.
- **Favorit** — setiap video dan folder yang telah Anda tandai sebagai favorit, dengan Lebih banyak tindakan cepat pada setiap item.
- **Perpustakaan Media** — perpustakaan media Anda, dikelompokkan berdasarkan Album, Genre, Daftar Putar, Terbaru, dan Favorit. Anda juga dapat menelusuri video dari perpustakaan Foto (Semua Video, Pendek, Sedang, Panjang, Rekaman Layar, dan Album Foto Anda) dan perpustakaan Music (Album, Genre, Daftar Putar).
- **Daftar Putar** — kelola setiap daftar putar video di satu tempat: buat, ganti nama, edit seni sampul, susun ulang, aktifkan mode offline, impor file M3U / M3U8 / CUE, dan ekspor ke M3U / CSV / TXT.
- **File** — manajer file all-in-one. Tab File menggabungkan apa yang sebagian besar aplikasi akan pisahkan menjadi 'Koneksi' dan 'File Lokal': penyimpanan cloud (Google Drive, Dropbox, OneDrive, iCloud Drive, MEGA, Box, pCloud, Yandex Disk, dan banyak lagi), berbagi NAS melalui SMB / WebDAV / FTP / SFTP / NFS / DLNA, server media yang dihosting sendiri (Plex, Jellyfin, Emby, Subsonic, Navidrome), stream kamera IP RTSP, penyimpanan objek kompatibel S3, folder Dokumen Anda, file di iPhone / iPad / Mac ini, antrean transfer, dan bagian Perangkat yang Tersedia. Ikon Transfer berbentuk panah berputar di pojok kiri atas memberikan akses instan ke unduhan dan unggahan aktif.
- **Pengaturan** — ubah pengaturan aplikasi: pemutar media (Picture-in-Picture, decoder perangkat keras, equalizer video, penskalaan video, rotasi), audio (equalizer, laju sampel, saluran, buffer IO), subtitle (trek primer / sekunder, font, file eksternal), perpustakaan media, manajer file, widget Layar Utama, personalisasi, bahasa, kode sandi, dan cadangan & pemulihan.
- **Akses Cepat** — terletak di bagian atas layar File dan Perpustakaan Media untuk akses cepat ke terbaru dan favorit dalam setiap konteks.

## Pemutar Media Kompak

Evervideo menyertakan pemutar media kompak unik yang tetap terlihat di bagian atas aplikasi saat Anda menelusuri perpustakaan, manajer file, pengaturan, atau tab lainnya. Ketuk pemutar kompak untuk memperluasnya menjadi pemutar media layar penuh; ketuk dan geser ke bawah pada pemutar layar penuh untuk melipatnya kembali ke tampilan kompak tanpa menghentikan pemutaran.

Ini berarti Anda dapat terus menonton video sambil menelusuri perpustakaan untuk apa yang akan diputar berikutnya, mengelola unduhan, atau menyesuaikan pengaturan aplikasi — tanpa pernah menghentikan pemutaran atau kehilangan posisi Anda.

Di macOS, pemutar kompak juga dapat dilepas sebagai jendela mini-player mengambang terpisah sehingga tetap di atas setiap aplikasi lain yang Anda gunakan — sempurna untuk tutorial, kuliah, atau menonton film sambil bekerja.

## Picture-in-Picture (PiP)

Di iPhone dan iPad, Evervideo sepenuhnya mendukung Picture-in-Picture (PiP), jadi saat Anda beralih ke aplikasi lain, video terus diputar dalam jendela mengambang yang dapat dipindahkan. Ketuk ikon PiP pada pemutar layar penuh untuk masuk ke mode PiP secara manual; seret jendela mengambang ke sudut mana pun; cubit untuk mengubah ukuran; ketuk sekali untuk memunculkan kontrol dasar putar / jeda / lewati; dan ketuk tombol perluas kecil untuk kembali ke Evervideo penuh.

PiP bekerja dengan semua format video yang diputar Evervideo, termasuk file yang di-streaming cloud dan stream RTSP, dan terus berjalan saat ponsel Anda terkunci.

## Lebih banyak tindakan

Hampir setiap item konten di layar memiliki tombol Lebih banyak tindakan (ikon tiga titik "⋯"). Ketuk untuk membuka menu yang sensitif konteks dengan setiap tindakan yang tersedia untuk item tersebut — putar berikutnya, putar nanti, tambahkan ke daftar putar, tambahkan ke favorit, edit tag, unduh, bagikan, ganti nama, pindahkan, dan sebagainya. Daftar panjang bergulir secara vertikal sehingga Anda dapat menjangkau tindakan yang kurang umum tanpa memenuhi UI utama.

## Toolbar Atas

Toolbar atas, terletak tepat di bawah bilah navigasi, menyediakan akses cepat ke tindakan per layar yang paling umum. Anda dapat menampilkan atau menyembunyikannya dengan gerakan geser ke bawah sederhana.

- **Cari** — mulai pencarian dalam layar saat ini (folder, daftar putar, album, genre, perpustakaan, atau antrean).
- **Lanjutkan Pemutaran** — Evervideo dapat memulihkan status pemutar dan posisi terakhir untuk direktori, album, genre, atau daftar putar saat ini. Opsi ini muncul saat **Simpan Status Audio / Pemutar Media** diaktifkan di pengaturan.
- **Putar Semua** — menambahkan setiap video dari halaman saat ini ke antrean pemutar, mempertahankan urutan sortir saat ini.
- **Acak Semua** — menambahkan setiap video dari halaman saat ini ke antrean pemutar, diacak. Bagus untuk menemukan kembali album klip yang sudah lama terlupakan.

## Menu Konteks

Menu konteks menyediakan akses cepat ke opsi dan tindakan tambahan untuk interaksi yang mulus di seluruh perangkat.

### Menu Konteks di iOS / iPadOS

Tekan dan Tahan pada sel, pemutar kompak, atau thumbnail mana pun untuk memunculkan menu konteks. Anda dapat mengubah gaya presentasi menu konteks di Pengaturan → Personalisasi → Gaya Menu Konteks.

### Menu Konteks di macOS

Klik kanan pada sel, pemutar kompak, atau thumbnail video untuk menampilkan menu konteks. Klik kanan juga berfungsi pada video yang sedang diputar di pemutar layar penuh.

## Widget Layar Utama

Evervideo mendukung widget di iPhone, iPad, dan Mac sehingga Anda dapat melihat video yang sedang diputar sekilas dan melanjutkan pemutaran dengan satu ketuk.

Pertama, aktifkan widget di aplikasi:

1. Buka **Pengaturan → Widget → Aktifkan Widget** (pembaruan widget menggunakan sedikit energi, sehingga tombol ini mati secara default).

Kemudian tambahkan widget di perangkat Anda:

**Di iPhone atau iPad**
1. Tekan lama area kosong **Layar Utama** atau **Layar Kunci** Anda.
2. Ketuk tombol **+** di pojok atas dan cari **Evervideo**.
3. Pilih ukuran widget dan ketuk **Tambahkan Widget**.

**Di Mac**
1. Klik tanggal dan waktu di bilah menu untuk membuka **Notification Center**, lalu klik **Edit Widget** di bagian bawah. (Anda juga bisa Control-klik desktop dan pilih **Edit Widget** untuk menempatkannya langsung di desktop Anda.)
2. Cari **Evervideo** di galeri widget.
3. Klik ukuran widget, atau seret ke Notification Center atau desktop.

Widget menampilkan judul video saat ini, thumbnail, dan kontrol dasar di setiap platform.

## Aksesibilitas

Evervideo sepenuhnya dapat diakses dengan **VoiceOver**: setiap komponen memiliki label deskriptif dan petunjuk aksesibilitas yang masuk akal. Saat VoiceOver aktif, aplikasi secara otomatis beralih ke **Mode Teks** — elemen dekoratif yang berat gambar disembunyikan dan hanya kontrol yang bermakna yang ditampilkan, meningkatkan kecepatan navigasi dan kejelasan bagi pengguna pembaca layar. Anda juga dapat mengaktifkan Mode Teks secara manual di Pengaturan → Aksesibilitas → Mode Teks.

### Menyesuaikan Slider dengan VoiceOver

1. **Pilih slider** — geser kiri atau kanan hingga VoiceOver mengumumkannya.
2. **Sesuaikan nilainya** — ketuk dua kali dan tahan slider, lalu seret ke atas atau ke bawah untuk mengubah nilai dengan cepat. VoiceOver mengumumkan setiap nilai baru saat Anda melakukan perubahan.

### Menyesuaikan Posisi Trek dalam Daftar Putar dengan VoiceOver

1. Buka daftar putar dan ketuk tombol **Lainnya**.
2. Pilih **Atur Ulang Lagu**. Daftar beralih ke mode pengeditan.
3. Ketuk ikon indikator urutan ulang dekat judul trek untuk memberikan fokus.
4. **Ketuk dua kali** indikator urutan ulang. Pada ketukan kedua, jangan lepaskan jari Anda — tahan hingga Anda mendengar suara yang menunjukkan sel siap dipindahkan.
5. Anda sekarang dapat memindahkan sel ke posisi barunya.

Komponen lain bekerja seperti yang diharapkan, menggunakan pola VoiceOver yang disediakan sistem.

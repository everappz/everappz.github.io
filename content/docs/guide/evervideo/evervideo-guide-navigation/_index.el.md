Το Evervideo προσφέρει μια καθαρή, διαισθητική διεπαφή που φαίνεται οικεία σε οποιονδήποτε έχει χρησιμοποιήσει μουσικό ή media player σε iOS — αλλά προσθέτει έναν πραγματικό ενσωματωμένο διαχειριστή αρχείων ώστε να μπορείτε να επεξεργαστείτε, να μετονομάσετε, να μετακινήσετε και να οργανώσετε αρχεία βίντεο στον χώρο αποθήκευσης cloud και στη συσκευή σας χωρίς να φύγετε από την εφαρμογή.

Σε αντίθεση με τις περισσότερες εφαρμογές πολυμέσων, το Evervideo ενοποιεί τους λογαριασμούς cloud, τα κοινόχρηστα NAS, τους media servers και τα τοπικά αρχεία σε ένα ενιαίο, ενοποιημένο tab Αρχείων — ώστε να μην πηδάτε μεταξύ ξεχωριστών οθονών. Αυτό κάνει τη μετακίνηση ενός βίντεο από έναν Plex server, σε έναν φάκελο iCloud Drive, στον φάκελο Έγγραφα του iPhone σας μια λειτουργία μιας οθόνης με ένα άγγιγμα.

## Tabs

Είτε χρησιμοποιείτε iPhone, iPad ή τη λειτουργία compact παραθύρου σε Mac, όλες οι λειτουργίες της εφαρμογής είναι προσβάσιμες από τη γραμμή tabs στο κάτω μέρος της οθόνης. Σε iPad και Mac, το ίδιο μενού είναι διαθέσιμο στην αριστερή πλαϊνή μπάρα ώστε να μπορείτε να αφήνετε περισσότερο χώρο οθόνης για το περιεχόμενο βίντεο.

- **Πρόσφατα** — το Evervideo ανοίγει εδώ από προεπιλογή. Μια λίστα με κάθε βίντεο που παρακολουθήσατε πρόσφατα, με μικρογραφία και πρόοδο παρακολούθησης ανά αρχείο, ώστε να μπορείτε να συνεχίσετε οποιοδήποτε με ένα άγγιγμα.
- **Αγαπημένα** — κάθε βίντεο και φάκελος που έχετε επισημάνει ως αγαπημένο, με γρήγορες Περισσότερες ενέργειες σε κάθε στοιχείο.
- **Βιβλιοθήκη Πολυμέσων** — η βιβλιοθήκη πολυμέσων σας, ομαδοποιημένη κατά Άλμπουμ, Είδη, Λίστες αναπαραγωγής, Πρόσφατα και Αγαπημένα. Μπορείτε επίσης να περιηγηθείτε σε βίντεο από τη βιβλιοθήκη Φωτογραφιών (Όλα τα Βίντεο, Κοντά, Μεσαία, Μεγάλα, Εγγραφές Οθόνης και τα Φωτογραφικά Άλμπουμ σας) και τη βιβλιοθήκη Μουσικής (Άλμπουμ, Είδη, Λίστες αναπαραγωγής).
- **Λίστες αναπαραγωγής** — διαχείριση κάθε λίστας αναπαραγωγής βίντεο σε ένα μέρος: δημιουργία, μετονομασία, επεξεργασία εξωφύλλου, αναδιάταξη, ενεργοποίηση offline λειτουργίας, εισαγωγή αρχείων M3U / M3U8 / CUE και εξαγωγή σε M3U / CSV / TXT.
- **Αρχεία** — ο all-in-one διαχειριστής αρχείων. Το tab Αρχεία συνδυάζει ό,τι οι περισσότερες εφαρμογές θα χωρίζουν σε "Συνδέσεις" και "Τοπικά Αρχεία": αποθήκευση cloud (Google Drive, Dropbox, OneDrive, iCloud Drive, MEGA, Box, pCloud, Yandex Disk και πολλά άλλα), κοινόχρηστα NAS μέσω SMB / WebDAV / FTP / SFTP / NFS / DLNA, αυτο-φιλοξενούμενους media servers (Plex, Jellyfin, Emby, Subsonic, Navidrome), ροές RTSP IP κάμερας, αντικειμενική αποθήκευση συμβατή με S3, τον φάκελο Έγγραφα, αρχεία σε αυτό το iPhone / iPad / Mac, την ουρά μεταφορών και την ενότητα Διαθέσιμες συσκευές. Ένα εικονίδιο Μεταφορές με περιστρεφόμενα βέλη στην επάνω αριστερή γωνία παρέχει άμεση πρόσβαση σε ενεργές λήψεις και αποστολές.
- **Ρυθμίσεις** — αλλαγή ρυθμίσεων εφαρμογής: media player (Picture-in-Picture, αποκωδικοποιητής υλικού, ισοσταθμιστής βίντεο, κλιμάκωση βίντεο, περιστροφή), ήχος (ισοσταθμιστής, ρυθμός δειγματοληψίας, κανάλια, IO buffer), υπότιτλοι (πρωτεύων / δευτερεύων τράκ, γραμματοσειρά, εξωτερικό αρχείο), βιβλιοθήκη πολυμέσων, διαχειριστής αρχείων, widgets Αρχικής Οθόνης, εξατομίκευση, γλώσσα, κωδικός πρόσβασης και Δημιουργία αντιγράφου ασφαλείας & Επαναφορά.
- **Γρήγορη Πρόσβαση** — βρίσκεται στην κορυφή των οθονών Αρχεία και Βιβλιοθήκη Πολυμέσων για γρήγορη πρόσβαση σε πρόσφατα και αγαπημένα σε κάθε πλαίσιο.

## Ο Compact Media Player

Το Evervideo περιλαμβάνει έναν μοναδικό compact media player που παραμένει ορατός στην κορυφή της εφαρμογής ενώ περιηγείστε στη βιβλιοθήκη, τον διαχειριστή αρχείων, τις ρυθμίσεις ή οποιοδήποτε άλλο tab. Πατήστε τον compact player για να τον επεκτείνετε σε πλήρη οθόνη· πατήστε και σύρετε προς τα κάτω τον player πλήρους οθόνης για να τον συμπτύξετε στην compact προβολή χωρίς να σταματήσει η αναπαραγωγή.

Αυτό σημαίνει ότι μπορείτε να συνεχίσετε να παρακολουθείτε ένα βίντεο ενώ περιηγείστε στη βιβλιοθήκη για το επόμενο, διαχειρίζεστε λήψεις ή προσαρμόζετε ρυθμίσεις εφαρμογής — χωρίς ποτέ να σταματήσετε την αναπαραγωγή ή να χάσετε τη θέση σας.

Στο macOS, ο compact player μπορεί επίσης να αποσπαστεί ως ξεχωριστό αιωρούμενο παράθυρο mini-player που παραμένει πάνω από κάθε άλλη εφαρμογή που χρησιμοποιείτε — ιδανικό για οδηγίες, διαλέξεις ή για να έχετε μια ταινία στο πλάι ενώ εργάζεστε.

## Picture-in-Picture (PiP)

Σε iPhone και iPad, το Evervideo υποστηρίζει πλήρως το Picture-in-Picture (PiP). Όταν μεταβείτε σε άλλη εφαρμογή, το βίντεο συνεχίζει να παίζει σε ένα κινητό αιωρούμενο παράθυρο. Πατήστε το εικονίδιο PiP στον player πλήρους οθόνης για να εισέλθετε στη λειτουργία PiP χειροκίνητα· σύρετε το αιωρούμενο παράθυρο σε οποιαδήποτε γωνία· τσιμπήστε για να αλλάξετε μέγεθος· πατήστε μία φορά για να εμφανιστούν βασικά στοιχεία ελέγχου αναπαραγωγής / παύσης / παράλειψης· και πατήστε το μικρό κουμπί ανάπτυξης για να επιστρέψετε στην πλήρη εφαρμογή Evervideo.

Το PiP λειτουργεί με όλες τις μορφές βίντεο που αναπαράγει το Evervideo, συμπεριλαμβανομένων αρχείων που μεταδίδονται από cloud και ροών RTSP, και συνεχίζει να λειτουργεί ενώ το τηλέφωνό σας είναι κλειδωμένο.

## Περισσότερες ενέργειες

Σχεδόν κάθε στοιχείο περιεχομένου στην οθόνη έχει ένα κουμπί Περισσότερες ενέργειες (το εικονίδιο "⋯" με τρεις τελείες). Πατήστε το για να ανοίξει ένα μενού ευαίσθητο στο πλαίσιο με κάθε διαθέσιμη ενέργεια για αυτό το στοιχείο — αναπαραγωγή επόμενου, αναπαραγωγή αργότερα, προσθήκη σε λίστα αναπαραγωγής, προσθήκη στα αγαπημένα, επεξεργασία ετικετών, λήψη, κοινή χρήση, μετονομασία, μετακίνηση και ούτω καθεξής. Οι μεγάλες λίστες κυλούν κατακόρυφα ώστε να μπορείτε να φτάσετε σε λιγότερο συνηθισμένες ενέργειες χωρίς να φορτώνετε την κύρια διεπαφή.

## Επάνω Γραμμή Εργαλείων

Η επάνω γραμμή εργαλείων, ακριβώς κάτω από τη γραμμή πλοήγησης, παρέχει γρήγορη πρόσβαση στις πιο συνηθισμένες ενέργειες ανά οθόνη. Μπορείτε να την εμφανίσετε ή να την αποκρύψετε με μια απλή χειρονομία σάρωσης προς τα κάτω.

- **Αναζήτηση** — έναρξη αναζήτησης στην τρέχουσα οθόνη (φάκελος, λίστα αναπαραγωγής, άλμπουμ, είδος, βιβλιοθήκη ή ουρά).
- **Συνέχεια Αναπαραγωγής** — το Evervideo μπορεί να επαναφέρει την κατάσταση του player και την τελευταία θέση για τον τρέχοντα κατάλογο, άλμπουμ, είδος ή λίστα αναπαραγωγής. Αυτή η επιλογή εμφανίζεται όταν η **Αποθήκευση Κατάστασης Αναπαραγωγής** είναι ενεργοποιημένη στις ρυθμίσεις.
- **Αναπαραγωγή Όλων** — προσθέτει κάθε βίντεο από την τρέχουσα σελίδα στην ουρά του player, διατηρώντας την τρέχουσα σειρά ταξινόμησης.
- **Ανακάτεμα Όλων** — προσθέτει κάθε βίντεο από την τρέχουσα σελίδα στην ουρά του player, ανακατεμένα. Ιδανικό για επανανακάλυψη ενός παλιά ξεχασμένου άλμπουμ κλιπ.

## Μενού Περιβάλλοντος

Το μενού περιβάλλοντος παρέχει γρήγορη πρόσβαση σε πρόσθετες επιλογές και ενέργειες για απρόσκοπτη αλληλεπίδραση σε όλες τις συσκευές.

### Μενού Περιβάλλοντος σε iOS / iPadOS

Πατήστε και κρατήστε σε κελιά, τον compact player ή οποιαδήποτε μικρογραφία για να εμφανιστεί το μενού περιβάλλοντος. Μπορείτε να αλλάξετε το στυλ παρουσίασης του μενού περιβάλλοντος στις Ρυθμίσεις → Προσωποποίηση → Στυλ Μενού Περιβάλλοντος.

### Μενού Περιβάλλοντος σε macOS

Κάντε δεξί κλικ σε κελιά, τον compact player ή μικρογραφίες βίντεο για να εμφανιστεί το μενού περιβάλλοντος. Το δεξί κλικ λειτουργεί επίσης στο βίντεο που παίζει στον player πλήρους οθόνης.

## Widgets Αρχικής Οθόνης

Το Evervideo υποστηρίζει widgets σε iPhone, iPad και Mac ώστε να μπορείτε να βλέπετε το βίντεο που παίζει με μια ματιά και να συνεχίσετε την αναπαραγωγή με ένα μόνο άγγιγμα.

Πρώτα, ενεργοποιήστε τα widgets στην εφαρμογή:

1. Ανοίξτε **Ρυθμίσεις → Widgets → Ενεργοποίηση Widgets** (οι ενημερώσεις widget χρησιμοποιούν μικρή ποσότητα ενέργειας, οπότε η εναλλαγή είναι απενεργοποιημένη από προεπιλογή).

Στη συνέχεια προσθέστε το widget στη συσκευή σας:

**Σε iPhone ή iPad**
1. Πατήστε και κρατήστε μια κενή περιοχή στην **Αρχική Οθόνη** ή στην **Οθόνη Κλειδώματος**.
2. Πατήστε το κουμπί **+** στην επάνω γωνία και αναζητήστε **Evervideo**.
3. Επιλέξτε μέγεθος widget και πατήστε **Προσθήκη Widget**.

**Σε Mac**
1. Κάντε κλικ στην ημερομηνία και ώρα στη γραμμή μενού για να ανοίξετε το **Κέντρο Γνωστοποιήσεων**, στη συνέχεια κάντε κλικ στο **Επεξεργασία Widgets** στο κάτω μέρος. (Μπορείτε επίσης να κάνετε Control-κλικ στην επιφάνεια εργασίας και να επιλέξετε **Επεξεργασία Widgets** για να τοποθετήσετε ένα απευθείας στην επιφάνεια εργασίας σας.)
2. Αναζητήστε **Evervideo** στη γκαλερί widget.
3. Κάντε κλικ σε ένα μέγεθος widget ή σύρετέ το στο Κέντρο Γνωστοποιήσεων ή στην επιφάνεια εργασίας.

Το widget εμφανίζει τον τίτλο του τρέχοντος βίντεο, τη μικρογραφία και βασικά στοιχεία ελέγχου σε κάθε πλατφόρμα.

## Προσβασιμότητα

Το Evervideo είναι πλήρως προσβάσιμο με **VoiceOver**: κάθε στοιχείο έχει μια περιγραφική ετικέτα και μια λογική υπόδειξη προσβασιμότητας. Όταν το VoiceOver είναι ενεργό, η εφαρμογή μεταβαίνει αυτόματα στη **Λειτουργία Κειμένου** — τα εικονογραφημένα διακοσμητικά στοιχεία αποκρύπτονται και εμφανίζονται μόνο τα σημαντικά στοιχεία ελέγχου, βελτιώνοντας την ταχύτητα πλοήγησης και τη σαφήνεια για χρήστες αναγνώστη οθόνης. Μπορείτε επίσης να ενεργοποιήσετε τη Λειτουργία Κειμένου χειροκίνητα στις Ρυθμίσεις → Προσβασιμότητα → Λειτουργία Κειμένου.

### Ρύθμιση Ρυθμιστικών με VoiceOver

1. **Επιλέξτε το ρυθμιστικό** — σαρώστε αριστερά ή δεξιά μέχρι το VoiceOver να το ανακοινώσει.
2. **Ρυθμίστε την τιμή** — διπλοπατήστε και κρατήστε το ρυθμιστικό, στη συνέχεια σύρετε πάνω ή κάτω για να αλλάξετε γρήγορα την τιμή. Το VoiceOver ανακοινώνει κάθε νέα τιμή καθώς προχωράτε.

### Ρύθμιση Θέσης Τράκ σε Λίστα Αναπαραγωγής με VoiceOver

1. Ανοίξτε μια λίστα αναπαραγωγής και πατήστε το κουμπί **Περισσότερα**.
2. Επιλέξτε **Αναδιάταξη Τραγουδιών**. Η λίστα μεταβαίνει σε λειτουργία επεξεργασίας.
3. Πατήστε το εικονίδιο δείκτη αναδιάταξης κοντά στον τίτλο τράκ για να του δώσετε εστίαση.
4. **Διπλοπατήστε** τον δείκτη αναδιάταξης. Στο δεύτερο πάτημα, μην αφήσετε το δάχτυλό σας — κρατήστε το μέχρι να ακούσετε έναν ήχο που υποδηλώνει ότι το κελί είναι έτοιμο να μετακινηθεί.
5. Τώρα μπορείτε να μετακινήσετε το κελί στη νέα του θέση.

Άλλα στοιχεία λειτουργούν όπως αναμένεται, χρησιμοποιώντας παρεχόμενα από το σύστημα μοτίβα VoiceOver.

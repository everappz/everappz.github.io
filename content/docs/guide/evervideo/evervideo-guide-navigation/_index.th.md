Evervideo มีอินเทอร์เฟซที่สะอาดและใช้งานง่าย ซึ่งคุ้นเคยสำหรับทุกคนที่เคยใช้โปรแกรมเล่นเพลงหรือสื่อบน iOS — แต่เพิ่มตัวจัดการไฟล์ในตัวที่แท้จริง เพื่อให้คุณสามารถแก้ไข, เปลี่ยนชื่อ, ย้าย และจัดระเบียบไฟล์วิดีโอในที่เก็บข้อมูลคลาวด์และบนอุปกรณ์ของคุณโดยไม่ต้องออกจากแอป

ต่างจากแอปมีเดียส่วนใหญ่ Evervideo รวมบัญชีคลาวด์, NAS share, มีเดียเซิร์ฟเวอร์ และไฟล์ในเครื่องเข้าเป็นแท็บไฟล์เดียวที่เป็นหนึ่งเดียว — เพื่อไม่ให้คุณต้องกระโดดระหว่างหน้าจอแยกต่างหาก ทำให้การย้ายวิดีโอจากเซิร์ฟเวอร์ Plex ไปยังโฟลเดอร์ iCloud Drive ไปยังโฟลเดอร์ Documents ของ iPhone เป็นการดำเนินการหนึ่งหน้าจอ หนึ่งแตะ

## แท็บ

ไม่ว่าจะใช้ iPhone, iPad หรือโหมดหน้าต่างขนาดกะทัดรัดบน Mac คุณสมบัติทั้งหมดของแอปสามารถเข้าถึงได้จากแถบแท็บที่ด้านล่างของหน้าจอ บน iPad และ Mac เมนูเดียวกันนี้อยู่ในแถบด้านข้างซ้าย เพื่อให้คุณมีพื้นที่หน้าจอมากขึ้นสำหรับเนื้อหาวิดีโอ

- **ล่าสุด** — Evervideo เปิดที่นี่ตามค่าเริ่มต้น รายการวิดีโอทุกรายการที่คุณเพิ่งดู พร้อมภาพขนาดเล็กและความคืบหน้าการดูต่อไฟล์ เพื่อให้คุณกลับมาดูต่อได้ด้วยการแตะครั้งเดียว
- **รายการโปรด** — ทุกวิดีโอและโฟลเดอร์ที่คุณทำเครื่องหมายเป็นรายการโปรด พร้อมดำเนินการเพิ่มเติมที่รวดเร็วสำหรับแต่ละรายการ
- **คลังมีเดีย** — คลังมีเดียของคุณ จัดกลุ่มตาม Albums, Genres, Playlists, ล่าสุด และรายการโปรด คุณยังสามารถเรียกดูวิดีโอจากไลบรารี Photos (วิดีโอทั้งหมด, สั้น, กลาง, ยาว, การบันทึกหน้าจอ และ Photo Albums) และ Music library (Albums, Genres, Playlists)
- **เพลย์ลิสต์** — จัดการเพลย์ลิสต์วิดีโอทุกรายการในที่เดียว: สร้าง, เปลี่ยนชื่อ, แก้ไขภาพปก, จัดเรียงใหม่, เปิดโหมดออฟไลน์, นำเข้าไฟล์ M3U / M3U8 / CUE และส่งออกเป็น M3U / CSV / TXT
- **ไฟล์** — ตัวจัดการไฟล์ครบวงจร แท็บไฟล์รวมสิ่งที่แอปส่วนใหญ่จะแยกออกเป็น "การเชื่อมต่อ" และ "ไฟล์ในเครื่อง": ที่เก็บข้อมูลคลาวด์ (Google Drive, Dropbox, OneDrive, iCloud Drive, MEGA, Box, pCloud, Yandex Disk และอีกมากมาย), NAS share ผ่าน SMB / WebDAV / FTP / SFTP / NFS / DLNA, มีเดียเซิร์ฟเวอร์แบบโฮสต์เอง (Plex, Jellyfin, Emby, Subsonic, Navidrome), สตรีม RTSP IP กล้อง, S3-compatible object storage, โฟลเดอร์ Documents, ไฟล์บน iPhone / iPad / Mac นี้, คิวการถ่ายโอน และส่วนอุปกรณ์ที่มีให้บริการ ไอคอนการถ่ายโอนลูกศรหมุนที่มุมซ้ายบนให้การเข้าถึงทันทีสำหรับการดาวน์โหลดและอัปโหลดที่ใช้งานอยู่
- **การตั้งค่า** — เปลี่ยนการตั้งค่าแอป: โปรแกรมเล่นมีเดีย (Picture-in-Picture, ตัวถอดรหัสฮาร์ดแวร์, อีควอไลเซอร์วิดีโอ, การปรับขนาดวิดีโอ, การหมุน), เสียง (อีควอไลเซอร์, อัตราการสุ่มตัวอย่าง, ช่องสัญญาณ, IO buffer), ซับไตเติ้ล (แทร็กหลัก / รอง, แบบอักษร, ไฟล์ภายนอก), คลังมีเดีย, ตัวจัดการไฟล์, วิดเจ็ตหน้าจอหลัก, การปรับแต่ง, ภาษา, รหัสผ่าน และสำรองข้อมูล & กู้คืน
- **การเข้าถึงด่วน** — อยู่ที่ด้านบนของหน้าจอไฟล์และคลังมีเดียเพื่อการเข้าถึงอย่างรวดเร็วไปยังล่าสุดและรายการโปรดในแต่ละบริบท

## โปรแกรมเล่นมีเดียขนาดกะทัดรัด

Evervideo มีโปรแกรมเล่นมีเดียขนาดกะทัดรัดที่ไม่เหมือนใคร ซึ่งยังคงมองเห็นได้ที่ด้านบนของแอปขณะที่คุณเรียกดูไลบรารี, ตัวจัดการไฟล์, การตั้งค่า หรือแท็บอื่นๆ แตะโปรแกรมเล่นขนาดกะทัดรัดเพื่อขยายเป็นโปรแกรมเล่นแบบเต็มหน้าจอ แตะและปัดลงบนโปรแกรมเล่นแบบเต็มหน้าจอเพื่อยุบกลับเป็นมุมมองขนาดกะทัดรัดโดยไม่หยุดการเล่น

ซึ่งหมายความว่าคุณสามารถดูวิดีโอต่อไปในขณะที่เรียกดูไลบรารีเพื่อหาสิ่งที่จะเล่นต่อไป, จัดการการดาวน์โหลด หรือปรับการตั้งค่าแอป — โดยไม่ต้องหยุดการเล่นหรือเสียตำแหน่ง

บน macOS โปรแกรมเล่นขนาดกะทัดรัดยังสามารถแยกออกเป็นหน้าต่าง mini player แบบลอยแยกต่างหาก ที่อยู่เหนือทุกแอปอื่นที่คุณใช้ — เหมาะสำหรับบทแนะนำ, การบรรยาย หรือมีภาพยนตร์ไว้ข้างๆ ขณะทำงาน

## Picture-in-Picture (PiP)

บน iPhone และ iPad Evervideo รองรับ Picture-in-Picture (PiP) อย่างเต็มรูปแบบ ดังนั้นเมื่อคุณสลับไปยังแอปอื่น วิดีโอยังคงเล่นต่อในหน้าต่างลอยที่เคลื่อนย้ายได้ แตะไอคอน PiP บนโปรแกรมเล่นแบบเต็มหน้าจอเพื่อเข้าสู่โหมด PiP ด้วยตนเอง ลากหน้าต่างลอยไปยังมุมใดก็ได้ หยิกเพื่อปรับขนาด แตะครั้งเดียวเพื่อแสดงตัวควบคุมเล่น / หยุดชั่วคราว / ข้ามพื้นฐาน และแตะปุ่มขยายขนาดเล็กเพื่อกลับสู่ Evervideo เต็มรูปแบบ

PiP ทำงานกับทุกรูปแบบวิดีโอที่ Evervideo เล่น รวมถึงไฟล์ที่สตรีมจากคลาวด์และสตรีม RTSP และยังคงทำงานต่อในขณะที่โทรศัพท์ของคุณถูกล็อก

## ดำเนินการเพิ่มเติม

เนื้อหาเกือบทุกรายการบนหน้าจอมีปุ่มดำเนินการเพิ่มเติม (ไอคอน "⋯" สามจุด) แตะเพื่อเปิดเมนูที่ตอบสนองต่อบริบทพร้อมทุกการดำเนินการที่มีสำหรับรายการนั้น — เล่นถัดไป, เล่นทีหลัง, เพิ่มในเพลย์ลิสต์, เพิ่มในรายการโปรด, แก้ไขแท็ก, ดาวน์โหลด, แชร์, เปลี่ยนชื่อ, ย้าย และอื่นๆ รายการยาวจะเลื่อนแนวตั้ง เพื่อให้คุณเข้าถึงการดำเนินการที่พบน้อยกว่าได้โดยไม่ทำให้ UI หลักแออัด

## แถบเครื่องมือด้านบน

แถบเครื่องมือด้านบนที่อยู่ใต้แถบนำทาง ให้การเข้าถึงอย่างรวดเร็วสำหรับการดำเนินการทั่วไปที่สุดต่อหน้าจอ คุณสามารถแสดงหรือซ่อนได้ด้วยการปัดลงง่ายๆ

- **ค้นหา** — เริ่มการค้นหาในหน้าจอปัจจุบัน (โฟลเดอร์, เพลย์ลิสต์, อัลบั้ม, แนวเพลง, ไลบรารี หรือคิว)
- **ดำเนินการเล่นต่อ** — Evervideo สามารถคืนค่าสถานะโปรแกรมเล่นและตำแหน่งล่าสุดสำหรับไดเรกทอรี, อัลบั้ม, แนวเพลง หรือเพลย์ลิสต์ปัจจุบัน ตัวเลือกนี้จะปรากฏเมื่อเปิดใช้งาน **บันทึกสถานะเสียง / โปรแกรมเล่นมีเดีย** ในการตั้งค่า
- **เล่นทั้งหมด** — เพิ่มวิดีโอทุกรายการจากหน้าปัจจุบันในคิวโปรแกรมเล่น โดยรักษาลำดับการเรียงที่ปัจจุบัน
- **สลับทั้งหมด** — เพิ่มวิดีโอทุกรายการจากหน้าปัจจุบันในคิวโปรแกรมเล่นแบบสลับ เหมาะสำหรับการค้นพบอัลบั้มคลิปที่ลืมเลือนนานอีกครั้ง

## เมนูบริบท

เมนูบริบทให้การเข้าถึงอย่างรวดเร็วสำหรับตัวเลือกและการดำเนินการเพิ่มเติมเพื่อการโต้ตอบที่ราบรื่นบนอุปกรณ์ต่างๆ

### เมนูบริบทบน iOS / iPadOS

แตะค้างบนเซลล์, โปรแกรมเล่นขนาดกะทัดรัด หรือภาพขนาดเล็กใดๆ เพื่อแสดงเมนูบริบท คุณสามารถเปลี่ยนรูปแบบการนำเสนอเมนูบริบทได้ใน การตั้งค่า → การปรับแต่งส่วนบุคคล → รูปแบบเมนูบริบท

### เมนูบริบทบน macOS

คลิกขวาบนเซลล์, โปรแกรมเล่นขนาดกะทัดรัด หรือภาพขนาดเล็กวิดีโอเพื่อแสดงเมนูบริบท การคลิกขวายังใช้งานได้กับวิดีโอที่กำลังเล่นอยู่ในโปรแกรมเล่นแบบเต็มหน้าจอ

## วิดเจ็ตหน้าจอหลัก

Evervideo รองรับวิดเจ็ตบน iPhone, iPad และ Mac เพื่อให้คุณเห็นวิดีโอที่กำลังเล่นอยู่อย่างรวดเร็วและกลับมาเล่นต่อด้วยการแตะครั้งเดียว

ขั้นแรก เปิดใช้งานวิดเจ็ตในแอป:

1. เปิด **การตั้งค่า → Widgets → เปิดใช้งาน Widgets** (การอัปเดตวิดเจ็ตใช้พลังงานเล็กน้อย ดังนั้นสวิตช์จึงปิดตามค่าเริ่มต้น)

จากนั้นเพิ่มวิดเจ็ตบนอุปกรณ์ของคุณ:

**บน iPhone หรือ iPad**
1. กดค้างในพื้นที่ว่างของ **หน้าจอหลัก** หรือ **หน้าจอล็อก**
2. แตะปุ่ม **+** ที่มุมบนและค้นหา **Evervideo**
3. เลือกขนาดวิดเจ็ตและแตะ **เพิ่มวิดเจ็ต**

**บน Mac**
1. คลิกวันที่และเวลาในแถบเมนูเพื่อเปิด **ศูนย์การแจ้งเตือน** จากนั้นคลิก **แก้ไขวิดเจ็ต** ที่ด้านล่าง (คุณยังสามารถ Control-คลิกเดสก์ท็อปและเลือก **แก้ไขวิดเจ็ต** เพื่อวางโดยตรงบนเดสก์ท็อปได้)
2. ค้นหา **Evervideo** ในแกลเลอรีวิดเจ็ต
3. คลิกขนาดวิดเจ็ต หรือลากไปยังศูนย์การแจ้งเตือนหรือเดสก์ท็อป

วิดเจ็ตแสดงชื่อวิดีโอปัจจุบัน, ภาพขนาดเล็ก และตัวควบคุมพื้นฐานบนทุกแพลตฟอร์ม

## การช่วยสำหรับการเข้าถึง

Evervideo สามารถเข้าถึงได้อย่างสมบูรณ์ด้วย **VoiceOver**: ทุกส่วนประกอบมีป้ายกำกับที่อธิบายไว้และคำแนะนำการช่วยสำหรับการเข้าถึงที่เหมาะสม เมื่อ VoiceOver ทำงาน แอปจะสลับไปยัง **โหมดข้อความ** โดยอัตโนมัติ — องค์ประกอบตกแต่งที่มีรูปภาพมากจะถูกซ่อน และแสดงเฉพาะตัวควบคุมที่มีความหมาย ซึ่งช่วยปรับปรุงความเร็วในการนำทางและความชัดเจนสำหรับผู้ใช้โปรแกรมอ่านหน้าจอ คุณยังสามารถเปิดใช้งานโหมดข้อความด้วยตนเองใน การตั้งค่า → การช่วยสำหรับการเข้าถึง → โหมดข้อความ

### การปรับ Slider ด้วย VoiceOver

1. **เลือก slider** — ปัดซ้ายหรือขวาจนกว่า VoiceOver จะประกาศ
2. **ปรับค่า** — แตะสองครั้งและกดค้าง slider จากนั้นลากขึ้นหรือลงเพื่อเปลี่ยนค่าอย่างรวดเร็ว VoiceOver จะประกาศค่าใหม่แต่ละค่าขณะที่คุณดำเนินการ

### การปรับตำแหน่งแทร็กในเพลย์ลิสต์ด้วย VoiceOver

1. เปิดเพลย์ลิสต์และแตะปุ่ม **เพิ่มเติม**
2. เลือก **จัดเรียงเพลงใหม่** รายการจะสลับเป็นโหมดแก้ไข
3. แตะไอคอนตัวบ่งชี้การจัดเรียงใหม่ใกล้ชื่อแทร็กเพื่อให้มันได้รับโฟกัส
4. **แตะสองครั้ง** ที่ตัวบ่งชี้การจัดเรียงใหม่ ในการแตะครั้งที่สอง อย่าปล่อยนิ้ว — กดค้างไว้จนกว่าคุณจะได้ยินเสียงที่บ่งบอกว่าเซลล์พร้อมที่จะย้าย
5. ตอนนี้คุณสามารถย้ายเซลล์ไปยังตำแหน่งใหม่ได้

ส่วนประกอบอื่นๆ ทำงานตามที่คาดหวัง โดยใช้รูปแบบ VoiceOver ที่ระบบจัดเตรียมไว้

Evervideo는 iOS에서 음악 또는 미디어 플레이어를 사용해 본 사람이라면 누구에게나 친숙하게 느껴지는 깔끔하고 직관적인 인터페이스를 제공합니다 — 하지만 클라우드 스토리지 및 기기에서 앱을 떠나지 않고 비디오 파일을 편집, 이름 변경, 이동 및 구성할 수 있는 실제 내장 파일 관리자가 추가되어 있습니다.

대부분의 미디어 앱과 달리, Evervideo는 클라우드 계정, NAS 공유, 미디어 서버 및 로컬 파일을 하나의 통합된 파일 탭으로 병합합니다 — 별도의 화면 사이를 왔다 갔다 할 필요가 없습니다. 이를 통해 Plex 서버에서 iCloud Drive 폴더로, iPhone의 Documents 폴더로 비디오를 이동하는 것이 한 화면, 한 번 탭으로 가능합니다.

## 탭

iPhone, iPad 또는 Mac의 컴팩트 창 모드를 사용하든, 모든 앱 기능은 화면 하단의 탭 바에서 접근할 수 있습니다. iPad 및 Mac에서는 동일한 메뉴가 왼쪽 사이드바에서 사용 가능하여 비디오 콘텐츠를 위한 화면 공간을 더 확보할 수 있습니다.

- **최근 항목** — Evervideo는 기본적으로 여기에서 열립니다. 최근에 시청한 모든 비디오 목록으로, 썸네일과 파일별 시청 진행 상황이 표시되어 한 번 탭으로 재개할 수 있습니다.
- **즐겨찾기** — 즐겨찾기로 표시한 모든 비디오와 폴더로, 각 항목에 빠른 추가 작업이 있습니다.
- **미디어 라이브러리** — 앨범, 장르, 재생 목록, 최근 항목, 즐겨찾기별로 그룹화된 미디어 라이브러리. 사진 라이브러리의 비디오(모든 비디오, 짧음, 보통, 긴 영상, 화면 녹화 및 사진 앨범)와 음악 라이브러리(앨범, 장르, 재생 목록)도 탐색할 수 있습니다.
- **재생 목록** — 모든 비디오 재생 목록을 한 곳에서 관리: 만들기, 이름 변경, 커버 아트 편집, 순서 변경, 오프라인 모드 활성화, M3U / M3U8 / CUE 파일 가져오기, M3U / CSV / TXT로 내보내기.
- **파일** — 올인원 파일 관리자. 파일 탭은 대부분의 앱이 "연결하기"와 "로컬 파일"로 분리하는 것을 하나로 결합합니다: 클라우드 스토리지 (Google Drive, Dropbox, OneDrive, iCloud Drive, MEGA, Box, pCloud, Yandex Disk 등), SMB / WebDAV / FTP / SFTP / NFS / DLNA를 통한 NAS 공유, 자체 호스팅 미디어 서버 (Plex, Jellyfin, Emby, Subsonic, Navidrome), RTSP IP 카메라 스트림, S3 호환 객체 스토리지, Documents 폴더, iPhone / iPad / Mac의 파일, 전송 대기열 및 사용 가능한 장치 섹션. 상단 왼쪽의 회전 화살표 전송 아이콘으로 활성 다운로드 및 업로드에 즉시 접근할 수 있습니다.
- **설정** — 애플리케이션 설정 변경: 미디어 플레이어 (화면 속 화면, 하드웨어 디코더, 비디오 이퀄라이저, 비디오 스케일링, 회전), 오디오 (이퀄라이저, 샘플 레이트, 채널, IO 버퍼), 자막 (기본 / 보조 트랙, 폰트, 외부 파일), 미디어 라이브러리, 파일 관리자, 홈 화면 위젯, 개인화, 언어, 비밀번호, 백업 및 복원.
- **빠른 접근** — 각 컨텍스트에서 최근 항목 및 즐겨찾기에 빠르게 접근하기 위해 파일 및 미디어 라이브러리 화면 상단에 있습니다.

## 소형 미디어 플레이어

Evervideo에는 라이브러리, 파일 관리자, 설정 또는 기타 탭을 탐색하는 동안 앱 상단에 계속 표시되는 고유한 소형 미디어 플레이어가 포함되어 있습니다. 소형 플레이어를 탭하면 전체 화면 미디어 플레이어로 확장되고, 전체 화면 플레이어를 아래로 탭하고 스와이프하면 재생을 중지하지 않고 소형 보기로 다시 접힙니다.

이것은 라이브러리에서 다음에 재생할 항목을 탐색하거나, 다운로드를 관리하거나, 앱 설정을 조정하면서도 비디오를 계속 시청할 수 있다는 것을 의미합니다 — 재생을 중지하거나 위치를 잃지 않고.

macOS에서는 소형 플레이어를 별도의 부동 미니 플레이어 창으로 분리하여 다른 모든 앱 위에 항상 표시되게 할 수도 있습니다 — 튜토리얼, 강의 또는 작업하면서 영화를 곁에 두기에 완벽합니다.

## 화면 속 화면 (PiP)

iPhone 및 iPad에서 Evervideo는 화면 속 화면 (PiP)을 완전히 지원하여, 다른 앱으로 전환할 때 비디오가 이동 가능한 부동 창에서 계속 재생됩니다. 전체 화면 플레이어의 PiP 아이콘을 탭하여 PiP 모드로 수동으로 진입하고; 부동 창을 모서리로 드래그하고; 핀치하여 크기를 조정하고; 한 번 탭하면 기본 재생 / 일시 정지 / 건너뛰기 컨트롤이 표시되며; 작은 확장 버튼을 탭하면 전체 Evervideo 앱으로 돌아갑니다.

PiP는 클라우드 스트리밍 파일 및 RTSP 스트림을 포함하여 Evervideo가 재생하는 모든 비디오 형식에서 작동하며, 휴대폰이 잠긴 상태에서도 계속 실행됩니다.

## 추가 작업

화면의 거의 모든 콘텐츠 항목에는 추가 작업 버튼 ("⋯" 세 점 아이콘)이 있습니다. 탭하면 해당 항목에 사용 가능한 모든 작업이 포함된 컨텍스트 메뉴가 열립니다 — 다음에 재생, 나중에 재생, 재생 목록에 추가, 즐겨찾기에 추가, 태그 편집, 다운로드, 공유, 이름 변경, 이동 등. 긴 목록은 세로로 스크롤되어 메인 UI를 혼잡하게 하지 않고도 자주 사용하지 않는 작업에 접근할 수 있습니다.

## 상단 툴바

내비게이션 바 바로 아래에 위치한 상단 툴바는 화면별로 가장 일반적인 작업에 빠르게 접근할 수 있습니다. 간단한 아래로 스와이프 제스처로 표시하거나 숨길 수 있습니다.

- **검색** — 현재 화면 (폴더, 재생 목록, 앨범, 장르, 라이브러리 또는 대기열) 내에서 검색을 시작합니다.
- **계속 재생** — Evervideo는 현재 디렉토리, 앨범, 장르 또는 재생 목록의 플레이어 상태와 마지막 위치를 복원할 수 있습니다. 이 옵션은 설정에서 **오디오 / 미디어 플레이어 상태 저장**이 활성화된 경우 나타납니다.
- **모두 재생** — 현재 페이지의 모든 비디오를 현재 정렬 순서를 유지하면서 플레이어 대기열에 추가합니다.
- **모두 셔플** — 현재 페이지의 모든 비디오를 셔플하여 플레이어 대기열에 추가합니다. 오래된 클립 앨범을 다시 발견하기에 좋습니다.

## 컨텍스트 메뉴

컨텍스트 메뉴는 기기 전반에 걸쳐 원활한 상호 작용을 위한 추가 옵션 및 작업에 빠르게 접근할 수 있게 해줍니다.

### iOS / iPadOS에서의 컨텍스트 메뉴

셀, 소형 플레이어 또는 임의의 썸네일을 길게 눌러 컨텍스트 메뉴를 표시합니다. 설정 → 개인화 → 컨텍스트 메뉴 스타일에서 컨텍스트 메뉴 표시 스타일을 변경할 수 있습니다.

### macOS에서의 컨텍스트 메뉴

셀, 소형 플레이어 또는 비디오 썸네일을 마우스 오른쪽 버튼으로 클릭하여 컨텍스트 메뉴를 표시합니다. 전체 화면 플레이어에서 현재 재생 중인 비디오에도 마우스 오른쪽 버튼 클릭이 작동합니다.

## 홈 화면 위젯

Evervideo는 iPhone, iPad 및 Mac에서 위젯을 지원하여 현재 재생 중인 비디오를 한 눈에 볼 수 있고 한 번 탭으로 재생을 재개할 수 있습니다.

먼저 앱에서 위젯을 활성화하세요:

1. **설정 → 위젯 → 위젯 활성화**를 열세요 (위젯 업데이트는 약간의 에너지를 사용하므로 토글은 기본적으로 꺼져 있습니다).

그런 다음 기기에 위젯을 추가하세요:

**iPhone 또는 iPad에서**
1. **홈 화면** 또는 **잠금 화면**의 빈 영역을 길게 누릅니다.
2. 상단 모서리의 **+** 버튼을 탭하고 **Evervideo**를 검색합니다.
3. 위젯 크기를 선택하고 **위젯 추가**를 탭합니다.

**Mac에서**
1. 메뉴 바의 날짜와 시간을 클릭하여 **알림 센터**를 열고 하단의 **위젯 편집**을 클릭합니다. (데스크탑을 Control 클릭하고 **위젯 편집**을 선택하여 데스크탑에 직접 배치할 수도 있습니다.)
2. 위젯 갤러리에서 **Evervideo**를 검색합니다.
3. 위젯 크기를 클릭하거나 알림 센터 또는 데스크탑으로 드래그합니다.

위젯은 모든 플랫폼에서 현재 비디오의 제목, 썸네일 및 기본 컨트롤을 표시합니다.

## 접근성

Evervideo는 **VoiceOver**로 완전히 접근 가능합니다: 모든 구성 요소에 설명적인 레이블과 합리적인 접근성 힌트가 있습니다. VoiceOver가 활성화되면 앱이 자동으로 **텍스트 모드**로 전환됩니다 — 이미지가 많은 장식 요소는 숨겨지고 의미 있는 컨트롤만 표시되어 화면 읽기 사용자의 탐색 속도와 명확성이 향상됩니다. 설정 → 접근성 → 텍스트 모드에서 텍스트 모드를 수동으로 활성화할 수도 있습니다.

### VoiceOver로 슬라이더 조정

1. **슬라이더 선택** — VoiceOver가 발표할 때까지 왼쪽 또는 오른쪽으로 스와이프합니다.
2. **값 조정** — 슬라이더를 두 번 탭하고 길게 누른 다음, 위나 아래로 드래그하여 값을 빠르게 변경합니다. VoiceOver는 변경할 때마다 새 값을 발표합니다.

### VoiceOver로 재생 목록에서 트랙 위치 조정

1. 재생 목록을 열고 **더보기** 버튼을 탭합니다.
2. **노래 재배열**을 선택합니다. 목록이 편집 모드로 전환됩니다.
3. 트랙 제목 근처의 재순서 표시기 아이콘을 탭하여 포커스를 줍니다.
4. 재순서 표시기를 **두 번 탭**합니다. 두 번째 탭 시 손가락을 떼지 마세요 — 셀이 이동 준비가 됐다는 소리가 들릴 때까지 누르고 있으세요.
5. 이제 셀을 새 위치로 이동할 수 있습니다.

다른 구성 요소는 시스템 제공 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

แท็บไฟล์คือตัวจัดการไฟล์ครบวงจรของ Evervideo ต่างจากแอปวิดีโอส่วนใหญ่ที่แยกที่เก็บข้อมูลคลาวด์จากไฟล์ในเครื่องออกเป็นแท็บต่างๆ Evervideo รวมทั้งสองเข้าเป็นหน้าจอเดียวที่เลื่อนได้ เพื่อให้คุณย้ายจากเซิร์ฟเวอร์ Plex ไปยังโฟลเดอร์คลาวด์ ไปยังโฟลเดอร์ Documents ของ iPhone ได้โดยไม่ต้องสลับแท็บ

แท็บไฟล์แบ่งออกเป็นส่วนที่ชัดเจนซึ่งปรากฏตามลำดับนี้บนหน้าจอของคุณ:

- **การเข้าถึงด่วน** — ไฟล์และโฟลเดอร์ที่เพิ่งเปิดล่าสุดและรายการโปรด
- **ไฟล์ในแอปพลิเคชันนี้** — สิ่งที่อยู่ในโฟลเดอร์ Documents แบบ sandbox ของ Evervideo
- **ไฟล์บน iPhone / iPad / Mac นี้** — วิดีโอที่อยู่ที่อื่นบนอุปกรณ์ของคุณ ผ่านตัวเลือกไฟล์ระบบ
- **ที่เก็บข้อมูลคลาวด์** — ทุกบัญชีคลาวด์, NAS และมีเดียเซิร์ฟเวอร์ที่คุณเชื่อมต่อ
- **อุปกรณ์ที่มีให้บริการ** — เซิร์ฟเวอร์และอุปกรณ์บนเครือข่ายท้องถิ่นของคุณที่ค้นพบอัตโนมัติผ่าน Bonjour / mDNS

ที่มุมขวาบนของหน้าจอไฟล์มีปุ่มการถ่ายโอน (ไอคอนลูกศรหมุน) แตะเพื่อเปิดคิวการถ่ายโอน ซึ่งคุณสามารถตรวจสอบการดาวน์โหลดและอัปโหลดทุกรายการจากแหล่งทั้งหมดของคุณ

## บริการที่เก็บข้อมูลคลาวด์, มีเดียเซิร์ฟเวอร์ และโปรโตคอลที่รองรับ

Evervideo รองรับแหล่งข้อมูลสำหรับวิดีโอของคุณในช่วงกว้างเป็นพิเศษ ทุกอย่างด้านล่างทำงานผ่านขั้นตอนเชื่อมต่อกับที่เก็บข้อมูลคลาวด์เดียว

**ที่เก็บข้อมูลคลาวด์ส่วนตัว:** iCloud Drive · Dropbox · OneDrive · Google Drive · MEGA · Box · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘)

**มีเดียเซิร์ฟเวอร์แบบโฮสต์เอง:** Plex · Jellyfin · Emby · Subsonic (และทุกเซิร์ฟเวอร์ที่เข้ากันได้กับ Subsonic — Airsonic, Funkwhale, Gonic, Ampache) · Navidrome · Nextcloud (และ ownCloud ผ่าน shared API) · Synology Drive · QNAP

**NAS และโปรโตคอลแชร์ไฟล์:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP · FTPS · SFTP (SSH File Transfer Protocol, รหัสผ่านหรือ public-key auth) · NFS · DLNA / UPnP (การเล่นและดาวน์โหลด)

**สตรีม Live และกล้อง IP:** RTSP — ชี้ Evervideo ไปที่ URL `rtsp://` ใดก็ได้และจะเล่นทันที เหมาะสำหรับกล้องรักษาความปลอดภัย, สตรีม IPTV, กล้องดอร์เบลล์, เครื่องตรวจสอบเด็กและแหล่งสดที่คล้ายกัน

**ที่เก็บข้อมูลออบเจกต์ที่เข้ากันได้กับ S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces และ S3-API endpoint อื่นๆ

**ไลบรารีบนอุปกรณ์:** ไลบรารี Photos (วิดีโอทั้งหมด, การบันทึกหน้าจอ, อัลบั้มรูปภาพ) และไลบรารี Music app (Albums, Genres, Playlists) — ทั้งสองแสดงในคลังมีเดียเพื่อไม่ให้คุณต้องคัดลอกอะไร

**การค้นพบเครือข่ายท้องถิ่น:** ส่วนอุปกรณ์ที่มีให้บริการจะแสดงรายการบริการ Bonjour / mDNS ทุกรายการบนเครือข่าย Wi-Fi ของคุณโดยอัตโนมัติ — Plex, Jellyfin, เซิร์ฟเวอร์ Emby, Synology, QNAP, AirPort router ที่มีไดรฟ์ติดอยู่, Apple Time Capsule — เพื่อให้คุณแตะเพื่อเชื่อมต่อโดยไม่ต้องพิมพ์ IP address

การเชื่อมต่อแต่ละรายการใช้ SDK อย่างเป็นทางการหรือโปรโตคอลเปิดของบริการ พร้อมการอนุญาตแบบ OAuth ในกรณีที่รองรับ คุณสามารถเชื่อมต่อบัญชีหลายบัญชีของบริการเดียวกัน (เช่น บัญชี Google Drive สองบัญชี หรือทั้งเซิร์ฟเวอร์ Plex และ Jellyfin) และเรียกดูพร้อมกันในส่วนที่เก็บข้อมูลคลาวด์

## ความปลอดภัยและความเป็นส่วนตัว

Evervideo ใช้เฉพาะ SDK อย่างเป็นทางการและการเชื่อมต่อที่ปลอดภัยในการโต้ตอบกับบริการคลาวด์ที่เชื่อมต่อ ชื่อผู้ใช้และรหัสผ่านของคุณไม่สามารถเข้าถึงได้โดยแอปพลิเคชัน — การถ่ายโอนทั้งหมดได้รับการเข้ารหัส TLS

เมื่อคุณป้อนชื่อผู้ใช้และรหัสผ่าน แอปพลิเคชันจะแสดงหน้าการอนุญาตอย่างเป็นทางการที่ผู้ให้บริการคลาวด์ให้มา และกระบวนการอนุญาตทั้งหมดเกิดขึ้นนอกแอปพลิเคชัน ผู้ให้บริการคลาวด์จะส่ง auth-token ไปยังแอปพลิเคชันหลังการอนุญาตที่สำเร็จ และ token นั้นถูกใช้เพื่อทำ API call

auth-token คือกุญแจดิจิทัลที่ช่วยให้แอปพลิเคชันของบุคคลที่สามโต้ตอบกับที่เก็บข้อมูลคลาวด์ในนามของคุณ Token จะถูกเก็บไว้บนอุปกรณ์ของคุณใน Apple's secure system storage (Keychain) ซึ่งเข้ารหัสขณะพักและได้รับการป้องกันด้วยรหัสผ่านอุปกรณ์หรือการล็อกชีวมาตร คุณสามารถดาวน์โหลดไฟล์จากบริการคลาวด์ที่เชื่อมต่อมายังอุปกรณ์ของคุณ ไฟล์เหล่านั้นจะถูกวางไว้ในไดเรกทอรี Documents ของแอปและสามารถลบออกได้ตลอดเวลาโดยใช้ตัวจัดการไฟล์ในตัว

แอปพลิเคชันไม่แชร์ข้อมูลใดๆ จากบัญชีคลาวด์ที่เชื่อมต่อของคุณกับ Everappz, ผู้โฆษณา หรือบุคคลที่สาม คุณสามารถเพิกถอนการเข้าถึงบัญชีคลาวด์ได้ตลอดเวลาโดยเปิดหน้าการตั้งค่าบัญชีในเว็บเบราว์เซอร์ของคุณ

## เพิกถอน Auth-Token

หากต้องการเพิกถอน auth-token ให้เข้าสู่ระบบบัญชีคลาวด์ของคุณในเว็บเบราว์เซอร์และไปที่หน้าความปลอดภัยหรือแอปที่เชื่อมต่อ ที่นั่นคุณสามารถค้นหาแอปของบุคคลที่สามทุกรายการที่เชื่อมต่อกับบัญชีคลาวด์ของคุณและลบรายการที่คุณไม่ต้องการใช้อีกต่อไป คำแนะนำโดยละเอียดสำหรับบัญชี Google มีให้ [ที่นี่](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account)

คุณยังสามารถตัดการเชื่อมต่อบัญชีคลาวด์ภายในแอปพลิเคชันเองได้ — เมื่อทำเช่นนั้น auth-token จะถูกลบออกจากอุปกรณ์ของคุณทันที หากคุณถอนการติดตั้งแอปพลิเคชันจากอุปกรณ์ ข้อมูลที่ดาวน์โหลดและ access token ทั้งหมดจะถูกลบออกอัตโนมัติพร้อมกัน

## ตัดการเชื่อมต่อที่เก็บข้อมูลคลาวด์หรือเปลี่ยนการกำหนดค่า

- **เข้าถึงตัวเลือกที่เก็บข้อมูลคลาวด์** — ค้นหาบริการที่เชื่อมต่อในส่วน **ที่เก็บข้อมูลคลาวด์** ของแท็บไฟล์
- **แตะปุ่ม "..."** ถัดจากชื่อบริการเพื่อเปิดตัวเลือกเพิ่มเติม:
  - **เปลี่ยนชื่อ** — เปลี่ยนชื่อบริการคลาวด์ที่ปรากฏในรายการของคุณ
  - **การตั้งค่า** — แก้ไขการกำหนดค่าหรือข้อมูลการตรวจสอบสิทธิ์ บางครั้งคุณอาจต้องอนุญาตบริการคลาวด์ที่เชื่อมต่อใหม่หาก authorization token หมดอายุ
  - **ตัดการเชื่อมต่อ** — ตัดการเชื่อมต่อระหว่างแอปและบริการคลาวด์โดยสมบูรณ์ การดำเนินการนี้จะลบวิดีโอทั้งหมดที่เกี่ยวข้องกับบริการนั้นออกจากคลังมีเดียของคุณ แต่จะไม่แตะต้องบนเซิร์ฟเวอร์

## เชื่อมต่อกับคอมพิวเตอร์หรือ NAS

คุณสามารถเชื่อมต่อคอมพิวเตอร์, NAS ส่วนตัว หรืออุปกรณ์เครือข่ายอื่นๆ โดยใช้โปรโตคอล SMB, WebDAV, FTP / FTPS, SFTP, NFS หรือ DLNA นี่เป็นวิธีที่ง่ายที่สุดในการนำคลังมีเดียในบ้านที่มีอยู่ — ไม่ว่าจะอยู่บน Mac, Windows PC, Synology, QNAP, Apple Time Capsule หรือ WD My Cloud Home — เข้าสู่ Evervideo โดยไม่ต้องคัดลอกอะไร

### เชื่อมต่อกับคอมพิวเตอร์โดยใช้ SMB

- แตะ **เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → SMB**
- ป้อน IP address ของคอมพิวเตอร์และชื่อโฟลเดอร์ที่แชร์โดยใช้รูปแบบ `smb://computer-ip-address/shared-folder-name`
- เลือกเวอร์ชันโปรโตคอล: **Auto**, **SMB1** หรือ **SMB2**
- ป้อนชื่อผู้ใช้และรหัสผ่าน (ถ้าจำเป็น)
- แตะ **เสร็จสิ้น**

หากการเชื่อมต่อสำเร็จ share จะปรากฏในส่วนที่เก็บข้อมูลคลาวด์ บทช่วยสอนฉบับสมบูรณ์เกี่ยวกับวิธีการเชื่อมต่อ Mac หรือ PC โดยใช้ SMB มีให้ [ที่นี่](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)

### เชื่อมต่อกับ NAS โดยใช้ WebDAV

ขั้นตอนทั้งหมดเหมือนกับ SMB ยกเว้นฟิลด์ URL ใช้ `http://server-name` หรือ `https://server-name` หากเซิร์ฟเวอร์รองรับ SSL WebDAV ทำงานกับ Synology, QNAP, Nextcloud, ownCloud, WD My Cloud Home และเซิร์ฟเวอร์อื่นๆ ที่มี WebDAV endpoint

บทช่วยสอนฉบับสมบูรณ์เกี่ยวกับ WebDAV มีให้ [ที่นี่](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)

### เชื่อมต่อผ่าน DLNA / UPnP

แชร์คลังมีเดียบน Windows PC หรือ NAS ของคุณโดยใช้โปรโตคอล DLNA / UPnP และเข้าถึงมันใน Evervideo ตามที่อธิบายไว้ [ที่นี่](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone) DLNA ได้รับการรองรับอย่างกว้างขวาง แต่อนุญาตให้คุณเล่นหรือดาวน์โหลดวิดีโอเท่านั้น — คุณไม่สามารถอัปโหลดไฟล์หรือสร้างโฟลเดอร์ใหม่บนเซิร์ฟเวอร์ DLNA

### เชื่อมต่อโดยใช้ FTP, FTPS หรือ SFTP

Evervideo ยังรองรับโปรโตคอลการถ่ายโอนไฟล์แบบคลาสสิก ในการเชื่อมต่อเซิร์ฟเวอร์ผ่าน FTP หรือ FTPS แตะ เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → FTP ป้อน URL host ในรูปแบบ `ftp://server-name` (หรือ `ftps://server-name` สำหรับการเชื่อมต่อที่เข้ารหัส) ป้อนชื่อผู้ใช้และรหัสผ่าน จากนั้นแตะ เสร็จสิ้น สำหรับ SFTP (SSH File Transfer Protocol) เลือก SFTP แทนและป้อนรหัสผ่านหรือคู่คีย์ SSH

### เชื่อมต่อกับ NFS Share

Evervideo รองรับ NFS (Network File System) — สะดวกสำหรับ Linux host, BSD server และอุปกรณ์ NAS ที่ต้องการเปิดเผยคลังวิดีโอผ่าน NFS แทน SMB เลือก NFS ในเมนูเชื่อมต่อกับที่เก็บข้อมูลคลาวด์ ป้อนที่อยู่เซิร์ฟเวอร์และเส้นทาง export และแตะ เสร็จสิ้น

## เชื่อมต่อ Plex Media Server

Evervideo สามารถเชื่อมต่อโดยตรงกับ Plex Media Server และเรียกดูคลังภาพยนตร์, รายการทีวี และวิดีโอที่บ้านของคุณ แตะ เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → Plex ลงชื่อเข้าใช้ด้วยบัญชี Plex ของคุณ เลือกเซิร์ฟเวอร์ และคลังจะปรากฏพร้อมกับบริการคลาวด์ของคุณ เซิร์ฟเวอร์ Plex ในเครือข่ายท้องถิ่นเดียวกันยังถูกค้นพบอัตโนมัติในส่วนอุปกรณ์ที่มีให้บริการ

## เชื่อมต่อเซิร์ฟเวอร์ Jellyfin หรือ Emby

ทั้ง Jellyfin (โอเพนซอร์ส) และ Emby (เชิงพาณิชย์) มีเดียเซิร์ฟเวอร์ทำงานได้อย่างเป็นธรรมชาติใน Evervideo แตะ เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → Jellyfin หรือ Emby ป้อน URL เซิร์ฟเวอร์ของคุณ (โดยทั่วไปจะเป็นเช่น `http://server-ip:8096`) และข้อมูลประจำตัว และคลังของคุณพร้อมสตรีม

## เชื่อมต่อเซิร์ฟเวอร์ Subsonic หรือ Navidrome

Evervideo รองรับ Subsonic API ซึ่งหมายความว่าทำงานกับ Subsonic, Navidrome และทุกเซิร์ฟเวอร์ที่เข้ากันได้กับ Subsonic — รวมถึง Airsonic, Funkwhale, Gonic, Logitech Media Server (LMS) และ Ampache แตะ เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → Subsonic ป้อน URL เซิร์ฟเวอร์และข้อมูลประจำตัว และคลังจะปรากฏในส่วนที่เก็บข้อมูลคลาวด์

## เชื่อมต่อสตรีม RTSP (กล้อง IP, Live TV, IPTV)

Evervideo รองรับ RTSP แบบเนทีฟ เพื่อให้คุณสามารถชี้ไปที่แหล่ง RTSP ใดก็ได้ — กล้องรักษาความปลอดภัย, กล้องดอร์เบลล์, ผู้ให้บริการ IPTV, เครื่องตรวจสอบเด็ก, ฟีดการออกอากาศ — และ Evervideo จะดึงและถอดรหัสสตรีมสด แตะ ลิงก์ออนไลน์ → เพิ่มลิงก์ วาง URL ฉบับสมบูรณ์ (`rtsp://camera-ip:port/stream-path`) ป้อนชื่อผู้ใช้และรหัสผ่านหากจำเป็น และแตะ เสร็จสิ้น

## เชื่อมต่อกับที่เก็บข้อมูลออบเจกต์ที่เข้ากันได้กับ S3

สำหรับผู้โฮสต์เองและผู้ใช้ขั้นสูงที่ใช้ที่เก็บข้อมูลออบเจกต์คลาวด์ Evervideo มี S3-compatible connector แตะ เชื่อมต่อกับที่เก็บข้อมูลคลาวด์ → S3 storage จากนั้นป้อน endpoint URL, ภูมิภาค, access key, secret key และชื่อ bucket ใช้งานได้กับ AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces และ S3-API endpoint อื่นๆ

## อุปกรณ์ที่มีให้บริการ

ส่วนนี้แสดงทุกอุปกรณ์บนเครือข่ายท้องถิ่นของคุณที่คุณสามารถเชื่อมต่อได้จาก Evervideo ผ่านการค้นพบ Bonjour / mDNS — Plex, Jellyfin, เซิร์ฟเวอร์ Emby, Synology, QNAP, AirPort router ที่มีไดรฟ์ติดอยู่, Apple Time Capsule และอื่นๆ วิธีสร้างการเชื่อมต่อ:

- เลื่อนไปที่ส่วนอุปกรณ์ที่มีให้บริการในแท็บไฟล์
- แตะอุปกรณ์ที่คุณต้องการเชื่อมต่อ
- ถ้าจำเป็น ป้อนข้อมูลล็อกอินเพื่อสร้างการเชื่อมต่อ

## Wi-Fi Drive

Wi-Fi Drive ให้คุณถ่ายโอนไฟล์แบบไร้สายจากคอมพิวเตอร์ไปยังอุปกรณ์ iOS ผ่านเว็บเบราว์เซอร์บนเดสก์ท็อป, Finder หรือ File Explorer อุปกรณ์และคอมพิวเตอร์ของคุณต้องอยู่ในเครือข่าย Wi-Fi เดียวกัน

### เปิดใช้งาน Wi-Fi Drive

- ในแท็บไฟล์ เลื่อนไปที่ ที่เก็บข้อมูลคลาวด์ → เชื่อมต่อผ่าน Wi-Fi เพื่อเปิดหน้าหลักของ Wi-Fi Drive
- (ไม่บังคับ) ตั้งชื่อผู้ใช้และรหัสผ่านสำหรับเว็บเซิร์ฟเวอร์ในตัว
- แตะ เริ่ม Wi-Fi Drive

### เข้าถึง Wi-Fi Drive บนคอมพิวเตอร์ของคุณ

- เปิดเว็บเบราว์เซอร์บนคอมพิวเตอร์ (Chrome, Firefox, Safari ฯลฯ)
- ป้อน URL ที่แอปพลิเคชันแสดง
- ลากและวางไฟล์จากคอมพิวเตอร์ลงบนเว็บเพจ — ไฟล์จะเริ่มถ่ายโอนไปยังอุปกรณ์ iOS ของคุณ

คุณยังสามารถ mount Wi-Fi Drive โดยตรงใน **Finder** บน macOS (ไป → เชื่อมต่อกับเซิร์ฟเวอร์…) หรือ File Explorer บน Windows (แมปไดรฟ์เครือข่าย…) และจัดการ iPhone หรือ iPad ของคุณเป็นไดรฟ์เครือข่ายปกติ

คำแนะนำโดยละเอียดมีให้ [ที่นี่](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)

## การแชร์ไฟล์ iTunes / Finder

iTunes File Sharing (ปัจจุบันคือ Finder File Sharing บน macOS Catalina และใหม่กว่า) ให้คุณถ่ายโอนไฟล์โดยใช้สาย Lightning หรือ USB-C เชื่อมต่ออุปกรณ์ เปิด Finder บน Mac (หรือ iTunes บน Windows) ค้นหา Evervideo ในรายการแอป และคัดลอกไฟล์ไปยังโฟลเดอร์ที่แชร์

## เชื่อมต่อ USB flash drive หรือ SD card

เสียบ USB drive หรือ SD card เข้ากับ iPhone, iPad หรือ Mac ผ่านอะแดปเตอร์ Lightning-to-USB / USB-C หรือเครื่องอ่านการ์ด เปิด ไฟล์ → ไฟล์บน iPhone นี้ → เปิดโฟลเดอร์ นำทางไปยังไดรฟ์ และเลือกไฟล์วิดีโอหรือโฟลเดอร์ Evervideo เล่นไฟล์โดยตรงจากไดรฟ์โดยไม่ต้องคัดลอกไปยังที่เก็บข้อมูลภายใน — สะดวกสำหรับคลัง 4K ขนาดใหญ่

## ไฟล์บน iPhone / iPad / Mac นี้

ส่วนนี้แสดงวิดีโอที่อยู่บนอุปกรณ์ของคุณแต่ในแอปพลิเคชันอื่นๆ คุณสามารถนำเข้าสู่ Evervideo โดยใช้ตัวเลือกไฟล์ระบบ:

- แตะ เปิดไฟล์… เพื่อเลือกไฟล์เฉพาะ

## โฟลเดอร์พิเศษ

ในแท็บไฟล์คุณจะเห็นโฟลเดอร์พิเศษหลายรายการที่ Evervideo สร้างและใช้โดยอัตโนมัติ:

- **ดาวน์โหลด** — ทุกไฟล์ที่ดาวน์โหลดจากคลาวด์จะปรากฏที่นี่ตามค่าเริ่มต้น กำหนดเองได้ใน การตั้งค่า → ตัวจัดการไฟล์ → บันทึกไฟล์ที่ดาวน์โหลดไปที่
- **แคชโปรแกรมเล่น** — แคชของโปรแกรมเล่นมีเดีย ตามค่าเริ่มต้น โปรแกรมเล่นจะดาวน์โหลดวิดีโอที่กำลังจะมาถึงเพื่อการเล่นที่ราบรื่น คุณสามารถปิดใช้งานแคชในการตั้งค่าแอปหรือเพียงแค่ลบโฟลเดอร์นี้
- **iCloud** — ไฟล์ในโฟลเดอร์นี้จะซิงค์ข้ามอุปกรณ์ทั้งหมดของคุณที่เชื่อมต่อกับบัญชี iCloud เดียวกัน
- **โฟลเดอร์ออฟไลน์** — เมื่อคุณทำเครื่องหมายโฟลเดอร์, เพลย์ลิสต์, อัลบั้ม หรือแนวเพลงเป็นออฟไลน์ ไฟล์จะถูกดาวน์โหลดไปยังโฟลเดอร์นี้

## แถบเครื่องมือด้านบน

แถบเครื่องมือด้านบนที่อยู่ใต้แถบนำทางมีการดำเนินการหลายอย่างที่คุณสามารถแสดงหรือซ่อนด้วยการปัดลง:

- **ค้นหา** — ทำการค้นหาในโฟลเดอร์หรือส่วนปัจจุบัน
- **ดำเนินการเล่นต่อ** — หากเปิดใช้งานในการตั้งค่า จะคืนค่าคิวโปรแกรมเล่นและตำแหน่งวิดีโอล่าสุดสำหรับโฟลเดอร์ปัจจุบัน
- **เล่นทั้งหมด** — สแกนโฟลเดอร์ปัจจุบันและโฟลเดอร์ย่อยและเพิ่มไฟล์ในคิวโปรแกรมเล่น
- **สลับทั้งหมด** — เหมือนเล่นทั้งหมด แต่สลับก่อนเพิ่ม

## ตัวเลือกโฟลเดอร์

เมื่อคุณเปิดโฟลเดอร์ แตะปุ่ม **"..."** ที่มุมขวาบนสำหรับการดำเนินการเหล่านี้:

- **เลือก** — เปิดใช้งานโหมดการเลือกไฟล์
- **โฟลเดอร์ใหม่** — สร้างโฟลเดอร์ใหม่ภายในโฟลเดอร์ปัจจุบัน
- **เปิดโหมดออฟไลน์** — เปิดการซิงค์ออฟไลน์สำหรับโฟลเดอร์ปัจจุบัน ไฟล์ใหม่ที่เพิ่มออนไลน์จะถูกดาวน์โหลดโดยอัตโนมัติ
- **อัปโหลดไฟล์** — อัปโหลดไฟล์จากอุปกรณ์ของคุณไปยังโฟลเดอร์ออนไลน์
- **ค้นหา** — ค้นหาภายในโฟลเดอร์
- **จัดเรียง** — จัดเรียงไฟล์ตามชื่อ, ขนาด, วันที่แก้ไข หรือข้อมูลเมตา
- **มุมมองกริด / รายการ** — สลับระหว่างมุมมองตารางและมุมมองภาพขนาดเล็ก มุมมองภาพขนาดเล็กแสดงตัวอย่างโปสเตอร์วิดีโอขนาดใหญ่

## โหมดการเลือก

แตะ **"..."** ที่มุมขวาบนและเลือก **เลือก** เพื่อเข้าสู่โหมดการเลือก กล่องกาเครื่องหมายจะปรากฏถัดจากไฟล์และโฟลเดอร์ทุกรายการ แตะเพื่อเลือกหนึ่งหรือหลายรายการ จากนั้นดำเนินการแบบกลุ่ม: เล่นถัดไป, เล่นทีหลัง, เพิ่มในคลังมีเดีย, เพิ่มในเพลย์ลิสต์, คัดลอก, อัปโหลด, ย้าย, เปลี่ยนชื่อ หรือลบ

{{< cards cols="1">}}
  {{< card title="" subtitle="Evervideo โหมดการเลือกในตัวจัดการไฟล์" image="/docs/guide/evervideo/img/evervideo-selection-mode-in-files.webp" >}}
{{< /cards >}}

หากคุณต้องการให้ที่เก็บข้อมูลคลาวด์ที่เชื่อมต่อเป็นแบบอ่านอย่างเดียว (เพื่อป้องกันการลบโดยไม่ตั้งใจ) ให้เปิดใช้งาน การตั้งค่า → ตัวจัดการไฟล์ → แก้ไขไฟล์ออนไลน์ → ปิด เพื่อซ่อนการดำเนินการที่ทำลายล้างทั้งหมดจาก UI

## การดำเนินการไฟล์

แตะไอคอน **"..."** ใกล้ชื่อไฟล์เพื่อแสดงเมนูการดำเนินการ:

- **เล่นถัดไป** — แทรกไฟล์ที่ด้านบนของคิวโปรแกรมเล่น
- **เล่นทีหลัง** — ต่อท้ายไฟล์ที่ด้านล่างของคิวโปรแกรมเล่น
- **เพิ่มในคลังมีเดีย** — รวมไฟล์ในคลังมีเดียของคุณ จัดระเบียบตาม Album และ Genre
- **เพิ่มในเพลย์ลิสต์** — เพิ่มไฟล์ในเพลย์ลิสต์ที่มีอยู่หรือสร้างใหม่
- **แก้ไขแท็ก** — เปิดตัวแก้ไขแท็กในตัวเพื่อแก้ไขข้อมูลเมตา สำหรับไฟล์ออนไลน์ ไฟล์จะถูกดาวน์โหลดชั่วคราว แก้ไข และอัปโหลดใหม่หลังจากที่คุณยืนยัน
- **เพิ่มในรายการโปรด** — เพิ่มไฟล์ในรายการโปรดของคุณเพื่อการเข้าถึงอย่างรวดเร็ว
- **ดาวน์โหลด** — ดาวน์โหลดไฟล์หรือโฟลเดอร์ไปยังอุปกรณ์ของคุณเพื่อใช้งานออฟไลน์
- **เปลี่ยนชื่อ** — เปลี่ยนชื่อไฟล์โดยตรงบนที่เก็บข้อมูลระยะไกล
- **ย้าย** — ย้ายไฟล์ไปยังโฟลเดอร์อื่นในที่เก็บข้อมูลคลาวด์ของคุณ
- **เพิ่มในไฟล์เก็บถาวร** — บีบอัดไฟล์เป็นไฟล์ ZIP เดียว (คุณสมบัติ Premium)
- **เปิดใน** — ส่งออกไฟล์ไปยังแอปอื่นที่เข้ากันได้ผ่านระบบ Share sheet
- **ลบ** — ลบไฟล์ถาวร **การดำเนินการนี้ไม่สามารถยกเลิกได้**

## การดำเนินการโฟลเดอร์

สำหรับแต่ละโฟลเดอร์ในที่เก็บข้อมูลคลาวด์ของคุณ มีการดำเนินการมากมายโดยการแตะไอคอน **"..."** ถัดจากชื่อโฟลเดอร์

- **เล่นทั้งหมด** — แทนที่คิวโปรแกรมเล่นปัจจุบันด้วยวิดีโอทุกรายการในโฟลเดอร์
- **เล่นถัดไป / เล่นทีหลัง** — เพิ่มโฟลเดอร์ทั้งหมดในคิว
- **เพิ่มในคลังมีเดีย** — รวมเนื้อหาของโฟลเดอร์ในคลังมีเดียของคุณ
- **เพิ่มในเพลย์ลิสต์** — เพิ่มโฟลเดอร์ทั้งหมดในเพลย์ลิสต์
- **เพิ่มในรายการโปรด** — เพิ่มโฟลเดอร์ในรายการโปรดของคุณ
- **เปิดโหมดออฟไลน์** — นอกเหนือจากการดาวน์โหลดง่ายๆ สิ่งนี้จะตรวจสอบโฟลเดอร์สำหรับไฟล์ใหม่อย่างต่อเนื่องและดาวน์โหลดอัตโนมัติเมื่อปรากฏออนไลน์
- **ดาวน์โหลด** — ดาวน์โหลดเนื้อหาทั้งหมดของโฟลเดอร์ไปยังอุปกรณ์ของคุณเพื่อการเข้าถึงออฟไลน์
- **เปลี่ยนชื่อ / ย้าย** — เปลี่ยนชื่อหรือย้ายโฟลเดอร์บนที่เก็บข้อมูลระยะไกล
- **เพิ่มในไฟล์เก็บถาวร** — บีบอัดเนื้อหาโฟลเดอร์เป็นไฟล์ ZIP (คุณสมบัติ Premium)
- **ลบ** — ลบโฟลเดอร์และเนื้อหาถาวร **การดำเนินการนี้ไม่สามารถยกเลิกได้**

## คิวการถ่ายโอน

ที่มุมขวาบนของแท็บไฟล์มีปุ่ม **การถ่ายโอน** (ไอคอนลูกศรหมุน) แตะเพื่อเปิดคิวการถ่ายโอน — รายการการดาวน์โหลดและอัปโหลดที่ใช้งานทุกรายการจากแหล่งทั้งหมดของคุณ พร้อมความคืบหน้าแบบเรียลไทม์, ความเร็ว และ ETA ต่อไฟล์

## โหมดออฟไลน์และโฟลเดอร์ออฟไลน์ที่ซิงโครไนซ์

โหมดออฟไลน์เป็นคุณสมบัติที่สะดวกที่ให้คุณดูวิดีโอโปรดแม้เมื่อคุณไม่ได้เชื่อมต่ออินเทอร์เน็ต เมื่อคุณเปิดใช้งานโหมดออฟไลน์สำหรับโฟลเดอร์, เพลย์ลิสต์, อัลบั้ม หรือแนวเพลงใดๆ ไฟล์ทั้งหมดในคอลเลกชันนั้นจะถูกดาวน์โหลดไปยังอุปกรณ์ของคุณโดยอัตโนมัติเพื่อการเล่นออฟไลน์ ไฟล์จะปรากฏในส่วนโฟลเดอร์ออฟไลน์

เมื่อคุณเพิ่มไฟล์ใหม่บนเซิร์ฟเวอร์ระยะไกล ไฟล์เหล่านั้นก็จะถูกดาวน์โหลดโดยอัตโนมัติเช่นกัน — คอลเลกชันออฟไลน์ของคุณจึงอัปเดตอยู่เสมอโดยไม่ต้องทำอะไร ในการซิงค์ใหม่ด้วยตนเอง แตะสามจุดที่มุมขวาบนของโฟลเดอร์ออฟไลน์และเลือก ซิงโครไนซ์

คุณสามารถปรับ timeout การซิงโครไนซ์ได้ใน การตั้งค่า → ตัวจัดการไฟล์ → โฟลเดอร์ออฟไลน์ที่ซิงโครไนซ์ → ช่วงเวลา

คำแนะนำโดยละเอียดมีให้ [ที่นี่](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)

- **รูปแบบหน้าจอไฟล์** — เมนูธรรมดา (แสดงรายการโฟลเดอร์โดยตรง) หรือเมนูที่จัดกลุ่ม (จัดกลุ่มเนื้อหาตามหมวดหมู่ — การเข้าถึงด่วน, โฟลเดอร์พิเศษ, ที่เก็บข้อมูลคลาวด์ ฯลฯ)

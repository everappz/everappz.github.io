Το tab Αρχεία είναι ο all-in-one διαχειριστής αρχείων του Evervideo. Σε αντίθεση με τις περισσότερες εφαρμογές βίντεο που χωρίζουν τον αποθηκευτικό χώρο cloud από τα τοπικά αρχεία σε διαφορετικά tabs, το Evervideo συγχωνεύει και τα δύο σε μία ενιαία, κυλιόμενη οθόνη, ώστε να μπορείτε να μετακινείστε από έναν Plex server σε έναν φάκελο cloud έως τον φάκελο Έγγραφα του iPhone σας χωρίς να εναλλάσσετε tabs.

Το tab Αρχεία χωρίζεται σε σαφείς ενότητες που εμφανίζονται με αυτή τη σειρά στην οθόνη σας:

- **Γρήγορη Πρόσβαση** — πρόσφατα και αγαπημένα για αρχεία και φακέλους που ανοίξατε πιο πρόσφατα.
- **Αρχεία σε Αυτήν την Εφαρμογή** — ό,τι βρίσκεται στον φάκελο Έγγραφα με sandbox του Evervideo.
- **Αρχεία σε Αυτό το iPhone / iPad / Mac** — βίντεο αλλού στη συσκευή σας, που εμφανίζονται μέσω του επιλογέα αρχείων συστήματος.
- **Αποθήκευση Cloud** — κάθε λογαριασμός cloud, NAS και media server που έχετε συνδέσει.
- **Διαθέσιμες συσκευές** — servers και μονάδες δίσκου που ανακαλύφθηκαν αυτόματα μέσω Bonjour / mDNS στο τοπικό σας δίκτυο.

Στην επάνω δεξιά γωνία της οθόνης Αρχεία υπάρχει ένα κουμπί Μεταφορές (εικονίδιο με περιστρεφόμενα βέλη). Πατήστε το για να ανοίξετε την Ουρά Μεταφορών όπου παρακολουθείτε κάθε λήψη και αποστολή σε όλες τις πηγές σας.

## Σύνδεση σε Αποθήκευση Cloud

Η ενότητα Αποθήκευση Cloud του tab Αρχεία είναι ο τόπος όπου βρίσκεται κάθε συνδεδεμένος λογαριασμός, NAS, media server και ροή — δίπλα-δίπλα, σε μία κυλιόμενη λίστα.

{{< /cards >}}

- Ανοίξτε το tab **Αρχεία**.
- Κυλήστε στην ενότητα **Αποθήκευση Cloud**.
- Πατήστε **Σύνδεση με αποθήκευση cloud** από το μενού.
- Επιλέξτε μια υπηρεσία αποθήκευσης cloud από τη λίστα.
- Εισάγετε τα διαπιστευτήρια σας στην επίσημη σελίδα εξουσιοδότησης του παρόχου cloud, στη συνέχεια πατήστε **Ολοκλήρωση**.

## Υποστηριζόμενες Υπηρεσίες Αποθήκευσης Cloud, Media Servers και Πρωτόκολλα

Το Evervideo υποστηρίζει μια εξαιρετικά ευρεία γκάμα πηγών για τα βίντεό σας. Όλα τα παρακάτω λειτουργούν μέσω μιας ενιαίας ροής Σύνδεσης με αποθήκευση cloud.

**Προσωπικός χώρος αποθήκευσης cloud:** iCloud Drive · Dropbox · OneDrive · Google Drive · MEGA · Box · pCloud · Yandex Disk · WD My Cloud Home · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘).

**Αυτο-φιλοξενούμενοι media servers:** Plex · Jellyfin · Emby · Subsonic (και κάθε Subsonic-συμβατός server — Airsonic, Funkwhale, Gonic, Ampache) · Navidrome · Nextcloud (και ownCloud μέσω του κοινού API) · Synology Drive · QNAP.

**NAS και πρωτόκολλα κοινής χρήσης αρχείων:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP · FTPS · SFTP (SSH File Transfer Protocol, κωδικός πρόσβασης ή αυθεντικοποίηση δημοσίου κλειδιού) · NFS · DLNA / UPnP (αναπαραγωγή και λήψη).

**Live και ροές IP κάμερας:** RTSP — στρέψτε το Evervideo σε οποιαδήποτε URL `rtsp://` και απλά αναπαράγεται. Ιδανικό για κάμερες ασφαλείας, ροές IPTV, κάμερες κουδουνιών, ενδοεπικοινωνίες βρεφών και παρόμοιες ζωντανές πηγές.

**Αντικειμενική αποθήκευση συμβατή με S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces και οποιοδήποτε άλλο endpoint S3-API.

**Βιβλιοθήκες στη συσκευή:** η βιβλιοθήκη Φωτογραφιών (όλα τα βίντεο, εγγραφές οθόνης, φωτογραφικά άλμπουμ) και η βιβλιοθήκη εφαρμογής Μουσική (Άλμπουμ, Είδη, Λίστες αναπαραγωγής) — και οι δύο εμφανίζονται μέσα στη Βιβλιοθήκη Πολυμέσων, οπότε δεν χρειάζεται να αντιγράψετε τίποτα.

**Ανακάλυψη τοπικού δικτύου:** η ενότητα Διαθέσιμες συσκευές παραθέτει αυτόματα κάθε υπηρεσία Bonjour / mDNS στο Wi-Fi δίκτυό σας — Plex, Jellyfin, Emby servers, Synology, QNAP, δρομολογητές AirPort με συνδεδεμένες μονάδες, Apple Time Capsule — ώστε να μπορείτε να πατάτε για σύνδεση χωρίς να πληκτρολογείτε διεύθυνση IP.

Κάθε σύνδεση χρησιμοποιεί το επίσημο SDK ή ανοιχτό πρωτόκολλο της υπηρεσίας, με OAuth-βασισμένη εξουσιοδότηση όπου υποστηρίζεται. Μπορείτε να συνδέσετε πολλαπλούς λογαριασμούς της ίδιας υπηρεσίας (για παράδειγμα, δύο λογαριασμούς Google Drive ή τόσο έναν Plex όσο και έναν Jellyfin server) και να τους περιηγείστε δίπλα-δίπλα στην ενότητα Αποθήκευση Cloud.

## Ασφάλεια και Απόρρητο

Το Evervideo χρησιμοποιεί μόνο επίσημα SDK και ασφαλείς συνδέσεις για να αλληλεπιδρά με τις συνδεδεμένες υπηρεσίες cloud. Τα στοιχεία σύνδεσης και ο κωδικός πρόσβασής σας δεν είναι προσβάσιμα στην εφαρμογή — όλες οι μεταφορές είναι κρυπτογραφημένες με TLS.

Όταν εισάγετε τα στοιχεία σύνδεσης και τον κωδικό πρόσβασής σας, η εφαρμογή σάς δείχνει την επίσημη σελίδα εξουσιοδότησης που παρέχεται από τον πάροχο της υπηρεσίας cloud, και ολόκληρη η διαδικασία εξουσιοδότησης πραγματοποιείται εκτός της εφαρμογής. Ο πάροχος της υπηρεσίας cloud στέλνει στη συνέχεια ένα auth-token στην εφαρμογή μετά από επιτυχή εξουσιοδότηση, και αυτό το token χρησιμοποιείται για κλήσεις API.

Ένα auth-token είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με την αποθήκευση cloud για λογαριασμό σας. Το token αποθηκεύεται στη συσκευή σας στον ασφαλή χώρο αποθήκευσης συστήματος της Apple (Keychain), ο οποίος είναι κρυπτογραφημένος κατά την ανάπαυση και προστατεύεται από τον κωδικό πρόσβασης της συσκευής ή το βιομετρικό κλείδωμά σας. Μπορείτε να κατεβάσετε αρχεία από τις συνδεδεμένες υπηρεσίες cloud στη συσκευή σας· αυτά τα αρχεία τοποθετούνται στον κατάλογο Έγγραφα της εφαρμογής και μπορούν να αφαιρεθούν ανά πάσα στιγμή χρησιμοποιώντας τον ενσωματωμένο διαχειριστή αρχείων.

Η εφαρμογή δεν μοιράζεται καμία πληροφορία από τους συνδεδεμένους λογαριασμούς cloud σας με την Everappz, διαφημιστές ή οποιοδήποτε τρίτο μέρος. Μπορείτε να ανακαλέσετε την πρόσβαση στον λογαριασμό cloud σας ανά πάσα στιγμή ανοίγοντας τη σελίδα ρυθμίσεων λογαριασμού στο πρόγραμμα περιήγησής σας.

## Απόρριψη Auth-Token

Για να ανακαλέσετε ένα auth-token, συνδεθείτε στον λογαριασμό cloud σας σε ένα πρόγραμμα περιήγησης και μεταβείτε στη σελίδα ασφαλείας ή συνδεδεμένων εφαρμογών. Εκεί μπορείτε να βρείτε κάθε εφαρμογή τρίτου που είναι συνδεδεμένη με τον λογαριασμό cloud σας και να αφαιρέσετε οποιαδήποτε από αυτές αν δεν θέλετε πλέον να τη χρησιμοποιείτε. Λεπτομερείς οδηγίες για λογαριασμούς Google είναι διαθέσιμες [εδώ](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account).

Μπορείτε επίσης να αποσυνδέσετε τον λογαριασμό cloud μέσα στην εφαρμογή — όταν το κάνετε, το auth-token διαγράφεται αμέσως από τη συσκευή σας. Αν απεγκαταστήσετε την εφαρμογή από τη συσκευή σας, όλα τα ληφθέντα δεδομένα και τα access tokens αφαιρούνται αυτόματα μαζί της.

## Αποσύνδεση Αποθήκευσης Cloud ή Αλλαγή Ρυθμίσεων

- **Πρόσβαση σε επιλογές αποθήκευσης cloud** — εντοπίστε τη συνδεδεμένη υπηρεσία στην ενότητα **Αποθήκευση Cloud** του tab Αρχεία.
- **Πατήστε το κουμπί "..."** δίπλα στον τίτλο της υπηρεσίας για να ανοίξετε πρόσθετες επιλογές:
  - **Μετονομασία** — αλλαγή του ονόματος της υπηρεσίας cloud όπως εμφανίζεται στη λίστα σας.
  - **Ρυθμίσεις** — τροποποίηση της ρύθμισης ή των δεδομένων ελέγχου ταυτότητας. Μερικές φορές μπορεί να χρειαστεί να επανεξουσιοδοτήσετε τη συνδεδεμένη υπηρεσία cloud αν έχει λήξει το token εξουσιοδότησης.
  - **Αποσύνδεση** — πλήρης διακοπή της σύνδεσης μεταξύ της εφαρμογής και της υπηρεσίας cloud. Αυτό αφαιρεί όλα τα βίντεο που σχετίζονται με αυτή την υπηρεσία από τη βιβλιοθήκη σας, αλλά τα αφήνει ανέπαφα στον server.

## Σύνδεση σε Υπολογιστή ή NAS

Μπορείτε να συνδέσετε τον υπολογιστή σας, την προσωπική NAS ή άλλη συσκευή δικτύου χρησιμοποιώντας το πρωτόκολλο SMB, WebDAV, FTP / FTPS, SFTP, NFS ή DLNA. Αυτός είναι ο ευκολότερος τρόπος να φέρετε μια υπάρχουσα βιβλιοθήκη οικιακών πολυμέσων — είτε βρίσκεται σε Mac, Windows PC, Synology, QNAP, Apple Time Capsule ή WD My Cloud Home — στο Evervideo χωρίς να αντιγράψετε τίποτα.

### Σύνδεση σε Υπολογιστή Χρησιμοποιώντας SMB

- Πατήστε **Σύνδεση με αποθήκευση cloud → SMB**.
- Εισάγετε τη διεύθυνση IP του υπολογιστή και το όνομα κοινόχρηστου φακέλου χρησιμοποιώντας τη μορφή `smb://computer-ip-address/shared-folder-name`.
- Επιλέξτε την έκδοση πρωτοκόλλου: **Auto**, **SMB1** ή **SMB2**.
- Εισάγετε τα στοιχεία σύνδεσης και τον κωδικό πρόσβασής σας (αν απαιτείται).
- Πατήστε **Ολοκλήρωση**.

Αν η σύνδεση είναι επιτυχής, το κοινόχρηστο εμφανίζεται στην ενότητα Αποθήκευση Cloud. Ένα πλήρες σεμινάριο για τη σύνδεση του Mac ή PC σας χρησιμοποιώντας SMB είναι διαθέσιμο [εδώ](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).

### Σύνδεση σε NAS Χρησιμοποιώντας WebDAV

Όλα τα βήματα είναι τα ίδια με το SMB, εκτός από το πεδίο URL. Χρησιμοποιήστε `http://server-name` ή `https://server-name` αν ο server υποστηρίζει SSL. Το WebDAV λειτουργεί με Synology, QNAP, Nextcloud, ownCloud, WD My Cloud Home και οποιοδήποτε άλλο server με endpoint WebDAV.

Ένα πλήρες σεμινάριο για WebDAV είναι διαθέσιμο [εδώ](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).

### Σύνδεση μέσω DLNA / UPnP

Μοιραστείτε μια βιβλιοθήκη πολυμέσων στον Windows PC ή NAS σας χρησιμοποιώντας το πρωτόκολλο DLNA / UPnP και αποκτήστε πρόσβαση σε αυτήν μέσα στο Evervideo όπως περιγράφεται [εδώ](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone). Το DLNA υποστηρίζεται ευρέως, αλλά σας επιτρέπει μόνο να αναπαράγετε ή να κατεβάζετε βίντεο — δεν μπορείτε να ανεβάσετε αρχεία ή να δημιουργήσετε νέους φακέλους σε έναν DLNA server.

### Σύνδεση Χρησιμοποιώντας FTP, FTPS ή SFTP

Το Evervideo υποστηρίζει επίσης τα κλασικά πρωτόκολλα μεταφοράς αρχείων. Για να συνδέσετε έναν server μέσω FTP ή FTPS, πατήστε Σύνδεση με αποθήκευση cloud → FTP, εισάγετε το URL host στη μορφή `ftp://server-name` (ή `ftps://server-name` για κρυπτογραφημένη σύνδεση), δώστε τα στοιχεία σύνδεσης και τον κωδικό πρόσβασής σας, στη συνέχεια πατήστε Ολοκλήρωση. Για SFTP (SSH File Transfer Protocol), επιλέξτε SFTP και δώστε είτε κωδικό πρόσβασης είτε ζεύγος SSH κλειδιών.

### Σύνδεση σε Κοινόχρηστο NFS

Το Evervideo περιλαμβάνει υποστήριξη NFS (Network File System) — βολικό για Linux hosts, BSD servers και συσκευές NAS που προτιμούν να εκθέτουν βιβλιοθήκες βίντεο μέσω NFS αντί για SMB. Επιλέξτε NFS στο μενού Σύνδεση με αποθήκευση cloud, εισάγετε τη διεύθυνση server και τον εξαγόμενο διαδρομή, και πατήστε Ολοκλήρωση.

## Σύνδεση Plex Media Server

Το Evervideo μπορεί να συνδεθεί απευθείας σε έναν Plex Media Server και να περιηγηθεί στις βιβλιοθήκες Ταινιών, Τηλεοπτικών Σειρών και Οικιακών Βίντεο. Πατήστε Σύνδεση με αποθήκευση cloud → Plex, συνδεθείτε με τον λογαριασμό Plex, επιλέξτε έναν server, και η βιβλιοθήκη εμφανίζεται δίπλα στις υπηρεσίες cloud. Οι Plex servers στο ίδιο τοπικό δίκτυο ανακαλύπτονται επίσης αυτόματα στην ενότητα Διαθέσιμες συσκευές.

## Σύνδεση Jellyfin ή Emby Server

Τόσο το Jellyfin (ανοιχτού κώδικα) όσο και το Emby (εμπορικό) λειτουργούν εγγενώς στο Evervideo. Πατήστε Σύνδεση με αποθήκευση cloud → Jellyfin ή Emby, εισάγετε το URL του server σας (συνήθως κάτι σαν `http://server-ip:8096`) και τα διαπιστευτήρια, και η βιβλιοθήκη είναι έτοιμη για ροή.

## Σύνδεση Subsonic ή Navidrome Server

Το Evervideo μιλά το Subsonic API, που σημαίνει ότι λειτουργεί με το Subsonic ίδιο, Navidrome και κάθε άλλον Subsonic-συμβατό server — συμπεριλαμβανομένων των Airsonic, Funkwhale, Gonic, Logitech Media Server (LMS) και Ampache. Πατήστε Σύνδεση με αποθήκευση cloud → Subsonic, εισάγετε το URL server και τα διαπιστευτήρια, και η βιβλιοθήκη εμφανίζεται στην ενότητα Αποθήκευση Cloud.

## Σύνδεση RTSP Ροής (IP Κάμερες, Live TV, IPTV)

Το Evervideo έχει εγγενή υποστήριξη RTSP, οπότε μπορείτε να το στρέψετε σε οποιαδήποτε πηγή RTSP — κάμερες ασφαλείας, κάμερες κουδουνιών, παρόχους IPTV, ενδοεπικοινωνίες βρεφών, εκπομπές — και το Evervideo θα τραβήξει και θα αποκωδικοποιήσει τη ροή ζωντανά. Πατήστε Online Σύνδεσμοι → Προσθήκη συνδέσμου, επικολλήστε την πλήρη URL (`rtsp://camera-ip:port/stream-path`), δώστε στοιχεία σύνδεσης και κωδικό πρόσβασης αν απαιτείται, και πατήστε Ολοκλήρωση.

## Σύνδεση σε Αντικειμενική Αποθήκευση Συμβατή με S3

Για αυτο-φιλοξενούμενους και προχωρημένους χρήστες που χρησιμοποιούν αντικειμενική αποθήκευση cloud, το Evervideo περιλαμβάνει συνδετήρα συμβατό με S3. Πατήστε Σύνδεση με αποθήκευση cloud → Αποθήκευση S3, στη συνέχεια εισάγετε το URL endpoint, περιοχή, κλειδί πρόσβασης, μυστικό κλειδί και όνομα bucket. Αυτό λειτουργεί με AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces και οποιοδήποτε άλλο endpoint S3-API.

## Διαθέσιμες Συσκευές

Αυτή η ενότητα εμφανίζει κάθε συσκευή στο τοπικό σας δίκτυο στην οποία μπορείτε να συνδεθείτε από το Evervideo μέσω ανακάλυψης Bonjour / mDNS — Plex, Jellyfin, Emby servers, Synology, QNAP, δρομολογητές AirPort με συνδεδεμένες μονάδες, Apple Time Capsule και ούτω καθεξής. Για να εγκαταστήσετε μια σύνδεση:

- Κυλήστε στην ενότητα Διαθέσιμες συσκευές στο tab Αρχεία.
- Πατήστε τη συσκευή στην οποία θέλετε να συνδεθείτε.
- Αν χρειάζεται, εισάγετε τα στοιχεία σύνδεσής σας για να ολοκληρώσετε τη σύνδεση.

## Wi-Fi Drive

Το Wi-Fi Drive σας επιτρέπει να μεταφέρετε αρχεία ασύρματα από τον υπολογιστή σας στη συσκευή iOS μέσω οποιουδήποτε προγράμματος περιήγησης επιτραπέζιου υπολογιστή, Finder ή Εξερεύνηση Αρχείων. Η συσκευή και ο υπολογιστής σας πρέπει να βρίσκονται στο ίδιο δίκτυο Wi-Fi.

### Ενεργοποίηση Wi-Fi Drive

- Στο tab Αρχεία, κυλήστε στο Αποθήκευση Cloud → Σύνδεση μέσω Wi-Fi για να ανοίξετε την κύρια οθόνη του Wi-Fi Drive.
- (Προαιρετικό) Ορίστε όνομα χρήστη και κωδικό πρόσβασης για τον ενσωματωμένο web server.
- Πατήστε Έναρξη Wi-Fi Drive.

### Πρόσβαση στο Wi-Fi Drive από τον Υπολογιστή σας

- Ανοίξτε ένα πρόγραμμα περιήγησης στον υπολογιστή σας (Chrome, Firefox, Safari κ.λπ.).
- Εισάγετε τη διεύθυνση URL που εμφανίζεται από την εφαρμογή.
- Σύρετε και αποθέστε αρχεία από τον υπολογιστή σας στην ιστοσελίδα — θα αρχίσουν να μεταφέρονται στη συσκευή iOS σας.

Μπορείτε επίσης να προσαρτήσετε το Wi-Fi Drive απευθείας στο **Finder** σε macOS (Μετάβαση → Σύνδεση στον Server…) ή στην Εξερεύνηση Αρχείων στα Windows (Αντιστοίχιση Μονάδας Δικτύου…) και να αντιμετωπίζετε το iPhone ή iPad σας ως κανονική μονάδα δικτύου.

Λεπτομερείς οδηγίες είναι διαθέσιμες [εδώ](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).

Ανοίξτε Αρχεία → Αρχεία σε Αυτό το iPhone → Άνοιγμα Φακέλου, πλοηγηθείτε στη μονάδα, και επιλέξτε ένα αρχείο βίντεο ή φάκελο. Το Evervideo αναπαράγει αρχεία απευθείας από τη μονάδα χωρίς να τα αντιγράψει στην εσωτερική αποθήκευση — βολικό για πολύ μεγάλες βιβλιοθήκες 4K.

## Ειδικοί Φάκελοι

Μέσα στο tab Αρχεία θα δείτε αρκετούς ειδικούς φακέλους που το Evervideo δημιουργεί και χρησιμοποιεί αυτόματα:

- **Λήψεις** — κάθε αρχείο που κατεβαίνει από το cloud εμφανίζεται εδώ από προεπιλογή. Προσαρμόστε στις Ρυθμίσεις → Διαχειριστής Αρχείων → Αποθήκευση Ληφθέντων Αρχείων Σε.
- **Cache Player** — η κρυφή μνήμη του media player. Από προεπιλογή ο player κατεβάζει επερχόμενα βίντεο για ομαλή αναπαραγωγή. Μπορείτε να απενεργοποιήσετε την κρυφή μνήμη στις ρυθμίσεις εφαρμογής ή απλά να διαγράψετε αυτόν τον φάκελο.
- **iCloud** — αρχεία σε αυτόν τον φάκελο συγχρονίζονται σε όλες τις συσκευές που είναι συνδεδεμένες στον ίδιο λογαριασμό iCloud.
- **Εκτός σύνδεσης φάκελοι** — όταν επισημάνετε έναν φάκελο, λίστα αναπαραγωγής, άλμπουμ ή είδος ως διαθέσιμο εκτός σύνδεσης, τα αρχεία κατεβαίνουν σε αυτόν τον φάκελο.

## Επάνω Γραμμή Εργαλείων

Η επάνω γραμμή εργαλείων, που βρίσκεται κάτω από τη γραμμή πλοήγησης, προσφέρει αρκετές ενέργειες που μπορείτε να εμφανίσετε ή να αποκρύψετε με χειρονομία σάρωσης προς τα κάτω:

- **Αναζήτηση** — εκτέλεση αναζήτησης εντός του τρέχοντος φακέλου ή ενότητας.
- **Συνέχεια Αναπαραγωγής** — αν είναι ενεργοποιημένο στις ρυθμίσεις, επαναφορά ουράς player και τελευταίας θέσης βίντεο για τον τρέχοντα φάκελο.
- **Αναπαραγωγή Όλων** — σάρωση του τρέχοντος φακέλου και των υποφακέλων του και προσθήκη αρχείων στην ουρά player.
- **Ανακάτεμα Όλων** — όπως Αναπαραγωγή Όλων, αλλά ανακατεύει πριν την προσθήκη.

## Επιλογές Φακέλου

Όταν ανοίγετε έναν φάκελο, πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία για αυτές τις ενέργειες:

- **Επιλογή** — ενεργοποίηση λειτουργίας επιλογής αρχείου.
- **Νέος Φάκελος** — δημιουργία νέου φακέλου εντός του τρέχοντος φακέλου.
- **Ενεργοποίηση εκτός σύνδεσης λειτουργίας** — ενεργοποίηση offline συγχρονισμού για τον τρέχοντα φάκελο. Νέα αρχεία που προστίθενται online κατεβαίνουν αυτόματα.
- **Αποστολή Αρχείων** — αποστολή αρχείων από τη συσκευή στον online φάκελο.
- **Αναζήτηση** — αναζήτηση εντός του φακέλου.
- **Ταξινόμηση** — ταξινόμηση αρχείων κατά όνομα, μέγεθος, ημερομηνία τροποποίησης ή μεταδεδομένα.
- **Πλέγμα / Προβολή Λίστας** — εναλλαγή μεταξύ προβολής πίνακα και προβολής μικρογραφιών. Η προβολή μικρογραφιών εμφανίζει μεγάλες προεπισκοπήσεις αφίσας βίντεο.

## Ενέργειες Αρχείου

Πατήστε το εικονίδιο **"..."** κοντά στον τίτλο ενός αρχείου για να αποκαλύψετε το μενού ενεργειών του:

- **Αναπαραγωγή Επόμενου** — εισαγωγή του αρχείου στην κορυφή της ουράς player.
- **Αναπαραγωγή Αργότερα** — προσάρτηση του αρχείου στο κάτω μέρος της ουράς player.
- **Προσθήκη στη Βιβλιοθήκη Πολυμέσων** — ενσωμάτωση του αρχείου στη βιβλιοθήκη πολυμέσων, οργανωμένη κατά Άλμπουμ και Είδος.
- **Προσθήκη σε Λίστα Αναπαραγωγής** — προσθήκη του αρχείου σε υπάρχουσα λίστα αναπαραγωγής ή δημιουργία νέας.
- **Επεξεργασία Ετικετών** — άνοιγμα του ενσωματωμένου επεξεργαστή ετικετών για τροποποίηση μεταδεδομένων. Για online αρχεία, το αρχείο κατεβαίνει προσωρινά, επεξεργάζεται και στη συνέχεια επαναφορτώνεται αφού επιβεβαιώσετε.
- **Προσθήκη στα Αγαπημένα** — προσθήκη του αρχείου στη λίστα αγαπημένων για γρήγορη πρόσβαση.
- **Λήψη** — λήψη του αρχείου ή φακέλου στη συσκευή για offline χρήση.
- **Μετονομασία** — μετονομασία του αρχείου απευθείας στον απομακρυσμένο χώρο αποθήκευσης.
- **Μετακίνηση** — μεταφορά του αρχείου σε διαφορετικό φάκελο εντός του χώρου αποθήκευσης cloud.
- **Προσθήκη στο Αρχείο** — ομαδοποίηση του αρχείου σε ένα μοναδικό αρχείο ZIP (λειτουργία Premium).
- **Άνοιγμα Σε** — εξαγωγή του αρχείου σε άλλη συμβατή εφαρμογή μέσω του φύλλου κοινής χρήσης συστήματος.
- **Διαγραφή** — μόνιμη αφαίρεση του αρχείου. **Αυτή η ενέργεια δεν μπορεί να αναιρεθεί.**

## Ενέργειες Φακέλου

Για κάθε φάκελο στον χώρο αποθήκευσης cloud, υπάρχουν πολλές διαθέσιμες ενέργειες πατώντας το εικονίδιο **"..."** δίπλα στον τίτλο φακέλου.

- **Αναπαραγωγή Όλων** — αντικατάσταση της τρέχουσας ουράς player με κάθε βίντεο στον φάκελο.
- **Αναπαραγωγή Επόμενου / Αναπαραγωγή Αργότερα** — προσθήκη ολόκληρου του φακέλου στην ουρά.
- **Προσθήκη στη Βιβλιοθήκη Πολυμέσων** — ενσωμάτωση του περιεχομένου φακέλου στη βιβλιοθήκη πολυμέσων.
- **Προσθήκη σε Λίστα Αναπαραγωγής** — προσθήκη ολόκληρου του φακέλου σε μια λίστα αναπαραγωγής.
- **Προσθήκη στα Αγαπημένα** — προσθήκη του φακέλου στα αγαπημένα σας.
- **Ενεργοποίηση εκτός σύνδεσης λειτουργίας** — πέρα από μια απλή λήψη, αυτό παρακολουθεί συνεχώς τον φάκελο για νέα αρχεία και τα κατεβάζει αυτόματα καθώς εμφανίζονται online.
- **Λήψη** — λήψη όλων των περιεχομένων του φακέλου στη συσκευή για offline πρόσβαση.
- **Μετονομασία / Μετακίνηση** — μετονομασία ή μεταφορά του φακέλου στον απομακρυσμένο χώρο αποθήκευσης.
- **Προσθήκη στο Αρχείο** — ομαδοποίηση περιεχομένων φακέλου σε αρχείο ZIP (λειτουργία Premium).
- **Διαγραφή** — μόνιμη αφαίρεση του φακέλου και των περιεχομένων του. **Αυτή η ενέργεια δεν μπορεί να αναιρεθεί.**

## Offline Λειτουργία και Συγχρονισμένοι Εκτός Σύνδεσης Φάκελοι

Η offline λειτουργία είναι μια εύχρηστη λειτουργία που σας επιτρέπει να παρακολουθείτε τα αγαπημένα σας βίντεο ακόμα και όταν δεν είστε συνδεδεμένοι στο διαδίκτυο. Όταν ενεργοποιείτε offline λειτουργία για οποιοδήποτε φάκελο, λίστα αναπαραγωγής, άλμπουμ ή είδος, όλα τα αρχεία εντός αυτής της συλλογής κατεβαίνουν αυτόματα στη συσκευή σας για offline αναπαραγωγή. Εμφανίζονται στην ενότητα Εκτός σύνδεσης φάκελοι.

Όταν προσθέτετε νέα αρχεία στον απομακρυσμένο server, κατεβαίνουν επίσης αυτόματα — ώστε η offline συλλογή σας να παραμένει ενημερωμένη χωρίς να κάνετε τίποτα. Για χειροκίνητο επανασυγχρονισμό, πατήστε τις τρεις τελείες στην επάνω δεξιά γωνία ενός offline φακέλου και επιλέξτε Συγχρονισμός.

Μπορείτε να ρυθμίσετε το χρονικό διάστημα συγχρονισμού στις Ρυθμίσεις → Διαχειριστής Αρχείων → Συγχρονισμένοι εκτός σύνδεσης φάκελοι → Χρονικό Διάστημα.

Λεπτομερείς οδηγίες είναι διαθέσιμες [εδώ](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

### ยินดีต้อนรับสู่คู่มือ Evervideo

Evervideo คือโปรแกรมเล่นสื่อคลาวด์แบบครบครันสำหรับ iPhone, iPad และ Mac ที่เปลี่ยนบัญชีคลาวด์สตอเรจ, NAS หรือมีเดียเซิร์ฟเวอร์ใดๆ ให้กลายเป็นคลังมีเดียส่วนตัวของคุณ — โดยไม่จำเป็นต้องอัปโหลดใหม่ และยังคงควบคุมไฟล์ของคุณได้อย่างเต็มที่

สร้างบนเครื่องยนต์เล่นแบบ FFmpeg แบบกำหนดเองพร้อมการถอดรหัส H.264 และ HEVC ด้วยฮาร์ดแวร์ Evervideo เล่นได้แทบทุก container และ codec สมัยใหม่ — MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS และรูปแบบ FFmpeg อีกมากมาย — ด้วยคุณภาพเต็ม พร้อมการบัฟเฟอร์อัจฉริยะเพื่อการสตรีมที่ราบรื่นผ่าน Wi-Fi หรือเครือข่ายมือถือ Picture-in-Picture ให้วิดีโอของคุณแสดงอยู่เหนือทุกแอป โปรแกรมเล่นขนาดเล็กที่แสดงตลอดเวลาให้คุณดูต่อขณะเรียกดูไลบรารี และ Chromecast กับ AirPlay 2 ส่งการเล่นไปยังทีวี, HomePod หรือลำโพงของคุณด้วยการแตะเพียงครั้งเดียว

Evervideo เชื่อมต่อกับแหล่งที่มาที่หลากหลายอย่างน่าทึ่ง ทั้งหมดจากแอปเดียว:

- **ที่เก็บข้อมูลคลาวด์ส่วนตัว:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · pCloud · Yandex Disk · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘)
- **มีเดียเซิร์ฟเวอร์แบบโฮสต์เอง:** Plex · Jellyfin · Emby · Subsonic · Navidrome · Nextcloud (และ ownCloud ผ่าน shared API) · Synology Drive · QNAP
- **NAS และโปรโตคอลแชร์ไฟล์:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP · FTPS · SFTP (รหัสผ่านหรือ public-key auth) · NFS · DLNA · UPnP
- **สตรีม Live และกล้อง IP:** RTSP — ชี้ Evervideo ไปที่ RTSP URL ใดก็ได้ (`rtsp://camera-ip/stream`) และจะเล่นทันที
- **ที่เก็บข้อมูลออบเจกต์ที่เข้ากันได้กับ S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces และ S3-API endpoint อื่นๆ
- **แหล่งบนอุปกรณ์:** ไลบรารี Photos (วิดีโอทั้งหมด, สั้น / กลาง / ยาว, การบันทึกหน้าจอ รวมถึง Photo Albums), ไลบรารี Music app (Albums, Genres, Playlists), USB / SD card และ Local Files

### โปรแกรมเล่นเดียวสำหรับทุกรูปแบบและ codec

ในขณะที่แอป iOS ส่วนใหญ่หยุดที่ H.264 / H.265 ใน MP4 / MOV, Evervideo รวม FFmpeg เข้ากับ codec ระบบของ Apple เพื่อให้คุณเล่นรูปแบบที่ระบบไม่รู้จัก — MKV containers, VP9, AV1, MPEG-2, OGG, Vorbis และอีกมากมาย — และสลับระหว่างการถอดรหัส H.264 / HEVC ด้วยฮาร์ดแวร์และการถอดรหัสซอฟต์แวร์โดยอัตโนมัติตามไฟล์และอุปกรณ์

คุณสามารถเลือกแทร็กวิดีโอ (Blu-ray rips หลายสตรีม), แทร็กเสียง (แทร็กคอมเมนต์, ดับเสียงอื่น) และแทร็กซับไตเติ้ล ไฟล์ซับไตเติ้ล SRT, VTT และ ASS / SSA ภายนอกโหลดได้ด้วยการแตะครั้งเดียว การจัดสไตล์ ASS / SSA ขั้นสูงแสดงผลได้อย่างถูกต้องด้วย libass ที่รวมมาด้วย

### ซับไตเติ้ลอัจฉริยะ

- **การเลือกแทร็กซับไตเติ้ล** — เหมาะสำหรับการเรียนภาษา
- **ไฟล์ซับไตเติ้ลภายนอก** (`.srt`, `.vtt`, `.ass`, `.ssa`) โหลดจากที่ใดก็ได้บนอุปกรณ์หรือจากคลาวด์
- **libass** สำหรับการแสดงผล ASS / SSA แบบมีสไตล์เต็มรูปแบบ
- **การเลือกแบบอักษรต่อแทร็กและระดับโลก** สำหรับซับไตเติ้ล
- **การเลือกแทร็กเสียง** — เลือกดับ, คอมเมนต์ หรือแทร็กของผู้กำกับ
- **การเลือกแทร็กวิดีโอ** สำหรับไฟล์หลายมุมมองหรือหลายเวอร์ชัน

### ปรับแต่งภาพและเสียง

อีควอไลเซอร์สองตัวเคียงข้างกัน: อีควอไลเซอร์เสียงสำหรับปรับเสียงเบสและเสียงแหลม (พร้อมนำเข้า / ส่งออก preset ที่กำหนดเอง) และอีควอไลเซอร์วิดีโอสำหรับปรับความสว่าง, คอนทราสต์, ความอิ่มตัวของสี และสีโทน (พร้อมนำเข้า / ส่งออกเช่นกัน) ทั้งสองเครื่องยนต์ยังเปิดเผยการควบคุมแบบ audiophile: อัตราการสุ่มตัวอย่างเอาต์พุตเสียง, จำนวนช่องสัญญาณ, ระยะเวลา IO buffer และ mixed mode — สำหรับผู้ใช้ที่มี DAC ภายนอกและ receiver โฮมเธียเตอร์

### Cast, AirPlay และ Picture-in-Picture

- **Picture-in-Picture (PiP):** ดูต่อขณะใช้งานแอปอื่น
- **AirPlay 2:** ส่งวิดีโอไปยัง Apple TV, HomePod หรือลำโพง / ทีวีที่รองรับ AirPlay 2
- **Google Chromecast:** cast โดยตรงไปยัง Chromecast หรือทีวีที่รองรับ Cast
- **โปรแกรมเล่นขนาดกะทัดรัด:** mini player ถาวรด้านบนทุกหน้าจอ ให้คุณเรียกดูไลบรารีโดยไม่เสียวิดีโอ

### ความเป็นส่วนตัวและความปลอดภัย

Evervideo ใช้ SDK อย่างเป็นทางการและการเข้าสู่ระบบแบบ OAuth จากทุกผู้ให้บริการคลาวด์ เพื่อให้รหัสผ่านของคุณไม่ถึงแอป token การเข้าถึงถูกเข้ารหัสใน iOS/MacOS Keychain ทุกการถ่ายโอนได้รับการป้องกัน TLS และการเพิกถอนการเข้าถึงจากบัญชีคลาวด์ (หรือการลบ Evervideo ออกจากอุปกรณ์) จะลบทุกอย่างทันที ปกป้องไลบรารีของคุณด้วยรหัสผ่าน 4 หลักเสริมเพื่อความเป็นส่วนตัวเพิ่มเติม

### เริ่มต้นกับ Evervideo

คู่มือนี้จะนำคุณผ่านทุกส่วนของ Evervideo บน iPhone, iPad และ Mac — ตั้งแต่การเชื่อมต่อบริการคลาวด์, การเรียกดูไลบรารี, การดาวน์โหลดและถ่ายโอนไฟล์, การจัดการเพลย์ลิสต์, ไปจนถึงการปรับแต่งโปรแกรมเล่นมีเดีย, อีควอไลเซอร์, ซับไตเติ้ล และ Picture-in-Picture ใช้การ์ดด้านล่างเพื่อข้ามไปยังส่วนที่คุณต้องการโดยตรง<br><br>

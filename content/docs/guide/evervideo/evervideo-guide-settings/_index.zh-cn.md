## 在 iOS 和 Mac 之间共享购买

终身购买和订阅通过使用 **iCloud** 同步信息在 iOS 和 Mac 之间共享。如果您在 iOS 设备上拥有 Premium，请确保已安装最新版本并启用了 iCloud。在 iOS 上启动应用并等待一分钟让购买信息上传到 iCloud，然后启动 Mac 版本——Premium 应该会自动激活。

您也可以点击应用设置中的**恢复购买**按钮。确保您有网络连接，并在两台设备上登录了相同的 iCloud 和 App Store 账户。

## 在新设备上恢复购买

要在新设备上恢复购买，请使用**购买 → 恢复购买**菜单。您将看到购买列表。如果看不到所有购买，请确认设备已连接到用于购买的同一 Apple ID，并确保 iCloud 已启用。

## 播放器

与播放相关的所有内容都在这里——音频、视频、字幕、设备、个性化、缓存和睡眠定时器。

### 通用

这些设置涵盖播放器的基本方面。

- **重复模式** — 选择视频结束时播放器的行为：
  - **全部重复** — 重播队列中的每个视频。
  - **单曲重复** — 仅重复当前视频。
  - **重复停止** — 当前视频结束时暂停。
  - **不重复** — 播放完队列后不重复。
- **随机模式** — 随机排列队列中视频的顺序（**开启**或**关闭**）。
- **保存播放位置** — 保存和恢复资料库中视频的播放位置。
- **保存播放器状态** — 在您关闭应用之前保留播放器状态，以便您可以从上次中断的地方继续。

### 视频

配置 Evervideo 解码和渲染视频的方式。

- **硬件解码 H.264** — 开启 / 关闭硬件加速的 AVC 解码。在电池和温度性能重要时使用；关闭以兼容特殊配置文件。
- **硬件解码 H.265 (HEVC)** — 与 HEVC 相同。现代 iPhone、iPad 和 Mac 可高效解码 HEVC。
- **首选像素格式** — 选择播放器向渲染器呈现的像素格式（默认值已针对设备进行优化）。
- **首选 FPS** — 设置目标播放 FPS。适用于匹配特定显示器刷新率。
- **视频缩放模式** — 适合 / 填充 / 拉伸 / 原始。确定画面如何填充显示区域。
- **视频显示模式** (iOS / iPadOS) — 视频在播放器视图中的呈现方式。
- **视频旋转** — 手动将画面旋转 0° / 90° / 180° / 270°（适用于方向错误录制的视频）。
- **视频均衡器** — 通过实时预览调整亮度、对比度、饱和度和色调。自定义预设可以**导出和导入**。
- **VR 视口** (iOS / iPadOS) — 适用于 360° 球形视频。拖动以环顾四周，捏合以缩放。
- **画中画 (PiP)** (iOS / iPadOS) — 启用 PiP 支持；当您将应用置于后台或点击 PiP 按钮时，应用将切换到浮动播放器窗口。

### 音频

配置音频播放和输出。

- **音轨** — 当视频有多个音轨时选择音频流（解说、配音等）。
- **音频均衡器** — 带预设和前置放大器的 10 频段均衡器。自定义预设可以导出 / 导入。
- **音频输出采样率** — 44.1 kHz、48 kHz、64 kHz、88.2 kHz、96 kHz。适用于外置 DAC。
- **音频输出声道数** — 单声道、立体声、5.1、ITU BS.775-1、SDDS 等。
- **音频输出首选 IO 缓冲区时长** — 低延迟高清播放的典型值约为 5 毫秒（0.005 秒）。根据您的硬件调整。
- **音频输出模式**（仅 iOS）— 混合模式让 Evervideo 的音频与其他应用混合。

### 字幕

Evervideo 包含全面的字幕支持。

- **字幕轨道** — 从文件中的嵌入字幕轨道中选择。
- **外部字幕文件** — 从设备或任何已连接的云服务加载外部 `.srt`、`.vtt`、`.ass` 或 `.ssa` 文件。
- **字体** — 为主字幕轨道选择字体。

### 设备（仅 iOS/iPadOS）

选择 **AirPlay** 或 **Google Chromecast** 设备进行视频输出。

### 个性化

- **播放器布局样式** — 简约（Evervideo 默认）、底部、古典、经典等。
- **主屏幕操作** — 选择主播放器屏幕上显示的按钮。
- **锁屏控件** — 跳过时间、添加书签、添加到收藏夹。
- **跳过时间间隔** — 选择跳过按钮的时间间隔（5 秒、10 秒、15 秒、30 秒等）。
- **专辑封面滚动样式** — 封面图片的首选滚动样式。
- **附加元素** — 音频格式信息、音量滑块。

### 文件加载

配置视频数据如何从网络流式传输。

- **网络类型** — 仅 Wi-Fi，或 Wi-Fi + 蜂窝网络。
- **预加载时间** — 缓冲区长度，用于在慢速网络上实现更流畅的播放。
- **使用直接 URL** — 在支持的情况下，使用直接 URL 更快加载。

### 播放缓存

播放器队列中的视频会自动下载以确保流畅播放。您可以在此禁用此选项或配置缓存大小。

### 睡眠定时器

激活定时器以在指定时间后自动停止播放。点击配置图标可使用**精确模式**，具有逐分钟精度。

## 媒体资料库

管理自动同步、元数据、专辑封面、播放列表、最近使用和收藏夹。

### 元数据读取

当您将视频添加到资料库时，元数据读取器会在后台扫描它们，并按专辑和流派整理。您可以调整扫描速度、禁用读取器或使用**重新加载元数据**触发完整重新扫描。**规范化元数据编码**自动修复损坏的字符编码（常见于来自 Windows 电脑的文件）。

### 在线同步

自动将已连接云服务和媒体服务器中的视频添加到您的资料库。选择要扫描的文件夹，配置应用同步频率，并手动启动 / 停止同步。在线同步仅在应用活跃时运行——对于大型资料库，请使用桌面版本，然后使用**备份与恢复**传输同步的资料库。

### 离线同步

当您将云文件夹标记为可离线使用时，它会出现在**文件 → 离线文件夹**中并自动下载。在线新增的文件也会下载。从此屏幕配置时间间隔并触发手动同步。

### 个性化

- **媒体资料库屏幕样式** — 纯菜单、分组菜单、标签菜单。
- 切换是否在详情屏幕中显示大专辑封面。

### 专辑封面

- **网络类型** — Wi-Fi 或 Wi-Fi + 蜂窝网络。
- **为在线文件加载专辑封面** — 从云文件中提取嵌入的图稿（使用额外数据）。
- **在文件夹中搜索** — 当视频没有嵌入封面时，使用同一文件夹中的 JPEG / PNG 图片。
- **封面质量** — 调整封面缓存的分辨率。
- **在文件夹中显示** / **删除全部** — 管理图稿缓存。

### 播放列表

启用将同一视频添加到播放列表两次（默认关闭）。

### 最近使用

管理最近播放的视频列表——更改大小、删除或导出为 M3U / CSV / TXT。

### 收藏夹

- **同步编辑** — 在媒体资料库和文件部分之间镜像收藏夹。
- **删除列表** — 清除收藏夹列表。
- **导出歌曲列表** — 将收藏夹导出为 M3U / CSV / TXT。

### 删除媒体资料库

清除媒体资料库数据库。您存储中的视频和音频文件保持不变。

## 密码锁

使用 **4 位数字密码**保护 Evervideo。启用后，每次应用启动时都会提示输入密码。结合设备上的 iOS Face ID / Touch ID 以获得额外保护。

## 文件管理器

控制文件的传输、存储和预览方式。

- **文件传输** — 网络偏好（仅 Wi-Fi 或 Wi-Fi + 蜂窝网络）。
- **最大并行任务数** — 设置并行下载线程数。
- **文件传输任务** — 查看当前活跃的上传和下载。
- **后台传输** — 允许在应用处于后台时下载。
- **将下载文件保存到** — 默认下载目录。
- **同步离线文件夹** — 管理离线模式文件夹。
- **时间间隔** — 检查离线文件夹变化的频率。
- **显示完整文件名** — 在文件管理器中显示扩展名。
- **编辑在线文件** — 禁用以切换到已连接云服务的只读模式。
- **打开时复制文件** — 如何处理从其他应用导入的文件。
- **文件缩略图** — 管理生成的文件缩略图。
- **删除临时文件** — 清除应用缓存文件夹。

## Wi-Fi Drive

激活 Wi-Fi Drive，使用桌面网络浏览器、Finder 或文件资源管理器从电脑向设备传输文件。详细说明[在此处](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)提供。

## 小部件

启用小部件使应用在播放期间更新小部件数据。小部件更新需要额外电量，因此默认关闭——仅在您实际使用主屏幕或锁屏上的小部件时启用。

您可以通过长按主屏幕或锁屏，点击 **+**，搜索 **Evervideo** 并选择小部件大小来添加 Evervideo 小部件。小部件显示正在播放的视频并直接点击进入全屏播放器。小部件也适用于 macOS 通知中心。

## 个性化

根据您的偏好自定义用户界面。

- **应用图标** — 备用主屏幕图标（Premium）。
- **配色方案** — 深色、浅色或默认（跟随系统外观）。
- **背景样式** — 模糊专辑封面或纯色。
- **列表中项目的外观** — 自动调整行高；显示较小的缩略图。
- **内容加载限制** — 开启 / 关闭分页。
- **文件屏幕样式** — 纯菜单或分组菜单。
- **媒体资料库屏幕样式** — 纯 / 分组 / 标签菜单。
- **播放器屏幕样式** — 简约 / 底部 / 古典 / 经典。
- **上下文菜单样式** — 系统菜单或底部表格样式。

## 窗口

在 Mac 和 Catalyst 上可用。配置与窗口相关的偏好，例如默认大小和启动时的行为。

## 屏幕

选择在使用应用时屏幕是否应保持活跃。

## 辅助功能

激活**文字模式**以隐藏用户界面中的图像。当 VoiceOver 激活时，文字模式会自动启用。

## 语言

更改应用语言并覆盖系统默认值。更改在您完全退出并重新打开 Evervideo 后生效。支持超过 120 种语言。

## 备份与恢复

备份所有应用数据或将其迁移到另一台设备。选择要包含的内容：

- **数据库** — 媒体资料库条目、播放列表、评分、收藏夹、观看历史。离线文件不包括在内以保持文件大小可管理。
- **专辑封面** — 缓存的图稿。
- **设置** — 应用设置。

点击**备份应用数据**开始备份。要在新设备上恢复，通过 iCloud Drive、AirDrop 或任何已连接的云服务移动备份文件，并在新设备上打开它。

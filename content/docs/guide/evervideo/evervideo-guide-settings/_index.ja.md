設定画面はEvervideoのコントロールセンターです。ここからプレミアムへのアップグレード、ビデオおよびオーディオエンジンの設定（システムコーデックまたはFFmpeg）、ピクチャーインピクチャーの管理、字幕の設定（プライマリ、セカンダリ、libass、外部ファイル、フォント）、メディアライブラリの整理、ファイルマネージャーの設定、ホーム画面ウィジェットの有効化、データのバックアップ、ヘルプと法的情報へのアクセスができます。セクションはヘッダーの下にグループ化されています：購入とアップデート、アプリ設定、ヘルプ、法的事項とプライバシー。

## プレミアムへのアップグレード

すべての制限を解除するためにアプリをプレミアムバージョンにアップグレードしてください。アプリの無料バージョンは、すべての制限を解除してプレミアムにアップグレードするための1回限りの永続的なアプリ内購入と2つのサブスクリプションオプション（1か月と1年）を提供しています。

## iOSとMac間での購入の共有

永続購入とサブスクリプションは、この情報を同期するために**iCloud**を使用してiOSとMac間で共有されます。iOSデバイスにプレミアムがある場合は、最新バージョンがインストールされており、iCloudが有効になっていることを確認してください。iOSでアプリを起動し、購入情報がiCloudにアップロードされるまで1分待ってから、Macバージョンを起動してください — プレミアムが自動的にアクティブになるはずです。

アプリ設定の**購入を復元**ボタンをタップすることもできます。インターネット接続があり、両方のデバイスで同じiCloudおよびApp Storeアカウントにサインインしていることを確認してください。

## 新しいデバイスでの購入の復元

新しいデバイスで購入を復元するには、**購入 → 購入を復元**メニューを使用してください。購入のリストが表示されます。すべて表示されない場合は、デバイスが購入に使用したApple IDと同じものに接続されていることを確認し、iCloudが有効になっていることを確認してください。

## プレーヤー

再生に関するすべてがここにあります — オーディオ、ビデオ、字幕、デバイス、パーソナライゼーション、キャッシュ、スリープタイマー。

### 一般

これらの設定はプレーヤーの基本的な側面をカバーします。

- **繰り返しモード** — ビデオが終了したときのプレーヤーの動作を選択します：
  - **すべて繰り返し** — キュー内のすべてのビデオを再生します。
  - **1つ繰り返し** — 現在のビデオのみ繰り返します。
  - **繰り返し停止** — 現在のビデオが終了すると一時停止します。
  - **繰り返しなし** — 繰り返さずにキューを一度再生します。
- **シャッフルモード** — キュー内のビデオの順序をランダム化します（**オン**または**オフ**）。
- **再生位置を保存** — ライブラリ内のビデオの再生位置を保存および復元します。
- **プレーヤー状態を保存** — アプリを閉じる前のプレーヤーの状態を保存し、中断した場所から再開できます。

### ビデオ

Evervideoがビデオをデコードおよびレンダリングする方法を設定します。

- **ハードウェアデコードH.264** — ハードウェアアクセラレーションAVCデコードをオン / オフにします。バッテリーと熱性能が重要な場合に使用します。エキゾチックなプロファイルとの互換性のためにオフにします。
- **ハードウェアデコードH.265 (HEVC)** — HEVCも同様です。最新のiPhone、iPad、MacはHEVCを効率的にデコードします。
- **優先ピクセルフォーマット** — プレーヤーがレンダラーに提示するピクセルフォーマットを選択します（デフォルトはデバイスに合わせて調整されています）。
- **優先FPS** — 目標再生FPSを設定します。特定のモニターリフレッシュレートに合わせるのに便利です。
- **ビデオスケーリングモード** — フィット / フィル / ストレッチ / オリジナル。画像が表示エリアをどのように満たすかを決定します。
- **ビデオ表示モード** (iOS / iPadOS) — ビデオがプレーヤービューにどのように表示されるか。
- **ビデオ回転** — 画像を0° / 90° / 180° / 270°手動で回転します（間違った向きで録画されたビデオに便利）。
- **ビデオイコライザー** — リアルタイムプレビューで明るさ、コントラスト、彩度、色相を調整します。カスタムプリセットは**エクスポートおよびインポート**できます。
- **VRビューポート** (iOS / iPadOS) — 360°球面ビデオ用。ドラッグして周囲を見渡し、ピンチしてズームします。
- **ピクチャーインピクチャー (PiP)** (iOS / iPadOS) — PIPサポートを有効にします。アプリをバックグラウンドにするか、PIPボタンをタップすると、フローティングプレーヤーウィンドウに切り替わります。

### オーディオ

オーディオ再生と出力を設定します。

- **オーディオトラック** — ビデオに複数のオーディオストリームがある場合に選択します（コメンタリー、吹き替えなど）。
- **オーディオイコライザー** — プリセットとプリアンプ付きの10バンドEQ。カスタムプリセットはエクスポート / インポートできます。
- **オーディオ出力サンプルレート** — 44.1 kHz、48 kHz、64 kHz、88.2 kHz、96 kHz。外部DACと併用するのに便利です。
- **オーディオ出力チャンネル数** — モノ、ステレオ、5.1、ITU BS.775-1、SDDSなど。
- **オーディオ出力優先IOバッファ時間** — 低遅延ハイレゾ再生の典型的な値は約5 ms（0.005 s）です。ハードウェアに合わせて調整してください。
- **オーディオ出力モード** (iOSのみ) — ミックスモードにより、Evervideoのオーディオをほかのアプリのオーディオとブレンドできます。

### 字幕

Evervideoには包括的な字幕サポートが含まれています。

- **字幕トラック** — ファイルに埋め込まれた字幕トラックから選択します。
- **外部字幕ファイル** — デバイスまたは接続されたクラウドサービスから外部の`.srt`、`.vtt`、`.ass`、または`.ssa`ファイルを読み込みます。
- **フォント** — プライマリ字幕トラックのフォントを選択します。

### デバイス (iOS/iPadOSのみ)

ビデオ出力用の**AirPlay**または**Google Chromecast**デバイスを選択します。

### パーソナライゼーション

- **プレーヤーレイアウトスタイル** — ミニマル（Evervideoのデフォルト）、ボトム、アンティーク、クラシックなど。
- **メイン画面アクション** — メインプレーヤー画面に表示するボタンを選択します。
- **ロック画面コントロール** — スキップ時間、ブックマークを追加、お気に入りに追加。
- **スキップ時間間隔** — スキップボタンの時間間隔を選択します（5秒、10秒、15秒、30秒など）。
- **アルバムカバースクロールスタイル** — カバーアートの優先スクロールスタイル。
- **追加要素** — オーディオフォーマット情報、音量スライダー。

### ファイル読み込み

ネットワークからビデオデータをストリーミングする方法を設定します。

- **ネットワークタイプ** — Wi-Fiのみ、またはWi-Fi + モバイルデータ。
- **プリロード時間** — 低速ネットワークでのスムーズな再生のためのバッファ長。
- **直接URLを使用** — サポートされている場合、より速い読み込みのために直接URLを使用します。

### 再生キャッシュ

スムーズな再生を確保するために、プレーヤーキュー内のビデオが自動的にダウンロードされます。このオプションを無効にするか、ここでキャッシュサイズを設定できます。

### スリープタイマー

指定した時間後に再生を自動的に停止するタイマーを起動します。分単位の粒度の**精密モード**の設定アイコンをタップします。

## メディアライブラリ

自動同期、メタデータ、アルバムカバー、プレイリスト、最近再生したもの、お気に入りを管理します。

### メタデータの読み込み

ライブラリにビデオを追加すると、メタデータリーダーがバックグラウンドでスキャンし、アルバムとジャンルごとに整理します。スキャン速度を調整したり、リーダーを無効にしたり、**メタデータをリロード**で完全な再スキャンをトリガーしたりできます。**メタデータエンコーディングを正規化**すると、壊れた文字エンコーディングを自動的に修正します（WindowsのPCからのファイルでよく見られます）。

### オンライン同期

接続されたクラウドサービスとメディアサーバーからビデオをライブラリに自動的に追加します。スキャンするフォルダーを選択し、アプリの同期頻度を設定し、手動で同期を開始 / 停止します。オンライン同期はアプリがアクティブの間のみ実行されます — 大きなライブラリには、デスクトップバージョンを使用し、次に**バックアップと復元**で同期されたライブラリを転送してください。

### オフライン同期

クラウドフォルダーをオフラインで利用可能としてマークすると、**ファイル → オフラインフォルダー**に表示され、自動的にダウンロードされます。オンラインで追加された新しいファイルもダウンロードされます。この画面から時間間隔を設定し、手動同期をトリガーできます。

### パーソナライゼーション

- **メディアライブラリ画面スタイル** — プレーンメニュー、グループ化メニュー、タブ付きメニュー。
- 詳細画面で大きなアルバムカバーを表示するかどうかを切り替えます。

### アルバムカバー

- **ネットワークタイプ** — Wi-FiまたはWi-Fi + モバイルデータ。
- **オンラインファイルのアルバムカバーを読み込む** — クラウドファイルから埋め込みアートワークを取得します（追加データを使用）。
- **フォルダー内を検索** — ビデオに埋め込みカバーがない場合、同じフォルダー内のJPEG / PNG画像を使用します。
- **カバー品質** — カバーがキャッシュされる解像度を調整します。
- **フォルダーに表示** / **すべて削除** — アートワークキャッシュを管理します。

### プレイリスト

同じビデオをプレイリストに2回追加することを有効にします（デフォルトではオフ）。

### 最近再生したもの

最近再生したビデオのリストを管理します — サイズを変更したり、削除したり、M3U / CSV / TXTとしてエクスポートしたりできます。

### お気に入り

- **同時編集** — メディアライブラリとファイルセクション間でお気に入りをミラーリングします。
- **リストを削除** — お気に入りリストをクリアします。
- **曲リストをエクスポート** — お気に入りをM3U / CSV / TXTとしてエクスポートします。

### メディアライブラリを削除

メディアライブラリデータベースを消去します。ストレージ上のビデオおよびオーディオファイルは影響を受けません。

## パスコード

**4桁の数値パスコード**でEvervideoを保護します。有効にすると、アプリを起動するたびにパスコードの入力を求められます。追加の保護のためにデバイスのiOS Face ID / Touch IDと組み合わせてください。

## ファイルマネージャー

ファイルの転送、保存、プレビューの方法を制御します。

- **ファイル転送** — ネットワーク設定（Wi-Fiのみ、またはWi-Fi + モバイルデータ）。
- **並列タスクの最大数** — 並列ダウンロードスレッドの数を設定します。
- **ファイル転送タスク** — 現在アクティブなアップロードとダウンロードを表示します。
- **バックグラウンド転送** — アプリがバックグラウンドにある間のダウンロードを許可します。
- **ダウンロードしたファイルの保存先** — デフォルトのダウンロードディレクトリ。
- **同期済みオフラインフォルダー** — オフラインモードフォルダーを管理します。
- **時間間隔** — オフラインフォルダーが変更を確認する頻度。
- **完全なファイル名を表示** — ファイルマネージャーに拡張子を表示します。
- **オンラインファイルを編集** — 接続されたクラウドサービスの読み取り専用モードに切り替えるには無効にします。
- **開くときにファイルをコピー** — 他のアプリからインポートされたファイルの処理方法。
- **ファイルのサムネイル** — 生成されたファイルサムネイルを管理します。
- **一時ファイルを削除** — アプリのキャッシュフォルダーをクリアします。

## Wi-Fiドライブ

Wi-Fiドライブを有効にして、デスクトップウェブブラウザー、Finder、またはエクスプローラーを使用してコンピューターからデバイスにファイルを転送します。詳細な手順は[こちら](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)で利用可能です。

## ウィジェット

再生中にアプリがウィジェットデータを更新できるようにウィジェットを有効にします。ウィジェットの更新には追加のエネルギーが必要なため、トグルはデフォルトでオフになっています — ホーム画面またはロック画面でウィジェットを実際に使用する場合にのみ有効にしてください。

Evervideoウィジェットを追加するには、ホーム画面またはロック画面を長押しし、**+**をタップし、**Evervideo**を検索して、ウィジェットのサイズを選択します。ウィジェットには現在再生中のビデオが表示され、フルスクリーンプレーヤーに直接移動できます。ウィジェットはmacOSの通知センターでも機能します。

## パーソナライゼーション

ユーザーインターフェースを好みに合わせてカスタマイズします。

- **アプリケーションアイコン** — 代替ホーム画面アイコン（プレミアム）。
- **カラースキーム** — ダーク、ライト、またはデフォルト（システムの外観に従います）。
- **背景スタイル** — ぼかしアルバムカバーまたは単色。
- **リスト内のアイテムの外観** — 行の高さを自動調整し、小さいサムネイルを表示します。
- **コンテンツ読み込み制限** — ページネーションをオン / オフにします。
- **ファイル画面スタイル** — プレーンメニューまたはグループ化メニュー。
- **メディアライブラリ画面スタイル** — プレーン / グループ化 / タブ付きメニュー。
- **プレーヤー画面スタイル** — ミニマル / ボトム / アンティーク / クラシック。
- **コンテキストメニュースタイル** — システムメニューまたはボトムシートスタイル。

## ウィンドウ

MacおよびCatalystで利用可能。デフォルトサイズや起動時の動作などのウィンドウ関連の設定を行います。

## 画面

## アクセシビリティ

ユーザーインターフェースの画像を非表示にするために**テキストモード**を有効にします。テキストモードはVoiceOverがアクティブな場合に自動的に有効になります。

## 言語

アプリケーションの言語を変更し、システムのデフォルトを上書きします。変更はEvervideoを完全に終了して再起動した後に適用されます。120以上の言語がサポートされています。

## バックアップと復元

すべてのアプリケーションデータをバックアップするか、別のデバイスに移行します。含める内容を選択します：

- **データベース** — メディアライブラリエントリ、プレイリスト、評価、お気に入り、視聴履歴。オフラインファイルはファイルサイズを管理しやすくするために含まれません。
- **アルバムカバー** — キャッシュされたアートワーク。
- **設定** — アプリケーションの設定。

**アプリケーションデータをバックアップ**をタップしてバックアップを開始します。新しいデバイスで復元するには、iCloud Drive、AirDrop、または接続されたクラウドサービスを通じてバックアップファイルを移動し、新しいデバイスで開きます。

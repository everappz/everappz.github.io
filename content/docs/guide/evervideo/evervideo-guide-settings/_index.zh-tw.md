## 在 iOS 和 Mac 之間共享購買

終身購買和訂閱透過使用 **iCloud** 同步資訊在 iOS 和 Mac 之間共享。如果您在 iOS 裝置上擁有 Premium，請確保已安裝最新版本並啟用了 iCloud。在 iOS 上啟動應用程式並等待一分鐘讓購買資訊上傳到 iCloud，然後啟動 Mac 版本——Premium 應該會自動啟動。

您也可以點選應用程式設定中的**還原購買項目**按鈕。確保您有網路連線，並在兩台裝置上登入了相同的 iCloud 和 App Store 帳號。

## 在新裝置上還原購買項目

要在新裝置上還原購買項目，請使用**購買項目 → 還原購買項目**選單。您將看到購買列表。如果看不到所有購買項目，請確認裝置已連線到用於購買的同一 Apple ID，並確保 iCloud 已啟用。

## 播放器

與播放相關的所有內容都在這裡——音訊、視訊、字幕、裝置、個人化、快取和睡眠定時器。

### 一般

這些設定涵蓋播放器的基本方面。

- **重複模式** — 選擇視訊結束時播放器的行為：
  - **重複全部** — 重播佇列中的每個視訊。
  - **重複一個** — 僅重複目前視訊。
  - **重複停止** — 目前視訊結束時暫停。
  - **不重複** — 播放完佇列後不重複。
- **隨機模式** — 隨機排列佇列中視訊的順序（**開啟**或**關閉**）。
- **儲存播放位置** — 儲存並還原資料庫中視訊的播放位置。
- **儲存播放器狀態** — 在您關閉應用程式之前保留播放器狀態，以便您可以從上次中斷的地方繼續。

### 視訊

設定 Evervideo 解碼和渲染視訊的方式。

- **硬體解碼 H.264** — 開啟 / 關閉硬體加速的 AVC 解碼。在電池和溫度效能重要時使用；關閉以相容特殊設定檔。
- **硬體解碼 H.265 (HEVC)** — 與 HEVC 相同。現代 iPhone、iPad 和 Mac 可高效解碼 HEVC。
- **首選像素格式** — 選擇播放器向渲染器呈現的像素格式（預設值已針對裝置進行最佳化）。
- **首選 FPS** — 設定目標播放 FPS。適用於匹配特定顯示器更新率。
- **視訊縮放模式** — 符合 / 填滿 / 拉伸 / 原始。決定畫面如何填滿顯示區域。
- **視訊顯示模式** (iOS / iPadOS) — 視訊在播放器檢視中的呈現方式。
- **視訊旋轉** — 手動將畫面旋轉 0° / 90° / 180° / 270°（適用於方向錯誤錄製的視訊）。
- **視訊均衡器** — 透過即時預覽調整亮度、對比度、飽和度和色調。自訂預設可以**匯出和匯入**。
- **VR 視埠** (iOS / iPadOS) — 適用於 360° 球形視訊。拖移以環顧四周，捏合以縮放。
- **子母畫面 (PiP)** (iOS / iPadOS) — 啟用 PiP 支援；當您將應用程式置於背景或點選 PiP 按鈕時，應用程式將切換到浮動播放器視窗。

### 音訊

設定音訊播放和輸出。

- **音軌** — 當視訊有多個音軌時選擇音訊串流（解說、配音等）。
- **音訊均衡器** — 帶預設和前置放大器的 10 頻段均衡器。自訂預設可以匯出 / 匯入。
- **音訊輸出取樣率** — 44.1 kHz、48 kHz、64 kHz、88.2 kHz、96 kHz。適用於外接 DAC。
- **音訊輸出聲道數** — 單聲道、立體聲、5.1、ITU BS.775-1、SDDS 等。
- **音訊輸出首選 IO 緩衝區時長** — 低延遲高解析度播放的典型值約為 5 毫秒（0.005 秒）。根據您的硬體調整。
- **音訊輸出模式**（僅 iOS）— 混合模式讓 Evervideo 的音訊與其他應用程式混合。

### 字幕

Evervideo 包含全面的字幕支援。

- **字幕軌道** — 從檔案中的嵌入字幕軌道中選擇。
- **外部字幕檔案** — 從裝置或任何已連線的雲端服務載入外部 `.srt`、`.vtt`、`.ass` 或 `.ssa` 檔案。
- **字型** — 為主字幕軌道選擇字型。

### 裝置（僅 iOS/iPadOS）

選擇 **AirPlay** 或 **Google Chromecast** 裝置進行視訊輸出。

### 個人化

- **播放器版面樣式** — 簡約（Evervideo 預設）、底部、古典、經典等。
- **主畫面操作** — 選擇主播放器畫面上顯示的按鈕。
- **鎖定畫面控制項** — 跳過時間、新增書籤、新增到最愛項目。
- **跳過時間間隔** — 選擇跳過按鈕的時間間隔（5 秒、10 秒、15 秒、30 秒等）。
- **專輯封面捲動樣式** — 封面圖片的首選捲動樣式。
- **附加元素** — 音訊格式資訊、音量滑桿。

### 檔案載入

設定視訊資料如何從網路串流傳輸。

- **網路類型** — 僅 Wi-Fi，或 Wi-Fi + 行動數據。
- **預載時間** — 緩衝區長度，用於在慢速網路上實現更流暢的播放。
- **使用直接 URL** — 在支援的情況下，使用直接 URL 更快載入。

### 播放快取

播放器佇列中的視訊會自動下載以確保流暢播放。您可以在此停用此選項或設定快取大小。

### 睡眠定時器

啟動定時器以在指定時間後自動停止播放。點選設定圖示可使用**精確模式**，具有逐分鐘精度。

## 媒體資料庫

管理自動同步、中繼資料、專輯封面、播放清單、最近使用的和最愛項目。

### 中繼資料讀取

當您將視訊新增到資料庫時，中繼資料讀取器會在背景掃描它們，並按專輯和流派整理。您可以調整掃描速度、停用讀取器或使用**重新載入中繼資料**觸發完整重新掃描。**規範化中繼資料編碼**自動修復損壞的字元編碼（常見於來自 Windows 電腦的檔案）。

### 線上同步

自動將已連線雲端服務和媒體伺服器中的視訊新增到您的資料庫。選擇要掃描的資料夾，設定應用程式同步頻率，並手動開始 / 停止同步。線上同步僅在應用程式活躍時執行——對於大型資料庫，請使用桌面版本，然後使用**備份與還原**傳輸同步的資料庫。

### 離線同步

當您將雲端資料夾標記為可離線使用時，它會出現在**檔案 → 離線資料夾**中並自動下載。在線上新增的檔案也會下載。從此畫面設定時間間隔並觸發手動同步。

### 個人化

- **媒體資料庫畫面樣式** — 純選單、分組選單、分頁選單。
- 切換是否在詳情畫面中顯示大專輯封面。

### 專輯封面

- **網路類型** — Wi-Fi 或 Wi-Fi + 行動數據。
- **為線上檔案載入專輯封面** — 從雲端檔案中擷取嵌入的圖稿（使用額外資料）。
- **在資料夾中搜尋** — 當視訊沒有嵌入封面時，使用同一資料夾中的 JPEG / PNG 圖片。
- **封面品質** — 調整封面快取的解析度。
- **在資料夾中顯示** / **全部刪除** — 管理圖稿快取。

### 播放清單

啟用將同一視訊新增到播放清單兩次（預設關閉）。

### 最近使用的

管理最近播放的視訊清單——變更大小、刪除或匯出為 M3U / CSV / TXT。

### 最愛項目

- **同步編輯** — 在媒體資料庫和檔案部分之間鏡像最愛項目。
- **刪除清單** — 清除最愛項目清單。
- **匯出歌曲清單** — 將最愛項目匯出為 M3U / CSV / TXT。

### 刪除媒體資料庫

清除媒體資料庫。您儲存空間上的視訊和音訊檔案保持不變。

## 密碼鎖

使用 **4 位數字密碼**保護 Evervideo。啟用後，每次應用程式啟動時都會提示輸入密碼。結合裝置上的 iOS Face ID / Touch ID 以獲得額外保護。

## 檔案管理員

控制檔案的傳輸、儲存和預覽方式。

- **檔案傳輸** — 網路偏好設定（僅 Wi-Fi 或 Wi-Fi + 行動數據）。
- **最大並行任務數** — 設定並行下載執行緒數。
- **檔案傳輸任務** — 查看目前活躍的上傳和下載。
- **背景傳輸** — 允許在應用程式處於背景時下載。
- **將下載檔案儲存到** — 預設下載目錄。
- **同步離線資料夾** — 管理離線模式資料夾。
- **時間間隔** — 檢查離線資料夾變更的頻率。
- **顯示完整檔案名稱** — 在檔案管理員中顯示副檔名。
- **編輯線上檔案** — 停用以切換到已連線雲端服務的唯讀模式。
- **開啟時複製檔案** — 如何處理從其他應用程式匯入的檔案。
- **檔案縮圖** — 管理產生的檔案縮圖。
- **刪除暫存檔案** — 清除應用程式快取資料夾。

## Wi-Fi Drive

啟動 Wi-Fi Drive，使用桌面網路瀏覽器、Finder 或檔案總管從電腦向裝置傳輸檔案。詳細說明[在此處](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)提供。

## 小工具

啟用小工具使應用程式在播放期間更新小工具資料。小工具更新需要額外電量，因此預設關閉——僅在您實際使用主畫面或鎖定畫面上的小工具時啟用。

您可以透過長按主畫面或鎖定畫面，點選 **+**，搜尋 **Evervideo** 並選擇小工具大小來新增 Evervideo 小工具。小工具顯示正在播放的視訊並直接點選進入全螢幕播放器。小工具也適用於 macOS 通知中心。

## 個人化

根據您的偏好自訂使用者介面。

- **應用程式圖示** — 備用主畫面圖示（Premium）。
- **配色方案** — 深色、淺色或預設（跟隨系統外觀）。
- **背景樣式** — 模糊專輯封面或純色。
- **清單中項目的外觀** — 自動調整列高；顯示較小的縮圖。
- **內容載入限制** — 開啟 / 關閉分頁。
- **檔案畫面樣式** — 純選單或分組選單。
- **媒體資料庫畫面樣式** — 純 / 分組 / 分頁選單。
- **播放器畫面樣式** — 簡約 / 底部 / 古典 / 經典。
- **情境選單樣式** — 系統選單或底部工作表樣式。

## 視窗

## 輔助使用

## 語言

變更應用程式語言並覆蓋系統預設值。變更在您完全退出並重新開啟 Evervideo 後生效。支援超過 120 種語言。

## 備份與還原

備份所有應用程式資料或將其遷移到另一台裝置。選擇要包含的內容：

- **資料庫** — 媒體資料庫條目、播放清單、評分、最愛項目、觀看紀錄。離線檔案不包含在內以保持檔案大小可管理。
- **專輯封面** — 快取的圖稿。
- **設定** — 應用程式設定。

點選**備份應用程式資料**開始備份。要在新裝置上還原，透過 iCloud Drive、AirDrop 或任何已連線的雲端服務移動備份檔案，並在新裝置上開啟它。

Экран «Настройки» — центр управления Evervideo. Здесь можно перейти на Premium, настроить движки воспроизведения видео и аудио (системные кодеки или FFmpeg), управлять «Картинкой в картинке», настроить субтитры (основные, дополнительные, libass, внешние файлы, шрифты), организовать медиатеку, настроить менеджер файлов, включить виджеты для экрана «Домой», создать резервную копию данных и получить доступ к справке и юридической информации. ## Общий доступ к покупкам между iOS и Mac

Пожизненные покупки и подписки распространяются между iOS и Mac с помощью **iCloud** для синхронизации этих данных. Если у вас Premium на устройстве iOS, убедитесь, что установлена последняя версия и iCloud включён. Запустите приложение на iOS и подождите одну минуту для загрузки информации о покупке в iCloud, затем запустите версию для Mac — Premium должен активироваться автоматически.

Также можно нажать кнопку **Восстановить покупки** в настройках приложения. Убедитесь, что есть интернет-соединение, и на обоих устройствах выполнен вход в один и тот же аккаунт iCloud и App Store.

## Восстановление покупок на новом устройстве

Для восстановления покупки на новом устройстве воспользуйтесь меню **Покупки → Восстановить покупки**. Отобразится список покупок. Если не все из них видны, убедитесь, что устройство подключено к тому же Apple ID, с которого осуществлялись покупки, и iCloud включён.

## Плеер

Здесь находится всё, что связано с воспроизведением, — аудио, видео, субтитры, устройства, персонализация, кэш и таймер сна.

### Основные

Эти настройки охватывают фундаментальные аспекты плеера.

- **Режим повтора** — выберите поведение плеера по завершении видео:
  - **Повторять все** — воспроизводит заново каждое видео в очереди.
  - **Повторять одно** — повторяет только текущее видео.
  - **Стоп по завершении** — делает паузу, когда текущее видео заканчивается.
  - **Без повтора** — воспроизводит очередь один раз без повтора.
- **Режим случайного порядка** — перемешивает порядок видео в очереди (**Вкл.** или **Выкл.**).
- **Сохранять позицию воспроизведения** — сохраняет и восстанавливает позицию воспроизведения для видео в библиотеке.
- **Сохранять состояние плеера** — сохраняет состояние плеера перед закрытием приложения, чтобы можно было продолжить с того места, где остановились.

### Видео

Настройте декодирование и отрисовку видео в Evervideo.

- **Аппаратное декодирование H.264** — включить / выключить аппаратное ускорение декодирования AVC. Используйте, когда важны энергопотребление и температурный режим; отключите при проблемах совместимости с нестандартными профилями.
- **Аппаратное декодирование H.265 (HEVC)** — то же для HEVC. Современные iPhone, iPad и Mac эффективно декодируют HEVC.
- **Предпочтительный формат пикселей** — выберите формат пикселей, который плеер передаёт рендереру (значения по умолчанию настроены под устройство).
- **Предпочтительная частота кадров** — задайте целевую частоту кадров воспроизведения. Полезно для совпадения с частотой обновления монитора.
- **Режим масштабирования видео** — По размеру / Заполнить / Растянуть / Оригинал. Определяет, как изображение заполняет область отображения.
- **Режим отображения видео** (iOS / iPadOS) — способ отображения видео в представлении плеера.
- **Поворот видео** — вручную поверните изображение на 0° / 90° / 180° / 270° (полезно для видео, снятых в неправильной ориентации).
- **Видеоэквалайзер** — регулируйте яркость, контрастность, насыщенность и оттенок с предварительным просмотром в реальном времени. Пользовательские пресеты можно **экспортировать и импортировать**.
- **VR-вьюпорт** (iOS / iPadOS) — для сферических видео 360°. Перетаскивайте для обзора, сводите/разводите для масштабирования.
- **Картинка в картинке (PiP)** (iOS / iPadOS) — включить поддержку PiP; приложение переключится на плавающее окно плеера при сворачивании приложения или нажатии кнопки PiP.

### Аудио

Настройте воспроизведение и вывод аудио.

- **Аудиодорожка** — выберите аудиопоток, если в видео их несколько (комментарий, дубляж и т.д.).
- **Аудиоэквалайзер** — 10-полосный эквалайзер с пресетами и предусилителем. Пользовательские пресеты можно экспортировать / импортировать.
- **Частота дискретизации аудиовыхода** — 44,1 кГц, 48 кГц, 64 кГц, 88,2 кГц, 96 кГц. Полезно при использовании внешних ЦАП.
- **Число каналов аудиовыхода** — Моно, Стерео, 5.1, ITU BS.775-1, SDDS и другие.
- **Предпочтительная длительность буфера IO аудиовыхода** — типичное значение для малой задержки при воспроизведении hi-res — около 5 мс (0,005 с). Настройте под своё оборудование.
- **Режим аудиовыхода** (только iOS) — режим смешивания позволяет аудио Evervideo воспроизводиться вместе с другими приложениями.

### Субтитры

Evervideo включает полноценную поддержку субтитров.

- **Дорожка субтитров** — выберите из встроенных дорожек субтитров в файле.
- **Внешний файл субтитров** — загрузите внешний файл `.srt`, `.vtt`, `.ass` или `.ssa` с устройства или любого подключённого облачного сервиса.
- **Шрифт** — выберите шрифт для основной дорожки субтитров.

### Устройства (только iOS/iPadOS)

Выберите устройство **AirPlay** или **Google Chromecast** для вывода видео.

### Персонализация

- **Стиль раскладки плеера** — Минимальный (по умолчанию для Evervideo), Нижний, Антикварный, Классический и другие.
- **Действия главного экрана** — выберите, какие кнопки отображаются на главном экране плеера.
- **Элементы управления экраном блокировки** — Пропустить время, Добавить закладку, Добавить в избранное.
- **Интервалы пропуска** — выберите временной интервал для кнопок перемотки (5 с, 10 с, 15 с, 30 с и т.д.).
- **Стиль прокрутки обложек альбомов** — предпочтительный стиль прокрутки для обложек.
- **Дополнительные элементы** — Информация об аудиоформате, Ползунок громкости.

### Загрузка файла

Настройте, как данные видео передаются из сети.

- **Тип сети** — только Wi-Fi или Wi-Fi + сотовая связь.
- **Время предзагрузки** — длина буфера для более плавного воспроизведения на медленных сетях.
- **Использовать прямой URL** — при наличии поддержки использовать прямой URL для более быстрой загрузки.

### Кэш воспроизведения

Видео в очереди плеера автоматически загружаются для обеспечения плавного воспроизведения. Здесь можно отключить эту опцию или настроить размер кэша.

### Таймер сна

Активируйте таймер для автоматической остановки воспроизведения через заданный период. Нажмите на значок настройки для **точного режима** с поминутной точностью.

## Медиатека

Управление автоматической синхронизацией, метаданными, обложками альбомов, плейлистами, последними записями и избранным.

### Чтение метаданных

При добавлении видео в библиотеку считыватель метаданных сканирует их в фоновом режиме и организует по Альбому и Жанру. Можно настроить скорость сканирования, отключить считыватель или запустить полное пересканирование с помощью **Перезагрузить метаданные**. **Нормализация кодировки метаданных** автоматически исправляет неверные кодировки символов (характерно для файлов с Windows-компьютеров).

### Онлайн-синхронизация

Автоматически добавляйте видео из подключённых облачных сервисов и медиасерверов в библиотеку. Выберите папки для сканирования, настройте периодичность синхронизации и запускайте / останавливайте синхронизацию вручную. Онлайн-синхронизация работает только пока приложение активно — для больших библиотек используйте версию для ПК, а затем перенесите синхронизированную библиотеку с помощью **Резервное копирование и восстановление**.

### Офлайн-синхронизация

Когда вы помечаете папку в облаке как доступную офлайн, она появляется в **Файлы → Офлайн-папки** и загружается автоматически. Новые файлы, добавленные онлайн, также загружаются. Настройте временной интервал и запускайте ручную синхронизацию с этого экрана.

### Персонализация

- **Стиль экрана медиатеки** — Обычное меню, Сгруппированное меню, Меню с вкладками.
- Переключение отображения больших обложек альбомов на экранах деталей.

### Обложки альбомов

- **Тип сети** — Wi-Fi или Wi-Fi + сотовая связь.
- **Загружать обложки для онлайн-файлов** — получать встроенные обложки из файлов в облаке (расходует дополнительные данные).
- **Искать в папке** — использовать изображения JPEG / PNG из той же папки, если у видео нет встроенной обложки.
- **Качество обложки** — настройте разрешение кэширования обложек.
- **Показать в папке** / **Удалить все** — управление кэшем обложек.

### Плейлисты

Включить добавление одного и того же видео в плейлист дважды (по умолчанию отключено).

### Последние

Управление списком последних воспроизведённых видео — изменение размера, удаление или экспорт в M3U / CSV / TXT.

### Избранное

- **Одновременное редактирование** — синхронизировать избранное между медиатекой и разделом файлов.
- **Удалить список** — очистить список избранного.
- **Экспортировать список** — экспортировать избранное в M3U / CSV / TXT.

### Удалить медиатеку

Удалить базу данных медиатеки. Видео- и аудиофайлы в хранилище остаются нетронутыми.

## Код доступа

Защитите Evervideo **4-значным цифровым кодом**. После включения при каждом запуске приложения будет запрашиваться код доступа. Для дополнительной защиты используйте совместно с iOS Face ID / Touch ID на устройстве.

## Менеджер файлов

Управляет передачей, хранением и предпросмотром файлов.

- **Передача файлов** — сетевые предпочтения (только Wi-Fi или Wi-Fi + сотовая связь).
- **Максимальное число параллельных задач** — установите количество параллельных потоков загрузки.
- **Задачи передачи файлов** — просмотр активных загрузок и выгрузок.
- **Фоновые передачи** — разрешить загрузку, пока приложение в фоне.
- **Сохранять загруженные файлы в** — каталог загрузок по умолчанию.
- **Синхронизированные офлайн-папки** — управление папками в офлайн-режиме.
- **Временной интервал** — как часто проверяются офлайн-папки на предмет изменений.
- **Показывать полные имена файлов** — отображать расширения в менеджере файлов.
- **Редактировать онлайн-файлы** — отключите для переключения в режим только для чтения для подключённых облачных сервисов.
- **Копировать файлы при открытии** — как обрабатывать импортированные файлы из других приложений.
- **Миниатюры для файлов** — управление созданными миниатюрами файлов.
- **Удалить временные файлы** — очистить папку кэша приложения.

## Wi-Fi Drive

Активируйте Wi-Fi Drive для передачи файлов с компьютера на устройство через браузер, Finder или Проводник. Подробные инструкции доступны [здесь](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).

## Виджеты

Включите виджеты, чтобы приложение обновляло их данные во время воспроизведения. Обновление виджетов требует дополнительной энергии, поэтому переключатель по умолчанию выключен — включайте только если действительно используете виджеты на экране «Домой» или экране блокировки.

Виджеты Evervideo можно добавить, удерживая экран «Домой» или экран блокировки, нажав **+**, выбрав **Evervideo** и выбрав размер виджета. Виджеты отображают текущее воспроизводимое видео и ведут прямо к полноэкранному плееру. Виджеты также работают в Центре уведомлений macOS.

## Персонализация

Настройте пользовательский интерфейс под свои предпочтения.

- **Иконка приложения** — альтернативная иконка для экрана «Домой» (Premium).
- **Цветовая схема** — Тёмная, Светлая или По умолчанию (следует оформлению системы).
- **Стиль фона** — Размытая обложка альбома или однотонный цвет.
- **Вид элементов в списке** — автонастройка высоты строки; показ уменьшенных миниатюр.
- **Лимит загрузки контента** — включить / выключить пагинацию.
- **Стиль экрана «Файлы»** — Обычное меню или Сгруппированное меню.
- **Стиль экрана медиатеки** — Обычное / Сгруппированное / Меню с вкладками.
- **Стиль экрана плеера** — Минимальный / Нижний / Антикварный / Классический.
- **Стиль контекстного меню** — системное меню или стиль нижнего листа.

## Окно

Доступно на Mac и Catalyst. Настройте параметры окна, такие как размер по умолчанию и поведение при запуске.

## Экран

Выберите, должен ли экран оставаться активным во время использования приложения.

## Доступность

Активируйте **Текстовый режим** для скрытия изображений в пользовательском интерфейсе. Текстовый режим включается автоматически при активации VoiceOver.

## Язык

Изменить язык приложения, переопределив системный. Изменение применяется после полного закрытия и повторного открытия Evervideo. Поддерживается более 120 языков.

## Резервное копирование и восстановление

Создайте резервную копию всех данных приложения или перенесите их на другое устройство. Выберите, что включить:

- **База данных** — записи медиатеки, плейлисты, рейтинги, избранное, история просмотров. Офлайн-файлы не включаются для оптимального размера файла.
- **Обложки альбомов** — кэшированные обложки.
- **Настройки** — настройки приложения.

Нажмите **Создать резервную копию данных приложения**, чтобы начать резервное копирование. Для восстановления на новом устройстве перенесите файл резервной копии через iCloud Drive, AirDrop или любой подключённый облачный сервис и откройте его на новом устройстве.

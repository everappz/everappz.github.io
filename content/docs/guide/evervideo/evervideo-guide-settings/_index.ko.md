설정 화면은 Evervideo의 제어 센터입니다. 여기서 Premium으로 업그레이드하고, 비디오 및 오디오 엔진 (시스템 코덱 또는 FFmpeg)을 구성하고, 화면 속 화면을 관리하고, 자막 (기본, 보조, libass, 외부 파일, 폰트)을 설정하고, 미디어 라이브러리를 구성하고, 파일 관리자를 설정하고, 홈 화면 위젯을 활성화하고, 데이터를 백업하고, 도움말 및 법률 정보에 접근할 수 있습니다. 섹션은 구매 및 업데이트, 앱 기본 설정, 도움말, 법률 및 개인정보 보호 헤더로 그룹화됩니다.

## Premium으로 업그레이드

모든 제한을 제거하기 위해 애플리케이션을 Premium 버전으로 업그레이드하세요. 애플리케이션의 무료 버전은 모든 제한을 제거하고 Premium으로 업그레이드하기 위한 일회성 평생 인앱 구매와 두 가지 구독 옵션 (1개월 및 1년)을 제공합니다.

## iOS와 Mac 간 구매 공유

평생 구매 및 구독은 **iCloud**를 사용하여 iOS와 Mac 간에 공유됩니다. iOS 기기에 Premium이 있으면 최신 버전이 설치되어 있고 iCloud가 활성화되어 있는지 확인하세요. iOS에서 앱을 시작하고 1분 정도 기다리면 구매 정보가 iCloud에 업로드되고, Mac 버전을 실행하면 Premium이 자동으로 활성화됩니다.

앱 설정에서 **구매 복원** 버튼을 탭할 수도 있습니다. 인터넷 연결이 있고 두 기기 모두에서 동일한 iCloud 및 App Store 계정에 로그인되어 있는지 확인하세요.

## 새 기기에서 구매 복원

새 기기에서 구매를 복원하려면 **구매 → 구매 복원** 메뉴를 사용하세요. 구매 목록이 표시됩니다. 모두 표시되지 않으면 기기가 구매에 사용된 것과 동일한 Apple ID에 연결되어 있는지 확인하고 iCloud가 활성화되어 있는지 확인하세요.

## 무료로 Premium 체험

이 메뉴를 사용하여 제한된 기간 동안 무료로 Premium 버전으로 업그레이드할 수 있습니다. 광고를 보거나 친구들에게 앱을 알리기만 하면 됩니다.

## 소프트웨어 업데이트

**소프트웨어 업데이트**를 탭하여 App Store에서 Evervideo의 최신 버전을 사용할 수 있는지 확인합니다.

## 새로운 기능

이 메뉴는 새 버전이 출시된 후 사용 가능합니다. 가장 최근 업데이트의 변경 사항 및 새 기능 요약을 표시합니다.

## 플레이어

재생과 관련된 모든 것이 여기에 있습니다 — 오디오, 비디오, 자막, 장치, 개인화, 캐시 및 슬립 타이머.

### 일반

이 설정은 플레이어의 기본 측면을 다룹니다.

- **반복 모드** — 비디오가 끝났을 때 플레이어가 어떻게 동작하는지 선택합니다:
  - **모두 반복** — 대기열의 모든 비디오를 다시 재생합니다.
  - **하나 반복** — 현재 비디오만 반복합니다.
  - **반복 정지** — 현재 비디오가 끝나면 일시 정지합니다.
  - **반복 없음** — 반복 없이 대기열을 한 번 재생합니다.
- **셔플 모드** — 대기열의 비디오 순서를 무작위로 설정합니다 (**켜기** 또는 **끄기**).
- **재생 위치 저장** — 라이브러리의 비디오에 대한 재생 위치를 저장하고 복원합니다.
- **플레이어 상태 저장** — 앱을 닫기 전에 플레이어 상태를 보존하여 종료한 곳에서 재개할 수 있습니다.

### 비디오

Evervideo가 비디오를 디코딩하고 렌더링하는 방법을 구성합니다.

- **하드웨어 H.264 디코딩** — 하드웨어 가속 AVC 디코딩을 켜거나 끕니다. 배터리와 열 성능이 중요할 때 사용합니다; 희귀한 프로파일과의 호환성을 위해 끕니다.
- **하드웨어 H.265 (HEVC) 디코딩** — HEVC와 동일합니다. 최신 iPhone, iPad, Mac은 HEVC를 효율적으로 디코딩합니다.
- **선호 픽셀 형식** — 플레이어가 렌더러에 제공하는 픽셀 형식을 선택합니다 (기본값은 기기에 맞게 조정됩니다).
- **선호 FPS** — 목표 재생 FPS를 설정합니다. 특정 모니터 새로 고침 속도와 일치시키는 데 유용합니다.
- **비디오 스케일링 모드** — 맞춤 / 채우기 / 늘리기 / 원본. 화면이 디스플레이 영역을 채우는 방법을 결정합니다.
- **비디오 디스플레이 모드** (iOS / iPadOS) — 비디오가 플레이어 보기에서 표시되는 방법.
- **비디오 회전** — 화면을 수동으로 0° / 90° / 180° / 270° 회전합니다 (잘못된 방향으로 녹화된 비디오에 유용합니다).
- **비디오 이퀄라이저** — 재생 중 실시간 미리 보기로 밝기, 대비, 채도, 색조를 조정합니다. 맞춤 프리셋을 **내보내고 가져올** 수 있습니다.
- **VR 뷰포트** (iOS / iPadOS) — 360° 구면 비디오용. 드래그하여 주변을 보고, 핀치하여 줌합니다.
- **화면 속 화면 (PiP)** (iOS / iPadOS) — PiP 지원을 활성화합니다. 앱을 백그라운드로 하거나 PiP 버튼을 탭하면 앱이 부동 플레이어 창으로 전환됩니다.

### 오디오

오디오 재생 및 출력을 구성합니다.

- **오디오 트랙** — 비디오에 여러 개가 있을 때 오디오 스트림을 선택합니다 (코멘터리, 더빙 등).
- **오디오 이퀄라이저** — 프리앰프가 있는 10밴드 EQ와 프리셋. 맞춤 프리셋을 내보내거나 가져올 수 있습니다.
- **오디오 출력 샘플 레이트** — 44.1 kHz, 48 kHz, 64 kHz, 88.2 kHz, 96 kHz. 외부 DAC에 유용합니다.
- **오디오 출력 채널 수** — 모노, 스테레오, 5.1, ITU BS.775-1, SDDS 등.
- **오디오 출력 선호 IO 버퍼 지속 시간** — 저지연 고해상도 재생을 위한 일반적인 값은 약 5 ms (0.005 s)입니다. 하드웨어에 맞게 조정합니다.
- **오디오 출력 모드** (iOS만 해당) — 혼합 모드를 사용하면 Evervideo의 오디오가 다른 앱과 혼합됩니다.

### 자막

Evervideo는 포괄적인 자막 지원을 포함합니다.

- **자막 트랙** — 파일의 임베디드 자막 트랙에서 선택합니다.
- **외부 자막 파일** — 기기 또는 연결된 클라우드 서비스에서 외부 `.srt`, `.vtt`, `.ass`, `.ssa` 파일을 불러옵니다.
- **폰트** — 기본 자막 트랙의 폰트를 선택합니다.

### 장치 (iOS/iPadOS만 해당)

비디오 출력을 위한 **AirPlay** 또는 **Google Chromecast** 장치를 선택합니다.

### 개인화

- **플레이어 레이아웃 스타일** — 최소 (Evervideo의 기본값), 하단, 앤틱, 클래식 등.
- **메인 화면 작업** — 메인 플레이어 화면에 나타나는 버튼을 선택합니다.
- **잠금 화면 컨트롤** — 건너뛰기 시간, 북마크 추가, 즐겨찾기에 추가.
- **건너뛰기 시간 간격** — 건너뛰기 버튼의 시간 간격을 선택합니다 (5초, 10초, 15초, 30초 등).
- **앨범 커버 스크롤 스타일** — 커버 아트에 선호하는 스크롤 스타일.
- **추가 요소** — 오디오 형식 정보, 볼륨 슬라이더.

### 파일 로딩

네트워크에서 비디오 데이터가 스트리밍되는 방법을 구성합니다.

- **네트워크 유형** — Wi-Fi만, 또는 Wi-Fi + 셀룰러.
- **미리 로딩 시간** — 느린 네트워크에서 더 부드러운 재생을 위한 버퍼 길이.
- **직접 URL 사용** — 지원되는 경우 더 빠른 로딩을 위해 직접 URL을 사용합니다.

### 재생 캐시

대기열의 비디오는 부드러운 재생을 보장하기 위해 자동으로 다운로드됩니다. 여기서 이 옵션을 비활성화하거나 캐시 크기를 구성할 수 있습니다.

### 슬립 타이머

지정된 기간 후 재생을 자동으로 중지하는 타이머를 활성화합니다. 분 단위 세밀도의 **정밀 모드**를 위해 구성 아이콘을 탭합니다.

## 미디어 라이브러리

자동 동기화, 메타데이터, 앨범 커버, 재생 목록, 최근 항목, 즐겨찾기를 관리합니다.

### 메타데이터 읽기

라이브러리에 비디오를 추가하면 메타데이터 리더가 백그라운드에서 스캔하여 앨범과 장르별로 구성합니다. 스캔 속도를 조정하고, 리더를 비활성화하거나, **메타데이터 새로 고침**으로 전체 재스캔을 트리거할 수 있습니다. **메타데이터 인코딩 정규화**는 손상된 문자 인코딩 (Windows PC의 파일에서 일반적)을 자동으로 수정합니다.

### 온라인 동기화

연결된 클라우드 서비스 및 미디어 서버의 비디오를 라이브러리에 자동으로 추가합니다. 스캔할 폴더를 선택하고, 앱이 얼마나 자주 동기화해야 하는지 구성하고, 동기화를 수동으로 시작 / 중지합니다. 온라인 동기화는 앱이 활성 상태인 동안에만 실행됩니다 — 대형 라이브러리의 경우 데스크탑 버전을 사용하고 **백업 및 복원**으로 동기화된 라이브러리를 전송하세요.

### 오프라인 동기화

클라우드 폴더를 오프라인으로 사용 가능하게 표시하면 **파일 → 오프라인 폴더**에 나타나고 자동으로 다운로드됩니다. 온라인에서 추가된 새 파일도 다운로드됩니다. 이 화면에서 시간 간격을 구성하고 수동 동기화를 트리거합니다.

### 개인화

- **미디어 라이브러리 화면 스타일** — 일반 메뉴, 그룹화된 메뉴, 탭 메뉴.
- 세부 화면에서 큰 앨범 커버 표시 여부를 토글합니다.

### 앨범 커버

- **네트워크 유형** — Wi-Fi 또는 Wi-Fi + 셀룰러.
- **온라인 파일의 앨범 커버 불러오기** — 클라우드 파일에서 임베디드 아트워크를 가져옵니다 (추가 데이터 사용).
- **폴더에서 검색** — 비디오에 임베디드 커버가 없을 때 동일한 폴더의 JPEG / PNG 이미지를 사용합니다.
- **커버 품질** — 커버가 캐시되는 해상도를 조정합니다.
- **폴더에 표시** / **모두 삭제** — 아트워크 캐시를 관리합니다.

### 재생 목록

동일한 비디오를 재생 목록에 두 번 추가하는 것을 활성화합니다 (기본적으로 꺼져 있음).

### 최근 항목

최근 재생된 비디오 목록 관리 — 크기 변경, 삭제 또는 M3U / CSV / TXT로 내보내기.

### 즐겨찾기

- **동시 편집** — 미디어 라이브러리와 파일 섹션 간에 즐겨찾기를 미러링합니다.
- **목록 삭제** — 즐겨찾기 목록을 지웁니다.
- **노래 목록 내보내기** — 즐겨찾기를 M3U / CSV / TXT로 내보냅니다.

### 미디어 라이브러리 삭제

미디어 라이브러리 데이터베이스를 지웁니다. 스토리지의 비디오 및 오디오 파일은 그대로 유지됩니다.

## 비밀번호

**4자리 숫자 비밀번호**로 Evervideo를 보호합니다. 활성화되면 앱이 실행될 때마다 비밀번호를 입력하라는 메시지가 표시됩니다. 추가 보호를 위해 기기의 iOS Face ID / Touch ID와 함께 사용하세요.

## 파일 관리자

파일이 전송, 저장 및 미리 보기되는 방법을 제어합니다.

- **파일 전송** — 네트워크 기본 설정 (Wi-Fi만 또는 Wi-Fi + 셀룰러).
- **최대 병렬 작업 수** — 병렬 다운로드 스레드 수를 설정합니다.
- **파일 전송 작업** — 현재 활성 업로드 및 다운로드를 봅니다.
- **백그라운드 전송** — 앱이 백그라운드에 있는 동안 다운로드를 허용합니다.
- **다운로드한 파일 저장 위치** — 기본 다운로드 디렉토리.
- **동기화된 오프라인 폴더** — 오프라인 모드 폴더를 관리합니다.
- **시간 간격** — 오프라인 폴더의 변경 사항을 확인하는 빈도.
- **전체 파일명 표시** — 파일 관리자에서 확장자를 표시합니다.
- **온라인 파일 편집** — 연결된 클라우드 서비스의 읽기 전용 모드로 전환하려면 비활성화합니다.
- **열 때 파일 복사** — 다른 앱에서 가져온 파일을 처리하는 방법.
- **파일 미리 보기** — 생성된 파일 미리 보기를 관리합니다.
- **임시 파일 삭제** — 애플리케이션의 캐시 폴더를 지웁니다.

## Wi-Fi Drive

Wi-Fi Drive를 활성화하여 데스크탑 웹 브라우저, Finder 또는 File Explorer를 사용하여 컴퓨터에서 기기로 파일을 전송합니다. 자세한 지침은 [여기](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)에서 확인할 수 있습니다.

## 위젯

재생 중에 앱이 위젯 데이터를 업데이트하도록 위젯을 활성화합니다. 위젯 업데이트에는 추가 에너지가 필요하므로 토글은 기본적으로 꺼져 있습니다 — 홈 화면 또는 잠금 화면에서 실제로 위젯을 사용하는 경우에만 활성화하세요.

홈 화면 또는 잠금 화면을 길게 눌러, **+**를 탭하고, **Evervideo**를 검색하고, 위젯 크기를 선택하여 Evervideo 위젯을 추가할 수 있습니다. 위젯은 현재 재생 중인 비디오를 표시하고 전체 화면 플레이어로 직접 연결됩니다. 위젯은 macOS의 알림 센터에서도 작동합니다.

## 개인화

사용자 인터페이스를 기본 설정에 맞게 사용자 정의합니다.

- **애플리케이션 아이콘** — 대체 홈 화면 아이콘 (Premium).
- **색 구성표** — 어둡게, 밝게 또는 기본값 (시스템 외관을 따릅니다).
- **배경 스타일** — 블러 앨범 커버 또는 단색.
- **목록의 항목 외관** — 행 높이 자동 조정; 더 작은 썸네일 표시.
- **콘텐츠 로딩 제한** — 페이지 매김을 켜거나 끕니다.
- **파일 화면 스타일** — 일반 메뉴 또는 그룹화된 메뉴.
- **미디어 라이브러리 화면 스타일** — 일반 / 그룹화 / 탭 메뉴.
- **플레이어 화면 스타일** — 최소 / 하단 / 앤틱 / 클래식.
- **컨텍스트 메뉴 스타일** — 시스템 메뉴 또는 바텀 시트 스타일.

## 창

Mac 및 Catalyst에서 사용 가능합니다. 기본 크기 및 실행 시 동작과 같은 창 관련 기본 설정을 구성합니다.

## 화면

애플리케이션을 사용하는 동안 화면이 계속 활성 상태여야 하는지 여부를 선택합니다.

## 접근성

사용자 인터페이스에서 이미지를 숨기려면 **텍스트 모드**를 활성화합니다. VoiceOver가 활성화되면 텍스트 모드가 자동으로 활성화됩니다.

## 언어

애플리케이션 언어를 변경하고 시스템 기본값을 재정의합니다. 변경 사항은 Evervideo를 완전히 종료하고 다시 열면 적용됩니다. 120개 이상의 언어가 지원됩니다.

## 백업 및 복원

모든 애플리케이션 데이터를 백업하거나 다른 기기로 마이그레이션합니다. 포함할 내용을 선택합니다:

- **데이터베이스** — 미디어 라이브러리 항목, 재생 목록, 평점, 즐겨찾기, 시청 기록. 파일 크기를 관리하기 위해 오프라인 파일은 포함되지 않습니다.
- **앨범 커버** — 캐시된 아트워크.
- **설정** — 애플리케이션 설정.

백업을 시작하려면 **애플리케이션 데이터 백업**을 탭합니다. 새 기기에서 복원하려면 iCloud Drive, AirDrop 또는 연결된 클라우드 서비스를 통해 백업 파일을 이동하고 새 기기에서 엽니다.

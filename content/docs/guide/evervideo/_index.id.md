### Selamat Datang di Panduan Evervideo

Evervideo adalah pemutar media cloud berfitur lengkap untuk iPhone, iPad, dan Mac yang mengubah akun penyimpanan cloud, NAS, atau server media apa pun menjadi perpustakaan media pribadi Anda, tanpa perlu mengunggah ulang apa pun sambil tetap menjaga kendali penuh atas file Anda.

Dibangun di atas mesin pemutar berbasis FFmpeg kustom dengan dekoding H.264 dan HEVC yang dipercepat perangkat keras, Evervideo memutar hampir semua kontainer dan codec modern — MP4, MKV, AVI, MOV, FLV, WMV, WebM, TS, M2TS, dan berbagai format FFmpeg lainnya — dengan kualitas penuh, dengan buffering cerdas untuk streaming yang mulus melalui Wi-Fi atau seluler. Picture-in-Picture menjaga video Anda di atas setiap aplikasi lain, pemutar kompak yang selalu terlihat memungkinkan Anda terus menonton sambil menjelajahi perpustakaan, serta Chromecast dan AirPlay 2 mengirimkan pemutaran ke TV, HomePod, atau speaker hanya dengan satu ketuk.

Evervideo terhubung ke daftar sumber yang sangat luas, semuanya dari satu aplikasi:

- **Penyimpanan cloud pribadi:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · pCloud · Yandex Disk · MediaFire · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘).
- **Server media yang dihosting sendiri:** Plex · Jellyfin · Emby · Subsonic · Navidrome · Nextcloud (dan ownCloud melalui API bersama) · Synology Drive · QNAP.
- **Protokol NAS dan berbagi file:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP · FTPS · SFTP (autentikasi kata sandi atau kunci publik) · NFS · DLNA · UPnP.
- **Stream langsung dan kamera IP:** RTSP — arahkan Evervideo ke URL RTSP mana pun (`rtsp://camera-ip/stream`) dan langsung diputar.
- **Penyimpanan objek kompatibel S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces, dan endpoint S3-API lainnya.
- **Sumber di perangkat:** perpustakaan Foto (Semua Video, Pendek / Sedang / Panjang, Rekaman Layar, dan Album Foto), perpustakaan aplikasi Music (Album, Genre, Daftar Putar), drive USB / kartu SD, dan File Lokal.

### Satu Pemutar untuk Semua Format dan Codec

Sementara sebagian besar aplikasi iOS berhenti di H.264 / H.265 dalam MP4 / MOV, Evervideo menyertakan FFmpeg bersama codec sistem Apple sehingga Anda dapat memutar format yang tidak dikenali sistem — kontainer MKV, VP9, AV1, MPEG-2, OGG, Vorbis, dan banyak lagi — dan secara otomatis beralih antara dekoding hardware H.264 / HEVC dan dekoding perangkat lunak berdasarkan file dan perangkat.

Anda dapat memilih trek video (rip Blu-ray multi-stream), trek audio (trek komentar, dubbing alternatif), dan trek subtitle. File subtitle SRT, VTT, dan ASS / SSA eksternal dapat dimuat dengan satu ketuk; gaya ASS / SSA lanjutan dirender dengan benar berkat libass yang disertakan.

### Subtitle Cerdas

- **Pemilihan trek subtitle** sempurna untuk belajar bahasa.
- **File subtitle eksternal** (`.srt`, `.vtt`, `.ass`, `.ssa`) dimuat dari mana saja di perangkat Anda atau dari cloud.
- **libass** untuk rendering ASS / SSA yang sepenuhnya bergaya.
- **Pemilihan font per trek dan global** untuk subtitle.
- **Pemilihan trek audio** — pilih dubbing, komentar, atau trek sutradara.
- **Pemilihan trek video** untuk file multi-sudut atau multi-versi.

### Sesuaikan Gambar dan Suara

Dua equalizer berdampingan: equalizer audio untuk menyetel bass dan treble (dengan impor / ekspor preset kustom), dan equalizer video untuk mengatur kecerahan, kontras, saturasi, dan rona (juga dengan impor / ekspor). Kedua mesin juga menyediakan kontrol audiofil: laju sampel output audio, jumlah saluran, durasi buffer IO, dan mode campuran — untuk pengguna dengan DAC eksternal dan receiver home theater.

### Cast, AirPlay, dan Picture-in-Picture

- **Picture-in-Picture (PiP):** terus menonton sambil menggunakan aplikasi lain.
- **AirPlay 2:** kirim video ke Apple TV, HomePod, atau speaker / TV AirPlay 2 mana pun.
- **Google Chromecast:** cast langsung ke Chromecast atau TV berkemampuan Cast.
- **Pemutar kompak:** pemutar mini persisten di atas setiap layar sehingga Anda dapat menjelajahi perpustakaan tanpa kehilangan video.

### Privasi dan Keamanan

Evervideo menggunakan SDK resmi dan login berbasis OAuth dari setiap penyedia cloud sehingga kata sandi Anda tidak pernah sampai ke aplikasi. Token akses tersimpan terenkripsi di iOS/MacOS Keychain, setiap transfer dilindungi TLS, dan mencabut akses dari akun cloud Anda (atau menghapus Evervideo dari perangkat) akan menghapus semuanya secara instan. Lindungi perpustakaan Anda dengan kode sandi 4 digit opsional untuk lapisan privasi tambahan.

### Memulai dengan Evervideo

Panduan ini memandu Anda melalui setiap bagian Evervideo di iPhone, iPad, dan Mac — mulai dari menghubungkan layanan cloud, menjelajahi perpustakaan, mengunduh dan mentransfer file, mengelola daftar putar, hingga menyempurnakan pemutar media, equalizer, subtitle, dan Picture-in-Picture.

**メディアライブラリ**はEvervideoの中心です。接続されたクラウドサービス、NAS共有、ダウンロードされたファイル、**写真ライブラリ**、**ミュージックライブラリ**からアクセスできるすべてのビデオと音楽ファイルを単一の検索可能なハブに集約します。アルバム、ジャンル、プレイリスト、最近再生したもの、お気に入り、すべてのビデオ、未再生ビデオで閲覧でき、Evervideoは**視聴の進捗**を自動的に追跡するので、どのタイトルも中断した場所から正確に再開できます。

ライブラリにメディアを追加する方法は2つあります：**手動追加**（追加するものを正確に選択）または**自動同期**（Evervideoが指定されたクラウドフォルダーをスキャンし、新しいファイルが現れると自動的に追加）。

## 手動追加

ビデオと音楽を手動で追加するには、左上隅の**メディアを追加**アイコンをタップし、接続されたクラウドストレージサービス、NAS共有、メディアサーバー、またはデバイス上のファイルからフォルダーやファイルを選択します。ライブラリにファイルを追加すると、それらへの**リンク**のみが作成されます — 実際のコンテンツは貴重なディスクスペースを節約するために元の場所に残ります。メディアをオフラインで利用可能にしたい場合は、オプションメニューの**ダウンロード**アクションを使用するか、フォルダー、アルバム、ジャンル、またはプレイリストの**オフラインモード**を有効にしてください。

Macでは、ファイルをライブラリウィンドウにドラッグアンドドロップすることもできます。

## 再生を続行

設定 → プレーヤー → 一般で**プレーヤー状態を保存**が有効になっている場合、最後に保存された位置からプレーヤーの状態を復元します。有効にすると、すべてのフォルダー、アルバム、ジャンル、プレイリスト画面の上部に**再生を続行**ボタンが表示されます — タップすると、最後に離れた正確なビデオと位置にすぐに戻ります。

Evervideoはファイルごとの再生進捗も追跡するので、再生を続行を使用しなかった場合でも、リスト内のすべてのビデオに既にどれくらい視聴したかを示す小さな進捗バーが表示されます。これにより、長いTVシリーズ、講義シリーズ、チュートリアルコレクションの管理が特に簡単になります。

## 場所

ライブラリ内のすべてのビデオはソースタイプ別に整理されています。右上隅の**その他の操作**ボタンを使用し、**フィルター**を選択して場所でビデオをフィルタリングできます。

### オンラインビデオ

このセクションでは、接続されたクラウドストレージサービス、NAS共有、メディアサーバー（Plex / Jellyfin / Emby / Subsonic / Navidrome）からのビデオが表示されます。ここのビデオはオンデマンドでストリーミングされ、明示的にダウンロードするか、含まれるフォルダーのオフラインモードを有効にするまで、ローカルストレージを占有しません。

### このアプリのファイル

ここでは、ローカルファイルから提供されるオフライン再生に利用可能なビデオが見つかります。これにはアプリの**書類**ディレクトリ内のファイルが含まれます — ダウンロードしたもの、Wi-Fiドライブで転送したもの、またはFinderファイル共有でインポートしたものすべて。

### このiPhone / iPad / Mac上のファイル

このカテゴリには、**ファイルを開く…**システムダイアログを通じて追加された、デバイスからアプリケーションにインポートされたビデオが含まれます。元のファイルは元の場所に残ります；Evervideoはそれらへのリンクを保持するだけです。

### 写真ライブラリ

Evervideoは、iOSの写真ライブラリに保存されているすべてのビデオを閲覧および再生できます — すべてのカメラ録画、画面録画、ダウンロードしたクリップ、AirDropされたビデオファイル。ライブラリは便利なバケットに分割されています：

- **すべてのビデオ** — 長さに関係なく、写真内のすべてのビデオ。
- **短いビデオ** — 60秒未満のクリップ（ミームや素早い撮影に最適）。
- **中程度のビデオ** — 1分から5分の間。
- **長いビデオ** — 5分以上。
- **画面録画** — iOS画面録画のみ。
- **アルバム** — フォトアルバム（iCloud共有アルバムとスマートアルバムを含む）別にグループ化されたビデオ。

これは、字幕、ピクチャーインピクチャー、イコライザー、またはChromecastでEvervideoで視聴するために、写真から何もコピーする必要がないことを意味します。

### iOSミュージックライブラリ

Evervideoはミュージックアプリライブラリからもビデオを読み取ります（iTunesから購入したもの、自分のDVDからリッピングしたもの、または該当する場合はApple Musicのオフラインダウンロードで追加したもの）。コピーやトランスコードなしにアルバム、ジャンル、プレイリストで閲覧できます。

## カテゴリ

ビデオをメディアライブラリに追加すると、Evervideoは自動的にメタデータを読み取り、カテゴリーに整理します。

## タグのグループ化

これらのカテゴリーはメタデータでビデオを整理するのに役立ちます。ライブラリにビデオを追加すると、アプリはタグを読み取り、以下によってグループ化します：

- **すべてのビデオ** — タイトルで並べ替えられたライブラリ内のすべてのビデオ。
- **未再生ビデオ** — 最後まで再生されたことがないビデオ。
- **アルバム** — ALBUM_NAMEタグでグループ化されたビデオ（シリーズやボックスセットに便利）。
- **ジャンル** — GENREタグでグループ化されたビデオ。
- **プレイリスト** — 作成した順序での独自のプレイリスト。
- **最近再生したもの** — 最近再生したビデオ。
- **お気に入り** — お気に入りとしてマークしたビデオ。

すべてのタイトルが表示されない場合は、アプリがすべてのファイルをスキャンしたことを確認してください。設定 → メディアライブラリ → メタデータの読み込み → 処理されたファイル数でスキャンの進捗を確認できます。

## 最近再生したもの

このセクションでは、最後の再生位置とともにすべての最近再生したビデオが表示されるので、1回のタップでどれも再開できます。設定 → メディアライブラリ → 最近再生したもの → リストサイズを変更でリストが保持するエントリ数を変更でき、視聴履歴をバックアップするためにリストをM3U / CSV / TXTにエクスポートできます。

## お気に入り

プレーヤー画面またはオプションメニューを通じてビデオをお気に入りとしてマークします。お気に入りは独自のセクションに表示されるので、1回のタップで見つけられます。

## 視聴の進捗

Evervideoは視聴したすべてのビデオの再生位置を追跡します。どのリストのビデオも — 最近再生したもの、お気に入り、アルバム、ジャンル、プレイリスト、フォルダー — どのくらい視聴済みかを一目で確認できる小さな進捗バーが表示されます。これにより、長いTVシリーズの季節、コースのプレイリスト、一気視聴の夜の管理が楽になります。

## トップツールバー

ナビゲーションバーのすぐ下に位置するトップツールバーは、いくつかの便利なアクションを提供します：検索、すべて再生、すべてシャッフル、再生を続行。このツールバーは、単純な下にスワイプするジェスチャーで表示または非表示にできます。

## 検索

検索機能を使用すると、メディアライブラリ内の特定のタイトル、アルバム、ジャンル、またはプレイリストを見つけることができます。検索画面内では、並べ替え、フィルター、グリッド / リストビューアクションにアクセスできます。検索はメディアライブラリデータベースに対してローカルで実行されるので、完全にオフラインで機能し、入力中に結果を返します。

## オプションメニュー

ライブラリ内の各ビデオには、ビデオタイトル近くの3点ボタンをタップしてアクセスできるその他のアクションメニューがあります。これらのアクションは、個々のビデオかコレクションの一部かによって異なります。

### 個々のビデオの場合

- **次に再生** — ビデオをプレーヤーキューの先頭に追加します。
- **後で再生** — ビデオをプレーヤーキューの末尾に追加します。
- **プレイリストに追加** — ビデオをプレイリストに追加します。
- **お気に入りに追加** — ビデオをお気に入りとしてマークします。
- **ダウンロード** — ビデオをオフライン再生用のローカルファイルに保存します。
- **タグを編集** — 欠落したメタデータを修正するために組み込みタグエディターを開きます。
- **フォルダーに表示** — ビデオファイルが保存されているフォルダーを表示します。
- **Finderに表示** — Macからインポートされたファイルの場合、macOS Finderのフォルダーを表示します。
- **で開く** — ビデオファイルを別のアプリにエクスポートします。
- **クラウドサービスから削除** — メディアライブラリとクラウドストレージの両方からファイルを削除します。**このアクションは元に戻せません。**
- **メディアライブラリから削除** — ライブラリからビデオを削除しますが、ファイルはストレージに残ります。自動同期が有効で、ファイルがリモートストレージに存在する場合、同期後に再表示されます。

### コレクション（アルバム、ジャンル、プレイリスト）の場合

- **すべて再生** — プレーヤーキューを選択したコレクションのビデオで置き換えます。
- **次に再生** — ビデオをプレーヤーキューの先頭に追加します。
- **後で再生** — キューの末尾に追加します。
- **プレイリストに追加** — 新しいプレイリストを作成するオプションとともに、プレイリストに追加します。
- **オフラインモードを有効にする** — コレクションのビデオをローカルファイルにダウンロードします。サーバーに追加された新しいアイテムは自動的にダウンロードされます。
- **画像を編集** — コレクションのカバー画像を変更します。
- **アーカイブに追加** — コレクション全体を1つのZIPにまとめます（プレミアム）。
- **曲リストをエクスポート** — コレクションをM3U、CSV、またはTXTにエクスポートします。
- **メディアライブラリから削除** — ライブラリからコレクションを削除しますが、ソースファイルは削除しません。

## 選択モード

右上隅の**その他の操作**ボタンを使用して選択モードを有効にできます。このモードでは、一度に複数のビデオを選択し、バッチアクションを実行できます — プレイリストに追加、お気に入りとしてマーク、ライブラリから削除、ダウンロードなど。

## 設定

その他の操作 → 設定をタップして、メディアライブラリの設定を行います — メタデータ読み込み速度、オンラインおよびオフラインの同期フォルダー、ライブラリの表示スタイル、アルバムカバーの品質と検索、プレイリストの重複動作、最近再生したもののリストサイズ、お気に入りの同時編集など。詳細については[設定ガイド](/docs/guide/evervideo/evervideo-guide-settings/)を参照してください。

## メディアライブラリを削除

設定 → メディアライブラリ → ライブラリを削除から、メディアライブラリデータベースを消去できます。ストレージ上のビデオファイルは影響を受けません。

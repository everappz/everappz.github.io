**Perpustakaan Media** adalah jantung Evervideo. Ini menggabungkan setiap file video dan musik yang dapat Anda akses — dari layanan cloud yang terhubung, berbagi NAS, file yang diunduh, **perpustakaan Photos**, dan **perpustakaan Music** — ke dalam satu hub yang dapat dicari. Anda dapat menelusuri berdasarkan Album, Genre, Daftar Putar, Terbaru, Favorit, Semua Video, dan Video Belum Ditonton, dan Evervideo secara otomatis melacak **progres menonton** sehingga Anda dapat melanjutkan judul apa pun tepat di tempat Anda berhenti.

Anda memiliki dua cara untuk menambahkan media ke perpustakaan: **penambahan manual** (Anda memilih dengan tepat apa yang ditambahkan) atau **sinkronisasi otomatis** (Evervideo memindai folder cloud yang ditentukan dan menambahkan file baru secara otomatis saat muncul).

## Penambahan Manual

Untuk menambahkan video dan musik secara manual, ketuk ikon **Tambah Media** di sudut kiri atas dan pilih folder atau file dari layanan penyimpanan cloud yang terhubung, berbagi NAS, server media, atau file yang ada di perangkat Anda. Saat Anda menambahkan file ke perpustakaan, hanya **tautan** ke file tersebut yang dibuat — konten sebenarnya tetap di lokasi aslinya untuk menghemat ruang disk yang berharga. Jika Anda ingin membuat media tersedia offline, gunakan tindakan **Unduh** dari menu opsi atau aktifkan **Mode Offline** untuk folder, album, genre, atau daftar putar.

Di Mac, Anda juga dapat menyeret dan meletakkan file ke jendela perpustakaan.

## Lanjutkan Pemutaran

Pulihkan status pemutar dari posisi tersimpan terakhir jika **Simpan Status Pemutar** diaktifkan di Pengaturan → Pemutar → Umum. Setelah diaktifkan, tombol **Lanjutkan Pemutaran** muncul di bagian atas setiap folder, album, genre, dan layar daftar putar — ketuk untuk langsung kembali ke video dan posisi yang terakhir Anda tinggalkan.

Evervideo juga melacak progres pemutaran per file sehingga meskipun Anda tidak menggunakan Lanjutkan Pemutaran, setiap video dalam daftar menampilkan bilah progres kecil yang menunjukkan seberapa banyak yang telah Anda tonton. Ini membuat serial TV panjang, seri kuliah, dan koleksi tutorial sangat mudah dikelola.

## Lokasi

Semua video di perpustakaan Anda dikelompokkan dengan cermat berdasarkan jenis sumber. Anda dapat memfilter video berdasarkan lokasi menggunakan tombol **Tindakan Lainnya** di sudut kanan atas dan memilih **Filter**.

### Video Online

Bagian ini menampilkan video dari layanan penyimpanan cloud yang terhubung, berbagi NAS, dan server media (Plex / Jellyfin / Emby / Subsonic / Navidrome). Video di sini di-stream sesuai permintaan; tidak ada yang mengambil penyimpanan lokal hingga Anda mengunduhnya secara eksplisit atau mengaktifkan mode offline untuk folder yang berisi.

### File dalam Aplikasi Ini

Di sini Anda dapat menemukan video yang tersedia untuk pemutaran offline, bersumber dari file lokal Anda. Ini termasuk file dalam direktori **Dokumen** aplikasi — apa pun yang Anda unduh, transfer melalui Wi-Fi Drive, atau impor melalui Berbagi File Finder.

### File di iPhone / iPad / Mac Ini

Kategori ini mencakup video yang diimpor ke aplikasi dari perangkat Anda, ditambahkan melalui dialog sistem **Buka File…**. File asli tetap di lokasi aslinya; Evervideo hanya menyimpan tautan ke file tersebut.

### Perpustakaan Photos

Evervideo dapat menelusuri dan memutar setiap video yang tersimpan di perpustakaan iOS Photos Anda — semua rekaman kamera, rekaman layar, klip yang diunduh, dan file video AirDrop. Perpustakaan dibagi menjadi kelompok yang nyaman:

- **Semua Video** — setiap video di Photos, terlepas dari durasinya.
- **Video Pendek** — klip di bawah 60 detik (bagus untuk meme dan tangkapan cepat).
- **Video Sedang** — antara 1 menit dan 5 menit.
- **Video Panjang** — 5 menit ke atas.
- **Rekaman Layar** — hanya Rekaman Layar iOS.
- **Album** — video dikelompokkan berdasarkan Album Foto Anda (termasuk album yang dibagikan iCloud dan Album Cerdas).

Ini berarti Anda tidak perlu menyalin apa pun dari Photos untuk menontonnya di Evervideo dengan subtitle, Picture-in-Picture, equalizer, atau Chromecast.

### Perpustakaan Music iOS

Evervideo juga membaca video dari perpustakaan aplikasi Music (apa pun yang Anda beli dari iTunes, rip dari DVD Anda sendiri, atau tambahkan melalui unduhan offline Apple Music jika berlaku). Anda dapat menelusuri berdasarkan Album, Genre, dan Daftar Putar — tanpa menyalin atau mengonversi.

## Kategori

Saat Anda menambahkan video ke perpustakaan media, Evervideo secara otomatis membaca metadata dan mengaturnya ke dalam kategori.

## Pengelompokan Tag

Kategori-kategori ini membantu Anda mengatur video berdasarkan metadata. Saat Anda menambahkan video ke perpustakaan, aplikasi membaca tag dan mengelompokkannya berdasarkan:

- **Semua Video** — setiap video di perpustakaan, diurutkan berdasarkan judul.
- **Video Belum Ditonton** — video yang belum pernah diputar sampai habis.
- **Album** — video dikelompokkan berdasarkan tag ALBUM_NAME (berguna untuk serial dan box set).
- **Genre** — video dikelompokkan berdasarkan tag GENRE.
- **Daftar Putar** — daftar putar Anda sendiri, dalam urutan pembuatan.
- **Terbaru** — video yang baru saja Anda putar.
- **Favorit** — video yang Anda tandai sebagai favorit.

Jika Anda tidak melihat semua judul, pastikan aplikasi telah memindai setiap file. Anda dapat memeriksa progres pemindaian di Pengaturan → Perpustakaan Media → Pembacaan Metadata → Jumlah File yang Diproses.

## Terbaru

Bagian ini menampilkan semua video yang baru saja diputar beserta posisi pemutaran terakhirnya, sehingga Anda dapat melanjutkan salah satunya dengan satu ketukan. Anda dapat mengubah berapa banyak entri yang disimpan daftar di Pengaturan → Perpustakaan Media → Terbaru → Ubah Ukuran Daftar, dan mengekspor daftar ke M3U / CSV / TXT untuk mencadangkan riwayat tontonan Anda.

## Favorit

Tandai video sebagai favorit di layar pemutar atau melalui menu opsi. Favorit muncul di bagian tersendiri sehingga Anda dapat menemukannya dengan satu ketukan.

## Progres Menonton

Evervideo melacak posisi pemutaran setiap video yang Anda tonton. Setiap video dalam daftar apa pun — Terbaru, Favorit, album, genre, daftar putar, folder — menampilkan bilah progres kecil sehingga Anda dapat melihat sekilas berapa banyak yang telah Anda tonton. Ini membuat musim serial TV panjang, daftar putar kursus, dan malam maraton sangat mudah dikelola.

## Toolbar Atas

Terletak tepat di bawah bilah navigasi, toolbar atas menawarkan beberapa tindakan yang nyaman: Cari, Putar Semua, Acak Semua, dan Lanjutkan Pemutaran. Anda dapat menampilkan atau menyembunyikan toolbar ini dengan gerakan geser ke bawah yang sederhana.

## Pencarian

Fitur pencarian memungkinkan Anda menemukan judul, album, genre, atau daftar putar tertentu dalam perpustakaan media Anda. Dalam layar Pencarian, Anda memiliki akses ke tindakan Urutkan, Filter, dan tampilan Grid / Daftar. Pencarian berjalan secara lokal terhadap database perpustakaan media, sehingga berfungsi sepenuhnya offline dan menampilkan hasil saat Anda mengetik.

## Menu Opsi

Setiap video di perpustakaan Anda memiliki menu tindakan lainnya, diakses dengan mengetuk tombol tiga titik di dekat judul video. Tindakan ini bervariasi tergantung pada apakah itu video tunggal atau bagian dari koleksi.

### Untuk Video Individual

- **Putar Berikutnya** — menambahkan video ke bagian atas antrean pemutar.
- **Putar Nanti** — menambahkan video ke bagian bawah antrean pemutar.
- **Tambah ke Daftar Putar** — menambahkan video ke daftar putar.
- **Tambah ke Favorit** — menandai video sebagai favorit.
- **Unduh** — menyimpan video ke file lokal untuk pemutaran offline.
- **Edit Tag** — membuka editor tag bawaan untuk memperbaiki metadata yang hilang.
- **Tampilkan di Folder** — menampilkan folder tempat file video disimpan.
- **Tampilkan di Finder** — untuk file yang diimpor dari Mac, menampilkan folder di macOS Finder.
- **Buka Di** — mengekspor file video ke aplikasi lain.
- **Hapus dari Layanan Cloud** — menghapus file dari perpustakaan media dan penyimpanan cloud. **Tindakan ini tidak dapat dipulihkan.**
- **Hapus dari Perpustakaan Media** — menghapus video dari perpustakaan, tetapi file tetap di penyimpanan. Jika sinkronisasi otomatis diaktifkan dan file ada di penyimpanan jarak jauh, file akan muncul kembali setelah sinkronisasi.

### Untuk Koleksi (Album, Genre, Daftar Putar)

- **Putar Semua** — mengganti antrean pemutar dengan video dari koleksi yang dipilih.
- **Putar Berikutnya** — menambahkan video ke bagian atas antrean pemutar.
- **Putar Nanti** — menambahkannya ke bagian bawah antrean.
- **Tambah ke Daftar Putar** — menambahkannya ke daftar putar, dengan opsi membuat yang baru.
- **Aktifkan Mode Offline** — mengunduh video dari koleksi ke file lokal. Item baru yang ditambahkan di server diunduh secara otomatis.
- **Edit Gambar** — mengubah gambar sampul untuk koleksi.
- **Tambah ke Arsip** — mengemas seluruh koleksi menjadi satu ZIP (Premium).
- **Ekspor Daftar Lagu** — mengekspor koleksi ke M3U, CSV, atau TXT.
- **Hapus dari Perpustakaan Media** — menghapus koleksi dari perpustakaan tetapi tidak menghapus file sumber.

## Mode Seleksi

Anda dapat mengaktifkan mode seleksi menggunakan tombol **Tindakan Lainnya** di sudut kanan atas. Dalam mode ini, Anda dapat memilih beberapa video sekaligus dan melakukan tindakan batch — tambah ke daftar putar, tandai sebagai favorit, hapus dari perpustakaan, unduh, dan lainnya.

## Pengaturan

Ketuk Tindakan Lainnya → Pengaturan untuk mengonfigurasi preferensi perpustakaan media Anda — termasuk kecepatan pembacaan metadata, folder sinkronisasi online dan offline, gaya presentasi perpustakaan, kualitas dan pencarian sampul album, perilaku duplikat daftar putar, ukuran daftar terbaru, dan pengeditan bersamaan favorit. Lihat [panduan Pengaturan](/docs/guide/evervideo/evervideo-guide-settings/) untuk penjelasan lengkap.

## Hapus Perpustakaan Media

Di Pengaturan → Perpustakaan Media → Hapus Perpustakaan, Anda dapat menghapus database perpustakaan media. File video Anda di penyimpanan tidak tersentuh.

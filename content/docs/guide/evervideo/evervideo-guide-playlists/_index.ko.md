재생 목록 섹션에는 비디오 컬렉션을 관리하는 데 유용한 도구가 있습니다 — 시리즈, 강좌, 튜토리얼 시퀀스, 뮤직 비디오 및 큐레이션된 마라톤에 완벽합니다. 이 영역에는 모든 재생 목록이 한 곳에 표시됩니다. 상단에는 재생 목록 옵션이 있는 메뉴를 여는 내비게이션 바의 "..." 버튼과 검색, 모두 재생, 모두 셔플과 같은 빠른 작업이 있는 툴바가 있습니다. 각 재생 목록에는 제목 옆에 해당 재생 목록에 대한 더 많은 옵션을 제공하는 "..." 버튼도 있습니다.

Evervideo의 재생 목록에는 온라인 클라우드 비디오, 오프라인 다운로드된 파일, 로컬 파일, 사진 라이브러리 비디오, iOS 음악 라이브러리 비디오를 혼합하여 포함할 수 있습니다 — 모두 하나의 재생 목록에서 — 원활하게 함께 재생됩니다.

## 재생 목록 만들기

새 재생 목록 만들기는 간단합니다. 두 가지 옵션이 있습니다: **+** 버튼을 탭하거나, 내비게이션 바 오른쪽 상단 모서리의 **"..."** 버튼을 탭하고 **새 재생 목록**을 선택합니다. 재생 목록에 의미 있는 이름을 지정하고 **저장하다**를 탭합니다.

그러면 포함하려는 비디오를 선택할 수 있는 **파일 추가** 대화 상자가 열립니다. 비디오는 소스 유형별로 편리하게 분류됩니다:

- **라이브러리** — 미디어 라이브러리에 이미 있는 파일.
- **이 애플리케이션의 파일** — 앱의 Documents 폴더에 있는 비디오.
- **이 기기의 파일** — 이 애플리케이션이 아닌 기기의 다른 곳에 있는 비디오.
- **연결하기** — 연결된 클라우드 스토리지 서비스 및 미디어 서버 내에 위치한 온라인 비디오.

기본적으로 비디오를 재생 목록에 한 번만 추가할 수 있습니다. 중복을 허용하려면 설정 → 라이브러리 → 재생 목록 → 재생 목록의 중복 → 활성화에서 활성화하세요.

## 재생 목록 가져오기

Evervideo는 M3U / M3U8 / CUE 재생 목록 파일 가져오기를 지원하므로 수동으로 다시 만들지 않고도 기존 재생 목록을 가져올 수 있습니다.

먼저 재생 목록 섹션으로 이동합니다. 그런 다음 오른쪽 상단 모서리의 더보기 버튼을 탭합니다. 메뉴에서 재생 목록 가져오기를 선택합니다.

다음 화면에서 파일 위치를 선택합니다. 지원되는 옵션:

- 연결된 클라우드 스토리지
- 애플리케이션의 파일
- 기기의 파일

위치를 선택하고 재생 목록 파일이 있는 폴더를 열고 .m3u, .m3u8 또는 .cue 파일을 선택한 다음 완료됨을 탭합니다. Evervideo는 파일을 파싱하고, 스토리지에서 참조된 각 비디오를 찾고, 라이브러리에 실제 재생 목록을 만듭니다. 중요: 재생 목록 파일은 지정된 경로에 실제로 존재하는 파일을 참조해야 합니다.

## 재생 목록 세부 화면

재생 목록을 열면 재생 목록 세부 화면이 나타납니다. 오른쪽 상단 모서리에 재생 목록 옵션이 있는 **"..."** 버튼과 커버 이미지 아래에 **검색**, **계속 재생**, **모두 재생**, **모두 셔플** 세 버튼이 있습니다. 전체 재생 목록 오프라인 동기화를 전환하는 **오프라인 모드** 확인란도 있습니다.

- **계속 재생** — 이 재생 목록의 마지막 저장된 재생 위치를 복원합니다.
- **검색** — 현재 재생 목록 내에서 검색을 수행합니다.
- **모두 재생** — 현재 재생 목록의 모든 비디오를 플레이어 대기열에 추가합니다.
- **모두 셔플** — **모두 재생**과 같지만 재생 목록에 추가하기 전에 비디오를 셔플합니다.
- **오프라인 모드** — 이 재생 목록의 모든 비디오를 로컬 파일로 다운로드합니다. 재생 목록에 추가된 새 항목도 자동으로 다운로드됩니다.

## 재생 목록 화면에서 재생 목록에 대한 추가 작업

재생 목록 제목 근처의 **"..."** 버튼을 탭하여 재생 목록의 작업에 접근할 수 있습니다. 사용 가능한 작업:

- **모두 재생** — 새로운 플레이어 대기열에 재생 목록 비디오를 추가합니다.
- **다음에 재생** — 기존 플레이어 대기열의 상단에 재생 목록 비디오를 추가합니다.
- **나중에 재생** — 기존 플레이어 대기열의 하단에 재생 목록 비디오를 추가합니다.
- **이미지 편집** — 재생 목록의 커버 아트워크를 변경합니다.
- **오프라인 모드 활성화** — 재생 목록의 오프라인 모드를 켭니다. 기존 비디오와 새 비디오 모두 자동으로 다운로드됩니다.
- **노래 목록 내보내기** — 이 재생 목록을 **M3U / M3U8 / CSV / TXT**로 내보냅니다.
- **아카이브에 추가** — 이 재생 목록 (m3u 파일, 커버 이미지 및 모든 비디오 포함)을 단일 ZIP으로 아카이브합니다 (Premium).
- **비디오 추가** — 현재 재생 목록에 더 많은 비디오를 추가합니다.
- **이름 변경** — 재생 목록 이름을 변경합니다.
- **재생 목록 삭제** — 미디어 라이브러리에서 재생 목록을 삭제합니다. **이 작업은 취소할 수 없습니다.**

## 재생 목록 세부 화면에서 재생 목록에 대한 추가 작업

재생 목록 세부 화면 오른쪽 상단 모서리의 **"..."** 버튼을 탭하여 작업에 접근할 수 있습니다. 사용 가능한 작업:

- **선택하다** — 비디오 선택 모드를 활성화합니다. 여러 비디오를 삭제하거나 순서를 변경하는 데 유용합니다.
- **다음에 재생 / 나중에 재생** — 재생 목록 비디오를 대기열에 추가합니다.
- **비디오 추가** — 재생 목록에 새 비디오를 추가합니다.
- **비디오 재배열** — 드래그 앤 드롭을 사용하여 비디오 순서를 수동으로 변경합니다.
- **이름 변경** — 현재 재생 목록 이름을 변경합니다.
- **이미지 편집** — 커버 아트워크를 변경합니다.
- **노래 목록 내보내기** — 재생 목록을 M3U / CSV / TXT로 내보냅니다.
- **아카이브에 추가** — 모든 비디오와 m3u 파일을 포함하여 현재 재생 목록을 ZIP으로 압축합니다 (Premium).
- **정렬** — 재생 목록에서 비디오 순서를 변경합니다. 정렬 옵션에는 **제목, 앨범, 장르, 평점, 연도, 재생 시간, 파일 이름, 파일 수정 날짜, 파일 생성 날짜** 및 **수동**이 포함됩니다.
- **검색** — 현재 재생 목록 내에서 특정 비디오를 검색합니다.
- **그리드 / 목록** — 화면 레이아웃 표현을 변경합니다.
- **재생 목록 삭제** — 미디어 라이브러리에서 재생 목록을 삭제합니다. 이 작업은 스토리지에서 비디오를 삭제하지 않지만 **취소할 수 없습니다**.

## 재생 목록에서 파일 순서 변경

재생 목록에서 비디오 순서를 변경하려면 오른쪽 상단 모서리의 **"..."** 버튼을 탭하고 **선택하다**를 선택하여 선택 모드로 전환합니다. 각 비디오 근처의 재순서 컨트롤과 드래그 앤 드롭 제스처를 사용하여 위나 아래로 이동합니다. 재순서 컨트롤을 탭하면 비디오가 목록 상단으로 이동합니다. 선택 모드를 종료하고 변경 사항을 적용하려면 **완료됨**을 탭합니다.

## 재생 목록 커버 이미지 변경

재생 목록의 커버 이미지를 변경하려면 오른쪽 상단 모서리의 **"..."** 버튼을 탭하고 **이미지 편집**을 선택합니다. 사용 가능한 소스 (사진, 파일, 클라우드 스토리지 또는 재생 목록의 비디오에서 생성된 아트워크)에서 이미지를 선택한 다음 **완료됨**을 탭하여 확인합니다.

## 재생 목록에 비디오 추가

재생 목록을 열고 오른쪽 상단 모서리의 **"..."** 버튼을 탭한 다음 **비디오 추가**를 선택하여 대화 상자를 엽니다. 추가할 비디오를 선택하고 **완료됨**을 탭하여 확인합니다.

## 재생 목록에서 여러 비디오 삭제

재생 목록을 열고, 오른쪽 상단 모서리의 **"..."** 버튼을 탭하고 **선택하다**를 선택하여 선택 모드로 전환합니다. 삭제할 비디오를 선택하고 화면 하단의 **재생 목록에서 삭제**를 탭합니다. **완료됨**을 탭하여 확인합니다.

## 비디오 옵션

재생 목록의 각 비디오에는 **"..."** 버튼을 탭하여 접근할 수 있는 작업 목록이 있습니다. 모든 작업이 보이지 않으면 아래로 스크롤하여 봅니다. 재생 목록에서 비디오를 삭제하고, 다운로드하고, 메타데이터를 편집하는 등의 작업을 수행할 수 있습니다.

- **다음에 재생** — 플레이어 대기열의 상단에 비디오를 추가합니다.
- **나중에 재생** — 플레이어 대기열의 하단에 비디오를 추가합니다.
- **재생 목록에 추가** — 다른 재생 목록에 비디오를 추가합니다.
- **즐겨찾기에 추가** — 비디오를 즐겨찾기로 표시합니다.
- **다운로드** — 비디오를 오프라인으로 사용 가능하게 합니다.
- **태그 편집** — 메타데이터 변경을 위한 내장 태그 편집기를 엽니다.
- **다른 앱에서 열기** — 비디오를 내보내고 다른 앱에서 엽니다.
- **폴더에 표시** — 파일이 있는 폴더를 표시합니다.
- **Finder에 표시** — Mac에서 가져온 파일의 경우 Mac의 폴더를 표시합니다.
- **재생 목록에서 삭제** — 재생 목록에서 비디오를 제거합니다.
- **클라우드 서비스에서 삭제** — 재생 목록과 클라우드 서비스에서 비디오를 삭제합니다. **되돌릴 수 없습니다.**
- **미디어 라이브러리에서 삭제** — 라이브러리에서 비디오를 삭제하고 스토리지의 파일은 그대로 둡니다.

## 접근성

Evervideo는 **VoiceOver**로 완전히 접근 가능합니다. 모든 구성 요소에 잘 설계된 레이블과 설명이 있습니다. VoiceOver가 활성화되면 앱이 **텍스트 모드**로 전환되어 의미 있는 요소만 표시됩니다 — 탐색 속도와 명확성을 향상시킵니다. **설정 → 접근성 → 텍스트 모드**에서 텍스트 모드를 활성화할 수도 있습니다.

### VoiceOver를 사용하여 재생 목록에서 비디오 위치 조정

1. 재생 목록을 열고 **더보기** 버튼을 탭합니다.
2. **비디오 재배열**을 선택합니다. 보기가 편집 모드로 전환됩니다.
3. 비디오 제목 근처의 재순서 표시기 아이콘을 탭하여 포커스를 줍니다.
4. 재순서 표시기 아이콘을 빠르게 **두 번 탭**합니다. 두 번째 탭 시 손가락을 떼지 마세요 — 셀이 이동 준비가 됐다는 소리가 들릴 때까지 누르고 있으세요.
5. 셀을 새 위치로 이동합니다.

다른 구성 요소는 시스템 제공 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

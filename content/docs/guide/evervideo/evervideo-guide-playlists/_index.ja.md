プレイリストセクションでは、ビデオコレクションを管理するための便利なツールが見つかります — シリーズ、コース、チュートリアルシーケンス、ミュージックビデオ、キュレートされたマラソンに最適です。このエリアにはすべてのプレイリストが1か所に表示されます。上部にはナビゲーションバーに「...」ボタンがあり、プレイリストオプションのメニューが開きます。また、検索、すべて再生、すべてシャッフルなどのクイックアクションを含むツールバーもあります。各プレイリストにも、そのタイトルの横に独自の「...」ボタンがあり、そのプレイリスト専用のオプションを提供します。

Evervideoのプレイリストには、オンラインクラウドビデオ、オフラインでダウンロードされたファイル、ローカルファイル、写真ライブラリビデオ、iOSミュージックライブラリビデオのミックスを含めることができます — すべて1つのプレイリストに — そしてシームレスに一緒に再生されます。

## プレイリストの作成

新しいプレイリストの作成は簡単です。2つのオプションがあります：**+**ボタンをタップするか、ナビゲーションバーの右上隅の**「...」ボタン**をタップして**新しいプレイリスト**を選択します。プレイリストに意味のある名前を付け、**保存する**をタップします。

これにより**ファイルを追加**ダイアログが起動し、含めたいビデオを手動で選択できます。ビデオはソースタイプ別に便利にカテゴリー分けされています：

- **ライブラリ** — すでにメディアライブラリにあるファイル。
- **このアプリのファイル** — アプリのドキュメントフォルダーにあるビデオ。
- **このデバイスのファイル** — このアプリではなく、デバイスの他の場所にあるビデオ。
- **接続** — 接続されたクラウドストレージサービスとメディアサーバー内のオンラインビデオ。

デフォルトでは、プレイリストに1つのビデオを1回しか追加できません。重複を許可したい場合は、設定 → ライブラリ → プレイリスト → プレイリスト内の重複 → 有効にするで有効にします。

## プレイリストのインポート

EvervideoはM3U / M3U8 / CUEプレイリストファイルのインポートをサポートしているため、既存のプレイリストを手動で再作成することなく持ち込むことができます。

まず、プレイリストセクションに移動します。次に、右上隅のその他ボタンをタップします。メニューから、プレイリストをインポートを選択します。

次の画面でファイルの場所を選択します。サポートされているオプション：

- 接続されたクラウドストレージ
- アプリ内のファイル
- デバイス上のファイル

場所を選択し、プレイリストファイルが含まれるフォルダーを開き、.m3u、.m3u8、または.cueファイルを選択して完了をタップします。Evervideoはファイルを解析し、ストレージ上の参照された各ビデオを見つけ、ライブラリに実際のプレイリストを作成します。重要：プレイリストファイルは指定されたパスに実際に存在するファイルを参照している必要があります。

## プレイリスト詳細画面

プレイリストを開くと、プレイリスト詳細画面が表示されます。右上隅にプレイリストオプションの**「...」ボタン**があり、カバー画像の下に3つのボタンがあります：**検索**、**再生を続行**、**すべて再生**、**すべてシャッフル**。また、プレイリスト全体のオフライン同期を切り替える**オフラインモード**チェックボックスもあります。

- **再生を続行** — このプレイリストの最後に保存された再生位置を復元します。
- **検索** — 現在のプレイリスト内で検索を実行します。
- **すべて再生** — 現在のプレイリストのすべてのビデオをプレーヤーキューに追加します。
- **すべてシャッフル** — **すべて再生**と同様ですが、プレーヤーキューに追加する前にビデオをシャッフルします。
- **オフラインモード** — このプレイリストのすべてのビデオをローカルファイルにダウンロードします。プレイリストに追加された新しいアイテムも自動的にダウンロードされます。

## プレイリスト画面のプレイリストのその他のアクション

プレイリストのタイトル付近の**「...」ボタン**をタップしてプレイリストのアクションにアクセスできます。利用可能なアクション：

- **すべて再生** — プレイリストのビデオを新しいプレーヤーキューに追加します。
- **次に再生** — プレイリストのビデオを既存のプレーヤーキューの先頭に追加します。
- **後で再生** — プレイリストのビデオを既存のプレーヤーキューの末尾に追加します。
- **画像を編集** — プレイリストのカバーアートワークを変更します。
- **オフラインモードを有効にする** — プレイリストのオフラインモードをオンにします。既存のビデオと新しいビデオの両方が自動的にダウンロードされます。
- **曲リストをエクスポート** — このプレイリストを**M3U / M3U8 / CSV / TXT**にエクスポートします。
- **アーカイブに追加** — このプレイリスト（m3uファイル、カバー画像、すべてのビデオを含む）を1つのZIPにアーカイブします（Premium）。
- **ビデオを追加** — 現在のプレイリストにビデオを追加します。
- **名前を変更** — プレイリストの名前を変更します。
- **プレイリストを削除** — メディアライブラリからプレイリストを削除します。**このアクションは元に戻せません。**

## プレイリスト詳細画面のプレイリストのその他のアクション

プレイリスト詳細画面の右上隅の**「...」ボタン**をタップしてアクションにアクセスできます。利用可能なアクション：

- **選択する** — ビデオ選択モードをアクティブにします。複数のビデオを削除したり順序を変更したりするのに便利です。
- **次に再生 / 後で再生** — プレイリストのビデオをキューに追加します。
- **ビデオを追加** — プレイリストに新しいビデオを追加します。
- **ビデオを並べ替える** — ドラッグ＆ドロップを使用してビデオの順序を手動で変更します。
- **名前を変更** — 現在のプレイリストの名前を変更します。
- **画像を編集** — カバーアートワークを変更します。
- **曲リストをエクスポート** — プレイリストをM3U / CSV / TXTにエクスポートします。
- **アーカイブに追加** — すべてのビデオとm3uファイルを含む現在のプレイリストをZIP化します（Premium）。
- **並べ替え** — プレイリスト内のビデオの順序を変更します。並べ替えオプションには**タイトル、アルバム、ジャンル、評価、年、時間、ファイル名、ファイル更新日、ファイル作成日**および**手動**が含まれます。
- **検索** — 現在のプレイリスト内の特定のビデオを検索します。
- **グリッド / リスト** — 画面レイアウト表示を変更します。
- **プレイリストを削除** — メディアライブラリからプレイリストを削除します。このアクションはビデオをストレージから削除しませんが、**元に戻せません**。

## プレイリスト内のファイル順序の変更

プレイリスト内のビデオの順序を変更するには、右上隅の**「...」ボタン**をタップして**選択する**を選択し、選択モードに入ります。各ビデオ付近の並べ替えコントロールとドラッグ＆ドロップジェスチャーを使用して上下に移動します。並べ替えコントロールをタップするとビデオがリストの先頭に移動します。選択モードを終了して変更を適用するには、**完了**をタップします。

## プレイリストのカバー画像の変更

プレイリストのカバー画像を変更するには、右上隅の**「...」ボタン**をタップして**画像を編集**を選択します。利用可能なソース（写真、ファイル、クラウドストレージ、またはプレイリスト内のビデオから生成されたアートワーク）から画像を選択し、**完了**をタップして確認します。

## プレイリストへのビデオの追加

プレイリストを開いて右上隅の**「...」ボタン**をタップし、**ビデオを追加**を選択してダイアログを開きます。追加したいビデオを選択し、**完了**をタップして確認します。

## プレイリストからの複数ビデオの削除

プレイリストを開き、右上隅の**「...」ボタン**をタップして**選択する**を選択し、選択モードに入ります。削除したいビデオを選択し、画面下部の**プレイリストから削除**をタップします。**完了**をタップして確認します。

## ビデオオプション

プレイリスト内の各ビデオには、**「...」ボタン**をタップしてアクセスできるアクションのリストがあります。すべてのアクションが見えない場合は、下にスクロールして確認します。プレイリストからビデオを削除したり、ダウンロードしたり、メタデータを編集したりできます。

- **次に再生** — プレーヤーキューの先頭にビデオを追加します。
- **後で再生** — プレーヤーキューの末尾にビデオを追加します。
- **プレイリストに追加** — 別のプレイリストにビデオを追加します。
- **お気に入りに追加** — ビデオをお気に入りとしてマークします。
- **ダウンロード** — ビデオをオフラインで利用可能にします。
- **タグを編集** — メタデータを変更するために組み込みタグエディターを開きます。
- **で開く** — ビデオをエクスポートして別のアプリで開きます。
- **フォルダーに表示** — ファイルがある場所のフォルダーを表示します。
- **Finderに表示** — Macからインポートされたファイルの場合、Mac上のフォルダーを表示します。
- **プレイリストから削除** — プレイリストからビデオを削除します。
- **クラウドサービスから削除** — プレイリストとクラウドサービスからビデオを削除します。**元に戻せません。**
- **メディアライブラリから削除** — ライブラリからビデオを削除しますが、ストレージ上のファイルはそのままにします。

## アクセシビリティ

Evervideoは**VoiceOver**で完全にアクセス可能です。すべてのコンポーネントには適切に設計されたラベルと説明があります。VoiceOverが有効な場合、アプリは**テキストモード**に切り替わり、意味のある要素のみが表示されます — ナビゲーション速度と明確さが向上します。**設定 → アクセシビリティ → テキストモード**でテキストモードを有効にすることもできます。

### VoiceOverでプレイリスト内のビデオ位置を調整する

1. プレイリストを開き、**その他**ボタンをタップします。
2. **ビデオを並べ替える**を選択します。ビューが編集モードに切り替わります。
3. ビデオタイトル付近の並べ替えインジケーターアイコンをタップしてフォーカスを与えます。
4. 並べ替えインジケーターアイコンを素早く**ダブルタップ**します。2回目のタップでは指を離さず — セルが移動できる状態になったことを示す音が聞こえるまで押さえ続けます。
5. セルを新しい位置に移動します。

その他のコンポーネントは、システムが提供するVoiceOverパターンを使用して期待通りに機能します。

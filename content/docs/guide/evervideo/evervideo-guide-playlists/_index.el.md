Στην ενότητα Λίστες αναπαραγωγής, θα βρείτε χρήσιμα εργαλεία για τη διαχείριση των συλλογών βίντεο — ιδανικά για σειρές, μαθήματα, αλληλουχίες tutorials, βίντεο μουσικής και επιμελημένα marathons. Αυτή η περιοχή εμφανίζει όλες τις λίστες αναπαραγωγής σε ένα μέρος. Στην κορυφή υπάρχει κουμπί "..." στη γραμμή πλοήγησης που ανοίγει μενού με επιλογές λίστας αναπαραγωγής, συν γραμμή εργαλείων με γρήγορες ενέργειες όπως Αναζήτηση, Αναπαραγωγή Όλων και Ανακάτεμα Όλων. Κάθε λίστα αναπαραγωγής έχει επίσης το δικό της κουμπί "..." δίπλα στον τίτλο της.

Οι λίστες αναπαραγωγής στο Evervideo μπορούν να περιέχουν μίξη online cloud βίντεο, offline ληφθέντα αρχεία, τοπικά αρχεία, βίντεο βιβλιοθήκης Φωτογραφιών και βίντεο βιβλιοθήκης iOS Μουσικής — όλα σε μία λίστα — και αναπαράγονται άψογα μαζί.

## Δημιουργία Λίστας Αναπαραγωγής

Η δημιουργία νέας λίστας αναπαραγωγής είναι απλή. Έχετε δύο επιλογές: πατήστε το κουμπί **+**, ή πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία της γραμμής πλοήγησης και επιλέξτε **Νέα Λίστα Αναπαραγωγής**. Δώστε στη λίστα αναπαραγωγής ένα σημαντικό όνομα, στη συνέχεια πατήστε **Αποθήκευση**.

Αυτό ενεργοποιεί τον διάλογο **Προσθήκη Αρχείων**, όπου μπορείτε να επιλέξετε χειροκίνητα τα βίντεο που θέλετε να συμπεριλάβετε. Τα βίντεο κατηγοριοποιούνται κατά τύπο πηγής:

- **Βιβλιοθήκη** — αρχεία ήδη στη βιβλιοθήκη πολυμέσων σας.
- **Αρχεία σε Αυτήν την Εφαρμογή** — βίντεο στον φάκελο Έγγραφα της εφαρμογής.
- **Αρχεία σε Αυτή τη Συσκευή** — βίντεο αλλού στη συσκευή σας, όχι σε αυτήν την εφαρμογή.
- **Συνδέσεις** — online βίντεο στις συνδεδεμένες υπηρεσίες αποθήκευσης cloud και media servers.

Από προεπιλογή, μπορείτε να προσθέσετε ένα βίντεο σε μια λίστα αναπαραγωγής μόνο μία φορά. Αν θέλετε να επιτρέπετε διπλότυπα, ενεργοποιήστε αυτό στις Ρυθμίσεις → Βιβλιοθήκη → Λίστες αναπαραγωγής → Διπλότυπα σε Λίστα Αναπαραγωγής → Ενεργοποίηση.

## Εισαγωγή Λίστας Αναπαραγωγής

Το Evervideo υποστηρίζει εισαγωγή αρχείων λίστας αναπαραγωγής M3U / M3U8 / CUE, ώστε να μπορείτε να εισαγάγετε υπάρχουσες λίστες αναπαραγωγής χωρίς να τις αναδημιουργήσετε χειροκίνητα.

Πρώτα πηγαίνετε στην ενότητα Λίστες αναπαραγωγής. Στη συνέχεια πατήστε το κουμπί Περισσότερα στην επάνω δεξιά γωνία. Από το μενού, επιλέξτε Εισαγωγή Λίστας Αναπαραγωγής.

Στην επόμενη οθόνη, επιλέξτε τη θέση αρχείου. Υποστηριζόμενες επιλογές:

- Συνδεδεμένη αποθήκευση cloud
- Αρχεία στην εφαρμογή
- Αρχεία στη συσκευή σας

Επιλέξτε τη θέση, ανοίξτε τον φάκελο που περιέχει το αρχείο λίστας αναπαραγωγής, στη συνέχεια επιλέξτε το αρχείο .m3u, .m3u8 ή .cue και πατήστε Ολοκλήρωση. Το Evervideo αναλύει το αρχείο, εντοπίζει κάθε βίντεο στην αποθήκευση και δημιουργεί πραγματική λίστα αναπαραγωγής στη βιβλιοθήκη σας. Σημαντικό: το αρχείο λίστας αναπαραγωγής πρέπει να αναφέρεται σε αρχεία που υπάρχουν πραγματικά στις διαδρομές που καθορίζει.

## Οθόνη Λεπτομέρειας Λίστας Αναπαραγωγής

Όταν ανοίγετε λίστα αναπαραγωγής, εμφανίζεται η Οθόνη Λεπτομέρειας Λίστας Αναπαραγωγής. Θα βρείτε κουμπί **"..."** στην επάνω δεξιά γωνία με επιλογές λίστας αναπαραγωγής, και τρία κουμπιά κάτω από την εικόνα εξωφύλλου: **Αναζήτηση**, **Συνέχεια Αναπαραγωγής**, **Αναπαραγωγή Όλων** και **Ανακάτεμα Όλων**. Υπάρχει επίσης πλαίσιο ελέγχου **Offline Λειτουργία** για εναλλαγή πλήρους offline συγχρονισμού λίστας αναπαραγωγής.

- **Συνέχεια Αναπαραγωγής** — επαναφορά τελευταίας αποθηκευμένης θέσης αναπαραγωγής για αυτήν τη λίστα.
- **Αναζήτηση** — εκτέλεση αναζήτησης εντός της τρέχουσας λίστας αναπαραγωγής.
- **Αναπαραγωγή Όλων** — προσθήκη όλων των βίντεο από την τρέχουσα λίστα στην ουρά player.
- **Ανακάτεμα Όλων** — όπως **Αναπαραγωγή Όλων**, αλλά ανακατεύει βίντεο πριν την προσθήκη στην ουρά.
- **Offline Λειτουργία** — λήψη όλων των βίντεο από αυτή τη λίστα σε τοπικά αρχεία. Νέα στοιχεία που προστίθενται στη λίστα κατεβαίνουν επίσης αυτόματα.

## Περισσότερες Ενέργειες για Λίστα Αναπαραγωγής στην Οθόνη Λιστών

Μπορείτε να αποκτήσετε πρόσβαση σε ενέργειες για λίστα αναπαραγωγής πατώντας το κουμπί **"..."** κοντά στον τίτλο λίστας. Διαθέσιμες ενέργειες:

- **Αναπαραγωγή Όλων** — προσθήκη βίντεο λίστας σε νέα ουρά player.
- **Αναπαραγωγή Επόμενου** — προσθήκη βίντεο λίστας στην κορυφή της υπάρχουσας ουράς player.
- **Αναπαραγωγή Αργότερα** — προσθήκη βίντεο λίστας στο κάτω μέρος της υπάρχουσας ουράς player.
- **Επεξεργασία Εικόνας** — αλλαγή εξωφύλλου λίστας.
- **Ενεργοποίηση εκτός σύνδεσης λειτουργίας** — ενεργοποίηση offline λειτουργίας για τη λίστα. Τόσο υπάρχοντα όσο και νέα βίντεο κατεβαίνουν αυτόματα.
- **Εξαγωγή Λίστας Τραγουδιών** — εξαγωγή αυτής της λίστας σε **M3U / M3U8 / CSV / TXT**.
- **Προσθήκη στο Αρχείο** — αρχειοθέτηση αυτής της λίστας (συμπεριλαμβανομένου αρχείου m3u, εικόνας εξωφύλλου και όλων των βίντεο) σε ένα ZIP (Premium).
- **Προσθήκη Βίντεο** — προσθήκη περισσότερων βίντεο στην τρέχουσα λίστα.
- **Μετονομασία** — μετονομασία λίστας.
- **Διαγραφή Λίστας Αναπαραγωγής** — διαγραφή λίστας από τη βιβλιοθήκη. **Αυτή η ενέργεια δεν μπορεί να αναιρεθεί.**

## Περισσότερες Ενέργειες για Λίστα Αναπαραγωγής στην Οθόνη Λεπτομέρειας

Μπορείτε να αποκτήσετε πρόσβαση σε ενέργειες πατώντας το κουμπί **"..."** στην επάνω δεξιά γωνία της Οθόνης Λεπτομέρειας. Διαθέσιμες ενέργειες:

- **Επιλογή** — ενεργοποίηση λειτουργίας επιλογής βίντεο, χρήσιμη για διαγραφή πολλαπλών βίντεο ή αλλαγή σειράς.
- **Αναπαραγωγή Επόμενου / Αναπαραγωγή Αργότερα** — προσθήκη βίντεο λίστας στην ουρά.
- **Προσθήκη Βίντεο** — προσθήκη νέων βίντεο στη λίστα.
- **Αναδιάταξη Βίντεο** — χειροκίνητη αλλαγή σειράς βίντεο με drag-and-drop.
- **Μετονομασία** — μετονομασία τρέχουσας λίστας.
- **Επεξεργασία Εικόνας** — αλλαγή εξωφύλλου.
- **Εξαγωγή Λίστας Τραγουδιών** — εξαγωγή λίστας σε M3U / CSV / TXT.
- **Προσθήκη στο Αρχείο** — συμπίεση τρέχουσας λίστας συμπεριλαμβανομένων όλων των βίντεο και αρχείου m3u (Premium).
- **Ταξινόμηση** — αλλαγή σειράς βίντεο στη λίστα. Επιλογές ταξινόμησης περιλαμβάνουν **Τίτλο, Άλμπουμ, Είδος, Αξιολόγηση, Έτος, Διάρκεια, Όνομα Αρχείου, Ημερομηνία Τροποποίησης Αρχείου, Ημερομηνία Δημιουργίας Αρχείου** και **Χειροκίνητα**.
- **Αναζήτηση** — αναζήτηση συγκεκριμένου βίντεο εντός τρέχουσας λίστας.
- **Πλέγμα / Λίστα** — αλλαγή παρουσίασης διάταξης οθόνης.
- **Διαγραφή Λίστας Αναπαραγωγής** — διαγραφή λίστας από βιβλιοθήκη. Αυτή η ενέργεια δεν διαγράφει βίντεο από αποθήκευσή σας, αλλά **δεν μπορεί να αναιρεθεί**.

## Αλλαγή Σειράς Αρχείων σε Λίστα Αναπαραγωγής

Για αλλαγή σειράς βίντεο σε λίστα αναπαραγωγής, πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία και επιλέξτε **Επιλογή** για λειτουργία επιλογής. Χρησιμοποιήστε τον έλεγχο αναδιάταξης και χειρονομίες drag-and-drop κοντά σε κάθε βίντεο για μετακίνηση πάνω ή κάτω. Πατώντας τον έλεγχο αναδιάταξης μετακινεί το βίντεο στην κορυφή της λίστας. Για έξοδο από λειτουργία επιλογής και εφαρμογή αλλαγών, πατήστε **Ολοκλήρωση**.

## Αλλαγή Εικόνας Εξωφύλλου Λίστας Αναπαραγωγής

Για αλλαγή εικόνας εξωφύλλου λίστας, πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία και επιλέξτε **Επεξεργασία Εικόνας**. Επιλέξτε εικόνα από τις διαθέσιμες πηγές (Φωτογραφίες, Αρχεία, αποθήκευση cloud ή εικόνα που δημιουργήθηκε από βίντεο στη λίστα), στη συνέχεια επιβεβαιώστε πατώντας **Ολοκλήρωση**.

## Προσθήκη Βίντεο σε Λίστα Αναπαραγωγής

Ανοίξτε τη λίστα και πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία, στη συνέχεια επιλέξτε **Προσθήκη Βίντεο** για να ανοίξετε διάλογο. Επιλέξτε τα βίντεο που θέλετε να προσθέσετε και επιβεβαιώστε πατώντας **Ολοκλήρωση**.

## Διαγραφή Πολλαπλών Βίντεο από Λίστα Αναπαραγωγής

Ανοίξτε τη λίστα, πατήστε το κουμπί **"..."** στην επάνω δεξιά γωνία, και επιλέξτε **Επιλογή** για λειτουργία επιλογής. Επιλέξτε τα βίντεο που θέλετε να διαγράψετε και πατήστε **Διαγραφή από Λίστα Αναπαραγωγής** στο κάτω μέρος. Επιβεβαιώστε πατώντας **Ολοκλήρωση**.

## Επιλογές Βίντεο

Κάθε βίντεο σε λίστα έχει κατάλογο ενεργειών, που είναι προσβάσιμος πατώντας **"..."**. Αν δεν βλέπετε όλες τις ενέργειες, κυλήστε για να τις δείτε. Μπορείτε να διαγράψετε βίντεο από λίστα, να το κατεβάσετε, να επεξεργαστείτε μεταδεδομένα και άλλα.

- **Αναπαραγωγή Επόμενου** — προσθήκη βίντεο στην κορυφή ουράς player.
- **Αναπαραγωγή Αργότερα** — προσάρτηση βίντεο στο κάτω μέρος ουράς player.
- **Προσθήκη σε Λίστα Αναπαραγωγής** — προσθήκη βίντεο σε άλλη λίστα.
- **Προσθήκη στα Αγαπημένα** — επισήμανση βίντεο ως αγαπημένο.
- **Λήψη** — κάνει το βίντεο διαθέσιμο offline.
- **Επεξεργασία Ετικετών** — άνοιγμα ενσωματωμένου επεξεργαστή ετικετών για αλλαγή μεταδεδομένων.
- **Άνοιγμα Σε** — εξαγωγή βίντεο και άνοιγμα σε άλλη εφαρμογή.
- **Εμφάνιση σε Φάκελο** — αποκάλυψη φακέλου όπου βρίσκεται το αρχείο.
- **Εμφάνιση στο Finder** — για αρχεία εισαγμένα από Mac, αποκαλύπτει τον φάκελο στο Mac.
- **Διαγραφή από Λίστα Αναπαραγωγής** — αφαίρεση βίντεο από λίστα.
- **Διαγραφή από Υπηρεσία Cloud** — διαγραφή βίντεο από λίστα και υπηρεσία cloud. **Μη αναστρέψιμο.**
- **Διαγραφή από Βιβλιοθήκη Πολυμέσων** — διαγραφή βίντεο από βιβλιοθήκη, αφήνοντας αρχείο στην αποθήκευση ανέπαφο.

## Προσβασιμότητα

Το Evervideo είναι πλήρως προσβάσιμο με **VoiceOver**. Κάθε στοιχείο έχει καλά σχεδιασμένη ετικέτα και περιγραφή. Όταν το VoiceOver είναι ενεργό, η εφαρμογή μεταβαίνει στη **Λειτουργία Κειμένου** ώστε να εμφανίζονται μόνο σημαντικά στοιχεία — βελτιώνοντας ταχύτητα πλοήγησης και σαφήνεια. Μπορείτε επίσης να ενεργοποιήσετε Λειτουργία Κειμένου στις **Ρυθμίσεις → Προσβασιμότητα → Λειτουργία Κειμένου**.

### Ρύθμιση Θέσης Βίντεο σε Λίστα Αναπαραγωγής με VoiceOver

1. Ανοίξτε λίστα αναπαραγωγής και πατήστε το κουμπί **Περισσότερα**.
2. Επιλέξτε **Αναδιάταξη Βίντεο**. Η προβολή μεταβαίνει σε λειτουργία επεξεργασίας.
3. Πατήστε εικονίδιο δείκτη αναδιάταξης κοντά στον τίτλο βίντεο για να του δώσετε εστίαση.
4. **Διπλοπατήστε** γρήγορα το εικονίδιο δείκτη αναδιάταξης. Στο δεύτερο πάτημα, μην αφήσετε το δάχτυλό σας — κρατήστε το μέχρι να ακούσετε ήχο που υποδηλώνει ότι κελί είναι έτοιμο να μετακινηθεί.
5. Μετακινήστε το κελί στη νέα θέση.

Άλλα στοιχεία λειτουργούν όπως αναμένεται, χρησιμοποιώντας παρεχόμενα από το σύστημα μοτίβα VoiceOver.

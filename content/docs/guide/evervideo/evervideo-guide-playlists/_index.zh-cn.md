在播放列表部分，您会找到管理视频合集的实用工具——非常适合系列、课程、教程序列、音乐视频和精心策划的马拉松观影。此区域在一处显示您所有的播放列表。顶部有一个「...」按钮在导航栏上，打开带有播放列表选项的菜单，还有一个工具栏，提供搜索、播放全部和随机播放等快速操作。每个播放列表的标题旁边也有自己的「...」按钮，提供专属于该播放列表的更多选项。

Evervideo 中的播放列表可以包含在线云端视频、离线下载文件、本地文件、Photos 库视频和 iOS Music 库视频的混合——全部在一个播放列表中——并无缝一起播放。

## 创建播放列表

创建新播放列表非常简单。您有两个选项：点击 **+** 按钮，或点击导航栏右上角的 **「...」** 按钮并选择**新建播放列表**。给播放列表起一个有意义的名称，然后点击**保存**。

这将触发**添加文件**对话框，您可以在其中手动选择要包含的视频。视频按来源类型方便地分类：

- **资料库** — 已在媒体资料库中的文件。
- **此应用中的文件** — 应用 Documents 文件夹中的视频。
- **此设备上的文件** — 不在此应用中的设备其他位置的视频。
- **连接** — 已连接云存储服务和媒体服务器中的在线视频。

默认情况下，每个视频只能添加到播放列表一次。如果您想允许重复，请在设置 → 资料库 → 播放列表 → 播放列表中的重复 → 启用中启用此功能。

## 导入播放列表

Evervideo 支持 M3U / M3U8 / CUE 播放列表文件导入，因此您可以在不手动重新创建的情况下引入现有播放列表。

首先，转到播放列表部分。然后，点击右上角的更多按钮。从菜单中选择导入播放列表。

在下一个屏幕上，选择文件位置。支持的选项：

- 已连接的云存储
- 应用中的文件
- 设备上的文件

选择位置，打开包含播放列表文件的文件夹，然后选择 .m3u、.m3u8 或 .cue 文件并点击完成。Evervideo 解析文件，在存储上定位每个引用的视频，并在您的资料库中创建真实的播放列表。重要提示：播放列表文件必须引用实际存在于其指定路径的文件。

## 播放列表详情屏幕

当您打开播放列表时，会出现播放列表详情屏幕。您会在右上角找到带有播放列表选项的 **「...」** 按钮，以及封面图片下面的三个按钮：**搜索**、**继续播放**、**播放全部**和**随机播放**。还有一个**离线模式**复选框来切换全播放列表离线同步。

- **继续播放** — 恢复此播放列表的最后保存播放位置。
- **搜索** — 在当前播放列表中执行搜索。
- **播放全部** — 将当前播放列表中的所有视频添加到播放器队列。
- **随机播放** — 与**播放全部**类似，但在将视频添加到播放器队列前先随机排序。
- **离线模式** — 将此播放列表中的所有视频下载到本地文件。新增到播放列表的项目也会自动下载。

## 播放列表屏幕中播放列表的更多操作

您可以通过点击播放列表标题旁边的 **「...」** 按钮访问播放列表操作。可用操作：

- **播放全部** — 将播放列表视频添加到新的播放器队列。
- **播放下一个** — 将播放列表视频添加到现有播放器队列的顶部。
- **稍后播放** — 将播放列表视频添加到现有播放器队列的底部。
- **编辑图片** — 更改播放列表的封面图片。
- **启用离线模式** — 为播放列表开启离线模式。现有和新视频都会自动下载。
- **导出歌曲列表** — 将此播放列表导出为 **M3U / M3U8 / CSV / TXT**。
- **添加到归档** — 将此播放列表（包括 m3u 文件、封面图片和所有视频）打包成单个 ZIP（Premium）。
- **添加视频** — 向当前播放列表添加更多视频。
- **重命名** — 重命名播放列表。
- **删除播放列表** — 从媒体资料库中删除播放列表。**此操作无法撤销。**

## 播放列表详情屏幕中播放列表的更多操作

您可以通过点击播放列表详情屏幕右上角的 **「...」** 按钮访问操作。可用操作：

- **选择** — 激活视频选择模式，用于删除多个视频或更改其顺序。
- **播放下一个 / 稍后播放** — 将播放列表视频添加到队列。
- **添加视频** — 向播放列表添加新视频。
- **重新排列视频** — 使用拖放手动更改视频顺序。
- **重命名** — 重命名当前播放列表。
- **编辑图片** — 更改封面图片。
- **导出歌曲列表** — 将播放列表导出为 M3U / CSV / TXT。
- **添加到归档** — 将当前播放列表包括所有视频和 m3u 文件压缩为 ZIP（Premium）。
- **排序** — 更改播放列表中视频的顺序。排序选项包括**标题、专辑、流派、评分、年份、时长、文件名、文件修改日期、文件创建日期**和**手动**。
- **搜索** — 在当前播放列表中搜索特定视频。
- **网格 / 列表** — 更改屏幕布局展示。
- **删除播放列表** — 从媒体资料库中删除播放列表。此操作不会从存储中删除视频，但**无法撤销**。

## 更改播放列表中的文件顺序

要更改播放列表中视频的顺序，点击右上角的 **「...」** 按钮，选择**选择**进入选择模式。使用每个视频旁边的重新排序控件和拖放手势上下移动它们。点击重新排序控件会将视频移到列表顶部。要退出选择模式并应用更改，点击**完成**。

## 更改播放列表封面图片

要更改播放列表的封面图片，点击右上角的 **「...」** 按钮，选择**编辑图片**。从可用来源（Photos、Files、云存储或从播放列表中视频生成的图稿）选择图片，然后点击**完成**确认。

## 向播放列表添加视频

打开播放列表，点击右上角的 **「...」** 按钮，然后选择**添加视频**打开对话框。选择您要添加的视频并点击**完成**确认。

## 从播放列表中删除多个视频

打开播放列表，点击右上角的 **「...」** 按钮，选择**选择**进入选择模式。选择您要删除的视频，点击屏幕底部的**从播放列表中删除**。点击**完成**确认。

## 视频选项

播放列表中的每个视频都有一个操作列表，通过点击 **「...」** 按钮访问。如果您看不到所有操作，请向下滚动查看。您可以从播放列表中删除视频、下载视频、编辑元数据等。

- **播放下一个** — 将视频添加到播放器队列顶部。
- **稍后播放** — 将视频追加到播放器队列底部。
- **添加到播放列表** — 将视频添加到另一个播放列表。
- **添加到收藏夹** — 将视频标记为收藏。
- **下载** — 使视频可离线使用。
- **编辑标签** — 打开内置标签编辑器更改元数据。
- **在...中打开** — 导出视频并在另一个应用中打开。
- **在文件夹中显示** — 显示文件所在的文件夹。
- **在 Finder 中显示** — 对于从 Mac 导入的文件，显示 Mac 上的文件夹。
- **从播放列表中删除** — 从播放列表中删除视频。
- **从云服务中删除** — 从播放列表和云服务中删除视频。**不可逆。**
- **从媒体资料库中删除** — 从资料库中删除视频，但文件在存储上保持不变。

## 无障碍功能

Evervideo 完全支持 **VoiceOver**。每个组件都有精心设计的标签和描述。VoiceOver 激活时，应用切换到**文字模式**，只显示有意义的元素——提升导航速度和清晰度。您也可以在**设置 → 无障碍 → 文字模式**中激活文字模式。

### 使用 VoiceOver 调整播放列表中视频位置

1. 打开播放列表并点击**更多**按钮。
2. 选择**重新排列视频**。视图切换到编辑模式。
3. 点击视频标题旁边的重新排序指示器图标以聚焦。
4. 快速**双击**重新排序指示器图标。第二次点击时不要松开手指——按住直到听到声音表示单元格可以移动。
5. 将单元格移动到新位置。

其他组件按预期工作，使用系统提供的 VoiceOver 模式。

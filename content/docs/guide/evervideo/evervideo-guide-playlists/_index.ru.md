В разделе «Плейлисты» вы найдёте удобные инструменты для управления видеоколлекциями — идеально для сериалов, курсов, последовательностей обучающих видео, клипов и тщательно подобранных марафонов. В этом разделе все ваши плейлисты отображаются в одном месте. Вверху есть кнопка «...» на навигационной панели для открытия меню с опциями плейлистов, а также панель инструментов с быстрыми действиями: «Поиск», «Воспроизвести все» и «Перемешать все». У каждого плейлиста есть своя кнопка «...» рядом с названием для дополнительных опций.

Плейлисты в Evervideo могут содержать смесь онлайн-видео из облака, скачанных офлайн-файлов, локальных файлов, видео из библиотеки «Фото» и видео из библиотеки iOS «Музыка» — всё в одном плейлисте — и воспроизводиться без прерываний.

## Создание плейлиста

Создать новый плейлист просто. Есть два варианта: нажмите кнопку **+** или нажмите кнопку **«...»** в правом верхнем углу навигационной панели и выберите **Новый плейлист**. Дайте плейлисту понятное имя, затем нажмите **Сохранить**.

Откроется диалог **Добавить файлы**, где можно вручную выбрать видео для включения. Видео удобно разделены по типу источника:

- **Библиотека** — файлы, уже находящиеся в медиатеке.
- **Файлы в этом приложении** — видео в папке «Документы» приложения.
- **Файлы на этом устройстве** — видео в других местах на устройстве, не в этом приложении.
- **Подключения** — онлайн-видео в подключённых облачных сервисах и медиасерверах.

По умолчанию одно видео можно добавить в плейлист только один раз. Для разрешения дубликатов включите это в «Настройки → Библиотека → Плейлисты → Дубликаты в плейлисте → Включить».

## Импорт плейлиста

Evervideo поддерживает импорт файлов плейлистов M3U / M3U8 / CUE, позволяя перенести существующие плейлисты без ручного пересоздания.

Сначала перейдите в раздел «Плейлисты». Затем нажмите кнопку «Ещё» в правом верхнем углу. В меню выберите «Импортировать плейлист».

На следующем экране выберите расположение файла. Поддерживаемые варианты:

- Подключённое облачное хранилище
- Файлы в приложении
- Файлы на устройстве

Выберите расположение, откройте папку с файлом плейлиста, затем выберите файл .m3u, .m3u8 или .cue и нажмите «Готово». Evervideo анализирует файл, находит каждое указанное видео в хранилище и создаёт реальный плейлист в библиотеке. Важно: файл плейлиста должен ссылаться на файлы, которые действительно существуют по указанным путям.

## Экран деталей плейлиста

При открытии плейлиста появляется экран «Детали плейлиста». Здесь есть кнопка **«...»** в правом верхнем углу с опциями плейлиста и три кнопки под обложкой: **Поиск**, **Продолжить воспроизведение**, **Воспроизвести все** и **Перемешать все**. Также есть флажок **Офлайн-режим** для включения полной синхронизации плейлиста офлайн.

- **Продолжить воспроизведение** — восстановить последнюю сохранённую позицию воспроизведения для этого плейлиста.
- **Поиск** — выполнить поиск в текущем плейлисте.
- **Воспроизвести все** — добавить все видео из текущего плейлиста в очередь плеера.
- **Перемешать все** — как **Воспроизвести все**, но перемешивает видео перед добавлением в очередь.
- **Офлайн-режим** — скачать все видео из этого плейлиста в локальные файлы. Любые новые элементы, добавленные в плейлист, также скачиваются автоматически.

## Другие действия для плейлиста на экране «Плейлисты»

Доступ к действиям плейлиста — нажатие кнопки **«...»** рядом с названием плейлиста. Доступные действия:

- **Воспроизвести все** — добавляет видео плейлиста в новую очередь плеера.
- **Воспроизвести следующим** — добавляет видео плейлиста в начало существующей очереди.
- **Воспроизвести позже** — добавляет видео плейлиста в конец существующей очереди.
- **Редактировать обложку** — изменить обложку плейлиста.
- **Включить офлайн режим** — включить офлайн-режим для плейлиста. Как существующие, так и новые видео скачиваются автоматически.
- **Экспортировать список песен** — экспортировать плейлист в **M3U / M3U8 / CSV / TXT**.
- **Добавить в архив** — архивировать плейлист (включая файл m3u, обложку и все видео) в один ZIP (Premium).
- **Добавить видео** — добавить больше видео в текущий плейлист.
- **Переименовать** — переименовать плейлист.
- **Удалить плейлист** — удалить плейлист из медиатеки. **Это действие нельзя отменить.**

## Другие действия для плейлиста на экране «Детали плейлиста»

Доступ к действиям — нажатие кнопки **«...»** в правом верхнем углу экрана «Детали плейлиста». Доступные действия:

- **Выбрать** — активировать режим выбора видео, удобный для удаления нескольких видео или изменения их порядка.
- **Воспроизвести следующим / Воспроизвести позже** — добавить видео плейлиста в очередь.
- **Добавить видео** — добавить новые видео в плейлист.
- **Изменить порядок видео** — вручную изменить порядок видео с помощью перетаскивания.
- **Переименовать** — переименовать текущий плейлист.
- **Редактировать обложку** — изменить обложку.
- **Экспортировать список песен** — экспортировать плейлист в M3U / CSV / TXT.
- **Добавить в архив** — упаковать текущий плейлист со всеми видео и файлом m3u в ZIP (Premium).
- **Сортировка** — изменить порядок видео в плейлисте. Варианты сортировки: **Заголовок, Альбом, Жанр, Рейтинг, Год, Длительность, Имя файла, Дата изменения файла, Дата создания файла** и **Вручную**.
- **Поиск** — поиск конкретного видео в текущем плейлисте.
- **Сетка / Список** — изменить способ отображения экрана.
- **Удалить плейлист** — удалить плейлист из медиатеки. Это действие не удаляет видео из хранилища, но **его нельзя отменить**.

## Изменение порядка файлов в плейлисте

Для изменения порядка видео в плейлисте нажмите кнопку **«...»** в правом верхнем углу и выберите **Выбрать** для входа в режим выбора. Используйте элемент управления изменением порядка и жесты перетаскивания рядом с каждым видео для перемещения вверх или вниз. Нажатие элемента управления изменением порядка перемещает видео в начало списка. Для выхода из режима выбора и применения изменений нажмите **Готово**.

## Изменение обложки плейлиста

Для изменения обложки плейлиста нажмите кнопку **«...»** в правом верхнем углу и выберите **Редактировать обложку**. Выберите изображение из доступных источников (Фото, Файлы, облачное хранилище или сгенерированная обложка из видео в плейлисте), затем подтвердите, нажав **Готово**.

## Добавление видео в плейлист

Откройте плейлист и нажмите кнопку **«...»** в правом верхнем углу, затем выберите **Добавить видео** для открытия диалога. Выберите видео для добавления и подтвердите, нажав **Готово**.

## Удаление нескольких видео из плейлиста

Откройте плейлист, нажмите кнопку **«...»** в правом верхнем углу и выберите **Выбрать** для входа в режим выбора. Выберите видео для удаления и нажмите **Удалить из плейлиста** внизу экрана. Подтвердите, нажав **Готово**.

## Опции видео

У каждого видео в плейлисте есть список действий, доступных при нажатии кнопки **«...»**. Если не все действия видны, прокрутите вниз. Можно удалить видео из плейлиста, скачать его, редактировать метаданные и другое.

- **Воспроизвести следующим** — добавляет видео в начало очереди плеера.
- **Воспроизвести позже** — добавляет видео в конец очереди плеера.
- **Добавить в плейлист** — добавляет видео в другой плейлист.
- **Добавить в избранное** — отмечает видео как избранное.
- **Скачать** — делает видео доступным офлайн.
- **Редактировать теги** — открывает встроенный редактор тегов для изменения метаданных.
- **Открыть в** — экспортирует видео и открывает его в другом приложении.
- **Показать в папке** — открывает папку, где находится файл.
- **Показать в Finder** — для файлов, импортированных с Mac, открывает папку на Mac.
- **Удалить из плейлиста** — удаляет видео из плейлиста.
- **Удалить из облачного сервиса** — удаляет видео из плейлиста и из облачного сервиса. **Необратимо.**
- **Удалить из медиатеки** — удаляет видео из библиотеки, оставляя файл в хранилище нетронутым.

## Доступность

Evervideo полностью доступна с **VoiceOver**. Каждый компонент имеет хорошо продуманную метку и описание. Когда VoiceOver активен, приложение переключается в **Текстовый режим**, чтобы отображались только значимые элементы — улучшая скорость навигации и читаемость. Также можно активировать «Текстовый режим» в **Настройки → Доступность → Текстовый режим**.

### Настройка позиции видео в плейлисте с VoiceOver

1. Откройте плейлист и нажмите кнопку **«Ещё»**.
2. Выберите **Изменить порядок видео**. Представление переключается в режим редактирования.
3. Нажмите на иконку индикатора изменения порядка рядом с названием видео, чтобы дать ему фокус.
4. **Дважды быстро нажмите** на иконку индикатора изменения порядка. При втором касании не отпускайте палец — удерживайте до звукового сигнала, означающего готовность ячейки к перемещению.
5. Переместите ячейку на новую позицию.

Другие компоненты работают как ожидается, используя паттерны VoiceOver, предоставляемые системой.

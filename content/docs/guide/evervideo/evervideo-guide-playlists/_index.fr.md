Dans la section Listes de lecture, vous trouverez des outils utiles pour gérer vos collections vidéo — parfaites pour les séries, les cours, les séquences de tutoriels, les clips musicaux et les marathons organisés. Cette zone affiche toutes vos listes de lecture en un seul endroit. En haut, il y a un bouton « ... » dans la barre de navigation qui ouvre un menu avec des options de liste de lecture, plus une barre d'outils avec des actions rapides comme Rechercher, Tout lire et Tout mélanger. Chaque liste de lecture a également son propre bouton « ... » à côté de son titre, vous donnant plus d'options juste pour cette liste de lecture.

Les listes de lecture dans Evervideo peuvent contenir un mélange de vidéos cloud en ligne, de fichiers téléchargés hors ligne, de fichiers locaux, de vidéos de la bibliothèque Photos et de vidéos de la bibliothèque iOS Music — toutes dans une seule liste de lecture — et se lisent parfaitement ensemble.

## Créer une liste de lecture

Créer une nouvelle liste de lecture est simple. Vous avez deux options : appuyez sur le bouton **+**, ou appuyez sur le bouton **« ... »** dans le coin supérieur droit de la barre de navigation et choisissez **Nouvelle liste de lecture**. Donnez un nom significatif à votre liste de lecture, puis appuyez sur **Enregistrer**.

Cela déclenche la boîte de dialogue **Ajouter des fichiers**, où vous pouvez sélectionner manuellement les vidéos que vous souhaitez inclure. Les vidéos sont commodément classées par type de source :

- **Bibliothèque** — fichiers déjà dans votre médiathèque.
- **Fichiers dans cette application** — vidéos dans le dossier Documents de l'application.
- **Fichiers sur cet appareil** — vidéos situées ailleurs sur votre appareil, pas dans cette application.
- **Connexions** — vidéos en ligne situées dans vos services de stockage cloud et serveurs multimédias connectés.

Par défaut, vous ne pouvez ajouter une seule vidéo à une liste de lecture qu'une seule fois. Si vous souhaitez autoriser les doublons, activez-le dans Paramètres → Bibliothèque → Listes de lecture → Doublons dans une liste de lecture → Activer.

## Importer une liste de lecture

Evervideo prend en charge l'importation de fichiers de liste de lecture M3U / M3U8 / CUE, vous permettant d'apporter des listes de lecture existantes sans les recréer manuellement.

Commencez par aller dans la section Listes de lecture. Ensuite, appuyez sur le bouton Plus dans le coin supérieur droit. Dans le menu, sélectionnez Importer une liste de lecture.

Sur l'écran suivant, choisissez l'emplacement du fichier. Options prises en charge :

- Stockage cloud connecté
- Fichiers dans l'application
- Fichiers sur votre appareil

Sélectionnez l'emplacement, ouvrez le dossier contenant le fichier de liste de lecture, puis choisissez le fichier .m3u, .m3u8, ou .cue et appuyez sur Fait. Evervideo analyse le fichier, localise chaque vidéo référencée sur le stockage, et crée une vraie liste de lecture dans votre bibliothèque. Important : le fichier de liste de lecture doit référencer des fichiers qui existent réellement aux chemins spécifiés.

## Écran de détail de la liste de lecture

Lorsque vous ouvrez une liste de lecture, l'écran de détail de la liste de lecture apparaît. Vous trouverez un bouton **« ... »** dans le coin supérieur droit avec les options de liste de lecture, et trois boutons sous l'image de couverture : **Rechercher**, **Continuer la lecture**, **Tout lire** et **Tout mélanger**. Il y a également une case à cocher **Mode hors ligne** pour basculer la synchronisation hors ligne de toute la liste de lecture.

- **Continuer la lecture** — restaurer la dernière position de lecture sauvegardée pour cette liste de lecture.
- **Rechercher** — effectuer une recherche dans la liste de lecture actuelle.
- **Tout lire** — ajouter toutes les vidéos de la liste de lecture actuelle à la file du lecteur.
- **Tout mélanger** — comme **Tout lire**, mais mélange les vidéos avant de les ajouter à la file du lecteur.
- **Mode hors ligne** — télécharger toutes les vidéos de cette liste de lecture dans les fichiers locaux. Les nouveaux éléments ajoutés à la liste de lecture sont également téléchargés automatiquement.

## Plus d'actions pour une liste de lecture dans l'écran Listes de lecture

Vous pouvez accéder aux actions d'une liste de lecture en appuyant sur le bouton **« ... »** près du titre de la liste de lecture. Actions disponibles :

- **Tout lire** — ajoute les vidéos de la liste de lecture à une nouvelle file du lecteur.
- **Lire ensuite** — ajoute les vidéos de la liste de lecture en haut de la file du lecteur existante.
- **Lire plus tard** — ajoute les vidéos de la liste de lecture en bas de la file du lecteur existante.
- **Modifier l'image** — changer la pochette de la liste de lecture.
- **Activer le mode hors ligne** — activer le mode hors ligne pour la liste de lecture. Les vidéos existantes et nouvelles sont téléchargées automatiquement.
- **Exporter la liste de chansons** — exporter cette liste de lecture vers **M3U / M3U8 / CSV / TXT**.
- **Ajouter à l'archive** — archiver cette liste de lecture (y compris le fichier m3u, la couverture et toutes les vidéos) dans un seul ZIP (Premium).
- **Ajouter des vidéos** — ajouter plus de vidéos à la liste de lecture actuelle.
- **Renommer** — renommer la liste de lecture.
- **Supprimer la liste de lecture** — supprimer la liste de lecture de la médiathèque. **Cette action ne peut pas être annulée.**

## Plus d'actions pour une liste de lecture dans l'écran de détail

Vous pouvez accéder aux actions en appuyant sur le bouton **« ... »** dans le coin supérieur droit de l'écran de détail de la liste de lecture. Actions disponibles :

- **Sélectionner** — activer le mode de sélection de vidéos, utile pour supprimer plusieurs vidéos ou modifier leur ordre.
- **Lire ensuite / Lire plus tard** — ajouter les vidéos de la liste de lecture à la file.
- **Ajouter des vidéos** — ajouter de nouvelles vidéos à la liste de lecture.
- **Réorganiser les vidéos** — modifier manuellement l'ordre des vidéos avec glisser-déposer.
- **Renommer** — renommer la liste de lecture actuelle.
- **Modifier l'image** — changer la couverture.
- **Exporter la liste de chansons** — exporter la liste de lecture vers M3U / CSV / TXT.
- **Ajouter à l'archive** — compresser la liste de lecture actuelle incluant toutes les vidéos et le fichier m3u (Premium).
- **Trier** — changer l'ordre des vidéos dans la liste de lecture. Les options de tri incluent **Titre, Album, Genre, Note, Année, Durée, Nom de fichier, Date de modification de fichier, Date de création de fichier** et **Manuel**.
- **Rechercher** — rechercher une vidéo spécifique dans la liste de lecture actuelle.
- **Grille / Liste** — changer la présentation de la mise en page.
- **Supprimer la liste de lecture** — supprimer la liste de lecture de la médiathèque. Cette action ne supprime pas les vidéos de votre stockage, mais **elle ne peut pas être annulée**.

## Changer l'ordre des fichiers dans une liste de lecture

Pour changer l'ordre des vidéos dans une liste de lecture, appuyez sur le bouton **« ... »** dans le coin supérieur droit et sélectionnez **Sélectionner** pour entrer en mode sélection. Utilisez le contrôle de réorganisation et les gestes de glisser-déposer près de chaque vidéo pour les déplacer vers le haut ou le bas. Appuyer sur le contrôle de réorganisation déplace la vidéo en haut de la liste. Pour quitter le mode sélection et appliquer les modifications, appuyez sur **Fait**.

## Changer l'image de couverture de la liste de lecture

Pour changer l'image de couverture d'une liste de lecture, appuyez sur le bouton **« ... »** dans le coin supérieur droit et sélectionnez **Modifier l'image**. Choisissez une image parmi les sources disponibles (Photos, Fichiers, stockage cloud ou une illustration générée à partir d'une vidéo dans la liste de lecture), puis confirmez en appuyant sur **Fait**.

## Ajouter des vidéos à une liste de lecture

Ouvrez la liste de lecture et appuyez sur le bouton **« ... »** dans le coin supérieur droit, puis sélectionnez **Ajouter des vidéos** pour ouvrir une boîte de dialogue. Choisissez les vidéos que vous souhaitez ajouter et confirmez en appuyant sur **Fait**.

## Supprimer plusieurs vidéos d'une liste de lecture

Ouvrez la liste de lecture, appuyez sur le bouton **« ... »** dans le coin supérieur droit, et sélectionnez **Sélectionner** pour entrer en mode sélection. Choisissez les vidéos que vous souhaitez supprimer et appuyez sur **Supprimer de la liste de lecture** en bas de l'écran. Confirmez en appuyant sur **Fait**.

## Options vidéo

Chaque vidéo dans une liste de lecture a une liste d'actions, accessible en appuyant sur le bouton **« ... »**. Si vous ne voyez pas toutes les actions, faites défiler vers le bas pour les voir. Vous pouvez supprimer la vidéo de la liste de lecture, la télécharger, modifier les métadonnées, et plus encore.

- **Lire ensuite** — ajoute la vidéo en haut de la file du lecteur.
- **Lire plus tard** — ajoute la vidéo en bas de la file du lecteur.
- **Ajouter à une liste de lecture** — ajoute la vidéo à une autre liste de lecture.
- **Ajouter aux favoris** — marque la vidéo comme favorite.
- **Télécharger** — rend la vidéo disponible hors ligne.
- **Modifier les tags** — ouvre l'éditeur de tags intégré pour modifier les métadonnées.
- **Ouvrir dans** — exporte la vidéo et l'ouvre dans une autre application.
- **Afficher dans le dossier** — révèle le dossier où se trouve le fichier.
- **Afficher dans Finder** — pour les fichiers importés depuis votre Mac, révèle le dossier sur votre Mac.
- **Supprimer de la liste de lecture** — supprime la vidéo de la liste de lecture.
- **Supprimer du service cloud** — supprime la vidéo de la liste de lecture et du service cloud. **Irréversible.**
- **Supprimer de la médiathèque** — supprime la vidéo de la bibliothèque, laissant le fichier sur le stockage intouché.

## Accessibilité

Evervideo est entièrement accessible avec **VoiceOver**. Chaque composant a une étiquette et une description bien conçues. Lorsque VoiceOver est actif, l'application passe en **Mode texte** afin que seuls les éléments significatifs soient affichés — améliorant la vitesse de navigation et la clarté. Vous pouvez également activer le Mode texte dans **Paramètres → Accessibilité → Mode texte**.

### Ajuster la position vidéo dans une liste de lecture avec VoiceOver

1. Ouvrez une liste de lecture et appuyez sur le bouton **Plus**.
2. Sélectionnez **Réorganiser les vidéos**. La vue passe en mode d'édition.
3. Appuyez sur l'icône de l'indicateur de réorganisation près du titre de la vidéo pour lui donner le focus.
4. **Double-appuyez** rapidement sur l'icône de l'indicateur de réorganisation. Au second tap, ne relâchez pas votre doigt — maintenez-le jusqu'à entendre un son indiquant que la cellule est prête à être déplacée.
5. Déplacez la cellule vers sa nouvelle position.

Les autres composants fonctionnent comme prévu, en utilisant les modèles VoiceOver fournis par le système.

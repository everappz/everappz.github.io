在播放清單部分，您會找到管理視訊集合的實用工具——非常適合影集、課程、教學序列、音樂視訊和精心策劃的馬拉松觀影。此區域在一處顯示您所有的播放清單。頂部有一個「...」按鈕在導覽列上，開啟帶有播放清單選項的選單，還有一個工具列，提供搜尋、播放全部和隨機播放等快速操作。每個播放清單的標題旁邊也有自己的「...」按鈕，提供專屬於該播放清單的更多選項。

Evervideo 中的播放清單可以包含線上雲端視訊、離線下載檔案、本機檔案、Photos 資料庫視訊和 iOS Music 資料庫視訊的混合——全部在一個播放清單中——並無縫一起播放。

## 建立播放清單

建立新播放清單非常簡單。您有兩個選項：點選 **+** 按鈕，或點選導覽列右上角的 **「...」** 按鈕並選擇**新建播放清單**。給播放清單起一個有意義的名稱，然後點選**儲存**。

這將觸發**新增檔案**對話框，您可以在其中手動選擇要包含的視訊。視訊按來源類型方便地分類：

- **資料庫** — 已在媒體資料庫中的檔案。
- **此應用程式中的檔案** — 應用程式 Documents 資料夾中的視訊。
- **此裝置上的檔案** — 不在此應用程式中的裝置其他位置的視訊。
- **連接** — 已連線雲端儲存服務和媒體伺服器中的線上視訊。

預設情況下，每個視訊只能新增到播放清單一次。如果您想允許重複，請在設定 → 資料庫 → 播放清單 → 播放清單中的重複 → 啟用中啟用此功能。

## 匯入播放清單

Evervideo 支援 M3U / M3U8 / CUE 播放清單檔案匯入，因此您可以在不手動重新建立的情況下引入現有播放清單。

首先，前往播放清單部分。然後，點選右上角的更多按鈕。從選單中選擇匯入播放清單。

在下一個畫面上，選擇檔案位置。支援的選項：

- 已連線的雲端儲存
- 應用程式中的檔案
- 裝置上的檔案

選擇位置，開啟包含播放清單檔案的資料夾，然後選擇 .m3u、.m3u8 或 .cue 檔案並點選完成。Evervideo 解析檔案，在儲存空間上定位每個引用的視訊，並在您的資料庫中建立真實的播放清單。重要提示：播放清單檔案必須引用實際存在於其指定路徑的檔案。

## 播放清單詳情畫面

當您開啟播放清單時，會出現播放清單詳情畫面。您會在右上角找到帶有播放清單選項的 **「...」** 按鈕，以及封面圖片下面的三個按鈕：**搜尋**、**繼續播放**、**播放全部**和**隨機播放**。還有一個**離線模式**核取方塊來切換全播放清單離線同步。

- **繼續播放** — 恢復此播放清單的最後儲存播放位置。
- **搜尋** — 在目前播放清單中執行搜尋。
- **播放全部** — 將目前播放清單中的所有視訊新增到播放器佇列。
- **隨機播放** — 與**播放全部**類似，但在將視訊新增到播放器佇列前先隨機排序。
- **離線模式** — 將此播放清單中的所有視訊下載到本機檔案。新增到播放清單的項目也會自動下載。

## 播放清單畫面中播放清單的更多操作

您可以透過點選播放清單標題旁邊的 **「...」** 按鈕存取播放清單操作。可用操作：

- **播放全部** — 將播放清單視訊新增到新的播放器佇列。
- **播放下一個** — 將播放清單視訊新增到現有播放器佇列的頂部。
- **稍後播放** — 將播放清單視訊新增到現有播放器佇列的底部。
- **編輯圖片** — 更改播放清單的封面圖片。
- **啟用離線模式** — 為播放清單開啟離線模式。現有和新視訊都會自動下載。
- **匯出歌曲清單** — 將此播放清單匯出為 **M3U / M3U8 / CSV / TXT**。
- **新增到封存** — 將此播放清單（包括 m3u 檔案、封面圖片和所有視訊）打包成單個 ZIP（Premium）。
- **新增視訊** — 向目前播放清單新增更多視訊。
- **重新命名** — 重新命名播放清單。
- **刪除播放清單** — 從媒體資料庫中刪除播放清單。**此操作無法復原。**

## 播放清單詳情畫面中播放清單的更多操作

您可以透過點選播放清單詳情畫面右上角的 **「...」** 按鈕存取操作。可用操作：

- **選擇** — 啟動視訊選擇模式，用於刪除多個視訊或更改其順序。
- **播放下一個 / 稍後播放** — 將播放清單視訊新增到佇列。
- **新增視訊** — 向播放清單新增新視訊。
- **重新排列視訊** — 使用拖放手動更改視訊順序。
- **重新命名** — 重新命名目前播放清單。
- **編輯圖片** — 更改封面圖片。
- **匯出歌曲清單** — 將播放清單匯出為 M3U / CSV / TXT。
- **新增到封存** — 將目前播放清單包括所有視訊和 m3u 檔案壓縮為 ZIP（Premium）。
- **排序** — 更改播放清單中視訊的順序。排序選項包括**標題、專輯、流派、評分、年份、時長、檔案名稱、檔案修改日期、檔案建立日期**和**手動**。
- **搜尋** — 在目前播放清單中搜尋特定視訊。
- **格線 / 清單** — 更改畫面版面配置展示。
- **刪除播放清單** — 從媒體資料庫中刪除播放清單。此操作不會從儲存空間中刪除視訊，但**無法復原**。

## 更改播放清單中的檔案順序

要更改播放清單中視訊的順序，點選右上角的 **「...」** 按鈕，選擇**選擇**進入選擇模式。使用每個視訊旁邊的重新排序控制項和拖放手勢上下移動它們。點選重新排序控制項會將視訊移到清單頂部。要退出選擇模式並套用更改，點選**完成**。

## 更改播放清單封面圖片

要更改播放清單的封面圖片，點選右上角的 **「...」** 按鈕，選擇**編輯圖片**。從可用來源（Photos、Files、雲端儲存或從播放清單中視訊產生的圖稿）選擇圖片，然後點選**完成**確認。

## 向播放清單新增視訊

開啟播放清單，點選右上角的 **「...」** 按鈕，然後選擇**新增視訊**開啟對話框。選擇您要新增的視訊並點選**完成**確認。

## 從播放清單中刪除多個視訊

開啟播放清單，點選右上角的 **「...」** 按鈕，選擇**選擇**進入選擇模式。選擇您要刪除的視訊，點選畫面底部的**從播放清單中刪除**。點選**完成**確認。

## 視訊選項

播放清單中的每個視訊都有一個操作清單，透過點選 **「...」** 按鈕存取。如果您看不到所有操作，請向下捲動查看。您可以從播放清單中刪除視訊、下載視訊、編輯元資料等。

- **播放下一個** — 將視訊新增到播放器佇列頂部。
- **稍後播放** — 將視訊附加到播放器佇列底部。
- **新增到播放清單** — 將視訊新增到另一個播放清單。
- **新增到最愛** — 將視訊標記為最愛。
- **下載** — 使視訊可離線使用。
- **編輯標籤** — 開啟內建標籤編輯器更改元資料。
- **以...開啟** — 匯出視訊並在另一個應用程式中開啟。
- **在資料夾中顯示** — 顯示檔案所在的資料夾。
- **在 Finder 中顯示** — 對於從 Mac 匯入的檔案，顯示 Mac 上的資料夾。
- **從播放清單中刪除** — 從播放清單中刪除視訊。
- **從雲端服務中刪除** — 從播放清單和雲端服務中刪除視訊。**不可逆。**
- **從媒體資料庫中刪除** — 從資料庫中刪除視訊，但檔案在儲存空間上保持不變。

## 無障礙功能

Evervideo 完全支援 **VoiceOver**。每個元件都有精心設計的標籤和描述。VoiceOver 啟動時，應用程式切換到**文字模式**，只顯示有意義的元素——提升導覽速度和清晰度。您也可以在**設定 → 無障礙 → 文字模式**中啟動文字模式。

### 使用 VoiceOver 調整播放清單中視訊位置

1. 開啟播放清單並點選**更多**按鈕。
2. 選擇**重新排列視訊**。視圖切換到編輯模式。
3. 點選視訊標題旁邊的重新排序指示器圖示以聚焦。
4. 快速**雙擊**重新排序指示器圖示。第二次點選時不要放開手指——按住直到聽到聲音表示儲存格可以移動。
5. 將儲存格移動到新位置。

其他元件按預期工作，使用系統提供的 VoiceOver 模式。

โปรแกรมเล่นมีเดียคือหน้าจอหลักของแอปที่คุณควบคุมการเล่นและคุณสมบัติส่วนใหญ่ของ Evervideo โปรแกรมเล่นนี้เล่นทั้งไฟล์วิดีโอและเสียง และสร้างบนโปรแกรมเล่นแบบ FFmpeg แบบกำหนดเองพร้อมการถอดรหัส H.264 และ HEVC ด้วยฮาร์ดแวร์ที่ทำงานหนัก มาสำรวจวิธีใช้กัน

## การเข้าถึงโปรแกรมเล่นมีเดีย

คุณสามารถเข้าถึงโปรแกรมเล่นแบบเต็มหน้าจอจากแถบโปรแกรมเล่นขนาดกะทัดรัด บน iPhone โปรแกรมเล่นขนาดกะทัดรัดอยู่ที่ด้านบนของหน้าจอหลัก บน iPad และ Mac อยู่ทางด้านซ้าย (หรือด้านบนของแผงหลัก) หากต้องการยุบโปรแกรมเล่นแบบเต็มหน้าจอกลับสู่มุมมองขนาดกะทัดรัด แตะปุ่มปิดที่มุมขวาล่างหรือปัดลง หากต้องการซ่อนโปรแกรมเล่นขนาดกะทัดรัดโดยสมบูรณ์ แตะและปัดลงอีกครั้ง

โปรแกรมเล่นขนาดกะทัดรัดยังคงมองเห็นได้ขณะเรียกดูไลบรารี, ตัวจัดการไฟล์ หรือการตั้งค่า เพื่อไม่ให้คุณสูญเสียวิดีโอขณะค้นหาวิดีโอถัดไป

## รูปแบบวิดีโอและเสียงที่รองรับ

Evervideo เล่นได้แทบทุก container และ codec สมัยใหม่ผ่าน FFmpeg ที่รวมมา พร้อมการถอดรหัส H.264 และ HEVC ด้วยฮาร์ดแวร์บนอุปกรณ์ที่รองรับ

- **Video container:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT — และอีกมากมาย
- **Video codec:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV — รวมถึง codec อื่นๆ ที่ FFmpeg รองรับ
- **Audio codec:** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE, TTA, MPC, WV — และอีกมากมาย
- **รูปแบบซับไตเติ้ล:** SRT, VTT (WebVTT), ASS / SSA และแทร็กซับไตเติ้ลข้อความหรือรูปภาพที่ฝังอยู่
- **โปรโตคอลการสตรีม:** HTTP / HTTPS, HLS (m3u8), RTSP (กล้อง IP และ IPTV) และการสตรีมไฟล์โดยตรงผ่าน SMB / WebDAV / FTP / SFTP / NFS / DLNA

ครอบคลุมไฟล์วิดีโอแทบทุกรายการที่คุณอาจพบ — รวมถึง MKV rip, สตรีม RTSP จากกล้องรักษาความปลอดภัย และการดาวน์โหลด AV1 webm จากเว็บ

## ตัวควบคุมการเล่น

ที่ด้านล่างของหน้าจอโปรแกรมเล่น คุณจะเห็นปุ่มสำหรับ เล่น, หยุดชั่วคราว, ถัดไป และ ก่อนหน้า นอกจากนี้ยังมีปุ่มเสริมอย่างข้ามไปข้างหน้าและข้ามไปข้างหลัง (ค่าเริ่มต้น 10 วินาที) ที่คุณสามารถเปิดใช้งานได้ในการตั้งค่าแอป กดค้างปุ่มถัดไป / ก่อนหน้าเพื่อเดินหน้าเร็วหรือถอยหลัง ลาก slider การเล่นเพื่อข้ามไปยังตำแหน่งใดก็ได้

## ทำซ้ำและสลับ

แตะปุ่มทำซ้ำเพื่อวนผ่านโหมดการทำซ้ำ:

- **ทำซ้ำทั้งหมด** — วนซ้ำวิดีโอทุกรายการในคิวของคุณ
- **ทำซ้ำหนึ่ง** — ทำซ้ำเฉพาะวิดีโอปัจจุบัน
- **หยุดทำซ้ำ** — หยุดชั่วคราวเมื่อวิดีโอปัจจุบันสิ้นสุด
- **ไม่ทำซ้ำ** — เล่นคิวหนึ่งครั้งโดยไม่ทำซ้ำ

ใช้ปุ่ม **สลับ** เพื่อสุ่มลำดับวิดีโอในคิว

## Picture-in-Picture (PiP)

บน iPhone และ iPad Evervideo รองรับ Picture-in-Picture (PiP) อย่างเต็มรูปแบบ แตะไอคอน PiP บนหน้าจอโปรแกรมเล่น หรือเพียงแค่ปัด Evervideo ลงพื้นหลัง — วิดีโอยังคงเล่นต่อในหน้าต่างลอยอยู่เหนือแอปอื่นทุกแอป ลากหน้าต่างลอยไปยังมุมใดก็ได้ หยิกเพื่อปรับขนาด แตะครั้งเดียวเพื่อแสดงตัวควบคุมเล่น / หยุดชั่วคราว / ข้ามพื้นฐาน แตะปุ่มขยายขนาดเล็กเพื่อกลับสู่ Evervideo เต็มรูปแบบ

PiP ทำงานกับทุกรูปแบบวิดีโอที่ Evervideo เล่น รวมถึงไฟล์ที่สตรีมจากคลาวด์และสตรีม RTSP PiP ยังคงทำงานต่อเมื่อโทรศัพท์ของคุณถูกล็อก (ขึ้นอยู่กับการตั้งค่า Auto-Lock ของคุณ)

## โปรแกรมเล่นขนาดกะทัดรัด

โปรแกรมเล่นขนาดกะทัดรัดคือ mini-player ถาวรที่ยังคงมองเห็นได้ที่ด้านบนของทุกหน้าจอในแอปขณะที่คุณเรียกดูไลบรารี, ตัวจัดการไฟล์ หรือการตั้งค่า แตะเพื่อขยายเป็นโปรแกรมเล่นแบบเต็มหน้าจอ ปัดลงเพื่อยุบกลับ

## AirPlay 2

สำหรับ AirPlay แตะปุ่ม AirPlay บนโปรแกรมเล่น Evervideo รองรับ AirPlay 2 เพื่อให้คุณสตรีมวิดีโอไปยัง Apple TV, HomePod หรือลำโพงหรือสมาร์ทีวีที่เข้ากันได้กับ AirPlay 2 ในการตั้งค่าที่มีอุปกรณ์ AirPlay หลายเครื่อง คุณสามารถส่งเสียงไปยังหลายเครื่องรับพร้อมกัน

## Google Chromecast

สำหรับผู้ใช้ Google Cast ไอคอน Cast จะปรากฏบนโปรแกรมเล่น แตะเพื่อเลือกอุปกรณ์และเริ่ม casting ตรวจสอบให้แน่ใจว่าโทรศัพท์และเครื่องรับ Cast อยู่ในเครือข่าย Wi-Fi เดียวกัน ไม่ใช่ทุก codec ที่ฮาร์ดแวร์ Chromecast รองรับ — ไฟล์บางรายการอาจต้องการการแปลงรหัส

## สตรีม RTSP สด

Evervideo สามารถเล่นแหล่ง RTSP โดยตรง — กล้อง IP, กล้องดอร์เบลล์, สตรีม IPTV, ฟีดการออกอากาศ และ URL `rtsp://` ใดก็ได้ เพิ่มสตรีมเป็นการเชื่อมต่อ RTSP ในไฟล์ → ลิงก์ออนไลน์ → เพิ่มลิงก์ จากนั้นแตะเพื่อเริ่มดู สตรีม RTSP ทำงานใน Picture-in-Picture, โปรแกรมเล่นขนาดกะทัดรัด และ cast ผ่าน AirPlay 2 และ Chromecast เหมือนวิดีโอปกติ

## การเลือกแทร็กเสียง

สำหรับวิดีโอที่มีแทร็กเสียงหลายรายการ (คอมเมนต์, ดับเสียงภาษาอื่น, แทร็กของผู้กำกับ) แตะปุ่มดำเนินการเพิ่มเติมบนโปรแกรมเล่นและเลือก แทร็กเสียง — จากนั้นเลือกแทร็กที่ต้องการ สิ่งนี้จำเป็นสำหรับภาพยนตร์ภาษาต่างประเทศ, ไฟล์ BDMV / MKV ที่มีดับเสียงหลายรายการ และเนื้อหาการสอนที่มีแทร็กคอมเมนต์

## การเลือกแทร็กวิดีโอ

ไฟล์วิดีโอบางรายการมีสตรีมวิดีโอหลายรายการ (Blu-ray หลายมุม, การตัดต่ออื่น) เลือก แทร็กวิดีโอ จากเมนูดำเนินการเพิ่มเติมเพื่อสลับระหว่างกันระหว่างการเล่น

## ซับไตเติ้ล — ภายในและภายนอก

Evervideo ให้การควบคุมซับไตเติ้ลอย่างละเอียด:

- **แทร็กซับไตเติ้ล** — เลือกจากแทร็กที่ฝังอยู่ในไฟล์
- **ไฟล์ซับไตเติ้ลภายนอก** — โหลดไฟล์ `.srt`, `.vtt`, `.ass` หรือ `.ssa` จากอุปกรณ์, iCloud Drive หรือบริการคลาวด์ที่เชื่อมต่อ
- **การแสดงผล Libass** — การจัดสไตล์ ASS / SSA ขั้นสูง (แบบอักษร, สี, ตำแหน่ง, เอฟเฟกต์คาราโอเกะ) แสดงผลได้อย่างถูกต้องด้วย libass ที่รวมมา
- **การเลือกแบบอักษร** — เลือกแบบอักษรกำหนดเองสำหรับซับไตเติ้ลหลักและแบบอักษรแยกสำหรับซับไตเติ้ลรอง มีแบบอักษรที่รวมมาและแบบอักษรที่ติดตั้งบนอุปกรณ์ของคุณ

คุณสามารถกำหนดค่าทั้งหมดนี้ใน การตั้งค่า → ซับไตเติ้ล ก่อนการเล่น หรือใช้ ดำเนินการเพิ่มเติม → ซับไตเติ้ล จากโปรแกรมเล่นเพื่อเปลี่ยนได้ทันที

## อีควอไลเซอร์เสียง

Evervideo มีอีควอไลเซอร์เสียงแบบเต็มรูปแบบเพื่อปรับแต่งซาวด์แทร็กวิดีโอสำหรับหูฟัง, ลำโพง หรือชุด hi-fi ของคุณ แตะไอคอนอีควอไลเซอร์บนมุมมองระดับเสียง (หรือการดำเนินการอีควอไลเซอร์เสียงในเมนูดำเนินการเพิ่มเติมของโปรแกรมเล่น) เปิดอีควอไลเซอร์ด้วยสวิตช์ที่มุมขวาบน และเลือก preset หรือลาก slider แถบเพื่อสร้าง preset ของตัวเอง preset กำหนดเองสามารถส่งออกและนำเข้าได้เพื่อแชร์ข้ามอุปกรณ์หรือสำรองข้อมูล

## อีควอไลเซอร์วิดีโอ

สำหรับการปรับแต่งภาพ Evervideo มีอีควอไลเซอร์วิดีโอแบบเฉพาะ — ปรับความสว่าง, คอนทราสต์, ความอิ่มตัวของสี และสีโทนแบบเรียลไทม์ระหว่างการเล่น เหมือน อีควอไลเซอร์เสียง preset วิดีโอกำหนดเองสามารถส่งออกและนำเข้าได้เพื่อแชร์หรือสำรองข้อมูล ใช้เพื่อทำให้ฉากมืดสว่างขึ้นในวันที่แดดออก, เพิ่มความอิ่มตัวบนเนื้อหาที่ซีดจาง หรือให้ความอบอุ่นกับสีโทนเย็น

## โหมดการปรับขนาดวิดีโอ

เลือกวิธีที่วิดีโอเต็มหน้าจอ:

- **พอดี** — รักษาอัตราส่วนภาพดั้งเดิม แถบดำตรงที่จำเป็น
- **เต็ม** — เต็มหน้าจอทั้งหมด ตัดวิดีโอหากจำเป็น
- **ยืด** — ยืดวิดีโอเพื่อเต็มหน้าจอ บิดเบือนหากจำเป็น
- **ต้นฉบับ** — รักษาความละเอียดดั้งเดิมในแบบ 1:1

## การหมุนวิดีโอ

สำหรับวิดีโอที่บันทึกด้วยการวางแนวที่ผิด เลือก **ดำเนินการเพิ่มเติม → การหมุน** และเลือก **0°**, **90°**, **180°** หรือ **270°** เพื่อหมุนภาพโดยไม่ต้องออกจากโปรแกรมเล่น

## การถอดรหัสฮาร์ดแวร์ (H.264 และ HEVC)

ใน การตั้งค่า → โปรแกรมเล่น → วิดีโอ คุณสามารถเปิด / ปิดการถอดรหัสฮาร์ดแวร์ H.264 และการถอดรหัสฮาร์ดแวร์ H.265 (HEVC) ได้อย่างอิสระ การถอดรหัสฮาร์ดแวร์เร็วกว่า ใช้แบตเตอรี่น้อยกว่า และทำงานเย็นกว่า — แต่ในกรณีที่หายาก (ไฟล์เสียหาย, profile แปลกๆ) คุณอาจต้องปิดการถอดรหัสฮาร์ดแวร์และกลับมาใช้การถอดรหัสซอฟต์แวร์ FFmpeg Evervideo ให้คุณทำสิ่งนั้นได้ทีละไฟล์จากเมนูดำเนินการเพิ่มเติมของโปรแกรมเล่น

## VR 360° Viewport

Evervideo มี VR / 360° viewport สำหรับไฟล์วิดีโอทรงกลม เมื่อเล่นวิดีโอ 360° คุณสามารถลากเพื่อมองรอบๆ หยิกเพื่อซูม และ Evervideo จะบิดเบือนการแสดงผลแบบเรียลไทม์

## ความเร็วการเล่น

แตะตัวควบคุมความเร็วบนแถบเครื่องมือโปรแกรมเล่นเพื่อเปลี่ยนความเร็วการเล่น — ชะลอสำหรับการวิเคราะห์ (0.25× หรือ 0.5×) หรือเร่งสำหรับบทแนะนำและการบรรยาย (1.25×, 1.5×, 2× และสูงสุด 3×) แตะไอคอนการกำหนดค่าที่มุมขวาบนของหน้าจอความเร็วเพื่อสลับเป็นโหมดแม่นยำพร้อมการปรับแต่งที่ละเอียดกว่า มีการแก้ไขระดับเสียงต่อแทร็กด้วย

## ตัวจับเวลาการนอน

เปิด การตั้งค่า → โปรแกรมเล่น → ตัวจับเวลาการนอน เปิดใช้งาน และเลือกระยะเวลาที่ต้องการให้การเล่นดำเนินต่อก่อนหยุดโดยอัตโนมัติ คุณยังสามารถเพิ่มปุ่มตัวจับเวลาการนอนโดยตรงในหน้าจอโปรแกรมเล่นหลักผ่าน การตั้งค่า → โปรแกรมเล่น → การปรับแต่ง → การดำเนินการหน้าจอหลัก

## บุ๊คมาร์คโปรแกรมเล่น

บันทึกตำแหน่งของคุณในวิดีโอยาว — การบรรยาย, หนังสือเสียงแบบวิดีโอ, บทแนะนำ, การดาวน์โหลด YouTube ชั่วโมงยาว — โดยแตะ เพิ่มบุ๊คมาร์ค จากเมนูดำเนินการเพิ่มเติม บุ๊คมาร์คจะปรากฏในรายการ ดำเนินการเพิ่มเติม → บุ๊คมาร์ค ของวิดีโอและยังคงอยู่ระหว่างเซสชัน

## เมนูดำเนินการเพิ่มเติม

แตะปุ่ม **ดำเนินการเพิ่มเติม "..."** บนโปรแกรมเล่นเพื่อเข้าถึงฟังก์ชันเพิ่มเติม

- **ดำเนินการเล่นต่อ** — กลับมาเล่นต่อคิวจากตำแหน่งล่าสุด
- **ค้นหา** — ค้นหาวิดีโอเฉพาะในคิวของคุณ
- **บุ๊คมาร์ค** — ดูและข้ามไปยังบุ๊คมาร์ค
- **ความเร็ว** — ปรับความเร็วการเล่น
- **ล่าสุด** — วิดีโอที่เล่นล่าสุด
- **รายการโปรด** — วิดีโอที่ถูกกำหนดเป็นรายการโปรด
- **อีควอไลเซอร์เสียง** — เปิดใช้งานอีควอไลเซอร์เสียง
- **อีควอไลเซอร์วิดีโอ** — เปิดใช้งานอีควอไลเซอร์วิดีโอ
- **แทร็กเสียง** — เลือกสตรีมเสียง
- **แทร็กวิดีโอ** — เลือกสตรีมวิดีโอ
- **ซับไตเติ้ล** — แทร็กหลัก / รอง, ไฟล์ภายนอก, แบบอักษร
- **การหมุน** — หมุนภาพ 0° / 90° / 180° / 270°
- **โหมดการปรับขนาด** — พอดี / เต็ม / ยืด / ต้นฉบับ
- **Picture-in-Picture** — เข้าสู่โหมด PiP
- **AirPlay** / **Chromecast** — ส่งไปยังอุปกรณ์
- **ตัวจับเวลาการนอน** — ตั้งค่าตัวจับเวลาเพื่อหยุดการเล่น
- **บันทึกคิวเป็นเพลย์ลิสต์** — บันทึกคิวปัจจุบันเป็นเพลย์ลิสต์ใหม่
- **ลบคิว** — ล้างคิวและหยุดการเล่น
- **การตั้งค่า** — ข้ามไปยังการตั้งค่าโปรแกรมเล่นโดยตรง
- **ความช่วยเหลือ** — เปิดคำแนะนำ

## การตั้งค่าโปรแกรมเล่น

แผนผังการตั้งค่าโปรแกรมเล่นแบบเต็มรูปแบบมีเอกสารอยู่ใน [คู่มือการตั้งค่า](/docs/guide/evervideo/evervideo-guide-settings/) ส่วนที่ใช้บ่อยที่สุด:

- **ทั่วไป** — โหมดทำซ้ำ, โหมดสลับ, บันทึกสถานะโปรแกรมเล่น, บันทึกตำแหน่งการเล่น, ช่วงเวลาการข้าม
- **วิดีโอ** — การถอดรหัสฮาร์ดแวร์ H.264 / HEVC, อีควอไลเซอร์วิดีโอ, โหมดการปรับขนาด, การหมุน, โหมดการแสดงผล, FPS ที่ต้องการ, รูปแบบพิกเซลที่ต้องการ, VR viewport
- **เสียง** — อีควอไลเซอร์เสียง, อัตราการสุ่มตัวอย่าง, ช่องสัญญาณ, ระยะเวลา IO buffer, โหมดผสม
- **ซับไตเติ้ล** — แทร็กหลัก / รอง, การเลือกไฟล์ภายนอก, แบบอักษร, แบบอักษรรอง
- **อุปกรณ์** (iOS) — AirPlay / Chromecast
- **การปรับแต่ง** — รูปแบบเลย์เอาต์โปรแกรมเล่น (ขั้นต่ำ / ล่าง / โบราณ / คลาสสิก), การดำเนินการหน้าจอหลัก, ตัวควบคุมหน้าจอล็อก
- **แคชการเล่น** — แคชดิสก์สำหรับการสตรีมที่ราบรื่นยิ่งขึ้น
- **ตัวจับเวลาการนอน** — หยุดอัตโนมัติ

## การช่วยสำหรับการเข้าถึง

Evervideo สามารถเข้าถึงได้อย่างสมบูรณ์ด้วย VoiceOver ทุกส่วนประกอบมีป้ายกำกับและคำอธิบายที่ออกแบบมาดี เมื่อ VoiceOver ทำงาน แอปจะสลับไปยังโหมดข้อความเพื่อให้แสดงเฉพาะองค์ประกอบที่มีความหมาย — ซึ่งช่วยปรับปรุงความเร็วในการนำทางและความชัดเจน คุณยังสามารถเปิดใช้งานโหมดข้อความใน การตั้งค่า → การช่วยสำหรับการเข้าถึง → โหมดข้อความ

### การปรับ Slider ด้วย VoiceOver

1. **เลือก slider** — ปัดซ้ายหรือขวาจนกว่า VoiceOver จะประกาศ
2. **ปรับค่า** — แตะสองครั้งและกดค้าง slider จากนั้นลากขึ้นหรือลงเพื่อเปลี่ยนค่าอย่างรวดเร็ว VoiceOver ประกาศค่าใหม่ขณะที่คุณดำเนินการ

ส่วนประกอบอื่นๆ ทำงานตามที่คาดหวัง โดยใช้รูปแบบ VoiceOver ที่ระบบจัดเตรียมไว้

媒體播放器是應用程式的主畫面，您在這裡控制播放和 Evervideo 的大多數功能。它同時播放視訊和音訊檔案，基於定製的 FFmpeg 播放引擎構建，配備執行繁重工作的硬體加速 H.264 和 HEVC 解碼。讓我們來了解如何使用它。

## 存取媒體播放器

您可以從緊湊播放器列進入全螢幕播放器。在 iPhone 上，緊湊播放器位於主畫面頂部。在 iPad 和 Mac 上，它在左側（或主面板頂部）。要將全螢幕播放器折疊回緊湊視圖，請點選右下角的關閉按鈕或向下滑動。要完全隱藏緊湊播放器，請再次點選並向下滑動。

緊湊播放器在您瀏覽資料庫、檔案管理員或設定時保持可見，因此您在查找下一個視訊時永遠不會失去當前視訊。

## 支援的視訊和音訊格式

Evervideo 透過捆綁的 FFmpeg 引擎幾乎可以播放所有現代容器和編解碼器，在支援的裝置上具有硬體加速 H.264 和 HEVC 解碼。

- **視訊容器：** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT — 以及更多。
- **視訊編解碼器：** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV — 加上 FFmpeg 支援的幾乎所有其他編解碼器。
- **音訊編解碼器：** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE, TTA, MPC, WV — 以及更多。
- **字幕格式：** SRT, VTT (WebVTT), ASS / SSA 以及嵌入的文字或圖像字幕軌道。
- **串流通訊協定：** HTTP / HTTPS, HLS (m3u8), RTSP（IP 攝影機和 IPTV），以及透過 SMB / WebDAV / FTP / SFTP / NFS / DLNA 的直接檔案串流。

這涵蓋了您可能遇到的幾乎所有視訊檔案——包括 MKV 轉錄、安全攝影機 RTSP 串流和 AV1 webm 網路下載。

## 播放控制

在播放器畫面底部，您會看到播放、暫停、下一個和上一個按鈕。還有可選的跳過前進和跳過後退按鈕（預設 10 秒），您可以在應用程式設定中啟用。按住下一個 / 上一個按鈕可快進或快退。拖動播放滑桿跳轉到任何位置。

## 重複和隨機播放

點選重複按鈕在重複模式之間切換：

- **重複全部** — 循環播放佇列中的每個視訊。
- **重複一個** — 只重複當前視訊。
- **重複停止** — 當前視訊結束時暫停。
- **不重複** — 播放一遍佇列而不重複。

使用**隨機播放**按鈕隨機化佇列中視訊的順序。

## 子母畫面（PiP）

在 iPhone 和 iPad 上，Evervideo 完全支援子母畫面（PiP）。點選播放器畫面上的 PiP 圖示或直接將 Evervideo 切換到背景——視訊繼續在所有其他應用程式上方的浮動視窗中播放。將浮動視窗拖到任何角落；用兩指捏合來調整大小；點選一次調出基本播放 / 暫停 / 跳過控制；點選小展開按鈕返回完整的 Evervideo。

PiP 適用於 Evervideo 播放的每種視訊格式，包括雲端串流檔案和 RTSP 串流。PiP 在手機鎖定時也會繼續執行（取決於自動鎖定設定）。

## 緊湊播放器

緊湊播放器是一個持久的迷你播放器，在您瀏覽資料庫、檔案管理員或設定時始終顯示在應用程式的每個畫面頂部。點選它展開到全螢幕播放器；向下滑動再次折疊。

## AirPlay 2

要使用 AirPlay，點選播放器上的 AirPlay 按鈕。Evervideo 支援 AirPlay 2，因此您可以將視訊串流到 Apple TV、HomePod 或任何相容 AirPlay 2 的音箱或智慧電視。在具有多個 AirPlay 裝置的設定中，您可以同時將音訊路由到多個接收器。

## Google Chromecast

對於 Google Cast 使用者，播放器上會出現投影圖示。點選它選擇裝置並開始投影。確保您的手機和 Cast 接收器在同一 Wi-Fi 網路上。不是每種編解碼器都受到 Chromecast 硬體支援——某些檔案可能需要轉碼。

## RTSP 直播串流

Evervideo 可以直接播放 RTSP 來源——IP 攝影機、門鈴攝影機、IPTV 串流、廣播來源以及任何 `rtsp://` URL。在檔案 → 線上連結 → 新增連結中將串流新增為 RTSP 連線，然後點選它開始觀看。RTSP 串流在子母畫面、緊湊播放器中工作，並像普通視訊一樣透過 AirPlay 2 和 Chromecast 投影。

## 音訊軌道選擇

對於包含多個音訊軌道的視訊（評論、備用語言配音、導演軌道），點選播放器上的更多操作按鈕並選擇音訊軌道——然後選擇您想要的軌道。這對於外語電影、包含多個配音的 BDMV / MKV 檔案以及帶有評論軌道的教學內容至關重要。

## 視訊軌道選擇

某些視訊檔案包含多個視訊串流（多角度藍光、備用剪輯）。從更多操作選單中選擇視訊軌道，在播放途中在它們之間切換。

## 字幕——內部和外部

Evervideo 為您提供對字幕的細粒度控制：

- **字幕軌道** — 從檔案中嵌入的軌道中選擇。
- **外部字幕檔案** — 從您的裝置、iCloud Drive 或任何已連線的雲端服務載入 `.srt`、`.vtt`、`.ass` 或 `.ssa` 檔案。
- **Libass 渲染** — 高級 ASS / SSA 樣式（字型、顏色、位置、卡拉 OK 效果）由於捆綁的 libass 可以正確渲染。
- **字型選擇** — 為主字幕軌道選擇自訂字型，為輔助字幕選擇單獨的字型。捆綁字型以及裝置上安裝的任何字型均可使用。

您可以在設定 → 字幕中在播放前設定所有這些，或使用播放器中的更多操作 → 字幕在播放時切換。

## 音訊等化器

Evervideo 包含完整的音訊等化器，用於為您的耳機、音箱或高保真系統調整視訊音軌。點選音量視圖上的等化器圖示（或播放器更多操作選單上的音訊等化器操作），用右上角的開關打開等化器，然後選擇預設或拖動頻帶滑桿建立自己的預設。自訂預設可以匯出和匯入，以便在裝置之間共享或備份。

## 視訊等化器

為了調整畫面，Evervideo 提供專用的視訊等化器——在播放過程中即時調整亮度、對比度、飽和度和色調。與音訊等化器一樣，自訂視訊預設可以匯出和匯入以便共享或備份。使用它在晴天亮化暗景、增強褪色內容的飽和度，或暖化冷色調。

## 視訊縮放模式

選擇視訊如何填充畫面：

- **適配** — 保持原始寬高比；必要時顯示黑邊。
- **填充** — 填充整個畫面，必要時裁剪視訊。
- **拉伸** — 拉伸視訊填充畫面，必要時變形。
- **原始** — 以 1:1 保持原始解析度。

## 視訊旋轉

對於方向錯誤的視訊，選擇**更多操作 → 旋轉**，選擇 **0°**、**90°**、**180°** 或 **270°** 旋轉畫面，而無需離開播放器。

## 硬體解碼（H.264 和 HEVC）

在設定 → 播放器 → 視訊中，您可以獨立啟用 / 停用硬體解碼 H.264 和硬體解碼 H.265 (HEVC)。硬體解碼更快、耗電更少、執行更涼——但在極少數情況下（損壞的檔案、奇特的設定檔）您可能需要停用硬體解碼並回退到軟體 FFmpeg 解碼。Evervideo 允許您從播放器更多操作選單逐個檔案執行此操作。

## VR 360° 視口

Evervideo 包含用於球形視訊檔案的 VR / 360° 視口。播放 360° 視訊時，您可以拖動四處張望，捏合縮放，Evervideo 會即時扭曲渲染。

## 播放速度

點選播放器工具列上的速度控制來更改播放速度——放慢用於分析（0.25× 或 0.5×）或加速用於教學和講座（1.25×、1.5×、2× 和高達 3×）。點選速度畫面右上角的設定圖示切換到具有更精細調整的精確模式。還提供每軌道音調校正。

## 睡眠定時器

開啟設定 → 播放器 → 睡眠定時器，開啟它，選擇您希望播放在自動停止前持續多長時間。您也可以透過設定 → 播放器 → 個人化 → 主畫面操作將睡眠定時器按鈕直接新增到主播放器畫面。

## 播放器書籤

在長視訊中儲存您的位置——講座、視訊有聲書、教學、一小時的 YouTube 下載——透過在更多操作選單中點選新增書籤。書籤出現在視訊的更多操作 → 書籤清單中，並在工作階段之間持久保存。

## 更多操作選單

點選播放器上的**更多操作「...」**按鈕存取其他功能。

- **繼續播放** — 從最後位置恢復佇列。
- **搜尋** — 在佇列中查找特定視訊。
- **書籤** — 查看並跳轉到書籤。
- **速度** — 調整播放速度。
- **最近使用的** — 最近播放的視訊。
- **最愛項目** — 收藏的視訊。
- **音訊等化器** — 啟動音訊等化器。
- **視訊等化器** — 啟動視訊等化器。
- **音訊軌道** — 選擇音訊串流。
- **視訊軌道** — 選擇視訊串流。
- **字幕** — 主 / 副軌道、外部檔案、字型。
- **旋轉** — 旋轉畫面 0° / 90° / 180° / 270°。
- **縮放模式** — 適配 / 填充 / 拉伸 / 原始。
- **子母畫面** — 進入 PiP 模式。
- **AirPlay** / **Chromecast** — 傳送到裝置。
- **睡眠定時器** — 設定定時器停止播放。
- **將佇列儲存為播放清單** — 將當前佇列儲存為新播放清單。
- **刪除佇列** — 清除佇列並停止播放。
- **設定** — 直接跳轉到播放器設定。
- **說明** — 開啟指南。

## 播放器設定

完整的播放器設定樹在[設定指南](/docs/guide/evervideo/evervideo-guide-settings/)中有記錄。最常用的部分：

- **一般** — 重複模式、隨機播放模式、儲存播放器狀態、儲存播放位置、跳過時間間隔。
- **視訊** — 硬體解碼 H.264 / HEVC、視訊等化器、縮放模式、旋轉、顯示模式、首選 FPS、首選像素格式、VR 視口。
- **音訊** — 音訊等化器、取樣率、聲道、IO 緩衝時長、混合模式。
- **字幕** — 主 / 副軌道、外部檔案選擇、字型、輔助字型。
- **裝置**（iOS）— AirPlay / Chromecast。
- **個人化** — 播放器版面配置樣式（最小 / 底部 / 古典 / 經典）、主畫面操作、鎖定螢幕控制項。
- **播放快取** — 用於更流暢串流的磁碟快取。
- **睡眠定時器** — 自動停止。

## 無障礙功能

Evervideo 完全支援 VoiceOver。每個元件都有精心設計的標籤和描述。VoiceOver 啟動時，應用程式切換到文字模式，只顯示有意義的元素——提升導覽速度和清晰度。您也可以在設定 → 無障礙 → 文字模式中啟動文字模式。

### 使用 VoiceOver 調整滑桿

1. **選擇滑桿** — 向左或向右滑動，直到 VoiceOver 宣告它。
2. **調整值** — 雙擊並按住滑桿，然後向上或向下拖動以快速變更值。VoiceOver 會在您操作時宣告新值。

其他元件按預期工作，使用系統提供的 VoiceOver 模式。

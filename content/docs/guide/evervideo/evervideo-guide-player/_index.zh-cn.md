媒体播放器是应用的主屏幕，您在这里控制播放和 Evervideo 的大多数功能。它同时播放视频和音频文件，基于定制的 FFmpeg 播放引擎构建，配备执行繁重工作的硬件加速 H.264 和 HEVC 解码。让我们来了解如何使用它。

## 访问媒体播放器

您可以从紧凑播放器栏进入全屏播放器。在 iPhone 上，紧凑播放器位于主屏幕顶部。在 iPad 和 Mac 上，它在左侧（或主面板顶部）。要将全屏播放器折叠回紧凑视图，请点击右下角的关闭按钮或向下滑动。要完全隐藏紧凑播放器，请再次点击并向下滑动。

紧凑播放器在您浏览资料库、文件管理器或设置时保持可见，因此您在查找下一个视频时永远不会失去当前视频。

## 支持的视频和音频格式

Evervideo 通过捆绑的 FFmpeg 引擎几乎可以播放所有现代容器和编解码器，在支持的设备上具有硬件加速 H.264 和 HEVC 解码。

- **视频容器：** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT — 以及更多。
- **视频编解码器：** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV — 加上 FFmpeg 支持的几乎所有其他编解码器。
- **音频编解码器：** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE, TTA, MPC, WV — 以及更多。
- **字幕格式：** SRT, VTT (WebVTT), ASS / SSA 以及嵌入的文字或图像字幕轨道。
- **流媒体协议：** HTTP / HTTPS, HLS (m3u8), RTSP（IP 摄像头和 IPTV），以及通过 SMB / WebDAV / FTP / SFTP / NFS / DLNA 的直接文件流媒体。

这涵盖了您可能遇到的几乎所有视频文件——包括 MKV 翻录、安全摄像头 RTSP 流和 AV1 webm 网络下载。

## 播放控制

在播放器屏幕底部，您会看到播放、暂停、下一个和上一个按钮。还有可选的跳过前进和跳过后退按钮（默认 10 秒），您可以在应用设置中启用。按住下一个 / 上一个按钮可快进或快退。拖动播放滑块跳转到任何位置。

## 重复和随机播放

点击重复按钮在重复模式之间切换：

- **重复全部** — 循环播放队列中的每个视频。
- **重复一个** — 只重复当前视频。
- **重复停止** — 当前视频结束时暂停。
- **不重复** — 播放一遍队列而不重复。

使用**随机播放**按钮随机化队列中视频的顺序。

## 画中画（PiP）

在 iPhone 和 iPad 上，Evervideo 完全支持画中画（PiP）。点击播放器屏幕上的 PiP 图标或直接将 Evervideo 切换到后台——视频继续在所有其他应用上方的浮动窗口中播放。将浮动窗口拖到任何角落；用两指捏合来调整大小；点击一次调出基本播放 / 暂停 / 跳过控制；点击小展开按钮返回完整的 Evervideo。

PiP 适用于 Evervideo 播放的每种视频格式，包括云端流媒体文件和 RTSP 流。PiP 在手机锁定时也会继续运行（取决于自动锁定设置）。

## 紧凑播放器

紧凑播放器是一个持久的迷你播放器，在您浏览资料库、文件管理器或设置时始终显示在应用的每个屏幕顶部。点击它展开到全屏播放器；向下滑动再次折叠。

## AirPlay 2

要使用 AirPlay，点击播放器上的 AirPlay 按钮。Evervideo 支持 AirPlay 2，因此您可以将视频流式传输到 Apple TV、HomePod 或任何兼容 AirPlay 2 的音箱或智能电视。在具有多个 AirPlay 设备的设置中，您可以同时将音频路由到多个接收器。

## Google Chromecast

对于 Google Cast 用户，播放器上会出现投射图标。点击它选择设备并开始投射。确保您的手机和 Cast 接收器在同一 Wi-Fi 网络上。不是每种编解码器都受到 Chromecast 硬件支持——某些文件可能需要转码。

## RTSP 直播流

Evervideo 可以直接播放 RTSP 来源——IP 摄像头、门铃摄像头、IPTV 流、广播源以及任何 `rtsp://` URL。在文件 → 在线链接 → 添加链接中将流添加为 RTSP 连接，然后点击它开始观看。RTSP 流在画中画、紧凑播放器中工作，并像普通视频一样通过 AirPlay 2 和 Chromecast 投射。

## 音频轨道选择

对于包含多个音频轨道的视频（评论、备用语言配音、导演轨道），点击播放器上的更多操作按钮并选择音频轨道——然后选择您想要的轨道。这对于外语电影、包含多个配音的 BDMV / MKV 文件以及带有评论轨道的教学内容至关重要。

## 视频轨道选择

某些视频文件包含多个视频流（多角度蓝光、备用剪辑）。从更多操作菜单中选择视频轨道，在播放中途在它们之间切换。

## 字幕——内部和外部

Evervideo 为您提供对字幕的细粒度控制：

- **字幕轨道** — 从文件中嵌入的轨道中选择。
- **外部字幕文件** — 从您的设备、iCloud Drive 或任何已连接的云服务加载 `.srt`、`.vtt`、`.ass` 或 `.ssa` 文件。
- **Libass 渲染** — 高级 ASS / SSA 样式（字体、颜色、位置、卡拉 OK 效果）由于捆绑的 libass 可以正确渲染。
- **字体选择** — 为主字幕轨道选择自定义字体，为辅助字幕选择单独的字体。捆绑字体以及设备上安装的任何字体均可使用。

您可以在设置 → 字幕中在播放前配置所有这些，或使用播放器中的更多操作 → 字幕在播放时切换。

## 音频均衡器

Evervideo 包含完整的音频均衡器，用于为您的耳机、音箱或高保真系统调整视频音轨。点击音量视图上的均衡器图标（或播放器更多操作菜单上的音频均衡器操作），用右上角的开关打开均衡器，然后选择预设或拖动频带滑块建立自己的预设。自定义预设可以导出和导入，以便在设备之间共享或备份。

## 视频均衡器

为了调整画面，Evervideo 提供专用的视频均衡器——在播放过程中实时调整亮度、对比度、饱和度和色调。与音频均衡器一样，自定义视频预设可以导出和导入以便共享或备份。使用它在晴天亮化暗景、增强褪色内容的饱和度，或暖化冷色调。

## 视频缩放模式

选择视频如何填充屏幕：

- **适配** — 保持原始宽高比；必要时显示黑边。
- **填充** — 填充整个屏幕，必要时裁剪视频。
- **拉伸** — 拉伸视频填充屏幕，必要时变形。
- **原始** — 以 1:1 保持原始分辨率。

## 视频旋转

对于方向错误的视频，选择**更多操作 → 旋转**，选择 **0°**、**90°**、**180°** 或 **270°** 旋转画面，而无需离开播放器。

## 硬件解码（H.264 和 HEVC）

在设置 → 播放器 → 视频中，您可以独立启用 / 禁用硬件解码 H.264 和硬件解码 H.265 (HEVC)。硬件解码更快、耗电更少、运行更凉——但在极少数情况下（损坏的文件、奇特的配置文件）您可能需要禁用硬件解码并回退到软件 FFmpeg 解码。Evervideo 允许您从播放器更多操作菜单逐个文件执行此操作。

## VR 360° 视口

Evervideo 包含用于球形视频文件的 VR / 360° 视口。播放 360° 视频时，您可以拖动四处张望，捏合缩放，Evervideo 会实时扭曲渲染。

## 播放速度

点击播放器工具栏上的速度控制来更改播放速度——放慢用于分析（0.25× 或 0.5×）或加速用于教程和讲座（1.25×、1.5×、2× 和高达 3×）。点击速度屏幕右上角的配置图标切换到具有更精细调整的精确模式。还提供每轨道音调校正。

## 睡眠定时器

打开设置 → 播放器 → 睡眠定时器，开启它，选择您希望播放在自动停止前持续多长时间。您也可以通过设置 → 播放器 → 个性化 → 主屏幕操作将睡眠定时器按钮直接添加到主播放器屏幕。

## 播放器书签

在长视频中保存您的位置——讲座、视频有声书、教程、一小时的 YouTube 下载——通过在更多操作菜单中点击添加书签。书签出现在视频的更多操作 → 书签列表中，并在会话之间持久保存。

## 更多操作菜单

点击播放器上的**更多操作「...」**按钮访问其他功能。

- **继续播放** — 从最后位置恢复队列。
- **搜索** — 在队列中查找特定视频。
- **书签** — 查看并跳转到书签。
- **速度** — 调整播放速度。
- **最近使用** — 最近播放的视频。
- **收藏夹** — 收藏的视频。
- **音频均衡器** — 激活音频均衡器。
- **视频均衡器** — 激活视频均衡器。
- **音频轨道** — 选择音频流。
- **视频轨道** — 选择视频流。
- **字幕** — 主 / 副轨道、外部文件、字体。
- **旋转** — 旋转画面 0° / 90° / 180° / 270°。
- **缩放模式** — 适配 / 填充 / 拉伸 / 原始。
- **画中画** — 进入 PiP 模式。
- **AirPlay** / **Chromecast** — 发送到设备。
- **睡眠定时器** — 设置定时器停止播放。
- **将队列保存为播放列表** — 将当前队列保存为新播放列表。
- **删除队列** — 清除队列并停止播放。

## 播放器设置

完整的播放器设置树在[设置指南](/docs/guide/evervideo/evervideo-guide-settings/)中有记录。最常用的部分：

- **一般** — 重复模式、随机播放模式、保存播放器状态、保存播放位置、跳过时间间隔。
- **视频** — 硬件解码 H.264 / HEVC、视频均衡器、缩放模式、旋转、显示模式、首选 FPS、首选像素格式、VR 视口。
- **音频** — 音频均衡器、采样率、声道、IO 缓冲时长、混合模式。
- **字幕** — 主 / 副轨道、外部文件选择、字体、辅助字体。
- **设备**（iOS）— AirPlay / Chromecast。
- **个性化** — 播放器布局样式（最小 / 底部 / 古典 / 经典）、主屏幕操作、锁屏控制。
- **播放缓存** — 用于更流畅流媒体的磁盘缓存。
- **睡眠定时器** — 自动停止。

## 无障碍功能

Evervideo 完全支持 VoiceOver。每个组件都有精心设计的标签和描述。VoiceOver 激活时，应用切换到文字模式，只显示有意义的元素——提升导航速度和清晰度。您也可以在设置 → 无障碍 → 文字模式中激活文字模式。

### 使用 VoiceOver 调整滑块

1. **选择滑块** — 向左或向右滑动，直到 VoiceOver 宣布它。
2. **调整值** — 双击并按住滑块，然后向上或向下拖动以快速更改值。VoiceOver 会在您操作时宣布新值。

其他组件按预期工作，使用系统提供的 VoiceOver 模式。

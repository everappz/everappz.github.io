미디어 플레이어는 재생을 제어하고 Evervideo의 대부분의 기능을 사용하는 앱의 메인 화면입니다. 비디오 및 오디오 파일을 재생하며 하드웨어 가속 H.264 및 HEVC 디코딩을 갖춘 맞춤형 FFmpeg 기반 플레이어 위에 구축되어 있습니다. 사용 방법을 살펴보겠습니다.

## 미디어 플레이어 접근

소형 플레이어 바에서 전체 화면 플레이어로 접근할 수 있습니다. iPhone에서 소형 플레이어는 메인 화면 상단에 있습니다. iPad 및 Mac에서는 왼쪽 (또는 메인 패널 상단)에 있습니다. 전체 화면 플레이어를 소형 보기로 다시 접으려면 오른쪽 하단 모서리의 닫기 버튼을 탭하거나 아래로 스와이프합니다. 소형 플레이어를 완전히 숨기려면 한 번 더 탭하고 아래로 스와이프합니다.

소형 플레이어는 라이브러리, 파일 관리자 또는 설정을 탐색하는 동안에도 계속 표시되어 다음 것을 찾는 동안 비디오를 잃지 않습니다.

## 지원되는 비디오 및 오디오 형식

Evervideo는 번들된 FFmpeg 엔진을 통해 거의 모든 최신 컨테이너와 코덱을 재생하며, 지원 기기에서 하드웨어 가속 H.264 및 HEVC 디코딩을 사용합니다.

- **비디오 컨테이너:** MP4, M4V, MKV, MOV, AVI, FLV, WMV, ASF, WebM, TS, M2TS, MTS, MPG, MPEG, OGV, 3GP, 3G2, F4V, RM, RMVB, VOB, DAT — 기타 다수.
- **비디오 코덱:** H.264 (AVC), H.265 (HEVC), VP9, VP8, AV1, MPEG-2, MPEG-4, MJPEG, ProRes, Theora, WMV — FFmpeg가 지원하는 거의 모든 다른 코덱 포함.
- **오디오 코덱:** AAC, MP3, FLAC, ALAC, OGG / Vorbis, OPUS, AC-3, EAC-3, DTS, AMR, WMA, APE, TTA, MPC, WV — 기타 다수.
- **자막 형식:** SRT, VTT (WebVTT), ASS / SSA, 임베디드 텍스트 또는 이미지 자막 트랙.
- **스트리밍 프로토콜:** HTTP / HTTPS, HLS (m3u8), RTSP (IP 카메라 및 IPTV), SMB / WebDAV / FTP / SFTP / NFS / DLNA를 통한 직접 파일 스트리밍.

이는 MKV 립, 보안 카메라 RTSP 스트림 및 AV1 webm 웹 다운로드를 포함하여 접할 가능성이 있는 거의 모든 비디오 파일을 포괄합니다.

## 재생 컨트롤

플레이어 화면 하단에 재생, 일시 정지, 다음 및 이전 버튼이 있습니다. 앱 설정에서 활성화할 수 있는 앞으로 건너뛰기 및 뒤로 건너뛰기 (기본값 10초)와 같은 선택적 버튼도 있습니다. 다음 / 이전 버튼을 길게 눌러 빠르게 앞으로/뒤로 감습니다. 재생 슬라이더를 드래그하여 어떤 위치로든 이동합니다.

## 반복 및 셔플

반복 버튼을 탭하여 반복 모드를 순환합니다:

- **모두 반복** — 대기열의 모든 비디오를 반복합니다.
- **하나 반복** — 현재 비디오만 반복합니다.
- **반복 정지** — 현재 비디오가 끝나면 일시 정지합니다.
- **반복 없음** — 반복 없이 대기열을 한 번 재생합니다.

**셔플** 버튼을 사용하여 대기열의 비디오 순서를 무작위로 설정합니다.

## 화면 속 화면 (PiP)

iPhone 및 iPad에서 Evervideo는 화면 속 화면 (PiP)을 완전히 지원합니다. 플레이어 화면의 PiP 아이콘을 탭하거나 Evervideo를 백그라운드로 스와이프하기만 하면 됩니다 — 비디오는 다른 모든 앱 위에 부동 창에서 계속 재생됩니다. 부동 창을 모서리로 드래그하고; 핀치하여 크기를 조정하고; 한 번 탭하면 기본 재생 / 일시 정지 / 건너뛰기 컨트롤이 표시되며; 작은 확장 버튼을 탭하면 전체 Evervideo로 돌아갑니다.

PiP는 클라우드 스트리밍 파일 및 RTSP 스트림을 포함하여 Evervideo가 재생하는 모든 비디오 형식에서 작동합니다. PiP는 또한 자동 잠금 설정에 따라 휴대폰이 잠긴 동안에도 계속 실행됩니다.

## 소형 플레이어

소형 플레이어는 라이브러리, 파일 관리자 또는 설정을 탐색하는 동안 앱의 모든 화면 상단에 계속 표시되는 지속적인 미니 플레이어입니다. 탭하면 전체 화면 플레이어로 확장되고; 아래로 스와이프하면 다시 접힙니다.

## AirPlay 2

AirPlay의 경우 플레이어의 AirPlay 버튼을 탭합니다. Evervideo는 AirPlay 2를 지원하므로 Apple TV, HomePod 또는 AirPlay 2 호환 스피커나 스마트 TV로 비디오를 스트리밍할 수 있습니다. 여러 AirPlay 장치가 있는 설정에서는 여러 수신기로 동시에 오디오를 라우팅할 수 있습니다.

## Google Chromecast

Google Cast 사용자의 경우 플레이어에 Cast 아이콘이 나타납니다. 탭하여 장치를 선택하고 캐스팅을 시작합니다. 휴대폰과 Cast 수신기가 동일한 Wi-Fi 네트워크에 있는지 확인하세요. Chromecast 하드웨어에서 모든 코덱이 지원되는 것은 아닙니다 — 일부 파일은 트랜스코딩이 필요할 수 있습니다.

## RTSP 라이브 스트림

Evervideo는 RTSP 소스를 직접 재생할 수 있습니다 — IP 카메라, 도어벨 카메라, IPTV 스트림, 방송 피드, 기타 모든 `rtsp://` URL. 파일 → 온라인 링크 → 링크 추가에서 스트림을 RTSP 연결로 추가하고 탭하여 시청을 시작합니다. RTSP 스트림은 화면 속 화면, 소형 플레이어에서 작동하며 일반 비디오처럼 AirPlay 2와 Chromecast로 캐스트됩니다.

## 오디오 트랙 선택

여러 오디오 트랙이 있는 비디오 (코멘터리, 언어 더빙, 감독 트랙)의 경우 플레이어의 추가 작업 버튼을 탭하고 오디오 트랙을 선택한 다음 원하는 트랙을 선택합니다. 이것은 외국어 영화, 여러 더빙이 있는 BDMV / MKV 파일, 코멘터리 트랙이 있는 교육 콘텐츠에 필수적입니다.

## 비디오 트랙 선택

일부 비디오 파일에는 여러 비디오 스트림이 포함됩니다 (다중 앵글 블루레이, 대안 컷). 추가 작업 메뉴에서 비디오 트랙을 선택하여 재생 중에 전환합니다.

## 자막 — 내부 및 외부

Evervideo는 자막에 대한 세밀한 제어를 제공합니다:

- **자막 트랙** — 파일에 임베디드된 트랙에서 선택합니다.
- **외부 자막 파일** — 기기, iCloud Drive 또는 연결된 클라우드 서비스에서 `.srt`, `.vtt`, `.ass`, `.ssa` 파일을 불러옵니다.
- **Libass 렌더링** — 고급 ASS / SSA 스타일링 (폰트, 색상, 위치, 가라오케 효과)이 번들된 libass 덕분에 올바르게 렌더링됩니다.
- **폰트 선택** — 기본 자막에 맞춤 폰트를 선택하고 보조 자막에 별도의 폰트를 선택합니다. 번들 폰트와 기기에 설치된 폰트를 사용할 수 있습니다.

재생 전에 설정 → 자막에서 이 모든 것을 구성하거나, 플레이어에서 추가 작업 → 자막을 사용하여 즉시 전환할 수 있습니다.

## 오디오 이퀄라이저

Evervideo에는 헤드폰, 스피커 또는 하이파이 설정에 맞게 비디오 사운드트랙을 조정하는 완전한 오디오 이퀄라이저가 포함되어 있습니다. 볼륨 보기의 이퀄라이저 아이콘을 탭하거나 (또는 플레이어 추가 작업 메뉴의 오디오 이퀄라이저 작업), 오른쪽 상단 모서리의 스위치로 이퀄라이저를 켜고 프리셋을 선택하거나 밴드 슬라이더를 드래그하여 자신만의 프리셋을 만듭니다. 맞춤 프리셋은 기기 간에 공유하거나 백업하기 위해 내보내고 가져올 수 있습니다.

## 비디오 이퀄라이저

화면 조정을 위해 Evervideo는 전용 비디오 이퀄라이저를 제공합니다 — 재생 중에 실시간으로 밝기, 대비, 채도 및 색조를 조정합니다. 오디오 이퀄라이저와 마찬가지로 맞춤 비디오 프리셋은 공유 또는 백업을 위해 내보내고 가져올 수 있습니다. 맑은 날에 어두운 장면을 밝히거나, 색이 바랜 콘텐츠의 채도를 높이거나, 차가운 색 캐스트를 따뜻하게 하는 데 사용하세요.

## 비디오 스케일링 모드

비디오가 화면을 채우는 방법을 선택합니다:

- **맞춤** — 원래 화면비를 유지합니다; 필요한 경우 검은 막대가 표시됩니다.
- **채우기** — 전체 화면을 채우며, 필요한 경우 비디오를 자릅니다.
- **늘리기** — 필요한 경우 비디오를 왜곡하여 화면을 채웁니다.
- **원본** — 1:1의 기본 해상도를 유지합니다.

## 비디오 회전

잘못된 방향으로 녹화된 비디오의 경우 **추가 작업 → 회전**을 선택하고 플레이어를 나가지 않고 **0°**, **90°**, **180°** 또는 **270°**를 선택하여 화면을 회전합니다.

## 하드웨어 디코딩 (H.264 및 HEVC)

설정 → 플레이어 → 비디오에서 하드웨어 H.264 디코딩과 하드웨어 H.265 (HEVC) 디코딩을 독립적으로 활성화 / 비활성화할 수 있습니다. 하드웨어 디코딩은 더 빠르고, 배터리를 덜 사용하며, 더 시원하게 실행됩니다 — 하지만 드물게 (손상된 파일, 희귀한 프로파일) 하드웨어 디코딩을 비활성화하고 소프트웨어 FFmpeg 디코딩으로 대체해야 할 수 있습니다. Evervideo는 플레이어 추가 작업 메뉴에서 파일별로 이를 수행할 수 있습니다.

## VR 360° 뷰포트

Evervideo는 구면 비디오 파일을 위한 VR / 360° 뷰포트를 포함합니다. 360° 비디오를 재생할 때 드래그하여 주변을 둘러보고, 핀치하여 줌하면 Evervideo가 실시간으로 렌더링을 변형합니다.

## 재생 속도

플레이어 툴바의 속도 컨트롤을 탭하여 재생 속도를 변경합니다 — 분석을 위해 느리게 하거나 (0.25× 또는 0.5×), 튜토리얼과 강의를 위해 빠르게 합니다 (1.25×, 1.5×, 2×, 최대 3×). 속도 화면 오른쪽 상단 모서리의 구성 아이콘을 탭하여 더 세밀한 조정을 위한 정밀 모드로 전환합니다. 트랙별 피치 보정도 사용 가능합니다.

## 플레이어 대기열

플레이어 대기열을 보려면 플레이어의 대기열 버튼을 탭합니다. 대기열의 각 비디오에는 더 많은 작업이 있습니다 — 세 점을 탭하여 볼 수 있습니다. 대기열에서 비디오 순서를 바꾸려면 제목 근처의 재순서 표시기를 사용하여 새 위치로 드래그합니다.

## 슬립 타이머

설정 → 플레이어 → 슬립 타이머를 열고 켜고 재생이 자동으로 중지되기까지 원하는 시간을 선택합니다. 설정 → 플레이어 → 개인화 → 메인 화면 작업을 통해 슬립 타이머 버튼을 메인 플레이어 화면에 직접 추가할 수도 있습니다.

## 플레이어 북마크

플레이어의 추가 작업 메뉴에서 북마크 추가를 탭하여 긴 비디오 — 강의, 비디오 오디오북, 튜토리얼, 시간 단위 YouTube 다운로드 — 에서 자리를 저장합니다. 북마크는 비디오의 추가 작업 → 북마크 목록에 나타나며 세션 간에 유지됩니다.

## 추가 작업 메뉴

플레이어의 **추가 작업 "..."** 버튼을 탭하여 추가 기능에 접근합니다.

- **계속 재생** — 마지막 위치에서 대기열을 재개합니다.
- **검색** — 대기열에서 특정 비디오를 찾습니다.
- **북마크** — 북마크를 보고 이동합니다.
- **속도** — 재생 속도를 조정합니다.
- **최근 항목** — 최근 재생한 비디오.
- **즐겨찾기** — 즐겨찾기한 비디오.
- **오디오 이퀄라이저** — 오디오 이퀄라이저를 활성화합니다.
- **비디오 이퀄라이저** — 비디오 이퀄라이저를 활성화합니다.
- **오디오 트랙** — 오디오 스트림을 선택합니다.
- **비디오 트랙** — 비디오 스트림을 선택합니다.
- **자막** — 기본 / 보조 트랙, 외부 파일, 폰트.
- **회전** — 화면을 0° / 90° / 180° / 270° 회전합니다.
- **스케일링 모드** — 맞춤 / 채우기 / 늘리기 / 원본.
- **화면 속 화면** — PiP 모드로 전환합니다.
- **AirPlay** / **Chromecast** — 장치로 전송합니다.
- **슬립 타이머** — 재생을 중지하는 타이머를 설정합니다.
- **대기열을 재생 목록으로 저장** — 현재 대기열을 새 재생 목록으로 저장합니다.
- **대기열 삭제** — 대기열을 지우고 재생을 중지합니다.
- **설정** — 플레이어 설정으로 바로 이동합니다.
- **도움말** — 안내를 엽니다.

## 플레이어 설정

전체 플레이어 설정 트리는 [설정 가이드](/docs/guide/evervideo/evervideo-guide-settings/)에 문서화되어 있습니다. 가장 많이 사용되는 섹션:

- **일반** — 반복 모드, 셔플 모드, 플레이어 상태 저장, 재생 위치 저장, 건너뛰기 시간 간격.
- **비디오** — 하드웨어 H.264 / HEVC 디코딩, 비디오 이퀄라이저, 스케일링 모드, 회전, 디스플레이 모드, 선호 FPS, 선호 픽셀 형식, VR 뷰포트.
- **오디오** — 오디오 이퀄라이저, 샘플 레이트, 채널, IO 버퍼 지속 시간, 혼합 모드.
- **자막** — 기본 / 보조 트랙, 외부 파일 선택, 폰트, 보조 폰트.
- **장치** (iOS) — AirPlay / Chromecast.
- **개인화** — 플레이어 레이아웃 스타일 (최소 / 하단 / 클래식 / 앤틱), 메인 화면 작업, 잠금 화면 컨트롤.
- **재생 캐시** — 더 부드러운 스트리밍을 위한 디스크 캐시.
- **슬립 타이머** — 자동 정지.

## 접근성

Evervideo는 VoiceOver로 완전히 접근 가능합니다. 모든 구성 요소에 잘 설계된 레이블과 설명이 있습니다. VoiceOver가 활성화되면 앱이 텍스트 모드로 전환되어 의미 있는 요소만 표시됩니다 — 탐색 속도와 명확성을 향상시킵니다. 설정 → 접근성 → 텍스트 모드에서 텍스트 모드를 활성화할 수도 있습니다.

### VoiceOver로 슬라이더 조정

1. **슬라이더 선택** — VoiceOver가 발표할 때까지 왼쪽 또는 오른쪽으로 스와이프합니다.
2. **값 조정** — 슬라이더를 두 번 탭하고 길게 누른 다음 위나 아래로 드래그하여 값을 빠르게 변경합니다. VoiceOver는 변경할 때 새 값을 발표합니다.

다른 구성 요소는 시스템 제공 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

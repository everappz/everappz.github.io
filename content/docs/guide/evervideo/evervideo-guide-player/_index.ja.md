メディアプレーヤーは、再生を制御し、Evervideoのほとんどの機能を使用するアプリのメイン画面です。ビデオとオーディオファイルの両方を再生し、ハードウェアアクセラレートH.264およびHEVCデコードを備えたカスタムFFmpegベースのプレーヤー上に構築されています。使い方を探ってみましょう。

## メディアプレーヤーへのアクセス

コンパクトプレーヤーバーからフルスクリーンプレーヤーにアクセスできます。iPhoneでは、コンパクトプレーヤーはメイン画面の上部にあります。iPadとMacでは、左側（またはメインパネルの上部）にあります。フルスクリーンプレーヤーをコンパクトビューに戻すには、右下隅の閉じるボタンをタップするか、下にスワイプします。コンパクトプレーヤーを完全に非表示にするには、もう一度タップして下にスワイプします。

コンパクトプレーヤーはライブラリ、ファイルマネージャー、設定を閲覧している間も表示され続けるため、次のビデオを探している間もビデオを失うことはありません。

## 対応ビデオおよびオーディオ形式

Evervideoは、サポートされているデバイスでのハードウェアアクセラレートH.264およびHEVCデコードにより、バンドルされたFFmpegエンジンを通じてほぼすべての現代的なコンテナとコーデックを再生します。

- **ビデオコンテナ：** MP4、M4V、MKV、MOV、AVI、FLV、WMV、ASF、WebM、TS、M2TS、MTS、MPG、MPEG、OGV、3GP、3G2、F4V、RM、RMVB、VOB、DAT — その他多数。
- **ビデオコーデック：** H.264 (AVC)、H.265 (HEVC)、VP9、VP8、AV1、MPEG-2、MPEG-4、MJPEG、ProRes、Theora、WMV — さらにFFmpegがサポートするほぼすべての他のコーデック。
- **オーディオコーデック：** AAC、MP3、FLAC、ALAC、OGG / Vorbis、OPUS、AC-3、EAC-3、DTS、AMR、WMA、APE、TTA、MPC、WV — その他多数。
- **字幕フォーマット：** SRT、VTT (WebVTT)、ASS / SSA、および埋め込みテキストまたは画像字幕トラック。
- **ストリーミングプロトコル：** HTTP / HTTPS、HLS (m3u8)、RTSP（IPカメラとIPTV）、SMB / WebDAV / FTP / SFTP / NFS / DLNA経由の直接ファイルストリーミング。

これにより、MKVリップ、防犯カメラRTSPストリーム、AV1 webmウェブダウンロードを含む、遭遇する可能性のあるほぼすべてのビデオファイルをカバーします。

## 再生コントロール

プレーヤー画面の下部に、再生、一時停止、次へ、前へのボタンが表示されます。アプリ設定で有効にできる前にスキップと後ろにスキップ（デフォルト10秒）などのオプションボタンもあります。次へ / 前へボタンを長押しして早送りまたは巻き戻し。再生スライダーをドラッグして任意の位置にジャンプします。

## リピートとシャッフル

リピートボタンをタップしてリピートモードを切り替えます：

- **すべてリピート** — キュー内のすべてのビデオをループします。
- **1曲リピート** — 現在のビデオだけを繰り返します。
- **リピート停止** — 現在のビデオが終わったら一時停止します。
- **リピートなし** — キューを1回再生して繰り返しません。

**シャッフル**ボタンを使用してキュー内のビデオの順序をランダム化します。

## ピクチャーインピクチャー（PiP）

iPhoneとiPadでは、Evervideoはピクチャーインピクチャー（PiP）を完全にサポートしています。プレーヤー画面でPiPアイコンをタップするか、単にEvervideoをバックグラウンドに送ります — ビデオはすべての他のアプリの上にフローティングウィンドウで再生を続けます。フローティングウィンドウを任意のコーナーにドラッグし、ピンチでサイズ変更し、1回タップして基本的な再生 / 一時停止 / スキップコントロールを表示し、小さな展開ボタンをタップしてEvervideo全体に戻ります。

PiPはEvervideoが再生するすべてのビデオ形式で機能し、クラウドストリーミングファイルやRTSPストリームも含まれます。PiPは電話がロックされていても実行し続けます（Auto-Lock設定によります）。

## コンパクトプレーヤー

コンパクトプレーヤーは、ライブラリ、ファイルマネージャー、設定を閲覧している間もアプリのすべての画面の上部に表示される永続的なミニプレーヤーです。タップしてフルスクリーンプレーヤーに展開し、下にスワイプして再び折りたたみます。

## AirPlay 2

AirPlayのためにプレーヤーのAirPlayボタンをタップします。EvervideoはAirPlay 2をサポートしているため、Apple TV、HomePod、またはAirPlay 2互換スピーカーやスマートTVにビデオをストリーミングできます。複数のAirPlayデバイスを持つ設定では、複数のレシーバーに同時にオーディオをルーティングできます。

## Google Chromecast

Google Castユーザーには、プレーヤーにキャストアイコンが表示されます。タップしてデバイスを選択してキャストを開始します。電話とキャストレシーバーが同じWi-Fiネットワーク上にあることを確認してください。すべてのコーデックがChromecastハードウェアでサポートされているわけではありません — 一部のファイルはトランスコーディングが必要な場合があります。

## RTSPライブストリーム

EvervideoはRTSPソースを直接再生できます — IPカメラ、ドアベルカメラ、IPTVストリーム、放送フィード、その他の`rtsp://` URL。ファイル → オンラインリンク → リンクを追加でストリームをRTSP接続として追加し、タップして視聴を開始します。RTSPストリームはピクチャーインピクチャー、コンパクトプレーヤーで機能し、通常のビデオと同様にAirPlay 2とChromecastでキャストできます。

## オーディオトラック選択

複数のオーディオトラックを持つビデオ（コメンタリー、代替言語吹き替え、ディレクターのトラック）の場合、プレーヤーのその他のアクションボタンをタップしてオーディオトラックを選択し、希望するトラックを選択します。これは外国語映画、複数の吹き替えを持つBDMV / MKVファイル、コメンタリートラック付きの教育コンテンツに不可欠です。

## ビデオトラック選択

一部のビデオファイルには複数のビデオストリームが含まれています（マルチアングルBlu-ray、代替カット）。その他のアクションメニューからビデオトラックを選択して、再生中に切り替えます。

## 字幕 — 内部と外部

Evervideoは字幕のきめ細かい制御を提供します：

- **字幕トラック** — ファイルに埋め込まれたトラックから選択。
- **外部字幕ファイル** — デバイス、iCloud Drive、または接続されたクラウドサービスから`.srt`、`.vtt`、`.ass`、`.ssa`ファイルを読み込み。
- **Libassレンダリング** — 高度なASS / SSAスタイリング（フォント、色、位置、カラオケエフェクト）はバンドルされたlibassのおかげで正しくレンダリングされます。
- **フォント選択** — プライマリ字幕のカスタムフォントとセカンダリ字幕用の別フォントを選択。バンドルされたフォントとデバイスにインストールされているフォントが利用可能。

これらすべてを再生前に設定 → 字幕で設定するか、プレーヤーからその他のアクション → 字幕を使用してその場で切り替えます。

## オーディオイコライザー

Evervideoには、ヘッドフォン、スピーカー、またはハイファイ設備向けにビデオサウンドトラックを調整するためのフルオーディオイコライザーが含まれています。ボリュームビューのイコライザーアイコンをタップ（またはプレーヤーのその他のアクションメニューのオーディオイコライザーアクション）し、右上隅のスイッチでイコライザーをオンにし、プリセットを選択するかバンドスライダーをドラッグして独自のプリセットを作成します。カスタムプリセットはエクスポートとインポートができるため、デバイス間で共有したりバックアップしたりできます。

## ビデオイコライザー

画像を調整するために、Evervideoは専用のビデオイコライザーを提供します — 再生中にリアルタイムで輝度、コントラスト、彩度、色相を調整します。オーディオイコライザーと同様に、カスタムビデオプリセットは共有またはバックアップのためにエクスポートとインポートができます。晴れた日に暗いシーンを明るくしたり、色褪せたコンテンツの彩度を上げたり、冷たい色合いを温めたりするために使用します。

## ビデオスケーリングモード

ビデオが画面を埋める方法を選択します：

- **フィット** — 元のアスペクト比を維持し、必要に応じてレターボックス。
- **フィル** — 必要に応じてビデオをトリミングして画面全体を埋める。
- **ストレッチ** — 必要に応じて歪めて画面を埋めるようにビデオを伸ばす。
- **オリジナル** — ネイティブ解像度を1:1で維持。

## ビデオ回転

向きが間違って録画されたビデオには、**その他のアクション → 回転**を選択して、**0°**、**90°**、**180°**、または**270°**を選択してプレーヤーを離れずに画像を回転させます。

## ハードウェアデコード（H.264およびHEVC）

設定 → プレーヤー → ビデオで、ハードウェアデコードH.264とハードウェアデコードH.265 (HEVC)を独立して有効化 / 無効化できます。ハードウェアデコードは速く、バッテリーをより少なく使用し、より低温で動作します — しかし稀なケース（破損したファイル、エキゾチックなプロファイル）ではハードウェアデコードを無効にしてソフトウェアFFmpegデコードにフォールバックする必要があるかもしれません。Evervideoではプレーヤーのその他のアクションメニューからファイルごとにそれができます。

## VR 360°ビューポート

Evervideoには球面ビデオファイル用のVR / 360°ビューポートが含まれています。360°ビデオを再生する際、ドラッグして見回し、ピンチしてズームでき、Evervideoはリアルタイムでレンダリングを歪めます。

## 再生速度

プレーヤーツールバーの速度コントロールをタップして再生速度を変更します — 分析のために遅くする（0.25×または0.5×）か、チュートリアルや講義を速くする（1.25×、1.5×、2×、最大3×まで）。速度画面の右上隅の設定アイコンをタップして、より細かい調整のある精密モードに切り替えます。トラックごとのピッチ補正も利用可能です。

プレーヤーキューを見るには、プレーヤーのキューボタンをタップします。キュー内の各ビデオにはその他のアクションがあります — 三点をタップして表示します。キュー内のビデオを並べ替えるには、タイトル付近の並べ替えインジケーターを使用して新しい位置にドラッグします。

## スリープタイマー

設定 → プレーヤー → スリープタイマーを開き、オンにして、自動停止前に再生を続ける時間を選択します。設定 → プレーヤー → パーソナライズ → メイン画面アクションを通じて、スリープタイマーボタンをメインプレーヤー画面に直接追加することもできます。

## プレーヤーブックマーク

その他のアクションメニューからブックマークを追加をタップして、長いビデオ — 講義、ビデオ上のオーディオブック、チュートリアル、1時間のYouTubeダウンロード — の場所を保存します。ブックマークはビデオのその他のアクション → ブックマークリストに表示され、セッション間で持続します。

## その他のアクションメニュー

プレーヤーの**その他のアクション「...」**ボタンをタップして追加機能にアクセスします。

- **再生を続行** — 最後の位置からキューを再開。
- **検索** — キュー内の特定のビデオを検索。
- **ブックマーク** — ブックマークを表示してジャンプ。
- **速度** — 再生速度を調整。
- **最近の項目** — 最近再生したビデオ。
- **お気に入り** — お気に入りにしたビデオ。
- **オーディオイコライザー** — オーディオイコライザーを有効化。
- **ビデオイコライザー** — ビデオイコライザーを有効化。
- **オーディオトラック** — オーディオストリームを選択。
- **ビデオトラック** — ビデオストリームを選択。
- **字幕** — プライマリ / セカンダリトラック、外部ファイル、フォント。
- **回転** — 画像を0° / 90° / 180° / 270°回転。
- **スケーリングモード** — フィット / フィル / ストレッチ / オリジナル。
- **ピクチャーインピクチャー** — PiPモードに入る。
- **AirPlay** / **Chromecast** — デバイスに送信。
- **スリープタイマー** — 再生を停止するタイマーを設定。
- **キューをプレイリストに保存** — 現在のキューを新しいプレイリストとして保存。
- **キューを削除** — キューをクリアして再生を停止。
- **設定** — プレーヤー設定に直接ジャンプ。
- **ヘルプ** — ガイダンスを開く。

## プレーヤー設定

プレーヤー設定ツリー全体は[設定ガイド](/docs/guide/evervideo/evervideo-guide-settings/)に記載されています。最もよく使用されるセクション：

- **一般** — リピートモード、シャッフルモード、プレーヤー状態を保存、再生位置を保存、スキップ時間間隔。
- **ビデオ** — ハードウェアデコードH.264 / HEVC、ビデオイコライザー、スケーリングモード、回転、表示モード、優先FPS、優先ピクセルフォーマット、VRビューポート。
- **オーディオ** — オーディオイコライザー、サンプルレート、チャンネル、IOバッファ時間、混合モード。
- **字幕** — プライマリ / セカンダリトラック、外部ファイル選択、フォント、セカンダリフォント。
- **デバイス**（iOS）— AirPlay / Chromecast。
- **パーソナライズ** — プレーヤーレイアウトスタイル（ミニマル / ボトム / アンティーク / クラシカル）、メイン画面アクション、ロック画面コントロール。
- **再生キャッシュ** — スムーズなストリーミングのためのディスクキャッシュ。
- **スリープタイマー** — 自動停止。

## アクセシビリティ

EvervideoはVoiceOverで完全にアクセス可能です。すべてのコンポーネントには適切に設計されたラベルと説明があります。VoiceOverが有効な場合、アプリはテキストモードに切り替わり、意味のある要素のみが表示されます — ナビゲーション速度と明確さが向上します。設定 → アクセシビリティ → テキストモードでテキストモードを有効にすることもできます。

### VoiceOverでスライダーを調整する

1. **スライダーを選択** — VoiceOverが読み上げるまで左右にスワイプします。
2. **値を調整** — スライダーをダブルタップして長押しし、上下にドラッグして素早く値を変更します。VoiceOverは変更ごとに新しい値を読み上げます。

その他のコンポーネントは、システムが提供するVoiceOverパターンを使用して期待通りに機能します。

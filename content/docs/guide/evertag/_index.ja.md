EvertagはiPhone、iPad、Macユーザーのための最高の音楽タグエディタです。音楽コレクションを整理し、壊れたメタデータを素早く修正するのに役立ちます。

## 30以上のオーディオフォーマット

Evertagは、MP3、OGG、OGA、FLAC、MPC、WV、SPX、OPUS、TTA、M4A、M4R、M4B、M4P、MP4、3G2、M4V、WMA、ASF、AIF、AIFF、AFC、AIFC、WAV、APE、MOD、MODULE、NTS、WOW、S3M、IT、XMなど、幅広い人気のオーディオフォーマットをシームレスに処理します。

## 120以上のタグ

Evertagでは、TITLE、ARTIST、ALBUM、COMMENT、GENRE、TRACK_NUMBER、YEAR、FRONT_COVER、DURATION、BITRATE、SAMPLE_RATE、CHANNELS、ALBUM_ARTIST、COMPOSER、BPM、DISK_NUMBER、RATING、歌詞（タイムスタンプ付き歌詞を含む）など、最も人気のあるオーディオタグを変更できます。拡張エディタに切り替えると、MusicBrainz識別子やreplay-gain値を含む、サポートされているすべてのフィールドにアクセスできます。

## バッチモード

複数のファイルを選択して1つのエンティティのように編集することで時間を節約できます。音楽コレクション全体の共通メタデータへの変更を迅速かつ効率的に適用できます。

## アルバムカバー

アプリ内でアルバムカバーを簡単に更新して音楽ファイルをカスタマイズし、写真ライブラリ、クラウド、またはダウンロードしたファイルからアートワークを選択し、オーディオタグに保存する前のカバーのスケール方法を選択できます。

## 自動修正

Evertagは不足しているオーディオタグを識別し、壊れたメタデータを自動的に修正して、音楽ライブラリを常に完璧な状態に保ちます。MusicBrainzデータベースを使った自動検索または手動検索を使用し、文字化けを修正するためにエンコーディングを正規化し、最大のプレーヤー互換性のためにID3v2.3とID3v2.4の間で選択できます。

## クラウド統合

音楽ライブラリがクラウドにある場合は、サポートされているサービス全ラインナップとEvertagをシームレスに接続できます：iCloud Drive、Google Drive、Dropbox、OneDrive、Box、MEGA、Yandex.Disk、pCloud、Synology Drive、MediaFire、WD My Cloud Home、InfiniCLOUD (TeraCLOUD)、HiDrive、OpenDrive、MyDrive、Put.io、Cloud Mail.ru、Baidu Pan（百度网盘）。SMBおよびWebDAVを使って個人NAS、Apple Time Capsule、またはコンピュータに接続することもできます。

## ファイル管理

Evertagには、すべての基本的なファイル操作をサポートする組み込みファイルマネージャーが含まれています。また、便利なopen-in-place機能を使ってデバイス上で直接オーディオファイルを編集でき、外部アプリケーションからファイルをコピーする必要がなくなります。クイックアクセス、最近開いたファイル、お気に入りにより、最もよく使うファイルにすばやく戻ることができます。

## Wi-Fi転送

Wi-Fi Drive機能を使ってコンピュータのウェブブラウザでオーディオファイルをデバイスに簡単に転送するか、iPhone、iPadではケーブルを使ってiTunes File Sharingを利用できます。

## データセキュリティ

データが安全であることを安心して知ることができます。Evertagではアプリケーションにパスワードを設定でき、音楽コレクションに追加の保護レイヤーを加えることができます。

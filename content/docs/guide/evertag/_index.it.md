Evertag è il miglior editor di tag musicali per gli utenti di iPhone, iPad e Mac. Ti aiuta a mantenere la tua collezione musicale organizzata e a correggere rapidamente i metadati danneggiati.

## 30+ formati audio

Evertag gestisce senza problemi un'ampia gamma di formati audio popolari, tra cui MP3, OGG, OGA, FLAC, MPC, WV, SPX, OPUS, TTA, M4A, M4R, M4B, M4P, MP4, 3G2, M4V, WMA, ASF, AIF, AIFF, AFC, AIFC, WAV, APE, MOD, MODULE, NTS, WOW, S3M, IT, XM.

## 120+ tag

Evertag ti permette di modificare i tag audio più popolari come TITLE, ARTIST, ALBUM, COMMENT, GENRE, TRACK_NUMBER, YEAR, FRONT_COVER, DURATION, BITRATE, SAMPLE_RATE, CHANNELS, ALBUM_ARTIST, COMPOSER, BPM, DISK_NUMBER, RATING, testi (inclusi i testi con timestamp) e molti altri. Passa all'editor esteso per accedere a ogni campo supportato, inclusi gli identificatori MusicBrainz e i valori replay-gain.

## Modalità batch

Risparmia tempo selezionando più file e modificandoli come se fossero una singola entità. Applica rapidamente ed efficacemente le modifiche ai metadati comuni nella tua collezione musicale.

## Copertine degli album

Personalizza i tuoi file musicali aggiornando facilmente le copertine degli album nell'app, scegli le copertine dalla libreria Foto, dal cloud o dai file scaricati, e scegli come vengono ridimensionate prima di essere salvate nei tag audio.

## Correzione automatica

Evertag identifica i tag audio mancanti e corregge automaticamente i metadati danneggiati, garantendo che la tua libreria musicale rimanga impeccabile. Usa la ricerca automatica o la ricerca manuale basata sul database MusicBrainz, normalizza le codifiche per correggere i caratteri illeggibili, e scegli tra ID3v2.3 e ID3v2.4 per la massima compatibilità con i lettori.

## Integrazione cloud

Se la tua libreria musicale si trova nel cloud, connetti Evertag senza problemi con la gamma completa di servizi supportati: iCloud Drive, Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk, pCloud, Synology Drive, MediaFire, WD My Cloud Home, InfiniCLOUD (TeraCLOUD), HiDrive, OpenDrive, MyDrive, Put.io, Cloud Mail.ru e Baidu Pan (百度网盘). Puoi anche connettere NAS personali, Apple Time Capsule o computer tramite SMB e WebDAV.

## Gestione dei file

Evertag include un gestore file integrato che supporta tutte le operazioni fondamentali sui file. Puoi anche modificare i file audio direttamente sul tuo dispositivo usando la comoda funzione di apertura in loco, eliminando la necessità di copiare file da applicazioni esterne. Accesso rapido, Aperti di recente e Preferiti ti aiutano a tornare rapidamente ai file con cui lavori di più.

## Trasferimento Wi-Fi

Trasferisci file audio sul tuo dispositivo senza difficoltà usando un browser web sul computer con la funzione Wi-Fi Drive, oppure usa iTunes File Sharing con un cavo su iPhone e iPad.

## Sicurezza dei dati

Stai tranquillo sapendo che i tuoi dati sono al sicuro. Evertag ti consente di impostare una password per l'applicazione, aggiungendo un ulteriore livello di protezione alla tua collezione musicale.

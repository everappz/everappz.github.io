## Passer à Premium

Mettez à niveau l'application vers la version Premium pour supprimer toutes les limites. La version gratuite de l'application propose un achat intégré unique à vie et deux options d'abonnement qui vous permettent de supprimer toutes les restrictions et de passer à la version Premium.

Aussi, gardez à l'esprit que **Family Sharing** est activé pour tous les achats et plans, vous pouvez donc partager la version Premium avec les membres de votre famille.

Vous pouvez en savoir plus sur les achats et la version Premium ici : [Quelle est la différence entre Evertag et Evertag Premium](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/).

## Partage des achats entre iOS et Mac

Gardez à l'esprit que les achats à vie et les abonnements sont partagés entre iOS et Mac, en utilisant iCloud pour synchroniser ces informations. Si vous avez la version premium sur votre appareil iOS, assurez-vous d'avoir la dernière version installée et qu'iCloud est activé. Démarrez l'application sur iOS et attendez une minute pour que vos informations d'achat soient téléversées sur iCloud.

Vous pouvez également essayer d'appuyer sur le bouton «Restaurer les achats» dans les paramètres de l'application. Ensuite, installez la dernière version de l'application depuis l'App Store sur votre Mac et démarrez l'application. Assurez-vous d'avoir une connexion Internet et d'utiliser le même compte iCloud et App Store sur votre Mac que celui utilisé sur votre appareil iOS. Attendez une minute pour que l'application télécharge les informations d'achat depuis iCloud. La version premium devrait s'activer automatiquement sur votre Mac.

## Restaurer les achats sur un nouvel appareil

Pour restaurer votre achat sur le nouvel appareil, utilisez simplement le menu **Achats -> Restaurer les achats**. Vous verrez la liste de vos achats. Si vous ne voyez pas tous vos achats, vérifiez si l'appareil est connecté au même compte iTunes que celui utilisé pour effectuer les achats, et assurez-vous qu'iCloud est activé.

## Essayer Premium gratuitement

Vous pouvez passer à la version Premium gratuitement mais pour une durée limitée uniquement en utilisant ce menu. Regardez simplement une publicité ou parlez de cette application à vos amis pour obtenir la version Premium gratuitement.

## Gestionnaire de fichiers

Le gestionnaire de fichiers prend en charge l'accès aux comptes de stockage cloud connectés et offre des opérations par lot pour une gestion rapide de plusieurs fichiers.

### Transferts de fichiers

Choisissez votre préférence réseau lors du téléchargement de fichiers sur votre appareil — Wi-Fi uniquement ou Wi-Fi et données cellulaires.

### Limite de transfert de fichiers en parallèle

Définissez le nombre maximum de fichiers transférés simultanément. Un nombre plus élevé peut accélérer les transferts mais peut utiliser plus de ressources système et de batterie.

### Tâches de transfert de fichiers

Ouvrez la file de transfert pour afficher et gérer les tâches de téléchargement/téléversement en cours et terminées.

### Transferts en arrière-plan

Vous pouvez activer les téléchargements même lorsque l'application s'exécute en arrière-plan. Sachez que si cette opération consomme une quantité significative d'énergie, le système d'exploitation peut suspendre l'application.

### Enregistrer les fichiers téléchargés dans

Définissez votre répertoire de téléchargements par défaut ou choisissez d'être invité à chaque fois pour choisir où enregistrer les fichiers.

### Afficher les noms de fichiers complets

Activez cette option si vous avez besoin de voir les noms de fichiers complets, y compris les extensions, dans le gestionnaire de fichiers.

### Miniatures pour les fichiers

Choisissez si les miniatures des fichiers dans le stockage cloud sont chargées (et si les données cellulaires sont autorisées), ouvrez le dossier de cache ou supprimez les miniatures en cache pour libérer de l'espace de stockage.

### Supprimer les fichiers temporaires

Videz le dossier de cache de l'application pour récupérer de l'espace de stockage.

## Éditeur de tags audio

Dans cette section, vous pouvez configurer l'éditeur de tags audio intégré.

### Mise à l'échelle des pochettes d'album

Choisissez la méthode de mise à l'échelle (Petite / Moyenne / Grande / Très grande) pour la pochette d'album lors de son enregistrement dans les tags audio, ou désactivez la mise à l'échelle pour conserver la taille originale. La mise à l'échelle à taille originale fait partie des fonctionnalités de personnalisation Premium.

### Options d'enregistrement des tags

- **ID3v2.4** — Enregistrez les tags au format ID3v2.4 lorsque possible. Désactivez si les anciens lecteurs ne lisent pas les tags correctement et l'application reviendra à ID3v2.3.
- **Tags dupliqués** — Dupliquez les champs de métadonnées communs dans les deux sections de tags du fichier pour la compatibilité avec les anciens lecteurs.

### Mettre à jour les fichiers en ligne

Contrôlez la façon dont l'application met à jour les métadonnées pour les fichiers audio stockés sur un service cloud connecté : demander avant chaque sauvegarde, toujours téléverser ou ne jamais téléverser (modifications locales uniquement).

### Modifier les fichiers en ligne

Décidez si la copie téléchargée localement d'un fichier cloud est supprimée automatiquement après avoir terminé l'édition, ou conservée sur votre appareil pour un accès hors ligne.

### Boutons de l'écran principal

Choisissez quelles actions apparaissent sur l'écran principal de l'éditeur de tags audio — Recherche automatique des tags audio, Recherche manuelle des tags audio, Rechercher la pochette d'album, Enregistrer la pochette d'album, Normaliser l'encodage, Supprimer les tags audio — et si **Afficher les tags étendus** et **Modifier les fichiers simultanément** sont toujours activés par défaut.

Dans cette section, vous pouvez personnaliser les paramètres de l'interface utilisateur selon vos préférences.

### Icône de l'application

Choisissez une autre icône d'écran d'accueil / Dock pour l'application, ou conservez celle par défaut. Certaines icônes alternatives font partie des fonctionnalités de personnalisation Premium.

### Schéma de couleurs

Personnalisez le thème de l'interface utilisateur et activez le mode sombre ici. Lorsque l'option par défaut est sélectionnée, l'application adaptera son schéma de couleurs en fonction des paramètres d'apparence de votre appareil.

### Apparence des éléments dans la liste

Personnalisez la façon dont les éléments sont affichés dans la liste, ce qui est particulièrement utile pour les appareils avec des écrans plus petits. Vous pouvez choisir si l'application doit ajuster automatiquement la hauteur en fonction du contenu ou afficher des icônes plus petites dans les cellules de la liste pour fournir plus d'espace pour le texte.

### Limite de chargement du contenu

Définissez le nombre maximum d'éléments que l'application charge par page dans un dossier (100, 500, 1000, 5000 ou sans limite). La pagination maintient la rapidité de chargement des dossiers ; désactivez la limite si vous préférez tout charger d'un coup.

### Style du menu principal (iPad et Mac)

Sur iPad et Mac, choisissez entre un menu principal simple ou un menu groupé extensible dans la barre latérale.

### Style de l'écran des fichiers locaux

Choisissez une mise en page simple ou groupée pour l'écran Fichiers locaux.

### Style du menu contextuel

Sélectionnez un style pour le menu contextuel qui apparaît lorsque vous appuyez sur le bouton «Plus d'actions» — menu système ou la feuille de fond personnalisée de l'application.

## Paramètres de fenêtre (iPad et Mac)

Sur iPad et Mac, décidez si l'écran doit rester actif pendant que vous utilisez l'application, et configurez d'autres options au niveau de la fenêtre.

## Aggiorna a Premium

Aggiorna l'applicazione alla versione Premium per rimuovere tutti i limiti. La versione gratuita dell'applicazione offre un acquisto in-app a vita e due opzioni di abbonamento che ti consentono di rimuovere tutte le restrizioni e aggiornare alla versione Premium.  

Tieni inoltre presente che **Family Sharing** è abilitato per tutti gli acquisti e i piani, quindi puoi condividere la versione Premium con i membri della tua famiglia.  

Puoi leggere di più sugli acquisti e sulla versione Premium qui: [Qual è la differenza tra Evertag e Evertag Premium](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/).

## Condivisione degli acquisti tra iOS e Mac

Tieni presente che gli acquisti a vita e gli abbonamenti sono condivisi tra iOS e Mac, usando iCloud per sincronizzare queste informazioni. Se hai la versione premium sul tuo dispositivo iOS, assicurati di avere installata la versione più recente e che iCloud sia abilitato. Avvia l'app su iOS e attendi un minuto per il caricamento delle informazioni di acquisto su iCloud.

Puoi anche provare a premere il pulsante "Ripristina acquisti" nelle impostazioni dell'app. Dopodiché, installa la versione più recente dell'app dall'App Store sul tuo Mac e avvia l'app. Assicurati di avere una connessione Internet e di utilizzare lo stesso account iCloud e App Store sul tuo Mac che hai usato sul tuo dispositivo iOS. Attendi un minuto per il download delle informazioni di acquisto da iCloud. La versione premium dovrebbe attivarsi automaticamente sul tuo Mac.

## Ripristina acquisti su un nuovo dispositivo

Per ripristinare il tuo acquisto sul nuovo dispositivo, usa semplicemente il menu **Acquisti -> Ripristina acquisti**. Vedrai l'elenco dei tuoi acquisti. Se non vedi tutti i tuoi acquisti, controlla se il dispositivo è connesso allo stesso account iTunes che è stato utilizzato per effettuare gli acquisti e assicurati che iCloud sia abilitato.

## Prova Premium gratuitamente

## Gestore file

Il gestore file supporta l'accesso agli account di archiviazione cloud connessi e offre operazioni batch per la gestione rapida di più file.

### Trasferimenti file

Scegli la tua preferenza di rete quando scarichi file sul dispositivo — solo Wi-Fi o Wi-Fi e dati cellulare.

### Limite di trasferimento file in parallelo

Imposta il numero massimo di file trasferiti contemporaneamente. Un numero più alto può velocizzare i trasferimenti ma potrebbe usare più risorse di sistema e batteria.

### Attività di trasferimento file

Apri la coda di trasferimento per visualizzare e gestire le attività di upload/download in corso e completate.

### Trasferimenti in background

Puoi abilitare i download anche quando l'app è in esecuzione in background. Tieni presente che se questa operazione consuma una quantità significativa di energia, il sistema operativo potrebbe sospendere l'app.

### Salva i file scaricati in

Definisci la directory di download predefinita o scegli di ricevere una richiesta ogni volta per scegliere dove salvare i file.

### Mostra nomi file completi

Attiva questa opzione se devi visualizzare i nomi file completi, incluse le estensioni, nel gestore file.

### Miniature per i file

Scegli se le miniature per i file nell'archiviazione cloud vengono caricate (e se i dati cellulare sono consentiti), apri la cartella cache, o elimina le miniature memorizzate nella cache per liberare spazio di archiviazione.

### Elimina file temporanei

Svuota la cartella cache dell'applicazione per recuperare spazio di archiviazione.

## Editor tag audio

In questa sezione puoi configurare l'editor di tag audio integrato.

{{< cards cols="1">}}
  {{< card title="" subtitle="Schermata Editor tag Impostazioni Evertag" image="/docs/guide/evertag/img/settings-tag-editor.webp" >}}
{{< /cards >}}

### Ridimensionamento copertina album

Scegli il metodo di ridimensionamento (Piccolo / Medio / Grande / Extra grande) per la copertina dell'album quando la salvi nei tag audio, o disattiva il ridimensionamento per mantenere la dimensione originale. Il ridimensionamento della dimensione originale fa parte delle funzionalità di personalizzazione Premium.

### Opzioni di salvataggio tag

- **ID3v2.4** — Salva i tag nel formato ID3v2.4 quando possibile. Disabilita se i lettori più vecchi non leggono correttamente i tag e l'app tornerà a ID3v2.3.
- **Tag duplicati** — Duplica i campi di metadati comuni in entrambe le sezioni tag del file per la compatibilità con i lettori più vecchi.

### Aggiorna file online

Controlla come l'app aggiorna i metadati per i file audio archiviati su un servizio cloud connesso: chiedi prima di ogni salvataggio, carica sempre, o non caricare mai (solo modifiche locali).

### Modifica file online

Decidi se la copia scaricata localmente di un file cloud viene eliminata automaticamente dopo aver finito di modificare, o mantenuta sul dispositivo per l'accesso offline.

### Pulsanti schermata principale

Scegli quali azioni appaiono nella schermata principale dell'editor tag audio — Ricerca automatica tag audio, Ricerca manuale tag audio, Cerca copertina album, Salva copertina album, Normalizza codifica, Elimina tag audio — e se **Mostra tag estesi** e **Modifica file contemporaneamente** sono sempre attivi per impostazione predefinita.

## Personalizzazione

In questa sezione puoi personalizzare le impostazioni dell'interfaccia utente secondo le tue preferenze.

{{< cards cols="1">}}
  {{< card title="" subtitle="Schermata Personalizzazione Impostazioni Evertag" image="/docs/guide/evertag/img/settings-personalization.webp" >}}
{{< /cards >}}

### Icona applicazione

Scegli un'icona alternativa per la schermata Home / Dock dell'app, o mantieni quella predefinita. Alcune icone alternative fanno parte delle funzionalità di personalizzazione Premium.

### Schema colori

Personalizza il tema dell'interfaccia utente e abilita la modalità scura qui. Quando è selezionata l'opzione predefinita, l'applicazione adatterà il suo schema di colori in base alle impostazioni di aspetto del dispositivo.

### Aspetto degli elementi nell'elenco

Personalizza come vengono visualizzati gli elementi nell'elenco, il che è particolarmente utile per i dispositivi con schermi più piccoli. Puoi scegliere se l'app deve regolare automaticamente l'altezza in base al contenuto o mostrare icone più piccole nelle celle dell'elenco per fornire più spazio per il testo.

### Limite di caricamento del contenuto

Imposta il numero massimo di elementi che l'app carica per pagina in una cartella (100, 500, 1000, 5000 o nessun limite). La paginazione mantiene il caricamento delle cartelle veloce; disabilita il limite se preferisci caricare tutto in una volta.

### Stile del menu principale (iPad e Mac)

Su iPad e Mac, scegli tra un menu principale semplice o un menu espandibile e raggruppato nella barra laterale.

### Stile della schermata file locali

Scegli un layout semplice o raggruppato per la schermata File locali.

### Stile del menu contestuale

Seleziona uno stile per il menu contestuale che appare quando tocchi il pulsante 'Altre azioni' — menu di sistema o il foglio inferiore personalizzato dell'app.

## Impostazioni finestra (iPad e Mac)

Su iPad e Mac, decidi se lo schermo deve rimanere attivo mentre usi l'applicazione, e configura altre opzioni a livello di finestra.

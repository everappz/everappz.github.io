इस स्क्रीन पर, आप एप्लिकेशन सेटिंग्स एक्सेस कर सकते हैं और इसे Premium संस्करण में अपग्रेड कर सकते हैं। Settings root निम्नलिखित अनुभागों में समूहीकृत है:

## Premium में अपग्रेड करें

सभी सीमाएँ हटाने के लिए एप्लिकेशन को Premium संस्करण में अपग्रेड करें। एप्लिकेशन का मुफ़्त संस्करण एक lifetime in-app purchase और दो subscription विकल्प प्रदान करता है जो आपको सभी प्रतिबंध हटाने और Premium संस्करण में अपग्रेड करने की अनुमति देते हैं।

साथ ही, कृपया ध्यान रखें कि सभी खरीदारी और plan के लिए **Family Sharing** सक्षम है, इसलिए आप अपने परिवार के सदस्यों के साथ Premium संस्करण साझा कर सकते हैं।

आप खरीदारी और Premium संस्करण के बारे में यहाँ अधिक पढ़ सकते हैं: [Evertag और Evertag Premium में क्या अंतर है](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/)।

## iOS और Mac के बीच खरीदारी साझा करना

कृपया ध्यान रखें कि lifetime purchases और subscriptions iOS और Mac के बीच साझा होते हैं, इस जानकारी को sync करने के लिए iCloud का उपयोग करते हुए। यदि आपके iOS डिवाइस पर premium संस्करण है, तो कृपया सुनिश्चित करें कि आपने latest version इंस्टॉल किया हुआ है और iCloud सक्षम है। iOS पर ऐप शुरू करें और अपनी खरीदारी की जानकारी iCloud पर upload होने के लिए एक मिनट प्रतीक्षा करें।

आप ऐप सेटिंग्स में "Restore Purchases" बटन दबाने का भी प्रयास कर सकते हैं। उसके बाद, अपने Mac पर App Store से latest app version इंस्टॉल करें और ऐप शुरू करें। सुनिश्चित करें कि आपके पास internet connection है और आप अपने Mac पर उसी iCloud और App Store account का उपयोग कर रहे हैं जो आपने अपने iOS डिवाइस पर उपयोग किया था। iCloud से खरीदारी की जानकारी download होने के लिए एक मिनट प्रतीक्षा करें। premium संस्करण आपके Mac पर स्वचालित रूप से activate हो जाना चाहिए।

## नए डिवाइस पर खरीदारी पुनर्स्थापित करें

नए डिवाइस पर अपनी खरीदारी पुनर्स्थापित करने के लिए बस **खरीदारी → खरीदारी पुनर्स्थापित करें** मेनू का उपयोग करें। आपको अपनी खरीदारी की सूची दिखाई देगी। यदि आपको सभी खरीदारी दिखाई नहीं देती हैं, तो जाँचें कि डिवाइस उसी iTunes account से जुड़ा है जो खरीदारी करने के लिए उपयोग किया गया था, और सुनिश्चित करें कि iCloud सक्षम है।

## मुफ़्त में Premium आज़माएँ

आप इस मेनू का उपयोग करके सीमित समय के लिए Premium संस्करण में मुफ़्त अपग्रेड कर सकते हैं। Premium संस्करण मुफ़्त में पाने के लिए बस एक advertisement देखें या अपने दोस्तों को इस ऐप के बारे में बताएँ।

## खरीदारी

आप इस मेनू से पिछली खरीदारी पुनर्स्थापित कर सकते हैं। यदि आपको activation errors आती हैं, तो "Restore Purchases at App Launch" विकल्प सक्षम करने का प्रयास करें।

## नया क्या है

यह मेनू नई version जारी होने के बाद उपलब्ध है। आप अपडेट किए गए एप्लिकेशन में नया क्या है देख सकते हैं।

## पासकोड

यदि आप अपना एप्लिकेशन डेटा सुरक्षित करना चाहते हैं तो पासवर्ड सुरक्षा स्क्रीन सक्रिय करता है।

## फ़ाइल मैनेजर

फ़ाइल मैनेजर कनेक्टेड cloud storage accounts तक पहुँच का समर्थन करता है और कई फ़ाइलों के त्वरित प्रबंधन के लिए batch operations प्रदान करता है।

### फ़ाइल ट्रांसफर

अपने डिवाइस पर फ़ाइलें download करते समय अपनी नेटवर्क प्राथमिकता चुनें — केवल Wi-Fi या Wi-Fi और Cellular Data।

### समानांतर फ़ाइल ट्रांसफर सीमा

एक साथ transfer की जाने वाली फ़ाइलों की अधिकतम संख्या सेट करें। अधिक संख्या transfers को गति दे सकती है लेकिन अधिक system resources और बैटरी का उपयोग कर सकती है।

### फ़ाइल ट्रांसफर कार्य

चल रहे और पूर्ण हुए upload/download कार्यों को देखने और प्रबंधित करने के लिए transfer queue खोलें।

### बैकग्राउंड ट्रांसफर

आप ऐप बैकग्राउंड में चलने पर भी downloads सक्षम कर सकते हैं। कृपया ध्यान रखें कि यदि यह operation महत्वपूर्ण मात्रा में ऊर्जा खपत करता है, तो operating system ऐप को suspend कर सकता है।

### डाउनलोड की गई फ़ाइलें यहाँ सेव करें

अपनी डिफ़ॉल्ट downloads directory परिभाषित करें या हर बार फ़ाइलें कहाँ सेव करनी हैं चुनने के लिए prompt करने का विकल्प चुनें।

### पूर्ण फ़ाइलनाम दिखाएँ

यदि आपको फ़ाइल मैनेजर में extensions सहित पूर्ण फ़ाइलनाम देखने की ज़रूरत है तो यह विकल्प सक्रिय करें।

### फ़ाइलों के लिए thumbnails

चुनें कि cloud storage में फ़ाइलों के thumbnails लोड किए जाएँ या नहीं (और क्या cellular data की अनुमति है), cache folder खोलें, या storage space खाली करने के लिए cached thumbnails हटाएँ।

### अस्थायी फ़ाइलें हटाएँ

storage space पुनः प्राप्त करने के लिए एप्लिकेशन का cache folder साफ़ करें।

## ऑडियो टैग एडिटर

इस अनुभाग में, आप बिल्ट-इन ऑडियो टैग एडिटर कॉन्फ़िगर कर सकते हैं।

{{< cards cols="1">}}
  {{< card title="" subtitle="Evertag सेटिंग्स टैग एडिटर स्क्रीन" image="/docs/guide/evertag/img/settings-tag-editor.webp" >}}
{{< /cards >}}

### एल्बम कवर स्केलिंग

ऑडियो टैग में सेव करते समय एल्बम कवर के लिए scaling method (Small / Medium / Large / Extra Large) चुनें, या original size रखने के लिए scaling निष्क्रिय करें। Original-size scaling Premium personalization features का हिस्सा है।

### टैग सेविंग विकल्प

- **ID3v2.4** — जब संभव हो ID3v2.4 format में टैग सेव करें। यदि पुराने players टैग सही से नहीं पढ़ते तो disable करें और ऐप ID3v2.3 पर वापस आ जाएगा।
- **Duplicate tags** — पुराने players के साथ compatibility के लिए फ़ाइल के दोनों tag sections में सामान्य metadata fields duplicate करें।

### ऑनलाइन फ़ाइलें अपडेट करें

नियंत्रित करें कि ऐप connected cloud service पर संग्रहीत ऑडियो फ़ाइलों के लिए metadata कैसे अपडेट करता है: प्रत्येक save से पहले पूछें, हमेशा upload करें, या कभी upload न करें (केवल local changes)।

### ऑनलाइन फ़ाइलें संपादित करें

तय करें कि संपादन समाप्त करने के बाद cloud file की locally downloaded copy स्वचालित रूप से हटाई जाए या ऑफलाइन पहुँच के लिए आपके डिवाइस पर रखी जाए।

### मुख्य स्क्रीन बटन

ऑडियो टैग एडिटर की मुख्य स्क्रीन पर कौन-सी actions दिखाई दें चुनें — Auto-search audio tags, Manual search audio tags, Search album artwork, Save album artwork, Normalize encoding, Delete audio tags — और क्या **Show extended tags** और **Edit files simultaneously** डिफ़ॉल्ट रूप से हमेशा on रहें।

## WiFi-Drive

इस अनुभाग में, आप WiFi Drive feature सक्रिय कर सकते हैं, जो आपको desktop web browser का उपयोग करके अपने computer से इस डिवाइस पर फ़ाइलें transfer करने की अनुमति देता है। WiFi Drive का उपयोग करने के विस्तृत निर्देश [यहाँ](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive) उपलब्ध हैं।

## व्यक्तिगतकरण

इस अनुभाग में, आप अपनी प्राथमिकताओं के अनुसार user interface सेटिंग्स कस्टमाइज़ कर सकते हैं।

### एप्लिकेशन आइकन

ऐप के लिए alternate Home Screen / Dock icon चुनें, या default रखें। कुछ alternate icons Premium personalization features का हिस्सा हैं।

### रंग योजना

यहाँ user interface theme कस्टमाइज़ करें और dark mode सक्षम करें। जब default विकल्प चुना जाता है, तो एप्लिकेशन अपनी color scheme को आपके डिवाइस की appearance settings के आधार पर अनुकूलित करेगा।

### सूची में items की उपस्थिति

सूची में items कैसे प्रदर्शित हों कस्टमाइज़ करें, जो छोटी स्क्रीन वाले डिवाइस के लिए विशेष रूप से उपयोगी है। आप चुन सकते हैं कि ऐप content के आधार पर ऊँचाई स्वचालित रूप से समायोजित करे या text के लिए अधिक space प्रदान करने के लिए list cells में smaller icons दिखाए।

### सामग्री लोडिंग सीमा

एक folder में ऐप प्रति page load होने वाले items की अधिकतम संख्या सेट करें (100, 500, 1000, 5000, या कोई सीमा नहीं)। Pagination folder loading को तेज़ रखती है; यदि आप सब कुछ एक बार में load करना चाहते हैं तो सीमा disable करें।

### मुख्य मेनू style (iPad और Mac)

iPad और Mac पर, sidebar में plain main menu या expandable, grouped menu के बीच चुनें।

### स्थानीय फ़ाइलें स्क्रीन style

Local Files स्क्रीन के लिए plain या grouped layout चुनें।

### Context menu style

उस context menu के लिए style चुनें जो 'More Actions' बटन टैप करने पर दिखाई देता है — system menu या ऐप का custom bottom sheet।

## विंडो सेटिंग्स (iPad और Mac)

iPad और Mac पर, तय करें कि एप्लिकेशन का उपयोग करते समय स्क्रीन active रहे या नहीं, और अन्य window-level विकल्प कॉन्फ़िगर करें।

## 프리미엄으로 업그레이드

모든 제한을 없애기 위해 애플리케이션을 프리미엄 버전으로 업그레이드하세요. 무료 버전의 애플리케이션은 모든 제한을 없애고 프리미엄 버전으로 업그레이드할 수 있는 평생 인앱 구매 한 가지와 두 가지 구독 옵션을 제공합니다.  

또한 **Family Sharing**은 모든 구매 및 플랜에 활성화되어 있으므로 가족 구성원과 프리미엄 버전을 공유할 수 있습니다.  

구매 및 프리미엄 버전에 대해 자세히 알아보려면 여기를 참조하세요: [Evertag와 Evertag 프리미엄의 차이점](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/).

## iOS와 Mac 간 구매 공유

평생 구매 및 구독은 iOS와 Mac 간에 iCloud를 사용하여 동기화된다는 점을 명심하세요. iOS 기기에 프리미엄 버전이 있는 경우 최신 버전이 설치되어 있고 iCloud가 활성화되어 있는지 확인하세요. iOS에서 앱을 시작하고 1분 정도 기다려 구매 정보가 iCloud에 업로드될 때까지 기다리세요.

앱 설정에서 "구매 복원" 버튼을 눌러 볼 수도 있습니다. 그 후 Mac에서 App Store에서 최신 앱 버전을 설치하고 앱을 시작하세요. 인터넷 연결이 있고 iOS 기기에서 구매 시 사용한 것과 동일한 iCloud 및 App Store 계정을 Mac에서 사용하고 있는지 확인하세요. 앱이 iCloud에서 구매 정보를 다운로드할 때까지 1분 정도 기다리세요. 프리미엄 버전이 Mac에서 자동으로 활성화됩니다.

## 새 기기에서 구매 복원

새 기기에서 구매를 복원하려면 **구매 -> 구매 복원** 메뉴를 사용하세요. 구매 목록이 표시됩니다. 모든 구매가 보이지 않으면 기기가 구매 시 사용한 것과 동일한 iTunes 계정에 연결되어 있는지 확인하고 iCloud가 활성화되어 있는지 확인하세요.

## 무료로 프리미엄 체험 

제한된 시간 동안만 이 메뉴를 사용하여 무료로 프리미엄 버전으로 업그레이드할 수 있습니다. 광고를 시청하거나 이 앱에 대해 친구들에게 알리기만 하면 무료로 프리미엄 버전을 받을 수 있습니다.

## 구매 

이 메뉴에서 이전 구매를 복원할 수 있습니다. 활성화 오류가 발생하면 "앱 시작 시 구매 복원" 옵션을 활성화해 보세요.

## 새로운 기능 

이 메뉴는 새 버전이 출시된 후에 사용 가능합니다. 업데이트된 애플리케이션의 새로운 기능을 볼 수 있습니다.

## 암호

애플리케이션 데이터를 보호하려면 비밀번호 보호 화면을 활성화하세요.

## 파일 관리자

파일 관리자는 연결된 클라우드 스토리지 계정에 대한 액세스를 지원하고 여러 파일의 빠른 관리를 위한 일괄 작업을 제공합니다.

### 파일 전송

기기에 파일을 다운로드할 때 네트워크 기본 설정을 선택하세요 — Wi-Fi만 또는 Wi-Fi 및 셀룰러 데이터.

### 병렬 파일 전송 제한

동시에 전송되는 최대 파일 수를 설정하세요. 숫자가 높을수록 전송 속도가 빠를 수 있지만 더 많은 시스템 리소스와 배터리를 사용할 수 있습니다.

### 파일 전송 작업

전송 대기열을 열어 진행 중이고 완료된 업로드/다운로드 작업을 보고 관리하세요.

### 백그라운드 전송

앱이 백그라운드에서 실행되는 동안에도 다운로드를 활성화할 수 있습니다. 이 작업이 상당한 양의 에너지를 소비하는 경우 운영 체제가 앱을 일시 중단할 수 있습니다.

### 다운로드된 파일 저장 위치

기본 다운로드 디렉토리를 정의하거나 파일 저장 위치를 선택하도록 매번 메시지를 표시하도록 선택하세요.

### 전체 파일 이름 표시

파일 관리자에서 확장자를 포함한 전체 파일 이름을 보려면 이 옵션을 활성화하세요.

### 파일 썸네일

클라우드 스토리지의 파일 썸네일이 로드되는지 여부(셀룰러 데이터가 허용되는지 여부), 캐시 폴더를 열거나 캐시된 썸네일을 삭제하여 스토리지 공간을 확보하세요.

### 임시 파일 삭제

스토리지 공간을 회복하기 위해 애플리케이션의 캐시 폴더를 지우세요.

## 오디오 태그 편집기

이 섹션에서 내장 오디오 태그 편집기를 구성할 수 있습니다.

  {{< card title="" subtitle="Evertag 설정 태그 편집기 화면" image="/docs/guide/evertag/img/settings-tag-editor.webp" >}}

### 앨범 커버 크기 조정

오디오 태그에 저장할 때 앨범 커버의 크기 조정 방법(소 / 중 / 대 / 특대)을 선택하거나 원래 크기를 유지하려면 크기 조정을 비활성화하세요. 원본 크기 크기 조정은 프리미엄 개인화 기능의 일부입니다.

### 태그 저장 옵션

- **ID3v2.4** — 가능한 경우 ID3v2.4 형식으로 태그를 저장합니다. 이전 플레이어가 태그를 올바르게 읽지 못하면 비활성화하면 앱이 ID3v2.3으로 대체됩니다.
- **중복 태그** — 이전 플레이어와의 호환성을 위해 파일의 두 태그 섹션 모두에 공통 메타데이터 필드를 복제합니다.

### 온라인 파일 업데이트

연결된 클라우드 서비스에 저장된 오디오 파일의 메타데이터를 앱이 업데이트하는 방법을 제어합니다: 각 저장 전에 묻기, 항상 업로드, 또는 업로드하지 않음(로컬 전용 변경 사항).

### 온라인 파일 편집

편집 후 클라우드 파일의 로컬 다운로드 복사본을 자동으로 삭제할지, 또는 오프라인 액세스를 위해 기기에 보관할지 결정하세요.

### 메인 화면 버튼

오디오 태그 편집기의 메인 화면에 표시되는 작업을 선택하세요 — 오디오 태그 자동 검색, 오디오 태그 수동 검색, 앨범 아트워크 검색, 앨범 아트워크 저장, 인코딩 정규화, 오디오 태그 삭제 — 그리고 **확장 태그 표시** 및 **파일 동시 편집**이 항상 기본으로 켜져 있는지 여부.

## WiFi-Drive

이 섹션에서 데스크톱 웹 브라우저를 사용하여 컴퓨터에서 이 기기로 파일을 전송할 수 있는 WiFi Drive 기능을 활성화할 수 있습니다. WiFi Drive 사용 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)에서 확인할 수 있습니다.

### 앱 아이콘

앱의 대체 홈 화면/독 아이콘을 선택하거나 기본값을 유지하세요. 일부 대체 아이콘은 프리미엄 개인화 기능의 일부입니다.

### 색상 구성표

여기서 사용자 인터페이스 테마를 사용자 지정하고 다크 모드를 활성화하세요. 기본 옵션을 선택하면 애플리케이션이 기기의 모양 설정에 따라 색상 구성표를 조정합니다.

### 목록의 항목 모양

목록에서 항목이 표시되는 방식을 사용자 지정하세요. 특히 화면이 작은 기기에 유용합니다. 앱이 콘텐츠를 기반으로 높이를 자동으로 조정할지 또는 텍스트를 위한 공간을 더 제공하기 위해 목록 셀에 더 작은 아이콘을 표시할지 선택할 수 있습니다.

### 콘텐츠 로드 제한

앱이 폴더에서 페이지당 로드하는 최대 항목 수를 설정하세요(100, 500, 1000, 5000 또는 제한 없음). 페이지 매김은 폴더 로드를 빠르게 유지합니다; 모든 것을 한 번에 로드하려면 제한을 비활성화하세요.

### 메인 메뉴 스타일 (iPad 및 Mac)

iPad 및 Mac에서 사이드바의 일반 메인 메뉴와 확장 가능한 그룹 메뉴 중에서 선택하세요.

### 로컬 파일 화면 스타일

로컬 파일 화면의 일반 또는 그룹화된 레이아웃을 선택하세요.

### 컨텍스트 메뉴 스타일

'추가 작업' 버튼을 탭할 때 나타나는 컨텍스트 메뉴 스타일을 선택하세요 — 시스템 메뉴 또는 앱의 커스텀 하단 시트.

## 창 설정 (iPad 및 Mac)

iPad 및 Mac에서 애플리케이션을 사용하는 동안 화면이 활성 상태를 유지해야 하는지 결정하고 다른 창 수준 옵션을 구성하세요.

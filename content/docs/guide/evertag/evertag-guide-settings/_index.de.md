## Auf Premium upgraden

Upgraden Sie die Anwendung auf die Premium-Version, um alle Einschränkungen zu entfernen. Die kostenlose Version der Anwendung bietet einen lebenslangen In-App-Kauf und zwei Abonnementoptionen, mit denen Sie alle Einschränkungen entfernen und auf die Premium-Version upgraden können.

Bitte bedenken Sie auch, dass **Family Sharing** für alle Käufe und Pläne aktiviert ist, sodass Sie die Premium-Version mit Familienmitgliedern teilen können.

Weitere Informationen zu Käufen und der Premium-Version finden Sie hier: [Was ist der Unterschied zwischen Evertag und Evertag Premium](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/).

## Käufe zwischen iOS und Mac teilen

Bitte beachten Sie, dass lebenslange Käufe und Abonnements zwischen iOS und Mac geteilt werden, wobei iCloud zur Synchronisierung dieser Informationen verwendet wird. Wenn Sie die Premium-Version auf Ihrem iOS-Gerät haben, stellen Sie sicher, dass die neueste Version installiert ist und iCloud aktiviert ist. Starten Sie die App auf iOS und warten Sie eine Minute, bis Ihre Kaufinformationen in iCloud hochgeladen werden.

Sie können auch versuchen, in den App-Einstellungen die Schaltfläche „Käufe wiederherstellen" zu drücken. Installieren Sie anschließend die neueste App-Version vom App Store auf Ihrem Mac und starten Sie die App. Stellen Sie sicher, dass Sie eine Internetverbindung haben und dasselbe iCloud- und App Store-Konto auf Ihrem Mac verwenden, das Sie auf Ihrem iOS-Gerät verwendet haben. Warten Sie eine Minute, bis die App die Kaufinformationen von iCloud heruntergeladen hat. Die Premium-Version sollte sich automatisch auf Ihrem Mac aktivieren.

## Käufe auf einem neuen Gerät wiederherstellen

Um Ihren Kauf auf dem neuen Gerät wiederherzustellen, verwenden Sie einfach das Menü **Käufe → Käufe wiederherstellen**. Sie sehen die Liste Ihrer Käufe. Wenn nicht alle Käufe angezeigt werden, prüfen Sie, ob das Gerät mit demselben iTunes-Konto verbunden ist, das für die Käufe verwendet wurde, und stellen Sie sicher, dass iCloud aktiviert ist.

## Premium kostenlos testen

Sie können die Premium-Version für eine begrenzte Zeit kostenlos upgraden, indem Sie dieses Menü verwenden. Sehen Sie sich einfach eine Werbung an oder empfehlen Sie diese App Ihren Freunden, um die Premium-Version kostenlos zu erhalten.

## Dateimanager

Der Dateimanager unterstützt den Zugriff auf verbundene Cloud-Speicherkonten und bietet Stapeloperationen für die schnelle Verwaltung mehrerer Dateien.

### Dateiübertragungen

Wählen Sie Ihre Netzwerkeinstellung beim Herunterladen von Dateien auf Ihr Gerät — nur WLAN oder WLAN & Mobilfunkdaten.

### Paralleles Dateiübertragungslimit

Legen Sie die maximale Anzahl von gleichzeitig übertragenen Dateien fest. Eine höhere Zahl kann Übertragungen beschleunigen, verbraucht jedoch möglicherweise mehr Systemressourcen und Akku.

### Dateiübertragungsaufgaben

Öffnen Sie die Übertragungswarteschlange, um laufende und abgeschlossene Upload-/Download-Aufgaben anzuzeigen und zu verwalten.

### Hintergrundübertragungen

Sie können Downloads auch aktivieren, wenn die App im Hintergrund ausgeführt wird. Bitte beachten Sie, dass das Betriebssystem die App anhalten kann, wenn dieser Vorgang viel Energie verbraucht.

### Heruntergeladene Dateien speichern in

Definieren Sie Ihr Standard-Download-Verzeichnis oder wählen Sie, dass Sie jedes Mal gefragt werden, wo Dateien gespeichert werden sollen.

### Vollständige Dateinamen anzeigen

Aktivieren Sie diese Option, wenn Sie vollständige Dateinamen einschließlich Erweiterungen im Dateimanager anzeigen möchten.

### Miniaturansichten für Dateien

Wählen Sie, ob Miniaturansichten für Dateien im Cloud-Speicher geladen werden (und ob Mobilfunkdaten erlaubt sind), öffnen Sie den Cache-Ordner oder löschen Sie zwischengespeicherte Miniaturansichten, um Speicherplatz freizugeben.

### Temporäre Dateien löschen

Löschen Sie den Cache-Ordner der Anwendung, um Speicherplatz zurückzugewinnen.

## Audio-Tags-Editor

In diesem Abschnitt können Sie den integrierten Audio-Tags-Editor konfigurieren.

### Albumcover-Skalierung

Wählen Sie die Skalierungsmethode (Klein / Mittel / Groß / Sehr groß) für das Albumcover beim Speichern in Audio-Tags oder deaktivieren Sie die Skalierung, um die Originalgröße beizubehalten. Die Originalgröße-Skalierung ist Teil der Premium-Personalisierungsfunktionen.

### Tag-Speicheroptionen

- **ID3v2.4** — Speichern Sie Tags im ID3v2.4-Format, wenn möglich. Deaktivieren Sie dies, wenn ältere Player die Tags nicht korrekt lesen, und die App fällt auf ID3v2.3 zurück.
- **Doppelte Tags** — Duplizieren Sie allgemeine Metadatenfelder in beide Tag-Abschnitte der Datei für die Kompatibilität mit älteren Playern.

### Online-Dateien aktualisieren

Steuern Sie, wie die App Metadaten für Audiodateien aktualisiert, die in einem verbundenen Cloud-Dienst gespeichert sind: vor jedem Speichern fragen, immer hochladen oder nie hochladen (nur lokale Änderungen).

### Online-Dateien bearbeiten

Entscheiden Sie, ob die lokal heruntergeladene Kopie einer Cloud-Datei nach Abschluss der Bearbeitung automatisch gelöscht oder auf Ihrem Gerät für den Offline-Zugriff behalten wird.

### Hauptbildschirmschaltflächen

Wählen Sie, welche Aktionen auf dem Hauptbildschirm des Audio-Tags-Editors angezeigt werden — Automatische Suche nach Audio-Tags, Manuelle Suche nach Audio-Tags, Album-Artwork suchen, Album-Artwork speichern, Kodierung normalisieren, Audio-Tags löschen — und ob **Erweiterte Tags anzeigen** und **Dateien gleichzeitig bearbeiten** standardmäßig immer aktiviert sind.

## Personalisierung

In diesem Abschnitt können Sie die Benutzeroberflächen-Einstellungen nach Ihren Wünschen anpassen.

{{< cards cols="1">}}
  {{< card title="" subtitle="Evertag Settings Personalization Screen" image="/docs/guide/evertag/img/settings-personalization.webp" >}}
{{< /cards >}}

### Anwendungssymbol

Wählen Sie ein alternatives Startbildschirm-/Dock-Symbol für die App oder behalten Sie das Standard-Symbol. Einige alternative Symbole sind Teil der Premium-Personalisierungsfunktionen.

### Farbschema

Passen Sie hier das Benutzeroberflächen-Thema an und aktivieren Sie den Dunkelmodus. Wenn die Standardoption ausgewählt ist, passt die Anwendung ihr Farbschema basierend auf den Erscheinungseinstellungen Ihres Geräts an.

### Erscheinungsbild von Elementen in der Liste

Passen Sie an, wie Elemente in der Liste angezeigt werden, was besonders nützlich für Geräte mit kleineren Bildschirmen ist. Sie können wählen, ob die App die Höhe basierend auf dem Inhalt automatisch anpassen oder kleinere Symbole in Listenzellen anzeigen soll, um mehr Platz für Text zu schaffen.

### Inhaltsladelimit

Legen Sie die maximale Anzahl von Elementen fest, die die App pro Seite in einem Ordner lädt (100, 500, 1000, 5000 oder kein Limit). Die Paginierung hält das Laden von Ordnern schnell; deaktivieren Sie das Limit, wenn Sie lieber alles auf einmal laden möchten.

### Hauptmenüstil (iPad und Mac)

Wählen Sie auf iPad und Mac zwischen einem einfachen Hauptmenü oder einem erweiterbaren, gruppierten Menü in der Seitenleiste.

### Stil des lokalen Datei-Bildschirms

Wählen Sie ein einfaches oder gruppiertes Layout für den Bildschirm „Lokale Dateien".

### Kontextmenüstil

Wählen Sie einen Stil für das Kontextmenü, das angezeigt wird, wenn Sie auf die Schaltfläche „Weitere Aktionen" tippen — Systemmenü oder das benutzerdefinierte untere Blatt der App.

## Fenstereinstellungen (iPad und Mac)

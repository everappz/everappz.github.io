## Upgrade ke Premium

Upgrade aplikasi ke versi Premium untuk menghapus semua batasan. Versi gratis aplikasi menawarkan satu pembelian dalam aplikasi seumur hidup dan dua opsi langganan yang memungkinkan Anda menghapus semua pembatasan dan upgrade ke versi Premium.  

Selain itu, harap ingat bahwa **Family Sharing** diaktifkan untuk semua pembelian dan paket, sehingga Anda dapat berbagi versi Premium dengan anggota keluarga Anda.  

Anda dapat membaca lebih lanjut tentang pembelian dan versi Premium di sini: [Apa perbedaan antara Evertag dan Evertag Premium](/docs/faq/evertag/what-is-the-difference-between-evertag-and-evertag-premium/).

## Berbagi Pembelian Antara iOS dan Mac

Harap ingat bahwa pembelian seumur hidup dan langganan dibagikan antara iOS dan Mac, menggunakan iCloud untuk menyinkronkan informasi ini. Jika Anda memiliki versi premium di perangkat iOS, pastikan Anda telah menginstal versi terbaru dan iCloud diaktifkan. Mulai aplikasi di iOS dan tunggu satu menit agar informasi pembelian Anda diunggah ke iCloud.

Anda juga dapat mencoba menekan tombol "Pulihkan Pembelian" di pengaturan aplikasi. Setelah itu, instal versi aplikasi terbaru dari App Store di Mac Anda dan mulai aplikasi. Pastikan Anda memiliki koneksi internet dan menggunakan akun iCloud dan App Store yang sama di Mac Anda yang Anda gunakan di perangkat iOS. Tunggu satu menit agar aplikasi mengunduh info pembelian dari iCloud. Versi premium harus aktif di Mac Anda secara otomatis.

## Pulihkan pembelian di Perangkat Baru

Untuk memulihkan pembelian Anda di perangkat baru, cukup gunakan menu **Pembelian -> Pulihkan pembelian**. Anda akan melihat daftar pembelian Anda. Jika Anda tidak melihat semua pembelian, periksa apakah perangkat terhubung ke akun iTunes yang sama yang digunakan untuk melakukan pembelian, dan pastikan iCloud diaktifkan.

## Coba Premium Gratis 

Anda dapat upgrade ke versi Premium secara gratis tetapi hanya untuk waktu terbatas menggunakan menu ini. Cukup tonton iklan atau ceritakan kepada teman-teman Anda tentang aplikasi ini untuk mendapatkan versi Premium secara gratis.

## Pengelola file

Pengelola file mendukung akses ke akun penyimpanan cloud yang terhubung dan menawarkan operasi batch untuk manajemen cepat beberapa file.

### Transfer file

Pilih preferensi jaringan Anda saat mengunduh file ke perangkat — Wi-Fi saja atau Wi-Fi & Data Seluler.

### Batas transfer file paralel

Tetapkan jumlah maksimum file yang ditransfer secara bersamaan. Angka yang lebih tinggi dapat mempercepat transfer tetapi mungkin menggunakan lebih banyak sumber daya sistem dan baterai.

### Tugas transfer file

Buka antrian transfer untuk melihat dan mengelola tugas upload/download yang sedang berlangsung dan selesai.

### Transfer latar belakang

Anda dapat mengaktifkan unduhan bahkan saat aplikasi berjalan di latar belakang. Harap perhatikan bahwa jika operasi ini menghabiskan banyak energi, sistem operasi mungkin menangguhkan aplikasi.

### Simpan file yang diunduh ke

Tentukan direktori unduhan default Anda atau pilih untuk ditanya setiap saat di mana menyimpan file.

### Tampilkan nama file lengkap

Aktifkan opsi ini jika Anda perlu melihat nama file lengkap, termasuk ekstensi, di pengelola file.

### Thumbnail untuk file

Pilih apakah thumbnail untuk file di penyimpanan cloud dimuat (dan apakah data seluler diizinkan), buka folder cache, atau hapus thumbnail yang di-cache untuk mengosongkan ruang penyimpanan.

### Hapus file sementara

Bersihkan folder cache aplikasi untuk mendapatkan kembali ruang penyimpanan.

Di bagian ini, Anda dapat mengonfigurasi editor tag audio bawaan.

### Penskalaan sampul album

Pilih metode penskalaan (Kecil / Sedang / Besar / Ekstra Besar) untuk sampul album saat menyimpannya ke tag audio, atau nonaktifkan penskalaan untuk mempertahankan ukuran asli. Penskalaan ukuran asli adalah bagian dari fitur personalisasi Premium.

### Opsi penyimpanan tag

- **ID3v2.4** — Simpan tag dalam format ID3v2.4 jika memungkinkan. Nonaktifkan jika pemutar lama tidak membaca tag dengan benar dan aplikasi akan kembali ke ID3v2.3.
- **Tag duplikat** — Duplikasikan kolom metadata umum ke kedua bagian tag file untuk kompatibilitas dengan pemutar lama.

### Perbarui file online

Kontrol bagaimana aplikasi memperbarui metadata untuk file audio yang tersimpan di layanan cloud yang terhubung: tanya sebelum setiap penyimpanan, selalu unggah, atau jangan pernah unggah (hanya perubahan lokal).

### Edit file online

Putuskan apakah salinan lokal yang diunduh dari file cloud dihapus secara otomatis setelah Anda selesai mengedit, atau disimpan di perangkat Anda untuk akses offline.

### Tombol layar utama

Pilih tindakan mana yang muncul di layar utama editor tag audio — Pencarian otomatis tag audio, Pencarian manual tag audio, Cari karya seni album, Simpan karya seni album, Normalisasi pengkodean, Hapus tag audio — dan apakah **Tampilkan tag diperluas** dan **Edit file secara bersamaan** selalu aktif secara default.

## Personalisasi

Di bagian ini, Anda dapat menyesuaikan pengaturan antarmuka pengguna sesuai preferensi Anda.

### Ikon aplikasi

Pilih ikon Layar Beranda / Dock alternatif untuk aplikasi, atau pertahankan default. Beberapa ikon alternatif adalah bagian dari fitur personalisasi Premium.

### Skema warna

Sesuaikan tema antarmuka pengguna dan aktifkan mode gelap di sini. Ketika opsi default dipilih, aplikasi akan menyesuaikan skema warnanya berdasarkan pengaturan tampilan perangkat Anda.

### Tampilan item dalam daftar

Sesuaikan bagaimana item dalam daftar ditampilkan, yang sangat berguna untuk perangkat dengan layar lebih kecil. Anda dapat memilih apakah aplikasi harus secara otomatis menyesuaikan tinggi berdasarkan konten atau menampilkan ikon lebih kecil dalam sel daftar untuk menyediakan lebih banyak ruang untuk teks.

### Batas pemuatan konten

Tetapkan jumlah maksimum item yang dimuat aplikasi per halaman dalam folder (100, 500, 1000, 5000, atau tanpa batas). Paginasi menjaga pemuatan folder tetap cepat; nonaktifkan batas jika Anda lebih suka memuat semuanya sekaligus.

### Gaya menu utama (iPad dan Mac)

Di iPad dan Mac, pilih antara menu utama biasa atau menu yang dapat diperluas dan dikelompokkan di sidebar.

### Gaya layar file lokal

Pilih tata letak biasa atau dikelompokkan untuk layar File Lokal.

### Gaya menu konteks

Pilih gaya untuk menu konteks yang muncul ketika Anda mengetuk tombol 'Lebih Banyak Tindakan' — menu sistem atau lembar bawah kustom aplikasi.

## Pengaturan jendela (iPad dan Mac)

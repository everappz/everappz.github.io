Evertag 擁有直觀的使用者介面。它與許多熱門應用程式的不同之處在於內建檔案管理器，使使用者能夠編輯音訊檔案並與雲端儲存之間無縫傳輸。

## 各部分

無論您使用 iPhone、iPad 還是 Mac 上的緊湊模式，所有應用程式功能都可以透過螢幕底部的標籤列輕鬆存取。對於 iPad 和 Mac 使用者，同樣的選單可以在螢幕左側找到。這種周到的組織方式將所有應用程式功能分類到易於存取的部分，確保友好且高效的體驗。

**最近使用的：** 應用程式啟動時開啟的預設標籤頁。列出您最近開啟的音訊檔案，讓您可以從上次停下的地方繼續編輯。

**最愛項目：** 快速跳轉到您標記為最愛的檔案和資料夾——包括深度嵌套的雲端資料夾。

**連接：** 您可以從此畫面輕鬆連接 iCloud Drive、Google Drive、MEGA、OneDrive 和 Dropbox 等雲端儲存服務，以及您的電腦和個人 NAS。畫面頂部還顯示**快速存取**列表，其中包含您新增到最愛的雲端資料夾，可一鍵存取。

**本機檔案：** 輕鬆尋找和管理已下載的檔案，完整控制傳輸佇列。您可以使用多種檔案管理操作編輯這些本機檔案。要存取傳輸佇列，請點選「本機檔案」畫面左上角的旋轉箭頭圖示。

本機檔案部分分為兩類：**此應用程式中的檔案**，顯示應用程式「文件」目錄中的本機檔案；以及**此 iPad/iPhone/Mac 上的檔案**，顯示裝置上但位於應用程式「文件」目錄之外的本機檔案。

**設定：** 修改應用程式設定，包括檔案管理器、標籤編輯器偏好設定（專輯封面縮放、ID3v2.4 / 重複標籤、主畫面按鈕）、Wi-Fi Drive、個人化（應用程式圖示、主題、列表密度、右鍵選單樣式）、語言、密碼、輔助功能和視窗設定（iPad/Mac）。

**標籤編輯器：** 當您點選檔案或從選單中選擇「編輯音訊標籤」選項時，標籤編輯器會開啟。在 iPad 和 Mac 上，標籤編輯器以主分割檢視的詳細側面呈現，因此當您在左側瀏覽檔案時，它保持在右側。

## 右鍵選單

Evertag 中的每個儲存格——本機檔案、雲端檔案、資料夾、最近使用的/最愛項目條目，甚至已連接的雲端帳戶——都附有右鍵選單。這是在不進入選擇模式的情況下對單一項目進行操作的最快方式，它顯示該項目類型的所有相關操作。

### 如何開啟它

有兩種方式可以叫出右鍵選單，兩種方式產生相同的操作集：

- **「...」按鈕**。每個儲存格在標題旁邊顯示一個「...」（更多操作）按鈕。點選它可開啟選單。這在 iPhone、iPad 和 Mac 上的工作方式相同，VoiceOver 使用者也可以存取。
- **長按 / 右鍵按一下**。在 iPhone 和 iPad 上，**長按**儲存格可開啟選單。在 Mac 上，**右鍵按一下**（或 Control-按一下）儲存格。選單旁邊會出現項目的目標預覽，讓您可以確認正在操作正確的檔案。

同樣的手勢也適用於畫面右上角資料夾自己的標題列——該選單作用於**目前開啟的資料夾**，讓您可以存取選擇、新增資料夾、搜尋、排序、格狀/清單和資料夾級操作。

### 裡面有什麼

確切的列表取決於項目，但您通常會看到以下操作：

- **編輯音訊標籤** — 在內建標籤編輯器中開啟檔案。
- **新增到最愛項目** — 將檔案或資料夾固定到最愛項目標籤頁。
- **下載** — 取得雲端檔案或資料夾以供離線使用；下載會出現在傳輸佇列中。
- **上傳** — 將本機檔案或資料夾推送到已連接的雲端服務。
- **重新命名 / 移動 / 複製** — 在其原始儲存上管理檔案。
- **開啟方式** — 透過系統分享表將檔案匯出到其他應用程式。
- **在 Finder 中顯示** *(Mac 和「此 Mac 上的檔案」)* — 在 Finder 中顯示檔案。
- **刪除** — 從目前位置刪除檔案。此操作不可逆。

如果操作列表溢出畫面，請在選單內捲動以查看其餘內容。

### 右鍵選單樣式

您可以從**設定 → 個人化 → 右鍵選單樣式**中選擇右鍵選單的呈現方式：

- **系統選單** — 使用帶有目標預覽的原生 iOS / macOS 右鍵選單。iPhone 和 iPad 上的預設設定。
- **底部彈出頁** — Evertag 的自訂底部彈出選單，當您偏好更大的觸控目標或希望標籤更易閱讀時很有用。

選擇是按裝置獨立的，因此您可以在 iPhone 上使用系統選單，在 iPad 上使用底部彈出頁（反之亦然）。

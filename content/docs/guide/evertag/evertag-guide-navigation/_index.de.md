Evertag bietet eine intuitive Benutzeroberfläche. Was es von vielen beliebten Apps unterscheidet, ist sein integrierter Dateimanager, der Benutzern die Möglichkeit gibt, Audiodateien zu bearbeiten und sie nahtlos zum und vom Cloud-Speicher zu übertragen.

## Abschnitte

Ob Sie ein iPhone, iPad oder den kompakten Modus auf einem Mac verwenden – alle App-Funktionen sind über die Tab-Leiste am unteren Bildschirmrand leicht zugänglich. Für iPad- und Mac-Benutzer finden Sie dasselbe Menü auf der linken Seite des Bildschirms. Diese durchdachte Organisation kategorisiert alle App-Funktionen in leicht zugängliche Abschnitte und sorgt so für eine benutzerfreundliche und effiziente Erfahrung.

**Aktuell:** Standard-Tab, der sich beim Start der App öffnet. Listet Ihre zuletzt geöffneten Audiodateien auf, damit Sie dort weitermachen können, wo Sie aufgehört haben.

**Favoriten:** Springen Sie schnell zu Dateien und Ordnern, die Sie als Favoriten markiert haben – einschließlich tief verschachtelter Cloud-Ordner.

**Verbindungen:** Sie können auf diesem Bildschirm mühelos Cloud-Speicherdienste wie iCloud Drive, Google Drive, MEGA, OneDrive und Dropbox sowie Ihren Computer und persönliches NAS verbinden. Am oberen Rand des Bildschirms wird auch eine **Schnellzugriff**-Liste mit den Cloud-Ordnern angezeigt, die Sie zu Favoriten hinzugefügt haben, für den Einzel-Tipp-Zugriff.

**Lokale Dateien:** Mühelos Ihre heruntergeladenen Dateien auffinden und verwalten, mit Kontrolle über die Übertragungswarteschlange. Sie können diese lokalen Dateien mit einer Reihe von Dateiverwaltungsaktionen bearbeiten. Um auf die Übertragungswarteschlange zuzugreifen, tippen Sie auf das Symbol der drehenden Pfeile in der oberen linken Ecke des Bildschirms „Lokale Dateien".

Der Abschnitt „Lokale Dateien" ist in zwei Kategorien unterteilt: **Dateien in dieser Anwendung**, die lokale Dateien im Verzeichnis „Documents" der Anwendung anzeigt, und **Dateien auf diesem iPad/iPhone/Mac**, die lokale Dateien auf dem Gerät außerhalb des Verzeichnisses „Documents" der Anwendung anzeigt.

**Einstellungen:** Anwendungseinstellungen ändern, einschließlich Dateimanager, Tag-Editor-Einstellungen (Albumcover-Skalierung, ID3v2.4 / doppelte Tags, Hauptbildschirmschaltflächen), Wi-Fi Drive, Personalisierung (App-Symbol, Thema, Listendichte, Kontextmenüstil), Sprache, Passcode, Barrierefreiheit und Fenstereinstellungen (iPad/Mac).

**Tag-Editor:** Der Tag-Editor öffnet sich, wenn Sie auf eine Datei tippen oder die Option „Audio-Tags bearbeiten" aus dem Menü auswählen. Auf iPad und Mac wird der Tag-Editor als Detailseite der Haupt-Split-Ansicht präsentiert, sodass er rechts bleibt, während Sie links Dateien navigieren.

## Weitere Aktionen

Praktisch jedes Inhaltselement auf dem Bildschirm hat eine Schaltfläche „Weitere Aktionen". Tippen Sie darauf, um auf alle verfügbaren Aktionen zuzugreifen.

## Obere Symbolleiste

Die obere Symbolleiste, die sich praktischerweise direkt unter der Navigationsleiste befindet, bietet schnellen Zugriff auf mehrere nützliche Aktionen.
Sie können diese Symbolleiste ganz einfach mit einer Wischgeste nach unten ein- oder ausblenden. Hier ist eine Liste der Aktionen:

- **Suche:** Starten Sie eine Suche im aktuellen Kontext.

## Kontextmenü

Jede Zelle in Evertag – eine lokale Datei, eine Cloud-Datei, ein Ordner, ein Eintrag in „Zuletzt geöffnet"/„Favoriten", sogar ein verbundenes Cloud-Konto – hat ein Kontextmenü. Es ist der schnellste Weg, mit einem einzelnen Element zu arbeiten, ohne den Auswahlmodus zu aktivieren, und zeigt alle relevanten Aktionen für den Elementtyp an.

### So öffnen Sie es

Es gibt zwei Möglichkeiten, das Kontextmenü aufzurufen, und beide zeigen denselben Satz von Aktionen:

- **Die Schaltfläche „..."** Jede Zelle zeigt eine Schaltfläche „..." (Weitere Aktionen) neben ihrem Titel. Tippen Sie darauf, um das Menü aufzurufen. Dies funktioniert auf iPhone, iPad und Mac gleichermaßen und ist für VoiceOver-Benutzer zugänglich.
- **Langes Drücken / Rechtsklick.** Auf iPhone und iPad **tippen und halten** Sie die Zelle, um das Menü zu öffnen. Auf dem Mac **klicken Sie mit der rechten Maustaste** (oder mit Control-Klick) auf die Zelle. Eine gezielte Vorschau des Elements erscheint neben dem Menü, damit Sie bestätigen können, dass Sie mit der richtigen Datei arbeiten.

Dieselbe Geste funktioniert auch auf der eigenen Titelleiste des Ordners oben rechts auf dem Bildschirm – dieses Menü wirkt auf den **aktuell geöffneten Ordner** und bietet Ihnen Zugriff auf Auswählen, Neuer Ordner, Suche, Sortieren, Raster/Liste und Aktionen auf Ordnerebene.

### Was darin enthalten ist

Die genaue Liste hängt vom Element ab, aber Sie sehen typischerweise Aktionen wie:

- **Audio-Tags bearbeiten** — Datei im integrierten Tag-Editor öffnen.
- **Zu Favoriten hinzufügen** — Datei oder Ordner an die Registerkarte „Favoriten" anheften.
- **Herunterladen** — eine Cloud-Datei oder einen Ordner für die Offline-Nutzung abrufen; Downloads erscheinen in der Übertragungswarteschlange.
- **Hochladen** — eine lokale Datei oder einen Ordner in einen verbundenen Cloud-Dienst übertragen.
- **Umbenennen / Verschieben / Kopieren** — die Datei in ihrem Ursprungsspeicher verwalten.
- **Öffnen in** — die Datei über das System-Share-Sheet in eine andere App exportieren.
- **Im Finder anzeigen** *(Mac und „Dateien auf diesem Mac")* — die Datei im Finder anzeigen.
- **Löschen** — die Datei von ihrem aktuellen Speicherort entfernen. Diese Aktion ist nicht umkehrbar.

Wenn die Aktionsliste den Bildschirm überschreitet, scrollen Sie innerhalb des Menüs, um den Rest anzuzeigen.

### Kontextmenüstil

Sie können auswählen, wie das Kontextmenü präsentiert wird unter **Einstellungen → Personalisierung → Kontextmenüstil**:

- **Systemmenü** — verwendet das native iOS/macOS-Kontextmenü mit einer gezielten Vorschau. Standard auf iPhone und iPad.
- **Unteres Blatt** — Evertags benutzerdefiniertes Bottom-Sheet-Menü, nützlich wenn Sie ein größeres Berührungsziel bevorzugen oder Beschriftungen leichter lesbar sein sollen.

Die Auswahl gilt pro Gerät, sodass Sie Systemmenüs auf dem iPhone mit dem unteren Blatt auf dem iPad kombinieren können (oder umgekehrt).

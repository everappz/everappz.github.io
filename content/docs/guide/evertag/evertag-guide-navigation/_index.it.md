Evertag offre un'interfaccia utente intuitiva. Ciò che la distingue da molte app popolari è il suo gestore file integrato, che offre agli utenti la possibilità di modificare i file audio e trasferirli da e verso l'archiviazione cloud senza problemi.

## Sezioni

Che tu stia usando un iPhone, iPad o la modalità compatta su un Mac, tutte le funzionalità dell'app sono facilmente accessibili tramite la tab bar nella parte inferiore dello schermo. Per gli utenti iPad e Mac, lo stesso menu si trova sul lato sinistro dello schermo. Questa organizzazione attenta categorizza tutte le funzionalità dell'app in sezioni facilmente accessibili, garantendo un'esperienza intuitiva ed efficiente.

**Recenti:** La scheda predefinita che si apre all'avvio dell'app. Elenca i file audio aperti di recente in modo da poter riprendere la modifica dove avevi lasciato.

**Preferiti:** Accedi rapidamente ai file e alle cartelle che hai contrassegnato come preferiti — incluse le cartelle cloud profondamente annidate.

**Connessioni:** Puoi connettere facilmente servizi di archiviazione cloud come iCloud Drive, Google Drive, MEGA, OneDrive e Dropbox, nonché il tuo computer e NAS personale da questa schermata. La parte superiore della schermata mostra anche un elenco **Accesso rapido** con le cartelle cloud che hai aggiunto ai preferiti per l'accesso con un tocco.

**File locali:** Individua e gestisci facilmente i tuoi file scaricati, con il controllo della coda di trasferimento. Puoi modificare questi file locali usando una serie di azioni di gestione dei file. Per accedere alla coda di trasferimento, tocca l'icona delle frecce rotanti nell'angolo in alto a sinistra della schermata "File locali".

La sezione File locali è divisa in due categorie: **File in questa applicazione**, che mostra i file locali nella directory Documents dell'applicazione, e **File su questo iPad/iPhone/Mac**, che mostra i file locali situati sul dispositivo ma al di fuori della directory Documents dell'applicazione.

**Impostazioni:** Modifica le impostazioni dell'applicazione, tra cui il gestore file, le preferenze dell'editor tag (ridimensionamento copertina album, ID3v2.4 / tag duplicati, pulsanti schermata principale), Wi-Fi Drive, personalizzazione (icona app, tema, densità elenco, stile menu contestuale), lingua, codice d'accesso, accessibilità e impostazioni finestra (iPad/Mac).

**Editor tag:** L'editor tag si apre quando tocchi un file o scegli l'opzione "Modificare i tag audio" dal menu. Su iPad e Mac l'editor tag è presentato come il lato dettaglio della vista divisa principale, quindi rimane a destra mentre navighi tra i file a sinistra.

## Barra degli strumenti superiore

La barra degli strumenti superiore, posizionata comodamente appena sotto la barra di navigazione, fornisce accesso rapido a diverse azioni utili.  
Puoi facilmente mostrare o nascondere questa barra degli strumenti con un semplice gesto di scorrimento verso il basso. Ecco un elenco delle azioni:

- **Cerca:** inizia una ricerca nel contesto corrente.

## Menu contestuale

Ogni cella in Evertag — un file locale, un file cloud, una cartella, una voce Recenti/Preferiti, persino un account cloud connesso — ha un menu contestuale allegato. È il modo più veloce per agire su un singolo elemento senza entrare in modalità di selezione, e mostra tutte le azioni rilevanti per il tipo di elemento.

### Come aprirlo

Ci sono due modi per richiamare il menu contestuale, ed entrambi producono lo stesso insieme di azioni:

- **Il pulsante "..."** Ogni cella mostra un pulsante "..." (Altre azioni) vicino al suo titolo. Toccalo per aprire il menu. Funziona allo stesso modo su iPhone, iPad e Mac ed è accessibile agli utenti di VoiceOver.
- **Pressione lunga / clic destro.** Su iPhone e iPad, **tieni premuta** la cella per aprire il menu. Su Mac, **fai clic destro** (o Control-clic) sulla cella. Accanto al menu appare un'anteprima mirata dell'elemento per confermare di stare agendo sul file corretto.

Lo stesso gesto funziona anche sulla barra del titolo della cartella stessa in alto a destra dello schermo — quel menu agisce sulla **cartella attualmente aperta** e ti dà accesso a Selezionare, Nuova cartella, Cerca, Ordina, Griglia/Elenco e azioni a livello di cartella.

### Cosa c'è dentro

L'elenco esatto dipende dall'elemento, ma vedrai tipicamente azioni come:

- **Modificare i tag audio** — apri il file nell'editor di tag integrato.
- **Aggiungi ai preferiti** — aggiungi il file o la cartella alla scheda Preferiti.
- **Scaricare** — recupera un file o una cartella cloud per uso offline; i download appaiono nella coda di trasferimento.
- **Carica** — invia un file o una cartella locale a un servizio cloud connesso.
- **Rinomina / Sposta / Copia** — gestisci il file nel suo archivio di origine.
- **Apri in** — esporta il file in un'altra app tramite il foglio di condivisione di sistema.
- **Mostra nel Finder** *(Mac e "File su questo Mac")* — mostra il file nel Finder.
- **Eliminare** — rimuovi il file dalla sua posizione attuale. Questa azione è irreversibile.

Se l'elenco delle azioni supera lo schermo, scorri nel menu per mostrare il resto.

### Stile del menu contestuale

Puoi scegliere come viene presentato il menu contestuale da **Impostazioni → Personalizzazione → Stile del menu contestuale**:

- **Menu di sistema** — utilizza il menu contestuale nativo iOS / macOS con un'anteprima mirata. Predefinito su iPhone e iPad.
- **Foglio inferiore** — il menu a foglio inferiore personalizzato di Evertag, utile quando preferisci un target di tocco più grande o vuoi che le etichette siano più facili da leggere.

La scelta è per dispositivo, quindi puoi combinare menu di sistema su iPhone con il foglio inferiore su iPad (o viceversa).

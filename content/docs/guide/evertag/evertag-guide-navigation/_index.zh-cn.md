## 各部分

无论您使用 iPhone、iPad 还是 Mac 上的紧凑模式，所有应用功能都可以通过屏幕底部的标签栏轻松访问。对于 iPad 和 Mac 用户，同样的菜单可以在屏幕左侧找到。这种周到的组织方式将所有应用功能分类到易于访问的部分，确保友好且高效的体验。

**最近使用：** 应用启动时打开的默认标签页。列出您最近打开的音频文件，让您可以从上次停下的地方继续编辑。

**收藏夹：** 快速跳转到您标记为收藏的文件和文件夹——包括深度嵌套的云文件夹。

**连接：** 您可以从此屏幕轻松连接 iCloud Drive、Google Drive、MEGA、OneDrive 和 Dropbox 等云存储服务，以及您的电脑和个人 NAS。屏幕顶部还显示**快速访问**列表，其中包含您添加到收藏夹的云文件夹，可一键访问。

**本地文件：** 轻松查找和管理下载的文件，完整控制传输队列。您可以使用多种文件管理操作编辑这些本地文件。要访问传输队列，请点击「本地文件」屏幕左上角的旋转箭头图标。

本地文件部分分为两类：**此应用程序中的文件**，显示应用程序「文档」目录中的本地文件；以及**此 iPad/iPhone/Mac 上的文件**，显示设备上但位于应用程序「文档」目录之外的本地文件。

**设置：** 修改应用程序设置，包括文件管理器、标签编辑器偏好设置（专辑封面缩放、ID3v2.4 / 重复标签、主屏幕按钮）、Wi-Fi Drive、个性化（应用图标、主题、列表密度、上下文菜单样式）、语言、密码、辅助功能和窗口设置（iPad/Mac）。

**标签编辑器：** 当您点击文件或从菜单中选择「编辑音频标签」选项时，标签编辑器会打开。在 iPad 和 Mac 上，标签编辑器以主分屏视图的详细侧面呈现，因此当您在左侧浏览文件时，它保持在右侧。

## 顶部工具栏

顶部工具栏位于导航栏正下方，提供对几个有用操作的快速访问。  
您可以通过简单的向下滑动手势轻松显示或隐藏此工具栏。以下是操作列表：

- **搜索：** 在当前上下文中开始搜索。

## 上下文菜单

Evertag 中的每个单元格——本地文件、云文件、文件夹、最近使用/收藏夹条目，甚至已连接的云账户——都附有上下文菜单。这是在不进入选择模式的情况下对单个项目进行操作的最快方式，它显示该项目类型的所有相关操作。

### 如何打开它

有两种方式可以调出上下文菜单，两种方式产生相同的操作集：

- **「...」按钮**。每个单元格在标题旁边显示一个「...」（更多操作）按钮。点击它可打开菜单。这在 iPhone、iPad 和 Mac 上的工作方式相同，VoiceOver 用户也可以访问。
- **长按 / 右键单击**。在 iPhone 和 iPad 上，**长按**单元格可打开菜单。在 Mac 上，**右键单击**（或 Control-单击）单元格。菜单旁边会出现项目的目标预览，让您可以确认正在操作正确的文件。

同样的手势也适用于屏幕右上角文件夹自己的标题栏——该菜单作用于**当前打开的文件夹**，让您可以访问选择、新建文件夹、搜索、排序、网格/列表和文件夹级操作。

### 里面有什么

确切的列表取决于项目，但您通常会看到以下操作：

- **编辑音频标签** — 在内置标签编辑器中打开文件。
- **添加到收藏夹** — 将文件或文件夹固定到收藏夹标签页。
- **下载** — 获取云文件或文件夹以供离线使用；下载会出现在传输队列中。
- **上传** — 将本地文件或文件夹推送到已连接的云服务。
- **重命名 / 移动 / 复制** — 在其原始存储上管理文件。
- **打开方式** — 通过系统分享表将文件导出到其他应用程序。
- **在 Finder 中显示** *(Mac 和「此 Mac 上的文件」)* — 在 Finder 中显示文件。
- **删除** — 从当前位置删除文件。此操作不可逆。

如果操作列表溢出屏幕，请在菜单内滚动以查看其余内容。

### 上下文菜单样式

您可以从**设置 → 个性化 → 上下文菜单样式**中选择上下文菜单的呈现方式：

- **系统菜单** — 使用带有目标预览的原生 iOS / macOS 上下文菜单。iPhone 和 iPad 上的默认设置。
- **底部弹出页** — Evertag 的自定义底部弹出菜单，当您偏好更大的触摸目标或希望标签更易阅读时很有用。

选择是按设备独立的，因此您可以在 iPhone 上使用系统菜单，在 iPad 上使用底部弹出页（反之亦然）。

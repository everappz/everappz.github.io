Evertag offre une interface utilisateur intuitive. Ce qui le distingue de nombreuses applications populaires est son gestionnaire de fichiers intégré, donnant aux utilisateurs le pouvoir de modifier les fichiers audio et de les transférer vers et depuis le stockage cloud.

## Sections

Que vous utilisiez un iPhone, un iPad ou le mode compact sur un Mac, toutes les fonctionnalités de l'application sont facilement accessibles via la barre d'onglets au bas de l'écran. Pour les utilisateurs d'iPad et de Mac, le même menu se trouve sur le côté gauche de l'écran. Cette organisation réfléchie catégorise toutes les fonctionnalités de l'application en sections facilement accessibles, garantissant une expérience conviviale et efficace.

**Récents :** Onglet par défaut qui s'ouvre au démarrage de l'application. Liste vos fichiers audio récemment ouverts afin que vous puissiez reprendre l'édition là où vous vous êtes arrêté.

**Favoris :** Accédez rapidement aux fichiers et dossiers que vous avez marqués comme favoris — y compris les dossiers cloud profondément imbriqués.

**Connexions :** Vous pouvez facilement connecter des services de stockage cloud tels qu'iCloud Drive, Google Drive, MEGA, OneDrive et Dropbox, ainsi que votre ordinateur et votre NAS personnel depuis cet écran. En haut de l'écran, vous trouverez également une liste d'**Accès rapide** avec les dossiers cloud que vous avez ajoutés aux favoris pour un accès en un seul appui.

**Fichiers locaux :** Localisez et gérez facilement vos fichiers téléchargés, avec le contrôle de la file de transfert. Vous pouvez modifier ces fichiers locaux à l'aide d'une gamme d'actions de gestion de fichiers. Pour accéder à la file de transfert, appuyez sur l'icône de flèches tournantes dans le coin supérieur gauche de l'écran «Fichiers locaux».

La section Fichiers locaux est divisée en deux catégories : **Fichiers dans cette application**, affichant les fichiers locaux dans le répertoire Documents de l'application, et **Fichiers sur cet iPad/iPhone/Mac**, révélant les fichiers locaux situés sur l'appareil mais en dehors du répertoire Documents de l'application.

**Paramètres :** Modifiez les paramètres de l'application, notamment le gestionnaire de fichiers, les préférences de l'éditeur de tags (mise à l'échelle des pochettes d'album, ID3v2.4 / tags dupliqués, boutons de l'écran principal), Wi-Fi Drive, la personnalisation (icône d'application, thème, densité de liste, style du menu contextuel), la langue, le code d'accès, l'accessibilité et les paramètres de fenêtre (iPad/Mac).

**Éditeur de tags :** L'éditeur de tags s'ouvre lorsque vous appuyez sur un fichier ou choisissez l'option «Modifier les balises audio» dans le menu. Sur iPad et Mac, l'éditeur de tags est présenté comme le côté détail de la vue fractionnée principale, il reste donc à droite pendant que vous naviguez dans les fichiers à gauche.

## Plus d'actions

Pratiquement chaque élément de contenu à l'écran dispose d'un bouton «Plus d'actions». Appuyez dessus pour accéder à toutes les actions disponibles.

## Barre d'outils supérieure

La barre d'outils supérieure, commodément située juste sous la barre de navigation, permet un accès rapide à plusieurs actions utiles.
Vous pouvez facilement afficher ou masquer cette barre d'outils avec un simple geste de balayage vers le bas. Voici la liste des actions :

- **Rechercher :** Commencer une recherche dans le contexte actuel.

## Menu contextuel

Chaque cellule dans Evertag — un fichier local, un fichier cloud, un dossier, une entrée Récents/Favoris, même un compte cloud connecté — dispose d'un menu contextuel associé. C'est le moyen le plus rapide d'agir sur un seul élément sans entrer en mode de sélection, et il présente chaque action pertinente pour le type d'élément.

### Comment l'ouvrir

Il y a deux façons d'ouvrir le menu contextuel, et les deux produisent le même ensemble d'actions :

- **Le bouton «...».** Chaque cellule affiche un bouton «...» (Plus d'actions) près de son titre. Appuyez dessus pour ouvrir le menu. Cela fonctionne de la même manière sur iPhone, iPad et Mac et est accessible aux utilisateurs de VoiceOver.
- **Appui long / clic droit.** Sur iPhone et iPad, **appuyez et maintenez** la cellule pour ouvrir le menu. Sur Mac, **cliquez avec le bouton droit** (ou Control-cliquez) sur la cellule à la place. Un aperçu ciblé de l'élément apparaît à côté du menu pour vous permettre de confirmer que vous agissez sur le bon fichier.

Le même geste fonctionne également sur la barre de titre du dossier dans le coin supérieur droit de l'écran — ce menu agit sur le **dossier actuellement ouvert** et vous donne accès à Sélectionner, Nouveau dossier, Rechercher, Trier, Grille/Liste et les actions au niveau du dossier.

### Ce qu'il contient

La liste exacte dépend de l'élément, mais vous verrez généralement des actions telles que :

- **Modifier les balises audio** — ouvrir le fichier dans l'éditeur de tags intégré.
- **Ajouter aux favoris** — épingler le fichier ou le dossier à l'onglet Favoris.
- **Télécharger** — récupérer un fichier ou dossier cloud pour une utilisation hors ligne ; les téléchargements apparaissent dans la file de transfert.
- **Téléverser** — envoyer un fichier ou dossier local vers un service cloud connecté.
- **Renommer / Déplacer / Copier** — gérer le fichier sur son stockage d'origine.
- **Ouvrir dans** — exporter le fichier vers une autre application via la feuille de partage système.
- **Afficher dans le Finder** *(Mac et «Fichiers sur ce Mac»)* — révéler le fichier dans le Finder.
- **Supprimer** — supprimer le fichier de son emplacement actuel. Cette action est irréversible.

Si la liste des actions dépasse l'écran, faites défiler dans le menu pour révéler le reste.

### Style du menu contextuel

Vous pouvez choisir comment le menu contextuel est présenté depuis **Paramètres → Personnalisation → Style du menu contextuel** :

- **Menu système** — utilise le menu contextuel natif iOS/macOS avec un aperçu ciblé. Par défaut sur iPhone et iPad.
- **Feuille de fond** — le menu personnalisé en feuille de fond d'Evertag, utile lorsque vous préférez une cible tactile plus grande ou souhaitez que les étiquettes soient plus faciles à lire.

Le choix est par appareil, vous pouvez donc utiliser les menus système sur iPhone avec la feuille de fond sur iPad (ou vice versa).

Evertag menawarkan antarmuka pengguna yang intuitif. Yang membedakannya dari banyak aplikasi populer adalah pengelola file bawaannya, yang memberikan pengguna kemampuan untuk mengedit file audio dan mentransfernya ke dan dari penyimpanan cloud dengan mulus.

## Bagian

Baik Anda menggunakan iPhone, iPad, atau mode kompak di Mac, semua fitur aplikasi mudah diakses melalui tab bar di bagian bawah layar. Untuk pengguna iPad dan Mac, menu yang sama dapat ditemukan di sisi kiri layar. Organisasi yang cermat ini mengkategorikan semua fitur aplikasi ke dalam bagian yang mudah diakses, memastikan pengalaman yang ramah pengguna dan efisien.

**Terbaru:** Tab default yang terbuka saat aplikasi dimulai. Mencantumkan file audio yang baru saja Anda buka sehingga Anda dapat melanjutkan pengeditan dari tempat Anda berhenti.

**Favorit:** Langsung menuju file dan folder yang telah Anda tandai sebagai favorit — termasuk folder cloud yang sangat tersarang.

**Koneksi:** Anda dapat dengan mudah menghubungkan layanan penyimpanan cloud seperti iCloud Drive, Google Drive, MEGA, OneDrive, dan Dropbox, serta komputer dan NAS pribadi Anda dari layar ini. Bagian atas layar juga menampilkan daftar **Akses cepat** dengan folder cloud yang telah Anda tambahkan ke favorit untuk akses satu ketukan.

**File Lokal:** Temukan dan kelola file yang diunduh dengan mudah, lengkap dengan kontrol antrian transfer. Anda dapat mengedit file lokal ini menggunakan berbagai tindakan manajemen file. Untuk mengakses antrian transfer, ketuk ikon panah berputar yang ada di sudut kiri atas layar "File Lokal".

Bagian File Lokal dibagi menjadi dua kategori: **File di aplikasi ini**, menampilkan file lokal dalam direktori Documents aplikasi, dan **File di iPad/iPhone/Mac ini**, mengungkapkan file lokal yang terletak di perangkat tetapi di luar direktori Documents aplikasi.

**Pengaturan:** Ubah pengaturan aplikasi, termasuk pengelola file, preferensi editor tag (penskalaan sampul album, ID3v2.4 / tag duplikat, tombol layar utama), Wi-Fi Drive, personalisasi (ikon aplikasi, tema, kepadatan daftar, gaya menu konteks), bahasa, kode sandi, aksesibilitas, dan pengaturan jendela (iPad/Mac).

**Editor Tag:** Editor tag terbuka saat Anda mengetuk file atau memilih opsi "Mengedit tag audio" dari menu. Di iPad dan Mac, editor tag ditampilkan sebagai sisi detail dari tampilan split utama, sehingga tetap di sebelah kanan saat Anda menavigasi file di sebelah kiri.

## Toolbar Atas

Toolbar atas yang berada tepat di bawah bilah navigasi menyediakan akses cepat ke beberapa tindakan berguna.  
Anda dapat dengan mudah menampilkan atau menyembunyikan toolbar ini dengan gerakan geser ke bawah. Berikut daftar tindakan:

- **Cari:** Mulai pencarian dalam konteks saat ini.

## Menu Konteks

Setiap sel di Evertag — file lokal, file cloud, folder, entri Terbaru/Favorit, bahkan akun cloud yang terhubung — memiliki menu konteks yang terpasang. Ini adalah cara tercepat untuk bertindak pada satu item tanpa memasuki mode pemilihan, dan menampilkan setiap tindakan yang relevan untuk jenis item.

### Cara membukanya

Ada dua cara untuk memanggil menu konteks, dan keduanya menghasilkan serangkaian tindakan yang sama:

- **Tombol "..."** Setiap sel menampilkan tombol "..." (Lebih Banyak Tindakan) dekat judulnya. Ketuk untuk memunculkan menu. Ini berfungsi sama di iPhone, iPad, dan Mac dan dapat diakses oleh pengguna VoiceOver.
- **Tekan lama / klik kanan.** Di iPhone dan iPad, **tekan dan tahan** sel untuk membuka menu. Di Mac, **klik kanan** (atau Control-klik) sel. Pratinjau item yang ditargetkan muncul bersama menu sehingga Anda dapat mengkonfirmasi bahwa Anda sedang menindaklanjuti file yang benar.

Gerakan yang sama juga berfungsi pada bilah judul folder sendiri di kanan atas layar — menu tersebut bertindak pada **folder yang sedang dibuka** dan memberi Anda akses ke Pilih, Folder Baru, Cari, Urutkan, Kisi/Daftar, dan tindakan tingkat folder.

### Apa yang ada di dalamnya

Daftar tepatnya tergantung pada item, tetapi Anda biasanya akan melihat tindakan seperti:

- **Mengedit tag audio** — buka file di editor tag bawaan.
- **Tambahkan ke Favorit** — sematkan file atau folder ke tab Favorit.
- **Unduh** — ambil file atau folder cloud untuk penggunaan offline; unduhan muncul di antrian transfer.
- **Unggah** — dorong file atau folder lokal ke layanan cloud yang terhubung.
- **Ganti Nama / Pindahkan / Salin** — kelola file di penyimpanan asalnya.
- **Buka Di** — ekspor file ke aplikasi lain melalui lembar berbagi sistem.
- **Tampilkan di Finder** *(Mac dan "File di Mac ini")* — tampilkan file di Finder.
- **Hapus** — hapus file dari lokasi saat ini. Tindakan ini tidak dapat dibatalkan.

Jika daftar tindakan melampaui layar, gulir di dalam menu untuk menampilkan sisanya.

### Gaya menu konteks

Anda dapat memilih bagaimana menu konteks disajikan dari **Pengaturan → Personalisasi → Gaya menu konteks**:

- **Menu sistem** — menggunakan menu konteks iOS / macOS asli dengan pratinjau yang ditargetkan. Default di iPhone dan iPad.
- **Lembar bawah** — menu lembar bawah kustom Evertag, berguna saat Anda lebih suka target sentuh yang lebih besar atau ingin label lebih mudah dibaca.

Pilihan ini per perangkat, sehingga Anda dapat memadukan menu sistem di iPhone dengan lembar bawah di iPad (atau sebaliknya).

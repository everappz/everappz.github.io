Evertag एक सहज उपयोगकर्ता इंटरफेस प्रदान करता है। जो इसे कई लोकप्रिय ऐप से अलग करता है वह है इसका बिल्ट-इन फ़ाइल मैनेजर, जो उपयोगकर्ताओं को ऑडियो फ़ाइलें संपादित करने और उन्हें क्लाउड स्टोरेज से और उस पर सहजता से ट्रांसफर करने की शक्ति देता है।

## अनुभाग

चाहे आप iPhone, iPad, या Mac पर compact mode का उपयोग कर रहे हों, सभी ऐप सुविधाएँ स्क्रीन के निचले भाग में tab bar के माध्यम से आसानी से पहुँच योग्य हैं। iPad और Mac उपयोगकर्ताओं के लिए, वही मेनू स्क्रीन के बाईं ओर पाया जा सकता है। यह सुविचारित संगठन सभी ऐप सुविधाओं को आसानी से पहुँच योग्य अनुभागों में वर्गीकृत करता है, एक user-friendly और efficient अनुभव सुनिश्चित करता है।

**हाल के:** डिफ़ॉल्ट टैब जो ऐप शुरू होने पर खुलता है। आपकी हाल ही में खोली गई ऑडियो फ़ाइलें सूचीबद्ध करता है ताकि आप जहाँ छोड़ा था वहाँ से संपादन फिर से शुरू कर सकें।

**पसंदीदा:** उन फ़ाइलों और फ़ोल्डर पर जल्दी जाएँ जिन्हें आपने favorites के रूप में चिह्नित किया है — जिसमें गहराई से nested cloud folders भी शामिल हैं।

**संपर्क:** आप इस स्क्रीन से iCloud Drive, Google Drive, MEGA, OneDrive और Dropbox जैसी cloud storage सेवाओं के साथ-साथ अपने कंप्यूटर और व्यक्तिगत NAS को आसानी से कनेक्ट कर सकते हैं। स्क्रीन के शीर्ष पर one-tap access के लिए favorites में जोड़े गए cloud folders के साथ एक **त्वरित पहुँच** सूची भी दिखाई देती है।

**स्थानीय फ़ाइलें:** transfer queue पर नियंत्रण के साथ अपनी डाउनलोड की गई फ़ाइलें आसानी से ढूंढें और प्रबंधित करें। आप file management actions की एक श्रृंखला का उपयोग करके इन local files को संपादित कर सकते हैं। transfer queue तक पहुँचने के लिए, "Local Files" स्क्रीन के ऊपरी बाएँ कोने में spinning arrows icon टैप करें।

Local Files अनुभाग दो categories में विभाजित है: **इस एप्लिकेशन में फ़ाइलें**, एप्लिकेशन की Documents directory में local files प्रदर्शित करता है, और **इस iPad/iPhone/Mac पर फ़ाइलें**, डिवाइस पर लेकिन एप्लिकेशन की Documents directory के बाहर स्थित local files प्रकट करता है।

**सेटिंग्स:** एप्लिकेशन सेटिंग्स संशोधित करें, जिसमें file manager, tag editor preferences (album cover scaling, ID3v2.4 / duplicate tags, main-screen buttons), Wi-Fi Drive, personalization (app icon, theme, list density, context menu style), language, passcode, accessibility और window settings (iPad/Mac) शामिल हैं।

**टैग एडिटर:** टैग एडिटर तब खुलता है जब आप एक फ़ाइल पर टैप करते हैं या मेनू से "ऑडियो टैग संपादित करें" विकल्प चुनते हैं। iPad और Mac पर टैग एडिटर main split view के detail side के रूप में प्रस्तुत किया जाता है, इसलिए यह दाईं ओर रहता है जबकि आप बाईं ओर फ़ाइलों में नेविगेट करते हैं।

## अधिक क्रियाएँ

स्क्रीन पर व्यावहारिक रूप से प्रत्येक content item में एक "अधिक क्रियाएँ" बटन होता है। सभी उपलब्ध क्रियाओं तक पहुँचने के लिए उसे टैप करें।

## शीर्ष टूलबार

शीर्ष टूलबार, नेविगेशन बार के ठीक नीचे सुविधाजनक रूप से स्थित, कई उपयोगी क्रियाओं तक त्वरित पहुँच प्रदान करता है।
आप एक सीधे swipe-down gesture के साथ इस टूलबार को आसानी से प्रकट या छुपा सकते हैं। यहाँ क्रियाओं की सूची है:

- **खोजें:** वर्तमान context में खोज शुरू करें।

## Context Menu

Evertag में प्रत्येक cell — एक local file, एक cloud file, एक folder, एक Recents/Favorites entry, यहाँ तक कि एक connected cloud account — में एक context menu जुड़ा हुआ है। यह selection mode में प्रवेश किए बिना एकल आइटम पर act करने का सबसे तेज़ तरीका है, और यह item type के लिए हर प्रासंगिक action दिखाता है।

### इसे कैसे खोलें

context menu को बुलाने के दो तरीके हैं, और दोनों एक ही set of actions produce करते हैं:

- **"..." बटन।** प्रत्येक cell अपने शीर्षक के पास एक "..." (More Actions) बटन दिखाता है। मेनू लाने के लिए उसे टैप करें। यह iPhone, iPad और Mac पर समान रूप से काम करता है और VoiceOver उपयोगकर्ताओं के लिए accessible है।
- **Long press / right click.** iPhone और iPad पर, मेनू खोलने के लिए cell को **tap and hold** करें। Mac पर, इसके बजाय cell पर **right-click** (या Control-click) करें। आइटम का एक targeted preview मेनू के साथ दिखाई देता है ताकि आप confirm कर सकें कि आप सही फ़ाइल पर act कर रहे हैं।

यही gesture स्क्रीन के ऊपरी दाएँ कोने में folder के अपने title bar पर भी काम करता है — वह मेनू **वर्तमान में खुले फ़ोल्डर** पर act करता है और आपको Select, New Folder, Search, Sort, Grid/List और folder-level actions तक पहुँच देता है।

### अंदर क्या है

exact list आइटम पर निर्भर करती है, लेकिन आप आमतौर पर इस तरह की actions देखेंगे:

- **ऑडियो टैग संपादित करें** — बिल्ट-इन टैग एडिटर में फ़ाइल खोलें।
- **पसंदीदा में जोड़ें** — Favorites टैब में फ़ाइल या फ़ोल्डर pin करें।
- **डाउनलोड करें** — ऑफलाइन उपयोग के लिए cloud file या folder fetch करें; downloads transfer queue में दिखाई देते हैं।
- **अपलोड करें** — एक connected cloud service पर local file या folder push करें।
- **नाम बदलें / स्थानांतरित करें / कॉपी करें** — इसके origin storage पर फ़ाइल प्रबंधित करें।
- **इसमें खोलें** — system share sheet के माध्यम से किसी अन्य ऐप में फ़ाइल export करें।
- **Finder में दिखाएँ** *(Mac और "Files on this Mac")* — Finder में फ़ाइल reveal करें।
- **हटाना** — अपने वर्तमान स्थान से फ़ाइल हटाएँ। यह क्रिया अपरिवर्तनीय है।

यदि actions list screen overflow करती है, तो बाकी reveal करने के लिए menu के अंदर scroll करें।

### Context menu style

आप **सेटिंग्स → व्यक्तिगतकरण → Context menu style** से context menu कैसे present किया जाए यह चुन सकते हैं:

- **System menu** — targeted preview के साथ native iOS/macOS context menu उपयोग करता है। iPhone और iPad पर default।
- **Bottom sheet** — Evertag का custom bottom-sheet menu, उपयोगी जब आप एक बड़ा touch target पसंद करते हैं या labels को पढ़ने में आसान चाहते हैं।

विकल्प per-device है, इसलिए आप iPad पर bottom sheet के साथ iPhone पर system menus mix कर सकते हैं (या उल्टा)।

Σε αυτή την οθόνη, μπορείτε να συνδέσετε διάφορες πηγές που περιέχουν τα αρχεία ήχου σας. Μπορείτε να ενσωματώσετε δημοφιλείς υπηρεσίες cloud όπως Google Drive, Dropbox, OneDrive, iCloud και άλλες, καθώς και να συνδέσετε τον Mac ή τον PC σας. Επιπλέον, έχετε τη δυνατότητα να επεξεργαστείτε αρχεία ήχου που βρίσκονται στο Apple Time Capsule, WD Cloud Home ή σε οποιοδήποτε NAS που υποστηρίζει SMB ή WebDAV.

## Γρήγορη πρόσβαση

Στο επάνω μέρος της οθόνης Συνδέσεις θα βρείτε μια λίστα **Γρήγορης πρόσβασης**. Κάθε φάκελος cloud που προσθέτετε στα αγαπημένα — ακόμα κι αν βρίσκεται πολλά επίπεδα βαθύτερα — εμφανίζεται εδώ, ώστε να μπορείτε να μεταβείτε σε αυτόν χωρίς να πλοηγείστε κάθε φορά μέσα από γονικούς φακέλους.

## Σύνδεση με αποθήκευση cloud

- Ανοίξτε την καρτέλα «Συνδέσεις»
- Επιλέξτε «Σύνδεση με αποθήκευση cloud» από το μενού
- Επιλέξτε μια υπηρεσία αποθήκευσης cloud από τη λίστα
- Εισαγάγετε τα διαπιστευτήριά σας και πατήστε «Ολοκλήρωση».

Εάν αντιμετωπίσετε προβλήματα, ελέγξτε τη σύνδεσή σας στο Διαδίκτυο και το όνομα χρήστη/κωδικό πρόσβασης.
Στην Premium έκδοση της εφαρμογής, μπορείτε να προσθέσετε απεριόριστο αριθμό υπηρεσιών.

## Υποστηριζόμενες υπηρεσίες αποθήκευσης cloud

Επί του παρόντος, η εφαρμογή υποστηρίζει τις πιο δημοφιλείς υπηρεσίες αποθήκευσης cloud: iCloud Drive, Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk, pCloud, Synology Drive, MediaFire, WD My Cloud Home, InfiniCLOUD (TeraCLOUD), HiDrive, OpenDrive, MyDrive, Put.io, Cloud Mail.ru, Baidu Pan (百度网盘), καθώς και οποιονδήποτε διακομιστή SMB ή WebDAV.

## Ασφάλεια και απόρρητο

Χρησιμοποιούμε μόνο επίσημα SDKs και ασφαλείς συνδέσεις για αλληλεπίδραση με συνδεδεμένες υπηρεσίες cloud. Το όνομα χρήστη και ο κωδικός πρόσβασής σας δεν είναι προσβάσιμα από την εφαρμογή. Όλα τα αιτήματα από την εφαρμογή προς την υπηρεσία cloud είναι κρυπτογραφημένα.

Όταν εισάγετε το όνομα χρήστη και τον κωδικό πρόσβασής σας, η εφαρμογή σάς εμφανίζει την επίσημη σελίδα εξουσιοδότησης του παρόχου υπηρεσιών cloud, και ολόκληρη η διαδικασία εξουσιοδότησης πραγματοποιείται εκτός της εφαρμογής. Ο πάροχος υπηρεσιών cloud στέλνει ένα διακριτικό ελέγχου ταυτότητας στην εφαρμογή μετά την επιτυχή εξουσιοδότηση, και αυτό το διακριτικό χρησιμοποιείται για κλήσεις API.

Ένα διακριτικό ελέγχου ταυτότητας είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με αποθήκευση cloud. Το διακριτικό αποθηκεύεται στη συσκευή σας σε ασφαλή αποθήκευση συστήματος που ονομάζεται Keychain. Μπορείτε να κατεβάσετε αρχεία από τη συνδεδεμένη υπηρεσία cloud στη συσκευή, και αυτά τα αρχεία θα τοποθετηθούν στον κατάλογο «Documents» της εφαρμογής. Μπορείτε να αφαιρέσετε αυτά τα αρχεία ανά πάσα στιγμή χρησιμοποιώντας τον ενσωματωμένο διαχειριστή αρχείων.

Η εφαρμογή δεν μοιράζεται πληροφορίες από τον συνδεδεμένο λογαριασμό cloud. Μπορείτε να ανακαλέσετε την πρόσβαση στον λογαριασμό cloud σας ανά πάσα στιγμή ανοίγοντας τη σελίδα ρυθμίσεων λογαριασμού στο πρόγραμμα περιήγησής σας.

## Ανάκληση διακριτικού ελέγχου ταυτότητας

Συνδεθείτε στον λογαριασμό σας σε πρόγραμμα περιήγησης και μεταβείτε στη σελίδα ρυθμίσεων. Εκεί μπορείτε να βρείτε όλες τις εφαρμογές τρίτων που είναι συνδεδεμένες στον λογαριασμό cloud σας και να αφαιρέσετε οποιαδήποτε από αυτές εάν δεν θέλετε πλέον να χρησιμοποιείτε εκείνη την εφαρμογή. Λεπτομερείς οδηγίες είναι διαθέσιμες [εδώ](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account).

Μπορείτε επίσης να αποσυνδέσετε τους συνδεδεμένους λογαριασμούς cloud στην εφαρμογή, και το διακριτικό ελέγχου ταυτότητας θα αφαιρεθεί επίσης από τη συσκευή σας. Εάν αφαιρέσετε την εφαρμογή από τη συσκευή σας, όλα τα ληφθέντα δεδομένα και τα διακριτικά πρόσβασης θα αφαιρεθούν επίσης.

## Αποσύνδεση αποθήκευσης cloud ή αλλαγή διαμόρφωσης

- Πρόσβαση στις Επιλογές Αποθήκευσης Cloud: Πρώτα, εντοπίστε την αποθήκευση cloud που θέλετε να διαχειριστείτε στη διεπαφή της εφαρμογής.
- Πατήστε το κουμπί «...»: Δίπλα στον τίτλο της υπηρεσίας, θα δείτε ένα κουμπί «...». Πατήστε το για να αποκτήσετε πρόσβαση σε πρόσθετες επιλογές.
  - **Μετονομασία**: Εάν θέλετε να αλλάξετε το όνομα της υπηρεσίας cloud όπως εμφανίζεται στη λίστα σας, επιλέξτε «Μετονομασία».
  - **Ρυθμίσεις**: Για να τροποποιήσετε τη διαμόρφωση ή τα δεδομένα ελέγχου ταυτότητας για την υπηρεσία cloud, επιλέξτε «Ρυθμίσεις». Μερικές φορές, ίσως χρειαστεί να εκ νέου εξουσιοδοτήσετε τη συνδεδεμένη υπηρεσία cloud εάν το διακριτικό εξουσιοδότησης έχει λήξει.
  - **Αποσύνδεση**: Εάν θέλετε να διακόψετε εντελώς τη σύνδεση μεταξύ της εφαρμογής και της υπηρεσίας cloud, επιλέξτε «Αποσύνδεση». Να γνωρίζετε ότι η επιλογή αυτής της επιλογής θα αφαιρέσει όλα τα τραγούδια που σχετίζονται με αυτή την υπηρεσία cloud από τη βιβλιοθήκη μουσικής της εφαρμογής, αλλά θα παραμείνουν στον διακομιστή.

## Σύνδεση σε Υπολογιστή ή NAS

Μπορείτε επίσης να συνδέσετε τον υπολογιστή σας, προσωπικό NAS ή άλλες συσκευές δικτύου χρησιμοποιώντας το πρωτόκολλο SMB, DLNA ή WebDAV.

## Σύνδεση σε υπολογιστή μέσω SMB

- Πατήστε «Σύνδεση με αποθήκευση cloud» → SMB.
- Εισαγάγετε τη διεύθυνση IP του υπολογιστή και το όνομα του κοινόχρηστου φακέλου στο πεδίο URL χρησιμοποιώντας τη μορφή smb://διεύθυνση-ip-υπολογιστή/όνομα-κοινόχρηστου-φακέλου
- Επιλέξτε έκδοση πρωτοκόλλου: Auto, SMB1, SMB2
- Εισαγάγετε όνομα χρήστη και κωδικό πρόσβασης (εάν απαιτείται)
- Πατήστε «Ολοκλήρωση».

Εάν η σύνδεσή σας είναι επιτυχής, θα δείτε την αποθήκευση που συνδέθηκε στην ενότητα «Αποθήκευση cloud».
Ένα πλήρες εκπαιδευτικό βήμα-βήμα για τον τρόπο σύνδεσης του Mac ή PC σας μέσω SMB είναι διαθέσιμο [εδώ](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/).

## Σύνδεση σε NAS μέσω WebDAV

Όλα τα βήματα είναι ίδια εκτός από το πεδίο URL.
Η URL πρέπει να έχει τη μορφή http://όνομα-διακομιστή ή https://όνομα-διακομιστή εάν ο διακομιστής υποστηρίζει SSL.
Ένα πλήρες εκπαιδευτικό βήμα-βήμα για τον τρόπο σύνδεσης NAS χρησιμοποιώντας το πρωτόκολλο WebDAV είναι διαθέσιμο [εδώ](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac).

## Διαθέσιμες συσκευές

Αυτή η ενότητα εμφανίζει όλες τις συσκευές στο τοπικό δίκτυό σας στις οποίες μπορείτε να συνδεθείτε μέσω της εφαρμογής.
Για να δημιουργήσετε σύνδεση με μια συσκευή, ακολουθήστε αυτά τα βήματα:

- Ανοίξτε την εφαρμογή και μεταβείτε στην ενότητα «Διαθέσιμες συσκευές».
- Πατήστε τη συσκευή με την οποία θέλετε να συνδεθείτε από τη λίστα.
- Εάν χρειαστεί, εισαγάγετε τα στοιχεία σύνδεσής σας για να ολοκληρώσετε τη σύνδεση.

## Wi-Fi Drive

Το Wi-Fi Drive είναι μια βολική τεχνολογία που επιτρέπει ασύρματες μεταφορές αρχείων από τον υπολογιστή σας στη συσκευή iOS μέσω προγράμματος περιήγησης επιφάνειας εργασίας.
Για να χρησιμοποιήσετε αποτελεσματικά αυτή τη λειτουργία, βεβαιωθείτε ότι η συσκευή και ο υπολογιστής σας είναι συνδεδεμένοι στο ίδιο δίκτυο Wi-Fi.
Εδώ είναι ένας οδηγός βήμα-βήμα για τον τρόπο χρήσης του Wi-Fi Drive.

## Ενεργοποίηση Wi-Fi Drive

- Ανοίξτε την εφαρμογή και μεταβείτε στην καρτέλα «Συνδέσεις».
- Επιλέξτε «Σύνδεση μέσω Wi-Fi» για πρόσβαση στην κύρια οθόνη Wi-Fi Drive.
- Πατήστε «Έναρξη Wi-Fi Drive» για ενεργοποίηση του Wi-Fi Drive.

## Πρόσβαση στο Wi-Fi Drive από τον Υπολογιστή σας

- Στον υπολογιστή σας (επιτραπέζιος ή φορητός), ανοίξτε ένα πρόγραμμα περιήγησης (όπως Chrome, Firefox ή Safari).
- Στη γραμμή διευθύνσεων του προγράμματος περιήγησης, εισαγάγετε τη διεύθυνση URL που παρέχεται από την εφαρμογή.

## Ασύρματη Μεταφορά Αρχείων

Μόλις ανοίξει στο πρόγραμμα περιήγησης η ιστοσελίδα που αντιστοιχεί στη συσκευή iOS σας, μπορείτε εύκολα να σύρετε και να αποθέσετε αρχεία από τον υπολογιστή σας στην ιστοσελίδα.
Τα αρχεία που σύρετε και αποθέτετε θα αρχίσουν να μεταφέρονται στη συσκευή iOS σας και θα είναι προσβάσιμα μέσα από την εφαρμογή.

Λεπτομερείς οδηγίες για τον τρόπο ασύρματης μεταφοράς αρχείων χρησιμοποιώντας το Wi-Fi Drive είναι διαθέσιμες [εδώ](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).

## iTunes File Sharing

Το iTunes File Sharing είναι μια άλλη τεχνολογία που σας επιτρέπει να μεταφέρετε αρχεία από υπολογιστή σε συσκευή χρησιμοποιώντας την εφαρμογή Finder στο Mac σας και καλώδιο Lightning.
- Απλώς συνδέστε τη συσκευή στον υπολογιστή με καλώδιο και εκτελέστε την εφαρμογή Finder στο Mac σας.
- Ανοίξτε «Τοποθεσίες» → «Η Συνδεδεμένη Συσκευή σας» → «Αρχεία» → και βρείτε την τρέχουσα εφαρμογή.
- Πατήστε στο εικονίδιο εφαρμογής για να δείτε όλους τους κοινόχρηστους φακέλους.
- Αντιγράψτε αρχεία από τον υπολογιστή στον κοινόχρηστο φάκελο στη συσκευή με σύρσιμο και απόθεση.

Λεπτομερείς οδηγίες για τον τρόπο χρήσης του iTunes File Sharing είναι διαθέσιμες [εδώ](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/).

## Σύνδεση κάρτας USB flash

Εάν έχετε κάρτα SD ή στικ USB, μπορείτε να τη συνδέσετε χρησιμοποιώντας αναγνώστη καρτών Lightning ή USB-C σε iPhone/iPad ή να τη συνδέσετε απευθείας σε Mac. Η εφαρμογή υποστηρίζει επί του παρόντος πιστοποιημένους αναγνώστες καρτών Apple. Έχουμε λεπτομερείς οδηγίες για τον τρόπο σύνδεσης κάρτας USB flash στο iPhone ή iPad σας και διαχείρισης αρχείων σε αυτή, διαθέσιμες [εδώ](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it). Συνδεδεμένες μονάδες εμφανίζονται στην ενότητα **Συνδεδεμένα αξεσουάρ** της οθόνης Συνδέσεις.

## Διαχειριστής αρχείων

Μόλις συνδέσετε μια υπηρεσία αποθήκευσης cloud, πατήστε στο εικονίδιο για να δείτε όλα τα διαθέσιμα αρχεία και φακέλους. Μπορείτε να χρησιμοποιήσετε τον ενσωματωμένο διαχειριστή αρχείων για να εργαστείτε με αυτά τα αρχεία — λήψη, μετονομασία, μετακίνηση και άλλα. Όταν ξεκινάτε μια λήψη, το αρχείο θα εμφανιστεί στην ουρά μεταφοράς. Για να δείτε την ουρά μεταφοράς, μεταβείτε στην καρτέλα «Τοπικά αρχεία» και πατήστε τα περιστρεφόμενα βέλη στην επάνω αριστερή γωνία. Όλα τα ληφθέντα αρχεία και φάκελοι είναι διαθέσιμα στην ενότητα «Τοπικά αρχεία».

## Επάνω Γραμμή Εργαλείων

Η επάνω γραμμή εργαλείων, βολικά τοποθετημένη κάτω από τη γραμμή πλοήγησης, προσφέρει αρκετές χρήσιμες ενέργειες για εύκολη πρόσβαση. Μπορείτε να εμφανίσετε ή να αποκρύψετε αυτή τη γραμμή εργαλείων με μια απλή κίνηση σάρωσης προς τα κάτω. Εδώ είναι οι διαθέσιμες ενέργειες:

- **Αναζήτηση**: Αυτή η επιλογή σάς επιτρέπει να πραγματοποιείτε γρήγορη αναζήτηση μέσα στον τρέχοντα κατάλογο, καθιστώντας εύκολο τον εντοπισμό συγκεκριμένων αρχείων ή φακέλων.

## Επιλογές Φακέλου

Όταν ανοίγετε ένα φάκελο μέσα στην εφαρμογή, θα βρείτε ένα χρήσιμο σύνολο ενεργειών που είναι διαθέσιμες πατώντας το κουμπί «...» στην επάνω δεξιά γωνία της οθόνης.
Εδώ είναι μια ανάλυση αυτών των ενεργειών:

- **Επιλογή**: Ενεργοποιήστε τη λειτουργία επιλογής αρχείων. Αυτή η λειτουργία σάς επιτρέπει να επιλέξετε ένα ή περισσότερα αρχεία μέσα στο φάκελο, καθιστώντας εύκολη την εκτέλεση ενεργειών σε επιλεγμένα στοιχεία.
- **Νέος φάκελος**: Δημιουργήστε νέο φάκελο μέσα στον τρέχοντα φάκελο. Αυτή η λειτουργία σάς επιτρέπει να οργανώσετε τα αρχεία σας και να διατηρήσετε καλά δομημένη τη βιβλιοθήκη σας.
- **Αποστολή αρχείων**: Ανεβάστε αρχεία από τη συσκευή σας στον διαδικτυακό φάκελο. Αυτή η ενέργεια σάς επιτρέπει να μεταφέρετε αρχεία στο cloud ή τον διακομιστή, καθιστώντας τα προσβάσιμα από οπουδήποτε.
- **Αναζήτηση**: Αναζητήστε συγκεκριμένα αρχεία μέσα στον τρέχοντα φάκελο. Αυτό είναι ιδιαίτερα χρήσιμο για γρήγορο εντοπισμό συγκεκριμένων στοιχείων σε μεγάλη συλλογή.
- **Ταξινόμηση**: Ταξινομήστε αρχεία μέσα στο φάκελο βάσει κριτηρίων όπως όνομα, μέγεθος ή ημερομηνία επεξεργασίας. Η προεπιλεγμένη λειτουργία ταξινόμησης αντικατοπτρίζει συνήθως τη σειρά ταξινόμησης στον διακομιστή, αλλά μπορείτε να την αλλάξετε σύμφωνα με τις προτιμήσεις σας.
- **Προβολή πλέγματος/λίστας**: Εναλλάξτε μεταξύ δύο λειτουργιών προβολής: προβολή πίνακα και προβολή μικρογραφιών. Η προβολή πίνακα παρουσιάζει αρχεία σε λίστα, ενώ η προβολή μικρογραφιών εμφανίζει οπτικές αναπαραστάσεις των αρχείων, καθιστώντας ευκολότερη την αναγνώριση περιεχομένου με μια ματιά.

## Επεξεργασία διαδικτυακών αρχείων

Όταν χρειάζεται να διαχειριστείτε πολλά αρχεία στον αποθηκευτικό χώρο cloud σε αυτή την εφαρμογή, μπορείτε να χρησιμοποιήσετε τη λειτουργία επιλογής για να βελτιστοποιήσετε τις ενέργειές σας. Ακολουθήστε αυτά τα βήματα για αποτελεσματική διαχείριση αρχείων:

- **Ενεργοποίηση Λειτουργίας Επιλογής**: Ανοίξτε την εφαρμογή στη συσκευή σας και μεταβείτε στην ενότητα που περιέχει τον αποθηκευτικό χώρο cloud σας. Αναζητήστε στην επάνω δεξιά γωνία το κουμπί «...» (αποσιωπητικά). Πατήστε το και επιλέξτε το στοιχείο μενού «Επιλογή» για ενεργοποίηση της λειτουργίας επιλογής.
- **Επιλογή Αρχείων**: Θα παρατηρήσετε πλαίσια ελέγχου να εμφανίζονται δίπλα σε κάθε αρχείο ή φάκελο που αναφέρεται. Επιλέξτε ένα ή πολλά αρχεία ή φακέλους απλώς πατώντας στα πλαίσια ελέγχου δίπλα τους.
- **Εκτέλεση Διαφόρων Ενεργειών**: Μόλις επιλέξετε τα αρχεία ή τους φακέλους που θέλετε να διαχειριστείτε, θα έχετε πρόσβαση σε αρκετές ενέργειες προσαρμοσμένες στις ανάγκες σας:

## Ενέργειες αρχείου

Κοντά στον τίτλο του αρχείου, θα παρατηρήσετε το σύμβολο αποσιωπητικών «...» (τρεις τελείες) – αυτό είναι το μενού ενεργειών.
Πατήστε το για να εμφανίσετε μια λίστα διαθέσιμων ενεργειών:

- **Επεξεργασία ετικετών ήχου**: Επιλέγοντας αυτή την επιλογή, μπορείτε να αποκτήσετε πρόσβαση στον ενσωματωμένο επεξεργαστή ετικετών, επιτρέποντάς σας να τροποποιήσετε τις ετικέτες ήχου για το επιλεγμένο αρχείο. Το αρχείο θα ληφθεί προσωρινά σε προσωρινό κατάλογο και στη συνέχεια θα ανέβει στην αποθήκευση αφού επιβεβαιώσετε τις αλλαγές.
- **Προσθήκη στα Αγαπημένα**: Αυτή η ενέργεια προσθέτει το αρχείο στη λίστα αγαπημένων αρχείων σας για γρήγορη και βολική πρόσβαση.
- **Λήψη**: Επιλέξτε αυτή την ενέργεια για λήψη του αρχείου ή φακέλου στη συσκευή σας, καθιστώντας το προσβάσιμο για εκτός σύνδεσης χρήση.
- **Μετονομασία**: Αυτή η επιλογή σάς επιτρέπει να μετονομάσετε το αρχείο απευθείας στον απομακρυσμένο χώρο αποθήκευσης, επιτρέποντας προσαρμοσμένη ονομασία αρχείων.
- **Μετακίνηση**: Επιλέξτε αυτή την ενέργεια για να μετεγκαταστήσετε το αρχείο σε διαφορετικό φάκελο μέσα στον αποθηκευτικό χώρο cloud σας, βοηθώντας στη διατήρηση οργανωμένης δομής αρχείων.
- **Άνοιγμα σε**: Χρησιμοποιήστε αυτή την ενέργεια για εξαγωγή του αρχείου σε άλλη συμβατή εφαρμογή. Το αρχείο θα ληφθεί στη συσκευή σας και στη συνέχεια θα εμφανιστεί το παράθυρο διαλόγου κοινοποίησης για περαιτέρω κοινοποίηση ή αλληλεπίδραση.
- **Διαγραφή**: Να είστε προσεκτικοί με αυτή την ενέργεια, καθώς αφαιρεί μόνιμα το αρχείο από τον αποθηκευτικό χώρο cloud σας. **Αυτή η διαγραφή δεν μπορεί να αναιρεθεί**.

## Ενέργειες φακέλου

Για κάθε φάκελο στον αποθηκευτικό χώρο cloud σας, διατίθενται διάφορες ενέργειες. Για να αποκτήσετε πρόσβαση σε αυτές τις επιλογές, απλώς πατήστε το εικονίδιο αποσιωπητικών «...» που βρίσκεται δίπλα στον τίτλο φακέλου. Εάν δεν βλέπετε όλες τις ενέργειες, μετακινηθείτε προς τα κάτω για να αποκαλύψετε περισσότερες επιλογές. Εδώ είναι οι διαθέσιμες ενέργειες:

- **Προσθήκη στα Αγαπημένα**: Χρησιμοποιήστε αυτή την ενέργεια για να προσθέσετε τον φάκελο στη λίστα αγαπημένων αρχείων σας για γρήγορη και βολική πρόσβαση.
- **Λήψη**: Λήψη όλου του περιεχομένου του φακέλου στη συσκευή σας για εκτός σύνδεσης πρόσβαση.
- **Μετονομασία**: Μετονομάστε τον φάκελο απευθείας στον απομακρυσμένο χώρο αποθήκευσης.
- **Μετακίνηση**: Μετεγκαταστήστε τον φάκελο σε διαφορετική τοποθεσία μέσα στον αποθηκευτικό χώρο cloud σας.
- **Διαγραφή**: Να είστε προσεκτικοί με αυτή την ενέργεια, καθώς αφαιρεί μόνιμα τον φάκελο και το περιεχόμενό του από τον αποθηκευτικό χώρο cloud σας. **Αυτή η ενέργεια δεν μπορεί να αναιρεθεί**.

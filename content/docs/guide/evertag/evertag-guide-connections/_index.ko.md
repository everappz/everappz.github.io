이 화면에서 오디오 파일이 포함된 다양한 소스를 연결할 수 있습니다. Google Drive, Dropbox, OneDrive, iCloud 등 인기 있는 클라우드 서비스를 통합하고 Mac 또는 PC를 연결할 수 있습니다. 또한 Apple Time Capsule, WD Cloud Home 또는 SMB나 WebDAV를 사용하는 모든 NAS에 있는 오디오 파일을 편집하는 옵션도 있습니다.

## 빠른 액세스

연결 화면 상단에는 **빠른 액세스** 목록이 있습니다. 즐겨찾기에 추가한 클라우드 폴더는 — 여러 단계 깊이에 있어도 — 상위 폴더를 매번 탐색하지 않고도 바로 이동할 수 있도록 여기에 표시됩니다.

## 클라우드 스토리지 연결

- '연결하기' 탭 열기  
- 메뉴에서 '클라우드 스토리지 연결' 선택  
- 목록에서 클라우드 스토리지 서비스 선택  
- 자격 증명을 입력하고 '완료됨'을 탭하세요.

문제가 발생하면 인터넷 연결 및 로그인/비밀번호를 확인하세요.  
프리미엄 버전에서는 무제한 서비스를 추가할 수 있습니다.

## 지원되는 클라우드 스토리지 서비스

현재 애플리케이션은 가장 인기 있는 클라우드 스토리지 서비스를 지원합니다: iCloud Drive, Google Drive, Dropbox, OneDrive, Box, MEGA, Yandex.Disk, pCloud, Synology Drive, MediaFire, WD My Cloud Home, InfiniCLOUD (TeraCLOUD), HiDrive, OpenDrive, MyDrive, Put.io, Cloud Mail.ru, Baidu Pan (百度网盘), 그리고 모든 SMB 또는 WebDAV 서버.

## 보안 및 개인 정보 보호

연결된 클라우드 서비스와 상호 작용하기 위해 공식 SDK와 보안 연결만 사용합니다. 애플리케이션에서 로그인과 비밀번호에 액세스할 수 없습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.

로그인과 비밀번호를 입력하면 애플리케이션이 클라우드 서비스 제공업체가 제공하는 공식 인증 페이지를 표시하며, 전체 인증 프로세스는 애플리케이션 외부에서 이루어집니다. 클라우드 서비스 제공업체는 성공적인 인증 후 애플리케이션에 인증 토큰을 보내며, 해당 토큰은 API 호출에 사용됩니다.

인증 토큰은 타사 애플리케이션이 클라우드 스토리지와 상호 작용할 수 있게 하는 디지털 키입니다. 인증 토큰은 Keychain이라는 안전한 시스템 스토리지에 기기에 저장됩니다. 연결된 클라우드 서비스에서 기기로 파일을 다운로드할 수 있으며, 해당 파일은 앱의 "문서" 디렉토리에 배치됩니다. 내장 파일 관리자를 사용하여 언제든지 이러한 파일을 제거할 수 있습니다.

애플리케이션은 연결된 클라우드 계정의 어떤 정보도 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정에 대한 액세스를 취소할 수 있습니다.

## 인증 토큰 거부

웹 브라우저에서 계정에 로그인하고 설정 페이지로 이동하세요. 거기서 클라우드 계정에 연결된 모든 타사 앱을 찾고 더 이상 사용하지 않으려는 앱을 제거할 수 있습니다. 자세한 지침은 [여기](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account)에서 확인할 수 있습니다.

애플리케이션에서 연결된 클라우드 계정을 연결 끊기하면 인증 토큰도 기기에서 제거됩니다. 기기에서 애플리케이션을 제거하면 모든 다운로드된 데이터와 액세스 토큰도 제거됩니다.

## 클라우드 스토리지 연결 끊기 또는 구성 변경

- 클라우드 스토리지 옵션 액세스: 먼저 앱 인터페이스에서 관리하려는 클라우드 스토리지를 찾으세요.  
- '...' 버튼 탭: 서비스 제목 옆에 '...' 버튼이 있습니다. 탭하여 추가 옵션에 액세스하세요.  
  - **이름 바꾸기**: 목록에 표시되는 클라우드 서비스 이름을 변경하려면 '이름 바꾸기'를 선택하세요.  
  - **설정**: 클라우드 서비스의 구성 또는 인증 데이터를 수정하려면 '설정'을 선택하세요. 인증 토큰이 만료된 경우 연결된 클라우드 서비스를 재인증해야 할 수 있습니다.  
  - **연결 끊기**: 앱과 클라우드 서비스 간의 연결을 완전히 끊으려면 '연결 끊기'를 선택하세요. 이 옵션을 선택하면 이 클라우드 서비스와 관련된 모든 노래가 앱의 음악 라이브러리에서 제거되지만 서버에는 남아 있습니다.

## 컴퓨터 또는 NAS 연결

SMB, DLNA 또는 WebDAV 프로토콜을 사용하여 컴퓨터, 개인 NAS 또는 기타 네트워크 장치를 연결할 수도 있습니다.

## SMB를 사용하여 컴퓨터에 연결

- "클라우드 스토리지 연결" → SMB를 탭하세요.  
- URL 필드에 smb://컴퓨터-IP-주소/공유-폴더-이름 형식으로 컴퓨터 IP 주소와 공유 폴더 이름을 입력하세요  
- 프로토콜 버전 선택: 자동, SMB1, SMB2  
- 로그인과 비밀번호 입력 (필요한 경우)  
- "완료됨"을 탭하세요.

연결에 성공하면 "클라우드 스토리지" 섹션에 연결된 스토리지가 표시됩니다.  
SMB를 사용하여 Mac 또는 PC를 연결하는 방법에 대한 전체 튜토리얼은 [여기](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)에서 확인할 수 있습니다.

## WebDAV를 사용하여 NAS 연결

모든 단계는 URL 필드를 제외하고 동일합니다.  
URL은 http://서버-이름 형식이어야 하며, 서버가 SSL을 지원하는 경우 https://서버-이름이어야 합니다.  
WebDAV 프로토콜을 사용하여 NAS를 연결하는 방법에 대한 전체 튜토리얼은 [여기](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)에서 확인할 수 있습니다.

## 사용 가능한 장치

이 섹션에는 애플리케이션을 통해 연결할 수 있는 로컬 네트워크 내의 모든 장치가 표시됩니다.  
장치와 연결을 설정하려면 다음 단계를 따르세요:

- 앱을 열고 "사용 가능한 장치" 섹션으로 이동하세요.  
- 목록에서 연결하려는 장치를 탭하세요.  
- 필요한 경우 연결을 완료하기 위해 로그인 세부 정보를 입력하세요.

## Wi-Fi Drive 

Wi-Fi Drive는 데스크톱 브라우저를 통해 컴퓨터에서 iOS 기기로 무선 파일 전송을 가능하게 하는 편리한 기술입니다.  
이 기능을 효과적으로 사용하려면 기기와 컴퓨터가 동일한 Wi-Fi 네트워크에 연결되어 있는지 확인하세요.  
Wi-Fi Drive 사용 방법에 대한 단계별 가이드입니다.

## Wi-Fi Drive 활성화

- 애플리케이션을 열고 "연결하기" 탭으로 이동하세요.  
- "Wi-Fi를 통해 연결"을 선택하여 Wi-Fi Drive 메인 화면에 접근하세요.  
- "Wi-Fi Drive 시작"을 탭하여 Wi-Fi Drive를 활성화하세요.

## 컴퓨터에서 Wi-Fi Drive 액세스

- 컴퓨터(데스크톱 또는 노트북)에서 웹 브라우저(Chrome, Firefox, Safari 등)를 여세요.  
- 브라우저 주소 표시줄에 애플리케이션이 제공하는 URL을 입력하세요.

## 무선으로 파일 전송

iOS 기기에 해당하는 웹 페이지가 브라우저에서 열리면 컴퓨터에서 웹 페이지로 파일을 쉽게 끌어다 놓을 수 있습니다.  
끌어다 놓은 파일은 iOS 기기로 전송되기 시작하며 애플리케이션 내에서 액세스할 수 있게 됩니다.

Wi-Fi Drive를 사용하여 무선으로 파일을 전송하는 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)에서 확인할 수 있습니다.

## iTunes 파일 공유

iTunes 파일 공유는 Mac의 Finder 앱과 Lightning 케이블을 사용하여 컴퓨터에서 기기로 파일을 전송할 수 있게 하는 또 다른 기술입니다.  
- 케이블을 사용하여 기기를 컴퓨터에 연결하고 Mac에서 Finder 앱을 실행하세요.  
- "위치" → "연결된 기기" → "파일"을 열고 현재 앱을 찾으세요.  
- 앱 아이콘을 탭하여 모든 공유 폴더를 보세요.  
- 끌어다 놓기를 사용하여 컴퓨터에서 기기의 공유 폴더로 파일을 복사하세요.

iTunes 파일 공유 사용 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-play-local-itunes-files-on-my-iphone/)에서 확인할 수 있습니다.

## USB 플래시카드 연결

SD 카드나 USB 스틱이 있는 경우 iPhone/iPad에서 Lightning 또는 USB-C 카드 리더기를 사용하여 연결하거나 Mac에 직접 연결할 수 있습니다. 앱은 현재 Apple 인증 카드 리더기를 지원합니다. iPhone 또는 iPad에 USB 플래시카드를 연결하고 그 위에 있는 파일을 관리하는 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it)에서 확인할 수 있습니다. 연결된 드라이브는 연결 화면의 **외부 액세서리** 섹션에 나타납니다.

## 파일 관리자

클라우드 스토리지 서비스를 연결한 후 아이콘을 탭하면 사용 가능한 모든 파일과 폴더가 표시됩니다. 내장 파일 관리자를 사용하여 이러한 파일을 작업할 수 있습니다 — 다운로드, 이름 바꾸기, 이동 등. 다운로드를 시작하면 파일이 전송 대기열에 나타납니다. 전송 대기열을 보려면 "로컬 파일" 탭으로 이동하여 왼쪽 상단의 회전 화살표를 탭하세요. 다운로드된 모든 파일과 폴더는 "로컬 파일" 섹션에서 사용할 수 있습니다.

## 상단 도구 모음

내비게이션 바 아래에 편리하게 위치한 상단 도구 모음은 쉬운 액세스를 위한 여러 유용한 작업을 제공합니다. 간단한 아래로 스와이프 제스처를 사용하여 이 도구 모음을 표시하거나 숨길 수 있습니다. 사용 가능한 작업은 다음과 같습니다:

- **검색**: 이 옵션을 사용하면 현재 디렉토리 내에서 빠른 검색을 수행하여 특정 파일이나 폴더를 쉽게 찾을 수 있습니다.  

## 폴더 옵션

앱에서 폴더를 열면 화면 오른쪽 상단의 "..." 버튼을 탭하여 사용 가능한 유용한 작업 세트를 찾을 수 있습니다.  
다음은 이러한 작업에 대한 설명입니다:

- **선택하다**: 파일 선택 모드를 활성화하세요. 이 모드에서는 폴더 내의 하나 이상의 파일을 선택하여 선택한 항목에 대한 작업을 쉽게 수행할 수 있습니다.  
- **새 폴더**: 현재 폴더 내에 새 폴더를 만드세요. 이 기능을 사용하면 파일을 구성하고 라이브러리를 잘 정리할 수 있습니다.   
- **파일 업로드**: 기기에서 온라인 폴더로 파일을 업로드하세요. 이 작업을 통해 파일을 클라우드나 서버로 전송하여 어디서나 액세스할 수 있게 합니다.  
- **검색**: 현재 폴더 내에서 특정 파일을 검색하세요. 이는 대규모 컬렉션에서 특정 항목을 빠르게 찾는 데 특히 유용합니다.  
- **정렬**: 이름, 크기 또는 편집 날짜와 같은 기준으로 폴더 내의 파일을 정렬하세요. 기본 정렬 모드는 일반적으로 서버의 정렬 순서를 반영하지만 기본 설정에 맞게 변경할 수 있습니다.  
- **그리드/목록 보기**: 테이블 보기와 썸네일 보기 두 가지 보기 모드 간에 전환하세요. 테이블 보기는 파일을 목록으로 표시하고, 썸네일 보기는 파일의 시각적 표현을 보여주어 한 눈에 내용을 쉽게 식별할 수 있습니다.

## 온라인 파일 편집

이 앱에서 클라우드 스토리지 내의 여러 파일을 관리해야 하는 경우 선택 모드를 사용하여 작업을 간소화할 수 있습니다. 효과적인 파일 관리를 위해 다음 단계를 따르세요:

- **선택 모드 활성화**: 기기에서 앱을 열고 클라우드 스토리지가 포함된 섹션으로 이동하세요. "..." (줄임표) 버튼이 있는 오른쪽 상단 모서리를 찾으세요. 탭한 후 "선택하다" 메뉴 항목을 선택하여 선택 모드를 활성화하세요.  
- **파일 선택**: 나열된 모든 파일이나 폴더 옆에 체크박스가 나타납니다. 옆의 체크박스를 탭하여 하나 또는 여러 파일이나 폴더를 선택하세요.  
- **다양한 작업 수행**: 관리하려는 파일이나 폴더를 선택한 후 필요에 맞게 맞춤화된 여러 작업에 액세스할 수 있습니다:

## 파일 작업

파일 제목 옆에 줄임표 기호 "..." (점 세 개)가 표시됩니다 – 이것이 작업 메뉴입니다.  
탭하면 사용 가능한 작업 목록이 표시됩니다:

- **오디오 태그 편집**: 이 옵션을 선택하면 내장 태그 편집기에 액세스하여 선택한 파일의 오디오 태그를 수정할 수 있습니다. 파일은 임시 디렉토리에 일시적으로 다운로드되고 변경 사항을 확인한 후 스토리지에 업로드됩니다.  
- **즐겨찾기에 추가**: 이 작업은 파일을 즐겨찾기 파일 목록에 추가하여 빠르고 편리하게 액세스할 수 있게 합니다.  
- **다운로드**: 파일이나 폴더를 기기에 다운로드하여 오프라인으로 사용할 수 있게 하려면 이 작업을 선택하세요.  
- **이름 바꾸기**: 이 옵션을 사용하면 원격 스토리지에서 직접 파일 이름을 바꾸어 맞춤 파일 이름 지정을 할 수 있습니다.  
- **이동**: 이 작업을 선택하면 파일을 클라우드 스토리지 내의 다른 폴더로 이동하여 체계적인 파일 구조를 유지하는 데 도움이 됩니다.  
- **다른 앱에서 열기**: 이 작업을 사용하여 파일을 다른 호환 앱으로 내보내세요. 파일이 기기에 다운로드되고 추가 공유나 상호 작용을 위해 공유 대화 상자가 나타납니다.  
- **삭제하기**: 이 작업은 파일을 클라우드 스토리지에서 영구적으로 제거하므로 주의하세요. **이 삭제는 취소할 수 없습니다**.

## 폴더 작업

클라우드 스토리지의 각 폴더에 대해 다양한 작업을 사용할 수 있습니다. 이러한 옵션에 액세스하려면 폴더 제목 옆에 있는 줄임표 아이콘 "..."을 탭하세요. 모든 작업이 보이지 않으면 아래로 스크롤하여 더 많은 선택 사항을 확인하세요. 사용 가능한 작업은 다음과 같습니다:

- **즐겨찾기에 추가**: 이 작업을 사용하여 폴더를 즐겨찾기 파일 목록에 추가하여 빠르고 편리하게 액세스할 수 있게 하세요.  
- **다운로드**: 오프라인 액세스를 위해 폴더의 모든 내용을 기기에 다운로드하세요.  
- **이름 바꾸기**: 원격 스토리지에서 직접 폴더 이름을 바꾸세요.  
- **이동**: 폴더를 클라우드 스토리지 내의 다른 위치로 이동하세요.  
- **삭제하기**: 이 작업은 폴더와 그 내용을 클라우드 스토리지에서 영구적으로 제거하므로 주의하세요. **이 작업은 취소할 수 없습니다**.

**Редактор тегов** — это главный экран приложения Evertag, где можно просматривать и редактировать метаданные аудиофайлов. Откройте этот экран, нажав на файл в разделе **Локальные файлы** или в любом подключённом аккаунте **облачного хранилища**.

## Режимы редактирования

Evertag предоставляет два режима редактирования:

- **Режим одного файла**  
  - Переходите между файлами, смахивая влево или вправо по карусели обложек.  

- **Пакетный режим**  
  - Редактируйте несколько файлов одновременно и применяйте общие метаданные.  
  - Для активации прокрутите вниз и нажмите **Редактировать файлы одновременно**.

## Режим одного файла

По умолчанию приложение открывает редактор тегов в режиме одного файла с включёнными только основными параметрами редактирования. В этом режиме можно редактировать наиболее распространённые поля метаданных:

**Название трека, Подзаголовок, Исполнитель альбома, Альбом, Исполнитель, Композитор, Исполнитель, Жанр, Комментарий, Удары в минуту, Подкаст, Компиляция, Номер диска, Номер трека, Всего треков, Рейтинг, Год**

Для доступа ко всем доступным тегам прокрутите вниз экрана и нажмите параметр **Показать расширенные теги**. Это переключит редактор в расширенный режим, позволяя редактировать более **120 полей метаданных**, включая **Теги MusicBrainz**, **Тексты песен**, **Консультативные рейтинги**, значения replay-gain, порядки сортировки, метаданные подкаста и многое другое. Используйте **Настройки → Редактор аудиотегов → Кнопки на главном экране**, чтобы постоянно включить «Показать расширенные теги» так, чтобы он всегда был активен.

## Пакетный режим

Войти в пакетное редактирование можно двумя способами:

1. **Из файлового менеджера**  
   - Нажмите **Другие действия** (•••) в верхнем правом углу.  
   - Нажмите **Выбрать**, выберите несколько файлов, затем нажмите **Редактировать теги**.

2. **Из редактора тегов**  
   - Откройте любой файл, прокрутите вниз и нажмите **Редактировать файлы одновременно** для загрузки всех файлов из той же папки.

## Редактирование текстов песен

В расширенном редакторе доступно поле **Тексты песен**. Нажмите на него, чтобы открыть список текстов — каждая запись имеет свой язык и описание, поэтому один трек может содержать несколько переводов.

Не нужно вводить тексты с нуля. Редактор включает быстрые ссылки для поиска одним касанием в самых популярных базах данных текстов в интернете, предварительно заполненных исполнителем и названием текущего трека:

- **Lrclib** — главная публичная база данных для **синхронизированных текстов (формат LRC)**. Используйте её, когда нужны синхронизированные тексты, прокручивающиеся строка за строкой в поддерживающих их плеерах.
- **Genius** — большой каталог с аннотациями и точными текстами в виде простого текста.
- **Lyricsify** — база данных сообщества с простыми и временными текстами.
- **Google** — общий веб-поиск как запасной вариант, когда специализированные базы данных не имеют соответствия.

Каждая ссылка отображается только тогда, когда соответствующий сервис доступен с вашего устройства. Нажмите на сервис, скопируйте нужные тексты (или временные метки LRC), вернитесь в Evertag и вставьте их в текстовое поле — затем нажмите **Сохранить** для записи текстов обратно в теги аудиофайла.

## Установка рейтинга и консультативного рейтинга

Расширенный редактор предлагает звёздный элемент управления **Рейтинг** вместе с сегментированным элементом управления **Консультативный рейтинг**.

### Звёздный рейтинг

Используйте поле **Рейтинг**, чтобы дать треку личную оценку от одной до пяти звёзд. Значение записывается в стандартное поле рейтинга файла (POPM для ID3, `rate` для MP4, `RATING` для Vorbis/APE и т.д.), поэтому другие приложения, читающие этот тег — включая приложение Music, Plex, Roon и большинство настольных редакторов тегов — немедленно увидят ваши оценки.

### Консультативный рейтинг

**Консультативный рейтинг** отмечает содержимое трека с помощью одного из значений стандарта Parental Advisory, используемого iTunes Store и большинством музыкальных платформ:

- **Без ограничений** — значение по умолчанию для треков без информации о родительском контроле. Файл рассматривается как подходящий для всех слушателей и не отображает никакой метки контроля в плеерах, поддерживающих этот тег.
- **Explicit** — трек содержит нецензурный язык или контент. Плееры, поддерживающие родительский контроль (приложение Music, приложение Apple TV, экспорт Spotify, Plex и т.д.), будут показывать значок **E** рядом с названием и при включённых ограничениях на устройстве или в аккаунте могут скрывать трек из детских профилей или отказываться от воспроизведения.
- **Чистый** — цензурированная или отредактированная версия изначально явного трека. Плееры показывают значок **C**, чтобы слушатели могли отличить чистую версию от оригинального явного мастера — полезно, когда обе версии находятся в одной библиотеке.

Нужно установить или исправить это поле, когда:

- Файл имеет неправильную метку (например, чистое радиоредактирование, ошибочно помеченное как Explicit, или наоборот).
- Треки были перекодированы или скачаны без тега и теперь отображаются как «Без ограничений», хотя содержат явный контент — необходимо для корректной работы родительского контроля и семейных общих библиотек.
- Вы готовите альбом к выпуску или совместному использованию и вам нужно, чтобы каждый трек имел согласованные метаданные.
- Вы хотите, чтобы CarPlay, экран блокировки, плееры в стиле Apple Music или DJ-программы отображали правильный значок **E** / **C** рядом с названием трека.

Значение хранится в стандартном поле консультативного рейтинга для формата файла (`rtng` для MP4, `TXXX:ITUNESADVISORY` для ID3, `ITUNESADVISORY` для Vorbis), поэтому любой плеер, читающий метаданные родительского контроля, увидит ваше обновление.

2. Выберите расположение изображения: Локальные файлы, Облако, Загрузки или Библиотека фотографий.

## Настройки редактора тегов

Перейдите в **Настройки → Редактор аудиотегов** для настройки поведения.

### Масштабирование обложки альбома

Выберите способ масштабирования обложек альбомов при сохранении в аудиофайлы. Можно отключить масштабирование для сохранения оригинального размера изображения, но имейте в виду, что некоторые плееры могут не поддерживать большие файлы обложек. Опция «оригинальный размер» является частью функций Premium персонализации.

### Параметры сохранения тегов

- **ID3v2.4** — при включении приложение сохраняет теги в формате ID3v2.4, когда это возможно. Отключите для возврата к более широко поддерживаемому ID3v2.3, если аудиотеги некорректно отображаются на старых плеерах или устройствах.
- **Дублировать теги** — при включении общие поля метаданных дублируются в обе секции тегов файла. Это улучшает совместимость со старыми аудиоплеерами, но в большинстве случаев не требуется.

### Параметры обновления метаданных облачных файлов

Можно управлять тем, как приложение обновляет метаданные аудиофайлов в облачных сервисах:

- **Показывать сообщение с подтверждением**  
  Спрашивать перед применением изменений метаданных к облачным файлам.

- **Автоматически обновлять метаданные файла**  
  Сохранять изменения метаданных в облачном файле автоматически после редактирования.

- **Не обновлять метаданные файла**  
  Пропускать обновление облачных файлов — изменения будут применяться только локально.

### Редактирование онлайн-файлов

Выберите, что происходит с локально скачанными копиями облачных файлов после редактирования:

- **Всегда удалять скачанный файл**  
  Удалять локальную копию после сохранения изменений.

- **Не удалять скачанный файл**  
  Сохранять скачанный файл на устройстве после редактирования.

### Кнопки главного экрана

Настройте, какие действия отображаются на главном экране редактора тегов (Автоматический поиск тегов, Ручной поиск тегов, Поиск обложки альбома, Сохранить обложку альбома, Нормализовать кодировку, Удалить аудиотеги). Можно также переключить **Показать расширенные теги** и **Редактировать файлы одновременно**, чтобы редактор всегда открывался в предпочтительном режиме.

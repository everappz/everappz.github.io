**태그 편집기**는 Evertag 앱의 메인 화면으로 오디오 파일 메타데이터를 보고 편집할 수 있습니다. **로컬 파일** 섹션에서 파일을 탭하거나 연결된 **클라우드 스토리지** 계정에서 파일을 선택하여 이 화면을 여세요.

## 편집 모드

Evertag는 두 가지 편집 모드를 제공합니다:

- **단일 파일 모드**  
  - 아트워크 캐러셀에서 왼쪽 또는 오른쪽으로 스와이프하여 파일 간에 이동합니다.  

- **일괄 모드**  
  - 여러 파일을 한 번에 편집하고 공유 메타데이터를 적용합니다.  
  - 활성화하려면 아래로 스크롤하여 **파일 동시 편집**을 탭하세요.

## 단일 파일 모드

기본적으로 앱은 주요 편집 옵션만 활성화된 단일 파일 모드에서 태그 편집기를 엽니다. 이 모드에서는 다음과 같은 가장 일반적인 메타데이터 필드를 편집할 수 있습니다:

**트랙 제목, 부제, 앨범 아티스트, 앨범, 아티스트, 작곡가, 연주자, 장르, 댓글, 분당 비트 수, 팟캐스트, 편집집, 디스크 번호, 트랙 번호, 트랙 총수, 평점, 연도**

모든 사용 가능한 태그에 액세스하려면 화면 하단으로 스크롤하여 **확장 태그 표시** 옵션을 탭하세요. 그러면 편집기가 확장 모드로 전환되어 **MusicBrainz 태그**, **가사**, **보호자 등급**, 재생 게인 값, 정렬 순서, 팟캐스트 메타데이터 등을 포함한 **120개 이상의 메타데이터 필드**를 편집할 수 있습니다. **설정 → 오디오 태그 편집기 → 메인 화면 버튼**을 사용하여 확장 태그 표시를 항상 켜두도록 영구적으로 설정하세요.

## 일괄 모드

두 가지 방법으로 일괄 편집을 시작할 수 있습니다:

1. **파일 관리자에서**  
   - 오른쪽 상단에서 **추가 작업** (•••)을 탭하세요.  
   - **선택하다**를 탭하고 여러 파일을 선택한 다음 **오디오 태그 편집**을 탭하세요.

2. **태그 편집기에서**  
   - 파일을 열고 아래로 스크롤한 다음 **파일 동시 편집**을 탭하여 같은 폴더의 모든 파일을 로드하세요.

## 가사 편집

확장 편집기는 **가사** 필드를 노출합니다. 탭하여 가사 목록을 열면 — 각 가사 항목에는 자체 언어와 설명이 있으므로 단일 트랙이 여러 번역을 저장할 수 있습니다.

가사를 처음부터 입력할 필요가 없습니다. 편집기에는 현재 트랙의 아티스트와 제목으로 미리 채워진 웹에서 가장 인기 있는 가사 데이터베이스로의 원클릭 검색 단축키가 포함됩니다:

- **Lrclib** — **시간 지정 (LRC 스타일) 가사**를 위한 공개 데이터베이스. 지원하는 플레이어에서 줄 단위로 스크롤되는 동기화된 가사를 원할 때 사용하세요.
- **Genius** — 주석과 정확한 일반 텍스트 가사가 있는 대규모 카탈로그.
- **Lyricsify** — 일반 및 타임스탬프 가사가 있는 커뮤니티 기반 데이터베이스.
- **Google** — 전용 데이터베이스에 일치 항목이 없을 때 대안으로 사용하는 일반 웹 검색.

각 단축키는 해당 서비스가 기기에서 연결 가능할 때만 나타납니다. 서비스를 탭하고 원하는 가사(또는 LRC 타임스탬프)를 복사하고 Evertag로 돌아와서 텍스트 필드에 붙여 넣으면 — **저장하다**를 탭하여 오디오 파일 태그에 가사를 씁니다.

## 평점 및 보호자 등급 설정

확장 편집기는 별 **평점** 컨트롤과 **보호자 등급** 세그먼트 컨트롤을 제공합니다.

### 별 평점

**평점** 필드를 사용하여 트랙에 1~5개의 별로 개인 점수를 부여하세요. 값은 파일의 표준 평점 태그 (ID3의 경우 POPM, MP4의 경우 `rate`, Vorbis/APE의 경우 `RATING` 등)에 기록되므로, 이 태그를 읽는 다른 앱 — Music 앱, Plex, Roon 및 대부분의 데스크톱 태그 편집기 포함 — 이 즉시 점수를 인식합니다.

{{< /cards >}}

### 보호자 등급

**보호자 등급**은 iTunes 스토어와 대부분의 음악 플랫폼이 사용하는 부모 자문 표준의 값 중 하나를 사용하여 트랙의 콘텐츠를 표시합니다:

- **무해** — 부모 자문 정보가 없는 트랙의 기본값. 파일은 모든 청취자에게 적합한 것으로 취급되며 태그를 존중하는 플레이어에서 자문 레이블을 표시하지 않습니다.
- **선정적** — 트랙에 선정적인 언어나 콘텐츠가 포함되어 있습니다. 부모 통제를 존중하는 플레이어(Music 앱, Apple TV 앱, Spotify 내보내기, Plex 등)는 제목 옆에 **E** 배지를 표시하고 기기나 계정에서 제한이 활성화된 경우 아이들 프로필에서 트랙을 숨기거나 재생을 거부할 수 있습니다.
- **클린** — 명시적인 트랙의 검열되거나 편집된 버전. 플레이어는 청취자가 클린 편집을 원래 명시적 마스터와 구별할 수 있도록 **C** 배지를 표시합니다. 이는 동일한 라이브러리에 두 버전이 함께 있을 때 유용합니다.

다음 경우에 이 필드를 설정하거나 수정하려고 할 것입니다:

- 파일에 잘못된 레이블이 있는 경우 (예: 명시적으로 잘못 태그된 클린 라디오 편집본, 또는 그 반대).
- 트랙이 태그 없이 리핑되거나 다운로드되어 명시적인 콘텐츠가 포함되어 있음에도 무해로 표시되는 경우 — 부모 통제 및 가족 공유 라이브러리가 올바르게 작동하려면 필요합니다.
- 앨범을 제출하거나 공유하기 위해 준비하고 모든 트랙에 일관된 메타데이터가 필요한 경우.
- CarPlay, 잠금 화면, Apple Music 스타일 플레이어 또는 DJ 소프트웨어가 트랙 제목 옆에 올바른 **E** / **C** 배지를 표시하도록 하려는 경우.

값은 파일 형식의 표준 보호자 등급 필드에 저장됩니다 (MP4의 경우 `rtng`, ID3의 경우 `TXXX:ITUNESADVISORY`, Vorbis의 경우 `ITUNESADVISORY`), 따라서 부모 자문 메타데이터를 읽는 모든 플레이어가 업데이트를 볼 것입니다.

## 앨범 커버 편집

앨범 커버를 변경하려면:

1. 아트워크 캐러셀에서 **카메라 아이콘**을 탭하세요.  
2. 이미지 위치를 선택하세요: 로컬 파일, 클라우드, 다운로드 또는 사진 라이브러리.  
3. 커버 아트로 적용할 이미지를 선택하세요.

## 태그 편집기 설정

**설정 → 오디오 태그 편집기**로 이동하여 동작을 사용자 지정하세요.

### 앨범 커버 크기 조정

오디오 파일에 저장할 때 앨범 커버의 크기를 어떻게 조정할지 선택하세요. 크기 조정을 비활성화하여 원본 이미지 크기를 유지할 수 있지만 일부 플레이어가 큰 아트워크 파일을 지원하지 않을 수 있다는 점을 참고하세요. "원본 크기" 옵션은 프리미엄 개인화 기능의 일부입니다.

### 태그 저장 옵션

- **ID3v2.4** — 활성화되면 앱이 가능한 경우 ID3v2.4 형식으로 태그를 저장합니다. 이전 플레이어나 기기에서 오디오 태그가 올바르게 표시되지 않는 경우 더 널리 지원되는 ID3v2.3으로 대체하려면 비활성화하세요.
- **중복 태그** — 활성화되면 파일의 두 태그 섹션 모두에 공통 메타데이터 필드가 복제됩니다. 이는 이전 오디오 플레이어와의 호환성을 개선하지만 대부분의 경우 필요하지 않습니다.

### 클라우드 파일 메타데이터 업데이트 옵션

클라우드 서비스에 저장된 오디오 파일의 메타데이터를 앱이 업데이트하는 방법을 제어할 수 있습니다:

- **확인 메시지 표시**  
  클라우드 파일에 메타데이터 변경 사항을 적용하기 전에 묻습니다.

- **파일의 메타데이터 자동 업데이트**  
  편집 후 클라우드 파일에 메타데이터 변경 사항을 자동으로 저장합니다.

- **파일의 메타데이터를 업데이트하지 않음**  
  클라우드 파일 업데이트 건너뛰기 — 변경 사항은 로컬로만 적용됩니다.

### 온라인 파일 편집

편집 후 클라우드 파일의 로컬 다운로드 복사본에 어떤 일이 발생하는지 선택하세요:

- **항상 다운로드된 파일 삭제**  
  변경 사항을 저장한 후 로컬 복사본을 제거합니다.

- **다운로드된 파일 삭제 안 함**  
  편집 후 기기에 다운로드된 파일을 유지합니다.

### 메인 화면 버튼

태그 편집기의 메인 화면에 표시되는 작업을 사용자 지정하세요 (오디오 태그 자동 검색, 오디오 태그 수동 검색, 앨범 아트워크 검색, 앨범 아트워크 저장, 인코딩 정규화, 오디오 태그 삭제). **확장 태그 표시** 및 **파일 동시 편집**을 설정하여 편집기가 항상 원하는 모드로 열리도록 할 수도 있습니다.

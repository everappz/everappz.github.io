La section Fichiers locaux sert de hub pour gérer les fichiers situés dans le dossier «Documents» de l'application, ainsi que les fichiers ajoutés depuis votre appareil, disponibles hors ligne.

Ce gestionnaire de fichiers intégré vous permet de modifier des fichiers et offre diverses méthodes pour importer des fichiers audio dans l'application. Les fichiers que vous avez récemment ouverts apparaissent automatiquement dans l'onglet **Récents** et les éléments que vous marquez d'une étoile apparaissent sous **Favoris**, vous permettant de passer directement aux fichiers avec lesquels vous travaillez le plus sans revenir à cet écran.

## Télécharger des fichiers depuis le stockage cloud

Vous pouvez ajouter des fichiers audio à l'application en les téléchargeant depuis des services de stockage cloud connectés comme décrit [ici](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag).

## Importer des fichiers depuis votre appareil

Importez facilement des fichiers depuis votre appareil comme décrit [ici](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac), ou depuis un lecteur USB connecté comme décrit [ici](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).

## iTunes File Sharing

Transférez des fichiers via une connexion câblée comme décrit [ici](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder).

## Wi-Fi Drive

Transférez des fichiers sans fil comme décrit [ici](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive).

## File de transfert

Dans le coin supérieur gauche de la barre de navigation, vous trouverez un bouton «Transferts». Appuyez dessus pour accéder à la file de transfert, où vous pouvez surveiller et gérer tous vos téléchargements et téléversements. De plus, vous avez la flexibilité d'ajuster la vitesse de la file de transfert et le type de réseau dans les paramètres de l'application.

## Barre d'outils supérieure

La barre d'outils supérieure, située sous la barre de navigation, offre plusieurs actions :

- **Rechercher :** Effectuez une recherche dans le dossier actuel.

Vous pouvez afficher ou masquer la barre d'outils supérieure à l'aide d'un geste de balayage vers le bas.

## Dossiers spéciaux

Dans l'écran «Fichiers locaux», vous rencontrerez plusieurs dossiers spéciaux.

## Téléchargements

Tous les fichiers téléchargés depuis un service de stockage cloud apparaîtront ici par défaut. Vous pouvez personnaliser ce comportement dans les paramètres de l'application.

## iCloud

Placez des fichiers dans ce dossier et ils se synchroniseront sur tous vos appareils connectés au même compte iCloud.

## Deux sections distinctes

L'écran «Fichiers locaux» est organisé en deux sections distinctes.

## Fichiers dans cette application

Affiche les fichiers et dossiers stockés dans le répertoire Documents de l'application et iCloud Drive.

## Fichiers sur cet iPhone/iPad/Mac

Affiche les fichiers situés sur votre appareil mais dans différentes applications. Vous pouvez les importer dans cette application à l'aide du sélecteur de fichiers système. Pour activer le sélecteur, choisissez «Ouvrir des fichiers...» pour sélectionner des fichiers ou «Ouvrir des dossiers...» pour sélectionner des dossiers. Des instructions détaillées sur la façon d'importer de la musique locale stockée sur votre iPhone ou Mac sont disponibles [ici](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac). Vous pouvez également connecter un dossier situé sur votre appareil et avoir un accès rapide au contenu du dossier. Utilisez l'élément de menu «Connecter un dossier» et choisissez un dossier situé sur votre appareil. Appuyez sur «Terminer» et l'application créera un lien vers ce dossier avec un accès en lecture/écriture, vous permettant de gérer les fichiers directement depuis cette application. Pour déconnecter un dossier situé sur votre appareil, appuyez sur le bouton «Plus d'actions» et choisissez «Déconnecter».

## Importer des fichiers depuis des clés USB connectées

Des instructions détaillées sur la façon de connecter une clé USB à votre iPhone et de gérer les fichiers qui s'y trouvent sont disponibles [ici](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).

## Plus d'actions

Le menu Plus d'actions pour le dossier actuellement ouvert est situé dans le coin supérieur droit et donne accès à diverses actions.

- **Sélectionner :** Passer en mode de sélection pour les fichiers et dossiers.
- **Nouveau dossier :** Créer un nouveau dossier dans le dossier actuel.
- **Rechercher :** Activer la fonction de recherche pour trouver un fichier ou dossier spécifique.
- **Trier :** Organiser les fichiers par nom, taille, date ou en utilisant des informations de métadonnées.
- **Liste/Grille :** Modifier la présentation du contenu d'une vue tableau à une vue grille.
- **Aide :** Accéder aux instructions d'aide.

## Mode de sélection

Si vous devez modifier plusieurs fichiers, activez le mode de sélection en appuyant sur le bouton «Plus d'actions» «...» dans la barre de navigation dans le coin supérieur droit, puis choisissez l'élément de menu «Sélectionner». Cela affichera des cases à cocher près de chaque fichier. Sélectionnez les fichiers souhaités en appuyant sur leurs cases à cocher. Vous pouvez effectuer les actions suivantes sur les fichiers sélectionnés :

- **Modifier les balises audio :** Modifiez les tags audio pour tous les fichiers sélectionnés simultanément (édition par lot).
- **Copier :** Créez une copie du fichier ou dossier sélectionné.
- **Téléverser :** Téléversez le fichier ou dossier sélectionné vers un stockage distant.
- **Déplacer :** Déplacez le fichier ou dossier sélectionné vers un autre emplacement sur l'appareil.
- **Renommer :** Modifiez le nom du fichier ou dossier sélectionné.
- **Supprimer :** Supprimez le fichier ou dossier sélectionné de l'appareil. Veuillez noter que cette action est irréversible.

## Menu des options

Pour chaque fichier ou dossier dans l'application, plusieurs actions sont disponibles, accessibles en appuyant sur le bouton «...». Si vous ne voyez pas toutes les actions, faites défiler vers le bas pour les afficher :

- **Ajouter aux favoris :** Marquez les fichiers ou dossiers sélectionnés comme favoris.
- **Modifier les balises audio :** Ouvrez l'éditeur de tags intégré pour modifier les tags audio du fichier sélectionné.
- **Renommer :** Modifiez le nom du fichier ou dossier sélectionné.
- **Déplacer :** Déplacez le fichier ou dossier sélectionné vers un autre emplacement sur l'appareil.
- **Copier :** Créez une copie du fichier ou dossier sélectionné.
- **Téléverser :** Téléversez le fichier ou dossier sélectionné vers un stockage distant.
- **Afficher dans le Finder :** Pour les fichiers importés depuis votre Mac, cette action révèle le répertoire dans le Finder contenant le fichier.
- **Ouvrir dans :** Exportez le fichier vers une autre application.
- **Supprimer :** Supprimez le fichier ou dossier de l'appareil. **Veuillez noter que cette action est irréversible** et vous ne pouvez pas restaurer les fichiers supprimés.

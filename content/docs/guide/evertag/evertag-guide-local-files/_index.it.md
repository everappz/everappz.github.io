La sezione File locali funge da hub per gestire i file situati nella cartella "Documents" dell'applicazione, nonché i file aggiunti dal dispositivo disponibili offline.

Questo gestore file integrato consente di modificare i file e offre vari metodi per importare file audio nell'app. I file aperti di recente appaiono automaticamente nella scheda **Recenti** e gli elementi contrassegnati con una stella compaiono sotto **Preferiti**, così puoi accedere direttamente ai file con cui lavori di più senza dover tornare a questa schermata.

## Coda di trasferimento

Nell'angolo in alto a sinistra della barra di navigazione troverai il pulsante 'Trasferimenti'. Toccalo per accedere alla coda di trasferimento, dove puoi monitorare e gestire tutti i download e upload. Inoltre, hai la flessibilità di regolare la velocità della coda di trasferimento e il tipo di rete nelle impostazioni dell'app.

## Barra degli strumenti superiore

La barra degli strumenti superiore, posizionata sotto la barra di navigazione, offre diverse azioni:

- **Cerca:** esegui una ricerca all'interno della cartella corrente.  

Puoi mostrare o nascondere la barra degli strumenti superiore usando un gesto di scorrimento verso il basso.

## Cartelle speciali

Nella schermata "File locali", incontrerai diverse cartelle speciali.

## Scaricare

Tutti i file scaricati da un servizio di archiviazione cloud appariranno qui per impostazione predefinita. Puoi personalizzare questo comportamento nelle impostazioni dell'applicazione.

## iCloud

Inserisci i file in questa cartella e verranno sincronizzati su tutti i tuoi dispositivi connessi allo stesso account iCloud.

## Due sezioni distinte

La schermata "File locali" è organizzata in due sezioni distinte.

## File in questa applicazione

Mostra i file e le cartelle archiviati nella directory Documents dell'app e su iCloud Drive.

## File su questo iPhone/iPad/Mac

Mostra i file situati sul dispositivo ma in applicazioni diverse. Puoi importarli in questa applicazione usando il selettore di file di sistema. Per attivare il selettore, scegli "Apri file..." per selezionare i file o "Apri cartelle..." per selezionare le cartelle. Istruzioni dettagliate su come importare la musica locale salvata su iPhone o Mac sono disponibili [qui](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac). Puoi anche connettere una cartella sul dispositivo e avere accesso rapido al contenuto della cartella. Usa la voce di menu "Connetti una cartella" e scegli una cartella sul dispositivo. Tocca "Fatto," e l'app creerà un collegamento a quella cartella con accesso in lettura/scrittura, permettendoti di gestire i file direttamente da questa app. Per disconnettere una cartella sul tuo dispositivo, tocca il pulsante "Altre azioni" e scegli "Disconnettere."

## Altre azioni

Il menu Altre azioni per la cartella attualmente aperta si trova nell'angolo in alto a destra e fornisce accesso a varie azioni.

- **Selezionare:** passa alla modalità di selezione per file e cartelle.  
- **Nuova cartella:** crea una nuova cartella all'interno della cartella corrente.  
- **Cerca:** attiva la funzione di ricerca per trovare un file o una cartella specifica.  
- **Ordina:** organizza i file per nome, dimensione, data o usando informazioni sui metadati.  
- **Elenco/Griglia:** cambia la presentazione dei contenuti da una tabella a una vista griglia.  
- **Guida:** accedi alle istruzioni della guida.

## Modalità di selezione

Se devi modificare più file, attiva la modalità di selezione toccando il pulsante Altre azioni "..." nella barra di navigazione nell'angolo in alto a destra e poi scegliendo la voce di menu "Selezionare". Verranno visualizzate le caselle di controllo vicino a ogni file. Seleziona i file desiderati toccando le relative caselle di controllo. Puoi eseguire le seguenti azioni sui file selezionati:

- **Modificare i tag audio:** modifica i tag audio per tutti i file selezionati contemporaneamente (modifica batch).  
- **Copia:** crea una copia del file o della cartella selezionata.  
- **Carica:** carica il file o la cartella selezionata nell'archiviazione remota.  
- **Sposta:** sposta il file o la cartella selezionata in un'altra posizione sul dispositivo.  
- **Rinomina:** cambia il nome del file o della cartella selezionata.  
- **Eliminare:** rimuovi il file o la cartella selezionata dal dispositivo. Si noti che questa azione è irreversibile.

## Menu opzioni

Per ogni file o cartella nell'app, sono disponibili diverse azioni accessibili toccando il pulsante Altro "...". Se non vedi tutte le azioni, scorri verso il basso per visualizzarle:

- **Aggiungi ai preferiti:** contrassegna i file o le cartelle selezionati come preferiti.  
- **Modificare i tag audio:** apri l'editor di tag integrato per modificare i tag audio del file selezionato.  
- **Rinomina:** modifica il nome del file o della cartella selezionata.  
- **Sposta:** sposta il file o la cartella selezionata in un'altra posizione sul dispositivo.  
- **Copia:** crea una copia del file o della cartella selezionata.  
- **Carica:** carica il file o la cartella selezionata nell'archiviazione remota.  
- **Mostra nel Finder:** per i file importati dal Mac, questa azione mostra la directory nel Finder che contiene il file.  
- **Apri in:** esporta il file in un'altra app.  
- **Eliminare:** elimina il file o la cartella dal dispositivo. **Si noti che questa azione è irreversibile** e non è possibile ripristinare i file eliminati.

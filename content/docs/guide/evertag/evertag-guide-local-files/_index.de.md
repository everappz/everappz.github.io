Der Abschnitt „Lokale Dateien" dient als Zentrale für die Verwaltung von Dateien im Ordner „Documents" der Anwendung sowie von Dateien, die von Ihrem Gerät hinzugefügt wurden und offline verfügbar sind.

Dieser integrierte Dateimanager ermöglicht Ihnen die Bearbeitung von Dateien und bietet verschiedene Methoden zum Importieren von Audiodateien in die App. Dateien, die Sie zuletzt geöffnet haben, erscheinen automatisch auf der Registerkarte **Aktuell** und Elemente, die Sie mit einem Stern markieren, werden unter **Favoriten** angezeigt, sodass Sie direkt zu den Dateien springen können, mit denen Sie am häufigsten arbeiten, ohne zu diesem Bildschirm zurücknavigieren zu müssen.

## Dateien aus Cloud-Speicher herunterladen

Sie können Audiodateien zur App hinzufügen, indem Sie sie aus verbundenen Cloud-Speicherdiensten herunterladen, wie [hier](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag) beschrieben.

## Dateien von Ihrem Gerät importieren

Importieren Sie Dateien ganz einfach von Ihrem Gerät, wie [hier](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac) beschrieben, oder von einem angeschlossenen USB-Laufwerk, wie [hier](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it) beschrieben.

## iTunes File Sharing

Übertragen Sie Dateien über eine Kabelverbindung, wie [hier](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder) beschrieben.

## Wi-Fi Drive

Übertragen Sie Dateien drahtlos, wie [hier](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive) beschrieben.

## Übertragungswarteschlange

In der oberen linken Ecke der Navigationsleiste finden Sie eine Schaltfläche „Übertragungen". Tippen Sie darauf, um auf die Übertragungswarteschlange zuzugreifen, in der Sie alle Ihre Downloads und Uploads überwachen und verwalten können. Darüber hinaus haben Sie die Möglichkeit, die Geschwindigkeit der Übertragungswarteschlange und den Netzwerktyp in den App-Einstellungen anzupassen.

## Obere Symbolleiste

Die obere Symbolleiste, die sich unter der Navigationsleiste befindet, bietet mehrere Aktionen:

- **Suche:** Führen Sie eine Suche im aktuellen Ordner durch.

Sie können die obere Symbolleiste mit einer Wischgeste nach unten ein- oder ausblenden.

## Spezielle Ordner

Im Bildschirm „Lokale Dateien" finden Sie mehrere spezielle Ordner.

## Downloads

Alle Dateien, die aus einem Cloud-Speicherdienst heruntergeladen wurden, erscheinen hier standardmäßig. Sie können dieses Verhalten in den Anwendungseinstellungen anpassen.

## iCloud

Legen Sie Dateien in diesem Ordner ab, und sie werden auf allen Ihren Geräten synchronisiert, die mit demselben iCloud-Konto verbunden sind.

## Zwei Separate Abschnitte

Der Bildschirm „Lokale Dateien" ist in zwei separate Abschnitte unterteilt.

## Dateien in dieser Anwendung

Zeigt Dateien und Ordner an, die im Verzeichnis „Documents" der App und in iCloud Drive gespeichert sind.

## Dateien auf diesem iPhone/iPad/Mac

Zeigt Dateien an, die sich auf Ihrem Gerät befinden, aber in verschiedenen Anwendungen. Sie können sie mit der Systemdateiauswahl in diese Anwendung importieren. Um die Auswahl zu aktivieren, wählen Sie „Dateien öffnen..." um Dateien auszuwählen, oder „Ordner öffnen..." um Ordner auszuwählen. Detaillierte Anweisungen zum Importieren lokaler Musik, die auf Ihrem iPhone oder Mac gespeichert ist, sind [hier](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac) verfügbar. Sie können auch einen auf Ihrem Gerät befindlichen Ordner verbinden und schnell auf den Inhalt des Ordners zugreifen. Verwenden Sie den Menüpunkt „Ordner verbinden" und wählen Sie einen auf Ihrem Gerät befindlichen Ordner aus. Tippen Sie auf „Fertig," und die App erstellt einen Link zu diesem Ordner mit Lese-/Schreibzugriff, sodass Sie Dateien direkt aus dieser App verwalten können. Um einen auf Ihrem Gerät befindlichen Ordner zu trennen, tippen Sie auf die Schaltfläche „Weitere Aktionen" und wählen Sie „Trennen."

- **Auswählen:** Wechseln Sie in den Auswahlmodus für Dateien und Ordner.
- **Neuer Ordner:** Erstellen Sie einen neuen Ordner im aktuellen Ordner.
- **Suche:** Aktivieren Sie die Suchfunktion, um eine bestimmte Datei oder einen bestimmten Ordner zu finden.
- **Sortieren:** Organisieren Sie Dateien nach Name, Größe, Datum oder mithilfe von Metadateninformationen.
- **Liste/Raster:** Ändern Sie die Inhaltsdarstellung von einer Tabelle in eine Rasteransicht.
- **Hilfe:** Greifen Sie auf Hilfeanweisungen zu.

## Auswahlmodus

Wenn Sie mehrere Dateien bearbeiten müssen, aktivieren Sie den Auswahlmodus, indem Sie auf die Schaltfläche „Weitere Aktionen" „..." in der Navigationsleiste in der oberen rechten Ecke tippen und dann den Menüpunkt „Auswählen" auswählen. Dadurch werden Kontrollkästchen neben jeder Datei angezeigt. Wählen Sie die gewünschten Dateien aus, indem Sie auf deren Kontrollkästchen tippen. Sie können folgende Aktionen mit den ausgewählten Dateien durchführen:

- **Audio-Tags bearbeiten:** Bearbeiten Sie Audio-Tags für alle ausgewählten Dateien gleichzeitig (Stapelbearbeitung).
- **Kopieren:** Erstellen Sie eine Kopie der ausgewählten Datei oder des ausgewählten Ordners.
- **Hochladen:** Laden Sie die ausgewählte Datei oder den ausgewählten Ordner in den Remote-Speicher hoch.
- **Verschieben:** Verschieben Sie die ausgewählte Datei oder den ausgewählten Ordner an einen anderen Speicherort auf dem Gerät.
- **Umbenennen:** Ändern Sie den Namen der ausgewählten Datei oder des ausgewählten Ordners.
- **Löschen:** Entfernen Sie die ausgewählte Datei oder den ausgewählten Ordner vom Gerät. Bitte beachten Sie, dass diese Aktion nicht rückgängig gemacht werden kann.

## Optionenmenü

Für jede Datei oder jeden Ordner in der App stehen mehrere Aktionen zur Verfügung, die durch Tippen auf die Schaltfläche „Weitere Aktionen" „..." zugänglich sind. Wenn Sie nicht alle Aktionen sehen, scrollen Sie nach unten:

- **Zu Favoriten hinzufügen:** Markieren Sie ausgewählte Dateien oder Ordner als Favoriten.
- **Audio-Tags bearbeiten:** Öffnen Sie den integrierten Tag-Editor, um Audio-Tags für die ausgewählte Datei zu ändern.
- **Umbenennen:** Ändern Sie den Namen der ausgewählten Datei oder des ausgewählten Ordners.
- **Verschieben:** Verschieben Sie die ausgewählte Datei oder den ausgewählten Ordner an einen anderen Speicherort auf dem Gerät.
- **Kopieren:** Erstellen Sie eine Kopie der ausgewählten Datei oder des ausgewählten Ordners.
- **Hochladen:** Laden Sie die ausgewählte Datei oder den ausgewählten Ordner in den Remote-Speicher hoch.
- **Im Finder anzeigen:** Für Dateien, die von Ihrem Mac importiert wurden, zeigt diese Aktion das Verzeichnis im Finder an, das die Datei enthält.
- **Öffnen in:** Exportieren Sie die Datei in eine andere App.
- **Löschen:** Löschen Sie die Datei oder den Ordner vom Gerät. **Bitte beachten Sie, dass diese Aktion nicht rückgängig gemacht werden kann** und Sie gelöschte Dateien nicht wiederherstellen können.

Bagian File Lokal berfungsi sebagai hub untuk mengelola file yang terletak di folder "Documents" aplikasi, serta file yang ditambahkan dari perangkat Anda yang tersedia secara offline.

Pengelola file bawaan ini memungkinkan Anda mengedit file dan menawarkan berbagai metode untuk mengimpor file audio ke dalam aplikasi. File yang baru saja Anda buka secara otomatis muncul di tab **Terbaru** dan item yang Anda tandai dengan bintang muncul di bawah **Favorit**, sehingga Anda dapat langsung menuju file yang paling sering Anda gunakan tanpa harus kembali ke layar ini.

## Antrian Transfer

Di sudut kiri atas bilah navigasi, Anda akan menemukan tombol 'Transfer'. Ketuk untuk mengakses antrian transfer, di mana Anda dapat memantau dan mengelola semua unduhan dan unggahan Anda. Selain itu, Anda memiliki fleksibilitas untuk menyesuaikan kecepatan antrian transfer dan jenis jaringan di pengaturan aplikasi.

## Toolbar Atas

Toolbar atas yang terletak di bawah bilah navigasi menawarkan beberapa tindakan:

- **Cari:** Lakukan pencarian di dalam folder saat ini.  

Anda dapat menampilkan atau menyembunyikan toolbar atas menggunakan gerakan geser ke bawah.

## Folder Khusus

Di dalam layar "File Lokal", Anda akan menemukan beberapa folder khusus.

## Unduh

Semua file yang diunduh dari layanan penyimpanan cloud akan muncul di sini secara default. Anda dapat menyesuaikan perilaku ini di pengaturan aplikasi.

## iCloud

Tempatkan file di folder ini, dan file tersebut akan disinkronkan di semua perangkat Anda yang terhubung ke akun iCloud yang sama.

## Dua Bagian yang Berbeda

Layar "File Lokal" diorganisasikan menjadi dua bagian yang berbeda.

## File di Aplikasi Ini

Menampilkan file dan folder yang tersimpan di direktori Documents aplikasi dan iCloud Drive.

## File di iPhone/iPad/Mac Ini

Menampilkan file yang terletak di perangkat Anda tetapi di aplikasi yang berbeda. Anda dapat mengimpornya ke dalam aplikasi ini menggunakan pemilih file sistem. Untuk mengaktifkan pemilih, pilih "Buka file..." untuk memilih file atau "Buka folder..." untuk memilih folder. Instruksi terperinci tentang cara mengimpor musik lokal yang tersimpan di iPhone atau Mac Anda tersedia [di sini](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac). Anda juga dapat menghubungkan folder yang terletak di perangkat Anda dan memiliki akses cepat ke konten folder tersebut. Gunakan item menu "Hubungkan folder" dan pilih folder yang terletak di perangkat Anda. Ketuk "Selesai," dan aplikasi akan membuat tautan ke folder tersebut dengan akses baca/tulis, memungkinkan Anda mengelola file langsung dari aplikasi ini. Untuk memutus koneksi folder yang terletak di perangkat Anda, ketuk tombol "Lebih banyak tindakan" dan pilih "Putuskan Koneksi."

## Lebih Banyak Tindakan

Menu Lebih Banyak Tindakan untuk folder yang sedang dibuka terletak di sudut kanan atas dan menyediakan akses ke berbagai tindakan.

- **Pilih:** Beralih ke mode pemilihan untuk file dan folder.  
- **Folder Baru:** Buat folder baru di dalam folder saat ini.  
- **Cari:** Aktifkan fungsi pencarian untuk menemukan file atau folder tertentu.  
- **Urutkan:** Atur file berdasarkan nama, ukuran, tanggal, atau menggunakan informasi metadata.  
- **Daftar/Kisi:** Ubah tampilan konten dari tabel ke tampilan kisi.  
- **Bantuan:** Akses instruksi bantuan.

## Mode Pemilihan

Jika Anda perlu mengedit beberapa file, aktifkan mode pemilihan dengan mengetuk tombol Lebih Banyak Tindakan "..." di bilah navigasi di sudut kanan atas dan kemudian memilih item menu "Pilih". Ini akan menampilkan kotak centang di dekat setiap file. Pilih file yang diinginkan dengan mengetuk kotak centang mereka. Anda dapat melakukan tindakan berikut pada file yang dipilih:

- **Mengedit tag audio:** Edit tag audio untuk semua file yang dipilih secara bersamaan (pengeditan batch).  
- **Salin:** Buat salinan file atau folder yang dipilih.  
- **Unggah:** Unggah file atau folder yang dipilih ke penyimpanan jarak jauh.  
- **Pindahkan:** Pindahkan file atau folder yang dipilih ke lokasi lain di perangkat.  
- **Ganti Nama:** Ubah nama file atau folder yang dipilih.  
- **Hapus:** Hapus file atau folder yang dipilih dari perangkat. Perlu diperhatikan bahwa tindakan ini tidak dapat dibatalkan.

## Menu Opsi

Untuk setiap file atau folder di aplikasi, beberapa tindakan tersedia, dapat diakses dengan mengetuk tombol Lebih "...". Jika Anda tidak melihat semua tindakan, gulir ke bawah untuk melihatnya:

- **Tambahkan ke Favorit:** Tandai file atau folder yang dipilih sebagai favorit.  
- **Mengedit tag audio:** Buka editor tag bawaan untuk mengubah tag audio file yang dipilih.  
- **Ganti Nama:** Ubah nama file atau folder yang dipilih.  
- **Pindahkan:** Pindahkan file atau folder yang dipilih ke lokasi lain di perangkat.  
- **Salin:** Buat salinan file atau folder yang dipilih.  
- **Unggah:** Unggah file atau folder yang dipilih ke penyimpanan jarak jauh.  
- **Tampilkan di Finder:** Untuk file yang diimpor dari Mac Anda, tindakan ini menampilkan direktori di Finder yang berisi file tersebut.  
- **Buka di:** Ekspor file ke aplikasi lain.  
- **Hapus:** Hapus file atau folder dari perangkat. **Perlu diperhatikan bahwa tindakan ini tidak dapat dibatalkan** dan Anda tidak dapat memulihkan file yang dihapus.

Nella sezione Playlist troverai strumenti utili per gestire le tue raccolte musicali. Quest'area mostra tutte le tue playlist in un unico posto. In alto c'è un pulsante **"..."** nella barra di navigazione che apre un menu con diverse opzioni per le playlist, più una barra degli strumenti con azioni rapide come Cerca, Riproduci Tutto e Mescola Tutto. Ogni playlist ha anche il suo pulsante **"..."** accanto al titolo, che offre più opzioni specifiche per quella playlist.

Le playlist in Flacbox possono contenere un mix di tracce cloud online, file scaricati offline e file locali dal tuo dispositivo — tutto in una playlist — e vengono riprodotte insieme senza interruzioni.

## Creare una Playlist

Creare una nuova playlist è semplice. Hai due opzioni: tocca il pulsante **+**, oppure tocca il pulsante **"..."** nell'angolo in alto a destra della barra di navigazione e scegli Nuova Playlist. Dai alla tua playlist un nome significativo, poi tocca Salva.

Questo apre il dialogo Aggiungi Brani, dove puoi selezionare manualmente le tracce da includere nella nuova playlist. Le tracce sono convenientemente categorizzate per tipo di sorgente:

- **Libreria** — tracce già presenti nella tua libreria musicale.
- **File in Questa Applicazione** — file audio nella cartella Documenti dell'app (scaricati da cloud storage, importati tramite Wi-Fi Drive, o aggiunti tramite Finder File Sharing).
- **File su Questo Dispositivo** — file audio situati altrove sul tuo dispositivo, non in questa applicazione.
- **Connessioni** — file online situati nei servizi di cloud storage connessi.

Per impostazione predefinita, puoi aggiungere un singolo brano a una playlist solo una volta. Se vuoi consentire i duplicati, abilita questa funzione in Impostazioni → Libreria Musicale → Playlist → Duplicati in una Playlist → Abilita.

## Importare una Playlist

In Flacbox abbiamo aggiunto l'importazione di file M3U / M3U8 / CUE in modo che non sia necessario ricreare le playlist manualmente dopo il passaggio da un altro lettore.

Prima, vai alla sezione Playlist. Poi tocca il pulsante Altro nell'angolo in alto a destra. Dal menu, seleziona Importa Playlist.

Nella schermata successiva, scegli la posizione del file. Le opzioni supportate includono:

- Cloud storage connesso
- File nell'applicazione
- File sul tuo dispositivo

Seleziona il cloud storage connesso e apri la cartella contenente il file playlist. Le estensioni dei file playlist supportate includono M3U, M3U8 e CUE. Seleziona il file playlist e tocca Fine per confermare la selezione.

L'app analizza il file playlist, crea un elenco di tracce e individua quei file nello storage per compilare una playlist finale, che viene poi importata nella libreria musicale. Importante: il file M3U / CUE deve contenere i percorsi corretti ai file multimediali, e i file devono effettivamente esistere in quei percorsi nel tuo storage. Leggi di più sull'importazione delle playlist [qui](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox).

## Schermata Dettaglio Playlist

Quando apri una playlist, appare la schermata Dettaglio Playlist. Troverai un pulsante **"..."** nell'angolo in alto a destra con le opzioni della playlist, e tre pulsanti sotto l'artwork: Cerca, Continua Riproduzione, Riproduci Tutto e Mescola Tutto. C'è anche una casella di controllo Modalità Offline per attivare la sincronizzazione offline dell'intera playlist.

- **Continua Riproduzione** — ripristina l'ultima posizione di riproduzione salvata per questa playlist.
- **Cerca** — esegue una ricerca all'interno della playlist corrente.
- **Riproduci Tutto** — aggiunge tutte le tracce della playlist corrente alla coda del lettore.
- **Mescola Tutto** — come **Riproduci Tutto**, ma mescola le tracce prima di aggiungerle alla coda del lettore audio.
- **Modalità Offline** — scarica tutte le tracce di questa playlist nei file locali. I nuovi elementi aggiunti alla playlist vengono scaricati automaticamente.

## Altre Azioni per una Playlist nella Schermata Playlist

Puoi accedere alle azioni per una playlist toccando il pulsante **"..."** vicino al titolo della playlist. Azioni disponibili:

- **Riproduci Tutto** — aggiunge le tracce della playlist a una nuova coda del lettore.
- **Riproduci Successivo** — aggiunge le tracce della playlist in cima alla coda del lettore esistente.
- **Riproduci Dopo** — aggiunge le tracce della playlist in fondo alla coda del lettore esistente.
- **Modifica Immagine** — cambia l'artwork della playlist.
- **Abilita Modalità Offline** — attiva la modalità offline per la playlist. Sia le tracce esistenti che quelle nuove vengono scaricate automaticamente. Leggi di più sulla modalità offline [qui](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).
- **Esporta Lista Brani** — esporta questa playlist in **M3U / M3U8 / CSV / TXT**.
- **Aggiungi all'Archivio** — archivia questa playlist (incluso il file m3u, la copertina e tutte le tracce) in un unico ZIP. Funzione Premium.
- **Aggiungi Brani** — aggiunge altri brani alla playlist corrente.
- **Rinomina** — rinomina la playlist.
- **Elimina Playlist** — elimina la playlist dalla libreria musicale. **Questa azione non può essere annullata.**

## Altre Azioni per una Playlist nella Schermata Dettaglio Playlist

Puoi accedere alle azioni per una playlist toccando il pulsante **"..."** nell'angolo in alto a destra. Azioni disponibili:

- **Seleziona** — attiva la modalità di selezione tracce, utile per eliminare più tracce dalla playlist o cambiare il loro ordine.
- **Riproduci Successivo** — aggiunge le tracce della playlist in cima alla coda del lettore esistente.
- **Riproduci Dopo** — aggiunge le tracce della playlist in fondo alla coda del lettore esistente.
- **Aggiungi Brani** — aggiunge nuovi brani alla playlist.
- **Riordina Brani** — cambia manualmente l'ordine dei brani nella playlist usando il trascinamento.
- **Rinomina** — rinomina la playlist corrente.
- **Modifica Immagine** — cambia l'artwork dell'album per la playlist corrente.
- **Esporta Lista Brani** — esporta questa playlist in **M3U / M3U8 / CSV / TXT**.
- **Aggiungi all'Archivio** — ZIP della playlist corrente incluse tutte le tracce e il file m3u.
- **Ordina** — cambia l'ordine delle tracce nella playlist. Le opzioni di ordinamento includono **Titolo Brano, Numero Brano, Album, Artista, Album Artist, Genere, Compositore, Valutazione, Anno, Beats per Minuto, Durata, Nome File, Data Modifica File, Data Creazione File,** e **Manuale**. L'opzione di ordinamento **Manuale** consente il riordino manuale dei brani tramite trascinamento.
- **Cerca** — cerca un brano specifico all'interno della playlist corrente.
- **Griglia / Lista** — cambia la presentazione del layout dello schermo.
- **Elimina Playlist** — elimina la playlist dalla libreria musicale. Questa azione non elimina le tracce dal tuo storage, ma **non può essere annullata**.

## Modifica dell'Ordine dei Brani in una Playlist

Per cambiare l'ordine dei brani in una playlist, tocca il pulsante **"..."** nell'angolo in alto a destra e seleziona **Seleziona** per entrare in modalità di selezione. Usa il controllo di riordino e i gesti di trascinamento vicino a ogni traccia per spostarle su o giù. Toccare il controllo di riordino sposta la traccia in cima all'elenco. Per uscire dalla modalità di selezione e applicare le modifiche, tocca **Fine**.

Per un flusso di lavoro ancora più semplice su playlist lunghe, scegli Altre Azioni → Riordina Brani per entrare in modalità dedicata di riordino tramite trascinamento.

## Modifica dell'Immagine di Copertina della Playlist

Per cambiare l'immagine di copertina di una playlist, tocca il pulsante **"..."** nell'angolo in alto a destra e seleziona **Modifica Immagine**. Scegli un'immagine dalle fonti disponibili (Foto, File, cloud storage, o un artwork generato da una traccia nella playlist), poi conferma toccando **Fine**.

## Aggiungere Brani a una Playlist

Apri la playlist e tocca il pulsante **"..."** nell'angolo in alto a destra, poi seleziona **Aggiungi Brani** per aprire un dialogo. Scegli le tracce che vuoi aggiungere e conferma toccando **Fine**.

## Eliminare più Brani da una Playlist

Apri la playlist, tocca il pulsante **"..."** nell'angolo in alto a destra, e seleziona **Seleziona** per entrare in modalità di selezione. Scegli le tracce che vuoi eliminare e tocca **Elimina dalla Playlist** in fondo allo schermo. Conferma toccando **Fine**.

## Opzioni Traccia

Ogni traccia in una playlist ha un elenco di azioni, accessibile toccando il pulsante **"..."**. Se non riesci a vedere tutte le azioni, scorri verso il basso per visualizzarle. Puoi eliminare la traccia dalla playlist, scaricarla, modificare i tag audio e altro.

- **Riproduci Successivo** — aggiunge la traccia in cima alla coda del lettore.
- **Riproduci Dopo** — aggiunge la traccia in fondo alla coda del lettore.
- **Aggiungi a una Playlist** — aggiunge la traccia a un'altra playlist.
- **Aggiungi ai Preferiti** — segna la traccia come preferita per un accesso rapido.
- **Scarica** — rende la traccia disponibile offline.
- **Modifica Tag Audio** — apre l'editor di tag integrato per modificare i metadati della traccia.
- **Apri In** — esporta la traccia e la apre in un'altra app.
- **Mostra nella Cartella** — mostra la cartella dove si trova il file audio.
- **Mostra nel Finder** — per i file importati dal tuo Mac, mostra la cartella dove si trova il file audio sul tuo Mac.
- **Elimina dalla Playlist** — elimina la traccia dalla playlist.
- **Elimina dal Servizio Cloud** — elimina la traccia dalla playlist e dal servizio cloud associato. **Questa azione non può essere annullata.**
- **Elimina dalla Libreria Musicale** — elimina la traccia dalla libreria musicale, lasciando il file nello storage intatto.

## Accessibilità

Flacbox è completamente accessibile con la tecnologia **VoiceOver**, garantendo che ogni componente abbia un'etichetta e una descrizione ben progettate. Quando VoiceOver è attivo, l'app traduce l'interfaccia utente in **Modalità Testo**, mostrando solo elementi accessibili e utili per migliorare la velocità di navigazione e la comodità. Puoi anche attivare la Modalità Testo in Impostazioni → Accessibilità → Modalità Testo.

### Regolare la Posizione di un Brano in una Playlist con VoiceOver

1. Apri una playlist e tocca il pulsante **Altro**.
2. Seleziona **Cambia Ordine Brani**. La vista passa in modalità modifica.
3. Tocca l'icona dell'indicatore di riordino vicino al titolo del brano per dargli focus.
4. **Tocca due volte** rapidamente l'icona dell'indicatore di riordino. Al secondo tocco, non rilasciare il dito — tienilo finché non senti un suono che indica che la cella è pronta per essere spostata.
5. Ora puoi spostare la cella nella sua nuova posizione.

Gli altri componenti funzionano come previsto, usando i pattern VoiceOver forniti dal sistema.

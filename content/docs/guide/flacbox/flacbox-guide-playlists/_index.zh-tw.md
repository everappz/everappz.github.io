在播放清單部分，您會找到管理音樂收藏的實用工具。此區域將所有播放清單顯示在一處。頂部有一個 **「...」** 按鈕，位於導覽列中，會開啟包含不同播放清單選項的選單，還有一個工具列，提供搜尋、全部播放和隨機播放全部等快速操作。每個播放清單在其標題旁也有自己的 **「...」** 按鈕，為該播放清單提供更多選項。

Flacbox 中的播放清單可以包含來自雲端的線上曲目、已下載的離線檔案和裝置本機檔案的混合——所有內容在一個播放清單中——並無縫一起播放。

## 建立播放清單

建立新播放清單非常簡單。您有兩個選擇：點擊 **+** 按鈕，或點擊導覽列右上角的 **「...」** 按鈕並選擇新建播放清單。為播放清單起一個有意義的名稱，然後點擊儲存。

這將觸發「新增歌曲」對話框，您可以在其中手動挑選要包含在新播放清單中的曲目。曲目按來源類型方便地分類：

- **音樂庫** — 已在您音樂庫中的曲目。
- **此應用程式中的檔案** — 應用程式文件資料夾中的音訊檔案（從雲端儲存下載、透過 Wi-Fi Drive 匯入或透過 Finder 檔案分享新增）。
- **此裝置上的檔案** — 位於裝置其他位置的音訊檔案，不在此應用程式中。
- **連接** — 位於連接的雲端儲存服務中的線上檔案。

預設情況下，您只能將單首曲目新增到播放清單一次。如果您想允許重複，請在設定 → 音樂庫 → 播放清單 → 播放清單中的重複 → 啟用中啟用此功能。

## 匯入播放清單

在 Flacbox 中，我們新增了 M3U / M3U8 / CUE 檔案匯入功能，這樣您從其他播放器切換後不必手動重新建立播放清單。

首先，前往播放清單部分。然後，點擊右上角的更多按鈕。從選單中選擇匯入播放清單。

在下一個畫面，選擇檔案位置。支援的選項包括：

- 連接的雲端儲存
- 應用程式中的檔案
- 您裝置上的檔案

選擇連接的雲端儲存並開啟包含播放清單檔案的資料夾。支援的播放清單檔案副檔名包括 M3U、M3U8 和 CUE。選擇播放清單檔案並點擊完成確認您的選擇。

應用程式解析播放清單檔案，建立曲目清單，並在儲存中定位這些檔案以編譯最終播放清單，然後將其匯入到音樂庫中。重要：M3U / CUE 檔案必須包含媒體檔案的正確路徑，並且這些檔案應實際存在於您儲存的這些路徑中。了解更多關於播放清單匯入的資訊[在這裡](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox)。

## 播放清單詳情畫面

當您開啟播放清單時，會出現播放清單詳情畫面。您會在右上角找到 **「...」** 按鈕，提供播放清單選項，還有封面下方的三個按鈕：搜尋、繼續播放、全部播放和隨機播放全部。還有一個離線模式核取方塊，用於切換全播放清單離線同步。

- **繼續播放** — 恢復此播放清單上次儲存的播放位置。
- **搜尋** — 在當前播放清單中執行搜尋。
- **全部播放** — 將當前播放清單的所有曲目新增到播放器佇列。
- **隨機播放全部** — 與**全部播放**類似，但在新增到音訊播放器佇列前先隨機排列曲目。
- **離線模式** — 將此播放清單的所有曲目下載到本機檔案。新增到播放清單的任何新項目也會自動下載。

## 播放清單畫面中播放清單的更多操作

您可以透過點擊播放清單標題旁的 **「...」** 按鈕存取播放清單的操作。可用操作：

- **全部播放** — 將播放清單曲目新增到全新的播放器佇列。
- **下一首播放** — 將播放清單曲目新增到現有播放器佇列的頂部。
- **稍後播放** — 將播放清單曲目新增到現有播放器佇列的底部。
- **編輯圖片** — 更改播放清單的封面。
- **啟用離線模式** — 為播放清單開啟離線模式。現有和新曲目都會自動下載。了解更多關於離線模式的資訊[在這裡](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)。
- **匯出歌曲清單** — 將此播放清單匯出為 **M3U / M3U8 / CSV / TXT** 格式，如[這裡](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)所述。
- **新增到封存** — 將此播放清單（包括 m3u 檔案、封面和所有曲目）封存為單個 ZIP，如[這裡](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)所述。Premium 功能。
- **新增歌曲** — 向當前播放清單新增更多歌曲。
- **重新命名** — 重新命名播放清單。
- **刪除播放清單** — 從音樂庫中刪除播放清單。**此操作無法復原。**

## 播放清單詳情畫面中播放清單的更多操作

您可以透過點擊右上角的 **「...」** 按鈕存取播放清單的操作。可用操作：

- **選擇** — 啟用曲目選擇模式，適用於從播放清單中刪除多首曲目或更改其順序。
- **下一首播放** — 將播放清單曲目新增到現有播放器佇列的頂部。
- **稍後播放** — 將播放清單曲目新增到現有播放器佇列的底部。
- **新增歌曲** — 向播放清單新增新歌曲。
- **重新排列歌曲** — 使用拖放手動更改播放清單中歌曲的順序。
- **重新命名** — 重新命名當前播放清單。
- **編輯圖片** — 更改當前播放清單的專輯封面。
- **匯出歌曲清單** — 將此播放清單匯出為 **M3U / M3U8 / CSV / TXT** 格式。了解更多[在這裡](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。
- **新增到封存** — 將當前播放清單（包括所有曲目和 m3u 檔案）壓縮為 ZIP。了解更多[在這裡](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)。
- **排序** — 更改播放清單中曲目的順序。排序選項包括**歌曲標題、歌曲編號、專輯、藝術家、專輯藝術家、類型、作曲家、評分、年份、每分鐘拍數、時長、檔名、檔案修改日期、檔案建立日期**和**手動**。**手動**排序選項允許使用拖放手動重新排序歌曲。
- **搜尋** — 在當前播放清單中搜尋特定歌曲。
- **格線 / 清單** — 更改畫面版面配置呈現方式。
- **刪除播放清單** — 從音樂庫中刪除播放清單。此操作不會從儲存中刪除曲目，但**無法復原**。

## 更改播放清單中的歌曲順序

要更改播放清單中歌曲的順序，點擊右上角的 **「...」** 按鈕並選擇**選擇**進入選擇模式。使用每首曲目旁邊的重新排序控制項和拖放手勢來上下移動它們。點擊重新排序控制項會將曲目移到清單頂部。要退出選擇模式並套用變更，點擊**完成**。

對於長播放清單，選擇更多操作 → 重新排列歌曲進入專用的拖放重新排序模式，工作流程更簡單。

## 更改播放清單封面圖片

要更改播放清單的封面圖片，點擊右上角的 **「...」** 按鈕並選擇**編輯圖片**。從可用來源（照片、檔案、雲端儲存或播放清單中曲目產生的封面）選擇圖片，然後點擊**完成**確認。

## 向播放清單新增歌曲

開啟播放清單並點擊右上角的 **「...」** 按鈕，然後選擇**新增歌曲**以開啟對話框。選擇要新增的曲目並點擊**完成**確認。

## 從播放清單刪除多首歌曲

開啟播放清單，點擊右上角的 **「...」** 按鈕，並選擇**選擇**進入選擇模式。選擇要刪除的曲目並點擊畫面底部的**從播放清單刪除**。點擊**完成**確認。

## 曲目選項

播放清單中的每首曲目都有一系列操作，透過點擊 **「...」** 按鈕存取。如果看不到所有操作，請向下捲動查看。您可以從播放清單中刪除曲目、下載曲目、編輯音訊標籤等。

- **下一首播放** — 將曲目新增到播放器佇列頂部。
- **稍後播放** — 將曲目附加到播放器佇列底部。
- **新增到播放清單** — 將曲目新增到另一個播放清單。
- **新增到最愛** — 將曲目標記為最愛以便快速存取。
- **下載** — 使曲目可離線使用。它出現在傳輸佇列和音樂庫**已下載音樂**部分的**本機檔案**標籤頁中。
- **編輯音訊標籤** — 開啟內建標籤編輯器以更改曲目元資料。
- **以...開啟** — 匯出曲目並在另一個應用程式中開啟。
- **在資料夾中顯示** — 顯示音訊檔案所在的資料夾。
- **在 Finder 中顯示** — 對於從 Mac 匯入的檔案，這將顯示音訊檔案在 Mac 上所在的資料夾。
- **從播放清單刪除** — 從播放清單中刪除曲目。
- **從雲端服務刪除** — 從播放清單和關聯的雲端服務中刪除曲目。**此操作無法復原。**
- **從音樂庫刪除** — 從音樂庫中刪除曲目，保留儲存中的檔案不受影響。

## 輔助功能

Flacbox 與 **VoiceOver** 技術完全相容，確保每個元件都有設計良好的標籤和描述。當 VoiceOver 啟動時，應用程式將使用者介面轉換為**文字模式**，只顯示可存取和有用的元素，以提高導覽速度和便利性。您也可以在設定 → 輔助功能 → 文字模式中啟用文字模式。

### 使用 VoiceOver 調整播放清單中的曲目位置

1. 開啟播放清單並點擊**更多**按鈕。
2. 選擇**更改歌曲順序**。清單切換到編輯模式。
3. 點擊曲目標題旁的重新排序指示器圖示使其獲得焦點。
4. 快速**按兩下**重新排序指示器。第二次點擊時，不要放開手指——保持按住直到聽到聲音指示儲存格可以移動。
5. 現在可以將儲存格移動到新位置。

其他元件按預期工作，使用系統提供的 VoiceOver 模式。

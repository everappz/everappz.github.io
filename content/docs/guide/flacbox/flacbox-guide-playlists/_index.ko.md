재생 목록 섹션에서는 음악 컬렉션을 관리하는 데 도움이 되는 유용한 도구를 찾을 수 있습니다. 이 영역에는 모든 재생 목록이 한 곳에 표시됩니다. 상단에는 다양한 재생 목록 옵션이 있는 메뉴를 여는 **"..."** 버튼이 탐색 모음에 있으며, 검색, 모두 재생, 모두 셔플과 같은 빠른 작업이 있는 도구 모음도 있습니다. 각 재생 목록에는 해당 재생 목록만을 위한 더 많은 옵션을 제공하는 제목 옆의 **"..."** 버튼도 있습니다.

Flacbox의 재생 목록에는 온라인 클라우드 트랙, 오프라인 다운로드 파일 및 기기의 로컬 파일이 모두 하나의 재생 목록에 혼합될 수 있으며 끊김 없이 함께 재생됩니다.

## 재생 목록 만들기

새 재생 목록 만들기는 간단합니다. 두 가지 옵션이 있습니다: **+** 버튼을 누르거나, 탐색 모음의 오른쪽 상단에 있는 **"..."** 버튼을 누르고 새 재생 목록을 선택하세요. 재생 목록에 의미 있는 이름을 지정하고 저장을 누르세요.

그러면 노래 추가 대화 상자가 열리며, 새 재생 목록에 포함할 트랙을 직접 선택할 수 있습니다. 트랙은 소스 유형별로 편리하게 분류됩니다:

- **라이브러리** — 음악 라이브러리에 이미 있는 트랙.
- **이 앱의 파일** — 앱의 Documents 폴더에 있는 오디오 파일 (클라우드 스토리지에서 다운로드, Wi-Fi Drive를 통해 가져오거나 Finder 파일 공유를 통해 추가된 것).
- **이 기기의 파일** — 이 앱이 아닌 기기의 다른 곳에 위치한 오디오 파일.
- **연결하기** — 연결된 클라우드 스토리지 서비스에 있는 온라인 파일.

기본적으로 재생 목록에 단일 트랙을 한 번만 추가할 수 있습니다. 중복을 허용하려면 설정 → 음악 라이브러리 → 재생 목록 → 재생 목록의 중복 → 활성화에서 이 기능을 활성화하세요.

## 재생 목록 가져오기

Flacbox에서는 다른 플레이어에서 전환한 후 재생 목록을 수동으로 다시 만들 필요가 없도록 M3U / M3U8 / CUE 파일 가져오기를 추가했습니다.

먼저 재생 목록 섹션으로 이동하세요. 그런 다음 오른쪽 상단에 있는 더 보기 버튼을 누르세요. 메뉴에서 재생 목록 가져오기를 선택하세요.

다음 화면에서 파일 위치를 선택하세요. 지원되는 옵션은 다음과 같습니다:

- 연결된 클라우드 스토리지
- 앱의 파일
- 기기의 파일

연결된 클라우드 스토리지를 선택하고 재생 목록 파일이 포함된 폴더를 여세요. 지원되는 재생 목록 파일 확장자는 M3U, M3U8 및 CUE입니다. 재생 목록 파일을 선택하고 완료됨을 눌러 선택을 확인하세요.

앱은 재생 목록 파일을 분석하고, 트랙 목록을 만들고, 스토리지에서 해당 파일을 찾아 최종 재생 목록을 컴파일하고 이를 음악 라이브러리로 가져옵니다. 중요: M3U / CUE 파일에는 미디어 파일의 올바른 경로가 포함되어 있어야 하며, 파일이 실제로 스토리지의 해당 경로에 있어야 합니다. 재생 목록 가져오기에 대해 더 알아보려면 [여기](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox)를 읽어보세요.

## 재생 목록 세부 정보 화면

재생 목록을 열면 재생 목록 세부 정보 화면이 나타납니다. 오른쪽 상단에는 재생 목록 옵션이 있는 **"..."** 버튼이 있으며, 아트워크 아래에는 검색, 계속 재생, 모두 재생, 모두 셔플 세 개의 버튼이 있습니다. 전체 재생 목록 오프라인 동기화를 전환하는 오프라인 모드 체크박스도 있습니다.

- **계속 재생** — 이 재생 목록의 마지막 저장된 재생 위치를 복원합니다.
- **검색** — 현재 재생 목록에서 검색을 수행합니다.
- **모두 재생** — 현재 재생 목록의 모든 트랙을 플레이어 큐에 추가합니다.
- **모두 셔플** — **모두 재생**과 유사하지만 오디오 플레이어 큐에 추가하기 전에 트랙을 섞습니다.
- **오프라인 모드** — 이 재생 목록의 모든 트랙을 로컬 파일로 다운로드합니다. 재생 목록에 새로 추가된 항목도 자동으로 다운로드됩니다.

## 재생 목록 화면의 재생 목록에 대한 추가 작업

재생 목록 제목 옆의 **"..."** 버튼을 눌러 재생 목록에 대한 작업에 액세스할 수 있습니다. 사용 가능한 작업:

- **모두 재생** — 재생 목록 트랙을 새 플레이어 큐에 추가합니다.
- **다음에 재생** — 재생 목록 트랙을 기존 플레이어 큐의 맨 위에 추가합니다.
- **나중에 재생** — 재생 목록 트랙을 기존 플레이어 큐의 맨 아래에 추가합니다.
- **이미지 편집** — 재생 목록의 아트워크를 변경합니다.
- **오프라인 모드 활성화** — 재생 목록에 대한 오프라인 모드를 켭니다. 기존 및 새 트랙 모두 자동으로 다운로드됩니다. 오프라인 모드에 대해 더 알아보려면 [여기](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)를 읽어보세요.
- **노래 목록 내보내기** — 이 재생 목록을 [여기](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)에 설명된 대로 **M3U / M3U8 / CSV / TXT**로 내보냅니다.
- **보관함에 추가** — [여기](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)에 설명된 대로 이 재생 목록(m3u 파일, 앨범 커버 및 모든 트랙 포함)을 단일 ZIP으로 보관합니다. 프리미엄 기능.
- **노래 추가** — 현재 재생 목록에 노래를 더 추가합니다.
- **이름 바꾸기** — 재생 목록의 이름을 바꿉니다.
- **재생 목록 삭제** — 음악 라이브러리에서 재생 목록을 삭제합니다. **이 작업은 취소할 수 없습니다.**

## 재생 목록 세부 정보 화면의 재생 목록에 대한 추가 작업

오른쪽 상단에 있는 **"..."** 버튼을 눌러 재생 목록에 대한 작업에 액세스할 수 있습니다. 사용 가능한 작업:

- **선택하다** — 트랙 선택 모드를 활성화하여 재생 목록에서 여러 트랙을 삭제하거나 순서를 변경하는 데 유용합니다.
- **다음에 재생** — 재생 목록 트랙을 기존 플레이어 큐의 맨 위에 추가합니다.
- **나중에 재생** — 재생 목록 트랙을 기존 플레이어 큐의 맨 아래에 추가합니다.
- **노래 추가** — 재생 목록에 새 노래를 추가합니다.
- **노래 재정렬** — 드래그 앤 드롭을 사용하여 재생 목록에서 노래 순서를 수동으로 변경합니다.
- **이름 바꾸기** — 현재 재생 목록의 이름을 바꿉니다.
- **이미지 편집** — 현재 재생 목록의 앨범 아트워크를 변경합니다.
- **노래 목록 내보내기** — 이 재생 목록을 **M3U / M3U8 / CSV / TXT**로 내보냅니다. 더 알아보려면 [여기](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)를 읽어보세요.
- **보관함에 추가** — 모든 트랙과 m3u 파일을 포함하여 현재 재생 목록을 ZIP으로 압축합니다. 더 알아보려면 [여기](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)를 읽어보세요.
- **정렬** — 재생 목록에서 트랙 순서를 변경합니다. 정렬 옵션: **노래 제목, 노래 번호, 앨범, 아티스트, 앨범 아티스트, 장르, 작곡가, 평점, 연도, BPM, 재생 시간, 파일 이름, 파일 수정 날짜, 파일 생성 날짜** 및 **수동**. **수동** 정렬 옵션을 사용하면 드래그 앤 드롭으로 노래를 수동으로 재정렬할 수 있습니다.
- **검색** — 현재 재생 목록 내에서 특정 노래를 검색합니다.
- **그리드 / 목록** — 화면 레이아웃 프레젠테이션을 변경합니다.
- **재생 목록 삭제** — 음악 라이브러리에서 재생 목록을 삭제합니다. 이 작업은 스토리지에서 트랙을 삭제하지 않지만 **취소할 수 없습니다**.

## 재생 목록에서 노래 순서 변경

재생 목록에서 노래 순서를 변경하려면 오른쪽 상단의 **"..."** 버튼을 누르고 **선택하다**를 선택하여 선택 모드로 진입하세요. 각 트랙 옆의 재정렬 컨트롤과 드래그 앤 드롭 제스처를 사용하여 위아래로 이동하세요. 재정렬 컨트롤을 누르면 트랙이 목록의 맨 위로 이동합니다. 선택 모드를 종료하고 변경 사항을 적용하려면 **완료됨**을 누르세요.

긴 재생 목록에서 더 간단한 워크플로우를 위해 추가 작업 → 노래 재정렬을 선택하여 전용 드래그 앤 드롭 재정렬 모드로 전환하세요.

## 재생 목록 커버 이미지 변경

재생 목록의 커버 이미지를 변경하려면 오른쪽 상단의 **"..."** 버튼을 누르고 **이미지 편집**을 선택하세요. 사용 가능한 소스(사진, 파일, 클라우드 스토리지 또는 재생 목록의 트랙에서 생성된 아트워크)에서 이미지를 선택한 다음 **완료됨**을 눌러 확인하세요.

## 재생 목록에 노래 추가

재생 목록을 열고 오른쪽 상단의 **"..."** 버튼을 누른 다음 **노래 추가**를 선택하여 대화 상자를 여세요. 추가할 트랙을 선택하고 **완료됨**을 눌러 확인하세요.

## 재생 목록에서 여러 노래 삭제

재생 목록을 열고 오른쪽 상단의 **"..."** 버튼을 누른 다음 **선택하다**를 선택하여 선택 모드로 진입하세요. 삭제할 트랙을 선택하고 화면 하단의 **재생 목록에서 삭제**를 누르세요. **완료됨**을 눌러 확인하세요.

## 트랙 옵션

재생 목록의 각 트랙에는 **"..."** 버튼을 눌러 액세스할 수 있는 작업 목록이 있습니다. 모든 작업이 보이지 않으면 스크롤하여 확인하세요. 재생 목록에서 트랙을 삭제하거나, 다운로드하거나, 오디오 태그를 편집하는 등의 작업을 할 수 있습니다.

- **다음에 재생** — 트랙을 플레이어 큐의 맨 위에 추가합니다.
- **나중에 재생** — 트랙을 플레이어 큐의 맨 아래에 추가합니다.
- **재생 목록에 추가** — 트랙을 다른 재생 목록에 추가합니다.
- **즐겨찾기에 추가** — 빠른 액세스를 위해 트랙을 즐겨찾기로 표시합니다.
- **다운로드** — 트랙을 오프라인에서 사용할 수 있도록 합니다. 전송 큐와 음악 라이브러리의 **다운로드된 음악** 섹션의 **로컬 파일** 탭에 나타납니다.
- **오디오 태그 편집** — 트랙 메타데이터를 변경하기 위한 내장 태그 편집기를 엽니다.
- **다른 앱에서 열기** — 트랙을 내보내고 다른 앱에서 엽니다.
- **폴더에서 표시** — 오디오 파일이 있는 폴더를 표시합니다.
- **Finder에서 표시** — Mac에서 가져온 파일의 경우, Mac에서 오디오 파일이 있는 폴더를 표시합니다.
- **재생 목록에서 삭제** — 재생 목록에서 트랙을 삭제합니다.
- **클라우드 서비스에서 삭제** — 재생 목록과 관련 클라우드 서비스에서 트랙을 삭제합니다. **이 작업은 취소할 수 없습니다.**
- **음악 라이브러리에서 삭제** — 음악 라이브러리에서 트랙을 삭제하고, 스토리지의 파일은 그대로 둡니다.

## 접근성

Flacbox는 **VoiceOver** 기술로 완전히 접근 가능하며, 모든 구성 요소에 잘 설계된 레이블과 설명이 있습니다. VoiceOver가 활성화되면 앱은 사용자 인터페이스를 **텍스트 모드**로 전환하여 접근 가능하고 유용한 요소만 표시함으로써 탐색 속도와 편의성을 향상시킵니다. 설정 → 접근성 → 텍스트 모드에서 텍스트 모드를 수동으로 활성화할 수도 있습니다.

### VoiceOver로 재생 목록에서 트랙 위치 조정

1. 재생 목록을 열고 **더 보기** 버튼을 누르세요.
2. **노래 순서 변경**을 선택하세요. 보기가 편집 모드로 전환됩니다.
3. 트랙 제목 옆의 재정렬 표시기 아이콘을 눌러 포커스를 줍니다.
4. 재정렬 표시기 아이콘을 빠르게 **두 번 탭**하세요. 두 번째 탭에서 손가락을 놓지 마세요 — 셀이 이동할 준비가 되었음을 알리는 소리가 들릴 때까지 유지하세요.
5. 이제 셀을 새 위치로 이동할 수 있습니다.

다른 구성 요소는 시스템 제공 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

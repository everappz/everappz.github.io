在播放列表部分，您会找到管理音乐收藏的实用工具。此区域将所有播放列表显示在一处。顶部有一个 **「...」** 按钮，位于导航栏中，会打开包含不同播放列表选项的菜单，还有一个工具栏，提供搜索、全部播放和随机播放全部等快速操作。每个播放列表在其标题旁也有自己的 **「...」** 按钮，为该播放列表提供更多选项。

Flacbox 中的播放列表可以包含来自云端的在线曲目、已下载的离线文件和设备本地文件的混合——所有内容在一个播放列表中——并无缝一起播放。

## 创建播放列表

创建新播放列表非常简单。您有两个选择：点击 **+** 按钮，或点击导航栏右上角的 **「...」** 按钮并选择新建播放列表。为播放列表起一个有意义的名称，然后点击保存。

这将触发"添加歌曲"对话框，您可以在其中手动挑选要包含在新播放列表中的曲目。曲目按来源类型方便地分类：

- **音乐库** — 已在您音乐库中的曲目。
- **此应用程序中的文件** — 应用程序文档文件夹中的音频文件（从云存储下载、通过 Wi-Fi Drive 导入或通过 Finder 文件共享添加）。
- **此设备上的文件** — 位于设备其他位置的音频文件，不在此应用程序中。
- **连接** — 位于连接的云存储服务中的在线文件。

默认情况下，您只能将单首曲目添加到播放列表一次。如果您想允许重复，请在设置 → 音乐库 → 播放列表 → 播放列表中的重复 → 启用中启用此功能。

## 导入播放列表

在 Flacbox 中，我们添加了 M3U / M3U8 / CUE 文件导入功能，这样您从其他播放器切换后不必手动重新创建播放列表。

首先，前往播放列表部分。然后，点击右上角的更多按钮。从菜单中选择导入播放列表。

在下一个界面，选择文件位置。支持的选项包括：

- 连接的云存储
- 应用程序中的文件
- 您设备上的文件

选择连接的云存储并打开包含播放列表文件的文件夹。支持的播放列表文件扩展名包括 M3U、M3U8 和 CUE。选择播放列表文件并点击完成确认您的选择。

应用程序解析播放列表文件，创建曲目列表，并在存储中定位这些文件以编译最终播放列表，然后将其导入到音乐库中。重要：M3U / CUE 文件必须包含媒体文件的正确路径，并且这些文件应实际存在于您存储的这些路径中。了解更多关于播放列表导入的信息[在这里](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox)。

## 播放列表详情界面

当您打开播放列表时，会出现播放列表详情界面。您会在右上角找到 **「...」** 按钮，提供播放列表选项，还有封面下方的三个按钮：搜索、继续播放、全部播放和随机播放全部。还有一个离线模式复选框，用于切换全播放列表离线同步。

- **继续播放** — 恢复此播放列表上次保存的播放位置。
- **搜索** — 在当前播放列表中执行搜索。
- **全部播放** — 将当前播放列表的所有曲目添加到播放器队列。
- **随机播放全部** — 与**全部播放**类似，但在添加到音频播放器队列前先随机排列曲目。
- **离线模式** — 将此播放列表的所有曲目下载到本地文件。添加到播放列表的任何新项目也会自动下载。

## 播放列表界面中播放列表的更多操作

您可以通过点击播放列表标题旁的 **「...」** 按钮访问播放列表的操作。可用操作：

- **全部播放** — 将播放列表曲目添加到全新的播放器队列。
- **下一首播放** — 将播放列表曲目添加到现有播放器队列的顶部。
- **稍后播放** — 将播放列表曲目添加到现有播放器队列的底部。
- **编辑图片** — 更改播放列表的封面。
- **启用离线模式** — 为播放列表开启离线模式。现有和新曲目都会自动下载。了解更多关于离线模式的信息[在这里](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)。
- **导出歌曲列表** — 将此播放列表导出为 **M3U / M3U8 / CSV / TXT** 格式，如[这里](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)所述。
- **添加到归档** — 将此播放列表（包括 m3u 文件、封面和所有曲目）归档为单个 ZIP，如[这里](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)所述。Premium 功能。
- **添加歌曲** — 向当前播放列表添加更多歌曲。
- **重命名** — 重命名播放列表。
- **删除播放列表** — 从音乐库中删除播放列表。**此操作无法撤销。**

## 播放列表详情界面中播放列表的更多操作

您可以通过点击右上角的 **「...」** 按钮访问播放列表的操作。可用操作：

- **选择** — 激活曲目选择模式，适用于从播放列表中删除多首曲目或更改其顺序。
- **下一首播放** — 将播放列表曲目添加到现有播放器队列的顶部。
- **稍后播放** — 将播放列表曲目添加到现有播放器队列的底部。
- **添加歌曲** — 向播放列表添加新歌曲。
- **重新排列歌曲** — 使用拖放手动更改播放列表中歌曲的顺序。
- **重命名** — 重命名当前播放列表。
- **编辑图片** — 更改当前播放列表的专辑封面。
- **导出歌曲列表** — 将此播放列表导出为 **M3U / M3U8 / CSV / TXT** 格式。了解更多[在这里](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。
- **添加到归档** — 将当前播放列表（包括所有曲目和 m3u 文件）压缩为 ZIP。了解更多[在这里](/docs/howto/how-to-archive-zip-playlists-albums-artists-and-genres-in-evermusic-flacbox-and-transfer-to)。
- **排序** — 更改播放列表中曲目的顺序。排序选项包括**歌曲标题、歌曲编号、专辑、艺术家、专辑艺术家、流派、作曲家、评分、年份、每分钟拍数、时长、文件名、文件修改日期、文件创建日期**和**手动**。**手动**排序选项允许使用拖放手动重新排序歌曲。
- **搜索** — 在当前播放列表中搜索特定歌曲。
- **网格 / 列表** — 更改界面布局呈现方式。
- **删除播放列表** — 从音乐库中删除播放列表。此操作不会从存储中删除曲目，但**无法撤销**。

## 更改播放列表中的歌曲顺序

要更改播放列表中歌曲的顺序，点击右上角的 **「...」** 按钮并选择**选择**进入选择模式。使用每首曲目旁边的重新排序控件和拖放手势来上下移动它们。点击重新排序控件会将曲目移到列表顶部。要退出选择模式并应用更改，点击**完成**。

对于长播放列表，选择更多操作 → 重新排列歌曲进入专用的拖放重新排序模式，工作流程更简单。

## 更改播放列表封面图片

要更改播放列表的封面图片，点击右上角的 **「...」** 按钮并选择**编辑图片**。从可用来源（照片、文件、云存储或播放列表中曲目生成的封面）选择图片，然后点击**完成**确认。

## 向播放列表添加歌曲

打开播放列表并点击右上角的 **「...」** 按钮，然后选择**添加歌曲**以打开对话框。选择要添加的曲目并点击**完成**确认。

## 从播放列表删除多首歌曲

打开播放列表，点击右上角的 **「...」** 按钮，并选择**选择**进入选择模式。选择要删除的曲目并点击屏幕底部的**从播放列表删除**。点击**完成**确认。

## 曲目选项

播放列表中的每首曲目都有一系列操作，通过点击 **「...」** 按钮访问。如果看不到所有操作，请向下滚动查看。您可以从播放列表中删除曲目、下载曲目、编辑音频标签等。

- **下一首播放** — 将曲目添加到播放器队列顶部。
- **稍后播放** — 将曲目追加到播放器队列底部。
- **添加到播放列表** — 将曲目添加到另一个播放列表。
- **添加到收藏夹** — 将曲目标记为收藏以便快速访问。
- **下载** — 使曲目可离线使用。它出现在传输队列和音乐库**已下载音乐**部分的**本地文件**标签页中。
- **编辑音频标签** — 打开内置标签编辑器以更改曲目元数据。
- **打开方式** — 导出曲目并在另一个应用中打开。
- **在文件夹中显示** — 显示音频文件所在的文件夹。
- **在 Finder 中显示** — 对于从 Mac 导入的文件，这将显示音频文件在 Mac 上所在的文件夹。
- **从播放列表删除** — 从播放列表中删除曲目。
- **从云服务删除** — 从播放列表和关联的云服务中删除曲目。**此操作无法撤销。**
- **从音乐库删除** — 从音乐库中删除曲目，保留存储中的文件不受影响。

## 辅助功能

Flacbox 与 **VoiceOver** 技术完全兼容，确保每个组件都有设计良好的标签和描述。当 VoiceOver 激活时，应用程序将用户界面转换为**文字模式**，只显示可访问和有用的元素，以提高导航速度和便利性。您也可以在设置 → 辅助功能 → 文字模式中激活文字模式。

### 使用 VoiceOver 调整播放列表中的曲目位置

1. 打开播放列表并点击**更多**按钮。
2. 选择**更改歌曲顺序**。列表切换到编辑模式。
3. 点击曲目标题旁的重新排序指示器图标使其获得焦点。
4. 快速**双击**重新排序指示器。第二次点击时，不要松开手指——保持按住直到听到声音指示单元格可以移动。
5. 现在可以将单元格移动到新位置。

其他组件按预期工作，使用系统提供的 VoiceOver 模式。

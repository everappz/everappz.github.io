Di bagian Playlist, Anda akan menemukan alat yang berguna untuk mengelola koleksi musik Anda. Area ini menampilkan semua playlist Anda di satu tempat. Di bagian atas, ada tombol **"..."** di bilah navigasi yang membuka menu dengan berbagai opsi playlist, ditambah bilah alat dengan aksi cepat seperti Cari, Putar Semua, dan Acak Semua. Setiap playlist juga memiliki tombol **"..."** sendiri di sebelah judulnya, memberikan lebih banyak opsi khusus untuk playlist tersebut.

Playlist di Flacbox dapat berisi campuran trek cloud online, file yang diunduh secara offline, dan file lokal dari perangkat Anda — semuanya dalam satu playlist — dan diputar dengan mulus bersama-sama.

## Membuat Playlist

Membuat playlist baru sangat mudah. Anda memiliki dua pilihan: ketuk tombol **+**, atau ketuk tombol **"..."** yang terletak di sudut kanan atas bilah navigasi dan pilih Playlist Baru. Beri nama playlist Anda yang bermakna, lalu ketuk Simpan.

Ini memicu dialog Tambah Lagu, di mana Anda dapat memilih sendiri trek yang ingin Anda sertakan dalam playlist baru Anda. Trek dikategorikan berdasarkan jenis sumber:

- **Perpustakaan** — trek yang sudah ada di perpustakaan musik Anda.
- **File di Aplikasi Ini** — file audio di folder Dokumen aplikasi (diunduh dari penyimpanan cloud, diimpor melalui Wi-Fi Drive, atau ditambahkan melalui Finder File Sharing).
- **File di Perangkat Ini** — file audio yang terletak di tempat lain di perangkat Anda, bukan di aplikasi ini.
- **Koneksi** — file online yang terletak di layanan penyimpanan cloud yang terhubung.

Secara default, Anda hanya dapat menambahkan satu trek ke playlist sekali saja. Jika Anda ingin mengizinkan duplikat, aktifkan fitur ini di Pengaturan → Perpustakaan Musik → Playlist → Duplikat dalam Playlist → Aktifkan.

## Impor Playlist

Di Flacbox, kami telah menambahkan impor file M3U / M3U8 / CUE sehingga Anda tidak perlu membuat ulang playlist secara manual setelah beralih dari pemutar lain.

Pertama, pergi ke bagian Playlist. Kemudian, ketuk tombol Lainnya di sudut kanan atas. Dari menu, pilih Impor Playlist.

Di layar berikutnya, pilih lokasi file. Opsi yang didukung meliputi:

- Penyimpanan cloud yang terhubung
- File di aplikasi
- File di perangkat Anda

Pilih penyimpanan cloud yang terhubung dan buka folder yang berisi file playlist. Ekstensi file playlist yang didukung meliputi M3U, M3U8, dan CUE. Pilih file playlist dan ketuk Selesai untuk mengonfirmasi pilihan Anda.

Aplikasi mengurai file playlist, membuat daftar trek, dan menemukan file-file tersebut di penyimpanan untuk menyusun playlist akhir, yang kemudian diimpor ke perpustakaan musik. Penting: file M3U / CUE harus berisi jalur yang benar ke file media, dan file-file tersebut harus benar-benar ada di jalur tersebut di penyimpanan Anda. Baca lebih lanjut tentang impor playlist [di sini](/docs/howto/how-to-import-m3u-playlist-to-evermusic-and-flacbox).

## Layar Detail Playlist

Saat Anda membuka playlist, layar Detail Playlist muncul. Anda akan menemukan tombol **"..."** di sudut kanan atas dengan opsi playlist, dan tiga tombol di bawah seni sampul: Cari, Lanjutkan Pemutaran, Putar Semua, dan Acak Semua. Ada juga kotak centang Mode Offline untuk mengaktifkan sinkronisasi offline seluruh playlist.

- **Lanjutkan Pemutaran** — memulihkan posisi pemutaran terakhir yang tersimpan untuk playlist ini.
- **Cari** — melakukan pencarian dalam playlist saat ini.
- **Putar Semua** — menambahkan semua trek dari playlist saat ini ke antrean pemutar.
- **Acak Semua** — seperti **Putar Semua**, tetapi mengacak trek sebelum menambahkannya ke antrean pemutar audio.
- **Mode Offline** — mengunduh semua trek dari playlist ini ke file lokal. Item baru yang ditambahkan ke playlist juga diunduh secara otomatis.

## Aksi Lainnya untuk Playlist di Layar Playlist

Anda dapat mengakses aksi untuk playlist dengan mengetuk tombol **"..."** di dekat judul playlist. Aksi yang tersedia:

- **Putar Semua** — menambahkan trek playlist ke antrean pemutar baru.
- **Putar Berikutnya** — menambahkan trek playlist ke bagian atas antrean pemutar yang ada.
- **Putar Nanti** — menambahkan trek playlist ke bagian bawah antrean pemutar yang ada.
- **Edit Gambar** — mengubah seni sampul playlist.
- **Aktifkan Mode Offline** — mengaktifkan mode offline untuk playlist. Baik trek yang ada maupun yang baru diunduh secara otomatis. Baca lebih lanjut tentang mode offline [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).
- **Ekspor Daftar Lagu** — ekspor playlist ini ke **M3U / M3U8 / CSV / TXT**.
- **Tambah ke Arsip** — mengarsipkan playlist ini (termasuk file m3u, sampul album, dan semua trek) menjadi satu ZIP. Fitur Premium.
- **Tambah Lagu** — menambahkan lebih banyak lagu ke playlist saat ini.
- **Ubah Nama** — mengubah nama playlist.
- **Hapus Playlist** — menghapus playlist dari perpustakaan musik. **Tindakan ini tidak dapat dibatalkan.**

## Aksi Lainnya untuk Playlist di Layar Detail Playlist

Anda dapat mengakses aksi untuk playlist dengan mengetuk tombol **"..."** di sudut kanan atas. Aksi yang tersedia:

- **Pilih** — mengaktifkan mode pemilihan trek, berguna untuk menghapus beberapa trek dari playlist atau mengubah urutannya.
- **Putar Berikutnya** — menambahkan trek playlist ke bagian atas antrean pemutar yang ada.
- **Putar Nanti** — menambahkan trek playlist ke bagian bawah antrean pemutar yang ada.
- **Tambah Lagu** — menambahkan lagu baru ke playlist.
- **Susun Ulang Lagu** — mengubah urutan lagu secara manual dalam playlist menggunakan seret dan lepas.
- **Ubah Nama** — mengubah nama playlist saat ini.
- **Edit Gambar** — mengubah seni sampul album untuk playlist saat ini.
- **Ekspor Daftar Lagu** — ekspor playlist ini ke **M3U / M3U8 / CSV / TXT**.
- **Tambah ke Arsip** — ZIP playlist saat ini termasuk semua trek dan file m3u.
- **Urutkan** — mengubah urutan trek dalam playlist. Opsi pengurutan meliputi **Judul Lagu, Nomor Lagu, Album, Artis, Artis Album, Genre, Komposer, Rating, Tahun, Beats per Menit, Durasi, Nama File, Tanggal Modifikasi File, Tanggal Pembuatan File,** dan **Manual**. Opsi urutan **Manual** memungkinkan pengurutan ulang lagu secara manual menggunakan seret dan lepas.
- **Cari** — mencari lagu tertentu dalam playlist saat ini.
- **Grid / Daftar** — mengubah tampilan presentasi layar.
- **Hapus Playlist** — menghapus playlist dari perpustakaan musik. Tindakan ini tidak menghapus trek dari penyimpanan Anda, tetapi **tidak dapat dibatalkan**.

## Mengubah Urutan Lagu dalam Playlist

Untuk mengubah urutan lagu dalam playlist, ketuk tombol **"..."** di sudut kanan atas dan pilih **Pilih** untuk masuk ke mode pemilihan. Gunakan kontrol pengurutan ulang dan gestur seret dan lepas di dekat setiap trek untuk memindahkannya ke atas atau ke bawah. Mengetuk kontrol pengurutan ulang memindahkan trek ke bagian atas daftar. Untuk keluar dari mode pemilihan dan menerapkan perubahan, ketuk **Selesai**.

Untuk alur kerja yang lebih sederhana pada playlist panjang, pilih Aksi Lainnya → Susun Ulang Lagu untuk masuk ke mode pengurutan ulang seret dan lepas khusus.

## Mengubah Gambar Sampul Playlist

Untuk mengubah gambar sampul playlist, ketuk tombol **"..."** di sudut kanan atas dan pilih **Edit Gambar**. Pilih gambar dari sumber yang tersedia (Foto, File, penyimpanan cloud, atau seni sampul yang dihasilkan dari trek dalam playlist), lalu konfirmasi dengan mengetuk **Selesai**.

## Menambahkan Lagu ke Playlist

Buka playlist dan ketuk tombol **"..."** di sudut kanan atas, lalu pilih **Tambah Lagu** untuk membuka dialog. Pilih trek yang ingin Anda tambahkan dan konfirmasi dengan mengetuk **Selesai**.

## Menghapus Beberapa Lagu dari Playlist

Buka playlist, ketuk tombol **"..."** di sudut kanan atas, dan pilih **Pilih** untuk masuk ke mode pemilihan. Pilih trek yang ingin Anda hapus dan ketuk **Hapus dari Playlist** di bagian bawah layar. Konfirmasi dengan mengetuk **Selesai**.

## Opsi Trek

Setiap trek dalam playlist memiliki daftar aksi, dapat diakses dengan mengetuk tombol **"..."**. Jika Anda tidak dapat melihat semua aksi, gulir ke bawah untuk melihatnya. Anda dapat menghapus trek dari playlist, mengunduhnya, mengedit tag audio, dan lainnya.

- **Putar Berikutnya** — menambahkan trek ke bagian atas antrean pemutar.
- **Putar Nanti** — menambahkan trek ke bagian bawah antrean pemutar.
- **Tambah ke Playlist** — menambahkan trek ke playlist lain.
- **Tambah ke Favorit** — menandai trek sebagai favorit untuk akses cepat.
- **Unduh** — membuat trek tersedia secara offline.
- **Edit Tag Audio** — membuka editor tag bawaan untuk mengubah metadata trek.
- **Buka Di** — mengekspor trek dan membukanya di aplikasi lain.
- **Tampilkan di Folder** — menampilkan folder tempat file audio berada.
- **Tampilkan di Finder** — untuk file yang diimpor dari Mac Anda, ini menampilkan folder tempat file audio berada di Mac Anda.
- **Hapus dari Playlist** — menghapus trek dari playlist.
- **Hapus dari Layanan Cloud** — menghapus trek dari playlist dan layanan cloud terkait. **Tindakan ini tidak dapat dibatalkan.**
- **Hapus dari Perpustakaan Musik** — menghapus trek dari perpustakaan musik, membiarkan file di penyimpanan tidak tersentuh.

## Aksesibilitas

Flacbox dapat diakses sepenuhnya dengan teknologi **VoiceOver**, memastikan setiap komponen memiliki label dan deskripsi yang dirancang dengan baik. Saat VoiceOver aktif, aplikasi menerjemahkan antarmuka pengguna ke **Mode Teks**, hanya menampilkan elemen yang dapat diakses dan berguna untuk meningkatkan kecepatan navigasi dan kenyamanan. Anda juga dapat mengaktifkan Mode Teks di Pengaturan → Aksesibilitas → Mode Teks.

### Menyesuaikan Posisi Trek dalam Playlist dengan VoiceOver

1. Buka playlist dan ketuk tombol **Lainnya**.
2. Pilih **Ubah Urutan Lagu**. Tampilan beralih ke mode pengeditan.
3. Ketuk ikon indikator pengurutan ulang di dekat judul trek untuk memberinya fokus.
4. **Ketuk dua kali** ikon indikator pengurutan ulang dengan cepat. Pada ketukan kedua, jangan lepaskan jari Anda — tahan hingga Anda mendengar suara yang menunjukkan sel siap dipindahkan.
5. Sekarang, Anda dapat memindahkan sel ke posisi baru.

Komponen lain berfungsi seperti yang diharapkan, menggunakan pola VoiceOver yang disediakan sistem.

Flacbox는 기본 음악 앱과 매우 유사한 직관적인 사용자 인터페이스를 제공하지만, 오디오 파일을 편집하고, 폴더로 구성하고, 앱을 벗어나지 않고도 클라우드 스토리지와 기기 간에 원활하게 이동할 수 있는 진정한 내장 파일 관리자를 추가합니다.

Flacbox의 기능은 두 가지 논리적 섹션으로 나뉩니다: 음악 라이브러리 및 재생 목록 탭을 통해 표시되는 음악 라이브러리와, 연결하기 및 로컬 파일 탭을 통해 표시되는 파일 섹션. 음악 라이브러리는 트랙을 메타데이터 인식 미디어 항목(아티스트 / 앨범 / 장르 / 작곡가 계층 구조)으로 처리하며, 파일 섹션은 해당 항목 뒤에 있는 원시 오디오 파일에 대한 완전한 파일 및 폴더 제어를 제공합니다.

## 섹션

iPhone을 사용하든, iPad를 사용하든, Mac의 컴팩트 창 모드를 사용하든, 모든 앱 기능은 화면 하단의 탭 바에서 접근할 수 있습니다. iPad와 Mac에서는 동일한 메뉴가 왼쪽 사이드바에서 사용할 수 있어 콘텐츠에 더 많은 화면 공간을 확보할 수 있습니다. 이 체계적인 구성은 모든 기능을 명확하게 이름 붙여진 섹션으로 분류하여 50개 트랙 라이브러리나 50,000개 트랙 라이브러리를 탐색하더라도 사용자 친화적인 경험을 보장합니다.

- **연결하기** — Google Drive, MEGA, OneDrive, Dropbox, Box, pCloud, Yandex Disk, Synology Drive 등 클라우드 스토리지 서비스와 SMB, WebDAV 또는 DLNA를 통한 컴퓨터나 NAS에 연결합니다.
- **음악 라이브러리** — 아티스트, 앨범 아티스트, 앨범, 장르, 작곡가, 곡, 미재생 곡으로 그룹화되고, 최근 항목, 즐겨찾기, 북마크를 위한 전용 섹션이 있는 라이브러리의 모든 트랙을 깔끔하게 표시합니다. 수동으로 곡을 추가하거나 온라인 동기화를 통해 자동으로 표시되도록 할 수 있습니다.
- **재생 목록** — 재생 목록 만들기, 이름 바꾸기, 커버 아트 편집, 곡 재정렬, 오프라인 모드 활성화, M3U / M3U8 / CUE 파일 가져오기, M3U / CSV / TXT로 내보내기, 플레이어 큐에 곡 추가 등 모든 재생 목록을 한 곳에서 관리합니다.
- **로컬 파일** — 다운로드된 파일을 찾고 관리하며, 전송 큐에 대한 완전한 제어권을 갖습니다. 전송 큐를 열려면 로컬 파일 화면의 왼쪽 상단 모서리에 있는 회전 화살표 아이콘을 탭합니다. 로컬 파일 화면은 이 애플리케이션의 파일(앱의 샌드박스 Documents 디렉토리)과 이 iPhone / iPad / Mac의 파일(Apple의 시스템 파일 선택기를 통해 액세스하는 기기의 다른 폴더)로 분류됩니다.
- **설정** — 오디오 엔진 및 코덱, 이퀄라이저, 음악 라이브러리 동기화, 파일 전송, 앨범 아트워크 캐시, 홈 화면 위젯, CarPlay, 사용자 인터페이스, 언어, 비밀번호, 백업 및 복원을 포함한 애플리케이션 설정을 수정합니다.
- **빠른 액세스** — 즐겨찾기 및 최근에 재생한 트랙과 파일로 이동합니다. Mac과 iPad에서 빠른 액세스는 전용 섹션이 있으며, iPhone에서는 로컬 파일, 연결하기, 음악 라이브러리 화면 상단에서 찾을 수 있습니다.

## iPhone의 미니 플레이어

iPhone 화면 하단의 미니 플레이어 바를 탭하면 전체 화면 플레이어로 확장되고, 아래로 스와이프하면 다시 접힙니다. iPad와 Mac에서 미니 플레이어는 창의 상단에 있으며 메인 메뉴에서 전체 화면 플레이어를 열면 숨길 수 있습니다. 미니 플레이어는 현재 아트워크, 제목, 아티스트 및 기본 재생 컨트롤을 표시하므로 일시 중지하거나 트랙을 건너뛰기 위해 탐색 중인 것을 떠날 필요가 없습니다.

## 미니 플레이어 창 (Mac 전용)

macOS에서는 Flacbox를 다른 앱 위에 항상 떠 있는 컴팩트한 미니 플레이어로 축소할 수 있습니다 — 작업하는 동안 현재 재생 중인 트랙을 볼 수 있어 완벽합니다.

1. 커서를 Flacbox 창의 오른쪽 하단 모서리로 이동하고 최소 크기로 줄입니다.
2. **접기 버튼**(아래 방향 화살표)을 탭하여 미니 플레이어 창으로 전환합니다.
3. 다른 모든 앱 위에 유지하려면 macOS 메뉴 바에서 창 → 창을 항상 위에 표시를 선택합니다.

이것은 읽거나 쓰는 동안 오디오 강의, 팟캐스트 또는 오디오북을 듣기에 좋습니다 — Flacbox는 방해가 되지 않지만 항상 재생 컨트롤에서 클릭 한 번이면 됩니다.

## 추가 작업

화면의 거의 모든 콘텐츠 항목에는 **추가 작업** 버튼("⋯" 점 세 개 아이콘)이 있습니다. 이것을 탭하면 해당 항목에 대해 사용 가능한 모든 작업(다음 재생, 나중에 재생, 라이브러리 또는 재생 목록에 추가, 태그 편집, 다운로드, 공유, 이름 바꾸기, 이동 등)이 있는 컨텍스트 감지 메뉴가 열립니다.

## 상단 도구 모음

탐색 바 바로 아래에 있는 상단 도구 모음은 가장 일반적인 화면별 작업에 빠르게 액세스할 수 있습니다. 아래로 스와이프하여 표시하거나 숨길 수 있습니다.

- **검색** — 현재 화면(폴더, 재생 목록, 앨범, 아티스트, 장르, 라이브러리 또는 큐) 내에서 검색을 시작합니다.
- **계속 재생** — Flacbox는 마지막으로 저장된 위치에서 현재 디렉토리, 앨범, 아티스트 또는 재생 목록에 대한 오디오 플레이어의 상태를 복원할 수 있습니다. 이 옵션은 설정 → 오디오 플레이어 → 일반에서 오디오 플레이어 상태 저장이 활성화된 경우에만 나타납니다.
- **모두 재생** — 현재 정렬 순서를 유지하면서 현재 페이지의 모든 트랙을 플레이어 큐에 추가합니다.
- **모두 셔플** — 현재 페이지의 모든 트랙을 섞어서 플레이어 큐에 추가합니다.

## 컨텍스트 메뉴

컨텍스트 메뉴는 기기 전체에서 원활한 상호 작용을 위한 추가 옵션과 작업에 빠르게 액세스할 수 있습니다.

### iOS / iPadOS의 컨텍스트 메뉴

셀, 미니 플레이어 또는 컴팩트 플레이어를 탭하고 있으면 컨텍스트 메뉴가 표시됩니다. 설정 → 개인화 → 컨텍스트 메뉴 스타일에서 컨텍스트 메뉴 표시 스타일을 변경할 수 있습니다.

### macOS의 컨텍스트 메뉴

셀, 미니 플레이어 또는 컴팩트 플레이어를 마우스 오른쪽 버튼으로 클릭하면 컨텍스트 메뉴가 표시됩니다. 전체 화면 플레이어의 앨범 아트워크에서도 마우스 오른쪽 버튼 클릭이 작동합니다.

## 홈 화면 위젯 (iPhone 및 iPad)

Flacbox는 iOS 홈 화면 및 잠금 화면 위젯을 지원하므로 현재 재생 중인 트랙을 한눈에 볼 수 있습니다.

1. **설정 → 위젯 → 위젯 활성화**에서 **위젯**이 활성화되어 있는지 확인합니다(위젯 업데이트는 소량의 에너지를 사용하므로 기본적으로 꺼져 있습니다).
2. **홈 화면** 또는 **잠금 화면**의 빈 공간을 길게 탭합니다.
3. 상단 모서리의 **+** 버튼을 탭하고 **Flacbox**를 검색합니다.
4. 위젯 크기를 선택하고 **위젯 추가**를 탭합니다.

위젯은 현재 아트워크, 트랙 제목, 아티스트를 표시하며 전체 화면 플레이어로 직접 탭할 수 있습니다. 위젯은 **macOS**의 알림 센터에서도 작동합니다.

## Apple CarPlay

Flacbox는 안전한 차내 사용을 위해 설계된 간소화된 인터페이스로 Apple CarPlay를 완벽하게 지원합니다. Lightning / USB-C 케이블 또는 무선 CarPlay로 iPhone을 연결하면 Flacbox 아이콘이 CarPlay 홈 화면에 나타납니다. CarPlay UI에는 라이브러리, 연결하기, 로컬 파일, 설정을 위한 전용 탭이 포함되어 있으므로 전화기를 들지 않고도 클라우드, 오프라인 또는 로컬 전체 음악 컬렉션을 완전히 제어할 수 있습니다.

설정 → CarPlay에서 CarPlay 레이아웃을 세밀하게 조정할 수 있습니다 — 정렬 옵션, 페이지 매김, 메뉴 아이콘 그라데이션 색상, 앨범 아트 표시 여부, CarPlay에 연결되는 순간 재생을 자동으로 일시 중지하는 옵션.

## 접근성

Flacbox는 **VoiceOver**로 완벽하게 접근할 수 있습니다: 모든 구성 요소에 설명적인 레이블과 적절한 접근성 힌트가 있습니다. VoiceOver가 활성화되면 앱이 자동으로 **텍스트 모드**로 전환됩니다 — 이미지가 많은 장식적 요소가 숨겨지고 의미 있는 컨트롤만 표시되어 화면 리더 사용자의 탐색 속도와 명확성이 향상됩니다. 설정 → 접근성 → 텍스트 모드에서 수동으로 텍스트 모드를 활성화할 수도 있습니다.

### VoiceOver로 슬라이더 조정

1. **슬라이더 선택** — VoiceOver가 알릴 때까지 왼쪽이나 오른쪽으로 스와이프합니다.
2. **값 조정** — 슬라이더를 두 번 탭하고 길게 누른 다음 위나 아래로 드래그하여 빠르게 값을 변경합니다. VoiceOver가 진행하는 동안 각 새 값을 알립니다.

### VoiceOver로 재생 목록에서 트랙 위치 조정

1. 재생 목록을 열고 **더보기** 버튼을 탭합니다.
2. **곡 순서 변경**을 선택합니다. 목록이 편집 모드로 전환됩니다.
3. 트랙 제목 근처의 재정렬 표시기 아이콘을 탭하여 포커스를 줍니다.
4. 재정렬 표시기를 **두 번 탭**합니다. 두 번째 탭 시 손가락을 떼지 않고 셀이 이동 준비가 되었다는 소리가 들릴 때까지 누르고 있습니다.
5. 이제 셀을 새 위치로 이동할 수 있습니다.

다른 구성 요소는 시스템에서 제공하는 VoiceOver 패턴을 사용하여 예상대로 작동합니다.

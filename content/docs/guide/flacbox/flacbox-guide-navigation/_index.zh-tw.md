Flacbox 提供直覺的使用者介面，與原生音樂應用程式非常相似，但增加了真正的內建檔案管理器，讓您可以編輯音訊檔案、將其整理到資料夾中，以及在雲端儲存空間和裝置之間無縫移動——無需離開應用程式。

Flacbox 的功能被巧妙地分為兩個邏輯部分：透過「音樂庫」和「播放清單」標籤頁展示的音樂庫，以及透過「連接」和「本機檔案」標籤頁展示的檔案部分。

## 各部分

無論您使用的是 iPhone、iPad 還是 Mac 上的緊湊視窗模式，所有應用程式功能都可以從畫面底部的標籤列存取。在 iPad 和 Mac 上，同樣的選單在左側側邊欄也可用。這種周全的組織將每個功能分類到清晰命名的部分中。

- **連接** — 連接 Google Drive、MEGA、OneDrive、Dropbox、Box、pCloud、Yandex Disk、Synology Drive 等雲端儲存服務，以及透過 SMB、WebDAV 或 DLNA 連接您的電腦或 NAS。
- **音樂庫** — 整齊地顯示音樂庫中的每首曲目，按藝術家、專輯藝術家、專輯、類型、作曲家、歌曲和未播放歌曲分組，另有最近使用的、最愛項目和書籤的專屬部分。
- **播放清單** — 在一處管理所有播放清單：建立、重新命名、編輯封面、重新排序歌曲、啟用離線模式、匯入 M3U / M3U8 / CUE、匯出為 M3U / CSV / TXT。
- **本機檔案** — 尋找和管理已下載的檔案，完全控制傳輸佇列。
- **設定** — 修改應用程式設定，包括音訊引擎和編解碼器、等化器、音樂庫同步、檔案傳輸、專輯封面快取、主畫面小工具、CarPlay、語言、密碼以及備份與還原。
- **快速存取** — 跳轉到您最愛的和最近播放的曲目和檔案。

## iPhone 上的迷你播放器

點擊 iPhone 畫面底部的迷你播放器列，將其展開為全螢幕播放器；向下滑動可將其收起。在 iPad 和 Mac 上，迷你播放器位於視窗頂部。迷你播放器顯示當前專輯封面、標題、藝術家和基本播放控制項。

## 迷你播放器視窗（Mac 專屬）

在 macOS 上，您可以將 Flacbox 縮小為緊湊的始終置頂迷你播放器，懸浮在其他應用程式之上。

1. 將游標移到 Flacbox 視窗的右下角，將其縮小至最小尺寸。
2. 點擊**折疊按鈕**（向下箭頭）切換到迷你播放器視窗。
3. 要將其保持在所有其他應用程式之上，從 macOS 選單列選擇視窗 → 始終置頂顯示視窗。

## 更多操作

畫面上幾乎每個內容項目都有**更多操作**按鈕（「⋯」三點圖示）。點擊它可開啟情境選單，包含該項目可用的所有操作。

## 頂部工具列

頂部工具列緊靠導覽列下方，提供對最常見的每畫面操作的快速存取。您可以透過簡單的向下輕掃手勢顯示或隱藏它。

- **搜尋** — 在當前畫面中開始搜尋。
- **繼續播放** — Flacbox 可以從上次儲存的位置恢復音訊播放器狀態。僅當「設定 → 音訊播放器 → 一般」中啟用了「儲存音訊播放器狀態」時才會出現此選項。
- **播放全部** — 將當前頁面的每首曲目新增到播放器佇列，保持當前排序順序。
- **隨機播放全部** — 將當前頁面的每首曲目新增到播放器佇列，隨機排列。

## 情境選單

情境選單提供對附加選項的快速存取。

### iOS / iPadOS 上的情境選單

在儲存格、迷你播放器或緊湊播放器上輕觸並按住以顯示情境選單。您可以在「設定 → 個人化 → 情境選單樣式」中更改情境選單的呈現樣式。

### macOS 上的情境選單

在儲存格、迷你播放器或緊湊播放器上按一下滑鼠右鍵以顯示情境選單。

## 主畫面小工具（iPhone & iPad）

Flacbox 支援 iOS 主畫面和鎖定畫面小工具。

1. 確保在**設定 → 小工具 → 啟用小工具**中啟用了**小工具**。
2. 長按**主畫面**或**鎖定畫面**的空白區域。
3. 點擊頂角的 **+** 按鈕並搜尋 **Flacbox**。
4. 選擇小工具尺寸並點擊**新增小工具**。

小工具顯示當前專輯封面、曲目標題和藝術家，並可直接點擊進入全螢幕播放器。小工具在 **macOS** 通知中心也可用。

## Apple CarPlay

Flacbox 完全支援 Apple CarPlay，具有專為安全駕駛設計的精簡介面。透過 Lightning / USB-C 連接線或無線 CarPlay 連接您的 iPhone，Flacbox 圖示將出現在 CarPlay 主畫面上。CarPlay 介面包括用於音樂庫、連接、本機檔案和設定的專用標籤頁。

您可以在「設定 → CarPlay」中微調 CarPlay 佈局——排序選項、分頁、選單圖示漸層顏色、是否顯示專輯封面，以及在連接 CarPlay 時自動暫停播放的選項。

## 輔助功能

Flacbox 與 **VoiceOver** 完全相容：每個元件都有描述性標籤和合理的輔助功能提示。當 VoiceOver 啟動時，應用程式自動切換到**文字模式**——隱藏大量圖片的裝飾元素，只顯示有意義的控制項。您也可以在「設定 → 輔助功能 → 文字模式」中手動啟用文字模式。

### 使用 VoiceOver 調整滑桿

1. **選擇滑桿** — 向左或向右滑動直到 VoiceOver 宣讀它。
2. **調整值** — 快速按兩下並按住滑桿，然後向上或向下拖曳以快速更改值。

### 使用 VoiceOver 調整播放清單中的曲目位置

1. 開啟播放清單並點擊**更多**按鈕。
2. 選擇**更改歌曲順序**。清單切換到編輯模式。
3. 點擊曲目標題旁的重新排序指示器圖示使其獲得焦點。
4. 快速**按兩下**重新排序指示器。第二次點擊時，不要放開手指——保持按住直到聽到聲音指示儲存格可以移動。
5. 現在可以將儲存格移動到新位置。

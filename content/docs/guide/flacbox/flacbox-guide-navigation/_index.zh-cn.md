Flacbox 提供直观的用户界面，与原生音乐应用非常相似，但增加了真正的内置文件管理器，让您可以编辑音频文件、将其整理到文件夹中，以及在云存储和设备之间无缝移动——无需离开应用。

Flacbox 的功能被巧妙地分为两个逻辑部分：通过「音乐库」和「播放列表」标签页展示的音乐库，以及通过「连接」和「本地文件」标签页展示的文件部分。

## 各部分

无论您使用的是 iPhone、iPad 还是 Mac 上的紧凑窗口模式，所有应用功能都可以从屏幕底部的标签栏访问。在 iPad 和 Mac 上，同样的菜单在左侧边栏也可用。这种周全的组织将每个功能分类到清晰命名的部分中。

- **连接** — 连接 Google Drive、MEGA、OneDrive、Dropbox、Box、pCloud、Yandex Disk、Synology Drive 等云存储服务，以及通过 SMB、WebDAV 或 DLNA 连接您的电脑或 NAS。
- **音乐库** — 整洁地显示库中的每首曲目，按艺术家、专辑艺术家、专辑、流派、作曲家、歌曲和未播放歌曲分组，另有最近使用、收藏夹和书签的专属部分。
- **播放列表** — 在一处管理所有播放列表：创建、重命名、编辑封面、重新排序歌曲、启用离线模式、导入 M3U / M3U8 / CUE、导出为 M3U / CSV / TXT。
- **本地文件** — 查找和管理已下载的文件，完全控制传输队列。
- **设置** — 修改应用程序设置，包括音频引擎和编解码器、均衡器、音乐库同步、文件传输、专辑封面缓存、主屏幕小组件、CarPlay、语言、密码以及备份与恢复。
- **快速访问** — 跳转到您收藏的和最近播放的曲目和文件。

## iPhone 上的迷你播放器

点击 iPhone 屏幕底部的迷你播放器栏，将其展开为全屏播放器；向下滑动可将其收起。在 iPad 和 Mac 上，迷你播放器位于窗口顶部。迷你播放器显示当前专辑封面、标题、艺术家和基本播放控件。

## 迷你播放器窗口（Mac 专属）

在 macOS 上，您可以将 Flacbox 缩小为紧凑的始终置顶迷你播放器，悬浮在其他应用之上。

1. 将光标移到 Flacbox 窗口的右下角，将其缩小至最小尺寸。
2. 点击**折叠按钮**（向下箭头）切换到迷你播放器窗口。
3. 要将其保持在所有其他应用之上，从 macOS 菜单栏选择窗口 → 始终置顶显示窗口。

## 更多操作

屏幕上几乎每个内容项都有**更多操作**按钮（「⋯」三点图标）。点击它可打开上下文敏感菜单，包含该项目可用的所有操作。

## 顶部工具栏

顶部工具栏紧靠导航栏下方，提供对最常见的每屏操作的快速访问。您可以通过简单的向下滑动手势显示或隐藏它。

- **搜索** — 在当前屏幕中开始搜索。
- **继续播放** — Flacbox 可以从上次保存的位置恢复音频播放器状态。仅当「设置 → 音频播放器 → 一般」中启用了「保存音频播放器状态」时才会出现此选项。
- **播放全部** — 将当前页面的每首曲目添加到播放器队列，保持当前排序顺序。
- **随机播放全部** — 将当前页面的每首曲目添加到播放器队列，随机排列。

## 上下文菜单

上下文菜单提供对附加选项的快速访问。

### iOS / iPadOS 上的上下文菜单

在单元格、迷你播放器或紧凑播放器上轻触并按住以显示上下文菜单。您可以在「设置 → 个性化 → 上下文菜单样式」中更改上下文菜单的呈现样式。

### macOS 上的上下文菜单

在单元格、迷你播放器或紧凑播放器上右键单击以显示上下文菜单。

## 主屏幕小组件（iPhone & iPad）

Flacbox 支持 iOS 主屏幕和锁屏小组件。

1. 确保在**设置 → 小组件 → 启用小组件**中启用了**小组件**。
2. 长按**主屏幕**或**锁屏**的空白区域。
3. 点击顶角的 **+** 按钮并搜索 **Flacbox**。
4. 选择小组件尺寸并点击**添加小组件**。

小组件显示当前专辑封面、曲目标题和艺术家，并可直接点击进入全屏播放器。小组件在 **macOS** 通知中心也可用。

## Apple CarPlay

Flacbox 完全支持 Apple CarPlay，具有专为安全驾驶设计的精简界面。通过 Lightning / USB-C 线缆或无线 CarPlay 连接您的 iPhone，Flacbox 图标将出现在 CarPlay 主屏幕上。CarPlay 界面包括用于音乐库、连接、本地文件和设置的专用标签页。

您可以在「设置 → CarPlay」中微调 CarPlay 布局——排序选项、分页、菜单图标渐变颜色、是否显示专辑封面，以及在连接 CarPlay 时自动暂停播放的选项。

## 辅助功能

Flacbox 与 **VoiceOver** 完全兼容：每个组件都有描述性标签和合理的辅助功能提示。当 VoiceOver 激活时，应用程序自动切换到**文字模式**——隐藏大量图片的装饰元素，只显示有意义的控件。您也可以在「设置 → 辅助功能 → 文字模式」中手动启用文字模式。

### 使用 VoiceOver 调整滑块

1. **选择滑块** — 向左或向右滑动直到 VoiceOver 宣读它。
2. **调整值** — 双击并按住滑块，然后向上或向下拖动以快速更改值。

### 使用 VoiceOver 调整播放列表中的曲目位置

1. 打开播放列表并点击**更多**按钮。
2. 选择**更改歌曲顺序**。列表切换到编辑模式。
3. 点击曲目标题旁的重新排序指示器图标使其获得焦点。
4. 快速**双击**重新排序指示器。第二次点击时，不要松开手指——保持按住直到听到声音指示单元格可以移动。
5. 现在可以将单元格移动到新位置。

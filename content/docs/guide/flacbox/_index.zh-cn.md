### 欢迎使用 Flacbox 指南

Flacbox 是一款适用于 iPhone、iPad 和 Mac 的高分辨率音乐播放器，可让您将喜爱的云存储、NAS 和媒体服务器变成专属的个人流媒体服务。

Flacbox 可连接极为丰富的音源，一切尽在一款应用中：

- **个人云存储：** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · pCloud · Yandex Disk · MediaFire · WD My Cloud Home · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘).
- **自托管服务器：** Plex · Jellyfin · Emby · Subsonic（及所有 Subsonic 兼容服务器——Airsonic、Funkwhale、Gonic、Logitech Media Server、Ampache）· Navidrome · Nextcloud（以及通过共享 API 的 ownCloud）· Synology Drive · QNAP.
- **NAS 和文件共享协议：** SMB（SMB1、SMB2、Auto）· WebDAV（HTTP / HTTPS）· FTP / FTPS · SFTP（密码或公钥认证）· NFS · DLNA / UPnP（播放和下载）。兼容 Apple Time Capsule、Synology、QNAP、WD My Cloud 以及任何 Linux Samba / NFS / SSH 主机或 Mac / Windows PC 上的共享文件夹。
- **S3 兼容对象存储：** AWS S3、Backblaze B2、Wasabi、Cloudflare R2、MinIO、DigitalOcean Spaces 及任何其他 S3-API 端点。
- **局域网发现：** 「可用设备」区域自动列出 Wi-Fi 上的所有 Bonjour / mDNS 服务——Plex、Jellyfin、Emby、Synology、QNAP、带外置硬盘的 AirPort 路由器、Time Capsule——无需输入 IP 地址，轻触即可连接。

大多数音乐应用仅使用 Apple 内置音频引擎，而 Flacbox 在系统编解码器的基础上还集成了 **FFmpeg**，可播放 iOS 默认不支持的格式——WMA、OGG、OPUS、APE、DSD、DSF、DFF、TTA、MPC、WV，以及常见的 MP3 / AAC / M4A / WAV / AIFF / ALAC / FLAC。结合可配置的音频输出采样率（44.1 / 48 / 64 / 88.2 / 96 kHz）、多声道输出（Mono → 5.1 → SDDS → ITU BS.775-1）、IO 缓冲调节和音调校正，Flacbox 为发烧友提供了大多数消费类音乐应用所没有的精细控制。

### 享受流畅的流媒体和离线播放

Flacbox 具备智能缓冲，可确保流畅的流媒体播放，同时内置下载管理器，让您可将整个播放列表、艺术家、专辑、文件夹或单曲下载到设备以供离线使用。

### 自动整理的音乐库

导入音轨时，Flacbox 会扫描其元数据并整理成清晰的音乐库——按歌曲、未播放歌曲、专辑、专辑艺术家、艺术家、流派和作曲家分组。

### 修复元数据并为音乐打标签

内置 ID3 标签编辑器可修复损坏的标签——手动或通过自动 MusicBrainz 查找。还可以规范化字符编码、搜索缺失的专辑封面，并自动将更改写回云端的原始文件。

### 轻松从 Mac、PC 或 NAS 传输

通过以下工具在电脑与 iPhone 或 iPad 之间移动音乐：SMB、WebDAV、DLNA、Wi-Fi Drive（在任意桌面浏览器中拖放）、通过 Lightning 或 USB-C 线缆进行 iTunes / Finder 文件共享、USB 闪存驱动器或 iXpand 闪存驱动器。

### 通过均衡器自定义音效

Flacbox 内置 10 段均衡器，提供 iPod 风格的预设（Acoustic、Bass Booster、Classical、Dance、Rock、Pop、Jazz 等）、前置放大器和保存自定义预设的功能。

### 专为有声书设计

Flacbox 是出色的有声书播放器——每首曲目支持多个书签、精细的播放速度调节（0.02× 至 3.00×）、从停止处精确恢复播放、可自定义跳过时间按钮以及渐出睡眠定时器。完整支持 M4B 章节标记和长文件。

### 随时随地流媒体——包括车内和主屏幕

通过 AirPlay 2 向 Apple TV / HomePod 流媒体，发送到 Google Chromecast 和支持 Cast 的电视，并通过 Apple CarPlay 在路上享受音乐。

### 隐私与安全

Flacbox 仅使用每家云提供商的官方 SDK 和 OAuth 登录——您的密码永远不会传输到应用。访问令牌加密存储在 iOS Keychain 中，所有传输均受 TLS 保护。

### Flacbox 入门

本指南将带您了解 Flacbox 在 iPhone、iPad 和 Mac 上的每个功能——从连接云服务、整理音乐库、传输文件和管理播放列表，到微调均衡器和配置音频引擎。

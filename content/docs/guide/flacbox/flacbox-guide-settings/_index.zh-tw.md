設定畫面是 Flacbox 的控制中心。從這裡您可以升級到 Premium、配置音訊引擎（系統編解碼器或 FFmpeg）、管理音樂庫、設定檔案管理員、自訂音訊標籤編輯器、啟用主畫面小工具和 Apple CarPlay、備份資料，以及存取說明和法律資訊。各部分按以下標題分組：購買與更新、應用程式偏好、說明以及法律與隱私。

將應用程式升級到 Premium 版本以移除所有限制。免費版本提供一次性終身應用程式內購買和兩種訂閱選項（1 個月和 1 年），以移除所有限制並升級到 Premium。

## 在 iOS 和 Mac 之間共享購買

終身購買和訂閱在 iOS 和 Mac 之間共享，使用 iCloud 同步此資訊。如果您在 iOS 裝置上有 Premium 版本，請確保您安裝了最新版本並且 iCloud 已啟用。在 iOS 上啟動應用程式並等待一分鐘，讓您的購買資訊上傳到 iCloud。

您也可以嘗試點擊應用程式設定中的恢復購買按鈕。之後，在 Mac 上從 App Store 安裝最新版本並啟動應用程式。確保您有網路連線，並且在 Mac 上使用與 iOS 相同的 iCloud 和 App Store 帳號。等待一分鐘讓應用程式從 iCloud 下載購買資訊——Premium 版本應該會在 Mac 上自動啟用。

## 在新裝置上恢復購買

要在新裝置上恢復購買，請使用購買 → 恢復購買選單。您將看到您的購買清單。如果您沒有看到所有購買，請確認裝置連接到了進行購買時使用的相同 Apple ID，並且 iCloud 已啟用。

## 免費試用 Premium

您可以使用此選單在限定時間內免費升級到 Premium 版本。只需觀看廣告或向朋友介紹應用程式即可免費獲得 Premium。

## 購買

您可以從此選單恢復之前的購買。如果遇到啟用錯誤，請嘗試啟用「啟動時恢復購買」選項。

## 軟體更新

點擊軟體更新以檢查是否有更新版本的 Flacbox 可用。應用程式將比較您安裝的版本與 App Store 上的最新版本，並告知您是否建議更新。

## 新功能

此選單在新版本發佈後可用。它顯示最新更新中的變更和新功能摘要。

## 音訊播放器

此部分配置音訊播放器和底層音訊引擎，包括 FFmpeg / 系統編解碼器選擇和 Hi-Res 音訊輸出選項。

### 一般

這些設定涵蓋音訊播放器的基本方面——播放佇列、音訊輸出和儲存播放器狀態。

- **重複模式** — 選擇曲目結束時音訊播放器的行為：
  - **全部重複** — 重播佇列中的所有曲目。
  - **單曲重複** — 僅重複當前曲目。
  - **重複停止** — 當前曲目結束時暫停播放。
  - **不重複** — 允許佇列不重複地播放一遍。
- **隨機模式** — 隨機排列佇列中曲目的順序。您可以**關閉隨機**或**開啟隨機**。
- **音訊編解碼器** — 選擇播放使用的音訊引擎：
  - **系統編解碼器 + FFmpeg** — 盡可能優先使用系統音訊編解碼器，增強相容性和穩定性。音調校正和音訊輸出採樣率可能受限。
  - **FFmpeg** — 對所有音訊檔案強制使用 FFmpeg 編解碼器，解鎖音調校正和音訊輸出採樣率。
- **音訊輸出採樣率** — 調整音訊輸出採樣率以最佳化音質，特別適合外接 DAC。可用值：**44.1 kHz、48 kHz、64 kHz、88.2 kHz** 和 **96 kHz**。
- **音訊輸出聲道數** — 對於帶外接 DAC 的多聲道音訊系統，指定聲道數：單聲道、立體聲、中 / 左 / 右、中 / 左 / 右 / 環繞、ITU BS.775-1、5.1 環繞聲和 SDDS。
- **音訊輸出首選 IO 緩衝時長** — 配置輸入 / 輸出緩衝時長。播放高解析度音訊時最小化延遲的典型值約為 **5 毫秒（0.005 秒）**。在目標裝置上測試不同時長以找到低延遲和無故障播放之間的最佳平衡。
- **音訊輸出模式（僅 iOS）** — 配置混合音訊輸出模式，使 Flacbox 的音訊與其他應用程式混合。詳細說明[在這裡](/docs/howto/how-to-record-video-while-playing-music-on-iphone)。
- **儲存播放位置** — 儲存和恢復音樂庫中歌曲的播放位置。
- **儲存音訊播放器狀態** — 在關閉應用程式前保留音訊播放器的狀態，方便從上次停止的地方繼續。

啟用**儲存播放位置**和**儲存音訊播放器狀態**後，開啟任意資料夾、專輯、藝術家或類型，您將在畫面頂部看到**繼續播放**按鈕。

### 個人化

個人化允許您自訂音訊播放器畫面的外觀，更改主畫面、鎖定畫面和狀態列上的可用控制項，並配置跳過時間控制項。

- **音訊播放器畫面樣式** — 配置音訊播放器畫面上元素的定位。
- **專輯封面捲動樣式** — 配置專輯封面的首選捲動樣式。
- **附加元素** — 在音訊播放器畫面上啟用額外元素。音訊格式資訊在封面上方顯示正在播放曲目的音訊資訊；音訊音量滑桿在播放控制項下方顯示音訊輸出滑桿。
- **主畫面操作** — 配置預設情況下主音訊播放器畫面上應顯示哪些按鈕：重複和隨機模式、下一首和上一首、跳過時間、睡眠定時器、Google Chromecast、AirPlay 和藍牙、音訊等化器、搜尋、書籤、速度、新增書籤、新增到最愛、評論等。
- **鎖定畫面上的播放控制項** — 選擇鎖定畫面上顯示哪些控制項。可能的值：跳過時間、新增書籤、新增到最愛。
- **跳過時間按鈕** — 選擇跳過時間按鈕的時間間隔。

### 檔案載入

在檔案載入過程中，您可以更改用於載入歌曲的網路類型。可用選項：**Wi-Fi**、**Wi-Fi 和行動數據**。

- **預載入時間** — 設定緩衝時間間隔。如果網路連線較差，請增加此值。
- **使用直接 URL** — 啟用後，如果伺服器支援，將使用直接 URL 載入歌曲。這可以加快歌曲載入速度，但可能影響播放穩定性。

### 音訊等化器

配置 10 段音訊等化器、預設和前置放大器。您可以在[這裡](/docs/howto/how-to-use-the-audio-equalizer-on-your-iphone-ipad-mac-with-evermusic-and-flacbox)閱讀更多關於配置音訊等化器的內容。

### 播放速度

將音訊播放器的播放速度從 **0.02× 調整到 3.00×**。點擊右上角的設定圖示切換到**精確模式**以進行更精細的調整。

### 音調校正

使用預定義值更改音調校正設定，或透過點擊右上角的按鈕切換到**精確模式**。音調校正在 FFmpeg 編解碼器路徑中可用，範圍從 **-1000 到 +1000**。

### 播放快取

音訊播放器佇列中的歌曲會自動下載以確保流暢播放。如果您手動下載音訊檔案，可以停用此選項以避免重複。您也可以在此配置音訊播放器快取大小。

### 睡眠定時器

啟用定時器以在指定時間後自動停止播放。點擊右上角的設定圖示以**精確模式**進行分鐘級精細控制。

## 音樂庫

音樂庫設定——自動同步、元資料讀取、封面載入、播放清單、最近和最愛——位於此處。

### 元資料讀取

當您將曲目新增到音樂庫時，**元資料讀取器**開始工作。此背景程序讀取曲目中的所有元資料，並按藝術家、專輯、類型和作曲家進行整理。您可以調整元資料讀取速度以更快地載入資料（以消耗更多電量為代價）。您也可以停用元資料讀取器，改為顯示檔名而非標籤資訊。

元資料讀取器**僅更新音樂庫中的元資料**，不會修改儲存在雲端帳號或本機儲存中的檔案。要編輯音訊檔案本身的元資料，請透過選項選單中的相應操作使用內建的**標籤編輯器**。

當**背景元資料讀取**開啟時，讀取器在背景模式下繼續工作。如果應用程式在音訊播放期間消耗過多能量，iOS 可能會暫停它。

如果您有大型音樂收藏，請在應用程式的桌面版本上執行元資料同步，並使用**備份與還原**將同步的音樂庫傳輸到 iOS。

當啟用**規範化元資料編碼**時，應用程式會自動規範化所有歌曲的元資料編碼。這可以修復損壞的標籤編碼（例如在 Windows PC 上編輯檔案後），並防止錯誤字元出現在曲目資訊中。

**重新載入元資料**將音樂庫中的每個檔案標記為缺少元資料，這會導致元資料讀取器重新整理每筆記錄。

**開始元資料讀取**手動觸發元資料讀取器。進度顯示在操作下方。

### 線上同步

自動線上音樂同步會自動將來自連接的雲端服務的曲目新增到音樂庫。要啟用，請開啟音樂庫設定並選擇同步資料夾。

啟用此選項後，應用程式會掃描所選資料夾，識別支援的音訊檔案，並將其新增到您的音樂庫。使用相應的選單操作開始或停止同步。

線上同步僅在應用程式處於前台時執行，因此同步大型音樂庫可能需要一些時間。要加快速度，請保持 Flacbox 開啟、將裝置連接到電源，並啟用**設定 → 螢幕 → 始終活躍**。

或者，在應用程式的桌面版本上執行線上同步，並使用**備份與還原**將結果傳輸到 iOS。

您還可以選擇線上同步執行的頻率。將間隔設定為**立即**會在每次開啟應用程式時觸發同步。

### 離線同步

配置離線音樂同步。

#### 同步的離線資料夾

當您將雲端伺服器上的線上資料夾標記為可離線存取（使用**更多操作**選單）時，它會出現在此處。資料夾內容下載到**本機檔案 → 離線資料夾**。當線上資料夾發生變更（檔案新增、刪除或更新）時，應用程式會檢查變更並更新裝置上的本機副本。

在此畫面您可以手動啟動離線資料夾同步、顯示離線資料夾所在的資料夾，以及為資料夾停用離線模式。停用離線模式會從您的裝置中刪除所有檔案的本機副本。

#### 時間間隔

選擇應用程式檢查離線資料夾是否有修改的頻率。

#### 開始本機資料夾掃描

掃描應用程式**文件**目錄中的所有本機資料夾以尋找支援的音訊檔案。找到的檔案會自動新增到音樂庫。位於您裝置上但在應用程式文件目錄之外的檔案必須手動新增到音樂庫，因為由於 iOS / macOS 安全限制，應用程式無法存取它們。

**重要：**建議定期啟動離線音樂同步，以使音樂庫與本機檔案保持最新。

### 個人化

配置音樂庫畫面樣式。有三個選項：**簡單選單、分組選單、索引標籤式選單**。您還可以在專輯詳情畫面中啟用或停用專輯封面。

### 專輯封面

配置應用程式載入和儲存專輯封面的方式。

- **網路類型** — 選擇**Wi-Fi** 或 **Wi-Fi 和行動數據**用於封面下載。
- **為線上檔案載入專輯封面** — 啟用後，會為儲存在雲端儲存中的檔案載入嵌入的專輯封面。這可能使用額外的網路資料。
- **在資料夾中搜尋** — 啟用後，如果曲目沒有嵌入封面，應用程式會在同一資料夾中尋找 JPEG 或 PNG 圖片，並將其用作專輯封面。
- **封面品質** — 選擇儲存在裝置上的專輯封面的品質。
- **在資料夾中顯示** — 開啟專輯封面快取所在的資料夾。
- **全部刪除** — 刪除快取的專輯封面以釋放儲存空間，並強制應用程式按需重新載入。

### 播放清單

啟用將相同歌曲兩次新增到播放清單的選項。預設情況下，此選項被停用。

### 最近

管理您最近播放的歌曲清單。

- **刪除清單** — 刪除整個最近播放歌曲清單。
- **更改清單大小** — 設定清單中顯示的項目數量。
- **匯出歌曲清單** — 將最近播放的歌曲清單匯出為 M3U、CSV 或 TXT。詳細說明請見[這裡](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。

### 最愛

管理您喜愛的歌曲清單。

- **同時編輯** — 啟用後，將歌曲新增到最愛會同時在音樂庫和檔案部分中新增。
- **刪除清單** — 刪除整個最愛歌曲清單。
- **匯出歌曲清單** — 與最近類似，將最愛匯出為 M3U、CSV 或 TXT。

### 刪除音樂庫

清除音樂庫資料庫。您儲存中的音樂檔案保持不變。

## 密碼

啟用密碼畫面，使用 4 位數字密碼保護您的應用程式資料。啟用後，每次應用程式啟動時都會提示您輸入密碼。與裝置上的 iOS Face ID / Touch ID 結合使用可獲得額外保護。

## 檔案管理員

檔案管理員部分控制檔案的傳輸、儲存和預覽方式。

### 檔案傳輸

選擇將檔案下載到裝置時的網路偏好。

### 最大並行任務數

設定並行下載執行緒數。數量越多下載越快，但使用更多電量。

### 檔案傳輸任務

顯示目前活躍的上傳和下載任務。

### 背景傳輸

允許在應用程式處於背景時進行下載。如果背景傳輸消耗大量能量，iOS 可能會暫停應用程式。

### 將下載檔案儲存到

選擇預設下載目錄，或讓應用程式每次提示您。

### 同步的離線資料夾

管理所選資料夾的離線同步。要啟用離線同步，點擊資料夾旁邊的三點按鈕並選擇**可離線存取模式**。新增到雲端資料夾的新檔案會自動下載到您的裝置。了解更多關於離線模式的資訊[在這裡](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)。

### 時間間隔

選擇離線資料夾同步的頻率。**立即**會在每次開啟應用程式時觸發同步。

### 顯示完整檔名

在檔案管理員中顯示完整檔名，包括副檔名。

### 編輯線上檔案

停用線上檔案編輯以切換到連接的雲端服務的唯讀模式，防止意外刪除。此操作從使用者介面中移除檔案編輯操作。

### 開啟時複製檔案

指定應用程式如何處理從其他應用程式匯入的檔案。

### 檔案縮圖

管理並刪除產生的檔案縮圖以釋放儲存空間。

### 刪除暫存檔案

清除應用程式的快取資料夾以回收儲存空間。

## 音訊標籤編輯器

配置內建音訊標籤編輯器——方便批次修復雲端和本機檔案中的藝術家 / 專輯 / 年份 / 類型 / 封面問題。

### 專輯封面縮放

## 小工具

啟用小工具，使應用程式在播放期間更新小工具資料。小工具更新需要額外能量，因此預設情況下該開關處於關閉狀態——僅在您實際使用主畫面或鎖定畫面上的小工具時啟用。

您可以透過長按主畫面或鎖定畫面、點擊 **+**、搜尋 **Flacbox** 並選擇小工具大小來新增 Flacbox 小工具。小工具顯示目前封面、曲目標題和藝術家，並可直接點擊進入全螢幕播放器。小工具在 macOS 通知中心也可用。

## 備份與還原

使用此功能備份所有應用程式資料或將其遷移到另一台裝置。您可以選擇包含的內容：

- **資料庫** — 音樂庫中的所有曲目，包括播放清單。離線曲目不包含在內，以使備份檔案大小保持可管理。
- **專輯封面** — 所有快取的專輯封面。
- **設定** — 所有應用程式設定。

點擊**備份應用程式資料**開始備份。應用程式資料寫入單個檔案，您可以在以後用於在新裝置上或重新安裝應用程式後恢復應用程式狀態。

要在新裝置上恢復應用程式資料，請使用連接的雲端服務或 iCloud 將備份檔案移動到新裝置，然後在新裝置上開啟它。

詳細的逐步說明：[如何在裝置之間傳輸音樂庫：逐步指南](/docs/howto/how-to-transfer-your-music-library-between-devices-in-evermusic-step-by-step-guide)。

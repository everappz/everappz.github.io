Gestire la libreria musicale è semplicissimo con Flacbox, dove puoi organizzare facilmente tutte le tracce — FLAC locale, ALAC, DSD, MP3, M4A, OGG, WMA, APE e decine di altri formati — in un'unica collezione ricercabile. Hai due opzioni per costruire la libreria musicale: aggiunta manuale (scegli esattamente cosa aggiungere) o sincronizzazione automatica (Flacbox scansiona le cartelle cloud designate e aggiunge i nuovi file automaticamente man mano che appaiono).

## Aggiunta Manuale

Per aggiungere tracce manualmente, tocca l'icona **Aggiungi Musica** nell'angolo in alto a sinistra e scegli cartelle o file da un servizio cloud storage connesso o da file sul tuo dispositivo. Quando aggiungi tracce alla libreria, vengono creati solo link a quelle tracce — i file effettivi rimangono nelle loro posizioni originali per risparmiare spazio su disco.

Puoi anche trascinare e rilasciare file nella libreria nella versione Mac, o usare **Apri File…** / **Apri Cartella…** dal selettore file di sistema su iPhone e iPad.

## Continua Riproduzione

Ripristina la coda del lettore audio dall'ultima posizione salvata se questa funzione è abilitata nelle impostazioni dell'applicazione. Abilita sia **Salva Stato Lettore Audio** che **Salva Posizione di Riproduzione** in Impostazioni → Lettore Audio → Generale per la migliore esperienza.

## Posizioni

Tutte le tracce nella libreria sono raggruppate per tipo di sorgente e tag musicali. Puoi filtrare i brani per posizione usando il pulsante **Altre azioni** nell'angolo in alto a destra e selezionando **Filtro**.

### Musica Online

Questa sezione mostra la musica dai servizi cloud storage connessi.

### File in Questa Applicazione

Qui puoi trovare musica disponibile per la riproduzione offline, proveniente dai file locali.

### File su Questo iPhone / iPad / Mac

Questa categoria include la musica importata nell'applicazione dal tuo dispositivo.

## Categorie

Quando aggiungi tracce alla libreria musicale, l'app legge automaticamente i tag audio e li organizza in categorie come Brani, Brani non riprodotti, Album, Album Artist, Artisti, Generi e Compositori.

## Raggruppamento per Tag

Queste categorie ti aiutano a organizzare le tracce per tag musicali. Se non vedi tutti gli album, assicurati che l'app abbia scansionato ogni traccia.

- **Brani** — tutti i brani raggruppati per il tag TRACK_TITLE.
- **Brani non riprodotti** — tutti i brani che non sono mai stati riprodotti.
- **Album** — brani raggruppati per il tag ALBUM_NAME.
- **Album Artist** — brani raggruppati solo per ALBUM_ARTIST_TAG.
- **Artisti** — brani raggruppati solo per ARTIST_TAG.
- **Generi** — brani raggruppati per il tag GENRE.
- **Compositori** — brani raggruppati per il tag COMPOSER.

## Preferiti

Puoi contrassegnare i brani come preferiti nella schermata del lettore audio o usando il menu opzioni. I preferiti appaiono nella loro sezione dedicata così puoi trovarli con un tocco.

## Recenti

Questa sezione mostra tutte le tracce riprodotte di recente. Puoi personalizzare quanti elementi mantiene l'elenco recenti in Impostazioni → Libreria → Recenti → Cambia Dimensione Elenco.

## Segnalibri

Puoi creare segnalibri audio mentre una canzone è in riproduzione e gestirli in questa schermata — perfetto per audiolibri, lunghi mix, lezioni o per segnare il ritornello di una traccia preferita.

## Barra degli Strumenti Superiore

Situata appena sotto la barra di navigazione, la barra degli strumenti superiore offre diverse azioni convenienti: Cerca, Riproduci Tutto, Mescola Tutto e Continua Riproduzione.

## Ricerca

La funzione di ricerca ti consente di trovare una traccia, artista, album o genere specifico nella libreria musicale. La ricerca viene eseguita localmente sul database della libreria musicale, quindi funziona completamente offline e restituisce risultati mentre digiti.

## Menu Opzioni

Ogni brano nella libreria musicale ha un menu con più azioni, accessibile toccando il pulsante a tre punti vicino al titolo del brano.

### Per Brani Singoli

- **Riproduci Successivo** — aggiunge il brano in cima alla coda del lettore.
- **Riproduci Dopo** — aggiunge il brano in fondo alla coda del lettore.
- **Aggiungi alla Playlist** — aggiunge il brano a una playlist.
- **Aggiungi ai Preferiti** — contrassegna il brano come preferito.
- **Scaricare** — salva il brano nei file locali.
- **Modificare i tag audio** — apre l'editor di tag audio integrato.
- **Mostra nella Cartella** — rivela la cartella dove è archiviato il file audio.
- **Mostra nel Finder** — per i file importati dal Mac.
- **Apri In** — esporta il file audio in un'altra app.
- **Elimina dal Servizio Cloud** — rimuove il file dalla libreria musicale e dal cloud storage. **Questa azione è irreversibile.**
- **Elimina dalla Libreria Musicale** — elimina il brano dalla libreria musicale, ma il file rimane nello storage.

### Per Collezioni di Brani

- **Riproduci Tutto** — sostituisce la coda del lettore con i brani della collezione selezionata.
- **Riproduci Successivo** — aggiunge i brani di questa collezione in cima alla coda del lettore.
- **Riproduci Dopo** — aggiunge i brani di questa collezione in fondo alla coda del lettore.
- **Aggiungi alla Playlist** — include i brani di questa collezione in una playlist.
- **Abilitare la modalità offline** — scarica i brani di questa collezione nei file locali.
- **Modifica Immagine** — cambia la copertina dell'album per la collezione di brani.
- **Aggiungi all'Archivio** — raggruppa l'intera collezione in un singolo file ZIP (funzione Premium).
- **Esporta Elenco Brani** — esporta la collezione in M3U, CSV o TXT.
- **Elimina dalla Libreria Musicale** — rimuove la collezione di brani dalla libreria musicale.

## Modalità Selezione

Puoi attivare la modalità selezione usando il pulsante Altre azioni nell'angolo in alto a destra. In questa modalità, puoi selezionare più tracce contemporaneamente ed eseguire azioni batch.

## Dettaglio Album

Quando apri le sezioni Artista, Album Artist o Compositore, puoi vedere un selettore per Brani / Tutti gli Album / Album Esclusivi / Album da Solista.

- **Brani** — mostra tutti i brani dove questo Artista / Album Artist / Compositore appare nei tag audio.
- **Tutti gli Album** — mostra gli album di compilazione e tutti gli album dove l'artista è presente.
- **Album Esclusivi** — mostra gli album dove l'artista specificato è l'unico artista con quel nome album.
- **Album da Solista** — mostra gli album dove appaiono solo le tracce dell'artista specificato.

## Impostazioni

Tocca Altre azioni → Impostazioni per configurare le preferenze della libreria musicale.

### Lettura Metadati

Quando aggiungi tracce alla libreria, il lettore di metadati si mette al lavoro. Questo processo in background legge tutti i metadati dalle tracce e li organizza per Artista, Album, Genere e Compositore. Puoi regolare la velocità di lettura dei metadati. Il lettore di metadati aggiorna solo i metadati nella libreria musicale e non altera i file archiviati nel cloud.

### Sincronizzazione Online

La sincronizzazione musica online automatica consente di aggiungere tracce dai servizi cloud connessi alla libreria musicale automaticamente. Per abilitarla, apri le impostazioni della libreria musicale e seleziona le cartelle di sincronizzazione.

### Sincronizzazione Offline

Qui puoi configurare la sincronizzazione musica offline.

#### Cartelle offline sincronizzate

Quando rendi disponibile offline una cartella cloud (tramite il menu Altre azioni), appare in File Locali → Cartelle Offline.

#### Intervallo di Tempo

Puoi impostare l'intervallo di tempo per quanto spesso l'app deve controllare le cartelle offline per le modifiche.

#### Avvia Scansione Cartelle Locali

Questa opzione scansiona tutte le cartelle locali nella directory Documenti dell'applicazione per trovare file audio supportati.

### Personalizzazione

In questa sezione, puoi configurare lo stile della schermata della libreria musicale. Sono disponibili tre opzioni: Menu Semplice, Menu Raggruppato, Menu a Schede.

### Copertine Album

Qui puoi configurare come l'applicazione carica e memorizza le copertine degli album.

### Playlist

Puoi abilitare l'opzione di aggiungere lo stesso brano a una playlist due volte.

### Recenti

Puoi gestire l'elenco dei brani riprodotti di recente.

### Preferiti

Puoi gestire l'elenco dei tuoi brani preferiti.

### Elimina Libreria

Questa azione cancellerà il database della libreria musicale, ma i file musicali sullo storage rimarranno intatti.

### Limite di Caricamento Contenuto

Per impostazione predefinita, l'applicazione usa la paginazione per ridurre i tempi di caricamento del contenuto. Tuttavia, puoi disabilitare questa opzione e consentire all'applicazione di caricare tutti i dati disponibili contemporaneamente.

在 Flacbox 中管理音乐库非常简单，您可以轻松地将所有 FLAC、ALAC、DSD、MP3、M4A、OGG、WMA、APE 等数十种格式的曲目整理成一个可搜索的统一收藏。您有两种构建音乐库的方式：手动添加（您精确选择要添加的内容）或自动同步（Flacbox 扫描指定的云文件夹并在新文件出现时自动添加）。

## 手动添加

要手动添加曲目，点击左上角的**添加音乐**图标，从连接的云存储服务或设备上的文件中选择文件夹或文件。向库中添加曲目时，只会创建指向这些曲目的链接——实际文件保留在原始位置以节省宝贵的磁盘空间。

{{< /cards >}}

在 Mac 版本上，您还可以将文件拖放到库中，或在 iPhone 和 iPad 上使用系统文件选取器中的**打开文件…** / **打开文件夹…**。

## 继续播放

如果在应用设置中启用了此功能，则从上次保存的位置恢复音频播放器队列。在「设置 → 音频播放器 → 一般」中同时启用**保存音频播放器状态**和**保存播放位置**以获得最佳体验。

## 位置

您库中的所有曲目都按来源类型和音乐标签进行细致分组。您可以使用右上角的**更多操作**按钮并选择**筛选**来按位置筛选歌曲。

### 在线音乐

此部分展示来自已连接云存储服务的音乐。这里的曲目按需流式传输；在您明确下载或启用离线模式之前，不会占用本地存储。

### 此应用程序中的文件

在这里您可以找到可供离线播放的音乐，来自本地文件。包括应用程序 Documents 目录中的文件。

### 此 iPhone / iPad / Mac 上的文件

此类别包括通过**打开文件…**系统对话框从设备导入应用程序的音乐。原始文件保留在原始位置；Flacbox 只保留指向它们的链接。

## 类别

当您向音乐库添加曲目时，应用程序自动读取其音频标签并将其整理成类别：歌曲、未播放歌曲、专辑、专辑艺术家、艺术家、流派和作曲家。

## 标签分组

这些类别帮助您按音乐标签整理曲目。添加曲目到音乐库时，应用程序读取其元数据并按这些类别分组。如果您没有看到所有专辑，请确保应用程序已扫描每个曲目。元数据读取器完成所有操作后，您将在音乐库中看到以下类别：

- **歌曲** — 按 TRACK_TITLE 标签分组的所有歌曲。
- **未播放歌曲** — 所有从未播放过的歌曲。
- **专辑** — 按 ALBUM_NAME 标签分组的歌曲，忽略艺术家、专辑艺术家和作曲家标签。
- **专辑艺术家** — 仅按 ALBUM_ARTIST_TAG 分组的歌曲。适用于保持合辑和合作与独奏作品清晰分离。
- **艺术家** — 仅按 ARTIST_TAG 分组的歌曲。
- **流派** — 按 GENRE 标签分组的歌曲。
- **作曲家** — 按 COMPOSER 标签分组的歌曲——对于以「作曲家」为主要导航轴的古典音乐库非常有价值。

## 收藏夹

您可以在音频播放器屏幕上或使用选项菜单将歌曲标记为收藏。收藏出现在各自的部分，方便一键找到。

## 最近使用

此部分显示所有最近播放的曲目。您可以在「设置 → 音乐库 → 最近使用 → 更改列表大小」中自定义最近使用列表保留的项目数，并将列表导出为 M3U / CSV / TXT 以备份您的收听历史。

## 书签

您可以在歌曲播放时创建音频书签，并在此屏幕上管理它们——非常适合有声书、长混音、讲座，或只是标记最喜欢曲目的副歌。

## 顶部工具栏

紧靠导航栏下方，顶部工具栏提供几个便捷操作：搜索、播放全部、随机播放全部和继续播放。您可以通过向下轻扫手势显示或隐藏此工具栏。

## 搜索

搜索功能让您可以在音乐库中找到特定曲目、艺术家、专辑或流派。

## 选项菜单

您音乐库中的每首歌曲都有一个更多操作菜单，通过点击歌曲标题旁边的三点按钮访问。

### 对于单首歌曲

- **播放下一首** — 将歌曲添加到播放器队列顶部。
- **稍后播放** — 将歌曲追加到播放器队列底部。
- **添加到播放列表** — 将歌曲添加到播放列表。
- **添加到收藏夹** — 将歌曲标记为收藏以便快速访问。
- **下载** — 将歌曲保存到本地文件。
- **编辑音频标签** — 打开内置音频标签编辑器以修复缺失的元数据。
- **在文件夹中显示** — 显示存储音频文件的文件夹。
- **在 Finder 中显示** — 对于从 Mac 导入的文件，显示 Mac 上的文件夹。
- **打开方式** — 将音频文件导出到另一个应用。
- **从云服务中删除** — 从音乐库和云存储中删除文件。**此操作不可逆。**
- **从音乐库中删除** — 从音乐库中删除歌曲，但文件仍留在存储中。

### 对于歌曲集合

对于专辑、艺术家、流派或作曲家等歌曲集合：

- **播放全部** — 用所选集合中的歌曲替换播放器队列。
- **播放下一首** — 将此集合中的歌曲添加到播放器队列顶部。
- **稍后播放** — 将此集合中的歌曲追加到播放器队列底部。
- **添加到播放列表** — 将此集合中的歌曲包含在播放列表中。
- **启用离线模式** — 将此集合中的歌曲下载到本地文件。
- **编辑图片** — 更改歌曲集合的专辑封面。
- **添加到归档** — 将整个集合打包成单个 ZIP 文件（Premium 功能）。
- **导出歌曲列表** — 将集合导出为 M3U、CSV 或 TXT。
- **从音乐库中删除** — 从音乐库中删除歌曲集合。存储中的实际文件保持不变。

## 选择模式

您可以使用右上角的「更多操作」按钮激活选择模式。

## 专辑详情

打开艺术家、专辑艺术家或作曲家部分时，您可以看到歌曲 / 所有专辑 / 独有专辑 / 独奏专辑的切换器。

- **歌曲** — 显示此艺术家 / 专辑艺术家 / 作曲家出现在音频标签中的所有歌曲。
- **所有专辑** — 显示合辑和艺术家出现的所有专辑。
- **独有专辑** — 显示指定艺术家是该专辑名称唯一艺术家的专辑。
- **独奏专辑** — 显示只有指定艺术家曲目出现的专辑。

## 设置

点击「更多操作 → 设置」以配置您的音乐库首选项。

### 元数据读取

当您向库中添加曲目时，元数据读取器开始工作。此后台进程读取曲目的所有元数据，并按艺术家、专辑、流派和作曲家整理。

元数据读取器**仅更新音乐库中的元数据**，不会更改存储在云账户或本地存储中的文件。

当**在后台读取元数据**开关开启时，元数据读取器继续在后台模式工作。

当**规范化元数据编码**选项启用时，应用程序自动规范化音乐库中所有歌曲的元数据编码。

**重新加载元数据**将音乐库中的每个文件标记为缺少元数据，触发元数据读取器刷新每条记录。

点击**开始元数据读取**手动触发读取器。

### 在线同步

自动在线音乐同步允许从连接的云服务自动向音乐库添加曲目。要激活此功能，前往音乐库设置并选择同步文件夹。

在线音乐同步仅在应用程序处于前台时运行。要加快进程，保持 Flacbox 打开，将设备接上电源，并启用**设置 → 屏幕 → 始终活跃**。

### 离线同步

在此配置离线音乐同步。

#### 同步离线文件夹

当您将在线文件夹标记为可离线使用时，它会显示在这里。文件夹内容下载到「本地文件 → 离线文件夹」。

#### 时间间隔

设置应用程序检查离线文件夹是否有修改的时间间隔。

#### 开始本地文件夹扫描

此选项扫描应用程序 Documents 目录中的所有本地文件夹以查找支持的音频文件。

#### 重要

建议定期启动离线音乐同步以保持音乐库与本地文件同步更新。

### 个性化

在此部分，您可以配置音乐库屏幕样式。有三个选项：简单菜单、分组菜单、选项卡菜单。

### 专辑封面

在此，您可以配置应用程序如何加载和存储专辑封面。

- **网络类型** — 选择 Wi-Fi 或 Wi-Fi 和蜂窝数据进行封面下载。
- **为在线文件加载专辑封面** — 启用时，为存储在云存储中的文件加载嵌入式专辑封面。
- **在文件夹中搜索** — 启用时，如果曲目没有嵌入式封面，应用程序会在相同文件夹中查找 JPEG 或 PNG 图像。
- **封面质量** — 选择设备上存储的专辑封面质量。
- **在文件夹中显示** — 打开专辑封面缓存所在的文件夹。
- **全部删除** — 删除缓存的专辑封面以释放存储空间。

### 播放列表

您可以启用将同一首歌曲添加到播放列表两次的选项。默认情况下，此选项处于禁用状态。

### 最近使用

您可以管理最近播放的歌曲列表。

- **删除列表** — 删除最近播放歌曲的整个列表。
- **更改列表大小** — 设置列表中显示的项目数。
- **导出歌曲列表** — 将最近播放的歌曲列表导出为 M3U、CSV 或 TXT。详细说明请参阅[此处](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。

### 收藏夹

您可以管理您喜欢的歌曲列表。

- **同时编辑** — 启用时，将歌曲添加到收藏夹会同时将其添加到音乐库和文件部分。
- **删除列表** — 删除收藏歌曲的整个列表。
- **导出歌曲列表** — 类似最近使用，将收藏夹导出为 M3U、CSV 或 TXT。

### 删除音乐库

此操作将清除音乐库数据库，但会将您的音乐文件在存储中保持不变。

### 内容加载限制

默认情况下，应用程序使用分页来减少内容加载时间并保持大型音乐库的响应速度。但是，您可以禁用此选项并允许应用程序一次加载所有可用数据。要执行此操作，打开应用程序设置，向下滚动到「个性化 → 内容加载限制」，然后选择「已停用」。

在 Flacbox 中管理音樂庫非常輕鬆——您可以輕鬆地將所有 FLAC、ALAC、DSD、MP3、M4A、OGG、WMA、APE 等數十種格式的曲目整理成一個可搜尋的統一收藏。您有兩種建立音樂庫的方式：手動新增（您精確選擇要新增的內容）或自動同步（Flacbox 掃描指定的雲端資料夾並在新檔案出現時自動新增）。

## 手動新增

要手動新增曲目，點擊左上角的**新增音樂**圖示，從已連接的雲端儲存服務或裝置上的檔案中選擇資料夾或檔案。向音樂庫新增曲目時，只會建立指向這些曲目的連結——實際檔案保留在原始位置以節省寶貴的磁碟空間。

在 Mac 版本上，您還可以將檔案拖放到音樂庫，或在 iPhone 和 iPad 上使用系統檔案選取器中的**開啟檔案…** / **開啟資料夾…**。

## 繼續播放

如果在應用程式設定中啟用了此功能，則從上次儲存的位置恢復音訊播放器佇列。在「設定 → 音訊播放器 → 一般」中同時啟用**儲存音訊播放器狀態**和**儲存播放位置**以獲得最佳體驗。

## 位置

您音樂庫中的所有曲目都按來源類型和音樂標籤進行分組。您可以使用右上角的**更多操作**按鈕並選擇**篩選**來按位置篩選歌曲。

### 線上音樂

此部分展示來自已連接雲端儲存服務的音樂。這裡的曲目按需串流；在您明確下載或啟用離線模式之前，不會占用本機儲存空間。

### 此應用程式中的檔案

在這裡您可以找到可供離線播放的音樂，來自本機檔案。包括應用程式 Documents 目錄中的檔案。

### 此 iPhone / iPad / Mac 上的檔案

此類別包括透過**開啟檔案…**系統對話框從裝置匯入應用程式的音樂。原始檔案保留在原始位置；Flacbox 只保留指向它們的連結。

## 類別

當您向音樂庫新增曲目時，應用程式自動讀取其音訊標籤並將其整理成類別：歌曲、未播放歌曲、專輯、專輯藝術家、藝術家、類型和作曲家。

## 標籤分組

這些類別幫助您按音樂標籤整理曲目。元數據讀取器完成所有操作後，您將在音樂庫中看到以下類別：

- **歌曲** — 按 TRACK_TITLE 標籤分組的所有歌曲。
- **未播放歌曲** — 所有從未播放過的歌曲。
- **專輯** — 按 ALBUM_NAME 標籤分組的歌曲，忽略藝術家、專輯藝術家和作曲家標籤。
- **專輯藝術家** — 僅按 ALBUM_ARTIST_TAG 分組的歌曲。適用於保持合輯和合作與獨奏作品清晰分離。
- **藝術家** — 僅按 ARTIST_TAG 分組的歌曲。
- **類型** — 按 GENRE 標籤分組的歌曲。
- **作曲家** — 按 COMPOSER 標籤分組的歌曲——對於以「作曲家」為主要導覽軸的古典音樂庫非常有價值。

## 最愛項目

您可以在音訊播放器畫面上或使用選項選單將歌曲標記為最愛。最愛出現在各自的部分，方便一鍵找到。

## 最近使用的

此部分顯示所有最近播放的曲目。您可以在「設定 → 音樂庫 → 最近使用的 → 變更清單大小」中自訂最近使用清單保留的項目數，並將清單匯出為 M3U / CSV / TXT。

## 書籤

您可以在歌曲播放時建立音訊書籤，並在此畫面上管理它們——非常適合有聲書、長混音、講座，或只是標記最喜歡曲目的副歌。

## 頂部工具列

緊靠導覽列下方，頂部工具列提供幾個便捷操作：搜尋、播放全部、隨機播放全部和繼續播放。您可以透過向下輕掃手勢顯示或隱藏此工具列。

## 搜尋

搜尋功能讓您可以在音樂庫中找到特定曲目、藝術家、專輯或類型。

## 選項選單

您音樂庫中的每首歌曲都有一個更多操作選單，透過點擊歌曲標題旁邊的三點按鈕存取。

### 對於單首歌曲

- **播放下一首** — 將歌曲新增到播放器佇列頂部。
- **稍後播放** — 將歌曲附加到播放器佇列底部。
- **新增到播放清單** — 將歌曲新增到播放清單。
- **新增到最愛** — 將歌曲標記為最愛以便快速存取。
- **下載** — 將歌曲儲存到本機檔案。
- **編輯音訊標籤** — 開啟內建音訊標籤編輯器以修復缺失的後設資料。
- **在資料夾中顯示** — 顯示儲存音訊檔案的資料夾。
- **在 Finder 中顯示** — 對於從 Mac 匯入的檔案，顯示 Mac 上的資料夾。
- **以...開啟** — 將音訊檔案匯出到另一個應用程式。
- **從雲端服務中刪除** — 從音樂庫和雲端儲存空間中刪除檔案。**此操作不可逆。**
- **從音樂庫中刪除** — 從音樂庫中刪除歌曲，但檔案仍留在儲存空間中。

### 對於歌曲集合

對於專輯、藝術家、類型或作曲家等歌曲集合：

- **播放全部** — 用所選集合中的歌曲替換播放器佇列。
- **播放下一首** — 將此集合中的歌曲新增到播放器佇列頂部。
- **稍後播放** — 將此集合中的歌曲附加到播放器佇列底部。
- **新增到播放清單** — 將此集合中的歌曲包含在播放清單中。
- **啟用離線模式** — 將此集合中的歌曲下載到本機檔案。
- **編輯圖片** — 變更歌曲集合的專輯封面。
- **新增到封存** — 將整個集合打包成單個 ZIP 檔案（Premium 功能）。
- **匯出歌曲清單** — 將集合匯出為 M3U、CSV 或 TXT。
- **從音樂庫中刪除** — 從音樂庫中刪除歌曲集合。儲存空間中的實際檔案保持不變。

## 選擇模式

您可以使用右上角的「更多操作」按鈕啟動選擇模式。

## 專輯詳情

開啟藝術家、專輯藝術家或作曲家部分時，您可以看到歌曲 / 所有專輯 / 獨有專輯 / 獨奏專輯的切換器。

- **歌曲** — 顯示此藝術家 / 專輯藝術家 / 作曲家出現在音訊標籤中的所有歌曲。
- **所有專輯** — 顯示合輯和藝術家出現的所有專輯。
- **獨有專輯** — 顯示指定藝術家是該專輯名稱唯一藝術家的專輯。
- **獨奏專輯** — 顯示只有指定藝術家曲目出現的專輯。

## 設定

點擊「更多操作 → 設定」以設定您的音樂庫偏好設定。

### 後設資料讀取

當您向音樂庫新增曲目時，後設資料讀取器開始工作。此背景程序讀取曲目的所有後設資料，並按藝術家、專輯、類型和作曲家整理。

後設資料讀取器**僅更新音樂庫中的後設資料**，不會更改儲存在雲端帳戶或本機儲存空間中的檔案。

當**在背景讀取後設資料**開關開啟時，後設資料讀取器繼續在背景模式工作。

當**規範化後設資料編碼**選項啟用時，應用程式自動規範化音樂庫中所有歌曲的後設資料編碼。

**重新加載元數據**將音樂庫中的每個檔案標記為缺少後設資料，觸發後設資料讀取器重新整理每條記錄。

點擊**開始後設資料讀取**手動觸發讀取器。

### 線上同步

自動線上音樂同步允許從已連接的雲端服務自動向音樂庫新增曲目。要啟動，開啟音樂庫設定並選擇同步資料夾。

線上音樂同步僅在應用程式處於前台時運行。要加快進程，保持 Flacbox 開啟，將裝置接上電源，並啟用**設定 → 螢幕 → 始終活躍**。

### 離線同步

在此設定離線音樂同步。

#### 同步離線資料夾

當您將線上資料夾標記為可離線使用時，它會顯示在這裡。資料夾內容下載到「本機檔案 → 離線資料夾」。

#### 時間間隔

設定應用程式檢查離線資料夾是否有修改的時間間隔。

#### 開始本機資料夾掃描

此選項掃描應用程式 Documents 目錄中的所有本機資料夾以查找支援的音訊檔案。

#### 重要

建議定期啟動離線音樂同步以保持音樂庫與本機檔案同步更新。

### 個人化

在此部分，您可以設定音樂庫畫面樣式。有三個選項：簡單選單、分組選單、索引標籤選單。

### 專輯封面

在此，您可以設定應用程式如何載入和儲存專輯封面。

- **網路類型** — 選擇 Wi-Fi 或 Wi-Fi 和行動數據進行封面下載。
- **為線上檔案載入專輯封面** — 啟用時，為存放在雲端儲存空間中的檔案載入內嵌專輯封面。
- **在資料夾中搜尋** — 啟用時，如果曲目沒有內嵌封面，應用程式會在相同資料夾中尋找 JPEG 或 PNG 影像。
- **封面品質** — 選擇儲存在裝置上的專輯封面品質。
- **在資料夾中顯示** — 開啟專輯封面快取所在的資料夾。
- **全部刪除** — 刪除快取的專輯封面以釋放儲存空間。

### 播放清單

您可以啟用將同一首歌曲新增到播放清單兩次的選項。預設情況下，此選項處於停用狀態。

### 最近使用的

您可以管理最近播放的歌曲清單。

- **刪除清單** — 刪除最近播放歌曲的整個清單。
- **變更清單大小** — 設定清單中顯示的項目數。
- **匯出歌曲清單** — 將最近播放的歌曲清單匯出為 M3U、CSV 或 TXT。詳細說明請參閱[此處](/docs/howto/export-tracks-collection-from-evermusic-flacbox-to-m3u-csv-txt/)。

### 最愛項目

您可以管理您喜愛的歌曲清單。

- **同時編輯** — 啟用時，將歌曲新增到最愛項目會同時將其新增到音樂庫和檔案部分。
- **刪除清單** — 刪除最愛歌曲的整個清單。
- **匯出歌曲清單** — 類似最近使用的，將最愛清單匯出為 M3U、CSV 或 TXT。

### 刪除音樂庫

此動作將清除音樂庫資料庫，但會將您的音樂檔案在儲存空間中保持不變。

### 內容載入限制

預設情況下，應用程式使用分頁來減少內容載入時間並保持大型音樂庫的回應速度。但是，您可以停用此選項並允許應用程式一次載入所有可用資料。要執行此操作，開啟應用程式設定，向下捲動到「個人化 → 內容載入限制」，然後選擇「已停用」。

La section Fichiers locaux sert de hub pour gérer les fichiers situés dans le dossier Documents de l'application, ainsi que les fichiers ajoutés depuis votre appareil disponibles pour une lecture hors ligne.

Ce gestionnaire de fichiers intégré vous permet de modifier des fichiers (renommer, déplacer, copier, téléverser, supprimer), de surveiller les transferts et propose plusieurs méthodes pour importer des fichiers audio dans l'application — téléchargements directs depuis le nuage, synchronisation en mode hors ligne, clés USB, Wi-Fi Drive et Finder File Sharing.

## Télécharger des fichiers depuis le stockage en nuage

Vous pouvez ajouter des fichiers audio à l'application en les téléchargeant depuis des services de stockage en nuage connectés comme décrit [ici](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag).

## Mode hors ligne

Synchronisez automatiquement des dossiers entiers, des listes de lecture, des albums, des artistes ou des genres pour une lecture hors ligne comme décrit [ici](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/). Le mode hors ligne maintient votre copie locale synchronisée avec le nuage — lorsque de nouveaux fichiers sont ajoutés sur le serveur, ils sont téléchargés automatiquement.

## Importer des fichiers depuis votre appareil

Importez facilement des fichiers depuis votre appareil comme décrit [ici](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac), ou depuis une clé USB connectée comme décrit [ici](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it). Vous pouvez également connecter un dossier situé sur votre appareil — Flacbox crée un lien vers ce dossier (lecture/écriture) pour vous permettre de gérer son contenu directement depuis Flacbox.

## iTunes / Finder File Sharing

Transférez des fichiers à l'aide d'un câble Lightning ou USB-C comme décrit [ici](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder). Sur macOS Catalina et versions ultérieures, le Finder remplace iTunes pour cette opération — la procédure est par ailleurs identique.

## Wi-Fi Drive

Transférez des fichiers sans fil depuis n'importe quel navigateur de bureau comme décrit [ici](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive). Sur macOS, vous pouvez également monter Wi-Fi Drive directement dans le Finder (Aller → Se connecter au serveur…) et sur Windows dans l'Explorateur de fichiers (Connecter un lecteur réseau…).

## File de transferts

Dans le coin supérieur gauche de la barre de navigation, vous trouverez un bouton Transferts. Appuyez dessus pour accéder à la file de transferts, où vous pouvez surveiller et gérer chaque téléchargement et téléversement — mettre en pause, reprendre, réessayer les transferts échoués et afficher la progression détaillée par élément. Vous pouvez également ajuster la vitesse de la file de transferts (nombre maximal de tâches parallèles) et le type de réseau dans Paramètres → Gestionnaire de fichiers.

## Section Accès rapide

En haut de l'écran, une section Accès rapide fournit des liens pratiques vers vos fichiers et dossiers récents et favoris.

## Récents

Cette section affiche tous les fichiers ou dossiers récemment ouverts, pour vous permettre de revenir rapidement à votre dernier espace de travail en un seul appui.

## Favoris

Vous pouvez marquer des fichiers ou des dossiers comme favoris et y accéder dans cette section. Vous pouvez également ajouter un dossier situé sur votre appareil à vos favoris. Pour cela, ouvrez la section Favoris, appuyez sur les trois points dans le coin supérieur droit et choisissez Ajouter un dossier. Suivez les instructions pour ajouter un dossier de votre appareil aux favoris pour un accès rapide.

## Barre d'outils supérieure

La barre d'outils supérieure, située sous la barre de navigation, propose plusieurs actions :

- **Rechercher** — effectuer une recherche dans le dossier actuel.
- **Reprendre la lecture** — si activée dans les paramètres de l'application, cette fonctionnalité restaure la file de l'audio player et la dernière position média pour le dossier actuel, vous permettant de reprendre là où vous vous étiez arrêté.
- **Tout lire** — parcourir le dossier actuel et ses sous-dossiers, puis ajouter les fichiers à la file de l'audio player dans l'ordre de tri actuel.
- **Tout mélanger** — similaire à Tout lire, mais mélange les fichiers avant de les ajouter à la file de l'audio player.

Vous pouvez afficher ou masquer la barre d'outils supérieure avec un geste de balayage vers le bas.

## Dossiers spéciaux

Dans l'écran Fichiers locaux, vous rencontrerez plusieurs dossiers spéciaux que Flacbox utilise pour organiser le contenu.

## Téléchargements

Tous les fichiers téléchargés depuis un service de stockage en nuage apparaîtront ici par défaut. Vous pouvez personnaliser ce comportement dans Paramètres → Gestionnaire de fichiers → Enregistrer les fichiers téléchargés dans.

## Audio Player

Il s'agit du dossier de cache de l'audio player. Par défaut, l'audio player télécharge les prochaines chansons de la file de lecture pour appliquer le fondu enchaîné et optimiser la lecture. Vous pouvez désactiver le cache de l'audio player dans les paramètres de l'application ou simplement supprimer ce dossier des Fichiers locaux pour libérer de l'espace.

## iCloud

Placez des fichiers dans ce dossier et ils se synchroniseront sur tous vos appareils connectés au même compte iCloud — pratique pour garder un petit ensemble de pistes indispensables disponibles partout sans les téléverser vers un nuage tiers.

## Dossiers hors ligne

Lorsque vous rendez une liste de lecture, un album, un artiste ou un dossier en ligne disponible pour une utilisation hors ligne, tous ces fichiers sont téléchargés dans ce dossier. Si vous devez désactiver le mode hors ligne pour des éléments spécifiques, vous pouvez utiliser le menu Plus d'actions ou supprimer le dossier correspondant situé dans Dossiers hors ligne.

## Deux sections distinctes

L'écran Fichiers locaux est organisé en deux sections distinctes.

### Fichiers dans cette application

Affiche les fichiers et dossiers stockés dans le répertoire Documents de l'application et dans iCloud Drive.

### Fichiers sur cet iPhone / iPad / Mac

Affiche les fichiers situés sur votre appareil mais dans d'autres applications. Vous pouvez les importer dans cette application à l'aide du sélecteur de fichiers système. Pour activer le sélecteur, choisissez **Ouvrir des fichiers…** pour sélectionner des fichiers ou **Ouvrir des dossiers…** pour sélectionner des dossiers. Des instructions détaillées sur l'importation de musique locale stockée sur votre iPhone ou Mac sont disponibles [ici](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).

Vous pouvez également connecter un dossier situé sur votre appareil et avoir un accès rapide à son contenu. Utilisez l'élément de menu **Connecter un dossier** et choisissez un dossier situé sur votre appareil. Appuyez sur **Fait**, et l'application crée un lien vers ce dossier avec accès en lecture / écriture, vous permettant de gérer les fichiers directement depuis cette application. Pour déconnecter un dossier situé sur votre appareil, appuyez sur le bouton **Plus d'actions** et choisissez **Se déconnecter**.

## Plus d'actions

Le menu Plus d'actions pour le dossier actuellement ouvert est situé dans le coin supérieur droit et donne accès à diverses actions.

- **Sélectionner** — passer en mode de sélection pour les fichiers et dossiers.
- **Nouveau dossier** — créer un nouveau dossier dans le dossier actuel.
- **Rechercher** — activer la fonction de recherche pour trouver un fichier ou dossier spécifique.
- **Trier** — organiser les fichiers par nom, taille, date ou à l'aide des informations de métadonnées.
- **Liste / Grille** — modifier la présentation du contenu d'une vue tableau à une vue grille (ou vice versa).
- **Aide** — accéder aux instructions d'aide.

## Mode de sélection

Si vous devez modifier plusieurs fichiers, activez le mode de sélection en appuyant sur le bouton **Plus d'actions** **«&nbsp;...&nbsp;»** dans la barre de navigation dans le coin supérieur droit et en choisissant l'élément de menu **Sélectionner**. Des cases à cocher apparaissent à côté de chaque fichier. Sélectionnez les fichiers souhaités en appuyant sur leurs cases. Vous pouvez effectuer les actions suivantes sur les fichiers sélectionnés :

- **Lire ensuite** — ajouter les fichiers ou dossiers sélectionnés en haut de la file de l'audio player dans l'ordre de tri actuel.
- **Lire plus tard** — ajouter les fichiers ou dossiers sélectionnés en bas de la file de l'audio player dans l'ordre de tri actuel.
- **Ajouter à la bibliothèque musicale** — inclure les fichiers ou dossiers sélectionnés dans la bibliothèque musicale.
- **Ajouter à une liste de lecture** — ajouter les fichiers ou dossiers sélectionnés à une liste de lecture.
- **Copier** — créer une copie du fichier ou du dossier sélectionné.
- **Téléverser** — téléverser le fichier ou le dossier sélectionné vers un stockage distant.
- **Déplacer** — déplacer le fichier ou le dossier sélectionné vers un autre emplacement sur l'appareil.
- **Renommer** — modifier le nom du fichier ou du dossier sélectionné.
- **Supprimer** — supprimer le fichier ou le dossier sélectionné de l'appareil. **Cette action est irréversible.**

## Menu Options

Pour chaque fichier ou dossier de l'application, plusieurs actions sont disponibles en appuyant sur le bouton **Plus** **«&nbsp;...&nbsp;»**. Si vous ne voyez pas toutes les actions, faites défiler vers le bas pour les afficher :

- **Tout lire** — remplacer la file de l'audio player par les éléments du dossier sélectionné.
- **Lire ensuite** — ajouter le fichier ou le dossier en haut de la file de l'audio player.
- **Lire plus tard** — ajouter le fichier ou le dossier en bas de la file de l'audio player.
- **Ajouter à la bibliothèque musicale** — inclure les fichiers ou dossiers sélectionnés dans la bibliothèque musicale.
- **Ajouter à une liste de lecture** — ajouter le fichier ou le dossier à une liste de lecture. Si vous créez une liste de lecture à partir d'un dossier, un nom lui sera automatiquement attribué.
- **Ajouter aux favoris** — marquer les fichiers ou dossiers sélectionnés comme favoris.
- **Modifier les balises audio** — ouvrir l'éditeur de balises intégré pour modifier les balises audio du fichier sélectionné.
- **Renommer** — modifier le nom du fichier ou du dossier sélectionné.
- **Déplacer** — déplacer le fichier ou le dossier sélectionné vers un autre emplacement sur l'appareil.
- **Copier** — créer une copie du fichier ou du dossier sélectionné.
- **Téléverser** — téléverser le fichier ou le dossier sélectionné vers un stockage distant.
- **Ajouter à l'archive** — regrouper le fichier ou le dossier dans un seul fichier ZIP (fonctionnalité Premium).
- **Afficher dans le Finder** — pour les fichiers importés depuis votre Mac, cette action révèle le répertoire dans le Finder contenant le fichier.
- **Ouvrir dans** — exporter le fichier vers une autre application.
- **Supprimer** — supprimer le fichier ou le dossier de l'appareil. **Cette action est irréversible** et vous ne pouvez pas restaurer les fichiers supprimés.

## Dossiers hors ligne

Le mode hors ligne est une fonctionnalité pratique qui vous permet d'accéder à votre musique préférée même lorsque vous n'êtes pas connecté à internet. Lorsque vous activez le mode hors ligne pour un album, un artiste, une liste de lecture, un genre ou un dossier distant, tous les fichiers de cette collection sont automatiquement téléchargés sur votre appareil pour une lecture hors ligne. Vous pouvez accéder commodément à ces fichiers dans la section **Dossiers hors ligne** de l'application.

Ce qui est encore plus pratique, c'est que lorsque vous ajoutez de nouveaux fichiers au serveur distant dans le dossier que vous avez marqué pour une utilisation hors ligne, ces nouveaux ajouts sont également automatiquement téléchargés sur votre appareil. Cela garantit que vous disposez toujours de la musique la plus récente prête pour une écoute hors ligne — idéal pour les voyages, les vols ou les déplacements entre réseaux Wi-Fi.

Si vous devez synchroniser manuellement un dossier hors ligne depuis votre service cloud, voici comment procéder : appuyez simplement sur les trois points dans le coin supérieur droit de l'application et sélectionnez **Synchroniser**. Cela met à jour votre collection hors ligne avec les dernières pistes du service cloud à votre convenance.

Pour ceux qui souhaitent personnaliser davantage leur expérience, vous pouvez ajuster les paramètres de délai de synchronisation dans les paramètres de l'application selon vos préférences. Vous avez ainsi un contrôle total sur quand et comment votre collection hors ligne est mise à jour, vous assurant que vos morceaux préférés sont toujours disponibles, même sans connexion internet.

Des instructions détaillées sur la façon de lire de la musique hors ligne dans Flacbox — y compris comment télécharger et synchroniser depuis le nuage vers les fichiers locaux — sont disponibles [ici](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Personnalisation

En ouvrant les paramètres de l'application et en accédant à Personnalisation → Style de l'écran Fichiers locaux, vous pouvez configurer la présentation des Fichiers locaux. 
Les options disponibles sont :

- **Menu simple** — affiche directement les dossiers situés dans le répertoire Documents de l'application.
- **Menu groupé** — affiche le contenu de l'écran regroupé par catégories (Accès rapide, Dossiers spéciaux, Fichiers dans cette application, Fichiers sur cet appareil, etc.) pour une vue d'ensemble plus ordonnée.

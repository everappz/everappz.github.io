La sezione File Locali funge da hub per la gestione dei file nella cartella Documenti dell'applicazione, nonché dei file aggiunti dal tuo dispositivo disponibili per la riproduzione offline.

Questo gestore file integrato consente di modificare i file (rinominare, spostare, copiare, caricare, eliminare), monitorare i trasferimenti e offre diversi metodi per importare file audio nell'app — download diretti dal cloud, sincronizzazione in modalità offline, drive flash USB, Wi-Fi Drive e Finder File Sharing.

## Scaricare File dal Cloud Storage

Puoi aggiungere file audio all'app scaricandoli dai servizi cloud storage connessi come descritto [qui](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag).

## Modalità Offline

Sincronizza automaticamente intere cartelle, playlist, album, artisti o generi per la riproduzione offline come descritto [qui](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/). La modalità offline mantiene la copia locale sincronizzata con il cloud — quando vengono aggiunti nuovi file sul server, vengono scaricati automaticamente.

## Importare File dal Dispositivo

Importa facilmente file dal tuo dispositivo come descritto [qui](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac), o da un drive USB collegato come descritto [qui](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it). Puoi anche connettere una cartella sul tuo dispositivo — Flacbox crea un collegamento ad essa (lettura/scrittura) così puoi gestire il contenuto della cartella direttamente da Flacbox.

## iTunes / Finder File Sharing

Trasferisci file usando un cavo Lightning o USB-C come descritto [qui](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder). Su macOS Catalina e versioni successive, Finder sostituisce iTunes per questo — la procedura è altrimenti identica.

## Wi-Fi Drive

Trasferisci file in modalità wireless usando qualsiasi browser desktop come descritto [qui](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive). Su macOS puoi anche montare Wi-Fi Drive direttamente nel Finder (Vai → Connetti al server…) e su Windows in Esplora file (Connetti unità di rete…).

## Coda di Trasferimento

Nell'angolo in alto a sinistra della barra di navigazione troverai un pulsante Trasferimenti. Toccalo per accedere alla coda di trasferimento, dove puoi monitorare e gestire ogni download e upload — mettere in pausa, riprendere, riprovare i trasferimenti falliti e visualizzare il progresso dettagliato per elemento. Puoi anche regolare la velocità della coda di trasferimento (numero massimo di attività parallele) e il tipo di rete in Impostazioni → Gestore File.

## Sezione Accesso Rapido

Nella parte superiore dello schermo, una sezione Accesso Rapido fornisce collegamenti comodi ai file e alle cartelle recenti e preferiti.

## Recenti

Questa sezione mostra ogni file o cartella aperto di recente, così puoi tornare al tuo ultimo spazio di lavoro con un tocco.

## Preferiti

Puoi contrassegnare file o cartelle come preferiti e accedervi in questa sezione. Puoi anche aggiungere una cartella sul tuo dispositivo ai preferiti. Per farlo, apri la sezione Preferiti, tocca i tre punti nell'angolo in alto a destra e scegli Aggiungi Cartella.

## Barra degli Strumenti Superiore

La barra degli strumenti superiore, situata sotto la barra di navigazione, offre diverse azioni:

- **Cerca** — esegui una ricerca nella cartella corrente.
- **Continua Riproduzione** — se abilitata nelle impostazioni dell'applicazione, questa funzione ripristina la coda del lettore audio e l'ultima posizione multimediale per la cartella corrente.
- **Riproduci Tutto** — scansiona la cartella corrente e le sue sottocartelle, poi aggiungi file alla coda del lettore con l'ordine di sorteggio corrente.
- **Mescola Tutto** — simile a Riproduci Tutto, ma mescola i file prima di aggiungerli alla coda del lettore audio.

Puoi mostrare o nascondere la barra degli strumenti superiore usando un gesto di scorrimento verso il basso.

## Cartelle Speciali

All'interno della schermata File Locali, incontrerai diverse cartelle speciali che Flacbox utilizza per organizzare i contenuti.

## Scaricare

Tutti i file scaricati da un servizio cloud storage appariranno qui per impostazione predefinita. Puoi personalizzare questo comportamento in Impostazioni → Gestore File → Salva File Scaricati In.

## Lettore Audio

Questa è la cartella cache del lettore audio. Per impostazione predefinita, il lettore audio scarica i brani successivi nella coda del lettore per applicare il crossfade e ottimizzare la riproduzione. Puoi disabilitare la cache del lettore audio nelle impostazioni dell'app o semplicemente eliminare questa cartella da File Locali per liberare spazio.

## iCloud

Inserisci file in questa cartella e si sincronizzeranno su tutti i tuoi dispositivi connessi allo stesso account iCloud — utile per mantenere un piccolo set di brani indispensabili disponibili ovunque senza caricarli su un cloud di terze parti.

## Cartelle Offline

Quando rendi disponibile offline una playlist, album, artista o cartella online, tutti quei file vengono scaricati in questa cartella. Se devi disabilitare la modalità offline per elementi specifici, puoi usare il menu Altre azioni o eliminare la cartella corrispondente in Cartelle Offline.

## Due Sezioni Distinte

La schermata File Locali è organizzata in due sezioni distinte.

### File in Questa Applicazione

Mostra file e cartelle memorizzati nella directory Documenti dell'app e in iCloud Drive.

### File su Questo iPhone / iPad / Mac

Mostra i file situati sul tuo dispositivo ma in applicazioni diverse. Puoi importarli in questa applicazione usando il selettore file di sistema. Per attivare il selettore, scegli **Apri File…** per selezionare file o **Apri Cartelle…** per selezionare cartelle. Le istruzioni dettagliate su come importare musica locale archiviata su iPhone o Mac sono disponibili [qui](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).

## Altre Azioni

Il menu Altre azioni per la cartella attualmente aperta si trova nell'angolo in alto a destra e fornisce accesso a varie azioni.

- **Selezionare** — passa alla modalità di selezione per file e cartelle.
- **Nuova Cartella** — crea una nuova cartella all'interno della cartella corrente.
- **Cerca** — attiva la funzione di ricerca per trovare un file o una cartella specifici.
- **Ordina** — organizza i file per nome, dimensione, data o usando le informazioni dei metadati.
- **Elenco / Griglia** — cambia la presentazione del contenuto da vista tabella a vista griglia (o viceversa).
- **Aiuto** — accedi alle istruzioni di aiuto.

## Modalità Selezione

Se devi modificare più file, attiva la modalità selezione toccando il pulsante **Altre azioni** **"..."** sulla barra di navigazione nell'angolo in alto a destra e scegliendo la voce di menu **Selezionare**. Vengono visualizzate caselle di controllo vicino a ogni file. Seleziona i file desiderati toccando le loro caselle di controllo. Puoi eseguire le seguenti azioni sui file selezionati:

- **Riproduci Successivo** — aggiunge file o cartelle selezionati in cima alla coda del lettore con l'ordine di sorteggio corrente.
- **Riproduci Dopo** — aggiunge file o cartelle selezionati in fondo alla coda del lettore.
- **Aggiungi alla Libreria Musicale** — include file o cartelle selezionati nella libreria musicale.
- **Aggiungi a una Playlist** — aggiunge file o cartelle selezionati a una playlist.
- **Copia** — crea una copia del file o della cartella selezionati.
- **Carica** — carica il file o la cartella selezionati sullo storage remoto.
- **Sposta** — sposta il file o la cartella selezionati in una posizione diversa sul dispositivo.
- **Rinomina** — cambia il nome del file o della cartella selezionati.
- **Eliminare** — rimuove il file o la cartella selezionati dal dispositivo. **Questa azione è irreversibile.**

## Menu Opzioni

Per ogni file o cartella nell'app, sono disponibili diverse azioni, accessibili toccando il pulsante **Altro** **"..."**. Se non vedi tutte le azioni, scorri verso il basso per visualizzarle:

- **Riproduci Tutto** — sostituisce la coda del lettore con elementi dalla cartella selezionata.
- **Riproduci Successivo** — aggiunge il file o la cartella in cima alla coda del lettore.
- **Riproduci Dopo** — aggiunge il file o la cartella in fondo alla coda del lettore.
- **Aggiungi alla Libreria Musicale** — include file o cartelle selezionati nella libreria musicale.
- **Aggiungi a una Playlist** — aggiunge il file o la cartella a una playlist.
- **Aggiungi ai Preferiti** — contrassegna file o cartelle selezionati come preferiti.
- **Modificare i tag audio** — apre l'editor di tag integrato per modificare i tag audio del file selezionato.
- **Rinomina** — modifica il nome del file o della cartella selezionati.
- **Sposta** — sposta il file o la cartella selezionati in una posizione diversa sul dispositivo.
- **Copia** — crea una copia del file o della cartella selezionati.
- **Carica** — carica il file o la cartella selezionati sullo storage remoto.
- **Aggiungi all'Archivio** — raggruppa il file o la cartella in un singolo file ZIP (funzione Premium).
- **Mostra nel Finder** — per i file importati dal Mac, questa azione rivela la directory nel Finder che contiene il file.
- **Apri In** — esporta il file in un'altra app.
- **Eliminare** — elimina il file o la cartella dal dispositivo. **Questa azione è irreversibile** e non puoi ripristinare i file eliminati.

## Cartelle Offline

La modalità offline è una funzione utile che ti permette di accedere alla tua musica preferita anche quando non sei connesso a internet. Quando abiliti la modalità offline per un album, artista, playlist, genere o cartella remota, tutti i file in quella collezione vengono scaricati automaticamente sul tuo dispositivo per la riproduzione offline. Puoi accedere comodamente a questi file nella sezione **Cartelle offline** dell'app.

La cosa ancora più comoda è che quando aggiungi nuovi file al server remoto nella cartella che hai contrassegnato per l'uso offline, anche quelle nuove aggiunte vengono scaricate automaticamente sul tuo dispositivo. Questo assicura di avere sempre la musica più recente pronta per il godimento offline — perfetto per viaggi, voli o spostamenti tra reti Wi-Fi.

Se hai bisogno di sincronizzare manualmente una cartella offline dal tuo servizio cloud, ecco come fare: basta toccare i tre punti nell'angolo in alto a destra dell'app e selezionare **Sincronizzare**. Questo aggiorna la tua collezione offline con i brani più recenti dal servizio cloud quando preferisci.

Le istruzioni dettagliate su come riprodurre musica offline in Flacbox — incluso come scaricare e sincronizzare dal cloud ai file locali — sono disponibili [qui](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Personalizzazione

Aprendo le impostazioni dell'applicazione e andando a Personalizzazione → Stile Schermata File Locali, puoi configurare la presentazione dei File Locali.
Le opzioni disponibili sono:

- **Menu Semplice** — mostra direttamente le cartelle nella directory Documenti dell'app.
- **Menu Raggruppato** — mostra il contenuto della schermata raggruppato per categorie (Accesso Rapido, Cartelle Speciali, File in Questa Applicazione, File su Questo Dispositivo, ecc.) per una panoramica più ordinata.

Bagian File Lokal berfungsi sebagai pusat untuk mengelola file yang terletak di folder Dokumen aplikasi, serta file yang ditambahkan dari perangkat Anda yang tersedia untuk pemutaran offline.

Manajer file bawaan ini memungkinkan Anda mengedit file (ganti nama, pindah, salin, unggah, hapus), memantau transfer, dan menawarkan beberapa metode untuk mengimpor file audio ke aplikasi — unduhan langsung dari cloud, sinkronisasi mode offline, drive flash USB, Wi-Fi Drive, dan Finder File Sharing.

## Mengunduh File dari Cloud Storage

Anda dapat menambahkan file audio ke aplikasi dengan mengunduhnya dari layanan cloud storage yang terhubung seperti yang dijelaskan [di sini](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag).

## Mode Offline

Sinkronkan seluruh folder, daftar putar, album, artis, atau genre secara otomatis untuk pemutaran offline seperti yang dijelaskan [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/). Mode offline menjaga salinan lokal Anda tetap sinkron dengan cloud — ketika file baru ditambahkan di server, file tersebut diunduh secara otomatis.

## Mengimpor File dari Perangkat Anda

Impor file dengan mudah dari perangkat Anda seperti yang dijelaskan [di sini](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac), atau dari drive USB yang terpasang seperti yang dijelaskan [di sini](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it). Anda juga dapat menghubungkan folder yang terletak di perangkat Anda — Flacbox membuat tautan ke folder tersebut (baca/tulis) sehingga Anda dapat mengelola konten folder langsung dari dalam Flacbox.

## iTunes / Finder File Sharing

Transfer file menggunakan kabel Lightning atau USB-C seperti yang dijelaskan [di sini](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder). Di macOS Catalina dan yang lebih baru, Finder menggantikan iTunes untuk ini — prosedurnya identik.

## Wi-Fi Drive

Transfer file secara nirkabel menggunakan browser desktop apa pun seperti yang dijelaskan [di sini](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive). Di macOS Anda juga dapat memasang Wi-Fi Drive langsung di Finder (Pergi → Hubungkan ke Server…) dan di Windows di File Explorer (Petakan Drive Jaringan…).

## Antrian Transfer

Di sudut kiri atas bilah navigasi, Anda akan menemukan tombol Transfer. Ketuk untuk mengakses antrian transfer, di mana Anda dapat memantau dan mengelola setiap unduhan dan unggahan — jeda, lanjutkan, coba ulang transfer yang gagal, dan lihat kemajuan terperinci per item. Anda juga dapat menyesuaikan kecepatan antrian transfer (tugas paralel maksimum) dan jenis jaringan di Pengaturan → Manajer File.

## Bagian Akses Cepat

Di bagian atas layar, bagian Akses Cepat menyediakan tautan yang mudah ke file dan folder terbaru dan favorit Anda.

## Terbaru

Bagian ini menampilkan setiap file atau folder yang baru dibuka, sehingga Anda dapat kembali ke ruang kerja terbaru dengan satu ketukan.

## Favorit

Anda dapat menandai file atau folder sebagai favorit dan mengaksesnya di bagian ini. Anda juga dapat menambahkan folder yang terletak di perangkat Anda ke favorit. Untuk melakukannya, buka bagian Favorit, ketuk tiga titik di sudut kanan atas, dan pilih Tambah Folder.

## Toolbar Atas

Toolbar atas, yang terletak di bawah bilah navigasi, menawarkan beberapa tindakan:

- **Cari** — lakukan pencarian dalam folder saat ini.
- **Lanjutkan Pemutaran** — jika diaktifkan dalam pengaturan aplikasi, fitur ini memulihkan antrian pemutar audio dan posisi media terakhir untuk folder saat ini.
- **Putar Semua** — pindai folder saat ini dan subfoldernya, lalu tambahkan file ke antrian pemutar dengan urutan sortir saat ini.
- **Acak Semua** — mirip dengan Putar Semua, tetapi mengacak file sebelum menambahkannya ke antrian pemutar audio.

Anda dapat menampilkan atau menyembunyikan toolbar atas menggunakan gerakan geser ke bawah.

## Folder Khusus

Di dalam layar File Lokal, Anda akan menemukan beberapa folder khusus yang digunakan Flacbox untuk mengatur konten.

## Unduh

Semua file yang diunduh dari layanan cloud storage akan muncul di sini secara default. Anda dapat menyesuaikan perilaku ini di Pengaturan → Manajer File → Simpan File yang Diunduh Ke.

## Pemutar Audio

Ini adalah folder cache pemutar audio. Secara default, pemutar audio mengunduh lagu-lagu berikutnya dalam antrian pemutar untuk menerapkan crossfade dan mengoptimalkan pemutaran. Anda dapat menonaktifkan cache pemutar audio di pengaturan aplikasi atau cukup hapus folder ini dari File Lokal untuk mengosongkan ruang.

## iCloud

Tempatkan file di folder ini, dan file tersebut akan disinkronkan di semua perangkat Anda yang terhubung ke akun iCloud yang sama — berguna untuk menjaga satu set trek yang harus dimiliki tersedia di mana-mana tanpa mengunggahnya ke cloud pihak ketiga.

## Folder Offline

Ketika Anda membuat daftar putar, album, artis, atau folder online tersedia untuk penggunaan offline, semua file tersebut diunduh ke folder ini. Jika Anda perlu menonaktifkan mode offline untuk item tertentu, Anda dapat menggunakan menu Lebih banyak tindakan atau menghapus folder yang sesuai yang terletak di Folder Offline.

## Dua Bagian yang Berbeda

Layar File Lokal diorganisir menjadi dua bagian yang berbeda.

### File di Aplikasi Ini

Menampilkan file dan folder yang disimpan di direktori Dokumen aplikasi dan iCloud Drive.

### File di iPhone / iPad / Mac Ini

Menampilkan file yang terletak di perangkat Anda tetapi di aplikasi yang berbeda. Anda dapat mengimpornya ke aplikasi ini menggunakan pemilih file sistem. Untuk mengaktifkan pemilih, pilih **Buka File…** untuk memilih file atau **Buka Folder…** untuk memilih folder. Instruksi terperinci tentang cara mengimpor musik lokal yang tersimpan di iPhone atau Mac Anda tersedia [di sini](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).

Anda juga dapat menghubungkan folder yang terletak di perangkat Anda dan memiliki akses cepat ke kontennya. Gunakan item menu **Hubungkan Folder** dan pilih folder yang terletak di perangkat Anda. Ketuk **Selesai**, dan aplikasi membuat tautan ke folder tersebut dengan akses baca / tulis, memungkinkan Anda mengelola file langsung dari aplikasi ini. Untuk memutuskan koneksi folder yang terletak di perangkat Anda, ketuk tombol **Lebih banyak tindakan** dan pilih **Putuskan Koneksi**.

## Mengimpor File yang Terletak di Drive Flash USB yang Terhubung

Instruksi terperinci tentang cara menghubungkan drive flash USB ke iPhone dan mendengarkan musik atau mengelola file yang ada di dalamnya tersedia [di sini](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it).

## Lebih Banyak Tindakan

Menu Lebih banyak tindakan untuk folder yang sedang dibuka terletak di sudut kanan atas dan menyediakan akses ke berbagai tindakan.

- **Pilih** — beralih ke mode pemilihan untuk file dan folder.
- **Folder Baru** — buat folder baru di dalam folder saat ini.
- **Cari** — aktifkan fungsi pencarian untuk menemukan file atau folder tertentu.
- **Urutkan** — atur file berdasarkan nama, ukuran, tanggal, atau menggunakan informasi metadata.
- **Daftar / Kisi** — ubah presentasi konten dari tampilan tabel ke tampilan kisi (atau sebaliknya).
- **Bantuan** — akses instruksi bantuan.

## Mode Pemilihan

Jika Anda perlu mengedit beberapa file, aktifkan mode pemilihan dengan mengetuk tombol **Lebih banyak tindakan** **"..."** pada bilah navigasi di sudut kanan atas dan memilih item menu **Pilih**. Ini menampilkan kotak centang di dekat setiap file. Pilih file yang diinginkan dengan mengetuk kotak centangnya. Anda dapat melakukan tindakan berikut pada file yang dipilih:

- **Putar Berikutnya** — tambahkan file atau folder yang dipilih ke atas antrian pemutar dengan urutan sortir saat ini.
- **Putar Nanti** — tambahkan file atau folder yang dipilih ke bagian bawah antrian pemutar.
- **Tambah ke Perpustakaan Musik** — sertakan file atau folder yang dipilih dalam perpustakaan musik.
- **Tambah ke Daftar Putar** — tambahkan file atau folder yang dipilih ke daftar putar.
- **Salin** — buat salinan file atau folder yang dipilih.
- **Unggah** — unggah file atau folder yang dipilih ke penyimpanan jarak jauh.
- **Pindah** — pindahkan file atau folder yang dipilih ke lokasi yang berbeda di perangkat.
- **Ganti Nama** — ubah nama file atau folder yang dipilih.
- **Hapus** — hapus file atau folder yang dipilih dari perangkat. **Tindakan ini tidak dapat dibalik.**

## Menu Opsi

Untuk setiap file atau folder di aplikasi, beberapa tindakan tersedia, dapat diakses dengan mengetuk tombol **Lebih** **"..."**. Jika Anda tidak melihat semua tindakan, gulir ke bawah untuk melihatnya:

- **Putar Semua** — ganti antrian pemutar dengan item dari folder yang dipilih.
- **Putar Berikutnya** — tambahkan file atau folder ke atas antrian pemutar.
- **Putar Nanti** — tambahkan file atau folder ke bagian bawah antrian pemutar.
- **Tambah ke Perpustakaan Musik** — sertakan file atau folder yang dipilih dalam perpustakaan musik.
- **Tambah ke Daftar Putar** — tambahkan file atau folder ke daftar putar.
- **Tambah ke Favorit** — tandai file atau folder yang dipilih sebagai favorit.
- **Mengedit tag audio** — buka editor tag bawaan untuk mengubah tag audio untuk file yang dipilih.
- **Ganti Nama** — modifikasi nama file atau folder yang dipilih.
- **Pindah** — pindahkan file atau folder yang dipilih ke lokasi yang berbeda di perangkat.
- **Salin** — buat salinan file atau folder yang dipilih.
- **Unggah** — unggah file atau folder yang dipilih ke penyimpanan jarak jauh.
- **Tambah ke Arsip** — bundel file atau folder ke dalam satu file ZIP (fitur Premium).
- **Tampilkan di Finder** — untuk file yang diimpor dari Mac Anda, tindakan ini memperlihatkan direktori di Finder yang berisi file.
- **Buka Di** — ekspor file ke aplikasi lain.
- **Hapus** — hapus file atau folder dari perangkat. **Tindakan ini tidak dapat dibalik** dan Anda tidak dapat memulihkan file yang dihapus.

## Folder Offline

Mode offline adalah fitur yang berguna yang memungkinkan Anda mengakses musik favorit meskipun tidak terhubung ke internet. Ketika Anda mengaktifkan mode offline untuk album, artis, daftar putar, genre, atau folder jarak jauh mana pun, semua file dalam koleksi tersebut secara otomatis diunduh ke perangkat Anda untuk pemutaran offline. Anda dapat mengakses file-file ini dengan mudah di bagian **Folder Offline** aplikasi.

Yang lebih nyaman adalah ketika Anda menambahkan file baru ke server jarak jauh di folder yang telah Anda tandai untuk penggunaan offline, penambahan baru tersebut juga secara otomatis diunduh ke perangkat Anda. Ini memastikan Anda selalu memiliki musik terbaru siap untuk dinikmati secara offline — sempurna untuk perjalanan, penerbangan, atau berpindah antara jaringan Wi-Fi.

Jika Anda perlu menyinkronkan folder offline dari layanan cloud secara manual, begini caranya: cukup ketuk tiga titik di sudut kanan atas aplikasi dan pilih **Sinkronkan**. Ini memperbarui koleksi offline Anda dengan trek terbaru dari layanan cloud sesuai keinginan Anda.

Instruksi terperinci tentang cara memutar musik offline di Flacbox — termasuk cara mengunduh dan menyinkronkan dari cloud ke file lokal — tersedia [di sini](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Personalisasi

Dengan membuka pengaturan aplikasi dan pergi ke Personalisasi → Gaya Layar File Lokal, Anda dapat mengonfigurasi tampilan File Lokal.
Opsi yang tersedia:

- **Menu Biasa** — menampilkan folder yang terletak di direktori Dokumen aplikasi secara langsung.
- **Menu Dikelompokkan** — menampilkan konten layar yang dikelompokkan berdasarkan kategori (Akses Cepat, Folder Khusus, File di Aplikasi Ini, File di Perangkat Ini, dan sebagainya) untuk ikhtisar yang lebih rapi.

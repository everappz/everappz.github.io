Der Bereich „Lokale Dateien" dient als Zentrale zur Verwaltung von Dateien im Dokumentenordner der Anwendung sowie von Dateien, die von Ihrem Gerät für die Offline-Wiedergabe hinzugefügt wurden.

Dieser integrierte Datei-Manager ermöglicht Ihnen das Bearbeiten von Dateien (Umbenennen, Verschieben, Kopieren, Hochladen, Löschen), das Überwachen von Übertragungen und bietet mehrere Methoden zum Importieren von Audiodateien in die App.

## Dateien aus Cloud-Speicher herunterladen

Sie können Audiodateien zur App hinzufügen, indem Sie sie aus verbundenen Cloud-Speicherdiensten herunterladen, wie [hier](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag) beschrieben.

## Offline-Modus

Synchronisieren Sie ganze Ordner, Playlists, Alben, Künstler oder Genres automatisch für die Offline-Wiedergabe, wie [hier](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/) beschrieben. Der Offline-Modus hält Ihre lokale Kopie mit der Cloud synchron.

## Dateien von Ihrem Gerät importieren

Importieren Sie Dateien einfach von Ihrem Gerät, wie [hier](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac) beschrieben, oder von einem angeschlossenen USB-Laufwerk, wie [hier](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it) beschrieben. Sie können auch einen Ordner auf Ihrem Gerät verbinden — Flacbox erstellt einen Link dazu (Lesen/Schreiben).

## iTunes / Finder File Sharing

Übertragen Sie Dateien per Lightning- oder USB-C-Kabel, wie [hier](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder) beschrieben.

## Wi-Fi Drive

Übertragen Sie Dateien drahtlos über jeden Desktop-Browser, wie [hier](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive) beschrieben. Unter macOS können Sie Wi-Fi Drive auch direkt in Finder einbinden (Gehe zu → Mit Server verbinden…) und unter Windows im Datei-Explorer (Netzlaufwerk verbinden…).

## Übertragungs-Warteschlange

In der oberen linken Ecke der Navigationsleiste finden Sie eine Schaltfläche für Übertragungen. Tippen Sie darauf, um die Übertragungs-Warteschlange zu öffnen, wo Sie jeden Download und Upload überwachen und verwalten können — pausieren, fortsetzen, fehlgeschlagene Übertragungen wiederholen. Sie können auch die Übertragungsgeschwindigkeit in **Einstellungen → Datei-Manager** anpassen.

## Schnellzugriff-Bereich

Am oberen Rand des Bildschirms bietet ein Schnellzugriff-Bereich bequeme Links zu Ihren aktuellen und Lieblingsdateien und -ordnern.

## Aktuell

Dieser Bereich zeigt alle zuletzt geöffneten Dateien oder Ordner an.

## Favoriten

Sie können Dateien oder Ordner als Favoriten markieren und in diesem Bereich aufrufen. Um einen Ordner von Ihrem Gerät hinzuzufügen, öffnen Sie den Favoriten-Bereich, tippen Sie auf die drei Punkte oben rechts und wählen Sie „Ordner hinzufügen".

## Obere Symbolleiste

Die obere Symbolleiste bietet mehrere Aktionen:

- **Suchen** — eine Suche im aktuellen Ordner durchführen.
- **Wiedergabe fortsetzen** — stellt die Audio-Player-Warteschlange und die letzte Medienposition für den aktuellen Ordner wieder her.
- **Alle abspielen** — den aktuellen Ordner und seine Unterordner scannen und Dateien in die Warteschlange hinzufügen.
- **Alles zufällig abspielen** — ähnlich wie „Alle abspielen", mischt aber Dateien vorher.

Sie können die Symbolleiste durch eine Wisch-nach-unten-Geste ein- oder ausblenden.

## Spezielle Ordner

Im Bildschirm „Lokale Dateien" gibt es mehrere spezielle Ordner, die Flacbox zum Organisieren von Inhalten verwendet.

## Downloads

Alle aus einem Cloud-Speicherdienst heruntergeladenen Dateien erscheinen hier standardmäßig. Sie können dieses Verhalten unter **Einstellungen → Datei-Manager → Heruntergeladene Dateien speichern unter** anpassen.

## Audio-Player

Dies ist der Audio-Player-Cache-Ordner. Standardmäßig lädt der Audio-Player kommende Songs in der Warteschlange herunter, um Crossfade anzuwenden und die Wiedergabe zu optimieren. Sie können den Audio-Player-Cache in den App-Einstellungen deaktivieren oder diesen Ordner einfach aus „Lokale Dateien" löschen.

## iCloud

Legen Sie Dateien in diesem Ordner ab, und sie werden auf allen Ihren Geräten synchronisiert, die mit demselben iCloud-Konto verbunden sind.

## Offline-Ordner

Wenn Sie eine Playlist, ein Album, einen Künstler oder einen Online-Ordner für die Offline-Nutzung verfügbar machen, werden alle diese Dateien in diesem Ordner heruntergeladen.

## Zwei verschiedene Bereiche

Der Bildschirm „Lokale Dateien" ist in zwei verschiedene Bereiche unterteilt.

### Dateien in dieser Anwendung

Zeigt Dateien und Ordner an, die im Dokumentenverzeichnis der App und iCloud Drive gespeichert sind.

### Dateien auf diesem iPhone / iPad / Mac

Zeigt Dateien auf Ihrem Gerät in anderen Anwendungen an. Sie können sie mit der Systemdateiauswahl importieren. Aktivieren Sie die Auswahl mit **Dateien öffnen…** für Dateien oder **Ordner öffnen…** für Ordner. Detaillierte Anweisungen sind [hier](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac) verfügbar.

## Weitere Aktionen

Das Menü „Weitere Aktionen" für den aktuell geöffneten Ordner befindet sich in der oberen rechten Ecke.

- **Auswählen** — in den Auswahlmodus für Dateien und Ordner wechseln.
- **Neuer Ordner** — einen neuen Ordner im aktuellen Ordner erstellen.
- **Suchen** — die Suchfunktion aktivieren.
- **Sortieren** — Dateien nach Name, Größe, Datum oder Metadaten organisieren.
- **Liste / Raster** — die Inhaltsdarstellung von Tabellenansicht auf Rasteransicht ändern (oder umgekehrt).
- **Hilfe** — Hilfeanweisungen aufrufen.

## Auswahlmodus

Um mehrere Dateien zu bearbeiten, aktivieren Sie den Auswahlmodus über die Schaltfläche **Weitere Aktionen** **"..."**. Folgende Aktionen sind möglich:

- **Weiter abspielen** — ausgewählte Dateien oben in die Warteschlange hinzufügen.
- **Später abspielen** — ausgewählte Dateien unten in die Warteschlange hinzufügen.
- **Zur Musikbibliothek hinzufügen** — ausgewählte Dateien in die Musikbibliothek aufnehmen.
- **Zu einer Playlist hinzufügen** — ausgewählte Dateien einer Playlist hinzufügen.
- **Kopieren** — eine Kopie der ausgewählten Datei oder des Ordners erstellen.
- **Hochladen** — die ausgewählte Datei auf Remote-Speicher hochladen.
- **Verschieben** — die ausgewählte Datei an einen anderen Speicherort verschieben.
- **Umbenennen** — den Namen der ausgewählten Datei ändern.
- **Löschen** — die ausgewählte Datei vom Gerät entfernen. **Diese Aktion ist irreversibel.**

## Optionsmenü

Für jede Datei oder jeden Ordner in der App sind Aktionen verfügbar, die über die Schaltfläche **Weitere Aktionen** **"..."** zugänglich sind:

- **Alle abspielen** — die Warteschlange durch Elemente aus dem ausgewählten Ordner ersetzen.
- **Weiter abspielen** — die Datei oder den Ordner oben in die Warteschlange hinzufügen.
- **Später abspielen** — die Datei oder den Ordner unten in die Warteschlange hinzufügen.
- **Zur Musikbibliothek hinzufügen** — ausgewählte Dateien in die Musikbibliothek aufnehmen.
- **Zu einer Playlist hinzufügen** — die Datei einer Playlist hinzufügen.
- **Zu Favoriten hinzufügen** — ausgewählte Dateien als Favoriten markieren.
- **Audio-Tags bearbeiten** — den integrierten Tag-Editor öffnen.
- **Umbenennen** — den Namen ändern.
- **Verschieben** — an einen anderen Speicherort verschieben.
- **Kopieren** — eine Kopie erstellen.
- **Hochladen** — auf Remote-Speicher hochladen.
- **Zum Archiv hinzufügen** — in eine ZIP-Datei bündeln (Premium-Funktion).
- **Im Finder anzeigen** — für importierte Mac-Dateien: das Verzeichnis im Finder anzeigen.
- **Öffnen in** — die Datei in eine andere App exportieren.
- **Löschen** — die Datei vom Gerät löschen. **Diese Aktion ist irreversibel** und Sie können gelöschte Dateien nicht wiederherstellen.

## Offline-Ordner

Der Offline-Modus ist eine praktische Funktion, die es Ihnen ermöglicht, auf Ihre Lieblingsmusik zuzugreifen, auch wenn Sie keine Internetverbindung haben. Wenn Sie den Offline-Modus für ein Album, einen Künstler, eine Playlist, ein Genre oder einen Remote-Ordner aktivieren, werden alle Dateien in dieser Sammlung automatisch auf Ihr Gerät heruntergeladen.

Wenn Sie neue Dateien zum Remote-Server im Ordner hinzufügen, den Sie für den Offline-Modus markiert haben, werden diese neuen Hinzufügungen ebenfalls automatisch auf Ihr Gerät heruntergeladen.

Um einen Offline-Ordner manuell zu synchronisieren, tippen Sie auf die drei Punkte oben rechts und wählen Sie **Synchronisieren**.

Detaillierte Anweisungen sind [hier](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/) verfügbar.

## Personalisierung

Durch Öffnen der Anwendungseinstellungen und Navigieren zu **Personalisierung → Stil des Lokale-Dateien-Bildschirms** können Sie die Darstellung von „Lokale Dateien" konfigurieren.
Verfügbare Optionen sind:

- **Einfaches Menü** — zeigt Ordner direkt im Dokumentenverzeichnis der App an.
- **Gruppiertes Menü** — zeigt den Bildschirminhalt gruppiert nach Kategorien an (Schnellzugriff, Spezielle Ordner, Dateien in dieser Anwendung, Dateien auf diesem Gerät usw.) für eine übersichtlichere Darstellung.

Phần File cục bộ đóng vai trò là trung tâm quản lý các file nằm trong thư mục Documents của ứng dụng, cũng như các file được thêm từ thiết bị của bạn để phát ngoại tuyến.

Trình quản lý file tích hợp này cho phép chỉnh sửa file (đổi tên, di chuyển, sao chép, tải lên, xóa), theo dõi truyền tải và cung cấp một số phương pháp nhập file âm thanh vào ứng dụng — tải xuống trực tiếp từ đám mây, đồng bộ chế độ ngoại tuyến, USB flash drive, Wi-Fi Drive và Finder File Sharing.

## Tải xuống File từ Lưu trữ Đám mây

Bạn có thể thêm file âm thanh vào ứng dụng bằng cách tải xuống từ các dịch vụ lưu trữ đám mây đã kết nối như mô tả [tại đây](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag).

## Chế độ Ngoại tuyến

Tự động đồng bộ toàn bộ thư mục, danh sách phát, album, nghệ sĩ hoặc thể loại để phát ngoại tuyến như mô tả [tại đây](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/). Chế độ ngoại tuyến giữ bản sao cục bộ của bạn đồng bộ với đám mây — khi file mới được thêm trên máy chủ, chúng tự động được tải xuống.

## Nhập File từ Thiết bị

Dễ dàng nhập file từ thiết bị như mô tả [tại đây](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac), hoặc từ USB drive đính kèm như mô tả [tại đây](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it). Bạn cũng có thể kết nối một thư mục trên thiết bị — Flacbox tạo liên kết (đọc/ghi) để bạn có thể quản lý nội dung thư mục đó trực tiếp từ Flacbox.

## iTunes / Finder File Sharing

Truyền file qua cáp Lightning hoặc USB-C như mô tả [tại đây](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder). Trên macOS Catalina trở lên, Finder thay thế iTunes cho việc này — quy trình giống hệt nhau.

## Wi-Fi Drive

Truyền file không dây qua bất kỳ trình duyệt máy tính nào như mô tả [tại đây](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive). Trên macOS bạn cũng có thể gắn Wi-Fi Drive trực tiếp trong Finder (Đi → Kết nối với Máy chủ…) và trên Windows trong File Explorer (Ánh xạ Ổ đĩa Mạng…).

## Hàng đợi Truyền tải

Ở góc trên bên trái của thanh điều hướng, bạn sẽ thấy nút Truyền tải. Nhấn nó để truy cập hàng đợi truyền tải, nơi bạn có thể theo dõi và quản lý mọi tải xuống và tải lên — tạm dừng, tiếp tục, thử lại các truyền tải thất bại và xem tiến trình chi tiết. Bạn cũng có thể điều chỉnh tốc độ hàng đợi truyền tải trong Cài đặt → Trình quản lý File.

## Phần Truy cập Nhanh

Ở đầu màn hình, phần Truy cập Nhanh cung cấp các liên kết thuận tiện đến file và thư mục gần đây và yêu thích của bạn.

## Gần đây

Phần này hiển thị mọi file hoặc thư mục được mở gần đây, để bạn có thể quay lại không gian làm việc mới nhất với một lần chạm.

## Yêu thích

Bạn có thể đánh dấu file hoặc thư mục là yêu thích và truy cập chúng trong phần này. Bạn cũng có thể thêm thư mục trên thiết bị vào yêu thích. Để làm điều này, mở phần Yêu thích, nhấn ba chấm ở góc trên bên phải và chọn Thêm Thư mục.

## Thanh Công cụ Trên cùng

Thanh công cụ trên cùng, nằm bên dưới thanh điều hướng, cung cấp một số hành động:

- **Tìm kiếm** — thực hiện tìm kiếm trong thư mục hiện tại.
- **Tiếp tục phát** — nếu được bật trong cài đặt ứng dụng, tính năng này khôi phục hàng đợi phát và vị trí phát cuối cùng.
- **Phát tất cả** — quét thư mục hiện tại và thư mục con, rồi thêm file vào hàng đợi phát.
- **Phát ngẫu nhiên** — tương tự Phát tất cả, nhưng xáo trộn file trước khi thêm vào hàng đợi.

Bạn có thể hiển thị hoặc ẩn thanh công cụ trên cùng bằng cử chỉ vuốt xuống.

## Thư mục Đặc biệt

Trong màn hình File cục bộ, bạn sẽ gặp một số thư mục đặc biệt mà Flacbox sử dụng để tổ chức nội dung.

## Tải xuống

Tất cả file tải xuống từ dịch vụ lưu trữ đám mây sẽ mặc định xuất hiện ở đây. Bạn có thể tùy chỉnh hành vi này trong Cài đặt → Trình quản lý File → Lưu File Đã tải xuống vào.

## Trình phát âm thanh

Đây là thư mục bộ nhớ đệm của trình phát âm thanh. Theo mặc định, trình phát âm thanh tải xuống các bài hát tiếp theo trong hàng đợi để áp dụng crossfade và tối ưu hóa phát lại. Bạn có thể tắt bộ nhớ đệm trình phát âm thanh trong cài đặt ứng dụng hoặc đơn giản là xóa thư mục này từ File cục bộ để giải phóng dung lượng.

## iCloud

Đặt file vào thư mục này, chúng sẽ đồng bộ trên tất cả thiết bị kết nối với cùng tài khoản iCloud — tiện lợi để giữ một bộ nhạc nhỏ phải có ở mọi nơi.

## Thư mục Ngoại tuyến

Khi bạn làm cho danh sách phát, album, nghệ sĩ hoặc thư mục trực tuyến có thể dùng ngoại tuyến, tất cả file đó được tải xuống vào thư mục này. Nếu bạn cần tắt chế độ ngoại tuyến cho các mục cụ thể, hãy sử dụng menu Thêm Hành động hoặc xóa thư mục tương ứng trong Thư mục Ngoại tuyến.

## Hai Phần Riêng biệt

Màn hình File cục bộ được tổ chức thành hai phần riêng biệt.

### File trong Ứng dụng này

Hiển thị file và thư mục được lưu trong thư mục Documents của ứng dụng và iCloud Drive.

### File trên iPhone / iPad / Mac này

Hiển thị các file nằm trên thiết bị nhưng trong các ứng dụng khác. Bạn có thể nhập chúng vào ứng dụng này bằng cách sử dụng bộ chọn file hệ thống. Để kích hoạt bộ chọn, chọn **Mở File…** để chọn file hoặc **Mở Thư mục…** để chọn thư mục. Hướng dẫn chi tiết về cách nhập nhạc cục bộ có sẵn [tại đây](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac).

Nhấn **Hoàn tất**, ứng dụng tạo liên kết đến thư mục đó với quyền đọc / ghi. Để ngắt kết nối thư mục, nhấn nút **Thêm Hành động** và chọn **Ngắt kết nối**.

## Thêm Hành động

Menu Thêm Hành động cho thư mục hiện đang mở nằm ở góc trên bên phải và cung cấp quyền truy cập vào các hành động khác nhau.

- **Chọn** — chuyển sang chế độ chọn file và thư mục.
- **Thư mục mới** — tạo thư mục mới trong thư mục hiện tại.
- **Tìm kiếm** — kích hoạt chức năng tìm kiếm.
- **Sắp xếp** — tổ chức file theo tên, kích thước, ngày hoặc siêu dữ liệu.
- **Danh sách / Lưới** — thay đổi cách trình bày nội dung.
- **Trợ giúp** — truy cập hướng dẫn.

## Chế độ Chọn

Nếu bạn cần chỉnh sửa nhiều file, kích hoạt chế độ chọn bằng cách nhấn nút **«...»** ở góc trên bên phải và chọn mục **Chọn**. Hộp kiểm xuất hiện bên cạnh mỗi file. Chọn các file bạn muốn và thực hiện các hành động sau:

- **Phát tiếp** — thêm file hoặc thư mục đã chọn vào đầu hàng đợi phát.
- **Phát sau** — thêm file hoặc thư mục đã chọn vào cuối hàng đợi phát.
- **Thêm vào Thư viện Nhạc** — bao gồm file hoặc thư mục đã chọn trong thư viện nhạc.
- **Thêm vào Danh sách phát** — thêm file hoặc thư mục đã chọn vào danh sách phát.
- **Sao chép** — tạo bản sao của file hoặc thư mục đã chọn.
- **Tải lên** — tải file hoặc thư mục đã chọn lên bộ lưu trữ từ xa.
- **Di chuyển** — chuyển file hoặc thư mục đã chọn đến vị trí khác trên thiết bị.
- **Đổi tên** — thay đổi tên file hoặc thư mục đã chọn.
- **Xóa** — xóa file hoặc thư mục đã chọn khỏi thiết bị. **Hành động này không thể đảo ngược.**

## Menu Tùy chọn

Đối với mọi file hoặc thư mục trong ứng dụng, có thể thực hiện một số hành động bằng cách nhấn nút **«...»**:

- **Phát tất cả** — thay thế hàng đợi phát bằng các mục từ thư mục đã chọn.
- **Phát tiếp** — thêm file hoặc thư mục vào đầu hàng đợi phát.
- **Phát sau** — thêm file hoặc thư mục vào cuối hàng đợi phát.
- **Thêm vào Thư viện Nhạc** — bao gồm file hoặc thư mục đã chọn trong thư viện nhạc.
- **Thêm vào Danh sách phát** — thêm file hoặc thư mục vào danh sách phát.
- **Thêm vào Yêu thích** — đánh dấu file hoặc thư mục đã chọn là yêu thích.
- **Chỉnh sửa thẻ âm thanh** — mở trình chỉnh sửa thẻ tích hợp.
- **Đổi tên** — chỉnh sửa tên file hoặc thư mục đã chọn.
- **Di chuyển** — chuyển file hoặc thư mục đã chọn đến vị trí khác trên thiết bị.
- **Sao chép** — tạo bản sao của file hoặc thư mục đã chọn.
- **Tải lên** — tải file hoặc thư mục đã chọn lên bộ lưu trữ từ xa.
- **Thêm vào Kho lưu trữ** — đóng gói file hoặc thư mục thành một file ZIP duy nhất (tính năng Premium).
- **Hiển thị trong Finder** — đối với file được nhập từ Mac, hành động này hiển thị thư mục chứa file trong Finder.
- **Mở bằng** — xuất file sang ứng dụng khác.
- **Xóa** — xóa file hoặc thư mục khỏi thiết bị. **Hành động này không thể đảo ngược** và bạn không thể khôi phục file đã xóa.

## Thư mục Ngoại tuyến

Chế độ ngoại tuyến là tính năng tiện lợi cho phép bạn truy cập nhạc yêu thích ngay cả khi không có kết nối internet. Khi bạn bật chế độ ngoại tuyến cho bất kỳ album, nghệ sĩ, danh sách phát, thể loại hoặc thư mục từ xa nào, tất cả file trong bộ sưu tập đó được tự động tải xuống thiết bị để phát ngoại tuyến. Bạn có thể thuận tiện truy cập các file này trong phần **Thư mục Ngoại tuyến** của ứng dụng.

Khi bạn thêm file mới vào máy chủ từ xa trong thư mục đã đánh dấu để dùng ngoại tuyến, những bổ sung mới đó cũng tự động được tải xuống thiết bị của bạn — hoàn hảo cho việc đi du lịch, chuyến bay hoặc di chuyển giữa các mạng Wi-Fi.

Nếu bạn cần đồng bộ thủ công thư mục ngoại tuyến, chỉ cần nhấn ba chấm ở góc trên bên phải và chọn **Đồng bộ hóa**.

Hướng dẫn chi tiết về cách phát nhạc ngoại tuyến trong Flacbox có sẵn [tại đây](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/).

## Cá nhân hóa

Bằng cách mở cài đặt ứng dụng và đi đến Cá nhân hóa → Kiểu Màn hình File cục bộ, bạn có thể cấu hình cách trình bày File cục bộ.
Các tùy chọn có sẵn:

- **Menu Đơn giản** — hiển thị thư mục nằm trong thư mục Documents của ứng dụng trực tiếp.
- **Menu Nhóm** — hiển thị nội dung màn hình được nhóm theo danh mục (Truy cập Nhanh, Thư mục Đặc biệt, File trong Ứng dụng này, File trên Thiết bị này, v.v.) để có cái nhìn tổng quan gọn gàng hơn.

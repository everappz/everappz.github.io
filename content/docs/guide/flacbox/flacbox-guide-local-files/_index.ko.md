로컬 파일 섹션은 애플리케이션의 Documents 폴더에 있는 파일과 기기에서 추가된 오프라인 재생 가능한 파일을 관리하는 허브 역할을 합니다.

이 내장 파일 관리자를 사용하면 파일을 편집(이름 바꾸기, 이동, 복사, 업로드, 삭제)하고, 전송을 모니터링하며, 클라우드에서 직접 다운로드, 오프라인 모드 동기화, USB 플래시 드라이브, Wi-Fi Drive, Finder 파일 공유 등 여러 방법으로 오디오 파일을 앱으로 가져올 수 있습니다.

## 클라우드 스토리지에서 파일 다운로드

[여기](/docs/howto/how-to-upload-my-files-to-the-cloud-storage-and-connect-them-to-evermusic-flacbox-evertag)에서 설명한 대로 연결된 클라우드 스토리지 서비스에서 오디오 파일을 다운로드하여 앱에 추가할 수 있습니다.

## 오프라인 모드

[여기](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)에서 설명한 대로 오프라인 재생을 위해 전체 폴더, 재생 목록, 앨범, 아티스트 또는 장르를 자동으로 동기화합니다. 오프라인 모드는 로컬 복사본을 클라우드와 동기화 상태로 유지합니다 — 서버에 새 파일이 추가되면 자동으로 다운로드됩니다.

## 기기에서 파일 가져오기

[여기](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac)에서 설명한 대로 기기에서 파일을 쉽게 가져오거나, [여기](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it)에서 설명한 대로 연결된 USB 드라이브에서 파일을 가져올 수 있습니다. 기기에 있는 폴더에 연결할 수도 있으며 — Flacbox는 해당 폴더에 대한 링크(읽기/쓰기)를 생성하여 Flacbox 내에서 직접 폴더 내용을 관리할 수 있습니다.

## iTunes / Finder 파일 공유

[여기](/docs/howto/how-to-transfer-files-from-my-mac-to-iphone-or-ipad-using-finder)에서 설명한 대로 Lightning 또는 USB-C 케이블을 사용하여 파일을 전송합니다. macOS Catalina 이상에서는 Finder가 iTunes를 대체하며 절차는 동일합니다.

## Wi-Fi Drive

[여기](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)에서 설명한 대로 모든 데스크톱 브라우저를 사용하여 무선으로 파일을 전송합니다. macOS에서는 Finder(이동 → 서버에 연결…)에서, Windows에서는 파일 탐색기(네트워크 드라이브 연결…)에서 Wi-Fi Drive를 직접 마운트할 수도 있습니다.

## 전송 큐

탐색 바의 왼쪽 상단 모서리에 전송 버튼이 있습니다. 이것을 탭하면 전송 큐에 액세스할 수 있으며, 여기서 모든 다운로드와 업로드를 모니터링하고 관리할 수 있습니다 — 일시 중지, 재개, 실패한 전송 재시도, 항목별 세부 진행률 확인. 설정 → 파일 관리자에서 전송 큐 속도(최대 병렬 작업)와 네트워크 유형을 조정할 수도 있습니다.

## 빠른 액세스 섹션

화면 상단의 빠른 액세스 섹션에서 최근 파일과 즐겨찾기 파일 및 폴더에 대한 편리한 링크를 제공합니다.

## 최근 항목

이 섹션에는 최근에 열린 모든 파일이나 폴더가 표시되므로 한 번의 탭으로 최신 작업 공간으로 돌아갈 수 있습니다.

## 즐겨찾기

파일이나 폴더를 즐겨찾기로 표시하고 이 섹션에서 액세스할 수 있습니다. 기기에 있는 폴더를 즐겨찾기에 추가할 수도 있습니다. 이를 위해 즐겨찾기 섹션을 열고, 오른쪽 상단 모서리의 점 세 개를 탭하고, 폴더 추가를 선택하세요.

## 상단 도구 모음

탐색 바 아래에 있는 상단 도구 모음은 여러 작업을 제공합니다:

- **검색** — 현재 폴더 내에서 검색을 수행합니다.
- **계속 재생** — 애플리케이션 설정에서 활성화된 경우 이 기능은 오디오 플레이어 큐와 현재 폴더의 마지막 미디어 위치를 복원하여 중단된 곳에서 다시 시작할 수 있습니다.
- **모두 재생** — 현재 폴더와 하위 폴더를 스캔한 다음 현재 정렬 순서로 파일을 플레이어 큐에 추가합니다.
- **모두 셔플** — 모두 재생과 유사하지만 파일을 오디오 플레이어 큐에 추가하기 전에 섞습니다.

아래로 스와이프하여 상단 도구 모음을 표시하거나 숨길 수 있습니다.

## 특수 폴더

로컬 파일 화면에서 Flacbox가 내용을 구성하는 데 사용하는 여러 특수 폴더를 만날 수 있습니다.

## 다운로드

클라우드 스토리지 서비스에서 다운로드한 모든 파일은 기본적으로 여기에 나타납니다. 설정 → 파일 관리자 → 다운로드 파일 저장 위치에서 이 동작을 맞춤 설정할 수 있습니다.

## 오디오 플레이어

이것은 오디오 플레이어 캐시 폴더입니다. 기본적으로 오디오 플레이어는 크로스페이드를 적용하고 재생을 최적화하기 위해 플레이어 큐의 다음 곡을 다운로드합니다. 앱 설정에서 오디오 플레이어 캐시를 비활성화하거나 로컬 파일에서 이 폴더를 삭제하여 공간을 확보할 수 있습니다.

## iCloud

이 폴더에 파일을 넣으면 동일한 iCloud 계정에 연결된 모든 기기에서 동기화됩니다 — 제3자 클라우드에 업로드하지 않고도 꼭 필요한 트랙을 어디서나 사용할 수 있게 하는 편리한 기능입니다.

## 오프라인 폴더

재생 목록, 앨범, 아티스트 또는 온라인 폴더를 오프라인에서 사용할 수 있도록 설정하면 모든 파일이 이 폴더로 다운로드됩니다. 특정 항목에 대한 오프라인 모드를 비활성화해야 하는 경우 추가 작업 메뉴를 사용하거나 오프라인 폴더에 있는 해당 폴더를 삭제할 수 있습니다.

## 두 개의 별개 섹션

로컬 파일 화면은 두 개의 별개 섹션으로 구성됩니다.

### 이 애플리케이션의 파일

앱의 Documents 디렉토리와 iCloud Drive에 저장된 파일 및 폴더를 표시합니다.

### 이 iPhone / iPad / Mac의 파일

기기에 있지만 다른 애플리케이션에 있는 파일을 표시합니다. 시스템 파일 선택기를 사용하여 이 애플리케이션으로 가져올 수 있습니다. 선택기를 활성화하려면 파일을 선택하려면 **파일 열기…**를, 폴더를 선택하려면 **폴더 열기…**를 선택하세요. iPhone 또는 Mac에 저장된 로컬 음악을 가져오는 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-play-local-music-stored-on-your-iphone-or-mac)에서 확인할 수 있습니다.

기기에 있는 폴더에 연결하고 그 내용에 빠르게 액세스할 수도 있습니다. **폴더 연결** 메뉴 항목을 사용하고 기기에 있는 폴더를 선택하세요. **완료됨**을 탭하면 앱이 읽기 / 쓰기 액세스 권한이 있는 해당 폴더에 대한 링크를 만들어 이 앱에서 직접 파일을 관리할 수 있습니다. 기기에 있는 폴더의 연결을 끊으려면 **추가 작업** 버튼을 탭하고 **연결 끊기**를 선택하세요.

## 연결된 USB 플래시 드라이브에 있는 파일 가져오기

iPhone에 USB 플래시 드라이브를 연결하고 음악을 듣거나 그 안에 있는 파일을 관리하는 방법에 대한 자세한 지침은 [여기](/docs/howto/how-to-connect-a-usb-flashcard-to-the-iphone-and-listen-to-music-or-manage-files-located-on-it)에서 확인할 수 있습니다.

## 추가 작업

현재 열린 폴더의 추가 작업 메뉴는 오른쪽 상단 모서리에 있으며 다양한 작업에 대한 액세스를 제공합니다.

- **선택하다** — 파일 및 폴더에 대한 선택 모드로 전환합니다.
- **새 폴더** — 현재 폴더 내에 새 폴더를 만듭니다.
- **검색** — 특정 파일이나 폴더를 찾기 위해 검색 기능을 활성화합니다.
- **정렬** — 이름, 크기, 날짜 또는 메타데이터 정보를 사용하여 파일을 구성합니다.
- **목록 / 그리드** — 테이블 보기에서 그리드 보기로(또는 반대로) 콘텐츠 표시를 변경합니다.
- **도움말** — 도움말 지침에 액세스합니다.

## 선택 모드

여러 파일을 편집해야 하는 경우 탐색 바의 오른쪽 상단 모서리에 있는 **추가 작업** 버튼 **"..."**을 탭하고 **선택하다** 메뉴 항목을 선택하여 선택 모드를 활성화합니다. 이렇게 하면 각 파일 옆에 체크박스가 표시됩니다. 체크박스를 탭하여 원하는 파일을 선택합니다. 선택한 파일에 대해 다음 작업을 수행할 수 있습니다:

- **다음 재생** — 선택한 파일이나 폴더를 현재 정렬 순서로 플레이어 큐의 상단에 추가합니다.
- **나중에 재생** — 선택한 파일이나 폴더를 현재 정렬 순서로 플레이어 큐의 하단에 추가합니다.
- **음악 라이브러리에 추가** — 선택한 파일이나 폴더를 음악 라이브러리에 포함합니다.
- **재생 목록에 추가** — 선택한 파일이나 폴더를 재생 목록에 추가합니다.
- **복사** — 선택한 파일이나 폴더의 복사본을 만듭니다.
- **업로드** — 선택한 파일이나 폴더를 원격 스토리지에 업로드합니다.
- **이동** — 선택한 파일이나 폴더를 기기의 다른 위치로 이동합니다.
- **이름 바꾸기** — 선택한 파일이나 폴더의 이름을 변경합니다.
- **삭제하기** — 기기에서 선택한 파일이나 폴더를 제거합니다. **이 작업은 되돌릴 수 없습니다.**

## 옵션 메뉴

앱의 모든 파일이나 폴더에 대해 **추가 작업** 버튼 **"..."**을 탭하여 액세스할 수 있는 여러 작업이 있습니다:

- **모두 재생** — 선택한 폴더의 항목으로 플레이어 큐를 교체합니다.
- **다음 재생** — 파일이나 폴더를 플레이어 큐의 상단에 추가합니다.
- **나중에 재생** — 파일이나 폴더를 플레이어 큐의 하단에 추가합니다.
- **음악 라이브러리에 추가** — 선택한 파일이나 폴더를 음악 라이브러리에 포함합니다.
- **재생 목록에 추가** — 파일이나 폴더를 재생 목록에 추가합니다. 폴더에서 재생 목록을 만들면 재생 목록 이름이 자동으로 지정됩니다.
- **즐겨찾기에 추가** — 선택한 파일이나 폴더를 즐겨찾기로 표시합니다.
- **오디오 태그 편집** — 선택한 파일의 오디오 태그를 변경하기 위해 내장 태그 편집기를 엽니다.
- **이름 바꾸기** — 선택한 파일이나 폴더의 이름을 수정합니다.
- **이동** — 선택한 파일이나 폴더를 기기의 다른 위치로 이동합니다.
- **복사** — 선택한 파일이나 폴더의 복사본을 만듭니다.
- **업로드** — 선택한 파일이나 폴더를 원격 스토리지에 업로드합니다.
- **아카이브에 추가** — 파일이나 폴더를 단일 ZIP 파일로 묶습니다(프리미엄 기능).
- **Finder에서 보기** — Mac에서 가져온 파일의 경우 파일이 포함된 디렉토리를 Finder에 표시합니다.
- **다른 앱으로 열기** — 파일을 다른 앱으로 내보냅니다.
- **삭제하기** — 기기에서 파일이나 폴더를 삭제합니다. **이 작업은 되돌릴 수 없으며** 삭제된 파일을 복원할 수 없습니다.

## 오프라인 폴더

오프라인 모드는 인터넷에 연결되지 않아도 즐겨찾는 음악에 액세스할 수 있는 편리한 기능입니다. 앨범, 아티스트, 재생 목록, 장르 또는 원격 폴더에 대해 오프라인 모드를 활성화하면 해당 컬렉션의 모든 파일이 오프라인 재생을 위해 기기로 자동 다운로드됩니다. 앱의 **오프라인 폴더** 섹션에서 편리하게 이러한 파일에 액세스할 수 있습니다.

더욱 편리한 점은 오프라인 사용으로 표시한 폴더의 원격 서버에 새 파일을 추가하면 해당 새 파일도 기기로 자동으로 다운로드됩니다. 이를 통해 여행, 비행 또는 Wi-Fi 네트워크 사이를 이동할 때도 항상 최신 음악을 오프라인으로 즐길 수 있습니다.

클라우드 서비스에서 오프라인 폴더를 수동으로 동기화해야 하는 경우 다음과 같이 하세요: 앱의 오른쪽 상단 모서리에 있는 점 세 개를 탭하고 **동기화하다**를 선택합니다.

자세한 지침은 [여기](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)에서 확인할 수 있습니다.

## 개인화

애플리케이션 설정을 열고 개인화 → 로컬 파일 화면 스타일로 이동하여 로컬 파일 표시를 구성할 수 있습니다.
사용 가능한 옵션:

- **일반 메뉴** — 앱의 Documents 디렉토리에 있는 폴더를 직접 표시합니다.
- **그룹화 메뉴** — 더 깔끔한 개요를 위해 카테고리별로 그룹화된 화면 내용을 표시합니다(빠른 액세스, 특수 폴더, 이 애플리케이션의 파일, 이 기기의 파일 등).

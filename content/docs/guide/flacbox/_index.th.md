### ยินดีต้อนรับสู่คู่มือ Flacbox

Flacbox คือเครื่องเล่นเพลงความละเอียดสูงสำหรับ iPhone, iPad และ Mac ที่ช่วยให้คุณเปลี่ยนที่เก็บข้อมูลคลาวด์, NAS และเซิร์ฟเวอร์มีเดียที่คุณชื่นชอบให้เป็นบริการสตรีมมิ่งส่วนตัวของคุณเอง

Flacbox เชื่อมต่อกับแหล่งที่มาที่หลากหลายอย่างน่าทึ่ง ทั้งหมดในแอปเดียว:

- **ที่เก็บข้อมูลคลาวด์ส่วนตัว:** iCloud Drive · Google Drive · Dropbox · OneDrive · Box · MEGA · pCloud · Yandex Disk · MediaFire · WD My Cloud Home · TeraCLOUD (InfiniCLOUD) · HiDrive · IceDrive · Koofr · OpenDrive · MyDrive · Put.io · Cloud Mail.ru · Internxt · Proton Drive · AliDrive (阿里云盘) · Baidu Pan (百度网盘)
- **เซิร์ฟเวอร์โฮสต์เอง:** Plex · Jellyfin · Emby · Subsonic (และทุกเซิร์ฟเวอร์ที่เข้ากันได้กับ Subsonic — Airsonic, Funkwhale, Gonic, Logitech Media Server, Ampache) · Navidrome · Nextcloud (และ ownCloud ผ่าน API ที่แชร์) · Synology Drive · QNAP
- **NAS และโปรโตคอลการแชร์ไฟล์:** SMB (SMB1, SMB2, Auto) · WebDAV (HTTP / HTTPS) · FTP / FTPS · SFTP (รหัสผ่านหรือการตรวจสอบสิทธิ์ด้วยคีย์สาธารณะ) · NFS · DLNA / UPnP (การเล่นและดาวน์โหลด) ทำงานกับ Apple Time Capsule, Synology, QNAP, WD My Cloud, โฮสต์ Linux Samba / NFS / SSH ใดๆ หรือโฟลเดอร์ที่แชร์บน Mac หรือ Windows PC ของคุณ
- **ที่เก็บข้อมูลอ็อบเจกต์ที่เข้ากันได้กับ S3:** AWS S3, Backblaze B2, Wasabi, Cloudflare R2, MinIO, DigitalOcean Spaces และ S3-API endpoint อื่นๆ
- **การค้นพบเครือข่ายท้องถิ่น:** ส่วน อุปกรณ์ที่มีให้บริการ จะแสดงรายการบริการ Bonjour / mDNS ทุกรายการบนเครือข่าย Wi-Fi ของคุณโดยอัตโนมัติ — เซิร์ฟเวอร์ Plex, Jellyfin, Emby, Synology, QNAP, เราเตอร์ AirPort ที่มีไดรฟ์เชื่อมต่อ, Time Capsule — เพื่อให้คุณแตะเพื่อเชื่อมต่อโดยไม่ต้องพิมพ์ที่อยู่ IP

ในขณะที่แอปเพลงส่วนใหญ่ใช้เครื่องมือเสียงในตัวของ Apple เท่านั้น Flacbox รวม **FFmpeg** เข้ากับโคเดกของระบบเพื่อให้คุณสามารถเล่นรูปแบบที่ iOS ไม่รองรับ — WMA, OGG, OPUS, APE, DSD, DSF, DFF, TTA, MPC, WV รวมถึงตระกูล MP3 / AAC / M4A / WAV / AIFF / ALAC / FLAC ปกติ เมื่อรวมกับอัตราการสุ่มตัวอย่างเสียงออกพุตที่กำหนดค่าได้ (44.1 / 48 / 64 / 88.2 / 96 kHz), เอาต์พุตหลายช่อง (Mono → 5.1 → SDDS → ITU BS.775-1), การปรับบัฟเฟอร์ IO และการแก้ไขระดับเสียง Flacbox ให้การควบคุมแก่นักเล่นเสียงที่แอปเพลงสำหรับผู้บริโภคส่วนใหญ่ไม่มีให้

### เพลิดเพลินกับการสตรีมที่ราบรื่นและการเล่นออฟไลน์

Flacbox มีการบัฟเฟอร์อัจฉริยะสำหรับการสตรีมที่ราบรื่นและตัวจัดการดาวน์โหลดในตัวเพื่อให้คุณดึงเพลย์ลิสต์, ศิลปิน, อัลบั้ม, โฟลเดอร์หรือแทร็คแต่ละรายการลงในอุปกรณ์ของคุณเพื่อใช้ออฟไลน์ เมื่อพื้นที่จัดเก็บเหลือน้อย ล้างไฟล์แคชด้วยการแตะครั้งเดียวและสตรีมต่อจากคลาวด์ โหมดออฟไลน์สำหรับโฟลเดอร์, เพลย์ลิสต์, อัลบั้มและศิลปินยังซิงค์แทร็คใหม่โดยอัตโนมัติทันทีที่ปรากฏในคลาวด์ ทำให้คอลเลกชันออฟไลน์ของคุณเป็นปัจจุบันเสมอ

### คลังเพลงที่จัดระเบียบโดยอัตโนมัติ

เมื่อคุณนำเข้าแทร็คเสียง Flacbox จะสแกนข้อมูลเมตาและจัดระเบียบเป็นคลังเพลงที่สะอาด — จัดกลุ่มตาม เพลง, เพลงที่ยังไม่ได้เล่น, อัลบั้ม, ศิลปินอัลบั้ม, ศิลปิน, แนวเพลง และนักแต่งเพลง ใช้การค้นหาในตัวเพื่อค้นหาอะไรก็ได้ภายในไม่กี่วินาที กรองตามแหล่งที่มา (คลาวด์ออนไลน์ / ออฟไลน์ / อุปกรณ์) และเลือกระหว่างเลย์เอาต์คลัง Plain / Grouped / Tabbed สำหรับคลังที่มีอัลบั้มรวมเพลง "various artists" แบบผสม Flacbox มีมุมมอง อัลบั้มทั้งหมด / อัลบั้มเฉพาะ / อัลบั้มเดี่ยว เพื่อแสดงผลลัพธ์ที่ถูกต้องโดยไม่มีสัญญาณรบกวน

### แก้ไขข้อมูลเมตาและแท็กเพลงของคุณ

หากคุณพบแท็กที่เสียหายหรือการเข้ารหัสที่ยุ่งเหยิง (ปัญหาทั่วไปสำหรับไฟล์ที่ริพหรือไฟล์เก่า) ตัวแก้ไขแท็ก ID3 ในตัวสามารถทำความสะอาดได้ — ด้วยตนเองหรือด้วยการค้นหา MusicBrainz อัตโนมัติ คุณยังสามารถทำให้การเข้ารหัสอักขระที่เสียหายเป็นมาตรฐาน (เหมาะสำหรับแท็กภาษาซีริลลิก, ญี่ปุ่น หรือจีนที่มาจาก Windows PC) ค้นหาภาพปกอัลบั้มที่หายไป และเขียนการเปลี่ยนแปลงกลับไปยังไฟล์ต้นฉบับในคลาวด์โดยอัตโนมัติ สำหรับการแก้ไขแบบกลุ่มที่ลึกกว่านั้น ติดตั้งแอปเพื่อนร่วมทีม Evertag

### การโอนย้ายง่ายจาก Mac, PC หรือ NAS

ย้ายเพลงระหว่างคอมพิวเตอร์กับ iPhone หรือ iPad ของคุณด้วยเครื่องมือใดก็ได้เหล่านี้: SMB, WebDAV, DLNA, Wi-Fi Drive (ลากและวางในเว็บเบราว์เซอร์เดสก์ท็อปใดก็ได้), iTunes / Finder File Sharing ผ่านสาย Lightning หรือ USB-C, แฟลชไดรฟ์ USB หรือ iXpand Flash Drives มี Apple Time Capsule, WD My Cloud, Synology, QNAP หรือ NAS อื่นๆ ที่เปิดเผย SMB / WebDAV / DLNA / FTP / SFTP ไหม? เชื่อมต่อครั้งเดียวและสตรีมคลังเพลงทั้งหมดโดยไม่ใช้พื้นที่จัดเก็บในอุปกรณ์

### ปรับแต่งเสียงด้วย Equalizer

Flacbox มี equalizer 10 แบนด์พร้อมการตั้งค่าล่วงหน้าสไตล์ iPod (Acoustic, Bass Booster, Classical, Dance, Rock, Pop, Jazz และอีกมากมาย), พรีแอมป์ และความสามารถในการบันทึกการตั้งค่าล่วงหน้าของคุณเอง ไม่ว่าจะปรับสำหรับหูฟัง IEM ระดับ audiophile, HomePod mini หรือระบบเสียงในรถ คุณสามารถปรับแต่งเสียงตามที่คุณต้องการได้

### เหมาะสำหรับหนังสือเสียง

Flacbox เป็นเครื่องเล่นหนังสือเสียงที่ยอดเยี่ยม — บุ๊คมาร์คหลายรายการต่อแทร็ค, การควบคุมความเร็วในการเล่นที่ละเอียด (0.02× ถึง 3.00×), ดำเนินการเล่นต่อจากจุดที่หยุดไว้, ปุ่มข้ามเวลาที่ปรับแต่งได้ และตัวจับเวลาสลีปที่ค่อยๆ ลดระดับเสียงก่อนนอน รองรับเครื่องหมายบทของ M4B และไฟล์ยาวได้อย่างสมบูรณ์

### สตรีมได้ทุกที่ — รวมถึงในรถและหน้าจอหลัก

สตรีมเพลงไปยัง Apple TV / HomePod ผ่าน AirPlay 2 ส่งไปยังลำโพง Google Chromecast และทีวีที่รองรับ Cast และนำทุกอย่างไปด้วยบนท้องถนนด้วย Apple CarPlay ใช้วิดเจ็ตหน้าจอหลักบน iPhone และ iPad เพื่อดูแทร็คที่กำลังเล่นอยู่ในขณะนี้แวบเดียว และ Last.fm scrobbling เพื่อเก็บประวัติการฟังของคุณในทุกแอป

### ความเป็นส่วนตัวและความปลอดภัย

Flacbox ใช้เฉพาะ SDK อย่างเป็นทางการและการเข้าสู่ระบบแบบ OAuth จากผู้ให้บริการคลาวด์แต่ละราย — รหัสผ่านของคุณจะไม่ถึงแอปเลย โทเคนการเข้าถึงถูกเข้ารหัสใน iOS Keychain, การถ่ายโอนทั้งหมดได้รับการป้องกันด้วย TLS และการเพิกถอนการเข้าถึงในบัญชีคลาวด์หรือการลบ Flacbox ออกจากอุปกรณ์จะลบทุกอย่างทันที ปกป้องคลังเพลงของคุณด้วยรหัสผ่านที่เป็นตัวเลือกสำหรับชั้นความเป็นส่วนตัวเพิ่มเติม

### เริ่มต้นใช้งาน Flacbox

คู่มือนี้จะพาคุณผ่านทุกส่วนของ Flacbox บน iPhone, iPad และ Mac — ตั้งแต่การเชื่อมต่อบริการคลาวด์, การจัดระเบียบคลัง, การถ่ายโอนไฟล์ และการจัดการเพลย์ลิสต์ ไปจนถึงการปรับแต่ง equalizer และการตั้งค่า audio engine ใช้การ์ดด้านล่างเพื่อข้ามไปยังส่วนที่คุณต้องการโดยตรง

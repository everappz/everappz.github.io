## 3 Juta Unduhan

**Evermusic** telah mencapai **3 juta unduhan di seluruh dunia**. Ribuan pengguna mengandalkannya setiap hari untuk streaming musik dari penyimpanan cloud, mengelola file audio, dan mendengarkan offline di iOS dan macOS.

**Ringkasan:** Evermusic adalah pemutar musik dan buku audio yang terhubung ke cloud untuk iPhone, iPad, dan Mac. Ia melakukan streaming dari Google Drive, Dropbox, OneDrive, NAS, dan lainnya. Fitur meliputi unduhan offline, equalizer 10-band, pengeditan tag ID3, manajer file, dan pemutaran buku audio dengan bookmark.

## Terima Kasih kepada Pengguna Kami

Umpan balik pengguna telah membentuk hampir setiap pembaruan besar. Laporan bug, permintaan fitur, dan pengujian di dunia nyata telah mendorong peningkatan dari penyesuaian UI kecil hingga kemampuan baru yang besar.

## Fitur Utama yang Membedakan Evermusic

### Streaming Cloud Pribadi

Bangun layanan musik cloud Anda sendiri dengan menghubungkan semua sumber penyimpanan:

- Dropbox, Google Drive, OneDrive, Box, MEGA
- Yandex.Disk, MyDrive, pCloud, HiDrive
- Berbagi file SMB, server WebDAV
- Drive Wi-Fi eksternal atau kartu SD melalui pembaca Lightning

Streaming atau unduh musik kapan saja tanpa menyimpan semuanya di perangkat Anda.

### Pemutar Buku Audio dengan Bookmark

Evermusic juga berfungsi sebagai pemutar buku audio dengan fitur yang dirancang untuk mendengarkan jangka panjang:

- Penyimpanan posisi pemutaran otomatis
- Beberapa **bookmark** per file
- Kontrol **kecepatan pemutaran** (0,5x hingga 2,0x)
- **Timer tidur** (1 menit hingga 24 jam)
- Mode pemutaran **berhenti setelah pengulangan**
- Casting ke **Chromecast** atau **Apple TV**

### Unduhan Offline

Unduh musik dari cloud atau penyimpanan lokal untuk penggunaan offline:

- Unduhan album atau artis satu ketukan
- Hingga 20 koneksi unduhan paralel
- Dukungan cache dan resume lanjutan

### Equalizer 10-Band dengan Preset

Sesuaikan output audio Anda:

- EQ 10-band dengan kontrol visual
- Profil kustom dan preset
- Kontrol gain preamplifier (0 hingga 2,0)

### Manajer File Bawaan

Kelola seluruh koleksi audio Anda langsung di aplikasi:

- Ganti nama, pindahkan, hapus, unggah, dan unduh file
- Atur folder di dalam penyimpanan cloud yang terhubung
- Bagikan file dengan aplikasi lain menggunakan **Buka Di**

### Editor Tag ID3

Perbaiki masalah metadata tanpa meninggalkan aplikasi:

- Edit atau lengkapi otomatis artis, album, genre, dan tahun
- Ambil otomatis sampul album
- Deteksi dan perbaiki otomatis encoding teks yang rusak

### Penyimpanan Eksternal melalui Kartu SD

Bebaskan ruang perangkat dengan menyimpan musik secara eksternal:

- Gunakan **pembaca Lightning SD/microSD** untuk mengakses musik di kartu
- Streaming langsung dari kartu melalui Evermusic
- Jaga penyimpanan iPhone atau iPad Anda tetap tersedia untuk konten lain

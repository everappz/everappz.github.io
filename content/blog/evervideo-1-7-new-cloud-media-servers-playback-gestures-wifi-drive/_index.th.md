**สรุปสั้นๆ:** [Evervideo 1.7](/products/evervideo) คือการอัปเดตครั้งใหญ่สำหรับเครื่องเล่นวิดีโอ HD บน iPhone, iPad และ Mac เวอร์ชันนี้เพิ่มการเชื่อมต่อใหม่กว่า 10 รายการ ทั้งคลาวด์, NAS และมีเดียเซิร์ฟเวอร์ — **Internxt**, **Proton Drive**, **QNAP**, **Nextcloud**, **Amazon S3** รวมถึงมีเดียเซิร์ฟเวอร์ยอดนิยม **Plex**, **Subsonic**, **Navidrome**, **Jellyfin** และ **Emby** และโปรโตคอลเครือข่ายอีกสามตัว: **FTP**, **SFTP** และ **NFS** **ท่าทางการเล่นวิดีโอ** ใหม่ช่วยให้คุณแตะสองครั้งเพื่อกระโดดไปข้างหน้าหรือถอยหลัง, แตะค้างเพื่อเล่นที่ความเร็ว 2x และแตะหนึ่งครั้งเพื่อสลับการแสดงปุ่มควบคุม — ทั้งหมดโดยไม่ต้องออกจากโหมดเต็มจอ Wi-Fi Drive ได้รับ UI ใหม่พร้อมโหมดเลือกและคิวอัปโหลดที่ฉลาดขึ้น แอปทั้งหมดได้รับการปรับให้เหมาะกับดีไซน์ **Liquid Glass** ใหม่ของ Apple

## สวัสดีทุกคน!

การอัปเดตครั้งใหญ่ของ Evervideo มาถึงแล้ว นี่เป็นหนึ่งในเวอร์ชันที่ใหญ่ที่สุดที่เราเคยปล่อยออกมา: **การเชื่อมต่อคลาวด์, เซิร์ฟเวอร์ และเครือข่ายใหม่กว่า 10 รายการ**, **ท่าทางการเล่นวิดีโอ** ใหม่หมดเพื่อการควบคุมที่รวดเร็วขึ้นในโหมดเต็มจอ, ประสบการณ์ **Wi-Fi Drive** ที่ได้รับการปรับปรุงใหม่ และ UI ที่ปรับให้เหมาะกับ **Liquid Glass** บน iPhone, iPad และ Mac

[ดาวน์โหลด Evervideo 1.7 จาก App Store](https://apps.apple.com/app/evervideo-hd-video-player/id6602897336) หรืออัปเดตจากเวอร์ชันที่มีอยู่ ผู้ใช้ Mac สามารถดาวน์โหลด [เวอร์ชันเดสก์ท็อปได้ที่นี่](https://apps.apple.com/app/evervideo/id6743504109)

## การเชื่อมต่อคลาวด์, NAS และมีเดียเซิร์ฟเวอร์ใหม่กว่า 10 รายการ

Evervideo 1.7 ขยายสิ่งที่นับเป็น «คลังวิดีโอ» ของคุณ หากภาพยนตร์, รายการ หรือบันทึกของคุณอยู่ในคลาวด์ที่คุณไว้วางใจ, NAS ที่บ้าน หรือมีเดียเซิร์ฟเวอร์ที่โฮสต์เอง Evervideo สามารถสตรีมจากแหล่งเหล่านั้นได้โดยตรง — ไม่ต้องดาวน์โหลด, ไม่ต้องแปลง, ไม่ต้องเข้ารหัสใหม่

### พื้นที่จัดเก็บคลาวด์ที่เน้นความเป็นส่วนตัว: Internxt และ Proton Drive

หากคุณใส่ใจการเข้ารหัสแบบ end-to-end และการจัดเก็บแบบ zero-knowledge สองในคลาวด์ที่เน้นความเป็นส่วนตัวที่ได้รับการยอมรับมากที่สุดตอนนี้รองรับใน Evervideo แบบเนทีฟ:

- **Internxt** — คลาวด์สเปนแบบโอเพนซอร์ส, เข้ารหัสแบบ post-quantum, และสอดคล้องกับ GDPR มีระดับฟรีให้ใช้
- **Proton Drive** — พื้นที่จัดเก็บที่เข้ารหัสแบบ end-to-end จากผู้สร้าง Proton Mail และ Proton VPN ซึ่งตั้งอยู่ในสวิตเซอร์แลนด์ มีระดับฟรีพร้อมแผนแบบจ่ายเงินสำหรับคลังที่ใหญ่ขึ้น

เชื่อมต่อครั้งเดียว และวิดีโอของคุณจะสตรีมผ่านอุโมงค์ที่เข้ารหัส — Evervideo ไม่เคยเห็นข้อมูลของคุณในรูปแบบที่ไม่ได้เข้ารหัส และเซิร์ฟเวอร์ของผู้ให้บริการก็เช่นกัน

### พื้นที่จัดเก็บที่โฮสต์เอง: QNAP, Nextcloud, Amazon S3

สำหรับผู้ใช้ที่รันโครงสร้างพื้นฐานของตนเอง:

- **QNAP** — การเชื่อมต่อ API เนทีฟกับอุปกรณ์ QNAP NAS เพื่อการเล่นวิดีโอที่รวดเร็วและเชื่อถือได้ผ่าน Wi-Fi ภายในหรือการเข้าถึงระยะไกล สตรีมไฟล์ 4K MKV โดยตรงโดยไม่ต้องเข้ารหัสใหม่
- **Nextcloud** — เชื่อมต่อกับอินสแตนซ์ Nextcloud ใดๆ ที่โฮสต์เองหรือมีการจัดการ ดีมากหากคุณย้ายออกจาก Google Drive หรือ Dropbox ด้วยเหตุผลด้านความเป็นส่วนตัวอยู่แล้ว
- **Amazon S3** — ชี้ Evervideo ไปยัง S3 bucket ใดๆ (หรือพื้นที่จัดเก็บที่เข้ากันได้กับ S3 เช่น Backblaze B2, Wasabi, MinIO, Cloudflare R2) และสตรีมคอลเลกชันของคุณได้โดยตรง

### <a id="media-servers"></a>มีเดียเซิร์ฟเวอร์: Plex, Subsonic, Navidrome, Jellyfin, Emby

นี่คือเรื่องใหญ่สำหรับแฟนวิดีโอที่โฮสต์เอง Evervideo 1.7 เปลี่ยน iPhone, iPad หรือ Mac ของคุณให้เป็นไคลเอนต์ระดับชั้นนำสำหรับมีเดียเซิร์ฟเวอร์โอเพนซอร์สและฟรีเมียมที่ได้รับความนิยมสูงสุด:

- **Plex** — Plex Media Server **ฟรี** ในการดาวน์โหลดและรัน พร้อมการสมัครสมาชิก **Plex Pass** เสริมสำหรับฟีเจอร์ต่างๆ เช่น การซิงค์มือถือ, การถอดรหัสด้วยฮาร์ดแวร์ และทีวีสด Evervideo ทำงานร่วมกับคลังทั้งแบบฟรีและ Plex Pass — สตรีมคอลเลกชันภาพยนตร์และทีวีทั้งหมดของคุณ
- **Subsonic** — เซิร์ฟเวอร์สตรีมมิ่งที่โฮสต์เองดั้งเดิม เซิร์ฟเวอร์ Subsonic อย่างเป็นทางการ **มีค่าใช้จ่าย** (1 ดอลลาร์/เดือนหลังทดลองใช้ 30 วัน) และ Evervideo ยังรองรับ Subsonic API กับเซิร์ฟเวอร์วิดีโอที่เข้ากันได้
- **Navidrome** — เซิร์ฟเวอร์สมัยใหม่, น้ำหนักเบา, **ฟรีและโอเพนซอร์สอย่างสมบูรณ์** ใช้ Subsonic API รันได้บน Raspberry Pi, NAS หรือเครื่อง Linux ใดๆ
- **Jellyfin** — มีเดียเซิร์ฟเวอร์ **ฟรีและโอเพนซอร์สอย่างสมบูรณ์** (เป็น community fork ของ Emby) จัดการภาพยนตร์, ทีวี, เพลง, หนังสือ และวิดีโอที่บ้าน ไม่มีบัญชี, ไม่มี telemetry, ไม่มีการสมัครสมาชิก ตัวเลือกอันดับแรกสำหรับผู้ใช้ที่ต้องการสตรีมมิ่งที่โฮสต์เองโดยไม่มีเงื่อนไขเชิงพาณิชย์ใดๆ
- **Emby** — มีเดียเซิร์ฟเวอร์ **freemium** เซิร์ฟเวอร์หลักฟรี; **Emby Premiere** เป็นการซื้อครั้งเดียวหรือรายปีที่ปลดล็อกแอปมือถือ, การซิงค์ออฟไลน์, Cinema Mode และอื่นๆ Evervideo เชื่อมต่อกับทั้งคลังฟรีและ Premiere

ไม่ว่าคุณจะรันเซิร์ฟเวอร์ใดก็ตาม Evervideo สตรีมคลังทั้งหมดของคุณ — ภาพยนตร์, รายการ, วิดีโอที่บ้าน และแทร็คคำบรรยายที่ฝังไว้ — พร้อมอีควอไลเซอร์วิดีโอ, รองรับ 360°, Picture-in-Picture, AirPlay และ Chromecast

### โปรโตคอลเครือข่ายใหม่: FTP, SFTP, NFS

สำหรับผู้ใช้ที่มีเซิร์ฟเวอร์ที่กำหนดเอง, ห้องแล็บที่บ้าน หรือ NAS ทั่วไปที่ไม่ได้มากับแอปมือถือที่ปรับให้เหมาะสม Evervideo 1.7 เพิ่มโปรโตคอลคลาสสิกสามตัว:

- **SFTP (SSH File Transfer Protocol)** — คำตอบที่ถูกต้องสำหรับ **การสตรีมวิดีโอจากเซิร์ฟเวอร์ของคุณเองอย่างปลอดภัยจากระยะไกล** SFTP ทำงานบน SSH ดังนั้นการถ่ายโอนทั้งหมด (การยืนยันตัวตนและข้อมูลวิดีโอ) จึงถูกเข้ารหัส หากคุณมี VPS, เซิร์ฟเวอร์ส่วนตัว หรือเครื่อง Linux ที่บ้านพร้อมการเข้าถึง SSH คุณสามารถวางโฟลเดอร์วิดีโอลงในเครื่องและสตรีมผ่านอินเทอร์เน็ตสาธารณะโดยไม่เปิดเผยสิ่งอื่น รองรับการยืนยันตัวตนด้วยรหัสผ่านและคีย์
- **FTP** — มาตรฐานสำหรับการถ่ายโอนไฟล์ที่ใช้มาอย่างยาวนาน มีประโยชน์หาก **NAS ที่บ้าน** ของคุณ (Synology, ASUS, D-Link, TerraMaster รุ่นเก่า หรืออุปกรณ์ทั่วไป) เปิดเซิร์ฟเวอร์ FTP แต่ไม่มีการรวม API เนทีฟ ใช้งานดีที่สุดภายในเครือข่ายในบ้านของคุณ
- **NFS (Network File System)** — โปรโตคอลการแชร์โดยพฤตินัยบน Linux และ NAS ส่วนใหญ่ การแชร์ NFS เป็นเรื่องปกติในห้องแล็บที่บ้านและเครือข่ายธุรกิจขนาดเล็ก ตอนนี้ Evervideo เมานต์ NFS และสตรีมวิดีโอ 4K และ HD ด้วยภาระต่ำ

> **เคล็ดลับ:** SFTP เป็นโปรโตคอลที่คุณต้องการสำหรับการสตรีมผ่านอินเทอร์เน็ตเปิด FTP และ NFS ใช้งานดีที่สุดภายในเครือข่ายในบ้าน — อย่าเปิดให้สาธารณะใช้ผ่านอินเทอร์เน็ตเว้นแต่จะห่อด้วย VPN

## ท่าทางการเล่นวิดีโอใหม่

การดูวิดีโอในโหมดเต็มจอควรรู้สึกเบาสบาย Evervideo 1.7 แนะนำชุดท่าทางแตะที่สะอาดซึ่งช่วยให้คุณควบคุมการเล่นได้โดยไม่ต้องแสดงปุ่มควบคุมบนหน้าจอ — เหมาะสำหรับภาพยนตร์, การบรรยาย หรืออะไรก็ตามที่คุณต้องการดูโดยไม่หยุดชะงัก

### แตะสองครั้ง — กระโดดไปข้างหน้าหรือถอยหลัง

แตะสองครั้งที่ **ด้านขวา** ของหน้าจอเพื่อ **ข้ามไปข้างหน้า** ตามจำนวนวินาทีที่กำหนดค่าได้ แตะสองครั้งที่ **ด้านซ้าย** เพื่อ **ถอยกลับ** คุณสามารถเปลี่ยนช่วงเวลา (ค่าเริ่มต้น: 10 วินาที) ใน **การตั้งค่า → การเล่น → ช่วงเวลาข้ามด้วยท่าทาง** — เลือกค่าจาก 5 วินาที (สำหรับการค้นหาที่ละเอียด) ไปจนถึง 60 วินาที (สำหรับการข้ามอินโทร)

นี่คือท่าทางที่ผู้ใช้ YouTube และ Netflix จะจดจำได้ทันที และตอนนี้รองรับแบบเนทีฟใน Evervideo สำหรับวิดีโอใดๆ จากแหล่งใดก็ได้

### แตะค้าง — ความเร็ว 2x ชั่วคราว

กดค้างที่ใดก็ได้บนหน้าจอเพื่อ **เพิ่มความเร็วการเล่นชั่วคราวเป็น 2x** ปล่อยเพื่อกลับสู่ความเร็วปกติ เหมาะสำหรับ:

- ข้ามฉากที่ช้าโดยไม่ต้องเปลี่ยนความเร็วถาวร
- สแกนผ่านการบรรยาย, บทเรียน หรือพอดแคสต์อย่างรวดเร็วเพื่อค้นหาส่วนที่เกี่ยวข้อง
- ลองชิมภาพยนตร์ก่อนตัดสินใจดูเวอร์ชันเต็ม

ท่าทางการกดค้างไม่ได้เปลี่ยนความเร็วการเล่นที่บันทึกไว้ — ปล่อยและคุณก็จะกลับมาที่จุดที่เริ่ม

### แตะหนึ่งครั้ง — แสดง/ซ่อนปุ่มควบคุม

การแตะหนึ่งครั้งบนหน้าจอจะสลับการแสดงปุ่มควบคุมการเล่น (เล่น, หยุดชั่วคราว, แถบค้นหา, คำบรรยาย, อีควอไลเซอร์) แตะหนึ่งครั้งเพื่อแสดง, แตะอีกครั้งเพื่อซ่อน เมื่อรวมกับแตะสองครั้งและกดค้าง หมายความว่าคุณสามารถใช้เวลาเกือบทั้งเรื่องในโหมดเต็มจอที่สะอาดและยังคงสามารถข้าม, หยุด และสแกนเร็วได้ทุกเมื่อที่ต้องการ

## Wi-Fi Drive: UI ใหม่และการอัปโหลดที่เร็วขึ้น

[**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/) คือฟีเจอร์ในตัวของ Evervideo สำหรับ **การถ่ายโอนวิดีโอแบบไร้สายจากคอมพิวเตอร์ไปยัง iPhone หรือ iPad ของคุณ — ไม่ต้องใช้ iTunes, ไม่ต้องใช้สายเคเบิล, ไม่ต้องมีบัญชีคลาวด์** อุปกรณ์ทั้งสองต้องอยู่บนเครือข่าย Wi-Fi เดียวกัน คุณเริ่มเซิร์ฟเวอร์ในแอป iOS และทำอย่างใดอย่างหนึ่ง:

- เปิด URL ในเบราว์เซอร์เดสก์ท็อปใดๆ (Safari, Chrome, Firefox, Edge), ลากไฟล์วิดีโอของคุณลงในหน้านั้น และไฟล์จะอัปโหลดโดยตรงไปยังอุปกรณ์ หรือ
- เมานต์อุปกรณ์เป็นการแชร์เครือข่ายผ่าน **Mac Finder** («เชื่อมต่อกับเซิร์ฟเวอร์…») หรือ **Windows File Explorer** (Map Network Drive) โดยใช้ WebDAV

เป็นวิธีที่เร็วที่สุดในการย้ายคอลเลกชันวิดีโอภายในเครื่องขนาดใหญ่ไปยังโทรศัพท์หรือแท็บเล็ตของคุณโดยไม่ต้องใช้บริการของบุคคลที่สาม ดู [คู่มือทีละขั้นตอนที่นี่](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)

ใน Evervideo 1.7, Wi-Fi Drive ได้รับ:

- **ส่วนติดต่อผู้ใช้ที่ออกแบบใหม่** — สะอาดยิ่งขึ้น, อ่านได้ง่ายขึ้นในพริบตา และอัปเดตสำหรับ Liquid Glass
- **โหมดเลือกใหม่สำหรับการดำเนินการเป็นชุด** — เลือกหลายไฟล์หรือหลายโฟลเดอร์และจัดการแบบกลุ่ม (ลบ, ย้าย, คัดลอก)
- **คิวการอัปโหลดไฟล์ที่ปรับปรุงแล้ว** — ตอบสนองความคืบหน้าที่ดีกว่าและทนต่อการสะดุดของเครือข่ายเพื่อให้การเชื่อมต่อ Wi-Fi ที่ไม่เสถียรไม่ทำลายการถ่ายโอน 30 GB อีกต่อไป
- **ประสิทธิภาพการถ่ายโอนโดยรวมที่ดีขึ้น** — การอัปโหลดเร็วขึ้นอย่างวัดได้สำหรับคลังขนาดใหญ่ โดยเฉพาะสำหรับไฟล์ 4K MKV และคอลเลกชันภาพยนตร์

## การปรับปรุงอื่นๆ

### การอัปเดตดีไซน์ Liquid Glass

ส่วนติดต่อของ Evervideo 1.7 ได้รับการอัปเดตสำหรับวัสดุ **Liquid Glass** ใหม่ของ Apple ทั่วทั้งแอป — พื้นผิวโปร่งใส, แอนิเมชั่นที่ราบรื่นขึ้น และปุ่มควบคุมที่ละเอียดอ่อนซึ่งเข้ากันได้ดีกับ iOS 26, iPadOS 26 และ macOS 26 หน้า Now Playing, ตัวเรียกดูไฟล์ และหน้าจอการตั้งค่าทั้งหมดได้รับการปรับให้เข้ากับสุนทรียศาสตร์ของระบบใหม่

### ไลบรารีการเชื่อมต่อที่อัปเดตแล้ว

เราได้รีเฟรชไลบรารีพื้นฐานที่ Evervideo ใช้ในการพูดคุยกับ **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive**, **iCloud Drive**, **MEGA** และบริการอื่นๆ ผลลัพธ์: ความล้มเหลวในการเชื่อมต่อในกรณีพิเศษน้อยลง, การรองรับเซิร์ฟเวอร์เวอร์ชันใหม่ที่ดีขึ้น และความน่าเชื่อถือที่ดีขึ้นเมื่อสตรีมวิดีโอบนการเชื่อมต่อที่ช้าลงหรืออยู่ไกลทางภูมิศาสตร์

### การแก้ไขบักการเล่น

- แก้ไขปัญหาการเล่นบนเซิร์ฟเวอร์ที่โฮสต์เองบางตัวที่สตรีมหยุดชะงักหลังจากค้นหาในไฟล์ MKV ขนาดใหญ่
- พฤติกรรมการเล่นต่อที่ดีขึ้นเมื่อเครือข่ายขาดหายไปสั้นๆ ระหว่างการเล่น
- การซิงค์คำบรรยายที่ราบรื่นยิ่งขึ้นในเนื้อหายาว

### การแก้ไขการแปลภาษา

การแก้ไขการแปลในหลายภาษาตามคำแนะนำโดยตรงจากผู้ใช้ การจัดวางข้อความที่ดีขึ้นบนปุ่มขนาดเล็กและภาษายุโรปที่ยาวกว่า (เยอรมัน, ดัตช์, ฝรั่งเศส)

### การปรับปรุงเล็กๆ ที่ได้รับแรงบันดาลใจจากคำแนะนำของคุณ

การปรับปรุงเล็กๆ น้อยๆ จำนวนมากตามรีวิวจาก App Store และอีเมลที่ส่งไปยัง support@everappz.com เราอ่านทุกข้อความ

## ทำไมการอัปเดตนี้จึงสำคัญ

Evervideo 1.7 ถูกสร้างขึ้นรอบสามแนวคิด:

1. **วิดีโอของคุณ ไม่ว่าคุณจะเก็บไว้ที่ไหน** ไม่ว่าคลังของคุณจะอยู่บน iCloud Drive, คลาวด์ที่เน้นความเป็นส่วนตัวเช่น Proton Drive หรือ Internxt, มีเดียเซิร์ฟเวอร์เช่น Plex หรือ Jellyfin, NAS ที่บ้าน หรือ Raspberry Pi ที่รัน Navidrome — ตอนนี้ Evervideo เชื่อมต่อกับมันแบบเนทีฟ พร้อมประสบการณ์การเล่นเหมือนกันทุกที่
2. **วิดีโอแบบเต็มจอที่รู้สึกเบาสบาย** ท่าทางการแตะใหม่ (แตะสองครั้ง, แตะค้าง, แตะหนึ่งครั้ง) นำการควบคุมที่ลื่นไหลแบบเดียวกับที่แอปสตรีมมิ่งอย่าง YouTube และ Netflix ฝึกผู้ใช้ให้คาดหวัง มาประยุกต์กับคอลเลกชันวิดีโอ *ของคุณ*
3. **การถ่ายโอนที่เร็วขึ้นจากคอมพิวเตอร์ของคุณ** Wi-Fi Drive ที่ปรับปรุงใหม่พร้อมการเลือกเป็นชุดและคิวอัปโหลดที่ฉลาดขึ้นทำให้การย้ายคอลเลกชันภาพยนตร์ 4K ขนาดใหญ่ไปยัง iPhone หรือ iPad ของคุณเร็วอย่างแท้จริง — ไม่มีสายเคเบิล, ไม่มี iTunes, ไม่มีการบีบอัด

{{% details title="การอัปเดตเป็น Evervideo 1.7 ฟรีหรือไม่?" closed="true" %}}
ใช่ Evervideo เป็นการดาวน์โหลดฟรีจาก App Store และ 1.7 เป็นการอัปเดตฟรีสำหรับผู้ใช้ที่มีอยู่ทั้งหมด การผสานรวมคลาวด์ใหม่, การรองรับมีเดียเซิร์ฟเวอร์, ท่าทางการเล่น, การปรับปรุง Wi-Fi Drive และ UI Liquid Glass เป็นส่วนหนึ่งของการอัปเดตพื้นฐาน

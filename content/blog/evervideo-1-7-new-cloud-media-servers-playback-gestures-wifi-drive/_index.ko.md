**요약:** [Evervideo 1.7](/products/evervideo)은 iPhone, iPad, Mac용 HD 비디오 플레이어의 큰 업데이트입니다. 이번 릴리스는 10개 이상의 새로운 클라우드, NAS, 미디어 서버 연결을 추가합니다 — **Internxt**, **Proton Drive**, **QNAP**, **Nextcloud**, **Amazon S3**, 그리고 가장 인기 있는 미디어 서버인 **Plex**, **Subsonic**, **Navidrome**, **Jellyfin**, **Emby**, 그리고 세 가지 네트워크 프로토콜 **FTP**, **SFTP**, **NFS**. 새로운 **재생 제스처**로 두 번 탭하여 앞뒤로 이동하고, 길게 눌러 2배속으로 재생하고, 한 번 탭하여 컨트롤을 켜고 끌 수 있습니다 — 모두 전체 화면을 떠나지 않고요. Wi-Fi Drive는 선택 모드와 더 똑똑한 업로드 큐를 갖춘 새로워진 UI를 얻습니다. 전체 앱이 Apple의 새로운 **Liquid Glass** 디자인에 맞춰 다듬어졌습니다.

## 10개 이상의 새로운 클라우드, NAS, 미디어 서버 연결

Evervideo 1.7은 「비디오 라이브러리」로 간주되는 범위를 확장합니다. 영화, 쇼, 녹화물이 신뢰하는 클라우드, 집의 NAS, 또는 자가 호스팅 미디어 서버에 있다면, 이제 Evervideo가 직접 스트리밍할 수 있습니다 — 다운로드, 변환, 재인코딩 없이.

### 프라이버시 중심 클라우드 저장소: Internxt와 Proton Drive

엔드 투 엔드 암호화와 제로 지식 저장에 신경을 쓴다면, 가장 인정받는 두 프라이버시 우선 클라우드가 이제 Evervideo에 네이티브로 들어왔습니다.

- **Internxt** — 오픈 소스, 포스트 양자 암호화, GDPR 준수의 스페인 클라우드. 무료 티어 제공.
- **Proton Drive** — Proton Mail과 Proton VPN 제작자가 만든 엔드 투 엔드 암호화 저장소(스위스 본사). 무료 티어 제공, 더 큰 라이브러리를 위한 유료 요금제도 있습니다.

한 번 연결하면 비디오가 암호화 터널을 통해 스트리밍됩니다 — Evervideo는 데이터를 평문으로 절대 보지 못하며, 제공자의 서버도 마찬가지입니다.

### 자가 호스팅 저장소: QNAP, Nextcloud, Amazon S3

자체 인프라를 운영하는 사용자라면:

- **QNAP** — QNAP NAS 장치에 네이티브 API 연결로 로컬 Wi-Fi 또는 원격 접속을 통해 빠르고 안정적인 비디오 재생을 제공합니다. 재인코딩 없이 4K MKV 파일을 직접 스트리밍하세요.
- **Nextcloud** — 자체 호스팅 또는 관리형 Nextcloud 인스턴스에 연결합니다. 프라이버시를 이유로 Google Drive 또는 Dropbox에서 이미 마이그레이션했다면 훌륭한 선택입니다.
- **Amazon S3** — Evertag를 임의의 S3 버킷(또는 Backblaze B2, Wasabi, MinIO, Cloudflare R2와 같은 S3 호환 저장소)에 연결하고 컬렉션을 직접 스트리밍하세요.

### <a id="media-servers"></a>미디어 서버: Plex, Subsonic, Navidrome, Jellyfin, Emby

자가 호스팅 비디오 팬에게 이것이 가장 큰 부분입니다. Evervideo 1.7은 가장 인기 있는 오픈 소스 및 프리미엄 미디어 서버를 위한 1급 클라이언트로 iPhone, iPad, Mac을 변모시킵니다.

- **Plex** — Plex Media Server는 다운로드하고 실행하는 데 **무료**이며, 모바일 동기화, 하드웨어 트랜스코딩, 라이브 TV와 같은 기능을 위한 선택형 **Plex Pass** 구독이 있습니다. Evervideo는 무료 라이브러리와 Plex Pass 라이브러리 모두에서 작동합니다 — 전체 영화 및 TV 컬렉션을 스트리밍하세요.
- **Subsonic** — 원조 자가 호스팅 스트리밍 서버. 공식 Subsonic 서버는 **유료**(30일 체험 후 월 1달러)이며, Evervideo는 Subsonic API를 사용하는 호환 비디오 서버와도 통신합니다.
- **Navidrome** — 현대적이고 가벼우며 **완전히 무료이고 오픈 소스**인 서버. Subsonic API를 구현합니다. Raspberry Pi, NAS, 또는 어떤 Linux 박스에서도 실행됩니다.
- **Jellyfin** — **완전히 무료이고 오픈 소스**인 미디어 서버(Emby의 커뮤니티 포크). 영화, TV, 음악, 책, 홈 비디오를 다룹니다. 계정도, 텔레메트리도, 구독도 없습니다. 상업적 제약 없이 자가 호스팅 스트리밍을 원하는 사용자에게 가장 적합한 선택입니다.
- **Emby** — **프리미엄** 미디어 서버. 코어 서버는 무료이며, **Emby Premiere**는 일회성 또는 연간 구매로 모바일 앱, 오프라인 동기화, 시네마 모드 등을 잠금 해제합니다. Evervideo는 무료 라이브러리와 Premiere 라이브러리 모두에 연결됩니다.

어떤 서버를 운영하든 Evervideo는 비디오 이퀄라이저, 360도 지원, Picture-in-Picture, AirPlay, Chromecast로 전체 라이브러리 — 영화, 쇼, 홈 비디오, 임베디드 자막 트랙 — 를 스트리밍합니다.

### 새로운 네트워크 프로토콜: FTP, SFTP, NFS

맞춤형 서버, 홈랩, 또는 정교한 모바일 앱 없이 출시되는 일반 NAS 박스를 사용하는 분들을 위해 Evervideo 1.7은 세 가지 클래식한 프로토콜을 추가했습니다.

- **SFTP(SSH File Transfer Protocol)** — **자체 서버에서 안전한 원격 비디오 스트리밍**을 위한 올바른 답입니다. SFTP는 SSH 위에서 동작하므로 전체 전송(인증과 비디오 데이터)이 암호화됩니다. SSH 접근이 가능한 VPS, 전용 서버, 집의 Linux 박스가 있다면, 비디오 폴더를 그 위에 두고 다른 어떤 것도 노출하지 않은 채 공개 인터넷을 통해 스트리밍할 수 있습니다. 비밀번호 및 키 기반 인증을 지원합니다.
- **FTP** — 파일 전송의 오래된 표준입니다. **가정용 NAS**(오래된 Synology, ASUS, D-Link, TerraMaster, 또는 일반 박스)가 FTP 서버를 노출하지만 네이티브 API 통합이 없는 경우에 유용합니다. 로컬 네트워크 내에서 사용하는 것이 가장 좋습니다.
- **NFS(Network File System)** — Linux와 대부분의 NAS 장치에서 사실상 표준이 된 공유 프로토콜입니다. NFS 공유는 홈랩과 소규모 기업 네트워크에서 흔히 볼 수 있습니다. 이제 Evervideo가 마운트하고 낮은 오버헤드로 4K 및 HD 비디오를 스트리밍합니다.

> **팁:** SFTP는 공개 인터넷에서 스트리밍할 때 원하는 프로토콜입니다. FTP와 NFS는 로컬 네트워크 내에서 사용하는 것이 가장 좋습니다 — VPN으로 감싸지 않는 한 공개 인터넷에 노출하지 마세요.

## 새로운 재생 제스처

전체 화면에서의 비디오 시청은 손쉽게 느껴져야 합니다. Evervideo 1.7은 화면 상의 컨트롤을 노출하지 않고도 재생을 제어할 수 있게 해주는 깔끔한 탭 제스처 세트를 도입합니다 — 영화, 강의, 또는 방해받지 않고 시청하고 싶은 어떤 것에든 완벽합니다.

### 두 번 탭 — 앞뒤로 이동

화면의 **오른쪽**을 두 번 탭하여 설정 가능한 초 단위로 **앞으로 건너뛰기**. **왼쪽**을 두 번 탭하여 **뒤로 이동**. **설정 → 재생 → 제스처 건너뛰기 간격**에서 간격(기본값: 10초)을 변경할 수 있습니다 — 5초(세밀한 탐색)부터 60초(인트로 건너뛰기)까지 선택하세요.

이것은 YouTube와 Netflix 사용자가 즉시 알아볼 수 있는 제스처이며, 이제 모든 비디오, 모든 소스에서 Evervideo에 네이티브로 들어왔습니다.

### 길게 누르기 — 일시적 2배속

화면 어디든 길게 눌러 **재생을 일시적으로 2배속으로** 가속하세요. 손을 떼면 정상 속도로 돌아갑니다. 다음에 완벽합니다.

- 영구적인 속도 변경을 약속하지 않고 느린 장면 건너뛰기.
- 강의, 튜토리얼, 팟캐스트를 빠르게 훑어보며 관련된 부분 찾기.
- 전체 버전 시청을 약속하기 전에 영화 샘플 보기.

길게 누르는 제스처는 저장된 재생 속도를 변경하지 않습니다 — 손을 떼면 시작한 곳으로 돌아옵니다.

### 한 번 탭 — 컨트롤 표시/숨김

화면을 한 번 탭하면 재생 컨트롤(재생, 일시 정지, 탐색 막대, 자막, 이퀄라이저)이 토글됩니다. 한 번 탭하여 표시하고, 다시 탭하여 숨깁니다. 두 번 탭 및 길게 누르기와 결합하면, 깔끔한 전체 화면 모드로 거의 영화 한 편 전체를 보낼 수 있으며 필요할 때마다 여전히 탐색, 일시 정지, 속도 스캔을 할 수 있다는 뜻입니다.

## Wi-Fi Drive: 새로운 UI와 더 빠른 업로드

[**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)는 **iTunes, 케이블, 클라우드 계정 없이 컴퓨터에서 iPhone 또는 iPad로 비디오를 무선 전송**하기 위한 Evervideo 내장 기능입니다. 두 기기는 같은 Wi-Fi 네트워크에 있어야 합니다. iOS 앱에서 서버를 시작하고 다음 중 하나를 수행합니다.

- 데스크톱 브라우저(Safari, Chrome, Firefox, Edge)에서 URL을 열고 비디오 파일을 페이지로 드래그하면 기기로 직접 업로드되거나,
- WebDAV를 사용하여 **Mac Finder**(「서버에 연결…」) 또는 **Windows 파일 탐색기**(네트워크 드라이브 매핑)를 통해 기기를 네트워크 공유로 마운트합니다.

타사 서비스 없이 큰 로컬 비디오 컬렉션을 휴대폰이나 태블릿으로 옮기는 가장 빠른 방법입니다. [단계별 가이드는 여기에서 확인하세요](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/).

Evervideo 1.7에서 Wi-Fi Drive는 다음을 갖춥니다.

- **새로워진 사용자 인터페이스** — 더 깔끔하고, 한눈에 더 쉽게 읽을 수 있으며, Liquid Glass에 맞춰 업데이트되었습니다.
- **일괄 작업을 위한 새로운 선택 모드** — 여러 파일이나 폴더를 선택하여 일괄로 처리합니다(삭제, 이동, 복사).
- **개선된 파일 업로드 큐** — 더 나은 진행 상황 피드백과 네트워크 변동에 대한 회복력으로, 불안정한 Wi-Fi 연결이 더 이상 30 GB 전송을 망치지 않습니다.
- **전반적인 전송 성능 향상** — 큰 라이브러리, 특히 4K MKV 파일과 영화 컬렉션에서 측정 가능하게 더 빠른 업로드.

## 그 외 개선

### Liquid Glass 디자인 업데이트

Evervideo 1.7의 인터페이스는 앱 전체에 걸쳐 Apple의 새로운 **Liquid Glass** 머티리얼에 맞춰 업데이트되었습니다 — 반투명 표면, 더 부드러운 애니메이션, iOS 26, iPadOS 26, macOS 26에 자연스럽게 어울리는 정제된 컨트롤이 적용됩니다. 지금 재생, 파일 브라우저, 설정 화면 모두 새로운 시스템 미학에 맞춰 다시 다듬어졌습니다.

### 업데이트된 연결 라이브러리

Evervideo가 **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive**, **iCloud Drive**, **MEGA** 등 다른 서비스와 통신하는 데 사용하는 내부 라이브러리를 새로 다듬었습니다. 결과: 엣지 케이스의 연결 실패가 줄어들고, 최신 서버 버전에 대한 지원이 향상되며, 더 느리거나 지리적으로 먼 연결에서 비디오를 스트리밍할 때 신뢰성이 향상됩니다.

### 재생 버그 수정

- 큰 MKV 파일에서 탐색 후 스트림이 멈추던 특정 자가 호스팅 서버의 재생 문제를 수정했습니다.
- 재생 도중 네트워크가 잠시 끊겼을 때 더 나은 재개 동작.
- 긴 형식 콘텐츠에서 더 부드러운 자막 동기화.

### 현지화 수정

직접적인 사용자 피드백을 바탕으로 한 여러 언어의 번역 수정. 더 작은 버튼과 더 긴 유럽 언어(독일어, 네덜란드어, 프랑스어)에서 텍스트가 더 잘 맞도록 개선되었습니다.

### 여러분의 피드백에서 영감을 받은 작은 다듬기

App Store 리뷰와 support@everappz.com으로 받은 이메일을 바탕으로 한 많은 작은 개선 사항. 우리는 모든 메시지를 읽습니다.

## 이번 업데이트가 중요한 이유

Evervideo 1.7은 세 가지 아이디어를 중심으로 만들어졌습니다.

1. **어디에 보관하든 비디오를.** 라이브러리가 iCloud Drive, Proton Drive 또는 Internxt 같은 프라이버시 우선 클라우드, Plex 또는 Jellyfin 같은 미디어 서버, 집의 NAS, 또는 Navidrome을 실행하는 Raspberry Pi에 있든 — Evervideo는 이제 어디서나 동일한 재생 경험으로 네이티브로 연결됩니다.
2. **손쉽게 느껴지는 전체 화면 비디오.** 새로운 탭 제스처(두 번 탭, 길게 누르기, 한 번 탭)는 YouTube와 Netflix 같은 스트리밍 앱이 사용자에게 기대하도록 훈련시킨 유연한 제어를 *여러분의* 비디오 컬렉션에 적용해 줍니다.
3. **컴퓨터에서 더 빠른 전송.** 일괄 선택과 더 똑똑한 업로드 큐를 갖춘 새로워진 Wi-Fi Drive는 큰 4K 영화 컬렉션을 iPhone 또는 iPad로 옮기는 것을 진정으로 빠르게 만듭니다 — 케이블도, iTunes도, 압축도 없이.

## 자주 묻는 질문

{{% details title="Evervideo 1.7의 새로운 점은?" closed="true" %}}
Evervideo 1.7은 10개 이상의 새로운 연결(Plex, Jellyfin, Emby, Subsonic, Navidrome, Internxt, Proton Drive, QNAP, Nextcloud, Amazon S3, FTP, SFTP, NFS) 지원, 새로운 재생 제스처(두 번 탭하여 탐색, 길게 눌러 2배속, 한 번 탭하여 컨트롤 토글), 선택 모드와 더 똑똑한 업로드 큐를 갖춘 새로 디자인된 Wi-Fi Drive, Liquid Glass 디자인 업데이트, 업데이트된 연결 라이브러리, 그리고 많은 버그 수정을 도입합니다.
{{% /details %}}

{{% details title="Evervideo는 Plex와 작동하나요?" closed="true" %}}
네. Evervideo 1.7부터 Plex Media Server에 연결하고 전체 비디오 라이브러리 — 영화, TV 쇼, 홈 비디오 — 를 스트리밍할 수 있습니다. Plex Media Server는 실행에 무료이며, Plex Pass는 선택 사항입니다. Evervideo는 재인코딩 없이 MKV, MP4, AVI, MOV 및 기타 형식의 직접 재생을 포함하여 무료 및 Plex Pass 설정 모두를 지원합니다.
{{% /details %}}

{{% details title="Evervideo에서 Jellyfin 또는 Navidrome이 지원되나요?" closed="true" %}}
네. Jellyfin과 Navidrome 모두 Evervideo 1.7에서 완전히 지원됩니다. Jellyfin은 비디오와 오디오를 처리하는 무료 오픈 소스 미디어 서버입니다. Navidrome은 Subsonic API를 구현하는 무료 오픈 소스 서버입니다. Evervideo는 두 가지 모두에 네이티브로 연결됩니다.
{{% /details %}}

{{% details title="Plex, Jellyfin, Emby, Navidrome, Subsonic은 무료인가요?" closed="true" %}}
- **Plex** — 서버는 무료이며, Plex Pass는 선택형 유료 업그레이드입니다.
- **Jellyfin** — 완전히 무료이고 오픈 소스.
- **Emby** — 서버는 무료이며, Emby Premiere는 유료이고 모바일 동기화와 오프라인을 잠금 해제합니다.
- **Navidrome** — 완전히 무료이고 오픈 소스.
- **Subsonic** — 공식 서버는 30일 체험 후 월 1달러이지만, API는 개방되어 있으며 많은 무료 서버(Navidrome 포함)가 이를 구현합니다.
{{% /details %}}

{{% details title="SFTP, FTP, 또는 NFS를 통해 가정용 NAS에서 스트리밍할 수 있나요?" closed="true" %}}
네. Evervideo 1.7은 SFTP, FTP, NFS를 네이티브 연결 유형으로 추가합니다. 모든 트래픽이 SSH를 통해 암호화되기 때문에 공개 인터넷을 통해 자체 서버에서 스트리밍할 때 SFTP가 권장되는 선택입니다. FTP와 NFS는 로컬 네트워크 내부나 VPN 뒤에서 사용하는 것이 가장 좋습니다.
{{% /details %}}

{{% details title="SFTP를 사용하여 Evervideo를 맞춤형 서버에 어떻게 연결하나요?" closed="true" %}}
Evervideo를 열고, 연결하기 탭으로 이동해 SFTP를 선택한 다음, 서버의 호스트 이름이나 IP, 포트(보통 22), 사용자 이름, 그리고 비밀번호 또는 SSH 개인 키를 입력하세요. Evervideo가 원격 폴더를 탐색하고 엔드 투 엔드 암호화로 비디오 파일을 직접 스트리밍합니다.
{{% /details %}}

{{% details title="Evervideo는 Internxt와 Proton Drive를 지원하나요?" closed="true" %}}
네. 두 프라이버시 중심 클라우드 모두 Evervideo 1.7부터 지원됩니다. 이들은 앱에서 이미 사용 가능한 MEGA 및 기타 프라이버시 우선 서비스에 합류합니다.
{{% /details %}}

{{% details title="새로운 재생 제스처는 어떻게 작동하나요?" closed="true" %}}
전체 화면 비디오 재생에서, **오른쪽을 두 번 탭**하여 앞으로 이동하고 **왼쪽을 두 번 탭**하여 설정 가능한 간격(기본값 10초 — 설정에서 변경 가능)만큼 뒤로 이동합니다. 화면 어디든 **길게 눌러** 일시적으로 2배속으로 가속합니다. 손을 떼면 정상으로 돌아갑니다. 어디든 **한 번 탭**하여 재생 컨트롤을 토글합니다(표시 또는 숨김).
{{% /details %}}

{{% details title="두 번 탭하기 건너뛰기 간격을 변경할 수 있나요?" closed="true" %}}
네. **설정 → 재생 → 제스처 건너뛰기 간격**으로 이동하여 5초에서 60초 사이의 값을 선택하세요. 대부분의 사용자는 10초 또는 15초로 유지합니다.
{{% /details %}}

{{% details title="Evervideo의 Wi-Fi Drive란 무엇인가요?" closed="true" %}}
Wi-Fi Drive는 Evervideo의 내장 무선 파일 전송 기능입니다. iTunes, 케이블, 클라우드 계정 없이 로컬 Wi-Fi 네트워크를 통해 컴퓨터에서 iPhone 또는 iPad로 비디오를 업로드할 수 있게 해줍니다. 모든 데스크톱 브라우저 또는 Mac Finder, Windows 파일 탐색기와 같은 WebDAV 클라이언트를 사용할 수 있습니다. [전체 Wi-Fi Drive 가이드](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)를 참조하세요.
{{% /details %}}

{{% details title="Evervideo는 Plex 또는 Jellyfin에서 MKV, AVI, 기타 형식을 재생하나요?" closed="true" %}}
네. Evervideo는 사실상 모든 비디오 형식 — MKV, AVI, MP4, MOV, FLV, WMV, WEBM, M4V, TS, 3GP — 을 재생하며, 대부분의 코덱에 대해 트랜스코딩 없이 Plex, Jellyfin, Emby 및 기타 미디어 서버에서 직접 스트리밍합니다. 이는 서버의 CPU 부하가 낮아지고 시작 시간이 빨라진다는 뜻입니다.
{{% /details %}}

{{% details title="Evervideo 1.7 업데이트는 무료인가요?" closed="true" %}}
네. Evervideo는 App Store에서 무료로 다운로드할 수 있으며, 1.7은 모든 기존 사용자를 위한 무료 업데이트입니다. 새로운 클라우드 통합, 미디어 서버 지원, 재생 제스처, Wi-Fi Drive 개선, Liquid Glass UI는 기본 업데이트의 일부입니다.

Evervideo 1.7은 iPhone, iPad, Mac에서 실행됩니다. AirPlay와 Chromecast로 재생을 더 큰 화면으로 캐스트할 수 있습니다. iCloud Drive 동기화는 기기 간에 라이브러리와 설정을 일관되게 유지합니다.

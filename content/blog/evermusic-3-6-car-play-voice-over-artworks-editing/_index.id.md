**Ringkasan:** Evermusic 3.6 menambahkan integrasi Apple CarPlay, aksesibilitas VoiceOver penuh, output audio campuran, lanjutan pemutaran otomatis, pengeditan artwork dan tag untuk FLAC/MP3/AIFF, dan impor file dari iCloud Drive.

## Yang Baru di Evermusic 3.6

Evermusic 3.6 menghadirkan enam fitur utama yang berfokus pada pemutaran di mobil, aksesibilitas, dan manajemen perpustakaan. Berikut yang berubah.

## Dukungan Apple CarPlay

Evermusic kini bekerja dengan Apple CarPlay. Akses daftar putar dan album Anda langsung dari layar bawaan mobil Anda.

- Jelajahi dan putar musik dari layar mobil
- Kontrol dirancang untuk penggunaan aman dengan mata di jalan
- Bekerja dengan kendaraan apa pun yang kompatibel dengan CarPlay

## Aksesibilitas VoiceOver Penuh

Evermusic 3.6 menyertakan dukungan VoiceOver lengkap untuk pengguna dengan gangguan penglihatan:

- Semua elemen UI memiliki label deskriptif dan petunjuk aksesibilitas
- Mode **Simple User Interface** di **Settings > Accessibility** menghilangkan kekacauan visual
- Navigasi lancar di seluruh aplikasi

Ini menjadikan Evermusic salah satu dari sedikit pemutar musik pihak ketiga dengan dukungan VoiceOver menyeluruh di iOS.

## Output Audio Campuran

Putar audio Evermusic bersamaan dengan aplikasi lain secara bersamaan.

- Aktifkan di **Settings → Audio Player → General → Audio Output Mode**
- Berguna untuk mencampur musik dengan audio game, navigasi, atau aplikasi DJ

## Lanjutan Pemutaran Otomatis

Evermusic menyimpan posisi mendengarkan Anda secara otomatis. Saat Anda membuka kembali aplikasi:

- **Antrean**, **trek saat ini**, dan **posisi pemutaran** Anda dipulihkan
- Ideal untuk buku audio dan podcast

Aktifkan di: **Settings → Audio Player → General → Save Audio Player State**

## Pengeditan Artwork dan Tag

Edit sampul album dan metadata langsung di Evermusic:

- Ubah artwork untuk file **FLAC**, **MP3**, **AIFF** dan daftar putar
- Edit judul lagu, artis, dan album dengan editor tag bawaan
- Perbarui artwork dalam satu ketukan

## Impor File iCloud Drive

Impor file musik langsung dari iCloud Drive dan aplikasi lain:

- Jelajahi file melalui aplikasi **File** bawaan iOS
- Impor trek ke Evermusic dalam satu ketukan
- Akses seluruh koleksi musik cloud Anda

## Dapatkan Pembaruan

[Unduh Evermusic](https://apps.apple.com/app/evermusic-offline-music-player/id885367198) dari App Store atau perbarui ke versi 3.6 untuk mengakses semua fitur baru.

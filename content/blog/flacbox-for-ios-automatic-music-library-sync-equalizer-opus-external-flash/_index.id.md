**Flacbox 1.6** membawa fitur baru utama ke pemutar musik FLAC untuk iPhone dan iPad.

**Ringkasan:** Pembaruan ini menambahkan sinkronisasi perpustakaan cloud otomatis, equalizer 10-band dengan preset kustom, dukungan format OPUS, dan kemampuan memutar musik dari kartu SD eksternal. Bug pemutaran FLAC juga telah diperbaiki.

## Yang Baru di Flacbox 1.6

### Sinkronisasi Perpustakaan Musik Cloud Otomatis

Tidak perlu lagi menyegarkan perpustakaan secara manual. Mesin sinkronisasi baru secara otomatis memindai folder cloud yang dipilih dan memperbarui perpustakaan musik Anda secara real time.

- Aktifkan melalui: **Settings → Music Library → Automatic Sync**
- Pilih folder melalui **Sync Folder → Change Settings**
- Pilih mode sinkronisasi: hanya Wi-Fi atau Wi-Fi + Seluler
- Aktifkan **Background Sync** untuk pembaruan otomatis saat pemutar aktif (meningkatkan penggunaan baterai)

Koleksi musik cloud Anda tetap terkini tanpa intervensi manual.

### Equalizer 10-Band untuk Suara Kustom

Bentuk audio Anda dengan **equalizer 10-band** bawaan, dapat diakses dari layar Pemutar atau Pengaturan.

- Sesuaikan frekuensi antara **-12 dB** dan **+12 dB**
- Gunakan preset bawaan atau buat milik Anda sendiri
- Sesuaikan daya preamp (gunakan dengan hati-hati untuk melindungi pendengaran)

Ini memberikan kontrol audio level studio di iPhone atau iPad Anda.

### Dukungan Format File OPUS

Flacbox sekarang memutar file **OPUS** bersama FLAC, ALAC, MP3, dan format lainnya. OPUS adalah codec efisiensi tinggi yang banyak digunakan untuk suara dan musik. Dukungan equalizer penuh disertakan untuk pemutaran OPUS.

### Putar Musik dari Kartu SD Eksternal

Putar musik langsung dari kartu SD atau microSD yang terhubung ke iPhone atau iPad Anda:

- Gunakan **Lightning to SD Card Camera Reader Adapter**
- Masukkan kartu dan hubungkan ke perangkat Anda
- Buka Flacbox -- secara otomatis mengenali kartu
- Jelajahi dan putar file melalui bagian **Services → PowerDrive**

Ini ideal untuk memperluas perpustakaan musik Anda tanpa menggunakan penyimpanan perangkat.

### Perbaikan Bug Pemutaran FLAC

**Lompatan 2 detik di akhir file FLAC** telah sepenuhnya diperbaiki. Trek sekarang diputar dengan mulus dari awal hingga akhir.

Flacbox mendukung FLAC, ALAC, MP3, AAC, OGG, OPUS, WAV, AIFF, DSD, dan format audio populer lainnya. Semua format berfungsi dengan equalizer bawaan.

Ya. Hubungkan kartu SD atau microSD menggunakan Lightning to SD Card Camera Reader Adapter. Flacbox mendeteksi kartu secara otomatis dan memungkinkan Anda menjelajahi dan memutar file langsung dari penyimpanan eksternal.
{{% /details %}}

{{% details title="Apakah Flacbox disinkronkan dengan penyimpanan cloud secara otomatis?" closed="true" %}}
Ya. Mulai versi 1.6, Flacbox dapat secara otomatis menyinkronkan perpustakaan musik Anda dari folder cloud. Aktifkan Sinkronisasi Otomatis di Pengaturan dan pilih folder yang ingin Anda pantau.
{{% /details %}}

{{% details title="Apakah equalizer Flacbox dapat disesuaikan?" closed="true" %}}
Ya. Equalizer 10-band memungkinkan Anda menyesuaikan level frekuensi individual antara -12 dB dan +12 dB. Anda dapat menggunakan preset bawaan atau menyimpan pengaturan kustom Anda sendiri.

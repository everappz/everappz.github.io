**Flacbox 1.6** 為 iPhone 和 iPad 的 FLAC 音樂播放器帶來了重大新功能。

**摘要：** 此更新新增自動雲端資料庫同步、帶自訂預設的 10 段等化器、OPUS 格式支援以及從外接 SD 卡播放音樂的功能。FLAC 播放錯誤也已修復。

## Flacbox 1.6 的新功能

### 自動雲端音樂庫同步

不再需要手動重新整理音樂庫。新的同步引擎自動掃描選定的雲端資料夾並即時更新您的音樂庫。

- 透過以下位置啟用：**Settings → Music Library → Automatic Sync**
- 透過 **Sync Folder → Change Settings** 選擇資料夾
- 選擇同步模式：僅 Wi-Fi 或 Wi-Fi + 行動數據
- 啟用 **Background Sync** 在播放器運作時自動更新（增加電池消耗）

您的雲端音樂收藏無需手動操作即可保持最新。

### 10 段等化器自訂音效

使用內建的 **10 段等化器**調整音效，可從播放器畫面或設定中存取。

- 在 **-12 dB** 和 **+12 dB** 之間調整頻率
- 使用內建預設或建立自己的
- 微調前級放大器功率（注意保護聽力）

這讓您在 iPhone 或 iPad 上擁有錄音室級別的音訊控制。

### OPUS 檔案格式支援

Flacbox 現在可以播放 **OPUS** 檔案，以及 FLAC、ALAC、MP3 和其他格式。OPUS 是一種廣泛用於語音和音樂的高效編解碼器。OPUS 播放完全支援等化器。

### 從外接 SD 卡播放音樂

直接從連接到 iPhone 或 iPad 的 SD 或 microSD 卡播放音樂：

- 使用 **Lightning to SD Card Camera Reader Adapter**
- 插入卡片並連接到裝置
- 開啟 Flacbox -- 自動辨識卡片
- 透過 **Services → PowerDrive** 部分瀏覽和播放檔案

這非常適合在不使用裝置儲存空間的情況下擴展音樂庫。

### FLAC 播放錯誤修復

**FLAC 檔案末尾的 2 秒跳過**問題已完全解決。曲目現在從頭到尾無縫播放。

## 常見問題

{{% details title="Flacbox 支援哪些音訊格式？" closed="true" %}}
Flacbox 支援 FLAC、ALAC、MP3、AAC、OGG、OPUS、WAV、AIFF、DSD 和其他熱門音訊格式。所有格式均可使用內建等化器。
{{% /details %}}

{{% details title="我可以在 iPhone 上從 SD 卡播放音樂嗎？" closed="true" %}}
可以。使用 Lightning to SD Card Camera Reader Adapter 連接 SD 或 microSD 卡。Flacbox 自動偵測卡片，讓您直接從外部儲存裝置瀏覽和播放檔案。
{{% /details %}}

{{% details title="Flacbox 能自動與雲端儲存同步嗎？" closed="true" %}}
能。從 1.6 版本開始，Flacbox 可以自動從雲端資料夾同步您的音樂庫。在設定中啟用自動同步並選擇要監控的資料夾。
{{% /details %}}

{{% details title="Flacbox 等化器可以自訂嗎？" closed="true" %}}
可以。10 段等化器允許您在 -12 dB 和 +12 dB 之間調整各個頻率級別。您可以使用內建預設或儲存自己的自訂設定。
{{% /details %}}

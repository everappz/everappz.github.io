## App Store कीवर्ड आपके डाउनलोड नंबर क्यों निर्धारित करते हैं

**संक्षेप में:** आपके App Store टाइटल, सबटाइटल और कीवर्ड फील्ड का हर कैरेक्टर सर्च रैंकिंग को प्रभावित करता है। यह गाइड प्रत्येक फील्ड को ऑप्टिमाइज़ करने के नियम कवर करती है और [AppKeywords.pro](https://appkeywords.pro) पेश करती है — एक मुफ़्त, प्राइवेट, ब्राउज़र-आधारित टूल जो मेटाडेटा को मान्य करता है, डुप्लिकेट डिटेक्ट करता है और Fastlane वर्कफ़्लो के लिए JSON एक्सपोर्ट करता है।

ऑप्टिमाइज़्ड मेटाडेटा से मिलता है: अधिक सर्च विज़िबिलिटी, अधिक ऑर्गेनिक डाउनलोड, व्यापक पहुंच, बिना पेड एड्स के बेहतर रैंकिंग।

## AppKeywords.pro क्या है?

[AppKeywords.pro](https://appkeywords.pro) एक मुफ़्त, हल्का ASO टूल है जो पूरी तरह आपके ब्राउज़र में चलता है।

### मुख्य फीचर्स

- **100% लोकल** — कोई लॉगिन नहीं, कोई डेटा कलेक्शन नहीं
- **रियल-टाइम कैरेक्टर काउंट** टाइटल (30), सबटाइटल (30), कीवर्ड (100)
- **वन-क्लिक ऑप्टिमाइज़ेशन** — कॉमा नॉर्मलाइज़, व्हाइटस्पेस ट्रिम, डुप्लिकेट हटाए
- **विज़ुअल कीवर्ड बबल्स** — यूनीक के लिए नीला, डुप्लिकेट के लिए लाल
- **ऑटोसेव** localStorage के माध्यम से
- **JSON इम्पोर्ट/एक्सपोर्ट** Fastlane CI/CD इंटीग्रेशन के लिए

## App Store मेटाडेटा कैसे ऑप्टिमाइज़ करें (स्टेप बाय स्टेप)

### 1. टाइटल, सबटाइटल और कीवर्ड दर्ज करें

- **टाइटल** — अधिकतम 30 कैरेक्टर
- **सबटाइटल** — अधिकतम 30 कैरेक्टर
- **कीवर्ड** — अधिकतम 100 कैरेक्टर

### 2. ऑप्टिमाइज़र चलाएं

**Optimize** क्लिक करें: स्पेस को कॉमा से बदलें, डुप्लिकेट डिटेक्ट करें, कीवर्ड बबल्स दिखाएं।

### 3. ऑटोसेव पर भरोसा करें

सभी बदलाव localStorage में सेव रहते हैं।

### 4. JSON इम्पोर्ट और एक्सपोर्ट करें

### 5. Fastlane के साथ इंटीग्रेट करें

```bash
meta_to_json_dict.sh       # Converts Fastlane metadata into JSON
json_dict_to_meta.sh       # Converts JSON back into Fastlane folders
```

## बेहतर रैंकिंग के लिए ASO बेस्ट प्रैक्टिसेज़

- **इंटेंट-आधारित कीवर्ड उपयोग करें**
- **कभी कीवर्ड डुप्लिकेट न करें** टाइटल, सबटाइटल और कीवर्ड फील्ड में
- **सभी 100 कैरेक्टर भरें**
- **हर प्रमुख बाज़ार के लिए मेटाडेटा लोकलाइज़ करें**
- **तिमाही में कीवर्ड रिफ्रेश करें**
- **कीवर्ड को कॉमा से अलग करें, स्पेस नहीं**

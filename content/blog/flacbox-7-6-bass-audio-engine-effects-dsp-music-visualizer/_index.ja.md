## 新しいBASS™オーディオエンジン

Flacbox 7.6の目玉は、**新しい[BASS™](https://www.un4seen.com)オーディオエンジン**です。これは[un4seen Developments](https://www.un4seen.com)による高く評価される**BASS™**オーディオライブラリを基盤としたプロ仕様の再生コアで、既存のエンジンに代わる選択肢として選ぶことができます。必要なときはクラシックエンジンを使い続けることも、オーディオファイル向けのフルツールキットが欲しいときはBASS™に切り替えることもできます。選択はあなた次第で、あなたの好みに合わせて、切り替えは設定ひとつで完了します。

BASS™エンジンを選ぶと、本リリースの中心となる機能がすべて利用できるようになります。

- **リアルタイムオーディオエフェクトのフルチェーン** — スタジオ品質のプロセッサーを重ねて、変化を瞬時に聴くことができます（下記の[エフェクトスイート](#new-audio-effects-suite)を参照）。
- **リアルタイムDSPプロセッサー** — [14種類のフィルター](#new-dsp-processor)から独自のシグナルチェーンを構築できます。
- **ライブビジュアライゼーション** — 音楽にリアルタイムで反応する、フルスクリーンのアニメーションビジュアル。
- **トラッカーおよびMOD音楽の再生** — クラシックなモジュール形式を再生できます：**MOD、XM、IT、S3M、MTM、UMX、MO3**。これらはレンダリング済みの波形ではなくパターンとサンプルを保存するチップチューンおよびデモシーンの形式で、Flacboxはネイティブに再生できるようになりました。
- **高品質なリサンプリング** — 既存のロスレスライブラリを、クリーンで高品質なサンプルレート変換で再生します。
- **正確で独立したピッチとテンポのコントロール** — 一方に影響を与えることなく、キーを最大**±60セミトーン**シフトする、*または*速度を**0.1×から4×**まで変更できます。練習、耳コピ、DJスタイルのリスニングに最適です。

Flacboxは**ハイレゾ・ロスレスプレーヤー**であるため、BASS™エンジンは**FLAC、DSD、ALAC、APE、WAV、AIFF、MP3、AAC、Opus**ファイルを期待どおりの忠実度で再生します。そして今、その上にリアルタイム処理チェーンが加わりました。

## 自動音量レベリング

あるアルバムが大音量で鳴り、次の曲はささやくような音量、というプレイリストを作ってしまったことはありませんか？Flacbox 7.6は**自動音量レベリング**でこれを解決します。**EBU R128**規格（放送やストリーミングサービスで使われるのと同じITU-R BS.1770ラウドネス測定）に基づく**ラウドネスベースの正規化**を用いて、各トラックの実際の知覚ラウドネスを**LUFS**で測定し、一貫したターゲットへと緩やかに調整します。これにより、曲の合間に絶えず音量ボタンに手を伸ばす必要がなくなります。

**ローカルファイルについては、Flacboxがライブラリを事前スキャン**するため、再生を開始した時点ですでにレベリングされた状態で始まります。トラック開始後にアプリが適切なラウドネスに「追いつく」のを待つ必要はありません。その結果、混在したライブラリ、コンピレーション、シャッフルセッションを通じて、滑らかで均一なリスニング音量が得られます。

一般的なケースをカバーする4つのプリセットがあります。

- **ライト** — 緩やかなレベリング（ターゲット −20 LUFS、最大 +6 dB のリフト）。
- **スタンダード** — デフォルトの、ストリーミング風のラウドネス（ターゲット −16 LUFS、静かなトラックには最大 +12 dB のリフト）。
- **ストロング** — 非常に混在したライブラリ向けのアグレッシブなマッチング（ターゲット −14 LUFS、最大 +18 dB）。
- **ナイト** — 夜間の低音量リスニング向けの、より静かで一貫した設定（ターゲット −23 LUFS）。

## <a id="new-audio-effects-suite"></a>新しいオーディオエフェクトスイート

Flacbox 7.6は、音楽を再生しながら重ねて調整できる**11種類のリアルタイムオーディオエフェクト**のスイートを追加します。各エフェクトはBASS™エンジン内でライブに動作するため、ファイルを再エンコードすることなく、聴くすべてのものに適用されます。すべてのエフェクトには**専用の画面、プリセットのライブラリ、そして即座のオン/オフ切り替え**が備わっており、ワンタップでサウンドを試聴し、同じくらい素早くオフにできます。

- **リバーブ** — 狭い部屋から広いホールまで、空間の広がりを加えます。
- **ディレイ** — 時間とフィードバックを調整できるクラシックなエコー。
- **マルチタップエコー** — 単一のディレイラインよりも豊かなエコーを生み出す、層状でリズミカルな繰り返し。
- **コーラス** — わずかにデチューンしたコピーをブレンドして、サウンドに厚みと広がりを与えます。
- **フランジャー** — 変調された短いディレイによって生まれる、うねるようなジェット機の「ヒューッ」という音。
- **フェイザー** — 周波数スペクトル全体を横切る、滑らかで渦を巻くような動き。
- **オートワウ** — 音楽のダイナミクスに反応する、エンベロープ駆動のフィルタースイープ。
- **ディストーション** — 繊細な温かみとザラつきから、フルなローファイキャラクターまで。
- **コンプレッサー** — 大きい部分と小さい部分を均して、よりコントロールされた前に出るサウンドにします。
- **ステレオローテーション** — サウンドをステレオフィールド内で動かし、広がりのある動くイメージを作ります。
- **クロスフィード** — 各チャンネルを互いに少しずつミックスすることで、ハードにパンされた録音を**ヘッドホンでより自然に**、部屋で実際のスピーカーを聴くのに近い感覚で聴かせます。

どのコントロールを調整してもFlacboxが設定を記憶するので、サウンドはあなたが残したとおりに正確に呼び戻されます。

## <a id="new-dsp-processor"></a>新しいDSPプロセッサー

さらに深く掘り下げたいリスナーのために、Flacbox 7.6は**14種類のフィルターから独自のシグナルチェーンを構築できる**、**リアルタイムDSPプロセッサー**を導入します。任意の順序で組み合わせて、部屋を補正したり、耳障りな録音を整えたり、完全にカスタムなトーンを作り込んだりできます。エフェクトと同様に、各フィルターには**プリセットと即座のオン/オフ切り替え**が備わっています。

14種類のリアルタイムDSPフィルターは以下のとおりです。

1. **ゲイン** — チェーンのレベル調整。
2. **ローパスフィルター** — 選択した周波数より上の高域をカットします。
3. **ハイパスフィルター** — 選択した周波数より下の低域をカットします。
4. **バンドパスフィルター** — ある帯域を残し、それ以外を除去します。
5. **ノッチフィルター** — 狭い問題のある周波数を外科的に除去します。
6. **ピーキングEQ** — ベル型の帯域をブースト、またはカットします。
7. **ローシェルフEQ** — ある周波数より下すべてを整えます（低音のコントロール）。
8. **ハイシェルフEQ** — ある周波数より上すべてを整えます（高音のコントロール）。
9. **ソフトクリップサチュレーション** — 緩やかで音楽的な温かみとラウドネス。
10. **ビットクラッシャー** — 意図的なローファイでレトロなデジタルキャラクター。
11. **トレモロ** — リズミカルな音量変調。
12. **ディレイ** — チェーンのどこにでも配置できるDSPレベルのエコー。
13. **リングモジュレーター** — 金属的でベルのような、実験的なテクスチャ。
14. **ステレオワイズ** — ステレオイメージを狭めたり広げたりします。

[10バンドイコライザー](#refreshed-design)と[エフェクトスイート](#new-audio-effects-suite)と組み合わせることで、DSPプロセッサーはFlacboxをロスレスライブラリのための本格的な**モバイルオーディオワークステーション**へと変えます。

パーティーで立てかけたスマートフォンやiPad、デスクの上のiPad、大画面のMacにぴったりです。あなたの音楽は今、その音と同じくらい素晴らしい見た目になりました。

## このアップデートが重要な理由

Flacbox 7.6は、ひとつの考え方を中心に構築されています。**あなたのロスレスライブラリはプロ品質の再生体験に値する — そしてその見た目は、その音と同じくらい素晴らしくあるべきだ。**

1. **選べるプロ用エンジン。** 新しいBASS™オーディオエンジンはクラシックなエンジンと並んで存在するため、望むときはいつでもオーディオファイル向けのフルツールキットに切り替え、そうでないときはシンプルなままにしておけます。
2. **ポケットの中のスタジオ。** 11種類のリアルタイムエフェクト、14フィルターのDSPプロセッサー、再設計されたイコライザー、そして自動音量レベリングが、単なるオン/オフのスイッチではなく、あなたのサウンドに対する本当のコントロールをもたらします。
3. **探索する新しい音楽。** ネイティブな**トラッカーおよびMOD**の再生は、ほとんどのiPhoneプレーヤーが単に再生できない、何十年にもわたるチップチューンとデモシーンの音楽への扉を開きます。
4. **どこでも音楽を目で見る。** リアルタイムビジュアライザーはエンジンとデバイスをまたいで動作し、画面をオンのまま保つのでショーは続きます。

[BASS™](https://www.un4seen.com)オーディオエンジンは、un4seen DevelopmentsによるBASS™オーディオライブラリを基盤としたプロ仕様の再生コアで、**Flacboxの既存エンジンに代わる選択肢**として選ぶことができます。これを選択すると、リアルタイムオーディオエフェクトのフルチェーン、DSPプロセッサー、ライブビジュアライゼーションが利用でき、さらにクラシックなトラッカーおよびMOD音楽の再生が加わります。既存のロスレスおよびハイレゾライブラリ（FLAC、DSD、ALAC、APEなど）を、**高品質なリサンプリング**と**正確なピッチとテンポのコントロール**で再生します。いつでもクラシックエンジンに戻すことができます。
{{% /details %}}

{{% details title="Flacbox 7.6はどのオーディオ形式とトラッカー/MOD形式を再生できますか？" closed="true" %}}
Flacboxはハイレゾ・ロスレスプレーヤーであり続け、**FLAC、DSD、APE、ALAC、WAV、AIFF、MP3、AAC、Opus**などを扱います。7.6での新機能として、BASS™エンジンはクラシックな**トラッカーおよびモジュール音楽**も再生します：**MOD、XM、IT、S3M、MTM、UMX、MO3** — これらはチップチューンやデモシーンの音楽で使われるパターン＆サンプル形式で、ほとんどのiPhoneプレーヤーは開くことができません。
{{% /details %}}

{{% details title="Flacboxの自動音量レベリングはどのように機能しますか？" closed="true" %}}
Flacbox 7.6は**EBU R128ラウドネス測定**（ITU-R BS.1770規格）を用いて、異なるアルバムのトラックを一貫した知覚音量に保つので、曲の合間に音量を調整する必要がありません。**ローカルファイルについては、ライブラリが事前スキャン**されるため、再生を開始した時点ですでにレベリングされた状態で始まります。トラック開始後にアプリがラウドネスを測定する間の遅延はありません。4つのプリセット — **ライト**（−20 LUFS）、**スタンダード**（−16 LUFS）、**ストロング**（−14 LUFS）、**ナイト**（−23 LUFS） — が用意されており、混在したライブラリ、コンピレーション、シャッフルセッションを通じて機能します。
{{% /details %}}

{{% details title="Flacbox 7.6にはどのオーディオエフェクトがありますか？" closed="true" %}}
音楽を再生しながら重ねて調整できる11種類のリアルタイムエフェクトがあります：**リバーブ、ディレイ、マルチタップエコー、コーラス、フランジャー、フェイザー、オートワウ、ディストーション、コンプレッサー、ステレオローテーション、クロスフィード**。各エフェクトには**専用の画面、プリセットのライブラリ、そして即座のオン/オフ切り替え**が備わっており、Flacboxはセッション間で設定を記憶します。特にクロスフィードは、ハードにパンされた録音をヘッドホンでより自然に聴かせます。
{{% /details %}}

{{% details title="DSPプロセッサーとは何で、どのフィルターが含まれていますか？" closed="true" %}}
DSPプロセッサーでは、**14種類のフィルターから独自のリアルタイムシグナルチェーンを構築**できます：ゲイン、ローパス、ハイパス、バンドパス、ノッチの各フィルター、ピーキングEQ、ローシェルフおよびハイシェルフEQ、ソフトクリップサチュレーション、ビットクラッシャー、トレモロ、ディレイ、リングモジュレーター、ステレオワイズ。各フィルターには**プリセットと即座のオン/オフ切り替え**が備わっているので、部屋を補正したり、耳障りな録音を整えたり、完全にカスタムなトーンを設計したりできます。
{{% /details %}}

{{% details title="クロスフィードとは何で、なぜヘッドホンで使うのですか？" closed="true" %}}
クロスフィードは、各ステレオチャンネルをフィルターしたわずかな量だけ互いにミックスします。これは、あなたの耳が部屋で実際のスピーカーを自然に聴くのと同じ働きです。ヘッドホンではこれが、ハードにパンされた録音の誇張された「頭の中で鳴る」ような分離を軽減し、長時間のリスニングをより快適にします。特に1960年代や1970年代の古いステレオミックスで効果的です。
{{% /details %}}

{{% details title="Flacboxの音楽ビジュアライザーはすべてのデバイスで動作しますか？" closed="true" %}}
はい。**リアルタイム音楽ビジュアライザー**は、音楽にライブで反応するフルスクリーンのアニメーションビジュアルを表示し、選べる、または自動で切り替えられる豊富なプリセットのライブラリを備えています。**すべてのデバイスの再生エンジンをまたいで**利用でき、内蔵の**画面スリープ防止機能**がディスプレイをオンのまま保つので、曲の途中でビジュアルが途切れることはありません。
{{% /details %}}

{{% details title="一方に影響を与えずにピッチとテンポを変更できますか？" closed="true" %}}
はい。新しいBASS™エンジンを使うと、Flacbox 7.6は**正確で独立したピッチとテンポのコントロール**を提供します。キーを変えずにトラックの速度を変更したり、速度を変えずにキーをシフトしたりできます。練習、耳コピ、DJスタイルのリスニングに役立ちます。
{{% /details %}}

{{% details title="Flacbox 7.6でCarPlayは何が改善されましたか？" closed="true" %}}
CarPlayには、車から設定にアクセスできる**DSP設定**、**アルバムアートワークと再生中画面**の表示の修正、そして再生と同期を保つ**ロック画面・Apple Watch・車載コントロールのより正確な表示**が加わりました。より信頼性の高いストリーミング基盤と組み合わさることで、移動中にロスレスライブラリを聴くのがより滑らかになります。
{{% /details %}}

{{% details title="エフェクト、DSP、イコライザーはクラウドストリーミングで動作しますか？" closed="true" %}}
はい。エフェクト、DSPフィルター、イコライザー、音量レベリングは、BASS™再生エンジン内でリアルタイムに動作するため、Flacboxが再生するすべてのもの — **ローカルファイル、クラウドドライブ（iCloud Drive、Google Drive、Dropbox、OneDriveなど）、メディアサーバー、ネットワーク共有** — に、ファイルを再エンコードすることなく適用されます。
{{% /details %}}

{{% details title="Flacbox 7.6は無料アップデートですか、また、どのデバイスに対応していますか？" closed="true" %}}
はい。FlacboxはApp Storeから**無料でダウンロード**でき、7.6は既存ユーザー向けの**無料アップデート**です。高度な機能はオプションのアプリ内アップグレードでご利用いただけます。**iPhone、iPad、Mac**で動作します。CarPlayにはCarPlay対応の車両またはヘッドユニットが必要です。

**요약:** [App Store에서 Flacbox](https://apps.apple.com/us/app/flacbox-flac-player-music/id1097564256?mt=8)를 설치하여 iPhone과 Mac에서 FLAC, DSD, ALAC 및 120개 이상의 오디오 형식을 재생하세요. iTunes 파일 공유, Wi-Fi Drive 또는 클라우드 저장소를 통해 파일을 가져오세요. 파일 변환이 필요 없습니다. Flacbox는 무손실 형식을 네이티브로 디코딩하여 완전한 스튜디오 품질 재생을 제공합니다.

100만 건 이상의 다운로드와 4.6점의 별점으로, Flacbox는 Apple 생태계에서 가장 신뢰받는 오디오 플레이어 중 하나입니다. 무손실과 손실 형식을 모두 지원하여 iPhone, iPad 또는 Mac에서 거의 모든 음악 파일을 재생할 수 있는 자유를 제공합니다.

## 무손실 오디오란?

무손실 오디오는 원본 스튜디오 녹음의 모든 디테일을 보존합니다. 파일 크기를 줄이기 위해 오디오 신호의 일부를 버리는 MP3와 AAC와 달리, 무손실 압축은 완전한 파형을 유지합니다. 디코딩된 출력은 소스와 비트 단위로 동일합니다.

무손실과 손실 오디오의 주요 차이점:

- **다이나믹 레인지**: 무손실은 조용한 부분에서 큰 부분까지 전체 범위를 보존
- **스테레오 이미징**: 악기의 공간적 위치가 정확하게 유지
- **디테일**: 어쿠스틱 악기, 보컬, 오케스트라 녹음의 미묘한 질감이 유지

일반적인 무손실 형식과 해상도:

- **FLAC / ALAC**: CD 품질 16-bit/44.1 kHz, hi-res는 24-bit/192 kHz 이상
- **DSD**: 매우 높은 샘플 레이트(DSD64, DSD128, DSD256) 사용, Super Audio CD를 위해 개발
- **WAV / AIFF**: 비압축 무손실 오디오

## iPhone에서 FLAC 파일 재생 방법

Apple의 기본 Music 앱은 FLAC를 지원하지 않습니다. Flacbox는 기기에서 직접 실행되는 네이티브 FLAC 디코더를 제공합니다.

[App Store](https://apps.apple.com/us/app/flacbox-flac-player-music/id1097564256?mt=8)에서 다운로드 후 다음 방법으로 파일을 가져오세요:

- **iTunes 파일 공유** — Mac의 Finder에서 파일 드래그
- **Wi-Fi Drive** — 같은 네트워크의 모든 브라우저에서 업로드
- **클라우드 저장소** — Google Drive, Dropbox, iCloud, WebDAV 등 연결
- **외부 드라이브** — Lightning 또는 USB-C 어댑터로 연결

Flacbox는 메타데이터를 자동으로 스캔하고 앨범, 아티스트, 장르별로 라이브러리를 정리합니다. 오프라인 재생을 위해 트랙을 로컬로 캐시하므로 인터넷 연결 없이도 무손실 라이브러리가 작동합니다.

## 지원 오디오 형식

Flacbox는 120개 이상의 오디오 형식을 지원합니다. 가장 일반적인 형식:

| 형식 | 유형 | 최대 해상도 | Flacbox 지원 |
|--------|----------|----------------------|-----------------|
| FLAC   | 무손실 | 32-bit / 384 kHz     | 예             |
| DSD    | 무손실 | DSD64 / DSD128 / DSD256 | 예          |
| ALAC   | 무손실 | 32-bit / 384 kHz     | 예             |
| APE    | 무손실 | 24-bit / 192 kHz     | 예             |
| WAV    | 무손실 | 32-bit / 384 kHz     | 예             |
| AIFF   | 무손실 | 32-bit / 384 kHz     | 예             |
| OPUS   | 손실    | 48 kHz               | 예             |
| MP3    | 손실    | 320 kbps / 48 kHz    | 예             |
| AAC    | 손실    | 320 kbps / 96 kHz    | 예             |
| OGG    | 손실    | 500 kbps / 192 kHz   | 예             |

파일 변환이 필요 없습니다. Flacbox는 DSD64, DSD128, DSD256를 네이티브로 처리하여 진정한 DSD 재생이 가능한 몇 안 되는 모바일 플레이어 중 하나입니다.

## iOS에서 시작하기

1. App Store에서 Flacbox 다운로드
2. 소스 메뉴를 열고 가져오기 방법 선택 (iTunes 파일 공유, Wi-Fi Drive 또는 클라우드 저장소)
3. Flacbox가 내장 태그와 커버 아트를 자동으로 읽음
4. 앨범, 아티스트, 장르 또는 폴더 구조로 라이브러리 탐색
5. 재생 목록 만들기, 메타데이터 편집, 즐겨찾기 표시

지원 클라우드 서비스: Google Drive, Dropbox, OneDrive, pCloud, MEGA, iCloud 등.

## Mac에서 Flacbox 설정

Mac App Store에서 Flacbox를 다운로드하세요. macOS 버전은 iOS와 동일한 재생 품질을 제공합니다.

- 로컬 폴더, 외부 드라이브 또는 네트워크 위치 추가
- Flacbox가 지원되는 모든 오디오 파일을 자동으로 인덱싱
- SMB, WebDAV, DLNA를 통해 NAS 또는 홈 서버에 연결
- 사용자 정의 프리셋이 있는 10밴드 이퀄라이저 사용
- 라이브 앨범과 연속 믹스를 위한 갭리스 재생
- 외부 USB 또는 Thunderbolt DAC를 통한 고해상도 출력

## Apple 기기에서 Flacbox를 선택해야 하는 이유

- 정밀한 톤 제어와 커스텀 프리셋이 있는 **10밴드 이퀄라이저**

- 재생 목록과 설정을 위한 iOS와 macOS 간 **크로스 플랫폼 동기화**
- iCloud, pCloud, HiDrive, MEGA, Synology, DLNA, SMB와 **클라우드 통합**
- 다운로드한 트랙에 빠르게 접근할 수 있는 **오프라인 캐싱**
- 메타데이터와 앨범 아트워크 관리를 위한 **내장 태그 편집기**
- **100만 건 이상 다운로드**와 App Store 4.6점 평점

## 더 나은 무손실 재생을 위한 팁

- **전용 DAC 사용**: USB-C와 Lightning DAC 어댑터가 기기의 내장 컨버터를 우회하여 더 좋은 출력 품질 제공
- **유선 헤드폰 사용**: Bluetooth는 오디오를 압축하여 무손실 파일의 장점을 감소
- **이퀄라이저 조정**: 헤드폰이나 스피커에 맞게 10밴드 EQ 조정
- **메타데이터 정리**: 정확한 태그와 일관된 폴더 구조로 대규모 라이브러리 탐색 용이
- **hi-res 파일 선택**: 24-bit/96 kHz 이상이 최대 디테일 제공
- **NAS 스트리밍을 위한 안정적인 네트워크**: 대용량 DSD나 hi-res FLAC 파일의 버퍼링 방지를 위해 유선 이더넷이나 강한 Wi-Fi 사용

## 자주 묻는 질문

{{< details title="Flacbox는 무손실 파일 재생에 구독이 필요한가요?" closed="true" >}}
Flacbox는 구독 없이 핵심 재생 기능을 제공합니다. 앱 다운로드 후 바로 FLAC, DSD, ALAC 및 기타 무손실 형식을 가져와 재생할 수 있습니다. 클라우드 스트리밍과 추가 사용자 정의 옵션 같은 일부 고급 기능은 프리미엄 업그레이드가 필요할 수 있지만, 기본 무손실 재생은 즉시 사용 가능합니다.
{{< /details >}}

{{< details title="Flacbox는 DSD 파일을 PCM으로 변환하지 않고 재생할 수 있나요?" closed="true" >}}
네, Flacbox는 DSD64, DSD128, DSD256 형식을 포함한 네이티브 DSD 재생을 지원합니다. 앱은 DSD 스트림을 직접 디코딩하여 형식의 고유한 음질 특성을 보존합니다. 최상의 결과를 위해 DSD 호환 외부 DAC와 함께 사용하세요.
{{< /details >}}

{{< details title="대규모 무손실 음악 컬렉션을 iPhone으로 어떻게 전송하나요?" closed="true" >}}
Flacbox는 대규모 라이브러리를 위한 여러 전송 옵션을 제공합니다. Wi-Fi Drive를 사용하면 로컬 네트워크의 모든 브라우저에서 파일을 업로드할 수 있습니다. Mac의 Finder를 통한 iTunes 파일 공유를 사용하거나 Google Drive나 Dropbox 같은 클라우드 저장소 서비스를 연결할 수도 있습니다. 매우 큰 컬렉션의 가장 빠른 전송을 위해 Lightning 또는 USB-C 어댑터를 사용하여 외부 드라이브를 직접 연결하세요.
{{< /details >}}

{{< details title="Flacbox에서 FLAC과 ALAC의 음질 차이가 있나요?" closed="true" >}}
FLAC과 ALAC 모두 무손실 코덱이므로 디코딩 시 동일한 오디오 출력을 생성합니다. 차이점은 호환성과 압축 효율성에 있습니다. FLAC은 플랫폼 간 더 널리 사용되며 일반적으로 약간 더 나은 압축률을 달성하고, ALAC는 Apple의 네이티브 무손실 형식입니다. Flacbox는 두 형식 모두 동일한 충실도로 처리합니다.
{{< /details >}}

{{< details title="iPhone에서 FLAC 파일을 재생하는 가장 좋은 방법은 무엇인가요?" closed="true" >}}
App Store에서 Flacbox를 설치한 다음 iTunes 파일 공유, Wi-Fi Drive, 클라우드 저장소 또는 USB/Lightning 외부 드라이브를 사용하여 FLAC 파일을 가져오세요. Flacbox는 32-bit/384 kHz까지의 해상도를 지원하며 변환 없이 FLAC를 네이티브로 디코딩합니다. 최상의 오디오 품질을 위해 전용 USB-C 또는 Lightning DAC와 함께 사용하세요.
{{< /details >}}

{{< details title="Flacbox는 NAS와 홈 서버에서 작동하나요?" closed="true" >}}
네. Flacbox는 SMB, WebDAV, DLNA 프로토콜을 통해 NAS 장치와 홈 서버에 연결됩니다. Mac에서는 네트워크 위치를 직접 추가할 수 있습니다. iOS에서는 클라우드/네트워크 소스 메뉴를 통해 연결하세요. 이를 통해 파일을 기기에 복사하지 않고도 무손실 라이브러리를 스트리밍할 수 있습니다.
{{< /details >}}

## Apa Itu Audio Lossless?

Audio lossless mempertahankan setiap detail dari rekaman studio asli. Tidak seperti MP3 dan AAC yang membuang bagian sinyal audio, kompresi lossless mempertahankan gelombang lengkap.

Perbedaan utama: **Rentang dinamis** penuh dipertahankan, **pencitraan stereo** tetap akurat, **detail** halus pada instrumen akustik tetap terjaga.

Format lossless umum: **FLAC / ALAC** (kualitas CD hingga hi-res), **DSD** (DSD64, DSD128, DSD256), **WAV / AIFF** (tidak terkompresi).

## Cara Memutar File FLAC di iPhone

Aplikasi Musik bawaan Apple tidak mendukung FLAC. Flacbox menyediakan dekoder FLAC native. Metode impor: **iTunes File Sharing**, **Wi-Fi Drive**, **penyimpanan cloud**, **drive eksternal**.

## Tips untuk Pemutaran Lossless Lebih Baik

- **Gunakan DAC khusus** untuk kualitas output lebih baik
- **Gunakan headphone berkabel**: Bluetooth mengompresi audio
- **Sesuaikan equalizer**: Cocokkan EQ 10-band dengan headphone Anda
- **Organisir metadata**: Tag akurat memudahkan navigasi
- **Pilih file hi-res**: 24-bit/96 kHz atau lebih tinggi memberikan detail maksimal

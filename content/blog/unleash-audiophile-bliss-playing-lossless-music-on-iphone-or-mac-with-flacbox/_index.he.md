**בקצרה:** התקינו את [Flacbox מה-App Store](https://apps.apple.com/us/app/flacbox-flac-player-music/id1097564256?mt=8) כדי לנגן FLAC, DSD, ALAC ו-120+ פורמטי אודיו נוספים ב-iPhone ו-Mac. ייבאו קבצים דרך iTunes File Sharing, Wi-Fi Drive או אחסון ענן. ללא צורך בהמרת קבצים. Flacbox מפענח פורמטים lossless באופן מקורי להשמעה באיכות סטודיו.

עם למעלה ממיליון הורדות ודירוג של 4.6 כוכבים, Flacbox הוא אחד מנגני האודיו המהימנים ביותר באקוסיסטם של Apple. הוא תומך גם בפורמטים lossless וגם lossy, ומעניק לכם חופש לנגן כמעט כל קובץ מוזיקה ב-iPhone, iPad או Mac.

## מהו אודיו lossless?

אודיו lossless שומר על כל פרט מההקלטה המקורית בסטודיו. בניגוד ל-MP3 ו-AAC, שמשמיטים חלקים מאות השמע כדי להקטין את גודל הקובץ, דחיסת lossless שומרת על צורת הגל המלאה. הפלט המפוענח זהה ביט-לביט למקור.

הבדלים עיקריים בין אודיו lossless ל-lossy:

- **טווח דינמי**: Lossless שומר על הטווח המלא מקטעים שקטים לקטעים חזקים
- **הדמיה סטריאו**: מיקום מרחבי של כלי נגינה נשאר מדויק
- **פירוט**: מרקמים עדינים בכלי נגינה אקוסטיים, שירה והקלטות תזמורתיות נשמרים

פורמטים lossless נפוצים והרזולוציות שלהם:

- **FLAC / ALAC**: איכות CD ב-16-bit/44.1 kHz, hi-res עד 24-bit/192 kHz ומעלה
- **DSD**: משתמש בקצבי דגימה גבוהים במיוחד (DSD64, DSD128, DSD256), פותח במקור עבור Super Audio CD
- **WAV / AIFF**: אודיו lossless לא דחוס

## כיצד לנגן קבצי FLAC באייפון

אפליקציית המוזיקה המובנית של Apple אינה תומכת ב-FLAC. Flacbox מספק מפענח FLAC מקורי שפועל ישירות על המכשיר שלכם.

לאחר ההורדה מה-[App Store](https://apps.apple.com/us/app/flacbox-flac-player-music/id1097564256?mt=8), ייבאו קבצים באחת מהשיטות הבאות:

- **iTunes File Sharing** — גררו קבצים מ-Finder ב-Mac
- **Wi-Fi Drive** — העלו מכל דפדפן באותה רשת
- **אחסון ענן** — חברו Google Drive, Dropbox, iCloud, WebDAV ועוד
- **כוננים חיצוניים** — חברו דרך מתאם Lightning או USB-C

Flacbox סורק אוטומטית מטא-נתונים ומארגן את הספרייה שלכם לפי אלבום, אמן וז׳אנר. הוא שומר רצועות במטמון מקומי להשמעה לא מקוונת, כך שספריית ה-lossless שלכם עובדת ללא חיבור אינטרנט.

## פורמטי אודיו נתמכים

Flacbox תומך ביותר מ-120 פורמטי אודיו. הנה הנפוצים ביותר:

| פורמט | סוג     | רזולוציה מקסימלית       | תמיכת Flacbox |
|--------|----------|----------------------|-----------------|
| FLAC   | Lossless | 32-bit / 384 kHz     | כן             |
| DSD    | Lossless | DSD64 / DSD128 / DSD256 | כן          |
| ALAC   | Lossless | 32-bit / 384 kHz     | כן             |
| APE    | Lossless | 24-bit / 192 kHz     | כן             |
| WAV    | Lossless | 32-bit / 384 kHz     | כן             |
| AIFF   | Lossless | 32-bit / 384 kHz     | כן             |
| OPUS   | Lossy    | 48 kHz               | כן             |
| MP3    | Lossy    | 320 kbps / 48 kHz    | כן             |
| AAC    | Lossy    | 320 kbps / 96 kHz    | כן             |
| OGG    | Lossy    | 500 kbps / 192 kHz   | כן             |

ללא צורך בהמרת קבצים. Flacbox מטפל ב-DSD64, DSD128 ו-DSD256 באופן מקורי, מה שהופך אותו לאחד מנגני הנייד הבודדים המסוגלים להשמעת DSD אמיתית.

## התחלה ב-iOS

1. הורידו את Flacbox מה-App Store
2. פתחו את תפריט המקורות ובחרו שיטת ייבוא (iTunes File Sharing, Wi-Fi Drive או אחסון ענן)
3. Flacbox קורא תגיות מוטבעות ועטיפות אוטומטית
4. דפדפו בספרייה לפי אלבום, אמן, ז׳אנר או מבנה תיקיות
5. צרו רשימות השמעה, ערכו מטא-נתונים וסמנו מועדפים

שירותי ענן נתמכים: Google Drive, Dropbox, OneDrive, pCloud, MEGA, iCloud ואחרים.

## הגדרת Flacbox ב-Mac

הורידו את Flacbox מ-Mac App Store. גרסת macOS מציעה את אותה איכות השמעה כמו iOS.

- הוסיפו תיקיות מקומיות, כוננים חיצוניים או מיקומי רשת
- Flacbox מאנדקס אוטומטית את כל קבצי האודיו הנתמכים
- התחברו ל-NAS או שרתים ביתיים דרך SMB, WebDAV ו-DLNA
- השתמשו באקולייזר 10 פסים עם הגדרות מותאמות אישית
- השמעה ללא הפסקות עבור אלבומים חיים ומיקסים רציפים
- פלט ברזולוציה גבוהה דרך DAC חיצוני USB או Thunderbolt

## טיפים להשמעת lossless טובה יותר

- **השתמשו ב-DAC ייעודי**: מתאמי DAC USB-C ו-Lightning עוקפים את הממיר המובנה של המכשיר לאיכות פלט טובה יותר
- **השתמשו באוזניות חוטיות**: Bluetooth דוחס אודיו, מפחית את היתרון של קבצי lossless
- **כווננו את האקולייזר**: התאימו את ה-EQ 10 פסים לאוזניות או רמקולים שלכם
- **ארגנו מטא-נתונים**: תגיות מדויקות ומבני תיקיות עקביים מקלים על ניווט בספריות גדולות
- **בחרו קבצי hi-res**: 24-bit/96 kHz ומעלה מספק פירוט מקסימלי
- **רשת יציבה להזרמת NAS**: השתמשו ב-Ethernet חוטי או Wi-Fi חזק כדי למנוע חציצה עם קבצי DSD או FLAC hi-res גדולים

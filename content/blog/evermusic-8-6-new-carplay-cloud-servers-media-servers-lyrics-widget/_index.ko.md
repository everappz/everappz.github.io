**요약:** [Evermusic 8.6](/products/evermusic)은 iPhone, iPad, Mac을 위한 대규모 업데이트입니다. CarPlay를 처음부터 다시 만들었습니다 — 빠른 정렬, 다양한 색상 구성표, 새롭게 디자인된 재생 중 화면, 재생 대기열 전체 보기, 빠른 알파벳 인덱스 스크롤이 추가되었습니다. 이번 릴리스는 **Plex**, **Jellyfin**, **Emby**, **Subsonic**, **Navidrome**, **Internxt**, **Proton Drive**, **QNAP**, **Nextcloud**, **Amazon S3** 외에 **FTP**, **SFTP**, **NFS** 프로토콜 등 10개 이상의 새로운 연결을 추가합니다. 새로운 **동기화 가사 홈 화면 위젯**은 음악을 들으면서 시간 동기화된 가사를 보여줍니다. Wi-Fi Drive는 새로운 UI, 선택 모드, 더 빠른 업로드 큐를 갖췄습니다. 앱 전체가 **Liquid Glass** 디자인에 맞게 업데이트되었으며 **Baidu Netdisk(百度网盘)**, **Aliyun Drive(阿里云盘)**와 같은 중국 서버에서의 스트리밍이 더 안정적으로 동작합니다.

## 완전히 새로운 CarPlay 경험

**Apple CarPlay**용 Evermusic을 처음부터 다시 만들었습니다. 결과는 더 빠르고, 더 부드러우며, 운전 중 사용하기에 훨씬 쉬워졌습니다.

- **빠른 정렬** — 앨범, 아티스트, 재생 목록, 폴더 전반에서 어떤 트랙도 몇 초 안에 찾을 수 있습니다.
- **다양한 색상 구성표** — 대시보드나 차량 실내 조명에 어울리는 테마를 선택하세요.
- **새롭게 디자인된 재생 중 화면** — 더 큰 아트워크, 더 명확한 컨트롤, 한 번의 탭으로 실행할 수 있는 새로운 재생 동작.
- **재생 대기열 한눈에 보기** — 현재 화면을 떠나지 않고 다음 곡을 확인할 수 있습니다.
- **빠른 알파벳 인덱스 스크롤** — 1만 곡이 넘는 거대한 라이브러리도 끝없는 스크롤 없이 이동할 수 있습니다.
- **더 부드러운 콘텐츠 로딩** — 큰 폴더, 클라우드 디렉터리, 미디어 서버 라이브러리를 열 때 더 이상 멈추지 않습니다.

운전 중에 클라우드(Dropbox, Google Drive, OneDrive, iCloud Drive, [Plex](#media-servers), [Jellyfin](#media-servers) 등)에서 음악을 스트리밍한다면 새로운 CarPlay 경험은 모든 흐름이 차량에 원래부터 있던 것처럼 자연스럽게 느껴집니다.

## 10개 이상의 새로운 클라우드, NAS, 서버 연동

Evermusic 8.6은 ‘음악 라이브러리’의 의미를 확장합니다. 음악이 신뢰하는 클라우드, 집의 NAS, 직접 호스팅한 미디어 서버 어디에 있든 Evermusic이 동기화나 내보내기 없이 직접 재생할 수 있습니다.

### 프라이버시 중심 클라우드: Internxt와 Proton Drive

엔드 투 엔드 암호화와 제로 지식 저장에 신경을 쓴다면, 프라이버시 우선 클라우드 가운데 가장 인정받는 두 곳이 이제 Evermusic에 네이티브로 들어왔습니다.

- **Internxt** — 오픈 소스, 포스트 양자 암호화, GDPR 준수의 스페인 클라우드. 무료 티어 제공.
- **Proton Drive** — Proton Mail과 Proton VPN 제작자가 만든 엔드 투 엔드 암호화 저장소(스위스 본사). 무료 티어 제공, 더 큰 라이브러리를 위한 유료 요금제도 있습니다.

한 번 연결하면 음악이 암호화된 터널을 통해 전달됩니다. Evermusic은 데이터를 평문으로 보지 않고, 제공자 서버 역시 마찬가지입니다.

### 직접 호스팅 저장소: QNAP, Nextcloud, Amazon S3

자체 인프라를 운영하는 사용자라면:

- **QNAP** — QNAP NAS 장치에 네이티브 API 연결. 로컬 Wi-Fi 또는 원격 접속을 통해 빠르고 안정적인 재생을 제공합니다.
- **Nextcloud** — 자체 호스팅 또는 관리형 Nextcloud 인스턴스 모두에 연결할 수 있습니다. 프라이버시 때문에 Google Drive나 Dropbox에서 이미 이주했다면 안성맞춤입니다.
- **Amazon S3** — 모든 S3 버킷(또는 Backblaze B2, Wasabi, MinIO, Cloudflare R2와 같은 S3 호환 저장소)을 Evermusic에 연결하고 컬렉션을 직접 스트리밍하세요.

### <a id="media-servers"></a>미디어 서버: Plex, Subsonic, Navidrome, Jellyfin, Emby

자체 호스팅 음악 팬에게는 가장 큰 변화입니다. Evermusic 8.6은 iPhone, iPad, Mac을 가장 인기 있는 오픈 소스 및 프리미엄 미디어 서버를 위한 일류 클라이언트로 만듭니다.

- **Plex** — Plex Media Server는 다운로드와 실행이 **무료**입니다. **Plex Pass** 구독은 선택 사항이며 모바일 동기화, 하드웨어 트랜스코딩, 라이브 TV 등의 기능을 잠금 해제합니다. Evermusic은 무료 라이브러리와 Plex Pass 라이브러리 모두에서 작동합니다.
- **Subsonic** — 자체 호스팅 음악 스트리밍 서버의 원조입니다. 공식 Subsonic 서버는 **유료**(30일 체험 후 월 1달러)이지만, Evermusic은 수십 종의 호환 서버와 Subsonic API로 통신합니다.
- **Navidrome** — Go로 작성된 현대적이고 가벼운 음악 서버로, **완전 무료이며 오픈 소스**입니다. Subsonic API를 구현하며 Raspberry Pi, NAS 또는 모든 Linux 머신에서 실행됩니다. 수십만 곡 규모의 라이브러리까지 강력 추천합니다.
- **Jellyfin** — **완전 무료이고 오픈 소스인** 미디어 서버(Emby의 커뮤니티 포크)입니다. 음악, 영화, TV, 책, 오디오북을 다루며 계정도, 텔레메트리도, 구독도 없습니다.
- **Emby** — **프리미엄(Freemium)** 미디어 서버입니다. 코어 서버는 무료이며, **Emby Premiere**는 일회성 또는 연간 결제 형태로 모바일 앱, 오프라인 동기화, Cinema 모드 등을 잠금 해제합니다. Evermusic은 무료 라이브러리와 Premiere 라이브러리 모두에 연결됩니다.

어떤 서버를 사용하든 Evermusic은 앨범, 아티스트, 재생 목록, 장르, 임베디드 아트워크를 포함한 전체 라이브러리를 스트리밍합니다. 오프라인 캐시, 갭리스 및 크로스페이드 재생, 10밴드 이퀄라이저, AirPlay, Chromecast, **CarPlay**도 함께 제공합니다. 듣기 기록은 서버가 관리하며 Evermusic은 이를 존중합니다.

### 새로운 네트워크 프로토콜: FTP, SFTP, NFS

맞춤형 서버, 홈랩, 정교한 모바일 앱이 없는 일반 NAS를 사용하는 분들을 위해 Evermusic 8.6은 세 가지 클래식한 프로토콜을 추가했습니다.

- **SFTP(SSH File Transfer Protocol)** — **자체 서버에서 안전하게 원격 스트리밍**할 때 가장 적합한 답입니다. SFTP는 SSH 위에서 동작하므로 인증과 오디오 데이터 등 전체 전송이 암호화됩니다. SSH 접근이 가능한 VPS, 전용 서버, 집의 Linux 머신이 있다면 음악 폴더만 올려두고 다른 것은 노출하지 않은 채 공개 인터넷을 통해 스트리밍할 수 있습니다. 비밀번호 및 키 기반 인증을 모두 지원합니다.
- **FTP** — 오랫동안 자리 잡은 파일 전송 표준입니다. 오래된 Synology, ASUS, D-Link, TerraMaster, 일반 박스 같은 **가정용 NAS**가 FTP 서버는 제공하지만 네이티브 API 연동이 없는 경우에 유용합니다. 로컬 네트워크 내 사용을 권장합니다.
- **NFS(Network File System)** — Linux와 대부분의 NAS에서 사실상 표준이 된 공유 프로토콜입니다. NFS 공유는 홈랩과 소규모 비즈니스 네트워크에서 흔히 쓰이며, Evermusic은 이를 마운트해 직접 스트리밍합니다. 같은 하드웨어에서 SMB보다 오버헤드가 적습니다.

> **팁:** 공개 인터넷에서 스트리밍하려면 SFTP를 사용하세요. FTP와 NFS는 로컬 네트워크 안에서 사용하는 것이 좋습니다 — VPN으로 감싸지 않는 한 공개 인터넷에 노출하지 마세요.

## Wi-Fi Drive: 새로운 UI와 더 빠른 업로드

[**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)는 **iTunes, 케이블, 클라우드 계정 없이 컴퓨터에서 iPhone이나 iPad로 음악을 무선 전송하기 위한 Evermusic 내장 기능**입니다. 두 기기는 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다. iOS 앱에서 서버를 시작한 다음 다음 중 하나를 사용합니다:

- 임의의 데스크톱 브라우저(Safari, Chrome, Firefox, Edge)에서 URL을 열고 음악 파일을 페이지로 끌어다 놓으면 기기로 직접 업로드되거나
- WebDAV를 사용해 **Mac Finder**(‘서버 연결…’)나 **Windows File Explorer**(네트워크 드라이브 연결)에서 기기를 네트워크 공유로 마운트합니다.

서드파티 서비스 없이 큰 로컬 음악 라이브러리를 휴대폰으로 옮기는 가장 빠른 방법입니다. [단계별 가이드를 여기에서 확인](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/) 하세요.

Evermusic 8.6에서 Wi-Fi Drive는 다음을 갖춥니다.

- **새로운 사용자 인터페이스** — 깔끔하고 한눈에 읽기 쉽습니다.
- **선택 모드** — 여러 파일이나 폴더를 선택해 일괄 처리할 수 있습니다.
- **개선된 업로드 큐** — 진행 상황 피드백이 더 명확하고 네트워크 변동에 더 강합니다.
- **전반적으로 향상된 성능** — 큰 라이브러리에서 더 빠른 전송 속도.

## 새로운 동기화 가사 홈 화면 위젯

가장 많이 요청된 기능 중 하나가 마침내 도착했습니다 — **iPhone, iPad, Mac 홈 화면에 고정할 수 있는 동기화 가사 위젯**.

시간이 동기화된 가사가 있는 트랙을 재생하면, 위젯은 곡의 **현재 줄**을 실시간으로 보여주고, 음악이 진행됨에 따라 줄이 함께 움직입니다. 위젯을 탭하면 Evermusic이 전체 가사 보기로 열립니다.

이 위젯이 하는 일:

- 현재 재생 중인 트랙에 대한 **동기화된 타임스탬프 가사**를 표시합니다.
- 곡이 진행될 때마다 자동 업데이트 — 수동 새로 고침이나 스크롤이 필요 없습니다.
- 오디오 파일의 임베디드 LRC 가사와 앱 내에서 가져온 가사를 모두 지원합니다.
- 홈 화면에 표시할 텍스트 양을 선택할 수 있도록 **여러 위젯 크기**로 제공됩니다.
- **iPhone**, **iPad**, **Mac**(지원되는 경우 알림 센터 포함)에서 작동합니다.

긴 청취 세션 동안 책상 위에 휴대폰을 두거나 거치대에 올려두거나 잠금 화면 상태에서 가사를 따라 읽고 싶은 사용자에게 완벽합니다. 읽기로 이해를 돕는 오디오북이나 팟캐스트 형식의 콘텐츠와 특히 잘 어울립니다.

활성화하려면 홈 화면을 길게 누르고 **편집 > 위젯 추가**를 누른 뒤 **Evermusic**을 검색해 원하는 크기의 **가사** 위젯을 선택하세요.

## 그 외 개선

### Liquid Glass 디자인 업데이트

Evermusic 8.6의 인터페이스는 앱 전체에 걸쳐 Apple의 새로운 **Liquid Glass** 머티리얼에 맞춰 업데이트되었습니다. 반투명 표면, 더 부드러운 애니메이션, iOS, iPadOS, macOS에 자연스럽게 어울리는 정제된 컨트롤이 적용됩니다.

### 중국 서버 안정성 개선

이번 릴리스에서는 **Baidu Netdisk(百度网盘)**와 **Aliyun Drive(阿里云盘)**의 재생 안정성에 많은 노력을 기울였습니다. Evermusic에서 이 서비스를 사용해 본 적이 있다면 다음 변화를 느낄 수 있습니다.

- 수천 개의 파일이 있는 라이브러리에서도 디렉터리 목록이 더 빠릅니다.
- 느리거나 지리적으로 먼 연결에서도 스트리밍이 더 안정적입니다.
- 긴 재생 세션에서 연결이 끊기는 경우가 줄었습니다.
- 네트워크가 잠시 끊겼을 때 재시도와 재개 로직이 개선되었습니다.

이 두 클라우드는 중국에서 가장 인기 있는 일반 사용자 저장 서비스이며, Evermusic은 두 서비스 모두 일류 수준의 네이티브 지원을 제공하는 몇 안 되는 iOS 음악 플레이어 중 하나입니다.

### 연결 라이브러리 업데이트

Evermusic이 **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive** 등 다른 서비스와 통신하는 데 사용하는 내부 라이브러리를 업데이트했습니다. 즉, 엣지 케이스의 연결 실패가 줄어들고, 최신 서버 버전에 대한 지원이 향상되며, 전반적인 성능도 개선됩니다.

### 새로운 홈 화면 위젯과 더 똑똑한 업데이트

새로운 가사 위젯 외에도 기존 홈 화면 위젯 — 재생 중, 재생 대기열, 최근 재생 — 이 **더 나은 업데이트 로직**으로 다듬어져 추가 배터리 소모 없이 동기화 상태를 유지합니다. 한눈에 보고 싶은 정보를 더 세밀하게 제어하고 싶은 사용자를 위한 새로운 위젯 레이아웃도 추가되었습니다.

### 버그 수정과 다듬기

- 일부 미디어 서버와 자체 호스팅 환경에서의 재생 문제를 수정했습니다.
- 여러 언어의 현지화 수정.
- App Store 리뷰와 support@everappz.com으로 받은 직접적인 피드백을 바탕으로 한 다수의 작은 개선 사항.

## 이번 업데이트가 의미하는 것

Evermusic 8.6은 세 가지 아이디어를 중심으로 만들어졌습니다.

1. **어디에 보관하든, 당신의 음악.** 라이브러리가 iCloud Drive, Proton Drive나 Internxt 같은 프라이버시 중심 클라우드, Plex나 Jellyfin과 같은 미디어 서버, 집의 NAS, 또는 Navidrome을 실행하는 Raspberry Pi 어디에 있든 Evermusic은 네이티브로 연결되어 어디서나 동일한 재생 경험을 제공합니다.
2. **자동차에서 더 좋아진다.** CarPlay는 많은 사용자가 가장 자주 보는 화면이며, 다시 만든 경험은 그 점을 반영합니다.
3. **홈 화면의 가사.** 새로운 동기화 가사 위젯은 누구도 제대로 해내지 못한 무언가를 사용자의 메인 한눈 화면에 가져옵니다.

## 자주 묻는 질문

{{% details title="Evermusic 8.6의 새로운 점은?" closed="true" %}}
Evermusic 8.6은 완전히 새로 디자인된 CarPlay 경험, 10개 이상의 새로운 연결 지원(Plex, Jellyfin, Emby, Subsonic, Navidrome, Internxt, Proton Drive, QNAP, Nextcloud, Amazon S3, FTP, SFTP, NFS), 새로운 동기화 가사 홈 화면 위젯, 선택 모드를 갖춘 Wi-Fi Drive UI 개선, Liquid Glass 디자인 업데이트, Baidu Netdisk와 Aliyun Drive에 대한 안정성 향상, 다수의 버그 수정을 도입합니다.
{{% /details %}}

{{% details title="Evermusic은 Plex와 함께 작동하나요?" closed="true" %}}
네. Evermusic 8.6부터 Plex Media Server에 연결해 전체 음악 라이브러리를 스트리밍할 수 있습니다. Plex Media Server는 무료로 실행되며, Plex Pass는 선택 사항입니다. Evermusic은 무료 설정과 Plex Pass 설정을 모두 지원합니다.
{{% /details %}}

{{% details title="Evermusic에서 Jellyfin이나 Navidrome이 지원되나요?" closed="true" %}}
네. Jellyfin과 Navidrome 모두 Evermusic 8.6에서 완전히 지원됩니다. Jellyfin은 무료 오픈 소스 미디어 서버이고, Navidrome은 Subsonic API를 구현하는 무료 오픈 소스 음악 서버입니다. Evermusic은 두 가지 모두에 네이티브로 연결됩니다.
{{% /details %}}

{{% details title="Plex, Jellyfin, Emby, Navidrome, Subsonic은 무료인가요?" closed="true" %}}
- **Plex** — 서버는 무료이며, Plex Pass는 선택형 유료 업그레이드입니다.
- **Jellyfin** — 완전 무료이며 오픈 소스입니다.
- **Emby** — 서버는 무료이며, Emby Premiere는 유료로 모바일 동기화와 오프라인을 잠금 해제합니다.
- **Navidrome** — 완전 무료이며 오픈 소스입니다.
- **Subsonic** — 공식 서버는 30일 체험 후 월 1달러이지만 API는 공개되어 있고 Navidrome을 비롯한 다수의 무료 서버가 이를 구현합니다.
{{% /details %}}

{{% details title="가정용 NAS에서 SFTP, FTP, NFS로 스트리밍할 수 있나요?" closed="true" %}}
네. Evermusic 8.6은 SFTP, FTP, NFS를 네이티브 연결 유형으로 추가합니다. SFTP는 모든 트래픽이 SSH로 암호화되므로 공개 인터넷에서 자체 서버로부터 스트리밍하려는 경우 권장되는 선택입니다. FTP와 NFS는 로컬 네트워크 또는 VPN 뒤에서 사용하는 것이 가장 좋습니다.
{{% /details %}}

{{% details title="SFTP를 사용해 Evermusic을 사용자 정의 서버에 어떻게 연결하나요?" closed="true" %}}
Evermusic을 열고 연결하기 탭으로 이동해 SFTP를 선택한 다음 서버의 호스트 이름이나 IP, 포트(보통 22), 사용자 이름, 그리고 비밀번호 또는 SSH 개인 키를 입력하세요. Evermusic이 원격 폴더를 탐색하고 엔드 투 엔드 암호화로 오디오 파일을 직접 스트리밍합니다.
{{% /details %}}

{{% details title="Evermusic은 Internxt와 Proton Drive를 지원하나요?" closed="true" %}}
네. Evermusic 8.6부터 두 가지 프라이버시 중심 클라우드를 모두 지원합니다. 앱에서 이미 사용할 수 있는 Mega 등 다른 프라이버시 우선 서비스에 더해집니다.
{{% /details %}}

{{% details title="Evermusic의 Wi-Fi Drive란 무엇인가요?" closed="true" %}}
Wi-Fi Drive는 Evermusic에 내장된 무선 파일 전송 기능입니다. iTunes, 케이블, 클라우드 계정 없이 로컬 Wi-Fi 네트워크를 통해 컴퓨터에서 iPhone이나 iPad로 음악을 업로드할 수 있습니다. 모든 데스크톱 브라우저나 Mac Finder, Windows File Explorer 같은 WebDAV 클라이언트를 사용할 수 있습니다. [전체 Wi-Fi Drive 가이드](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)를 참고하세요.
{{% /details %}}

{{% details title="새로운 가사 위젯은 어떻게 작동하나요?" closed="true" %}}
가사 위젯은 현재 재생 중인 트랙에 대해 시간 동기화된 가사를 iPhone, iPad, Mac 홈 화면에 표시합니다. 표시되는 줄은 곡과 함께 자동으로 진행됩니다. 홈 화면을 길게 누르고 편집 > 위젯 추가를 탭한 뒤 Evermusic을 검색하고 가사 위젯을 선택해 추가하세요.
{{% /details %}}

{{% details title="Evermusic 8.6은 Baidu Netdisk와 Aliyun Drive 재생 문제를 해결하나요?" closed="true" %}}
네. 百度网盘(Baidu Netdisk)과 阿里云盘(Aliyun Drive) 모두에서 안정성을 크게 개선했습니다. 디렉터리 목록 표시 속도 향상, 약한 연결에서의 더 똑똑한 재시도, 긴 재생 세션 동안의 더 나은 재개 동작 등이 포함됩니다.
{{% /details %}}

{{% details title="Evermusic 8.6 업데이트는 무료인가요?" closed="true" %}}
네. Evermusic은 App Store에서 무료로 다운로드할 수 있고, 8.6은 모든 기존 사용자를 위한 무료 업데이트입니다. 새로운 CarPlay, 가사 위젯, 모든 새로운 서버 통합은 기본 업데이트에 포함되어 있습니다.
{{% /details %}}

{{% details title="Evermusic 8.6은 어떤 기기에서 사용할 수 있나요?" closed="true" %}}
Evermusic 8.6은 iPhone, iPad, Mac에서 작동합니다. CarPlay 지원에는 CarPlay 호환 차량이나 애프터마켓 헤드 유닛이 필요합니다.

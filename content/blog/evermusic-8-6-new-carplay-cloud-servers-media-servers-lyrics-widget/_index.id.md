## Pengalaman CarPlay yang sama sekali baru

Kami membangun ulang Evermusic untuk **Apple CarPlay** dari nol. Hasilnya lebih cepat, lebih halus, dan jauh lebih mudah digunakan saat berkendara.

- **Pengurutan cepat** — temukan trek apa pun dalam hitungan detik di album, artis, daftar putar, dan folder.
- **Beberapa skema warna** — pilih tema yang cocok dengan dasbor atau penerangan interior mobil Anda.
- **Layar Sedang Diputar didesain ulang** — sampul lebih besar, kontrol yang lebih jelas, dan aksi pemutaran baru cukup dengan satu ketukan.
- **Antrean putar penuh dalam satu pandangan** — lihat apa yang berikutnya tanpa meninggalkan layar saat ini.
- **Indeks huruf gulir cepat** — lompati pustaka raksasa (10.000+ trek) tanpa scroll tanpa akhir.
- **Pemuatan konten lebih halus** — tidak ada lagi macet saat membuka folder besar, direktori cloud, atau pustaka server media.

Jika Anda streaming musik dari cloud sambil berkendara — Dropbox, Google Drive, OneDrive, iCloud Drive, [Plex](#media-servers), [Jellyfin](#media-servers), dan lainnya — pengalaman CarPlay baru membuat seluruh alurnya terasa native di dalam mobil.

## Lebih dari 10 integrasi cloud, NAS, dan server baru

Evermusic 8.6 memperluas definisi "pustaka musik" Anda. Apakah audio Anda tersimpan di cloud yang Anda percayai, di NAS rumah, atau di server media yang Anda host sendiri — Evermusic kini bisa memutar langsung dari sana, tanpa sinkronisasi atau ekspor.

### Cloud yang fokus pada privasi: Internxt dan Proton Drive

Jika Anda peduli pada enkripsi ujung-ke-ujung dan penyimpanan tanpa pengetahuan, dua nama paling dihormati di kategori cloud privasi-utama kini hadir secara native di Evermusic:

- **Internxt** — cloud Spanyol open-source, terenkripsi pasca-kuantum, dan patuh GDPR. Tersedia tier gratis.
- **Proton Drive** — penyimpanan terenkripsi ujung-ke-ujung dari pembuat Proton Mail dan Proton VPN, berbasis di Swiss. Tersedia tier gratis, dengan paket berbayar untuk pustaka yang lebih besar.

Hubungkan sekali, dan musik Anda mengalir lewat tunnel terenkripsi — Evermusic tidak pernah melihat data Anda dalam bentuk teks polos, begitu pula server penyedia.

### Penyimpanan host sendiri: QNAP, Nextcloud, Amazon S3

Untuk pengguna yang menjalankan infrastruktur sendiri:

- **QNAP** — koneksi API native ke perangkat QNAP NAS untuk pemutaran cepat dan andal lewat Wi-Fi lokal atau akses jarak jauh.
- **Nextcloud** — terhubung ke instance Nextcloud apa pun, baik host sendiri maupun terkelola. Cocok jika Anda sudah pindah dari Google Drive atau Dropbox karena alasan privasi.
- **Amazon S3** — arahkan Evermusic ke bucket S3 mana pun (atau penyimpanan kompatibel S3 seperti Backblaze B2, Wasabi, MinIO, Cloudflare R2) dan streaming koleksi Anda secara langsung.

### <a id="media-servers"></a>Server media: Plex, Subsonic, Navidrome, Jellyfin, Emby

Inilah penambahan terbesar bagi penggemar musik host sendiri. Evermusic 8.6 menjadikan iPhone, iPad, atau Mac Anda klien kelas satu bagi server media open-source dan freemium yang paling populer:

- **Plex** — Plex Media Server **gratis** untuk diunduh dan dijalankan. Langganan **Plex Pass** opsional dan membuka fitur seperti sinkronisasi mobile, transcoding hardware, dan TV langsung. Evermusic bekerja dengan pustaka gratis maupun Plex Pass.
- **Subsonic** — server streaming musik host-sendiri orisinal. Server Subsonic resmi **berbayar** ($1/bulan setelah uji coba 30 hari), tetapi Evermusic juga berbicara API Subsonic dengan puluhan server kompatibel.
- **Navidrome** — server musik modern, ringan, **sepenuhnya gratis dan open-source**, ditulis dalam Go. Mengimplementasikan API Subsonic. Berjalan di Raspberry Pi, NAS, atau mesin Linux apa pun. Sangat direkomendasikan untuk pustaka hingga beberapa ratus ribu trek.
- **Jellyfin** — server media **sepenuhnya gratis dan open-source** (fork komunitas dari Emby). Menangani musik, film, TV, buku, dan buku audio. Tanpa akun, tanpa telemetri, tanpa langganan.
- **Emby** — server media **freemium**. Server inti gratis; **Emby Premiere** adalah pembelian sekali atau tahunan yang membuka aplikasi mobile, sinkronisasi offline, mode Cinema, dan lainnya. Evermusic terhubung ke pustaka gratis maupun Premiere.

Server apa pun yang Anda jalankan, Evermusic streaming seluruh pustaka Anda — album, artis, daftar putar, genre, dan sampul tertanam — dengan cache offline, pemutaran gapless dan crossfade, equalizer 10 band, AirPlay, Chromecast, dan **CarPlay**. Server Anda menyimpan riwayat dengar; Evermusic menghormatinya.

### Protokol jaringan baru: FTP, SFTP, NFS

Untuk pengguna dengan server kustom, homelab, atau NAS generik yang tidak punya aplikasi mobile yang dipoles, Evermusic 8.6 menambahkan tiga protokol klasik:

- **SFTP (SSH File Transfer Protocol)** — jawaban yang tepat untuk **streaming jarak jauh yang aman dari server Anda sendiri**. SFTP berjalan di atas SSH, sehingga seluruh transfer (autentikasi dan data audio) dienkripsi. Jika Anda punya VPS, server dedicated, atau mesin Linux di rumah dengan akses SSH, taruh saja folder musik di sana dan streaming melalui internet publik tanpa membuka apa pun yang lain. Mendukung autentikasi password maupun kunci.
- **FTP** — standar lama untuk transfer berkas. Berguna jika **NAS rumah** Anda (Synology lawas, ASUS, D-Link, TerraMaster, atau kotak generik) menyediakan server FTP namun tanpa integrasi API native. Paling baik digunakan di jaringan lokal.
- **NFS (Network File System)** — protokol berbagi de facto di Linux dan kebanyakan NAS. Share NFS umum di homelab dan jaringan bisnis kecil; Evermusic kini bisa me-mount dan streaming langsung. Beban lebih rendah dibandingkan SMB pada perangkat keras yang sama.

> **Tip:** SFTP adalah protokol yang Anda inginkan untuk streaming melalui internet terbuka. FTP dan NFS paling baik di jaringan lokal — jangan ekspos ke internet kecuali dibungkus VPN.

## Wi-Fi Drive: UI baru dan unggahan lebih cepat

[**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/) adalah fitur bawaan Evermusic untuk **memindahkan musik secara nirkabel dari komputer ke iPhone atau iPad — tanpa iTunes, tanpa kabel, tanpa akun cloud**. Kedua perangkat harus berada di jaringan Wi-Fi yang sama. Anda menyalakan server di aplikasi iOS, lalu:

- buka URL di browser desktop apa pun (Safari, Chrome, Firefox, Edge), seret berkas musik ke halaman, dan mereka langsung diunggah ke perangkat, atau
- pasang perangkat sebagai berbagi jaringan via **Mac Finder** ("Connect to Server…") atau **Windows File Explorer** (Map Network Drive) menggunakan WebDAV.

Ini adalah cara tercepat memindahkan pustaka musik lokal yang besar ke ponsel tanpa layanan pihak ketiga. Lihat [panduan langkah demi langkah di sini](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/).

Pada Evermusic 8.6, Wi-Fi Drive mendapatkan:

- **Antarmuka baru** — lebih bersih, lebih mudah dibaca sekilas.
- **Mode pilih** — pilih beberapa berkas atau folder dan tindak secara massal.
- **Antrean unggah lebih baik** — feedback progres lebih jelas dan tahan terhadap gangguan jaringan.
- **Performa keseluruhan lebih baik** — transfer lebih cepat untuk pustaka besar.

## Widget lirik tersinkron baru di Layar Beranda

Salah satu fitur yang paling banyak diminta akhirnya hadir: **widget lirik tersinkron yang bisa Anda tempelkan di Layar Beranda iPhone, iPad, atau Mac**.

Saat Anda memutar trek dengan lirik selaras waktu, widget menampilkan **baris saat ini** dari lagu secara real time — dan barisnya berlanjut seiring musik. Ketuk widget untuk membuka Evermusic dalam tampilan lirik penuh.

Apa yang dilakukannya:

- Menampilkan **lirik tersinkron, bertanda waktu** untuk trek yang sedang diputar.
- Memperbarui sesuai progres lagu — tanpa refresh manual, tanpa scroll.
- Berfungsi dengan lirik LRC tertanam di berkas audio Anda dan dengan lirik yang diambil di aplikasi.
- Tersedia dalam **beberapa ukuran widget** sehingga Anda memilih banyaknya teks yang muat di Layar Beranda.
- Berjalan di **iPhone**, **iPad**, dan **Mac** (Layar Beranda dan Pusat Notifikasi di tempat yang didukung).

Widget ini sempurna bagi pengguna yang suka mengikuti lirik saat ponsel di meja, di stand, atau di layar kunci selama sesi mendengarkan yang panjang. Sangat cocok dengan buku audio dan konten gaya podcast di mana membaca membantu pemahaman.

Untuk mengaktifkan, tahan Layar Beranda, ketuk **Edit > Tambah Widget**, cari **Evermusic**, dan pilih widget **Lirik** dalam ukuran yang Anda inginkan.

Antarmuka Evermusic 8.6 diperbarui untuk material **Liquid Glass** baru dari Apple di seluruh aplikasi — permukaan tembus cahaya, animasi yang lebih halus, dan kontrol yang dipoles agar pas dengan iOS, iPadOS, dan macOS.

Pada rilis ini kami memberi perhatian besar pada stabilitas pemutaran untuk **Baidu Netdisk (百度网盘)** dan **Aliyun Drive (阿里云盘)**. Jika Anda pernah menggunakan layanan ini dengan Evermusic, Anda akan menyadari:

- Listing direktori lebih cepat pada pustaka berisi ribuan berkas.
- Streaming lebih andal pada koneksi lambat atau jauh secara geografis.
- Lebih sedikit putus koneksi selama sesi pemutaran panjang.
- Logika percobaan ulang dan melanjutkan yang lebih baik saat jaringan sempat terputus.

Kedua cloud ini adalah layanan penyimpanan konsumen paling populer di Tiongkok, dan Evermusic adalah salah satu dari sedikit pemutar musik iOS dengan dukungan native kelas satu untuk keduanya.

### Pustaka koneksi diperbarui

Kami memperbarui pustaka internal yang digunakan Evermusic untuk berbicara dengan **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive**, dan layanan lain. Selain widget lirik baru, widget Layar Beranda yang sudah ada — Sedang Diputar, Antrean Putar, Baru Diputar — telah dipoles dengan **logika pembaruan yang lebih baik**, agar tetap tersinkron tanpa membakar baterai tambahan.

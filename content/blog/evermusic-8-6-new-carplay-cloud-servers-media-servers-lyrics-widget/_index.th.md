**สรุป:** [Evermusic 8.6](/products/evermusic) เป็นการอัปเดตครั้งใหญ่สำหรับ iPhone, iPad และ Mac เราสร้าง CarPlay ขึ้นใหม่ตั้งแต่ต้น พร้อมการเรียงลำดับด่วน, ชุดสีหลายแบบ, หน้าจอกำลังเล่นที่ออกแบบใหม่, มุมมองคิวการเล่นเต็มรูปแบบ และดัชนีตัวอักษรที่เลื่อนได้รวดเร็ว เวอร์ชันนี้เพิ่มการเชื่อมต่อใหม่กว่า 10 รายการ — **Plex**, **Jellyfin**, **Emby**, **Subsonic**, **Navidrome**, **Internxt**, **Proton Drive**, **QNAP**, **Nextcloud**, **Amazon S3** รวมถึงโปรโตคอล **FTP**, **SFTP** และ **NFS** วิดเจ็ตเนื้อเพลงแบบซิงค์บนหน้าจอโฮมจะแสดงเนื้อเพลงที่ตรงกับเวลาขณะคุณกำลังฟัง Wi-Fi Drive ได้รับ UI ใหม่, โหมดเลือกหลายรายการ และคิวอัปโหลดที่เร็วขึ้น ทั้งแอปได้รับการอัปเดตให้ใช้ดีไซน์ **Liquid Glass** และการสตรีมจากเซิร์ฟเวอร์จีนอย่าง **Baidu Netdisk (百度网盘)** และ **Aliyun Drive (阿里云盘)** มีความเสถียรขึ้น

## ประสบการณ์ CarPlay ใหม่ทั้งหมด

เราสร้าง Evermusic สำหรับ **Apple CarPlay** ขึ้นใหม่จากศูนย์ ผลลัพธ์คือเร็วขึ้น, ลื่นขึ้น และใช้งานง่ายขึ้นมากขณะขับรถ

- **เรียงลำดับอย่างรวดเร็ว** — ค้นหาแทร็กใดก็ได้ในไม่กี่วินาที ทั้งในอัลบั้ม, ศิลปิน, เพลย์ลิสต์ และโฟลเดอร์
- **ชุดสีหลายแบบ** — เลือกธีมที่เข้ากับแดชบอร์ดหรือไฟภายในรถของคุณ
- **หน้าจอกำลังเล่นที่ออกแบบใหม่** — ภาพปกใหญ่ขึ้น, ปุ่มควบคุมชัดเจนขึ้น และการกระทำการเล่นใหม่ ๆ ที่กดได้ในแตะเดียว
- **คิวการเล่นแบบเต็มในแวบเดียว** — ดูว่าจะมีอะไรเล่นต่อโดยไม่ต้องออกจากหน้าจอปัจจุบัน
- **ดัชนีตัวอักษรเลื่อนเร็ว** — กระโดดผ่านไลบรารีใหญ่ (10,000+ แทร็ก) โดยไม่ต้องเลื่อนไม่จบ
- **โหลดเนื้อหาลื่นขึ้น** — ไม่มีอาการค้างเมื่อเปิดโฟลเดอร์ใหญ่, ไดเรกทอรีคลาวด์ หรือไลบรารีเซิร์ฟเวอร์มีเดียอีกต่อไป

ถ้าคุณสตรีมเพลงจากคลาวด์ขณะขับรถ — Dropbox, Google Drive, OneDrive, iCloud Drive, [Plex](#media-servers), [Jellyfin](#media-servers) และอื่น ๆ — ประสบการณ์ CarPlay ใหม่จะทำให้ทั้งกระบวนการรู้สึกเหมือนเป็นส่วนหนึ่งของรถมาแต่แรก

## การเชื่อมต่อใหม่กับคลาวด์, NAS และเซิร์ฟเวอร์มากกว่า 10 รายการ

Evermusic 8.6 ขยายความหมายของ "ไลบรารีเพลง" ของคุณ ไม่ว่าไฟล์เสียงจะอยู่ในคลาวด์ที่คุณไว้ใจ, NAS ที่บ้าน หรือเซิร์ฟเวอร์มีเดียที่โฮสต์เอง — ตอนนี้ Evermusic สตรีมจากที่เหล่านั้นได้โดยตรง โดยไม่ต้องซิงค์หรือส่งออก

### คลาวด์ที่ให้ความสำคัญกับความเป็นส่วนตัว: Internxt และ Proton Drive

ถ้าคุณให้ความสำคัญกับการเข้ารหัสแบบ end-to-end และพื้นที่จัดเก็บแบบ zero-knowledge สองชื่อที่ได้รับความเคารพที่สุดในวงการคลาวด์เพื่อความเป็นส่วนตัวก็ใช้งานแบบเนทีฟใน Evermusic ได้แล้ว:

- **Internxt** — คลาวด์จากสเปน โอเพนซอร์ส มีการเข้ารหัสแบบ post-quantum และเป็นไปตาม GDPR มีแพลนฟรี
- **Proton Drive** — พื้นที่จัดเก็บแบบเข้ารหัสจากต้นทางถึงปลายทางจากผู้สร้าง Proton Mail และ Proton VPN ตั้งฐานในสวิตเซอร์แลนด์ มีระดับฟรีและแพลนเสียเงินสำหรับไลบรารีขนาดใหญ่

เชื่อมต่อครั้งเดียวแล้วเพลงของคุณจะวิ่งผ่านอุโมงค์ที่เข้ารหัส Evermusic ไม่เคยเห็นข้อมูลของคุณในรูปแบบโล่ง ๆ และเซิร์ฟเวอร์ของผู้ให้บริการก็เช่นกัน

### พื้นที่จัดเก็บแบบโฮสต์เอง: QNAP, Nextcloud, Amazon S3

สำหรับผู้ใช้ที่ดูแลโครงสร้างพื้นฐานของตัวเอง:

- **QNAP** — เชื่อมต่อ API เนทีฟกับอุปกรณ์ QNAP NAS เพื่อเล่นได้รวดเร็วและเสถียรผ่าน Wi-Fi ในบ้านหรือเข้าถึงระยะไกล
- **Nextcloud** — เชื่อมต่อกับอินสแตนซ์ Nextcloud ใด ๆ ก็ได้ ทั้งโฮสต์เองและจัดการให้ เหมาะมากถ้าคุณย้ายออกจาก Google Drive หรือ Dropbox เพื่อความเป็นส่วนตัวอยู่แล้ว
- **Amazon S3** — ชี้ Evermusic ไปยัง S3 bucket ใด ๆ (หรือสตอเรจที่เข้ากับ S3 ได้อย่าง Backblaze B2, Wasabi, MinIO, Cloudflare R2) แล้วสตรีมคอลเลกชันของคุณตรง ๆ

### <a id="media-servers"></a>เซิร์ฟเวอร์มีเดีย: Plex, Subsonic, Navidrome, Jellyfin, Emby

นี่คือไฮไลต์ใหญ่สำหรับคนที่ชอบโฮสต์เพลงเอง Evermusic 8.6 ทำให้ iPhone, iPad หรือ Mac ของคุณเป็นไคลเอนต์ระดับชั้นนำสำหรับเซิร์ฟเวอร์มีเดียโอเพนซอร์สและฟรีเมียมยอดนิยม:

- **Plex** — Plex Media Server ดาวน์โหลดและเรียกใช้ **ฟรี** การสมัคร **Plex Pass** เป็นทางเลือก ปลดล็อกฟีเจอร์อย่างซิงค์มือถือ, ทรานส์โค้ดด้วยฮาร์ดแวร์ และทีวีสด Evermusic ทำงานทั้งกับไลบรารีฟรีและ Plex Pass
- **Subsonic** — เซิร์ฟเวอร์สตรีมเพลงโฮสต์เองต้นแบบ เซิร์ฟเวอร์ Subsonic อย่างเป็นทางการ **เสียเงิน** (1 ดอลลาร์/เดือนหลังทดลอง 30 วัน) แต่ Evermusic ก็พูด API ของ Subsonic กับเซิร์ฟเวอร์ที่เข้ากันได้นับสิบรายการ
- **Navidrome** — เซิร์ฟเวอร์เพลงสมัยใหม่ น้ำหนักเบา **ฟรีโดยสมบูรณ์และโอเพนซอร์ส** เขียนด้วย Go รองรับ API ของ Subsonic ทำงานบน Raspberry Pi, NAS หรือเครื่อง Linux ใด ๆ ก็ได้ แนะนำอย่างยิ่งสำหรับไลบรารีไม่กี่แสนแทร็ก
- **Jellyfin** — เซิร์ฟเวอร์มีเดีย **ฟรีโดยสมบูรณ์และโอเพนซอร์ส** (ฟอร์กของ Emby โดยชุมชน) จัดการเพลง, หนัง, ทีวี, หนังสือ และหนังสือเสียงได้ ไม่มีบัญชี, ไม่มี telemetry, ไม่มีการสมัครสมาชิก
- **Emby** — เซิร์ฟเวอร์มีเดียแบบ **ฟรีเมียม** เซิร์ฟเวอร์หลักฟรี **Emby Premiere** เป็นการซื้อครั้งเดียวหรือรายปี ปลดล็อกแอปมือถือ, ซิงค์ออฟไลน์, โหมด Cinema และอื่น ๆ Evermusic ใช้งานได้ทั้งไลบรารีฟรีและ Premiere

ไม่ว่าคุณจะใช้เซิร์ฟเวอร์ใด Evermusic จะสตรีมไลบรารีทั้งหมดของคุณ — อัลบั้ม, ศิลปิน, เพลย์ลิสต์, แนวเพลง และภาพปกที่ฝังในไฟล์ — พร้อมแคชออฟไลน์, การเล่นแบบ gapless และ crossfade, อีควอไลเซอร์ 10 แบนด์, AirPlay, Chromecast และ **CarPlay** เซิร์ฟเวอร์เก็บประวัติการฟัง; Evermusic เคารพข้อมูลนั้น

### โปรโตคอลเครือข่ายใหม่: FTP, SFTP, NFS

สำหรับผู้ใช้ที่มีเซิร์ฟเวอร์ที่ปรับเอง, โฮมแล็บ หรือ NAS ทั่วไปที่ไม่มีแอปมือถือคุณภาพดี Evermusic 8.6 เพิ่มโปรโตคอลคลาสสิก 3 ตัว:

- **SFTP (SSH File Transfer Protocol)** — คำตอบที่ใช่สำหรับ **การสตรีมระยะไกลอย่างปลอดภัยจากเซิร์ฟเวอร์ของคุณเอง** SFTP ทำงานบน SSH ดังนั้นการถ่ายโอนทั้งหมด (ทั้งการยืนยันตัวและข้อมูลเสียง) ถูกเข้ารหัส ถ้าคุณมี VPS, เซิร์ฟเวอร์เฉพาะ หรือเครื่อง Linux ที่บ้านพร้อม SSH ก็วางโฟลเดอร์เพลงไว้แล้วสตรีมผ่านอินเทอร์เน็ตสาธารณะได้โดยไม่ต้องเปิดเผยอย่างอื่น รองรับการยืนยันตัวด้วยรหัสผ่านและคีย์
- **FTP** — มาตรฐานการถ่ายโอนไฟล์ที่ใช้งานมายาวนาน เหมาะถ้า **NAS ที่บ้าน** ของคุณ (Synology รุ่นเก่า, ASUS, D-Link, TerraMaster หรือกล่องทั่วไป) มีบริการ FTP แต่ไม่มีการเชื่อม API เนทีฟ ใช้ภายในเครือข่ายในบ้านดีที่สุด
- **NFS (Network File System)** — โปรโตคอลแชร์มาตรฐานโดยพฤตินัยของ Linux และ NAS ส่วนใหญ่ การแชร์ NFS เป็นเรื่องปกติในโฮมแล็บและเครือข่ายธุรกิจขนาดเล็ก ตอนนี้ Evermusic เมานต์และสตรีมตรง ๆ ได้ มีโอเวอร์เฮดน้อยกว่า SMB บนฮาร์ดแวร์เดียวกัน

> **เคล็ดลับ:** SFTP คือโปรโตคอลที่คุณต้องการสำหรับการสตรีมผ่านอินเทอร์เน็ตเปิด FTP และ NFS ใช้ในเครือข่ายภายในจะดีกว่า — อย่าเปิดให้อินเทอร์เน็ตสาธารณะใช้ตรง ๆ ยกเว้นจะหุ้มด้วย VPN

## Wi-Fi Drive: UI ใหม่และอัปโหลดเร็วขึ้น

[**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/) คือฟีเจอร์ในตัวของ Evermusic สำหรับ **โอนเพลงแบบไร้สายจากคอมพิวเตอร์ไปยัง iPhone หรือ iPad — ไม่ต้องใช้ iTunes, ไม่ต้องใช้สาย, ไม่ต้องมีบัญชีคลาวด์** ทั้งสองอุปกรณ์ต้องอยู่บนเครือข่าย Wi-Fi เดียวกัน คุณเริ่มเซิร์ฟเวอร์ในแอป iOS แล้วทำได้สองวิธี:

- เปิด URL ในเบราว์เซอร์เดสก์ท็อปใด ๆ (Safari, Chrome, Firefox, Edge), ลากไฟล์เพลงไปวางบนหน้าเว็บ ไฟล์จะถูกอัปโหลดไปยังอุปกรณ์โดยตรง หรือ
- เมานต์อุปกรณ์เป็นการแชร์เครือข่ายผ่าน **Mac Finder** ("Connect to Server…") หรือ **Windows File Explorer** (Map Network Drive) ด้วย WebDAV

นี่คือวิธีที่เร็วที่สุดในการย้ายไลบรารีเพลงในเครื่องขนาดใหญ่ลงโทรศัพท์โดยไม่ต้องพึ่งบริการบุคคลที่สาม ดู [คู่มือทีละขั้นตอนที่นี่](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/)

ใน Evermusic 8.6, Wi-Fi Drive ได้รับ:

- **อินเทอร์เฟซใหม่** — สะอาดตา อ่านเข้าใจง่ายในแวบเดียว
- **โหมดเลือก** — เลือกหลายไฟล์หรือหลายโฟลเดอร์แล้วทำงานเป็นชุด
- **คิวอัปโหลดที่ดีขึ้น** — แสดงความคืบหน้าได้ชัดเจนขึ้น และทนทานต่อปัญหาเครือข่ายมากขึ้น
- **ประสิทธิภาพรวมดีขึ้น** — โอนไลบรารีขนาดใหญ่ได้เร็วขึ้น

## วิดเจ็ตเนื้อเพลงแบบซิงค์ใหม่บนหน้าจอโฮม

หนึ่งในฟีเจอร์ที่ถูกเรียกร้องที่สุดมาถึงแล้ว: **วิดเจ็ตเนื้อเพลงแบบซิงค์ที่คุณตรึงไว้บนหน้าจอโฮมของ iPhone, iPad หรือ Mac ได้**

เมื่อคุณเล่นแทร็กที่มีเนื้อเพลงตรงตามเวลา วิดเจ็ตจะแสดง **บรรทัดปัจจุบัน** ของเพลงแบบเรียลไทม์ — และบรรทัดจะเลื่อนตามเสียงเพลง แตะวิดเจ็ตเพื่อเปิด Evermusic ในมุมมองเนื้อเพลงเต็ม

สิ่งที่วิดเจ็ตทำได้:

- แสดง **เนื้อเพลงพร้อมไทม์สแตมป์** ของแทร็กที่กำลังเล่น
- อัปเดตตามความคืบหน้าของเพลง — ไม่ต้องรีเฟรชเอง ไม่ต้องเลื่อน
- ใช้งานได้กับเนื้อเพลงแบบ LRC ที่ฝังในไฟล์เสียงและเนื้อเพลงที่ดึงในแอป
- มี **ขนาดวิดเจ็ตหลายขนาด** ให้เลือกว่าจะให้แสดงข้อความเท่าไรบนหน้าจอโฮม
- ใช้ได้บน **iPhone**, **iPad** และ **Mac** (หน้าจอโฮมและศูนย์การแจ้งเตือนในที่ที่รองรับ)

วิดเจ็ตเหมาะกับผู้ใช้ที่ชอบอ่านเนื้อเพลงตามไปขณะที่โทรศัพท์วางอยู่บนโต๊ะ, ในขาตั้ง หรือบนหน้าจอล็อกระหว่างฟังนาน ๆ และเข้ากันได้ดีเป็นพิเศษกับหนังสือเสียงและคอนเทนต์แบบพอดแคสต์ ที่การอ่านช่วยให้เข้าใจเนื้อหามากขึ้น

ในการเปิดใช้งาน กดค้างที่หน้าจอโฮม แตะ **Edit > Add Widget** ค้นหา **Evermusic** แล้วเลือกวิดเจ็ต **Lyrics** ขนาดที่คุณต้องการ

## การปรับปรุงอื่น ๆ

### อัปเดตดีไซน์ Liquid Glass

อินเทอร์เฟซของ Evermusic 8.6 ได้รับการอัปเดตให้สอดคล้องกับวัสดุ **Liquid Glass** ใหม่ของ Apple ทั่วทั้งแอป — พื้นผิวโปร่งแสง, แอนิเมชันลื่นไหลขึ้น และปุ่มควบคุมที่ปรับให้กลมกลืนกับ iOS, iPadOS และ macOS อย่างเป็นธรรมชาติ

### ความเสถียรกับเซิร์ฟเวอร์จีนดีขึ้น

ในเวอร์ชันนี้เราทุ่มเทกับเสถียรภาพของการเล่นจาก **Baidu Netdisk (百度网盘)** และ **Aliyun Drive (阿里云盘)** ถ้าคุณเคยใช้บริการเหล่านี้กับ Evermusic มาก่อน คุณจะสังเกตเห็น:

- การแสดงรายการไดเรกทอรีเร็วขึ้นในไลบรารีที่มีไฟล์หลายพันไฟล์
- การสตรีมเสถียรขึ้นบนการเชื่อมต่อช้าหรืออยู่ห่างไกล
- การหลุดของการเชื่อมต่อระหว่างเซสชันการเล่นยาว ๆ น้อยลง
- ลอจิกการลองใหม่และทำต่อปรับปรุงขึ้นเมื่อเครือข่ายดร็อปสั้น ๆ

คลาวด์ทั้งสองนี้เป็นบริการจัดเก็บข้อมูลผู้บริโภคยอดนิยมในจีน และ Evermusic เป็นหนึ่งในเครื่องเล่นเพลงบน iOS ไม่กี่ตัวที่รองรับทั้งสองตัวแบบเนทีฟระดับชั้นนำ

### ไลบรารีการเชื่อมต่ออัปเดต

เราอัปเดตไลบรารีภายในที่ Evermusic ใช้คุยกับ **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive** และบริการอื่น ๆ พูดอีกอย่างก็คือ: ความล้มเหลวของการเชื่อมต่อในกรณีพิเศษน้อยลง, รองรับเซิร์ฟเวอร์รุ่นใหม่ขึ้นได้ดีขึ้น และประสิทธิภาพโดยรวมดีขึ้น

### วิดเจ็ตหน้าจอโฮมใหม่และการอัปเดตที่ฉลาดขึ้น

นอกจากวิดเจ็ตเนื้อเพลงใหม่แล้ว วิดเจ็ตหน้าจอโฮมที่มีอยู่ — กำลังเล่น, คิวการเล่น, เล่นล่าสุด — ได้รับการขัดเกลาให้มี **ลอจิกการอัปเดตที่ดีขึ้น** เพื่อให้ซิงค์อยู่ตลอดโดยไม่กินแบตเตอรี่เพิ่ม นอกจากนี้ยังมีเลย์เอาต์วิดเจ็ตใหม่สำหรับผู้ใช้ที่ต้องการควบคุมสิ่งที่เห็นได้ในแวบเดียวมากขึ้น

### แก้ไขจุดบกพร่องและขัดเกลา

- แก้ปัญหาการเล่นบนเซิร์ฟเวอร์มีเดียบางตัวและการตั้งค่าโฮสต์เอง
- แก้คำแปลในหลายภาษา
- การปรับปรุงเล็ก ๆ จำนวนมากตามฟีดแบ็คตรงจากรีวิวใน App Store และอีเมลถึง support@everappz.com

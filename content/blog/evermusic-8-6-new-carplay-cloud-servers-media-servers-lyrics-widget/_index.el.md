Το CarPlay έχει χτιστεί από την αρχή — γρήγορη ταξινόμηση, πολλαπλά χρωματικά σχήματα, ανασχεδιασμένη οθόνη Παίζει Τώρα, πλήρης προβολή ουράς αναπαραγωγής και γρήγορος αλφαβητικός δείκτης κύλισης. ## Μια εντελώς νέα εμπειρία CarPlay

Φτιάξαμε ξανά το Evermusic για το **Apple CarPlay** από το μηδέν. Το αποτέλεσμα είναι ταχύτερο, πιο ομαλό και πολύ πιο εύκολο στη χρήση πίσω από το τιμόνι.

- **Γρήγορη ταξινόμηση** — βρείτε οποιοδήποτε κομμάτι σε δευτερόλεπτα μέσα σε άλμπουμ, καλλιτέχνες, λίστες αναπαραγωγής και φακέλους.
- **Πολλαπλά χρωματικά σχήματα** — επιλέξτε ένα θέμα που ταιριάζει στο ταμπλό ή στον εσωτερικό φωτισμό του αυτοκινήτου.
- **Ανασχεδιασμένη οθόνη Παίζει Τώρα** — μεγαλύτερο εξώφυλλο, σαφέστερα στοιχεία ελέγχου και νέες ενέργειες αναπαραγωγής με ένα μόνο άγγιγμα.
- **Πλήρης ουρά αναπαραγωγής με μια ματιά** — δείτε τι ακολουθεί χωρίς να βγείτε από την τρέχουσα οθόνη.
- **Γρήγορος αλφαβητικός δείκτης κύλισης** — προσπεράστε σε τεράστιες βιβλιοθήκες (10.000+ κομμάτια) χωρίς ατέρμονη κύλιση.
- **Ομαλότερη φόρτωση περιεχομένου** — τέλος στα παγώματα όταν ανοίγετε μεγάλους φακέλους, καταλόγους cloud ή βιβλιοθήκες media servers.

Αν κάνετε streaming μουσικής από το cloud ενώ οδηγείτε — Dropbox, Google Drive, OneDrive, iCloud Drive, [Plex](#media-servers), [Jellyfin](#media-servers) και άλλα — η νέα εμπειρία CarPlay κάνει ολόκληρη τη ροή να μοιάζει σαν ενσωματωμένη στο αυτοκίνητο.

## Πάνω από 10 νέες ενσωματώσεις cloud, NAS και διακομιστών

Το Evermusic 8.6 διευρύνει τι μετράει ως «μουσική βιβλιοθήκη» σας. Είτε τα αρχεία σας βρίσκονται σε ένα cloud που εμπιστεύεστε, σε ένα NAS στο σπίτι ή σε έναν media server που φιλοξενείτε μόνοι σας — το Evermusic μπορεί τώρα να αναπαράγει απευθείας από εκεί, χωρίς συγχρονισμό ή εξαγωγή.

### Cloud εστιασμένα στην ιδιωτικότητα: Internxt και Proton Drive

Αν σας ενδιαφέρει η κρυπτογράφηση από άκρο σε άκρο και η αποθήκευση μηδενικής γνώσης, δύο από τα πιο σεβαστά ονόματα στα cloud με προτεραιότητα στην ιδιωτικότητα είναι τώρα native στο Evermusic:

- **Internxt** — ισπανικό cloud ανοιχτού κώδικα, με μετα-κβαντική κρυπτογράφηση και συμμόρφωση GDPR. Διαθέσιμο δωρεάν επίπεδο.
- **Proton Drive** — αποθήκευση με κρυπτογράφηση από άκρο σε άκρο από τους δημιουργούς του Proton Mail και του Proton VPN, με έδρα την Ελβετία. Διαθέσιμο δωρεάν επίπεδο και επί πληρωμή πλάνα για μεγαλύτερες βιβλιοθήκες.

Συνδεθείτε μία φορά και η μουσική σας ταξιδεύει μέσα από το κρυπτογραφημένο τούνελ — το Evermusic ποτέ δεν βλέπει τα δεδομένα σας σε καθαρό κείμενο, ούτε και ο διακομιστής του παρόχου.

### Αυτοφιλοξενούμενη αποθήκευση: QNAP, Nextcloud, Amazon S3

Για χρήστες που τρέχουν τη δική τους υποδομή:

- **QNAP** — native σύνδεση API σε συσκευές QNAP NAS για γρήγορη και αξιόπιστη αναπαραγωγή μέσω τοπικού Wi-Fi ή απομακρυσμένης πρόσβασης.
- **Nextcloud** — συνδεθείτε σε οποιαδήποτε εγκατάσταση Nextcloud, αυτοφιλοξενούμενη ή διαχειριζόμενη. Ιδανικό αν έχετε ήδη μεταφερθεί από Google Drive ή Dropbox για λόγους ιδιωτικότητας.
- **Amazon S3** — στρέψτε το Evermusic σε οποιοδήποτε S3 bucket (ή σε αποθήκευση συμβατή με S3 όπως Backblaze B2, Wasabi, MinIO, Cloudflare R2) και κάντε streaming της συλλογής σας απευθείας.

### <a id="media-servers"></a>Media servers: Plex, Subsonic, Navidrome, Jellyfin, Emby

Αυτή είναι η μεγάλη προσθήκη για τους λάτρεις της αυτοφιλοξενούμενης μουσικής. Το Evermusic 8.6 κάνει το iPhone, iPad ή Mac σας πρώτης κατηγορίας πελάτη για τους πιο δημοφιλείς ανοιχτού κώδικα και freemium media servers:

- **Plex** — ο Plex Media Server είναι **δωρεάν** για λήψη και εκτέλεση. Η συνδρομή **Plex Pass** είναι προαιρετική και ξεκλειδώνει λειτουργίες όπως κινητός συγχρονισμός, transcoding υλικού και ζωντανή τηλεόραση. Το Evermusic λειτουργεί με δωρεάν βιβλιοθήκες και βιβλιοθήκες Plex Pass.
- **Subsonic** — ο πρωτότυπος αυτοφιλοξενούμενος server μουσικής. Ο επίσημος server Subsonic είναι **επί πληρωμή** (1 $/μήνα μετά από δοκιμή 30 ημερών), αλλά το Evermusic μιλά το API Subsonic με δεκάδες συμβατούς διακομιστές.
- **Navidrome** — μοντέρνος, ελαφρύς server μουσικής, **εντελώς δωρεάν και ανοιχτού κώδικα**, γραμμένος σε Go. Υλοποιεί το API Subsonic. Τρέχει σε Raspberry Pi, NAS ή σε οποιαδήποτε μηχανή Linux. Πολύ προτεινόμενος για βιβλιοθήκες έως μερικές εκατοντάδες χιλιάδων κομματιών.
- **Jellyfin** — **εντελώς δωρεάν και ανοιχτού κώδικα** media server (κοινοτικό fork του Emby). Διαχειρίζεται μουσική, ταινίες, τηλεόραση, βιβλία και ηχητικά βιβλία. Χωρίς λογαριασμούς, χωρίς τηλεμετρία, χωρίς συνδρομές.
- **Emby** — **freemium** media server. Ο βασικός server είναι δωρεάν· το **Emby Premiere** είναι εφάπαξ ή ετήσια αγορά που ξεκλειδώνει εφαρμογές κινητού, offline συγχρονισμό, λειτουργία Cinema και άλλα. Το Evermusic συνδέεται και σε δωρεάν και σε Premiere βιβλιοθήκες.

Όποιον server κι αν τρέχετε, το Evermusic στριμάρει όλη τη βιβλιοθήκη σας — άλμπουμ, καλλιτέχνες, λίστες, είδη και ενσωματωμένα εξώφυλλα — με offline cache, αναπαραγωγή gapless και crossfade, ισοσταθμιστή 10 ζωνών, AirPlay, Chromecast και **CarPlay**. Ο server σας κρατά το ιστορικό ακρόασης· το Evermusic το σέβεται.

### Νέα πρωτόκολλα δικτύου: FTP, SFTP, NFS

Για χρήστες με προσαρμοσμένους διακομιστές, homelabs ή γενικά NAS που δεν έρχονται με γυαλισμένη εφαρμογή κινητού, το Evermusic 8.6 προσθέτει τρία κλασικά πρωτόκολλα:

- **SFTP (SSH File Transfer Protocol)** — η σωστή απάντηση για **ασφαλές απομακρυσμένο streaming από τον δικό σας server**. Το SFTP τρέχει πάνω από SSH, οπότε ολόκληρη η μεταφορά (πιστοποίηση και δεδομένα ήχου) είναι κρυπτογραφημένη. Αν έχετε VPS, αφιερωμένο server ή μηχανή Linux στο σπίτι με πρόσβαση SSH, μπορείτε να βάλετε εκεί έναν φάκελο μουσικής και να κάνετε streaming μέσω του δημόσιου διαδικτύου χωρίς να εκθέσετε τίποτα άλλο. Υποστηρίζει πιστοποίηση με κωδικό και με κλειδί.
- **FTP** — το μακροχρόνιο πρότυπο για μεταφορά αρχείων. Χρήσιμο αν το **οικιακό σας NAS** (παλαιότερο Synology, ASUS, D-Link, TerraMaster ή γενικά κουτιά) εκθέτει έναν server FTP χωρίς όμως native ενσωμάτωση API. Καλύτερα μέσα στο τοπικό δίκτυο.
- **NFS (Σύστημα Αρχείων Δικτύου)** — το de facto πρωτόκολλο διαμοιρασμού στο Linux και τα περισσότερα NAS. Τα κοινόχρηστα NFS είναι συνηθισμένα σε homelabs και μικρά εταιρικά δίκτυα· το Evermusic μπορεί τώρα να τα προσαρτά και να κάνει streaming απευθείας. Λιγότερη επιβάρυνση από το SMB στο ίδιο υλικό.

> **Συμβουλή:** Το SFTP είναι το πρωτόκολλο που θέλετε για streaming μέσω ανοιχτού διαδικτύου. Τα FTP και NFS λειτουργούν καλύτερα μέσα στο τοπικό δίκτυο — μην τα εκθέτετε στο διαδίκτυο εκτός αν τα τυλίξετε σε VPN.

## Wi-Fi Drive: νέα διεπαφή και ταχύτερες μεταφορτώσεις

Το [**Wi-Fi Drive**](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/) είναι η ενσωματωμένη λειτουργία του Evermusic για **ασύρματη μεταφορά μουσικής από τον υπολογιστή στο iPhone ή στο iPad — χωρίς iTunes, χωρίς καλώδια, χωρίς λογαριασμό cloud**. Και οι δύο συσκευές πρέπει να είναι στο ίδιο δίκτυο Wi-Fi. Ξεκινάτε τον server στην εφαρμογή iOS και στη συνέχεια:

- ανοίγετε το URL σε οποιονδήποτε browser υπολογιστή (Safari, Chrome, Firefox, Edge), σύρετε τα μουσικά αρχεία στη σελίδα και ανεβαίνουν απευθείας στη συσκευή, ή
- προσαρτάτε τη συσκευή ως κοινόχρηστο δίκτυο μέσω **Mac Finder** («Σύνδεση σε διακομιστή…») ή **Windows File Explorer** (Map Network Drive) χρησιμοποιώντας WebDAV.

Είναι ο πιο γρήγορος τρόπος να μετακινήσετε μια μεγάλη τοπική μουσική βιβλιοθήκη στο τηλέφωνο χωρίς υπηρεσίες τρίτων. Δείτε τον [αναλυτικό οδηγό εδώ](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/).

Στο Evermusic 8.6, το Wi-Fi Drive αποκτά:

- **Νέα διεπαφή χρήστη** — πιο καθαρή και πιο ευανάγνωστη με μια ματιά.
- **Λειτουργία επιλογής** — επιλέξτε πολλά αρχεία ή φακέλους και ενεργήστε σε αυτά μαζικά.
- **Βελτιωμένη ουρά μεταφορτώσεων** — καλύτερη ανατροφοδότηση προόδου και αντοχή σε διακοπές δικτύου.
- **Καλύτερη συνολική απόδοση** — ταχύτερες μεταφορές για μεγάλες βιβλιοθήκες.

## Νέο συγχρονισμένο widget στίχων στην Αρχική Οθόνη

Μία από τις πιο απαιτούμενες λειτουργίες είναι επιτέλους εδώ: **ένα συγχρονισμένο widget στίχων που μπορείτε να καρφιτσώσετε στην Αρχική Οθόνη του iPhone, iPad ή Mac**.

Όταν αναπαράγετε ένα κομμάτι με χρονικά ευθυγραμμισμένους στίχους, το widget δείχνει την **τρέχουσα γραμμή** του τραγουδιού σε πραγματικό χρόνο — και η γραμμή προχωράει με τη μουσική. Πατήστε το widget για να ανοίξετε το Evermusic σε προβολή πλήρων στίχων.

Τι κάνει:

- Δείχνει **συγχρονισμένους στίχους με χρονοσημάνσεις** για το κομμάτι που παίζει.
- Ενημερώνεται με την πρόοδο του τραγουδιού — χωρίς χειροκίνητη ανανέωση και χωρίς κύλιση.
- Λειτουργεί με ενσωματωμένους στίχους LRC στα αρχεία ήχου σας και με στίχους που λαμβάνονται μέσα στην εφαρμογή.
- Διαθέσιμο σε **πολλά μεγέθη widget** ώστε να επιλέξετε πόσο κείμενο χωράει στην Αρχική Οθόνη.
- Λειτουργεί σε **iPhone**, **iPad** και **Mac** (Αρχική Οθόνη και Κέντρο Ειδοποιήσεων όπου υποστηρίζεται).

Το widget είναι ιδανικό για χρήστες που τους αρέσει να ακολουθούν τους στίχους όσο το τηλέφωνο είναι στο γραφείο, σε βάση ή στην οθόνη κλειδώματος κατά τη διάρκεια μιας μεγάλης ακρόασης. Ταιριάζει ιδιαίτερα με ηχητικά βιβλία και περιεχόμενο τύπου podcast, όπου η ανάγνωση βοηθά την κατανόηση.

Για να το ενεργοποιήσετε, κρατήστε πατημένη την Αρχική Οθόνη, πατήστε **Επεξεργασία > Προσθήκη widget**, αναζητήστε **Evermusic** και επιλέξτε το widget **Στίχοι** στο επιθυμητό μέγεθος.

## Άλλες βελτιώσεις

### Ενημερώσεις σχεδίασης Liquid Glass

Η διεπαφή του Evermusic 8.6 ενημερώθηκε για το νέο υλικό **Liquid Glass** της Apple σε ολόκληρη την εφαρμογή — ημιδιαφανείς επιφάνειες, πιο ομαλές κινήσεις και πιο εκλεπτυσμένα στοιχεία ελέγχου που ταιριάζουν φυσικά σε iOS, iPadOS και macOS.

### Καλύτερη αξιοπιστία για κινεζικούς servers

Σε αυτή την έκδοση επενδύσαμε σημαντικά στη σταθερότητα αναπαραγωγής για **Baidu Netdisk (百度网盘)** και **Aliyun Drive (阿里云盘)**. Αν έχετε ήδη χρησιμοποιήσει αυτές τις υπηρεσίες με το Evermusic, θα παρατηρήσετε:

- Ταχύτερη λίστα καταλόγων σε βιβλιοθήκες με χιλιάδες αρχεία.
- Πιο αξιόπιστο streaming σε αργές ή γεωγραφικά μακρινές συνδέσεις.
- Λιγότερες αποσυνδέσεις κατά τη διάρκεια μεγάλων συνεδριών αναπαραγωγής.
- Βελτιωμένη λογική επανάληψης και επανάληψης από εκεί που είχε σταματήσει όταν το δίκτυο πέσει στιγμιαία.

Αυτά τα δύο cloud είναι οι πιο δημοφιλείς υπηρεσίες αποθήκευσης για καταναλωτές στην Κίνα, και το Evermusic είναι ένας από τους λίγους iOS music players με κορυφαία native υποστήριξη και για τα δύο.

### Ενημερωμένες βιβλιοθήκες σύνδεσης

Ενημερώσαμε τις εσωτερικές βιβλιοθήκες που χρησιμοποιεί το Evermusic για να επικοινωνεί με **WebDAV**, **SMB**, **DLNA**, **Dropbox**, **Google Drive**, **OneDrive** και άλλες υπηρεσίες. Σε λίγα λόγια: λιγότερες αποτυχίες σύνδεσης σε ακραίες περιπτώσεις, καλύτερη υποστήριξη πιο πρόσφατων εκδόσεων διακομιστών και καλύτερη απόδοση συνολικά.

### Νέα widgets Αρχικής Οθόνης και πιο έξυπνες ενημερώσεις

Πέρα από το νέο widget στίχων, τα υπάρχοντα widgets Αρχικής Οθόνης — Παίζει Τώρα, Ουρά Αναπαραγωγής, Πρόσφατα — έχουν εκλεπτυνθεί με **καλύτερη λογική ενημέρωσης**, ώστε να παραμένουν συγχρονισμένα χωρίς να καίνε επιπλέον μπαταρία. Υπάρχουν επίσης νέες διατάξεις widget για όσους θέλουν περισσότερο έλεγχο σε αυτό που φαίνεται με μια ματιά.

### Διορθώσεις σφαλμάτων και γυάλισμα

- Διορθώθηκαν προβλήματα αναπαραγωγής σε ορισμένους media servers και αυτοφιλοξενούμενες ρυθμίσεις.
- Διορθώσεις τοπικοποίησης σε πολλές γλώσσες.
- Πολλές μικρότερες βελτιώσεις βάσει άμεσων σχολίων από κριτικές στο App Store και emails στο support@everappz.com.

## Γιατί έχει σημασία αυτή η ενημέρωση

Το Evermusic 8.6 χτίζεται γύρω από τρεις ιδέες:

1. **Η μουσική σας, οπουδήποτε κι αν τη φυλάτε.** Είτε η βιβλιοθήκη σας είναι στο iCloud Drive, σε ένα cloud εστιασμένο στην ιδιωτικότητα όπως Proton Drive ή Internxt, σε έναν media server όπως Plex ή Jellyfin, σε NAS στο σπίτι ή σε Raspberry Pi με Navidrome — το Evermusic συνδέεται τώρα native με όλα, με την ίδια εμπειρία αναπαραγωγής παντού.
2. **Καλύτερο στο αυτοκίνητο.** Το CarPlay είναι η οθόνη που πολλοί χρήστες κοιτούν περισσότερο, και η ξανακτισμένη εμπειρία το αντικατοπτρίζει.
3. **Στίχοι στην Αρχική Οθόνη.** Το νέο συγχρονισμένο widget στίχων φέρνει στην κύρια επιφάνεια ματιάς σας κάτι που κανένας άλλος δεν κάνει πραγματικά καλά.

Αν σας αρέσει η εφαρμογή, αφήστε μια αξιολόγηση στο App Store — βοηθάει πραγματικά. Έχετε σχόλια ή συναντήσατε πρόβλημα; Στείλτε μας email στο **support@everappz.com**. Διαβάζουμε κάθε μήνυμα.

## Συχνές Ερωτήσεις

{{% details title="Τι νέο υπάρχει στο Evermusic 8.6;" closed="true" %}}
Το Evermusic 8.6 παρουσιάζει εμπειρία CarPlay πλήρως ανασχεδιασμένη, υποστήριξη για πάνω από 10 νέες συνδέσεις (Plex, Jellyfin, Emby, Subsonic, Navidrome, Internxt, Proton Drive, QNAP, Nextcloud, Amazon S3, FTP, SFTP, NFS), νέο συγχρονισμένο widget στίχων για την Αρχική Οθόνη, βελτιώσεις διεπαφής Wi-Fi Drive με λειτουργία επιλογής, ενημερώσεις σχεδίασης Liquid Glass, καλύτερη αξιοπιστία για Baidu Netdisk και Aliyun Drive, και πολλές διορθώσεις σφαλμάτων.
{{% /details %}}

{{% details title="Λειτουργεί το Evermusic με το Plex;" closed="true" %}}
Ναι. Από το Evermusic 8.6, μπορείτε να συνδεθείτε σε έναν Plex Media Server και να κάνετε streaming όλη τη μουσική σας βιβλιοθήκη. Ο Plex Media Server είναι δωρεάν στη χρήση· το Plex Pass είναι προαιρετικό. Το Evermusic υποστηρίζει και δωρεάν ρυθμίσεις και Plex Pass.
{{% /details %}}

{{% details title="Υποστηρίζονται Jellyfin ή Navidrome στο Evermusic;" closed="true" %}}
Ναι. Τόσο το Jellyfin όσο και το Navidrome υποστηρίζονται πλήρως στο Evermusic 8.6. Το Jellyfin είναι δωρεάν, ανοιχτού κώδικα media server. Το Navidrome είναι δωρεάν, ανοιχτού κώδικα music server που υλοποιεί το API Subsonic. Το Evermusic συνδέεται και στα δύο native.
{{% /details %}}

{{% details title="Είναι δωρεάν τα Plex, Jellyfin, Emby, Navidrome και Subsonic;" closed="true" %}}
- **Plex** — ο server είναι δωρεάν· το Plex Pass είναι προαιρετική επί πληρωμή αναβάθμιση.
- **Jellyfin** — εντελώς δωρεάν και ανοιχτού κώδικα.
- **Emby** — ο server είναι δωρεάν· το Emby Premiere είναι επί πληρωμή και ξεκλειδώνει συγχρονισμό κινητού και offline.
- **Navidrome** — εντελώς δωρεάν και ανοιχτού κώδικα.
- **Subsonic** — ο επίσημος server κοστίζει 1 $/μήνα μετά από δοκιμή 30 ημερών, αλλά το API του είναι ανοιχτό και πολλοί δωρεάν διακομιστές (συμπεριλαμβανομένου του Navidrome) το υλοποιούν.
{{% /details %}}

{{% details title="Μπορώ να κάνω streaming από το οικιακό μου NAS μέσω SFTP, FTP ή NFS;" closed="true" %}}
Ναι. Το Evermusic 8.6 προσθέτει SFTP, FTP και NFS ως native τύπους σύνδεσης. Το SFTP είναι η συνιστώμενη επιλογή για streaming από τον δικό σας server μέσω ανοιχτού διαδικτύου, καθώς όλη η κίνηση είναι κρυπτογραφημένη μέσω SSH. Τα FTP και NFS λειτουργούν καλύτερα στο τοπικό δίκτυο ή πίσω από VPN.
{{% /details %}}

{{% details title="Πώς συνδέω το Evermusic σε έναν προσαρμοσμένο server μέσω SFTP;" closed="true" %}}
Ανοίξτε το Evermusic, μεταβείτε στην καρτέλα Συνδέσεις, επιλέξτε SFTP και εισαγάγετε το όνομα κεντρικού υπολογιστή ή IP του server, τη θύρα (συνήθως 22), όνομα χρήστη και είτε κωδικό είτε ιδιωτικό κλειδί SSH. Το Evermusic θα περιηγηθεί στους απομακρυσμένους φακέλους σας και θα κάνει streaming αρχείων ήχου απευθείας με κρυπτογράφηση από άκρο σε άκρο.
{{% /details %}}

{{% details title="Υποστηρίζει το Evermusic Internxt και Proton Drive;" closed="true" %}}
Ναι. Και τα δύο cloud εστιασμένα στην ιδιωτικότητα υποστηρίζονται από το Evermusic 8.6 και μετά. Προστίθενται στο Mega και σε άλλες υπηρεσίες με προτεραιότητα την ιδιωτικότητα που είναι ήδη διαθέσιμες στην εφαρμογή.
{{% /details %}}

{{% details title="Τι είναι το Wi-Fi Drive στο Evermusic;" closed="true" %}}
Το Wi-Fi Drive είναι η ενσωματωμένη ασύρματη λειτουργία μεταφοράς αρχείων του Evermusic. Σας επιτρέπει να ανεβάσετε μουσική από τον υπολογιστή σας στο iPhone ή στο iPad μέσω του τοπικού δικτύου Wi-Fi — χωρίς iTunes, καλώδια ή λογαριασμό cloud. Μπορείτε να χρησιμοποιήσετε οποιονδήποτε desktop browser ή έναν πελάτη WebDAV όπως το Mac Finder ή το Windows File Explorer. Δείτε τον [πλήρη οδηγό Wi-Fi Drive](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive/).
{{% /details %}}

{{% details title="Πώς λειτουργεί το νέο widget στίχων;" closed="true" %}}
Το widget στίχων δείχνει χρονικά συγχρονισμένους στίχους στην Αρχική Οθόνη iPhone, iPad ή Mac για το κομμάτι που παίζει. Η εμφανιζόμενη γραμμή προχωράει αυτόματα με το τραγούδι. Προσθέστε το κρατώντας πατημένη την Αρχική Οθόνη, πατώντας Επεξεργασία > Προσθήκη widget, αναζητώντας Evermusic και επιλέγοντας το widget Στίχοι.
{{% /details %}}

{{% details title="Διορθώνει το Evermusic 8.6 προβλήματα αναπαραγωγής στο Baidu Netdisk και στο Aliyun Drive;" closed="true" %}}
Ναι. Κάναμε σημαντικές βελτιώσεις αξιοπιστίας για 百度网盘 (Baidu Netdisk) και 阿里云盘 (Aliyun Drive), συμπεριλαμβανομένης ταχύτερης λίστας καταλόγων, εξυπνότερων επαναλήψεων σε αδύναμες συνδέσεις και καλύτερης συμπεριφοράς συνέχισης σε μεγάλες συνεδρίες αναπαραγωγής.
{{% /details %}}

Ναι. Το Evermusic είναι δωρεάν λήψη από το App Store, και το 8.6 είναι δωρεάν ενημέρωση για όλους τους υπάρχοντες χρήστες. Το νέο CarPlay, το widget στίχων και όλες οι νέες ενσωματώσεις διακομιστών είναι μέρος της βασικής ενημέρωσης.

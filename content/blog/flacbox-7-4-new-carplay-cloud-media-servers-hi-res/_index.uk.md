## 10+ нових способів підключити вашу музику

Flacbox 7.4 розширює поняття вашої «музичної бібліотеки». Якщо ваші hi-res файли лежать у хмарі, якій ви довіряєте, на NAS удома чи на self-hosted стримінговому сервері, Flacbox тепер відтворює з них напряму — без синхронізації, без експорту, без конвертації форматів.

### Приватні хмари: Internxt і Proton Drive

Якщо вам важливі наскрізне шифрування й сховище без знання провайдера, дві з найшанованіших приватних хмар тепер підтримуються у Flacbox нативно:

- **Internxt** — іспанська хмара з відкритим кодом, постквантовим шифруванням і відповідністю GDPR. Доступний безкоштовний рівень.
- **Proton Drive** — наскрізно зашифроване сховище від творців Proton Mail і Proton VPN з базою у Швейцарії. Є безкоштовний рівень і платні плани для більших бібліотек.

Підключіться один раз — і ваші файли FLAC, DSD чи ALAC йтимуть зашифрованим тунелем. Flacbox ніколи не бачить ваших даних у відкритому вигляді, як і сервер провайдера.

### Особисті сервери: QNAP, Nextcloud, Amazon S3

Для слухачів, які тримають власну інфраструктуру:

- **QNAP** — нативне з'єднання за API з пристроями QNAP NAS для швидкого та надійного відтворення локальною Wi-Fi чи віддалено. Стрімте FLAC і DSD з високим бітрейтом напряму, без перекодування.
- **Nextcloud** — підключіться до будь-якої Nextcloud — самостійно піднятої або керованої. Чудовий варіант, якщо ви вже мігрували з Google Drive чи Dropbox з міркувань приватності.
- **Amazon S3** — спрямуйте Flacbox на будь-який S3-bucket (або S3-сумісне сховище, як Backblaze B2, Wasabi, MinIO, Cloudflare R2) і стрімте свою колекцію напряму.

### <a id="media-servers"></a>Стримінгові сервери: Plex, Subsonic, Navidrome, Jellyfin, Emby

Це найбільша новина для прихильників self-hosted музики. Flacbox 7.4 перетворює ваш iPhone чи Mac на першокласного hi-res клієнта для найпопулярніших відкритих та freemium медіасерверів:

- **Plex** — Plex Media Server **безкоштовний** для завантаження та запуску. Підписка **Plex Pass** опціональна й відкриває мобільну синхронізацію, апаратне транскодування та інші додаткові функції. Flacbox працює і з безкоштовними бібліотеками, і з Plex Pass.
- **Subsonic** — оригінальний self-hosted сервер музичного стрімінгу. Офіційний сервер Subsonic **платний** (1 $/міс після 30-денного пробного періоду), а Flacbox також спілкується API Subsonic з десятками сумісних серверів.
- **Navidrome** — сучасний, легкий, **повністю безкоштовний і з відкритим кодом** музичний сервер, написаний на Go. Реалізує API Subsonic. Працює на Raspberry Pi, NAS чи будь-якій машині з Linux. Дуже рекомендуємо для колекцій без втрат до кількох сотень тисяч треків.
- **Jellyfin** — **повністю безкоштовний і з відкритим кодом** медіасервер (форк Emby спільнотою). Працює з музикою, фільмами, ТБ та аудіокнигами. Без облікових записів, без телеметрії, без підписок.
- **Emby** — медіасервер за моделлю **freemium**. Базовий сервер безкоштовний; **Emby Premiere** — разова або щорічна покупка, що відкриває мобільні застосунки, офлайн-синхронізацію тощо. Flacbox підключається до бібліотек і безкоштовного, і Premiere рівня.

Який би сервер ви не використовували, Flacbox стрімить вашу всю колекцію — альбоми, виконавців, плейлисти, жанри та вбудовані обкладинки — з bit-perfect виводом на USB DAC, 10-смуговим еквалайзером, crossfade і gapless відтворенням, AirPlay, Chromecast і перебудованим **CarPlay**. Сервер веде історію прослуховування; Flacbox її поважає.

### Нові методи передавання: FTP, SFTP, NFS

Для слухачів з власними серверами, домашніми лабораторіями чи універсальними NAS, які не постачаються з вилизаним мобільним застосунком, Flacbox 7.4 додає три класичні мережеві протоколи:

- **SFTP (SSH File Transfer Protocol)** — правильна відповідь для **безпечного віддаленого стрімінгу з власного сервера**. SFTP працює поверх SSH, тож уся передача (автентифікація і аудіодані) шифрується. Якщо у вас є VPS, виділений сервер чи Linux-машина вдома з SSH-доступом, можна покласти туди теку з FLAC або DSD й стрімити через відкритий інтернет, нічого більше не відкриваючи. Підтримує автентифікацію паролем і ключем.
- **FTP** — давній стандарт передавання файлів. Корисно, якщо ваш **домашній NAS** (старіший Synology, ASUS, D-Link, TerraMaster чи універсальні бокси) надає FTP-сервер, але без нативної API-інтеграції. Найкраще використовувати в локальній мережі.
- **NFS (Network File System)** — фактичний стандартний протокол спільного доступу в Linux і на більшості NAS. NFS-шарінги типові для homelab і мереж малих компаній; Flacbox тепер монтує їх і стрімить bit-perfect аудіо напряму. На тому самому обладнанні менше накладних витрат, ніж у SMB.

> **Порада:** SFTP — це протокол, який варто обирати для стрімінгу через відкритий інтернет. FTP та NFS краще тримати в локальній мережі — не виставляйте їх у відкритий інтернет, якщо тільки не загорнули у VPN.

Так. Обидві хмари, що зосереджені на приватності, підтримуються з Flacbox 7.4. Вони приєднуються до MEGA та інших privacy-first сервісів, які вже доступні в застосунку.
{{% /details %}}

{{% details title="Чи відтворює Flacbox файли DSD з Plex, Jellyfin або NAS?" closed="true" %}}
Так. Flacbox відтворює файли DSD64, DSD128 і DSD256 (контейнери DSF і DFF), що стрімяться з Plex, Jellyfin, Emby, Subsonic-сумісних серверів, QNAP, Nextcloud, Amazon S3 та через SFTP, FTP і NFS. Bit-perfect вивід на USB DAC підтримується на iPhone, iPad і Mac.
{{% /details %}}

{{% details title="Як працюють перепроєктовані екрани CarPlay?" closed="true" %}}
Інтерфейс CarPlay у Flacbox перебудовано зі швидким сортуванням за альбомами, виконавцями, плейлистами й теками; кількома кольоровими темами, що пасують різним інтер'єрам авто; новим екраном «Зараз грає» з новими керуваннями; повною чергою відтворення на одному екрані; абетковим індексом для переходу великими бібліотеками; та швидшим завантаженням великих тек і хмарних каталогів.
{{% /details %}}

{{% details title="Оновлення до Flacbox 7.4 безкоштовне?" closed="true" %}}
Так. Flacbox — безкоштовне завантаження з App Store, а 7.4 — безкоштовне оновлення для всіх наявних користувачів. Перебудоване CarPlay, усі нові хмарні та серверні підключення, оновлені віджети Головного екрана та Liquid Glass UI є частиною базового оновлення.
{{% /details %}}

{{% details title="На яких пристроях доступне Flacbox 7.4?" closed="true" %}}
Flacbox 7.4 працює на iPhone, iPad і Mac. Підтримка CarPlay вимагає сумісного з CarPlay автомобіля або стороннього головного пристрою. AirPlay та Chromecast дозволяють кастувати відтворення на більшу систему; USB DAC підтримуються для bit-perfect виводу без втрат.

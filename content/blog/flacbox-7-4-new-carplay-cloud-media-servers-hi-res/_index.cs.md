Přepracovali jsme **Apple CarPlay** pro Flacbox od základů, aby bylo poslouchání za jízdy bezpečnější a snadnější. Každý detail — od toho, jak se najde skladba, až po to, jak se ovládá přehrávání — byl vyladěn pro zážitek ve voze.

- **Rychlé třídění** — dosáhněte jakékoli skladby během okamžiku napříč alby, interprety, playlisty a složkami bez nekonečného posouvání.
- **Barevná schémata, která se hodí k vašemu autu** — vyberte si motiv, který odpovídá palubní desce nebo vnitřnímu osvětlení, ve dne i v noci.
- **Nová obrazovka Nyní hraje** — větší obal alba, jasnější ovládání a nové akce přehrávání jedním klepnutím.
- **Kompletní fronta přehrávání na jeden pohled** — vidíte, co bude následovat, aniž byste opustili aktuální obrazovku.
- **Abecední rejstřík pro velké knihovny** — proskočte desítky tisíc skladeb FLAC, DSD, ALAC a APE jediným dotykem.
- **Rychlejší načítání, žádné pauzy** — otevírání velkých složek, cloudových adresářů nebo knihoven mediálních serverů je okamžité.

Pokud streamujete bezztrátový zvuk během řízení — z Dropboxu, Google Drive, OneDrive, iCloud Drive, [Plexu](#media-servers), [Jellyfinu](#media-servers), Subsonicu nebo z jakéhokoli jiného zdroje — přebudované rozhraní CarPlay vytváří dojem, že celý tok je nativní součástí auta.

## Více než 10 nových způsobů, jak připojit hudbu

Flacbox 7.4 rozšiřuje, co se počítá za vaši «hudební knihovnu». Pokud vaše hi-res soubory žijí v cloudu, kterému důvěřujete, na domácím NAS nebo na samohostovaném streamovacím serveru, Flacbox z nich nyní přehrává přímo — bez synchronizace, exportu nebo konverze formátu.

### Soukromé cloudy: Internxt a Proton Drive

Pokud vám záleží na šifrování end-to-end a úložišti bez znalosti, dva z nejrespektovanějších cloudů zaměřených na soukromí jsou nyní ve Flacboxu nativní:

- **Internxt** — open-source, post-kvantově šifrovaný, s GDPR kompatibilní španělský cloud. Bezplatná úroveň k dispozici.
- **Proton Drive** — úložiště se šifrováním end-to-end od tvůrců Proton Mailu a Proton VPN, se sídlem ve Švýcarsku. Bezplatná úroveň k dispozici s placenými plány pro větší knihovny.

Připojte se jednou a vaše soubory FLAC, DSD nebo ALAC budou proudit šifrovaným tunelem — Flacbox nikdy nevidí vaše data v čistém textu, stejně tak server poskytovatele.

### Osobní servery: QNAP, Nextcloud, Amazon S3

Pro posluchače provozující vlastní infrastrukturu:

- **QNAP** — nativní API připojení k zařízením QNAP NAS pro rychlé a spolehlivé přehrávání přes místní Wi-Fi nebo vzdálený přístup. Streamujte FLAC a DSD s vysokým datovým tokem přímo bez překódování.
- **Nextcloud** — připojte se k jakékoli samohostované nebo spravované instanci Nextcloud. Skvělá volba, pokud jste z důvodů soukromí už opustili Google Drive nebo Dropbox.
- **Amazon S3** — namiřte Flacbox na libovolný bucket S3 (nebo úložiště kompatibilní s S3 jako Backblaze B2, Wasabi, MinIO, Cloudflare R2) a streamujte svou sbírku přímo.

### <a id="media-servers"></a>Streamovací servery: Plex, Subsonic, Navidrome, Jellyfin, Emby

Toto je velká novinka pro fanoušky samohostované hudby. Flacbox 7.4 promění váš iPhone nebo Mac v prvotřídního hi-res klienta pro nejoblíbenější open-source a freemium mediální servery:

- **Plex** — Plex Media Server je **zdarma** ke stažení a provozu. Předplatné **Plex Pass** je volitelné a odemyká mobilní synchronizaci, hardwarové překódování a další extra funkce. Flacbox funguje s bezplatnými i Plex Pass knihovnami.
- **Subsonic** — původní samohostovaný hudební streamovací server. Oficiální Subsonic server je **placený** (1 $/měsíc po 30denní zkušební době) a Flacbox také hovoří Subsonic API s desítkami kompatibilních serverů.
- **Navidrome** — moderní, lehký, **zcela zdarma a open-source** hudební server napsaný v Go. Implementuje Subsonic API. Běží na Raspberry Pi, NAS nebo jakémkoli linuxovém stroji. Vřele doporučeno pro bezztrátové sbírky až do několika set tisíc skladeb.
- **Jellyfin** — **zcela zdarma a open-source** mediální server (komunitní fork Emby). Zpracovává hudbu, filmy, televizi a audioknihy. Žádné účty, žádná telemetrie, žádná předplatná.
- **Emby** — **freemium** mediální server. Základní server je zdarma; **Emby Premiere** je jednorázový nebo roční nákup, který odemyká mobilní aplikace, offline synchronizaci a další. Flacbox se připojuje k bezplatným i Premiere knihovnám.

Ať už provozujete jakýkoli server, Flacbox streamuje celou vaši sbírku — alba, interprety, playlisty, žánry a vložené obaly — s bit-perfect výstupem do USB DAC, 10pásmovým ekvalizérem, crossfade a gapless přehráváním, AirPlay, Chromecastem a přebudovaným zážitkem **CarPlay**. Váš server vede historii poslechu; Flacbox ji respektuje.

### Nové metody přenosu: FTP, SFTP, NFS

Pro posluchače s vlastními servery, domácími laboratořemi nebo obecnými NAS boxy, které nejsou dodávány s vyladěnou mobilní aplikací, Flacbox 7.4 přidává tři klasické síťové protokoly:

- **SFTP (SSH File Transfer Protocol)** — správná odpověď pro **bezpečné vzdálené streamování z vlastního serveru**. SFTP běží přes SSH, takže celý přenos (autentizace i zvuková data) je šifrován. Pokud máte VPS, dedikovaný server nebo linuxový stroj doma s SSH přístupem, můžete na něj umístit složku s FLAC nebo DSD a streamovat přes veřejný internet, aniž byste cokoli jiného vystavovali. Podporuje autentizaci pomocí hesla i klíče.
- **FTP** — dlouholetý standard pro přenos souborů. Užitečné, pokud váš **domácí NAS** (starší Synology, ASUS, D-Link, TerraMaster nebo obecné boxy) zpřístupňuje FTP server, ale nemá nativní API integraci. Nejlépe uvnitř vaší lokální sítě.
- **NFS (Network File System)** — de-facto sdílecí protokol na Linuxu a většině NAS zařízení. NFS sdílení jsou běžná v domácích laboratořích a sítích malých firem; Flacbox je nyní připojí a streamuje z nich bit-perfect zvuk přímo. Nižší režie než SMB na stejném hardwaru.

> **Tip:** SFTP je protokol, který chcete pro streamování z otevřeného internetu. FTP a NFS fungují nejlépe uvnitř vaší lokální sítě — držte je mimo veřejný internet, pokud je neuzavřete do VPN.

Widgety domovské obrazovky — Nyní hraje, Fronta přehrávání, Naposledy přehráno — byly přepracovány s čistším rozvržením a chytřejší politikou obnovy.

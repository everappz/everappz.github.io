## Apa yang Dibawa Evermusic 2.3 untuk Perpustakaan Musik Anda

**TL;DR:** Evermusic 2.3 memperkenalkan sinkronisasi folder cloud otomatis, penyimpanan posisi pemutaran (ideal untuk buku audio), pembacaan metadata di latar belakang, dan koreksi tag ID3 otomatis. Bekerja dengan Dropbox, Google Drive, OneDrive, MEGA, WebDAV, dan SMB.

Evermusic adalah pemutar musik cloud untuk iOS dan macOS yang memungkinkan Anda streaming atau mengunduh audio dari Dropbox, Google Drive, OneDrive, MEGA, WebDAV, dan server SMB. Versi 2.3 menambahkan empat fitur yang telah diminta pengguna. Jika Anda belum mencoba Evermusic, [unduh dari App Store](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198).

## Sinkronisasi Perpustakaan Musik Otomatis

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_7ae554f5982649758176de1c994b6b05~mv2.jpg)

Tidak perlu lagi pemindaian folder manual. Evermusic 2.3 memantau folder cloud yang Anda pilih dan memperbarui perpustakaan musik Anda secara otomatis saat file berubah. Atur folder yang ingin Anda lacak dan aplikasi menangani sisanya di latar belakang.

Ini berarti:

- Unggahan baru muncul di perpustakaan Anda tanpa penyegaran manual
- File yang dihapus dihapus dari perpustakaan Anda secara otomatis
- Perubahan disinkronkan baik saat aplikasi di latar depan maupun latar belakang

## Simpan Posisi Pemutaran untuk Buku Audio dan Podcast

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_3baab435323443bcb788e5f6da0d39f3~mv2.jpg)

Jeda buku audio atau kuliah dan lanjutkan tepat di mana Anda berhenti. Saat diaktifkan, Evermusic mengingat posisi pemutaran untuk setiap file secara otomatis — tidak perlu penanda manual.

Terbaik untuk:

- Buku audio
- Kuliah dan kursus
- Podcast
- Rekaman panjang

Aktifkan ini di pengaturan audio aplikasi.

## Pembacaan Metadata di Latar Belakang

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_511ee81f8b094059a2c14f1cfc32a1cc~mv2.jpg)

Untuk koleksi musik besar, pembacaan metadata sekarang berjalan di latar belakang. Saat Anda menjelajah atau mendengarkan, Evermusic memindai file audio Anda dan mengorganisirnya berdasarkan Artis, Album, dan Genre. Ini secara signifikan mempercepat penjelajahan perpustakaan untuk koleksi dengan ribuan lagu.

## Koreksi Tag ID3 Otomatis

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_2374803418e74c8680fd3420fc1fc35a~mv2.jpg)

Evermusic mendeteksi dan memperbaiki tag ID3 yang tidak valid atau tidak lengkap menggunakan database online. File dengan metadata yang berantakan, hilang, atau salah dibersihkan secara otomatis, menjaga perpustakaan Anda tetap terorganisir tanpa pengeditan manual.

## Evermusic 2.3 為你的音樂庫帶來了什麼

**摘要：** Evermusic 2.3 引入自動雲端資料夾同步、播放位置儲存（適合有聲書）、背景元資料讀取和自動 ID3 標籤修正。支援 Dropbox、Google Drive、OneDrive、MEGA、WebDAV 和 SMB。

Evermusic 是一款 iOS 和 macOS 雲端音樂播放器，可讓你從 Dropbox、Google Drive、OneDrive、MEGA、WebDAV 和 SMB 伺服器串流或下載音訊。版本 2.3 新增了使用者一直要求的四項功能。如果你還沒有試過 Evermusic，[從 App Store 下載](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198)。

## 自動音樂庫同步

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_7ae554f5982649758176de1c994b6b05~mv2.jpg)

不再需要手動掃描資料夾。Evermusic 2.3 監控你選擇的雲端資料夾，並在檔案變更時自動更新音樂庫。設定要追蹤的資料夾，應用程式會在背景處理其餘工作。

這意味著：

- 新上傳的檔案無需手動重新整理即可出現在音樂庫中
- 已刪除的檔案會自動從音樂庫中移除
- 無論應用程式在前景還是背景，變更都會同步

## 為有聲書和播客儲存播放位置

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_3baab435323443bcb788e5f6da0d39f3~mv2.jpg)

暫停有聲書或講座，從上次停止的位置繼續。啟用後，Evermusic 會自動記住每個檔案的播放位置——無需手動書籤。

最適合：

- 有聲書
- 講座和課程
- 播客
- 長時間錄音

在應用程式的音訊設定中啟用此功能。

## 背景元資料讀取

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_511ee81f8b094059a2c14f1cfc32a1cc~mv2.jpg)

對於大型音樂收藏，元資料讀取現在在背景執行。在你瀏覽或收聽時，Evermusic 掃描音訊檔案並按藝人、專輯和類型進行組織。這大幅加速了擁有數千首曲目的收藏的音樂庫瀏覽速度。

## 自動 ID3 標籤修正

![](/blog/evermusic-sync-your-music-library-save-playback-position-correct-music-tags/21260c_2374803418e74c8680fd3420fc1fc35a~mv2.jpg)

Evermusic 使用線上資料庫偵測並修正無效或不完整的 ID3 標籤。元資料混亂、缺失或不正確的檔案會被自動清理，無需手動編輯即可保持音樂庫井然有序。

## 常見問題

{{% details title="Evermusic 自動同步適用於所有雲端服務嗎？" closed="true" %}}
是的。自動同步適用於 Dropbox、Google Drive、OneDrive、MEGA、WebDAV 和 SMB。選擇要監控的資料夾，Evermusic 會保持音樂庫處於最新狀態。
{{% /details %}}

{{% details title="Evermusic 可以儲存有聲書位置嗎？" closed="true" %}}
可以。在音訊設定中啟用播放位置儲存。Evermusic 會記住每個檔案停止的位置，讓你無需手動書籤即可繼續。
{{% /details %}}

{{% details title="背景元資料讀取如何運作？" closed="true" %}}
Evermusic 在你使用其他功能時在背景讀取 ID3 標籤和檔案元資料。它會自動按藝人、專輯和類型組織音樂庫。
{{% /details %}}

{{% details title="Evermusic 會修復損壞的音樂標籤嗎？" closed="true" %}}
會。自動標籤修正功能會將你的檔案與線上資料庫進行比對，修復無效、不完整或缺失的 ID3 元資料。
{{% /details %}}

{{% details title="Evermusic 可以免費下載嗎？" closed="true" %}}
Evermusic 可免費下載，提供可透過應用程式內購買取得的進階功能。
{{% /details %}}

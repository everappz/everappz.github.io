## Evermusic 3.1 : ce qui a changé et pourquoi c'est important

**En bref :** Evermusic 3.1 introduit le fondu enchaîné (3 à 15 secondes), une bibliothèque musicale repensée avec synchronisation, la sauvegarde/restauration complète vers le stockage cloud, la navigation dans la bibliothèque iPod, des contrôles de file d'attente améliorés et une meilleure édition des tags ID3.

Cette version apporte des améliorations significatives en matière de lecture, de gestion de bibliothèque et de sécurité des données. Voici toutes les nouveautés de la version 3.1.

### Fondu enchaîné

Le fondu enchaîné élimine le silence entre les pistes en mélangeant la fin d'une chanson avec le début de la suivante. Cela fonctionne bien pour :

- La musique classique et les enregistrements live
- Les playlists style DJ
- L'écoute continue en arrière-plan

Vous pouvez régler la durée du fondu enchaîné de **3 à 15 secondes** ou le désactiver entièrement.

`Settings → Audio Player → Crossfade Playback`

### Bibliothèque musicale repensée

L'interface de la bibliothèque musicale a été reconstruite pour plus de rapidité et de clarté :

- Parcourez les morceaux, artistes et albums dans une mise en page plus claire
- Appuyez sur le bouton **Synchronisation** pour actualiser après avoir téléversé de nouvelles pistes vers le stockage cloud
- Triez par nom, taille, numéro de piste, album ou artiste
- Ajustez la **vitesse de lecture des métadonnées** pour équilibrer performance et autonomie de la batterie

`Settings → Music Library → Metadata Reading`

### Sauvegarde et restauration de la bibliothèque musicale

Protégez vos données musicales avec une sauvegarde en un geste :

- Sauvegarde les playlists, les métadonnées des morceaux, les pochettes et les paramètres de l'application
- Enregistre un seul fichier d'archive vers votre stockage cloud connecté
- Restaurez tout en un seul geste sur n'importe quel appareil

`Settings → Backup & Restore`

### File d'attente du lecteur audio améliorée

Les nouveaux contrôles de file d'attente vous offrent plus de flexibilité :

- **Lire ensuite** — insérez un morceau juste après la piste en cours
- **Lire plus tard** — ajoutez des morceaux à la fin de la file d'attente
- Réordonnez ou supprimez des pistes avec des gestes simples

### Intégration de la bibliothèque iPod

Parcourez votre bibliothèque iPod directement depuis l'écran d'accueil d'Evermusic. Vous pouvez ajouter des pistes iPod à votre file d'attente de lecture ou les copier dans la section Fichiers de l'application.

Catégories disponibles :

- Playlists
- Albums
- Artistes
- Genres

### Modification des comptes cloud

Modifiez les identifiants de connexion de n'importe quel service cloud connecté sans supprimer et réajouter le compte. Cela fait gagner du temps lorsque les mots de passe changent ou que les jetons expirent.

### Éditeur de tags amélioré

L'**éditeur de tags audio** gère désormais la correction des métadonnées de manière plus fiable :

- Utilisez l'action **Identifier** pour analyser les noms de fichiers et mettre à jour automatiquement les tags ID3
- Corrigez les métadonnées erronées ou ajoutez les informations d'album manquantes en masse

[Téléchargez Evermusic depuis l'App Store](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198) et essayez ces fonctionnalités dès aujourd'hui.

## Questions fréquemment posées

{{% details title="Qu'est-ce que le fondu enchaîné dans Evermusic ?" closed="true" %}}
Le fondu enchaîné mélange la fin d'une piste avec le début de la suivante, créant des transitions fluides. Vous pouvez régler la durée entre 3 et 15 secondes dans Settings → Audio Player → Crossfade Playback.
{{% /details %}}

{{% details title="Puis-je sauvegarder mes playlists Evermusic vers le stockage cloud ?" closed="true" %}}
Oui. Evermusic 3.1 vous permet de sauvegarder toute votre bibliothèque — y compris les playlists, les métadonnées, les pochettes et les paramètres — vers n'importe quel service cloud connecté sous forme d'un seul fichier.
{{% /details %}}

{{% details title="Evermusic prend-il en charge la navigation dans la bibliothèque iPod ?" closed="true" %}}
Oui. Vous pouvez parcourir votre bibliothèque iPod par playlists, albums, artistes et genres directement depuis l'écran d'accueil d'Evermusic et ajouter des pistes à votre file d'attente.
{{% /details %}}

{{% details title="Comment corriger les tags de morceaux incorrects dans Evermusic ?" closed="true" %}}
Utilisez l'éditeur de tags intégré et appuyez sur l'action Identifier. Evermusic analyse vos noms de fichiers et met à jour automatiquement les tags ID3 avec les métadonnées corrigées.
{{% /details %}}

{{% details title="Quels services cloud Evermusic prend-il en charge ?" closed="true" %}}
Evermusic fonctionne avec Dropbox, Google Drive, OneDrive, MEGA, Box, Yandex.Disk, WebDAV, SMB/CIFS et les serveurs FTP.
{{% /details %}}

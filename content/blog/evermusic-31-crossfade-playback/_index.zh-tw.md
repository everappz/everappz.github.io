## Evermusic 3.1：有何變化及其重要性

**TL;DR：** Evermusic 3.1 引入交叉淡入淡出播放（3-15 秒）、帶同步功能的重新設計音樂庫、完整的雲端儲存備份/還原、iPod 音樂庫瀏覽、改進的佇列控制和更好的 ID3 標籤編輯。

此版本在播放、音樂庫管理和資料安全方面帶來了有意義的升級。以下是 3.1 版本的所有新內容。

### 交叉淡入淡出播放

交叉淡入淡出透過將一首歌的結尾與下一首歌的開頭混合來消除曲目之間的靜音。適用於：

- 古典音樂和現場錄音
- DJ 風格的播放清單
- 連續背景收聽

您可以將交叉淡入淡出時長設定為 **3 到 15 秒**，也可以完全關閉。

`Settings → Audio Player → Crossfade Playback`

### 重新設計的音樂庫

音樂庫介面已重建，更快速、更清晰：

- 在更簡潔的佈局中瀏覽歌曲、藝人和專輯
- 將新曲目上傳到雲端儲存後，點擊 **Synchronization** 按鈕重新整理
- 按名稱、大小、曲目編號、專輯或藝人排序
- 調整**中繼資料掃描速度**以平衡效能和電池續航

`Settings → Music Library → Metadata Reading`

### 音樂庫備份和還原

一鍵保護您的音樂資料：

- 備份播放清單、歌曲中繼資料、封面和應用程式設定
- 將單一封存檔案儲存到已連接的雲端儲存
- 在任何裝置上一鍵還原所有內容

`Settings → Backup & Restore`

### 增強的音訊播放器佇列

新的佇列控制提供更大的彈性：

- **Play Next** — 在目前曲目之後插入一首歌
- **Play Later** — 將歌曲新增到佇列末尾
- 透過簡單手勢重新排列或移除曲目

### iPod 音樂庫整合

直接從 Evermusic 主畫面瀏覽您的 iPod 音樂庫。您可以將 iPod 曲目新增到播放佇列或將其複製到應用程式的 Files 部分。

可用分類：

- 播放清單
- 專輯
- 藝人
- 類型

### 雲端帳戶編輯

編輯任何已連接雲端服務的登入憑證，無需刪除並重新新增帳戶。當密碼變更或權杖過期時可節省時間。

### 改進的標籤編輯器

**Audio Tags Editor** 現在更可靠地處理中繼資料修正：

- 使用 **Identify** 操作掃描檔案名稱並自動更新 ID3 標籤
- 修復損壞的中繼資料或批次新增遺失的專輯資訊

[從 App Store 下載 Evermusic](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198)，立即試用這些功能。

## 常見問題

{{% details title="Evermusic 中的交叉淡入淡出播放是什麼？" closed="true" %}}
交叉淡入淡出播放將一首曲目的結尾與下一首的開頭混合，創造無縫過渡。您可以在 Settings → Audio Player → Crossfade Playback 中將時長設定為 3 到 15 秒。
{{% /details %}}

{{% details title="我可以將 Evermusic 播放清單備份到雲端儲存嗎？" closed="true" %}}
可以。Evermusic 3.1 允許您將整個音樂庫——包括播放清單、中繼資料、封面和設定——備份到任何已連接的雲端服務，儲存為單一檔案。
{{% /details %}}

{{% details title="Evermusic 支援 iPod 音樂庫瀏覽嗎？" closed="true" %}}
支援。您可以直接從 Evermusic 主畫面按播放清單、專輯、藝人和類型瀏覽 iPod 音樂庫，並將曲目新增到佇列。
{{% /details %}}

{{% details title="如何修復 Evermusic 中不正確的歌曲標籤？" closed="true" %}}
使用內建的 Tags Editor 並點擊 Identify 操作。Evermusic 會掃描您的檔案名稱並自動用修正後的中繼資料更新 ID3 標籤。
{{% /details %}}

{{% details title="Evermusic 支援哪些雲端服務？" closed="true" %}}
Evermusic 支援 Dropbox、Google Drive、OneDrive、MEGA、Box、Yandex.Disk、WebDAV、SMB/CIFS 和 FTP 伺服器。
{{% /details %}}

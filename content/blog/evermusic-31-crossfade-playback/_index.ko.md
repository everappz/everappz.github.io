## Evermusic 3.1: 변경 사항과 중요한 이유

**요약:** Evermusic 3.1은 크로스페이드 재생(3~15초), 동기화 기능이 있는 재설계된 음악 라이브러리, 클라우드 스토리지로의 완전한 백업/복원, iPod 라이브러리 탐색, 개선된 대기열 컨트롤, 향상된 ID3 태그 편집 기능을 도입했습니다.

이번 릴리스는 재생, 라이브러리 관리, 데이터 안전성에 걸쳐 의미 있는 업그레이드를 제공합니다. 버전 3.1의 모든 새로운 기능을 소개합니다.

### 크로스페이드 재생

크로스페이드는 한 곡의 끝부분을 다음 곡의 시작 부분과 블렌딩하여 트랙 사이의 무음을 제거합니다. 다음과 같은 경우에 효과적입니다:

- 클래식 음악과 라이브 녹음
- DJ 스타일 재생목록
- 연속 배경 음악 청취

크로스페이드 지속 시간을 **3~15초**로 설정하거나 완전히 끌 수 있습니다.

`Settings → Audio Player → Crossfade Playback`

### 재설계된 음악 라이브러리

음악 라이브러리 인터페이스가 속도와 명확성을 위해 새롭게 구축되었습니다:

- 더 깔끔한 레이아웃으로 곡, 아티스트, 앨범 탐색
- 클라우드 스토리지에 새 트랙을 업로드한 후 **동기화** 버튼을 탭하여 새로고침
- 이름, 크기, 트랙 번호, 앨범 또는 아티스트별 정렬
- **메타데이터 스캔 속도**를 조정하여 성능과 배터리 수명의 균형을 맞춤

`Settings → Music Library → Metadata Reading`

### 음악 라이브러리 백업 및 복원

원탭 백업으로 음악 데이터를 보호하세요:

- 재생목록, 곡 메타데이터, 커버 아트, 앱 설정을 백업
- 연결된 클라우드 스토리지에 단일 아카이브 파일로 저장
- 모든 기기에서 원탭으로 모든 것을 복원

`Settings → Backup & Restore`

### 향상된 오디오 플레이어 대기열

새로운 대기열 컨트롤로 더 많은 유연성을 제공합니다:

- **다음에 재생** — 현재 트랙 바로 뒤에 곡 삽입
- **나중에 재생** — 대기열 끝에 곡 추가
- 간단한 제스처로 트랙 순서 변경 또는 제거

### iPod 라이브러리 통합

Evermusic 홈 화면에서 직접 iPod 라이브러리를 탐색하세요. iPod 트랙을 재생 대기열에 추가하거나 앱의 파일 섹션으로 복사할 수 있습니다.

사용 가능한 카테고리:

- 재생목록
- 앨범
- 아티스트
- 장르

### 클라우드 계정 편집

계정을 제거하고 다시 추가하지 않고도 연결된 클라우드 서비스의 로그인 자격 증명을 편집하세요. 비밀번호가 변경되거나 토큰이 만료될 때 시간을 절약합니다.

### 개선된 태그 편집기

**Audio Tags Editor**가 메타데이터 수정을 더 안정적으로 처리합니다:

- **식별** 작업을 사용하여 파일명을 스캔하고 ID3 태그를 자동으로 업데이트
- 손상된 메타데이터를 수정하거나 누락된 앨범 정보를 일괄 추가

[App Store에서 Evermusic을 다운로드](https://apps.apple.com/us/app/evermusic-cloud-music-player/id885367198)하고 오늘 이 기능들을 사용해 보세요.

## Δεδομένα App Store σε δευτερόλεπτα

**Σύντομη έκδοση:** Το [AppLookup.pro](https://applookup.pro) είναι ένα δωρεάν εργαλείο που ανακτά δημόσια δεδομένα για οποιαδήποτε εφαρμογή iOS, iPadOS, macOS ή tvOS. Παίρνετε τον τίτλο, την περιγραφή, τα νέα της έκδοσης, την έκδοση, την τιμή, τις βαθμολογίες, το εικονίδιο της εφαρμογής, τα στιγμιότυπα, τις υποστηριζόμενες συσκευές και την ακατέργαστη απόκριση του iTunes API. Κάθε πεδίο έχει ένα κουμπί αντιγραφής με ένα πάτημα. Ανοίξτε τον ιστότοπο, επικολλήστε έναν σύνδεσμο App Store ή πληκτρολογήστε το όνομα της εφαρμογής και έχετε τα δεδομένα.

Χρησιμοποιήστε το για:

- **Έρευνα ASO.** Δείτε πώς οι κορυφαίες εφαρμογές γράφουν τους τίτλους, τους υπότιτλους, τις περιγραφές και τις λέξεις-κλειδιά τους.
- **Παρακολούθηση ανταγωνιστών.** Ελέγξτε ενημερώσεις έκδοσης, βαθμολογίες και τιμές σε διάφορες αγορές.
- **Λήψη υλικού.** Αποθηκεύστε το επίσημο εικονίδιο της εφαρμογής και τα στιγμιότυπα πλήρους μεγέθους σε ένα ZIP.
- **Έλεγχοι τοπικής προσαρμογής.** Συγκρίνετε περιγραφή, νέα έκδοσης και στιγμιότυπα σε πάνω από 40 χώρες App Store.
- **Δοκιμή API.** Αντιγράψτε το ακατέργαστο JSON του iTunes Search API απευθείας στον κώδικά σας ή στο Postman.

## Πώς να χρησιμοποιήσετε το AppLookup.pro βήμα προς βήμα

### Βήμα 1. Εισαγάγετε το όνομα της εφαρμογής ή επικολλήστε ένα URL App Store

Ανοίξτε το [applookup.pro](https://applookup.pro) και ξεκινήστε να πληκτρολογείτε το όνομα της εφαρμογής. Η αυτόματη συμπλήρωση εμφανίζει ζωντανά αποτελέσματα App Store καθώς πληκτρολογείτε.

Μπορείτε επίσης να επικολλήσετε έναν απευθείας σύνδεσμο App Store όπως `https://apps.apple.com/us/app/instagram/id389801252` ή απλώς το ID της εφαρμογής. Το εργαλείο εξάγει το ID για εσάς. Χειρίζεται επίσης τα URL ανακατεύθυνσης Google.

![Εισαγάγετε όνομα εφαρμογής ή URL App Store στο AppLookup.pro](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/1-app-store-lookup-enter-app-name.webp)

### Βήμα 2. Ανακτήστε τις πληροφορίες της εφαρμογής και κατεβάστε το εικονίδιο

Κάντε κλικ στο **Lookup** ή πατήστε Enter. Το εργαλείο καλεί το επίσημο iTunes Search API και εμφανίζει το εικονίδιο της εφαρμογής, το όνομα του προγραμματιστή, τη βαθμολογία, την έκδοση και την τιμή σε λιγότερο από ένα δευτερόλεπτο.

Κάντε κύλιση στην ενότητα **App Icon**. Κάθε μέγεθος εικονιδίου που επιστρέφει η Apple εμφανίζεται ως κάρτα. Κάθε κάρτα έχει:

- **Direct Link.** Ανοίγει την εικόνα πλήρους μεγέθους σε νέα καρτέλα.
- **Download.** Αποθηκεύει το αρχείο στον υπολογιστή σας.

Χρησιμοποιήστε το **Download All Icons (ZIP)** για να πάρετε κάθε μέγεθος εικονιδίου σε ένα αρχείο. Το ίδιο ισχύει για τα στιγμιότυπα: κάθε ενότητα πλατφόρμας έχει το δικό της κουμπί **Download All (ZIP)**.

![Λήψη εικονιδίων εφαρμογής και στιγμιοτύπων από το AppLookup.pro](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/2-app-store-lookup-view-app-details-icons.webp)

### Βήμα 3. Διαβάστε τις λεπτομέρειες της εφαρμογής και αντιγράψτε οποιοδήποτε πεδίο

Κάντε κύλιση στο **App Details**. Θα δείτε bundle ID, έκδοση, τιμή, μέγεθος αρχείου, ελάχιστο OS, ημερομηνία κυκλοφορίας, ημερομηνία τελευταίας ενημέρωσης, ηλικιακή κατάταξη, κατηγορίες, IDs κατηγοριών, γλώσσες, πωλητή, ID καλλιτέχνη και ID κομματιού.

Πατήστε το κουμπί **Copy** σε οποιαδήποτε κάρτα. Η τιμή πηγαίνει στο πρόχειρό σας και το κουμπί εμφανίζει ένα πράσινο σημάδι 'Copied'.

Το ίδιο ισχύει για τις ενότητες **Description**, **What's New** και **Supported Devices**. Αυτές οι ενότητες κάνουν κύλιση ώστε να μπορείτε να διαβάσετε το πλήρες κείμενο χωρίς να χάσετε τη θέση σας και το κουμπί Copy τοποθετεί ολόκληρο το πεδίο στο πρόχειρό σας.

![Δείτε τις πλήρεις λεπτομέρειες App Store και αντιγράψτε οποιοδήποτε πεδίο με ένα πάτημα](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/3-app-store-lookup-view-app-details.webp)

### Βήμα 4. Δείτε την ακατέργαστη απόκριση του App Store API

Χρειάζεστε το ακριβές JSON που επιστρέφει η Apple; Κάντε κύλιση στο **Raw API Response**. Το πλήρες payload του iTunes Search API εμφανίζεται σε ένα παράθυρο προβολής με κύλιση με ένα κουμπί **Copy** στην κορυφή. Ένα πάτημα αντιγράφει ολόκληρο το αντικείμενο JSON.

Το **iTunes Lookup URL** εμφανίζεται ακριβώς πάνω από αυτό. Επικολλήστε το στο Postman, στο curl ή στο πρόγραμμα περιήγησής σας για να επαναλάβετε το ίδιο αίτημα.

![Δείτε και αντιγράψτε την ακατέργαστη απόκριση JSON του iTunes Search API](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/4-app-store-lookup-view-app-raw-api-response.webp)

### Βήμα 5. Αλλάξτε τη χώρα για να δείτε την τοπική έκδοση

Τα μεταδεδομένα App Store αλλάζουν ανά χώρα. Η ίδια εφαρμογή έχει συχνά διαφορετικό τίτλο, υπότιτλο, περιγραφή, στιγμιότυπα και τιμή σε κάθε αγορά.

Επιλέξτε μια χώρα από το αναπτυσσόμενο μενού στην κορυφή. Το URL στο πλαίσιο εισαγωγής ενημερώνεται αυτόματα. Κάντε κλικ ξανά στο **Lookup** για να ξανα-ανακτήσετε την εφαρμογή σε εκείνη την αγορά.

Αυτός είναι ο γρηγορότερος τρόπος να ελέγξετε πώς ένας ανταγωνιστής παρουσιάζει την εφαρμογή του στην Ιαπωνία, τη Γερμανία, τη Βραζιλία ή σε οποιαδήποτε από τις πάνω από 40 υποστηριζόμενες χώρες.

![Εναλλαγή καταστημάτων ανά χώρα για να δείτε τοπικά μεταδεδομένα App Store](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/5-app-store-lookup-change-app-country.webp)

### Βήμα 6. Αντιγράψτε τα τοπικά μεταδεδομένα

Μόλις φορτωθεί το αποτέλεσμα της χώρας, κάθε πεδίο λειτουργεί με τον ίδιο τρόπο. Πατήστε **Copy** στην περιγραφή, στα νέα της έκδοσης, στο όνομα εφαρμογής, στον προγραμματιστή, στο bundle ID ή σε οποιαδήποτε κάρτα λεπτομερειών για να αποτυπώσετε το τοπικό κείμενο.

Αυτό διευκολύνει τη δημιουργία υπολογιστικών φύλλων σύγκρισης δίπλα-δίπλα ή την τροφοδοσία τοπικών κειμένων στον έλεγχο μετάφρασης.

![Αντιγράψτε τοπική περιγραφή και μεταδεδομένα App Store με ένα πάτημα](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/6-app-store-lookup-copy-localized-app-description.webp)

## Απόρρητο και αποποίηση ευθύνης

Το AppLookup.pro τρέχει στο πρόγραμμα περιήγησής σας. Δεν υπάρχει σύνδεση. Δεν υπάρχει παρακολούθηση. Δεν υπάρχει καταγραφή διακομιστή των εφαρμογών που αναζητάτε. Τα αιτήματα πηγαίνουν απευθείας από το πρόγραμμα περιήγησής σας στο [iTunes Search API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html) της Apple.

Αυτό το εργαλείο προορίζεται **μόνο για εκπαιδευτικούς και ερευνητικούς σκοπούς**. Όλα τα δεδομένα ανακτώνται από το επίσημο δημόσιο API της Apple και παραμένουν ιδιοκτησία της Apple Inc. και των αναφερόμενων εκδοτών εφαρμογών. Η χρήση του εργαλείου υπόκειται στους [Όρους και Προϋποθέσεις του Apple Media Services](https://www.apple.com/legal/internet-services/terms/site.html). Σεβαστείτε τα όρια ρυθμού της Apple και μην αναδιανέμετε υλικό που προστατεύεται από πνευματικά δικαιώματα.

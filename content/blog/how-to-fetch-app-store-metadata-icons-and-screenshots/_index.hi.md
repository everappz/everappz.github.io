## सेकंडों में App Store डेटा प्राप्त करें

**संक्षेप में:** [AppLookup.pro](https://applookup.pro) एक मुफ्त टूल है जो किसी भी iOS, iPadOS, macOS या tvOS ऐप के सार्वजनिक डेटा को खींचता है। आपको शीर्षक, विवरण, क्या नया है, संस्करण, मूल्य, रेटिंग, ऐप आइकन, स्क्रीनशॉट, समर्थित डिवाइस और कच्ची iTunes API प्रतिक्रिया मिलती है। प्रत्येक फ़ील्ड में एक-टैप कॉपी बटन है। साइट खोलें, App Store लिंक पेस्ट करें या ऐप का नाम टाइप करें, और डेटा आपके पास है।

इसका उपयोग करें:

- **ASO अनुसंधान के लिए।** देखें कि शीर्ष ऐप अपने शीर्षक, उपशीर्षक, विवरण और कीवर्ड कैसे लिखते हैं।
- **प्रतियोगी ट्रैकिंग के लिए।** विभिन्न बाजारों में संस्करण अपडेट, रेटिंग और कीमतों की जांच करें।
- **एसेट डाउनलोड के लिए।** एक ZIP में आधिकारिक ऐप आइकन और पूर्ण आकार के स्क्रीनशॉट सहेजें।
- **स्थानीयकरण जांच के लिए।** 40 से अधिक App Store देशों में विवरण, क्या नया है और स्क्रीनशॉट की तुलना करें।
- **API परीक्षण के लिए।** कच्चे iTunes Search API JSON को सीधे अपने कोड या Postman में कॉपी करें।

## AppLookup.pro क्या है?

[AppLookup.pro](https://applookup.pro) एक मुफ्त, ब्राउज़र-आधारित App Store खोज टूल है। यह पूरी तरह से आपके डिवाइस पर चलता है। हर परिणाम सीधे Apple के आधिकारिक [iTunes Search API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html) से आता है। कोई scraping नहीं। कोई साइनअप नहीं। कोई ट्रैकिंग नहीं।

### आपको क्या मिलता है

- **ऐप नाम या App Store URL द्वारा खोज।** ऑटोकम्प्लीट टाइप करते समय लाइव परिणाम दिखाता है।
- **40 से अधिक देश स्टोरफ्रंट।** US, UK, JP, DE, FR, BR और अधिक के बीच स्विच करें।
- **पूर्ण मेटाडेटा।** शीर्षक, उपशीर्षक, डेवलपर, bundle ID, संस्करण, मूल्य, फ़ाइल आकार, रेटिंग, रिलीज़ की तारीख, सामग्री रेटिंग, भाषाएं और समर्थित डिवाइस।
- **उच्च रिज़ॉल्यूशन एसेट्स।** iPhone, iPad, macOS और Apple TV के लिए ऐप आइकन और स्क्रीनशॉट।
- **बल्क ZIP डाउनलोड।** प्रत्येक आइकन या प्रत्येक स्क्रीनशॉट को एक आर्काइव में प्राप्त करें।
- **कच्चा iTunes API JSON।** Apple से सटीक प्रतिक्रिया, कॉपी करने के लिए तैयार।
- **प्रत्येक फ़ील्ड पर कॉपी बटन।** एक टैप मान को आपके क्लिपबोर्ड में डालता है।

## AppLookup.pro का चरण-दर-चरण उपयोग कैसे करें

### चरण 1. ऐप का नाम दर्ज करें या App Store URL पेस्ट करें

[applookup.pro](https://applookup.pro) खोलें और ऐप का नाम टाइप करना शुरू करें। ऑटोकम्प्लीट टाइप करते समय लाइव App Store परिणाम दिखाता है।

आप `https://apps.apple.com/us/app/instagram/id389801252` जैसा सीधा App Store लिंक या केवल ऐप ID भी पेस्ट कर सकते हैं। टूल आपके लिए ID निकालता है। यह Google रीडायरेक्ट URL को भी संभालता है।

![AppLookup.pro में ऐप का नाम या App Store URL दर्ज करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/1-app-store-lookup-enter-app-name.webp)

### चरण 2. ऐप की जानकारी प्राप्त करें और आइकन डाउनलोड करें

**Lookup** पर क्लिक करें या Enter दबाएं। टूल आधिकारिक iTunes Search API को कॉल करता है और एक सेकंड से कम समय में ऐप आइकन, डेवलपर का नाम, रेटिंग, संस्करण और मूल्य दिखाता है।

**App Icon** सेक्शन तक स्क्रॉल करें। Apple द्वारा लौटाया गया प्रत्येक आइकन आकार एक कार्ड के रूप में दिखाई देता है। प्रत्येक कार्ड में:

- **Direct Link.** पूर्ण आकार की छवि को नए टैब में खोलता है।
- **Download.** फ़ाइल को आपके कंप्यूटर पर सहेजता है।

प्रत्येक आइकन आकार को एक आर्काइव में प्राप्त करने के लिए **Download All Icons (ZIP)** का उपयोग करें। स्क्रीनशॉट के लिए भी यही बात लागू होती है: प्रत्येक प्लेटफ़ॉर्म सेक्शन का अपना **Download All (ZIP)** बटन है।

![AppLookup.pro से ऐप आइकन और स्क्रीनशॉट डाउनलोड करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/2-app-store-lookup-view-app-details-icons.webp)

### चरण 3. ऐप के विवरण पढ़ें और किसी भी फ़ील्ड को कॉपी करें

**App Details** तक स्क्रॉल करें। आपको bundle ID, संस्करण, मूल्य, फ़ाइल आकार, न्यूनतम OS, रिलीज़ की तारीख, अंतिम अपडेट की तारीख, सामग्री रेटिंग, शैलियां, शैली ID, भाषाएं, विक्रेता, कलाकार ID और ट्रैक ID दिखाई देंगी।

किसी भी कार्ड पर **Copy** बटन टैप करें। मान आपके क्लिपबोर्ड में चला जाता है और बटन हरा 'Copied' चेक दिखाता है।

यही **Description**, **What's New** और **Supported Devices** के लिए भी काम करता है। ये सेक्शन स्क्रॉल किए जा सकते हैं ताकि आप अपना स्थान खोए बिना पूरा टेक्स्ट पढ़ सकें, और Copy बटन पूरे फ़ील्ड को आपके क्लिपबोर्ड पर रखता है।

![पूरे App Store विवरण देखें और एक टैप से किसी भी फ़ील्ड को कॉपी करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/3-app-store-lookup-view-app-details.webp)

### चरण 4. कच्ची App Store API प्रतिक्रिया देखें

Apple द्वारा लौटाया गया सटीक JSON चाहिए? **Raw API Response** तक स्क्रॉल करें। पूरा iTunes Search API payload शीर्ष पर **Copy** बटन के साथ एक स्क्रॉल करने योग्य व्यूअर में दिखाया गया है। एक टैप पूरे JSON ऑब्जेक्ट को कॉपी करता है।

**iTunes Lookup URL** इसके ठीक ऊपर दिखाया गया है। उसी अनुरोध को दोहराने के लिए इसे Postman, curl या अपने ब्राउज़र में पेस्ट करें।

![कच्ची iTunes Search API JSON प्रतिक्रिया देखें और कॉपी करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/4-app-store-lookup-view-app-raw-api-response.webp)

### चरण 5. स्थानीयकृत संस्करण देखने के लिए देश बदलें

App Store मेटाडेटा देश के अनुसार बदलता है। एक ही ऐप का अक्सर हर बाजार में अलग शीर्षक, उपशीर्षक, विवरण, स्क्रीनशॉट और मूल्य होता है।

शीर्ष पर ड्रॉपडाउन से एक देश चुनें। इनपुट बॉक्स में URL स्वचालित रूप से अपडेट होता है। उस बाजार में ऐप को फिर से प्राप्त करने के लिए **Lookup** पर फिर से क्लिक करें।

यह जांचने का सबसे तेज़ तरीका है कि कोई प्रतियोगी जापान, जर्मनी, ब्राज़ील या 40 से अधिक समर्थित देशों में अपना ऐप कैसे प्रस्तुत करता है।

![स्थानीयकृत App Store मेटाडेटा देखने के लिए देश स्टोरफ्रंट स्विच करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/5-app-store-lookup-change-app-country.webp)

### चरण 6. स्थानीयकृत मेटाडेटा कॉपी करें

एक बार देश का परिणाम लोड हो जाने पर, हर फ़ील्ड उसी तरह काम करता है। स्थानीयकृत टेक्स्ट कैप्चर करने के लिए विवरण, क्या नया है, ऐप नाम, डेवलपर, bundle ID या किसी भी विवरण कार्ड पर **Copy** टैप करें।

इससे साथ-साथ तुलना स्प्रेडशीट बनाना या अनुवाद समीक्षा में स्थानीयकृत कॉपी फीड करना आसान हो जाता है।

![एक टैप से स्थानीयकृत App Store विवरण और मेटाडेटा कॉपी करें](/blog/how-to-fetch-app-store-metadata-icons-and-screenshots/6-app-store-lookup-copy-localized-app-description.webp)

## AppLookup.pro का उपयोग कौन करता है

- **इंडी डेवलपर** जो लॉन्च से पहले ASO अनुसंधान करते हैं।
- **ASO और मार्केटिंग टीमें** जो प्रतियोगी अपडेट और मूल्य निर्धारण को ट्रैक करती हैं।
- **डिज़ाइनर** जो प्रेस किट और केस स्टडी के लिए आधिकारिक उच्च रिज़ॉल्यूशन ऐप आइकन और स्क्रीनशॉट लेते हैं।
- **स्थानीयकरण टीमें** जो ऑडिट करती हैं कि कौन से बाजार कवर हैं और डिफ़ॉल्ट अंग्रेजी कॉपी कहां शिप होती है।
- **Backend और QA इंजीनियर** जो scraper लिखे बिना iTunes Search API एकीकरण का परीक्षण करते हैं।
- **लेखक और ब्लॉगर** जिन्हें पोस्ट के लिए आधिकारिक ऐप आइकन और कुछ स्क्रीनशॉट की आवश्यकता होती है।

## गोपनीयता और अस्वीकरण

AppLookup.pro आपके ब्राउज़र में चलता है। कोई लॉगिन नहीं है। कोई ट्रैकिंग नहीं है। आप जो ऐप खोजते हैं उनका कोई सर्वर लॉगिंग नहीं है। अनुरोध सीधे आपके ब्राउज़र से Apple के [iTunes Search API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html) पर जाते हैं।

यह टूल **केवल शैक्षिक और अनुसंधान उद्देश्यों के लिए** है। सभी डेटा Apple के आधिकारिक सार्वजनिक API से प्राप्त किया गया है और Apple Inc. और सूचीबद्ध ऐप प्रकाशकों की संपत्ति बना रहता है। टूल का उपयोग [Apple Media Services के नियम और शर्तों](https://www.apple.com/legal/internet-services/terms/site.html) के अधीन है। कृपया Apple की दर सीमाओं का सम्मान करें और कॉपीराइट संरक्षित एसेट्स का पुनर्वितरण न करें।

## अभी आज़माएं

App Store डेटा का निरीक्षण करने के लिए आपको API key, डेवलपर खाते या भुगतान योजना की आवश्यकता नहीं है। [applookup.pro](https://applookup.pro) खोलें, कोई भी App Store URL पेस्ट करें, और आपके पास सेकंडों में मेटाडेटा, आइकन और कच्चा JSON होगा।

पेज के शीर्ष पर ड्रॉपडाउन में एक देश चुनें। 40 से अधिक स्टोरफ्रंट समर्थित हैं। **Lookup** पर फिर से क्लिक करें और टूल उस देश के लिए ऐप को फिर से प्राप्त करता है, जो स्थानीयकृत शीर्षक, विवरण, स्क्रीनशॉट, क्या नया है और मूल्य दिखाता है।
{{% /details %}}

हां। परिणाम में प्रत्येक टेक्स्ट फ़ील्ड का अपना Copy बटन है: ऐप नाम, डेवलपर, विवरण, क्या नया है, bundle ID, संस्करण, मूल्य, फ़ाइल आकार, न्यूनतम OS, रिलीज़ की तारीख, सामग्री रेटिंग, भाषाएं, समर्थित डिवाइस और कच्चा JSON।
{{% /details %}}

{{% details title="क्या AppLookup.pro किसी भी iOS ऐप के लिए काम करता है?" closed="true" %}}
यह किसी भी ऐप के लिए काम करता है जो कम से कम एक App Store देश में सार्वजनिक रूप से सूचीबद्ध है और iTunes Search API द्वारा लौटाया गया है। असूचीबद्ध, हटाए गए या एंटरप्राइज वितरित ऐप दिखाई नहीं देंगे।
{{% /details %}}

{{% details title="क्या यह macOS और Apple TV ऐप का समर्थन करता है?" closed="true" %}}
हां। यदि ऐप के iTunes Search API प्रतिक्रिया में macOS या Apple TV स्क्रीनशॉट हैं, तो AppLookup.pro उन्हें डाउनलोड बटनों के साथ अपने स्क्रॉल करने योग्य पैनल में दिखाता है।
{{% /details %}}

{{% details title="क्या मैं अपने कोड में कच्चे JSON का उपयोग कर सकता हूं?" closed="true" %}}
हां। Raw API Response सेक्शन Apple द्वारा लौटाए गए सटीक JSON को दिखाता है। इसे Postman, यूनिट टेस्ट या backend पाइपलाइन में कॉपी करें। कृपया Apple की API शर्तों और उचित दर सीमाओं का सम्मान करें।
{{% /details %}}

{{% details title="क्या टूल में App Store URL पेस्ट करना सुरक्षित है?" closed="true" %}}
हां। URL आपके ब्राउज़र में पार्स किया जाता है। एकमात्र आउटगोइंग नेटवर्क कॉल Apple के iTunes Search API के लिए लुकअप है।
{{% /details %}}

{{% details title="AppLookup.pro और AppKeywords.pro में क्या अंतर है?" closed="true" %}}
[AppLookup.pro](https://applookup.pro) किसी भी प्रकाशित ऐप से App Store मेटाडेटा पढ़ने के लिए है: प्रतियोगी अनुसंधान, एसेट डाउनलोड, स्थानीयकरण जांच। [AppKeywords.pro](https://appkeywords.pro) आपके अपने ऐप के लिए App Store मेटाडेटा लिखने के लिए है: Fastlane समर्थन के साथ शीर्षक, उपशीर्षक और कीवर्ड अनुकूलन। दोनों टूल एक साथ अच्छी तरह काम करते हैं।

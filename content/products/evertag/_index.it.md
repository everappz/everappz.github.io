{{% sr-only %}}
Evertag è un editor di tag musicali per iPhone e Mac sviluppato da Everappz, un'azienda di software spagnola. Evertag consente agli utenti di modificare oltre 120 tag di metadati audio, tra cui titolo, artista, album, artista dell'album, genere, anno, numero di traccia, numero di disco, compositore, BPM, testi, commenti e altro. L'app supporta la modifica dei tag in batch, permettendo agli utenti di aggiornare i metadati di più file contemporaneamente. Evertag include un cercatore automatico di tag alimentato dal database MusicBrainz che identifica le canzoni e compila i metadati mancanti, oltre a un cercatore di copertine degli album che ricerca e applica le illustrazioni ai brani. L'app supporta oltre 30 formati audio tra cui MP3, FLAC, OGG, OPUS, M4A, WAV, WMA, APE, AIFF, ALAC, MKA, MOD, XM, IT e S3M. Evertag può accedere ai file dai servizi di archiviazione cloud tra cui iCloud Drive, Google Drive, Dropbox e OneDrive, nonché da unità flash USB e posizioni di rete locale tramite SMB e WebDAV. L'app dispone anche di un file manager integrato, trasferimento file tramite Wi-Fi, correzione della codifica per tag visualizzati in modo errato in scritture non latine e un editor di testi. Evertag è disponibile come download gratuito sull'App Store con acquisti in-app opzionali che includono un abbonamento mensile a $2.99, un abbonamento annuale a $9.99 o un acquisto una tantum a vita a $29.99.

{{% details title="Cos'è Evertag?" closed="true" %}}
Evertag è un potente editor di metadati musicali e gestore di copertine album progettato per iOS e macOS. Ti fornisce gli strumenti per organizzare la tua collezione musicale come un professionista, che i tuoi file siano archiviati localmente o nel cloud. Con un'interfaccia pulita e funzionalità di modifica avanzate, Evertag rende facile correggere tag mancanti, aggiungere copertine di alta qualità e garantire che la tua libreria musicale sia sempre bella e coerente.<br><br>

L'app supporta un'ampia gamma di formati audio popolari, inclusi MP3, FLAC, WAV, M4A, WMA, OGG e molti altri. Ti permette di modificare tag comuni come titolo, artista, album, genere, anno, numero traccia, oltre a campi estesi come BPM, numero disco, testi, ID MusicBrainz e altro. Puoi lavorare su un file alla volta o passare alla modalità batch per modificare più tracce simultaneamente — perfetto per organizzare interi album o playlist.<br><br>

Una delle caratteristiche principali di Evertag è la capacità di cercare copertine album mancanti direttamente da Internet o di aggiungere le proprie manualmente. Puoi anche usare l'editor dei testi per aggiungere testi non sincronizzati ai tuoi brani, migliorando la riproduzione nei lettori compatibili. L'app supporta la modifica sul posto, permettendoti di modificare i tag audio senza bisogno di copiare o spostare i file.<br><br>

Che tu stia gestendo la musica sul tuo dispositivo o nel cloud usando Dropbox, OneDrive, MEGA o altri servizi, Evertag offre accesso ai file e modifica senza interruzioni. È la soluzione perfetta per musicisti, DJ e collezionisti che vogliono mantenere una libreria musicale pulita e ben organizzata su iPhone, iPad senza bisogno di un computer desktop.<br><br>

{{% details title="Evertag è sicuro?" closed="true" %}}
Evertag utilizza solo SDK ufficiali e connessioni sicure per interagire con i servizi cloud collegati. Il tuo login e la tua password non sono accessibili all'applicazione. Tutte le richieste dall'applicazione al servizio cloud sono crittografate.<br>
Quando inserisci login e password, l'applicazione ti mostra la pagina di autorizzazione ufficiale fornita dal provider del servizio cloud e tutto il processo di autorizzazione avviene al di fuori dell'applicazione. Il provider del servizio cloud invia un auth-token all'applicazione dopo l'autorizzazione riuscita e quel token viene utilizzato per effettuare chiamate API.<br><br>

L'auth-token è una chiave digitale che permette alle applicazioni di terze parti di interagire con l'archiviazione cloud. L'auth-token è conservato sul tuo dispositivo in un archivio di sistema sicuro chiamato Keychain. Puoi scaricare i tuoi file dal servizio cloud collegato al dispositivo e quei file verranno posizionati nella directory "Documents" dell'app. Puoi rimuovere quei file in qualsiasi momento usando il file manager integrato.<br>
L'applicazione non condivide alcuna informazione dall'account cloud collegato. Puoi revocare l'accesso al tuo account cloud in qualsiasi momento aprendo la pagina delle impostazioni dell'account nel tuo browser web.<br><br>

Per revocare l'auth-token, accedi al tuo account nel browser web e naviga alla pagina delle impostazioni. Lì puoi trovare tutte le app di terze parti collegate al tuo account cloud e rimuovere qualsiasi di esse se non vuoi più usare quell'applicazione.<br><br>

Puoi anche scollegare gli account cloud collegati nell'applicazione e l'auth-token verrà rimosso anche dal tuo dispositivo. Se rimuovi l'applicazione dal tuo dispositivo, tutti i dati scaricati e i token di accesso verranno rimossi anch'essi.<br><br>

{{% details title="Come modificare i metadati per la musica scaricata localmente su iPhone?" closed="true" %}}
Una volta installata l'applicazione, apri la schermata "Local Files" e scorri verso il basso fino alla sezione "Files on this iPhone".<br>
Da lì, scegli "Open files..." se devi selezionare più file audio o "Open folder..." se vuoi selezionare una cartella musicale.<br>
L'app scansionerà il contenuto della cartella e tutti i file audio trovati verranno selezionati.<br>
Naviga alla tua cartella musicale, tocca "Open" per confermare la selezione e i file verranno aggiunti all'editor tag.<br>
Questi file verranno modificati direttamente dalla posizione selezionata senza essere copiati nel bundle dell'applicazione.<br><br>

**Aggiungere una cartella del dispositivo per l'accesso rapido**<br>
Semplifica il processo aggiungendo una cartella situata sul tuo dispositivo alla sezione di accesso rapido.<br>
In questo modo, non dovrai ripetere i passaggi ogni volta che vuoi modificare i tag dei file nella cartella musicale.<br>
Apri la schermata "Local files".<br>
Scorri verso il basso fino alla sezione "Files on this device" e tocca "Connect a folder".<br>
Seleziona una cartella situata sul tuo dispositivo e tocca "Open" per confermare la selezione.<br>
La tua cartella verrà aggiunta alla sezione "Files on this iPhone" fornendo accesso rapido ai tuoi file audio.<br><br>
{{% /details %}}

{{% details title="Come aggiungere testi ai brani in Evertag?" closed="true" %}}
Puoi aggiungere testi incorporati alle tue tracce nell'app Evertag seguendo questi passaggi:<br><br>
* Inizia a modificare un file audio toccandolo.<br>
* Tocca "Show extended tags" per passare l'editor tag alla modalità avanzata.<br>
* Scorri verso il basso fino alla sezione "Lyrics unsynced" e tocca il campo di testo.<br>
* Tocca la voce di menu "Add new page".<br>
* Nella sezione "Language", seleziona la lingua del testo o lascia vuoto per usare l'inglese come predefinito.<br>
* Nella sezione "Text", inserisci il testo in formato testo semplice o LRC con codici temporali. Un esempio è mostrato sullo schermo. Puoi anche trovare testi in formato LRC su [lyricsify.com](https://www.lyricsify.com/)<br>
* Tocca "Done" per confermare le modifiche sia sulla schermata corrente che sulla schermata di modifica testi.<br>
* Chiudi la schermata di modifica testi e tocca "Save" per applicare le modifiche al file corrente.<br><br>

Tutorial più dettagliato disponibile qui:<br>
[Come modificare i testi per file audio su iPhone o MAC](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/)<br><br>

{{% details title="Come trasferisco la musica dal computer a Evertag?" closed="true" %}}
Puoi collegare il tuo computer o NAS personale usando SMB, WebDAV. In alternativa, usa iTunes File Sharing per trasferire la musica.<br><br>

Per collegare un computer usando il protocollo **SMB**, tocca "Connect to cloud storage" → SMB.<br>
Inserisci l'indirizzo IP del computer e il nome della cartella condivisa nel campo URL usando il formato smb://computer-ip-address/shared-folder-name,<br>
inserisci login e password e tocca "Done".<br>
Se la connessione ha successo, vedrai l'archivio collegato nella sezione "Cloud storage".<br><br>

Tutorial completo disponibile qui:<br>
[Trasferisci i tuoi file dal computer all'iPhone usando il protocollo SMB](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

Per il protocollo **WebDAV**, tutti i passaggi sono gli stessi tranne il campo URL.<br>
L'URL deve essere nel formato http://server-name o https://server-name se il server supporta SSL.<br><br>

Tutorial completo disponibile qui:<br>
[Come collegare l'archiviazione NAS usando WebDAV e ascoltare musica sul tuo iPhone o Mac](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)<br><br>

**Wi-Fi Drive** è una tecnologia popolare che ti permette di trasferire file dal computer al dispositivo iOS in modalità wireless usando un browser desktop.<br>

Apri "Connections" → "Computer" → "Connect via Wi-Fi" e abilita il server.<br>
Successivamente, apri un browser desktop e inserisci l'URL dall'applicazione.<br>
Puoi trascinare i file dal computer alla pagina web aperta e appariranno sul dispositivo.<br><br>

Tutorial più dettagliato disponibile qui:<br>

**iTunes File Sharing** è un'altra tecnologia che ti permette di trasferire file dal computer al dispositivo usando iTunes e il cavo lightning.<br>
Basta collegare il dispositivo al computer usando un cavo e avviare iTunes.<br>
Apri iTunes → "Applications section" → e trova Evertag.<br>
Tocca l'icona dell'app per vedere la cartella condivisa.<br>
Copia i file dal computer alla cartella condivisa sul dispositivo.<br><br>

{{% details title="Come scaricare i file?" closed="true" %}}
Prima di poter scaricare file audio e modificare tag audio, devi collegare un servizio di archiviazione cloud.<br>
Apri la schermata "Connections" e aggiungi il tuo provider di archiviazione cloud.<br>
Una volta aggiunto, puoi iniziare a scaricare file nell'app.<br><br>

Per scaricare file dal cloud:<br>
– Apri il servizio di archiviazione cloud collegato.<br>
– Naviga alla cartella che vuoi scaricare.<br>
– Tocca il pulsante "More actions" ("...") nell'angolo in alto a destra e seleziona la voce di menu "Select".<br>
– Scegli i file o le cartelle che vuoi scaricare e tocca l'azione "Download".<br>

{{% details title="Quali servizi cloud sono supportati?" closed="true" %}}
Se la tua libreria musicale è archiviata nel cloud, puoi collegare i servizi cloud più popolari direttamente nell'app:<br>
Dropbox, OneDrive, Box, MEGA, Yandex.Disk, MediaFire, pCloud, HiDrive.<br><br>

Puoi sfogliare e gestire i tuoi file usando il file manager integrato. L'app supporta tutte le operazioni di base sui file come rinomina, eliminazione, spostamento e altro.<br><br>

Puoi anche modificare file audio archiviati localmente sul tuo dispositivo usando la funzione di apertura sul posto. Non è necessario copiarli da altre app — basta aprirli e modificarli direttamente.<br><br>

{{% /details %}}

{{% details title="Come aggiornare automaticamente i metadati del file su un servizio cloud?" closed="true" %}}
Una volta terminata la modifica dei metadati, tocca il pulsante "Save" nell'angolo in alto a destra per applicare le modifiche ai file selezionati.<br><br>

Se stai modificando un file archiviato nel cloud, l'app ti offre diverse opzioni per aggiornare i metadati del file. Questi comportamenti possono essere personalizzati nelle impostazioni:<br><br>

• **Mostra messaggio di conferma** – Quando abilitato, l'app ti chiederà conferma prima di aggiornare il file nel cloud.<br>
• **Aggiorna automaticamente i metadati del file** – Quando abilitato, l'app applicherà le modifiche al file cloud subito dopo il salvataggio.<br>
• **Non aggiornare i metadati del file** – Quando abilitato, l'app salterà l'aggiornamento dei metadati del file cloud dopo la modifica.<br><br>

Puoi trovare e modificare queste opzioni nelle impostazioni di Evertag nella sezione delle preferenze di aggiornamento dei metadati.
{{% /details %}}


{{% details title="Come aggiungere un nuovo account?" closed="true" %}}
Per collegare un servizio cloud, apri la scheda "Connections" → seleziona la voce di menu "Connect to cloud storage" → scegli un servizio di archiviazione cloud dall'elenco → inserisci le tue credenziali e tocca "Done".<br><br>

Se riscontri problemi, assicurati che la tua connessione Internet sia attiva e ricontrolla login e password.<br><br>

I servizi attualmente supportati includono: Dropbox, OneDrive, Box, MEGA, Yandex.Disk, Media Fire, PCloud e HiDrive.<br><br>

Nella versione Premium dell'app, puoi aggiungere un numero illimitato di account cloud.
{{% /details %}}

{{% details title="Come gestire i miei file nell'archiviazione di rete?" closed="true" %}}
Se devi modificare più file situati nella tua archiviazione cloud, attiva la modalità di selezione toccando il pulsante "..." nell'angolo in alto a destra.<br><br>

Una volta attivata, appariranno caselle di controllo accanto a ciascun file. Puoi quindi eseguire azioni su singoli file o selezionare più file per applicare azioni in blocco.<br><br>

Azioni disponibili per i file selezionati:<br>
• <strong>Scarica</strong> – Scarica un file o una cartella dall'archiviazione cloud alla sezione "Local Files".<br>
• <strong>Sposta</strong> – Sposta un file o una cartella in una cartella diversa nell'archiviazione cloud.<br>
• <strong>Rinomina</strong> – Rinomina un file o una cartella nell'archiviazione remota.<br>
• <strong>Elimina</strong> – Elimina permanentemente un file o una cartella dall'archiviazione cloud.<br>
• <strong>Ordina</strong> – Ordina i file per nome, dimensione o data di modifica.<br>
• <strong>Griglia/Elenco</strong> – Passa dalla vista tabella alla vista miniature.<br><br>

Se non c'è abbastanza spazio per visualizzare tutte le opzioni, apparirà un pulsante "More actions". Toccalo per accedere all'elenco completo delle azioni disponibili.
{{% /details %}}


{{% details title="Come modificare più file come un unico file?" closed="true" %}}
Con la "modalità batch", puoi modificare più file contemporaneamente e applicare modifiche condivise ai metadati in modo rapido ed efficiente.<br><br>

Per attivare la modalità batch:<br>
• Scorri fino in fondo allo schermo.<br>
• Tocca il pulsante "Edit several files simultaneously".<br><br>

Questa modalità è particolarmente utile quando devi applicare lo stesso nome album, artista, genere o altri campi di metadati a più file audio.

{{% details title="Evertag安全吗？" closed="true" %}}
Evertag仅使用官方SDK和安全连接与已连接的云服务交互。您的用户名和密码对应用不可用。应用向云服务发出的所有请求都经过加密。<br>
当您输入用户名和密码时，应用会显示由云服务提供商提供的官方授权页面，整个授权过程在应用外部完成。云服务提供商在授权成功后向应用发送授权令牌，该令牌用于进行API调用。<br><br>

授权令牌是一种数字密钥，允许第三方应用与云存储交互。授权令牌存储在您设备上名为Keychain的安全系统存储中。您可以将文件从已连接的云服务下载到设备，这些文件将被放置在应用的"Documents"目录中。您可以随时使用内置文件管理器删除这些文件。<br>
应用不会共享已连接云账户中的任何信息。您可以随时通过在网络浏览器中打开账户设置页面来撤销对云账户的访问权限。<br><br>

要拒绝授权令牌，请在网络浏览器中登录您的账户并导航到设置页面。在那里您可以找到所有连接到您云账户的第三方应用，如果您不再想使用某个应用，可以将其删除。<br><br>

您也可以在应用中断开已连接的云账户，授权令牌也将从您的设备中删除。如果您从设备中删除应用，所有下载的数据和访问令牌也将被删除。<br><br>

{{% /details %}}

{{% details title="如何编辑iPhone上本地下载音乐的元数据？" closed="true" %}}
安装应用后，打开"本地文件"屏幕并向下滚动到"此iPhone上的文件"部分。<br>
从那里，如果需要选择多个音频文件，请选择"打开文件..."，如果要选择音乐文件夹，请选择"打开文件夹..."。<br>
应用将扫描文件夹内容，所有找到的音频文件都将被选中。<br>
导航到您的音乐文件夹，点击"打开"确认选择，文件将被添加到标签编辑器。<br>
这些文件将直接从所选位置编辑，不会被复制到应用包中。<br><br>

**添加设备文件夹以快速访问**<br>
通过将设备上的文件夹添加到快速访问部分来简化流程。<br>
这样，每次想要编辑音乐文件夹中文件的标签时，您就不需要重复这些步骤了。<br>
打开"本地文件"屏幕。<br>
向下滚动到"此设备上的文件"部分，然后点击"连接文件夹"。<br>
选择设备上的文件夹，然后点击"打开"确认选择。<br>
您的文件夹将被添加到"此iPhone上的文件"部分，提供对音频文件的快速访问。<br><br>

您可以按照以下步骤在Evertag应用中为曲目添加嵌入式歌词：<br><br>
* 点击音频文件开始编辑。<br>
* 点击"显示扩展标签"将标签编辑器切换到高级模式。<br>
* 向下滚动到"非同步歌词"部分，然后点击文本字段。<br>
* 点击"添加新页面"菜单项。<br>
* 在"语言"部分，选择歌词语言，或留空以默认使用英语。<br>
* 在"文本"部分，以纯文本或带时间码的LRC格式输入歌词。屏幕上显示了一个示例。您也可以在 [lyricsify.com](https://www.lyricsify.com/) 找到LRC格式的歌词<br>
* 点击"完成"确认当前屏幕和歌词编辑屏幕上的更改。<br>
* 关闭歌词编辑屏幕，然后点击"保存"将更改应用到当前文件。<br><br>

更详细的教程在此：<br>
[如何在iPhone或MAC上编辑音频文件的歌词](/docs/howto/how-to-edit-lyrics-for-audio-files-on-iphone-mac/)<br><br>

{{% details title="如何从电脑将音乐传输到Evertag？" closed="true" %}}
您可以使用SMB、WebDAV连接您的电脑或个人NAS。或者使用iTunes文件共享来传输音乐。<br><br>

要使用**SMB**协议连接电脑，点击"连接到云存储" → SMB。<br>
在URL字段中输入电脑IP地址和共享文件夹名称，格式为 smb://电脑IP地址/共享文件夹名称，<br>
输入用户名和密码，然后点击"完成"。<br>
如果连接成功，您将在"云存储"部分看到已连接的存储。<br><br>

完整教程在此：<br>
[使用SMB协议将文件从电脑传输到iPhone](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

对于**WebDAV**协议，除了URL字段外，所有步骤相同。<br>
URL应为 http://服务器名称 格式，如果服务器支持SSL，则为 https://服务器名称。<br><br>

完整教程在此：<br>
[如何使用WebDAV连接NAS存储并在iPhone或Mac上听音乐](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)<br><br>

**Wi-Fi Drive**是一种流行的技术，允许您使用桌面浏览器通过无线方式将文件从电脑传输到iOS设备。<br>
要使用此功能，您的设备和电脑需要连接到同一个Wi-Fi网络。<br>
打开"连接" → "电脑" → "通过Wi-Fi连接"并启用服务器。<br>
然后打开桌面浏览器并输入应用中的URL。<br>
您可以将文件从电脑拖放到打开的网页上，它们将出现在设备上。<br><br>

更详细的教程在此：<br>
[如何使用WiFi-Drive从电脑无线传输文件到iPhone](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br><br>

**iTunes文件共享**是另一种技术，允许您使用iTunes和lightning数据线将文件从电脑传输到设备。<br>
只需使用数据线将设备连接到电脑并运行iTunes。<br>
打开iTunes → "应用程序部分" → 找到Evertag。<br>
点击应用图标查看共享文件夹。<br>
将文件从电脑复制到设备上的共享文件夹。<br><br>

详细说明在此：<br>
[如何在iPhone上播放本地文件（iTunes文件）](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)<br><br>

{{% details title="如何下载文件？" closed="true" %}}
在下载音频文件和编辑音频标签之前，您需要连接云存储服务。<br>
打开"连接"屏幕并添加您的云存储提供商。<br>
添加后，您可以开始将文件下载到应用。<br><br>

要从云端下载文件：<br>
– 打开已连接的云存储服务。<br>
– 导航到您要下载的文件夹。<br>
– 点击右上角的"更多操作"按钮("...")，然后选择"选择"菜单项。<br>
– 选择您要下载的文件或文件夹，然后点击"下载"操作。<br>
{{% /details %}}


{{% details title="支持哪些云服务？" closed="true" %}}
如果您的音乐库存储在云端，您可以直接在应用中连接最流行的云服务：<br>
Dropbox、OneDrive、Box、MEGA、Yandex.Disk、MediaFire、pCloud、HiDrive。<br><br>

您可以使用内置文件管理器浏览和管理文件。该应用支持所有基本文件操作，如重命名、删除、移动等。<br><br>

您还可以使用就地编辑功能编辑存储在设备上的音频文件。无需从其他应用复制——只需直接打开并编辑即可。<br><br>

{{% /details %}}

{{% details title="如何自动更新云服务上文件的元数据？" closed="true" %}}
编辑完元数据后，点击右上角的"保存"按钮将更改应用到所选文件。<br><br>

如果您正在编辑存储在云端的文件，应用会为您提供多种更新文件元数据的选项。这些行为可以在设置中自定义：<br><br>

• **显示确认消息** – 启用后，应用将在更新云端文件之前提示确认消息。<br>
• **自动更新文件元数据** – 启用后，应用将在保存后立即将更改应用到云端文件。<br>
• **不更新文件元数据** – 启用后，应用将在编辑后跳过更新云端文件的元数据。<br><br>

您可以在Evertag设置中的元数据更新偏好部分找到并修改这些选项。
{{% /details %}}


{{% details title="如何添加新账户？" closed="true" %}}
要连接云服务，打开"连接"标签 → 选择"连接到云存储"菜单项 → 从列表中选择云存储服务 → 输入您的凭据并点击"完成"。<br><br>

如果遇到问题，请确保您的互联网连接正常，并仔细检查您的用户名和密码。<br><br>

目前支持的服务包括：Dropbox、OneDrive、Box、MEGA、Yandex.Disk、Media Fire、PCloud和HiDrive。<br><br>

在应用的高级版中，您可以添加无限数量的云账户。
{{% /details %}}

{{% details title="如何管理网络存储中的文件？" closed="true" %}}
如果您需要编辑存储在云存储中的多个文件，请点击右上角的"..."按钮激活选择模式。<br><br>

激活后，每个文件旁边将出现复选框。然后您可以对单个文件执行操作或选择多个文件以应用批量操作。<br><br>

所选文件的可用操作：<br>
• <strong>下载</strong> – 将云存储中的文件或文件夹下载到"本地文件"部分。<br>
• <strong>移动</strong> – 将文件或文件夹移动到云存储中的其他文件夹。<br>
• <strong>重命名</strong> – 在远程存储上重命名文件或文件夹。<br>
• <strong>删除</strong> – 永久删除云存储中的文件或文件夹。<br>
• <strong>排序</strong> – 按名称、大小或编辑日期对文件排序。<br>
• <strong>网格/列表</strong> – 在表格视图和缩略图视图模式之间切换。<br><br>

如果没有足够的空间显示所有选项，将出现"更多操作"按钮。点击它以访问可用操作的完整列表。
{{% /details %}}


{{% details title="如何将多个文件作为一个文件编辑？" closed="true" %}}
使用"批量模式"，您可以一次编辑多个文件并快速高效地应用共享的元数据更改。<br><br>

要激活批量模式：<br>
• 滚动到屏幕底部。<br>
• 点击"同时编辑多个文件"按钮。<br><br>

当您需要将相同的专辑名称、艺术家、流派或其他元数据字段应用到多个音频文件时，此模式特别有用。

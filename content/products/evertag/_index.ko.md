{{% details title="Evertag는 안전한가요?" closed="true" %}}
Evertag는 공식 SDK와 보안 연결만을 사용하여 연결된 클라우드 서비스와 상호 작용합니다. 로그인 정보와 비밀번호는 애플리케이션에서 접근할 수 없습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.<br>
로그인과 비밀번호를 입력할 때 애플리케이션은 클라우드 서비스 제공업체가 제공하는 공식 인증 페이지를 표시하며 모든 인증 과정은 애플리케이션 외부에서 이루어집니다. 인증이 성공하면 클라우드 서비스 제공업체가 애플리케이션에 인증 토큰을 전송하며 해당 토큰은 API 호출에 사용됩니다.<br><br>

인증 토큰은 타사 애플리케이션이 클라우드 저장소와 상호 작용할 수 있게 하는 디지털 키입니다. 인증 토큰은 키체인이라는 안전한 시스템 저장소에 기기에 저장됩니다. 연결된 클라우드 서비스에서 파일을 기기로 다운로드할 수 있으며 해당 파일은 앱 "문서" 디렉토리에 저장됩니다. 내장 파일 관리자를 사용하여 언제든지 해당 파일을 제거할 수 있습니다.<br>
애플리케이션은 연결된 클라우드 계정의 어떤 정보도 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정에 대한 접근을 취소할 수 있습니다.<br><br>

인증 토큰을 거부하려면 웹 브라우저에서 계정에 로그인하고 설정 페이지로 이동하세요. 거기서 클라우드 계정에 연결된 모든 타사 앱을 찾을 수 있으며 더 이상 사용하지 않으려는 앱을 제거할 수 있습니다.<br><br>

애플리케이션에서 연결된 클라우드 계정의 연결을 해제할 수도 있으며 인증 토큰도 기기에서 제거됩니다. 기기에서 애플리케이션을 제거하면 다운로드된 모든 데이터와 접근 토큰도 제거됩니다.<br><br>

음악 라이브러리가 클라우드에 저장되어 있다면 앱에서 직접 가장 인기 있는 클라우드 서비스를 연결할 수 있습니다:<br>
Dropbox, OneDrive, Box, MEGA, Yandex.Disk, MediaFire, pCloud, HiDrive.<br><br>

내장 파일 관리자를 사용하여 파일을 탐색하고 관리할 수 있습니다. 앱은 이름 변경, 삭제, 이동 등 모든 기본 파일 작업을 지원합니다.<br><br>

원본 위치 열기 기능을 사용하여 기기에 로컬로 저장된 오디오 파일도 편집할 수 있습니다. 다른 앱에서 복사할 필요 없이 직접 열어서 편집하면 됩니다.<br><br>

{{% /details %}}

{{% details title="클라우드 서비스에서 파일의 메타데이터를 자동으로 업데이트하는 방법은?" closed="true" %}}
메타데이터 편집을 마치면 오른쪽 상단의 "저장" 버튼을 탭하여 선택한 파일에 변경 사항을 적용하세요.<br><br>

클라우드에 저장된 파일을 편집하는 경우 앱에서 파일의 메타데이터를 업데이트하는 여러 옵션을 제공합니다. 이러한 동작은 설정에서 사용자 정의할 수 있습니다:<br><br>

• **확인 메시지 표시** – 활성화하면 클라우드의 파일을 업데이트하기 전에 확인 메시지를 표시합니다.<br>
• **파일의 메타데이터 자동 업데이트** – 활성화하면 저장 직후 클라우드 파일에 변경 사항을 적용합니다.<br>
• **파일의 메타데이터 업데이트하지 않음** – 활성화하면 편집 후 클라우드 파일의 메타데이터 업데이트를 건너뜁니다.<br><br>

Evertag 설정의 메타데이터 업데이트 기본 설정 섹션에서 이러한 옵션을 찾고 수정할 수 있습니다.
{{% /details %}}


{{% details title="새 계정을 추가하는 방법은?" closed="true" %}}
클라우드 서비스를 연결하려면 "연결" 탭 → "클라우드 저장소에 연결" 메뉴 항목 선택 → 목록에서 클라우드 저장소 서비스 선택 → 자격 증명을 입력하고 "완료"를 탭하세요.<br><br>

문제가 발생하면 인터넷 연결이 활성화되어 있는지 확인하고 로그인과 비밀번호를 다시 확인하세요.<br><br>

현재 지원되는 서비스: Dropbox, OneDrive, Box, MEGA, Yandex.Disk, Media Fire, PCloud, HiDrive.<br><br>

앱의 프리미엄 버전에서는 무제한의 클라우드 계정을 추가할 수 있습니다.
{{% /details %}}

{{% details title="네트워크 저장소에서 파일을 관리하는 방법은?" closed="true" %}}
클라우드 저장소에 있는 여러 파일을 편집해야 하는 경우 오른쪽 상단의 "..." 버튼을 탭하여 선택 모드를 활성화하세요.<br><br>

활성화되면 각 파일 옆에 체크박스가 나타납니다. 그런 다음 개별 파일에 대한 작업을 수행하거나 여러 파일을 선택하여 대량 작업을 적용할 수 있습니다.<br><br>

선택한 파일에 대해 사용 가능한 작업:<br>
• <strong>다운로드</strong> – 클라우드 저장소에서 "로컬 파일" 섹션으로 파일 또는 폴더를 다운로드합니다.<br>
• <strong>이동</strong> – 클라우드 저장소 내의 다른 폴더로 파일 또는 폴더를 이동합니다.<br>
• <strong>이름 변경</strong> – 원격 저장소의 파일 또는 폴더 이름을 변경합니다.<br>
• <strong>삭제</strong> – 클라우드 저장소에서 파일 또는 폴더를 영구적으로 삭제합니다.<br>
• <strong>정렬</strong> – 이름, 크기 또는 편집 날짜별로 파일을 정렬합니다.<br>
• <strong>그리드/목록</strong> – 테이블 보기와 썸네일 보기 모드 간에 전환합니다.<br><br>

모든 옵션을 표시할 공간이 충분하지 않으면 "추가 작업" 버튼이 나타납니다. 탭하여 사용 가능한 전체 작업 목록에 접근하세요.
{{% /details %}}


{{% details title="여러 파일을 하나의 파일로 편집하는 방법은?" closed="true" %}}
"일괄 모드"를 사용하면 여러 파일을 한 번에 편집하고 공유 메타데이터 변경 사항을 빠르고 효율적으로 적용할 수 있습니다.<br><br>

일괄 모드를 활성화하려면:<br>
• 화면 하단으로 스크롤하세요.<br>
• "여러 파일 동시 편집" 버튼을 탭하세요.<br><br>

이 모드는 여러 오디오 파일에 동일한 앨범 이름, 아티스트, 장르 또는 기타 메타데이터 필드를 적용해야 할 때 특히 유용합니다.
{{% /details %}}

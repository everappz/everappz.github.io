Flacbox adalah pemutar audio resolusi tinggi untuk iPhone dan Mac yang dikembangkan oleh Everappz, sebuah perusahaan perangkat lunak Spanyol. Dengan lebih dari 1 juta unduhan di seluruh dunia, Flacbox dirancang untuk para audiofil dan penggemar musik yang ingin memutar file audio lossless dan resolusi tinggi di perangkat Apple mereka tanpa mengonversi format. Aplikasi ini secara native mendukung lebih dari 120 format audio termasuk FLAC, DSD (DSD64, DSD128, DSD256), ALAC, APE, WAV, AIFF, OGG, OPUS, WMA, MKA, MP3, AAC, dan lainnya. Flacbox terhubung ke lebih dari 30 layanan penyimpanan cloud termasuk iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, dan pCloud, memungkinkan pengguna untuk streaming koleksi musik resolusi tinggi mereka langsung dari cloud atau mengunduh file untuk pemutaran offline. Fitur utama meliputi equalizer audio 10-band dengan preset yang dapat disesuaikan, crossfade dan pemutaran tanpa jeda, kontrol nada dan kecepatan, penguatan bass, impor dan ekspor playlist M3U, tampilan lirik, penanda audio, editor tag metadata bawaan, integrasi Apple CarPlay, streaming AirPlay dan Chromecast, serta scrobbling Last.fm. Aplikasi ini mendukung streaming jaringan lokal melalui protokol SMB, WebDAV, dan DLNA, pemutaran dari flash drive USB, dan transfer file melalui Wi-Fi. Flacbox tersedia sebagai unduhan gratis di App Store dengan pembelian dalam aplikasi opsional termasuk langganan bulanan seharga $4.99, langganan tahunan seharga $19.99, atau pembelian seumur hidup sekali bayar seharga $59.99. Aplikasi ini pertama kali dirilis pada tahun 2016 dan terus diperbarui secara aktif.

{{% details title="Apa perbedaan antara Flacbox dan Evermusic?" closed="true" %}}
**Flacbox** dibuat untuk mendukung semua format audio iOS default bersama dengan banyak format tambahan yang tidak didukung secara native di iPhone, seperti WMA, OGG, M4A, DSD, dan lainnya.<br>
Menggunakan mesin audio kustom untuk menangani hampir semua format dan menawarkan fitur seperti sample rate output audio yang dapat disesuaikan dan koreksi pitch.<br><br>

**Evermusic**, di sisi lain, mendukung format audio default dan menyertakan fitur canggih seperti pemutaran crossfade, pemutaran gapless, equalizer audio, kontrol kecepatan pemutaran, dan lainnya.<br><br>

Jika Anda terutama menggunakan MP3, ALAC, atau FLAC, **Evermusic** mungkin menjadi pilihan yang lebih baik karena fitur pemutarannya.<br>
Jika Anda membutuhkan kompatibilitas luas dengan berbagai jenis file audio, **Flacbox** adalah pilihan yang tepat.<br><br>

[Pelajari lebih lanjut tentang perbedaan antara Flacbox dan Evermusic](/docs/faq/evermusic/what-is-the-difference-between-evermusic-and-flacbox/)
{{% /details %}}

{{% details title="Bagaimana cara menyinkronkan Flacbox?" closed="true" %}}

**Sinkronisasi Metadata**  
Saat Anda menambahkan trek ke pustaka, pembaca metadata latar belakang memindai file Anda dan mengaturnya berdasarkan Artis, Album, Genre, dan Komposer.<br>
Anda dapat menyesuaikan kecepatan pemindaian untuk pemuatan lebih cepat, meskipun mungkin meningkatkan penggunaan energi. Anda juga dapat menonaktifkannya sepenuhnya untuk menampilkan hanya nama file sebagai ganti tag.<br><br>

Pembaca metadata **tidak mengubah** file audio asli Anda di cloud atau penyimpanan lokal. Jika Anda ingin mengedit tag, gunakan editor tag bawaan dari menu opsi file.<br><br>

Jika sinkronisasi latar belakang diaktifkan, pembaca metadata akan terus berjalan bahkan saat aplikasi di latar belakang, tetapi perlu diingat bahwa iOS dapat menangguhkannya jika mengonsumsi terlalu banyak energi.<br>
Untuk pustaka besar, disarankan untuk menyinkronkan metadata di versi desktop dan menggunakan fitur **cadangan & pemulihan data** untuk mentransfer ke perangkat mobile Anda.<br><br>

Gunakan tindakan **'Muat ulang metadata'** untuk memaksa penyegaran metadata penuh di seluruh pustaka Anda.<br><br>

**Sinkronisasi Musik Online**<br>
Sinkronisasi online memungkinkan Anda menambahkan musik dari penyimpanan cloud ke pustaka secara otomatis.<br>
Buka **Pengaturan Pustaka Musik → Folder sinkronisasi**, pilih folder untuk dipindai, dan aplikasi akan menemukan dan menambahkan file audio yang didukung.<br><br>

Sinkronisasi online hanya berfungsi **saat aplikasi aktif**, jadi sebaiknya biarkan aplikasi terbuka, terhubung ke daya, dan aktifkan **"Layar selalu aktif"** di pengaturan untuk sinkronisasi lebih cepat.<br>
Anda juga dapat melakukan ini di desktop dan kemudian mentransfer pustaka ke mobile menggunakan cadangan & pemulihan.<br>
Frekuensi sinkronisasi dapat dikonfigurasi (misalnya "segera" memulai sinkronisasi setiap kali aplikasi dibuka).

**Sinkronisasi Musik Offline**<br>  
Ini memindai semua folder di dalam direktori **Dokumen** aplikasi dan menambahkan file lokal yang didukung ke pustaka musik.<br>
File di luar sandbox aplikasi (karena keamanan iOS/macOS) harus ditambahkan secara manual.<br>
Jalankan sinkronisasi offline secara berkala untuk menjaga pustaka tetap terbaru dengan file lokal Anda.<br><br>

**Folder Offline**<br>  
Mengaktifkan mode offline untuk album, artis, genre, playlist, atau folder jarak jauh akan mengunduh semua file dalam grup tersebut untuk akses offline.<br>
Disimpan di bagian **Folder Offline**.<br><br>

File baru yang ditambahkan ke folder jarak jauh yang sama akan diunduh secara otomatis.<br>
Untuk **sinkronisasi ulang manual**, ketuk menu tiga titik dan pilih **"Sinkronisasi"**.<br><br>

Anda juga dapat menyesuaikan **interval timeout sinkronisasi** di pengaturan untuk kontrol lebih baik atas perilaku sinkronisasi dan memastikan musik offline Anda tetap terbaru.<br><br>

{{% details title="Apakah Flacbox Aman?" closed="true" %}}
Flacbox hanya menggunakan SDK resmi dan koneksi aman untuk berinteraksi dengan layanan cloud yang terhubung. Login dan kata sandi Anda tidak tersedia untuk aplikasi. Semua permintaan dari aplikasi ke layanan cloud dienkripsi.<br>
Saat Anda memasukkan login dan kata sandi, aplikasi menampilkan halaman otorisasi resmi yang disediakan oleh penyedia layanan cloud dan seluruh proses otorisasi dilakukan di luar aplikasi. Penyedia layanan cloud mengirimkan auth-token ke aplikasi setelah otorisasi berhasil dan token tersebut digunakan untuk melakukan panggilan API.<br><br>

Auth-token adalah kunci digital yang memungkinkan aplikasi pihak ketiga berinteraksi dengan penyimpanan cloud. Auth-token disimpan di perangkat Anda dalam penyimpanan sistem aman yang disebut Keychain. Anda dapat mengunduh file dari layanan cloud yang terhubung ke perangkat dan file tersebut akan ditempatkan di direktori "Dokumen" aplikasi. Anda dapat menghapus file tersebut kapan saja menggunakan pengelola file bawaan.<br>
Aplikasi tidak membagikan informasi apa pun dari akun cloud yang terhubung. Anda dapat mencabut akses ke akun cloud Anda kapan saja dengan membuka halaman pengaturan akun di browser web Anda.<br><br>

Untuk menolak auth-token, masuk ke akun Anda di browser web dan navigasi ke halaman pengaturan. Di sana Anda dapat menemukan semua aplikasi pihak ketiga yang terhubung ke akun cloud Anda dan menghapus salah satunya jika Anda tidak ingin menggunakan aplikasi tersebut lagi.<br><br>

Anda juga dapat memutuskan akun cloud yang terhubung di aplikasi dan auth-token juga akan dihapus dari perangkat Anda. Jika Anda menghapus aplikasi dari perangkat, semua data yang diunduh dan token akses juga akan dihapus.<br><br>

• Untuk **file cloud**, hubungkan layanan seperti iCloud Drive, Google Drive, Dropbox, dan lainnya di dalam aplikasi Flacbox untuk mengakses buku audio dari jarak jauh.<br><br>

**Fitur Pemutaran**<br>
• Navigasi ke buku audio melalui **Pustaka Musik** atau **File Lokal**.<br>
• Flacbox mendukung **putar, jeda, lewati**, dan **kecepatan pemutaran yang dapat disesuaikan**.<br>
• Anda dapat mengaktifkan tombol **Waktu Lewati** di pengaturan untuk navigasi cepat.<br>
• Buat **Penanda audio** untuk menyimpan posisi Anda.<br>
• Gunakan **Lanjutkan Pemutaran** untuk melanjutkan dari tempat Anda berhenti.<br><br>

**Pengaturan dan Akses Offline**<br>
• Atur buku audio ke dalam playlist berdasarkan **genre**, **penulis**, atau **seri**.<br>
• Unduh buku audio untuk **mendengarkan offline**, sempurna untuk perjalanan atau konektivitas terbatas.<br><br>

Flacbox menyediakan solusi lengkap untuk pecinta buku audio di iPhone, iPad, dan Mac.<br><br>

[Baca selengkapnya](/docs/howto/how-to-listen-to-audiobooks-on-iphone-ipad-mac-using-evermusic)

**Untuk menghubungkan NAS menggunakan protokol SMB:**<br>
• Ketuk **"Hubungkan layanan cloud" → SMB**.<br>
• Masukkan alamat IP server NAS dan nama folder bersama dalam format ini: `smb://ip-anda/folder-bersama`<br>
• Pilih versi protokol: **Auto**, **SMB1**, atau **SMB2**.<br>
• Masukkan login dan kata sandi (jika diperlukan).<br>
• Ketuk **"Selesai"**.<br><br>

Jika koneksi berhasil, Anda akan melihat NAS Anda di bagian **Layanan cloud**.<br>

**Untuk streaming melalui DLNA:**<br>
• Anda dapat streaming dari Windows PC dengan mengaktifkan server media DLNA-nya.<br>

{{% details title="Bagaimana cara menggunakan fitur Wi-Fi Drive di Flacbox?" closed="true" %}}

**Transfer nirkabel menggunakan browser desktop**<br>
1. Luncurkan aplikasi: Buka Flacbox.<br>
2. Hubungkan melalui Wi-Fi: Buka "Koneksi" → "Komputer" → "Hubungkan menggunakan Wi-Fi".<br>
3. (Opsional) Tambahkan keamanan: Masukkan nama pengguna dan kata sandi jika diperlukan.<br>
4. Mulai Wi-Fi Drive: Ketuk "Mulai Wi-Fi Drive" dan salin URL yang disediakan.<br>
5. Buka browser (Safari, Chrome, Opera, Firefox, dll.) dan masukkan URL.<br>
6. Gunakan pengelola file bawaan untuk mengunggah, mengunduh, mengganti nama, atau menghapus file. Anda dapat menarik dan melepas file langsung ke iPhone Anda.<br>
7. Setelah selesai, ketuk "Hentikan Wi-Fi Drive" di iPhone Anda.<br><br>

Catatan: Pastikan JavaScript diaktifkan dan Anda menggunakan versi browser terbaru untuk performa terbaik.<br><br>

**Transfer file menggunakan Mac Finder**<br>
1. Di Finder, klik "Go" → "Connect to Server…".<br>
2. Masukkan URL server yang ditampilkan di aplikasi Flacbox.<br>
3. Klik "Connect" dan kelola file di iPhone Anda seperti drive jaringan biasa.<br><br>

**Transfer file menggunakan Windows File Explorer**<br>
1. Klik kanan "This PC" → "Map Network Drive…".<br>
2. Masukkan URL server dari aplikasi Flacbox di bidang "Folder".<br>
3. Pilih huruf drive, klik "Finish", dan jelajahi file iPhone Anda secara langsung.<br><br>

[Baca selengkapnya](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)

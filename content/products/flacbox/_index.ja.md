{{% details title="Flacboxはどのように動作しますか？" closed="true" %}}
Flacboxは、オーディオトラックを通常のファイルのように管理できるハイレゾ音楽プレーヤーです。<br>
音楽コレクション全体をDropbox、Google Drive、個人NASなどのクラウドサービスにアップロードし、完全なコントロールでクラウドから直接音楽を再生できます。<br><br>
iTunes同期は不要です。PCやMacからの通常のファイルと同じようにアップロードするだけです。<br>
オフラインで聴くには、内蔵ダウンロードマネージャーを使用して音楽をデバイスに保存します。<br><br>

{{% details title="Flacboxは無料ですか？" closed="true" %}}
Flacboxは一部制限付きで無料で使用でき、プレミアム版にアップグレードすることで制限を解除できます。<br>
一回限りの永久購入または2つのサブスクリプションオプション（月額または年額）から選択できます。価格はお住まいの地域によって異なる場合があります。<br><br>

すべてのプランでファミリー共有が有効になっているため、家族とプレミアム版を共有できます。<br><br>

プレミアムの購入とサブスクリプションはiCloudを通じてiOSとMac間で共有されます。購入を同期するには、iCloudが有効になっていることを確認し、iOSデバイスでアプリを開いて、同期が完了するまで1分待ちます。<br><br>

主にMP3、ALAC、FLACを使用する場合、再生機能の面で**Evermusic**がより良い選択かもしれません。<br>
さまざまなオーディオファイルタイプとの幅広い互換性が必要な場合、**Flacbox**が正しい選択です。<br><br>

{{% details title="Flacboxの同期方法は？" closed="true" %}}

**メタデータ同期**  
ライブラリにトラックを追加すると、バックグラウンドのメタデータリーダーがファイルをスキャンし、アーティスト、アルバム、ジャンル、作曲者別に整理します。<br>
スキャン速度を調整してより速く読み込むことができますが、エネルギー消費が増加する可能性があります。タグの代わりにファイル名のみを表示するように完全に無効にすることもできます。<br><br>

メタデータリーダーはクラウドやローカルストレージ内の元のオーディオファイルを**変更しません**。タグを編集したい場合は、ファイルのオプションメニューから内蔵タグエディターを使用してください。<br><br>

バックグラウンド同期が有効な場合、メタデータリーダーはアプリがバックグラウンドにあるときも動作し続けますが、エネルギーを消費しすぎるとiOSが一時停止する可能性があることに注意してください。<br>
大きなライブラリの場合、デスクトップ版でメタデータを同期し、**データバックアップ＆リストア**機能を使用してモバイルデバイスに転送することをお勧めします。<br><br>

ライブラリ全体のメタデータの完全な更新を強制するには、**「メタデータを再読み込み」**アクションを使用してください。<br><br>

**オンライン音楽同期**<br>
オンライン同期により、クラウドストレージからライブラリに自動的に音楽を追加できます。<br>
**音楽ライブラリ設定 → 同期フォルダ**に移動し、スキャンするフォルダを選択すると、アプリがサポートされているオーディオファイルを見つけて追加します。<br><br>

オンライン同期は**アプリがアクティブな場合のみ**動作するため、アプリを開いたまま電源に接続し、設定で**「常時アクティブ画面」**を有効にして、より速い同期を実現することをお勧めします。<br>
デスクトップでも同じ操作を行い、バックアップ＆リストアを使用してライブラリをモバイルに転送できます。<br>
同期頻度は設定可能です（例：「即時」はアプリを開くたびに同期を開始します）。

**オフライン音楽同期**<br>  
これはアプリの**Documents**ディレクトリ内のすべてのフォルダをスキャンし、サポートされているローカルファイルを音楽ライブラリに追加します。<br>
アプリのサンドボックス外のファイル（iOS/macOSのセキュリティのため）は手動で追加する必要があります。<br>
ローカルファイルとライブラリを最新の状態に保つために、定期的にオフライン同期を実行してください。<br><br>

**オフラインフォルダ**<br>  
アルバム、アーティスト、ジャンル、プレイリスト、またはリモートフォルダのオフラインモードを有効にすると、そのグループ内のすべてのファイルがオフラインアクセス用にダウンロードされます。<br>
**オフラインフォルダ**セクションに保存されます。<br><br>

同じリモートフォルダに追加された新しいファイルは自動的にダウンロードされます。<br>
**手動で再同期**するには、三点メニューをタップして**「同期」**を選択します。<br><br>

設定で**同期タイムアウト間隔**をカスタマイズして、同期動作のコントロールを強化し、オフライン音楽を最新に保つこともできます。<br><br>

{{% details title="Flacboxは安全ですか？" closed="true" %}}
Flacboxは、接続されたクラウドサービスとのやり取りに公式SDKと安全な接続のみを使用しています。ログインとパスワードはアプリケーションからアクセスできません。アプリケーションからクラウドサービスへのすべてのリクエストは暗号化されています。<br>
ログインとパスワードを入力すると、アプリケーションはクラウドサービスプロバイダーが提供する公式認証ページを表示し、認証プロセス全体がアプリケーションの外部で行われます。クラウドサービスプロバイダーは認証成功後にアプリケーションにauth-tokenを送信し、そのトークンがAPIコールに使用されます。<br><br>

auth-tokenは、サードパーティアプリケーションがクラウドストレージとやり取りすることを可能にするデジタルキーです。auth-tokenはKeychain（キーチェーン）と呼ばれる安全なシステムストレージにデバイス上に保存されます。接続されたクラウドサービスからデバイスにファイルをダウンロードでき、それらのファイルはアプリの「Documents」ディレクトリに配置されます。内蔵ファイルマネージャーを使用していつでもそれらのファイルを削除できます。<br>
アプリケーションは接続されたクラウドアカウントからの情報を共有しません。Webブラウザでアカウント設定ページを開くことで、いつでもクラウドアカウントへのアクセスを取り消すことができます。<br><br>

auth-tokenを拒否するには、Webブラウザでアカウントにログインし、設定ページに移動します。そこで、クラウドアカウントに接続されているすべてのサードパーティアプリを見つけ、使用しなくなったアプリケーションを削除できます。<br><br>

アプリケーション内で接続されたクラウドアカウントを切断することもでき、auth-tokenもデバイスから削除されます。デバイスからアプリケーションを削除すると、ダウンロードしたすべてのデータとアクセストークンも削除されます。<br><br>

{{% details title="iPhoneでローカルにダウンロードした音楽を再生する方法は？" closed="true" %}}

{{% details title="Flacboxで曲の歌詞を表示する方法は？" closed="true" %}}
アプリ内のトラックの埋め込み歌詞は**以下の手順**で表示できます：<br>

2. フルスクリーンオーディオプレーヤーを開きます。<br>

{{% details title="コンピュータからFlacboxに音楽を転送する方法は？" closed="true" %}}

SMBプロトコルを使用してコンピュータを接続するには、「クラウドサービスを接続」→ SMBをタップします。URLフィールドにsmb://コンピュータのIPアドレス/共有フォルダ名の形式でコンピュータのIPアドレスと共有フォルダ名を入力し、ログインとパスワードを入力して「完了」をタップします。接続が成功すると、「クラウドサービス」セクションに接続されたストレージが表示されます。<br><br>

完全なチュートリアルはこちら：<br>
[SMBプロトコルを使用してコンピュータからiPhoneにファイルを転送](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

WebDAVプロトコルの場合、URLフィールド以外はすべて同じ手順です。URLはhttp://サーバー名またはサーバーがSSLをサポートしている場合はhttps://サーバー名の形式にする必要があります。<br><br>

Wi-Fi Driveは、デスクトップブラウザを使用してコンピュータからiOSデバイスにワイヤレスでファイルを転送できる人気の技術です。この機能を使用するには、デバイスとコンピュータが同じWi-Fiネットワークに接続されている必要があります。「接続」→「コンピュータ」→「Wi-Fiを使用して接続」を開いてサーバーを有効にします。その後、デスクトップブラウザを開いてアプリケーションからURLを入力します。コンピュータから開いたWebページにファイルをドラッグアンドドロップすると、デバイスに表示されます。<br>
より詳細なチュートリアルはこちら：<br>
[WiFi-Driveを使用してコンピュータからiPhoneにワイヤレスでファイルを転送する方法](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br><br>

iTunesファイル共有は、iTunesとlightningケーブルを使用してコンピュータからデバイスにファイルを転送できるもう1つの技術です。ケーブルを使用してデバイスをコンピュータに接続し、iTunesを起動するだけです。iTunes →「アプリケーションセクション」→を開き、Evermusicを見つけます。アプリアイコンをタップして共有フォルダを表示します。コンピュータからデバイス上の共有フォルダにファイルをコピーします。<br><br>

詳細な手順はこちら：<br>
[iPhoneでローカルファイル（iTunesファイル）を再生する方法](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)<br><br>

DLNA DLNAメディアサーバーを設定し、こちらの説明のようにWindows PCから音楽をストリーミングすることもできます：<br>
[Windows 10でDLNAメディアサーバーを有効にしてiPhoneで音楽を再生する方法](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)

{{% details title="FlacboxのWi-Fi Drive機能の使い方は？" closed="true" %}}

**デスクトップブラウザを使用したワイヤレス転送**<br>
1. アプリを起動：Flacboxを開きます。<br>
2. Wi-Fiで接続：「接続」→「コンピュータ」→「Wi-Fiを使用して接続」に移動します。<br>
3. （オプション）セキュリティを追加：必要に応じてユーザー名とパスワードを入力します。<br>
4. Wi-Fi Driveを開始：「Wi-Fi Driveを開始」をタップして提供されたURLをコピーします。<br>
5. ブラウザ（Safari、Chrome、Opera、Firefoxなど）を開いてURLを入力します。<br>
6. 内蔵ファイルマネージャーを使用してファイルのアップロード、ダウンロード、名前変更、削除ができます。ファイルを直接iPhoneにドラッグアンドドロップできます。<br>
7. 完了したら、iPhoneで「Wi-Fi Driveを停止」をタップします。<br><br>

注意：最高のパフォーマンスを得るために、JavaScriptが有効で最新のブラウザバージョンを使用していることを確認してください。<br><br>

**Mac Finderを使用したファイル転送**<br>
1. Finderで「移動」→「サーバへ接続…」をクリックします。<br>
2. Flacboxアプリに表示されたサーバーURLを入力します。<br>
3. 「接続」をクリックして、ネットワークドライブと同様にiPhone上のファイルを管理します。<br><br>

**Windows エクスプローラーを使用したファイル転送**<br>
1. 「PC」を右クリック →「ネットワークドライブの割り当て…」。<br>
2. Flacboxアプリからのサーバー URLを「フォルダ」フィールドに入力します。<br>
3. ドライブレターを選択して「完了」をクリックし、iPhoneのファイルを直接参照します。<br><br>

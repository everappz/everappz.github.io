{{% details title="Flacbox를 동기화하는 방법은?" closed="true" %}}

**메타데이터 동기화**  
라이브러리에 트랙을 추가하면 백그라운드 메타데이터 리더가 파일을 스캔하고 아티스트, 앨범, 장르, 작곡가별로 정리합니다.<br>
더 빠른 로딩을 위해 스캔 속도를 조정할 수 있지만 에너지 사용량이 증가할 수 있습니다. 태그 대신 파일 이름만 표시하도록 완전히 비활성화할 수도 있습니다.<br><br>

메타데이터 리더는 클라우드나 로컬 스토리지의 원본 오디오 파일을 **수정하지 않습니다**. 태그를 편집하려면 파일 옵션 메뉴에서 내장 태그 편집기를 사용하세요.<br><br>

백그라운드 동기화가 활성화된 경우 앱이 백그라운드에 있어도 메타데이터 리더가 계속 실행되지만 너무 많은 에너지를 소비하면 iOS가 일시 중단할 수 있습니다.<br>
대규모 라이브러리의 경우 데스크톱 버전에서 메타데이터를 동기화하고 **데이터 백업 및 복원** 기능을 사용하여 모바일 기기로 전송하는 것이 좋습니다.<br><br>

라이브러리 전체에서 메타데이터 새로 고침을 강제하려면 **'메타데이터 다시 로드'** 작업을 사용하세요.<br><br>

**온라인 음악 동기화**<br>
온라인 동기화를 통해 클라우드 스토리지에서 라이브러리에 자동으로 음악을 추가할 수 있습니다.<br>
**음악 라이브러리 설정 → 동기화 폴더**로 이동하여 스캔할 폴더를 선택하면 앱이 지원되는 오디오 파일을 찾아 추가합니다.<br><br>

온라인 동기화는 **앱이 활성 상태일 때만** 작동하므로 앱을 열어두고 전원에 연결하고 설정에서 **"항상 활성 화면"**을 활성화하여 더 빠른 동기화를 하는 것이 좋습니다.<br>
데스크톱에서도 이 작업을 수행한 다음 백업 및 복원을 사용하여 라이브러리를 모바일로 전송할 수 있습니다.<br>
동기화 빈도를 구성할 수 있습니다(예: "즉시"는 앱을 열 때마다 동기화를 시작합니다).

**오프라인 음악 동기화**<br>  
앱의 **Documents** 디렉토리 내 모든 폴더를 스캔하고 지원되는 로컬 파일을 음악 라이브러리에 추가합니다.<br>
앱 샌드박스 외부의 파일(iOS/macOS 보안)은 수동으로 추가해야 합니다.<br>
로컬 파일과 라이브러리를 최신 상태로 유지하려면 주기적으로 오프라인 동기화를 실행하세요.<br><br>

**오프라인 폴더**<br>  
앨범, 아티스트, 장르, 재생목록 또는 원격 폴더에 대해 오프라인 모드를 활성화하면 해당 그룹 내의 모든 파일이 오프라인 액세스를 위해 다운로드됩니다.<br>
**오프라인 폴더** 섹션에 저장됩니다.<br><br>

같은 원격 폴더에 추가된 새 파일은 자동으로 다운로드됩니다.<br>
**수동으로 재동기화**하려면 세 점 메뉴를 탭하고 **"동기화"**를 선택하세요.<br><br>

설정에서 **동기화 타임아웃 간격**을 사용자 정의하여 동기화 동작을 더 잘 제어하고 오프라인 음악을 최신 상태로 유지할 수도 있습니다.<br><br>

{{% details title="Flacbox는 안전한가요?" closed="true" %}}
Flacbox는 연결된 클라우드 서비스와의 상호 작용에 공식 SDK와 보안 연결만 사용합니다. 로그인과 비밀번호는 애플리케이션에서 사용할 수 없습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.<br>
로그인과 비밀번호를 입력하면 애플리케이션은 클라우드 서비스 제공업체가 제공하는 공식 인증 페이지를 표시하며 전체 인증 프로세스는 애플리케이션 외부에서 이루어집니다. 클라우드 서비스 제공업체는 성공적인 인증 후 애플리케이션에 auth-token을 보내고 해당 토큰은 API 호출에 사용됩니다.<br><br>

auth-token은 타사 애플리케이션이 클라우드 스토리지와 상호 작용할 수 있게 하는 디지털 키입니다. auth-token은 Keychain이라는 안전한 시스템 스토리지에 기기에 저장됩니다. 연결된 클라우드 서비스에서 기기로 파일을 다운로드할 수 있으며 해당 파일은 앱의 "Documents" 디렉토리에 배치됩니다. 내장 파일 관리자를 사용하여 언제든지 해당 파일을 제거할 수 있습니다.<br>
애플리케이션은 연결된 클라우드 계정의 정보를 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정에 대한 액세스를 취소할 수 있습니다.<br><br>

auth-token을 거부하려면 웹 브라우저에서 계정에 로그인하고 설정 페이지로 이동합니다. 거기에서 클라우드 계정에 연결된 모든 타사 앱을 찾을 수 있으며 더 이상 사용하지 않으려는 앱을 제거할 수 있습니다.<br><br>

애플리케이션에서 연결된 클라우드 계정을 분리할 수도 있으며 auth-token도 기기에서 제거됩니다. 기기에서 애플리케이션을 제거하면 모든 다운로드된 데이터와 액세스 토큰도 제거됩니다.<br><br>

{{% details title="iPhone에서 로컬로 다운로드한 음악을 재생하는 방법은?" closed="true" %}}
애플리케이션을 설치한 후 "로컬 파일" 화면을 열고 "이 iPhone의 파일" 섹션으로 스크롤합니다. 여러 오디오 파일을 선택해야 하는 경우 "파일 열기..."를, 음악 폴더를 선택하려면 "폴더 열기..."를 선택합니다. 앱이 폴더 콘텐츠를 스캔하고 발견된 모든 오디오 파일이 선택됩니다. 음악 폴더로 이동하여 "열기"를 탭하여 선택을 확인하면 파일이 플레이어 대기열에 추가됩니다. 이 파일들은 애플리케이션 번들에 복사되지 않고 선택한 위치에서 직접 재생됩니다.<br><br>

**빠른 액세스를 위해 폴더를 즐겨찾기에 추가**<br>
기기의 폴더를 즐겨찾기에 추가하여 프로세스를 간소화하세요. 이렇게 하면 음악을 재생할 때마다 단계를 반복할 필요가 없습니다. "로컬 파일" 화면을 열고 "빠른 액세스" 섹션으로 스크롤한 다음 "즐겨찾기"를 탭하여 "즐겨찾기 파일" 화면에 액세스합니다. 오른쪽 상단의 추가 작업 버튼(세 점)을 탭하고 "폴더 추가"를 선택합니다. 원하는 폴더로 이동하여 "열기"를 탭하여 확인합니다. 폴더가 "즐겨찾기 파일"에 추가되어 음악에 빠르게 액세스할 수 있습니다.<br><br>

**로컬 파일을 음악 라이브러리로 가져오기**<br>
로컬 파일을 음악 라이브러리 내에서 정리하려면 "음악 라이브러리" 화면을 열고 오른쪽 상단의 세 점 버튼을 탭하고 "+ 음악 추가"를 선택합니다. "로컬 파일" 메뉴 항목을 선택하고 "이 iPhone의 파일" 섹션으로 스크롤하여 "파일 열기..."를 탭합니다. 추가할 파일을 선택하고 "열기"를 탭하여 확인합니다. 앱이 선택한 파일을 스캔하고 아티스트, 앨범, 장르와 같은 메타데이터로 정리하여 음악 라이브러리에 추가합니다.<br><br>

**로컬 파일을 재생목록에 추가**<br>
로컬 파일을 재생목록에 추가하려면 "재생목록" 화면을 열고 오른쪽 상단의 추가 버튼을 탭합니다. "+ 새 재생목록"을 선택하고 새 재생목록의 이름을 입력한 다음 화면에서 "로컬 파일" 옵션을 선택합니다. "이 iPhone의 파일" 섹션으로 스크롤하여 "파일 열기..."를 탭합니다. 추가할 오디오 파일을 선택하고 "열기"를 탭하여 확인합니다. 파일이 재생목록에 추가되며 추가 버튼을 사용하여 곡 순서를 변경하고 기타 작업을 수행할 수 있습니다.<br><br>

이 간단한 단계들로 소중한 로컬 음악 컬렉션을 즐기기 위한 궁극의 플랫폼으로 iPhone과 Mac의 잠재력을 최대한 활용할 수 있습니다.<br><br>

3. LRC 파일 모드: 오디오 파일을 편집하는 대신 원본 오디오 파일과 같은 폴더에 LRC 파일을 놓을 수 있습니다. 두 파일은 이름이 같고 확장자가 달라야 합니다. 댓글 화면의 세 번째 페이지로 스와이프하면 앱이 같은 디렉토리에서 LRC 파일을 검색하여 내용을 표시합니다.<br><br>

{{% details title="Flacbox로 오디오북을 재생할 수 있나요?" closed="true" %}}

네, Flacbox는 강력한 오디오북 플레이어로 사용할 수 있습니다.<br><br>

**오디오북 추가 방법**<br>
• **로컬 파일**의 경우, iPhone 또는 iPad를 컴퓨터에 연결하고 iTunes(또는 macOS Catalina 이상에서는 Finder)를 열어 오디오북 파일을 "Flacbox" 섹션으로 드래그합니다.<br>
• **클라우드 파일**의 경우, Flacbox 앱 내에서 iCloud Drive, Google Drive, Dropbox 등의 서비스를 연결하여 원격으로 오디오북에 액세스합니다.<br><br>

**재생 기능**<br>
• **음악 라이브러리** 또는 **로컬 파일**을 통해 오디오북으로 이동합니다.<br>
• Flacbox는 **재생, 일시 정지, 건너뛰기** 및 **조절 가능한 재생 속도**를 지원합니다.<br>
• 설정에서 **건너뛰기 시간** 버튼을 활성화하여 빠른 탐색이 가능합니다.<br>
• **오디오 북마크**를 만들어 위치를 저장합니다.<br>
• **재생 계속**을 사용하여 중단한 곳에서 재개합니다.<br><br>

**정리 및 오프라인 액세스**<br>
• **장르**, **저자** 또는 **시리즈**별로 재생목록에 오디오북을 정리합니다.<br>
• **오프라인 청취**를 위해 오디오북을 다운로드합니다. 여행이나 제한된 연결 환경에 적합합니다.<br><br>

Flacbox는 iPhone, iPad, Mac에서 오디오북 애호가를 위한 완전한 솔루션을 제공합니다.<br><br>

{{% details title="Flacbox는 NAS 기기와 작동하나요?" closed="true" %}}

네, Flacbox는 **SMB**, **WebDAV**, **DLNA** 프로토콜을 사용한 NAS 연결을 지원합니다.<br><br>

**SMB 프로토콜로 NAS 연결:**<br>
• **"클라우드 서비스 연결" → SMB**를 탭합니다.<br>
• NAS 서버 IP 주소와 공유 폴더 이름을 다음 형식으로 입력합니다: `smb://your-ip/shared-folder`<br>
• 프로토콜 버전 선택: **자동**, **SMB1** 또는 **SMB2**.<br>
• 로그인과 비밀번호를 입력합니다(필요한 경우).<br>
• **"완료"**를 탭합니다.<br><br>

연결이 성공하면 **클라우드 서비스** 섹션에 NAS가 표시됩니다.<br>
[전체 튜토리얼 읽기](/docs/howto/stream-your-music-from-mac-or-pc-to-iphone-using-smb/)<br><br>

**WebDAV 프로토콜로 연결:**<br>
• SMB와 동일한 단계이지만 `http://your-server-name` 또는 SSL 사용 시 `https://your-server-name`과 같은 WebDAV URL을 사용합니다.<br><br>

**DLNA로 스트리밍:**<br>
• Windows PC의 DLNA 미디어 서버를 활성화하여 스트리밍할 수 있습니다.<br>
[DLNA 설정 가이드 읽기](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)<br><br>

**사용 가능한 기기 섹션:**<br>
• 로컬 네트워크에서 검색 가능한 모든 NAS 기기를 표시합니다.<br>
• 기기 이름을 탭하여 연결한 다음 필요한 경우 로그인 자격 증명을 입력합니다.<br><br>

**유용한 링크:**<br>

{{% details title="Flacbox에 음악을 가져오는 방법은?" closed="true" %}}

**클라우드 서비스 연결**<br>
• **연결** 탭을 엽니다.<br>
• 메뉴에서 **클라우드 서비스 연결**을 선택합니다.<br>
• 목록에서 원하는 클라우드 서비스를 선택합니다.<br>
• 자격 증명을 입력하고 **완료**를 탭합니다.<br><br>

**기기에서 파일 가져오기**<br>
기기, USB 플래시 드라이브, 컴퓨터 또는 NAS에서 파일을 가져올 수 있습니다.<br>
• **로컬 파일** 탭을 엽니다.<br>
• **이 기기의 파일** 메뉴에서 **파일 열기...** 또는 **폴더 열기...**를 탭합니다.<br>
• 기기의 파일 또는 폴더를 선택합니다.<br>
• **열기**를 탭하여 앱에 파일을 가져옵니다.<br><br>

**음악 추가**<br>
수동으로 또는 동기화 폴더를 사용하여 자동으로 음악을 추가할 수 있습니다.<br><br>

**수동 추가**<br>
• **음악 추가** 메뉴 항목을 탭합니다.<br>
• 연결된 클라우드 서비스 또는 기기에서 폴더 또는 파일을 선택합니다.<br>
• Flacbox는 복사하지 않고 트랙에 대한 링크를 생성합니다.<br>
• 트랙을 오프라인으로 사용하려면 다운로드 관리자를 사용하거나 **오프라인 모드**를 활성화합니다.<br><br>

**자동 동기화**<br>
• **음악 라이브러리 설정**을 엽니다.<br>
• **동기화 폴더**를 선택하여 자동 동기화를 켭니다.<br>
• 앱이 선택한 폴더를 스캔하고 지원되는 오디오 파일을 라이브러리에 추가합니다.<br><br>

**음악 라이브러리 탐색**<br>
• 라이브러리는 소스(온라인, 오프라인, 로컬)별로 정리되어 있습니다.<br>
• 곡, 앨범, 아티스트, 장르, 작곡가별로 탐색합니다.<br>
• 검색 바를 사용하여 모든 곡을 빠르게 찾습니다.<br><br>

{{% details title="Flacbox의 Wi-Fi Drive 기능 사용 방법은?" closed="true" %}}

**데스크톱 브라우저를 사용한 무선 전송**<br>
1. 앱 실행: Flacbox를 엽니다.<br>
2. Wi-Fi로 연결: "연결" → "컴퓨터" → "Wi-Fi로 연결"로 이동합니다.<br>
3. (선택 사항) 보안 추가: 필요한 경우 사용자 이름과 비밀번호를 입력합니다.<br>
4. Wi-Fi Drive 시작: "Wi-Fi Drive 시작"을 탭하고 제공된 URL을 복사합니다.<br>
5. 브라우저(Safari, Chrome, Opera, Firefox 등)를 열고 URL을 입력합니다.<br>
6. 내장 파일 관리자를 사용하여 파일을 업로드, 다운로드, 이름 변경 또는 삭제합니다. 파일을 iPhone으로 직접 드래그 앤 드롭할 수 있습니다.<br>
7. 완료되면 iPhone에서 "Wi-Fi Drive 중지"를 탭합니다.<br><br>

참고: 최상의 성능을 위해 JavaScript가 활성화되어 있고 최신 브라우저 버전을 사용하고 있는지 확인하세요.<br><br>

**Mac Finder를 사용한 파일 전송**<br>
1. Finder에서 "이동" → "서버에 연결..."을 클릭합니다.<br>
2. Flacbox 앱에 표시된 서버 URL을 입력합니다.<br>
3. "연결"을 클릭하고 네트워크 드라이브처럼 iPhone의 파일을 관리합니다.<br><br>

**Windows 파일 탐색기를 사용한 파일 전송**<br>
1. "내 PC"를 마우스 오른쪽 버튼으로 클릭 → "네트워크 드라이브 연결...".<br>
2. Flacbox 앱의 서버 URL을 "폴더" 필드에 입력합니다.<br>
3. 드라이브 문자를 선택하고 "마침"을 클릭한 다음 iPhone 파일을 직접 탐색합니다.<br><br>

CarPlay를 사용하려면 Siri가 활성화되어 있고 iPhone이 USB 또는 무선으로 연결되어 있는지 확인하세요.

要离线观看，请使用内置下载管理器将视频保存到设备。<br><br>

{{% details title="如何使用 Evervideo？" closed="true" %}}

**安装 Evervideo**<br>
从设备的应用商店下载并安装 Evervideo 应用。支持 iOS 和 Mac 设备。<br><br>

**打开应用**<br>

• 从 **Files on this device** 菜单中点击 **Open the files...** 或 **Open the folders...**。<br>
• 选择设备上的文件或文件夹。<br>
• 点击 **Open** 将文件导入应用。<br><br>

**添加视频**<br>
您有两种方式将视频添加到 Evervideo：手动添加或自动同步。<br><br>

**手动添加**<br>
• 打开 **Library** 部分。<br>
• 点击 **Add** 或类似菜单项。<br>
• 从已连接的云存储服务或本地文件中选择文件夹/文件。<br>

• 要离线可用，请使用内置下载管理器或启用离线模式。<br><br>

**自动同步**<br>

• 选择 **Online synchronization** -> **Wi-Fi** -> **Synchronized online folders** 启用自动同步。<br>
• 应用扫描选定的文件夹，识别媒体文件，并将其集成到您的库中。<br><br>

**浏览您的视频库**<br>
• 按来源类型和元数据标签整理。<br>
• 筛选器：在线文件、离线文件等。<br>
• 分类：所有视频、专辑、艺术家、类型等。<br>
• 使用 **Search** 快速查找媒体曲目。<br><br>

**自定义设置**<br>
• 点击三个点按钮访问 **More actions** 和 **Settings**。<br>
• 调整元数据同步、个性化、播放列表、最近播放和收藏。<br><br>

**享受您的视频**<br>
视频整理好后，使用顶部工具栏进行快速操作，如 **Search**、**Play All**、**Shuffle** 和 **Continue Playback**。<br><br>

{{% details title="Evervideo 安全吗？" closed="true" %}}
Evervideo 仅使用官方 SDK 和安全连接与已连接的云服务交互。您的用户名和密码不会被应用访问。从应用到云服务的所有请求都经过加密。<br>
当您输入用户名和密码时，应用会显示由云服务提供商提供的官方授权页面，整个授权过程在应用外部完成。云服务提供商在成功授权后向应用发送 auth-token，该令牌用于进行 API 调用。<br><br>

Auth-token 是一个数字密钥，允许第三方应用与云存储交互。Auth-token 存储在设备上名为 Keychain 的安全系统存储中。您可以从已连接的云服务下载文件到设备，这些文件将放置在应用的"Documents"目录中。您可以随时使用内置文件管理器删除这些文件。<br>
应用不会共享已连接云账户的任何信息。您可以随时通过在网页浏览器中打开账户设置页面来撤销对云账户的访问权限。<br><br>

要撤销 auth-token，请在网页浏览器中登录您的账户并导航到设置页面。在那里您可以找到所有连接到云账户的第三方应用，如果不想再使用某个应用，可以将其删除。<br><br>

您也可以在应用中断开已连接的云账户，auth-token 也将从设备中删除。如果您从设备中删除应用，所有下载的数据和访问令牌也将被删除。<br><br>

{{% details title="如何从电脑传输视频到 Evervideo？" closed="true" %}}

要使用 SMB 协议连接电脑，点击"Files""Connect to cloud storage" → SMB。在 URL 字段中输入电脑 IP 地址和共享文件夹名称，格式为 smb://computer-ip-address/shared-folder-name，输入用户名和密码并点击"Done"。如果连接成功，您将在"Cloud storage"部分看到已连接的存储。<br><br>

Wi-Fi Drive 是一种流行的技术，允许您使用桌面浏览器无线地将文件从电脑传输到 iOS 设备。要使用此功能，您的设备和电脑需要连接到同一 Wi-Fi 网络。打开"Files" → "Computer" → "Connect using Wi-Fi"并启用服务器。然后打开桌面浏览器并输入应用中的 URL。您可以从电脑拖放文件到打开的网页，文件将出现在设备上。<br>

iTunes 文件共享是另一种技术，允许您使用 iTunes 和 Lightning 线缆从电脑传输文件到设备。只需使用线缆将设备连接到电脑并运行 iTunes。打开 iTunes →"Applications section" → 找到 Evervideo。点击应用图标查看共享文件夹。从电脑复制文件到设备上的共享文件夹。<br><br>

{{% details title="Evervideo 是否支持 NAS 设备？" closed="true" %}}

• 点击 **Files** -> **"Connect to cloud storage" → SMB**。<br>

**通过 DLNA 串流：**<br>
• 您可以通过启用 Windows PC 的 DLNA 媒体服务器从其串流。<br>
[阅读 DLNA 设置指南](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)<br><br>

**可用设备部分：**<br>
• 显示本地网络上所有可发现的 NAS 设备。<br>

{{% details title="如何在 Evervideo 中使用 Wi-Fi Drive 功能？" closed="true" %}}

**使用桌面浏览器无线传输**<br>
1. 启动应用：打开 Evervideo。<br>
2. 通过 Wi-Fi 连接：前往 Files → "Computer" → "Connect via Wi-Fi"。<br>
3.（可选）添加安全设置：根据需要输入用户名和密码。<br>
4. 启动 Wi-Fi Drive：点击"Start Wi-Fi Drive"并复制提供的 URL。<br>
5. 打开浏览器（Safari、Chrome、Opera、Firefox 等）并输入 URL。<br>
6. 使用内置文件管理器上传、下载、重命名或删除文件。您可以直接将文件拖放到 iPhone。<br>
7. 完成后，在 iPhone 上点击"Stop Wi-Fi Drive"。<br><br>

注意：确保 JavaScript 已启用，并使用最新版本的浏览器以获得最佳性能。<br><br>

**使用 Mac Finder 传输文件**<br>
1. 在 Finder 中，点击"Go" → "Connect to Server…"。<br>
2. 输入 Evervideo 应用中显示的服务器 URL。<br>
3. 点击"Connect"，然后像管理任何网络驱动器一样管理 iPhone 上的文件。<br><br>

**使用 Windows 文件资源管理器传输文件**<br>
1. 右键点击"This PC" → "Map Network Drive…"。<br>
2. 在"Folder"字段中输入 Evervideo 应用中的服务器 URL。<br>
3. 选择驱动器号，点击"Finish"，然后直接浏览 iPhone 上的文件。<br><br>

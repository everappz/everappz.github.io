{{% details title="Come funziona Evervideo?" closed="true" %}}
Evervideo è un lettore video HD che ti permette di gestire le tracce video come normali file.<br>
Puoi caricare la tua intera collezione video su servizi cloud come Dropbox, OneDrive, iCloud o un NAS personale e riprodurre i video direttamente dal cloud con il pieno controllo.<br><br>
Non è necessaria la sincronizzazione con iTunes — basta caricare dal tuo PC o Mac come faresti con qualsiasi file.<br>
Per ascoltare offline, usa il download manager integrato per salvare i video sul tuo dispositivo.<br><br>

Esplora le nostre guide per maggiori dettagli:<br>
- [Guida Evervideo](/docs/guide/evervideo/)<br>
- [Come trasferire file in modalità wireless da un computer a un iPhone usando WiFi-Drive.](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br>

{{% /details %}}

{{% details title="Evervideo è gratuito?" closed="true" %}}
Evervideo è gratuito con alcune limitazioni, che possono essere rimosse passando alla versione Premium.<br>
Puoi scegliere tra un acquisto a vita una tantum o due opzioni di abbonamento (mensile o annuale). I prezzi possono variare in base alla tua regione.<br><br>

La condivisione In Famiglia è abilitata per tutti i piani, così puoi condividere la versione Premium con i membri della tua famiglia.<br><br>

Gli acquisti e gli abbonamenti Premium sono condivisi tra iOS e Mac tramite iCloud. Per sincronizzare il tuo acquisto, assicurati che iCloud sia abilitato, apri l'app sul tuo dispositivo iOS e attendi un minuto per il completamento della sincronizzazione.<br><br>

[Scopri di più sulle differenze tra Evervideo ed Evervideo Premium](/docs/faq/evervideo/what-is-the-difference-between-evervideo-and-evervideo-premium/)
{{% /details %}}

{{% details title="Come uso Evervideo?" closed="true" %}}

**Installa Evervideo**<br>
Scarica e installa l'app Evervideo dall'app store del tuo dispositivo. È disponibile per dispositivi iOS e Mac.<br><br>

**Apri l'app**<br>
Avvia Evervideo toccando la sua icona nella schermata principale del tuo dispositivo.<br><br>

**Collega il tuo servizio cloud**<br>
• Apri la scheda **Files**.<br>
• Seleziona **Connect to cloud storage** dal menu.<br>
• Scegli il servizio di cloud storage desiderato dall'elenco.<br>
• Inserisci le tue credenziali e tocca **Done**.<br><br>

**Importa file dal tuo dispositivo**<br>
Puoi importare file dal tuo dispositivo, da una chiavetta USB esterna, da un computer personale o da un NAS.<br>
• Apri la scheda **Files**.<br>
• Tocca **Open the files...** o **Open the folders...** dal menu **Files on this device**.<br>
• Scegli file o cartelle sul tuo dispositivo.<br>
• Tocca **Open** per importare i file nell'app.<br><br>

**Aggiungi i tuoi video**<br>
Hai due opzioni per aggiungere video a Evervideo: aggiunta manuale o sincronizzazione automatica.<br><br>

**Aggiunta manuale**<br>
• Apri la sezione **Library**.<br>
• Tocca **Add** o una voce di menu simile.<br>
• Seleziona cartelle/file dai servizi cloud collegati o dai file locali.<br>
• Evervideo crea collegamenti alle tue tracce, preservando le posizioni originali dei file.<br>
• Per rendere le tracce disponibili offline, usa il download manager integrato o abilita la modalità offline.<br><br>

**Sincronizzazione automatica**<br>
• Vai su **Settings** -> **Library**.<br>
• Seleziona **Online synchronization** -> **Wi-Fi** -> **Synchronized online folders** per abilitare la sincronizzazione automatica.<br>
• L'app scansiona le cartelle selezionate, identifica i file multimediali e li integra nella tua libreria.<br><br>

**Esplora la tua libreria video**<br>
• Organizzata per tipi di sorgente e tag dei metadati.<br>
• Filtri: file online, file offline, ecc.<br>
• Categorie: Tutti i video, Album, Artisti, Generi, ecc.<br>
• Usa **Search** per trovare rapidamente le tracce multimediali.<br><br>

**Personalizza le tue impostazioni**<br>
• Tocca il pulsante con i tre puntini per accedere a **More actions** e **Settings**.<br>
• Regola la sincronizzazione dei metadati, la personalizzazione, le playlist, i recenti e i preferiti.<br><br>

**Goditi i tuoi video**<br>
Una volta che i tuoi video sono organizzati, usa la barra degli strumenti superiore per azioni rapide come **Search**, **Play All**, **Shuffle** e **Continue Playback**.<br><br>

{{% /details %}}

{{% details title="Evervideo è sicuro?" closed="true" %}}
Evervideo utilizza solo SDK ufficiali e connessioni sicure per interagire con i servizi cloud collegati. Il tuo login e la tua password non sono accessibili all'applicazione. Tutte le richieste dall'applicazione al servizio cloud sono crittografate.<br>
Quando inserisci login e password, l'applicazione ti mostra la pagina di autorizzazione ufficiale fornita dal provider del servizio cloud e tutto il processo di autorizzazione avviene al di fuori dell'applicazione. Il provider del servizio cloud invia un auth-token all'applicazione dopo l'autorizzazione riuscita e quel token viene utilizzato per effettuare chiamate API.<br><br>

L'auth-token è una chiave digitale che consente alle applicazioni di terze parti di interagire con il cloud storage. L'auth-token è memorizzato sul tuo dispositivo in un archivio di sistema sicuro chiamato Keychain. Puoi scaricare i tuoi file dal servizio cloud collegato sul dispositivo e quei file saranno posizionati nella directory "Documents" dell'app. Puoi rimuovere quei file in qualsiasi momento usando il gestore file integrato.<br>
L'applicazione non condivide alcuna informazione dall'account cloud collegato. Puoi revocare l'accesso al tuo account cloud in qualsiasi momento aprendo la pagina delle impostazioni dell'account nel tuo browser web.<br><br>

Per rifiutare l'auth-token, accedi al tuo account nel browser web e vai alla pagina delle impostazioni. Lì puoi trovare tutte le app di terze parti collegate al tuo account cloud e rimuovere quelle che non desideri più utilizzare.<br><br>

Puoi anche disconnettere gli account cloud collegati nell'applicazione e l'auth-token verrà rimosso dal tuo dispositivo. Se rimuovi l'applicazione dal tuo dispositivo, tutti i dati scaricati e i token di accesso verranno anch'essi rimossi.<br><br>

{{% details title="Come uso la funzione Wi-Fi Drive in Evervideo?" closed="true" %}}

**Trasferimento wireless usando un browser desktop**<br>
1. Avvia l'app: Apri Evervideo.<br>
2. Collegati tramite Wi-Fi: Vai su Files → "Computer" → "Connect via Wi-Fi".<br>
3. (Opzionale) Aggiungi sicurezza: Inserisci nome utente e password se necessario.<br>
4. Avvia Wi-Fi Drive: Tocca "Start Wi-Fi Drive" e copia l'URL fornito.<br>
5. Apri un browser (Safari, Chrome, Opera, Firefox, ecc.) e inserisci l'URL.<br>
6. Usa il gestore file integrato per caricare, scaricare, rinominare o eliminare file. Puoi trascinare i file direttamente sul tuo iPhone.<br>
7. Quando hai finito, tocca "Stop Wi-Fi Drive" sul tuo iPhone.<br><br>

Nota: Assicurati che JavaScript sia abilitato e di usare l'ultima versione del browser per le migliori prestazioni.<br><br>

**Trasferire file usando Mac Finder**<br>
1. In Finder, clicca "Go" → "Connect to Server…".<br>
2. Inserisci l'URL del server mostrato nell'app Evervideo.<br>
3. Clicca "Connect" e gestisci i file sul tuo iPhone come qualsiasi unità di rete.<br><br>

**Trasferire file usando Windows File Explorer**<br>
1. Fai clic destro su "This PC" → "Map Network Drive…".<br>
2. Inserisci l'URL del server dall'app Evervideo nel campo "Folder".<br>
3. Scegli una lettera di unità, clicca "Finish" e sfoglia direttamente i file del tuo iPhone.<br><br>

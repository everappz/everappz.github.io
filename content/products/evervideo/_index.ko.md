Evervideo는 스페인 소프트웨어 회사인 Everappz가 개발한 iPhone 및 Mac용 무료 HD 비디오 플레이어입니다. Evervideo는 포맷 변환 없이 MKV, AVI, MP4, MOV, FLV, WMV, WEBM, M4V, TS, 3GP를 포함한 거의 모든 비디오 형식을 재생합니다. 이 앱은 360도 및 VR 비디오 재생, Picture-in-Picture 모드, 50개 이상의 프리셋을 갖춘 비디오 및 오디오 이퀄라이저, SRT, SSA, ASS 형식의 자막 지원, 재생 속도 조절 기능을 제공합니다. Evervideo는 iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA를 포함한 클라우드 저장 서비스에 연결하여 사용자가 클라우드에서 직접 비디오를 스트리밍하거나 오프라인 시청을 위해 다운로드할 수 있습니다. 또한 SMB, WebDAV, DLNA 프로토콜을 통한 로컬 네트워크 스트리밍, Lightning 또는 USB-C 어댑터를 통한 USB 플래시 드라이브 재생, 컴퓨터에서의 Wi-Fi 파일 전송도 지원합니다. 추가 기능으로는 재생 목록이 포함된 미디어 라이브러리, AirPlay 및 Chromecast 캐스팅, 내장 파일 관리자가 있습니다. Evervideo는 App Store에서 무료로 다운로드할 수 있으며, 월 $2.99 구독, 연간 $14.99 구독, 또는 $29.99 일회성 평생 구매를 포함한 선택적 인앱 구매가 제공됩니다.

{{% details title="Evervideo는 안전한가요?" closed="true" %}}
Evervideo는 연결된 클라우드 서비스와 상호 작용하기 위해 공식 SDK 및 보안 연결만 사용합니다. 로그인과 비밀번호는 애플리케이션에 제공되지 않습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.<br>
로그인과 비밀번호를 입력하면 애플리케이션은 클라우드 서비스 제공자가 제공하는 공식 인증 페이지를 표시하며 모든 인증 프로세스는 애플리케이션 외부에서 수행됩니다. 클라우드 서비스 제공자는 성공적인 인증 후 애플리케이션에 인증 토큰을 보내고 해당 토큰은 API 호출에 사용됩니다.<br><br>

인증 토큰은 타사 애플리케이션이 클라우드 저장소와 상호 작용할 수 있게 해주는 디지털 키입니다. 인증 토큰은 기기의 키체인이라는 보안 시스템 저장소에 저장됩니다. 연결된 클라우드 서비스에서 파일을 기기로 다운로드할 수 있으며 해당 파일은 앱 "문서" 디렉토리에 저장됩니다. 내장 파일 관리자를 사용하여 언제든지 해당 파일을 제거할 수 있습니다.<br>
애플리케이션은 연결된 클라우드 계정의 정보를 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정 액세스를 취소할 수 있습니다.<br><br>

인증 토큰을 거부하려면 웹 브라우저에서 계정에 로그인하고 설정 페이지로 이동하세요. 거기에서 클라우드 계정에 연결된 모든 타사 앱을 찾을 수 있으며 더 이상 사용하지 않으려는 앱을 제거할 수 있습니다.<br><br>

애플리케이션에서 연결된 클라우드 계정의 연결을 해제할 수도 있으며 인증 토큰도 기기에서 제거됩니다. 기기에서 애플리케이션을 제거하면 다운로드된 모든 데이터와 액세스 토큰도 제거됩니다.<br><br>

{{% details title="iPhone에서 로컬로 다운로드된 비디오를 재생하는 방법은?" closed="true" %}}
애플리케이션을 설치한 후 "파일" 화면을 열고 "이 iPhone의 파일" 섹션으로 스크롤하세요. 여러 파일을 선택해야 하는 경우 "파일 열기..."를 선택하고, 미디어 폴더를 선택하려면 "폴더 열기..."를 선택하세요. 앱이 폴더의 내용을 스캔하고 발견된 모든 미디어 파일이 선택됩니다. 미디어 폴더로 이동하고 "열기"를 탭하여 선택을 확인하면 파일이 플레이어 대기열에 추가됩니다. 이 파일들은 애플리케이션 번들에 복사되지 않고 선택한 위치에서 직접 재생됩니다.<br><br>

**빠른 접근을 위해 폴더를 즐겨찾기에 추가**<br>
기기에 있는 폴더를 즐겨찾기에 추가하여 프로세스를 간소화하세요. 이렇게 하면 음악을 재생할 때마다 단계를 반복할 필요가 없습니다. "파일" 화면을 열고 "빠른 접근" 섹션으로 스크롤한 다음 "즐겨찾기"를 탭하여 "즐겨찾기 파일" 화면에 접근하세요. 오른쪽 상단 모서리에 있는 추가 작업 버튼(세 개의 점)을 탭하고 "폴더 추가"를 선택하세요. 원하는 폴더로 이동하고 "열기"를 탭하여 확인하세요. 폴더가 "즐겨찾기 파일"에 추가되어 미디어 파일에 빠르게 접근할 수 있습니다.<br><br>

**로컬 파일을 라이브러리로 가져오기**<br>
로컬 파일을 라이브러리 내에서 정리하려면 "라이브러리" 화면을 열고 오른쪽 상단 모서리에 있는 세 개의 점 버튼을 탭한 다음 "+ 미디어 파일 추가"를 선택하세요. "이 기기의 파일" 메뉴 항목을 선택하고 "파일 열기..."를 탭하세요. 추가할 파일을 선택하고 "열기"를 탭하여 확인하세요. 앱이 선택한 파일을 스캔하고 메타데이터별로 정리하여 라이브러리에 추가합니다.<br><br>

**로컬 파일을 재생목록에 추가**<br>
로컬 파일을 재생목록에 추가하려면 "재생목록" 화면을 열고 오른쪽 상단 모서리에 있는 추가 버튼을 탭하세요. "+ 새 재생목록"을 선택하고 새 재생목록의 이름을 입력한 다음, 다음 화면에서 "이 기기의 파일" 옵션을 선택하고 "파일 열기..."를 탭하세요. 추가할 미디어 파일을 선택하고 "열기"를 탭하여 확인하세요. 파일이 재생목록에 추가되며 추가 버튼을 사용하여 트랙 순서를 변경하고 다른 작업을 수행할 수 있습니다.<br><br>

{{% details title="재생목록을 중단한 곳에서 어떻게 다시 시작하나요?" closed="true" %}}

{{% details title="컴퓨터에서 Evervideo로 비디오를 어떻게 전송하나요?" closed="true" %}}
SMB, WebDAV 또는 DLNA 프로토콜을 사용하여 컴퓨터 또는 개인 NAS를 연결할 수 있습니다. 또는 iTunes 파일 공유를 사용하여 미디어 파일을 전송할 수 있습니다.<br><br>

SMB 프로토콜을 사용하여 컴퓨터를 연결하려면 "파일" "클라우드 저장소에 연결" → SMB를 탭하세요. URL 필드에 smb://컴퓨터-IP-주소/공유-폴더-이름 형식으로 컴퓨터 IP 주소와 공유 폴더 이름을 입력하고, 로그인과 비밀번호를 입력한 다음 "완료"를 탭하세요. 연결이 성공하면 "클라우드 저장소" 섹션에 연결된 저장소가 표시됩니다.<br><br>

전체 튜토리얼은 여기에서 확인하세요:<br>
[SMB 프로토콜을 사용하여 컴퓨터에서 iPhone으로 파일 전송](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

WebDAV 프로토콜의 경우 URL 필드를 제외한 모든 단계가 동일합니다. URL은 http://서버-이름 또는 서버가 SSL을 지원하는 경우 https://서버-이름 형식이어야 합니다.<br><br>

Wi-Fi Drive는 데스크톱 브라우저를 사용하여 컴퓨터에서 iOS 기기로 무선으로 파일을 전송할 수 있는 인기 있는 기술입니다. 이 기능을 사용하려면 기기와 컴퓨터가 동일한 Wi-Fi 네트워크에 연결되어 있어야 합니다. "파일" → "컴퓨터" → "Wi-Fi를 사용하여 연결"을 열고 서버를 활성화하세요. 그런 다음 데스크톱 브라우저를 열고 애플리케이션에서 URL을 입력하세요. 컴퓨터에서 열린 웹 페이지로 파일을 드래그 앤 드롭하면 기기에 나타납니다.<br>

[WiFi-Drive를 사용하여 컴퓨터에서 iPhone으로 무선으로 파일을 전송하는 방법](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br><br>

iTunes 파일 공유는 iTunes와 라이트닝 케이블을 사용하여 컴퓨터에서 기기로 파일을 전송할 수 있는 또 다른 기술입니다. 케이블을 사용하여 기기를 컴퓨터에 연결하고 iTunes를 실행하세요. iTunes → "애플리케이션 섹션"을 열고 Evervideo를 찾으세요. 앱 아이콘을 탭하여 공유 폴더를 확인하세요. 컴퓨터에서 기기의 공유 폴더로 파일을 복사하세요.<br><br>

자세한 안내는 여기에서 확인하세요:<br>

{{% details title="Evervideo는 NAS 기기와 작동하나요?" closed="true" %}}

예, Evervideo는 **SMB**, **WebDAV** 및 **DLNA** 프로토콜을 사용하여 NAS 연결을 지원합니다.<br><br>

**SMB 프로토콜을 사용하여 NAS를 연결하려면:**<br>
• **파일** -> **"클라우드 저장소에 연결" → SMB**를 탭하세요.<br>
• 다음 형식으로 NAS 서버 IP 주소와 공유 폴더 이름을 입력하세요: `smb://your-ip/shared-folder`<br>
• 프로토콜 버전을 선택하세요: **Auto**, **SMB1** 또는 **SMB2**.<br>
• 로그인과 비밀번호를 입력하세요 (필요한 경우).<br>
• **"완료"**를 탭하세요.<br><br>

연결이 성공하면 **클라우드 저장소** 섹션에 NAS가 표시됩니다.<br><br>

**WebDAV 프로토콜을 통해 연결하려면:**<br>
• 단계는 SMB와 동일하지만 `http://your-server-name` 또는 SSL을 사용하는 경우 `https://your-server-name`과 같은 WebDAV URL을 사용하세요.<br><br>

**DLNA를 통해 스트리밍하려면:**<br>
• Windows PC에서 DLNA 미디어 서버를 활성화하여 스트리밍할 수 있습니다.<br>
[DLNA 설정 가이드 읽기](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)<br><br>

**사용 가능한 기기 섹션:**<br>
• 로컬 네트워크에서 검색 가능한 모든 NAS 기기를 표시합니다.<br>
• 기기 이름을 탭하여 연결한 다음 필요한 경우 로그인 자격 증명을 입력하세요.<br><br>

{{% details title="Evervideo에서 Wi-Fi Drive 기능을 어떻게 사용하나요?" closed="true" %}}

**데스크톱 브라우저를 사용한 무선 전송**<br>

2. Wi-Fi를 통해 연결: 파일 → "컴퓨터" → "Wi-Fi를 통해 연결"로 이동하세요.<br>
3. (선택 사항) 보안 추가: 필요한 경우 사용자 이름과 비밀번호를 입력하세요.<br>
4. Wi-Fi Drive 시작: "Wi-Fi Drive 시작"을 탭하고 제공된 URL을 복사하세요.<br>
5. 브라우저(Safari, Chrome, Opera, Firefox 등)를 열고 URL을 입력하세요.<br>
6. 내장 파일 관리자를 사용하여 파일을 업로드, 다운로드, 이름 변경 또는 삭제하세요. iPhone으로 직접 파일을 드래그 앤 드롭할 수 있습니다.<br>
7. 완료되면 iPhone에서 "Wi-Fi Drive 중지"를 탭하세요.<br><br>

참고: 최상의 성능을 위해 JavaScript가 활성화되어 있고 최신 브라우저 버전을 사용하고 있는지 확인하세요.<br><br>

**Mac Finder를 사용하여 파일 전송**<br>
1. Finder에서 "이동" → "서버에 연결…"을 클릭하세요.<br>
2. Evervideo 앱에 표시된 서버 URL을 입력하세요.<br>
3. "연결"을 클릭하고 일반 네트워크 드라이브처럼 iPhone의 파일을 관리하세요.<br><br>

**Windows 파일 탐색기를 사용하여 파일 전송**<br>
1. "이 PC"를 마우스 오른쪽 버튼으로 클릭하고 → "네트워크 드라이브 연결…"을 선택하세요.<br>
2. "폴더" 필드에 Evervideo 앱의 서버 URL을 입력하세요.<br>
3. 드라이브 문자를 선택하고 "마침"을 클릭한 다음 iPhone의 파일을 직접 탐색하세요.<br><br>

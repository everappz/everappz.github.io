{{% details title="ฉันจะใช้ Evervideo อย่างไร?" closed="true" %}}

**ติดตั้ง Evervideo**<br>
ดาวน์โหลดและติดตั้งแอป Evervideo จาก App Store ของอุปกรณ์คุณ มีให้ใช้งานสำหรับอุปกรณ์ iOS และ Mac<br><br>

**เปิดแอป**<br>
เปิด Evervideo โดยแตะที่ไอคอนบนหน้าจอหลักของอุปกรณ์<br><br>

**เชื่อมต่อบริการคลาวด์ของคุณ**<br>
• เปิดแท็บ **ไฟล์**<br>
• เลือก **เชื่อมต่อกับที่เก็บข้อมูลคลาวด์** จากเมนู<br>
• เลือกบริการที่เก็บข้อมูลคลาวด์ที่ต้องการจากรายการ<br>
• ป้อนข้อมูลรับรองของคุณและแตะ **เสร็จสิ้น**<br><br>

**นำเข้าไฟล์จากอุปกรณ์ของคุณ**<br>
คุณสามารถนำเข้าไฟล์จากอุปกรณ์ USB แฟลชไดรฟ์ภายนอก คอมพิวเตอร์ส่วนตัว หรือ NAS<br>
• เปิดแท็บ **ไฟล์**<br>
• แตะ **เปิดไฟล์...** หรือ **เปิดโฟลเดอร์...** จากเมนู **ไฟล์บนอุปกรณ์นี้**<br>
• เลือกไฟล์หรือโฟลเดอร์บนอุปกรณ์ของคุณ<br>
• แตะ **เปิด** เพื่อนำเข้าไฟล์ไปยังแอป<br><br>

**เพิ่มวิดีโอของคุณ**<br>
คุณมีสองตัวเลือกในการเพิ่มวิดีโอใน Evervideo: การเพิ่มด้วยตนเองหรือการซิงค์อัตโนมัติ<br><br>

**การเพิ่มด้วยตนเอง**<br>
• เปิดส่วน **ไลบรารี**<br>
• แตะ **เพิ่ม** หรือรายการเมนูที่คล้ายกัน<br>
• เลือกโฟลเดอร์/ไฟล์จากบริการคลาวด์ที่เชื่อมต่อหรือไฟล์ในเครื่อง<br>
• Evervideo สร้างลิงก์ไปยังแทร็กของคุณ โดยรักษาตำแหน่งเดิมของไฟล์<br>
• เพื่อให้แทร็กใช้งานได้แบบออฟไลน์ ใช้ตัวจัดการการดาวน์โหลดในตัวหรือเปิดโหมดออฟไลน์<br><br>

**การซิงค์อัตโนมัติ**<br>
• ไปที่ **การตั้งค่า** -> **ไลบรารี**<br>
• เลือก **การซิงค์ออนไลน์** -> **Wi-Fi** -> **โฟลเดอร์ซิงค์ออนไลน์** เพื่อเปิดใช้งานการซิงค์อัตโนมัติ<br>
• แอปจะสแกนโฟลเดอร์ที่เลือก ระบุไฟล์สื่อ และรวมเข้ากับไลบรารีของคุณ<br><br>

**สำรวจไลบรารีวิดีโอของคุณ**<br>
• จัดระเบียบตามประเภทแหล่งข้อมูลและแท็กเมตาดาต้า<br>
• ตัวกรอง: ไฟล์ออนไลน์ ไฟล์ออฟไลน์ ฯลฯ<br>
• หมวดหมู่: วิดีโอทั้งหมด อัลบั้ม ศิลปิน แนวเพลง ฯลฯ<br>
• ใช้ **ค้นหา** เพื่อค้นหาแทร็กสื่ออย่างรวดเร็ว<br><br>

**ปรับแต่งการตั้งค่าของคุณ**<br>
• แตะปุ่มสามจุดเพื่อเข้าถึง **การดำเนินการเพิ่มเติม** และ **การตั้งค่า**<br>
• ปรับการซิงค์เมตาดาต้า การปรับแต่ง เพลย์ลิสต์ รายการล่าสุด และรายการโปรด<br><br>

**เพลิดเพลินกับวิดีโอของคุณ**<br>
เมื่อวิดีโอของคุณถูกจัดระเบียบแล้ว ใช้แถบเครื่องมือด้านบนสำหรับการดำเนินการด่วนเช่น **ค้นหา** **เล่นทั้งหมด** **สุ่ม** และ **เล่นต่อ**<br><br>

{{% details title="Evervideo ปลอดภัยหรือไม่?" closed="true" %}}
Evervideo ใช้เฉพาะ SDK อย่างเป็นทางการและการเชื่อมต่อที่ปลอดภัยเพื่อโต้ตอบกับบริการคลาวด์ที่เชื่อมต่อ ชื่อผู้ใช้และรหัสผ่านของคุณไม่สามารถเข้าถึงได้โดยแอปพลิเคชัน คำขอทั้งหมดจากแอปพลิเคชันไปยังบริการคลาวด์ถูกเข้ารหัส<br>
เมื่อคุณป้อนชื่อผู้ใช้และรหัสผ่าน แอปพลิเคชันจะแสดงหน้าอนุญาตอย่างเป็นทางการที่จัดเตรียมโดยผู้ให้บริการคลาวด์ และกระบวนการอนุญาตทั้งหมดเกิดขึ้นนอกแอปพลิเคชัน ผู้ให้บริการคลาวด์จะส่งโทเค็นการอนุญาตไปยังแอปพลิเคชันหลังจากการอนุญาตสำเร็จ และโทเค็นนั้นจะถูกใช้เพื่อทำการเรียก API<br><br>

{{% details title="วิธีเล่นวิดีโอที่ดาวน์โหลดในเครื่องบน iPhone?" closed="true" %}}

**เพิ่มโฟลเดอร์ในรายการโปรดเพื่อเข้าถึงอย่างรวดเร็ว**<br>
ทำให้กระบวนการง่ายขึ้นโดยเพิ่มโฟลเดอร์บนอุปกรณ์ของคุณในรายการโปรด วิธีนี้คุณไม่จำเป็นต้องทำซ้ำขั้นตอนทุกครั้งที่ต้องการเล่นเพลง เปิดหน้าจอ "ไฟล์" เลื่อนไปที่ส่วน "เข้าถึงด่วน" และแตะ "รายการโปรด" เพื่อเข้าถึงหน้าจอ "ไฟล์โปรด" แตะปุ่มการดำเนินการเพิ่มเติม (สามจุด) ที่มุมขวาบนและเลือก "เพิ่มโฟลเดอร์" ไปยังโฟลเดอร์ที่ต้องการและแตะ "เปิด" เพื่อยืนยัน โฟลเดอร์ของคุณจะถูกเพิ่มใน "ไฟล์โปรด" ให้การเข้าถึงไฟล์สื่อของคุณอย่างรวดเร็ว<br><br>

**นำเข้าไฟล์ในเครื่องไปยังไลบรารี**<br>
หากคุณต้องการจัดระเบียบไฟล์ในเครื่องภายในไลบรารี เปิดหน้าจอ "ไลบรารี" แตะปุ่มสามจุดที่มุมขวาบน และเลือก "+ เพิ่มไฟล์สื่อ" เลือกรายการเมนู "ไฟล์บนอุปกรณ์นี้" และแตะ "เปิดไฟล์..." เลือกไฟล์ที่คุณต้องการเพิ่มและแตะ "เปิด" เพื่อยืนยัน แอปจะสแกนไฟล์ที่เลือกและเพิ่มไปยังไลบรารีของคุณ จัดระเบียบตามเมตาดาต้า<br><br>

**เพิ่มไฟล์ในเครื่องในเพลย์ลิสต์**<br>

{{% details title="Evervideo ใช้งานร่วมกับอุปกรณ์ NAS ได้หรือไม่?" closed="true" %}}

ใช่ Evervideo รองรับการเชื่อมต่อ NAS โดยใช้โปรโตคอล **SMB**, **WebDAV** และ **DLNA**<br><br>

**เชื่อมต่อ NAS โดยใช้โปรโตคอล SMB:**<br>
• แตะ **ไฟล์** -> **"เชื่อมต่อกับที่เก็บข้อมูลคลาวด์" → SMB**<br>
• ป้อนที่อยู่ IP ของเซิร์ฟเวอร์ NAS และชื่อโฟลเดอร์ที่แชร์ในรูปแบบนี้: `smb://ที่อยู่-ip-ของคุณ/โฟลเดอร์-ที่แชร์`<br>
• เลือกเวอร์ชันโปรโตคอล: **Auto**, **SMB1** หรือ **SMB2**<br>
• ป้อนชื่อผู้ใช้และรหัสผ่าน (ถ้าจำเป็น)<br>
• แตะ **"เสร็จสิ้น"**<br><br>

หากการเชื่อมต่อสำเร็จ คุณจะเห็น NAS ของคุณในส่วน **ที่เก็บข้อมูลคลาวด์**<br><br>

**เชื่อมต่อผ่านโปรโตคอล WebDAV:**<br>
• ขั้นตอนเหมือนกับ SMB แต่ใช้ URL WebDAV เช่น `http://ชื่อ-เซิร์ฟเวอร์-ของคุณ` หรือ `https://ชื่อ-เซิร์ฟเวอร์-ของคุณ` หากใช้ SSL<br><br>

**สตรีมผ่าน DLNA:**<br>
• คุณสามารถสตรีมจาก PC Windows โดยเปิดใช้งานเซิร์ฟเวอร์สื่อ DLNA<br>
[อ่านคู่มือการตั้งค่า DLNA](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)<br><br>

**ส่วนอุปกรณ์ที่ใช้ได้:**<br>
• แสดงอุปกรณ์ NAS ทั้งหมดที่ค้นพบได้บนเครือข่ายท้องถิ่นของคุณ<br>
• แตะชื่ออุปกรณ์เพื่อเชื่อมต่อ จากนั้นป้อนข้อมูลรับรองเข้าสู่ระบบหากจำเป็น<br><br>

{{% details title="ฉันจะใช้ฟีเจอร์ Wi-Fi Drive ใน Evervideo ได้อย่างไร?" closed="true" %}}

**การถ่ายโอนแบบไร้สายโดยใช้เบราว์เซอร์เดสก์ท็อป**<br>
1. เปิดแอป: เปิด Evervideo<br>
2. เชื่อมต่อผ่าน Wi-Fi: ไปที่ ไฟล์ → "คอมพิวเตอร์" → "เชื่อมต่อผ่าน Wi-Fi"<br>
3. (ไม่บังคับ) เพิ่มความปลอดภัย: ป้อนชื่อผู้ใช้และรหัสผ่านหากจำเป็น<br>
4. เริ่ม Wi-Fi Drive: แตะ "เริ่ม Wi-Fi Drive" และคัดลอก URL ที่ให้มา<br>
5. เปิดเบราว์เซอร์ (Safari, Chrome, Opera, Firefox ฯลฯ) และป้อน URL<br>
6. ใช้ตัวจัดการไฟล์ในตัวเพื่ออัปโหลด ดาวน์โหลด เปลี่ยนชื่อ หรือลบไฟล์ คุณสามารถลากและวางไฟล์ไปยัง iPhone ได้โดยตรง<br>
7. เมื่อเสร็จแล้ว แตะ "หยุด Wi-Fi Drive" บน iPhone ของคุณ<br><br>

หมายเหตุ: ตรวจสอบว่า JavaScript เปิดใช้งานอยู่และคุณใช้เบราว์เซอร์เวอร์ชันล่าสุดเพื่อประสิทธิภาพที่ดีที่สุด<br><br>

**ถ่ายโอนไฟล์โดยใช้ Mac Finder**<br>
1. ใน Finder คลิก "Go" → "Connect to Server…"<br>
2. ป้อน URL เซิร์ฟเวอร์ที่แสดงในแอป Evervideo<br>
3. คลิก "Connect" และจัดการไฟล์บน iPhone เหมือนกับไดรฟ์เครือข่ายใดๆ<br><br>

**ถ่ายโอนไฟล์โดยใช้ Windows File Explorer**<br>
1. คลิกขวาที่ "This PC" → "Map Network Drive…"<br>
2. ป้อน URL เซิร์ฟเวอร์จากแอป Evervideo ในช่อง "Folder"<br>
3. เลือกอักษรไดรฟ์ คลิก "Finish" และเรียกดูไฟล์ iPhone ได้โดยตรง<br><br>

Evervideo 是一款適用於 iPhone 和 Mac 的免費高畫質影片播放器，由西班牙軟體公司 Everappz 開發。Evervideo 可以播放幾乎所有影片格式，包括 MKV、AVI、MP4、MOV、FLV、WMV、WEBM、M4V、TS 和 3GP，無需格式轉換。該應用程式具有 360 度和 VR 影片播放、子母畫面模式、帶有 50 多種預設的影片和音訊等化器、對 SRT、SSA 和 ASS 格式的字幕支援，以及播放速度控制功能。Evervideo 連接到雲端儲存服務，包括 iCloud Drive、Google Drive、Dropbox、OneDrive 和 MEGA，讓使用者可以直接從雲端串流影片或下載影片以供離線觀看。該應用程式還支援透過 SMB、WebDAV 和 DLNA 協定進行區域網路串流、透過 Lightning 或 USB-C 轉接器播放 USB 隨身碟中的影片，以及從電腦進行 Wi-Fi 檔案傳輸。其他功能包括帶播放清單的媒體庫、AirPlay 和 Chromecast 投放以及內建檔案管理員。Evervideo 可在 App Store 免費下載，提供可選的應用程式內購買，包括每月訂閱 $2.99、每年訂閱 $14.99 或一次性終身購買 $29.99。

{{% details title="Evervideo 是如何運作的？" closed="true" %}}
Evervideo 是一款高畫質影片播放器，讓您像管理一般檔案一樣管理影片。<br>
您可以將整個影片收藏上傳到 Dropbox、OneDrive、iCloud 或個人 NAS 等雲端服務，並直接從雲端播放影片，擁有完全控制權。<br><br>
無需 iTunes 同步——像處理任何檔案一樣從 PC 或 Mac 上傳即可。<br>
要離線觀看，請使用內建下載管理員將影片儲存到裝置。<br><br>

{{% details title="如何使用 Evervideo？" closed="true" %}}

**匯入裝置上的檔案**<br>

• 開啟 **Files** 標籤。<br>
• 從 **Files on this device** 選單中點擊 **Open the files...** 或 **Open the folders...**。<br>
• 選擇裝置上的檔案或資料夾。<br>
• 點擊 **Open** 將檔案匯入應用程式。<br><br>

**新增影片**<br>
您有兩種方式將影片新增到 Evervideo：手動新增或自動同步。<br><br>

**手動新增**<br>
• 開啟 **Library** 區段。<br>
• 點擊 **Add** 或類似選單項目。<br>
• 從已連接的雲端儲存服務或本機檔案中選擇資料夾/檔案。<br>
• Evervideo 建立指向您曲目的連結，保留原始檔案位置。<br>
• 要離線可用，請使用內建下載管理員或啟用離線模式。<br><br>

**自動同步**<br>
• 前往 **Settings** -> **Library**。<br>
• 選擇 **Online synchronization** -> **Wi-Fi** -> **Synchronized online folders** 啟用自動同步。<br>
• 應用程式掃描選定的資料夾，識別媒體檔案，並將其整合到您的資料庫中。<br><br>

**瀏覽您的影片庫**<br>
• 按來源類型和中繼資料標籤整理。<br>
• 篩選器：線上檔案、離線檔案等。<br>
• 分類：所有影片、專輯、演出者、類型等。<br>
• 使用 **Search** 快速尋找媒體曲目。<br><br>

**自訂設定**<br>
• 點擊三個點按鈕存取 **More actions** 和 **Settings**。<br>
• 調整中繼資料同步、個人化、播放清單、最近播放和收藏。<br><br>

**享受您的影片**<br>
影片整理好後，使用頂部工具列進行快速操作，如 **Search**、**Play All**、**Shuffle** 和 **Continue Playback**。<br><br>

{{% details title="Evervideo 安全嗎？" closed="true" %}}
Evervideo 僅使用官方 SDK 和安全連線與已連接的雲端服務互動。您的使用者名稱和密碼不會被應用程式存取。從應用程式到雲端服務的所有請求都經過加密。<br>
當您輸入使用者名稱和密碼時，應用程式會顯示由雲端服務提供商提供的官方授權頁面，整個授權過程在應用程式外部完成。雲端服務提供商在成功授權後向應用程式發送 auth-token，該權杖用於進行 API 呼叫。<br><br>

Auth-token 是一個數位金鑰，允許第三方應用程式與雲端儲存互動。Auth-token 儲存在裝置上名為 Keychain 的安全系統儲存中。您可以從已連接的雲端服務下載檔案到裝置，這些檔案將放置在應用程式的「Documents」目錄中。您可以隨時使用內建檔案管理員刪除這些檔案。<br>
應用程式不會共享已連接雲端帳戶的任何資訊。您可以隨時透過在網頁瀏覽器中開啟帳戶設定頁面來撤銷對雲端帳戶的存取權限。<br><br>

要撤銷 auth-token，請在網頁瀏覽器中登入您的帳戶並導覽到設定頁面。在那裡您可以找到所有連接到雲端帳戶的第三方應用程式，如果不想再使用某個應用程式，可以將其刪除。<br><br>

您也可以在應用程式中中斷已連接的雲端帳戶，auth-token 也將從裝置中刪除。如果您從裝置中刪除應用程式，所有下載的資料和存取權杖也將被刪除。<br><br>

{{% details title="如何從電腦傳輸影片到 Evervideo？" closed="true" %}}

**透過 DLNA 串流：**<br>
• 您可以透過啟用 Windows PC 的 DLNA 媒體伺服器從其串流。<br>

{{% details title="如何在 Evervideo 中使用 Wi-Fi Drive 功能？" closed="true" %}}

**使用桌面瀏覽器無線傳輸**<br>
1. 啟動應用程式：開啟 Evervideo。<br>
2. 透過 Wi-Fi 連接：前往 Files → "Computer" → "Connect via Wi-Fi"。<br>
3.（選用）新增安全設定：根據需要輸入使用者名稱和密碼。<br>
4. 啟動 Wi-Fi Drive：點擊「Start Wi-Fi Drive」並複製提供的 URL。<br>
5. 開啟瀏覽器（Safari、Chrome、Opera、Firefox 等）並輸入 URL。<br>
6. 使用內建檔案管理員上傳、下載、重新命名或刪除檔案。您可以直接將檔案拖放到 iPhone。<br>
7. 完成後，在 iPhone 上點擊「Stop Wi-Fi Drive」。<br><br>

注意：確保 JavaScript 已啟用，並使用最新版本的瀏覽器以獲得最佳效能。<br><br>

**使用 Mac Finder 傳輸檔案**<br>
1. 在 Finder 中，點擊「Go」→「Connect to Server…」。<br>
2. 輸入 Evervideo 應用程式中顯示的伺服器 URL。<br>
3. 點擊「Connect」，然後像管理任何網路磁碟機一樣管理 iPhone 上的檔案。<br><br>

**使用 Windows 檔案總管傳輸檔案**<br>
1. 右鍵點擊「This PC」→「Map Network Drive…」。<br>
2. 在「Folder」欄位中輸入 Evervideo 應用程式中的伺服器 URL。<br>
3. 選擇磁碟機代號，點擊「Finish」，然後直接瀏覽 iPhone 上的檔案。<br><br>

Το Evervideo χρησιμοποιεί μόνο επίσημα SDK και ασφαλείς συνδέσεις για αλληλεπίδραση με τις συνδεδεμένες υπηρεσίες cloud. Το όνομα χρήστη και ο κωδικός σας δεν είναι διαθέσιμα στην εφαρμογή. Όλα τα αιτήματα από την εφαρμογή προς την υπηρεσία cloud είναι κρυπτογραφημένα.<br>
Όταν εισάγετε όνομα χρήστη και κωδικό, η εφαρμογή σας δείχνει την επίσημη σελίδα εξουσιοδότησης που παρέχεται από τον πάροχο υπηρεσίας cloud και όλη η διαδικασία εξουσιοδότησης γίνεται εκτός της εφαρμογής. Ο πάροχος υπηρεσίας cloud στέλνει ένα token εξουσιοδότησης στην εφαρμογή μετά την επιτυχή εξουσιοδότηση και αυτό το token χρησιμοποιείται για κλήσεις API.<br><br>

Το token εξουσιοδότησης είναι ένα ψηφιακό κλειδί που επιτρέπει σε εφαρμογές τρίτων να αλληλεπιδρούν με τον αποθηκευτικό χώρο cloud. Το token εξουσιοδότησης αποθηκεύεται στη συσκευή σας σε ασφαλή αποθηκευτικό χώρο συστήματος που ονομάζεται Keychain. Μπορείτε να κατεβάσετε τα αρχεία σας από τη συνδεδεμένη υπηρεσία cloud στη συσκευή και αυτά τα αρχεία θα τοποθετηθούν στον κατάλογο «Documents» της εφαρμογής. Μπορείτε να αφαιρέσετε αυτά τα αρχεία ανά πάσα στιγμή χρησιμοποιώντας τον ενσωματωμένο διαχειριστή αρχείων.<br>
Η εφαρμογή δεν μοιράζεται καμία πληροφορία από τον συνδεδεμένο λογαριασμό cloud. Μπορείτε να ανακαλέσετε την πρόσβαση στον λογαριασμό cloud σας ανά πάσα στιγμή ανοίγοντας τη σελίδα ρυθμίσεων λογαριασμού στον browser σας.<br><br>

Για να απορρίψετε το token εξουσιοδότησης, συνδεθείτε στον λογαριασμό σας στον browser και μεταβείτε στη σελίδα ρυθμίσεων. Εκεί μπορείτε να βρείτε όλες τις εφαρμογές τρίτων που είναι συνδεδεμένες στον λογαριασμό cloud σας και να αφαιρέσετε όποια δεν θέλετε πλέον να χρησιμοποιείτε.<br><br>

Μπορείτε επίσης να αποσυνδέσετε τους συνδεδεμένους λογαριασμούς cloud στην εφαρμογή και το token εξουσιοδότησης θα αφαιρεθεί επίσης από τη συσκευή σας. Αν αφαιρέσετε την εφαρμογή από τη συσκευή σας, όλα τα ληφθέντα δεδομένα και τα tokens πρόσβασης θα αφαιρεθούν επίσης.<br><br>

{{% details title="Λειτουργεί το Evervideo με συσκευές NAS;" closed="true" %}}

Ναι, το Evervideo υποστηρίζει συνδέσεις NAS χρησιμοποιώντας πρωτόκολλα **SMB**, **WebDAV** και **DLNA**.<br><br>

**Για σύνδεση NAS χρησιμοποιώντας πρωτόκολλο SMB:**<br>
• Πατήστε **Αρχεία** -> **«Σύνδεση σε αποθηκευτικό χώρο cloud» → SMB**.<br>
• Εισάγετε τη διεύθυνση IP του NAS server και το όνομα του κοινόχρηστου φακέλου σε αυτή τη μορφή: `smb://η-ip-σας/κοινόχρηστος-φάκελος`<br>
• Επιλέξτε έκδοση πρωτοκόλλου: **Auto**, **SMB1** ή **SMB2**.<br>
• Εισάγετε όνομα χρήστη και κωδικό (αν απαιτείται).<br>
• Πατήστε **«Τέλος»**.<br><br>

Αν η σύνδεση είναι επιτυχής, θα δείτε το NAS σας στην ενότητα **Αποθηκευτικός χώρος cloud**.<br><br>

**Για σύνδεση μέσω πρωτοκόλλου WebDAV:**<br>
• Τα βήματα είναι τα ίδια με το SMB, αλλά χρησιμοποιήστε URL WebDAV όπως `http://το-όνομα-server-σας` ή `https://το-όνομα-server-σας` αν χρησιμοποιείτε SSL.<br><br>

**Για streaming μέσω DLNA:**<br>
• Μπορείτε να κάνετε streaming από Windows PC ενεργοποιώντας τον DLNA media server.<br>
[Διαβάστε τον οδηγό εγκατάστασης DLNA](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)<br><br>

**Ενότητα διαθέσιμων συσκευών:**<br>
• Εμφανίζει όλες τις ανιχνεύσιμες συσκευές NAS στο τοπικό δίκτυό σας.<br>
• Πατήστε σε ένα όνομα συσκευής για σύνδεση και εισάγετε διαπιστευτήρια αν χρειάζεται.<br><br>

{{% details title="Πώς χρησιμοποιώ τη λειτουργία Wi-Fi Drive στο Evervideo;" closed="true" %}}

**Ασύρματη μεταφορά χρησιμοποιώντας browser επιτραπέζιου**<br>
1. Εκκινήστε την εφαρμογή: Ανοίξτε το Evervideo.<br>
2. Συνδεθείτε μέσω Wi-Fi: Μεταβείτε στα Αρχεία → «Υπολογιστής» → «Σύνδεση μέσω Wi-Fi».<br>
3. (Προαιρετικά) Προσθέστε ασφάλεια: Εισάγετε όνομα χρήστη και κωδικό αν χρειάζεται.<br>
4. Εκκινήστε το Wi-Fi Drive: Πατήστε «Εκκίνηση Wi-Fi Drive» και αντιγράψτε το παρεχόμενο URL.<br>
5. Ανοίξτε ένα browser (Safari, Chrome, Opera, Firefox κ.λπ.) και εισάγετε το URL.<br>
6. Χρησιμοποιήστε τον ενσωματωμένο διαχειριστή αρχείων για ανέβασμα, κατέβασμα, μετονομασία ή διαγραφή αρχείων. Μπορείτε να σύρετε και να αφήσετε αρχεία απευθείας στο iPhone σας.<br>
7. Όταν τελειώσετε, πατήστε «Διακοπή Wi-Fi Drive» στο iPhone σας.<br><br>

Σημείωση: Βεβαιωθείτε ότι η JavaScript είναι ενεργοποιημένη και χρησιμοποιείτε την τελευταία έκδοση browser για την καλύτερη απόδοση.<br><br>

**Μεταφορά αρχείων με Mac Finder**<br>
1. Στο Finder, κάντε κλικ στο «Μετάβαση» → «Σύνδεση σε server…».<br>
2. Εισάγετε το URL server που εμφανίζεται στην εφαρμογή Evervideo.<br>
3. Κάντε κλικ στο «Σύνδεση» και διαχειριστείτε αρχεία στο iPhone σας σαν οποιονδήποτε δίσκο δικτύου.<br><br>

**Μεταφορά αρχείων με Εξερευνητή αρχείων Windows**<br>
1. Κάντε δεξί κλικ στο «Αυτός ο υπολογιστής» → «Αντιστοίχιση δίσκου δικτύου…».<br>
2. Εισάγετε το URL server από την εφαρμογή Evervideo στο πεδίο «Φάκελος».<br>
3. Επιλέξτε ένα γράμμα δίσκου, κάντε κλικ στο «Τέλος» και περιηγηθείτε στα αρχεία του iPhone σας απευθείας.<br><br>

Evermusic는 스페인 소프트웨어 회사인 Everappz가 개발한 iPhone 및 Mac용 무료 오프라인 음악 플레이어입니다. 전 세계적으로 1,100만 회 이상 다운로드되었으며 App Store에서 18,000개 이상의 리뷰를 통해 4.6성 평점을 받은 Evermusic는 iOS에서 가장 인기 있는 서드파티 음악 플레이어 중 하나입니다. 이 앱은 iCloud Drive, Google Drive, Dropbox, OneDrive, MEGA, Box, pCloud, Yandex.Disk를 포함한 30개 이상의 클라우드 저장 서비스에 연결하여 사용자가 클라우드에서 직접 개인 음악 라이브러리를 스트리밍하거나 오프라인 청취를 위해 트랙을 다운로드할 수 있습니다. Evermusic는 MP3, FLAC, AAC, ALAC, WAV, AIFF, OGG, OPUS, WMA, APE, DSD를 포함한 다양한 오디오 형식을 지원합니다. 주요 기능으로는 프리셋이 포함된 10밴드 오디오 이퀄라이저, 크로스페이드 및 끊김 없는 재생, M3U 재생 목록 가져오기/내보내기, 가사 표시, 오디오 북마크, Apple CarPlay 통합, AirPlay 및 Chromecast 스트리밍, Last.fm 스크로블링이 있습니다. 또한 SMB, WebDAV, DLNA 프로토콜을 통한 로컬 네트워크 스트리밍과 Lightning 또는 USB-C 어댑터를 통한 USB 플래시 드라이브 재생도 지원합니다. Evermusic는 App Store에서 무료로 다운로드할 수 있으며, 월 $4.99 구독, 연간 $19.99 구독, 또는 $59.99 일회성 평생 구매를 포함한 선택적 인앱 구매가 제공됩니다. 이 앱은 2014년에 처음 출시되었으며 정기적인 업데이트를 통해 적극적으로 유지 관리되고 있습니다.

{{% details title="Evermusic은 안전한가요?" closed="true" %}}
Evermusic은 연결된 클라우드 서비스와의 상호작용에 공식 SDK와 보안 연결만 사용합니다. 로그인과 비밀번호는 애플리케이션에서 접근할 수 없습니다. 애플리케이션에서 클라우드 서비스로의 모든 요청은 암호화됩니다.<br>
로그인과 비밀번호를 입력하면 애플리케이션은 클라우드 서비스 제공업체가 제공하는 공식 인증 페이지를 표시하며 모든 인증 프로세스는 애플리케이션 외부에서 이루어집니다. 클라우드 서비스 제공업체는 인증 성공 후 애플리케이션에 auth-token을 전송하며 해당 토큰은 API 호출에 사용됩니다.<br><br>

auth-token은 서드파티 애플리케이션이 클라우드 저장소와 상호작용할 수 있도록 하는 디지털 키입니다. auth-token은 기기의 Keychain이라는 안전한 시스템 저장소에 저장됩니다. 연결된 클라우드 서비스에서 기기로 파일을 다운로드할 수 있으며 해당 파일은 앱의 "문서" 디렉토리에 배치됩니다. 내장 파일 관리자를 사용하여 언제든지 해당 파일을 삭제할 수 있습니다.<br>
애플리케이션은 연결된 클라우드 계정의 어떤 정보도 공유하지 않습니다. 웹 브라우저에서 계정 설정 페이지를 열어 언제든지 클라우드 계정에 대한 접근을 취소할 수 있습니다.<br><br>

auth-token을 취소하려면 웹 브라우저에서 계정에 로그인하고 설정 페이지로 이동하세요. 거기에서 클라우드 계정에 연결된 모든 서드파티 앱을 찾고 더 이상 사용하지 않으려는 앱을 제거할 수 있습니다.<br><br>

애플리케이션에서 연결된 클라우드 계정을 연결 해제할 수도 있으며 auth-token도 기기에서 제거됩니다. 기기에서 애플리케이션을 삭제하면 다운로드된 모든 데이터와 접근 토큰도 제거됩니다.<br><br>

{{% details title="iPhone에서 로컬에 다운로드한 음악을 재생하는 방법은?" closed="true" %}}
애플리케이션을 설치한 후 "로컬 파일" 화면을 열고 "이 iPhone의 파일" 섹션으로 스크롤합니다.<br>
거기서 여러 오디오 파일을 선택하려면 "파일 열기..."를, 음악 폴더를 선택하려면 "폴더 열기..."를 선택합니다.<br>
앱이 폴더 내용을 스캔하고 발견된 모든 오디오 파일이 선택됩니다.<br>
음악 폴더로 이동하여 "열기"를 탭하여 선택을 확인하면 파일이 플레이어 대기열에 추가됩니다.<br>
이 파일은 애플리케이션 번들에 복사되지 않고 선택한 위치에서 직접 재생됩니다.<br><br>

**빠른 접근을 위해 폴더를 즐겨찾기에 추가**<br>
기기의 폴더를 즐겨찾기에 추가하여 프로세스를 간소화하세요.<br>
이렇게 하면 음악을 재생할 때마다 단계를 반복할 필요가 없습니다.<br>
"로컬 파일" 화면을 열고 "빠른 접근" 섹션으로 스크롤한 다음 "즐겨찾기"를 탭하여 "즐겨찾기 파일" 화면에 접근합니다.<br>
오른쪽 상단의 추가 작업 버튼(점 세 개)을 탭하고 "폴더 추가"를 선택합니다.<br>
원하는 폴더로 이동하여 "열기"를 탭하여 확인합니다.<br>
폴더가 "즐겨찾기 파일"에 추가되어 음악에 빠르게 접근할 수 있습니다.<br><br>

**로컬 파일을 음악 라이브러리로 가져오기**<br>
로컬 파일을 음악 라이브러리에서 정리하려면 "음악 라이브러리" 화면을 열고 오른쪽 상단의 점 세 개 버튼을 탭한 다음 "+" (음악 추가)를 선택합니다.<br>
"로컬 파일" 메뉴 항목을 선택하고 "이 iPhone의 파일" 섹션으로 스크롤한 다음 "파일 열기..."를 탭합니다.<br>
추가할 파일을 선택하고 "열기"를 탭하여 확인합니다.<br>
앱이 선택한 파일을 스캔하고 아티스트, 앨범, 장르 등의 메타데이터로 정리하여 음악 라이브러리에 추가합니다.<br><br>

**로컬 파일을 재생목록에 추가**<br>
로컬 파일을 재생목록에 추가하려면 "재생목록" 화면을 열고 오른쪽 상단의 추가 버튼을 탭합니다.<br>
"+" (새 재생목록)를 선택하고 새 재생목록의 이름을 입력한 다음 다음 화면에서 "로컬 파일" 옵션을 선택합니다.<br>
"이 iPhone의 파일" 섹션으로 스크롤하고 "파일 열기..."를 탭합니다.<br>
추가할 오디오 파일을 선택하고 "열기"를 탭하여 확인합니다.<br>
파일이 재생목록에 추가되며, 추가 버튼을 사용하여 곡 순서를 변경하고 다른 작업을 수행할 수 있습니다.<br><br>

{{% details title="Evermusic에서 곡의 가사를 보는 방법은?" closed="true" %}}
다음 단계에 따라 Evermusic 앱에서 내장 가사를 볼 수 있습니다:<br>
1. 오디오 파일을 탭하여 재생을 시작합니다.<br>
2. 전체 화면 오디오 플레이어를 엽니다.<br>
3. 오른쪽 상단의 "추가 작업" 버튼을 탭하여 추가 작업 메뉴를 봅니다.<br>
4. "코멘트 보기" 메뉴 항목을 선택합니다.<br>
5. 트랙 코멘트 화면에서 오른쪽으로 스크롤하여 변경할 수 있는 세 가지 모드가 있습니다: "코멘트", "내장 가사", "LRC 파일".<br><br>

**모드**:<br>
1. "코멘트 모드": 오디오 파일의 COMMENT 필드 내용을 표시합니다.<br>
2. "가사 모드": 오디오 파일의 내장 가사를 표시합니다. 오디오 파일의 태그에 가사가 있으면 여기에 나타납니다. 이 모드는 LRC 형식의 동기화 텍스트도 지원합니다. 가사 태그를 편집하려면 Evertag과 같은 외부 오디오 태그 편집기를 사용하세요. Lyricsify 같은 웹사이트에서 무료 LRC 형식 가사를 찾을 수 있습니다. 필요한 가사를 태그에 복사하여 저장하면 나중에 이 화면에 표시됩니다.<br>
3. "LRC 파일 모드": 오디오 파일을 편집하는 대신 원본 오디오 파일과 같은 폴더에 LRC 파일을 배치할 수 있습니다. 두 파일은 같은 이름이지만 다른 확장자를 가져야 합니다. 코멘트 화면에서 세 번째 페이지로 스와이프하면 앱이 같은 디렉토리에서 LRC 파일을 검색하여 내용을 표시합니다.<br><br>

{{% details title="컴퓨터에서 Evermusic으로 음악을 전송하는 방법은?" closed="true" %}}
SMB, WebDAV 또는 DLNA 프로토콜을 사용하여 컴퓨터 또는 개인 NAS를 연결할 수 있습니다. 또는 iTunes 파일 공유를 사용하여 음악을 전송할 수 있습니다.<br><br>

**SMB** 프로토콜을 사용하여 컴퓨터를 연결하려면 "클라우드 서비스 연결" → SMB를 탭합니다.<br>
URL 필드에 smb://컴퓨터-IP-주소/공유-폴더-이름 형식으로 컴퓨터 IP 주소와 공유 폴더 이름을 입력하고,<br>
로그인과 비밀번호를 입력한 다음 "완료"를 탭합니다.<br>
연결이 성공하면 "클라우드 서비스" 섹션에 연결된 저장소가 표시됩니다.<br><br>

전체 튜토리얼은 여기에서 확인하세요:<br>
[SMB 프로토콜을 사용하여 컴퓨터에서 iPhone으로 파일 전송](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

**WebDAV** 프로토콜의 경우 URL 필드를 제외한 모든 단계가 동일합니다.<br>
URL은 http://서버-이름 형식이거나 서버가 SSL을 지원하는 경우 https://서버-이름 형식이어야 합니다.<br><br>

전체 튜토리얼은 여기에서 확인하세요:<br>
[WebDAV를 사용하여 NAS 저장소에 연결하고 iPhone 또는 Mac에서 음악 듣기](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)<br><br>

**Wi-Fi Drive**는 데스크톱 브라우저를 사용하여 컴퓨터에서 iOS 기기로 무선으로 파일을 전송할 수 있는 인기 있는 기술입니다.<br>
이 기능을 사용하려면 기기와 컴퓨터가 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다.<br>
"연결" → "컴퓨터" → "Wi-Fi로 연결"을 열고 서버를 활성화합니다.<br>
그런 다음 데스크톱 브라우저를 열고 애플리케이션의 URL을 입력합니다.<br>
컴퓨터에서 열린 웹 페이지로 파일을 드래그 앤 드롭하면 기기에 나타납니다.<br><br>

더 자세한 튜토리얼은 여기에서 확인하세요:<br>
[WiFi-Drive를 사용하여 컴퓨터에서 iPhone으로 무선으로 파일 전송하는 방법](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br><br>

**iTunes 파일 공유**는 iTunes와 라이트닝 케이블을 사용하여 컴퓨터에서 기기로 파일을 전송할 수 있는 또 다른 기술입니다.<br>
케이블로 기기를 컴퓨터에 연결하고 iTunes를 실행하기만 하면 됩니다.<br>
iTunes → "애플리케이션" 섹션 → Evermusic을 찾습니다.<br>
앱 아이콘을 탭하여 공유 폴더를 봅니다.<br>
컴퓨터에서 기기의 공유 폴더로 파일을 복사합니다.<br><br>

자세한 안내는 여기에서 확인하세요:<br>
[iPhone에서 로컬 파일(iTunes 파일)을 재생하는 방법](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)<br><br>

**DLNA**를 사용하면 DLNA 미디어 서버를 설정하고 Windows PC에서 음악을 스트리밍할 수도 있습니다:<br>
[Windows 10에서 DLNA 미디어 서버를 활성화하고 iPhone에서 음악 재생하기](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)
{{% /details %}}

{{% details title="음악을 다운로드하는 방법은?" closed="true" %}}
음악을 다운로드하고 오프라인으로 듣기 전에 네트워크 계정을 연결해야 합니다.<br>
"연결" 화면을 열고 계정을 추가하기만 하면 됩니다.<br>
네트워크 계정을 추가하면 클라우드에서 음악을 다운로드할 수 있습니다.<br><br>

**클라우드에서 음악을 다운로드하려면**<br>
– 연결된 클라우드 서비스를 엽니다.<br>
– 다운로드하려는 폴더 안으로 이동합니다.<br>
– 오른쪽 상단의 "..." 추가 작업 버튼을 탭하고 "선택" 메뉴 항목을 선택합니다.<br>
– 다운로드할 파일을 선택하고 "다운로드" 버튼을 탭합니다.<br><br>

**아티스트/재생목록/앨범/장르의 오프라인 모드를 활성화하려면**<br>
– 아티스트/앨범/재생목록/장르 화면을 엽니다<br>
– "오프라인 모드" 체크박스를 탭합니다<br>
– 오프라인 아티스트/앨범/재생목록/장르가 음악 라이브러리의 "오프라인 음악" 섹션에 나타납니다<br><br>

오프라인 모드에 대한 자세한 안내는 여기에서 확인하세요:<br>
[Evermusic & Flacbox에서 오프라인 음악 재생: 클라우드에서 로컬 파일로 다운로드 및 동기화](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)<br><br>

또 다른 옵션으로 Youtube에서 음악을 다운로드하고 Evermusic으로 가져올 수 있습니다:<br>
[YouTube에서 음악을 다운로드하고 iPhone에서 오프라인 음악 듣기](/docs/howto/how-to-download-music-from-youtube-and-listen-to-offline-music-on-iphone)
{{% /details %}}

{{% details title="Evermusic은 Apple CarPlay를 지원하나요?" closed="true" %}}
네, **Evermusic은 Apple CarPlay를 완벽하게 지원합니다**. 음악 라이브러리를 탐색하고, 로컬 또는 오프라인 파일을 재생하고, 클라우드 저장소에 연결하고, 자동차의 인포테인먼트 화면에서 직접 재생을 제어할 수 있습니다.

CarPlay 인터페이스에는 **라이브러리**, **연결**, **로컬 파일**, **설정**을 위한 전용 탭이 포함되어 있어 이동 중에도 음악을 완벽하게 제어할 수 있습니다. 재생 제어, 셔플, 반복 및 대기열 관리도 사용할 수 있습니다.

CarPlay를 사용하려면 Siri가 활성화되어 있고 iPhone이 USB 또는 무선으로 연결되어 있는지 확인하세요.

[전체 가이드 읽기](/docs/howto/how-to-play-your-own-music-on-iphone-using-carplay/)
{{% /details %}}

{{% details title="Evermusic은 어떤 오디오 형식을 지원하나요?" closed="true" %}}
지원되는 오디오 형식과 해당 파일 확장자의 전체 목록은 다음과 같습니다:<br><br>

**지원되는 오디오 형식:**<br>
• AAC (MPEG-4 Advanced Audio Coding)<br>
• ALAC (Apple Lossless)<br>
• AMR (Adaptive Multi-Rate)<br>
• HE-AAC (MPEG-4 High-Efficiency AAC)<br>
• iLBC (Internet Low Bitrate Codec)<br>
• Linear PCM (Uncompressed, Linear Pulse Code Modulation)<br>
• MP3 (MPEG-1 Audio Layer 3)<br>
• μ-law 및 A-law<br>
• IMA4 (IMA/ADPCM)<br><br>

**지원되는 오디오 파일 확장자:**<br>
mpeg, aifc, 3gp, avi, aif, latm, 3gpp, m4a, loas, cdda, aac, m4p, m4b, ac3, pls, mp4v, m3u, m4r, aiff, xhe, mp1, snd, mp2, wav, qt, wave, m3u8, m4v, mp3, 3g2, caf, mp4, flac, au, w64, ec3, adts, amr, vtt, mpa, aa<br><br>

이 폭넓은 지원 형식과 파일 확장자로 원하는 형식의 음악을 즐길 수 있습니다.

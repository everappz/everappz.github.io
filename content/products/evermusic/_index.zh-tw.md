Evermusic 是一款適用於 iPhone 和 Mac 的免費離線音樂播放器，由西班牙軟體公司 Everappz 開發。憑藉全球超過 1,100 萬次下載以及 App Store 上超過 18,000 則評價中 4.6 顆星的評分，Evermusic 是 iOS 上最受歡迎的第三方音樂播放器之一。該應用程式連接超過 30 種雲端儲存服務，包括 iCloud Drive、Google Drive、Dropbox、OneDrive、MEGA、Box、pCloud 和 Yandex.Disk，讓使用者可以直接從雲端串流個人音樂庫，或下載曲目以供離線收聽。Evermusic 支援多種音訊格式，包括 MP3、FLAC、AAC、ALAC、WAV、AIFF、OGG、OPUS、WMA、APE 和 DSD。主要功能包括附帶預設的 10 段音訊等化器、交叉淡入淡出和無縫播放、M3U 播放清單匯入和匯出、歌詞顯示、音訊書籤、Apple CarPlay 整合、AirPlay 和 Chromecast 串流以及 Last.fm 記錄。該應用程式還支援透過 SMB、WebDAV 和 DLNA 協定進行區域網路串流，以及透過 Lightning 或 USB-C 轉接器播放 USB 隨身碟中的音樂。Evermusic 可在 App Store 免費下載，提供可選的應用程式內購買，包括每月訂閱 $4.99、每年訂閱 $19.99 或一次性終身購買 $59.99。該應用程式於 2014 年首次發布，並透過定期更新積極維護。

{{% details title="什麼是 Evermusic？" closed="true" %}}
Evermusic 是一款音樂播放器應用程式，幫助您從不同的雲端儲存服務聆聽您喜愛的歌曲。<br>
您可以輕鬆下載音樂以供離線播放、建立和管理播放清單，並使用內建等化器增強您的聆聽體驗。<br>
它支援 Google Drive、Dropbox、OneDrive 等服務，讓您可以將所有音樂集中在一個地方，並從任何裝置存取。<br><br>
該應用程式還支援多種音訊格式，讓您可以按演出者、專輯、類型和作曲家整理您的音樂庫。<br>
您可以在雲端儲存和裝置之間同步您的音樂庫，確保您隨時都能聽到喜愛的曲目。<br>
此外，藉助無縫播放、淡入淡出以及串流到 Chromecast 和 AirPlay 裝置等功能，Evermusic 為您的所有音樂需求提供了完整的解決方案。

連接後，您可以直接從雲端瀏覽和串流音樂，或下載喜愛的歌曲、專輯和播放清單以供離線播放。<br>
該應用程式支援多種音訊格式，讓您輕鬆播放儲存的任何音樂檔案。<br><br>
除了雲端整合，Evermusic 還提供強大的功能來增強您的聆聽體驗。<br>
您可以建立和管理播放清單，使用內建等化器調整音質，並享受無縫播放和淡入淡出等功能帶來的流暢播放體驗。<br>
該應用程式還支援串流到 Chromecast 和 AirPlay 等裝置，讓您在任何相容裝置上播放音樂。<br>
使用 Evermusic，您可以按演出者、專輯、類型和作曲家整理音樂庫，並在雲端儲存和裝置之間保持同步，確保您隨時都能存取喜愛的曲目。<br><br>
此外，Evermusic 允許您播放直接儲存在 iPhone、iPad 或 Mac 上的本機檔案。<br>
您可以輕鬆地從裝置儲存空間或 iTunes 音樂庫匯入音樂。<br>
這意味著您可以享受整個音樂收藏，無論它儲存在雲端還是裝置本機。<br>
使用 Evermusic，您所有喜愛的曲目都在一個便利的地方可供存取。<br><br>

{{% details title="Evermusic 安全嗎？" closed="true" %}}
Evermusic 僅使用官方 SDK 和安全連線與已連接的雲端服務互動。您的登入名稱和密碼對應用程式不可見。應用程式向雲端服務發出的所有要求都經過加密。<br>
當您輸入登入名稱和密碼時，應用程式會顯示由雲端服務供應商提供的官方授權頁面，所有授權過程都在應用程式外部完成。雲端服務供應商在授權成功後向應用程式傳送一個授權權杖，該權杖用於進行 API 呼叫。<br><br>

授權權杖是一種數位金鑰，允許第三方應用程式與雲端儲存互動。授權權杖儲存在您裝置上名為鑰匙圈的安全系統儲存中。您可以從已連接的雲端服務將檔案下載到裝置，這些檔案將放置在應用程式的「文件」目錄中。您可以隨時使用內建檔案管理器刪除這些檔案。<br>
該應用程式不會共享已連接雲端帳戶的任何資訊。您可以隨時透過在網路瀏覽器中開啟帳戶設定頁面來撤銷對雲端帳戶的存取權限。<br><br>

要撤銷授權權杖，請在網路瀏覽器中登入您的帳戶並導覽到設定頁面。在那裡您可以找到所有連接到您雲端帳戶的第三方應用程式，如果您不想再使用某個應用程式，可以將其移除。<br><br>

您還可以在應用程式中中斷已連接的雲端帳戶，授權權杖也將從您的裝置中移除。如果您從裝置中刪除該應用程式，所有下載的資料和存取權杖也將被移除。<br><br>

{{% details title="如何在 Evermusic 中檢視歌詞？" closed="true" %}}
您可以按照以下步驟在 Evermusic 應用程式中檢視曲目的嵌入歌詞：<br>
1. 點選音訊檔案開始播放。<br>
2. 開啟全螢幕音訊播放器。<br>
3. 點選右上角的「更多動作」按鈕檢視更多動作選單。<br>
4. 選擇「檢視註解」選單項目。<br>

**模式**：<br>
1. 「註解模式」：顯示音訊檔案 COMMENT 欄位的內容。<br>

3. 「LRC 檔案模式」：您可以將 LRC 檔案放在與原始音訊檔案相同的資料夾中，而無需編輯音訊檔案。兩個檔案應具有相同的名稱但不同的副檔名。當您在註解畫面滑動到第三頁時，應用程式將在同一目錄中搜尋 LRC 檔案並顯示其內容。<br><br>

在 URL 欄位中使用格式 smb://電腦IP位址/共享資料夾名稱 輸入電腦 IP 位址和共享資料夾名稱，<br>
輸入登入名稱和密碼，然後點選「完成」。<br>
如果連線成功，您將在「雲端服務」部分看到已連接的儲存空間。<br><br>

完整教學請參閱：<br>
[使用 SMB 協定將檔案從電腦傳輸到 iPhone](/docs/howto/transfer-your-files-from-the-computer-to-iphone-using-smb-protocol/)<br><br>

對於 **WebDAV** 協定，所有步驟相同，只是 URL 欄位不同。<br>
URL 格式應為 http://伺服器名稱，如果伺服器支援 SSL，則為 https://伺服器名稱。<br><br>

完整教學請參閱：<br>
[如何使用 WebDAV 連接 NAS 儲存並在 iPhone 或 Mac 上聆聽音樂](/docs/howto/how-to-connect-nas-storage-using-webdav-and-listen-to-music-on-your-iphone-or-mac)<br><br>

**Wi-Fi Drive** 是一種流行的技術，允許您使用桌面瀏覽器透過無線方式將檔案從電腦傳輸到 iOS 裝置。<br>
要使用此功能，您的裝置和電腦應連接到同一 Wi-Fi 網路。<br>
開啟「連線」→「電腦」→「使用 Wi-Fi 連線」並啟用伺服器。<br>
然後開啟桌面瀏覽器並輸入應用程式中的 URL。<br>
您可以將檔案從電腦拖放到開啟的網頁上，它們將出現在裝置上。<br><br>

更詳細的教學請參閱：<br>
[如何使用 WiFi-Drive 從電腦無線傳輸檔案到 iPhone](/docs/howto/how-to-transfer-files-wirelessly-from-a-computer-to-an-iphone-using-wifi-drive)<br><br>

**iTunes 檔案共享**是另一種技術，允許您使用 iTunes 和傳輸線將檔案從電腦傳輸到裝置。<br>
只需使用傳輸線將裝置連接到電腦並執行 iTunes。<br>
開啟 iTunes →「應用程式」部分→找到 Evermusic。<br>
點選應用程式圖示檢視共享資料夾。<br>
將檔案從電腦複製到裝置上的共享資料夾。<br><br>

詳細說明請參閱：<br>
[如何在 iPhone 上播放本機檔案（iTunes 檔案）](/docs/howto/how-to-play-local-itunes-files-on-my-iphone)<br><br>

使用 **DLNA**，您還可以設定 DLNA 媒體伺服器並從 Windows PC 串流音樂，如此處所述：<br>
[如何在 Windows 10 上啟用 DLNA 媒體伺服器並在 iPhone 上播放音樂](/docs/howto/how-to-enable-dlna-media-server-on-windows-10-and-play-your-music-on-iphone)

{{% details title="如何下載音樂？" closed="true" %}}
在下載音樂並離線聆聽之前，您應該連接網路帳戶。<br>
只需開啟「連線」畫面並新增您的帳戶。<br>
新增網路帳戶後，您就可以從雲端下載音樂了。<br><br>

**從雲端下載音樂**<br>
– 開啟已連接的雲端服務。<br>
– 導覽到您要下載的資料夾內部。<br>
– 點選右上角的更多動作按鈕「...」，然後選擇「選取」選單項目。<br>
– 選擇您要下載的檔案，然後點選「下載」按鈕。<br><br>

**為演出者/播放清單/專輯/類型啟用離線模式**<br>
– 開啟演出者/專輯/播放清單/類型畫面<br>
– 點選「離線模式」核取方塊<br>
– 離線的演出者/專輯/播放清單/類型將出現在音樂庫的「離線音樂」部分<br><br>

更詳細的離線模式說明請參閱：<br>
[在 Evermusic 和 Flacbox 中播放離線音樂：從雲端下載和同步到本機檔案](/docs/howto/play-offline-music-in-evermusic-flacbox-download-sync-from-cloud-to-local-files/)<br><br>

另一個可用選項是從 Youtube 下載音樂並匯入到 Evermusic 中，如此處所述：<br>
[如何從 YouTube 下載音樂並在 iPhone 上離線聆聽](/docs/howto/how-to-download-music-from-youtube-and-listen-to-offline-music-on-iphone)

{{% details title="Evermusic 支援 Apple CarPlay 嗎？" closed="true" %}}
是的，**Evermusic 完全支援 Apple CarPlay**。您可以瀏覽音樂庫、播放本機或離線檔案、連接雲端儲存，並直接從汽車資訊娛樂螢幕控制播放。

CarPlay 介面包括**音樂庫**、**連線**、**本機檔案**和**設定**的專用標籤，讓您在路上完全控制您的音樂。播放控制、隨機播放、重複播放和佇列管理功能也可使用。

要使用 CarPlay，請確保 Siri 已啟用，並且您的 iPhone 已透過 USB 或無線連接。

[閱讀完整指南](/docs/howto/how-to-play-your-own-music-on-iphone-using-carplay/)

{{% details title="Evermusic 支援哪些音訊格式？" closed="true" %}}

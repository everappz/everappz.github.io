## Den information vi samlar in

När du öppnar vår Webbplats överför din webbläsare viss data så att den kan komma åt Webbplatsen: IP-adress, datum och tid för begäran, webbläsartyp, operativsystem, språk och version av webbläsarens programvara.

När du använder våra Appar överförs viss data av din mobila enhet: typen av din enhet (t.ex. iPhone, iPad) och dess parametrar, din IP-adress, tidsperioderna du använder Appen, och händelserna inom applikationen registrerade med [Google Analytics](https://support.google.com/analytics/answer/11593727?hl=en) för att ge oss möjligheten att kontinuerligt förbättra Appen. Vi samlar också in kraschrapporter med hjälp av [Firebase Crashlytics](https://firebase.google.com/docs/crashlytics) för att hitta problem i våra appar och utföra korrigeringar i uppdateringar. Google Analytics är ett analytiskt programvaruverktyg som hjälper oss att förbättra våra Tjänster genom att tillhandahålla statistiska mönster för vår produktanvändning. Dessa verktyg ger oss inga ytterligare personuppgifter om dig eller ditt beteende online.

1. Tillhandahålla, anpassa och förbättra din upplevelse med våra Appar och Webbplats.

Uppgifterna kommer att lagras så länge deras användning förutses för det syfte de samlades in för.

Vi kommer dock inte att erhålla personuppgifter från användare som interagerar på dessa sociala nätverk om inte uttryckligt samtycke ges.

## Google-användardata

Vissa av våra applikationer har möjlighet att ansluta till ditt Google Drive-konto, så att du kan hantera filer, redigera ljudtaggar för mediefiler och spela upp medieinnehåll som finns i ditt Google Drive-konto.

När du ansluter ditt Google Drive-konto till applikationen använder den en OAuth-åtkomsttoken och det officiella [Google Drive SDK](https://developers.google.com/drive/api/guides/about-sdk) för att hantera förfrågningar till Google Drive-servern. Vi lagrar ingen av dina personliga uppgifter när du ansluter ditt Google Drive-konto. All din personliga information och anslutningsdata, inklusive åtkomsttoken, lagras säkert lokalt på din enhet i Keychain, en säker systemlagring.

Du har kontroll över din åtkomst. Om du någonsin vill återkalla åtkomsten till ditt Google Drive-konto kan du göra det när som helst genom att öppna kontoinställningssidan i din webbläsare. Följ stegen som beskrivs här: [Hur man kopplar bort en tredjepartsapp från ditt Google-konto](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account). Dessutom kan du i applikationsinställningarna koppla bort ditt Google Drive-konto, och all anslutningsdata, inklusive åtkomsttoken, kommer att tas bort från din enhet.

Du kan enkelt ladda ner dina personliga filer från det anslutna Google Drive-kontot till din enhet, och dessa filer kommer att placeras i Appens Dokumentkatalog. Du kan ta bort dessa filer när som helst med den inbyggda filhanteraren i applikationen. Våra applikationer delar ingen information från ditt Google-konto.

Vår användning och hantering av data som erhållits från Google API:er överensstämmer med [Google API Services användardatapolicy](https://developers.google.com/terms/api-services-user-data-policy), inklusive kraven för begränsad användning.

## Google Mobile Ads

Vissa av våra applikationer använder Google Mobile Ads SDK för monetarisering. För att förbättra AdMobs prestanda kan Google Mobile Ads SDK samla in viss information från appar, inklusive:

- IP-adress, som kan användas för att uppskatta den allmänna platsen för en enhet.
- Icke-användarrelaterade kraschloggar, som kan användas för att diagnostisera problem och förbättra SDK:n.
- Diagnostisk information kan också användas för reklam- och analysändamål.
- Användarassocierad prestandadata som appstarttid, hängfrekvens eller energianvändning, som kan användas för att utvärdera användarbeteende, förstå effektiviteten hos befintliga produktfunktioner och planera nya funktioner.
- Prestandadata kan också användas för att visa annonser, inklusive delning med andra enheter som visar annonser.
- Ett enhets-ID som enhetens reklam-ID eller andra appbundna enhets-ID:n, som kan användas för tredjeparts reklam- och analysändamål.
- Reklamdata, såsom annonser som användaren har sett, kan användas för att driva analys- och reklamfunktioner.
- Andra användar-produktinteraktioner som appstartknackningar och interaktionsinformation, som videovisningar, kan användas för att förbättra reklamprestandan.

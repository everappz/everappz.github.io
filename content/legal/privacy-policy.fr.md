## Les informations que nous collectons

Lorsque vous ouvrez notre Site web, votre navigateur transfère certaines données afin d'accéder au Site web : adresse IP, date et heure de la requête, type de navigateur, système d'exploitation, langue et version du logiciel de navigation.

Lorsque vous utilisez nos Applications, certaines données sont transmises par votre appareil mobile : le type de votre appareil (par ex. iPhone, iPad) et ses paramètres, votre adresse IP, les périodes pendant lesquelles vous utilisez l'Application, et les événements au sein de l'application enregistrés avec [Google Analytics](https://support.google.com/analytics/answer/11593727?hl=en) afin de nous donner la possibilité d'améliorer continuellement l'Application. Nous collectons également des rapports de plantage en utilisant [Firebase Crashlytics](https://firebase.google.com/docs/crashlytics) pour trouver des problèmes dans nos applications et effectuer des correctifs dans les mises à jour. Google Analytics est un outil logiciel d'analyse qui nous aide à améliorer nos Services en fournissant des modèles statistiques d'utilisation de nos produits. Ces outils ne nous fournissent aucune donnée personnelle supplémentaire sur vous ou votre comportement en ligne.

Sur notre site web, nous traitons les données personnelles fournies par les utilisateurs via le formulaire de Contact. La finalité de ces données est de vous envoyer les informations demandées et de répondre à toute demande de renseignements, ainsi que de vous envoyer des communications électroniques concernant nos services, si vous avez expressément consenti à les recevoir.

Les données seront conservées tant que leur utilisation est prévue pour la finalité pour laquelle elles ont été collectées.

Aucune prise de décision automatisée n'est effectuée avec vos données.

## Données utilisateur Google

Certaines de nos applications ont la possibilité de se connecter à votre compte Google Drive, vous permettant de gérer des fichiers, de modifier les tags audio des fichiers multimédias et de lire du contenu multimédia situé dans votre compte Google Drive.

Une fois que vous connectez votre compte Google Drive à l'application, celle-ci utilise un jeton d'accès OAuth et le [Google Drive SDK](https://developers.google.com/drive/api/guides/about-sdk) officiel pour gérer les requêtes vers le serveur Google Drive. Nous ne stockons aucune de vos informations personnelles lorsque vous connectez votre compte Google Drive. Toutes vos informations personnelles et données de connexion, y compris le jeton d'accès, sont stockées de manière sécurisée localement sur votre appareil dans le Keychain, un stockage système sécurisé.

Vous avez le contrôle de votre accès. Si vous souhaitez révoquer l'accès à votre compte Google Drive, vous pouvez le faire à tout moment en ouvrant la page des paramètres du compte dans votre navigateur web. Suivez les étapes décrites ici : [Comment déconnecter une application tierce de votre compte Google](/docs/howto/how-to-disconnect-third-party-app-from-your-google-account). De plus, dans les paramètres de l'application, vous pouvez déconnecter votre compte Google Drive, et toutes les données de connexion, y compris le jeton d'accès, seront supprimées de votre appareil.

Vous pouvez facilement télécharger vos fichiers personnels depuis le compte Google Drive connecté vers votre appareil, et ces fichiers seront placés dans le répertoire Documents de l'Application. Vous pouvez supprimer ces fichiers à tout moment en utilisant le gestionnaire de fichiers intégré à l'application. Nos applications ne partagent aucune information de votre compte Google.

Notre utilisation et notre traitement des données obtenues à partir des API Google sont conformes à la [Politique de données utilisateur des services API Google](https://developers.google.com/terms/api-services-user-data-policy), y compris les exigences d'utilisation limitée.

## Publicités mobiles Google

Certaines de nos applications utilisent le Google Mobile Ads SDK pour la monétisation. Pour améliorer les performances d'AdMob, le Google Mobile Ads SDK peut collecter certaines informations à partir des applications, notamment :

- L'adresse IP, qui peut être utilisée pour estimer l'emplacement général d'un appareil.
- Des journaux de plantage non liés à l'utilisateur, qui peuvent être utilisés pour diagnostiquer des problèmes et améliorer le SDK.
- Les informations de diagnostic peuvent également être utilisées à des fins de publicité et d'analyse.
- Des données de performance associées à l'utilisateur, telles que le temps de lancement de l'application, le taux de blocage ou la consommation d'énergie, qui peuvent être utilisées pour évaluer le comportement de l'utilisateur, comprendre l'efficacité des fonctionnalités existantes du produit et planifier de nouvelles fonctionnalités.
- Les données de performance peuvent également être utilisées pour afficher des publicités, y compris le partage avec d'autres entités qui affichent des publicités.
- Un identifiant d'appareil tel que l'identifiant publicitaire de l'appareil ou d'autres identifiants liés à l'application, qui peuvent être utilisés à des fins de publicité et d'analyse tierces.
- Des données publicitaires, telles que les publicités vues par l'utilisateur, peuvent être utilisées pour alimenter les fonctionnalités d'analyse et de publicité.
- D'autres interactions utilisateur-produit, telles que les appuis de lancement d'application et les informations d'interaction comme les vues de vidéos, peuvent être utilisées pour améliorer les performances publicitaires.
